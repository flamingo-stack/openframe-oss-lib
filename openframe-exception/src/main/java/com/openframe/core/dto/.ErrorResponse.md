A standardized DTO for serializing API error responses in the OpenFrame core, supporting structured error metadata including error codes, HTTP status, request path, timestamp, and per-field validation errors.

## Key Components

| Component | Description |
|---|---|
| `ErrorResponse` | Root DTO class with `@JsonInclude(NON_NULL)` — null fields are omitted from JSON output |
| `ErrorResponse.of()` | Static factory method that creates an `ErrorResponse` from an `ErrorCode` enum with an auto-generated ISO-8601 timestamp |
| `ErrorResponseBuilder` | Fluent builder for constructing full `ErrorResponse` instances |
| `FieldError` | Nested DTO capturing per-field validation failures (`field` + `message`) |
| `FieldErrorBuilder` | Fluent builder for constructing `FieldError` instances |

## Fields

| Field | Type | Description |
|---|---|---|
| `code` | `String` | Application-level error code (from `ErrorCode`) |
| `message` | `String` | Human-readable error description |
| `status` | `Integer` | HTTP status code |
| `timestamp` | `String` | ISO-8601 instant of the error |
| `path` | `String` | Request path that triggered the error |
| `fieldErrors` | `List<FieldError>` | Validation errors per request field |

## Usage Example

```java
// Using the static factory with an ErrorCode
ErrorResponse response = ErrorResponse.of(ErrorCode.NOT_FOUND, "Resource not found");

// Using the builder for full control
ErrorResponse response = ErrorResponse.builder()
    .code("VALIDATION_ERROR")
    .message("Request validation failed")
    .status(400)
    .timestamp(Instant.now().toString())
    .path("/api/v1/devices")
    .fieldErrors(List.of(
        ErrorResponse.FieldError.builder()
            .field("email")
            .message("must be a valid email address")
            .build()
    ))
    .build();

// Simple message-only response
ErrorResponse response = new ErrorResponse("Unexpected error occurred");
```
