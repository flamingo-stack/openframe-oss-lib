Custom exception class for encryption-related failures within the OpenFrame core, extending `InternalException` to carry structured error codes alongside descriptive messages.

## Key Components

| Component | Description |
|-----------|-------------|
| `EncryptionException(ErrorCode, String)` | Constructs an exception with an error code and message |
| `EncryptionException(ErrorCode, String, Throwable)` | Constructs an exception with an error code, message, and root cause |

## Usage Example

```java
// Throw without a root cause
throw new EncryptionException(
    ErrorCode.ENCRYPTION_FAILED,
    "Failed to encrypt payload using AES-256"
);

// Wrap a low-level cause
try {
    cipher.doFinal(data);
} catch (BadPaddingException e) {
    throw new EncryptionException(
        ErrorCode.DECRYPTION_FAILED,
        "Decryption failed due to invalid padding",
        e
    );
}
```

> Prefer the three-argument constructor when wrapping a caught exception, so the original stack trace is preserved for diagnostics.
