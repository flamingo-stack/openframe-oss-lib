A specialized exception class for handling HTTP 403 Forbidden errors, extending `BaseException` with a fixed `HttpStatus.FORBIDDEN` status code.

## Key Components

| Constructor | Description |
|-------------|-------------|
| `ForbiddenException(String message)` | Creates a forbidden exception using the default `ErrorCode.FORBIDDEN` error code |
| `ForbiddenException(ErrorCode, String)` | Creates a forbidden exception with a custom `ErrorCode` |
| `ForbiddenException(ErrorCode, String, Throwable)` | Creates a forbidden exception with a custom `ErrorCode` and a root cause |

## Usage Example

```java
// Basic usage with default error code
throw new ForbiddenException("Access denied: insufficient permissions");

// Custom error code for specific forbidden scenarios
throw new ForbiddenException(ErrorCode.TENANT_ACCESS_DENIED, "User does not belong to this tenant");

// With root cause for debugging
throw new ForbiddenException(
    ErrorCode.ROLE_ACCESS_DENIED,
    "Admin role required for this operation",
    originalException
);
```

All constructors automatically bind `HttpStatus.FORBIDDEN` (HTTP 403), ensuring consistent error responses across the OpenFrame platform without requiring callers to specify the HTTP status manually.
