Custom exception class for NATS messaging errors within the OpenFrame core, extending `InternalException` with a fixed `INTERNAL_ERROR` error code.

## Key Components

| Component | Description |
|-----------|-------------|
| `NatsException(String message)` | Constructs exception with a descriptive message |
| `NatsException(String message, Throwable cause)` | Constructs exception with a message and a root cause for chained exceptions |

## Usage Example

```java
// Throw with message only
throw new NatsException("Failed to publish message to NATS subject");

// Throw with chained cause
try {
    natsConnection.publish(subject, payload);
} catch (Exception e) {
    throw new NatsException("NATS publish failed", e);
}
```

> All `NatsException` instances automatically use `ErrorCode.INTERNAL_ERROR`, ensuring consistent error classification across NATS-related failures in the OpenFrame platform.
