Specialized exception for API key-related failures within the OpenFrame core, extending `InternalException` with a fixed `INTERNAL_ERROR` error code.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `ApiKeyException(String message)` | Constructor | Creates an API key exception with a descriptive message |
| `ApiKeyException(String message, Throwable cause)` | Constructor | Creates an API key exception with a message and a root cause for exception chaining |

## Usage Example

```java
// Throw when an API key is missing or invalid
if (apiKey == null || apiKey.isBlank()) {
    throw new ApiKeyException("API key must not be null or empty");
}

// Throw with a root cause when key validation fails due to an underlying error
try {
    keyService.validate(apiKey);
} catch (Exception e) {
    throw new ApiKeyException("Failed to validate API key", e);
}
```

> Both constructors automatically assign `ErrorCode.INTERNAL_ERROR`, so all API key failures surface with a consistent error code through OpenFrame's exception handling pipeline.
