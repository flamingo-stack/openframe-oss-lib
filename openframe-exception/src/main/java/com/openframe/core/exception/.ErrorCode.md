Enum defining all standardized error codes used across the OpenFrame platform, pairing each error with a string code and its corresponding HTTP status code.

## Key Components

- **`code`** — String identifier for the error (e.g., `"NOT_FOUND"`, `"TICKET_INVALID_TRANSITION"`)
- **`httpStatus`** — Associated HTTP status integer (e.g., `404`, `409`, `500`)
- **`getCode()`** — Returns the string error code
- **`getHttpStatus()`** — Returns the HTTP status code

**Error categories defined:**

| Category | Examples |
|---|---|
| Generic HTTP | `NOT_FOUND`, `BAD_REQUEST`, `UNAUTHORIZED`, `FORBIDDEN` |
| Domain-specific | `USER_NOT_FOUND`, `DEVICE_NOT_FOUND`, `ENCRYPTION_FAILED` |
| External API | `ORGANIZATION_NOT_FOUND`, `EVENT_NOT_FOUND` |
| Tickets | `TICKET_NOT_FOUND`, `TICKET_INVALID_TRANSITION`, `TICKET_AI_DISABLED` |
| Authorization | `USER_ACTIVE_IN_ANOTHER_TENANT`, `OWNER_CANNOT_SWITCH_TENANT` |
| Infrastructure | `PINOT_QUERY_ERROR`, `DATABASE_ERROR` |

## Usage Example

```java
// Throwing a domain exception using an ErrorCode
if (ticket == null) {
    throw new OpenFrameException(ErrorCode.TICKET_NOT_FOUND);
}

// Accessing code and status programmatically
ErrorCode error = ErrorCode.VALIDATION_ERROR;
System.out.println(error.getCode());       // "VALIDATION_ERROR"
System.out.println(error.getHttpStatus()); // 400

// Mapping to an HTTP response
return ResponseEntity
    .status(error.getHttpStatus())
    .body(Map.of("error", error.getCode()));
```
