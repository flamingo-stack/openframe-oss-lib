Enum defining standardized error codes paired with their corresponding HTTP status codes for the OpenFrame platform's exception handling system.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `code` | `String` | Machine-readable error identifier string |
| `httpStatus` | `int` | Associated HTTP status code |
| `getCode()` | Method | Returns the string error code |
| `getHttpStatus()` | Method | Returns the HTTP status integer |

### Error Categories

| Category | Examples |
|----------|---------|
| Generic HTTP | `NOT_FOUND`, `BAD_REQUEST`, `UNAUTHORIZED`, `FORBIDDEN`, `CONFLICT`, `INTERNAL_ERROR` |
| Domain-specific | `USER_NOT_FOUND`, `DEVICE_NOT_FOUND`, `ENCRYPTION_FAILED`, `INITIAL_KEY_INVALID` |
| External API | `EVENT_NOT_FOUND`, `ORGANIZATION_NOT_FOUND`, `ORGANIZATION_HAS_MACHINES` |
| Tag | `TAG_NOT_FOUND`, `TAG_ALREADY_EXISTS` |
| Authorization | `USER_ACTIVE_IN_ANOTHER_TENANT` |
| Infrastructure | `PINOT_QUERY_ERROR`, `DATABASE_ERROR` |

## Usage Example

```java
// Throwing a domain exception using an ErrorCode
throw new OpenFrameException(ErrorCode.DEVICE_NOT_FOUND);

// Accessing code and status directly
ErrorCode error = ErrorCode.VALIDATION_ERROR;
System.out.println(error.getCode());       // "VALIDATION_ERROR"
System.out.println(error.getHttpStatus()); // 400

// Mapping to an HTTP response
ResponseEntity.status(error.getHttpStatus())
    .body(Map.of("error", error.getCode()));
```
