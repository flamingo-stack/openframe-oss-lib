Enum defining all standardized error codes used across the OpenFrame platform, pairing each error with a string identifier and its corresponding HTTP status code.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `code` | `String` | Machine-readable error identifier returned in API responses |
| `httpStatus` | `int` | HTTP status code associated with the error |
| `getCode()` | `String` | Returns the string error code |
| `getHttpStatus()` | `int` | Returns the HTTP status integer |

**Error categories:**

| Category | Examples |
|----------|---------|
| Generic HTTP | `NOT_FOUND`, `BAD_REQUEST`, `UNAUTHORIZED`, `FORBIDDEN` |
| Domain | `USER_NOT_FOUND`, `DEVICE_NOT_FOUND`, `ENCRYPTION_FAILED` |
| Tickets | `TICKET_NOT_FOUND`, `TICKET_INVALID_TRANSITION`, `TICKET_AI_DISABLED` |
| Authorization | `USER_ACTIVE_IN_ANOTHER_TENANT`, `OWNER_CANNOT_SWITCH_TENANT` |
| Infrastructure | `PINOT_QUERY_ERROR`, `DATABASE_ERROR` |

## Usage Example

```java
// Throwing a domain exception with a specific ErrorCode
if (user == null) {
    throw new OpenFrameException(ErrorCode.USER_NOT_FOUND);
}

// Reading code and status from the enum
ErrorCode error = ErrorCode.TICKET_INVALID_TRANSITION;
int status = error.getHttpStatus(); // 409
String code  = error.getCode();    // "TICKET_INVALID_TRANSITION"

// Using in an API error response builder
ApiErrorResponse response = ApiErrorResponse.builder()
    .code(ErrorCode.FORBIDDEN.getCode())
    .httpStatus(ErrorCode.FORBIDDEN.getHttpStatus())
    .build();
```
