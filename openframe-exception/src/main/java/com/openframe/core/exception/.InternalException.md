Represents a server-side internal error exception that maps to HTTP 500 responses. Extends `BaseException` to provide structured error handling for unexpected or unrecoverable failures within the OpenFrame platform.

## Key Components

| Constructor | Description |
|---|---|
| `InternalException(String message)` | Creates a 500 error with the default `INTERNAL_ERROR` error code |
| `InternalException(ErrorCode, String)` | Creates a 500 error with a custom `ErrorCode` and message |
| `InternalException(ErrorCode, String, Throwable)` | Creates a 500 error with a custom code, message, and root cause |

## Usage Example

```java
// Simple internal error
throw new InternalException("Unexpected failure during data processing");

// With a specific error code
throw new InternalException(ErrorCode.DATABASE_FAILURE, "Unable to connect to database");

// With root cause for stack trace propagation
try {
    externalService.call();
} catch (Exception e) {
    throw new InternalException(ErrorCode.SERVICE_UNAVAILABLE, "External service failed", e);
}
```

## Notes

- Always maps to `HttpStatus.INTERNAL_SERVER_ERROR` (HTTP 500)
- Use the `Throwable` constructor when wrapping lower-level exceptions to preserve the original stack trace
- Prefer specific `ErrorCode` values over the default `INTERNAL_ERROR` when the failure type is known, to aid in observability and debugging
