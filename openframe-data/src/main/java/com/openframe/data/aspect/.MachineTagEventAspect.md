An AOP aspect that intercepts repository save operations for Machine, MachineTag, and Tag entities and delegates processing to MachineTagEventService. Uses both @AfterReturning and @Around advice patterns with conditional activation based on configuration properties.

## Key Components

- **@Aspect**: Spring AOP aspect for cross-cutting repository event handling
- **@ConditionalOnProperty**: Configurable activation via `openframe.device.aspect.enabled` property
- **MachineTagEventService**: Injected service for processing intercepted save operations
- **afterMachineSave/afterMachineSaveAll**: @AfterReturning advice for Machine repository operations
- **afterMachineTagSave/afterMachineTagSaveAll**: @AfterReturning advice for MachineTag repository operations
- **aroundTagSave/aroundTagSaveAll**: @Around advice for Tag repository operations with state capture

## Usage Example

```java
// Configuration to enable the aspect
@Configuration
public class AspectConfig {
    // Aspect is auto-configured when property is true (default)
    // openframe.device.aspect.enabled=true
}

// The aspect automatically intercepts these repository calls:
@Service
public class MachineService {
    
    @Autowired
    private MachineRepository machineRepository;
    
    @Autowired  
    private MachineTagRepository machineTagRepository;
    
    public void updateMachine(Machine machine) {
        // This save operation is intercepted by afterMachineSave()
        machineRepository.save(machine);
    }
    
    public void bulkUpdateMachineTags(List<MachineTag> tags) {
        // This saveAll operation is intercepted by afterMachineTagSaveAll()
        machineTagRepository.saveAll(tags);
    }
}
```
