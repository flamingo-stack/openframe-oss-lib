An AOP aspect that intercepts repository save operations for Machine, MachineTag, and Tag entities, delegating processing to MachineTagEventService for cross-cutting concerns.

## Key Components

- **@Aspect**: Spring AOP component that intercepts repository operations
- **@ConditionalOnProperty**: Conditionally enabled via `openframe.device.aspect.enabled` property
- **@AfterReturning advisors**: Handle Machine and MachineTag save/saveAll operations after completion
- **@Around advisors**: Handle Tag save/saveAll operations with pre/post processing
- **MachineTagEventService**: Injected service for processing intercepted entity changes

## Usage Example

```java
// Enable the aspect in application.properties
// openframe.device.aspect.enabled=true

// The aspect automatically intercepts these repository calls:

@Autowired
private MachineRepository machineRepository;

@Autowired
private MachineTagRepository machineTagRepository;

@Autowired
private TagRepository tagRepository;

// These operations are automatically intercepted:
Machine machine = new Machine();
machineRepository.save(machine); // Triggers afterMachineSave()

List<MachineTag> tags = Arrays.asList(new MachineTag(), new MachineTag());
machineTagRepository.saveAll(tags); // Triggers afterMachineTagSaveAll()

Tag tag = new Tag();
tagRepository.save(tag); // Triggers aroundTagSave() with state capture
```

The aspect provides transparent event processing without requiring changes to existing repository usage patterns.
