Service interface for handling repository events related to machines, machine tags, and tags, with automatic Kafka message publishing for downstream processing.

## Key Components

- **processMachineSave/processMachineSaveAll**: Handles machine entity save events and publishes messages for individual machines or collections
- **processMachineTagSave/processMachineTagSaveAll**: Processes machine-tag relationship changes and triggers messaging for affected machines
- **processTagSave/processTagSaveAll**: Handles tag entity updates (specifically name changes) and publishes messages for all associated machines

## Usage Example

```java
@Service
public class MachineTagEventServiceImpl implements MachineTagEventService {
    
    @Autowired
    private KafkaProducer kafkaProducer;
    
    @Override
    public void processMachineSave(Machine machine) {
        // Fetch all tags for the machine
        List<MachineTag> machineTags = machineTagRepository.findByMachineId(machine.getId());
        
        // Create and send MachinePinotMessage
        MachinePinotMessage message = createMessage(machine, machineTags);
        kafkaProducer.send("machine-topic", message);
    }
    
    @EventListener
    public void handleMachineUpdate(Machine machine) {
        machineTagEventService.processMachineSave(machine);
    }
}
```

This service acts as an event handler that translates entity changes into Kafka messages, ensuring downstream systems stay synchronized with machine and tag relationship updates.
