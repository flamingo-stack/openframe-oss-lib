A Spring component that retrieves agent registration secrets from Fleet MDM servers for device enrollment. This service connects to configured Fleet MDM instances to obtain enrollment secrets needed for agent registration.

## Key Components

- **FleetMdmAgentRegistrationSecretRetriever** - Main component implementing `ToolAgentRegistrationSecretRetriever` interface
- **getSecret()** - Retrieves enrollment secret from Fleet MDM API using configured credentials
- **getToolId()** - Returns the static tool identifier "fleetmdm-server"
- **IntegratedToolService** - Manages tool configurations and credentials
- **ToolUrlService** - Handles API URL resolution for integrated tools
- **FleetMdmClient** - SDK client for Fleet MDM API interactions

## Usage Example

```java
@Autowired
private FleetMdmAgentRegistrationSecretRetriever secretRetriever;

// Get the tool identifier
String toolId = secretRetriever.getToolId(); // Returns "fleetmdm-server"

// Retrieve enrollment secret from Fleet MDM
String enrollSecret = secretRetriever.getSecret();

// Use secret for agent registration
agentService.registerWithSecret(enrollSecret);
```

The component is conditionally enabled via the `openframe.integration.tool.enabled=true` property and requires proper Fleet MDM tool configuration including API URL and authentication token. It handles errors gracefully by logging issues and throwing `IllegalStateException` for configuration or connectivity problems.
