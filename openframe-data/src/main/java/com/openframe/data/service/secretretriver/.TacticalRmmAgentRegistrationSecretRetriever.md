Retrieves agent registration secrets from TacticalRMM for automated agent deployment. This service integrates with TacticalRMM's API to generate installation secrets required for registering new agents.

## Key Components

- **getToolId()**: Returns the unique identifier "tactical-rmm" for this tool integration
- **getSecret()**: Main method that retrieves agent registration secret from TacticalRMM API
- **buildRequest()**: Creates a default registration request with Windows server configuration
- **TacticalRmmClient**: Injected client for TacticalRMM API communication
- **IntegratedToolService/ToolUrlService**: Services for managing tool configuration and URLs

## Usage Example

```java
@Autowired
private TacticalRmmAgentRegistrationSecretRetriever secretRetriever;

// Get registration secret for agent deployment
public String deployAgent() {
    try {
        String secret = secretRetriever.getSecret();
        // Use secret for agent installation
        return "Agent secret: " + secret;
    } catch (IllegalStateException e) {
        log.error("Failed to retrieve TacticalRMM secret: {}", e.getMessage());
        throw e;
    }
}
```

The component is conditionally enabled via the `openframe.integration.tool.enabled=true` property and creates a standard Windows server agent request with 40-hour expiration by default.
