Service component that retrieves agent registration secrets from TacticalRMM monitoring tools for automated agent enrollment. This implementation handles the authentication and API communication required to generate installation secrets for new monitoring agents.

## Key Components

- **`TacticalRmmAgentRegistrationSecretRetriever`** - Main service implementing `ToolAgentRegistrationSecretRetriever` interface
- **`getSecret()`** - Primary method that orchestrates secret retrieval from TacticalRMM API
- **`buildRequest()`** - Helper method that constructs default agent registration parameters
- **`getToolId()`** - Returns the static tool identifier "tactical-rmm"

## Usage Example

```java
@Autowired
private TacticalRmmAgentRegistrationSecretRetriever secretRetriever;

// Retrieve an agent registration secret
public void setupNewAgent() {
    try {
        String toolId = secretRetriever.getToolId(); // Returns "tactical-rmm"
        String enrollmentSecret = secretRetriever.getSecret();
        
        // Use the secret to register a new agent
        log.info("Agent enrollment secret: {}", enrollmentSecret);
        
    } catch (IllegalStateException e) {
        log.error("Failed to retrieve secret: {}", e.getMessage());
    }
}
```

The component automatically configures agents for Windows Server platform with 40-hour expiration (2400 minutes) and default organization/site settings. It requires the TacticalRMM integration to be enabled via the `openframe.integration.tool.enabled=true` property.
