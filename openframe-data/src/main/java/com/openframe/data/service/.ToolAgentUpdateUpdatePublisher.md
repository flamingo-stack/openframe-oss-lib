A Spring service that publishes tool agent update notifications via NATS messaging when integrated tool agents are updated, with configurable feature toggling.

## Key Components

- **publish()** - Main method that publishes tool agent update messages to NATS topics
- **buildTopicName()** - Constructs topic names using pattern `machine.all.tool.{toolId}.update`
- **buildMessage()** - Creates `ToolAgentUpdateMessage` objects with tool agent details
- **Feature Toggle** - Controlled by `openframe.tool-agent.update.feature.enabled` property
- **Conditional Loading** - Only active when `spring.cloud.stream.enabled` is true

## Usage Example

```java
@Autowired
private ToolAgentUpdateUpdatePublisher publisher;

// Publish update notification for a tool agent
IntegratedToolAgent toolAgent = getToolAgent();
publisher.publish(toolAgent);

// The publisher will:
// 1. Check if feature is enabled
// 2. Build topic: "machine.all.tool.myTool.update"  
// 3. Create message with tool ID, version, session type
// 4. Publish via NATS with download configurations
```

The service automatically handles feature flag validation and logs appropriate messages for both enabled/disabled states. It integrates with the NATS messaging infrastructure to broadcast tool updates across the system.
