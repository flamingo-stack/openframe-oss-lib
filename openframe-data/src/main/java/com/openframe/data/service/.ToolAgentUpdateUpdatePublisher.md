Publishes tool agent update notifications to NATS messaging system when tool agents are updated. Only active when streaming and the tool agent update feature are enabled.

## Key Components

- **publish()** - Main method that publishes tool agent updates to NATS topics
- **buildTopicName()** - Creates topic name using pattern `machine.all.tool.{toolId}.update`
- **buildMessage()** - Constructs `ToolAgentUpdateMessage` with agent details and download configurations
- **toolAgentUpdateFeatureEnabled** - Feature flag to enable/disable publishing
- **TOPIC_NAME_TEMPLATE** - Static template for generating topic names

## Usage Example

```java
@Autowired
private ToolAgentUpdateUpdatePublisher publisher;

public void updateToolAgent(IntegratedToolAgent toolAgent) {
    // Update the tool agent...
    toolAgentRepository.save(toolAgent);
    
    // Publish update notification
    publisher.publish(toolAgent);
}
```

The service is conditionally enabled via `spring.cloud.stream.enabled` property and can be toggled with `openframe.tool-agent.update.feature.enabled`. Messages include tool ID, version, session type, and mapped download configurations.
