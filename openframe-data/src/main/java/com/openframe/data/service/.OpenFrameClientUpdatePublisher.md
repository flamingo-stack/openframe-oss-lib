Publishes client update messages to all machines in the OpenFrame system via NATS messaging when new client configurations are available.

## Key Components

- **`publish(OpenFrameClientConfiguration)`** - Main method that publishes client update messages to all machines
- **`buildMessage(OpenFrameClientConfiguration)`** - Creates the update message from configuration data
- **Feature Flag Support** - Conditionally enabled via `openframe.client.update.feature.enabled` property
- **NATS Integration** - Uses `NatsMessagePublisher` to broadcast to `machine.all.client-update` topic

## Usage Example

```java
@Autowired
private OpenFrameClientUpdatePublisher publisher;

// Publish configuration update to all machines
OpenFrameClientConfiguration config = new OpenFrameClientConfiguration();
config.setVersion("2.1.0");
config.setDownloadConfiguration(downloadConfig);

publisher.publish(config);
// Publishes update message if feature is enabled
```

The service is automatically disabled when `spring.cloud.stream.enabled` is false, and individual publishing can be controlled via the `openframe.client.update.feature.enabled` property. Messages include version information and mapped download configurations for client updates.
