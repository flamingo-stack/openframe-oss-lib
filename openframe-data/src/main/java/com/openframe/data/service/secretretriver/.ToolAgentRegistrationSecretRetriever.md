This interface defines a contract for retrieving authentication credentials needed for tool agent registration in the system.

## Key Components

- **`getToolId()`** - Returns the unique identifier for the tool being registered
- **`getSecret()`** - Returns the authentication secret/token for the tool registration process

## Usage Example

```java
public class DatabaseSecretRetriever implements ToolAgentRegistrationSecretRetriever {
    
    @Override
    public String getToolId() {
        return "analytics-tool-v2";
    }
    
    @Override
    public String getSecret() {
        return configService.getSecret("tool.registration.token");
    }
}

// Usage in registration service
ToolAgentRegistrationSecretRetriever retriever = new DatabaseSecretRetriever();
String toolId = retriever.getToolId();
String secret = retriever.getSecret();

// Use credentials for tool registration
toolRegistrationService.register(toolId, secret);
```

This interface enables flexible secret retrieval strategies, allowing implementations to source credentials from databases, configuration files, environment variables, or external secret management services while maintaining a consistent API for the registration process.
