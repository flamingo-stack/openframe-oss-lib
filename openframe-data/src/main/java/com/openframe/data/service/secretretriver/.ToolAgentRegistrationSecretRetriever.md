A simple interface for retrieving authentication credentials needed for tool agent registration, providing access to both tool identifier and associated secret.

## Key Components

- **getToolId()** - Returns the unique identifier for the tool agent
- **getSecret()** - Returns the authentication secret/token for the tool agent

## Usage Example

```java
// Implementation example
public class DatabaseToolSecretRetriever implements ToolAgentRegistrationSecretRetriever {
    
    @Override
    public String getToolId() {
        return "database-connector-v1";
    }
    
    @Override
    public String getSecret() {
        return System.getenv("DB_TOOL_SECRET");
    }
}

// Usage in registration service
ToolAgentRegistrationSecretRetriever retriever = new DatabaseToolSecretRetriever();
String toolId = retriever.getToolId();
String secret = retriever.getSecret();

// Register tool agent with credentials
toolRegistry.register(toolId, secret);
```

This interface enables secure credential management for tool agent registration by abstracting the retrieval mechanism, allowing implementations to source secrets from environment variables, secure vaults, or configuration files.
