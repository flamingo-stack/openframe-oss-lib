A Spring service component that publishes tool installation messages to NATS topics for machine-specific tool deployment operations.

## Key Components

- **publish()** - Main methods to publish tool installation messages, with optional reinstall flag
- **buildTopicName()** - Constructs machine-specific NATS topic names using template pattern
- **buildMessage()** - Creates `ToolInstallationMessage` objects from tool agent and tool data
- **mapAssets()** - Converts `ToolAgentAsset` collections to message asset format
- **mapAssetSource()** - Maps asset source enums between domain and message models

## Usage Example

```java
@Autowired
private ToolInstallationNatsPublisher publisher;

// Publish standard tool installation
publisher.publish("machine-001", toolAgent, integratedTool);

// Publish tool reinstallation
publisher.publish("machine-001", toolAgent, integratedTool, true);
```

The service constructs topic names in the format `machine.{machineId}.tool-installation` and transforms tool configuration data into NATS messages containing download configurations, assets, and command arguments. It requires NATS streaming to be enabled via the `spring.cloud.stream.enabled` property and handles null-safe mapping of tool agent assets and their sources.
