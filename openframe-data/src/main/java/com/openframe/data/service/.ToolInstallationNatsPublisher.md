A Spring Boot service that publishes tool installation messages to NATS messaging system for remote machine tool deployment and management.

## Key Components

**Main Class:**
- `ToolInstallationNatsPublisher` - Core service for publishing tool installation messages to machine-specific NATS topics

**Dependencies:**
- `NatsMessagePublisher` - Handles NATS message publishing
- `DownloadConfigurationMapper` - Maps download configurations with version substitution
- `LocalFilenameConfigurationMapper` - Maps local filename configurations

**Key Methods:**
- `publish()` - Publishes tool installation messages (with optional reinstall flag)
- `buildMessage()` - Constructs `ToolInstallationMessage` from tool agent and tool data
- `mapAssets()` - Converts tool agent assets to message format
- `mapAssetSource()` - Maps asset source enums between domain and message models

## Usage Example

```java
@Autowired
private ToolInstallationNatsPublisher publisher;

// Basic tool installation
publisher.publish("machine-001", toolAgent, integrationTool);

// Force reinstallation
publisher.publish("machine-001", toolAgent, integrationTool, true);
```

The service automatically constructs machine-specific topics using the pattern `machine.{machineId}.tool-installation` and handles the complete mapping of tool agent data including assets, download configurations, and command arguments. It's conditionally enabled when NATS streaming is active via `spring.cloud.stream.enabled` property.
