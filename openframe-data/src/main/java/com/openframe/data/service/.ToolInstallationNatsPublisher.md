A Spring component that publishes tool installation messages to NATS messaging system for machine-specific tool deployment operations.

## Key Components

- **publish()** - Main methods to publish tool installation messages, with optional reinstall flag
- **buildTopicName()** - Constructs machine-specific NATS topic using template pattern
- **buildMessage()** - Creates ToolInstallationMessage from tool agent and tool data
- **mapAssets()** - Converts ToolAgentAsset objects to message-compatible format
- **mapAssetSource()** - Maps asset source enums between domain and message models

## Usage Example

```java
@Autowired
private ToolInstallationNatsPublisher publisher;

// Basic tool installation
publisher.publish("machine-123", toolAgent, integratedTool);

// Tool reinstallation
publisher.publish("machine-456", toolAgent, integratedTool, true);
```

The publisher creates machine-specific topics (`machine.{machineId}.tool-installation`) and transforms tool configuration data into NATS messages containing installation commands, assets, download configurations, and metadata. Only active when `spring.cloud.stream.enabled` property is set.
