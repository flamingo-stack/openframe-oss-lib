Spring Boot configuration class that defines beans for external tool SDK clients, specifically for Tactical RMM integration.

## Key Components

- **@Configuration**: Marks this class as a Spring configuration source
- **tacticalRmmClient()**: Bean factory method that creates a TacticalRmmClient instance
- **@Bean**: Registers the TacticalRmmClient as a Spring-managed bean

## Usage Example

This configuration automatically registers the Tactical RMM client in the Spring application context:

```java
@Service
public class MonitoringService {
    
    @Autowired
    private TacticalRmmClient tacticalRmmClient;
    
    public void performHealthCheck() {
        // Use the auto-configured client
        tacticalRmmClient.checkSystemStatus();
    }
}
```

The bean can also be accessed programmatically:

```java
@Component
public class SystemManager {
    
    @Autowired
    private ApplicationContext context;
    
    public void manageSystems() {
        TacticalRmmClient client = context.getBean(TacticalRmmClient.class);
        client.performRemoteManagement();
    }
}
```
