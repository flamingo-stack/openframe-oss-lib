Configuration class that sets up SDK client beans for external tool integrations, specifically for Tactical RMM connectivity.

## Key Components

- **ToolSdkConfig**: Spring configuration class that defines SDK client beans
- **tacticalRmmClient()**: Bean factory method that creates and configures a TacticalRmmClient instance

## Usage Example

```java
@Service
public class MonitoringService {
    
    @Autowired
    private TacticalRmmClient tacticalRmmClient;
    
    public void checkSystemStatus() {
        // Use the auto-configured client
        var status = tacticalRmmClient.getSystemStatus();
        // Process monitoring data...
    }
}
```

The configuration automatically makes the TacticalRmmClient available for dependency injection throughout the Spring application context. Simply autowire the client in any Spring-managed component to interact with Tactical RMM services.
