Spring Boot configuration class that sets up Apache Cassandra database connectivity with conditional activation and custom session configuration.

## Key Components

- **CassandraConfig**: Main configuration class extending `AbstractCassandraConfiguration`
- **@ConditionalOnProperty**: Conditionally enables Cassandra only when `spring.data.cassandra.enabled=true`
- **@EnableCassandraRepositories**: Activates Cassandra repositories in specified package
- **Configuration Properties**: Contact points, port, keyspace, and datacenter settings
- **cassandraSession()**: Custom session factory with load balancing and timestamp configuration
- **CassandraSessionLogger**: Bean for session logging functionality

## Usage Example

```java
# application.properties
spring.data.cassandra.enabled=true
spring.data.cassandra.contact-points=127.0.0.1
spring.data.cassandra.port=9042
spring.data.cassandra.keyspace-name=myapp
spring.data.cassandra.local-datacenter=datacenter1

@Repository
public interface UserRepository extends CassandraRepository<User, UUID> {
    // Repository methods automatically available
}

@Service
public class UserService {
    @Autowired
    private UserRepository userRepository;
    
    public User saveUser(User user) {
        return userRepository.save(user);
    }
}
```

The configuration automatically creates keyspaces if they don't exist and sets up server-side timestamp generation for optimal performance.
