Spring Boot configuration class that sets up Redis connectivity with both synchronous and reactive templates. Conditionally enabled based on the `spring.redis.enabled` property and automatically configures Redis repositories.

## Key Components

- **@ConditionalOnProperty**: Activates Redis configuration only when `spring.redis.enabled=true`
- **@EnableRedisRepositories**: Enables Redis repositories in the specified package
- **redisTemplate()**: Creates synchronous RedisTemplate with String serialization
- **reactiveStringRedisTemplate()**: Creates reactive template for String operations
- **reactiveRedisTemplate()**: Creates fully configured reactive template with custom serialization

## Usage Example

```java
// Enable Redis in application.properties
// spring.redis.enabled=true

// Inject and use the templates
@Autowired
private RedisTemplate<String, String> redisTemplate;

@Autowired
private ReactiveStringRedisTemplate reactiveTemplate;

// Synchronous operations
redisTemplate.opsForValue().set("key", "value");
String value = redisTemplate.opsForValue().get("key");

// Reactive operations
Mono<Boolean> result = reactiveTemplate.opsForValue()
    .set("reactive-key", "reactive-value");

Mono<String> reactiveValue = reactiveTemplate.opsForValue()
    .get("reactive-key");
```

All templates use String serialization for keys, values, and hash operations, making them suitable for text-based caching and data storage scenarios.
