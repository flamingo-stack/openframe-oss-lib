Spring Boot configuration class that sets up Redis connectivity with both synchronous and reactive templates, activated only when Redis is explicitly enabled.

## Key Components

- **@ConditionalOnProperty**: Conditionally loads Redis configuration only when `spring.redis.enabled=true`
- **@EnableRedisRepositories**: Enables Redis repository support for the specified package
- **redisTemplate()**: Creates a synchronous RedisTemplate with String serialization for keys and values
- **reactiveStringRedisTemplate()**: Provides a reactive String-based Redis template
- **reactiveRedisTemplate()**: Creates a fully configured reactive RedisTemplate with custom serialization context

## Usage Example

```java
// Application properties to enable Redis
spring.redis.enabled=true
spring.redis.host=localhost
spring.redis.port=6379

// Injecting templates in service classes
@Service
public class CacheService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    @Autowired
    private ReactiveStringRedisTemplate reactiveTemplate;
    
    // Synchronous operations
    public void cacheValue(String key, String value) {
        redisTemplate.opsForValue().set(key, value);
    }
    
    // Reactive operations
    public Mono<String> getCachedValue(String key) {
        return reactiveTemplate.opsForValue().get(key);
    }
}
```

The configuration ensures Redis repositories are scanned from `com.openframe.data.repository.redis` package and all templates use consistent String serialization for optimal performance and compatibility.
