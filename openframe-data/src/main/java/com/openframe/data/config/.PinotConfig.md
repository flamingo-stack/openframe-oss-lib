Spring configuration class that sets up Apache Pinot database connections for both broker and controller endpoints using externalized configuration properties.

## Key Components

- **`pinotBrokerConnection()`** - Creates a Pinot broker connection using properties-based configuration
- **`pinotControllerConnection()`** - Creates a Pinot controller connection using host list configuration
- **`brokerUrl`** - Injected broker endpoint URL from application properties
- **`controllerUrl`** - Injected controller endpoint URL from application properties

## Usage Example

```java
// Application properties configuration
// pinot.broker.url=localhost:8000
// pinot.controller.url=localhost:9000

@Service
public class PinotQueryService {
    
    @Autowired
    @Qualifier("pinotBrokerConnection")
    private Connection brokerConnection;
    
    @Autowired
    @Qualifier("pinotControllerConnection") 
    private Connection controllerConnection;
    
    public ResultSet executeQuery(String sql) {
        return brokerConnection.execute(sql);
    }
    
    public void createTable(String tableConfig) {
        // Use controller connection for admin operations
        controllerConnection.execute(tableConfig);
    }
}
```

The configuration automatically injects the connection beans into any Spring-managed components that need to interact with Pinot for real-time analytics queries or administrative operations.
