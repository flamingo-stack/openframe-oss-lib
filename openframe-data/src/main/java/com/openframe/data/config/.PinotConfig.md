Spring configuration class that establishes connections to Apache Pinot broker and controller services using externalized configuration properties.

## Key Components

- **@Configuration**: Marks this as a Spring configuration class
- **brokerUrl/controllerUrl**: Injected property values for Pinot service endpoints
- **pinotBrokerConnection()**: Creates a connection bean to the Pinot broker using Properties-based configuration
- **pinotControllerConnection()**: Creates a connection bean to the Pinot controller using host list configuration

## Usage Example

Configure the required properties in `application.properties`:

```java
pinot.broker.url=localhost:8000
pinot.controller.url=localhost:9000
```

Then inject the connections in your service classes:

```java
@Service
public class PinotQueryService {
    
    @Autowired
    @Qualifier("pinotBrokerConnection")
    private Connection brokerConnection;
    
    @Autowired
    @Qualifier("pinotControllerConnection")
    private Connection controllerConnection;
    
    public void executeQuery(String sql) {
        // Use brokerConnection for querying data
        // Use controllerConnection for admin operations
    }
}
```

The configuration automatically creates two connection beans that can be injected wherever Pinot database access is needed in your Spring application.
