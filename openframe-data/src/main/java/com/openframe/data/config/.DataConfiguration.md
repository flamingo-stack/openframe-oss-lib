A Spring Boot configuration class that conditionally enables Cassandra repository support based on application properties.

## Key Components

- **DataConfiguration**: Main configuration class containing nested database configurations
- **CassandraConfiguration**: Static nested configuration that conditionally enables Cassandra repositories when the `spring.data.cassandra.enabled` property is set to `true`

## Usage Example

To enable Cassandra repositories, add this property to your `application.properties`:

```java
spring.data.cassandra.enabled=true
```

The configuration will then automatically scan for Cassandra repositories in the specified package:

```java
// Repositories in com.openframe.data.repository.cassandra will be auto-discovered
@Repository
public interface UserRepository extends CassandraRepository<User, UUID> {
    List<User> findByEmail(String email);
}
```

If the property is not set or set to `false`, Cassandra repository support will be disabled, allowing the application to run without Cassandra dependencies.
