This Spring Boot configuration class conditionally enables Cassandra database repositories based on application properties.

## Key Components

- **`DataConfiguration`** - Main configuration class that houses database-specific configurations
- **`CassandraConfiguration`** - Nested static configuration class that enables Cassandra repositories when the feature is activated

## Usage Example

To enable Cassandra repositories, add this property to your `application.properties`:

```java
spring.data.cassandra.enabled=true
```

The configuration will automatically scan for repository interfaces in the specified package:

```java
// Repository interfaces in com.openframe.data.repository.cassandra will be auto-discovered
@Repository
public interface UserRepository extends CassandraRepository<User, UUID> {
    // Repository methods
}
```

If the property is not set or set to `false`, Cassandra repositories will be disabled, allowing the application to run without Cassandra dependencies. This follows the conditional configuration pattern for optional database integrations.
