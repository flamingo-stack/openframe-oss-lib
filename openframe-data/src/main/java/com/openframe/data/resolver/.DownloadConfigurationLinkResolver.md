Resolves download configuration link templates by replacing version placeholders with actual version values or handling "latest" version requests.

## Key Components

- **resolve(DownloadConfiguration, String)** - Main method that resolves a link template with the provided version
- **resolveLink(String, String)** - Internal method that performs the actual template resolution logic
- **isLatestVersion(String)** - Utility method to check if the version is the "latest" tag
- **Constants** - Predefined path patterns for version handling (`VERSION_TAG_PATH`, `LATEST_VERSION_TAG_PATH`)

## Usage Example

```java
@Autowired
private DownloadConfigurationLinkResolver resolver;

public void generateDownloadLinks() {
    DownloadConfiguration config = new DownloadConfiguration();
    config.setLinkTemplate("https://api.example.com/download/{version}/app.zip");
    
    // Resolve specific version
    String versionedLink = resolver.resolve(config, "1.2.3");
    // Result: "https://api.example.com/download/1.2.3/app.zip"
    
    // Resolve latest version
    String latestLink = resolver.resolve(config, "latest");
    // Result: "https://api.example.com/latest/download/app.zip"
}
```

The component automatically handles the special "latest" version case by replacing the version path pattern with a dedicated latest path structure.
