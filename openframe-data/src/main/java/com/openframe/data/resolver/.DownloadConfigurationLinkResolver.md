A Spring component that resolves download configuration links by substituting version placeholders with actual version values.

## Key Components

**Main Class:**
- `DownloadConfigurationLinkResolver` - Spring component for resolving download links with version substitution

**Core Methods:**
- `resolve(DownloadConfiguration, String)` - Public method that resolves a download link template with the provided version
- `resolveLink(String, String)` - Private helper method that performs the actual placeholder replacement

**Constants:**
- `VERSION_PLACEHOLDER` - Defines the `{version}` placeholder pattern used in link templates

## Usage Example

```java
@Autowired
private DownloadConfigurationLinkResolver linkResolver;

public void processDownload() {
    DownloadConfiguration config = new DownloadConfiguration();
    config.setLinkTemplate("https://releases.example.com/app-{version}.zip");
    
    String version = "2.1.0";
    String resolvedUrl = linkResolver.resolve(config, version);
    
    // Result: "https://releases.example.com/app-2.1.0.zip"
    System.out.println("Download URL: " + resolvedUrl);
}
```

The resolver enables dynamic download URLs by replacing version placeholders in configured link templates, making it easy to generate version-specific download links for software distributions or updates.
