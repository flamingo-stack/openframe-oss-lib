This enum defines the available types of data enrichment services within the OpenFrame data processing system.

## Key Components

- **INTEGRATED_TOOLS_EVENTS** - Represents a data enrichment service that processes and enhances events from integrated tools

## Usage Example

```java
import com.openframe.data.model.enums.DataEnrichmentServiceType;

// Using the enum in service configuration
public class DataEnrichmentConfig {
    private DataEnrichmentServiceType serviceType;
    
    public void configureService() {
        serviceType = DataEnrichmentServiceType.INTEGRATED_TOOLS_EVENTS;
    }
}

// In switch statements
public void processEnrichment(DataEnrichmentServiceType type) {
    switch (type) {
        case INTEGRATED_TOOLS_EVENTS:
            // Handle integrated tools event enrichment
            processIntegratedToolsEvents();
            break;
        default:
            throw new IllegalArgumentException("Unknown service type: " + type);
    }
}
```

The enum currently contains a single value but provides a foundation for expanding data enrichment service types as the system grows.
