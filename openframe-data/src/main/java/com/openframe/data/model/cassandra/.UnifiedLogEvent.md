A NIST-compliant Cassandra entity for storing unified log events from integrated tools (Fleet, Meshcentral, Tactical-RMM) with time-series partitioning for scalable storage and efficient querying.

## Key Components

- **UnifiedLogEvent** - Main entity class with Cassandra table mapping
- **UnifiedLogEventKey** - Composite primary key class with partitioning strategy
- **Partitioning** - By `ingest_day` and `tool_type` for horizontal scaling
- **Clustering** - By `event_type`, `event_timestamp` (descending), and `tool_event_id`
- **Core Fields** - User/device/organization identifiers, severity, message, and flexible details storage

## Usage Example

```java
@Autowired
private CassandraTemplate cassandraTemplate;

// Create a new log event
UnifiedLogEvent logEvent = new UnifiedLogEvent();
UnifiedLogEvent.UnifiedLogEventKey key = new UnifiedLogEvent.UnifiedLogEventKey();

key.setIngestDay("2024-01-15");
key.setToolType("meshcentral");
key.setEventType("login");
key.setEventTimestamp(Instant.now());
key.setToolEventId("mc_12345");

logEvent.setKey(key);
logEvent.setUserId("user123");
logEvent.setHostname("workstation01");
logEvent.setSeverity("info");
logEvent.setMessage("User login successful");

cassandraTemplate.insert(logEvent);

// Query recent events for a specific tool and day
Select query = QueryBuilder.selectFrom("unified_logs")
    .all()
    .whereColumn("ingest_day").isEqualTo(literal("2024-01-15"))
    .whereColumn("tool_type").isEqualTo(literal("meshcentral"));
```
