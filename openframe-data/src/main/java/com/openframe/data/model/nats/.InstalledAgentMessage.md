A simple data model representing a message about an installed agent, typically used for communication via NATS messaging system.

## Key Components

- **machineId** - Unique identifier for the machine where the agent is installed
- **agentType** - Type or category of the installed agent
- **version** - Version number of the installed agent
- **@Data** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Creating a new installed agent message
InstalledAgentMessage message = new InstalledAgentMessage();
message.setMachineId("server-001");
message.setAgentType("monitoring-agent");
message.setVersion("2.1.4");

// Using in NATS publisher
publisher.publish("agent.installed", message);

// Accessing properties
String machineId = message.getMachineId();
String type = message.getAgentType();
String version = message.getVersion();
```

This class is designed for serialization/deserialization when sending agent installation notifications through NATS messaging infrastructure, allowing systems to track which agents are deployed across different machines.
