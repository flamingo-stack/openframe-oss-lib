A message model for communicating tool installation instructions through NATS messaging, containing all necessary metadata and configuration for tool deployment.

## Key Components

- **ToolInstallationMessage** - Main message class containing tool metadata, installation commands, and assets
- **Asset** - Nested class representing downloadable tool components with source information
- **AssetSource** - Enum defining where assets can be retrieved from (ARTIFACTORY or TOOL_API)
- **Key fields**: toolAgentId, toolId, version, sessionType, downloadConfigurations, and various command argument lists

## Usage Example

```java
// Create installation message
ToolInstallationMessage message = new ToolInstallationMessage();
message.setToolAgentId("agent-123");
message.setToolId("my-tool");
message.setVersion("1.0.0");
message.setSessionType(SessionType.PERSISTENT);
message.setReinstall(true);

// Add installation commands
message.setInstallationCommandArgs(List.of("install", "--force"));
message.setRunCommandArgs(List.of("run", "--port", "8080"));

// Add asset
ToolInstallationMessage.Asset asset = new ToolInstallationMessage.Asset();
asset.setId("tool-binary");
asset.setLocalFilename("tool.jar");
asset.setSource(AssetSource.ARTIFACTORY);
asset.setExecutable(true);
message.setAssets(List.of(asset));
```
