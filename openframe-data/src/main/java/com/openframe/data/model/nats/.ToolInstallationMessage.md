A data model representing a message for tool installation operations in the OpenFrame system, containing configuration details for downloading, installing, and managing tools.

## Key Components

- **ToolInstallationMessage**: Main message class containing tool installation metadata and command configurations
- **Asset**: Nested class representing downloadable tool assets with source and execution properties  
- **AssetSource**: Enum defining asset source types (ARTIFACTORY, TOOL_API)
- **Core fields**: Tool identification, version info, session type, and command argument lists
- **Command configurations**: Installation, run, uninstallation, and tool agent ID command arguments

## Usage Example

```java
// Create a tool installation message
ToolInstallationMessage message = new ToolInstallationMessage();
message.setToolAgentId("agent-123");
message.setToolId("maven");
message.setToolType("build-tool");
message.setVersion("3.8.1");
message.setSessionType(SessionType.PERSISTENT);

// Configure installation commands
message.setInstallationCommandArgs(List.of("install", "--quiet"));
message.setRunCommandArgs(List.of("mvn", "clean", "compile"));
message.setUninstallationCommandArgs(List.of("uninstall", "--force"));

// Add asset configuration
ToolInstallationMessage.Asset asset = new ToolInstallationMessage.Asset();
asset.setId("maven-binary");
asset.setLocalFilename("maven-3.8.1.tar.gz");
asset.setSource(ToolInstallationMessage.AssetSource.ARTIFACTORY);
asset.setExecutable(true);

message.setAssets(List.of(asset));
```
