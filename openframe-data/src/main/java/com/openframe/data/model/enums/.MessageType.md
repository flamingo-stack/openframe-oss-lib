This enum defines message types for various integrated tools and their associated routing configurations in the OpenFrame data processing system.

## Key Components

**Enum Constants:**
- `MESHCENTRAL_EVENT` - MeshCentral tool events
- `TACTICAL_RMM_AUDIT_EVENT` - TacticalRMM audit events  
- `TACTICAL_RMM_AGENT_HISTORY_EVENT` - TacticalRMM agent history events
- `FLEET_MDM_EVENT` - Fleet MDM events
- `FLEET_MDM_QUERY_RESULT_EVENT` - Fleet MDM query result events

**Properties:**
- `integratedToolType` - Source tool type (MeshCentral, Tactical, Fleet)
- `dataEnrichmentServiceType` - Service type for data enrichment
- `destinationList` - Target destinations (Cassandra, Kafka)
- `eventHandlerType` - Handler type for processing

## Usage Example

```java
// Get message type configuration
MessageType messageType = MessageType.MESHCENTRAL_EVENT;

// Access routing information
IntegratedToolType toolType = messageType.getIntegratedToolType();
List<Destination> destinations = messageType.getDestinationList();
EventHandlerType handlerType = messageType.getEventHandlerType();

// Route message based on type
if (destinations.contains(Destination.KAFKA)) {
    // Send to Kafka
}
if (destinations.contains(Destination.CASSANDRA)) {
    // Store in Cassandra
}
```
