A simple data model class for storing various authentication credentials used by external tools and services.

## Key Components

- **ToolCredentials** - Main data class with Lombok `@Data` annotation for automatic getter/setter generation
- **Authentication Fields**:
  - `username` - Basic auth username
  - `password` - Basic auth password  
  - `token` - Bearer/access tokens
  - `apiKey` - API key authentication
  - `clientId` - OAuth client identifier
  - `clientSecret` - OAuth client secret

All fields are nullable to support different authentication schemes.

## Usage Example

```java
// Create credentials for API key authentication
ToolCredentials apiCredentials = new ToolCredentials();
apiCredentials.setApiKey("your-api-key-here");

// Create credentials for OAuth
ToolCredentials oauthCredentials = new ToolCredentials();
oauthCredentials.setClientId("client-123");
oauthCredentials.setClientSecret("secret-456");

// Create credentials for basic auth
ToolCredentials basicCredentials = new ToolCredentials();
basicCredentials.setUsername("user");
basicCredentials.setPassword("pass");
```
