A data model class for storing various types of authentication credentials used by tools in the OpenFrame system.

## Key Components

- **username/password** - Basic authentication credentials
- **token** - Bearer or session tokens for API access
- **apiKey** - API key for service authentication
- **clientId/clientSecret** - OAuth 2.0 client credentials
- All fields are nullable using Spring's `@Nullable` annotation
- Uses Lombok's `@Data` annotation for automatic getter/setter generation

## Usage Example

```java
// Create credentials for API key authentication
ToolCredentials apiCredentials = new ToolCredentials();
apiCredentials.setApiKey("sk-1234567890abcdef");

// Create OAuth 2.0 credentials
ToolCredentials oauthCredentials = new ToolCredentials();
oauthCredentials.setClientId("my-client-id");
oauthCredentials.setClientSecret("my-client-secret");

// Create basic auth credentials
ToolCredentials basicCredentials = new ToolCredentials();
basicCredentials.setUsername("admin");
basicCredentials.setPassword("secure-password");

// Use with a tool that requires authentication
tool.authenticate(apiCredentials);
```

This flexible model supports multiple authentication schemes, allowing tools to use the appropriate credential type for their specific authentication requirements.
