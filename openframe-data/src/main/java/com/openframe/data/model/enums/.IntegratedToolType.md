An enumeration that defines the types of integrated third-party management tools supported by the OpenFrame platform, mapping each tool type to its corresponding database identifier.

## Key Components

- **MESHCENTRAL** - Remote access and device management platform
- **TACTICAL** - TacticalRMM remote monitoring and management system  
- **FLEET** - Fleet mobile device management (MDM) solution
- **getDbName()** - Returns the database name string for the tool type

## Usage Example

```java
// Get the database name for a tool type
IntegratedToolType toolType = IntegratedToolType.MESHCENTRAL;
String dbIdentifier = toolType.getDbName(); // Returns "meshcentral"

// Use in conditional logic
if (toolType == IntegratedToolType.TACTICAL) {
    // Configure TacticalRMM integration
    setupTacticalIntegration(toolType.getDbName());
}

// Iterate through all supported tools
for (IntegratedToolType tool : IntegratedToolType.values()) {
    System.out.println("Tool: " + tool + ", DB Name: " + tool.getDbName());
}
```

This enum provides a type-safe way to reference integrated management tools while maintaining consistent database naming conventions across the OpenFrame system.
