An enumeration that defines the available types of integrated third-party management tools in the OpenFrame system, each with an associated database identifier.

## Key Components

- **MESHCENTRAL** - Remote management tool integration
- **TACTICAL** - Tactical RMM tool integration  
- **FLEET** - Fleet MDM (Mobile Device Management) integration
- **getDbName()** - Returns the database name string for the enum value

## Usage Example

```java
// Get the database name for a tool type
IntegratedToolType toolType = IntegratedToolType.MESHCENTRAL;
String dbIdentifier = toolType.getDbName(); // Returns "meshcentral"

// Use in switch statements for tool-specific logic
switch (toolType) {
    case MESHCENTRAL:
        // Handle MeshCentral integration
        break;
    case TACTICAL:
        // Handle Tactical RMM integration
        break;
    case FLEET:
        // Handle Fleet MDM integration
        break;
}

// Iterate through all available tool types
for (IntegratedToolType tool : IntegratedToolType.values()) {
    System.out.println("Tool: " + tool + ", DB Name: " + tool.getDbName());
}
```
