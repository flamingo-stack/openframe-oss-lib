A simple data transfer object representing an organization option with identifier and display name properties, commonly used in dropdown lists or selection components.

## Key Components

- **OrganizationOption** - Main data class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **id** - String field for unique organization identifier
- **name** - String field for human-readable organization name

## Usage Example

```java
// Create using builder pattern
OrganizationOption option = OrganizationOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationOption option2 = new OrganizationOption("org-456", "Tech Solutions Inc");

// Create empty and set values
OrganizationOption option3 = new OrganizationOption();
option3.setId("org-789");
option3.setName("Global Industries");

// Access properties
String orgId = option.getId();
String orgName = option.getName();
```

The class leverages Lombok annotations to minimize boilerplate code while providing flexible instantiation options through builders, constructors, and setters.
