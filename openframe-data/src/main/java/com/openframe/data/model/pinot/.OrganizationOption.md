A simple data model representing an organization option with identifier and name fields, designed for use with Apache Pinot data storage.

## Key Components

- **id** - String identifier for the organization option
- **name** - Human-readable name of the organization option
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create using builder pattern
OrganizationOption option = OrganizationOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationOption option2 = new OrganizationOption("org-456", "Tech Startup Inc");

// Access fields directly
String orgId = option.id;
String orgName = option.name;

// Modify fields
option.setName("Updated Organization Name");
```

This class serves as a lightweight data transfer object (DTO) for organization options in Pinot queries and data operations. The public fields and Lombok annotations make it suitable for JSON serialization/deserialization and database mapping scenarios.
