A simple POJO representing messages exchanged over NATS for tool connection events, containing tool type and agent identification information.

## Key Components

- **toolType** - String field identifying the type of tool being connected
- **agentToolId** - String field containing the unique identifier for the agent's tool instance
- Uses Lombok annotations (`@Getter`, `@Setter`) for automatic accessor method generation

## Usage Example

```java
// Creating a tool connection message
ToolConnectionMessage message = new ToolConnectionMessage();
message.setToolType("database-connector");
message.setAgentToolId("agent-123-db-tool-456");

// Reading message properties
String toolType = message.getToolType();
String agentId = message.getAgentToolId();

// Typical usage in NATS message handling
public void handleToolConnection(ToolConnectionMessage message) {
    System.out.println("Tool connected: " + message.getToolType() + 
                      " for agent: " + message.getAgentToolId());
}
```

This class is typically used in distributed messaging scenarios where agents need to communicate tool connection status across the system via NATS messaging infrastructure.
