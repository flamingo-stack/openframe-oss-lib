A simple data model class for representing tool connection messages in the NATS messaging system, containing essential identifiers for tool-to-agent communication.

## Key Components

- **toolType** - String identifier specifying the type of tool being connected
- **agentToolId** - String identifier linking the tool to a specific agent instance
- **@Getter/@Setter** - Lombok annotations providing automatic accessor methods

## Usage Example

```java
// Creating a new tool connection message
ToolConnectionMessage message = new ToolConnectionMessage();
message.setToolType("database-connector");
message.setAgentToolId("agent-123-db-tool");

// Reading message properties
String toolType = message.getToolType();
String agentId = message.getAgentToolId();

// Typically used in NATS message publishing/subscribing
publisher.publish("tool.connection", message);
```

This class serves as a lightweight data transfer object for coordinating tool connections within a distributed agent system, facilitating communication between different components through the NATS messaging infrastructure.
