Lightweight data transfer object for caching essential organization information in Redis, designed to minimize serialization overhead and cache size.

## Key Components

- **CachedOrganizationInfo** - Serializable DTO class with Lombok annotations for automatic getters/setters
- **organizationId** - Unique identifier for the organization
- **name** - Organization display name
- **serialVersionUID** - Version control for serialization compatibility

## Usage Example

```java
// Creating a cached organization info object
CachedOrganizationInfo orgInfo = new CachedOrganizationInfo();
orgInfo.setOrganizationId("org-123");
orgInfo.setName("Acme Corporation");

// Using constructor with all arguments
CachedOrganizationInfo orgInfo2 = new CachedOrganizationInfo("org-456", "Tech Solutions Inc");

// Typical usage in a Redis caching service
@Service
public class OrganizationCacheService {
    
    @Autowired
    private RedisTemplate<String, CachedOrganizationInfo> redisTemplate;
    
    public void cacheOrganization(CachedOrganizationInfo orgInfo) {
        String key = "org:" + orgInfo.getOrganizationId();
        redisTemplate.opsForValue().set(key, orgInfo);
    }
    
    public CachedOrganizationInfo getOrganization(String orgId) {
        return redisTemplate.opsForValue().get("org:" + orgId);
    }
}
```
