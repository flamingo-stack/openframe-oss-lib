A lightweight data transfer object designed for caching organization information in Redis, containing only essential fields to minimize serialization overhead and cache storage requirements.

## Key Components

- **CachedOrganizationInfo** - Main DTO class implementing `Serializable` for Redis caching
- **organizationId** - Unique identifier for the organization
- **name** - Organization display name
- **Lombok annotations** - `@Data`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Creating a cached organization info object
CachedOrganizationInfo orgInfo = new CachedOrganizationInfo("org-123", "Acme Corporation");

// Using with Redis template
@Autowired
private RedisTemplate<String, CachedOrganizationInfo> redisTemplate;

// Cache organization info
redisTemplate.opsForValue().set("org:org-123", orgInfo, Duration.ofHours(24));

// Retrieve from cache
CachedOrganizationInfo cached = redisTemplate.opsForValue().get("org:org-123");
if (cached != null) {
    System.out.println("Organization: " + cached.getName());
}

// Constructor usage
CachedOrganizationInfo info = new CachedOrganizationInfo();
info.setOrganizationId("org-456");
info.setName("Tech Solutions Inc");
```
