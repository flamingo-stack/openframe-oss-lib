A lightweight data transfer object (DTO) designed specifically for caching machine information in Redis, containing only essential fields to minimize serialization overhead and cache storage requirements.

## Key Components

- **CachedMachineInfo** - Serializable DTO class with Lombok annotations for automatic getters, setters, and constructors
- **machineId** - Unique identifier for the machine
- **hostname** - Network hostname of the machine  
- **organizationId** - Organization association for the machine
- **serialVersionUID** - Version control for serialization compatibility

## Usage Example

```java
// Create cached machine info
CachedMachineInfo machineInfo = new CachedMachineInfo(
    "machine-001",
    "web-server-01", 
    "org-12345"
);

// Store in Redis (example with RedisTemplate)
redisTemplate.opsForValue().set("machine:" + machineInfo.getMachineId(), machineInfo);

// Retrieve from Redis
CachedMachineInfo cached = (CachedMachineInfo) redisTemplate.opsForValue()
    .get("machine:machine-001");

// Access properties
String hostname = cached.getHostname();
String orgId = cached.getOrganizationId();
```

The class implements `Serializable` with explicit version control to ensure reliable Redis serialization and deserialization across application deployments.
