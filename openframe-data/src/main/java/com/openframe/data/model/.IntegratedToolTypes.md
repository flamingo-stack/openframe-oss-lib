A utility class that defines string constants for various integrated tool types used throughout the OpenFrame data model system.

## Key Components

**String Constants:**
- **Infrastructure Tools**: `MONGODB`, `REDIS`, `CASSANDRA`, `KAFKA`, `NIFI`, `PINOT`, `PROMETHEUS`, `GRAFANA`, `LOKI`
- **Integrated Tools**: `FLEET`, `AUTHENTIK`, `MYSQL`, `POSTGRESQL`
- **Private Constructor**: Prevents instantiation, making this a pure constants class

## Usage Example

```java
// Reference tool types in configuration or validation
public class ToolConfiguration {
    public void configureDatabase(String toolType) {
        switch (toolType) {
            case IntegratedToolTypes.MONGODB:
                setupMongoDB();
                break;
            case IntegratedToolTypes.POSTGRESQL:
                setupPostgreSQL();
                break;
            case IntegratedToolTypes.MYSQL:
                setupMySQL();
                break;
        }
    }
    
    public boolean isMonitoringTool(String type) {
        return type.equals(IntegratedToolTypes.PROMETHEUS) || 
               type.equals(IntegratedToolTypes.GRAFANA);
    }
}
```

This class provides a centralized way to reference tool types consistently across the application, reducing typos and enabling IDE autocompletion.
