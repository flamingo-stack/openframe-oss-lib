A utility class that defines string constants for various integrated tools and infrastructure components used within the OpenFrame platform.

## Key Components

- **Infrastructure Tools**: Database systems (MongoDB, Redis, Cassandra), messaging (Kafka), data processing (NiFi, Pinot), and monitoring tools (Prometheus, Grafana, Loki)
- **Integrated Tools**: Fleet management, authentication (Authentik), and relational databases (MySQL, PostgreSQL)
- **Utility Pattern**: Uses private constructor to prevent instantiation, following the utility class pattern

## Usage Example

```java
// Configure database connection
if (toolType.equals(IntegratedToolTypes.MONGODB)) {
    setupMongoConnection();
} else if (toolType.equals(IntegratedToolTypes.POSTGRESQL)) {
    setupPostgresConnection();
}

// Check monitoring tools
switch (monitoringTool) {
    case IntegratedToolTypes.PROMETHEUS:
        initializePrometheusMetrics();
        break;
    case IntegratedToolTypes.GRAFANA:
        configureDashboards();
        break;
}

// Service registration
List<String> supportedTools = Arrays.asList(
    IntegratedToolTypes.KAFKA,
    IntegratedToolTypes.REDIS,
    IntegratedToolTypes.AUTHENTIK
);
```

This class provides type-safe string constants for tool identification across the application, ensuring consistency when referencing different integrated services and preventing typos in tool names.
