Enumeration defining all unified event types used throughout the OpenFrame system for event logging and tracking. Each event type includes an associated severity level and descriptive summary.

## Key Components

- **Event Categories**: Authentication, device management, user management, script execution, policy compliance, file operations, remote access, monitoring, system events, and more
- **Severity Levels**: Each event type has an associated `Severity` (INFO, WARNING, ERROR, CRITICAL)
- **Summary Descriptions**: Human-readable descriptions for each event type
- **Lombok @Getter**: Automatic getter methods for `severity` and `summary` fields

## Usage Example

```java
// Get event severity and description
UnifiedEventType eventType = UnifiedEventType.LOGIN_FAILED;
Severity severity = eventType.getSeverity(); // WARNING
String description = eventType.getSummary(); // "Failed login attempt"

// Use in event logging
public void logEvent(UnifiedEventType eventType, String details) {
    logger.log(eventType.getSeverity(), 
              eventType.getSummary() + ": " + details);
}

// Example usage
logEvent(UnifiedEventType.DEVICE_OFFLINE, "Device ID: 12345");
logEvent(UnifiedEventType.SCRIPT_FAILED, "Script timeout after 30 seconds");
```
