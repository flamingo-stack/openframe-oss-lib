A data model class representing a client connection event in the NATS messaging system, containing timestamp information and client details.

## Key Components

- **ClientConnectionEvent** - Main event class with timestamp and client information
- **Client** - Nested static class containing client name
- **Lombok annotations** - `@Getter` and `@Setter` for automatic accessor method generation

## Usage Example

```java
// Create a new client connection event
ClientConnectionEvent event = new ClientConnectionEvent();
event.setTimestamp("2023-12-01T10:30:00Z");

// Set client information
ClientConnectionEvent.Client client = new ClientConnectionEvent.Client();
client.setName("client-service-1");
event.setClient(client);

// Access the data
String eventTime = event.getTimestamp();
String clientName = event.getClient().getName();

// Typically used for NATS event processing
System.out.println("Client " + clientName + " connected at " + eventTime);
```

This class is commonly used for tracking and logging client connection events in distributed systems using NATS messaging infrastructure.
