A POJO representing a client update message containing version information and download configurations for the OpenFrame system's NATS messaging.

## Key Components

- **version**: String field containing the client version information
- **downloadConfigurations**: List of `DownloadConfiguration` objects specifying available downloads
- **Lombok annotations**: `@Getter` and `@Setter` for automatic accessor method generation

## Usage Example

```java
// Create a client update message
OpenFrameClientUpdateMessage updateMessage = new OpenFrameClientUpdateMessage();
updateMessage.setVersion("1.2.3");

// Add download configurations
List<DownloadConfiguration> configs = new ArrayList<>();
configs.add(new DownloadConfiguration());
updateMessage.setDownloadConfigurations(configs);

// Access data
String currentVersion = updateMessage.getVersion();
List<DownloadConfiguration> availableDownloads = updateMessage.getDownloadConfigurations();

// Typically used with NATS messaging
publisher.publish("client.update.topic", objectMapper.writeValueAsString(updateMessage));
```

This message model is commonly used in microservice architectures to notify clients about available updates and their download options through NATS messaging infrastructure.
