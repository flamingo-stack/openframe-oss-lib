A message model for client update notifications sent via NATS messaging system, containing version information and download configurations for OpenFrame clients.

## Key Components

- **version** - String field representing the client version to update to
- **downloadConfigurations** - List of DownloadConfiguration objects specifying where and how to download update files
- **Lombok annotations** - @Getter and @Setter for automatic accessor method generation

## Usage Example

```java
// Create update message
OpenFrameClientUpdateMessage updateMessage = new OpenFrameClientUpdateMessage();
updateMessage.setVersion("2.1.0");

// Add download configurations
List<DownloadConfiguration> configs = new ArrayList<>();
configs.add(new DownloadConfiguration("https://releases.openframe.io/client-2.1.0.jar", "client"));
configs.add(new DownloadConfiguration("https://releases.openframe.io/config-2.1.0.xml", "config"));
updateMessage.setDownloadConfigurations(configs);

// Send via NATS
natsConnection.publish("openframe.client.update", 
    objectMapper.writeValueAsBytes(updateMessage));
```

This model serves as a data transfer object for coordinating client updates across the OpenFrame system, allowing centralized version management and distribution of update packages through the NATS messaging infrastructure.
