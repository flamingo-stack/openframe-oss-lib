A data model class for log projections in Apache Pinot, representing structured log events with metadata for analytics and querying.

## Key Components

- **toolEventId** - Unique identifier for the tool event
- **ingestDay** - Day-based partitioning field for time-series data
- **toolType** - Type of tool that generated the log event
- **eventType** - Classification of the event (e.g., ERROR, INFO, WARN)
- **severity** - Severity level of the log event
- **userId** - User associated with the event
- **deviceId** - Device identifier where the event occurred
- **hostname** - Server/host that generated the event
- **organizationId** - Organization identifier for multi-tenant support
- **organizationName** - Human-readable organization name
- **summary** - Brief description of the log event
- **eventTimestamp** - Precise timestamp when the event occurred

## Usage Example

```java
// Creating a log projection for analytics
LogProjection logEvent = LogProjection.builder()
    .toolEventId("evt_123456")
    .ingestDay("2024-01-15")
    .toolType("deployment-service")
    .eventType("ERROR")
    .severity("HIGH")
    .userId("user_789")
    .deviceId("device_456")
    .hostname("prod-server-01")
    .organizationId("org_123")
    .organizationName("Acme Corp")
    .summary("Failed to deploy application")
    .eventTimestamp(Instant.now())
    .build();

// Used for Pinot ingestion and time-series analytics
```
