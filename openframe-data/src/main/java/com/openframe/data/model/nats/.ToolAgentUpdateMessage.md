A NATS message model for communicating tool agent update information, including version details and download configurations.

## Key Components

- **ToolAgentUpdateMessage**: Main data transfer object containing tool agent update metadata
- **toolAgentId**: Unique identifier for the tool agent being updated
- **version**: Version string of the update
- **sessionType**: Enum indicating the type of session (from SessionType)
- **downloadConfigurations**: List of download configuration objects for the update

## Usage Example

```java
// Create a tool agent update message
ToolAgentUpdateMessage updateMessage = new ToolAgentUpdateMessage();
updateMessage.setToolAgentId("agent-123");
updateMessage.setVersion("2.1.0");
updateMessage.setSessionType(SessionType.PRODUCTION);

// Add download configurations
List<DownloadConfiguration> configs = Arrays.asList(
    new DownloadConfiguration("https://updates.example.com/agent-123-2.1.0.jar"),
    new DownloadConfiguration("https://updates.example.com/agent-123-2.1.0-config.xml")
);
updateMessage.setDownloadConfigurations(configs);

// Use with NATS publisher/subscriber
natsTemplate.publish("tool.agent.updates", updateMessage);
```
