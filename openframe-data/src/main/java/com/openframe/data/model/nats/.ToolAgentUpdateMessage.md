A data model class that represents update messages for ToolAgent instances, carrying version information and download configurations for NATS messaging.

## Key Components

- **toolAgentId** - Unique identifier for the target ToolAgent
- **version** - Version string for the update
- **sessionType** - Enum defining the session type from `SessionType`
- **downloadConfigurations** - List of `DownloadConfiguration` objects specifying download details

## Usage Example

```java
ToolAgentUpdateMessage updateMessage = new ToolAgentUpdateMessage();
updateMessage.setToolAgentId("agent-123");
updateMessage.setVersion("1.2.3");
updateMessage.setSessionType(SessionType.INTERACTIVE);

List<DownloadConfiguration> configs = Arrays.asList(
    new DownloadConfiguration("https://example.com/update.zip", "/tmp/agent-update")
);
updateMessage.setDownloadConfigurations(configs);

// Send via NATS messaging system
natsTemplate.convertAndSend("tool.agent.updates", updateMessage);
```

This message class enables coordinated updates of distributed ToolAgent instances through the NATS messaging infrastructure, supporting different session types and multiple download sources per update.
