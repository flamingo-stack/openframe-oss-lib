A simple data model class that represents configuration settings for downloading platform-specific files, particularly for agent distributions across different operating systems.

## Key Components

- **os** - Target operating system identifier
- **fileName** - Name of the main download file
- **agentFileName** - Name of the agent-specific file
- **link** - Download URL or path to the file

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods.

## Usage Example

```java
// Create download configuration for Windows
DownloadConfiguration windowsConfig = new DownloadConfiguration();
windowsConfig.setOs("windows");
windowsConfig.setFileName("openframe-installer.exe");
windowsConfig.setAgentFileName("openframe-agent.exe");
windowsConfig.setLink("https://releases.openframe.com/v1.0.0/windows/");

// Create configuration for Linux
DownloadConfiguration linuxConfig = new DownloadConfiguration();
linuxConfig.setOs("linux");
linuxConfig.setFileName("openframe-installer.tar.gz");
linuxConfig.setAgentFileName("openframe-agent");
linuxConfig.setLink("https://releases.openframe.com/v1.0.0/linux/");

// Access properties
String osType = windowsConfig.getOs(); // "windows"
String downloadUrl = linuxConfig.getLink(); // "https://releases.openframe.com/v1.0.0/linux/"
```

This model is typically used in NATS messaging contexts to distribute download configuration information across services in the OpenFrame ecosystem.
