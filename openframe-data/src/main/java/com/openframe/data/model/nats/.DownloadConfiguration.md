A simple data model class that represents configuration settings for downloading files, particularly for agent-related downloads across different operating systems.

## Key Components

- **os**: String field specifying the target operating system
- **fileName**: String field for the downloaded file name
- **agentFileName**: String field for the agent-specific file name
- **link**: String field containing the download URL
- **@Data**: Lombok annotation that auto-generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a download configuration for Windows
DownloadConfiguration config = new DownloadConfiguration();
config.setOs("windows");
config.setFileName("installer.exe");
config.setAgentFileName("agent-v1.2.exe");
config.setLink("https://example.com/downloads/agent-v1.2.exe");

// Access configuration data
String downloadUrl = config.getLink();
String targetOS = config.getOs();

// Use in a collection for multiple OS support
List<DownloadConfiguration> configs = Arrays.asList(
    new DownloadConfiguration("linux", "installer.tar.gz", "agent-linux", "https://example.com/linux-agent"),
    new DownloadConfiguration("windows", "installer.exe", "agent-win.exe", "https://example.com/win-agent")
);
```

This class is typically used in NATS messaging contexts to communicate download instructions between services.
