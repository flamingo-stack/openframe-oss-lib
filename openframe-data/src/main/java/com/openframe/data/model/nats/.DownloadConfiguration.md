A data model class that represents configuration settings for downloading and installing software packages or agents, designed for NATS messaging integration.

## Key Components

### Properties
- **`os`** - Target operating system identifier
- **`fileName`** - Original filename of the downloadable package
- **`targetFileName`** - Final filename after installation (mapped from JSON `agentFileName`)
- **`link`** - Download URL for the package
- **`installationType`** - Installation method (defaults to `STANDARD`)
- **`bundleId`** - Unique identifier for the software bundle

### Annotations
- **`@Data`** - Lombok annotation providing getters, setters, toString, equals, and hashCode
- **`@JsonProperty`** - Jackson annotation mapping `agentFileName` JSON field to `targetFileName` property

## Usage Example

```java
// Create download configuration
DownloadConfiguration config = new DownloadConfiguration();
config.setOs("linux");
config.setFileName("agent-installer.deb");
config.setTargetFileName("openframe-agent");
config.setLink("https://releases.openframe.ai/agent/latest/linux/agent.deb");
config.setInstallationType(InstallationType.DOCKER);
config.setBundleId("openframe-agent-v2.1.0");

// Serialize to JSON for NATS messaging
ObjectMapper mapper = new ObjectMapper();
String json = mapper.writeValueAsString(config);
```

The class supports JSON serialization/deserialization for messaging systems and provides flexible configuration for cross-platform software distribution in the OpenFrame ecosystem.
