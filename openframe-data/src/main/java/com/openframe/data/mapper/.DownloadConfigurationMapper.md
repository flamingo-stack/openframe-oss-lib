A Spring component that transforms download configuration objects from document format to NATS message format, resolving download links in the process.

## Key Components

- **`map()`** - Main mapping method that converts a list of document download configurations to NATS format
- **`mapToNatsDownloadConfiguration()`** - Private helper method that handles individual configuration mapping
- **`DownloadConfigurationLinkResolver`** - Injected dependency for resolving download links with version information

## Usage Example

```java
@Autowired
private DownloadConfigurationMapper mapper;

// Map document configurations to NATS format
List<com.openframe.data.document.clientconfiguration.DownloadConfiguration> docConfigs = 
    getDocumentConfigurations();
String version = "1.2.3";

List<com.openframe.data.model.nats.DownloadConfiguration> natsConfigs = 
    mapper.map(docConfigs, version);

// The mapper handles null input gracefully
List<com.openframe.data.model.nats.DownloadConfiguration> result = 
    mapper.map(null, version); // Returns null
```

The mapper preserves OS type, file names, and agent file names while using the resolver to generate appropriate download links based on the provided version.
