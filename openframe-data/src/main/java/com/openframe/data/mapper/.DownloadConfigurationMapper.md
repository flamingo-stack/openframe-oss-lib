A Spring component that maps download configuration data between document and NATS model representations, resolving download links for specific versions.

## Key Components

- **`map()`** - Transforms a list of document-based download configurations to NATS model format
- **`mapToNatsDownloadConfiguration()`** - Private method handling individual configuration mapping
- **`DownloadConfigurationLinkResolver`** - Injected dependency for resolving download links with version context

## Usage Example

```java
@Service
public class ConfigurationService {
    
    @Autowired
    private DownloadConfigurationMapper mapper;
    
    public void processConfigurations() {
        // Source configurations from document model
        List<com.openframe.data.document.clientconfiguration.DownloadConfiguration> docConfigs = 
            getDocumentConfigurations();
        
        String version = "1.2.3";
        
        // Map to NATS model with version-specific links
        List<com.openframe.data.model.nats.DownloadConfiguration> natsConfigs = 
            mapper.map(docConfigs, version);
        
        // Process mapped configurations
        natsConfigs.forEach(config -> {
            System.out.println("OS: " + config.getOs());
            System.out.println("File: " + config.getFileName());
            System.out.println("Link: " + config.getLink());
        });
    }
}
```

The mapper handles null-safe transformation and leverages the link resolver to generate version-specific download URLs for each configuration.
