A Spring component that maps local filename configurations from document model to NATS message model for tool installation messaging.

## Key Components

- **`map()`** - Main mapping method that converts a list of document-based `LocalFilenameConfiguration` objects to NATS message format
- **`mapToNatsLocalFilenameConfiguration()`** - Private helper method that handles individual object mapping between the two models

## Usage Example

```java
@Autowired
private LocalFilenameConfigurationMapper mapper;

public void processToolInstallation() {
    // Document model configurations
    List<com.openframe.data.document.toolagent.LocalFilenameConfiguration> docConfigs = 
        getLocalFilenameConfigurations();
    
    // Map to NATS message format
    List<com.openframe.data.model.nats.ToolInstallationMessage.LocalFilenameConfiguration> natsConfigs = 
        mapper.map(docConfigs);
    
    // Use in tool installation message
    ToolInstallationMessage message = new ToolInstallationMessage();
    message.setLocalFilenameConfigurations(natsConfigs);
    publishMessage(message);
}
```

The mapper handles null-safe conversion and preserves filename and operating system properties during the transformation from the document persistence layer to the NATS messaging layer.
