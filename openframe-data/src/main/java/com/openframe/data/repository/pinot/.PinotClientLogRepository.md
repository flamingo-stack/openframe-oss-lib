Repository implementation that provides log querying capabilities using Apache Pinot as the underlying data store. It implements the PinotLogRepository interface to handle log retrieval, filtering, and search operations.

## Key Components

- **PinotClientLogRepository**: Main repository class implementing PinotLogRepository interface
- **Connection pinotConnection**: Apache Pinot broker connection for executing queries
- **PinotQueryBuilder**: Internal query builder for constructing Pinot SQL queries
- **LogProjection**: Data model for log query results
- **OrganizationOption**: Data model for organization filter options

### Primary Methods

- `findLogs()`: Retrieves logs with filtering by date range, tool types, event types, severities, and organizations
- `searchLogs()`: Performs text search on logs with relevance-based filtering
- `getEventTypeOptions()`: Returns available event type filter options
- `getSeverityOptions()`: Returns available severity level filter options  
- `getToolTypeOptions()`: Returns available tool type filter options
- `getOrganizationOptions()`: Returns available organization filter options with ID and name
- `getAvailableDateRanges()`: Returns available date ranges for logs

## Usage Example

```java
@Autowired
private PinotLogRepository logRepository;

public void queryLogs() {
    LocalDate startDate = LocalDate.of(2024, 1, 1);
    LocalDate endDate = LocalDate.of(2024, 1, 31);
    
    // Find logs with filters
    List<LogProjection> logs = logRepository.findLogs(
        startDate, endDate,
        List.of("API", "WEB"),     // toolTypes
        List.of("ERROR", "WARN"),  // eventTypes
        List.of("HIGH", "MEDIUM"), // severities
        List.of("org1", "org2"),   // organizationIds
        "device123",               // deviceId
        null,                      // cursor
        100                        // limit
    );
    
    // Search logs with text query
    List<LogProjection> searchResults = logRepository.searchLogs(
        startDate, endDate,
        null, null, null, null,
        null,
        "authentication failed",   // searchTerm
        null, 50
    );
    
    // Get filter options
    List<String> eventTypes = logRepository.getEventTypeOptions(
        startDate, endDate, null, null, null, null
    );
}
```
