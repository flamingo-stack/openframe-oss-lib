Repository interface for querying log data from Apache Pinot, providing comprehensive log search, filtering, and metadata retrieval capabilities with multi-dimensional filtering support.

## Key Components

- **`findLogs()`** - Retrieves logs with date range, filtering, and pagination
- **`searchLogs()`** - Full-text search across logs with the same filtering options
- **Filter Option Methods** - Dynamic filter value retrieval for UI components:
  - `getToolTypeOptions()` - Available tool types for current filters
  - `getEventTypeOptions()` - Available event types for current filters  
  - `getSeverityOptions()` - Available severity levels for current filters
  - `getOrganizationOptions()` - Available organizations with metadata
- **`getAvailableDateRanges()`** - Returns valid date ranges for given filter criteria

## Usage Example

```java
@Service
public class LogService {
    
    @Autowired
    private PinotLogRepository logRepository;
    
    public List<LogProjection> getErrorLogs(String orgId) {
        return logRepository.findLogs(
            LocalDate.now().minusDays(7),  // startDate
            LocalDate.now(),               // endDate
            null,                          // all toolTypes
            null,                          // all eventTypes
            List.of("ERROR", "CRITICAL"),  // severities
            List.of(orgId),                // organizationIds
            null,                          // any deviceId
            null,                          // no cursor
            100                            // limit
        );
    }
    
    public List<String> getAvailableToolTypes(String orgId) {
        return logRepository.getToolTypeOptions(
            LocalDate.now().minusDays(30),
            LocalDate.now(),
            null, null, null,
            List.of(orgId)
        );
    }
}
```
