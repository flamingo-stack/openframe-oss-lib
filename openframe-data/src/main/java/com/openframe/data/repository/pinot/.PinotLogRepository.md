A Spring Data repository interface for querying log data from Apache Pinot, providing comprehensive filtering and search capabilities with support for pagination and faceted filtering options.

## Key Components

- **findLogs()** - Retrieves filtered log entries with pagination support
- **searchLogs()** - Performs text search across log entries with same filtering capabilities
- **getToolTypeOptions()** - Returns available tool type values for filter dropdowns
- **getEventTypeOptions()** - Returns available event type values for filter dropdowns  
- **getSeverityOptions()** - Returns available severity level values for filter dropdowns
- **getAvailableDateRanges()** - Returns available date ranges for temporal filtering
- **getOrganizationOptions()** - Returns organization filter options with metadata

## Usage Example

```java
@Service
public class LogService {
    
    @Autowired
    private PinotLogRepository logRepository;
    
    public List<LogProjection> getRecentLogs() {
        return logRepository.findLogs(
            LocalDate.now().minusDays(7),  // startDate
            LocalDate.now(),               // endDate
            Arrays.asList("API", "WEB"),   // toolTypes
            null,                          // eventTypes (all)
            Arrays.asList("ERROR"),        // severities
            null,                          // organizationIds (all)
            null,                          // deviceId
            null,                          // cursor
            100                            // limit
        );
    }
    
    public List<String> getFilterOptions() {
        return logRepository.getToolTypeOptions(
            LocalDate.now().minusDays(30),
            LocalDate.now(),
            null, null, null, null
        );
    }
}
```
