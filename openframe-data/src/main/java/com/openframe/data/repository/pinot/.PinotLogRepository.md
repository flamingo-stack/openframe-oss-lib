Spring Data repository interface for querying log data stored in Apache Pinot, providing advanced filtering, searching, and metadata operations for MSP platform analytics.

## Key Components

- **findLogs()** - Retrieves filtered log entries with pagination and sorting support
- **searchLogs()** - Performs full-text search across log entries with same filtering capabilities
- **Filter Option Methods** - Provides available values for tool types, event types, severities, organizations, and date ranges
- **Sorting Utilities** - Validates sortable fields and provides default sort configuration

## Usage Example

```java
@Service
public class LogAnalyticsService {
    
    @Autowired
    private PinotLogRepository logRepository;
    
    public List<LogProjection> getRecentErrors() {
        return logRepository.findLogs(
            LocalDate.now().minusDays(7),  // startDate
            LocalDate.now(),               // endDate
            List.of("RMM", "PSA"),        // toolTypes
            List.of("ERROR", "CRITICAL"),  // eventTypes
            List.of("HIGH", "CRITICAL"),   // severities
            null,                         // organizationIds (all)
            null,                         // deviceId (all)
            null,                         // cursor (first page)
            100,                          // limit
            "timestamp",                  // sortField
            "DESC"                        // sortDirection
        );
    }
    
    public List<String> getFilterOptions() {
        LocalDate start = LocalDate.now().minusDays(30);
        LocalDate end = LocalDate.now();
        
        return logRepository.getToolTypeOptions(start, end, null, null, null, null);
    }
}
```
