Repository for synchronizing API key statistics from Redis using blocking operations. Provides methods to retrieve, parse, and manage statistics data stored as Redis hash structures.

## Key Components

- **getAllStatsKeys()** - Retrieves all Redis keys matching the stats pattern
- **getStatsData(String redisKey)** - Fetches complete hash data for a specific stats key
- **deleteStatsKey(String redisKey)** - Removes a stats key from Redis
- **extractKeyId(String redisKey)** - Extracts the API key ID from the Redis key format
- **getLong(Map, String)** - Safely parses Long values from Redis hash data with error handling
- **getString(Map, String)** - Extracts String values from Redis hash data

## Usage Example

```java
@Service
public class StatsService {
    
    @Autowired
    private ApiKeyStatsSyncRepository statsRepository;
    
    public void syncApiKeyStats() {
        // Get all stats keys
        Set<String> statsKeys = statsRepository.getAllStatsKeys();
        
        for (String redisKey : statsKeys) {
            // Extract the API key ID
            String keyId = statsRepository.extractKeyId(redisKey);
            
            // Get hash data
            Map<Object, Object> data = statsRepository.getStatsData(redisKey);
            
            // Parse statistics
            Long requestCount = statsRepository.getLong(data, "requests");
            Long errorCount = statsRepository.getLong(data, "errors");
            String lastUsed = statsRepository.getString(data, "lastUsed");
            
            // Process and persist to database
            processStats(keyId, requestCount, errorCount, lastUsed);
            
            // Clean up Redis key after processing
            statsRepository.deleteStatsKey(redisKey);
        }
    }
}
```
