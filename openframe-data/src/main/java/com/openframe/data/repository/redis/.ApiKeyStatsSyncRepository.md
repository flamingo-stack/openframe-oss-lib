Provides synchronous Redis operations for managing API key statistics data with blocking read/write operations and data parsing utilities.

## Key Components

- **getAllStatsKeys()** - Retrieves all Redis keys matching the stats prefix pattern
- **getStatsData(String redisKey)** - Fetches hash entries for a specific stats key
- **deleteStatsKey(String redisKey)** - Removes a stats key from Redis
- **extractKeyId(String redisKey)** - Extracts the key ID by removing the stats prefix
- **getLong(Map, String)** - Safely parses Long values from Redis data with error handling
- **getString(Map, String)** - Extracts String values from Redis hash data

## Usage Example

```java
@Autowired
private ApiKeyStatsSyncRepository statsRepository;

// Get all API key statistics
Set<String> allKeys = statsRepository.getAllStatsKeys();

// Process each key's statistics
for (String key : allKeys) {
    Map<Object, Object> statsData = statsRepository.getStatsData(key);
    
    String keyId = statsRepository.extractKeyId(key);
    Long requestCount = statsRepository.getLong(statsData, "requests");
    String lastUsed = statsRepository.getString(statsData, "lastUsed");
    
    // Process statistics...
    
    // Cleanup if needed
    statsRepository.deleteStatsKey(key);
}
```
