A Spring service for publishing messages to NATS messaging system via Spring Cloud Stream, providing type-safe message publishing with error handling and logging.

## Key Components

- **NatsMessagePublisher**: Main service class for NATS message publishing
- **publish()**: Generic method for publishing typed payloads to specified NATS subjects
- **StreamBridge**: Spring Cloud Stream component for sending messages to message brokers
- **@ConditionalOnProperty**: Conditionally enables the service when Spring Cloud Stream is configured

## Usage Example

```java
@Autowired
private NatsMessagePublisher publisher;

// Publishing different payload types
public void publishUserEvent() {
    UserCreatedEvent event = new UserCreatedEvent("user123", "john@example.com");
    publisher.publish("user.created", event);
}

public void publishOrderNotification() {
    OrderStatus status = new OrderStatus("order456", "COMPLETED");
    publisher.publish("order.status", status);
}

// Publishing simple string messages
public void publishAlert() {
    publisher.publish("system.alerts", "High CPU usage detected");
}
```

The service automatically builds Spring messages from payloads, logs successful publications, and throws `NatsException` for any failures. It's only active when `spring.cloud.stream.enabled` property is set to true.
