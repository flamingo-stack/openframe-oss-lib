A Spring service that provides a simplified interface for publishing messages to NATS subjects using Spring Cloud Stream's StreamBridge. The service includes error handling and logging for message publication operations.

## Key Components

- **NatsMessagePublisher** - Main service class for publishing messages to NATS
- **publish(String subject, T payload)** - Generic method to publish any payload to a specified NATS subject
- **StreamBridge** - Spring Cloud Stream component used for actual message sending
- **@ConditionalOnProperty** - Only activates when Spring Cloud Stream is enabled

## Usage Example

```java
@Autowired
private NatsMessagePublisher natsPublisher;

// Publishing a simple string message
natsPublisher.publish("user.events", "User logged in");

// Publishing a complex object
UserEvent event = new UserEvent("user123", "LOGIN", Instant.now());
natsPublisher.publish("user.events", event);

// Publishing to different subjects
OrderCreated order = new OrderCreated(orderId, customerId);
natsPublisher.publish("order.created", order);
```

The service automatically wraps payloads in Spring Message objects and provides comprehensive error handling. Failed publications throw `NatsException` with descriptive error messages. Successful publications are logged for monitoring purposes.
