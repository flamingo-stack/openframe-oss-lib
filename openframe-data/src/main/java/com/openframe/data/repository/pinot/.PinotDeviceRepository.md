Repository interface for querying device data from Apache Pinot with advanced filtering capabilities. Provides methods for retrieving filter options and device counts based on multiple criteria.

## Key Components

- **Filter Option Methods**: Five methods (`getStatusFilterOptions`, `getDeviceTypeFilterOptions`, `getOsTypeFilterOptions`, `getOrganizationFilterOptions`, `getTagFilterOptions`) that return available filter values with their counts
- **Device Count Method**: `getFilteredDeviceCount` returns the total number of devices matching the specified filter criteria
- **Consistent Filter Parameters**: All methods accept the same five filter parameters for consistent querying

## Usage Example

```java
@Autowired
private PinotDeviceRepository deviceRepository;

public void getDeviceAnalytics() {
    List<String> statuses = Arrays.asList("active", "inactive");
    List<String> deviceTypes = Arrays.asList("mobile", "tablet");
    List<String> osTypes = Arrays.asList("iOS", "Android");
    List<String> organizationIds = Arrays.asList("org1", "org2");
    List<String> tagNames = Arrays.asList("production", "testing");
    
    // Get available status options with counts
    Map<String, Integer> statusOptions = deviceRepository.getStatusFilterOptions(
        statuses, deviceTypes, osTypes, organizationIds, tagNames
    );
    
    // Get total filtered device count
    int totalDevices = deviceRepository.getFilteredDeviceCount(
        statuses, deviceTypes, osTypes, organizationIds, tagNames
    );
    
    System.out.println("Status options: " + statusOptions);
    System.out.println("Total devices: " + totalDevices);
}
```
