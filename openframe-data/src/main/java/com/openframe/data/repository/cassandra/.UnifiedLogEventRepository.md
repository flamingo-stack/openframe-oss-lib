A Spring Data Cassandra repository interface for managing UnifiedLogEvent entities, conditionally enabled based on Cassandra configuration.

## Key Components

- **UnifiedLogEventRepository**: Main repository interface extending CassandraRepository
- **@ConditionalOnProperty**: Conditional bean creation based on Cassandra enablement
- **Generic Types**: 
  - Entity: `UnifiedLogEvent` 
  - Key: `UnifiedLogEvent.UnifiedLogEventKey` (composite key)

## Usage Example

```java
@Service
public class LogEventService {
    
    @Autowired(required = false)
    private UnifiedLogEventRepository repository;
    
    public void saveLogEvent(UnifiedLogEvent event) {
        if (repository != null) {
            repository.save(event);
        }
    }
    
    public List<UnifiedLogEvent> getAllEvents() {
        return repository != null ? 
            repository.findAll() : 
            Collections.emptyList();
    }
    
    public Optional<UnifiedLogEvent> findById(UnifiedLogEvent.UnifiedLogEventKey key) {
        return repository != null ? 
            repository.findById(key) : 
            Optional.empty();
    }
}
```

The repository is only instantiated when `spring.data.cassandra.enabled=true` is set in application properties, making it safe to use in environments where Cassandra may not be available. Inherits standard CRUD operations from CassandraRepository.
