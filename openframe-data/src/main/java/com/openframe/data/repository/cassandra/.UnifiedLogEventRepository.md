This repository interface provides Cassandra database access for unified log event entities, conditionally enabled based on application configuration.

## Key Components

- **UnifiedLogEventRepository** - Spring Data repository interface extending `CassandraRepository`
- **Entity Type** - `UnifiedLogEvent` with composite key `UnifiedLogEvent.UnifiedLogEventKey`
- **Conditional Activation** - `@ConditionalOnProperty` annotation ensures repository is only active when Cassandra is enabled

## Usage Example

```java
@Service
public class LogEventService {
    
    @Autowired
    private UnifiedLogEventRepository repository;
    
    public void saveLogEvent(UnifiedLogEvent event) {
        repository.save(event);
    }
    
    public Optional<UnifiedLogEvent> findById(UnifiedLogEvent.UnifiedLogEventKey key) {
        return repository.findById(key);
    }
    
    public List<UnifiedLogEvent> findAll() {
        return repository.findAll();
    }
}
```

**Configuration requirement:**
```properties
spring.data.cassandra.enabled=true
```

The repository inherits standard CRUD operations from `CassandraRepository` and will only be instantiated when Cassandra support is explicitly enabled in the application configuration.
