A fluent SQL query builder specifically designed for Apache Pinot database queries, providing type-safe construction of SELECT statements with comprehensive filtering, sorting, and aggregation capabilities.

## Key Components

- **Core Builder Methods**: `select()`, `where()`, `orderBy()`, `groupBy()`, `limit()` for basic SQL construction
- **Specialized Filters**: `whereEquals()`, `whereGreaterThan()`, `whereDateRange()`, `whereTextSearch()`, `whereLike()`, `whereIn()`, `whereOr()`  
- **Pinot-Specific Features**: Text search with `TEXT_MATCH` function, timestamp-based ordering, cursor-based pagination support
- **Aggregation Support**: `selectCount()`, `selectCountAll()`, `groupByWithCount()` for analytical queries
- **Security**: Built-in SQL injection protection through value escaping and input validation
- **Constants**: Predefined field names, operators, and limits specific to Pinot schema conventions

## Usage Example

```java
// Basic query with filtering and ordering
String query = new PinotQueryBuilder("device_events")
    .select("deviceId", "eventType", "eventTimestamp")
    .whereEquals("tenantId", "tenant123")
    .whereDateRange("eventTimestamp", LocalDate.now().minusDays(7), LocalDate.now())
    .whereTextSearch("error network", "message", "description")
    .orderByTimestampDesc()
    .limit(100)
    .build();

// Aggregation query with grouping
String countQuery = new PinotQueryBuilder("device_events")
    .selectCount()
    .select("deviceType")
    .whereGreaterThan("eventTimestamp", "1640995200000")
    .whereIn("status", "active", "pending")
    .groupBy("deviceType")
    .orderByCountDesc()
    .build();

// Complex filtering with multiple conditions
String complexQuery = new PinotQueryBuilder("audit_logs")
    .distinct()
    .select("userId", "action", "resourceId")
    .whereEquals("tenantId", "org456")
    .whereNotEquals("status", "deleted")
    .whereOr("priority", Arrays.asList("high", "critical"))
    .whereLike("action", "update")
    .orderBySortInput("createdAt", "DESC", "logId")
    .limit(500)
    .build();
```
