A fluent builder class for constructing Apache Pinot SQL queries with comprehensive filtering, ordering, and aggregation capabilities. Provides type-safe query construction with built-in validation and security features like SQL injection protection.

## Key Components

- **Core Builder Methods**: `select()`, `where()`, `orderBy()`, `groupBy()`, `limit()`
- **Specialized Filters**: `whereEquals()`, `whereTextSearch()`, `whereLike()`, `whereDateRange()`, `whereIn()`
- **Aggregation Support**: `selectCount()`, `selectCountAll()`, `orderByCountDesc()`
- **Timestamp Operations**: `orderByTimestampDesc()`, `orderByTimestampAsc()`
- **Security Features**: SQL injection prevention through `escapeSqlValue()`
- **Validation**: Input validation for table names, field names, and limits

## Usage Example

```java
// Basic query with filtering and ordering
String query = new PinotQueryBuilder("events_table")
    .select("userId", "eventType", "eventTimestamp")
    .whereEquals("status", "active")
    .whereDateRange("eventTimestamp", 
        LocalDate.now().minusDays(7), 
        LocalDate.now())
    .whereIn("category", "login", "logout", "signup")
    .orderByTimestampDesc()
    .limit(100)
    .build();

// Count query with grouping
String countQuery = new PinotQueryBuilder("user_events")
    .selectCount()
    .whereTextSearch("error", "message", "description")
    .groupBy("eventType")
    .orderByCountDesc()
    .build();

// Complex filtering with text search
String searchQuery = new PinotQueryBuilder("logs")
    .distinct()
    .select("toolId", "message")
    .whereGreaterThanOrEqual("severity", "3")
    .whereTextSearch("database connection", "message", "stackTrace")
    .whereLike("source", "api")
    .orderBy("severity DESC", "eventTimestamp DESC")
    .limit(50)
    .build();
```
