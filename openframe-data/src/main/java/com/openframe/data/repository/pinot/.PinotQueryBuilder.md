A fluent builder class for constructing SQL queries specifically designed for Apache Pinot database operations with built-in validation and security features.

## Key Components

**Main Class:**
- `PinotQueryBuilder` - Fluent builder for Pinot SQL queries with method chaining

**Core Methods:**
- `select(String... columns)` - Select specific columns
- `selectCount()` / `selectCountAll()` - Count queries with aliases
- `where*()` methods - Various where conditions (equals, greater than, date range, text search, etc.)
- `orderBy*()` methods - Sorting with timestamp-specific helpers
- `groupBy()` / `limit()` - Grouping and result limiting
- `build()` - Generates the final SQL query string

**Security Features:**
- SQL injection prevention through value escaping
- Input validation for table names, columns, and limits
- Parameterized query construction

## Usage Example

```java
// Basic query with filtering and ordering
String query = new PinotQueryBuilder("events_table")
    .select("userId", "eventType", "eventTimestamp")
    .whereEquals("deviceType", "mobile")
    .whereDateRange("eventTimestamp", startDate, endDate)
    .whereIn("region", Arrays.asList("US", "EU", "APAC"))
    .orderByTimestampDesc()
    .limit(1000)
    .build();

// Count query with grouping
String countQuery = new PinotQueryBuilder("events_table")
    .selectCount()
    .whereTextSearch("error", "message", "description")
    .groupBy("eventType")
    .orderByCountDesc()
    .build();

// Text search across multiple columns
String searchQuery = new PinotQueryBuilder("logs_table")
    .select("*")
    .whereTextSearch("exception", "logMessage", "stackTrace")
    .whereLike("service", "payment")
    .limit(500)
    .build();
```
