A Spring service that manages HTTP cookies for authentication and OAuth state management, providing secure cookie creation, clearing, and retrieval functionality for access tokens, refresh tokens, and OAuth state parameters.

## Key Components

- **Cookie Creation Methods**: `addAuthCookies()`, `addOAuthStateCookie()` - Creates secure, HTTP-only cookies with configurable domain and SameSite settings
- **Cookie Clearing Methods**: `addClearAuthCookies()`, `addClearSasCookies()`, `addClearOAuthStateCookie()` - Removes authentication and session cookies
- **Cookie Retrieval**: `getAccessTokenFromCookies()` - Extracts access tokens from incoming requests
- **Configuration Properties**: Supports customizable cookie domain, security flags, expiration times, and SameSite policies

## Usage Example

```java
@Autowired
private CookieService cookieService;

// Setting authentication cookies
public ResponseEntity<?> login(String accessToken, String refreshToken) {
    HttpHeaders headers = new HttpHeaders();
    cookieService.addAuthCookies(headers, accessToken, refreshToken);
    return ResponseEntity.ok().headers(headers).build();
}

// Clearing cookies on logout
public ResponseEntity<?> logout() {
    HttpHeaders headers = new HttpHeaders();
    cookieService.addClearAuthCookies(headers);
    return ResponseEntity.ok().headers(headers).build();
}

// Retrieving access token from request
public String getCurrentUserToken(ServerWebExchange exchange) {
    return cookieService.getAccessTokenFromCookies(exchange);
}
```
