A utility class that provides cryptographically secure methods for implementing PKCE (Proof Key for Code Exchange) in OAuth2 authentication flows, including generation of state parameters, code verifiers, and code challenges.

## Key Components

- **`generateState()`** - Creates a 128-bit random state parameter for CSRF protection
- **`generateCodeVerifier()`** - Generates a 256-bit cryptographically secure code verifier
- **`generateCodeChallenge(String)`** - Creates SHA-256 based code challenge from verifier
- **`urlEncode(String)`** - URL encodes strings using UTF-8 encoding
- **`base64UrlEncode(byte[])`** - Private helper for Base64URL encoding without padding

## Usage Example

```java
// Generate PKCE parameters for OAuth2 flow
String state = PKCEUtils.generateState();
String codeVerifier = PKCEUtils.generateCodeVerifier();
String codeChallenge = PKCEUtils.generateCodeChallenge(codeVerifier);

// Build authorization URL
String authUrl = "https://auth.example.com/oauth/authorize?" +
    "client_id=your_client&" +
    "redirect_uri=" + PKCEUtils.urlEncode("https://yourapp.com/callback") + "&" +
    "state=" + state + "&" +
    "code_challenge=" + codeChallenge + "&" +
    "code_challenge_method=S256";

// Store codeVerifier securely for token exchange
// Send user to authUrl for authorization
```

The class uses `SecureRandom` for cryptographic security and follows RFC 7636 PKCE specifications with SHA-256 hashing and Base64URL encoding.
