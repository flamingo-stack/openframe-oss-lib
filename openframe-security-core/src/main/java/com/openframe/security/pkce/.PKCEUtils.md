Utility class for implementing PKCE (Proof Key for Code Exchange) security measures in OAuth2 flows. Provides cryptographically secure methods for generating state parameters, code verifiers, and code challenges.

## Key Components

- **`generateState()`** - Creates a random state parameter for CSRF protection
- **`generateCodeVerifier()`** - Generates a cryptographically secure code verifier
- **`generateCodeChallenge(String)`** - Creates SHA-256 code challenge from verifier
- **`urlEncode(String)`** - URL encodes strings using UTF-8
- **`base64UrlEncode(byte[])`** - Base64URL encodes byte arrays (private helper)

## Usage Example

```java
// Generate PKCE parameters for OAuth2 flow
String state = PKCEUtils.generateState();
String codeVerifier = PKCEUtils.generateCodeVerifier();
String codeChallenge = PKCEUtils.generateCodeChallenge(codeVerifier);

// Use in authorization URL
String authUrl = "https://auth.example.com/oauth/authorize?"
    + "client_id=your_client_id"
    + "&state=" + state
    + "&code_challenge=" + codeChallenge
    + "&code_challenge_method=S256";

// Later, exchange code for token using the code verifier
// (verifier is sent in token request, challenge was sent in auth request)
```

The class follows OAuth2 PKCE RFC 7636 standards and uses secure random generation with proper Base64URL encoding.
