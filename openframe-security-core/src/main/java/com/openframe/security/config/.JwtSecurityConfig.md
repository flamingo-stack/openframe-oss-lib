Configures JWT encoding and decoding beans for Spring Security OAuth2 using RSA key pairs. This configuration class sets up the necessary components for JWT token creation and validation in a Spring Boot application.

## Key Components

- **`jwtEncoder(JwtConfig)`** - Creates a `JwtEncoder` bean using RSA public/private key pair for signing JWT tokens
- **`jwtDecoder(JwtConfig)`** - Creates a `JwtDecoder` bean using RSA public key for validating JWT tokens
- **RSA Key Management** - Utilizes Nimbus JOSE library for RSA key handling and JWK (JSON Web Key) operations

## Usage Example

```java
@Service
public class AuthService {
    
    @Autowired
    private JwtEncoder jwtEncoder;
    
    @Autowired
    private JwtDecoder jwtDecoder;
    
    public String generateToken(String subject) {
        JwtClaimsSet claims = JwtClaimsSet.builder()
            .subject(subject)
            .issuedAt(Instant.now())
            .expiresAt(Instant.now().plusSeconds(3600))
            .build();
            
        return jwtEncoder.encode(JwtEncoderParameters.from(claims)).getTokenValue();
    }
    
    public String validateToken(String token) {
        Jwt jwt = jwtDecoder.decode(token);
        return jwt.getSubject();
    }
}
```

The configuration requires a `JwtConfig` bean to provide RSA key loading functionality and automatically registers the JWT components with Spring's application context.
