Defines an enumeration that categorizes authentication principals into two distinct access levels within the OpenFrame security framework.

## Key Components

- **ADMIN** - Represents admin users with full access rights to system resources
- **AGENT** - Represents agent machines with limited, programmatic access rights

## Usage Example

```java
import com.openframe.security.authentication.ActorType;

// Determine access level based on actor type
public boolean hasFullAccess(ActorType actorType) {
    return actorType == ActorType.ADMIN;
}

// Example in authentication context
public void authenticateUser(String userId, ActorType type) {
    if (type == ActorType.ADMIN) {
        grantAdminPrivileges(userId);
    } else if (type == ActorType.AGENT) {
        grantLimitedAccess(userId);
    }
}

// Switch-based permission handling
public Set<Permission> getPermissions(ActorType actorType) {
    switch (actorType) {
        case ADMIN:
            return adminPermissions;
        case AGENT:
            return agentPermissions;
        default:
            return Collections.emptySet();
    }
}
```
