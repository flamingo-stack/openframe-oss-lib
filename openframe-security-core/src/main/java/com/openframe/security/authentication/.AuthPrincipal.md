A clean wrapper class that extracts authenticated user information from JWT tokens, providing convenient access to user claims without working with raw JWT objects.

## Key Components

- **`fromJwt(Jwt jwt)`** - Static factory method to create AuthPrincipal from JWT token
- **User Identity Fields** - `id`, `email`, `firstName`, `lastName` extracted from standard JWT claims
- **Authorization Data** - `roles` and `scopes` for permission checking
- **Tenant Information** - `tenantId` and `tenantDomain` for multi-tenant support
- **Actor Type** - Automatically determined as `ADMIN` (human) or `AGENT` (machine/service)
- **`getDisplayName()`** - Utility method for user-friendly name display

## Usage Example

```java
@RestController
public class UserController {
    
    @GetMapping("/profile")
    public UserProfile getProfile(JwtAuthenticationToken token) {
        AuthPrincipal principal = AuthPrincipal.fromJwt(token.getToken());
        
        return UserProfile.builder()
                .userId(principal.getId())
                .displayName(principal.getDisplayName())
                .email(principal.getEmail())
                .roles(principal.getRoles())
                .tenantId(principal.getTenantId())
                .build();
    }
    
    @PreAuthorize("hasRole('ADMIN')")
    @GetMapping("/admin-only")
    public String adminEndpoint(JwtAuthenticationToken token) {
        AuthPrincipal principal = AuthPrincipal.fromJwt(token.getToken());
        return "Welcome " + principal.getDisplayName();
    }
}
```
