A utility class containing constant string values for OAuth2 authentication and authorization parameters used throughout the security module.

## Key Components

- **AUTHORIZATION_QUERY_PARAM** - Query parameter name for authorization data
- **ACCESS_TOKEN** - Standard OAuth2 access token identifier
- **REFRESH_TOKEN** - Standard OAuth2 refresh token identifier  
- **ACCESS_TOKEN_HEADER** - HTTP header name for access token transmission
- **REFRESH_TOKEN_HEADER** - HTTP header name for refresh token transmission

## Usage Example

```java
// Using constants for token handling
String accessToken = request.getParameter(SecurityConstants.ACCESS_TOKEN);
String refreshToken = request.getHeader(SecurityConstants.REFRESH_TOKEN_HEADER);

// Setting response headers
response.setHeader(SecurityConstants.ACCESS_TOKEN_HEADER, newAccessToken);
response.setHeader(SecurityConstants.REFRESH_TOKEN_HEADER, newRefreshToken);

// Building authorization URLs
String authUrl = baseUrl + "?" + SecurityConstants.AUTHORIZATION_QUERY_PARAM + "=" + authCode;
```

The class follows the utility pattern with a private constructor to prevent instantiation and provides centralized string constants for consistent OAuth2 parameter naming across the application.
