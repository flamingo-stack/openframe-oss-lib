A utility class that defines OAuth security-related string constants used throughout the application for token management and authorization headers.

## Key Components

- **AUTHORIZATION_QUERY_PARAM** - Query parameter name for authorization requests
- **ACCESS_TOKEN** - Standard access token identifier
- **REFRESH_TOKEN** - Standard refresh token identifier  
- **ACCESS_TOKEN_HEADER** - HTTP header name for access tokens
- **REFRESH_TOKEN_HEADER** - HTTP header name for refresh tokens

## Usage Example

```java
// Setting authorization headers
request.setHeader(SecurityConstants.ACCESS_TOKEN_HEADER, userAccessToken);
request.setHeader(SecurityConstants.REFRESH_TOKEN_HEADER, userRefreshToken);

// Building OAuth URLs with query parameters
String authUrl = baseUrl + "?" + SecurityConstants.AUTHORIZATION_QUERY_PARAM + "=" + authCode;

// Extracting tokens from response
String accessToken = response.get(SecurityConstants.ACCESS_TOKEN);
String refreshToken = response.get(SecurityConstants.REFRESH_TOKEN);
```

This constants class ensures consistent naming across the OAuth implementation and reduces the risk of typos in security-critical string literals. The final class with private constructor follows the utility class pattern to prevent instantiation.
