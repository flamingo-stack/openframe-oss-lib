A Spring Boot configuration service that manages JWT RSA key pairs and metadata for JWT token operations, providing convenient methods to load public and private keys from configuration properties.

## Key Components

- **JwtConfig** - Main configuration service annotated with `@ConfigurationProperties(prefix = "jwt")`
- **publicKey/privateKey** - `KeyConfig` objects for storing RSA key data
- **issuer/audience** - JWT standard claims for token validation
- **loadPublicKey()** - Returns configured RSA public key for token verification
- **loadPrivateKey()** - Parses PEM-formatted private key string into `RSAPrivateKey` object

## Usage Example

```java
// Application configuration (application.yml)
jwt:
  issuer: "myapp.com"
  audience: "api-users"
  publicKey:
    value: "-----BEGIN PUBLIC KEY-----..."
  privateKey:
    value: "-----BEGIN PRIVATE KEY-----..."

// Using in a JWT service
@Autowired
private JwtConfig jwtConfig;

public String generateToken(String subject) throws Exception {
    RSAPrivateKey privateKey = jwtConfig.loadPrivateKey();
    return Jwts.builder()
        .setSubject(subject)
        .setIssuer(jwtConfig.getIssuer())
        .setAudience(jwtConfig.getAudience())
        .signWith(privateKey, SignatureAlgorithm.RS256)
        .compact();
}

public Claims validateToken(String token) throws Exception {
    RSAPublicKey publicKey = jwtConfig.loadPublicKey();
    return Jwts.parserBuilder()
        .setSigningKey(publicKey)
        .build()
        .parseClaimsJws(token)
        .getBody();
}
```

The configuration automatically binds properties with the `jwt` prefix and provides clean access to RSA keys for JWT signing and verification operations.
