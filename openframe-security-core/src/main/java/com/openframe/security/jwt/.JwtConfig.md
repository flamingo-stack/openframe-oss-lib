A Spring configuration service that manages JWT RSA key pairs and provides methods to load public and private keys for JWT token operations.

## Key Components

- **JwtConfig**: Main configuration class with JWT-related properties
- **publicKey/privateKey**: KeyConfig instances for RSA key pair management
- **issuer/audience**: JWT token validation properties
- **loadPublicKey()**: Converts configured public key to RSAPublicKey instance
- **loadPrivateKey()**: Parses PEM-formatted private key string to RSAPrivateKey instance

## Usage Example

```java
@Autowired
private JwtConfig jwtConfig;

// Load RSA keys for JWT operations
public void setupJwtSigning() throws Exception {
    RSAPublicKey publicKey = jwtConfig.loadPublicKey();
    RSAPrivateKey privateKey = jwtConfig.loadPrivateKey();
    
    // Use keys with JWT library
    Algorithm algorithm = Algorithm.RSA256(publicKey, privateKey);
    
    // Access other JWT properties
    String issuer = jwtConfig.getIssuer();
    String audience = jwtConfig.getAudience();
}
```

Configure in `application.yml`:
```yaml
jwt:
  issuer: "your-app"
  audience: "api-users"
  public-key:
    value: "-----BEGIN PUBLIC KEY-----..."
  private-key:
    value: "-----BEGIN PRIVATE KEY-----..."
```
