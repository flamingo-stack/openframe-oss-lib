A configuration class that holds RSA public key data in PEM format and provides conversion functionality to Java's RSAPublicKey object for JWT token verification.

## Key Components

- **value** - String field containing the PEM-formatted RSA public key
- **toRSAPublicKey()** - Converts the PEM string to a usable RSAPublicKey instance

## Usage Example

```java
// Configure with PEM-formatted public key
KeyConfig keyConfig = new KeyConfig();
keyConfig.setValue("""
    -----BEGIN PUBLIC KEY-----
    MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8AMIIBCgKCAQEA...
    -----END PUBLIC KEY-----
    """);

// Convert to RSAPublicKey for JWT verification
try {
    RSAPublicKey publicKey = keyConfig.toRSAPublicKey();
    // Use with JWT libraries like jjwt or java-jwt
    // Jwts.parserBuilder().setSigningKey(publicKey).build()
} catch (Exception e) {
    // Handle key parsing errors
    log.error("Failed to parse RSA public key", e);
}
```

The `toRSAPublicKey()` method strips PEM headers/footers and whitespace, decodes the Base64 content, and creates an RSAPublicKey using Java's KeyFactory. This is commonly used in JWT authentication workflows where public keys verify token signatures.
