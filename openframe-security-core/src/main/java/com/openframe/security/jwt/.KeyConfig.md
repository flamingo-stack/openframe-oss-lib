Configuration class for handling RSA public key conversion from PEM format strings. Provides utility methods to parse and convert PEM-formatted public keys into RSAPublicKey objects for cryptographic operations.

## Key Components

- **`value`** - String field containing the PEM-formatted public key
- **`toRSAPublicKey()`** - Converts the PEM string to an RSAPublicKey object by removing PEM headers/footers and decoding the Base64 content

## Usage Example

```java
// Create KeyConfig with PEM-formatted public key
KeyConfig keyConfig = new KeyConfig();
keyConfig.setValue("""
    -----BEGIN PUBLIC KEY-----
    MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8AMIIBCgKCAQEA...
    -----END PUBLIC KEY-----
    """);

// Convert to RSAPublicKey for verification
try {
    RSAPublicKey publicKey = keyConfig.toRSAPublicKey();
    // Use publicKey for JWT signature verification, etc.
} catch (Exception e) {
    // Handle key parsing errors
}

// Alternatively, using Lombok setter
KeyConfig config = new KeyConfig();
config.setValue(pemKeyString);
RSAPublicKey key = config.toRSAPublicKey();
```
