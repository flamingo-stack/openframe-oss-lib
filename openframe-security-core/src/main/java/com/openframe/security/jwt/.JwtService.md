A Spring service that provides JWT token generation and decoding functionality using Spring Security's OAuth2 JWT components.

## Key Components

- **`decodeToken(String token)`** - Decodes and validates a JWT token string, returning a parsed Jwt object
- **`generateToken(JwtClaimsSet claims)`** - Generates a JWT token string from provided claims
- **Dependencies**: Uses Spring's `JwtEncoder` and `JwtDecoder` for token operations

## Usage Example

```java
@Autowired
private JwtService jwtService;

// Generate a token
JwtClaimsSet claims = JwtClaimsSet.builder()
    .subject("user123")
    .claim("role", "ADMIN")
    .expiresAt(Instant.now().plusSeconds(3600))
    .build();
String token = jwtService.generateToken(claims);

// Decode a token
try {
    Jwt decodedJwt = jwtService.decodeToken(token);
    String subject = decodedJwt.getSubject();
    String role = decodedJwt.getClaimAsString("role");
} catch (JwtException e) {
    log.error("Invalid token: {}", e.getMessage());
}
```

The service includes debug logging to track token operations and handles JWT validation automatically through Spring Security's decoder.
