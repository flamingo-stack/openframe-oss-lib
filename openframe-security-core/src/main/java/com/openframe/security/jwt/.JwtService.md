A Spring Security service that handles JWT token encoding and decoding operations for authentication and authorization workflows.

## Key Components

- **JwtEncoder encoder** - Spring Security component for creating JWT tokens
- **JwtDecoder decoder** - Spring Security component for parsing and validating JWT tokens
- **decodeToken(String token)** - Parses a JWT string into a validated Jwt object with debug logging
- **generateToken(JwtClaimsSet claims)** - Creates a JWT token string from provided claims

## Usage Example

```java
@Autowired
private JwtService jwtService;

// Generate a token with custom claims
JwtClaimsSet claims = JwtClaimsSet.builder()
    .subject("user123")
    .claim("role", "ADMIN")
    .expiresAt(Instant.now().plusSeconds(3600))
    .build();

String token = jwtService.generateToken(claims);

// Decode and validate a token
try {
    Jwt decodedJwt = jwtService.decodeToken(token);
    String userId = decodedJwt.getSubject();
    String role = decodedJwt.getClaimAsString("role");
    Instant expiry = decodedJwt.getExpiresAt();
} catch (JwtException e) {
    // Handle invalid token
    log.error("Token validation failed", e);
}
```

This service provides a clean abstraction over Spring Security's JWT components, with built-in logging for debugging token operations and automatic validation during decoding.
