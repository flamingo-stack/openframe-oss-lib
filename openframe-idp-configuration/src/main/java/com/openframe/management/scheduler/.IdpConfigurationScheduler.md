A Spring Boot scheduler responsible for initializing default OAuth2 client configurations in the Identity Provider (IdP) system. It runs once on startup to ensure essential OAuth clients are properly registered.

## Key Components

- **@Scheduled method**: `initializeDefaultIdp()` - Creates default OAuth2 client registration if it doesn't exist
- **@SchedulerLock**: Prevents concurrent execution across multiple instances
- **@ConditionalOnProperty**: Only activates when `openframe.management.idp.init.enabled=true`
- **Dependencies**: `RegisteredClientMongoRepository` for data persistence, `PasswordEncoder` for secret encoding

## Usage Example

```java
// Configuration in application.yml
openframe:
  management:
    idp:
      init:
        enabled: true
  gateway:
    oauth:
      client-id: "gateway-client"
      client-secret: "secret123"
      redirect-uri: "http://localhost:8080/login/oauth2/code/gateway"

// The scheduler automatically creates a MongoRegisteredClient with:
// - Authorization code + refresh token grant types
// - PKCE requirement enabled
// - Standard OpenID Connect scopes
// - Configurable token expiration times
```

The scheduler runs with a fixed delay of `Long.MAX_VALUE` and initial delay of 5 seconds, effectively executing only once after application startup to bootstrap the OAuth2 client configuration.
