A data transfer object (DTO) that represents the status information of a Debezium connector, including its state and associated tasks.

## Key Components

- **ConnectorStatus** - Main DTO class containing connector name, connector details, and task list
- **Connector** - Nested static class representing connector state and worker assignment
- **TaskStatus** - Nested static class representing individual task status with ID, state, worker, and error trace

## Usage Example

```java
// Deserializing connector status from JSON response
ObjectMapper mapper = new ObjectMapper();
ConnectorStatus status = mapper.readValue(jsonResponse, ConnectorStatus.class);

// Accessing connector information
String connectorName = status.getName();
String connectorState = status.getConnector().getState();
String workerId = status.getConnector().getWorkerId();

// Checking task statuses
List<ConnectorStatus.TaskStatus> tasks = status.getTasks();
for (ConnectorStatus.TaskStatus task : tasks) {
    System.out.println("Task " + task.getId() + ": " + task.getState());
    if (task.getTrace() != null) {
        System.out.println("Error trace: " + task.getTrace());
    }
}

// Example states: "RUNNING", "PAUSED", "FAILED"
boolean isHealthy = "RUNNING".equals(status.getConnector().getState());
```

The class uses Jackson annotations for JSON serialization/deserialization and Lombok for automatic getter/setter generation. The `@JsonIgnoreProperties` annotation ensures compatibility when the API returns additional fields not defined in the DTO.
