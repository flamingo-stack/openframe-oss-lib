A data transfer object (DTO) representing a request to release a specific version, containing the image tag version information needed for deployment operations.

## Key Components

- **`imageTagVersion`** - String field storing the Docker image tag version to be released
- **`@Data`** - Lombok annotation providing automatic getter/setter methods, equals(), hashCode(), and toString()

## Usage Example

```java
// Create a release version request
ReleaseVersionRequest request = new ReleaseVersionRequest();
request.setImageTagVersion("v1.2.3");

// Or using builder pattern (if @Builder is added)
ReleaseVersionRequest request = ReleaseVersionRequest.builder()
    .imageTagVersion("v1.2.3-alpha")
    .build();

// Access the version
String version = request.getImageTagVersion();
```

This DTO is typically used in REST API endpoints for version management operations, allowing clients to specify which Docker image version should be deployed or released in the OpenFrame management system.
