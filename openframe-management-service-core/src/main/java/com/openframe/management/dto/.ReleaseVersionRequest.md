A simple data transfer object (DTO) for capturing image tag version information in release version requests.

## Key Components

- **imageTagVersion** - String field to store the version tag for container images or releases

## Usage Example

```java
// Creating a release version request
ReleaseVersionRequest request = new ReleaseVersionRequest();
request.setImageTagVersion("v1.2.3");

// Using with builder pattern (Lombok generates builder)
ReleaseVersionRequest request = ReleaseVersionRequest.builder()
    .imageTagVersion("v2.0.0-beta")
    .build();

// Accessing the version
String version = request.getImageTagVersion();
```

The `@Data` annotation automatically generates getters, setters, `toString()`, `equals()`, and `hashCode()` methods. This DTO is typically used in REST API endpoints or service methods that handle release version operations, allowing clients to specify which image tag version should be deployed or processed.
