Brief interface contract for supplying supplemental NATS stream configurations during application initialization.

## Key Components

- **`AdditionalStreamConfigurationProvider`** — Functional interface that allows modules or plugins to contribute additional `StreamConfiguration` instances beyond the default setup.
- **`provide()`** — Returns a `List<StreamConfiguration>` of NATS JetStream stream definitions to be registered at startup.

## Usage Example

```java
@Component
public class MyCustomStreamProvider implements AdditionalStreamConfigurationProvider {

    @Override
    public List<StreamConfiguration> provide() {
        StreamConfiguration config = StreamConfiguration.builder()
            .name("MY_CUSTOM_STREAM")
            .subjects("custom.events.>")
            .build();

        return List.of(config);
    }
}
```

Register this bean in your Spring context and the initializer will automatically pick up and apply these configurations alongside the platform defaults during startup.
