A Spring Boot application initializer that ensures the agent registration secret is properly set up during application startup.

## Key Components

- **AgentRegistrationSecretInitializer** - Main component implementing `ApplicationRunner` to execute initialization logic at startup
- **agentRegistrationSecretService** - Injected service responsible for creating and managing the initial registration secret
- **run()** - Entry point method that executes the secret initialization with proper error handling and logging

## Usage Example

```java
// This component runs automatically at application startup
// No manual instantiation required - Spring manages the lifecycle

@SpringBootApplication
public class ManagementApplication {
    public static void main(String[] args) {
        SpringApplication.run(ManagementApplication.class, args);
        // AgentRegistrationSecretInitializer.run() executes automatically
    }
}

// The initializer depends on the service being available
@Service
public class AgentRegistrationSecretManagementService {
    public void createInitialSecret() {
        // Implementation handles secret creation logic
    }
}
```

The component uses Spring's `@Order` annotation support (though not explicitly set) to control initialization sequence and `@RequiredArgsConstructor` for clean dependency injection. All initialization attempts are logged with appropriate success/error messages for monitoring and debugging purposes.
