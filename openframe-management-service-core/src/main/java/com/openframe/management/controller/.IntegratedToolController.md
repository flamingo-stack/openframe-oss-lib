REST API controller that manages integrated tool configurations and their associated Debezium connectors for data change capture.

## Key Components

- **IntegratedToolController**: Main REST controller handling tool management operations
- **GET /v1/tools**: Retrieves all available integrated tools
- **GET /v1/tools/{id}**: Fetches a specific tool by ID
- **POST /v1/tools/{id}**: Saves/updates tool configuration and triggers Debezium connector setup
- **SaveToolRequest**: Request DTO for tool save operations
- **IntegratedToolService**: Service for tool data operations
- **DebeziumService**: Service for managing Debezium connectors

## Usage Example

```java
// Get all tools
GET /v1/tools
Response: {"status": "success", "tools": [...]}

// Get specific tool
GET /v1/tools/my-database-tool
Response: {"status": "success", "tool": {...}}

// Save tool configuration
POST /v1/tools/my-database-tool
{
  "tool": {
    "name": "PostgreSQL Tool",
    "config": {...},
    "debeziumConnectors": [...]
  }
}
Response: {"status": "success", "tool": {...}}
```

The controller automatically enables saved tools and creates corresponding Debezium connectors for change data capture, with comprehensive error handling and logging.
