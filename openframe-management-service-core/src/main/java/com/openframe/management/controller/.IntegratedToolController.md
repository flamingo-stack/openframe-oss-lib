REST controller that manages integrated tool configurations in OpenFrame, providing endpoints to retrieve, save, and configure MSP tools with automated post-save processing and Debezium connector management.

## Key Components

- **IntegratedToolController**: Main REST controller handling tool management operations
- **SaveToolRequest**: Inner DTO class for tool save requests
- **IntegratedToolService**: Service layer for tool persistence operations
- **DebeziumService**: Service for managing CDC connectors
- **IntegratedToolPostSaveHook**: Hook interface for post-save processing

## Usage Example

```java
// GET all tools
// Request: GET /v1/tools
// Response: {"status": "success", "tools": [...]}

// GET specific tool
// Request: GET /v1/tools/slack-integration
// Response: {"status": "success", "tool": {...}}

// Save/update tool configuration
// Request: POST /v1/tools/slack-integration
{
  "tool": {
    "name": "Slack Integration",
    "config": {...},
    "debeziumConnectors": [...]
  }
}
// Response: {"status": "success", "tool": {...}}

// Tool is automatically enabled and post-save hooks execute
// Debezium connectors are created/updated for real-time sync
```

The controller implements error handling with appropriate HTTP status codes and executes post-save hooks for additional processing while gracefully handling hook failures to prevent cascading errors.
