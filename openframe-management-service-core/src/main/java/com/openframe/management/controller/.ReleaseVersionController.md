REST controller that handles cluster registration requests to update release versions based on provided image tag versions.

## Key Components

- **ReleaseVersionController**: Main controller class with `/v1/cluster-registrations` endpoint
- **updateReleaseVersion()**: POST endpoint method that processes release version updates
- **ReleaseVersionService**: Injected service dependency for business logic
- **ReleaseVersionRequest**: DTO for incoming request payload

## Usage Example

```java
// POST request to update release version
POST /v1/cluster-registrations
Content-Type: application/json

{
  "imageTagVersion": "v2.1.3"
}
```

```java
// Example request DTO structure
@Data
public class ReleaseVersionRequest {
    private String imageTagVersion;
}

// Service integration example
@Service
public class ReleaseVersionService {
    public void process(String imageTagVersion) {
        // Process the version update logic
        log.info("Processing release version: {}", imageTagVersion);
    }
}
```

The controller extracts the image tag version from the request body and delegates processing to the injected service layer for handling the actual release version update logic.
