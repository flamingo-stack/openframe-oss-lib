A data model class representing configuration settings for script execution within the OpenFrame management system.

## Key Components

- **ScriptConfig**: Main data class with Lombok annotations for automatic getter/setter generation
- **name**: Script identifier/name
- **resourcePath**: File system path to the script resource
- **description**: Human-readable script description
- **shell**: Shell interpreter to use for execution (e.g., bash, sh, powershell)
- **category**: Organizational category for script grouping
- **defaultTimeout**: Maximum execution time in seconds

## Usage Example

```java
// Create a new script configuration
ScriptConfig config = ScriptConfig.builder()
    .name("database-backup")
    .resourcePath("/scripts/db-backup.sh")
    .description("Daily database backup script")
    .shell("bash")
    .category("maintenance")
    .defaultTimeout(300)
    .build();

// Access configuration properties
String scriptPath = config.getResourcePath();
Integer timeout = config.getDefaultTimeout();

// Create with constructor
ScriptConfig config2 = new ScriptConfig(
    "system-check", 
    "/scripts/health-check.sh",
    "System health monitoring",
    "sh",
    "monitoring",
    60
);
```
