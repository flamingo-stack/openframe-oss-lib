A configuration model class that defines the structure and metadata for executable scripts within the OpenFrame management system.

## Key Components

**Main Class:**
- `ScriptConfig` - Lombok-powered data class with builder pattern support

**Fields:**
- `name` - Script identifier/display name
- `resourcePath` - File system path to the script executable
- `description` - Human-readable script description
- `shell` - Shell environment for script execution (e.g., "bash", "powershell")
- `category` - Logical grouping/classification for the script
- `defaultTimeout` - Maximum execution time in seconds before termination

**Annotations:**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Enables fluent builder pattern for object construction
- `@NoArgsConstructor/@AllArgsConstructor` - Provides default and full constructors

## Usage Example

```java
// Create script configuration using builder pattern
ScriptConfig config = ScriptConfig.builder()
    .name("system-backup")
    .resourcePath("/scripts/backup.sh")
    .description("Automated system backup utility")
    .shell("bash")
    .category("maintenance")
    .defaultTimeout(300)
    .build();

// Access configuration properties
String scriptPath = config.getResourcePath();
Integer timeout = config.getDefaultTimeout();

// Create with constructor
ScriptConfig quickConfig = new ScriptConfig(
    "health-check", "/scripts/health.py", 
    "System health monitor", "python3", 
    "monitoring", 60
);
```
