A Spring Boot component that automatically initializes Debezium connectors from MongoDB-stored configurations when the application starts up, ensuring data synchronization is properly established for integrated tools.

## Key Components

- **@EventListener(ApplicationReadyEvent.class)** - Listens for application startup completion
- **@ConditionalOnProperty** - Only activates when `openframe.debezium.health-check.enabled=true`
- **initializeConnectorsIfEmpty()** - Core method that checks for existing connectors and creates new ones from MongoDB
- **DebeziumService** - Handles connector management operations
- **IntegratedToolService** - Provides access to tool configurations with Debezium settings

## Usage Example

This component runs automatically on application startup. To enable it, configure:

```java
// application.yml
openframe:
  debezium:
    health-check:
      enabled: true
```

The component will:
1. Check if Debezium connectors already exist
2. If none exist, fetch all IntegratedTool documents from MongoDB
3. Create connectors for tools that have debeziumConnectors configured
4. Log the initialization process

```java
// Example tool configuration in MongoDB
IntegratedTool tool = new IntegratedTool();
tool.setName("PostgreSQL Database");
tool.setDebeziumConnectors(new Object[]{
    // Debezium connector configuration objects
});
```

The initializer ensures that data change streams are properly established for all configured integrated tools without manual intervention.
