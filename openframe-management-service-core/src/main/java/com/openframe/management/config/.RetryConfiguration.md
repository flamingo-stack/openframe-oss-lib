Enables Spring Retry support for the OpenFrame management application by activating the `@EnableRetry` annotation at the application context level.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `RetryConfiguration` | `@Configuration` class | Spring configuration class that enables retry behavior |
| `@EnableRetry` | Annotation | Activates Spring Retry's AOP-based retry mechanism across the application |

## Usage Example

Once this configuration is active, annotate any service method with `@Retryable` to automatically retry on failure:

```java
@Service
public class TicketSyncService {

    @Retryable(
        value = { ExternalApiException.class },
        maxAttempts = 3,
        backoff = @Backoff(delay = 2000)
    )
    public void syncTickets() {
        // Retried up to 3 times on ExternalApiException
    }

    @Recover
    public void recover(ExternalApiException e) {
        // Fallback logic after all retries are exhausted
    }
}
```

> This configuration requires `spring-retry` and `spring-aspects` on the classpath. No additional bean definitions are needed — `@EnableRetry` handles all proxy setup automatically.
