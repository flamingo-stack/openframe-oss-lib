Configuration properties class for Fleet MDM setup, binding external configuration values prefixed with `openframe.fleet-mdm.setup` to typed Java fields.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `adminName` | `String` | The administrator name for the Fleet MDM instance |
| `orgName` | `String` | The organization name for the Fleet MDM instance |

## Usage Example

```java
// application.yml
// openframe:
//   fleet-mdm:
//     setup:
//       admin-name: admin@example.com
//       org-name: Acme Corp

@Service
@RequiredArgsConstructor
public class FleetMdmSetupService {

    private final FleetMdmSetupProperties fleetMdmSetupProperties;

    public void initialize() {
        String admin = fleetMdmSetupProperties.getAdminName();
        String org = fleetMdmSetupProperties.getOrgName();
        // use values to configure Fleet MDM...
    }
}
```

## Notes

- Annotated with `@ConfigurationProperties(prefix = "openframe.fleet-mdm.setup")` for type-safe binding
- `@Data` (Lombok) auto-generates getters, setters, `equals`, `hashCode`, and `toString`
- Registered as a Spring `@Component`, making it injectable across the application context
