Configuration properties class that manages agent configuration settings for the OpenFrame management system using Spring Boot's configuration binding mechanism.

## Key Components

- **agentConfigurations**: List of strings containing agent configuration entries
- **@ConfigurationProperties**: Binds properties with prefix `openframe.management`
- **@Data**: Lombok annotation providing getters, setters, and other utility methods
- **@Component**: Marks class as a Spring-managed bean

## Usage Example

```java
// application.yml configuration
openframe:
  management:
    agent-configurations:
      - "agent1-config"
      - "agent2-config"
      - "agent3-config"

// Injecting and using the configuration
@Autowired
private AgentConfigurationProperties agentConfig;

public void processAgents() {
    List<String> configs = agentConfig.getAgentConfigurations();
    for (String config : configs) {
        // Process each agent configuration
        System.out.println("Processing: " + config);
    }
}
```

The class automatically populates the `agentConfigurations` list from external configuration sources (application.properties, application.yml, environment variables, etc.) making it easy to manage multiple agent configurations without hardcoding values.
