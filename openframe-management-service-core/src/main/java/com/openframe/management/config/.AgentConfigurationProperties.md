A Spring Boot configuration properties class that manages a list of agent configurations for the OpenFrame management system.

## Key Components

- **`AgentConfigurationProperties`** - Main configuration class annotated with `@ConfigurationProperties`
- **`agentConfigurations`** - List of strings containing agent configuration entries
- **Configuration prefix** - `openframe.management` for property binding

## Usage Example

```java
@Service
public class AgentManagementService {
    
    @Autowired
    private AgentConfigurationProperties agentConfig;
    
    public void processAgents() {
        List<String> configs = agentConfig.getAgentConfigurations();
        for (String config : configs) {
            // Process each agent configuration
            System.out.println("Processing agent: " + config);
        }
    }
}
```

**Application Properties Configuration:**
```yaml
openframe:
  management:
    agent-configurations:
      - "agent-1-config"
      - "agent-2-config"
      - "agent-3-config"
```

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, and other boilerplate methods, while Spring Boot's `@ConfigurationProperties` enables automatic binding of YAML/properties file values to the `agentConfigurations` list.
