A simple configuration class for Apache Pinot table definitions, containing the essential properties needed to set up Pinot tables including name, schema, and table configuration file paths.

## Key Components

- **PinotConfig**: Main configuration POJO with Lombok annotations
  - `name`: String identifier for the Pinot table
  - `schemaFile`: Path to the Pinot schema definition file
  - `tableConfigFile`: Path to the Pinot table configuration file

## Usage Example

```java
// Create a new Pinot configuration
PinotConfig config = new PinotConfig();
config.setName("user_events");
config.setSchemaFile("/config/schemas/user_events_schema.json");
config.setTableConfigFile("/config/tables/user_events_table.json");

// Or use the all-args constructor
PinotConfig config = new PinotConfig(
    "user_events",
    "/config/schemas/user_events_schema.json", 
    "/config/tables/user_events_table.json"
);

// Access properties
String tableName = config.getName();
String schemaPath = config.getSchemaFile();
```

This class is typically used in Spring Boot applications to externalize Pinot table configuration through properties files or configuration management systems.
