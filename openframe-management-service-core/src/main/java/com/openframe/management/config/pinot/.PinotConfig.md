This is a configuration data class for Apache Pinot table setup, containing the essential parameters needed to define and configure a Pinot table within the OpenFrame management system.

## Key Components

- **`name`** - The identifier/name for the Pinot table configuration
- **`schemaFile`** - Path or reference to the Pinot schema definition file
- **`tableConfigFile`** - Path or reference to the Pinot table configuration file
- **Lombok annotations** - `@Data`, `@AllArgsConstructor`, `@NoArgsConstructor` for automatic generation of getters, setters, constructors, and utility methods

## Usage Example

```java
// Create a new Pinot configuration
PinotConfig config = new PinotConfig();
config.setName("user_events");
config.setSchemaFile("schemas/user_events_schema.json");
config.setTableConfigFile("configs/user_events_table.json");

// Or using the all-args constructor
PinotConfig config = new PinotConfig(
    "user_events",
    "schemas/user_events_schema.json", 
    "configs/user_events_table.json"
);

// Access configuration details
String tableName = config.getName();
String schemaPath = config.getSchemaFile();
```

This class serves as a simple POJO for bundling Pinot table configuration parameters, likely used by configuration management services to initialize and deploy Pinot tables programmatically.
