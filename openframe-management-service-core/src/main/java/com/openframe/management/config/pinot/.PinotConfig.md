Configuration class for Apache Pinot table setup, defining the essential properties needed to create and configure Pinot tables including schema and table configuration files.

## Key Components

- **name**: String identifier for the Pinot table name
- **schemaFile**: Path to the Pinot schema definition file
- **tableRealtimeConfigFile**: Path to the real-time table configuration file
- **tableOfflineConfigFile**: Path to the offline table configuration file

## Usage Example

```java
// Create a new Pinot configuration
PinotConfig config = new PinotConfig();
config.setName("user_events");
config.setSchemaFile("/config/schemas/user_events_schema.json");
config.setTableRealtimeConfigFile("/config/tables/user_events_realtime.json");
config.setTableOfflineConfigFile("/config/tables/user_events_offline.json");

// Or using the all-args constructor
PinotConfig config = new PinotConfig(
    "user_events",
    "/config/schemas/user_events_schema.json",
    "/config/tables/user_events_realtime.json",
    "/config/tables/user_events_offline.json"
);

// Access configuration properties
String tableName = config.getName();
String schemaPath = config.getSchemaFile();
```

This configuration class is typically used in Spring Boot applications to define Pinot table configurations that can be loaded from application properties or configuration files for automated table creation and management.
