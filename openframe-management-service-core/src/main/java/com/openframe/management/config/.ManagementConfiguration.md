Spring Boot configuration class for the OpenFrame management module that sets up component scanning and security infrastructure.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `ManagementConfiguration` | `@Configuration` class | Root Spring configuration for the management module |
| `passwordEncoder()` | `@Bean` method | Registers a `BCryptPasswordEncoder` for secure password hashing |
| `@ComponentScan` | Annotation | Scans all `com.openframe` packages, excluding `CassandraHealthIndicator` |

## Details

- **Component Scan Scope:** Scans the entire `com.openframe` base package to auto-discover Spring-managed beans.
- **Cassandra Health Exclusion:** `CassandraHealthIndicator` is explicitly excluded from scanning via `FilterType.ASSIGNABLE_TYPE`, preventing it from being registered in the management context (likely to avoid health check conflicts or dependency issues at this layer).
- **Password Encoding:** Exposes a `BCryptPasswordEncoder` bean, making it available for injection anywhere authentication or credential management is needed.

## Usage Example

```java
// Inject the PasswordEncoder bean anywhere in the application context
@Service
public class UserService {

    private final PasswordEncoder passwordEncoder;

    public UserService(PasswordEncoder passwordEncoder) {
        this.passwordEncoder = passwordEncoder;
    }

    public String hashPassword(String rawPassword) {
        return passwordEncoder.encode(rawPassword);
    }

    public boolean verifyPassword(String rawPassword, String hashedPassword) {
        return passwordEncoder.matches(rawPassword, hashedPassword);
    }
}
```

> **Note:** The exclusion of `CassandraHealthIndicator` ensures the management context does not inadvertently register Cassandra-specific health probes, keeping infrastructure concerns separated from management-layer responsibilities.
