Spring Boot configuration class for OpenFrame management module that sets up component scanning and security components.

## Key Components

- **@Configuration**: Marks this as a Spring configuration class
- **@ComponentScan**: Scans `com.openframe` package for Spring components, excluding `CassandraHealthIndicator`
- **passwordEncoder()**: Bean method that provides BCrypt password encoding functionality

## Usage Example

```java
// The configuration is automatically loaded by Spring Boot
// Access the password encoder in other components:

@Service
public class UserService {
    
    @Autowired
    private PasswordEncoder passwordEncoder;
    
    public String hashPassword(String rawPassword) {
        return passwordEncoder.encode(rawPassword);
    }
    
    public boolean validatePassword(String rawPassword, String hashedPassword) {
        return passwordEncoder.matches(rawPassword, hashedPassword);
    }
}
```

This configuration enables automatic discovery of Spring components across the OpenFrame application while providing secure password hashing capabilities through BCrypt. The exclusion filter prevents the Cassandra health indicator from being automatically registered, likely to avoid conflicts with custom health check configurations.
