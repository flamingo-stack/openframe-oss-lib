Service responsible for managing agent registration secrets used for secure agent authentication and registration processes.

## Key Components

- **`createInitialSecret()`** - Creates and persists the initial agent registration secret if none exist
- **AgentRegistrationSecretRepository** - Repository for persisting registration secrets
- **AgentRegistrationSecretGenerator** - Generates secure registration keys
- **EncryptionService** - Handles encryption of sensitive secret data

## Usage Example

```java
@Autowired
private AgentRegistrationSecretManagementService secretService;

// Initialize agent registration secrets on application startup
@PostConstruct
public void initializeSecrets() {
    secretService.createInitialSecret();
}
```

The service ensures only one initial secret is created by checking for existing secrets before generation. Generated secrets are encrypted before storage and marked as active with a creation timestamp. This provides a secure foundation for agent registration workflows while preventing duplicate secret creation.
