Manages the lifecycle of agent registration secrets, specifically handling the initial creation of encrypted registration keys used to authenticate agents joining the OpenFrame platform.

## Key Components

| Element | Description |
|---|---|
| `createInitialSecret()` | Generates and persists a new encrypted registration secret if none exists |
| `AgentRegistrationSecretRepository` | Persistence layer for storing registration secret documents |
| `AgentRegistrationSecretGenerator` | Generates raw secret key strings |
| `EncryptionService` | Encrypts the raw secret before storage |
| `AgentRegistrationSecretManagementProcessor` | Executes post-processing hooks after initial secret creation |

## Usage Example

```java
// Typically called during application startup or bootstrapping
@Component
public class ApplicationStartupRunner implements ApplicationRunner {

    private final AgentRegistrationSecretManagementService secretManagementService;

    @Override
    public void run(ApplicationArguments args) {
        // Idempotent - skips generation if a secret already exists
        secretManagementService.createInitialSecret();
    }
}
```

## Behavior Notes

- **Idempotent by design** — `createInitialSecret()` checks for existing secrets via `existsAny()` and exits early if one is found, making it safe to call repeatedly on startup.
- **Encryption at rest** — the raw secret key is encrypted via `EncryptionService` before being persisted; the plaintext key is only passed downstream to `secretProcessor` for post-processing (e.g., displaying or distributing the key on first boot).
- **Audit-friendly** — each secret is stamped with `createdAt` and an `active` flag, supporting future rotation workflows.
