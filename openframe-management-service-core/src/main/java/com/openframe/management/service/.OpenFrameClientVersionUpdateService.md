Handles the processing and propagation of OpenFrame client version update events by delegating to the NATS message publisher when a new release version is detected.

## Key Components

| Element | Type | Description |
|---|---|---|
| `OpenFrameClientVersionUpdateService` | `@Component` | Spring-managed service for client version update orchestration |
| `openFrameClientUpdatePublisher` | Dependency | NATS publisher used to broadcast version update events |
| `process(String newReleaseVersion)` | Method | Entry point that handles a new release version string and triggers downstream notification |

## Usage Example

```java
@Service
@RequiredArgsConstructor
public class ReleaseWatcherService {

    private final OpenFrameClientVersionUpdateService versionUpdateService;

    public void onNewRelease(String version) {
        // Triggers client update propagation via NATS
        versionUpdateService.process(version);
    }
}
```

> **Note:** The `process` method body is currently a stub. The intended behavior is to invoke `openFrameClientUpdatePublisher` to notify connected OpenFrame clients of the new available version.
