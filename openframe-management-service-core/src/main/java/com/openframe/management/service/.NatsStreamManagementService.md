Manages NATS JetStream stream lifecycle operations, providing automatic creation and updating of stream configurations with existence checking.

## Key Components

- **`save(StreamConfiguration)`** - Creates new streams or updates existing ones based on stream name
- **`existsByName(String)`** - Checks if a stream exists by attempting to retrieve stream info
- **NATS Connection** - Injected connection for JetStream management operations
- **Exception Handling** - Wraps NATS exceptions in custom `NatsException` with context

## Usage Example

```java
@Autowired
private NatsStreamManagementService streamService;

// Create a new stream configuration
StreamConfiguration config = StreamConfiguration.builder()
    .name("user-events")
    .subjects("user.*")
    .build();

// Save will automatically create or update the stream
streamService.save(config);

// The service handles:
// - Checking if stream "user-events" exists
// - Creating new stream if it doesn't exist
// - Updating existing stream if it already exists
// - Logging operations and wrapping exceptions
```

The service automatically determines whether to create or update streams, making it safe for repeated deployments and configuration changes. All NATS-specific exceptions are wrapped in custom `NatsException` with descriptive error messages.
