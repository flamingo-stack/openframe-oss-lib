A Spring service component that manages NATS JetStream streams, providing functionality to create, update, and check existence of streams with proper error handling.

## Key Components

- **save(StreamConfiguration)** - Creates new streams or updates existing ones based on stream name availability
- **existsByName(String)** - Checks if a stream exists by attempting to retrieve stream information
- **natsConnection** - Injected NATS connection for accessing JetStream management operations

## Usage Example

```java
@Autowired
private NatsStreamManagementService streamService;

// Create a new stream configuration
StreamConfiguration config = StreamConfiguration.builder()
    .name("user-events")
    .subjects("user.*")
    .build();

// Save the stream (will create new or update existing)
streamService.save(config);

// The service automatically handles:
// - Checking if stream exists
// - Creating new stream if it doesn't exist
// - Updating existing stream if it exists
// - Proper error handling with custom NatsException
```

The service uses the NATS JetStream API to manage stream lifecycle operations, automatically determining whether to create or update streams based on their existence. All operations include comprehensive logging and error handling through custom `NatsException` wrapping.
