A Spring service that manages Debezium connector lifecycle by creating new connectors or updating existing ones via REST API calls.

## Key Components

- **`createOrUpdateDebeziumConnector(Object[] debeziumConnectors)`** - Main method that processes an array of connector configurations, checking if each exists and either updating or creating accordingly
- **`getDebeziumConnectorCreateUrl()`** - Returns the base URL for connector operations (`/connectors` endpoint)
- **`getDebeziumConnectorUrl(String name)`** - Constructs URL for specific connector operations
- **Configuration**: Uses `${openframe.debezium.base-url}` property for Debezium Connect API base URL

## Usage Example

```java
@Autowired
private DebeziumService debeziumService;

// Define connector configurations
Object[] connectors = {
    Map.of(
        "name", "my-postgres-connector",
        "config", Map.of(
            "connector.class", "io.debezium.connector.postgresql.PostgresConnector",
            "database.hostname", "localhost",
            "database.port", "5432",
            "database.user", "debezium",
            "database.password", "dbpass",
            "database.dbname", "mydb"
        )
    )
};

// Create or update connectors
debeziumService.createOrUpdateDebeziumConnector(connectors);
```

The service automatically handles connector existence checks, updates existing connectors' configurations, and creates new ones as needed while providing comprehensive logging throughout the process.
