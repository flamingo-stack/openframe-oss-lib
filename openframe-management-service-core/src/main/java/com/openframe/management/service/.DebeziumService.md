A Spring service for managing Debezium CDC (Change Data Capture) connectors through REST API interactions, providing functionality to create, update, monitor, and restart connectors and their tasks.

## Key Components

- **createOrUpdateDebeziumConnector()** - Creates new connectors or updates existing ones based on configuration
- **listConnectors()** - Retrieves all registered connector names
- **getConnectorStatus()** - Gets detailed status information for a specific connector
- **restartTask()** - Restarts a failed task for a given connector
- **checkAndRestartFailedTasks()** - Monitors all connectors and automatically restarts any failed tasks
- **checkConnectorAndRestartFailedTasks()** - Checks and restarts failed tasks for a specific connector

## Usage Example

```java
@Autowired
private DebeziumService debeziumService;

// Create or update connectors from configuration
Object[] connectorConfigs = loadConnectorConfigurations();
debeziumService.createOrUpdateDebeziumConnector(connectorConfigs);

// List all connectors
List<String> connectors = debeziumService.listConnectors();

// Check status and restart failed tasks
ConnectorStatus status = debeziumService.getConnectorStatus("my-connector");
if ("FAILED".equals(status.getConnector().getState())) {
    debeziumService.checkConnectorAndRestartFailedTasks("my-connector");
}

// Perform health check on all connectors
debeziumService.checkAndRestartFailedTasks();
```

The service requires the `openframe.debezium.base-url` configuration property and uses RestTemplate for HTTP communication with the Debezium Connect API.
