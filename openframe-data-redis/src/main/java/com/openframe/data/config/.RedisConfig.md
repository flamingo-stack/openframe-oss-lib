Spring Boot Redis configuration class that provides both synchronous and reactive Redis template beans for the OpenFrame data layer. Enables Redis repositories and configures serialization for string-based key-value operations.

## Key Components

- **@Configuration** - Spring configuration class with conditional enablement
- **@ConditionalOnProperty** - Only activates when `spring.redis.enabled=true`
- **@EnableRedisRepositories** - Enables Redis repositories in specified package
- **redisTemplate()** - Creates synchronous RedisTemplate with string serialization
- **reactiveStringRedisTemplate()** - Creates reactive string-based Redis template
- **reactiveRedisTemplate()** - Creates fully reactive Redis template with custom serialization

## Usage Example

```java
// Enable Redis in application.yml
spring:
  redis:
    enabled: true
    host: localhost
    port: 6379

// Inject and use the templates
@Service
public class CacheService {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    @Autowired
    private ReactiveRedisTemplate<String, String> reactiveRedisTemplate;
    
    public void cacheValue(String key, String value) {
        redisTemplate.opsForValue().set(key, value);
    }
    
    public Mono<String> getCacheValueReactive(String key) {
        return reactiveRedisTemplate.opsForValue().get(key);
    }
}
```

The configuration uses `@ConditionalOnMissingBean` to prevent conflicts with custom Redis configurations and ensures all serializers use `StringRedisSerializer` for consistent string-based storage.
