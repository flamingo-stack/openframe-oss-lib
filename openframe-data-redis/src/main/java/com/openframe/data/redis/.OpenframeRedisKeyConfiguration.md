Configures Redis key generation for the OpenFrame platform, providing a standardized way to build Redis keys with proper prefixing and structure based on application properties.

## Key Components

- **OpenframeRedisKeyConfiguration** - Spring configuration class that sets up Redis key management
- **openframeRedisKeyBuilder()** - Creates and configures the Redis key builder bean
- **@ConditionalOnMissingBean** - Ensures only one key builder instance exists in the application context
- **@EnableConfigurationProperties** - Enables binding of `OpenframeRedisProperties` configuration

## Usage Example

This configuration automatically provides a Redis key builder throughout the application:

```java
@Service
public class UserCacheService {
    
    private final OpenframeRedisKeyBuilder keyBuilder;
    
    public UserCacheService(OpenframeRedisKeyBuilder keyBuilder) {
        this.keyBuilder = keyBuilder;
    }
    
    public void cacheUser(String userId, User user) {
        String key = keyBuilder.buildKey("users", userId);
        // key might be: "openframe:prod:users:12345"
        redisTemplate.set(key, user);
    }
}
```

The configuration leverages Spring Boot's auto-configuration capabilities to provide consistent Redis key naming across all OpenFrame services, with keys structured according to environment, service context, and data type specifications defined in the properties.
