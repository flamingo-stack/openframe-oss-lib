Repository for synchronizing API key statistics from Redis using blocking operations. Provides methods to retrieve, parse, and manage API key statistics stored as Redis hash entries.

## Key Components

- **getAllStatsKeys()** - Retrieves all Redis keys matching the API key stats pattern
- **getStatsData(String)** - Fetches complete hash data for a specific stats key
- **deleteStatsKey(String)** - Removes a stats key from Redis storage
- **extractKeyId(String)** - Extracts the API key ID from a Redis key
- **getLong(Map, String)** - Safely parses Long values from Redis hash data with error handling
- **getString(Map, String)** - Safely parses String values from Redis hash data

## Usage Example

```java
@Service
public class ApiKeyStatsService {
    
    private final ApiKeyStatsSyncRepository statsRepo;
    
    public void processAllStats() {
        // Get all stats keys
        Set<String> keys = statsRepo.getAllStatsKeys();
        
        for (String redisKey : keys) {
            // Extract key ID and get stats data
            String keyId = statsRepo.extractKeyId(redisKey);
            Map<Object, Object> data = statsRepo.getStatsData(redisKey);
            
            // Parse specific metrics
            Long requestCount = statsRepo.getLong(data, "requests");
            String lastUsed = statsRepo.getString(data, "lastUsed");
            
            // Process stats...
            processKeyStats(keyId, requestCount, lastUsed);
            
            // Clean up if needed
            if (shouldCleanup(data)) {
                statsRepo.deleteStatsKey(redisKey);
            }
        }
    }
}
```
