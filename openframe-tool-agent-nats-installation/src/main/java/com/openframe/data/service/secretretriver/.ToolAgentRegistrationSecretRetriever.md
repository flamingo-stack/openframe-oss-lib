Interface for retrieving authentication credentials used during tool agent registration. Implementations provide a tool identifier and its associated secret for secure agent onboarding.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `getToolId()` | `String` | Returns the unique identifier for the tool agent |
| `getSecret()` | `String` | Returns the secret credential associated with the tool |

## Usage Example

```java
public class ApiKeyToolAgentSecretRetriever 
    implements ToolAgentRegistrationSecretRetriever {

    @Override
    public String getToolId() {
        return "my-tool-agent-id";
    }

    @Override
    public String getSecret() {
        return System.getenv("TOOL_AGENT_SECRET");
    }
}
```

> Implementations should avoid hardcoding secrets. Prefer reading from environment variables or a secrets manager (e.g., Vault, AWS Secrets Manager) to keep credentials out of source code.
