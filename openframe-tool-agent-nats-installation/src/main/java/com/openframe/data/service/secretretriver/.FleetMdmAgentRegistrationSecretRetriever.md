Retrieves the agent enrollment secret from a Fleet MDM integration by connecting to the configured Fleet MDM server using stored credentials.

## Key Components

| Element | Description |
|---|---|
| `TOOL_ID` | Constant identifier `"fleetmdm-server"` used to look up the Fleet MDM integration |
| `getToolId()` | Returns the tool identifier for registration routing |
| `getSecret()` | Fetches the Fleet MDM enroll secret via `FleetMdmClient` using the configured API URL and token |
| `@ConditionalOnProperty` | Only activates when `openframe.integration.tool.enabled=true` |

## Usage Example

```java
// Injected automatically via ToolAgentRegistrationSecretRetriever polymorphism
@Autowired
List<ToolAgentRegistrationSecretRetriever> retrievers;

// Locate and invoke the Fleet MDM retriever
retrievers.stream()
    .filter(r -> r.getToolId().equals("fleetmdm-server"))
    .findFirst()
    .ifPresent(r -> {
        String enrollSecret = r.getSecret();
        // use enrollSecret for agent provisioning
    });
```

## Runtime Behavior

1. Looks up the `fleetmdm-server` tool via `IntegratedToolService`
2. Resolves the API URL and port from `ToolUrlService` using `ToolUrlType.API`
3. Instantiates a `FleetMdmClient` with the API URL and stored API key
4. Calls `client.getEnrollSecret()` and returns the result
5. Throws `IllegalStateException` if the tool, URL, or secret retrieval fails — errors are logged via SLF4J before re-throwing
