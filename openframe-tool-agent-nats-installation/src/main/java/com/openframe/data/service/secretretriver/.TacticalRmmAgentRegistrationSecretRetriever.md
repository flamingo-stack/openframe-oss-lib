Retrieves the TacticalRMM agent registration secret by connecting to a configured TacticalRMM instance and fetching an installation token using the stored API credentials and URL configuration.

## Key Components

| Element | Description |
|---|---|
| `TOOL_ID` | Static identifier `"tactical-rmm"` used to look up the tool configuration |
| `getToolId()` | Returns the tool identifier for registry/dispatch purposes |
| `getSecret()` | Resolves the API URL and key from stored config, then calls the TacticalRMM client to retrieve the enrollment secret |
| `buildRequest()` | Constructs a default `AgentRegistrationSecretRequest` targeting Windows/amd64 with a 2400-minute expiry |

The bean is conditionally activated via `openframe.integration.tool.enabled=true`.

## Usage Example

```java
// Injected via ToolAgentRegistrationSecretRetriever interface
@Autowired
private TacticalRmmAgentRegistrationSecretRetriever retriever;

// Verify this retriever handles tactical-rmm
if (retriever.getToolId().equals("tactical-rmm")) {
    String secret = retriever.getSecret();
    // Use secret to generate agent install command
}
```

> **Note:** The `getSecret()` method throws `IllegalStateException` if no TacticalRMM tool or API URL is found in the integrated tool configuration. Ensure the tool is registered and an API-type URL is configured before invoking.
