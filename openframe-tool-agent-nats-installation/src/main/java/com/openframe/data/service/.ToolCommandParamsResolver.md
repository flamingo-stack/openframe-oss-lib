Resolves command-line parameter placeholders for tool agent registrations by injecting secrets retrieved from configured `ToolAgentRegistrationSecretRetriever` implementations.

## Key Components

| Element | Description |
|---|---|
| `REGISTRATION_SECRET_PLACEHOLDER` | Constant (`${server.registrationSecret}`) representing the placeholder string to be replaced in command arguments |
| `toolAgentRegistrationSecretRetrievers` | Injected list of `ToolAgentRegistrationSecretRetriever` implementations used to resolve secrets per tool ID |
| `process(toolId, commandArgs)` | Entry point; iterates over command arguments and delegates placeholder resolution |
| `processArgument(toolId, argument)` | Checks if a single argument contains the placeholder and replaces it with the resolved secret |
| `getRegistrationSecret(toolId)` | Finds the first matching retriever for the given tool ID and returns its secret, throwing `IllegalStateException` if none found |
| `isSuitable(toolId, retriever)` | Matches a retriever to a tool ID using strict string equality |

> **Note:** This component is only active when `openframe.integration.tool.enabled=true` is set in your application properties.

## Usage Example

```java
// Inject the resolver into your service
@Autowired
private ToolCommandParamsResolver resolver;

// Define command args with a registration secret placeholder
List<String> rawArgs = List.of(
    "--host=localhost",
    "--secret=${server.registrationSecret}",
    "--port=8080"
);

// Resolve placeholders for a specific tool
List<String> resolvedArgs = resolver.process("my-tool-id", rawArgs);
// Result: ["--host=localhost", "--secret=abc123secretValue", "--port=8080"]
```
