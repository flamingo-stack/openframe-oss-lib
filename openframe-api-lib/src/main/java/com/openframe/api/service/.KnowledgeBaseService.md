Provides CRUD and lifecycle management operations for knowledge base articles and folders, including querying, filtering, tag resolution, cursor-based pagination, and hierarchical folder operations.

## Key Components

### Public Methods

| Method | Description |
|--------|-------------|
| `queryItems` | Queries KB items with filtering, search, and cursor pagination; auto-detects recursive vs. flat traversal |
| `getItem` | Fetches a single KB item by ID |
| `getAllFolders` / `getAllArticles` | Returns all folders or articles sorted by name |
| `getTagsInSubtree` | Collects tags from all articles under a given folder subtree |
| `queryArchivedArticles` | Paged query for archived articles with search and tag filters |
| `createFolder` / `renameFolder` / `deleteFolder` | Folder lifecycle; delete supports `MOVE` or `ARCHIVE` children actions |
| `createArticle` | Creates an article with tags and multi-target assignments (org, device, ticket, article) |
| `updateArticle` | Partial update of article fields |
| `publishArticle` / `unpublishArticle` | Toggles article status between `PUBLISHED` and `DRAFT` |
| `archiveArticle` / `unarchiveArticle` | Soft-archives an article or restores it to a target folder |
| `moveToFolder` | Moves any KB item to a new parent, with circular-move guard for folders |

### Dependencies

- `KnowledgeBaseItemRepository` — data access layer
- `KnowledgeBaseTagService` — tag resolution and lookup
- `AssignmentService` — links articles to organizations, devices, tickets, and other articles

## Usage Example

```java
// Create a folder, then publish an article inside it
KnowledgeBaseItem folder = knowledgeBaseService.createFolder("Networking Guides", null);

CreateArticleCommand cmd = CreateArticleCommand.builder()
    .name("VPN Setup Guide")
    .parentId(folder.getId())
    .content("Step-by-step VPN configuration...")
    .status(KnowledgeBaseArticleStatus.DRAFT)
    .tagIds(List.of("tag-vpn", "tag-networking"))
    .assignedOrganizationIds(List.of("org-123"))
    .build();

KnowledgeBaseItem article = knowledgeBaseService.createArticle(currentUserId, cmd);
knowledgeBaseService.publishArticle(article.getId());

// Query articles with tag filter and pagination
KnowledgeBaseFilterCriteria filter = KnowledgeBaseFilterCriteria.builder()
    .parentId(folder.getId())
    .tagIds(List.of("tag-vpn"))
    .build();

CountedGenericQueryResult<KnowledgeBaseItem> result =
    knowledgeBaseService.queryItems(filter, "VPN", CursorPaginationCriteria.ofLimit(20));
```
