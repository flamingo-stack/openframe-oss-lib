Manages tag assignments for Knowledge Base articles within the OpenFrame platform, providing CRUD operations and lookup utilities scoped to the `KNOWLEDGE_ARTICLE` entity type.

## Key Components

| Method | Description |
|---|---|
| `getAllTags(boolean archived)` | Returns all tags associated with KB articles filtered by archived/active status |
| `getTagsForItemIds(List<String>)` | Fetches distinct tags assigned to a given list of article IDs |
| `findItemIdsByTags(List<String>)` | Resolves article IDs that match any of the provided tag IDs |
| `addTagToItem(String, String)` | Creates and persists a `TagAssignment` linking a tag to a KB article |
| `removeTagFromItem(String, String)` | Deletes the `TagAssignment` for a specific article/tag pair |
| `getTagsByItemIds(List<String>)` | Returns a positionally-ordered list of tag lists, one per input article ID |

## Usage Example

```java
// Retrieve all active (non-archived) tags used across KB articles
List<Tag> activeTags = knowledgeBaseTagService.getAllTags(false);

// Get tags for a specific set of articles
List<Tag> tags = knowledgeBaseTagService.getTagsForItemIds(List.of("article-1", "article-2"));

// Find all article IDs tagged with specific tags
List<String> articleIds = knowledgeBaseTagService.findItemIdsByTags(List.of("tag-id-abc"));

// Assign and remove a tag
knowledgeBaseTagService.addTagToItem("article-1", "tag-id-abc");
knowledgeBaseTagService.removeTagFromItem("article-1", "tag-id-abc");

// Batch-fetch tags per article, preserving input order
List<List<Tag>> tagsByArticle = knowledgeBaseTagService.getTagsByItemIds(
    List.of("article-1", "article-2", "article-3")
);
```

> `getTagsByItemIds` preserves the order of the input list, returning an empty list for any article ID with no assigned tags — safe for positional mapping in bulk responses.
