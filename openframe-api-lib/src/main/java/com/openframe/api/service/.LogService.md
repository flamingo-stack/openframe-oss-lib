A Spring service that handles querying, filtering, and retrieving unified log events by coordinating between Apache Pinot (for fast analytics queries) and Apache Cassandra (for detailed event storage).

## Key Components

| Method | Description |
|--------|-------------|
| `queryLogs()` | Fetches paginated log events from Pinot with support for full-text search, field filtering, cursor-based pagination, and sort validation |
| `findLogDetails()` | Retrieves full event details from Cassandra by composite key (ingestDay, toolType, eventType, timestamp, toolEventId) |
| `getLogFilters()` | Returns available filter options (tool types, event types, severities, organizations) for the current tenant |
| `buildLogQueryResult()` | Assembles paginated `GenericQueryResult<LogEvent>` with encoded start/end cursors |
| `validateSortField()` | Guards against invalid sort fields by falling back to the repository default |
| `mapToLogEvent()` | Maps a `LogProjection` (Pinot) to a `LogEvent` DTO |
| `mapToLogDetails()` | Maps a `UnifiedLogEvent` (Cassandra) to a `LogDetails` DTO |

> The service is conditionally activated via `@ConditionalOnProperty(name = "spring.data.cassandra.enabled", havingValue = "true")` and scoped to the current tenant via `TenantIdProvider`.

## Usage Example

```java
// Query paginated logs with search and sorting
LogFilterCriteria filter = LogFilterCriteria.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .severities(List.of("ERROR", "WARN"))
    .build();

CursorPaginationCriteria pagination = CursorPaginationCriteria.builder()
    .limit(50)
    .cursor(null) // first page
    .build();

SortInput sort = SortInput.builder()
    .field("eventTimestamp")
    .direction(SortDirection.DESC)
    .build();

GenericQueryResult<LogEvent> result = logService.queryLogs(filter, pagination, "login failed", sort);

// Fetch full details for a specific event
Optional<LogDetails> details = logService.findLogDetails(
    "2024-01-15", "rmm", "agent_disconnect",
    Instant.parse("2024-01-15T10:30:00Z"), "evt_abc123"
);

// Retrieve available filter options
LogFilters filters = logService.getLogFilters(filter);
```
