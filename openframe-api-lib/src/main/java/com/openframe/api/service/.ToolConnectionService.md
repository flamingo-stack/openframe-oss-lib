A Spring service that manages tool connections associated with machines, providing efficient batch and individual retrieval operations.

## Key Components

- **`getToolConnectionsForMachines(List<String> machineIds)`** - Retrieves tool connections for multiple machines in a single operation, returning results grouped by machine ID in the same order as input
- **`getToolConnectionsForMachine(String machineId)`** - Retrieves tool connections for a single machine
- **`ToolConnectionRepository`** - Injected repository for data access operations
- **`@Transactional(readOnly = true)`** - Optimizes database transactions for read-only operations

## Usage Example

```java
@Autowired
private ToolConnectionService toolConnectionService;

// Get connections for multiple machines (useful for batch processing)
List<String> machineIds = Arrays.asList("machine1", "machine2", "machine3");
List<List<ToolConnection>> batchResults = toolConnectionService
    .getToolConnectionsForMachines(machineIds);

// Results maintain order: batchResults.get(0) contains connections for "machine1"
List<ToolConnection> machine1Connections = batchResults.get(0);

// Get connections for a single machine
List<ToolConnection> singleMachineConnections = toolConnectionService
    .getToolConnectionsForMachine("machine1");
```

The batch method is particularly useful for DataLoader scenarios where you need to fetch tool connections for multiple machines efficiently while preserving the input order.
