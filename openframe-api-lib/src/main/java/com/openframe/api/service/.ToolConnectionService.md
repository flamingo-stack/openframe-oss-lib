Service class that manages tool connections for machines, providing methods to retrieve connection data either for individual machines or batches of machines efficiently.

## Key Components

- **ToolConnectionService** - Main service class with read-only transaction support
- **getToolConnectionsForMachines()** - Batch retrieval method optimized for DataLoader usage
- **getToolConnectionsForMachine()** - Single machine connection retrieval method
- **ToolConnectionRepository** - Injected repository for database operations

## Usage Example

```java
@Autowired
private ToolConnectionService toolConnectionService;

// Get connections for a single machine
List<ToolConnection> singleMachineConnections = 
    toolConnectionService.getToolConnectionsForMachine("machine-123");

// Get connections for multiple machines (optimized for batch operations)
List<String> machineIds = Arrays.asList("machine-123", "machine-456", "machine-789");
List<List<ToolConnection>> batchConnections = 
    toolConnectionService.getToolConnectionsForMachines(machineIds);

// Access connections for specific machine from batch result
List<ToolConnection> firstMachineConnections = batchConnections.get(0);
```

The service uses efficient grouping to minimize database queries when fetching connections for multiple machines, making it ideal for GraphQL DataLoader scenarios where batch loading is required.
