A facade service that handles organization command operations (Create, Update, Delete) by managing DTO-to-entity mapping and delegating business logic to the underlying OrganizationService.

## Key Components

- **createOrganization()** - Creates new organizations from CreateOrganizationRequest DTOs
- **updateOrganization()** - Performs partial updates using UpdateOrganizationRequest DTOs
- **deleteOrganization()** - Delegates organization deletion with validation
- **OrganizationMapper** - Handles DTO-to-entity conversions
- **OrganizationService** - Core business logic service dependency

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class OrganizationController {
    
    private final OrganizationCommandService commandService;
    
    @PostMapping("/organizations")
    public Organization create(@RequestBody CreateOrganizationRequest request) {
        return commandService.createOrganization(request);
    }
    
    @PutMapping("/organizations/{id}")
    public Organization update(@PathVariable String id, 
                             @RequestBody UpdateOrganizationRequest request) {
        return commandService.updateOrganization(id, request);
    }
    
    @DeleteMapping("/organizations/{id}")
    public void delete(@PathVariable String id) {
        commandService.deleteOrganization(id);
    }
}
```

This service acts as an API layer facade, preventing circular dependencies by handling DTOs while delegating core business logic to the data layer service.
