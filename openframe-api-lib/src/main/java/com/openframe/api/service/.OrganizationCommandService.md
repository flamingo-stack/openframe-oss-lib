This service provides a facade for organization command operations (Create, Update, Delete) that handles DTO-to-entity mapping and delegates persistence operations to the underlying OrganizationService.

## Key Components

- **createOrganization()** - Creates new organizations from CreateOrganizationRequest DTOs
- **updateOrganization()** - Performs partial updates using UpdateOrganizationRequest DTOs
- **deleteOrganization()** - Delegates organization deletion with association validation
- **OrganizationMapper** - Handles bidirectional mapping between DTOs and entities
- **OrganizationService** - Underlying service for persistence operations

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class OrganizationController {
    
    private final OrganizationCommandService commandService;
    
    @PostMapping("/organizations")
    public Organization createOrganization(@RequestBody CreateOrganizationRequest request) {
        return commandService.createOrganization(request);
    }
    
    @PutMapping("/organizations/{id}")
    public Organization updateOrganization(@PathVariable String id, 
                                         @RequestBody UpdateOrganizationRequest request) {
        return commandService.updateOrganization(id, request);
    }
    
    @DeleteMapping("/organizations/{id}")
    public void deleteOrganization(@PathVariable String id) {
        commandService.deleteOrganization(id);
    }
}
```

The service acts as an anti-corruption layer between the API and data layers, preventing circular dependencies by handling DTO transformations in the api-lib module while delegating business logic to the data layer.
