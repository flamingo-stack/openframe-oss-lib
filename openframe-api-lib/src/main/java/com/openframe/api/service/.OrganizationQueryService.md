Service for querying organizations with flexible filtering, search capabilities, and cursor-based pagination. Provides both paginated and non-paginated query methods with MongoDB-level filtering for optimal performance.

## Key Components

- **queryOrganizations(filterOptions, search)** - Returns all organizations without pagination for backward compatibility
- **queryOrganizations(filterOptions, paginationCriteria, search)** - Returns paginated results with cursor-based navigation
- **fetchPageItems()** - Handles cursor pagination logic with limit enforcement
- **buildPageInfo()** - Constructs pagination metadata including cursors and page flags
- **buildQueryFilter()** - Converts filter options to MongoDB query filter objects

## Usage Example

```java
@Autowired
private OrganizationQueryService organizationQueryService;

// Non-paginated query for all organizations
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("TECHNOLOGY")
    .minEmployees(50)
    .hasActiveContract(true)
    .build();

OrganizationList allOrgs = organizationQueryService.queryOrganizations(filters, "openframe");

// Paginated query with cursor navigation
CursorPaginationCriteria pagination = CursorPaginationCriteria.builder()
    .limit(20)
    .cursor("org123")
    .build();

CountedGenericQueryResult<Organization> pagedResults = organizationQueryService
    .queryOrganizations(filters, pagination, "openframe");

// Access results and pagination info
List<Organization> organizations = pagedResults.getItems();
CursorPageInfo pageInfo = pagedResults.getPageInfo();
boolean hasMore = pageInfo.isHasNextPage();
```
