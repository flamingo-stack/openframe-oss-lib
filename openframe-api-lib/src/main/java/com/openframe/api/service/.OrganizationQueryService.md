Service class that provides organization querying capabilities with filtering, searching, and cursor-based pagination. Implements MongoDB-level filtering for optimal performance following the ToolService pattern.

## Key Components

- **queryOrganizations(filterOptions, search)** - Returns all organizations without pagination for backward compatibility
- **queryOrganizations(filterOptions, paginationCriteria, search)** - Returns paginated organization results with cursor-based navigation
- **buildQueryFilter()** - Converts filter options into MongoDB query filters
- **fetchPageItems()** - Handles cursor pagination logic with limit + 1 strategy
- **buildPageInfo()** - Creates pagination metadata including cursor positions

## Usage Example

```java
@Autowired
private OrganizationQueryService organizationQueryService;

// Simple query without pagination
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("TECHNOLOGY")
    .minEmployees(100)
    .hasActiveContract(true)
    .build();

OrganizationList result = organizationQueryService.queryOrganizations(filters, "search term");

// Paginated query with cursor
CursorPaginationCriteria pagination = CursorPaginationCriteria.builder()
    .limit(20)
    .cursor("previous_page_end_cursor")
    .build();

CountedGenericQueryResult<Organization> pagedResult = 
    organizationQueryService.queryOrganizations(filters, pagination, "search term");
```
