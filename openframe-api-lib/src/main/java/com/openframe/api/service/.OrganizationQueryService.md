Service responsible for querying organizations with filtering, searching, and pagination capabilities using MongoDB-level filtering for optimal performance.

## Key Components

- **queryOrganizations(OrganizationFilterOptions, String)** - Returns all organizations without pagination for backward compatibility
- **queryOrganizations(OrganizationFilterOptions, CursorPaginationCriteria, String)** - Returns paginated organization results with cursor-based navigation
- **buildQueryFilter(OrganizationFilterOptions)** - Converts filter options into MongoDB query filters
- **fetchPageItems(Query, CursorPaginationCriteria)** - Retrieves paginated organization data
- **buildPageInfo(List<Organization>, boolean, boolean)** - Constructs cursor pagination metadata

## Usage Example

```java
@Autowired
private OrganizationQueryService organizationQueryService;

// Simple query without pagination
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("TECH")
    .minEmployees(10)
    .hasActiveContract(true)
    .build();

OrganizationList result = organizationQueryService.queryOrganizations(filters, "startup");

// Paginated query with cursor
CursorPaginationCriteria pagination = CursorPaginationCriteria.builder()
    .limit(20)
    .cursor("organization-id-123")
    .build();

CountedGenericQueryResult<Organization> paginatedResult = 
    organizationQueryService.queryOrganizations(filters, pagination, "tech company");
```
