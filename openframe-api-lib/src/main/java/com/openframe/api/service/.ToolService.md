A Spring service that manages integrated tool queries and filtering functionality, providing search capabilities and dynamic filter options for tool discovery.

## Key Components

- **queryTools(ToolFilterOptions, String)** - Searches and filters integrated tools based on criteria and text search
- **getToolFilters()** - Retrieves available filter values (types, categories, platform categories) for UI components
- **buildQueryFilter(ToolFilterOptions)** - Private helper that converts filter options into query filter objects
- **IntegratedToolRepository** - Injected repository for tool data access

## Usage Example

```java
@RestController
public class ToolController {
    
    private final ToolService toolService;
    
    @GetMapping("/tools")
    public ToolList searchTools(
            @RequestParam(required = false) String search,
            @RequestParam(required = false) Boolean enabled,
            @RequestParam(required = false) String type) {
        
        ToolFilterOptions filters = ToolFilterOptions.builder()
                .enabled(enabled)
                .type(type)
                .build();
                
        return toolService.queryTools(filters, search);
    }
    
    @GetMapping("/tools/filters")
    public ToolFilters getAvailableFilters() {
        return toolService.getToolFilters();
    }
}
```
