A Spring service that handles tool discovery and filtering operations for the OpenFrame API, providing search and filter capabilities for integrated tools.

## Key Components

**Main Service Methods:**
- `queryTools(ToolFilterOptions, String)` - Searches and filters integrated tools based on criteria and search terms
- `getToolFilters()` - Retrieves all available filter options (types, categories, platform categories)

**Private Helper:**
- `buildQueryFilter(ToolFilterOptions)` - Converts API filter options into repository query filters

**Dependencies:**
- `IntegratedToolRepository` - Data access layer for tool operations
- Various DTOs for request/response handling (`ToolList`, `ToolFilters`, `ToolFilterOptions`)

## Usage Example

```java
@RestController
public class ToolController {
    
    private final ToolService toolService;
    
    @GetMapping("/tools")
    public ToolList searchTools(
        @RequestParam(required = false) String search,
        @ModelAttribute ToolFilterOptions filters) {
        
        return toolService.queryTools(filters, search);
    }
    
    @GetMapping("/tools/filters")
    public ToolFilters getAvailableFilters() {
        return toolService.getToolFilters();
    }
}
```

The service acts as an intermediary between the REST API layer and the data repository, transforming filter options into appropriate query parameters and building structured responses for tool discovery functionality.
