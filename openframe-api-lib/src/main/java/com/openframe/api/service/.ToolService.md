A Spring service that handles tool management operations including searching, filtering, and retrieving metadata about integrated tools in the system.

## Key Components

- **queryTools()** - Searches and filters tools based on provided criteria and search terms
- **getToolFilters()** - Retrieves available filter options (types, categories, platform categories)
- **buildQueryFilter()** - Private helper method that converts filter options into query filter objects
- **IntegratedToolRepository** - Injected repository for database operations

## Usage Example

```java
@Autowired
private ToolService toolService;

// Query tools with filters
ToolFilterOptions filters = ToolFilterOptions.builder()
    .enabled(true)
    .type("API")
    .category("Development")
    .build();

ToolList results = toolService.queryTools(filters, "authentication");

// Get available filter options
ToolFilters availableFilters = toolService.getToolFilters();
List<String> categories = availableFilters.getCategories();

// Query all tools without filters
ToolList allTools = toolService.queryTools(null, null);
```

The service uses Spring's dependency injection and includes debug logging for monitoring filter operations and search queries.
