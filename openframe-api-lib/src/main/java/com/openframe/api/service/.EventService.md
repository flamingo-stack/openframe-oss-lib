A Spring service class that handles CRUD operations and querying for Event entities with support for filtering, searching, and cursor-based pagination.

## Key Components

- **queryEvents()** - Retrieves paginated events with filtering and search capabilities
- **findById()** - Fetches a single event by its unique identifier
- **createEvent()** - Creates new events with auto-generated ID and timestamp
- **updateEvent()** - Updates existing events by ID
- **getEventFilters()** - Returns available filter options (user IDs and event types)
- **Private helper methods** - Support pagination, query building, and page info construction

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class EventController {
    
    private final EventService eventService;
    
    public GenericQueryResult<Event> getEvents(
            EventFilterOptions filters,
            CursorPaginationCriteria pagination,
            String search) {
        return eventService.queryEvents(filters, pagination, search);
    }
    
    public Event createEvent(Event event) {
        return eventService.createEvent(event);
    }
    
    public Optional<Event> getEvent(String id) {
        return eventService.findById(id);
    }
    
    public EventFilters getAvailableFilters() {
        return eventService.getEventFilters();
    }
}
```

The service integrates with MongoDB through EventRepository and provides comprehensive event management with proper logging and error handling.
