A Spring service class that provides comprehensive event management operations including querying, CRUD operations, and filtering capabilities with cursor-based pagination support.

## Key Components

- **queryEvents()** - Executes filtered and paginated event queries with search functionality
- **findById()** - Retrieves a single event by its unique identifier
- **createEvent()** - Creates new events with auto-generated ID and timestamp
- **updateEvent()** - Updates existing events with validation
- **getEventFilters()** - Returns available filter options (user IDs and event types)
- **Helper methods** - Private utilities for pagination, query building, and cursor management

## Usage Example

```java
@Autowired
private EventService eventService;

// Query events with filters and pagination
EventFilterOptions filters = EventFilterOptions.builder()
    .userIds(List.of("user123"))
    .eventTypes(List.of("LOGIN", "LOGOUT"))
    .build();

CursorPaginationCriteria pagination = CursorPaginationCriteria.builder()
    .limit(10)
    .build();

GenericQueryResult<Event> result = eventService.queryEvents(filters, pagination, "search term");

// Create a new event
Event newEvent = Event.builder()
    .userId("user123")
    .eventType("LOGIN")
    .data(Map.of("ip", "192.168.1.1"))
    .build();

Event savedEvent = eventService.createEvent(newEvent);

// Get available filter options
EventFilters availableFilters = eventService.getEventFilters();
```
