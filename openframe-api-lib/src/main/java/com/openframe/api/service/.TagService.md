A Spring service class that manages tag retrieval operations for machines, providing both single and bulk tag lookup functionality with optimized database queries.

## Key Components

- **`getTagsForMachines(List<String> machineIds)`** - Bulk retrieval method that fetches tags for multiple machines efficiently using batch database operations
- **`getTagsForMachine(String machineId)`** - Single machine tag retrieval method
- **TagRepository** - Repository for Tag entity operations
- **MachineTagRepository** - Repository for MachineTag relationship operations

## Usage Example

```java
@Autowired
private TagService tagService;

// Get tags for a single machine
List<Tag> singleMachineTags = tagService.getTagsForMachine("machine-123");

// Get tags for multiple machines (bulk operation)
List<String> machineIds = Arrays.asList("machine-123", "machine-456", "machine-789");
List<List<Tag>> multipleMachineTags = tagService.getTagsForMachines(machineIds);

// Results are returned in the same order as requested machine IDs
for (int i = 0; i < machineIds.size(); i++) {
    String machineId = machineIds.get(i);
    List<Tag> tags = multipleMachineTags.get(i);
    System.out.println("Machine " + machineId + " has " + tags.size() + " tags");
}
```

The service uses optimized batch queries to minimize database calls and maintains result ordering for predictable behavior in bulk operations.
