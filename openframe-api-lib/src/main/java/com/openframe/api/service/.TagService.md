A Spring service that manages tag retrieval operations for machines, providing both single and bulk machine tag lookup functionality with optimized database queries.

## Key Components

- **`getTagsForMachines(List<String> machineIds)`** - Bulk retrieval method that fetches tags for multiple machines in a single operation, minimizing database calls through batched queries
- **`getTagsForMachine(String machineId)`** - Single machine tag retrieval method for simpler use cases
- **Repository dependencies** - `TagRepository` for tag data access and `MachineTagRepository` for machine-tag associations

## Usage Example

```java
@Autowired
private TagService tagService;

// Get tags for a single machine
List<Tag> singleMachineTags = tagService.getTagsForMachine("machine-123");

// Get tags for multiple machines (bulk operation)
List<String> machineIds = Arrays.asList("machine-123", "machine-456", "machine-789");
List<List<Tag>> multipleMachineTags = tagService.getTagsForMachines(machineIds);

// Process results - each list corresponds to tags for the machine at the same index
for (int i = 0; i < machineIds.size(); i++) {
    String machineId = machineIds.get(i);
    List<Tag> tags = multipleMachineTags.get(i);
    System.out.println("Machine " + machineId + " has " + tags.size() + " tags");
}
```

The service uses efficient bulk loading strategies to minimize database round trips and maintains result ordering for predictable behavior.
