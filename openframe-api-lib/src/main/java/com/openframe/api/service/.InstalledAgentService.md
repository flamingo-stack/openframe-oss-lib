Service layer component that manages CRUD operations for installed agent data across different machines in the OpenFrame system. It provides methods for retrieving agent information both individually and in bulk for efficient data loading operations.

## Key Components

- **getInstalledAgentsForMachines()** - Bulk retrieval method that returns agents grouped by machine IDs, optimized for DataLoader usage
- **getInstalledAgentsForMachine()** - Retrieves all agents installed on a specific machine
- **getAllInstalledAgents()** - Fetches complete list of all installed agents
- **getInstalledAgent()** - Finds a specific agent by ID
- **getInstalledAgentByMachineIdAndType()** - Locates agent by machine ID and agent type combination
- **InstalledAgentRepository** - Injected repository dependency for data access

## Usage Example

```java
@Autowired
private InstalledAgentService agentService;

// Get agents for multiple machines (bulk operation)
List<String> machineIds = Arrays.asList("machine1", "machine2", "machine3");
List<List<InstalledAgent>> agentsByMachine = agentService.getInstalledAgentsForMachines(machineIds);

// Get agents for single machine
List<InstalledAgent> machineAgents = agentService.getInstalledAgentsForMachine("machine1");

// Find specific agent type on machine
Optional<InstalledAgent> agent = agentService.getInstalledAgentByMachineIdAndType("machine1", "monitoring");
```
