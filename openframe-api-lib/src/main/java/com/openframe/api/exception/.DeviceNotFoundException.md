A custom runtime exception thrown when a requested device cannot be found in the OpenFrame API system.

## Key Components

- **DeviceNotFoundException** - Custom exception class extending `RuntimeException`
- **Constructor** - Single parameterized constructor accepting an error message

## Usage Example

```java
// Throwing the exception when device lookup fails
public Device getDeviceById(String deviceId) {
    Device device = deviceRepository.findById(deviceId);
    if (device == null) {
        throw new DeviceNotFoundException("Device with ID " + deviceId + " not found");
    }
    return device;
}

// Handling the exception in a service layer
try {
    Device device = deviceService.getDeviceById("invalid-id");
} catch (DeviceNotFoundException e) {
    logger.error("Device lookup failed: " + e.getMessage());
    // Handle appropriately - return error response, etc.
}
```

This exception follows standard Java exception patterns by extending `RuntimeException`, making it an unchecked exception that doesn't require explicit handling but can be caught when needed for error handling or logging purposes.
