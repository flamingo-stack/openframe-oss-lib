Custom exception thrown when a requested device cannot be located, extending `NotFoundException` with the specific `DEVICE_NOT_FOUND` error code.

## Key Components

| Component | Description |
|-----------|-------------|
| `DeviceNotFoundException` | Extends `NotFoundException` to represent a missing device resource |
| `ErrorCode.DEVICE_NOT_FOUND` | Standardized error code injected automatically on construction |

## Usage Example

```java
// Throw when a device lookup returns no result
Device device = deviceRepository.findById(deviceId)
    .orElseThrow(() -> new DeviceNotFoundException(
        "Device with ID " + deviceId + " was not found"
    ));
```
