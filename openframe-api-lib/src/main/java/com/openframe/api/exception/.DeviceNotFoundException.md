A custom runtime exception thrown when a requested device cannot be found in the system.

## Key Components

- **DeviceNotFoundException**: Custom exception class extending `RuntimeException`
- **Constructor**: Accepts a descriptive error message parameter

## Usage Example

```java
// Throwing the exception when device lookup fails
public Device findDeviceById(String deviceId) {
    Device device = deviceRepository.findById(deviceId);
    if (device == null) {
        throw new DeviceNotFoundException("Device with ID '" + deviceId + "' not found");
    }
    return device;
}

// Catching and handling the exception
try {
    Device device = deviceService.findDeviceById("invalid-id");
} catch (DeviceNotFoundException e) {
    logger.error("Device lookup failed: " + e.getMessage());
    return ResponseEntity.notFound().build();
}
```

This exception provides a clear, semantic way to handle device-not-found scenarios in the API, allowing callers to distinguish between general errors and specific device lookup failures.
