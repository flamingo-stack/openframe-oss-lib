Data Transfer Object (DTO) used to carry update parameters for a knowledge base article in the OpenFrame API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier of the article to update |
| `name` | `String` | Updated article title/name |
| `parentId` | `String` | Parent category or article identifier |
| `content` | `String` | Full article body content |
| `summary` | `String` | Brief description or excerpt of the article |

## Lombok Annotations

- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern instantiation
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors

## Usage Example

```java
// Builder pattern
UpdateArticleCommand command = UpdateArticleCommand.builder()
    .id("article-123")
    .name("VPN Troubleshooting Guide")
    .parentId("category-456")
    .content("## Overview\nThis guide covers common VPN issues...")
    .summary("Step-by-step VPN troubleshooting for end users")
    .build();

// All-args constructor
UpdateArticleCommand command = new UpdateArticleCommand(
    "article-123",
    "VPN Troubleshooting Guide",
    "category-456",
    "## Overview\nThis guide covers...",
    "Step-by-step VPN troubleshooting"
);
```
