Data Transfer Object (DTO) for encapsulating filter criteria when querying or listing tools in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `enabled` | `Boolean` | Filter tools by their enabled/disabled status |
| `type` | `String` | Filter by tool type |
| `category` | `String` | Filter by tool category |
| `platformCategory` | `String` | Filter by platform-specific category |

## Usage Example

```java
// Build filter criteria using the builder pattern
ToolFilterCriteria criteria = ToolFilterCriteria.builder()
    .enabled(true)
    .type("automation")
    .category("monitoring")
    .platformCategory("infrastructure")
    .build();

// Or use the no-args constructor with setters
ToolFilterCriteria criteria = new ToolFilterCriteria();
criteria.setEnabled(true);
criteria.setCategory("monitoring");
```

> **Note:** All fields are optional — only populate the fields you need to filter on. Unpopulated fields should be treated as "no filter applied" by the consuming service.
