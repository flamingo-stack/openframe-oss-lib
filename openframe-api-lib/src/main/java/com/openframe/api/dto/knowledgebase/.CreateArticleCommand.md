Data Transfer Object (DTO) representing the command payload for creating a new Knowledge Base article in the OpenFrame API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | Title of the article |
| `parentId` | `String` | ID of the parent article or category |
| `content` | `String` | Full body content of the article |
| `summary` | `String` | Short description or excerpt |
| `status` | `KnowledgeBaseArticleStatus` | Publication status (e.g., draft, published) |
| `tagIds` | `List<String>` | Associated tag identifiers |
| `assignedOrganizationIds` | `List<String>` | Organizations linked to the article |
| `assignedDeviceIds` | `List<String>` | Devices linked to the article |
| `assignedTicketIds` | `List<String>` | Support tickets linked to the article |
| `assignedKnowledgeArticleIds` | `List<String>` | Related knowledge base articles |

**Lombok Annotations:** `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
CreateArticleCommand command = CreateArticleCommand.builder()
    .name("How to Reset VPN Credentials")
    .parentId("kb-cat-network-001")
    .content("Step-by-step guide to resetting VPN credentials...")
    .summary("Quick guide for VPN credential reset")
    .status(KnowledgeBaseArticleStatus.PUBLISHED)
    .tagIds(List.of("tag-vpn", "tag-security"))
    .assignedOrganizationIds(List.of("org-acme-123"))
    .assignedDeviceIds(List.of("dev-win-456"))
    .assignedTicketIds(List.of("ticket-789"))
    .assignedKnowledgeArticleIds(List.of("kb-art-related-001"))
    .build();
```
