Data Transfer Object (DTO) for capturing the required inputs to cancel an ongoing command execution.

## Key Components

| Field | Type | Constraint | Description |
|-------|------|------------|-------------|
| `machineId` | `String` | `@NotBlank` | Identifies the target machine running the execution |
| `executionId` | `String` | `@NotBlank` | Identifies the specific execution to cancel |

## Usage Example

```java
CancelExecutionInput input = new CancelExecutionInput();
input.setMachineId("machine-abc123");
input.setExecutionId("exec-xyz789");

// Pass to command service
commandService.cancelExecution(input);
```

> Both fields are mandatory — validation will reject any request where either `machineId` or `executionId` is null or blank.
