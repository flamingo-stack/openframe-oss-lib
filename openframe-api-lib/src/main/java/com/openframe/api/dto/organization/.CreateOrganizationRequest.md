Data transfer object for creating new organizations in the OpenFrame API system. This record serves as a shared DTO between GraphQL and REST endpoints with built-in validation constraints.

## Key Components

- **CreateOrganizationRequest** - Immutable record with Lombok builder pattern
- **name** - Required organization name (validated with `@NotBlank`)
- **numberOfEmployees** - Optional employee count (must be non-negative)
- **contactInformation** - Nested DTO for contact details (validated with `@Valid`)
- **monthlyRevenue** - Optional revenue field (must be non-negative)
- **contractStartDate/contractEndDate** - Optional contract duration fields

## Usage Example

```java
// Create organization request with builder pattern
CreateOrganizationRequest request = CreateOrganizationRequest.builder()
    .name("Acme Corporation")
    .category("Technology")
    .numberOfEmployees(150)
    .websiteUrl("https://acme.com")
    .monthlyRevenue(new BigDecimal("50000.00"))
    .contactInformation(ContactInformationDto.builder()
        .email("contact@acme.com")
        .phone("+1-555-0123")
        .build())
    .contractStartDate(LocalDate.now())
    .contractEndDate(LocalDate.now().plusYears(1))
    .build();

// The organizationId will be auto-generated as UUID on the backend
```
