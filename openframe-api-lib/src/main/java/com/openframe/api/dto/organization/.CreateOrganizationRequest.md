A data transfer object (DTO) used for creating new organizations across both GraphQL and REST APIs, with automatic UUID generation for the organization ID.

## Key Components

- **CreateOrganizationRequest** - Record class with validation annotations for organization creation
- **Required fields** - `name` (with `@NotBlank` validation)
- **Optional fields** - `category`, `numberOfEmployees`, `websiteUrl`, `notes`, `monthlyRevenue`, contract dates
- **Nested object** - `ContactInformationDto` with `@Valid` annotation
- **Validation constraints** - `@PositiveOrZero` for numeric fields, `@NotBlank` for required strings

## Usage Example

```java
// Create organization request with required fields
CreateOrganizationRequest request = CreateOrganizationRequest.builder()
    .name("Acme Corporation")
    .category("Technology")
    .numberOfEmployees(50)
    .websiteUrl("https://acme.com")
    .contactInformation(ContactInformationDto.builder()
        .email("contact@acme.com")
        .phone("+1-555-0123")
        .build())
    .monthlyRevenue(new BigDecimal("100000.00"))
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .build();

// Minimal request with only required field
CreateOrganizationRequest minimal = CreateOrganizationRequest.builder()
    .name("Basic Company")
    .build();
```
