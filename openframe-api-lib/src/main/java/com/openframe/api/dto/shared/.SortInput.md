A GraphQL input DTO for specifying sort parameters in query operations, enabling clients to sort results by field name and direction.

## Key Components

- **`field`** - String specifying which field to sort by
- **`direction`** - Enum value determining sort order (ascending/descending)
- **Lombok annotations** - Provides automatic getter/setter generation, builder pattern, and constructors

## Usage Example

```java
// Using builder pattern to create sort input
SortInput nameSort = SortInput.builder()
    .field("name")
    .direction(SortDirection.ASC)
    .build();

SortInput createdDateSort = SortInput.builder()
    .field("createdAt")
    .direction(SortDirection.DESC)
    .build();

// In GraphQL resolver
@QueryMapping
public List<User> getUsers(
    @Argument List<SortInput> sort
) {
    return userService.findUsersWithSort(sort);
}
```

This DTO works with GraphQL queries to provide flexible sorting capabilities, allowing clients to specify multiple sort criteria for complex data retrieval operations.
