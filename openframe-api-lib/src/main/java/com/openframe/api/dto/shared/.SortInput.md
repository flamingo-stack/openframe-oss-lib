A DTO (Data Transfer Object) representing a sorting specification, encapsulating a field name and sort direction for use in API query operations.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `field` | `String` | The field/column name to sort by |
| `direction` | `SortDirection` | Enum indicating `ASC` or `DESC` ordering |
| `from(String, String)` | `static SortInput` | Factory method that safely constructs a `SortInput` from raw string inputs |

## Usage Example

```java
// Using the factory method (preferred)
SortInput sort = SortInput.from("createdAt", "DESC");

// Returns null if field is blank or null (safe for optional sort params)
SortInput nullSort = SortInput.from("", "ASC"); // → null

// Using the builder directly
SortInput sort = SortInput.builder()
    .field("lastName")
    .direction(SortDirection.ASC)
    .build();
```

## Notes

- The `from()` factory method guards against empty/null fields, returning `null` instead of an invalid sort object — useful for optional query parameters.
- Direction parsing is case-insensitive; any value other than `"ASC"` (e.g. `"desc"`, `"random"`) defaults to `SortDirection.DESC`.
- Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder, and constructors.
