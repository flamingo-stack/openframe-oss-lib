A DTO (Data Transfer Object) used to encapsulate filter parameters when querying events within the OpenFrame API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `userIds` | `List<String>` | Filter events by one or more user IDs |
| `eventTypes` | `List<String>` | Filter events by one or more event type identifiers |
| `startDate` | `LocalDate` | Inclusive start date for the event date range filter |
| `endDate` | `LocalDate` | Inclusive end date for the event date range filter |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder, and constructors.

## Usage Example

```java
// Using the builder pattern to construct filter criteria
EventFilterCriteria criteria = EventFilterCriteria.builder()
    .userIds(List.of("user-123", "user-456"))
    .eventTypes(List.of("LOGIN", "TICKET_CREATED"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 12, 31))
    .build();

// Or using the all-args constructor
EventFilterCriteria criteria = new EventFilterCriteria(
    List.of("user-123"),
    List.of("TICKET_CLOSED"),
    LocalDate.now().minusDays(30),
    LocalDate.now()
);
```
