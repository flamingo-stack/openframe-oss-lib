A simple data transfer object that wraps a list of IntegratedTool objects for API responses and requests.

## Key Components

- **ToolList class**: Main DTO class with Lombok annotations for boilerplate code generation
- **tools field**: List of `IntegratedTool` objects representing available tools
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic getter/setter/constructor generation

## Usage Example

```java
// Creating a ToolList with builder pattern
ToolList toolList = ToolList.builder()
    .tools(Arrays.asList(tool1, tool2, tool3))
    .build();

// Creating empty ToolList
ToolList emptyList = new ToolList();
emptyList.setTools(new ArrayList<>());

// Accessing tools
List<IntegratedTool> availableTools = toolList.getTools();
```

This DTO is typically used in REST API endpoints to return collections of tools or accept tool lists in request payloads, providing a clean wrapper around the raw List type.
