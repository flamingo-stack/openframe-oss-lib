Data Transfer Object (DTO) that wraps a collection of integrated tools for API responses and requests.

## Key Components

- **ToolList class**: Main DTO with Lombok annotations for automatic getter/setter generation
- **tools field**: List of `IntegratedTool` objects representing the collection of tools
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Creating a ToolList with builder pattern
ToolList toolList = ToolList.builder()
    .tools(Arrays.asList(tool1, tool2, tool3))
    .build();

// Creating empty ToolList
ToolList emptyList = new ToolList();
emptyList.setTools(new ArrayList<>());

// Accessing tools
List<IntegratedTool> tools = toolList.getTools();
for (IntegratedTool tool : tools) {
    // Process each tool
    System.out.println(tool.getName());
}

// In REST controller
@GetMapping("/tools")
public ToolList getAllTools() {
    return toolService.getAllToolsAsDTO();
}
```
