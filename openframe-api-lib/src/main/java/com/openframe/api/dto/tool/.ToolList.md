A simple data transfer object (DTO) that wraps a list of integrated tools for API responses and requests in the OpenFrame application.

## Key Components

- **ToolList** - Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **tools** - List of `IntegratedTool` objects representing available tools in the system

## Usage Example

```java
// Creating a tool list with builder pattern
ToolList toolList = ToolList.builder()
    .tools(Arrays.asList(
        new IntegratedTool("calculator"),
        new IntegratedTool("notepad")
    ))
    .build();

// Accessing tools
List<IntegratedTool> availableTools = toolList.getTools();

// Creating empty tool list
ToolList emptyList = new ToolList();
emptyList.setTools(new ArrayList<>());
```

The class uses Lombok annotations to reduce boilerplate code, automatically generating standard Java methods. This DTO is typically used in REST API endpoints to return collections of tools or receive tool lists in request payloads.
