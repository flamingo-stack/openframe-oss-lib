A data transfer object (DTO) that encapsulates filtering criteria for querying tools, supporting multiple filter types for flexible tool searches.

## Key Components

- **types**: List of tool type strings for filtering by tool classification
- **categories**: List of category strings for filtering by tool categories  
- **platformCategories**: List of platform-specific category strings for filtering by platform context
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filters using builder pattern
ToolFilters filters = ToolFilters.builder()
    .types(List.of("web", "mobile", "desktop"))
    .categories(List.of("development", "testing"))
    .platformCategories(List.of("java", "javascript"))
    .build();

// Create filters with constructor
ToolFilters filters2 = new ToolFilters(
    List.of("api", "cli"),
    List.of("productivity"),
    List.of("cloud", "on-premise")
);

// Use in service layer
List<Tool> filteredTools = toolService.findTools(filters);
```
