Data Transfer Object for filtering tools in the OpenFrame API, providing criteria for searching and categorizing tools by type, category, and platform.

## Key Components

- **types** - List of tool type filters to apply
- **categories** - List of general category filters for tool classification
- **platformCategories** - List of platform-specific category filters

## Usage Example

```java
// Create filters for specific tool types and categories
ToolFilters filters = ToolFilters.builder()
    .types(List.of("analyzer", "generator", "validator"))
    .categories(List.of("development", "testing"))
    .platformCategories(List.of("web", "mobile"))
    .build();

// Create empty filters for no filtering
ToolFilters noFilters = new ToolFilters();

// Apply specific platform filtering only
ToolFilters platformOnly = ToolFilters.builder()
    .platformCategories(List.of("desktop", "cli"))
    .build();
```

This DTO uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it easy to construct filter objects for tool search and discovery operations.
