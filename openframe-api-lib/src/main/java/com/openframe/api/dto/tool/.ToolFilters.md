Data Transfer Object (DTO) for filtering tools based on multiple criteria including types, categories, and platform categories.

## Key Components

- **types** - List of tool type filters
- **categories** - List of tool category filters  
- **platformCategories** - List of platform-specific category filters
- Uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter using builder pattern
ToolFilters filters = ToolFilters.builder()
    .types(List.of("automation", "testing"))
    .categories(List.of("development", "ci-cd"))
    .platformCategories(List.of("web", "mobile"))
    .build();

// Create empty filter
ToolFilters emptyFilter = new ToolFilters();

// Create filter with specific categories only
ToolFilters categoryFilter = ToolFilters.builder()
    .categories(List.of("database", "monitoring"))
    .build();
```
