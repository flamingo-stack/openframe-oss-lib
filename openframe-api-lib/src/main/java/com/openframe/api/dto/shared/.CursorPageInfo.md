A data transfer object that provides pagination metadata for cursor-based pagination, containing information about page boundaries and navigation availability.

## Key Components

- **hasNextPage** - Boolean indicating if more records exist after the current page
- **hasPreviousPage** - Boolean indicating if records exist before the current page  
- **startCursor** - String cursor pointing to the first item in the current page
- **endCursor** - String cursor pointing to the last item in the current page

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern support.

## Usage Example

```java
// Creating pagination info for a result set
CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("eyJpZCI6MTAwfQ==")
    .endCursor("eyJpZCI6MTIwfQ==")
    .build();

// Using in a paginated response
if (pageInfo.isHasNextPage()) {
    String nextCursor = pageInfo.getEndCursor();
    // Fetch next page using endCursor
}

// Checking for previous page navigation
if (pageInfo.isHasPreviousPage()) {
    String prevCursor = pageInfo.getStartCursor();
    // Navigate to previous page
}
```
