A data transfer object that provides pagination metadata for cursor-based pagination, indicating whether additional pages exist and the cursor positions for navigation.

## Key Components

- **hasNextPage**: Boolean flag indicating if more results exist after the current page
- **hasPreviousPage**: Boolean flag indicating if results exist before the current page  
- **startCursor**: String cursor pointing to the first item in the current page
- **endCursor**: String cursor pointing to the last item in the current page

## Usage Example

```java
// Creating page info for a paginated response
CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("cursor_abc123")
    .endCursor("cursor_xyz789")
    .build();

// Using in a paginated API response
public class PaginatedResponse<T> {
    private List<T> data;
    private CursorPageInfo pageInfo;
}

// Client-side navigation logic
if (pageInfo.isHasNextPage()) {
    // Fetch next page using endCursor
    fetchNextPage(pageInfo.getEndCursor());
}

if (pageInfo.isHasPreviousPage()) {
    // Fetch previous page using startCursor
    fetchPreviousPage(pageInfo.getStartCursor());
}
```
