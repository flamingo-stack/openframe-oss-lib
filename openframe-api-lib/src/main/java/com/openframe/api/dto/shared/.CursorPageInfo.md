Data Transfer Object (DTO) that provides cursor-based pagination information for GraphQL-style APIs. This class encapsulates navigation metadata for paginated result sets using cursor tokens.

## Key Components

- **hasNextPage** - Boolean flag indicating if more records exist after the current page
- **hasPreviousPage** - Boolean flag indicating if records exist before the current page  
- **startCursor** - String token representing the first item in the current page
- **endCursor** - String token representing the last item in the current page

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern methods.

## Usage Example

```java
// Creating pagination info for a result set
CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("eyJpZCI6MX0=")
    .endCursor("eyJpZCI6MTB9")
    .build();

// Using in a paginated response
public class PaginatedResponse<T> {
    private List<T> data;
    private CursorPageInfo pageInfo;
}

// Checking navigation capabilities
if (pageInfo.isHasNextPage()) {
    // Load next page using endCursor
    fetchNextPage(pageInfo.getEndCursor());
}
```

This DTO is commonly used in GraphQL implementations following the Relay specification for cursor-based pagination, providing efficient navigation through large datasets without offset-based pagination drawbacks.
