A simple DTO representing a structured error response for mutation operations, containing a field path and a descriptive error message.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `field` | `List<String>` | Path segments identifying the field(s) that caused the error |
| `message` | `String` | Human-readable description of the error |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Builder pattern
MutationError error = MutationError.builder()
    .field(List.of("user", "email"))
    .message("Email address is already in use.")
    .build();

// No-args constructor with setters
MutationError error = new MutationError();
error.setField(List.of("password"));
error.setMessage("Password must be at least 8 characters.");

// All-args constructor
MutationError error = new MutationError(
    List.of("device", "hostname"),
    "Hostname already exists in this tenant."
);
```
