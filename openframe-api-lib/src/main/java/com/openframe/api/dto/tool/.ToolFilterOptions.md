A data transfer object (DTO) that encapsulates filtering criteria for tool-related queries. This class provides a standardized way to pass multiple filter parameters when searching or retrieving tools from the API.

## Key Components

- **enabled**: Boolean flag to filter tools by their active/inactive status
- **type**: String filter for tool type classification
- **category**: String filter for tool category grouping
- **platformCategory**: String filter for platform-specific tool categorization

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation.

## Usage Example

```java
// Create filter options using builder pattern
ToolFilterOptions filters = ToolFilterOptions.builder()
    .enabled(true)
    .type("INTEGRATION")
    .category("DATABASE")
    .platformCategory("CLOUD")
    .build();

// Create with specific filters only
ToolFilterOptions enabledOnly = ToolFilterOptions.builder()
    .enabled(true)
    .build();

// Use in service method
public List<Tool> getFilteredTools(ToolFilterOptions filters) {
    return toolRepository.findByFilters(
        filters.getEnabled(),
        filters.getType(),
        filters.getCategory(),
        filters.getPlatformCategory()
    );
}
```
