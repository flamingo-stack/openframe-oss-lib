A data transfer object (DTO) that represents filter criteria for querying and filtering tools in the OpenFrame API system.

## Key Components

- **enabled** - Boolean filter to include/exclude enabled tools
- **type** - String filter for tool type classification
- **category** - String filter for tool category grouping
- **platformCategory** - String filter for platform-specific tool categories

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation.

## Usage Example

```java
// Create filter to find enabled web development tools
ToolFilterOptions webFilter = ToolFilterOptions.builder()
    .enabled(true)
    .type("development")
    .category("web")
    .platformCategory("frontend")
    .build();

// Create filter for all disabled tools
ToolFilterOptions disabledFilter = ToolFilterOptions.builder()
    .enabled(false)
    .build();

// Filter by category only
ToolFilterOptions categoryFilter = new ToolFilterOptions();
categoryFilter.setCategory("analytics");
categoryFilter.setEnabled(true);
```

This DTO is typically used in service layer methods or API endpoints to provide flexible tool filtering capabilities, allowing clients to search for tools based on multiple optional criteria.
