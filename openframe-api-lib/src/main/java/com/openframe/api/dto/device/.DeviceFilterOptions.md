A data transfer object (DTO) that encapsulates filtering criteria for device queries, providing multiple filter dimensions for device searches.

## Key Components

- **statuses** - List of `DeviceStatus` enums to filter by device operational states
- **deviceTypes** - List of `DeviceType` enums to filter by hardware categories
- **osTypes** - List of operating system type strings for OS-based filtering
- **organizationIds** - List of organization identifier strings for multi-tenant filtering
- **tagNames** - List of tag name strings for metadata-based filtering

## Usage Example

```java
// Create filter options using builder pattern
DeviceFilterOptions filters = DeviceFilterOptions.builder()
    .statuses(List.of(DeviceStatus.ONLINE, DeviceStatus.IDLE))
    .deviceTypes(List.of(DeviceType.TABLET, DeviceType.SMARTPHONE))
    .osTypes(List.of("Android", "iOS"))
    .organizationIds(List.of("org123", "org456"))
    .tagNames(List.of("production", "critical"))
    .build();

// Or create with constructor
DeviceFilterOptions filters = new DeviceFilterOptions(
    List.of(DeviceStatus.ONLINE),
    List.of(DeviceType.TABLET),
    List.of("Android"),
    List.of("org123"),
    List.of("production")
);
```
