A data transfer object for specifying filter criteria when querying devices, supporting multiple filter dimensions including status, type, operating system, organization, and tags.

## Key Components

- **statuses** - List of `DeviceStatus` enum values to filter by device status
- **deviceTypes** - List of `DeviceType` enum values to filter by device type
- **osTypes** - List of operating system type strings for OS-based filtering
- **organizationIds** - List of organization ID strings for multi-tenant filtering
- **tagNames** - List of tag name strings for tag-based device filtering

## Usage Example

```java
// Create filter for active Android devices in specific organizations
DeviceFilterOptions filter = DeviceFilterOptions.builder()
    .statuses(List.of(DeviceStatus.ACTIVE, DeviceStatus.ONLINE))
    .deviceTypes(List.of(DeviceType.TABLET, DeviceType.SMARTPHONE))
    .osTypes(List.of("Android", "iOS"))
    .organizationIds(List.of("org123", "org456"))
    .tagNames(List.of("production", "critical"))
    .build();

// Use with device service for filtered queries
List<Device> filteredDevices = deviceService.findDevices(filter);
```
