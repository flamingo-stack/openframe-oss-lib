Data Transfer Object (DTO) for encapsulating filter parameters used when querying audit logs within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `startDate` | `LocalDate` | Start of the date range filter |
| `endDate` | `LocalDate` | End of the date range filter |
| `eventTypes` | `List<String>` | Filter by specific audit event types |
| `toolTypes` | `List<String>` | Filter by tool/integration types |
| `severities` | `List<String>` | Filter by log severity levels |
| `organizationIds` | `List<String>` | Filter logs by organization scope |
| `deviceId` | `String` | Filter logs for a specific device |

## Usage Example

```java
// Build a filter to query critical logs for specific orgs within a date range
LogFilterCriteria criteria = LogFilterCriteria.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .severities(List.of("CRITICAL", "ERROR"))
    .organizationIds(List.of("org-001", "org-002"))
    .eventTypes(List.of("LOGIN_FAILURE", "CONFIG_CHANGE"))
    .deviceId("device-abc123")
    .build();

// Or use no-args constructor and setters
LogFilterCriteria criteria = new LogFilterCriteria();
criteria.setStartDate(LocalDate.now().minusDays(7));
criteria.setSeverities(List.of("WARNING"));
```

Leverages Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) to auto-generate getters, setters, constructors, and builder pattern boilerplate.
