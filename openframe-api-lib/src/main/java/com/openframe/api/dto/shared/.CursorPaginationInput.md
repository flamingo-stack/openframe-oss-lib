A data transfer object for cursor-based pagination parameters, providing limit and cursor fields with validation constraints.

## Key Components

- **limit** - Integer field with validation constraints (1-100 range)
- **cursor** - String field for pagination cursor position
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation
- **Jakarta validation** - `@Min` and `@Max` constraints on the limit field

## Usage Example

```java
// Create pagination input with builder pattern
CursorPaginationInput pagination = CursorPaginationInput.builder()
    .limit(25)
    .cursor("eyJpZCI6MTIzfQ==")
    .build();

// Or using constructor
CursorPaginationInput pagination = new CursorPaginationInput(25, "eyJpZCI6MTIzfQ==");

// For first page request (no cursor)
CursorPaginationInput firstPage = CursorPaginationInput.builder()
    .limit(50)
    .build();

// Validation will automatically enforce:
// - limit between 1 and 100
// - appropriate error messages for constraint violations
```
