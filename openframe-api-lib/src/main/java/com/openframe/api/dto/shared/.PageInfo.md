A DTO (Data Transfer Object) representing pagination metadata for cursor-based pagination responses in the OpenFrame API.

## Key Components

| Field | Type | Description |
|---|---|---|
| `hasNextPage` | `boolean` | Indicates if more pages exist after the current one |
| `hasPreviousPage` | `boolean` | Indicates if pages exist before the current one |
| `startCursor` | `String` | Cursor pointing to the first item in the current page |
| `endCursor` | `String` | Cursor pointing to the last item in the current page |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Builder pattern
PageInfo pageInfo = PageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("cursor_abc123")
    .endCursor("cursor_xyz789")
    .build();

// Constructor pattern
PageInfo pageInfo = new PageInfo(true, false, "cursor_abc123", "cursor_xyz789");

// Typically embedded in a paginated response
PagedResponse<TicketDTO> response = PagedResponse.<TicketDTO>builder()
    .data(tickets)
    .pageInfo(pageInfo)
    .build();
```
