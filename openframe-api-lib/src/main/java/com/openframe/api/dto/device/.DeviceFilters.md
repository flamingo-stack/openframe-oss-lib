A DTO (Data Transfer Object) that encapsulates available filter options for querying devices, along with the resulting filtered count.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `statuses` | `List<DeviceFilterOption>` | Available device status filter options |
| `deviceTypes` | `List<DeviceFilterOption>` | Available device type filter options |
| `osTypes` | `List<DeviceFilterOption>` | Available operating system type filter options |
| `organizationIds` | `List<DeviceFilterOption>` | Available organization filter options |
| `tagKeys` | `List<TagFilterOption>` | Available tag key filter options |
| `filteredCount` | `Integer` | Total count of devices matching applied filters |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate boilerplate for getters/setters, the builder pattern, and constructors.

## Usage Example

```java
// Build a DeviceFilters response to return available filter options
DeviceFilters filters = DeviceFilters.builder()
    .statuses(List.of(
        new DeviceFilterOption("online", "Online"),
        new DeviceFilterOption("offline", "Offline")
    ))
    .deviceTypes(List.of(
        new DeviceFilterOption("workstation", "Workstation"),
        new DeviceFilterOption("server", "Server")
    ))
    .osTypes(List.of(
        new DeviceFilterOption("windows", "Windows"),
        new DeviceFilterOption("linux", "Linux")
    ))
    .organizationIds(List.of(
        new DeviceFilterOption("org-123", "Acme Corp")
    ))
    .tagKeys(List.of(
        new TagFilterOption("environment", List.of("prod", "staging"))
    ))
    .filteredCount(42)
    .build();
```
