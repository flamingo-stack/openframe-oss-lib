A DTO (Data Transfer Object) class that represents filtering options and results for device queries, containing various filter categories and the count of filtered results.

## Key Components

- **statuses** - List of device status filter options
- **deviceTypes** - List of device type filter options  
- **osTypes** - List of operating system type filter options
- **organizationIds** - List of organization ID filter options
- **tags** - List of tag-based filter options
- **filteredCount** - Integer representing the number of devices matching the applied filters

## Usage Example

```java
// Building device filters for a query
DeviceFilters filters = DeviceFilters.builder()
    .statuses(Arrays.asList(
        new DeviceFilterOption("ACTIVE", "Active Devices"),
        new DeviceFilterOption("INACTIVE", "Inactive Devices")
    ))
    .deviceTypes(Arrays.asList(
        new DeviceFilterOption("MOBILE", "Mobile Devices"),
        new DeviceFilterOption("TABLET", "Tablets")
    ))
    .osTypes(Arrays.asList(
        new DeviceFilterOption("ANDROID", "Android"),
        new DeviceFilterOption("IOS", "iOS")
    ))
    .filteredCount(150)
    .build();

// Using in API responses
return ResponseEntity.ok(filters);
```
