A data transfer object that encapsulates various filter criteria for device queries, providing structured filtering options for device searches and listings.

## Key Components

- **statuses** - List of device status filter options (active, inactive, etc.)
- **deviceTypes** - List of device type filter options (mobile, tablet, desktop, etc.)
- **osTypes** - List of operating system filter options (iOS, Android, Windows, etc.)
- **organizationIds** - List of organization identifier filter options
- **tags** - List of tag-based filter options for device categorization
- **filteredCount** - Integer representing the total count of devices matching current filters

## Usage Example

```java
// Create device filters for a specific query
DeviceFilters filters = DeviceFilters.builder()
    .statuses(Arrays.asList(
        new DeviceFilterOption("ACTIVE", "Active Devices"),
        new DeviceFilterOption("INACTIVE", "Inactive Devices")
    ))
    .deviceTypes(Arrays.asList(
        new DeviceFilterOption("MOBILE", "Mobile Devices")
    ))
    .osTypes(Arrays.asList(
        new DeviceFilterOption("ANDROID", "Android")
    ))
    .filteredCount(42)
    .build();

// Use in API response
return ResponseEntity.ok(filters);
```
