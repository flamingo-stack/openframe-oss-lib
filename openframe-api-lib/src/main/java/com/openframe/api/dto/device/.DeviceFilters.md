A DTO class that encapsulates filtering options and metadata for device queries, providing structured filter criteria and result count information.

## Key Components

- **statuses** - List of device status filter options (e.g., active, inactive)
- **deviceTypes** - List of device type filter options (e.g., mobile, tablet, desktop)
- **osTypes** - List of operating system filter options (e.g., Android, iOS, Windows)
- **organizationIds** - List of organization identifier filter options
- **tags** - List of tag-based filter options for device categorization
- **filteredCount** - Integer representing the total count of devices matching the applied filters

## Usage Example

```java
// Create device filters for a search query
DeviceFilters filters = DeviceFilters.builder()
    .statuses(List.of(new DeviceFilterOption("active", "Active Devices")))
    .deviceTypes(List.of(new DeviceFilterOption("mobile", "Mobile Devices")))
    .osTypes(List.of(new DeviceFilterOption("android", "Android")))
    .organizationIds(List.of(new DeviceFilterOption("org123", "Tech Corp")))
    .tags(List.of(new TagFilterOption("production", "Production Environment")))
    .filteredCount(150)
    .build();

// Access filter options
List<DeviceFilterOption> availableStatuses = filters.getStatuses();
Integer totalMatches = filters.getFilteredCount();
```
