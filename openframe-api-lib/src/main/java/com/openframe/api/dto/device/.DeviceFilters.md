A Data Transfer Object (DTO) that encapsulates filtering options and criteria for querying devices in the OpenFrame API, providing structured filter parameters for device searches.

## Key Components

- **Filter Collections**: Lists of `DeviceFilterOption` objects for status, device type, OS type, and organization filtering
- **Tag Filtering**: Specialized `TagFilterOption` list for tag-based device filtering
- **Result Metadata**: `filteredCount` integer to track the number of devices matching the applied filters
- **Lombok Annotations**: Automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create device filters for searching
DeviceFilters filters = DeviceFilters.builder()
    .statuses(List.of(
        new DeviceFilterOption("ACTIVE"),
        new DeviceFilterOption("ONLINE")
    ))
    .deviceTypes(List.of(
        new DeviceFilterOption("SMARTPHONE"),
        new DeviceFilterOption("TABLET")
    ))
    .osTypes(List.of(
        new DeviceFilterOption("ANDROID"),
        new DeviceFilterOption("IOS")
    ))
    .organizationIds(List.of(
        new DeviceFilterOption("org123"),
        new DeviceFilterOption("org456")
    ))
    .tags(List.of(
        new TagFilterOption("production"),
        new TagFilterOption("mobile-fleet")
    ))
    .filteredCount(42)
    .build();

// Use in API service
deviceService.searchDevices(filters);
```
