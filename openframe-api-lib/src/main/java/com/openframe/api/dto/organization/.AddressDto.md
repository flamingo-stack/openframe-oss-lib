A data transfer object that encapsulates address information with standard geographical components. This record uses the builder pattern for flexible object construction.

## Key Components

- **AddressDto**: Immutable record containing six address fields
- **@Builder**: Lombok annotation enabling builder pattern construction
- **Fields**: street1, street2, city, state, postalCode, country (all String types)

## Usage Example

```java
// Using builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Apt 4B")
    .city("Springfield")
    .state("IL")
    .postalCode("62701")
    .country("USA")
    .build();

// Using record constructor
AddressDto simpleAddress = new AddressDto(
    "456 Oak Ave", 
    null, 
    "Chicago", 
    "IL", 
    "60601", 
    "USA"
);

// Accessing fields
String fullAddress = address.street1() + ", " + address.city();
```

The record provides immutability and automatic implementations of equals(), hashCode(), and toString() methods, making it ideal for data transfer scenarios in REST APIs or between service layers.
