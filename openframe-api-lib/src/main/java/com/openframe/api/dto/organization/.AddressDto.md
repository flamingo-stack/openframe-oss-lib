A data transfer object (DTO) that represents address information used across the organization module of the API. This record class provides an immutable structure for handling address data with Lombok's builder pattern.

## Key Components

- **AddressDto**: A record class with builder pattern support for address data
- **Fields**: 
  - `street1` - Primary street address line
  - `street2` - Secondary street address line (optional)
  - `city` - City name
  - `state` - State or province
  - `postalCode` - ZIP or postal code
  - `country` - Country name

## Usage Example

```java
// Creating an address using the builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Suite 100")
    .city("New York")
    .state("NY")
    .postalCode("10001")
    .country("USA")
    .build();

// Accessing address data
String fullStreet = address.street1() + 
    (address.street2() != null ? ", " + address.street2() : "");
System.out.println("Address: " + fullStreet + ", " + address.city());

    // Implementation using the address DTO
```
