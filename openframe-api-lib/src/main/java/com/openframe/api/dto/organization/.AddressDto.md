A data transfer object (DTO) that represents address information using Java records with Lombok's builder pattern for immutable address data structures.

## Key Components

- **AddressDto**: Immutable record class representing a complete address
- **@Builder**: Lombok annotation providing a fluent builder pattern
- **Address Fields**: 
  - `street1` - Primary street address
  - `street2` - Secondary street address (optional)
  - `city` - City name
  - `state` - State/province
  - `postalCode` - ZIP/postal code
  - `country` - Country name

## Usage Example

```java
// Create address using builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Apt 4B")
    .city("New York")
    .state("NY")
    .postalCode("10001")
    .country("USA")
    .build();

// Access immutable fields
String fullStreet = address.street1() + " " + address.street2();
String location = address.city() + ", " + address.state();
```
