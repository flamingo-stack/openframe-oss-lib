A data transfer object (DTO) representing address information with standard address fields for use in organization-related API operations.

## Key Components

- **AddressDto record**: Immutable data structure containing six address fields
- **@Builder annotation**: Provides fluent builder pattern via Lombok
- **Fields**: 
  - `street1` - Primary street address
  - `street2` - Secondary address line (apartment, suite, etc.)
  - `city` - City name
  - `state` - State/province
  - `postalCode` - ZIP/postal code
  - `country` - Country name

## Usage Example

```java
// Using the builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Apt 4B")
    .city("Springfield")
    .state("Illinois")
    .postalCode("62701")
    .country("USA")
    .build();

// Accessing fields (records provide automatic getters)
String fullAddress = address.street1() + ", " + address.city();

// Records are immutable - all fields are final
System.out.println(address.postalCode()); // "62701"
```

The record provides automatic implementations of `equals()`, `hashCode()`, and `toString()` methods, making it ideal for data transfer scenarios where immutability and value semantics are desired.
