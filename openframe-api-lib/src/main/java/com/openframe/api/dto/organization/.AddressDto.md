A simple data transfer object representing address information using Java records and the builder pattern for immutable address data.

## Key Components

- **AddressDto** - Record class with builder pattern for address data
- **Fields**:
  - `street1` - Primary street address line
  - `street2` - Secondary street address line (optional)
  - `city` - City name
  - `state` - State or province
  - `postalCode` - ZIP/postal code
  - `country` - Country name

## Usage Example

```java
// Create address using builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Suite 100")
    .city("San Francisco")
    .state("CA")
    .postalCode("94105")
    .country("USA")
    .build();

// Access fields (records provide automatic getters)
String fullStreet = address.street1() + " " + address.street2();
String location = address.city() + ", " + address.state() + " " + address.postalCode();

// Use in API responses or organization data structures
OrganizationDto org = OrganizationDto.builder()
    .name("Acme Corp")
    .address(address)
    .build();
```

The record provides immutability, automatic equals/hashCode/toString methods, and the Lombok `@Builder` annotation enables fluent construction syntax.
