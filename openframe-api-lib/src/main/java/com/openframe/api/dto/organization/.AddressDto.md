Data Transfer Object (DTO) representing address information for organization-related operations. This record class provides an immutable structure for transferring address data between different layers of the application.

## Key Components

- **AddressDto record**: Immutable data structure with builder pattern support
- **@Builder annotation**: Enables fluent builder pattern for object creation
- **Address fields**: street1, street2, city, state, postalCode, country

## Usage Example

```java
// Creating an address using the builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Suite 456")
    .city("Springfield")
    .state("IL")
    .postalCode("62701")
    .country("USA")
    .build();

// Creating a minimal address
AddressDto simpleAddress = AddressDto.builder()
    .street1("789 Oak Avenue")
    .city("Chicago")
    .state("IL")
    .postalCode("60601")
    .country("USA")
    .build();

// Accessing fields (record provides automatic getters)
String city = address.city();
String fullStreet = address.street1() + " " + address.street2();
```
