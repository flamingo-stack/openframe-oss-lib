A data transfer object representing address information with comprehensive location fields including street, city, state, postal code, and country details.

## Key Components

- **AddressDto**: Immutable record class with builder pattern support
- **Fields**:
  - `street1`: Primary street address line
  - `street2`: Secondary street address line (optional)
  - `city`: City name
  - `state`: State or province
  - `postalCode`: ZIP/postal code
  - `country`: Country name

## Usage Example

```java
// Create address using builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Suite 456")
    .city("Springfield")
    .state("IL")
    .postalCode("62701")
    .country("USA")
    .build();

// Create minimal address (street2 can be null)
AddressDto simpleAddress = AddressDto.builder()
    .street1("789 Oak Avenue")
    .city("Chicago")
    .state("IL")
    .postalCode("60601")
    .country("USA")
    .build();

// Access fields
String fullStreet = address.street1() + " " + address.street2();
String location = address.city() + ", " + address.state();
```
