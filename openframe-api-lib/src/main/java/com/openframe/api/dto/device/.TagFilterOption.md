A data transfer object (DTO) that represents a filter option for device tags, containing the option's value, display label, and occurrence count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the option
- **count** - Number of items associated with this tag option

The class uses Lombok annotations for automatic generation of:
- Getters/setters (`@Data`)
- Builder pattern (`@Builder`)
- Default and all-args constructors (`@NoArgsConstructor`, `@AllArgsConstructor`)

## Usage Example

```java
// Create a tag filter option using builder pattern
TagFilterOption option = TagFilterOption.builder()
    .value("ACTIVE")
    .label("Active Devices")
    .count(25)
    .build();

// Use in a filter dropdown or API response
List<TagFilterOption> filterOptions = Arrays.asList(
    TagFilterOption.builder()
        .value("ACTIVE")
        .label("Active")
        .count(15)
        .build(),
    TagFilterOption.builder()
        .value("INACTIVE")
        .label("Inactive")
        .count(5)
        .build()
);
```

This DTO is typically used in UI components for displaying filterable tag options with their respective counts, enabling users to see available filters and how many items match each option.
