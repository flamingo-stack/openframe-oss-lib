A simple data transfer object (DTO) that represents a tag filter option with associated metadata for device filtering operations.

## Key Components

- **TagFilterOption**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **value**: String field representing the actual filter value
- **label**: String field for display text
- **count**: Integer field indicating how many items match this filter option

## Usage Example

```java
// Creating a tag filter option using builder pattern
TagFilterOption filterOption = TagFilterOption.builder()
    .value("mobile")
    .label("Mobile Devices")
    .count(25)
    .build();

// Creating with constructor
TagFilterOption option = new TagFilterOption("tablet", "Tablet Devices", 12);

// Accessing properties
String filterValue = filterOption.getValue();
String displayLabel = filterOption.getLabel();
Integer itemCount = filterOption.getCount();

// Typical usage in a filter dropdown
List<TagFilterOption> deviceFilters = Arrays.asList(
    new TagFilterOption("mobile", "Mobile", 25),
    new TagFilterOption("desktop", "Desktop", 18),
    new TagFilterOption("tablet", "Tablet", 7)
);
```
