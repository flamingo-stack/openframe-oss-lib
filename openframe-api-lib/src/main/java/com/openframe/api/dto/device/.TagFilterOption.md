A data transfer object representing a filter option for device tags, containing display information and occurrence count.

## Key Components

- **value**: The actual filter value used in queries
- **label**: Human-readable display text for the filter option
- **count**: Number of items that match this filter option

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern support.

## Usage Example

```java
// Creating a tag filter option with builder pattern
TagFilterOption filterOption = TagFilterOption.builder()
    .value("sensor")
    .label("Sensor Devices")
    .count(25)
    .build();

// Creating filter options for a dropdown menu
List<TagFilterOption> tagFilters = Arrays.asList(
    new TagFilterOption("camera", "Camera Devices", 12),
    new TagFilterOption("sensor", "Sensor Devices", 25),
    new TagFilterOption("actuator", "Actuator Devices", 8)
);

// Accessing properties
String displayText = filterOption.getLabel(); // "Sensor Devices"
int deviceCount = filterOption.getCount(); // 25
```

This DTO is typically used in APIs that provide filtering capabilities for device management interfaces, allowing clients to display available filter options with their respective counts.
