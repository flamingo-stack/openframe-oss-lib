A data transfer object representing a filter option for device tags, containing the option value, display label, and occurrence count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the option
- **count** - Number of items matching this filter option

## Usage Example

```java
// Creating a tag filter option for device status
TagFilterOption statusFilter = TagFilterOption.builder()
    .value("active")
    .label("Active Devices")
    .count(42)
    .build();

// Using in a filter dropdown
List<TagFilterOption> tagOptions = Arrays.asList(
    TagFilterOption.builder().value("production").label("Production").count(15).build(),
    TagFilterOption.builder().value("staging").label("Staging").count(8).build(),
    TagFilterOption.builder().value("development").label("Development").count(3).build()
);

// Accessing properties
String filterValue = statusFilter.getValue();    // "active"
String displayText = statusFilter.getLabel();   // "Active Devices" 
Integer itemCount = statusFilter.getCount();    // 42
```

This DTO is typically used in API responses to provide frontend components with the necessary data to render filter dropdowns or tag selection interfaces, where users need to see both the display name and the count of available items for each option.
