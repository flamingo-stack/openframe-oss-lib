A data transfer object representing a filter option for device tags, containing the option value, display label, and associated count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the option
- **count** - Number of items associated with this filter option

## Usage Example

```java
// Create a tag filter option
TagFilterOption option = TagFilterOption.builder()
    .value("production")
    .label("Production Environment")
    .count(25)
    .build();

// Use in a filter dropdown
List<TagFilterOption> filterOptions = Arrays.asList(
    new TagFilterOption("dev", "Development", 10),
    new TagFilterOption("staging", "Staging", 5),
    new TagFilterOption("prod", "Production", 25)
);

// Access properties
String filterValue = option.getValue();     // "production"
String displayText = option.getLabel();     // "Production Environment"  
Integer itemCount = option.getCount();      // 25
```

This DTO is typically used in UI components like dropdowns or filter lists where users can select device tags, showing both the human-friendly label and the count of matching devices.
