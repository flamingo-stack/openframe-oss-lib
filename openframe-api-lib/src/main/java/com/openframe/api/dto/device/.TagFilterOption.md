A simple Data Transfer Object (DTO) representing a tag-based filter option for device queries, containing a key-value pair and the count of devices matching that tag combination.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `key` | `String` | The tag key (e.g., `"os"`, `"location"`) |
| `value` | `String` | The tag value (e.g., `"windows"`, `"us-east"`) |
| `count` | `Integer` | Number of devices matching this tag key-value pair |

**Lombok Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors

## Usage Example

```java
// Builder pattern
TagFilterOption option = TagFilterOption.builder()
    .key("os")
    .value("windows")
    .count(42)
    .build();

// All-args constructor
TagFilterOption option = new TagFilterOption("location", "us-east", 15);

// Accessing fields
String tagKey   = option.getKey();    // "location"
String tagValue = option.getValue();  // "us-east"
Integer count   = option.getCount();  // 15
```

> **Typical use case:** Returned in filter aggregation responses to populate device filter dropdowns in the OpenFrame UI, showing users how many devices match each available tag combination.
