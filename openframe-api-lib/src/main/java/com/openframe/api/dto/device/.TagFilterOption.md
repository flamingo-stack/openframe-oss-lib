A data transfer object (DTO) that represents a filter option for device tags, containing the option's value, display label, and occurrence count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the option
- **count** - Number of items matching this filter option

The class uses Lombok annotations for automatic generation of:
- Getters/setters (`@Data`)
- Builder pattern (`@Builder`)
- Default and all-args constructors (`@NoArgsConstructor`, `@AllArgsConstructor`)

## Usage Example

```java
// Creating a tag filter option using builder pattern
TagFilterOption option = TagFilterOption.builder()
    .value("production")
    .label("Production Environment")
    .count(25)
    .build();

// Creating multiple filter options for a dropdown
List<TagFilterOption> filterOptions = Arrays.asList(
    new TagFilterOption("dev", "Development", 10),
    new TagFilterOption("staging", "Staging", 5),
    new TagFilterOption("prod", "Production", 25)
);

// Accessing properties
String displayText = option.getLabel(); // "Production Environment"
Integer itemCount = option.getCount();  // 25
```

This DTO is typically used in REST APIs to provide filterable tag options with metadata about how many devices match each filter criterion.
