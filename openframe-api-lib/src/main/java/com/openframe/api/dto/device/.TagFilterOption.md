A simple data transfer object (DTO) representing a filter option for device tags, containing the option value, display label, and occurrence count.

## Key Components

- **value**: The actual filter value used in queries
- **label**: Human-readable display text for the UI
- **count**: Number of items matching this filter option

## Usage Example

```java
// Create a tag filter option programmatically
TagFilterOption option = TagFilterOption.builder()
    .value("production")
    .label("Production Environment")
    .count(25)
    .build();

// Use in a filter dropdown
List<TagFilterOption> filterOptions = Arrays.asList(
    new TagFilterOption("dev", "Development", 10),
    new TagFilterOption("staging", "Staging", 5),
    new TagFilterOption("prod", "Production", 25)
);

// Access properties
String filterValue = option.getValue();     // "production"
String displayText = option.getLabel();     // "Production Environment" 
Integer itemCount = option.getCount();      // 25
```

This DTO is typically used in REST API responses to populate filter dropdowns or search interfaces, providing both the technical value needed for filtering and user-friendly labels with counts for better UX.
