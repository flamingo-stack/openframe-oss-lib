A data transfer object (DTO) that represents contact person information for organizational entities, using Java's record syntax with Lombok's builder pattern for immutable data structures.

## Key Components

- **ContactPersonDto**: Immutable record class containing contact person details
- **@Builder**: Lombok annotation providing builder pattern functionality
- **contactName**: Person's full name
- **title**: Job title or position
- **phone**: Contact phone number
- **email**: Email address

## Usage Example

```java
// Create using builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Operations Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Access fields directly (immutable)
String name = contact.contactName();
String email = contact.email();

// Use in API responses or data mapping
public class OrganizationDto {
    private List<ContactPersonDto> contacts;
    // ... other fields
}
```

This DTO is typically used in REST APIs for transferring contact person data between client and server, ensuring type safety and immutability while providing convenient builder-based construction.
