A data transfer object that encapsulates contact person information for organizational entities, using Java's record syntax with Lombok's builder pattern for immutable data representation.

## Key Components

- **ContactPersonDto**: Immutable record containing contact person details
- **@Builder**: Lombok annotation enabling fluent builder pattern construction
- **Fields**:
  - `contactName`: Person's full name
  - `title`: Job title or position
  - `phone`: Contact phone number
  - `email`: Email address

## Usage Example

```java
// Create using builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Project Manager")
    .phone("555-0123")
    .email("john.smith@company.com")
    .build();

// Access fields (records provide automatic getters)
String name = contact.contactName();
String email = contact.email();

// Use in API responses or data transfer scenarios
public class OrganizationDto {
    private ContactPersonDto primaryContact;
    private List<ContactPersonDto> additionalContacts;
}
```

The record provides automatic implementations of `equals()`, `hashCode()`, and `toString()` methods, making it ideal for API data transfer and value object scenarios.
