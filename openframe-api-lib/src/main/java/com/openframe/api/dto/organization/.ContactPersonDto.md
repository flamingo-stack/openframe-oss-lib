A data transfer object that represents contact person information for organizations, built using Java records and Lombok's builder pattern for immutable data structures.

## Key Components

- **ContactPersonDto**: Immutable record class with builder support
- **Fields**:
  - `contactName`: The person's name
  - `title`: Their job title or position
  - `phone`: Contact phone number
  - `email`: Contact email address

## Usage Example

```java
// Create a contact person using the builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Technical Director")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Access immutable fields
String name = contact.contactName();
String email = contact.email();

// Use in organization contexts
List<ContactPersonDto> contacts = Arrays.asList(
    ContactPersonDto.builder()
        .contactName("Jane Doe")
        .title("Project Manager")
        .phone("+1-555-0124")
        .email("jane.doe@company.com")
        .build()
);
```
