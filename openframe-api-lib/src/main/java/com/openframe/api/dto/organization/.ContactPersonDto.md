A data transfer object (DTO) that represents contact person information for organizations, using Java records for immutable data structure.

## Key Components

- **ContactPersonDto**: Immutable record class with Lombok `@Builder` annotation
- **contactName**: String field for the person's name
- **title**: String field for the person's job title or role
- **phone**: String field for phone number
- **email**: String field for email address

## Usage Example

```java
// Create using builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Operations Manager")
    .phone("+1-555-123-4567")
    .email("john.smith@company.com")
    .build();

// Access fields (records provide automatic getters)
String name = contact.contactName();
String email = contact.email();

// Use in organization contexts
OrganizationDto organization = OrganizationDto.builder()
    .name("ACME Corp")
    .primaryContact(contact)
    .build();
```

The record structure ensures immutability and provides automatic implementations for `equals()`, `hashCode()`, and `toString()` methods, while the `@Builder` annotation enables flexible object construction.
