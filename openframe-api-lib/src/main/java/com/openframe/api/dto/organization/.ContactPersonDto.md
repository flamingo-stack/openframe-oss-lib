A data transfer object (DTO) that represents contact person information within an organization context, using Java records and Lombok's builder pattern for immutable data representation.

## Key Components

- **ContactPersonDto**: Immutable record with builder pattern support
- **contactName**: Person's full name
- **title**: Job title or position
- **phone**: Contact phone number
- **email**: Email address

## Usage Example

```java
// Create using builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Project Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Access fields directly
String name = contact.contactName();
String email = contact.email();

// Use in API responses or data transfer
public ResponseEntity<ContactPersonDto> getContact(Long id) {
    return ResponseEntity.ok(contactService.getContactById(id));
}
```

The `@Builder` annotation generates a fluent builder API while the record provides automatic implementations of equals, hashCode, toString, and getter methods, making it ideal for REST API data transfer scenarios.
