A Data Transfer Object (DTO) representing contact person information within an organization context, designed for API communication and data exchange.

## Key Components

- **ContactPersonDto**: Immutable record class with Lombok's `@Builder` annotation
- **contactName**: String field for the person's full name
- **title**: String field for the person's job title or position
- **phone**: String field for contact phone number
- **email**: String field for contact email address

## Usage Example

```java
// Creating a contact person using the builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Project Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Accessing contact information
String name = contact.contactName();
String email = contact.email();

// Using in API responses or requests
public ContactPersonDto getContactPerson(Long organizationId) {
    // Retrieve from database and map to DTO
    return ContactPersonDto.builder()
        .contactName("Jane Doe")
        .title("Sales Director")
        .phone("+1-555-0456")
        .email("jane.doe@company.com")
        .build();
}
```
