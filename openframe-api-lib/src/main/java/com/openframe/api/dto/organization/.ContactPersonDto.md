A data transfer object (DTO) that represents contact person information within an organization, using Java record syntax with Lombok's Builder pattern for immutable data structures.

## Key Components

- **ContactPersonDto**: Immutable record class representing a contact person
- **@Builder**: Lombok annotation enabling the builder pattern for object creation
- **Fields**:
  - `contactName` - The person's full name
  - `title` - Their job title or position
  - `phone` - Contact phone number
  - `email` - Email address

## Usage Example

```java
// Using the builder pattern to create a contact person
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("Project Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Accessing immutable fields
String name = contact.contactName();
String email = contact.email();

// Creating with partial information
ContactPersonDto minimalContact = ContactPersonDto.builder()
    .contactName("Jane Doe")
    .email("jane.doe@company.com")
    .build();
```

The record provides automatic implementations of `equals()`, `hashCode()`, and `toString()` methods, making it ideal for data transfer scenarios where immutability and value semantics are desired.
