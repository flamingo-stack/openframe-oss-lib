A data transfer object (DTO) that wraps a list of Organization entities for API responses, providing a structured container for organization collection endpoints.

## Key Components

- **organizations**: `List<Organization>` - Collection of organization entities
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern
- **Builder pattern support**: Enables fluent object construction

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = Arrays.asList(
    Organization.builder().name("Acme Corp").build(),
    Organization.builder().name("Tech Solutions").build()
);

OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// Using in a REST controller
@GetMapping("/organizations")
public ResponseEntity<OrganizationList> getAllOrganizations() {
    List<Organization> organizations = organizationService.findAll();
    return ResponseEntity.ok(
        OrganizationList.builder()
            .organizations(organizations)
            .build()
    );
}
```

This DTO follows standard API response patterns by wrapping collections in a named container, making it easier to extend with metadata like pagination information or total counts in the future.
