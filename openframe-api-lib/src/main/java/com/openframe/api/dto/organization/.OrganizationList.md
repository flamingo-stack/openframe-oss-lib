A DTO (Data Transfer Object) class that encapsulates a collection of organizations for API responses, providing a structured wrapper around a list of Organization entities.

## Key Components

- **OrganizationList** - Main DTO class with builder pattern support
- **organizations** - List field containing Organization entities
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder methods

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = organizationService.getAllOrganizations();
OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// Using in a REST controller
@GetMapping("/organizations")
public OrganizationList getOrganizations() {
    return OrganizationList.builder()
        .organizations(organizationRepository.findAll())
        .build();
}

// Accessing the data
OrganizationList orgList = getOrganizations();
List<Organization> organizations = orgList.getOrganizations();
```
