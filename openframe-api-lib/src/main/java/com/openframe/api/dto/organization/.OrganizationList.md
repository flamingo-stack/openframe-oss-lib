A data transfer object (DTO) that wraps a list of Organization entities for API responses, providing a structured container for organization collection data.

## Key Components

- **OrganizationList** - Main DTO class with Lombok annotations for boilerplate code generation
- **organizations** - List field containing Organization document entities
- **@Data, @Builder, @NoArgsConstructor, @AllArgsConstructor** - Lombok annotations providing getters, setters, builders, and constructors

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = organizationService.getAllOrganizations();
OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// Using builder pattern
OrganizationList orgList = new OrganizationList();
orgList.setOrganizations(fetchedOrganizations);

// Accessing the list
List<Organization> organizations = response.getOrganizations();
```

This DTO is typically used in REST API controllers to return multiple organizations in a consistent JSON structure, ensuring proper encapsulation of list data in API responses.
