A data transfer object (DTO) that encapsulates a list of organizations for API responses. This class provides a structured wrapper around organization collections returned by REST endpoints.

## Key Components

- **organizations**: `List<Organization>` - The main field containing the collection of organization entities
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern support

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = organizationService.getAllOrganizations();
OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// Using in a REST controller
@GetMapping("/organizations")
public OrganizationList getOrganizations() {
    return OrganizationList.builder()
        .organizations(organizationRepository.findAll())
        .build();
}

// Accessing the organizations
OrganizationList orgList = apiClient.getOrganizations();
List<Organization> organizations = orgList.getOrganizations();
```
