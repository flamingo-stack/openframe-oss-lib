A Data Transfer Object (DTO) that wraps a list of organizations for API responses. This class provides a structured container for returning multiple organization entities from REST endpoints.

## Key Components

- **organizations**: List of `Organization` entities
- **@Data**: Lombok annotation providing getters, setters, equals, hashCode, and toString methods
- **@Builder**: Enables builder pattern for object construction
- **@NoArgsConstructor/@AllArgsConstructor**: Provides default and full parameter constructors

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = organizationService.getAllOrganizations();
OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// Using in a REST controller
@GetMapping("/organizations")
public ResponseEntity<OrganizationList> getAllOrganizations() {
    List<Organization> organizations = organizationService.findAll();
    OrganizationList response = new OrganizationList(organizations);
    return ResponseEntity.ok(response);
}

// Accessing the organizations
OrganizationList list = getOrganizations();
List<Organization> orgs = list.getOrganizations();
```
