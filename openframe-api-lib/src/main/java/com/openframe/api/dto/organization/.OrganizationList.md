A simple Data Transfer Object (DTO) that wraps a list of Organization entities for API responses when returning multiple organizations.

## Key Components

- **OrganizationList** - Main DTO class with Lombok annotations for boilerplate code generation
- **organizations** - List field containing Organization document entities
- **@Data, @Builder, @NoArgsConstructor, @AllArgsConstructor** - Lombok annotations providing getters, setters, builder pattern, and constructors

## Usage Example

```java
// Creating an organization list response
List<Organization> orgs = organizationService.findAll();
OrganizationList response = OrganizationList.builder()
    .organizations(orgs)
    .build();

// In a REST controller
@GetMapping("/organizations")
public OrganizationList getAllOrganizations() {
    List<Organization> organizations = organizationService.getAllOrganizations();
    return new OrganizationList(organizations);
}

// Using builder pattern
OrganizationList orgList = OrganizationList.builder()
    .organizations(Arrays.asList(org1, org2, org3))
    .build();
```
