A data transfer object that wraps a list of organizations for API responses, providing a standardized structure for returning multiple organization records.

## Key Components

- **OrganizationList**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **organizations**: List field containing Organization entities
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Creating an organization list response
List<Organization> orgList = Arrays.asList(org1, org2, org3);
OrganizationList response = OrganizationList.builder()
    .organizations(orgList)
    .build();

// Using in a REST controller
@GetMapping("/organizations")
public ResponseEntity<OrganizationList> getOrganizations() {
    List<Organization> organizations = organizationService.findAll();
    OrganizationList response = new OrganizationList(organizations);
    return ResponseEntity.ok(response);
}
```
