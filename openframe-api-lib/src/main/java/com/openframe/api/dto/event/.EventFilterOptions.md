Data transfer object for filtering events in the OpenFrame API, providing flexible criteria for event queries including user-based, type-based, and date range filtering.

## Key Components

- **userIds** - List of user identifiers to filter events by specific users
- **eventTypes** - List of event type strings to filter by event categories
- **startDate** - Beginning date boundary for date range filtering (inclusive)
- **endDate** - End date boundary for date range filtering (inclusive)

## Usage Example

```java
// Filter events for specific users and date range
EventFilterOptions filter = EventFilterOptions.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Using with service layer
List<Event> filteredEvents = eventService.getFilteredEvents(filter);

// Minimal filter - only date range
EventFilterOptions dateOnlyFilter = EventFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .build();
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation, making it ideal for API request parameters and service method arguments.
