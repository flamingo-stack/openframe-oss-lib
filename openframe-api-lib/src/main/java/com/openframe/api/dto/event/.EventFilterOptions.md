A data transfer object (DTO) that encapsulates filtering criteria for querying events in the OpenFrame API. This class provides a structured way to pass multiple filter parameters for event retrieval operations.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter by event categories
- **startDate**: Beginning date boundary for time-based filtering
- **endDate**: Ending date boundary for time-based filtering
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter options using builder pattern
EventFilterOptions filterOptions = EventFilterOptions.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "PURCHASE"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Use with event service
List<Event> filteredEvents = eventService.getEvents(filterOptions);

// Create filter for specific date range only
EventFilterOptions dateFilter = EventFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .build();
```
