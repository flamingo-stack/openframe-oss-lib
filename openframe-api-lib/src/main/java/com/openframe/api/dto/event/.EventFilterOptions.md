A data transfer object (DTO) that encapsulates filtering criteria for querying events, supporting filtering by users, event types, and date ranges.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter by event categories
- **startDate**: Beginning date for date range filtering (inclusive)
- **endDate**: Ending date for date range filtering (inclusive)

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern support.

## Usage Example

```java
// Create filter using builder pattern
EventFilterOptions filter = EventFilterOptions.builder()
    .userIds(Arrays.asList("user123", "user456"))
    .eventTypes(Arrays.asList("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Create filter with partial criteria
EventFilterOptions simpleFilter = EventFilterOptions.builder()
    .eventTypes(Arrays.asList("ERROR", "WARNING"))
    .startDate(LocalDate.now().minusDays(7))
    .build();

// Use in service method
List<Event> events = eventService.findEvents(filter);
```
