A data transfer object (DTO) that encapsulates filtering criteria for querying events. This class uses Lombok annotations to provide a clean, builder-pattern implementation for event filter parameters.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter by event categories
- **startDate**: Beginning date boundary for time-based filtering
- **endDate**: Ending date boundary for time-based filtering
- **Builder pattern**: Enabled through `@Builder` for flexible object construction

## Usage Example

```java
// Create filter for specific users and date range
EventFilterOptions filter = EventFilterOptions.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "UPDATE"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Create filter with only event types
EventFilterOptions typeFilter = EventFilterOptions.builder()
    .eventTypes(List.of("ERROR", "WARNING"))
    .build();

// Use in service method
List<Event> events = eventService.getFilteredEvents(filter);
```
