A data transfer object (DTO) that encapsulates filtering criteria for querying events in the OpenFrame API. This class provides a flexible way to specify multiple filter parameters when retrieving events.

## Key Components

- **userIds** - List of user IDs to filter events by specific users
- **eventTypes** - List of event type strings to filter by event categories
- **startDate** - LocalDate representing the earliest date for event filtering
- **endDate** - LocalDate representing the latest date for event filtering
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter options using builder pattern
EventFilterOptions filters = EventFilterOptions.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Use with an event service
List<Event> filteredEvents = eventService.findEvents(filters);

// Create filter for specific date range only
EventFilterOptions dateFilter = EventFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .build();
```
