A data transfer object that encapsulates filtering criteria for querying events, supporting filtering by users, event types, and date ranges.

## Key Components

- **userIds** - List of user identifiers to filter events by specific users
- **eventTypes** - List of event type strings to filter by event categories
- **startDate** - LocalDate representing the beginning of the date range filter
- **endDate** - LocalDate representing the end of the date range filter

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern support.

## Usage Example

```java
// Create filter options using builder pattern
EventFilterOptions filterOptions = EventFilterOptions.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .build();

// Create with partial filters
EventFilterOptions dateOnlyFilter = EventFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .build();

// Use in event service
List<Event> filteredEvents = eventService.getEvents(filterOptions);
```
