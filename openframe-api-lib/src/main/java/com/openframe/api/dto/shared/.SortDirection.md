A simple enumeration that defines sorting direction options for API queries, designed to match GraphQL schema definitions.

## Key Components

- **ASC** - Ascending sort order (A-Z, 1-10, oldest to newest)
- **DESC** - Descending sort order (Z-A, 10-1, newest to oldest)

## Usage Example

```java
// Using in a query builder or service method
public List<User> getUsers(SortDirection direction) {
    if (direction == SortDirection.ASC) {
        return userRepository.findAllByOrderByNameAsc();
    } else {
        return userRepository.findAllByOrderByNameDesc();
    }
}

// Common usage in DTOs
public class UserQuery {
    private String sortBy;
    private SortDirection sortDirection = SortDirection.ASC; // default
    
    // getters and setters
}

// Switch statement usage
switch (sortDirection) {
    case ASC -> query.orderBy(field + " ASC");
    case DESC -> query.orderBy(field + " DESC");
}
```

This enum provides a type-safe way to specify sort direction in API endpoints and database queries, ensuring consistency between the Java backend and GraphQL schema.
