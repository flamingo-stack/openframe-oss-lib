A data transfer object (DTO) that encapsulates filtering criteria for querying events, specifically allowing filtering by user IDs and event types.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter events by category or type
- Uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filters using builder pattern
EventFilters filters = EventFilters.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "PURCHASE"))
    .build();

// Create filters with constructor
EventFilters filters2 = new EventFilters(
    List.of("admin001"), 
    List.of("SYSTEM_ERROR")
);

// Use in service method
public List<Event> getFilteredEvents(EventFilters filters) {
    return eventRepository.findByFilters(
        filters.getUserIds(), 
        filters.getEventTypes()
    );
}
```
