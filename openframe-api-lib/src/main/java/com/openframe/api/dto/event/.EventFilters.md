A Data Transfer Object (DTO) that encapsulates filtering criteria for querying events, providing optional filters for user IDs and event types.

## Key Components

- **userIds** - List of user identifiers to filter events by specific users
- **eventTypes** - List of event type strings to filter events by category or action type
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filters using builder pattern
EventFilters filters = EventFilters.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_UPDATE"))
    .build();

// Create empty filters for unfiltered queries
EventFilters allEvents = EventFilters.builder().build();

// Create filters for specific user only
EventFilters userSpecific = EventFilters.builder()
    .userIds(List.of("admin001"))
    .build();

// Access filter values
List<String> users = filters.getUserIds();
List<String> types = filters.getEventTypes();
```

This class is typically used as a parameter object in service methods or API endpoints that need to query events with optional filtering criteria, allowing for flexible and composable event retrieval operations.
