A data transfer object (DTO) that encapsulates filtering criteria for querying events. This class provides a structured way to specify which users and event types to include when filtering event data.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter events by category or type
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filters using builder pattern
EventFilters filters = EventFilters.builder()
    .userIds(Arrays.asList("user123", "user456"))
    .eventTypes(Arrays.asList("LOGIN", "LOGOUT", "PURCHASE"))
    .build();

// Create empty filters for all events
EventFilters allEvents = new EventFilters();

// Access filter criteria
List<String> users = filters.getUserIds();
List<String> types = filters.getEventTypes();

// Use in service method
List<Event> filteredEvents = eventService.getEvents(filters);
```
