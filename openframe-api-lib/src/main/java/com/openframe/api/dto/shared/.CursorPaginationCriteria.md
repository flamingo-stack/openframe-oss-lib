A data transfer object that encapsulates cursor-based pagination parameters with built-in validation and normalization capabilities.

## Key Components

- **`limit`** - Maximum number of items to return per page
- **`cursor`** - Opaque token representing the position in the result set
- **`normalize()`** - Validates and constrains the limit within acceptable bounds
- **`hasCursor()`** - Checks if a valid cursor is present
- **Constants** - `DEFAULT_PAGE_SIZE` (20), `MIN_PAGE_SIZE` (1), `MAX_PAGE_SIZE` (100)

## Usage Example

```java
// Create pagination criteria
CursorPaginationCriteria criteria = CursorPaginationCriteria.builder()
    .limit(50)
    .cursor("eyJpZCI6MTAwfQ==")
    .build();

// Normalize to enforce limits
CursorPaginationCriteria normalized = criteria.normalize();
// limit will be clamped between 1-100

// Check for cursor presence
if (criteria.hasCursor()) {
    // Continue from cursor position
    fetchDataFromCursor(criteria.getCursor(), criteria.getLimit());
} else {
    // Start from beginning
    fetchInitialData(criteria.getLimit());
}

// Handle invalid limits
CursorPaginationCriteria oversized = new CursorPaginationCriteria(500, null);
CursorPaginationCriteria safe = oversized.normalize(); // limit becomes 100
```
