A data transfer object that encapsulates cursor-based pagination parameters with built-in validation and normalization capabilities. This class provides a type-safe way to handle pagination requests with configurable limits and cursor-based navigation.

## Key Components

- **Fields**: `limit` (page size) and `cursor` (pagination token)
- **Constants**: Default, minimum, and maximum page size constraints
- **normalize()**: Validates and clamps the limit within acceptable bounds
- **hasCursor()**: Utility method to check if a valid cursor is present
- **Lombok annotations**: Auto-generates constructors, getters, setters, and builder pattern

## Usage Example

```java
// Create pagination criteria with builder
CursorPaginationCriteria criteria = CursorPaginationCriteria.builder()
    .limit(50)
    .cursor("eyJpZCI6MTIzfQ==")
    .build();

// Normalize to ensure valid limits
CursorPaginationCriteria normalized = criteria.normalize();

// Check if cursor-based pagination is requested
if (normalized.hasCursor()) {
    // Continue from cursor position
    return fetchResultsAfterCursor(normalized.getCursor(), normalized.getLimit());
} else {
    // First page request
    return fetchFirstPage(normalized.getLimit());
}

// Handle edge cases - limit too high gets capped at 100
CursorPaginationCriteria oversized = CursorPaginationCriteria.builder()
    .limit(500)  // Will be normalized to 100
    .build();
```
