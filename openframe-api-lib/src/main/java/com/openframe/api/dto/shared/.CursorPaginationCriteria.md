A data transfer object for cursor-based pagination parameters with built-in validation and normalization capabilities.

## Key Components

- **Fields**: `limit` (page size) and `cursor` (pagination token)
- **Constants**: Default (20), minimum (1), and maximum (100) page size limits
- **normalize()**: Validates and clamps the limit within acceptable bounds
- **hasCursor()**: Checks if a valid cursor token is present

## Usage Example

```java
// Create pagination criteria
CursorPaginationCriteria criteria = CursorPaginationCriteria.builder()
    .limit(50)
    .cursor("eyJpZCI6MTIzfQ")
    .build();

// Normalize to enforce limits
CursorPaginationCriteria normalized = criteria.normalize();

// Check for cursor presence
if (normalized.hasCursor()) {
    // Continue from cursor position
    return service.findItemsAfterCursor(normalized.getCursor(), normalized.getLimit());
} else {
    // First page request
    return service.findFirstPage(normalized.getLimit());
}

// Handle edge cases
CursorPaginationCriteria excessive = new CursorPaginationCriteria(500, null);
CursorPaginationCriteria clamped = excessive.normalize(); // limit becomes 100
```
