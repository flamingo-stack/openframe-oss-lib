A shared DTO class that encapsulates organization data for API responses, used across both GraphQL and REST endpoints to ensure consistent data structure.

## Key Components

- **OrganizationResponse**: Main DTO class with Lombok annotations for automatic getter/setter generation and builder pattern support
- **Core Fields**: Basic organization data (id, name, organizationId, category, numberOfEmployees)
- **Contact & Business Info**: Website URL, contact information, notes, and revenue data
- **Contract Management**: Start/end dates for contracts and financial information
- **Audit Fields**: Creation/update timestamps, deletion status and soft delete timestamp
- **Configuration**: Default organization flag for system settings

## Usage Example

```java
// Building an organization response
OrganizationResponse response = OrganizationResponse.builder()
    .id("org-123")
    .name("Acme Corporation")
    .organizationId("ACME-001")
    .category("Enterprise")
    .numberOfEmployees(500)
    .websiteUrl("https://acme.com")
    .monthlyRevenue(new BigDecimal("150000.00"))
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .isDefault(false)
    .deleted(false)
    .createdAt(Instant.now())
    .build();

// Accessing data
String orgName = response.getName();
BigDecimal revenue = response.getMonthlyRevenue();
boolean isActive = !response.getDeleted();
```
