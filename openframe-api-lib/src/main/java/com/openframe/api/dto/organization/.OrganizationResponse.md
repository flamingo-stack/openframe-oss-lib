A data transfer object (DTO) that represents organization information in API responses, shared between GraphQL and REST endpoints.

## Key Components

- **OrganizationResponse**: Main DTO class with comprehensive organization fields
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation
- **Core fields**: `id`, `name`, `organizationId`, `category`, `numberOfEmployees`
- **Contact & business data**: `contactInformation`, `websiteUrl`, `monthlyRevenue`
- **Contract lifecycle**: `contractStartDate`, `contractEndDate`
- **Audit fields**: `createdAt`, `updatedAt`, `deleted`, `deletedAt`
- **Status flags**: `isDefault`, `deleted`

## Usage Example

```java
// Building a response object
OrganizationResponse response = OrganizationResponse.builder()
    .id("org-123")
    .name("Acme Corp")
    .organizationId("ACME001")
    .category("Technology")
    .numberOfEmployees(250)
    .websiteUrl("https://acme.com")
    .monthlyRevenue(new BigDecimal("150000.00"))
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .createdAt(Instant.now())
    .isDefault(false)
    .deleted(false)
    .build();

// Using in API response
return ResponseEntity.ok(response);
```
