Shared DTO representing a complete organization record returned by both GraphQL and REST API responses across the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique organization identifier |
| `organizationId` | `String` | External/business organization ID |
| `name` | `String` | Organization display name |
| `category` | `String` | Organization classification |
| `numberOfEmployees` | `Integer` | Headcount |
| `websiteUrl` | `String` | Organization website |
| `notes` | `String` | Free-form notes |
| `contactInformation` | `ContactInformationDto` | Nested contact details |
| `monthlyRevenue` | `BigDecimal` | Monthly revenue figure |
| `contractStartDate` / `contractEndDate` | `LocalDate` | Contract term boundaries |
| `createdAt` / `updatedAt` | `Instant` | Audit timestamps |
| `isDefault` | `Boolean` | Flags the default organization |
| `status` | `String` | Current organization status |
| `statusChangedAt` | `Instant` | Timestamp of last status change |

## Usage Example

```java
// Building a response from a domain entity
OrganizationResponse response = OrganizationResponse.builder()
    .id(org.getId())
    .name(org.getName())
    .organizationId(org.getOrganizationId())
    .category(org.getCategory())
    .numberOfEmployees(org.getNumberOfEmployees())
    .websiteUrl(org.getWebsiteUrl())
    .contactInformation(contactMapper.toDto(org.getContact()))
    .monthlyRevenue(org.getMonthlyRevenue())
    .contractStartDate(org.getContractStartDate())
    .contractEndDate(org.getContractEndDate())
    .status(org.getStatus())
    .statusChangedAt(org.getStatusChangedAt())
    .isDefault(org.getIsDefault())
    .createdAt(org.getCreatedAt())
    .updatedAt(org.getUpdatedAt())
    .build();
```

> This DTO is intentionally shared between `api-service-core` (GraphQL) and `external-api` (REST) to ensure a consistent organization schema across all consumers.
