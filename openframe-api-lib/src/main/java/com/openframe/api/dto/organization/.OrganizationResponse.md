A shared Data Transfer Object (DTO) that represents organization data in API responses, used by both GraphQL and REST services to ensure consistent organization representation across different API endpoints.

## Key Components

- **OrganizationResponse**: Main DTO class with comprehensive organization fields
- **Core Fields**: `id`, `name`, `organizationId`, `category`, `numberOfEmployees`
- **Contact & Web**: `websiteUrl`, `contactInformation` (nested DTO)
- **Financial Data**: `monthlyRevenue`, contract dates
- **Metadata**: Creation/update timestamps, deletion tracking, default flag
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`

## Usage Example

```java
// Building an organization response
OrganizationResponse response = OrganizationResponse.builder()
    .id("org-123")
    .name("Acme Corporation")
    .organizationId("ACME-001")
    .category("Technology")
    .numberOfEmployees(500)
    .websiteUrl("https://acme.com")
    .monthlyRevenue(new BigDecimal("100000.00"))
    .contractStartDate(LocalDate.now())
    .isDefault(false)
    .deleted(false)
    .createdAt(Instant.now())
    .build();

// Accessing organization data
String orgName = response.getName();
BigDecimal revenue = response.getMonthlyRevenue();
boolean isDeleted = response.getDeleted();
```
