A data transfer object (DTO) representing an organization's complete information for API responses, shared between GraphQL and REST endpoints.

## Key Components

- **Core Fields**: `id`, `name`, `organizationId` for basic organization identification
- **Business Details**: `category`, `numberOfEmployees`, `websiteUrl`, `monthlyRevenue` for organizational metadata
- **Contract Information**: `contractStartDate` and `contractEndDate` for agreement tracking
- **Contact Data**: `contactInformation` field using `ContactInformationDto` for structured contact details
- **Audit Fields**: `createdAt`, `updatedAt`, `deleted`, `deletedAt` for entity lifecycle tracking
- **Configuration**: `isDefault` flag for default organization designation
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Building a complete organization response
OrganizationResponse organization = OrganizationResponse.builder()
    .id("org-123")
    .name("Acme Corporation")
    .organizationId("ACME-001")
    .category("Technology")
    .numberOfEmployees(250)
    .monthlyRevenue(new BigDecimal("150000.00"))
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .isDefault(true)
    .deleted(false)
    .createdAt(Instant.now())
    .build();

// Using in REST controller or GraphQL resolver
return ResponseEntity.ok(organization);
```
