A shared DTO (Data Transfer Object) representing organization data in API responses, designed for use across both GraphQL and REST endpoints.

## Key Components

- **OrganizationResponse**: Main response class with comprehensive organization fields including basic info, contact details, financial data, and audit timestamps
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern
- **ContactInformationDto**: Nested DTO for organization contact details
- **Audit fields**: Creation/update timestamps and soft deletion support

## Usage Example

```java
// Creating an organization response using builder pattern
OrganizationResponse response = OrganizationResponse.builder()
    .id("org-123")
    .name("Tech Solutions Inc")
    .organizationId("TS001")
    .category("Technology")
    .numberOfEmployees(150)
    .websiteUrl("https://techsolutions.com")
    .monthlyRevenue(new BigDecimal("50000.00"))
    .contractStartDate(LocalDate.of(2023, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .createdAt(Instant.now())
    .isDefault(false)
    .deleted(false)
    .contactInformation(contactDto)
    .build();

// Using in REST controller
@GetMapping("/organizations/{id}")
public OrganizationResponse getOrganization(@PathVariable String id) {
    return organizationService.getOrganizationResponse(id);
}
```
