A simple DTO (Data Transfer Object) that encapsulates the result of a knowledge base attachment upload operation, pairing the attachment metadata with its pre-signed or generated upload URL.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `attachment` | `KnowledgeBaseItemAttachment` | The attachment document/metadata object |
| `uploadUrl` | `String` | The URL to which the file should be uploaded |

## Usage Example

```java
KnowledgeBaseAttachmentUpload uploadResult = KnowledgeBaseAttachmentUpload.builder()
    .attachment(attachmentDocument)
    .uploadUrl("https://storage.example.com/upload?token=abc123")
    .build();

String url = uploadResult.getUploadUrl();
KnowledgeBaseItemAttachment meta = uploadResult.getAttachment();
```

## Notes

- Uses Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) to auto-generate boilerplate — getters, setters, constructors, and the builder pattern.
- Typically returned from a service or controller layer after initiating an upload, allowing the client to receive both the attachment record and the destination URL in a single response.
