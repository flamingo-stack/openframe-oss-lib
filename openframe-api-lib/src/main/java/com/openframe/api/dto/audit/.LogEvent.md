A data transfer object (DTO) representing audit log events in the OpenFrame API, containing essential metadata and identifiers for tracking system events across organizations and devices.

## Key Components

- **LogEvent class**: Main DTO with Lombok annotations for automatic getter/setter generation
- **Event identification fields**: `toolEventId`, `eventType`, `toolType` for categorizing events
- **Organization context**: `organizationId`, `organizationName` for multi-tenant support
- **User/device tracking**: `userId`, `deviceId`, `hostname` for audit trail
- **Event metadata**: `severity`, `summary`, `timestamp`, `ingestDay` for logging details

## Usage Example

```java
// Creating a new log event
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("USER_LOGIN")
    .toolType("AUTHENTICATION")
    .severity("INFO")
    .userId("user123")
    .deviceId("device456")
    .hostname("app-server-01")
    .organizationId("org789")
    .organizationName("Acme Corp")
    .summary("User successfully logged in")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Accessing event data
String eventSummary = logEvent.getSummary();
Instant eventTime = logEvent.getTimestamp();
```
