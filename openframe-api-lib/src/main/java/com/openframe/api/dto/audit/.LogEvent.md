A data transfer object (DTO) representing a structured audit log event, used to transfer log data across application layers within the OpenFrame audit subsystem.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the log event |
| `toolEventId` | `String` | Reference ID from the originating tool |
| `eventType` | `String` | Classification of the event |
| `ingestDay` | `String` | Date partition key for ingestion |
| `toolType` | `String` | Type of MSP tool that generated the event |
| `severity` | `String` | Event severity level |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Source hostname |
| `organizationId` | `String` | Tenant organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `summary` | `String` | Brief description of the event |
| `timestamp` | `Instant` | Precise UTC time of the event |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Builder pattern
LogEvent event = LogEvent.builder()
    .id("evt-001")
    .toolEventId("rmm-789")
    .eventType("DEVICE_OFFLINE")
    .toolType("RMM")
    .severity("HIGH")
    .userId("usr-123")
    .deviceId("dev-456")
    .hostname("workstation-01")
    .organizationId("org-999")
    .organizationName("Acme Corp")
    .summary("Device went offline unexpectedly")
    .timestamp(Instant.now())
    .build();

// No-args + setters
LogEvent event = new LogEvent();
event.setSeverity("LOW");
event.setTimestamp(Instant.now());
```
