A data transfer object (DTO) representing audit log events within the OpenFrame API system. This class encapsulates all essential information for tracking and auditing system events across different tools and organizations.

## Key Components

- **LogEvent** - Main DTO class with comprehensive audit fields
- **Builder Pattern** - Enabled through Lombok `@Builder` annotation for flexible object construction
- **Data Annotations** - Lombok annotations providing automatic getters, setters, equals, hashCode, and toString methods

### Fields
- `toolEventId` - Unique identifier for the event
- `eventType` - Classification of the event
- `severity` - Event severity level
- `userId/deviceId/hostname` - Source identification
- `organizationId/organizationName` - Organizational context
- `timestamp` - Event occurrence time using Java 8 Instant

## Usage Example

```java
// Creating a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user123")
    .organizationId("org-456")
    .organizationName("Acme Corp")
    .summary("User login successful")
    .timestamp(Instant.now())
    .build();

// Accessing properties
String eventId = logEvent.getToolEventId();
Instant when = logEvent.getTimestamp();
```
