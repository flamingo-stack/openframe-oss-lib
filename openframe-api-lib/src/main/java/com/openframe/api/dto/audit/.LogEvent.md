A data transfer object (DTO) representing audit log events in the OpenFrame API system. This class captures comprehensive logging information including user activity, device details, and event metadata for audit trail purposes.

## Key Components

- **Core Event Fields**: `toolEventId`, `eventType`, `toolType`, `severity` for event classification
- **User/Device Context**: `userId`, `deviceId`, `hostname` for tracking event sources  
- **Organization Data**: `organizationId`, `organizationName` for multi-tenant support
- **Temporal/Content**: `timestamp`, `ingestDay`, `summary` for event timing and description
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Creating a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt_12345")
    .eventType("LOGIN_SUCCESS")
    .toolType("WEB_APP")
    .severity("INFO")
    .userId("user123")
    .deviceId("device456")
    .organizationId("org789")
    .organizationName("Acme Corp")
    .summary("User successfully logged in")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Accessing fields (via Lombok @Data)
String eventId = logEvent.getToolEventId();
Instant eventTime = logEvent.getTimestamp();
```
