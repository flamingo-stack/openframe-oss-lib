A data transfer object (DTO) representing a structured log event for audit and monitoring purposes within the OpenFrame platform, capturing essential metadata about system events across different tools and organizational contexts.

## Key Components

- **Event Identification**: `id`, `toolEventId`, and `eventType` for unique identification and categorization
- **Temporal Data**: `timestamp` (Instant) for precise timing and `ingestDay` for partitioning
- **Tool Integration**: `toolType` to identify the source system or tool generating the event
- **Organizational Context**: `organizationId` and `organizationName` for multi-tenant support
- **Device/User Tracking**: `userId`, `deviceId`, and `hostname` for traceability
- **Event Classification**: `severity` level and `summary` description for quick assessment

## Usage Example

```java
LogEvent logEvent = LogEvent.builder()
    .id("evt_123456")
    .toolEventId("rmm_alert_789")
    .eventType("SECURITY_ALERT")
    .ingestDay("2024-01-15")
    .toolType("RMM")
    .severity("HIGH")
    .userId("user_456")
    .deviceId("device_789")
    .hostname("workstation-01")
    .organizationId("org_123")
    .organizationName("Acme Corp")
    .summary("Suspicious network activity detected")
    .timestamp(Instant.now())
    .build();

// Convert to JSON for API responses or storage
// The Lombok annotations provide automatic getters, setters, and builders
```

This DTO serves as the primary data structure for log aggregation, audit trails, and event monitoring across the Flamingo MSP platform's integrated tools.
