A data transfer object (DTO) representing an audit log event within the OpenFrame API system. This class encapsulates all essential information about security, system, or user events for auditing and monitoring purposes.

## Key Components

- **LogEvent** - Main DTO class with comprehensive audit event fields
- **toolEventId** - Unique identifier for the specific event
- **eventType** - Classification of the event (e.g., login, access, error)
- **severity** - Event severity level for filtering and alerting
- **userId/deviceId/hostname** - Identity and location context
- **organizationId/organizationName** - Multi-tenant organization context
- **summary** - Human-readable event description
- **timestamp** - Event occurrence time using `Instant`

## Usage Example

```java
// Creating a security audit event
LogEvent loginEvent = LogEvent.builder()
    .toolEventId("evt_12345")
    .eventType("USER_LOGIN")
    .severity("INFO")
    .userId("user123")
    .deviceId("device456")
    .hostname("app-server-01")
    .organizationId("org_789")
    .organizationName("Acme Corp")
    .summary("User successfully logged in")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .toolType("AUTH_SERVICE")
    .build();

// Accessing event data
String eventSummary = loginEvent.getSummary();
Instant eventTime = loginEvent.getTimestamp();
```
