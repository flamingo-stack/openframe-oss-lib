A data transfer object representing audit log events with comprehensive metadata for tracking and monitoring system activities.

## Key Components

**LogEvent Class**
- `@Data` - Generates getters, setters, toString, equals, and hashCode methods
- `@Builder` - Enables builder pattern for object creation
- `@NoArgsConstructor/@AllArgsConstructor` - Provides default and full constructors

**Core Fields**
- `toolEventId` - Unique identifier for the event
- `eventType` - Classification of the event
- `severity` - Event severity level
- `userId/deviceId/hostname` - Source identification
- `organizationId/organizationName` - Organization context
- `summary` - Event description
- `timestamp` - Event occurrence time

## Usage Example

```java
// Creating a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user123")
    .deviceId("device-abc")
    .hostname("web-server-01")
    .organizationId("org-456")
    .organizationName("Acme Corp")
    .summary("User login successful")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .toolType("WEB_APP")
    .build();

// Accessing properties
String summary = logEvent.getSummary();
Instant eventTime = logEvent.getTimestamp();
```
