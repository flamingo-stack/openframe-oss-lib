A data transfer object (DTO) representing an audit log event in the OpenFrame API, containing metadata about security, user activity, and system events.

## Key Components

**Class Structure**:
- `LogEvent` - Main DTO class with Lombok annotations for automatic getter/setter/constructor generation
- **Core Fields**: Event identification (`toolEventId`, `eventType`), temporal data (`timestamp`, `ingestDay`)
- **Context Fields**: User context (`userId`, `organizationId`, `organizationName`), device context (`deviceId`, `hostname`)
- **Classification Fields**: Event categorization (`toolType`, `severity`), event description (`summary`)

**Annotations**:
- `@Data` - Generates getters, setters, equals, hashCode, and toString methods
- `@Builder` - Enables fluent builder pattern for object creation
- `@NoArgsConstructor`/`@AllArgsConstructor` - Generates default and full parameter constructors

## Usage Example

```java
// Creating a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user-789")
    .deviceId("device-abc")
    .hostname("app-server-01")
    .organizationId("org-456")
    .organizationName("Acme Corp")
    .toolType("AUTHENTICATION")
    .summary("User successful login")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Accessing properties
String eventType = logEvent.getEventType();
Instant when = logEvent.getTimestamp();
```
