A Data Transfer Object (DTO) representing a structured audit log event within the OpenFrame platform, used for capturing and transferring event data across the system.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the log event |
| `toolEventId` | `String` | Reference ID from the originating tool |
| `eventType` | `String` | Classification of the event |
| `ingestDay` | `String` | Date partition for ingestion tracking |
| `toolType` | `String` | Source tool that generated the event |
| `severity` | `String` | Event severity level |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Originating host name |
| `organizationId` | `String` | Tenant/organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `summary` | `String` | Brief description of the event |
| `timestamp` | `Instant` | Precise UTC time the event occurred |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Build a log event using the Lombok builder
LogEvent logEvent = LogEvent.builder()
    .id("evt-001")
    .toolEventId("rmm-9876")
    .eventType("ALERT")
    .toolType("RMM")
    .severity("HIGH")
    .userId("user-123")
    .deviceId("device-456")
    .hostname("workstation-01")
    .organizationId("org-789")
    .organizationName("Acme Corp")
    .summary("Disk usage exceeded 90% threshold")
    .ingestDay("2024-01-15")
    .timestamp(Instant.now())
    .build();
```
