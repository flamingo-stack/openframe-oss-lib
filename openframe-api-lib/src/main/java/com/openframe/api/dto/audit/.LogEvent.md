A data transfer object (DTO) representing an audit log event within the OpenFrame API system, designed to capture and structure security and operational events.

## Key Components

- **Event Identification**: `toolEventId`, `eventType`, `toolType` for categorizing and tracking events
- **Temporal Data**: `timestamp` (Instant) for precise timing, `ingestDay` for daily aggregation
- **User Context**: `userId`, `deviceId`, `hostname` for identifying event sources
- **Organizational Context**: `organizationId`, `organizationName` for multi-tenant support
- **Event Details**: `severity` level classification, `summary` for human-readable descriptions
- **Lombok Annotations**: `@Data`, `@Builder`, constructors for automatic boilerplate generation

## Usage Example

```java
// Create a security event log entry
LogEvent securityEvent = LogEvent.builder()
    .toolEventId("evt_12345")
    .eventType("AUTHENTICATION")
    .toolType("SSO_PROVIDER")
    .severity("HIGH")
    .userId("user_789")
    .deviceId("device_abc")
    .hostname("auth.company.com")
    .organizationId("org_456")
    .organizationName("Acme Corp")
    .summary("Failed login attempt detected")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Use in audit logging system
auditService.logEvent(securityEvent);
```

This DTO provides a standardized structure for audit trail data that can be easily serialized for storage or transmission while maintaining type safety and immutability through the builder pattern.
