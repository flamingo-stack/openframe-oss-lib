A Data Transfer Object (DTO) representing audit log events in the OpenFrame MSP platform, capturing security, operational, and user activity data across integrated tools.

## Key Components

- **Event Identification**: `id`, `toolEventId`, `eventType` for unique tracking and categorization
- **Tool Integration**: `toolType` to identify which MSP tool generated the event
- **Organizational Context**: `organizationId`, `organizationName` for multi-tenant isolation
- **User & Device Tracking**: `userId`, `deviceId`, `hostname` for comprehensive audit trails
- **Event Metadata**: `severity`, `summary`, `timestamp` for filtering and analysis
- **Time-based Partitioning**: `ingestDay` for efficient data organization and querying

## Usage Example

```java
// Creating a security audit log event
LogEvent securityEvent = LogEvent.builder()
    .id("log-001")
    .toolEventId("auth-failed-123")
    .eventType("AUTHENTICATION_FAILURE")
    .toolType("IDENTITY_MANAGEMENT")
    .severity("HIGH")
    .userId("user-456")
    .deviceId("device-789")
    .hostname("workstation-01")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .summary("Failed login attempt from unknown location")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Processing operational events from monitoring tools
LogEvent monitoringEvent = LogEvent.builder()
    .eventType("DISK_SPACE_WARNING")
    .toolType("MONITORING")
    .severity("MEDIUM")
    .deviceId("server-001")
    .organizationId("org-123")
    .summary("Disk usage exceeded 80% threshold")
    .timestamp(Instant.now())
    .build();
```
