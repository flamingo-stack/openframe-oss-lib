A data transfer object (DTO) that represents an audit log event within the OpenFrame API system. This class encapsulates all the essential information needed to track and audit system events across different tools and organizations.

## Key Components

- **LogEvent**: Main DTO class with Lombok annotations for automatic getter/setter generation
- **Core Fields**:
  - `toolEventId` - Unique identifier for the event
  - `eventType` - Classification of the event
  - `severity` - Event severity level
  - `timestamp` - When the event occurred
- **Context Fields**:
  - `userId`, `deviceId`, `hostname` - Event source information  
  - `organizationId`, `organizationName` - Organizational context
  - `toolType` - Type of tool that generated the event
  - `summary` - Human-readable event description
  - `ingestDay` - Date partitioning field for data processing

## Usage Example

```java
// Creating a log event using the builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("USER_LOGIN")
    .severity("INFO")
    .userId("user123")
    .organizationId("org-456")
    .organizationName("Acme Corp")
    .toolType("AUTHENTICATION")
    .summary("User successfully logged in")
    .timestamp(Instant.now())
    .hostname("api-server-01")
    .ingestDay("2024-01-15")
    .build();

// Using in API responses or data processing pipelines
auditService.processLogEvent(logEvent);
```
