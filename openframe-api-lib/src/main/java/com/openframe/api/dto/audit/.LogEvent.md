A data transfer object (DTO) representing a structured log event in the audit system, containing essential metadata and context information for tracking and analyzing system activities.

## Key Components

**Primary Fields:**
- `toolEventId` - Unique identifier for the specific event
- `eventType` - Classification of the event (error, warning, info, etc.)
- `severity` - Event severity level for prioritization
- `timestamp` - When the event occurred (using `Instant` for precision)

**Context Fields:**
- `userId` - User associated with the event
- `deviceId` - Device identifier where event originated
- `hostname` - Server/host where event was generated
- `organizationId/organizationName` - Organizational context

**Metadata Fields:**
- `toolType` - Type of tool/service that generated the event
- `ingestDay` - Date partition for efficient querying
- `summary` - Human-readable event description

**Lombok Annotations:**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Enables fluent builder pattern for object creation
- `@NoArgsConstructor/@AllArgsConstructor` - Provides constructors

## Usage Example

```java
// Creating a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("authentication")
    .severity("HIGH")
    .userId("user-789")
    .deviceId("device-456")
    .hostname("api-server-01")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .toolType("auth-service")
    .summary("Failed login attempt")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .build();

// Accessing event data
String eventInfo = String.format("Event %s: %s at %s", 
    logEvent.getToolEventId(), 
    logEvent.getSummary(), 
    logEvent.getTimestamp());
```
