A data transfer object (DTO) that represents a structured log event for audit and monitoring purposes in the OpenFrame API system.

## Key Components

**Main Class:**
- `LogEvent` - Immutable data container with comprehensive logging metadata

**Core Fields:**
- `toolEventId` - Unique identifier for the log event
- `eventType` - Classification of the event (error, info, warning, etc.)
- `severity` - Event severity level for filtering and alerting
- `userId/deviceId/hostname` - Source identification fields
- `organizationId/organizationName` - Tenant/organization context
- `summary` - Human-readable event description
- `timestamp` - Precise event occurrence time
- `ingestDay` - Date partitioning field for efficient storage/querying
- `toolType` - Source system or tool that generated the event

**Lombok Annotations:**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Enables fluent object construction
- `@NoArgsConstructor/@AllArgsConstructor` - Default and full constructors

## Usage Example

```java
// Create a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt-12345")
    .eventType("USER_LOGIN")
    .severity("INFO")
    .userId("user-789")
    .deviceId("device-456")
    .hostname("api-server-01")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .summary("User successfully logged in")
    .timestamp(Instant.now())
    .ingestDay("2024-01-15")
    .toolType("authentication-service")
    .build();

// Use in audit logging
auditService.recordEvent(logEvent);
```
