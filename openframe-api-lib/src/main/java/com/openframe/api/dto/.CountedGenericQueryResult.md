A data transfer object that extends GenericQueryResult to include filtered count information for paginated query results with generic type support.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class that extends GenericQueryResult
- **filteredCount** - Integer field representing the number of items after filtering
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, equals, hashCode, and builder pattern

## Usage Example

```java
// Create a counted result for a list of users
CountedGenericQueryResult<User> result = CountedGenericQueryResult.<User>builder()
    .data(userList)
    .totalCount(1000)
    .filteredCount(250)
    .build();

// Access the filtered count
int filteredItems = result.getFilteredCount();

// Use in API responses for paginated data with search/filter applied
@GetMapping("/users")
public CountedGenericQueryResult<User> getUsers(
    @RequestParam String search,
    @RequestParam int page) {
    
    List<User> users = userService.findBySearch(search, page);
    int totalFiltered = userService.countBySearch(search);
    
    return CountedGenericQueryResult.<User>builder()
        .data(users)
        .filteredCount(totalFiltered)
        .build();
}
```
