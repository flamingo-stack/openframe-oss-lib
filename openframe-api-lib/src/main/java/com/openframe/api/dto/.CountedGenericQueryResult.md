A data transfer object that extends GenericQueryResult to include filtered count information for paginated query results with type safety.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class extending GenericQueryResult
- **filteredCount** - Integer field representing the number of filtered results
- **Lombok annotations** - Provides automatic generation of constructors, getters, setters, equals, hashCode, and builder pattern

## Usage Example

```java
// Create a counted result for user queries
CountedGenericQueryResult<User> userResult = CountedGenericQueryResult.<User>builder()
    .data(userList)
    .totalCount(1000)
    .filteredCount(25)
    .build();

// Access the filtered count
int filtered = userResult.getFilteredCount();

// Use with pagination info
CountedGenericQueryResult<Product> productResult = new CountedGenericQueryResult<>();
productResult.setData(products);
productResult.setFilteredCount(150); // Results after applying filters
productResult.setTotalCount(500);    // Total available results
```

This class is typically used in REST API responses where you need to return both the actual data and metadata about filtering applied to the dataset, enabling proper pagination controls in client applications.
