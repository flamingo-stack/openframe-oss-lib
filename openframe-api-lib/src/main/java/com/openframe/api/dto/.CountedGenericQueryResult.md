A data transfer object that extends GenericQueryResult to include filtered count information for paginated query results with generic type support.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class extending GenericQueryResult
- **filteredCount** - Integer field representing the number of filtered results
- **Lombok annotations** - Provides builders, constructors, and data access methods
- **Generic type T** - Allows type-safe handling of different result data types

## Usage Example

```java
// Create a counted result for User entities
CountedGenericQueryResult<User> userResult = CountedGenericQueryResult.<User>builder()
    .data(userList)
    .totalCount(1000)
    .filteredCount(250)
    .build();

// Access the filtered count
int filtered = userResult.getFilteredCount();

// Use with different entity types
CountedGenericQueryResult<Product> productResult = new CountedGenericQueryResult<>();
productResult.setFilteredCount(42);
productResult.setData(productList);

// Useful for pagination with search/filter applied
if (result.getFilteredCount() < result.getTotalCount()) {
    System.out.println("Search filters applied: " + result.getFilteredCount() + 
                      " of " + result.getTotalCount() + " total records");
}
```
