A data transfer object that extends `GenericQueryResult` to include additional count information for filtered query results, typically used for paginated API responses.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class extending `GenericQueryResult<T>`
- **filteredCount** - Integer field storing the count of filtered results
- **Lombok Annotations** - Auto-generates boilerplate code for constructors, getters, setters, and equality methods

## Usage Example

```java
// Creating a counted query result for paginated user data
CountedGenericQueryResult<User> userResults = CountedGenericQueryResult.<User>builder()
    .data(Arrays.asList(user1, user2, user3))  // From parent class
    .totalCount(1000)                          // From parent class
    .filteredCount(50)                         // Count after applying filters
    .build();

// Accessing the filtered count
int filteredResults = userResults.getFilteredCount();

// Typical API response pattern
@GetMapping("/users")
public CountedGenericQueryResult<User> getUsers(
    @RequestParam String searchTerm,
    @RequestParam int page,
    @RequestParam int size) {
    
    List<User> users = userService.findFiltered(searchTerm, page, size);
    int totalFiltered = userService.countFiltered(searchTerm);
    
    return CountedGenericQueryResult.<User>builder()
        .data(users)
        .totalCount(userService.getTotalCount())
        .filteredCount(totalFiltered)
        .build();
}
```
