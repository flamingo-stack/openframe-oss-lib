A data transfer object that extends GenericQueryResult to include filtered count information for paginated query responses with additional counting metrics.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class that adds filtered count functionality to query results
- **filteredCount** - Integer field representing the number of records that match filter criteria
- **Generic Type T** - Allows the class to handle any type of query result data
- **Lombok Annotations** - Provides automatic generation of constructors, getters, setters, and builder patterns

## Usage Example

```java
// Creating a counted query result for user data
CountedGenericQueryResult<User> userResult = CountedGenericQueryResult.<User>builder()
    .data(userList)
    .totalCount(1000)
    .filteredCount(250)
    .build();

// Accessing the filtered count
int matchingRecords = userResult.getFilteredCount();

// Using with pagination info
CountedGenericQueryResult<Product> productResult = new CountedGenericQueryResult<>();
productResult.setData(products);
productResult.setTotalCount(5000);
productResult.setFilteredCount(150); // Records matching current filter
```

This class is typically used in REST API responses where you need to return both the total number of records and the count of records that match specific filter criteria, enabling proper pagination controls in client applications.
