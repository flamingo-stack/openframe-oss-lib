A DTO class that extends GenericQueryResult to include pagination information by adding a filtered count field for query results.

## Key Components

- **CountedGenericQueryResult<T>** - Generic DTO class extending GenericQueryResult
- **filteredCount** - Integer field representing the total number of filtered records
- **Lombok Annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a counted query result for a list of users
CountedGenericQueryResult<User> result = CountedGenericQueryResult.<User>builder()
    .data(userList)
    .filteredCount(150)
    .build();

// Accessing the filtered count
int totalFiltered = result.getFilteredCount();

// Using with pagination info
CountedGenericQueryResult<Product> productResult = new CountedGenericQueryResult<>();
productResult.setData(products);
productResult.setFilteredCount(1250); // Total products matching filter criteria

// Typical API response usage
@GetMapping("/api/users")
public CountedGenericQueryResult<User> getUsers(@RequestParam String filter) {
    List<User> users = userService.findByFilter(filter);
    int totalCount = userService.countByFilter(filter);
    
    return CountedGenericQueryResult.<User>builder()
        .data(users)
        .filteredCount(totalCount)
        .build();
}
```
