A generic data transfer object that wraps query results with pagination information, providing a standardized response format for paginated API endpoints.

## Key Components

- **`items`** - Generic list containing the actual query results of type T
- **`pageInfo`** - Cursor-based pagination metadata (CursorPageInfo object)
- **Lombok annotations** - Provides builder pattern, constructors, and getters/setters

## Usage Example

```java
// Creating a paginated result for users
List<User> users = userService.findUsers();
CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .endCursor("cursor123")
    .build();

GenericQueryResult<User> result = GenericQueryResult.<User>builder()
    .items(users)
    .pageInfo(pageInfo)
    .build();

// Accessing results
List<User> retrievedUsers = result.getItems();
boolean hasMore = result.getPageInfo().isHasNextPage();
String nextCursor = result.getPageInfo().getEndCursor();
```

This class enables consistent pagination handling across different entity types while maintaining type safety through generics.
