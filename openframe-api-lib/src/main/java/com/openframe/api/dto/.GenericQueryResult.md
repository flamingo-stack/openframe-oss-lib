A generic data transfer object that wraps query results with pagination information, providing a standardized response format for paginated API endpoints.

## Key Components

- **`items`** - Generic list containing the actual query results of type T
- **`pageInfo`** - Cursor-based pagination metadata from `CursorPageInfo`
- **Generic type parameter `<T>`** - Allows type-safe wrapping of any result type
- **Lombok annotations** - Provides builders, constructors, and getters/setters

## Usage Example

```java
// Creating a paginated user query result
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// Accessing results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();

// Using with different entity types
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(pageInfo)
    .build();
```
