A generic wrapper for paginated query results that provides a type-safe container for returning collections of data with pagination metadata.

## Key Components

- **`items`**: Generic list of type `T` containing the actual query results
- **`pageInfo`**: Pagination metadata from the shared `PageInfo` class
- **Generic Type Parameter `<T>`**: Allows reuse for any entity type (devices, users, etc.)

## Usage Example

```java
// Return paginated device results
GenericQueryResult<Device> deviceResults = GenericQueryResult.<Device>builder()
    .items(deviceList)
    .pageInfo(PageInfo.builder()
        .currentPage(1)
        .totalPages(5)
        .totalItems(100)
        .pageSize(20)
        .build())
    .build();

// Return paginated user results  
GenericQueryResult<User> userResults = GenericQueryResult.<User>builder()
    .items(userList)
    .pageInfo(pageInfo)
    .build();

// Access the data
List<Device> devices = deviceResults.getItems();
int totalDevices = deviceResults.getPageInfo().getTotalItems();
```

This DTO provides a consistent structure for all paginated API responses in the OpenFrame platform, ensuring uniform handling of result sets and pagination across different entity types.
