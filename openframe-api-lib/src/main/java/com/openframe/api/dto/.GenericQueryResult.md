A generic DTO class that wraps query results with pagination information, providing a standardized response format for paginated API endpoints.

## Key Components

- **`items`** - Generic list containing the actual query results of type T
- **`pageInfo`** - Cursor-based pagination metadata from `CursorPageInfo`
- **Generic type parameter `<T>`** - Allows reuse with any data type
- **Lombok annotations** - Provides builder pattern, constructors, and getters/setters

## Usage Example

```java
// For a list of users
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(userList)
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// For a list of products
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(pageInfo)
    .build();

// Accessing results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
```
