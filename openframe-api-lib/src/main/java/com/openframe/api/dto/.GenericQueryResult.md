A generic DTO class for paginated query results that can hold any type of data items along with pagination metadata.

## Key Components

- **`items`** - List of generic type T containing the actual query results
- **`pageInfo`** - CursorPageInfo object containing pagination metadata (cursors, hasNext/hasPrevious flags, etc.)
- **Generic Type `<T>`** - Allows reuse with any data type
- **Lombok Annotations** - Provides builders, constructors, and getters/setters

## Usage Example

```java
// For a list of users
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// For a list of products
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(pageInfo)
    .build();

// Accessing results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
```

This class provides a standardized way to return paginated results from API endpoints while maintaining type safety through generics.
