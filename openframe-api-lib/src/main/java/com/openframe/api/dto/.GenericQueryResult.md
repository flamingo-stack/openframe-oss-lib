A generic data transfer object that encapsulates paginated query results with type-safe item collections and cursor-based pagination metadata.

## Key Components

- **GenericQueryResult<T>** - Generic wrapper class for paginated query responses
- **items** - List of typed result items
- **pageInfo** - Cursor-based pagination information
- **Lombok annotations** - Provides builders, constructors, and data access methods

## Usage Example

```java
// Create a paginated result for User entities
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(List.of(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// Access the results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
String nextCursor = userResult.getPageInfo().getEndCursor();

// Use in API responses
@GetMapping("/users")
public GenericQueryResult<User> getUsers(@RequestParam String cursor) {
    return userService.getUsersPaginated(cursor);
}
```
