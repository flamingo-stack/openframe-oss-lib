A generic DTO class that wraps query results with pagination metadata, providing a standardized response format for paginated data across the API.

## Key Components

**GenericQueryResult<T>** - Main generic wrapper class with the following properties:
- `items` - List of typed results from the query
- `pageInfo` - Cursor-based pagination metadata (CursorPageInfo)

**Annotations**:
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@SuperBuilder` - Enables builder pattern with inheritance support
- `@NoArgsConstructor/@AllArgsConstructor` - Generates default and full constructors

## Usage Example

```java
// Creating a paginated result for User entities
CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("cursor123")
    .endCursor("cursor456")
    .build();

List<User> users = getUsersFromDatabase();

GenericQueryResult<User> result = GenericQueryResult.<User>builder()
    .items(users)
    .pageInfo(pageInfo)
    .build();

// Using in API responses
@GetMapping("/api/users")
public GenericQueryResult<User> getUsers(@RequestParam String cursor) {
    return userService.findUsersWithPagination(cursor);
}
```

The class provides type safety for different entity types while maintaining consistent pagination structure across all API endpoints.
