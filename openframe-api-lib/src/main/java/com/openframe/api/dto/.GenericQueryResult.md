A generic data transfer object that wraps query results with pagination metadata, providing a standardized response format for paginated API queries.

## Key Components

- **`items`** - Generic list containing the actual query result data
- **`pageInfo`** - Cursor-based pagination metadata from `CursorPageInfo`
- **Generic type `<T>`** - Allows reuse with any data type
- **Lombok annotations** - Provides builder pattern, constructors, and getters/setters

## Usage Example

```java
// Create a paginated result for users
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// Access the data
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();

// Use with different types
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(pageInfo)
    .build();
```
