A generic data transfer object that provides a standardized structure for paginated query results across the OpenFrame API. This class encapsulates query results with their associated pagination metadata using a type-safe generic approach.

## Key Components

- **items** - Generic list containing the actual query result data of type T
- **pageInfo** - PageInfo object containing pagination metadata (total count, page size, current page, etc.)
- **Generic Type Parameter T** - Allows the class to work with any data type while maintaining type safety

## Usage Example

```java
// Example with user entities
GenericQueryResult<User> userResults = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(PageInfo.builder()
        .totalItems(150)
        .pageSize(10)
        .currentPage(1)
        .build())
    .build();

// Example with device DTOs
GenericQueryResult<DeviceDto> deviceResults = GenericQueryResult.<DeviceDto>builder()
    .items(deviceList)
    .pageInfo(paginationInfo)
    .build();

// Accessing results
List<User> users = userResults.getItems();
int totalPages = userResults.getPageInfo().getTotalPages();
```

This pattern ensures consistent pagination handling across all API endpoints in the OpenFrame platform, making client-side data processing predictable and standardized.
