A generic data transfer object that provides a standardized structure for paginated query results across the OpenFrame API, combining result items with pagination metadata.

## Key Components

- **`items`** - Generic list containing the actual query results of type T
- **`pageInfo`** - Pagination metadata including page size, current page, and total counts
- **Generic Type Support** - Uses type parameter T to work with any data type
- **Lombok Annotations** - Provides automatic generation of getters, setters, builders, and constructors

## Usage Example

```java
// Creating a paginated result for user queries
PageInfo pageInfo = PageInfo.builder()
    .currentPage(1)
    .pageSize(10)
    .totalItems(150)
    .build();

List<User> users = userService.findUsers(searchCriteria);

GenericQueryResult<User> result = GenericQueryResult.<User>builder()
    .items(users)
    .pageInfo(pageInfo)
    .build();

// Accessing the results
List<User> resultItems = result.getItems();
int totalPages = result.getPageInfo().getTotalPages();
```

This DTO is typically used by API endpoints to return consistent paginated responses, ensuring all query operations follow the same structure for client consumption.
