Generic DTO class for paginated query results that provides a standardized structure for API responses containing collections of items with cursor-based pagination metadata.

## Key Components

- **`items`** - Generic list containing the actual data items of type T
- **`pageInfo`** - Cursor pagination metadata including navigation information
- **Generic Type `<T>`** - Allows reuse with any data type while maintaining type safety
- **Lombok Annotations** - Provides automatic generation of constructors, getters, setters, and builder pattern

## Usage Example

```java
// Query result for user data
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .hasPreviousPage(false)
        .startCursor("cursor123")
        .endCursor("cursor456")
        .build())
    .build();

// Access the results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();

// Works with any type
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(paginationInfo)
    .build();
```

This class enables consistent pagination handling across different API endpoints while maintaining type safety through generics and reducing boilerplate code with Lombok annotations.
