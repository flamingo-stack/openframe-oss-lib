A generic container class for paginated query results that combines a list of items with cursor-based pagination metadata.

## Key Components

- **`items`** - List of generic type T containing the actual query result data
- **`pageInfo`** - CursorPageInfo object containing pagination metadata (cursors, page size, etc.)
- **Generic type `<T>`** - Allows type-safe usage with any data type

## Usage Example

```java
// Query result for User entities
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(List.of(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .hasPreviousPage(false)
        .startCursor("cursor123")
        .endCursor("cursor456")
        .build())
    .build();

// Access the data
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
String nextCursor = userResult.getPageInfo().getEndCursor();

// Can be used with any type
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(pageInfo)
    .build();
```
