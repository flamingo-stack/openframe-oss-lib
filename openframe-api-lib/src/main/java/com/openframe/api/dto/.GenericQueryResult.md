A generic DTO wrapper for paginated query results, combining a typed list of items with pagination metadata.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `items` | `List<T>` | Generic list of result items |
| `pageInfo` | `PageInfo` | Pagination metadata (page number, size, total, etc.) |

## Usage Example

```java
// Wrapping a paginated list of tickets
List<TicketDto> tickets = ticketService.findAll(pageable);
PageInfo pageInfo = new PageInfo(page, size, totalCount);

GenericQueryResult<TicketDto> result = GenericQueryResult.<TicketDto>builder()
    .items(tickets)
    .pageInfo(pageInfo)
    .build();

// Accessing results
List<TicketDto> items = result.getItems();
PageInfo info = result.getPageInfo();
```

> **Note:** Uses Lombok's `@SuperBuilder` to support inheritance — subclasses can extend `GenericQueryResult<T>` and inherit builder support without additional boilerplate.
