A generic container class for paginated query results that combines data items with cursor-based pagination information.

## Key Components

- **GenericQueryResult<T>** - Generic wrapper class for paginated results
- **items** - List of result items of type T
- **pageInfo** - Cursor-based pagination metadata (CursorPageInfo)
- **Lombok annotations** - Provides builder pattern, constructors, and getters/setters

## Usage Example

```java
// Creating a paginated result for User entities
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(List.of(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .endCursor("cursor123")
        .build())
    .build();

// Accessing results
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
String nextCursor = userResult.getPageInfo().getEndCursor();

// Using in API responses
@GetMapping("/users")
public GenericQueryResult<User> getUsers(@RequestParam String cursor) {
    return userService.findUsersPaginated(cursor);
}
```
