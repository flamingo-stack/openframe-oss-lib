A generic wrapper class for paginated query results that combines a list of items with cursor-based pagination metadata.

## Key Components

- **`items`** - List of generic type T containing the actual query results
- **`pageInfo`** - CursorPageInfo object containing pagination metadata (cursors, hasNext/hasPrevious flags)
- **Generic type parameter `<T>`** - Allows reuse with any data type
- **Lombok annotations** - Provides builder pattern, constructors, and getters/setters

## Usage Example

```java
// Creating a paginated result for users
GenericQueryResult<User> userResult = GenericQueryResult.<User>builder()
    .items(Arrays.asList(user1, user2, user3))
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .hasPreviousPage(false)
        .startCursor("cursor123")
        .endCursor("cursor456")
        .build())
    .build();

// Accessing the data
List<User> users = userResult.getItems();
boolean hasMore = userResult.getPageInfo().isHasNextPage();
String nextCursor = userResult.getPageInfo().getEndCursor();

// Can be used with any type
GenericQueryResult<Product> productResult = GenericQueryResult.<Product>builder()
    .items(productList)
    .pageInfo(paginationInfo)
    .build();
```
