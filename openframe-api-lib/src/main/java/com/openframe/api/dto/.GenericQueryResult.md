A generic DTO container for paginated query results, wrapping a typed list of items alongside pagination metadata.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `items` | `List<T>` | The paginated result set of any type `T` |
| `pageInfo` | `PageInfo` | Pagination metadata (e.g. total count, current page, page size) |

## Usage Example

```java
// Returning a paginated list of tickets
GenericQueryResult<TicketDto> result = GenericQueryResult.<TicketDto>builder()
    .items(ticketList)
    .pageInfo(PageInfo.builder()
        .totalCount(100)
        .currentPage(1)
        .pageSize(20)
        .build())
    .build();

// Accessing results
List<TicketDto> tickets = result.getItems();
PageInfo page = result.getPageInfo();
```

## Notes

- Uses Lombok `@Data` for getters/setters, `@SuperBuilder` for builder inheritance support, and `@NoArgsConstructor`/`@AllArgsConstructor` for flexible instantiation.
- `@SuperBuilder` allows subclasses to extend this result type while retaining builder functionality.
- Designed to be used across multiple query endpoints in the OpenFrame API layer as a standardized response envelope.
