A Data Transfer Object (DTO) representing detailed audit log entry information returned from the OpenFrame audit/event system.

## Key Components

| Field | Type | Description |
|---|---|---|
| `id` | `String` | Unique identifier for the log entry |
| `toolEventId` | `String` | Reference ID from the originating tool event |
| `eventType` | `String` | Classification of the event |
| `ingestDay` | `String` | Date partition key for log ingestion |
| `toolType` | `String` | Source tool that generated the event |
| `severity` | `String` | Log severity level (e.g. `INFO`, `WARN`, `ERROR`) |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Hostname of the affected machine |
| `organizationId` | `String` | Tenant/organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `summary` | `String` | Short description of the event |
| `timestamp` | `Instant` | Exact UTC time the event occurred |
| `message` | `String` | Detailed log message |
| `details` | `String` | Additional raw or structured event payload |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Builder pattern
LogDetails log = LogDetails.builder()
    .id("log-001")
    .toolEventId("evt-xyz")
    .eventType("LOGIN_FAILURE")
    .severity("WARN")
    .userId("user-42")
    .hostname("workstation-01")
    .organizationId("org-99")
    .organizationName("Acme Corp")
    .summary("Failed login attempt detected")
    .timestamp(Instant.now())
    .message("User exceeded max retry attempts")
    .build();

// Access fields
String severity = log.getSeverity(); // "WARN"
Instant when = log.getTimestamp();
```
