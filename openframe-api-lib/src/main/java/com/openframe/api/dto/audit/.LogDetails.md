A DTO class representing detailed audit log information within the OpenFrame platform, capturing comprehensive event data including user actions, device information, and organizational context.

## Key Components

- **@Data/@Builder/@NoArgsConstructor/@AllArgsConstructor**: Lombok annotations providing standard getters, setters, builder pattern, and constructors
- **Event Identification**: `id`, `toolEventId`, `eventType` for unique event tracking
- **Temporal Fields**: `ingestDay`, `timestamp` for time-based log organization and precise event timing
- **Context Fields**: `toolType`, `severity`, `userId`, `deviceId`, `hostname` for event classification and source identification
- **Organizational Fields**: `organizationId`, `organizationName` for multi-tenant log segregation
- **Content Fields**: `summary`, `message`, `details` for storing event descriptions and detailed information

## Usage Example

```java
LogDetails logEntry = LogDetails.builder()
    .id("log-12345")
    .toolEventId("event-67890")
    .eventType("USER_LOGIN")
    .toolType("AUTHENTICATION")
    .severity("INFO")
    .userId("user-abc123")
    .deviceId("device-xyz789")
    .hostname("workstation-01")
    .organizationId("org-456")
    .organizationName("Acme Corp")
    .summary("User successful login")
    .timestamp(Instant.now())
    .message("User logged in successfully")
    .details("Login from IP: 192.168.1.100")
    .build();

// Access log information
String eventSummary = logEntry.getSummary();
Instant logTime = logEntry.getTimestamp();
```
