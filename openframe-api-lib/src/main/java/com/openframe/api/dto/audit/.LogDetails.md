A data transfer object representing audit log details containing comprehensive information about security and system events within the OpenFrame platform.

## Key Components

- **Identification Fields**: `toolEventId`, `eventType`, `toolType` for event classification
- **Organizational Context**: `userId`, `deviceId`, `hostname`, `organizationId`, `organizationName` for tracing event sources
- **Temporal Data**: `timestamp` (Instant) and `ingestDay` for chronological tracking
- **Content Fields**: `summary`, `message`, `details` for event descriptions
- **Metadata**: `severity` for event prioritization

## Usage Example

```java
// Create a log entry for a security event
LogDetails securityLog = LogDetails.builder()
    .toolEventId("evt-12345")
    .eventType("authentication")
    .toolType("firewall")
    .severity("HIGH")
    .userId("user-789")
    .deviceId("device-456")
    .hostname("server01.example.com")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .summary("Failed login attempt")
    .timestamp(Instant.now())
    .message("Multiple failed authentication attempts detected")
    .details("Source IP: 192.168.1.100, Attempts: 5")
    .ingestDay("2024-01-15")
    .build();

// Access log information
String eventSummary = securityLog.getSummary();
Instant eventTime = securityLog.getTimestamp();
```

This DTO is typically used in audit trails, security monitoring, and compliance reporting within the OpenFrame API ecosystem.
