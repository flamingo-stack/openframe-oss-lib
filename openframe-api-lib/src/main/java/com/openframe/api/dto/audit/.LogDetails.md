A data transfer object (DTO) that represents audit log details with comprehensive information about system events, user activities, and organizational context.

## Key Components

- **LogDetails class**: Main DTO with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **Event identification fields**: `toolEventId`, `eventType`, `toolType` for categorizing and tracking events
- **Context fields**: `userId`, `deviceId`, `hostname`, `organizationId`, `organizationName` for identifying the source and scope
- **Temporal fields**: `timestamp`, `ingestDay` for time-based tracking and partitioning
- **Content fields**: `summary`, `message`, `details` for storing event information
- **Metadata fields**: `severity` for event classification

## Usage Example

```java
// Creating a log entry using the builder pattern
LogDetails logEntry = LogDetails.builder()
    .toolEventId("evt-12345")
    .eventType("LOGIN")
    .toolType("AUTH_SERVICE")
    .severity("INFO")
    .userId("user123")
    .organizationId("org456")
    .hostname("app-server-01")
    .timestamp(Instant.now())
    .summary("User login successful")
    .message("User authenticated successfully")
    .build();

// Accessing properties
String eventId = logEntry.getToolEventId();
Instant when = logEntry.getTimestamp();
```
