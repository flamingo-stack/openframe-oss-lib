A data transfer object that represents detailed audit log information for events tracked within the OpenFrame platform, including device, user, and organization context.

## Key Components

- **LogDetails**: Main DTO class with comprehensive audit log fields
- **Event Identification**: `id`, `toolEventId`, `eventType` for tracking specific events
- **Organizational Context**: `organizationId`, `organizationName`, `userId` for multi-tenant support
- **Device Information**: `deviceId`, `hostname` for asset tracking
- **Event Metadata**: `severity`, `toolType`, `ingestDay` for categorization
- **Content Fields**: `summary`, `message`, `details` for event descriptions
- **Temporal Data**: `timestamp` using `Instant` for precise event timing

## Usage Example

```java
// Creating a new log detail entry
LogDetails logDetail = LogDetails.builder()
    .id("log-123")
    .toolEventId("evt-456")
    .eventType("SECURITY_ALERT")
    .toolType("EDR")
    .severity("HIGH")
    .userId("user-789")
    .deviceId("device-abc")
    .hostname("workstation-01")
    .organizationId("org-def")
    .organizationName("Acme Corp")
    .summary("Suspicious process detected")
    .timestamp(Instant.now())
    .message("Malware detected on endpoint")
    .details("Process: malicious.exe, PID: 1234")
    .ingestDay("2024-01-15")
    .build();

// Accessing log information
String alertSeverity = logDetail.getSeverity();
Instant eventTime = logDetail.getTimestamp();
```
