A data transfer object (DTO) class that represents comprehensive audit log details for tracking security and operational events within the OpenFrame system.

## Key Components

### Class Annotations
- `@Data` - Generates getters, setters, toString, equals, and hashCode methods
- `@Builder` - Enables builder pattern for object creation
- `@NoArgsConstructor`/`@AllArgsConstructor` - Provides default and full constructors

### Core Fields
- **Event Identification**: `toolEventId`, `eventType`, `toolType`
- **Temporal Data**: `timestamp`, `ingestDay`
- **Entity Context**: `userId`, `deviceId`, `hostname`, `organizationId`, `organizationName`
- **Event Details**: `severity`, `summary`, `message`, `details`

## Usage Example

```java
// Creating log details using builder pattern
LogDetails auditLog = LogDetails.builder()
    .toolEventId("EVT-12345")
    .eventType("USER_LOGIN")
    .toolType("AUTHENTICATION")
    .severity("INFO")
    .userId("user123")
    .deviceId("device456")
    .hostname("api-server-01")
    .organizationId("org789")
    .organizationName("Acme Corp")
    .summary("Successful user login")
    .timestamp(Instant.now())
    .message("User authenticated successfully")
    .details("Login from IP: 192.168.1.100")
    .ingestDay("2024-01-15")
    .build();

// Using in audit service
auditService.recordEvent(auditLog);
```

This DTO is typically used in audit trails, security monitoring, and compliance reporting systems to capture detailed information about system events and user activities.
