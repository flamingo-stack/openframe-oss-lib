A data transfer object (DTO) that represents audit log entry details for the OpenFrame API audit system.

## Key Components

**LogDetails Class**
- `@Data` - Provides getters, setters, equals, hashCode, and toString methods
- `@Builder` - Enables fluent builder pattern for object construction
- `@NoArgsConstructor/@AllArgsConstructor` - Provides default and full constructors

**Core Fields**
- `toolEventId` - Unique identifier for the log event
- `eventType` - Classification of the audit event
- `severity` - Log severity level
- `timestamp` - Event occurrence time as Instant
- `userId/deviceId/hostname` - Identity and location tracking
- `organizationId/organizationName` - Organizational context
- `summary/message/details` - Event description at different detail levels

## Usage Example

```java
// Create log entry using builder pattern
LogDetails logEntry = LogDetails.builder()
    .toolEventId("evt-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user123")
    .organizationId("org456")
    .summary("User login successful")
    .timestamp(Instant.now())
    .build();

// Access fields
String eventId = logEntry.getToolEventId();
Instant when = logEntry.getTimestamp();
```
