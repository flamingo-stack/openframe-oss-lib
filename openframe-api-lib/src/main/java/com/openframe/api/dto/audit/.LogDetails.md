A data transfer object (DTO) that represents detailed log information for audit and monitoring purposes in the OpenFrame API system.

## Key Components

**Class Annotations**:
- `@Data` - Generates getters, setters, toString, equals, and hashCode methods
- `@Builder` - Provides builder pattern for object creation
- `@NoArgsConstructor/@AllArgsConstructor` - Generates constructors

**Primary Fields**:
- `toolEventId` - Unique identifier for the logged event
- `eventType` - Classification of the event type
- `severity` - Log level (INFO, WARN, ERROR, etc.)
- `timestamp` - When the event occurred (Instant for precision)
- `userId/deviceId/hostname` - Source identification fields
- `organizationId/organizationName` - Tenant context information
- `summary/message/details` - Event description at different detail levels

**Operational Fields**:
- `ingestDay` - Date partitioning field for log storage
- `toolType` - Source system that generated the log

## Usage Example

```java
// Create log details using builder pattern
LogDetails logDetails = LogDetails.builder()
    .toolEventId("evt-12345")
    .eventType("USER_LOGIN")
    .severity("INFO")
    .userId("user-456")
    .organizationId("org-789")
    .organizationName("Acme Corp")
    .hostname("api-server-01")
    .timestamp(Instant.now())
    .summary("User login successful")
    .message("User authenticated via OAuth")
    .details("{\"ip\":\"192.168.1.100\",\"userAgent\":\"Chrome/91.0\"}")
    .build();

// Use in audit logging service
auditService.recordEvent(logDetails);
```
