A Data Transfer Object (DTO) that represents comprehensive log details for audit and monitoring purposes in the OpenFrame platform, capturing security events, device activities, and organizational context.

## Key Components

**Primary Fields:**
- `id` - Unique log identifier
- `toolEventId` - Associated tool-specific event ID
- `eventType` - Category of the logged event
- `severity` - Event severity level (e.g., INFO, WARN, ERROR)
- `timestamp` - Event occurrence time
- `message` - Primary log message
- `details` - Additional event details

**Organizational Context:**
- `organizationId` / `organizationName` - Organization identification
- `userId` - Associated user identifier
- `deviceId` / `hostname` - Device/system identification

**Technical Fields:**
- `toolType` - Source tool that generated the log
- `ingestDay` - Date partition for log storage optimization

## Usage Example

```java
// Creating a security event log
LogDetails securityLog = LogDetails.builder()
    .id("log-12345")
    .toolEventId("tool-event-67890")
    .eventType("AUTHENTICATION")
    .severity("WARN")
    .userId("user-001")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .deviceId("device-456")
    .hostname("workstation-01")
    .toolType("SSO_PROVIDER")
    .timestamp(Instant.now())
    .message("Failed login attempt")
    .details("{\"ip\": \"192.168.1.100\", \"attempts\": 3}")
    .ingestDay("2024-01-15")
    .build();
```

This DTO is typically used in audit trails, security monitoring, and log aggregation systems within the OpenFrame MSP platform.
