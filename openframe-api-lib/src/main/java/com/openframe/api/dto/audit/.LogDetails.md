A data transfer object (DTO) that represents audit log details for the OpenFrame API system. This class encapsulates all relevant information about security and operational events.

## Key Components

- **Event Identification**: `toolEventId`, `eventType`, `ingestDay` for tracking and categorization
- **Context Information**: `toolType`, `severity`, `userId`, `deviceId`, `hostname` for environmental context
- **Organization Details**: `organizationId`, `organizationName` for multi-tenant support
- **Content Fields**: `summary`, `message`, `details` for storing event descriptions
- **Temporal Data**: `timestamp` using `Instant` for precise time tracking
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Creating a new log entry
LogDetails auditLog = LogDetails.builder()
    .toolEventId("EVT-001")
    .eventType("LOGIN_ATTEMPT")
    .severity("HIGH")
    .userId("user123")
    .deviceId("device456")
    .hostname("api.example.com")
    .organizationId("org789")
    .organizationName("Acme Corp")
    .summary("Failed login attempt")
    .timestamp(Instant.now())
    .message("User attempted login with invalid credentials")
    .details("{\"attempts\": 3, \"ip\": \"192.168.1.1\"}")
    .build();

// Accessing log information
String eventType = auditLog.getEventType();
Instant when = auditLog.getTimestamp();
```
