A Data Transfer Object (DTO) representing a structured audit log entry, used to transfer log data across application layers in the OpenFrame audit system.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the log entry |
| `toolEventId` | `String` | Reference to the originating tool event |
| `eventType` | `String` | Classification of the event |
| `ingestDay` | `String` | Day the log was ingested |
| `toolType` | `String` | Type of tool that generated the log |
| `severity` | `String` | Log severity level (e.g. ERROR, WARN) |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Source hostname |
| `organizationId` | `String` | Tenant organization identifier |
| `organizationName` | `String` | Tenant organization display name |
| `summary` | `String` | Brief description of the event |
| `timestamp` | `Instant` | UTC timestamp of the event |
| `message` | `String` | Primary log message |
| `details` | `String` | Extended log detail payload |

**Lombok Annotations:** `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` — provides getters/setters, builder pattern, and both constructors automatically.

## Usage Example

```java
// Builder pattern
LogDetails log = LogDetails.builder()
    .id("log-001")
    .toolEventId("evt-abc123")
    .eventType("DEVICE_ALERT")
    .toolType("RMM")
    .severity("WARN")
    .userId("user-42")
    .deviceId("dev-99")
    .hostname("workstation-01")
    .organizationId("org-flamingo")
    .organizationName("Flamingo MSP")
    .summary("Disk usage exceeded 90%")
    .timestamp(Instant.now())
    .message("High disk usage detected")
    .details("{\"used\": \"91%\", \"drive\": \"C:\"}")
    .build();
```
