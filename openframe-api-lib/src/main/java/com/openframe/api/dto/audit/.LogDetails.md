A data transfer object (DTO) that represents audit log details with comprehensive logging information including event metadata, user context, and organizational data.

## Key Components

**LogDetails Class**
- `@Data` - Lombok annotation providing getters, setters, equals, hashCode, and toString
- `@Builder` - Enables builder pattern for object construction
- `@NoArgsConstructor/@AllArgsConstructor` - Provides default and full parameter constructors

**Core Fields**
- Event identifiers: `toolEventId`, `eventType`, `ingestDay`
- System context: `toolType`, `severity`, `hostname`
- User/Organization: `userId`, `deviceId`, `organizationId`, `organizationName`
- Content: `summary`, `message`, `details`
- Timing: `timestamp` (Instant)

## Usage Example

```java
// Using builder pattern
LogDetails logEntry = LogDetails.builder()
    .toolEventId("EVT-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user123")
    .organizationId("org456")
    .hostname("web-server-01")
    .timestamp(Instant.now())
    .summary("User login successful")
    .message("User authenticated successfully")
    .build();

// Accessing fields
String eventId = logEntry.getToolEventId();
Instant when = logEntry.getTimestamp();
```
