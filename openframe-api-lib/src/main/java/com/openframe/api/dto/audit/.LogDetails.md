A data transfer object (DTO) that represents audit log details in the OpenFrame API system. This class encapsulates comprehensive logging information including event metadata, user context, and organizational details.

## Key Components

- **LogDetails**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **Event Fields**: `toolEventId`, `eventType`, `ingestDay`, `toolType`, `severity` for event classification
- **Context Fields**: `userId`, `deviceId`, `hostname`, `organizationId`, `organizationName` for tracking context
- **Content Fields**: `summary`, `message`, `details` for log content
- **Timestamp**: `Instant timestamp` for precise timing information

## Usage Example

```java
// Create log details using builder pattern
LogDetails logDetails = LogDetails.builder()
    .toolEventId("evt-12345")
    .eventType("USER_LOGIN")
    .severity("INFO")
    .userId("user-789")
    .organizationId("org-456")
    .organizationName("ACME Corp")
    .summary("User login attempt")
    .timestamp(Instant.now())
    .message("Successful login from mobile device")
    .build();

// Access properties
String eventType = logDetails.getEventType();
Instant when = logDetails.getTimestamp();
```
