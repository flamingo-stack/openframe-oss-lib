A data transfer object (DTO) representing detailed audit log information with comprehensive event tracking fields. This class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation.

## Key Components

- **Data Fields**: 14 audit-related properties including event identifiers, organizational context, device information, and log content
- **Lombok Annotations**: 
  - `@Data` - generates getters, setters, toString, equals, and hashCode
  - `@Builder` - enables fluent builder pattern for object construction
  - `@NoArgsConstructor/@AllArgsConstructor` - generates default and full parameter constructors
- **Key Fields**: `toolEventId`, `eventType`, `severity`, `userId`, `organizationId`, `timestamp`, `message`

## Usage Example

```java
// Using builder pattern
LogDetails logDetails = LogDetails.builder()
    .toolEventId("EVT-12345")
    .eventType("LOGIN_ATTEMPT")
    .severity("INFO")
    .userId("user123")
    .organizationId("org-456")
    .hostname("server01")
    .timestamp(Instant.now())
    .message("User login successful")
    .build();

// Using constructor
LogDetails details = new LogDetails();
details.setToolEventId("EVT-67890");
details.setEventType("DATA_ACCESS");
details.setSeverity("WARN");
details.setTimestamp(Instant.now());
```
