A data transfer object (DTO) that represents audit log details for tracking events and activities within the OpenFrame API system.

## Key Components

- **LogDetails class**: Main DTO with comprehensive audit log fields
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation
- **Event identification**: `toolEventId`, `eventType`, `toolType` for categorizing log entries
- **Context information**: `userId`, `deviceId`, `hostname`, `organizationId/Name` for tracing
- **Temporal data**: `timestamp`, `ingestDay` for time-based operations
- **Content fields**: `summary`, `message`, `details` for log content
- **Severity level**: For log prioritization and filtering

## Usage Example

```java
// Creating a log entry using builder pattern
LogDetails logEntry = LogDetails.builder()
    .toolEventId("evt_12345")
    .eventType("USER_LOGIN")
    .toolType("WEB_APP")
    .severity("INFO")
    .userId("user123")
    .hostname("api-server-01")
    .organizationId("org456")
    .summary("User authentication successful")
    .timestamp(Instant.now())
    .message("User logged in successfully")
    .build();

// Accessing log details
String eventId = logEntry.getToolEventId();
Instant logTime = logEntry.getTimestamp();
```
