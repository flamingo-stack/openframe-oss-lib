Data Transfer Object (DTO) for filtering audit log entries with criteria such as tool types, event types, severities, and organizations.

## Key Components

- **`toolTypes`** - List of tool type strings to filter by
- **`eventTypes`** - List of event type strings to filter by  
- **`severities`** - List of severity level strings to filter by
- **`organizations`** - List of organization filter options for scoping results

## Usage Example

```java
// Create filters for audit log queries
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("API", "DATABASE"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARN"))
    .organizations(List.of(
        new OrganizationFilterOption("org-123"),
        new OrganizationFilterOption("org-456")
    ))
    .build();

// Use with audit service
auditService.getLogs(filters);

// Or create empty filters for unfiltered results
LogFilters noFilters = new LogFilters();
```

The class uses Lombok annotations for automatic generation of getters, setters, builders, and constructors, making it ideal for use in REST API request bodies or service method parameters.
