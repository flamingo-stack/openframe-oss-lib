A data transfer object (DTO) that encapsulates filtering criteria for audit log queries. This class provides a structured way to define multiple filter parameters for retrieving and filtering log entries.

## Key Components

- **toolTypes**: List of tool type identifiers to filter logs by specific tools
- **eventTypes**: List of event type identifiers to filter by event categories
- **severities**: List of severity levels (e.g., INFO, WARN, ERROR) for log filtering
- **organizations**: List of organization filter options for multi-tenant log access

## Usage Example

```java
// Create log filters for audit query
LogFilters filters = LogFilters.builder()
    .toolTypes(Arrays.asList("DEPLOYMENT", "MONITORING"))
    .eventTypes(Arrays.asList("USER_LOGIN", "DATA_ACCESS"))
    .severities(Arrays.asList("ERROR", "WARN"))
    .organizations(Arrays.asList(
        new OrganizationFilterOption("org1", "Company A"),
        new OrganizationFilterOption("org2", "Company B")
    ))
    .build();

// Use with audit service
List<AuditLog> filteredLogs = auditService.getLogs(filters);
```

The class uses Lombok annotations to automatically generate constructors, getters, setters, and builder pattern methods, reducing boilerplate code while maintaining clean API design.
