A data transfer object (DTO) that encapsulates filtering criteria for audit log queries. This class provides a structured way to specify multiple filter parameters when searching or retrieving log entries.

## Key Components

- **toolTypes** - List of tool type identifiers to filter logs by specific tools
- **eventTypes** - List of event type identifiers to filter by event categories
- **severities** - List of severity levels (e.g., INFO, WARN, ERROR) for log filtering
- **organizations** - List of organization filter options for multi-tenant log filtering

## Usage Example

```java
// Create filters for error logs from specific tools
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("database", "authentication"))
    .eventTypes(List.of("login_failure", "connection_error"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizations(List.of(
        new OrganizationFilterOption("org-123", "Company A")
    ))
    .build();

// Use with audit service
auditService.getLogs(filters);
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation, making it convenient for creating filter configurations in audit logging scenarios.
