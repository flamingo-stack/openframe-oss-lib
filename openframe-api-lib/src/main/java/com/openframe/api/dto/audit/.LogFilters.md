A data transfer object that encapsulates filtering criteria for querying log entries in audit operations.

## Key Components

- **toolTypes**: List of tool type filters to apply to log queries
- **eventTypes**: List of event type filters for categorizing log entries  
- **severities**: List of severity level filters (e.g., ERROR, WARN, INFO)
- **organizations**: List of organization filter options for multi-tenant filtering

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern implementation.

## Usage Example

```java
// Create filters using builder pattern
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("DATABASE", "API_GATEWAY"))
    .eventTypes(List.of("LOGIN", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARN"))
    .organizations(List.of(
        new OrganizationFilterOption("org1", "Company A"),
        new OrganizationFilterOption("org2", "Company B")
    ))
    .build();

// Use with audit service
auditService.queryLogs(filters);

// Create empty filters
LogFilters emptyFilters = new LogFilters();
```
