A DTO class that defines filtering criteria for log queries in the audit system, containing filter options for tool types, event types, severities, and organizations.

## Key Components

- **toolTypes** - List of tool type identifiers to filter log entries by specific tools
- **eventTypes** - List of event type identifiers to filter by specific event categories
- **severities** - List of severity level identifiers (e.g., INFO, WARN, ERROR) for filtering log entries
- **organizations** - List of OrganizationFilterOption objects for filtering by organizational context

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation.

## Usage Example

```java
// Create filters for error logs from specific tools
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("authentication", "payment-processor"))
    .eventTypes(List.of("login-failure", "transaction-error"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizations(List.of(
        OrganizationFilterOption.builder()
            .organizationId("org-123")
            .build()
    ))
    .build();

// Use in audit service
auditService.searchLogs(filters);
```
