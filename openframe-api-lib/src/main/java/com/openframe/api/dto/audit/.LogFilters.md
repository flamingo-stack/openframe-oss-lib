A DTO (Data Transfer Object) used to encapsulate filter criteria when querying audit logs in the OpenFrame API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolTypes` | `List<String>` | Filter logs by tool/integration type |
| `eventTypes` | `List<String>` | Filter logs by event category |
| `severities` | `List<String>` | Filter logs by severity level |
| `organizations` | `List<OrganizationFilterOption>` | Filter logs by organization entries |

## Usage Example

```java
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("RMM", "PSA"))
    .eventTypes(List.of("LOGIN", "UPDATE"))
    .severities(List.of("ERROR", "WARNING"))
    .organizations(List.of(
        OrganizationFilterOption.builder().id("org-123").build()
    ))
    .build();
```
