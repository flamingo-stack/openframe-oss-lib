A data transfer object (DTO) that encapsulates filtering criteria for audit log queries. This class provides a structured way to pass multiple filter parameters for log searching and retrieval operations.

## Key Components

- **toolTypes** - List of tool type filters to apply
- **eventTypes** - List of event type filters for log entries  
- **severities** - List of severity level filters (e.g., INFO, WARN, ERROR)
- **organizations** - List of organization-based filter options using custom `OrganizationFilterOption` objects

## Usage Example

```java
// Create log filters for audit query
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("DATABASE", "API_GATEWAY"))
    .eventTypes(List.of("LOGIN", "DATA_ACCESS", "CONFIGURATION_CHANGE"))
    .severities(List.of("ERROR", "WARN"))
    .organizations(List.of(
        new OrganizationFilterOption("org-123", "Acme Corp"),
        new OrganizationFilterOption("org-456", "Tech Solutions")
    ))
    .build();

// Use in audit service
auditService.searchLogs(filters);

// Create empty filters for unfiltered search
LogFilters noFilters = new LogFilters();
```

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern implementation.
