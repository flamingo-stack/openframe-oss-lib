A data transfer object (DTO) that encapsulates filtering criteria for log-based audit queries. This class provides a structured way to specify multiple filter dimensions when searching through audit logs.

## Key Components

- **toolTypes** - List of tool type identifiers to filter logs by
- **eventTypes** - List of event type identifiers for filtering specific audit events  
- **severities** - List of severity levels to include in filtered results
- **organizations** - List of organization filter options for multi-tenant filtering

## Usage Example

```java
// Create filters for audit log queries
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("DATABASE", "API_GATEWAY"))
    .eventTypes(List.of("USER_LOGIN", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARNING"))
    .organizations(List.of(
        new OrganizationFilterOption("org-123", "INCLUDE"),
        new OrganizationFilterOption("org-456", "EXCLUDE")
    ))
    .build();

// Use with audit service
auditService.getLogs(filters);

// Create empty filters for unfiltered queries
LogFilters noFilters = new LogFilters();
```
