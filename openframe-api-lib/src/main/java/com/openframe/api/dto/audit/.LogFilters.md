A data transfer object (DTO) that encapsulates filtering criteria for audit log queries in the OpenFrame API. This class provides a structured way to specify multiple filter parameters for retrieving and displaying audit logs.

## Key Components

- **toolTypes** - List of tool type identifiers to filter logs by specific MSP tools
- **eventTypes** - List of event type identifiers to filter by specific audit event categories
- **severities** - List of severity level identifiers to filter by log importance/criticality
- **organizations** - List of organization filter options for multi-tenant log filtering

## Usage Example

```java
// Create comprehensive log filters for audit query
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("rmm", "psa", "documentation"))
    .eventTypes(List.of("user_login", "config_change", "alert_created"))
    .severities(List.of("error", "warning", "critical"))
    .organizations(List.of(
        OrganizationFilterOption.builder()
            .organizationId("org-123")
            .includeSubOrganizations(true)
            .build()
    ))
    .build();

// Apply filters to audit log service
AuditLogQuery query = AuditLogQuery.builder()
    .filters(filters)
    .dateRange(DateRange.lastWeek())
    .build();
```

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern implementation, making it easy to construct complex filter combinations for audit log queries across the OpenFrame platform.
