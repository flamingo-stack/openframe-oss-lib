A data transfer object (DTO) that encapsulates filtering criteria for audit log queries in the OpenFrame API. This class provides structured filtering options to narrow down log entries based on multiple criteria.

## Key Components

- **toolTypes** - List of tool type identifiers to filter logs by specific tools
- **eventTypes** - List of event type identifiers to filter by event categories
- **severities** - List of severity levels (e.g., INFO, WARN, ERROR) for log filtering
- **organizations** - List of organization filter options for multi-tenant log access

## Usage Example

```java
// Create log filters for error events from specific tools
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("TICKETING", "MONITORING"))
    .eventTypes(List.of("USER_LOGIN", "SYSTEM_ERROR"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizations(List.of(
        OrganizationFilterOption.builder()
            .id("org-123")
            .name("Acme Corp")
            .build()
    ))
    .build();

// Use with audit service
auditService.getLogs(filters);
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation, making it easy to construct complex filter combinations for audit log queries.
