A simple Data Transfer Object (DTO) used to capture the input for delete mutation operations, carrying a single required entity identifier.

## Key Components

| Element | Description |
|---|---|
| `id` | The unique identifier of the entity to delete; validated as non-blank |
| `@NotBlank` | Jakarta validation constraint ensuring `id` is not null, empty, or whitespace |
| `@Data` / `@Builder` | Lombok annotations generating getters, setters, equals, hashCode, toString, and a builder pattern |
| `@NoArgsConstructor` / `@AllArgsConstructor` | Lombok-generated constructors for both no-arg and full-arg instantiation |

## Usage Example

```java
// Using the builder pattern
MutationDeleteInput input = MutationDeleteInput.builder()
    .id("abc-123")
    .build();

// Using the all-args constructor
MutationDeleteInput input = new MutationDeleteInput("abc-123");

// Typical use in a GraphQL/REST mutation handler
public ResponseEntity<Void> deleteEntity(@Valid @RequestBody MutationDeleteInput input) {
    service.deleteById(input.getId());
    return ResponseEntity.noContent().build();
}
```
