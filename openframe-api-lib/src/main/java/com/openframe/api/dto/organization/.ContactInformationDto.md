A Data Transfer Object (DTO) that encapsulates contact information for organizations, including contact persons and address details.

## Key Components

- **ContactInformationDto** - Main record class with `@Builder` annotation for easy object construction
- **contacts** - List of `ContactPersonDto` objects representing contact persons
- **physicalAddress** - `AddressDto` for the organization's physical location
- **mailingAddress** - `AddressDto` for mailing purposes
- **mailingAddressSameAsPhysical** - Boolean flag indicating if mailing and physical addresses are identical

## Usage Example

```java
// Create contact information with builder pattern
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(contactPerson1, contactPerson2))
    .physicalAddress(physicalAddr)
    .mailingAddress(mailingAddr)
    .mailingAddressSameAsPhysical(false)
    .build();

// Access contact information
List<ContactPersonDto> contacts = contactInfo.contacts();
AddressDto physical = contactInfo.physicalAddress();
boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```

This immutable record leverages Lombok's `@Builder` for fluent object creation while maintaining type safety through Java records.
