A data transfer object that encapsulates contact information for an organization, including contact persons and physical/mailing addresses.

## Key Components

- **ContactInformationDto** - Immutable record class with Lombok @Builder annotation
- **contacts** - List of ContactPersonDto objects representing contact persons
- **physicalAddress** - AddressDto containing the organization's physical location
- **mailingAddress** - AddressDto for mailing correspondence
- **mailingAddressSameAsPhysical** - Boolean flag indicating if mailing and physical addresses are identical

## Usage Example

```java
// Create contact information using builder pattern
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(contactPerson1, contactPerson2))
    .physicalAddress(physicalAddr)
    .mailingAddress(mailingAddr)
    .mailingAddressSameAsPhysical(false)
    .build();

// Access immutable properties
List<ContactPersonDto> contacts = contactInfo.contacts();
AddressDto physical = contactInfo.physicalAddress();
Boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```

This DTO is typically used in REST APIs for transferring organization contact information between client and server, providing a clean separation between the data layer and presentation layer.
