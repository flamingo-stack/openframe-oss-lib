A data transfer object that encapsulates contact information for an organization, including contact persons and address details.

## Key Components

- **ContactInformationDto** - Record class with `@Builder` annotation for immutable contact data
- **contacts** - List of contact persons (`ContactPersonDto` objects)
- **physicalAddress** - Primary address location (`AddressDto` object)
- **mailingAddress** - Mailing address location (`AddressDto` object)  
- **mailingAddressSameAsPhysical** - Boolean flag indicating if addresses are identical

## Usage Example

```java
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(contactPerson1, contactPerson2))
    .physicalAddress(physicalAddr)
    .mailingAddress(mailingAddr)
    .mailingAddressSameAsPhysical(false)
    .build();

// Access data
List<ContactPersonDto> people = contactInfo.contacts();
AddressDto location = contactInfo.physicalAddress();
boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```

The record provides immutable data structure with automatic getters, equals, hashCode, and toString methods. The builder pattern enables flexible construction with optional fields.
