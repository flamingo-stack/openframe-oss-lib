A data transfer object (DTO) that encapsulates contact information for an organization, including contact persons, physical and mailing addresses, and a flag indicating if both addresses are the same.

## Key Components

- **contacts** - List of `ContactPersonDto` objects representing contact persons
- **physicalAddress** - `AddressDto` containing the physical location address
- **mailingAddress** - `AddressDto` containing the mailing address
- **mailingAddressSameAsPhysical** - Boolean flag indicating if mailing and physical addresses are identical

## Usage Example

```java
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("John Doe")
            .email("john@example.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("Springfield")
        .build())
    .mailingAddress(null)
    .mailingAddressSameAsPhysical(true)
    .build();

// Access contact information
List<ContactPersonDto> contacts = contactInfo.contacts();
AddressDto physical = contactInfo.physicalAddress();
boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```

The record is immutable and uses Lombok's `@Builder` annotation for convenient object construction.
