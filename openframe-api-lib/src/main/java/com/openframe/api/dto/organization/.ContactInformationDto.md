Data Transfer Object for encapsulating contact information within an organization context, including contact persons and address details.

## Key Components

- **ContactInformationDto** - Immutable record with builder pattern support
- **contacts** - List of contact persons associated with the organization
- **physicalAddress** - Physical location address
- **mailingAddress** - Mailing/correspondence address  
- **mailingAddressSameAsPhysical** - Flag indicating if mailing address matches physical address

## Usage Example

```java
// Create contact information with builder pattern
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("John Smith")
            .email("john@company.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("Springfield")
        .build())
    .mailingAddressSameAsPhysical(true)
    .build();

// Access contact details
List<ContactPersonDto> contacts = contactInfo.contacts();
AddressDto address = contactInfo.physicalAddress();
boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```
