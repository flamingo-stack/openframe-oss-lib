A data transfer object that encapsulates contact information for an organization, including contact persons and addresses.

## Key Components

- **ContactInformationDto** - Record class with Lombok `@Builder` annotation for immutable contact data
- **contacts** - List of `ContactPersonDto` objects representing contact persons
- **physicalAddress** - `AddressDto` for the physical location
- **mailingAddress** - `AddressDto` for mailing purposes
- **mailingAddressSameAsPhysical** - Boolean flag indicating if mailing address matches physical address

## Usage Example

```java
// Creating contact information with builder pattern
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("John Doe")
            .email("john@company.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("Springfield")
        .build())
    .mailingAddressSameAsPhysical(true)
    .build();

// Accessing data
List<ContactPersonDto> contacts = contactInfo.contacts();
boolean sameAddress = contactInfo.mailingAddressSameAsPhysical();
```

The record structure provides immutability and automatic generation of equals, hashCode, and toString methods, making it ideal for data transfer scenarios.
