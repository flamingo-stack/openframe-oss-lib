A data transfer object representing a filter option for device queries, containing the filter value, display label, and occurrence count.

## Key Components

- **value**: The actual filter value used in queries
- **label**: Human-readable display text for the filter option
- **count**: Number of items matching this filter option

## Usage Example

```java
// Creating a device filter option
DeviceFilterOption filterOption = DeviceFilterOption.builder()
    .value("ACTIVE")
    .label("Active Devices")
    .count(25)
    .build();

// Using in a filter dropdown
List<DeviceFilterOption> statusFilters = Arrays.asList(
    new DeviceFilterOption("ACTIVE", "Active", 25),
    new DeviceFilterOption("INACTIVE", "Inactive", 5),
    new DeviceFilterOption("MAINTENANCE", "Under Maintenance", 3)
);

// Accessing properties
String filterValue = filterOption.getValue(); // "ACTIVE"
String displayText = filterOption.getLabel(); // "Active Devices"
Integer itemCount = filterOption.getCount();   // 25
```

This DTO is typically used in APIs that provide filter options for device listings, allowing clients to display available filters with their respective counts for better user experience.
