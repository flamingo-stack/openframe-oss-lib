A data transfer object that represents a filter option for device filtering operations, containing the option value, display label, and occurrence count.

## Key Components

- **value**: The actual filter value used in queries
- **label**: Human-readable display text for the filter option
- **count**: Number of devices matching this filter option

## Usage Example

```java
// Creating a filter option with builder pattern
DeviceFilterOption filterOption = DeviceFilterOption.builder()
    .value("android")
    .label("Android Devices")
    .count(25)
    .build();

// Creating filter options for a device type dropdown
List<DeviceFilterOption> deviceTypes = Arrays.asList(
    new DeviceFilterOption("mobile", "Mobile Phones", 150),
    new DeviceFilterOption("tablet", "Tablets", 75),
    new DeviceFilterOption("laptop", "Laptops", 200)
);

// Accessing filter data
String filterValue = filterOption.getValue();    // "android"
String displayText = filterOption.getLabel();   // "Android Devices"
Integer deviceCount = filterOption.getCount();  // 25
```

This DTO is typically used in REST API responses to provide filter options for device management interfaces, where users can filter devices by various criteria and see how many devices match each option.
