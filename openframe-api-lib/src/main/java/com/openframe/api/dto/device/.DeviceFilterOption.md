A simple DTO representing a single filter option for device filtering, containing a value, display label, and result count.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `value` | `String` | The filter option's unique identifier or key used in filter queries |
| `label` | `String` | Human-readable display name for the filter option |
| `count` | `Integer` | Number of devices matching this filter option |

## Usage Example

```java
// Build a filter option using the builder pattern
DeviceFilterOption option = DeviceFilterOption.builder()
    .value("windows")
    .label("Windows")
    .count(42)
    .build();

// Or use the all-args constructor
DeviceFilterOption option = new DeviceFilterOption("windows", "Windows", 42);

// Accessing fields via Lombok-generated getters
String value = option.getValue();   // "windows"
String label = option.getLabel();   // "Windows"
Integer count = option.getCount();  // 42
```

Typically used in device filter dropdowns or faceted search responses within the [OpenFrame](https://openframe.ai) platform, where technicians need to quickly segment and locate managed devices by criteria such as OS type, status, or site.
