A data transfer object (DTO) representing a filter option for device queries, containing the option value, display label, and associated count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of devices matching this filter option

## Usage Example

```java
// Creating a device filter option for device type
DeviceFilterOption laptopFilter = DeviceFilterOption.builder()
    .value("laptop")
    .label("Laptops")
    .count(25)
    .build();

// Creating filter options for device status
List<DeviceFilterOption> statusFilters = Arrays.asList(
    new DeviceFilterOption("active", "Active", 150),
    new DeviceFilterOption("inactive", "Inactive", 12),
    new DeviceFilterOption("maintenance", "Under Maintenance", 3)
);

// Accessing filter properties
String filterValue = laptopFilter.getValue();    // "laptop"
String displayText = laptopFilter.getLabel();    // "Laptops"
Integer deviceCount = laptopFilter.getCount();   // 25
```

This DTO is typically used in API responses to provide filter options for device management interfaces, allowing clients to display available filters with their corresponding counts for better user experience.
