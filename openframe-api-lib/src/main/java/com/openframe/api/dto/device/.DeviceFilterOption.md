A simple data transfer object (DTO) that represents a filter option for device-related queries, typically used in dropdown menus or filter interfaces where users can select device criteria with associated counts.

## Key Components

- **value**: The actual filter value used in queries or API calls
- **label**: The human-readable display text shown to users
- **count**: Optional numeric indicator showing how many items match this filter option

## Usage Example

```java
// Creating filter options for device types
DeviceFilterOption mobileOption = DeviceFilterOption.builder()
    .value("mobile")
    .label("Mobile Devices")
    .count(42)
    .build();

DeviceFilterOption desktopOption = DeviceFilterOption.builder()
    .value("desktop")
    .label("Desktop Computers")
    .count(18)
    .build();

// Using in a filter interface
List<DeviceFilterOption> deviceTypeFilters = Arrays.asList(
    mobileOption,
    desktopOption,
    new DeviceFilterOption("tablet", "Tablets", 7)
);

// Accessing properties
String filterValue = mobileOption.getValue();     // "mobile"
String displayText = mobileOption.getLabel();     // "Mobile Devices"
Integer resultCount = mobileOption.getCount();    // 42
```

The class uses Lombok annotations to automatically generate getters, setters, constructors, and builder pattern methods, making it ideal for creating filter UI components where users need to see both the option name and the number of matching results.
