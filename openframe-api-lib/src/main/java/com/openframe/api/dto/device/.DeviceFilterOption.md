A simple data transfer object (DTO) that represents a filter option for device-related operations, containing a value, display label, and associated count.

## Key Components

- **value**: The actual filter value used in queries or operations
- **label**: Human-readable display text for the filter option
- **count**: Optional numeric count associated with the filter option
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create a device filter option using builder pattern
DeviceFilterOption filterOption = DeviceFilterOption.builder()
    .value("android")
    .label("Android Devices")
    .count(15)
    .build();

// Create using constructor
DeviceFilterOption statusFilter = new DeviceFilterOption("active", "Active Devices", 42);

// Use in a filter list
List<DeviceFilterOption> deviceTypes = Arrays.asList(
    new DeviceFilterOption("mobile", "Mobile", 25),
    new DeviceFilterOption("tablet", "Tablet", 8),
    new DeviceFilterOption("desktop", "Desktop", 12)
);

// Access properties
String filterValue = filterOption.getValue(); // "android"
String displayText = filterOption.getLabel(); // "Android Devices"
Integer deviceCount = filterOption.getCount(); // 15
```
