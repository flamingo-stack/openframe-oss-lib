A data transfer object (DTO) representing filter options for device queries, containing display information and result counts for UI filtering components.

## Key Components

- **value** - The internal filter value used in queries
- **label** - User-friendly display text for the filter option
- **count** - Number of devices matching this filter criteria

## Usage Example

```java
// Creating filter options for device types
DeviceFilterOption laptopFilter = DeviceFilterOption.builder()
    .value("laptop")
    .label("Laptop Computers")
    .count(245)
    .build();

DeviceFilterOption serverFilter = DeviceFilterOption.builder()
    .value("server")
    .label("Server Hardware")
    .count(89)
    .build();

// Using in a filter dropdown
List<DeviceFilterOption> deviceTypeFilters = Arrays.asList(
    laptopFilter,
    serverFilter
);

// Accessing properties
String filterValue = laptopFilter.getValue(); // "laptop"
String displayText = laptopFilter.getLabel(); // "Laptop Computers"
Integer deviceCount = laptopFilter.getCount(); // 245
```

This DTO is typically used to populate filter dropdowns in device management interfaces, providing both the query parameter value and user-friendly labels with result counts.
