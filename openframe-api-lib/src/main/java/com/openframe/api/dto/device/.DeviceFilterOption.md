A data transfer object that represents a filter option for device filtering operations, containing the option value, display label, and associated count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of items associated with this filter option
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter option using builder pattern
DeviceFilterOption statusFilter = DeviceFilterOption.builder()
    .value("ACTIVE")
    .label("Active Devices")
    .count(25)
    .build();

// Create filter option using constructor
DeviceFilterOption typeFilter = new DeviceFilterOption("MOBILE", "Mobile Devices", 12);

// Access properties
String filterValue = statusFilter.getValue(); // "ACTIVE"
String displayText = statusFilter.getLabel(); // "Active Devices"
Integer deviceCount = statusFilter.getCount(); // 25

// Typical usage in filter dropdown or API response
List<DeviceFilterOption> filterOptions = Arrays.asList(
    new DeviceFilterOption("ALL", "All Devices", 50),
    new DeviceFilterOption("ACTIVE", "Active", 25),
    new DeviceFilterOption("INACTIVE", "Inactive", 25)
);
```
