A simple DTO (Data Transfer Object) that represents a filter option for device filtering operations, containing the option's value, display label, and associated count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of items matching this filter option

## Usage Example

```java
// Create a device filter option for active devices
DeviceFilterOption activeFilter = DeviceFilterOption.builder()
    .value("active")
    .label("Active Devices")
    .count(42)
    .build();

// Use in a filter dropdown or search interface
List<DeviceFilterOption> statusFilters = Arrays.asList(
    DeviceFilterOption.builder()
        .value("active")
        .label("Active")
        .count(42)
        .build(),
    DeviceFilterOption.builder()
        .value("inactive")
        .label("Inactive")
        .count(8)
        .build()
);
```

This DTO is typically used in REST API responses to provide filter options for device management interfaces, enabling users to see available filter values along with their counts for better UX.
