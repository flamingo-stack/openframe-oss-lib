A data transfer object representing a filter option for device queries, containing a selectable value, display label, and result count.

## Key Components

- **value**: The actual filter value used in queries
- **label**: Human-readable display text for the filter option
- **count**: Number of devices matching this filter option

## Usage Example

```java
// Create a filter option for device status
DeviceFilterOption statusOption = DeviceFilterOption.builder()
    .value("ACTIVE")
    .label("Active Devices")
    .count(42)
    .build();

// Use in a filter dropdown or API response
List<DeviceFilterOption> statusFilters = Arrays.asList(
    DeviceFilterOption.builder()
        .value("ACTIVE")
        .label("Active")
        .count(150)
        .build(),
    DeviceFilterOption.builder()
        .value("INACTIVE")
        .label("Inactive")
        .count(25)
        .build()
);

// Access properties
String filterValue = statusOption.getValue();    // "ACTIVE"
String displayText = statusOption.getLabel();    // "Active Devices"
Integer resultCount = statusOption.getCount();   // 42
```

This DTO is typically used in device management UIs to populate filter dropdowns with both the technical filter values and user-friendly labels, along with counts showing how many devices match each filter option.
