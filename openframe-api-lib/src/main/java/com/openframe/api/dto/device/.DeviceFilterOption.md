A data transfer object (DTO) that represents a filter option for device queries, containing the option value, display label, and count of matching items.

## Key Components

- **DeviceFilterOption** - Main DTO class with Lombok annotations for automatic getter/setter generation
- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of devices that match this filter option

## Usage Example

```java
// Creating a filter option for device types
DeviceFilterOption mobileFilter = DeviceFilterOption.builder()
    .value("mobile")
    .label("Mobile Devices")
    .count(45)
    .build();

// Using constructor
DeviceFilterOption desktopFilter = new DeviceFilterOption("desktop", "Desktop Computers", 23);

// Accessing properties
String filterValue = mobileFilter.getValue();  // "mobile"
String displayText = mobileFilter.getLabel();  // "Mobile Devices"
Integer deviceCount = mobileFilter.getCount(); // 45

// Typical usage in filter lists
List<DeviceFilterOption> filterOptions = Arrays.asList(
    new DeviceFilterOption("mobile", "Mobile", 45),
    new DeviceFilterOption("tablet", "Tablet", 12),
    new DeviceFilterOption("desktop", "Desktop", 23)
);
```

This class is typically used in API responses to provide client applications with available filter options and their corresponding counts for device-related queries.
