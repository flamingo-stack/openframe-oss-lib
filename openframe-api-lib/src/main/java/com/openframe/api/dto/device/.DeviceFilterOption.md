A data transfer object representing a filter option for device filtering operations, containing the option value, display label, and count of matching items.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of devices matching this filter option

## Usage Example

```java
// Creating a device filter option for device type
DeviceFilterOption typeFilter = DeviceFilterOption.builder()
    .value("smartphone")
    .label("Smartphone")
    .count(42)
    .build();

// Using in a filter list
List<DeviceFilterOption> statusFilters = Arrays.asList(
    new DeviceFilterOption("active", "Active Devices", 156),
    new DeviceFilterOption("inactive", "Inactive Devices", 23),
    new DeviceFilterOption("maintenance", "Under Maintenance", 8)
);

// Accessing properties
String filterValue = typeFilter.getValue();     // "smartphone"
String displayText = typeFilter.getLabel();    // "Smartphone"
Integer deviceCount = typeFilter.getCount();   // 42
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it ideal for creating filter dropdown options in device management interfaces.
