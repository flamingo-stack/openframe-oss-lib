A simple data transfer object (DTO) that represents a filter option for device-related filtering operations, containing a value-label pair with an optional count for display purposes.

## Key Components

- **value** - The actual filter value used in queries or operations
- **label** - Human-readable display text for the filter option
- **count** - Optional numeric count associated with the filter (e.g., number of devices matching this filter)

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern support.

## Usage Example

```java
// Creating a device filter option using builder pattern
DeviceFilterOption filterOption = DeviceFilterOption.builder()
    .value("android")
    .label("Android Devices")
    .count(42)
    .build();

// Using in a filter dropdown context
List<DeviceFilterOption> osFilters = Arrays.asList(
    new DeviceFilterOption("ios", "iOS Devices", 25),
    new DeviceFilterOption("android", "Android Devices", 42),
    new DeviceFilterOption("windows", "Windows Devices", 8)
);

// Accessing properties
String filterValue = filterOption.getValue(); // "android"
String displayText = filterOption.getLabel(); // "Android Devices" 
Integer deviceCount = filterOption.getCount(); // 42
```
