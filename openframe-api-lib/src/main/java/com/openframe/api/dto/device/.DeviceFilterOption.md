A data transfer object (DTO) that represents a filter option for device queries, containing a value-label pair with an associated count for UI filtering components.

## Key Components

- **value**: String identifier for the filter option
- **label**: Human-readable display text for the filter option  
- **count**: Integer representing the number of items matching this filter option
- Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern

## Usage Example

```java
// Creating a filter option using builder pattern
DeviceFilterOption statusFilter = DeviceFilterOption.builder()
    .value("active")
    .label("Active Devices")
    .count(42)
    .build();

// Creating filter options for a dropdown
List<DeviceFilterOption> deviceTypeFilters = Arrays.asList(
    new DeviceFilterOption("smartphone", "Smartphone", 15),
    new DeviceFilterOption("tablet", "Tablet", 8),
    new DeviceFilterOption("laptop", "Laptop", 23)
);

// Accessing properties
String filterValue = statusFilter.getValue(); // "active"
String displayText = statusFilter.getLabel(); // "Active Devices"
Integer itemCount = statusFilter.getCount(); // 42
```

This DTO is typically used in REST API responses to provide frontend applications with structured filter options that include both the technical value and user-friendly labels along with result counts.
