A data transfer object (DTO) that encapsulates filtering criteria for querying audit logs. This class provides a structured way to pass multiple filter parameters for log searches and retrievals.

## Key Components

- **Date Range Filters**: `startDate` and `endDate` for temporal filtering
- **Category Filters**: `eventTypes`, `toolTypes`, and `severities` for log classification
- **Context Filters**: `organizationIds` and `deviceId` for scoping results
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Using builder pattern
LogFilterOptions filters = LogFilterOptions.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.now())
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARN"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Using constructor
LogFilterOptions filters2 = new LogFilterOptions(
    LocalDate.of(2024, 1, 1),
    LocalDate.now(),
    List.of("LOGIN"),
    List.of("WEB", "MOBILE"),
    List.of("INFO"),
    List.of("org-123"),
    "device-456"
);

// All fields are optional and can be null for flexible filtering
LogFilterOptions minimalFilter = LogFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .build();
```
