A Data Transfer Object (DTO) that encapsulates filtering criteria for querying audit logs. This class provides a structured way to pass multiple filter parameters for log searches and auditing operations.

## Key Components

- **Date Range Filtering**: `startDate` and `endDate` fields for temporal filtering
- **Event Classification**: `eventTypes` and `toolTypes` lists for categorizing log entries
- **Severity Filtering**: `severities` list for filtering by log importance levels
- **Organization Scope**: `organizationIds` list for multi-tenant filtering
- **Device Targeting**: `deviceId` string for device-specific log queries
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Create filter options using builder pattern
LogFilterOptions filterOptions = LogFilterOptions.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARN"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Use in service method
List<AuditLog> logs = auditService.findLogsByFilter(filterOptions);

// Create empty filter for all logs
LogFilterOptions allLogs = new LogFilterOptions();
```
