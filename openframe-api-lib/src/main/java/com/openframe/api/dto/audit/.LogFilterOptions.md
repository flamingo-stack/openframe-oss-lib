A data transfer object (DTO) that encapsulates filtering criteria for querying audit logs in the OpenFrame API system.

## Key Components

- **Date Range Fields**: `startDate` and `endDate` for temporal filtering
- **Category Filters**: `eventTypes`, `toolTypes`, and `severities` as string lists
- **Scope Filters**: `organizationIds` list and `deviceId` string for targeted queries
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate reduction

## Usage Example

```java
// Create filter for error logs from last week
LogFilterOptions filter = LogFilterOptions.builder()
    .startDate(LocalDate.now().minusWeeks(1))
    .endDate(LocalDate.now())
    .eventTypes(List.of("LOGIN_FAILURE", "SYSTEM_ERROR"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Use with audit service
auditService.getLogs(filter);
```

The builder pattern enables flexible construction of filter criteria, allowing clients to specify only the relevant filtering parameters for their audit log queries.
