A data transfer object (DTO) that encapsulates filtering criteria for querying audit logs. This class uses Lombok annotations to automatically generate boilerplate code for getters, setters, constructors, and builder pattern implementation.

## Key Components

- **Date Range Filtering**: `startDate` and `endDate` fields for temporal filtering
- **Category Filters**: `eventTypes`, `toolTypes`, and `severities` lists for log categorization
- **Scope Filters**: `organizationIds` list and `deviceId` string for filtering by organizational or device scope
- **Builder Pattern**: Enabled through `@Builder` annotation for flexible object construction

## Usage Example

```java
// Create filter options using builder pattern
LogFilterOptions filters = LogFilterOptions.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARNING"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Use in service layer for log querying
List<AuditLog> results = auditService.findLogs(filters);

// Create empty filter for all logs
LogFilterOptions allLogs = new LogFilterOptions();
```
