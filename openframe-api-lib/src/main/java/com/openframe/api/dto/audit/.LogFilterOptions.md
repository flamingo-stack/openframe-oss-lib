A data transfer object (DTO) that encapsulates filtering criteria for audit log queries, providing flexible search parameters for log retrieval operations.

## Key Components

- **Date Range Filtering**: `startDate` and `endDate` fields for temporal log filtering
- **Event Classification**: `eventTypes` list for filtering by specific event categories
- **Tool-based Filtering**: `toolTypes` list for filtering logs by originating tools or services
- **Severity Levels**: `severities` list for filtering by log severity or importance levels
- **Multi-tenant Support**: `organizationIds` list for filtering logs across multiple organizations
- **Device-specific Filtering**: `deviceId` string for filtering logs from a specific device or endpoint

## Usage Example

```java
// Create comprehensive log filter
LogFilterOptions filter = LogFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .endDate(LocalDate.now())
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .toolTypes(List.of("API_GATEWAY", "DATABASE"))
    .severities(List.of("ERROR", "WARN"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Create simple date-based filter
LogFilterOptions simpleFilter = LogFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(1))
    .endDate(LocalDate.now())
    .build();
```

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern, making it ideal for use with audit log services and repositories.
