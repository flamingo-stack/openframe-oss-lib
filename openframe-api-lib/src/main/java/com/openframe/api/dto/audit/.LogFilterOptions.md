A data transfer object that encapsulates filtering criteria for querying audit logs within the OpenFrame API system.

## Key Components

- **Date Range Filters**: `startDate` and `endDate` for temporal filtering
- **Event Classification**: `eventTypes` and `toolTypes` for categorizing log entries
- **Severity Filtering**: `severities` list for filtering by log level/importance
- **Organizational Scope**: `organizationIds` for multi-tenant filtering
- **Device Targeting**: `deviceId` for device-specific log retrieval
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter options using builder pattern
LogFilterOptions filterOptions = LogFilterOptions.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .eventTypes(Arrays.asList("LOGIN", "DATA_ACCESS", "SYSTEM_ERROR"))
    .severities(Arrays.asList("ERROR", "WARN"))
    .organizationIds(Arrays.asList("org123", "org456"))
    .deviceId("device789")
    .build();

// Use with audit service
List<AuditLog> logs = auditService.getLogs(filterOptions);

// Create minimal filter for recent errors only
LogFilterOptions errorFilter = LogFilterOptions.builder()
    .startDate(LocalDate.now().minusDays(7))
    .severities(Arrays.asList("ERROR", "CRITICAL"))
    .build();
```
