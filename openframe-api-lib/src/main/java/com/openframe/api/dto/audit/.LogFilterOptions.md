Data Transfer Object (DTO) that encapsulates filtering criteria for audit log queries. This class provides a structured way to specify date ranges, event types, and other parameters when retrieving audit logs.

## Key Components

- **Date Filtering**: `startDate` and `endDate` fields for temporal filtering using `LocalDate`
- **Event Classification**: `eventTypes` and `toolTypes` lists for categorizing log entries
- **Severity Levels**: `severities` list for filtering by log importance/criticality
- **Organization Scope**: `organizationIds` list for multi-tenant filtering
- **Device Targeting**: `deviceId` string for device-specific log retrieval
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate reduction

## Usage Example

```java
// Create filter using builder pattern
LogFilterOptions filter = LogFilterOptions.builder()
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 1, 31))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .severities(List.of("ERROR", "WARN"))
    .organizationIds(List.of("org-123", "org-456"))
    .deviceId("device-789")
    .build();

// Use with audit service
List<AuditLog> logs = auditService.getFilteredLogs(filter);
```
