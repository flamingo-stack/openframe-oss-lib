A data transfer object (DTO) that encapsulates filtering criteria for organization queries. This class provides a structured way to pass multiple filter parameters when searching or retrieving organizations.

## Key Components

- **category** - Filter organizations by category/type
- **minEmployees** - Minimum employee count threshold
- **maxEmployees** - Maximum employee count threshold  
- **hasActiveContract** - Filter by active contract status
- Lombok annotations for automatic getter/setter generation and builder pattern

## Usage Example

```java
// Create filter using builder pattern
OrganizationFilterOptions filter = OrganizationFilterOptions.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Use in service layer
List<Organization> organizations = organizationService.findByFilter(filter);

// Create minimal filter
OrganizationFilterOptions simpleFilter = OrganizationFilterOptions.builder()
    .hasActiveContract(true)
    .build();
```

The builder pattern allows for flexible filter creation where only relevant criteria need to be specified, making it ideal for dynamic search functionality in APIs or service layers.
