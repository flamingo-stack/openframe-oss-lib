A Data Transfer Object (DTO) that encapsulates filter criteria for querying organizations within the system. This class provides a structured way to pass multiple optional filtering parameters when searching or retrieving organization data.

## Key Components

- **category** - String filter for organization category/type
- **minEmployees** - Integer threshold for minimum employee count
- **maxEmployees** - Integer threshold for maximum employee count  
- **hasActiveContract** - Boolean flag to filter by active contract status
- Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter using builder pattern
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Use with service layer
List<Organization> results = organizationService.findOrganizations(filters);

// Create filter with partial criteria
OrganizationFilterOptions partialFilter = OrganizationFilterOptions.builder()
    .hasActiveContract(false)
    .build();
```

The builder pattern allows for flexible construction of filter objects with only the required criteria, making it easy to compose different search queries based on varying business requirements.
