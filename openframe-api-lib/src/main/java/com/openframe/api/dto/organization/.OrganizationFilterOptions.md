This DTO class encapsulates filtering criteria for organization queries, providing a structured way to specify search parameters for organization data retrieval.

## Key Components

- **category** - Filter organizations by category/type
- **minEmployees** - Minimum employee count threshold
- **maxEmployees** - Maximum employee count threshold  
- **hasActiveContract** - Filter by active contract status
- Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter using builder pattern
OrganizationFilterOptions filter = OrganizationFilterOptions.builder()
    .category("Technology")
    .minEmployees(10)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Create filter with constructor
OrganizationFilterOptions filter = new OrganizationFilterOptions(
    "Healthcare", 
    50, 
    1000, 
    false
);

// Use in service layer
List<Organization> organizations = organizationService.findByFilter(filter);
```
