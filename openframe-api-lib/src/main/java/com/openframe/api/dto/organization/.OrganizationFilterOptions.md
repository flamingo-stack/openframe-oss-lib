A Data Transfer Object (DTO) that encapsulates filtering criteria for organization queries. This class provides a structured way to pass multiple filter parameters when searching or retrieving organizations.

## Key Components

- **category** - String filter for organization category/type
- **minEmployees** - Integer minimum employee count threshold
- **maxEmployees** - Integer maximum employee count threshold  
- **hasActiveContract** - Boolean flag to filter by active contract status
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate reduction

## Usage Example

```java
// Create filter using builder pattern
OrganizationFilterOptions filter = OrganizationFilterOptions.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Create filter with partial criteria
OrganizationFilterOptions smallCompanyFilter = OrganizationFilterOptions.builder()
    .maxEmployees(25)
    .build();

// Use in service method
List<Organization> organizations = organizationService.findByFilter(filter);
```

The builder pattern enables flexible filter construction where only relevant criteria need to be specified, while null values can indicate "no filter" for that particular field.
