Data Transfer Object (DTO) for encapsulating filter criteria when querying organizations within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|---|---|---|
| `category` | `String` | Filter organizations by category type |
| `minEmployees` | `Integer` | Minimum employee count threshold |
| `maxEmployees` | `Integer` | Maximum employee count threshold |
| `hasActiveContract` | `Boolean` | Filter by active contract status |
| `status` | `String` | Filter by organization status |

## Usage Example

```java
// Build filter options using the fluent builder pattern
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("MSP")
    .minEmployees(10)
    .maxEmployees(500)
    .hasActiveContract(true)
    .status("ACTIVE")
    .build();

// Or use the no-args constructor with setters
OrganizationFilterOptions filters = new OrganizationFilterOptions();
filters.setCategory("Enterprise");
filters.setHasActiveContract(true);
```

> **Note:** All fields are optional. Null fields are typically ignored during query construction, allowing partial filter combinations.
