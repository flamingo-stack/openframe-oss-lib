A data transfer object (DTO) that encapsulates filtering criteria for organization queries. This class provides a structured way to pass multiple optional filter parameters when searching or retrieving organizations.

## Key Components

- **category**: String filter for organization category/type
- **minEmployees**: Minimum employee count threshold
- **maxEmployees**: Maximum employee count threshold  
- **hasActiveContract**: Boolean flag to filter by contract status
- Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Using builder pattern to create filter options
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Using constructor
OrganizationFilterOptions filters = new OrganizationFilterOptions(
    "Healthcare", 10, 100, false);

// Partial filtering (other fields remain null)
OrganizationFilterOptions filters = OrganizationFilterOptions.builder()
    .category("Finance")
    .hasActiveContract(true)
    .build();
```

The class is typically used as a parameter in service methods or repository queries to apply conditional filtering based on the provided criteria.
