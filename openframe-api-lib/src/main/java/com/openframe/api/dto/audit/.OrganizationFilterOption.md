A data transfer object representing an organization option for filtering audit logs, typically used in dropdown menus or filter UI components.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for automatic getter/setter generation
- **id** - String field representing the organization identifier
- **name** - String field representing the organization display name
- **@Builder** - Enables builder pattern for object creation
- **@Data** - Generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a list for dropdown population
List<OrganizationFilterOption> filterOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Marketing Dept"),
    new OrganizationFilterOption("org-2", "Engineering Dept")
);
```
