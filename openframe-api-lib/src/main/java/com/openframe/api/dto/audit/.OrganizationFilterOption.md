A simple data transfer object representing an organization filter option used in audit log filtering interfaces, containing an organization's ID and display name.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for boilerplate code generation
- **id** - String field storing the unique organization identifier
- **name** - String field storing the human-readable organization name
- **@Data** - Lombok annotation providing getters, setters, equals, hashCode, and toString methods
- **@Builder** - Lombok annotation enabling builder pattern construction
- **@NoArgsConstructor/@AllArgsConstructor** - Lombok annotations for default and full parameter constructors

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a list for dropdown population
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Company A"),
    new OrganizationFilterOption("org-2", "Company B")
);

// Access properties
String orgId = option.getId();
String orgName = option.getName();
```

This DTO is typically used to populate organization filter dropdowns in audit log interfaces, providing both the internal ID for filtering and the display name for user selection.
