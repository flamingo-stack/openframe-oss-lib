A simple DTO class that represents an organization filter option containing an ID and name, primarily used for populating organization dropdown menus in audit log filtering interfaces.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for automatic getter/setter generation
- **id** - String field representing the unique organization identifier
- **name** - String field representing the display name of the organization
- **@Builder** - Enables fluent builder pattern for object creation
- **@Data** - Generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a filter dropdown list
List<OrganizationFilterOption> organizations = Arrays.asList(
    new OrganizationFilterOption("org-1", "Company A"),
    new OrganizationFilterOption("org-2", "Company B")
);
```
