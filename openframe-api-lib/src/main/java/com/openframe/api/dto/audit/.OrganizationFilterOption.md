A simple DTO class that represents an organization option for filtering audit logs in dropdown menus and selection components.

## Key Components

- **id**: String identifier for the organization
- **name**: String display name for the organization
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating an organization filter option using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Using in a list for dropdown population
List<OrganizationFilterOption> organizations = Arrays.asList(
    OrganizationFilterOption.builder().id("org-1").name("Company A").build(),
    OrganizationFilterOption.builder().id("org-2").name("Company B").build()
);

// Accessing properties
String orgId = option.getId();
String displayName = option.getName();
```

This DTO is typically used in audit log filtering interfaces where users need to select specific organizations from a dropdown menu to filter log entries by organization.
