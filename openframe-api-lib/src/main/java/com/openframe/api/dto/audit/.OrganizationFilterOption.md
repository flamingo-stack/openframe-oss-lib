A simple DTO class representing an organization filter option that provides ID and name fields for organization dropdown selections in log filtering interfaces.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for data binding
- **id** - String field representing the unique organization identifier
- **name** - String field representing the organization display name
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Creating an organization filter option using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Creating using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Accessing fields (via Lombok-generated getters)
String orgId = option.getId();
String orgName = option.getName();

// Using in a list for dropdown population
List<OrganizationFilterOption> orgOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Company A"),
    new OrganizationFilterOption("org-2", "Company B")
);
```

This DTO is typically used in REST API responses to populate organization filter dropdowns in audit log interfaces, allowing users to filter logs by specific organizations.
