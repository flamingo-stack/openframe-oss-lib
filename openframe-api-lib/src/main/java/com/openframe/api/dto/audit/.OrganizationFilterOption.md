A simple DTO class representing an organization option for filtering audit logs, containing basic identification and display information.

## Key Components

- **OrganizationFilterOption**: Main DTO class with Lombok annotations for boilerplate code generation
- **id**: String field storing the unique organization identifier
- **name**: String field storing the display name for the organization
- **Lombok annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a filter dropdown
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Company A"),
    new OrganizationFilterOption("org-2", "Company B")
);
```
