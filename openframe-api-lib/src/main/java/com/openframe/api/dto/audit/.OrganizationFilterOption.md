A data transfer object that represents a selectable organization option for filtering audit logs in dropdown interfaces.

## Key Components

- **id** - Unique identifier for the organization
- **name** - Display name for the organization
- **Lombok annotations** - Automatically generates getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create organization filter option using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Use in audit log filtering
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    OrganizationFilterOption.builder()
        .id("org-001")
        .name("Tech Solutions Inc")
        .build(),
    OrganizationFilterOption.builder()
        .id("org-002") 
        .name("Digital Services LLC")
        .build()
);

// Typical usage in API response for populating organization dropdown
public List<OrganizationFilterOption> getOrganizationFilterOptions() {
    return organizationService.getAllOrganizations()
        .stream()
        .map(org -> OrganizationFilterOption.builder()
            .id(org.getId())
            .name(org.getDisplayName())
            .build())
        .collect(Collectors.toList());
}
```

This DTO is specifically designed for audit log filtering interfaces where users need to select organizations from a dropdown menu.
