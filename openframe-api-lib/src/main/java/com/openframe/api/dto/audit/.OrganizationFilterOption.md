A simple DTO (Data Transfer Object) that represents an organization filter option for audit log filtering functionality, containing basic organization identification and display information.

## Key Components

- **id** - String field representing the unique organization identifier
- **name** - String field containing the organization's display name
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern
- **Builder pattern** - Enables fluent object construction via `@Builder` annotation

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Access properties
String orgId = option.getId();
String orgName = option.getName();

// Modify properties (via generated setters)
option.setName("Acme Corp (Updated)");
```

This DTO is typically used in API responses for populating organization dropdown menus in audit log filter interfaces, providing both the backend identifier and user-friendly display name for each organization option.
