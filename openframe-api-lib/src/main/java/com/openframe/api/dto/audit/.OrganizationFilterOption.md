A simple DTO class representing an organization option used in audit log filtering interfaces, containing basic organization identification and display information.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for boilerplate code generation
- **id** - String field containing the organization's unique identifier
- **name** - String field containing the organization's display name
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a filter dropdown list
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Engineering Dept"),
    new OrganizationFilterOption("org-2", "Marketing Dept")
);

// Access properties
String orgId = option.getId();
String displayName = option.getName();
```
