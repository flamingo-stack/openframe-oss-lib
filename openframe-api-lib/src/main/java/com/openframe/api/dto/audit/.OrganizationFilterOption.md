A simple DTO class that represents an organization option for filtering in the audit logging system, providing the essential organization identifier and display name.

## Key Components

- **id** - String identifier for the organization
- **name** - Display name for the organization
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a filter dropdown list
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-001", "Flamingo MSP"),
    new OrganizationFilterOption("org-002", "Client Company A"),
    new OrganizationFilterOption("org-003", "Client Company B")
);

// Access properties
String orgId = option.getId();
String displayName = option.getName();
```

This DTO is typically used in audit log filtering interfaces where users need to select from available organizations to narrow down log results.
