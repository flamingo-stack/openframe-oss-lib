A DTO class that represents an organization option for filtering audit logs in dropdown components.

## Key Components

- **OrganizationFilterOption**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **id**: String field representing the unique organization identifier
- **name**: String field representing the display name of the organization

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Access fields (Lombok generates getters/setters)
String orgId = option.getId();
String displayName = option.getName();

// Typically used in collections for filter dropdowns
List<OrganizationFilterOption> orgFilters = Arrays.asList(
    new OrganizationFilterOption("org-1", "Organization A"),
    new OrganizationFilterOption("org-2", "Organization B")
);
```
