A data transfer object representing an organization filter option used in audit log filtering interfaces, containing organization ID and display name for dropdown selections.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for boilerplate code generation
- **id** - String field representing the unique organization identifier
- **name** - String field containing the human-readable organization name for display

## Usage Example

```java
// Create organization filter option manually
OrganizationFilterOption option = new OrganizationFilterOption();
option.setId("org-123");
option.setName("Acme Corporation");

// Create using builder pattern
OrganizationFilterOption builderOption = OrganizationFilterOption.builder()
    .id("org-456")
    .name("Tech Solutions Inc")
    .build();

// Create using all-args constructor
OrganizationFilterOption constructorOption = new OrganizationFilterOption(
    "org-789", 
    "Global Industries Ltd"
);

// Typical usage in filtering context
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-001", "Marketing Department"),
    new OrganizationFilterOption("org-002", "Engineering Team")
);
```
