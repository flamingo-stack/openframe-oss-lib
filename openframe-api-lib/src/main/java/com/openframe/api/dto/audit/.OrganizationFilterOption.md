A data transfer object (DTO) representing an organization filter option used in audit log filtering interfaces. This class provides a simple structure for organization dropdown selections with basic identification fields.

## Key Components

- **OrganizationFilterOption** - Main DTO class with Lombok annotations for boilerplate code generation
- **id** - String field representing the unique organization identifier
- **name** - String field for the human-readable organization name
- **@Data, @Builder, @NoArgsConstructor, @AllArgsConstructor** - Lombok annotations providing getters, setters, builder pattern, and constructors

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Use in a list for dropdown population
List<OrganizationFilterOption> organizationOptions = Arrays.asList(
    new OrganizationFilterOption("org-001", "Marketing Dept"),
    new OrganizationFilterOption("org-002", "Engineering Dept")
);

// Access fields
String orgId = option.getId();
String orgName = option.getName();
```
