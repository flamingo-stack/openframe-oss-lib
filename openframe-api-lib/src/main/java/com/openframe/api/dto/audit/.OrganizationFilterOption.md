A simple DTO representing a selectable organization entry used to populate organization filter dropdowns in audit log filtering interfaces.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the organization |
| `name` | `String` | Display name of the organization |

Lombok annotations provide `@Data` (getters/setters/equals/hashCode/toString), `@Builder` (builder pattern), `@NoArgsConstructor`, and `@AllArgsConstructor`.

## Usage Example

```java
// Using the builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Using the all-args constructor
OrganizationFilterOption option = new OrganizationFilterOption("org-123", "Acme Corporation");

// Typical usage in a filter list
List<OrganizationFilterOption> filterOptions = organizations.stream()
    .map(org -> OrganizationFilterOption.builder()
        .id(org.getId())
        .name(org.getName())
        .build())
    .collect(Collectors.toList());
```
