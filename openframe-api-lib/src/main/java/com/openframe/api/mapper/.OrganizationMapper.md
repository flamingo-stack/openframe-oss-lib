A Spring component that handles bidirectional mapping between organization DTOs and entities, supporting both creation and update operations with automatic UUID generation.

## Key Components

- **`toEntity(CreateOrganizationRequest)`** - Converts creation requests to Organization entities with auto-generated UUID
- **`updateEntity(Organization, UpdateOrganizationRequest)`** - Performs partial updates on existing entities (null-safe)
- **`toResponse(Organization)`** - Converts entities to response DTOs
- **Contact Information Mappers** - Handles nested mapping for contacts, addresses, and persons
- **`copyAddress(Address)`** - Duplicates physical address when mailing address is the same

## Usage Example

```java
@Autowired
private OrganizationMapper mapper;

// Create new organization
CreateOrganizationRequest request = CreateOrganizationRequest.builder()
    .name("Acme Corp")
    .category("Technology")
    .numberOfEmployees(100)
    .build();
Organization entity = mapper.toEntity(request);

// Update existing organization (partial update)
UpdateOrganizationRequest updateRequest = UpdateOrganizationRequest.builder()
    .name("Acme Corporation") // Only update name
    .build();
Organization updated = mapper.updateEntity(entity, updateRequest);

// Convert to response DTO
OrganizationResponse response = mapper.toResponse(updated);
```

The mapper automatically handles UUID generation for new organizations and supports partial updates where only non-null fields are modified.
