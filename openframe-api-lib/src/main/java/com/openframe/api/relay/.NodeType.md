Defines the enumeration of all node types used in the OpenFrame GraphQL relay layer, mapping Java enum constants to their corresponding GraphQL type name strings.

## Key Components

| Member | GraphQL Type Name | Description |
|---|---|---|
| `MACHINE` | `"Machine"` | Physical or virtual device node |
| `ORGANIZATION` | `"Organization"` | Organizational entity node |
| `EVENT` | `"Event"` | System or user event node |
| `INTEGRATED_TOOL` | `"IntegratedTool"` | Connected third-party tool node |
| `TENANT` | `"Tenant"` | MSP tenant/client node |
| `DIALOG` | `"Dialog"` | Conversation/chat dialog node |
| `MESSAGE` | `"Message"` | Individual message node |
| `TAG` | `"Tag"` | Classification tag node |
| `TOOL_CONNECTION` | `"ToolConnection"` | Tool integration connection node |
| `INSTALLED_AGENT` | `"InstalledAgent"` | Deployed agent instance node |
| `LOG_EVENT` | `"LogEvent"` | Audit/system log event node |
| `LOG_DETAILS` | `"LogDetails"` | Detailed log entry node |

**Methods:**

- `getGraphqlTypeName()` — Returns the GraphQL type name string for the enum constant
- `fromTypeName(String typeName)` — Resolves a `NodeType` from a GraphQL type name string; throws `IllegalArgumentException` if not found

## Usage Example

```java
// Resolve a NodeType from an incoming GraphQL type string
NodeType type = NodeType.fromTypeName("Tenant");
// type == NodeType.TENANT

// Retrieve the GraphQL type name from an enum constant
String gqlName = NodeType.INSTALLED_AGENT.getGraphqlTypeName();
// gqlName == "InstalledAgent"

// Use in a relay node switch
switch (NodeType.fromTypeName(incomingTypeName)) {
    case MACHINE    -> handleMachine(node);
    case TENANT     -> handleTenant(node);
    case LOG_EVENT  -> handleLogEvent(node);
    default         -> handleGeneric(node);
}
```
