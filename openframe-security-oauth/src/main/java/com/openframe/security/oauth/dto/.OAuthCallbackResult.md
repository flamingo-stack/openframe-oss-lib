A data transfer object that encapsulates the result of an OAuth callback operation, containing tenant information, redirect destination, and authentication tokens.

## Key Components

- **OAuthCallbackResult** - Record class representing the complete OAuth callback response
- **tenantId** - String identifier for the tenant/organization context
- **redirectTo** - String URL where the user should be redirected after successful authentication
- **tokens** - TokenResponse object containing the OAuth tokens (likely access/refresh tokens)

## Usage Example

```java
// Creating an OAuth callback result
TokenResponse tokens = new TokenResponse(accessToken, refreshToken, expiresIn);
OAuthCallbackResult result = new OAuthCallbackResult(
    "tenant-123",
    "https://app.example.com/dashboard",
    tokens
);

// Accessing the components
String tenantId = result.tenantId();
String redirectUrl = result.redirectTo();
TokenResponse authTokens = result.tokens();

// Typical usage in a controller
@RestController
public class OAuthController {
    @PostMapping("/oauth/callback")
    public ResponseEntity<OAuthCallbackResult> handleCallback() {
        OAuthCallbackResult result = oauthService.processCallback();
        return ResponseEntity.ok(result);
    }
}
```
