A data transfer object that encapsulates the result of an OAuth callback operation, containing tenant information, redirect destination, and authentication tokens.

## Key Components

- **tenantId**: String identifier for the tenant associated with the OAuth callback
- **redirectTo**: URL or path where the user should be redirected after successful authentication
- **tokens**: TokenResponse object containing the OAuth authentication tokens (access token, refresh token, etc.)

## Usage Example

```java
// Creating an OAuth callback result after successful authentication
TokenResponse tokens = new TokenResponse(accessToken, refreshToken, expiresIn);
OAuthCallbackResult result = new OAuthCallbackResult(
    "tenant-123",
    "/dashboard", 
    tokens
);

// Accessing the components
String tenantId = result.tenantId();
String redirectUrl = result.redirectTo();
TokenResponse authTokens = result.tokens();

// Typical usage in a controller
@PostMapping("/oauth/callback")
public ResponseEntity<OAuthCallbackResult> handleCallback(
    @RequestParam String code,
    @RequestParam String state
) {
    OAuthCallbackResult result = oauthService.processCallback(code, state);
    return ResponseEntity.ok(result);
}
```

This immutable record provides type-safe encapsulation of OAuth callback data, ensuring all three components are always present and accessible through the automatically generated accessor methods.
