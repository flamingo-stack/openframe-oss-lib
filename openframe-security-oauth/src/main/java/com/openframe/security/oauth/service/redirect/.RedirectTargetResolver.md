A service interface for resolving and validating redirect targets in OAuth flows, ensuring secure redirection based on tenant context and request parameters.

## Key Components

- **RedirectTargetResolver** - Main interface defining redirect target resolution contract
- **resolve()** - Core method that takes tenant ID, requested redirect URL, and HTTP request to return a validated redirect target

## Usage Example

```java
@Component
public class DefaultRedirectTargetResolver implements RedirectTargetResolver {
    
    @Override
    public Mono<String> resolve(String tenantId, String requestedRedirectTo, ServerHttpRequest request) {
        // Validate redirect URL against tenant's allowed URLs
        return validateRedirectUrl(tenantId, requestedRedirectTo)
            .map(isValid -> isValid ? requestedRedirectTo : getDefaultRedirectUrl(tenantId));

// Usage in OAuth controller
@Autowired
private RedirectTargetResolver redirectResolver;

public Mono<String> handleOAuthCallback(String tenantId, String redirectTo, ServerHttpRequest request) {
    return redirectResolver.resolve(tenantId, redirectTo, request);
}
```

This interface provides a reactive approach to redirect URL validation, helping prevent open redirect vulnerabilities in multi-tenant OAuth implementations.
