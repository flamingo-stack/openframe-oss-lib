Extension point interface for contributing forwarded or custom headers to outbound Authorization Server calls from the BFF (Backend for Frontend).

## Key Components

- **`contribute(HttpHeaders, ServerHttpRequest)`** - Main method for adding headers to outbound requests
- **Interface design** - Allows multiple implementations for different header forwarding strategies
- **Idempotent contract** - Implementations must be safe to call multiple times

## Usage Example

```java
@Component
public class XForwardedHeadersContributor implements ForwardedHeadersContributor {
    
    @Override
    public void contribute(HttpHeaders headers, ServerHttpRequest request) {
        if (request != null) {
            // Forward client IP
            String clientIp = request.getRemoteAddress().getAddress().getHostAddress();
            headers.add("X-Forwarded-For", clientIp);
            
            // Forward original host
            String host = request.getHeaders().getFirst("Host");
            if (host != null) {
                headers.add("X-Forwarded-Host", host);
            }
        }
    }
}

// Register multiple contributors
@Bean
public List<ForwardedHeadersContributor> headerContributors() {
    return Arrays.asList(
        new XForwardedHeadersContributor(),
        new CustomAuthHeadersContributor()
    );
}
```

This interface enables flexible header propagation patterns while maintaining clean separation of concerns between different forwarding strategies.
