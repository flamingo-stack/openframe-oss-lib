This interface provides an extension point for adding forwarded or custom headers to outbound calls from the Backend for Frontend (BFF) to the Authorization Server in OAuth flows.

## Key Components

- **`ForwardedHeadersContributor`** - Main interface for contributing headers to outbound OAuth requests
- **`contribute(HttpHeaders, ServerHttpRequest)`** - Method to modify headers based on the current inbound request

## Usage Example

```java
@Component
public class CustomHeadersContributor implements ForwardedHeadersContributor {

    @Override
    public void contribute(HttpHeaders headers, ServerHttpRequest request) {
        // Forward client IP address
        if (request != null) {
            String clientIp = request.getHeaders().getFirst("X-Real-IP");
            if (clientIp != null) {
                headers.add("X-Forwarded-For", clientIp);
            }
        }
        
        // Add custom authentication context
        headers.add("X-BFF-Client", "openframe-web");
        
        // Forward correlation ID for tracing
        String correlationId = request != null ? 
            request.getHeaders().getFirst("X-Correlation-ID") : null;
        if (correlationId != null) {
            headers.add("X-Correlation-ID", correlationId);
        }
    }
}
```

This allows customization of headers sent to OAuth providers while maintaining request traceability and context between the BFF and authorization services.
