A data transfer object (DTO) that represents the response from an OAuth 2.0 token endpoint, containing all standard OAuth token information.

## Key Components

- **TokenResponse**: Record class representing OAuth 2.0 token response structure
- **access_token**: The bearer token used for API authorization
- **refresh_token**: Token used to obtain new access tokens when current token expires
- **token_type**: Type of token (typically "Bearer")
- **expires_in**: Token lifetime in seconds from issuance
- **scope**: Space-delimited list of granted permissions/scopes

## Usage Example

```java
// Creating a token response
TokenResponse tokenResponse = new TokenResponse(
    "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refresh_token_abc123",
    "Bearer",
    3600,
    "read write"
);

// Accessing token information
String accessToken = tokenResponse.access_token();
int expiryTime = tokenResponse.expires_in();
String[] scopes = tokenResponse.scope().split(" ");

// Typical usage in OAuth flow
```
