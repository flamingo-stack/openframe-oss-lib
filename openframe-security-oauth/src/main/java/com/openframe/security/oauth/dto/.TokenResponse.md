A data transfer object (DTO) that represents an OAuth token response containing access credentials and metadata returned from authentication endpoints.

## Key Components

- **TokenResponse record** - Immutable data structure with the following fields:
  - `access_token` - The bearer token for API authentication
  - `refresh_token` - Token used to obtain new access tokens
  - `token_type` - Type of token (typically "Bearer")
  - `expires_in` - Token lifetime in seconds
  - `scope` - Authorized permissions/scopes

## Usage Example

```java
// Creating a token response
TokenResponse response = new TokenResponse(
    "eyJhbGciOiJIUzI1NiIs...", 
    "refresh_abc123",
    "Bearer",
    3600,
    "read write"
);

// Accessing token data
String token = response.access_token();
boolean isExpired = System.currentTimeMillis() > 
    (response.expires_in() * 1000);

// Typical usage in OAuth flow
@PostMapping("/token")
public TokenResponse generateToken(@RequestBody LoginRequest request) {
    // Authentication logic...
    return new TokenResponse(
        jwtService.generateAccessToken(user),
        jwtService.generateRefreshToken(user),
        "Bearer",
        3600,
        user.getScope()
    );
}
```
