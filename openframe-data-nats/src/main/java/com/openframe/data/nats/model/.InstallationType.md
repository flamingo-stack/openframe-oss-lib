Defines the `InstallationType` enum representing the possible installation modes for an OpenFrame deployment or component.

## Key Components

| Value | Description |
|-------|-------------|
| `STANDARD` | Default installation mode |
| `GUI_APP` | Installation as a graphical user interface application |
| `SERVICE` | Installation as a background service/daemon |

## Usage Example

```java
InstallationType type = InstallationType.SERVICE;

switch (type) {
    case STANDARD -> System.out.println("Standard install");
    case GUI_APP  -> System.out.println("GUI application install");
    case SERVICE  -> System.out.println("Background service install");
}
```
