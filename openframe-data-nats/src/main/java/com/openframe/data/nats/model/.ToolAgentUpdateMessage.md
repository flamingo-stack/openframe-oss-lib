Represents a NATS message payload for updating a tool agent, carrying version metadata, session configuration, and a list of asset updates with their download instructions.

## Key Components

| Element | Type | Description |
|---|---|---|
| `toolAgentId` | `String` | Unique identifier of the tool agent to update |
| `version` | `String` | Target version for the tool agent |
| `sessionType` | `SessionType` | Session classification for the agent |
| `downloadConfigurations` | `List<DownloadConfiguration>` | Top-level download sources for the agent |
| `assets` | `List<AssetUpdate>` | Individual asset updates bundled with the message |
| `AssetUpdate` | Inner `@Data` class | Describes a single asset: ID, version, executable flag, and its own download configurations |

## Usage Example

```java
ToolAgentUpdateMessage message = new ToolAgentUpdateMessage();
message.setToolAgentId("agent-123");
message.setVersion("2.1.0");
message.setSessionType(SessionType.REMOTE);

ToolAgentUpdateMessage.AssetUpdate asset = new ToolAgentUpdateMessage.AssetUpdate();
asset.setAssetId("binary-456");
asset.setVersion("2.1.0");
asset.setExecutable(true);
asset.setDownloadConfigurations(List.of(downloadConfig));

message.setAssets(List.of(asset));

// Publish via NATS
natsConnection.publish("tool-agent.update", objectMapper.writeValueAsBytes(message));
```

> Both `ToolAgentUpdateMessage` and `AssetUpdate` use Lombok `@Data`, providing auto-generated getters, setters, `equals`, `hashCode`, and `toString` without boilerplate.
