A NATS message model representing a client connection event, containing a timestamp and associated client information.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `ClientConnectionEvent` | Class | Root event model with Lombok-generated getters/setters |
| `timestamp` | `String` | Time at which the connection event occurred |
| `client` | `Client` | Nested object holding client metadata |
| `Client` | Static Inner Class | Represents the connected client, exposing a `name` field |

## Usage Example

```java
ClientConnectionEvent event = new ClientConnectionEvent();
event.setTimestamp("2024-01-15T10:30:00Z");

ClientConnectionEvent.Client client = new ClientConnectionEvent.Client();
client.setName("acme-corp-device");

event.setClient(client);

System.out.println(event.getTimestamp()); // "2024-01-15T10:30:00Z"
System.out.println(event.getClient().getName()); // "acme-corp-device"
```

> This model is typically deserialized from a NATS subject payload when a client connects to the OpenFrame platform, and consumed by event handlers to trigger downstream automation via Mingo AI or Fae workflows.
