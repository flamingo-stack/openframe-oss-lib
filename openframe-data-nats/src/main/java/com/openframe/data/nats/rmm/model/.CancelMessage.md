Wire payload model sent over NATS to the OpenFrame agent to abort an in-flight execution. Contains the identifier of the execution to cancel.

## Key Components

| Element | Description |
|---------|-------------|
| `executionId` | The unique identifier of the in-flight execution to be aborted |
| `@JsonInclude(NON_NULL)` | Omits null fields from the serialized JSON payload |
| `@Builder` | Enables fluent builder construction via Lombok |
| `@Data` | Generates getters, setters, `equals`, `hashCode`, and `toString` via Lombok |

## Usage Example

```java
// Build and publish a cancellation message over NATS
CancelMessage cancelMessage = CancelMessage.builder()
        .executionId("exec-7f3a1b2c")
        .build();

// Serialize and publish to the appropriate NATS subject
String payload = objectMapper.writeValueAsString(cancelMessage);
natsConnection.publish("openframe.agent.cancel", payload.getBytes());
```
