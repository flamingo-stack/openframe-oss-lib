NATS wire payload model used to request cancellation of an in-flight execution on the OpenFrame agent. Contains a single `executionId` field identifying the target execution to abort.

## Key Components

| Element | Description |
|---|---|
| `executionId` | Unique identifier of the execution to be cancelled |
| `@JsonInclude(NON_NULL)` | Suppresses null fields during JSON serialization |
| `@Builder` / `@Data` | Lombok-generated builder pattern and accessor methods |

## Usage Example

```java
// Build and publish a cancel message over NATS
CancelMessage cancel = CancelMessage.builder()
    .executionId("exec-7f3a1b2c")
    .build();

String json = objectMapper.writeValueAsString(cancel);
natsConnection.publish("openframe.agent.cancel", json.getBytes());
```
