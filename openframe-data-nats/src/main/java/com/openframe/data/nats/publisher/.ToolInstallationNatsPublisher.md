Publishes tool installation messages to NATS for a specific machine, translating `IntegratedToolAgent` and `IntegratedTool` domain objects into a `ToolInstallationMessage` payload.

## Key Components

| Member | Description |
|--------|-------------|
| `TOPIC_NAME_TEMPLATE` | Topic pattern `machine.{machineId}.tool-installation` used to route messages per machine |
| `publish(machineId, toolAgent, tool)` | Publishes an installation message with `reinstall = false` |
| `publish(machineId, toolAgent, tool, reinstall)` | Publishes an installation message with an explicit reinstall flag |
| `buildMessage(...)` | Assembles a `ToolInstallationMessage` from the tool agent, tool metadata, and reinstall flag |
| `mapAssets(...)` | Converts a list of `ToolAgentAsset` objects to `ToolInstallationMessage.Asset` entries |
| `mapAsset(...)` | Maps a single `ToolAgentAsset`, including download configs, local filename, source, path, and executable flag |
| `mapAssetSource(...)` | Translates `ToolAgentAssetSource` enum (`ARTIFACTORY`, `TOOL_API`, `GITHUB`) to the message model equivalent |

> **Note:** This component is conditionally loaded — it requires `spring.cloud.stream.enabled` to be set in application properties.

## Usage Example

```java
// Inject and call from a service
@Autowired
private ToolInstallationNatsPublisher publisher;

// Standard installation
publisher.publish(machineId, toolAgent, tool);

// Force reinstall
publisher.publish(machineId, toolAgent, tool, true);
// Publishes to: machine.<machineId>.tool-installation
```
