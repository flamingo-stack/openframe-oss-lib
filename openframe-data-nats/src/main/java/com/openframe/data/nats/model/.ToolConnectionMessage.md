A simple data transfer object (DTO) used to carry tool connection information over NATS messaging in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolType` | `String` | Identifies the category or kind of tool being connected |
| `agentToolId` | `String` | Unique identifier for the specific agent tool instance |

Lombok annotations `@Getter` and `@Setter` auto-generate accessor methods at compile time, keeping the class concise.

## Usage Example

```java
// Creating and publishing a tool connection message
ToolConnectionMessage message = new ToolConnectionMessage();
message.setToolType("remote-access");
message.setAgentToolId("agent-tool-789");

// Serialize and publish via NATS
String payload = objectMapper.writeValueAsString(message);
natsConnection.publish("tool.connection.events", payload.getBytes());

// Deserializing an incoming NATS message
ToolConnectionMessage received = objectMapper.readValue(
    new String(natsMsg.getData()),
    ToolConnectionMessage.class
);
String toolType = received.getToolType();
String agentToolId = received.getAgentToolId();
```
