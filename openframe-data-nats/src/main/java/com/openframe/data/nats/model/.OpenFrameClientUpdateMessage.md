A NATS message model representing a client update payload for the OpenFrame platform, containing version information and a list of download configurations.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `version` | `String` | The target version string for the client update |
| `downloadConfigurations` | `List<DownloadConfiguration>` | List of download configuration entries associated with the update |

Lombok `@Getter` and `@Setter` annotations are used to auto-generate accessors, keeping the model concise.

## Usage Example

```java
OpenFrameClientUpdateMessage updateMessage = new OpenFrameClientUpdateMessage();
updateMessage.setVersion("1.4.2");

DownloadConfiguration config = new DownloadConfiguration();
// configure config fields...

updateMessage.setDownloadConfigurations(List.of(config));

// Publish via NATS
natsConnection.publish("openframe.client.update", serialize(updateMessage));
```
