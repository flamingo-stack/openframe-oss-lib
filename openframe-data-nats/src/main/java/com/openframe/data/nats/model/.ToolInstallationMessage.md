NATS message model representing a tool installation request payload, including configuration for downloading, installing, running, and uninstalling a tool on a target agent.

## Key Components

### Main Class: `ToolInstallationMessage`

| Field | Type | Description |
|-------|------|-------------|
| `toolAgentId` | `String` | Target tool agent identifier |
| `toolId` | `String` | Unique tool identifier |
| `toolType` | `String` | Category/type of the tool |
| `version` | `String` | Tool version to install |
| `sessionType` | `SessionType` | Session context for installation |
| `downloadConfigurations` | `List<DownloadConfiguration>` | Download source configs |
| `assets` | `List<Asset>` | Associated binaries or files |
| `installationCommandArgs` | `List<String>` | Args for install command |
| `runCommandArgs` | `List<String>` | Args for run command |
| `toolAgentIdCommandArgs` | `List<String>` | Args to identify the agent |
| `uninstallationCommandArgs` | `List<String>` | Args for uninstall command |
| `reinstall` | `boolean` | Forces reinstall if `true` (default: `false`) |

### Nested Classes & Enums

- **`Asset`** — Represents a downloadable binary or file tied to the tool, including OS-specific filename mapping, source, path, and executable flag.
- **`LocalFilenameConfiguration`** — Maps a filename to a target OS string for platform-specific asset naming.
- **`AssetSource`** — Enum defining supported asset download origins: `ARTIFACTORY`, `TOOL_API`, `GITHUB`.

## Usage Example

```java
ToolInstallationMessage message = new ToolInstallationMessage();
message.setToolAgentId("agent-001");
message.setToolId("nmap");
message.setToolType("network-scanner");
message.setVersion("7.94");
message.setSessionType(SessionType.REMOTE);
message.setReinstall(false);

ToolInstallationMessage.Asset asset = new ToolInstallationMessage.Asset();
asset.setId("nmap-binary");
asset.setSource(ToolInstallationMessage.AssetSource.GITHUB);
asset.setExecutable(true);
asset.setPath("/usr/local/bin/nmap");

message.setAssets(List.of(asset));
```
