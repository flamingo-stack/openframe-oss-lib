A NATS message model representing an installed agent notification, used to communicate agent installation events across the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `machineId` | `String` | Unique identifier of the machine where the agent is installed |
| `agentType` | `String` | Type/category of the installed agent |
| `version` | `String` | Version string of the installed agent |

The class uses Lombok's `@Data` annotation, which auto-generates getters, setters, `equals()`, `hashCode()`, and `toString()` at compile time.

## Usage Example

```java
// Publishing an installed agent event via NATS
InstalledAgentMessage message = new InstalledAgentMessage();
message.setMachineId("machine-abc-123");
message.setAgentType("monitoring");
message.setVersion("2.1.0");

// Serialize and publish to NATS subject
String payload = objectMapper.writeValueAsString(message);
natsConnection.publish("agents.installed", payload.getBytes());

// Deserializing an incoming NATS message
InstalledAgentMessage received = objectMapper
    .readValue(natsMsg.getData(), InstalledAgentMessage.class);

System.out.println(received.getMachineId());  // "machine-abc-123"
System.out.println(received.getAgentType());  // "monitoring"
System.out.println(received.getVersion());    // "2.1.0"
```
