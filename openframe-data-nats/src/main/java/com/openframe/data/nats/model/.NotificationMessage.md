A NATS message model representing a notification payload transmitted across the OpenFrame messaging infrastructure.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the notification |
| `severity` | `NotificationSeverity` | Severity level of the notification |
| `title` | `String` | Short display title |
| `description` | `String` | Detailed notification body |
| `createdAt` | `Instant` | UTC timestamp of creation |
| `context` | `NotificationContext` | Contextual metadata associated with the notification |

## Usage Example

```java
NotificationMessage message = NotificationMessage.builder()
    .id(UUID.randomUUID().toString())
    .severity(NotificationSeverity.HIGH)
    .title("Disk Usage Alert")
    .description("Disk usage has exceeded 90% on server prod-01.")
    .createdAt(Instant.now())
    .context(NotificationContext.builder()
        .tenantId("tenant-abc")
        .build())
    .build();

// Publish via NATS
natsConnection.publish("notifications.alerts", serialize(message));
```

> Lombok annotations (`@Builder`, `@Getter`, `@Setter`, `@NoArgsConstructor`, `@AllArgsConstructor`) are used to reduce boilerplate, making this class suitable for both builder-pattern construction and deserialization frameworks.
