A Spring component that publishes RMM (Remote Monitoring & Management) ad-hoc command and cancellation messages to target agents over core NATS messaging.

## Key Components

| Member | Description |
|--------|-------------|
| `MACHINE_SUBJECT_TEMPLATE` | NATS subject pattern `machine.{machineId}.command-execution` |
| `CANCEL_SUBJECT_TEMPLATE` | NATS subject pattern `machine.{machineId}.command-cancel` |
| `publishCommand()` | Publishes a `CommandMessage` to the target agent for execution |
| `publishCancel()` | Publishes a `CancelMessage` to abort an in-flight execution |

**Activation:** Only loaded when `spring.cloud.stream.enabled` property is set.

**Validation:** Both methods guard against blank `machineId` and null/blank `executionId`, throwing `IllegalArgumentException` on violation.

## Usage Example

```java
@Autowired
private CommandNatsPublisher commandNatsPublisher;

// Publish a command to a remote agent
CommandMessage command = CommandMessage.builder()
    .executionId("exec-001")
    .script("ipconfig /all")
    .build();

commandNatsPublisher.publishCommand("machine-abc123", command);
// → publishes to: machine.machine-abc123.command-execution

// Cancel an in-flight execution
CancelMessage cancel = CancelMessage.builder()
    .executionId("exec-001")
    .build();

commandNatsPublisher.publishCancel("machine-abc123", cancel);
// → publishes to: machine.machine-abc123.command-cancel
```
