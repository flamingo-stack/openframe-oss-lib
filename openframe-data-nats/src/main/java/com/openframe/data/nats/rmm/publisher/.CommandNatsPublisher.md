A Spring component that publishes RMM (Remote Monitoring & Management) ad-hoc command and cancellation messages to specific agents over core NATS messaging.

## Key Components

| Member | Description |
|--------|-------------|
| `MACHINE_SUBJECT_TEMPLATE` | NATS subject pattern `machine.{machineId}.command-execution` for command dispatch |
| `CANCEL_SUBJECT_TEMPLATE` | NATS subject pattern `machine.{machineId}.command-cancel` for cancellation requests |
| `publishCommand()` | Publishes a `CommandMessage` to a target agent identified by `machineId` |
| `publishCancel()` | Publishes a `CancelMessage` to abort an in-flight execution on a target agent |

**Activation:** Only registered as a bean when `spring.cloud.stream.enabled=true`.

**Validation:** Both methods enforce non-blank `machineId` and non-null message with a valid `executionId`, throwing `IllegalArgumentException` otherwise.

## Usage Example

```java
// Dispatch a command to a remote agent
CommandMessage cmd = new CommandMessage();
cmd.setExecutionId("exec-001");
cmd.setCommand("ipconfig /all");

commandNatsPublisher.publishCommand("machine-abc123", cmd);
// Publishes to: machine.machine-abc123.command-execution

// Cancel an in-flight execution on the same agent
CancelMessage cancel = new CancelMessage();
cancel.setExecutionId("exec-001");

commandNatsPublisher.publishCancel("machine-abc123", cancel);
// Publishes to: machine.machine-abc123.command-cancel
```

> Both methods delegate to `NatsMessagePublisher.publish()` and log the `executionId`, `machineId`, and resolved NATS subject on success. A `NatsException` is propagated if the underlying publish fails.
