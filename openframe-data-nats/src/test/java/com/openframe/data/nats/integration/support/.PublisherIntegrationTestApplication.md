Minimal Spring Boot test application context used to bootstrap NATS publisher integration tests. It wires together the Jackson serialization config and the two NATS publisher beans without loading the full application.

## Key Components

| Element | Description |
|---|---|
| `@SpringBootConfiguration` | Marks this as a test-scoped Spring Boot configuration root |
| `@EnableAutoConfiguration` | Activates Spring Boot's auto-configuration for the test slice |
| `NotificationContextJacksonConfig` | Imported Jackson config for notification context serialization |
| `NatsMessagePublisher` | Core NATS message publishing bean |
| `NotificationNatsPublisher` | Higher-level notification-specific NATS publisher bean |

## Usage Example

```java
@SpringBootTest(classes = PublisherIntegrationTestApplication.class)
class NotificationNatsPublisherIT {

    @Autowired
    private NotificationNatsPublisher publisher;

    @Test
    void shouldPublishNotification() {
        // test against a real or embedded NATS server
        publisher.publish(new NotificationEvent(...));
    }
}
```

> This class is intentionally empty — its sole purpose is to act as a lightweight Spring context anchor for integration tests, importing only the beans required to exercise the NATS publisher stack.
