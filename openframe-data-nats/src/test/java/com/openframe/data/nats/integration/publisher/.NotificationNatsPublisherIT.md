Integration test suite for `NotificationNatsPublisher`, verifying that notifications are correctly routed and delivered over NATS subjects using the Spring Cloud Stream NATS binder.

## Key Components

| Component | Description |
|---|---|
| `NotificationNatsPublisher` | The publisher under test — publishes notifications to user/machine NATS subjects |
| `BaseIntegrationTest` | Base class providing `natsUri()` and shared NATS infrastructure |
| `NotificationMessage` | Deserialized payload asserted on received NATS messages |
| `GenericContext` | Notification context type carrying `type` and `payload` fields |
| `subscriber` | Raw `io.nats.client.Connection` used to assert message delivery per test |

## Test Coverage

| Test | Scenario |
|---|---|
| `publish_to_user_delivers_payload` | Full payload round-trip: id, title, severity, context deserialized correctly |
| `per_user_subjects_isolated` | Publishing to `bob` does not leak to `alice`'s subject |
| `publish_to_machine_delivers_to_machine_subject` | Machine-scoped subjects route exclusively to the correct subscriber |
| `unsaved_notification_throws` | `IllegalArgumentException` raised when notification has no id (not yet persisted) |

## Usage Example

```java
// Guard: only runs when -Dintegration.tests=true is passed to the JVM
// mvn verify -Dintegration.tests=true

// Each test subscribes directly via NATS client, then asserts on the raw message
Subscription sub = subscriber.subscribe("user.alice.notification");
subscriber.flush(Duration.ofSeconds(2));

publisher.publishToUser("alice", savedNotification);

Message received = sub.nextMessage(Duration.ofSeconds(5));
NotificationMessage decoded = objectMapper.readValue(received.getData(), NotificationMessage.class);
assertThat(decoded.getId()).isEqualTo(savedNotification.getId());
```

> **Note:** Tests are disabled by default. Enable with `-Dintegration.tests=true`. Notifications must be persisted (have a non-null `id`) before publishing — the publisher enforces this contract via `IllegalArgumentException`.
