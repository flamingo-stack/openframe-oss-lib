Unit test suite validating the construction and validation logic of `NotificationCommand`, a builder-pattern command object used to dispatch notifications via NATS in the OpenFrame platform.

## Key Components

| Test Method | Validates |
|---|---|
| `valid_command_builds_and_carries_fields` | Full happy-path build with all fields populated |
| `blank_title_rejected` / `null_title_rejected` | Title is required and must be non-blank |
| `null_severity_rejected` | Severity is required for UI styling |
| `null_context_rejected` / `blank_context_type_rejected` | Context and its `type` field are required for polymorphic dispatch |
| `empty_audiences_rejected` | At least one non-empty audience must be supplied |
| `admin_only_audience_valid` / `machine_only_audience_valid` | Either audience alone satisfies the requirement; the other defaults to empty |
| `audience_snapshot_isolated_from_caller_mutation` | Internal defensive copy prevents post-build mutation |
| `blank_admin/machine_audience_entry_rejected` | Blank IDs in audience sets are rejected to prevent malformed NATS subjects |
| `null_audience_normalizes_to_empty` | `null` audience sets are normalized to empty sets without NPE |

## Usage Example

```java
// Valid command with both audiences
NotificationCommand cmd = NotificationCommand.builder()
    .title("Approval required")
    .description("Please review the pending request")
    .severity(NotificationSeverity.INFO)
    .context(GenericContext.builder()
        .type("APPROVAL")
        .payload("{\"ticketId\":\"T-123\"}")
        .build())
    .adminAudience(Set.of("admin-1", "admin-2"))
    .machineAudience(Set.of("device-abc"))
    .build();

// Minimal valid command — admin audience only
NotificationCommand minimal = NotificationCommand.builder()
    .title("Alert")
    .severity(NotificationSeverity.WARNING)
    .context(GenericContext.builder().type("ALERT").payload("{}").build())
    .adminAudience(Set.of("admin-1"))
    .build();
```

## Validation Rules Enforced

- `title` — must be non-null and non-blank
- `severity` — must be non-null
- `context` — must be non-null; `context.type` must be non-blank
- `adminAudience` / `machineAudience` — at least one must be non-empty; individual entries must not be blank; `null` sets normalize to empty
- Audience sets are defensively copied at build time, isolating the command from caller mutations
