Provides reusable test fixture factory methods for creating pre-configured `Notification` instances in NATS integration tests.

## Key Components

- **`NotificationFixtures`** — Non-instantiable utility class (private constructor) containing static factory methods for test data.
- **`basic()`** — Returns a minimal `Notification` with a title, current timestamp, and a `GenericContext` of type `"welcome"` with a sample JSON payload.

## Usage Example

```java
// Create a basic notification fixture for use in integration tests
Notification notification = NotificationFixtures.basic();

// Use in assertions or as NATS message payload
assertNotNull(notification.getTitle());
assertEquals("welcome", notification.getContext().getType());
```
