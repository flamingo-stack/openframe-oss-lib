Defines the shared base URL constant for all content hub API endpoints, derived from the single canonical prefix source.

## Key Components

- **`CONTENT`** — Exported string constant combining `CONTENT_PREFIX` with `/api`, used as the root path for every hub endpoint call.

## Usage Example

```typescript
import { CONTENT } from './content'

const response = await fetch(`${CONTENT}/tickets`)
```

## Notes

The prefix is sourced exclusively from `proxy/content-prefix.mjs`, ensuring a namespace change requires only a single edit site-wide. The proxy layer maps these `CONTENT_PREFIX/api/*` paths back to `/api/*` on the server.
