Shared UI component for displaying error states on route pages using ODS design tokens.

## Key Components

- **`PageError`** — A styled error display component that renders a primary error title and an optional detail message.

## Usage Example

```typescript
import { PageError } from './page-state'

// Basic error
<PageError title="Something went wrong" />

// With detail message
<PageError
  title="Failed to load data"
  detail="The requested resource could not be found."
/>
```

## Notes

- Uses ODS design tokens (`ods-text-primary`, `ods-text-secondary`) — no hardcoded colors.
- The `detail` prop is optional; if omitted, no secondary paragraph is rendered.
