Renders the public **Contact Us** page by mounting the self-contained `ContactForm` component from the OpenFrame frontend core library.

## Key Components

- **`ContactPage`** — Page-level wrapper that provides layout padding and renders `ContactForm` with pre-configured title and subtitle props.
- **`ContactForm`** *(from `@flamingo-stack/openframe-frontend-core`)* — Fully self-contained form component; internally calls `useContactSubmission`, which resolves the submission endpoint via the mounted `EndpointsRuntime` provider. No host-side hooks or schema imports are required.

## Usage Example

```typescript
// Route this component from your page router (e.g. Next.js / React Router)
import { ContactPage } from './contact'

// In your router config:
{ path: '/contact', element: <ContactPage /> }
```

To customise the heading copy, edit the `title` and `subtitle` props directly:

```typescript
<ContactForm
  title="Get in touch"
  subtitle="Questions about OpenFrame? Send us a note."
/>
```

> **Note:** `ContactPage` relies on an `EndpointsRuntime` provider being present higher in the React tree. Ensure it is mounted at the application root so `useContactSubmission` can resolve `EP.contact` correctly.
