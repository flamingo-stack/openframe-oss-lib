Navigation hub component for the OpenFrame embedding demo app, rendering a grid of links to all available surfaces/routes with their descriptions.

## Key Components

- **`SURFACES`** — Static array of route descriptors (`to`, `title`, `desc`) covering all demo surfaces
- **`HomePage`** — Default export component rendering the landing page grid

## Usage Example

```typescript
import { HomePage } from './home'

// Used as the index route in your router config
const router = createBrowserRouter([
  { path: '/', element: <HomePage /> },
  // ... other surface routes
])
```

## Surface Routes

| Route | Surface |
|-------|---------|
| `/onboarding-guides` | Onboarding guides catalog |
| `/roadmap` | Voting grid |
| `/delivery` | Bug-fix & enhancement tables |
| `/releases` | Product releases list + detail |
| `/legal/privacy` | Privacy / Terms pages |
| `/contact` | Contact form |
| `/tickets` | Help center / tickets |

> Each card links to a real `@flamingo-stack/openframe-frontend-core` component proxied through `/content`, which injects the chat secret and a fixed identity for the floating **Ask AI** widget.
