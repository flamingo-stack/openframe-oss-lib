Provides static factory methods for creating pre-built `Notification` test fixtures for use in integration tests.

## Key Components

| Method | Description |
|--------|-------------|
| `basic()` | Creates a default notification with title `"Welcome aboard"` and a `welcome` context type |
| `basic(String type)` | Creates a notification using the provided `type` as both the title and `GenericContext` type |

Both methods set `createdAt` to `Instant.now()` and use an empty JSON payload (`{}`).

## Usage Example

```java
// Default welcome notification
Notification notification = NotificationFixtures.basic();

// Custom-typed notification
Notification alertNotification = NotificationFixtures.basic("alert");
```
