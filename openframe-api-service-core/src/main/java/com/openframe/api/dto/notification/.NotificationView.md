Immutable DTO representing a notification item returned to API consumers, built via Lombok's `@Builder` pattern.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique notification identifier |
| `severity` | `NotificationSeverity` | Severity level of the notification |
| `title` | `String` | Short notification title |
| `description` | `String` | Detailed notification message |
| `createdAt` | `Instant` | UTC timestamp of creation |
| `context` | `NotificationContext` | Contextual metadata attached to the notification |
| `read` | `boolean` | Read/unread status flag |

## Usage Example

```java
NotificationView view = NotificationView.builder()
    .id("notif-123")
    .severity(NotificationSeverity.WARNING)
    .title("Disk space low")
    .description("Server disk usage exceeded 90%.")
    .createdAt(Instant.now())
    .context(notificationContext)
    .read(false)
    .build();
```
