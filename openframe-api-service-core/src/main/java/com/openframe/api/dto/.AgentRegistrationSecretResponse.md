A DTO class that represents the response data for agent registration secrets, containing the secret's identification, key, creation timestamp, and status information.

## Key Components

- **`id`** - Unique identifier for the registration secret
- **`key`** - The actual secret key value used for agent registration
- **`createdAt`** - Timestamp when the secret was generated
- **`active`** - Boolean flag indicating if the secret is currently valid for use

## Usage Example

```java
// Building a response object
AgentRegistrationSecretResponse response = AgentRegistrationSecretResponse.builder()
    .id("secret-123")
    .key("abcd1234-secret-key-5678efgh")
    .createdAt(Instant.now())
    .active(true)
    .build();

// Accessing response data
String secretId = response.getId();
String registrationKey = response.getKey();
boolean isSecretActive = response.isActive();
Instant whenCreated = response.getCreatedAt();

// Typical usage in API controller
@GetMapping("/registration-secret")
public AgentRegistrationSecretResponse getRegistrationSecret() {
    return secretService.generateNewSecret();
}
```

This immutable response object is typically used in REST API endpoints to provide agents with the necessary credentials for registration while including metadata about the secret's validity and creation time.
