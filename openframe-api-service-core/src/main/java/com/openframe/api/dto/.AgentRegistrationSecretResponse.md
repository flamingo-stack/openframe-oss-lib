Data Transfer Object (DTO) that represents the response containing agent registration secret information, typically used in API responses for agent authentication setup.

## Key Components

- **id** - Unique identifier for the registration secret
- **key** - The actual secret key used for agent registration
- **createdAt** - Timestamp when the secret was created
- **active** - Boolean flag indicating if the secret is currently active

## Usage Example

```java
// Creating a registration secret response
AgentRegistrationSecretResponse response = AgentRegistrationSecretResponse.builder()
    .id("secret-123")
    .key("abc-def-ghi-jkl")
    .createdAt(Instant.now())
    .active(true)
    .build();

// Accessing response data
String secretId = response.getId();
String registrationKey = response.getKey();
boolean isActive = response.isActive();
Instant whenCreated = response.getCreatedAt();

// Typical API controller usage
@GetMapping("/agent/registration-secret/{id}")
public ResponseEntity<AgentRegistrationSecretResponse> getRegistrationSecret(@PathVariable String id) {
    AgentRegistrationSecretResponse secret = agentService.getRegistrationSecret(id);
    return ResponseEntity.ok(secret);
}
```
