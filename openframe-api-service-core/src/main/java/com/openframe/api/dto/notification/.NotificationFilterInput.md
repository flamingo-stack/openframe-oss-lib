Defines a Data Transfer Object (DTO) used to filter notifications by their read/unread status when querying the notifications API.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `read` | `Boolean` | Filter flag; `true` for read notifications, `false` for unread, `null` to return all |

## Usage Example

```java
// Filter for unread notifications only
NotificationFilterInput filter = NotificationFilterInput.builder()
    .read(false)
    .build();

// Filter for read notifications
NotificationFilterInput readFilter = new NotificationFilterInput(true);

// No filter applied (returns all notifications)
NotificationFilterInput noFilter = new NotificationFilterInput();
```

> **Note:** The `read` field is nullable (`Boolean` wrapper type), allowing callers to omit the filter entirely and retrieve all notifications regardless of read status.
