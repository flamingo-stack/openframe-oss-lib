A simple DTO representing the count of items assigned to a specific assignment target type.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `targetType` | `AssignmentTargetType` | The type of assignment target (e.g., user, group, device) |
| `count` | `int` | Number of items assigned to that target type |

## Usage Example

```java
// Building an AssignedItemCount instance
AssignedItemCount itemCount = AssignedItemCount.builder()
    .targetType(AssignmentTargetType.USER)
    .count(42)
    .build();

// Accessing fields
AssignmentTargetType type = itemCount.getTargetType();
int total = itemCount.getCount();
```

## Notes

- Uses Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) to auto-generate boilerplate (getters, setters, constructors, builder pattern).
- Typically used in assignment-related API responses to summarize how many items are assigned per target type.
