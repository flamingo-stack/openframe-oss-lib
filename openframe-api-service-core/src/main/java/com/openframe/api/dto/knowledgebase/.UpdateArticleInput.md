Data Transfer Object (DTO) for updating an existing knowledge base article, carrying the fields that can be modified in an update operation.

## Key Components

| Field | Type | Constraint | Description |
|-------|------|-----------|-------------|
| `id` | `String` | `@NotBlank` | Required unique identifier of the article to update |
| `name` | `String` | — | Updated article title/name |
| `parentId` | `String` | — | Updated parent node reference for hierarchy |
| `content` | `String` | — | Updated article body content |
| `summary` | `String` | — | Updated brief description of the article |

## Usage Example

```java
// Build via Lombok builder
UpdateArticleInput input = UpdateArticleInput.builder()
    .id("article-uuid-123")
    .name("Updated Article Title")
    .parentId("parent-uuid-456")
    .content("Updated article body content...")
    .summary("A brief summary of the updated article.")
    .build();

// Or via all-args constructor
UpdateArticleInput input = new UpdateArticleInput(
    "article-uuid-123",
    "Updated Article Title",
    "parent-uuid-456",
    "Updated article body content...",
    "A brief summary."
);
```

> Only `id` is required (`@NotBlank`). All other fields are optional — pass `null` to leave them unchanged in the update handler.
