A data transfer object representing SSO configuration settings for authentication providers, with validation constraints and auto-provisioning controls.

## Key Components

- **clientId** - Required OAuth client identifier with validation
- **clientSecret** - Required OAuth client secret with validation  
- **autoProvisionUsers** - Boolean flag controlling automatic user creation
- **msTenantId** - Optional Microsoft tenant identifier for Azure AD integration
- **allowedDomains** - List of email domains permitted for auto-provisioning

## Usage Example

```java
// Create SSO configuration for Azure AD
SSOConfigRequest ssoConfig = new SSOConfigRequest();
ssoConfig.setClientId("your-azure-client-id");
ssoConfig.setClientSecret("your-azure-client-secret");
ssoConfig.setAutoProvisionUsers(true);
ssoConfig.setMsTenantId("your-tenant-id");
ssoConfig.setAllowedDomains(List.of("company.com", "subsidiary.com"));

// Use in API call
ssoService.configureSSOProvider(ssoConfig);
```

The class uses Lombok's `@Data` annotation for automatic getter/setter generation and Jakarta validation annotations to ensure required fields are provided.
