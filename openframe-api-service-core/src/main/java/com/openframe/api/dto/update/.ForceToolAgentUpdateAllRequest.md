A data transfer object (DTO) that represents a request to force an update operation on all instances of a specific tool agent in the OpenFrame API system.

## Key Components

- **toolAgentId** - String field containing the unique identifier of the tool agent to be updated across all instances
- **@Data** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a request to force update all instances of a tool agent
ForceToolAgentUpdateAllRequest request = new ForceToolAgentUpdateAllRequest();
request.setToolAgentId("agent-123");

// Or using constructor if available through Lombok
ForceToolAgentUpdateAllRequest request = new ForceToolAgentUpdateAllRequest();
request.setToolAgentId("my-tool-agent-uuid");

// The request can then be sent to the appropriate API endpoint
// to trigger updates across all instances of the specified tool agent
```

This simple DTO follows the standard pattern for API request objects, encapsulating the minimal data needed to identify which tool agent should be updated across all its instances in the system.
