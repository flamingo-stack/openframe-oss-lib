A simple data transfer object (DTO) used to request a forced update of all components for a specific tool agent in the OpenFrame API system.

## Key Components

- **toolAgentId** (String) - The unique identifier of the tool agent that requires a forced update of all its components

## Usage Example

```java
// Create a force update request for a specific tool agent
ForceToolAgentUpdateAllRequest request = new ForceToolAgentUpdateAllRequest();
request.setToolAgentId("agent-123");

// Or using constructor if available via Lombok
ForceToolAgentUpdateAllRequest request = new ForceToolAgentUpdateAllRequest();
request.setToolAgentId("my-tool-agent-uuid");

// The request can then be sent to the appropriate API endpoint
// to trigger a forced update of all components for the specified agent
```

This DTO is typically used in REST API endpoints where administrators need to force a comprehensive update of a tool agent's configuration, dependencies, or runtime components without waiting for scheduled updates.
