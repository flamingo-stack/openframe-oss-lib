A data transfer object that encapsulates Single Sign-On (SSO) configuration details returned from API endpoints. This class represents the complete SSO setup for various identity providers with security and user management settings.

## Key Components

- **Core Identity Fields**: `id`, `provider`, `clientId`, `clientSecret` for OAuth/OIDC integration
- **Microsoft-Specific**: `msTenantId` for Azure AD/Microsoft 365 configurations  
- **User Management**: `autoProvisionUsers` flag for automatic user creation
- **Security Controls**: `enabled` status and `allowedDomains` whitelist
- **Lombok Annotations**: `@Data` for getters/setters/equals/hashCode, `@Builder` for fluent object creation

## Usage Example

```java
// Building an SSO configuration response
SSOConfigResponse ssoConfig = SSOConfigResponse.builder()
    .id("sso-config-123")
    .provider("microsoft")
    .clientId("app-client-id")
    .clientSecret("secret-key")
    .msTenantId("tenant-uuid")
    .autoProvisionUsers(true)
    .enabled(true)
    .allowedDomains(List.of("company.com", "subsidiary.com"))
    .build();

// Accessing configuration properties
if (ssoConfig.isEnabled() && ssoConfig.getAllowedDomains().contains("user@company.com")) {
    // Proceed with SSO authentication
}
```
