A data transfer object (DTO) record representing the response structure for release version information in the OpenFrame API.

## Key Components

- **Record Fields**:
  - `id` - Unique identifier for the release version
  - `releaseVersion` - The actual version string (e.g., "1.0.0")
  - `createdAt` - Timestamp when the release version was created
  - `updatedAt` - Timestamp when the release version was last modified

## Usage Example

```java
// Creating a release version response
ReleaseVersionResponse response = new ReleaseVersionResponse(
    "rv-123456",
    "2.1.0",
    Instant.now().minusSeconds(3600),
    Instant.now()
);

// Accessing fields (auto-generated getters)
String version = response.releaseVersion(); // "2.1.0"
String id = response.id(); // "rv-123456"
Instant created = response.createdAt();

// JSON serialization (when used with REST endpoints)
// {
//   "id": "rv-123456",
//   "releaseVersion": "2.1.0", 
//   "createdAt": "2024-01-15T10:30:00Z",
//   "updatedAt": "2024-01-15T11:30:00Z"
// }
```

This immutable record provides a clean, type-safe way to transfer release version data between API layers, with automatic implementations of `equals()`, `hashCode()`, and `toString()` methods.
