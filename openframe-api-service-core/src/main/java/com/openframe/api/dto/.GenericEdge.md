A generic edge data transfer object used in GraphQL-style pagination to wrap nodes with cursor information for efficient traversal of result sets.

## Key Components

- **Generic Type Parameter**: `<T>` allows wrapping any type of node object
- **Node Field**: `T node` - the actual data object being paginated
- **Cursor Field**: `String cursor` - pagination cursor for efficient traversal
- **Lombok Annotations**: Provides automatic getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a generic edge for a User object
GenericEdge<User> userEdge = GenericEdge.<User>builder()
    .node(new User("john_doe", "John Doe"))
    .cursor("eyJpZCI6MTIzfQ==")
    .build();

// Using in a pagination response
List<GenericEdge<Product>> productEdges = products.stream()
    .map(product -> GenericEdge.<Product>builder()
        .node(product)
        .cursor(encodeCursor(product.getId()))
        .build())
    .collect(Collectors.toList());

// Accessing the data
User user = userEdge.getNode();
String nextCursor = userEdge.getCursor();
```

This class follows the GraphQL Relay specification for connection-based pagination, where each edge contains both the data node and a cursor string used for requesting subsequent pages of results.
