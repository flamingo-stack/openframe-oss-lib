A data transfer object representing an edge in a GraphQL-style paginated response, containing a node of generic type and its cursor position.

## Key Components

- **GenericEdge<T>** - Generic edge class that wraps any node type with pagination metadata
- **node** - The actual data item of type T
- **cursor** - String identifier for the node's position in the paginated result set

## Usage Example

```java
// Create an edge for a User object
GenericEdge<User> userEdge = GenericEdge.<User>builder()
    .node(new User("john@example.com", "John Doe"))
    .cursor("eyJpZCI6MTIzfQ==")
    .build();

// Use in a paginated response
List<GenericEdge<User>> edges = Arrays.asList(
    GenericEdge.<User>builder()
        .node(user1)
        .cursor("cursor1")
        .build(),
    GenericEdge.<User>builder()
        .node(user2)
        .cursor("cursor2")
        .build()
);

// Access the data
User userData = userEdge.getNode();
String position = userEdge.getCursor();
```

This class is typically used in GraphQL pagination implementations where each item in a connection needs both the data and its cursor for subsequent page requests.
