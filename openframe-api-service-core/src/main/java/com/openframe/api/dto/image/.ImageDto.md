A simple data transfer object (DTO) for representing image information, containing the image URL and its hash value for identification and integrity verification.

## Key Components

- **imageUrl** - String field containing the URL or path to the image resource
- **hash** - String field containing a hash value (typically for content verification or caching)
- **@Data** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods
- **@Builder** - Lombok annotation that provides a builder pattern for object construction

## Usage Example

```java
// Creating an ImageDto using the builder pattern
ImageDto imageDto = ImageDto.builder()
    .imageUrl("https://example.com/images/photo.jpg")
    .hash("a1b2c3d4e5f6")
    .build();

// Accessing the properties
String url = imageDto.getImageUrl();
String hash = imageDto.getHash();

// Using in API responses or data transfer
public ImageDto getImageInfo(Long imageId) {
    // Fetch image details from service/repository
    return ImageDto.builder()
        .imageUrl(imageService.getImageUrl(imageId))
        .hash(imageService.calculateHash(imageId))
        .build();
}
```

This DTO is commonly used in REST APIs for transferring image metadata between different layers of the application or to external clients.
