A simple data transfer object representing image metadata with URL and hash information for the OpenFrame API.

## Key Components

- **imageUrl** - String field containing the URL path or reference to the image
- **hash** - String field containing a hash value for image identification or integrity verification
- **@Data** - Lombok annotation providing getters, setters, toString, equals, and hashCode methods
- **@Builder** - Lombok annotation enabling the builder pattern for object construction

## Usage Example

```java
// Creating an ImageDto using the builder pattern
ImageDto image = ImageDto.builder()
    .imageUrl("/api/images/photo123.jpg")
    .hash("a1b2c3d4e5f6")
    .build();

// Accessing image data
String url = image.getImageUrl();
String hashValue = image.getHash();

// Using in API responses or data transfer
public ImageDto getImageMetadata(String imageId) {
    return ImageDto.builder()
        .imageUrl(generateImageUrl(imageId))
        .hash(calculateImageHash(imageId))
        .build();
}
```

This DTO is typically used for transferring image metadata between API layers, providing a clean separation between internal image representation and external API contracts.
