Data Transfer Object (DTO) for capturing input when creating a temporary knowledge base attachment, carrying file metadata required for upload initiation.

## Key Components

| Field | Type | Constraint | Description |
|-------|------|------------|-------------|
| `fileName` | `String` | `@NotBlank` | Name of the file being attached |
| `contentType` | `String` | — | MIME type of the file |
| `fileSize` | `Long` | — | Size of the file in bytes |

## Usage Example

```java
// Build using the Lombok builder
CreateKnowledgeBaseTempAttachmentInput input = CreateKnowledgeBaseTempAttachmentInput.builder()
    .fileName("network-topology.pdf")
    .contentType("application/pdf")
    .fileSize(204800L)
    .build();

// Or use the all-args constructor
CreateKnowledgeBaseTempAttachmentInput input = new CreateKnowledgeBaseTempAttachmentInput(
    "network-topology.pdf",
    "application/pdf",
    204800L
);
```

> **Note:** `fileName` is required and must not be blank (`@NotBlank`). Submitting a request without a valid `fileName` will fail Bean Validation before reaching the service layer.
