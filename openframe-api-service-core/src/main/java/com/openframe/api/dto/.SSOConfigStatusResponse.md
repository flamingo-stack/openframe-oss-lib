A Data Transfer Object (DTO) that represents the response containing Single Sign-On (SSO) configuration status information for API clients.

## Key Components

- **enabled** - Boolean flag indicating whether SSO is currently enabled
- **provider** - String identifying the SSO provider (e.g., "google", "microsoft", "okta")
- **clientId** - String containing the OAuth client identifier for the SSO provider

## Usage Example

```java
// Building an SSO config status response
SSOConfigStatusResponse response = SSOConfigStatusResponse.builder()
    .enabled(true)
    .provider("google")
    .clientId("your-oauth-client-id")
    .build();

// Accessing response data
boolean ssoEnabled = response.isEnabled();
String providerName = response.getProvider();
String oauthClientId = response.getClientId();

// Typical API controller usage
@GetMapping("/api/sso/status")
public SSOConfigStatusResponse getSSOStatus() {
    return SSOConfigStatusResponse.builder()
        .enabled(ssoConfigService.isEnabled())
        .provider(ssoConfigService.getProvider())
        .clientId(ssoConfigService.getClientId())
        .build();
}
```

This DTO uses Lombok annotations to automatically generate getters, setters, equals, hashCode, toString methods (`@Data`) and a builder pattern (`@Builder`) for convenient object construction.
