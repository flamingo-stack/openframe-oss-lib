A data transfer object (DTO) that encapsulates information about Single Sign-On (SSO) providers, including the provider identifier and human-readable display name.

## Key Components

- **`provider`** - String identifier for the SSO provider (e.g., "google", "microsoft")
- **`displayName`** - Human-readable name for display in UI (e.g., "Google", "Microsoft Azure")
- **`@Data`** - Lombok annotation providing getters, setters, toString, equals, and hashCode
- **`@Builder`** - Lombok annotation enabling builder pattern for object construction

## Usage Example

```java
// Create SSO provider info using builder pattern
SSOProviderInfo googleProvider = SSOProviderInfo.builder()
    .provider("google")
    .displayName("Google")
    .build();

SSOProviderInfo azureProvider = SSOProviderInfo.builder()
    .provider("microsoft")
    .displayName("Microsoft Azure")
    .build();

// Use in a list of available providers
List<SSOProviderInfo> availableProviders = Arrays.asList(
    googleProvider,
    azureProvider
);

// Access properties
String providerId = googleProvider.getProvider(); // "google"
String displayText = googleProvider.getDisplayName(); // "Google"
```
