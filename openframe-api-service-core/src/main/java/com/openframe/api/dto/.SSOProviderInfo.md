A simple data transfer object (DTO) that represents Single Sign-On (SSO) provider information with basic identification details.

## Key Components

- **`provider`** - String identifier for the SSO provider (e.g., "google", "microsoft")
- **`displayName`** - Human-readable name for the provider (e.g., "Google", "Microsoft Azure")
- **`@Data`** - Lombok annotation generating getters, setters, toString, equals, and hashCode
- **`@Builder`** - Lombok annotation enabling fluent builder pattern construction

## Usage Example

```java
// Create SSO provider info using builder pattern
SSOProviderInfo googleProvider = SSOProviderInfo.builder()
    .provider("google")
    .displayName("Google")
    .build();

SSOProviderInfo azureProvider = SSOProviderInfo.builder()
    .provider("microsoft")
    .displayName("Microsoft Azure AD")
    .build();

// Access provider information
String providerId = googleProvider.getProvider();
String name = googleProvider.getDisplayName();

// Use in collections for SSO configuration
List<SSOProviderInfo> availableProviders = Arrays.asList(
    googleProvider, azureProvider
);
```

This DTO is typically used in API responses to inform clients about available SSO authentication options, allowing frontend applications to display appropriate login buttons with correct provider names and identifiers.
