A data transfer object (DTO) used to request forced updates for specific client machines through the OpenFrame API.

## Key Components

- **machineId**: List of machine identifiers that should receive the forced update
- **@Data**: Lombok annotation that automatically generates getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create a request to force update specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(Arrays.asList("machine-001", "machine-002", "kiosk-lobby"));

// Send the request through the API
updateService.forceClientUpdate(request);

// Or using builder pattern (if available)
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(List.of("terminal-1", "terminal-2"));
```

This DTO is typically used in administrative scenarios where specific client machines need to be updated immediately rather than waiting for their regular update cycle. The list structure allows for batch operations across multiple machines in a single API call.
