A data transfer object (DTO) that represents a request to force client updates for specific machines in the OpenFrame system.

## Key Components

- **`machineId`** - List of machine identifiers that should receive forced updates
- **`@Data`** - Lombok annotation providing automatic getter/setter methods, toString, equals, and hashCode implementations

## Usage Example

```java
// Create a force update request for multiple machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(Arrays.asList("machine-001", "machine-002", "machine-003"));

// Send the request to trigger updates
updateService.forceClientUpdate(request);

// Alternative construction approach
ForceClientUpdateRequest singleMachineRequest = new ForceClientUpdateRequest();
singleMachineRequest.setMachineId(Collections.singletonList("machine-001"));
```

This DTO is typically used in REST API endpoints or service methods where administrators need to push immediate updates to specific client machines rather than waiting for scheduled update cycles.
