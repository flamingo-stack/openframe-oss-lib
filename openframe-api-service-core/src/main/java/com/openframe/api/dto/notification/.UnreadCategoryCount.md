A Data Transfer Object (DTO) record that pairs a notification category with its corresponding unread notification count.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `category` | `NotificationCategory` | The notification category enum/type |
| `count` | `long` | Number of unread notifications in the category |

## Usage Example

```java
// Create an unread count summary for a specific category
UnreadCategoryCount alertCount = new UnreadCategoryCount(
    NotificationCategory.ALERTS,
    42L
);

// Access components
NotificationCategory category = alertCount.category();
long unread = alertCount.count();

// Typical use: aggregating unread counts per category
List<UnreadCategoryCount> summary = categories.stream()
    .map(cat -> new UnreadCategoryCount(cat, notificationService.countUnread(cat)))
    .toList();
```

> Being a Java `record`, `UnreadCategoryCount` is immutable and automatically provides `equals()`, `hashCode()`, and `toString()` implementations — making it ideal for lightweight API response payloads.
