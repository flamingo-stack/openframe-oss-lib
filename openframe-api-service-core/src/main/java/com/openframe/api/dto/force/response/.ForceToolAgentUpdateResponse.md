A response DTO that encapsulates the results of a force tool agent update operation, containing a list of update response items.

## Key Components

- **items** - List of `ForceToolAgentUpdateResponseItem` objects representing individual update results
- **@Data** - Lombok annotation providing automatic getter/setter methods and other boilerplate code

## Usage Example

```java
// Creating a response with update results
ForceToolAgentUpdateResponse response = new ForceToolAgentUpdateResponse();
List<ForceToolAgentUpdateResponseItem> updateItems = Arrays.asList(
    new ForceToolAgentUpdateResponseItem(),
    new ForceToolAgentUpdateResponseItem()
);
response.setItems(updateItems);

// Accessing update results
List<ForceToolAgentUpdateResponseItem> results = response.getItems();
for (ForceToolAgentUpdateResponseItem item : results) {
    // Process each update result
}

// In REST controller
@PostMapping("/force-update")
public ForceToolAgentUpdateResponse forceUpdate(@RequestBody UpdateRequest request) {
    ForceToolAgentUpdateResponse response = new ForceToolAgentUpdateResponse();
    response.setItems(updateService.performForceUpdate(request));
    return response;
}
```

This DTO is typically returned from API endpoints that perform batch force updates on tool agents, allowing clients to receive structured information about the success or failure of each individual update operation.
