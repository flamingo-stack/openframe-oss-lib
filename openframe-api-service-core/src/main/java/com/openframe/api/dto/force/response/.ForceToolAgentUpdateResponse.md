Response DTO that wraps a collection of force tool agent update results, typically returned after performing batch updates on tool agents.

## Key Components

- **`items`** - List of `ForceToolAgentUpdateResponseItem` objects containing individual update results
- **`@Data`** - Lombok annotation providing getter/setter methods and standard object methods

## Usage Example

```java
// Processing batch update response
ForceToolAgentUpdateResponse response = forceToolService.updateAgents(updateRequest);

// Iterate through update results
for (ForceToolAgentUpdateResponseItem item : response.getItems()) {
    if (item.isSuccess()) {
        logger.info("Agent {} updated successfully", item.getAgentId());
    } else {
        logger.error("Failed to update agent {}: {}", item.getAgentId(), item.getErrorMessage());
    }
}

// Check total results
int totalUpdates = response.getItems().size();
long successfulUpdates = response.getItems().stream()
    .filter(ForceToolAgentUpdateResponseItem::isSuccess)
    .count();
```

This response wrapper enables structured handling of bulk agent update operations, allowing clients to process both successful and failed updates appropriately.
