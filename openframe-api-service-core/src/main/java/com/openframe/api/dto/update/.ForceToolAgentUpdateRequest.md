A request DTO for forcing tool agent updates across multiple machines, containing the target machine IDs and the specific tool agent to update.

## Key Components

- **`machineIds`** - List of machine identifiers where the tool agent update should be forced
- **`toolAgentId`** - Identifier of the specific tool agent to be updated
- **`@Data`** - Lombok annotation providing automatic getter/setter methods, toString, equals, and hashCode

## Usage Example

```java
// Create a force update request for multiple machines
ForceToolAgentUpdateRequest request = new ForceToolAgentUpdateRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));
request.setToolAgentId("agent-v2.1.0");

// Send the request to force tool agent updates
updateService.forceToolAgentUpdate(request);

// Alternative construction using builder pattern (if available)
ForceToolAgentUpdateRequest request = new ForceToolAgentUpdateRequest();
request.setMachineIds(List.of("prod-server-01", "prod-server-02"));
request.setToolAgentId("monitoring-agent-latest");
```

This DTO is typically used in REST API endpoints or service methods that handle bulk tool agent updates across distributed systems or machine clusters.
