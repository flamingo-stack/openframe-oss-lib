A response DTO for force tool agent installation operations, containing a list of installation response items.

## Key Components

- **`items`**: A list of `ForceToolAgentInstallationResponseItem` objects representing individual installation results or status information for force tool agents

## Usage Example

```java
// Creating a response with installation results
ForceToolAgentInstallationResponse response = new ForceToolAgentInstallationResponse();
List<ForceToolAgentInstallationResponseItem> installationItems = Arrays.asList(
    new ForceToolAgentInstallationResponseItem(),
    new ForceToolAgentInstallationResponseItem()
);
response.setItems(installationItems);

// Processing installation results
for (ForceToolAgentInstallationResponseItem item : response.getItems()) {
    // Handle each installation result
    processInstallationResult(item);
}

// REST endpoint usage
@PostMapping("/force/install-agents")
public ForceToolAgentInstallationResponse installAgents(@RequestBody InstallRequest request) {
    ForceToolAgentInstallationResponse response = forceToolService.installAgents(request);
    return response; // Automatically serialized to JSON
}
```

This DTO uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods, making it ideal for REST API responses where multiple agent installation results need to be returned as a collection.
