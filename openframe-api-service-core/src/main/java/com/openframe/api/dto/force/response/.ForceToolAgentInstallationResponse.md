A response DTO that wraps a collection of force tool agent installation result items, typically returned from bulk installation operations.

## Key Components

- **`items`**: List of `ForceToolAgentInstallationResponseItem` objects containing individual installation results
- **`@Data`**: Lombok annotation providing automatic getter/setter generation and other utility methods

## Usage Example

```java
// Receiving installation response
ForceToolAgentInstallationResponse response = forceToolService.installAgents(installationRequest);

// Processing installation results
for (ForceToolAgentInstallationResponseItem item : response.getItems()) {
    if (item.isSuccess()) {
        log.info("Agent installed successfully: {}", item.getAgentId());
    } else {
        log.error("Installation failed: {}", item.getErrorMessage());
    }
}

// Checking overall installation status
boolean allSuccessful = response.getItems().stream()
    .allMatch(ForceToolAgentInstallationResponseItem::isSuccess);
```
