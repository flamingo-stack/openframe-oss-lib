A DTO (Data Transfer Object) class that represents the response structure for invitation-related API operations, containing invitation details and metadata.

## Key Components

- **InvitationResponse**: Main response class with Lombok annotations for data handling
- **Fields**:
  - `id`: Unique invitation identifier
  - `email`: Recipient's email address
  - `roles`: List of roles assigned to the invitation
  - `createdAt`: Invitation creation timestamp
  - `expiresAt`: Invitation expiration timestamp
  - `status`: Current invitation status (enum)

## Usage Example

```java
// Creating an invitation response
InvitationResponse response = InvitationResponse.builder()
    .id("inv-123")
    .email("user@example.com")
    .roles(Arrays.asList(Role.USER, Role.EDITOR))
    .createdAt(Instant.now())
    .expiresAt(Instant.now().plusSeconds(86400 * 7)) // 7 days
    .status(InvitationStatus.PENDING)
    .build();

// Accessing response data
String invitationId = response.getId();
List<Role> assignedRoles = response.getRoles();
boolean isExpired = response.getExpiresAt().isBefore(Instant.now());
```

The class uses Lombok's `@Data` and `@Builder` annotations to automatically generate getters, setters, constructors, and builder pattern methods, making it easy to create and manipulate invitation response objects in API controllers and services.
