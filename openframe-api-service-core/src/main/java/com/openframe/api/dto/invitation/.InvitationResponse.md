Data Transfer Object representing an invitation response with user details, roles, and status information for API communication.

## Key Components

- **`id`** - Unique identifier for the invitation
- **`email`** - Email address of the invited user
- **`roles`** - List of roles assigned to the invitation
- **`createdAt`** - Timestamp when the invitation was created
- **`expiresAt`** - Timestamp when the invitation expires
- **`status`** - Current status of the invitation (enum type)

## Usage Example

```java
// Create an invitation response
InvitationResponse response = InvitationResponse.builder()
    .id("inv-123")
    .email("user@example.com")
    .roles(List.of(Role.USER, Role.EDITOR))
    .createdAt(Instant.now())
    .expiresAt(Instant.now().plusSeconds(86400 * 7)) // 7 days
    .status(InvitationStatus.PENDING)
    .build();

// Access invitation details
String invitationId = response.getId();
boolean isExpired = response.getExpiresAt().isBefore(Instant.now());
List<Role> assignedRoles = response.getRoles();
```

This DTO uses Lombok annotations for automatic generation of getters, setters, and builder pattern methods, making it ideal for REST API responses containing invitation data.
