A simple enumeration that defines the possible status values for force agent operations in the OpenFrame API response system.

## Key Components

- **PROCESSED** - Indicates the force agent operation completed successfully
- **FAILED** - Indicates the force agent operation encountered an error or failure

## Usage Example

```java
// Check force agent operation result
ForceAgentStatus status = getForceAgentResult();

switch (status) {
    case PROCESSED:
        System.out.println("Force agent operation completed successfully");
        // Continue with next steps
        break;
    case FAILED:
        System.out.println("Force agent operation failed");
        // Handle error case
        break;
}

// Use in conditional logic
if (status == ForceAgentStatus.PROCESSED) {
    proceedWithNextOperation();
} else {
    handleFailure();
}
```

This enum is typically used in response DTOs to communicate the outcome of force agent operations back to API clients, providing a standardized way to represent operation status across the OpenFrame system.
