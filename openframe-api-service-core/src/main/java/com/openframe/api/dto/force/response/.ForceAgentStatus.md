Defines the possible status values for force agent operations in the OpenFrame API response system.

## Key Components

- **PROCESSED** - Indicates the force agent operation completed successfully
- **FAILED** - Indicates the force agent operation encountered an error and could not complete

## Usage Example

```java
// In a response handler
ForceAgentStatus status = response.getStatus();

switch (status) {
    case PROCESSED:
        logger.info("Force agent operation completed successfully");
        handleSuccess();
        break;
    case FAILED:
        logger.error("Force agent operation failed");
        handleFailure();
        break;
}

// In a response builder
ForceAgentResponse response = ForceAgentResponse.builder()
    .status(ForceAgentStatus.PROCESSED)
    .build();
```
