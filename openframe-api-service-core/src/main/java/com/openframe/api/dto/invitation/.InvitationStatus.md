This enum defines the possible states that an invitation can have within the OpenFrame API system, providing a standardized way to track invitation lifecycle.

## Key Components

- **PENDING** - Invitation has been sent but not yet responded to
- **ACCEPTED** - Invitation has been accepted by the recipient
- **EXPIRED** - Invitation has passed its validity period
- **REVOKED** - Invitation has been cancelled or withdrawn

## Usage Example

```java
// Setting invitation status
InvitationStatus status = InvitationStatus.PENDING;

// Checking invitation state
if (invitation.getStatus() == InvitationStatus.ACCEPTED) {
    // Process accepted invitation
    grantAccess(invitation.getRecipient());
} else if (invitation.getStatus() == InvitationStatus.EXPIRED) {
    // Clean up expired invitation
    removeInvitation(invitation);
}

// Switch statement for status handling
switch (invitation.getStatus()) {
    case PENDING:
        sendReminder(invitation);
        break;
    case ACCEPTED:
        activateUser(invitation);
        break;
    case EXPIRED:
    case REVOKED:
        cleanupInvitation(invitation);
        break;
}
```
