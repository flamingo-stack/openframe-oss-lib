A DTO record used to filter notifications by read status and search query, with a pre-built empty filter constant.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `read` | `Boolean` | Filter by read/unread status; `null` means no filter applied |
| `search` | `String` | Filter by search term; `null` means no filter applied |
| `EMPTY` | `NotificationFilter` | Convenience constant representing a no-op filter (all fields null) |

## Usage Example

```java
// No filtering — retrieve all notifications
NotificationFilter noFilter = NotificationFilter.EMPTY;

// Filter for unread notifications only
NotificationFilter unreadOnly = new NotificationFilter(false, null);

// Filter by search term across all notifications
NotificationFilter bySearch = new NotificationFilter(null, "ticket");

// Filter for read notifications matching a search term
NotificationFilter combined = new NotificationFilter(true, "deployment");
```
