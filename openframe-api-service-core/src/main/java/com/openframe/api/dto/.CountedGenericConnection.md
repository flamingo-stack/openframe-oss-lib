A generic paginated connection DTO that extends `GenericConnection` with an additional `filteredCount` field, representing the total number of items matching the current filter criteria.

## Key Components

| Component | Description |
|-----------|-------------|
| `CountedGenericConnection<T>` | Generic class extending `GenericConnection<T>`, bounded to types implementing `GenericEdge` |
| `filteredCount` | Integer representing the total count of filtered results, separate from the page size |

## Usage Example

```java
// Building a counted connection response
List<UserEdge> edges = userService.findEdges(filter);
PageInfo pageInfo = PageInfo.builder()
    .hasNextPage(true)
    .endCursor("cursor123")
    .build();

CountedGenericConnection<UserEdge> response = CountedGenericConnection.<UserEdge>builder()
    .edges(edges)
    .pageInfo(pageInfo)
    .filteredCount(142)
    .build();

// Accessing fields
int total = response.getFilteredCount(); // 142
List<UserEdge> pageEdges = response.getEdges();
```

> **Note:** `filteredCount` reflects the total matching records across all pages, not just the current page — useful for rendering pagination controls in the OpenFrame UI.
