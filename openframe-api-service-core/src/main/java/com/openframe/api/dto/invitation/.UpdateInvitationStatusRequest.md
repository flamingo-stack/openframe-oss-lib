A data transfer object (DTO) for updating the status of an invitation, providing validation constraints for API requests.

## Key Components

- **`status`** - The new invitation status to be set, validated as non-null
- **`@Data`** - Lombok annotation providing getters, setters, toString, equals, and hashCode methods
- **`@NotNull`** - Jakarta validation constraint ensuring the status field is required

## Usage Example

```java
// Creating an update request
UpdateInvitationStatusRequest request = new UpdateInvitationStatusRequest();
request.setStatus(InvitationStatus.ACCEPTED);

// Using in a REST controller
@PutMapping("/invitations/{id}/status")
public ResponseEntity<Void> updateInvitationStatus(
    @PathVariable Long id,
    @Valid @RequestBody UpdateInvitationStatusRequest request) {
    
    invitationService.updateStatus(id, request.getStatus());
    return ResponseEntity.ok().build();
}

// Validation will automatically reject null status values
UpdateInvitationStatusRequest invalidRequest = new UpdateInvitationStatusRequest();
// Will fail validation when processed by @Valid annotation
```

This DTO encapsulates invitation status updates with built-in validation, ensuring API consumers provide valid status transitions while maintaining type safety through the `InvitationStatus` enum.
