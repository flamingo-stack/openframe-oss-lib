A DTO class for updating the status of an invitation, containing validation constraints to ensure data integrity.

## Key Components

- **UpdateInvitationStatusRequest** - Main DTO class with Lombok `@Data` annotation for automatic getters, setters, and other boilerplate methods
- **status** - Required field of type `InvitationStatus` with `@NotNull` validation
- **InvitationStatus** - Enum type representing possible invitation states (referenced but not defined in this file)

## Usage Example

```java
// Create request to update invitation status
UpdateInvitationStatusRequest request = new UpdateInvitationStatusRequest();
request.setStatus(InvitationStatus.ACCEPTED);

// The @NotNull validation ensures status cannot be null
// This will trigger validation error:
UpdateInvitationStatusRequest invalidRequest = new UpdateInvitationStatusRequest();
// invalidRequest.setStatus(null); // Will fail validation

// Typically used in REST controllers
@PutMapping("/invitations/{id}/status")
public ResponseEntity<?> updateStatus(@RequestBody @Valid UpdateInvitationStatusRequest request) {
    // Service layer handles the status update
    invitationService.updateStatus(request.getStatus());
    return ResponseEntity.ok().build();
}
```
