GraphQL input type DTO for filtering organizations by various criteria, mapping to the `OrganizationFilterInput` type defined in the GraphQL schema.

## Key Components

| Field | Type | Description |
|---|---|---|
| `category` | `String` | Filter by organization category |
| `minEmployees` | `Integer` | Minimum employee count threshold |
| `maxEmployees` | `Integer` | Maximum employee count threshold |
| `hasActiveContract` | `Boolean` | Filter by active contract status |
| `status` | `String` | Filter by organization status |

## Usage Example

```java
OrganizationFilterInput filter = new OrganizationFilterInput();
filter.setCategory("MSP");
filter.setMinEmployees(10);
filter.setMaxEmployees(500);
filter.setHasActiveContract(true);
filter.setStatus("ACTIVE");

// Pass to a GraphQL query resolver
List<Organization> results = organizationService.findAll(filter);
```

> All fields are optional — only populated fields are applied as filter conditions. Lombok's `@Data` annotation auto-generates getters, setters, `equals`, `hashCode`, and `toString`.
