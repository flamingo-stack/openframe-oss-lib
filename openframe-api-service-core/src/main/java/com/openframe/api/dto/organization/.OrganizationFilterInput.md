GraphQL input DTO for filtering organization queries with category, employee count, and contract status criteria.

## Key Components

- **category** - String filter for organization category/type
- **minEmployees** - Integer minimum employee count threshold
- **maxEmployees** - Integer maximum employee count threshold  
- **hasActiveContract** - Boolean flag to filter by active contract status

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, equals, hashCode, and toString methods.

## Usage Example

```java
// Create filter for tech companies with 10-500 employees
OrganizationFilterInput filter = new OrganizationFilterInput();
filter.setCategory("Technology");
filter.setMinEmployees(10);
filter.setMaxEmployees(500);
filter.setHasActiveContract(true);

// Use in GraphQL query
String query = """
    query getOrganizations($filter: OrganizationFilterInput) {
        organizations(filter: $filter) {
            id
            name
            category
            employeeCount
        }
    }
    """;

// Filter for any size organization with active contracts
OrganizationFilterInput activeFilter = new OrganizationFilterInput();
activeFilter.setHasActiveContract(true);
```
