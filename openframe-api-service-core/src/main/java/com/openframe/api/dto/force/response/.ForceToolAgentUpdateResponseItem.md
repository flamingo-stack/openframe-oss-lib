A response DTO that represents the result of updating a single tool agent through a force operation, containing the agent's identification and current status.

## Key Components

- **machineId** - String identifier for the machine hosting the tool agent
- **toolAgentId** - String identifier for the specific tool agent being updated
- **status** - ForceAgentStatus enum indicating the current state of the agent after the update operation
- **@Data** - Lombok annotation providing automatic getter/setter generation and other utility methods

## Usage Example

```java
// Typically used as part of a response collection
List<ForceToolAgentUpdateResponseItem> updateResults = forceToolService.updateAgents(request);

for (ForceToolAgentUpdateResponseItem item : updateResults) {
    System.out.println("Machine: " + item.getMachineId());
    System.out.println("Agent: " + item.getToolAgentId());
    System.out.println("Status: " + item.getStatus());
    
    if (item.getStatus() == ForceAgentStatus.FAILED) {
        // Handle failed update
        handleFailedUpdate(item.getMachineId(), item.getToolAgentId());
    }
}
```

This DTO is commonly used in bulk operations where multiple tool agents are updated simultaneously, allowing clients to track the success or failure of each individual agent update.
