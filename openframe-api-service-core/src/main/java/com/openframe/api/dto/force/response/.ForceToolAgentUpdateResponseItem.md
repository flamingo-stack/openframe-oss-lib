A response DTO that represents the result of a force tool agent update operation for a specific machine and tool agent combination.

## Key Components

- **`machineId`** - String identifier for the target machine
- **`toolAgentId`** - String identifier for the specific tool agent being updated
- **`status`** - ForceAgentStatus enum indicating the current status of the force update operation

## Usage Example

```java
// Creating a response item for a successful force update
ForceToolAgentUpdateResponseItem responseItem = new ForceToolAgentUpdateResponseItem();
responseItem.setMachineId("machine-001");
responseItem.setToolAgentId("tool-agent-123");
responseItem.setStatus(ForceAgentStatus.SUCCESS);

// Accessing response data
String machineId = responseItem.getMachineId();
String toolAgentId = responseItem.getToolAgentId();
ForceAgentStatus updateStatus = responseItem.getStatus();

// Typically used in collection responses
List<ForceToolAgentUpdateResponseItem> updateResults = Arrays.asList(responseItem);
```

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods, making it ideal for API response serialization and logging.
