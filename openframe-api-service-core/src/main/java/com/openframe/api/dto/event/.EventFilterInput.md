A GraphQL input DTO class that defines filtering criteria for querying events in the OpenFrame API.

## Key Components

- **userIds** - List of user IDs to filter events by specific users
- **eventTypes** - List of event type strings to filter by event categories
- **startDate** - LocalDate for filtering events from a specific start date
- **endDate** - LocalDate for filtering events until a specific end date

## Usage Example

```java
// Create event filter for specific users and date range
EventFilterInput filter = EventFilterInput.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 12, 31))
    .build();

// Filter for all events in the last month
EventFilterInput recentFilter = EventFilterInput.builder()
    .startDate(LocalDate.now().minusMonths(1))
    .endDate(LocalDate.now())
    .build();
```

This DTO uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it easy to construct complex filtering criteria for GraphQL event queries.
