A GraphQL input DTO class used for filtering events based on multiple criteria including users, event types, and date ranges.

## Key Components

- **userIds**: List of user identifiers to filter events by specific users
- **eventTypes**: List of event type strings to filter by event categories
- **startDate**: LocalDate for filtering events from a specific start date
- **endDate**: LocalDate for filtering events until a specific end date
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Using builder pattern to create filter criteria
EventFilterInput filter = EventFilterInput.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "DATA_ACCESS"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 12, 31))
    .build();

// Using with GraphQL query
public List<Event> getFilteredEvents(EventFilterInput filter) {
    // Pass filter to GraphQL resolver or service layer
    return eventService.findEvents(filter);
}

// Creating filter with only date range
EventFilterInput dateFilter = EventFilterInput.builder()
    .startDate(LocalDate.now().minusDays(30))
    .endDate(LocalDate.now())
    .build();
```
