This data transfer object represents user information returned from OpenID Connect (OIDC) user info endpoints, supporting both successful user data and error responses.

## Key Components

- **UserInfo** - Main DTO class with Lombok `@Data` and `@Builder` annotations
- **User Fields**:
  - `sub` - Subject identifier (user ID)
  - `name` - Full display name
  - `email` - Email address
  - `givenName` - First name (mapped to `given_name`)
  - `familyName` - Last name (mapped to `family_name`) 
  - `emailVerified` - Email verification status (mapped to `email_verified`)
- **Error Fields**:
  - `error` - Error code
  - `errorDescription` - Human-readable error description (mapped to `error_description`)

## Usage Example

```java
// Building successful user info response
UserInfo userInfo = UserInfo.builder()
    .sub("123456789")
    .name("John Doe")
    .email("john.doe@example.com")
    .givenName("John")
    .familyName("Doe")
    .emailVerified(true)
    .build();

// Building error response
UserInfo errorResponse = UserInfo.builder()
    .error("invalid_token")
    .errorDescription("The access token is invalid")
    .build();
```

The class uses Jackson annotations for JSON serialization and excludes null values from the output.
