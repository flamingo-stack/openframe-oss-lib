A data transfer object (DTO) that represents the input payload for creating new events in the OpenFrame API. This class defines the required and optional fields needed when submitting event creation requests.

## Key Components

- **userId** - Required string identifier for the user creating the event
- **type** - Required non-blank string specifying the event type/category
- **data** - Optional string field for additional event payload or metadata
- **Validation annotations** - Jakarta Bean Validation constraints ensuring data integrity
- **Lombok annotations** - Auto-generated constructors, getters, setters, and builder pattern

## Usage Example

```java
// Create event input using builder pattern
CreateEventInput eventInput = CreateEventInput.builder()
    .userId("user-123")
    .type("USER_LOGIN")
    .data("{\"timestamp\": \"2024-01-15T10:30:00Z\", \"ip\": \"192.168.1.1\"}")
    .build();

// Create event input using constructor
CreateEventInput simpleEvent = new CreateEventInput(
    "user-456", 
    "PROFILE_UPDATE", 
    null
);

// Validation will automatically enforce:
// - userId cannot be null
// - type cannot be null or blank
// - data is optional and can be null
```

The class uses Jakarta validation to ensure required fields are provided before processing event creation requests, helping maintain data consistency across the API.
