A data transfer object (DTO) for creating new events in the OpenFrame API, with validation constraints to ensure required fields are provided.

## Key Components

- **userId** - Required string field identifying the user creating the event
- **type** - Required non-blank string specifying the event type
- **data** - Optional string field for additional event data
- **Validation annotations** - `@NotNull` and `@NotBlank` constraints with custom error messages
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Using builder pattern
CreateEventInput eventInput = CreateEventInput.builder()
    .userId("user123")
    .type("USER_LOGIN")
    .data("{\"timestamp\": \"2024-01-01T10:00:00Z\"}")
    .build();

// Using constructor
CreateEventInput eventInput = new CreateEventInput(
    "user123", 
    "USER_ACTION", 
    "Additional event metadata"
);

// Validation will fail if userId is null or type is blank
CreateEventInput invalid = CreateEventInput.builder()
    .type("") // Validation error: "Event type is required"
    .build(); // Validation error: "User ID is required"
```
