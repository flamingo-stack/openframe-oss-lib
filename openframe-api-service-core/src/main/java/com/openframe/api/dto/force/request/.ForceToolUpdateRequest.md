A data transfer object (DTO) that represents a request to force update tools on specified machines through a tool agent.

## Key Components

- **machineIds** - List of machine identifiers that should receive the forced tool update
- **toolAgentId** - Identifier of the tool agent responsible for performing the update
- **@Data** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a force tool update request for multiple machines
ForceToolUpdateRequest request = new ForceToolUpdateRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));
request.setToolAgentId("agent-123");

// Send the request to force update tools on specified machines
apiClient.forceToolUpdate(request);

// Or using builder pattern (if available)
ForceToolUpdateRequest request = new ForceToolUpdateRequest();
request.setMachineIds(List.of("prod-server-01", "prod-server-02"));
request.setToolAgentId("production-agent");
```

This DTO is typically used in REST API endpoints where administrators need to trigger immediate tool updates across multiple machines in a distributed system.
