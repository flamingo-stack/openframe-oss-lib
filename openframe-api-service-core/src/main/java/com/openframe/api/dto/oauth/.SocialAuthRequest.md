A data transfer object (DTO) that encapsulates the parameters required for OAuth social authentication requests, including authorization code and PKCE verification.

## Key Components

- **`code`** - The authorization code received from the OAuth provider
- **`codeVerifier`** - PKCE (Proof Key for Code Exchange) code verifier for enhanced security
- **`redirectUri`** - The URI where the OAuth provider should redirect after authentication
- **`@Data`** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods
- **`@JsonProperty`** - Jackson annotations for proper JSON serialization with snake_case naming

## Usage Example

```java
// Creating a social auth request
SocialAuthRequest request = new SocialAuthRequest();
request.setCode("auth_code_from_provider");
request.setCodeVerifier("dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk");
request.setRedirectUri("https://myapp.com/auth/callback");

// JSON representation:
// {
//   "code": "auth_code_from_provider",
//   "code_verifier": "dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk",
//   "redirect_uri": "https://myapp.com/auth/callback"
// }

// Typically used in REST controller methods
@PostMapping("/auth/social")
public ResponseEntity<?> authenticateWithSocial(@RequestBody SocialAuthRequest request) {
    // Process OAuth authentication with the provided parameters
    return authService.processOAuthAuthentication(request);
}
```
