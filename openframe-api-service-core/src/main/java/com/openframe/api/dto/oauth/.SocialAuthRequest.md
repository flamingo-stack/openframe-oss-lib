A data transfer object for handling OAuth 2.0 social authentication requests with PKCE (Proof Key for Code Exchange) support.

## Key Components

- **code** - Authorization code received from the OAuth provider
- **codeVerifier** - PKCE code verifier for enhanced security (mapped from `code_verifier`)
- **redirectUri** - URI where the OAuth provider redirects after authorization (mapped from `redirect_uri`)

## Usage Example

```java
// Creating a social auth request
SocialAuthRequest request = new SocialAuthRequest();
request.setCode("auth_code_123");
request.setCodeVerifier("dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk");
request.setRedirectUri("https://myapp.com/callback");

// JSON serialization will map to snake_case
// {
//   "code": "auth_code_123",
//   "code_verifier": "dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk",
//   "redirect_uri": "https://myapp.com/callback"
// }
```
