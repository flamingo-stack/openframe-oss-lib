A data transfer object for requesting forced client updates on specific machines in the OpenFrame API.

## Key Components

- **`machineIds`** - List of machine identifier strings that should receive forced updates

## Usage Example

```java
// Create a force update request for specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));

// Send the request to force updates on the specified machines
apiClient.forceClientUpdate(request);

// Or using builder pattern with Lombok
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineIds(List.of("kiosk-lobby", "kiosk-entrance"));
```

This DTO is typically used in administrative operations where specific client machines need to be updated immediately, bypassing normal update schedules. The `@Data` annotation from Lombok automatically generates getters, setters, equals, hashCode, and toString methods.
