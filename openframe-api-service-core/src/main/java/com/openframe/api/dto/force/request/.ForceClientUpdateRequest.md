A request DTO for forcing client updates on specified machines, containing a list of machine identifiers to target for the update operation.

## Key Components

- **machineIds**: `List<String>` - Collection of machine/client identifiers that should receive forced updates
- **@Data**: Lombok annotation providing automatic getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create request to force update on specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));

// Or using builder pattern (if Lombok @Builder is added)
List<String> targetMachines = Arrays.asList("server-01", "workstation-05");
ForceClientUpdateRequest updateRequest = new ForceClientUpdateRequest();
updateRequest.setMachineIds(targetMachines);

// Typically sent to an API endpoint
apiClient.forceClientUpdate(updateRequest);
```

This DTO is commonly used in administrative operations where system administrators need to push updates to specific client machines or force synchronization of client software across selected devices in a distributed system.
