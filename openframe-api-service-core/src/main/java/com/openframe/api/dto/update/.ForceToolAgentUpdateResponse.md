A data transfer object (DTO) that represents the response from a force tool agent update operation, containing a collection of update result items.

## Key Components

- **`items`** - List of `ForceToolAgentUpdateResponseItem` objects representing individual update results
- **`@Data`** - Lombok annotation generating getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Creating a response with update results
ForceToolAgentUpdateResponse response = new ForceToolAgentUpdateResponse();

List<ForceToolAgentUpdateResponseItem> updateItems = Arrays.asList(
    new ForceToolAgentUpdateResponseItem(), // Success item
    new ForceToolAgentUpdateResponseItem()  // Another result item
);

response.setItems(updateItems);

// Processing the response
if (response.getItems() != null && !response.getItems().isEmpty()) {
    for (ForceToolAgentUpdateResponseItem item : response.getItems()) {
        // Handle each update result item
        processUpdateResult(item);
    }
}

// JSON serialization (typical API response)
// {
//   "items": [
//     { /* ForceToolAgentUpdateResponseItem data */ },
//     { /* ForceToolAgentUpdateResponseItem data */ }
//   ]
// }
```

This response wrapper allows the API to return multiple update results in a single operation, providing batch update capabilities for force tool agent operations.
