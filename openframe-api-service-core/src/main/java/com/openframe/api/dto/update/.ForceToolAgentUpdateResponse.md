A response DTO that wraps a collection of force tool agent update response items, used for batch update operations in the OpenFrame API.

## Key Components

- **ForceToolAgentUpdateResponse**: Main response class containing a list of update response items
- **items**: List of `ForceToolAgentUpdateResponseItem` objects representing individual update results
- **@Data**: Lombok annotation providing automatic getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Processing a batch update response
ForceToolAgentUpdateResponse response = apiClient.updateForceToolAgents(updateRequest);

// Access the update results
List<ForceToolAgentUpdateResponseItem> updateResults = response.getItems();

// Iterate through individual update responses
for (ForceToolAgentUpdateResponseItem item : updateResults) {
    // Process each update result
    System.out.println("Update result: " + item);
}

// Check if any updates were processed
if (response.getItems().isEmpty()) {
    System.out.println("No items were updated");
}
```
