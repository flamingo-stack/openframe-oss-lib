Data Transfer Object (DTO) for capturing the required input when creating an attachment associated with a knowledge base article.

## Key Components

| Field | Type | Constraint | Description |
|-------|------|------------|-------------|
| `articleId` | `String` | `@NotBlank` | ID of the target knowledge base article |
| `fileName` | `String` | `@NotBlank` | Name of the file being attached |
| `contentType` | `String` | — | MIME type of the attachment (e.g. `image/png`) |
| `fileSize` | `Long` | — | Size of the file in bytes |

## Usage Example

```java
CreateKnowledgeBaseAttachmentInput input = CreateKnowledgeBaseAttachmentInput.builder()
    .articleId("article-123")
    .fileName("network-diagram.png")
    .contentType("image/png")
    .fileSize(204800L)
    .build();
```

> `articleId` and `fileName` are mandatory — validation will reject blank values before the request reaches the service layer.
