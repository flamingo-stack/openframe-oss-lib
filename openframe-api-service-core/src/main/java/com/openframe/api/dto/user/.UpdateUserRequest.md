A data transfer object (DTO) for handling user profile update requests with validation constraints for first and last name fields.

## Key Components

- **UpdateUserRequest**: Main DTO class for user update operations
- **firstName**: Optional field for updating user's first name (max 128 characters)
- **lastName**: Optional field for updating user's last name (max 128 characters)
- **@Size validation**: Ensures name fields don't exceed character limits
- **@Data annotation**: Lombok-generated getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create an update request
UpdateUserRequest updateRequest = new UpdateUserRequest();
updateRequest.setFirstName("John");
updateRequest.setLastName("Doe");

// Validation will ensure names are within 128 character limit
// Both fields are optional - can update just firstName or lastName

// Example with partial update
UpdateUserRequest partialUpdate = new UpdateUserRequest();
partialUpdate.setFirstName("Jane"); // Only updating first name
// lastName remains null, indicating no change requested
```

This DTO follows the pattern of allowing partial updates where only provided fields will be updated, while null values indicate no change is requested for those fields.
