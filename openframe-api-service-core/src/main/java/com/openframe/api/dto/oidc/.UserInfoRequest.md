A data transfer object (DTO) that encapsulates user information fields for OIDC (OpenID Connect) user info requests. This class uses Lombok annotations to automatically generate boilerplate code for data handling.

## Key Components

- **@Data**: Lombok annotation that generates getters, setters, toString, equals, and hashCode methods
- **@Builder**: Lombok annotation that implements the Builder pattern for object construction
- **userId**: String field for unique user identifier
- **email**: String field for user's email address
- **firstName**: String field for user's first name
- **lastName**: String field for user's last name

## Usage Example

```java
// Create a UserInfoRequest using the builder pattern
UserInfoRequest userInfo = UserInfoRequest.builder()
    .userId("user123")
    .email("john.doe@example.com")
    .firstName("John")
    .lastName("Doe")
    .build();

// Access user information
String email = userInfo.getEmail();
String fullName = userInfo.getFirstName() + " " + userInfo.getLastName();

// Update user information
userInfo.setEmail("john.updated@example.com");
```

This DTO is typically used in OIDC flows to transfer user profile information between services or when constructing user info responses for authentication requests.
