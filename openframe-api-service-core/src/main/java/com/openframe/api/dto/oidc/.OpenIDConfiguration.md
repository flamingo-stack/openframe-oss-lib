A data transfer object representing OpenID Connect discovery configuration metadata used for OAuth 2.0/OIDC provider discovery.

## Key Components

- **OpenIDConfiguration**: Main DTO class with Lombok annotations for automatic getters/setters and builder pattern
- **Core Endpoints**: Authorization, token, userinfo, and JWKS URI endpoints
- **Capability Lists**: Supported response types, subject types, signing algorithms, scopes, authentication methods, and claims
- **JSON Mapping**: Jackson annotations for proper JSON property name mapping following OIDC specification

## Usage Example

```java
// Building a configuration object
OpenIDConfiguration config = OpenIDConfiguration.builder()
    .issuer("https://example.com")
    .authorizationEndpoint("https://example.com/auth")
    .tokenEndpoint("https://example.com/token")
    .userinfoEndpoint("https://example.com/userinfo")
    .jwksUri("https://example.com/.well-known/jwks.json")
    .responseTypesSupported(List.of("code", "id_token", "token"))
    .subjectTypesSupported(List.of("public"))
    .idTokenSigningAlgValuesSupported(List.of("RS256", "HS256"))
    .scopesSupported(List.of("openid", "profile", "email"))
    .tokenEndpointAuthMethodsSupported(List.of("client_secret_basic"))
    .claimsSupported(List.of("sub", "name", "email"))
    .build();

// Accessing configuration data
String authEndpoint = config.getAuthorizationEndpoint();
List<String> supportedScopes = config.getScopesSupported();
```
