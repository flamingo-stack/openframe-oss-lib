Data Transfer Object for API key information responses, excluding sensitive secret data for secure client communication.

## Key Components

- **ApiKeyResponse**: Builder-pattern DTO class containing API key metadata
- **Security Fields**: ID, name, description, and enabled status
- **Timestamps**: Creation, update, last usage, and expiration tracking
- **Usage Statistics**: Request counters for total, successful, and failed operations
- **Lombok Annotations**: `@Data` for getters/setters and `@Builder` for construction

## Usage Example

```java
// Creating an API key response
ApiKeyResponse response = ApiKeyResponse.builder()
    .id("api_key_123")
    .name("Production API Key")
    .description("Main application API access")
    .enabled(true)
    .createdAt(Instant.now())
    .totalRequests(1500L)
    .successfulRequests(1485L)
    .failedRequests(15L)
    .expiresAt(Instant.now().plusSeconds(86400 * 30)) // 30 days
    .build();

// Accessing response data
String keyName = response.getName();
boolean isActive = response.isEnabled();
Long usage = response.getTotalRequests();
```

This DTO ensures API key secrets remain secure while providing clients with essential metadata and usage statistics for monitoring and management purposes.
