A Data Transfer Object (DTO) that represents API key information returned to clients, excluding sensitive data like the actual secret key.

## Key Components

- **@Data & @Builder**: Lombok annotations providing getters, setters, equals, hashCode, toString methods and builder pattern
- **Basic Info Fields**: `id`, `name`, `description` for API key identification
- **Status Fields**: `enabled` flag and timestamp fields (`createdAt`, `updatedAt`, `lastUsed`, `expiresAt`)
- **Usage Statistics**: Request counters (`totalRequests`, `successfulRequests`, `failedRequests`)

## Usage Example

```java
// Building an API key response
ApiKeyResponse response = ApiKeyResponse.builder()
    .id("key-123")
    .name("Production API Key")
    .description("Main API key for production services")
    .enabled(true)
    .createdAt(Instant.now().minusSeconds(86400))
    .lastUsed(Instant.now().minusSeconds(3600))
    .totalRequests(1500L)
    .successfulRequests(1450L)
    .failedRequests(50L)
    .build();

// Accessing fields
System.out.println(response.getName()); // "Production API Key"
boolean isActive = response.isEnabled(); // true
```

This DTO is typically used in REST API responses when listing API keys or retrieving API key details, ensuring sensitive information remains secure.
