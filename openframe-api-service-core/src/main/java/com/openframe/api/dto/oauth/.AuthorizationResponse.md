A data transfer object (DTO) representing the response from an OAuth 2.0 authorization server, containing the authorization code and related parameters returned after successful user authorization.

## Key Components

- **code** - The authorization code generated by the authorization server
- **state** - The state parameter used to prevent CSRF attacks and maintain client state
- **redirectUri** - The URI where the authorization response was sent

## Usage Example

```java
// Building an authorization response after successful OAuth flow
AuthorizationResponse response = AuthorizationResponse.builder()
    .code("SplxlOBeZQQYbYS6WxSbIA")
    .state("xyz123")
    .redirectUri("https://client-app.com/callback")
    .build();

// Accessing response data
String authCode = response.getCode();
String clientState = response.getState();
String callback = response.getRedirectUri();

// Typical usage in OAuth callback handler
@GetMapping("/oauth/callback")
public ResponseEntity<?> handleCallback(
    @RequestParam String code,
    @RequestParam String state,
    @RequestParam String redirect_uri) {
    
    AuthorizationResponse authResponse = AuthorizationResponse.builder()
        .code(code)
        .state(state)
        .redirectUri(redirect_uri)
        .build();
    
    return processAuthorizationResponse(authResponse);
}
```
