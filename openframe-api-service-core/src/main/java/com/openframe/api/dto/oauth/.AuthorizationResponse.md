A data transfer object (DTO) that represents an OAuth 2.0 authorization response containing the authorization code and related parameters returned after successful user authentication.

## Key Components

- **code**: The authorization code granted by the authorization server
- **state**: A value used to maintain state between the request and callback to mitigate CSRF attacks
- **redirectUri**: The URI where the authorization server redirects the user-agent after authorization

## Usage Example

```java
// Building an authorization response
AuthorizationResponse response = AuthorizationResponse.builder()
    .code("abc123xyz")
    .state("random-state-value")
    .redirectUri("https://example.com/callback")
    .build();

// Accessing response data
String authCode = response.getCode();
String stateParam = response.getState();
String callbackUri = response.getRedirectUri();

// Typically used in OAuth flow controllers
@GetMapping("/oauth/callback")
public ResponseEntity<?> handleCallback(AuthorizationResponse authResponse) {
    // Process the authorization response
    return processAuthorizationCode(authResponse.getCode(), authResponse.getState());
}
```
