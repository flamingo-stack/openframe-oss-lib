A response DTO class that encapsulates user information returned from API endpoints, including profile details, roles, and associated metadata.

## Key Components

- **UserResponse** - Main DTO class with Lombok annotations for automatic getters/setters and builder pattern
- **Core Fields**: `id`, `email`, `firstName`, `lastName` for basic user identification
- **Authorization**: `roles` list for permission management and `status` for account state
- **Media**: `image` field containing user profile picture data
- **Timestamps**: `createdAt` and `updatedAt` for audit tracking

## Usage Example

```java
// Building a user response
UserResponse response = UserResponse.builder()
    .id("user-123")
    .email("john.doe@example.com")
    .firstName("John")
    .lastName("Doe")
    .roles(List.of("USER", "ADMIN"))
    .status("ACTIVE")
    .image(imageDto)
    .createdAt(LocalDateTime.now())
    .updatedAt(LocalDateTime.now())
    .build();

// Accessing user data
String fullName = response.getFirstName() + " " + response.getLastName();
boolean isAdmin = response.getRoles().contains("ADMIN");
```
