A data transfer object (DTO) that represents the complete user information returned by API endpoints, including profile details, verification status, and associated metadata.

## Key Components

- **User Identity**: `id`, `email`, `firstName`, `lastName` for core user identification
- **Verification Status**: `emailVerified` boolean flag for account verification state
- **Authorization**: `roles` list containing user permission levels and access rights
- **Account Status**: `status` string indicating current account state (active, suspended, etc.)
- **Profile Image**: `image` field using `ImageDto` for user avatar or profile picture
- **Timestamps**: `createdAt` and `updatedAt` for audit trail and account lifecycle tracking
- **Lombok Annotations**: `@Data` and `@Builder` for automatic generation of getters, setters, and builder pattern

## Usage Example

```java
// Building a user response for API return
UserResponse userResponse = UserResponse.builder()
    .id("user123")
    .email("john.doe@company.com")
    .emailVerified(true)
    .firstName("John")
    .lastName("Doe")
    .roles(List.of("USER", "ADMIN"))
    .status("ACTIVE")
    .image(ImageDto.builder().url("/avatars/user123.jpg").build())
    .createdAt(LocalDateTime.now().minusDays(30))
    .updatedAt(LocalDateTime.now())
    .build();

// Accessing user information
String fullName = userResponse.getFirstName() + " " + userResponse.getLastName();
boolean isAdmin = userResponse.getRoles().contains("ADMIN");
boolean canLogin = userResponse.isEmailVerified() && "ACTIVE".equals(userResponse.getStatus());
```
