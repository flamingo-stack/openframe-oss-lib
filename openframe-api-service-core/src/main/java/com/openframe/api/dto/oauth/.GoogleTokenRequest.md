A data transfer object (DTO) representing the request payload for exchanging an OAuth authorization code for Google access tokens using the PKCE (Proof Key for Code Exchange) flow.

## Key Components

- **code** - The authorization code received from Google's OAuth authorization endpoint
- **code_verifier** - The PKCE code verifier used to validate the original code challenge
- **redirect_uri** - The redirect URI that must match the one used in the initial authorization request
- **@Data** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Creating a Google token request for PKCE flow
GoogleTokenRequest tokenRequest = new GoogleTokenRequest();
tokenRequest.setCode("4/0AX4XfWi...");  // Authorization code from Google
tokenRequest.setCode_verifier("dBjftJeZ4CVP-mB92K27uhbUJU1p1r_wW1gFWFOEjXk");  // PKCE verifier
tokenRequest.setRedirect_uri("https://myapp.com/oauth/callback");

// Typically used with REST client to exchange code for tokens
String accessToken = oauthService.exchangeCodeForToken(tokenRequest);

// The DTO can also be populated from JSON request body
@PostMapping("/oauth/token")
public ResponseEntity<?> exchangeToken(@RequestBody GoogleTokenRequest request) {
    return oauthService.processTokenExchange(request);
}
```
