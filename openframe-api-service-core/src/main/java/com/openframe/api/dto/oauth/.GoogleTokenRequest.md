A data transfer object (DTO) that encapsulates the required parameters for exchanging an authorization code for an access token in Google's OAuth 2.0 PKCE flow.

## Key Components

- **`code`** - The authorization code received from Google's authorization server
- **`code_verifier`** - The code verifier used in PKCE (Proof Key for Code Exchange) for enhanced security
- **`redirect_uri`** - The redirect URI that must match the one used in the initial authorization request
- **`@Data`** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a token request for Google OAuth
GoogleTokenRequest tokenRequest = new GoogleTokenRequest();
tokenRequest.setCode("authorization_code_from_google");
tokenRequest.setCode_verifier("generated_code_verifier");
tokenRequest.setRedirect_uri("https://yourapp.com/oauth/callback");

// Use in service layer to exchange code for tokens
String accessToken = oauthService.exchangeCodeForToken(tokenRequest);
```

This DTO is typically used in REST controllers to capture the OAuth callback parameters and pass them to service layers that handle the token exchange with Google's token endpoint.
