A data transfer object (DTO) that represents an OAuth token response, containing authentication tokens and metadata returned from OAuth authentication endpoints.

## Key Components

- **TokenResponse**: Main DTO class with builder pattern support
- **accessToken**: The OAuth access token for API authentication
- **refreshToken**: Token used to refresh expired access tokens
- **tokenType**: Type of token (typically "Bearer")
- **expiresIn**: Token expiration time in seconds

## Usage Example

```java
// Creating a token response
TokenResponse response = TokenResponse.builder()
    .accessToken("eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .refreshToken("def50200a8b7c9d1e2f3...")
    .tokenType("Bearer")
    .expiresIn(3600)
    .build();

// Accessing token information
String token = response.getAccessToken();
int expiry = response.getExpiresIn();

// JSON serialization automatically maps to snake_case
// {
//   "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
//   "refresh_token": "def50200a8b7c9d1e2f3...",
//   "token_type": "Bearer",
//   "expires_in": 3600
// }
```
