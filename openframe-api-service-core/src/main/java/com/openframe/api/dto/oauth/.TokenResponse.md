A data transfer object (DTO) representing OAuth 2.0 token responses, containing access tokens, refresh tokens, and related metadata with proper JSON serialization mappings.

## Key Components

- **accessToken** - The bearer token for API authentication
- **refreshToken** - Token used to obtain new access tokens
- **tokenType** - Type of token (typically "Bearer")
- **expiresIn** - Token expiration time in seconds
- **@JsonProperty** annotations - Map Java field names to OAuth 2.0 standard JSON property names
- **@Data** - Lombok annotation generating getters, setters, equals, hashCode, and toString
- **@Builder** - Lombok annotation providing builder pattern implementation

## Usage Example

```java
// Creating a token response using builder pattern
TokenResponse response = TokenResponse.builder()
    .accessToken("eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .refreshToken("def50200e3b0c44298fc1c149afbf4c8996fb924...")
    .tokenType("Bearer")
    .expiresIn(3600)
    .build();

// JSON serialization will produce OAuth 2.0 compliant format:
// {
//   "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
//   "refresh_token": "def50200e3b0c44298fc1c149afbf4c8996fb924...",
//   "token_type": "Bearer",
//   "expires_in": 3600
// }

// Accessing token data
String token = response.getAccessToken();
boolean isExpired = System.currentTimeMillis() > (response.getExpiresIn() * 1000);
```
