Data transfer object representing the response for a single machine's force client update operation status.

## Key Components

- **machineId** - String identifier for the target machine
- **status** - `ForceAgentStatus` enum indicating the current update status
- **@Data** - Lombok annotation providing automatic getter/setter generation

## Usage Example

```java
// Creating a response item
ForceClientUpdateResponseItem responseItem = new ForceClientUpdateResponseItem();
responseItem.setMachineId("machine-001");
responseItem.setStatus(ForceAgentStatus.UPDATE_IN_PROGRESS);

// Accessing response data
String machineId = responseItem.getMachineId();
ForceAgentStatus currentStatus = responseItem.getStatus();

// Typically used in collections for batch operations
List<ForceClientUpdateResponseItem> batchResults = Arrays.asList(
    new ForceClientUpdateResponseItem("machine-001", ForceAgentStatus.SUCCESS),
    new ForceClientUpdateResponseItem("machine-002", ForceAgentStatus.FAILED)
);
```

This DTO is commonly used in API responses when performing bulk force update operations across multiple OpenFrame client machines, allowing the caller to track the individual status of each machine in the update process.
