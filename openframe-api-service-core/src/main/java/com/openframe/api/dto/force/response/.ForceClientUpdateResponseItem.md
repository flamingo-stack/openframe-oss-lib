A response item representing the status of a force client update operation for a specific machine, containing the machine identifier and corresponding agent status.

## Key Components

- **`machineId`** - String identifier for the target machine
- **`status`** - `ForceAgentStatus` enum indicating the current status of the force update operation
- **`@Data`** - Lombok annotation providing automatic getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Creating a response item
ForceClientUpdateResponseItem responseItem = new ForceClientUpdateResponseItem();
responseItem.setMachineId("MACHINE_001");
responseItem.setStatus(ForceAgentStatus.SUCCESS);

// Processing response items in a collection
List<ForceClientUpdateResponseItem> responses = forceUpdateService.executeUpdate(targetMachines);
for (ForceClientUpdateResponseItem item : responses) {
    System.out.println("Machine " + item.getMachineId() + 
                      " update status: " + item.getStatus());
}

// Filtering failed updates
List<ForceClientUpdateResponseItem> failedUpdates = responses.stream()
    .filter(item -> item.getStatus() == ForceAgentStatus.FAILED)
    .collect(Collectors.toList());
```
