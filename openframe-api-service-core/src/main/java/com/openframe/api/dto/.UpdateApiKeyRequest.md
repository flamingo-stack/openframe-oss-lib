A data transfer object (DTO) representing a request to update an existing API key's properties. This record uses Lombok's `@Builder` annotation to provide a fluent builder pattern for object construction.

## Key Components

- **name** - Optional new name for the API key
- **description** - Optional new description text
- **enabled** - Optional boolean to enable/disable the key
- **expiresAt** - Optional expiration timestamp using `Instant`
- **@Builder** - Lombok annotation providing builder pattern support

## Usage Example

```java
// Update API key with new name and description
UpdateApiKeyRequest request = UpdateApiKeyRequest.builder()
    .name("Production API Key")
    .description("Key for production environment access")
    .enabled(true)
    .build();

// Update only the expiration date
UpdateApiKeyRequest expirationUpdate = UpdateApiKeyRequest.builder()
    .expiresAt(Instant.now().plusSeconds(86400 * 30)) // 30 days
    .build();

// Disable an API key
UpdateApiKeyRequest disableRequest = UpdateApiKeyRequest.builder()
    .enabled(false)
    .build();
```

All fields are optional, allowing partial updates where only specific properties need to be modified while leaving others unchanged.
