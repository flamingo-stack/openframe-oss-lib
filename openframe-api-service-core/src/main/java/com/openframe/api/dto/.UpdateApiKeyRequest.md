A Data Transfer Object (DTO) record that encapsulates the request parameters for updating an existing API key in the OpenFrame system.

## Key Components

- **Record Fields**: `name`, `description`, `enabled`, `expiresAt` - all optional fields for partial updates
- **@Builder**: Lombok annotation enabling fluent builder pattern construction
- **Immutable Design**: Record type ensures immutability and automatic generation of equals/hashCode/toString methods

## Usage Example

```java
// Create update request with builder pattern
UpdateApiKeyRequest updateRequest = UpdateApiKeyRequest.builder()
    .name("Updated Production Key")
    .description("Updated API key for production environment")
    .enabled(false)
    .expiresAt(Instant.now().plusDays(90))
    .build();

// Partial update - only change specific fields
UpdateApiKeyRequest partialUpdate = UpdateApiKeyRequest.builder()
    .enabled(true)
    .build();

// Access fields
String newName = updateRequest.name();
Boolean isEnabled = updateRequest.enabled();
```

The record supports partial updates by allowing any combination of fields to be set, with null values indicating fields that should remain unchanged during the update operation.
