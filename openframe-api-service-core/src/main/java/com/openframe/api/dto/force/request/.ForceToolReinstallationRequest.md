A DTO class for requesting forced reinstallation of tools on specified machines through the OpenFrame API.

## Key Components

- **machineIds** - List of machine identifiers where tool reinstallation should be forced
- **toolAgentId** - Identifier of the specific tool agent to be reinstalled
- **@Data** - Lombok annotation providing getter/setter methods and other utility methods

## Usage Example

```java
// Create a force tool reinstallation request
ForceToolReinstallationRequest request = new ForceToolReinstallationRequest();

// Set the machines where reinstallation should occur
List<String> targetMachines = Arrays.asList("machine-001", "machine-002", "machine-003");
request.setMachineIds(targetMachines);

// Specify which tool agent to reinstall
request.setToolAgentId("monitoring-agent-v2.1");

// Send request through API client
apiClient.forceToolReinstallation(request);
```

This request object is typically used when tools need to be forcibly reinstalled across multiple machines, such as during updates, configuration changes, or troubleshooting scenarios.
