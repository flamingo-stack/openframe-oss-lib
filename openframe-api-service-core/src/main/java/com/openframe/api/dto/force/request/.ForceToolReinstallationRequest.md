A simple DTO (Data Transfer Object) for requesting forced reinstallation of tools on specified machines.

## Key Components

- **machineIds** - List of machine identifiers where tool reinstallation should be forced
- **toolAgentId** - Identifier for the specific tool agent to reinstall
- **@Data** - Lombok annotation providing automatic getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create a request to force tool reinstallation
ForceToolReinstallationRequest request = new ForceToolReinstallationRequest();

// Set the machines that need tool reinstallation
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));

// Specify which tool agent to reinstall
request.setToolAgentId("monitoring-agent-v1.2.3");

// Send the request (typically via REST client or service layer)
apiClient.forceToolReinstallation(request);
```

```java
// Alternative constructor-based usage
List<String> targetMachines = List.of("server-alpha", "server-beta");
ForceToolReinstallationRequest request = new ForceToolReinstallationRequest();
request.setMachineIds(targetMachines);
request.setToolAgentId("security-scanner-v2.1.0");
```

This request object is typically used in administrative scenarios where tools need to be forcibly reinstalled across multiple machines, such as after configuration changes or when resolving deployment issues.
