DTO for linking temporary attachments to a knowledge base article, carrying the target article ID and a list of temporary file identifiers for permanent association.

## Key Components

| Field | Type | Constraints | Description |
|-------|------|-------------|-------------|
| `articleId` | `String` | `@NotBlank` | Target knowledge base article ID |
| `tempIds` | `List<String>` | `@NotEmpty`, `@Size(max=50)` | Temporary attachment IDs to link (max 50) |

## Usage Example

```java
LinkKnowledgeBaseTempAttachmentsInput input = LinkKnowledgeBaseTempAttachmentsInput.builder()
    .articleId("article-abc123")
    .tempIds(List.of("temp-001", "temp-002", "temp-003"))
    .build();
```
