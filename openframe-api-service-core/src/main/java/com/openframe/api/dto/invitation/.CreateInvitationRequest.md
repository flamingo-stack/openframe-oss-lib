A data transfer object (DTO) that represents a request to create a user invitation with email validation and role assignments.

## Key Components

- **CreateInvitationRequest**: Main DTO class with Lombok annotations for data handling
- **email**: Email field with custom `@ValidEmail` validation annotation
- **roles**: List of `Role` objects to be assigned to the invited user
- **@Builder**: Lombok annotation enabling builder pattern construction
- **@Data**: Lombok annotation providing getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create invitation request using builder pattern
CreateInvitationRequest request = CreateInvitationRequest.builder()
    .email("user@example.com")
    .roles(List.of(Role.USER, Role.EDITOR))
    .build();

// Alternative construction
CreateInvitationRequest request = new CreateInvitationRequest();
request.setEmail("admin@company.com");
request.setRoles(Arrays.asList(Role.ADMIN));

// Access fields
String email = request.getEmail();
List<Role> userRoles = request.getRoles();
```

The class leverages Lombok to reduce boilerplate code and includes built-in email validation through the custom `@ValidEmail` annotation, making it suitable for API endpoints that handle user invitation creation.
