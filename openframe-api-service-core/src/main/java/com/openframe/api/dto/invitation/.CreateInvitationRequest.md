A data transfer object for creating user invitation requests with email validation and role assignment capabilities.

## Key Components

- **`email`** - The recipient's email address with custom validation via `@ValidEmail`
- **`roles`** - A list of `Role` objects to be assigned to the invited user
- **Lombok annotations** - `@Data` and `@Builder` for automatic getter/setter generation and builder pattern support

## Usage Example

```java
// Create an invitation request using the builder pattern
CreateInvitationRequest request = CreateInvitationRequest.builder()
    .email("user@example.com")
    .roles(List.of(Role.USER, Role.MODERATOR))
    .build();

// Access the email field
String recipientEmail = request.getEmail();

// Access the roles
List<Role> assignedRoles = request.getRoles();
```

The class leverages custom validation through the `@ValidEmail` annotation to ensure email format compliance, while the builder pattern provides a clean and flexible way to construct invitation requests with varying role combinations.
