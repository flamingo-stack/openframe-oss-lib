DTO representing the GraphQL mutation payload for uploading a temporary attachment to a Knowledge Base entry, encapsulating either a successful `TempAttachment` result or a list of user-facing errors.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `tempAttachment` | `TempAttachment` | The created temporary attachment document on success |
| `userErrors` | `List<MutationError>` | List of mutation errors; defaults to an empty list |
| `success(TempAttachment)` | `static` factory | Builds a payload with a populated `tempAttachment` and no errors |
| `error(String)` | `static` factory | Builds a payload with a single `MutationError` and no attachment |

## Usage Example

```java
// Success case — attachment uploaded successfully
TempAttachment attachment = storageService.uploadTemp(file);
KnowledgeBaseTempAttachmentPayload payload =
    KnowledgeBaseTempAttachmentPayload.success(attachment);

// Error case — validation or upload failure
KnowledgeBaseTempAttachmentPayload payload =
    KnowledgeBaseTempAttachmentPayload.error("File size exceeds the allowed limit.");

// Checking the result
if (payload.getUserErrors().isEmpty()) {
    TempAttachment result = payload.getTempAttachment();
    // proceed with result
} else {
    payload.getUserErrors().forEach(e -> log.warn("Upload error: {}", e.getMessage()));
}
```
