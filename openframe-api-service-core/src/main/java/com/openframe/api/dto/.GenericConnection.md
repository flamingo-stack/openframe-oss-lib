A generic GraphQL connection wrapper that provides paginated data with cursor-based navigation, commonly used in GraphQL APIs to return lists of data with pagination metadata.

## Key Components

- **GenericConnection<T>** - Generic class that wraps paginated data where T must extend GenericEdge
- **edges** - List of edge objects containing the actual data items
- **pageInfo** - Cursor-based pagination information (hasNextPage, hasPreviousPage, cursors, etc.)
- **Lombok annotations** - Provides automatic generation of getters, setters, builders, and constructors

## Usage Example

```java
// Create a connection for user data
List<UserEdge> userEdges = Arrays.asList(
    UserEdge.builder().node(user1).cursor("cursor1").build(),
    UserEdge.builder().node(user2).cursor("cursor2").build()
);

CursorPageInfo pageInfo = CursorPageInfo.builder()
    .hasNextPage(true)
    .hasPreviousPage(false)
    .startCursor("cursor1")
    .endCursor("cursor2")
    .build();

GenericConnection<UserEdge> connection = GenericConnection.<UserEdge>builder()
    .edges(userEdges)
    .pageInfo(pageInfo)
    .build();

// Access paginated data
boolean hasMore = connection.getPageInfo().isHasNextPage();
List<UserEdge> data = connection.getEdges();
```
