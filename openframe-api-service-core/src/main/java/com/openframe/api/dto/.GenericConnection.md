A generic GraphQL connection wrapper that provides pagination support for any type of edge data structure.

## Key Components

- **GenericConnection<T>** - Main class that wraps paginated data with cursor-based pagination metadata
- **edges** - List of generic edges containing the actual data nodes
- **pageInfo** - Cursor-based pagination information including hasNextPage, hasPreviousPage, and cursor details

## Usage Example

```java
// Create a connection with user edges
GenericConnection<UserEdge> userConnection = GenericConnection.<UserEdge>builder()
    .edges(userEdges)
    .pageInfo(CursorPageInfo.builder()
        .hasNextPage(true)
        .hasPreviousPage(false)
        .startCursor("cursor1")
        .endCursor("cursor10")
        .build())
    .build();

// Access paginated data
List<UserEdge> users = userConnection.getEdges();
boolean hasMore = userConnection.getPageInfo().isHasNextPage();
```

This class follows the GraphQL Relay specification for cursor-based pagination, making it suitable for implementing paginated APIs that work efficiently with large datasets.
