A data transfer object (DTO) for requesting the forced installation of all tools for a specific agent in the OpenFrame API system.

## Key Components

**Main Class:**
- `ForceToolInstallationAllRequest` - Request DTO containing the target agent identifier

**Fields:**
- `toolAgentId` - String identifier specifying which agent should have all tools forcibly installed

**Annotations:**
- `@Data` - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a request to force install all tools for a specific agent
ForceToolInstallationAllRequest request = new ForceToolInstallationAllRequest();
request.setToolAgentId("agent-12345");

// The request can now be sent to the API endpoint
// POST /api/force/tool-installation/all
apiClient.forceInstallAllTools(request);

// Alternative constructor usage (if available through Lombok)
ForceToolInstallationAllRequest request2 = new ForceToolInstallationAllRequest();
request2.setToolAgentId("production-agent-001");
```

This DTO is typically used when an administrator needs to override normal installation processes and ensure all available tools are installed on a target agent, regardless of existing installation states or dependencies.
