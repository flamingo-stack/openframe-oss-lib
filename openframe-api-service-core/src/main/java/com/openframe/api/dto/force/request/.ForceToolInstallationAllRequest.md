A DTO (Data Transfer Object) for requesting forced installation of all tools on a specific tool agent in the OpenFrame API system.

## Key Components

- **`toolAgentId`** - String field identifying the target tool agent where all tools should be force-installed
- **`@Data`** - Lombok annotation providing automatic generation of getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a request to force install all tools on a specific agent
ForceToolInstallationAllRequest request = new ForceToolInstallationAllRequest();
request.setToolAgentId("agent-12345");

// The request can then be sent via REST API or service layer
// Example with a hypothetical service
toolService.forceInstallAllTools(request);

// Alternative constructor-like usage (if using Lombok @AllArgsConstructor)
// ForceToolInstallationAllRequest request = new ForceToolInstallationAllRequest("agent-12345");
```

This request object is typically used in scenarios where you need to trigger a bulk installation of all available tools on a particular agent, bypassing normal installation checks or prerequisites.
