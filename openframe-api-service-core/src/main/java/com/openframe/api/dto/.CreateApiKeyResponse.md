A response DTO that encapsulates both the API key metadata and the complete secret key value returned after creating a new API key.

## Key Components

- **CreateApiKeyResponse** - Record class that serves as a data transfer object for API key creation responses
- **apiKey** - Contains the API key metadata (likely ID, name, permissions, etc.)
- **fullKey** - The complete secret key string that should be securely stored by the client
- **@Builder** - Lombok annotation providing a builder pattern for object construction

## Usage Example

```java
// Creating a response after API key generation
CreateApiKeyResponse response = CreateApiKeyResponse.builder()
    .apiKey(ApiKeyResponse.builder()
        .id("key-123")
        .name("Production Key")
        .createdAt(Instant.now())
        .build())
    .fullKey("sk_live_abc123def456ghi789")
    .build();

// Accessing the components
String secretKey = response.fullKey();
ApiKeyResponse metadata = response.apiKey();

// Typical usage in API controller
@PostMapping("/api-keys")
public CreateApiKeyResponse createApiKey(@RequestBody CreateApiKeyRequest request) {
    // Generate key and metadata
    return keyService.createApiKey(request);
}
```
