A simple DTO (Data Transfer Object) class representing a client configuration response containing version information, using Lombok annotations for boilerplate code generation.

## Key Components

- **ClientConfigurationResponse**: Main response class with a single `version` field
- **Lombok Annotations**:
  - `@Data` - Generates getters, setters, toString, equals, and hashCode methods
  - `@Builder` - Provides builder pattern implementation
  - `@NoArgsConstructor` - Creates default constructor
  - `@AllArgsConstructor` - Creates constructor with all fields

## Usage Example

```java
// Create using builder pattern
ClientConfigurationResponse response = ClientConfigurationResponse.builder()
    .version("1.0.0")
    .build();

// Create using constructor
ClientConfigurationResponse response2 = new ClientConfigurationResponse("1.0.0");

// Access version
String version = response.getVersion();

// Modify version
response.setVersion("1.1.0");
```

This class is typically used in API responses to communicate the current version of client configuration to consuming applications, providing a standardized way to handle version information in the OpenFrame API.
