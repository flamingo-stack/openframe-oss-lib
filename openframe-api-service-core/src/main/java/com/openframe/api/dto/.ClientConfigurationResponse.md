A simple DTO (Data Transfer Object) class that represents a response containing client configuration information, specifically the version of the client configuration.

## Key Components

- **ClientConfigurationResponse** - Main response class with Lombok annotations for automatic code generation
- **version** - String field representing the configuration version
- **@Data** - Generates getters, setters, toString, equals, and hashCode methods
- **@Builder** - Enables builder pattern for object construction
- **@NoArgsConstructor/@AllArgsConstructor** - Generates default and full parameter constructors

## Usage Example

```java
// Using builder pattern
ClientConfigurationResponse response = ClientConfigurationResponse.builder()
    .version("1.2.3")
    .build();

// Using constructor
ClientConfigurationResponse response = new ClientConfigurationResponse("1.2.3");

// Using setters
ClientConfigurationResponse response = new ClientConfigurationResponse();
response.setVersion("1.2.3");

// Accessing the version
String configVersion = response.getVersion();
```

This class is typically used in REST API responses to communicate the current client configuration version to consuming applications.
