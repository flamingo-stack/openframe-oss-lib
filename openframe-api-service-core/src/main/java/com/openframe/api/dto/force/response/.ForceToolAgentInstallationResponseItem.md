A response data transfer object that represents the installation status of a Force tool agent on a specific machine. This class is used in API responses to communicate the outcome of agent installation operations.

## Key Components

- **machineId** - Unique identifier for the target machine where the agent is being installed
- **toolAgentId** - Unique identifier for the specific tool agent being installed
- **status** - Current installation status using the `ForceAgentStatus` enum
- **@Data** - Lombok annotation that automatically generates getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Creating a response item for a successful installation
ForceToolAgentInstallationResponseItem responseItem = new ForceToolAgentInstallationResponseItem();
responseItem.setMachineId("machine-001");
responseItem.setToolAgentId("agent-xyz-123");
responseItem.setStatus(ForceAgentStatus.INSTALLED);

// Typically used in a list response
List<ForceToolAgentInstallationResponseItem> installationResults = Arrays.asList(
    responseItem
);

// Accessing the data
String machineId = responseItem.getMachineId();
ForceAgentStatus currentStatus = responseItem.getStatus();
```

This DTO is commonly returned from force tool agent installation endpoints to provide clients with detailed status information for each machine-agent combination in batch operations.
