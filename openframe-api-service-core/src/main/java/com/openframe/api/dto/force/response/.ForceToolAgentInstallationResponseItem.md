A data transfer object representing the response for individual force tool agent installation operations, containing machine identification and installation status information.

## Key Components

- **machineId** - String identifier for the target machine where the agent is being installed
- **toolAgentId** - String identifier for the specific tool agent being installed
- **status** - ForceAgentStatus enum indicating the current installation status
- **@Data** - Lombok annotation providing automatic getter/setter methods, toString, equals, and hashCode

## Usage Example

```java
// Creating a response item for a successful installation
ForceToolAgentInstallationResponseItem responseItem = new ForceToolAgentInstallationResponseItem();
responseItem.setMachineId("machine-001");
responseItem.setToolAgentId("agent-toolkit-v2.1");
responseItem.setStatus(ForceAgentStatus.INSTALLED);

// Accessing the response data
String targetMachine = responseItem.getMachineId();
String agentVersion = responseItem.getToolAgentId();
ForceAgentStatus installationResult = responseItem.getStatus();

// Typically used in collections for batch installation responses
List<ForceToolAgentInstallationResponseItem> installationResults = 
    installationService.installAgents(targetMachines, agentConfig);
```
