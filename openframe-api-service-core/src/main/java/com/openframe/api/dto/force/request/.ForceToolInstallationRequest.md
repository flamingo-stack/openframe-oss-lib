A data transfer object that represents a request to force installation of a tool agent on specified machines in the OpenFrame system.

## Key Components

- **`machineIds`** - List of machine identifiers where the tool should be forcefully installed
- **`toolAgentId`** - Identifier of the specific tool agent to be installed
- **`@Data`** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a force installation request for multiple machines
ForceToolInstallationRequest request = new ForceToolInstallationRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));
request.setToolAgentId("diagnostic-tool-v2.1");

// Send request through API service
apiService.forceToolInstallation(request);

// Alternative: using builder pattern if available
ForceToolInstallationRequest request2 = new ForceToolInstallationRequest();
request2.setMachineIds(List.of("machine-004"));
request2.setToolAgentId("monitoring-agent-v1.5");
```

This DTO is typically used when standard tool installation processes have failed and administrators need to override normal deployment procedures to ensure critical tools are installed on target machines.
