A data transfer object that represents a request to force tool installation on multiple machines using a specific tool agent.

## Key Components

- **`machineIds`** - List of machine identifiers where the tool installation should be forced
- **`toolAgentId`** - Identifier of the tool agent that will perform the installation
- **`@Data`** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a force installation request for multiple machines
ForceToolInstallationRequest request = new ForceToolInstallationRequest();
request.setMachineIds(Arrays.asList("machine-001", "machine-002", "machine-003"));
request.setToolAgentId("agent-12345");

// Use in API call or service method
toolService.forceInstallation(request);

// Alternative construction using builder pattern (if available)
ForceToolInstallationRequest request = new ForceToolInstallationRequest();
request.setMachineIds(List.of("prod-server-01", "prod-server-02"));
request.setToolAgentId("deployment-agent-v2");
```

This DTO is typically used in REST API endpoints or service methods that need to trigger tool installations across multiple target machines simultaneously.
