A data transfer object that represents a paginated response containing invitation data, extending the core pagination functionality with invitation-specific typing.

## Key Components

- **InvitationPageResponse**: A paginated response wrapper specifically typed for `InvitationResponse` objects
- **@SuperBuilder**: Enables builder pattern inheritance from the parent `PageResponse` class
- **@EqualsAndHashCode**: Generates equality methods including parent class fields via `callSuper = true`

## Usage Example

```java
// Creating a paginated invitation response
InvitationPageResponse response = InvitationPageResponse.builder()
    .content(invitationList)
    .page(0)
    .size(10)
    .totalElements(25L)
    .totalPages(3)
    .build();

// Using in a REST controller
@GetMapping("/invitations")
public InvitationPageResponse getInvitations(
    @RequestParam(defaultValue = "0") int page,
    @RequestParam(defaultValue = "10") int size) {
    
    Page<Invitation> invitationPage = invitationService.findAll(
        PageRequest.of(page, size)
    );
    
    return InvitationPageResponse.builder()
        .content(invitationPage.getContent().stream()
            .map(this::toInvitationResponse)
            .collect(Collectors.toList()))
        .page(invitationPage.getNumber())
        .size(invitationPage.getSize())
        .totalElements(invitationPage.getTotalElements())
        .totalPages(invitationPage.getTotalPages())
        .build();
}
```
