A paginated response wrapper for invitation data that extends the base PageResponse class to provide invitation-specific pagination functionality.

## Key Components

- **InvitationPageResponse** - Main class extending `PageResponse<InvitationResponse>`
- **@SuperBuilder** - Lombok annotation enabling builder pattern with inheritance
- **@EqualsAndHashCode(callSuper = true)** - Lombok annotation for proper equality comparison including parent class fields

## Usage Example

```java
// Building a paginated invitation response
InvitationPageResponse response = InvitationPageResponse.builder()
    .content(invitationList)
    .totalElements(100)
    .totalPages(10)
    .currentPage(1)
    .pageSize(10)
    .build();

// Using in a REST controller
@GetMapping("/invitations")
public InvitationPageResponse getInvitations(
    @RequestParam(defaultValue = "0") int page,
    @RequestParam(defaultValue = "10") int size) {
    
    return invitationService.getInvitationsPaginated(page, size);
}

// Accessing pagination data
List<InvitationResponse> invitations = response.getContent();
int totalPages = response.getTotalPages();
boolean hasNext = response.hasNext();
```
