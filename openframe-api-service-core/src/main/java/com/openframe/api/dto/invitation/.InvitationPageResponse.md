A paginated response DTO for invitation listings, extending the generic `PageResponse` with `InvitationResponse` as its element type.

## Key Components

| Element | Description |
|---|---|
| `InvitationPageResponse` | Typed pagination wrapper for `InvitationResponse` items |
| `PageResponse<InvitationResponse>` | Parent class providing standard pagination metadata (page, size, total, content) |
| `@SuperBuilder` | Enables Lombok's builder pattern with inheritance support |
| `@EqualsAndHashCode(callSuper = true)` | Includes parent class fields in equality checks |

## Usage Example

```java
// Building a paginated invitation response
InvitationPageResponse response = InvitationPageResponse.builder()
        .content(List.of(invitationResponse1, invitationResponse2))
        .page(0)
        .size(10)
        .totalElements(50L)
        .totalPages(5)
        .build();

// Typically returned from a service or controller
@GetMapping("/invitations")
public InvitationPageResponse listInvitations(Pageable pageable) {
    return invitationService.findAll(pageable);
}
```

> **Note:** All pagination fields (content list, page index, page size, total counts) are inherited from `PageResponse`. This class serves as a concrete, type-safe specialization for invitation data — no additional fields are needed.
