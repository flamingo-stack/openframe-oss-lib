A simple data transfer object (DTO) record that encapsulates a request to update a device's status in the OpenFrame API, using Jakarta Bean Validation to ensure the status field is provided.

## Key Components

- **UpdateDeviceStatusRequest** - Record class representing the request payload
- **status** - Required `DeviceStatus` enum field with `@NotNull` validation
- **@NotNull** - Jakarta validation annotation ensuring the status cannot be null

## Usage Example

```java
// Create a request to update device status
UpdateDeviceStatusRequest request = new UpdateDeviceStatusRequest(DeviceStatus.ONLINE);

// Validation will ensure status is not null
UpdateDeviceStatusRequest invalidRequest = new UpdateDeviceStatusRequest(null); // Will fail validation

// Typical usage in a REST controller
@PutMapping("/devices/{deviceId}/status")
public ResponseEntity<Void> updateDeviceStatus(
    @PathVariable String deviceId,
    @Valid @RequestBody UpdateDeviceStatusRequest request) {
    
    deviceService.updateStatus(deviceId, request.status());
    return ResponseEntity.ok().build();
}
```

The record automatically provides constructor, getters, equals, hashCode, and toString methods, making it ideal for immutable request objects in REST APIs.
