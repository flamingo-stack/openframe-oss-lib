A data transfer object (DTO) record that encapsulates a request to update a device's status in the OpenFrame API. This immutable record ensures type safety and validation for device status update operations.

## Key Components

- **UpdateDeviceStatusRequest** - Record class representing the request payload
- **status** - Required field of type `DeviceStatus` with `@NotNull` validation
- **@NotNull** - Jakarta validation constraint ensuring the status cannot be null

## Usage Example

```java
// Create a status update request
DeviceStatus newStatus = DeviceStatus.ONLINE;
UpdateDeviceStatusRequest request = new UpdateDeviceStatusRequest(newStatus);

// In a REST controller
@PutMapping("/devices/{id}/status")
public ResponseEntity<Void> updateDeviceStatus(
    @PathVariable String id,
    @Valid @RequestBody UpdateDeviceStatusRequest request) {
    
    deviceService.updateStatus(id, request.status());
    return ResponseEntity.ok().build();
}

// Validation will automatically fail if status is null
UpdateDeviceStatusRequest invalidRequest = new UpdateDeviceStatusRequest(null); // Throws validation error
```

The record automatically generates constructor, accessor methods, equals(), hashCode(), and toString() methods, making it ideal for API request handling with minimal boilerplate code.
