A data transfer object (DTO) that encapsulates the request parameters needed to create a new API key in the OpenFrame system. This immutable record uses the Builder pattern for flexible object construction.

## Key Components

- **`name`** - String identifier for the API key
- **`description`** - Optional descriptive text about the API key's purpose
- **`expiresAt`** - Instant timestamp defining when the API key should expire
- **`@Builder`** - Lombok annotation enabling fluent builder pattern construction

## Usage Example

```java
// Create API key request with all fields
CreateApiKeyRequest request = CreateApiKeyRequest.builder()
    .name("production-api-key")
    .description("API key for production environment access")
    .expiresAt(Instant.now().plusSeconds(86400 * 30)) // 30 days from now
    .build();

// Create minimal request with just name
CreateApiKeyRequest minimalRequest = CreateApiKeyRequest.builder()
    .name("dev-key")
    .build();

// Access fields
String keyName = request.name();
Instant expiration = request.expiresAt();
```

The record provides immutability and automatic generation of equals, hashCode, and toString methods, making it ideal for API request handling where data integrity is important.
