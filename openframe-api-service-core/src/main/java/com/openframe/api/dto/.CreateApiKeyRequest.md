A data transfer object (DTO) that encapsulates the request parameters needed to create a new API key in the system.

## Key Components

- **name** - String identifier for the API key
- **description** - Optional descriptive text for the API key's purpose
- **expiresAt** - Instant timestamp when the API key should expire
- **@Builder** - Lombok annotation enabling fluent builder pattern construction

## Usage Example

```java
// Create an API key request with all fields
CreateApiKeyRequest request = CreateApiKeyRequest.builder()
    .name("production-service-key")
    .description("API key for production microservice authentication")
    .expiresAt(Instant.now().plusSeconds(86400 * 30)) // 30 days from now
    .build();

// Create with minimal required fields
CreateApiKeyRequest simpleRequest = CreateApiKeyRequest.builder()
    .name("dev-key")
    .build();

// Use in API controller
@PostMapping("/api-keys")
public ResponseEntity<ApiKeyResponse> createApiKey(@RequestBody CreateApiKeyRequest request) {
    return apiKeyService.createApiKey(request);
}
```
