A simple response DTO that wraps a list of force client update response items, typically used to return multiple update results from batch operations.

## Key Components

- **`items`**: List of `ForceClientUpdateResponseItem` objects containing individual update results
- **`@Data`**: Lombok annotation providing automatic getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Creating a response with multiple update results
ForceClientUpdateResponse response = new ForceClientUpdateResponse();
List<ForceClientUpdateResponseItem> updateItems = Arrays.asList(
    new ForceClientUpdateResponseItem("client1", true, "Updated successfully"),
    new ForceClientUpdateResponseItem("client2", false, "Update failed")
);
response.setItems(updateItems);

// Accessing update results
response.getItems().forEach(item -> {
    System.out.println("Client: " + item.getClientId() + 
                      ", Success: " + item.isSuccess());
});
```

This DTO follows the common pattern of wrapping collections in response objects, allowing for future extensibility with additional metadata while maintaining a clean API structure for batch force client update operations.
