A response DTO that contains a list of force client update response items, typically used to communicate batch update results from server to client.

## Key Components

- **`items`** - List of `ForceClientUpdateResponseItem` objects representing individual update results
- **`@Data`** - Lombok annotation providing automatic generation of getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Creating a response with multiple update results
ForceClientUpdateResponse response = new ForceClientUpdateResponse();
List<ForceClientUpdateResponseItem> updateItems = Arrays.asList(
    new ForceClientUpdateResponseItem(),
    new ForceClientUpdateResponseItem()
);
response.setItems(updateItems);

// Processing response items
if (response.getItems() != null) {
    for (ForceClientUpdateResponseItem item : response.getItems()) {
        // Process each update result
        System.out.println("Update item: " + item);
    }
}

// Typical usage in REST controller
@PostMapping("/force-update")
public ForceClientUpdateResponse handleForceUpdate(@RequestBody ForceUpdateRequest request) {
    // Process updates and return response
    return forceUpdateService.processUpdates(request);
}
```
