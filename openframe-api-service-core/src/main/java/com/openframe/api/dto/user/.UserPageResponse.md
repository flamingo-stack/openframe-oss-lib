A DTO class that extends the generic PageResponse to provide paginated responses specifically for user data.

## Key Components

- **UserPageResponse** - Extends `PageResponse<UserResponse>` to create a specialized paginated response container for user entities
- **@SuperBuilder** - Lombok annotation enabling the builder pattern with inheritance support
- **@EqualsAndHashCode(callSuper = true)** - Generates equals and hashCode methods that include parent class fields
