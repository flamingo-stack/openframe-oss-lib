A specialized page response class that extends the generic PageResponse to handle paginated collections of UserResponse objects.

## Key Components

- **UserPageResponse**: Extends `PageResponse<UserResponse>` to provide type-safe pagination for user data
- **@SuperBuilder**: Lombok annotation enabling builder pattern inheritance from parent class
- **@EqualsAndHashCode(callSuper = true)**: Lombok annotation for proper equality comparison including parent fields

## Usage Example

```java
// Building a paginated user response
UserPageResponse response = UserPageResponse.builder()
    .content(userList)
    .totalElements(100L)
    .pageNumber(0)
    .pageSize(10)
    .build();

// Accessing paginated user data
List<UserResponse> users = response.getContent();
long totalUsers = response.getTotalElements();
boolean hasNext = response.hasNext();

// Using in REST controller
@GetMapping("/users")
public UserPageResponse getUsers(Pageable pageable) {
    Page<User> userPage = userService.findAll(pageable);
    return UserPageResponse.builder()
        .content(userMapper.toResponseList(userPage.getContent()))
        .totalElements(userPage.getTotalElements())
        .pageNumber(userPage.getNumber())
        .pageSize(userPage.getSize())
        .build();
}
```
