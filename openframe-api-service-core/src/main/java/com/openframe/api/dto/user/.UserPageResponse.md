Paginated response wrapper for user data, extending the generic `PageResponse` with `UserResponse` as the element type.

## Key Components

| Element | Type | Description |
|---------|------|-------------|
| `UserPageResponse` | Class | Extends `PageResponse<UserResponse>` for paginated user listings |
| `@SuperBuilder` | Lombok | Enables builder pattern inheritance from `PageResponse` |
| `@EqualsAndHashCode(callSuper = true)` | Lombok | Includes parent class fields in equality checks |

## Usage Example

```java
// Build a paginated user response
UserPageResponse response = UserPageResponse.builder()
        .content(List.of(userResponse1, userResponse2))
        .page(0)
        .size(20)
        .totalElements(100L)
        .totalPages(5)
        .build();

// Typically returned from a service/controller
return ResponseEntity.ok(response);
```

> The builder fields (`content`, `page`, `size`, etc.) are inherited from `PageResponse` via `@SuperBuilder`, so no additional fields are defined in this class.
