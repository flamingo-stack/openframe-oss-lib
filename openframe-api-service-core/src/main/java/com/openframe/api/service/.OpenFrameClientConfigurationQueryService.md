A Spring service that retrieves the most recent OpenFrame client configuration from the database and transforms it into a response DTO.

## Key Components

- **OpenFrameClientConfigurationQueryService** - Main service class annotated with `@Service` for dependency injection
- **get()** - Public method that fetches the latest client configuration by creation date
- **OpenFrameClientConfigurationRepository** - Injected repository for database access
- **ClientConfigurationResponse** - Response DTO containing configuration version

## Usage Example

```java
@RestController
public class ConfigurationController {
    
    private final OpenFrameClientConfigurationQueryService configService;
    
    @GetMapping("/api/client/config")
    public ClientConfigurationResponse getClientConfiguration() {
        return configService.get();
    }
}

// Service automatically injected by Spring
@Autowired
private OpenFrameClientConfigurationQueryService configService;

// Get latest configuration
ClientConfigurationResponse config = configService.get();
String version = config.getVersion();
```

The service uses repository method `findFirstByOrderByCreatedAtDesc()` to retrieve the most recently created configuration record, throwing an `IllegalStateException` if none exists. The response is built using the builder pattern, currently only exposing the configuration version field.
