A Spring service responsible for querying and retrieving OpenFrame client configuration data, providing the latest configuration version to API consumers.

## Key Components

- **OpenFrameClientConfigurationQueryService** - Main service class annotated with `@Service`
- **get()** - Primary method that retrieves the most recent client configuration
- **OpenFrameClientConfigurationRepository** - Injected repository for data access
- **ClientConfigurationResponse** - DTO for returning configuration data

## Usage Example

```java
@RestController
public class ConfigController {
    
    private final OpenFrameClientConfigurationQueryService configService;
    
    @GetMapping("/api/client-config")
    public ClientConfigurationResponse getClientConfig() {
        return configService.get();
    }
}

// Service usage in business logic
@Component
public class ClientManager {
    
    private final OpenFrameClientConfigurationQueryService configService;
    
    public void processClient() {
        ClientConfigurationResponse config = configService.get();
        String version = config.getVersion();
        // Process with current configuration version
    }
}
```

The service retrieves the latest configuration by creation date and throws an `IllegalStateException` if no configuration exists. It follows Spring's dependency injection pattern and includes comprehensive logging capabilities.
