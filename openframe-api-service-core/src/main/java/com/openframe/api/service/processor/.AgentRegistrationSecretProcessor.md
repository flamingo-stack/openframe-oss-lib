Processor interface defining post-operation hooks for agent registration secret lifecycle events, enabling integration with external systems such as shared clusters via Kafka.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `postProcessSecretGenerated` | `default` method | Hook invoked after a new agent registration secret is generated; receives the active secret and its plaintext key |
| `postProcessSecretDeactivated` | `default` method | Hook invoked after an agent registration secret is deactivated; receives the deactivated secret |

Both methods provide no-op default implementations, allowing implementors to override only the hooks relevant to their integration.

## Usage Example

```java
@Component
public class KafkaAgentSecretProcessor implements AgentRegistrationSecretProcessor {

    private final KafkaProducer kafkaProducer;

    @Override
    public void postProcessSecretGenerated(AgentRegistrationSecret secret, String decryptedKey) {
        // Sync new secret to shared cluster via Kafka
        kafkaProducer.send("agent-secrets", new SecretGeneratedEvent(secret, decryptedKey));
    }

    @Override
    public void postProcessSecretDeactivated(AgentRegistrationSecret secret) {
        // Notify shared cluster of secret deactivation
        kafkaProducer.send("agent-secrets", new SecretDeactivatedEvent(secret));
    }
}
```

Implement this interface to propagate agent registration secret changes to external systems. The default no-op methods ensure backward compatibility when only a subset of lifecycle events requires handling.
