A Spring service that handles forced tool agent installation and reinstallation operations across machines in an OpenFrame system. It provides methods to install/reinstall tool agents on specific machines or all machines in the system.

## Key Components

- **process()** - Installs a tool agent on specified machines
- **processAll()** - Installs a tool agent on all machines in the system
- **processReinstall()** - Reinstalls a tool agent on specified machines
- **processReinstallAll()** - Reinstalls a tool agent on all machines
- **processMachine()** - Core logic for processing a single machine installation
- **validateToolAgentId()** / **validateMachineIds()** - Input validation methods

## Usage Example

```java
@Autowired
private ForceToolInstallationService forceService;

// Install tool on specific machines
ForceToolInstallationRequest request = new ForceToolInstallationRequest();
request.setToolAgentId("agent-123");
request.setMachineIds(Arrays.asList("machine-1", "machine-2"));

ForceToolAgentInstallationResponse response = forceService.process(request);

// Install on all machines
ForceToolAgentInstallationResponse allResponse = forceService.processAll("agent-123");

// Reinstall on specific machines
ForceToolAgentInstallationResponse reinstallResponse = forceService.processReinstall(request);

// Check results
response.getItems().forEach(item -> {
    System.out.println("Machine: " + item.getMachineId() + 
                      " Status: " + item.getStatus());
});
```
