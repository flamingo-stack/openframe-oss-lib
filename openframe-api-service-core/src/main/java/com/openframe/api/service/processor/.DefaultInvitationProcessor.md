A default implementation of the `InvitationProcessor` interface that provides basic logging functionality for invitation lifecycle events. This component serves as a fallback when no custom invitation processor is configured.

## Key Components

- **DefaultInvitationProcessor**: Spring component implementing `InvitationProcessor` interface
- **postProcessInvitationCreated()**: Logs invitation creation events at debug level
- **postProcessInvitationRevoked()**: Logs invitation revocation events at debug level

## Usage Example

This processor is automatically registered when no custom implementation exists:

```java
// Custom processor (prevents default from loading)
@Component
public class CustomInvitationProcessor implements InvitationProcessor {
    @Override
    public void postProcessInvitationCreated(Invitation invitation) {
        // Send email notification
        emailService.sendInvitation(invitation);
    }
    
    @Override
    public void postProcessInvitationRevoked(Invitation invitation) {
        // Send cancellation email
        emailService.sendCancellation(invitation);
    }
}

// If no custom processor exists, DefaultInvitationProcessor will be used
// and will log: "Invitation created: id=123, email=user@example.com"
```

The `@ConditionalOnMissingBean` annotation ensures this default implementation only loads when no other `InvitationProcessor` bean is present in the application context.
