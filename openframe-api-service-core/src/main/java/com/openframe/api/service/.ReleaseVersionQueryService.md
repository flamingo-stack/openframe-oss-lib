Retrieves the current release version from the database, providing a lightweight query service for accessing `ReleaseVersion` document records.

## Key Components

| Element | Description |
|---|---|
| `ReleaseVersionQueryService` | Spring `@Service` bean for release version lookups |
| `getReleaseVersion()` | Returns the first available `ReleaseVersion` entry wrapped in an `Optional` |
| `ReleaseVersionRepository` | Injected repository delegate handling the actual data access |

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class VersionController {

    private final ReleaseVersionQueryService releaseVersionQueryService;

    @GetMapping("/version")
    public ResponseEntity<ReleaseVersion> getVersion() {
        return releaseVersionQueryService.getReleaseVersion()
                .map(ResponseEntity::ok)
                .orElse(ResponseEntity.notFound().build());
    }
}
```

> **Note:** `getReleaseVersion()` uses `findFirstBy()` under the hood, meaning it assumes at most one version record exists in the datastore at a time. If no record is found, an empty `Optional` is returned — callers should handle the absent case explicitly.
