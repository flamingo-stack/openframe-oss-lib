A Spring service responsible for querying release version information from the database, providing access to the current release version data.

## Key Components

- **ReleaseVersionQueryService**: Main service class that handles release version queries
- **getReleaseVersion()**: Retrieves the current release version by finding the first record ordered by creation date
- **ReleaseVersionRepository**: Injected repository dependency for database operations
- **ReleaseVersion**: Document entity representing version information

## Usage Example

```java
@Autowired
private ReleaseVersionQueryService releaseVersionQueryService;

public void displayCurrentVersion() {
    Optional<ReleaseVersion> currentVersion = releaseVersionQueryService.getReleaseVersion();
    
    if (currentVersion.isPresent()) {
        ReleaseVersion version = currentVersion.get();
        log.info("Current release version: {}", version.getVersionNumber());
    } else {
        log.warn("No release version found in database");
    }
}

// In a REST controller
@GetMapping("/api/version")
public ResponseEntity<ReleaseVersion> getCurrentVersion() {
    return releaseVersionQueryService.getReleaseVersion()
        .map(ResponseEntity::ok)
        .orElse(ResponseEntity.notFound().build());
}
```

The service uses constructor injection for dependencies and includes debug logging for query operations. It returns an `Optional` to handle cases where no version data exists in the database.
