Provides read-only access to the current release version of the OpenFrame platform by querying the underlying data store.

## Key Components

| Element | Description |
|---|---|
| `ReleaseVersionQueryService` | Spring `@Service` bean for querying release version data |
| `getReleaseVersion()` | Fetches the singleton `ReleaseVersion` document using `ReleaseVersion.DEFAULT_ID` |
| `ReleaseVersionRepository` | Injected repository (via `@RequiredArgsConstructor`) used to perform the lookup |

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class VersionController {

    private final ReleaseVersionQueryService releaseVersionQueryService;

    @GetMapping("/version")
    public ResponseEntity<ReleaseVersion> getVersion() {
        return releaseVersionQueryService.getReleaseVersion()
                .map(ResponseEntity::ok)
                .orElse(ResponseEntity.notFound().build());
    }
}
```

> **Note:** `getReleaseVersion()` returns an `Optional<ReleaseVersion>`, so callers should handle the case where no version document has been persisted yet (e.g., on a fresh deployment).
