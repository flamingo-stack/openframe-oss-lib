A Spring Boot service that manages API key lifecycle operations including creation, validation, updates, and statistics tracking with secure credential generation.

## Key Components

- **createApiKey()** - Generates new API keys with secure random credentials
- **getApiKeysForUser()** - Retrieves all API keys for a specific user
- **updateApiKey()** - Modifies API key properties (name, description, enabled status)
- **deleteApiKey()** - Removes API keys and associated statistics
- **regenerateApiKey()** - Creates new credentials while preserving metadata
- **disableExpiredKeys()** - Batch disables expired keys (scheduled task ready)
- **Secure Generation** - Uses `SecureRandom` with prefixed identifiers (`ak_` for key IDs, `sk_` for secrets)

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class ApiKeyController {
    
    private final ApiKeyService apiKeyService;
    
    @PostMapping("/api-keys")
    public CreateApiKeyResponse createKey(@RequestBody CreateApiKeyRequest request) {
        String userId = getCurrentUserId();
        return apiKeyService.createApiKey(userId, request);
    }
    
    @GetMapping("/api-keys")
    public List<ApiKeyResponse> getUserKeys() {
        return apiKeyService.getApiKeysForUser(getCurrentUserId());
    }
    
    @PutMapping("/api-keys/{keyId}")
    public ApiKeyResponse updateKey(@PathVariable String keyId, 
                                  @RequestBody UpdateApiKeyRequest request) {
        return apiKeyService.updateApiKey(keyId, getCurrentUserId(), request);
    }
}
```

The service integrates with MongoDB repositories for persistence and includes comprehensive statistics tracking through `ApiKeyStats` aggregation.
