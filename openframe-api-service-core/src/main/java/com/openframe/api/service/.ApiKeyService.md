Manages API key lifecycle operations including creation, retrieval, updates, and deletion for user authentication. Handles secure key generation, password encoding, and statistics tracking.

## Key Components

- **createApiKey()** - Creates new API keys with secure random generation
- **getApiKeysForUser()** / **getApiKeyById()** - Retrieves API keys for users
- **updateApiKey()** - Updates API key properties (name, description, enabled status, expiration)
- **deleteApiKey()** - Removes API key and associated statistics
- **regenerateApiKey()** - Creates new credentials while preserving metadata
- **disableExpiredKeys()** - Maintenance method to disable expired keys
- **mapToResponse()** - Converts entities to DTOs with usage statistics

## Usage Example

```java
@Autowired
private ApiKeyService apiKeyService;

// Create new API key
CreateApiKeyRequest request = new CreateApiKeyRequest(
    "My API Key", 
    "For production use", 
    Instant.now().plus(365, ChronoUnit.DAYS)
);
CreateApiKeyResponse response = apiKeyService.createApiKey("user123", request);

// Get user's API keys
List<ApiKeyResponse> userKeys = apiKeyService.getApiKeysForUser("user123");

// Update API key
UpdateApiKeyRequest updateRequest = new UpdateApiKeyRequest(
    "Updated Name", null, false, null
);
apiKeyService.updateApiKey("ak_1234567890abcdef", "user123", updateRequest);

// Delete API key
apiKeyService.deleteApiKey("ak_1234567890abcdef", "user123");
```
