This service class provides comprehensive user management functionality for the OpenFrame MSP platform, handling CRUD operations, user validation, and business rule enforcement with soft deletion capabilities.

## Key Components

- **getUserByEmail()** - Retrieves user by email address
- **existsActiveUserByEmail()** - Checks if an active user exists with given email
- **listUsers()** - Returns paginated list of users with post-processing
- **getUserById()** - Fetches user by ID with response mapping
- **updateUser()** - Updates user first/last name with validation
- **softDeleteUser()** - Performs soft deletion with business rule validation

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class UserController {
    private final UserService userService;
    
    @GetMapping("/users")
    public UserPageResponse getUsers(@RequestParam int page, @RequestParam int size) {
        return userService.listUsers(page, size);
    }
    
    @PutMapping("/users/{id}")
    public UserResponse updateUser(@PathVariable String id, @RequestBody UpdateUserRequest request) {
        return userService.updateUser(id, request);
    }
    
    @DeleteMapping("/users/{id}")
    public void deleteUser(@PathVariable String id, Principal principal) {
        userService.softDeleteUser(id, principal.getName());
    }
}
```

**Key Features:**
- Prevents self-deletion and owner account deletion
- Integrates with UserProcessor for event handling
- Uses soft deletion to maintain data integrity
- Supports pagination for user listings
