A Spring service class that handles user management operations including retrieval, listing, and soft deletion with proper authorization checks.

## Key Components

- **getUserByEmail()** - Retrieves a user by email address
- **existsActiveUserByEmail()** - Checks if an active user exists with the given email
- **listUsers()** - Returns paginated list of users with post-processing
- **softDeleteUser()** - Performs soft deletion with business rule validation

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class UserController {
    
    private final UserService userService;
    
    @GetMapping("/users")
    public UserPageResponse getUsers(@RequestParam int page, @RequestParam int size) {
        return userService.listUsers(page, size);
    }
    
    @DeleteMapping("/users/{id}")
    public void deleteUser(@PathVariable String id, @AuthenticationPrincipal String requesterId) {
        userService.softDeleteUser(id, requesterId);
    }
    
    @GetMapping("/users/check/{email}")
    public boolean checkUserExists(@PathVariable String email) {
        return userService.existsActiveUserByEmail(email);
    }
}
```

The service enforces business rules like preventing self-deletion and protecting OWNER accounts from deletion. It integrates with `UserProcessor` for post-operation handling and uses `UserMapper` for DTO conversion.
