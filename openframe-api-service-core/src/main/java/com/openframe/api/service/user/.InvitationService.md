Manages user invitation lifecycle including creation, listing, and revocation of invitations with email notifications and validation.

## Key Components

- **createInvitation()** - Creates new invitation, validates email uniqueness, sends notification email, and triggers post-processing
- **listInvitations()** - Retrieves paginated list of invitations with metadata (page, size, total elements)
- **revokeInvitation()** - Revokes pending invitations and updates status to REVOKED

## Usage Example

```java
@Autowired
private InvitationService invitationService;

// Create a new invitation
CreateInvitationRequest request = CreateInvitationRequest.builder()
    .email("user@example.com")
    .build();
InvitationResponse invitation = invitationService.createInvitation(request);

// List invitations with pagination
InvitationPageResponse invitations = invitationService.listInvitations(0, 10);

// Revoke an invitation
invitationService.revokeInvitation("invitation-id-123");
```

The service integrates with `EmailService` for notifications, `UserService` for validation, and `InvitationProcessor` for custom business logic. It enforces business rules like preventing duplicate invitations for existing users and only allowing revocation of pending invitations.
