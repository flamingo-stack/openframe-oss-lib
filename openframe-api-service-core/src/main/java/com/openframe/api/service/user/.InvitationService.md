Service class responsible for managing user invitations in the OpenFrame MSP platform, handling invitation lifecycle operations including creation, revocation, and renewal with email notifications.

## Key Components

- **createInvitation()** - Creates new user invitations with duplicate email validation, sends notification emails, and triggers post-processing
- **listInvitations()** - Retrieves paginated list of active invitations (excludes accepted/revoked)
- **revokeInvitation()** - Cancels pending invitations and updates status to revoked
- **renewInvitation()** - Creates new invitation for expired ones while revoking the original

## Usage Example

```java
// Create a new invitation
CreateInvitationRequest request = CreateInvitationRequest.builder()
    .email("tech@company.com")
    .roles(List.of(Role.TECHNICIAN))
    .build();
InvitationResponse invitation = invitationService.createInvitation(request);

// List active invitations with pagination
InvitationPageResponse invitations = invitationService.listInvitations(0, 10);

// Revoke a pending invitation
invitationService.revokeInvitation("invitation-id");

// Renew an expired invitation
InvitationResponse renewed = invitationService.renewInvitation("expired-invitation-id");
```

The service integrates with email notifications, user validation, and invitation processing workflows to ensure secure and trackable user onboarding for MSP organizations.
