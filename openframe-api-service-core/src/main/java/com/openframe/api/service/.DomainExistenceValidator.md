This interface defines a contract for validating the existence of domains, allowing implementations to check if any domains from a provided list actually exist.

## Key Components

- **`anyExists(List<String> domains)`** - Method that takes a list of domain names and returns true if at least one domain exists, false otherwise

public class DNSValidator implements DomainExistenceValidator {
    @Override
