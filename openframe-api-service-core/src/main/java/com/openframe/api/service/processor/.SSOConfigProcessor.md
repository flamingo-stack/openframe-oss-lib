Interface for processing SSO configuration lifecycle events in the API service layer. Provides extensible hooks for post-processing operations like event publishing or notifications.

## Key Components

- **`postProcessConfigSaved(SSOConfig config)`** - Hook triggered after SSO config creation or updates
- **`postProcessConfigDeleted(SSOConfig config)`** - Hook triggered after SSO config deletion  
- **`postProcessConfigToggled(SSOConfig config)`** - Hook triggered after SSO config enable/disable operations

All methods have default no-op implementations, making the interface optional to implement.

## Usage Example

```java
@Component
public class KafkaSSOConfigProcessor implements SSOConfigProcessor {
    
    @Autowired
    private KafkaTemplate<String, Object> kafkaTemplate;
    
    @Override
    public void postProcessConfigSaved(SSOConfig config) {
        kafkaTemplate.send("sso-config-events", 
            new SSOConfigEvent("SAVED", config.getId()));
    }
    
    @Override
    public void postProcessConfigDeleted(SSOConfig config) {
        kafkaTemplate.send("sso-config-events", 
            new SSOConfigEvent("DELETED", config.getId()));
    }
}

// In your service class
@Service
public class SSOConfigService {
    
    @Autowired
    private SSOConfigProcessor processor;
    
    public void saveConfig(SSOConfig config) {
        repository.save(config);
        processor.postProcessConfigSaved(config);
    }
}
```
