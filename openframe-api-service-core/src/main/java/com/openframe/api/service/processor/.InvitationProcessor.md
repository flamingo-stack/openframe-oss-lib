An interface that provides hooks for processing invitation lifecycle events, designed for extensibility and future integration with event publishing systems like Kafka.

## Key Components

- **`postProcessInvitationCreated(Invitation)`** - Hook called after invitation creation with populated invitation data
- **`postProcessInvitationRevoked(Invitation)`** - Hook called after invitation revocation with REVOKED status
- **Default implementations** - No-op methods allowing selective override in implementations

## Usage Example

```java
@Component
public class KafkaInvitationProcessor implements InvitationProcessor {
    
    @Autowired
    private KafkaTemplate<String, Object> kafkaTemplate;
    
    @Override
    public void postProcessInvitationCreated(Invitation invitation) {
        InvitationEvent event = new InvitationEvent("CREATED", invitation);
        kafkaTemplate.send("invitation-events", event);
    }
    
    @Override
    public void postProcessInvitationRevoked(Invitation invitation) {
        InvitationEvent event = new InvitationEvent("REVOKED", invitation);
        kafkaTemplate.send("invitation-events", event);
    }
}

// Inject and use in service
@Service
public class InvitationService {
    @Autowired
    private InvitationProcessor processor;
    
    public void createInvitation(Invitation invitation) {
        // Create invitation logic...
        processor.postProcessInvitationCreated(invitation);
    }
}
```
