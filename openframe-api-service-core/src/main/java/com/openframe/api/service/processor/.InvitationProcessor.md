This interface defines a processor for handling invitation lifecycle events, providing extensible hooks for post-processing operations like event publishing or notifications.

## Key Components

- **InvitationProcessor** - Main interface defining invitation processing hooks
- **postProcessInvitationCreated()** - Default method for handling invitation creation events
- **postProcessInvitationRevoked()** - Default method for handling invitation revocation events

## Usage Example

```java
@Component
public class KafkaInvitationProcessor implements InvitationProcessor {
    
    @Autowired
    private KafkaTemplate<String, Object> kafkaTemplate;
    
    @Override
    public void postProcessInvitationCreated(Invitation invitation) {
        InvitationEvent event = new InvitationEvent("CREATED", invitation);
        kafkaTemplate.send("invitation-events", event);
    }
    
    @Override
    public void postProcessInvitationRevoked(Invitation invitation) {
        InvitationEvent event = new InvitationEvent("REVOKED", invitation);
        kafkaTemplate.send("invitation-events", event);
    }
}

// Usage in service
@Service
public class InvitationService {
    
    @Autowired
    private InvitationProcessor processor;
    
    public void createInvitation(Invitation invitation) {
        // Save invitation logic
        invitationRepository.save(invitation);
        
        // Post-process
        processor.postProcessInvitationCreated(invitation);
    }
}
```
