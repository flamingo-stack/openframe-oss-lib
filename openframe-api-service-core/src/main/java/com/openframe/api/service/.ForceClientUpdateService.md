Handles forced client update operations for OpenFrame-managed machines by fetching the latest client configuration and broadcasting update commands via NATS messaging.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `process(ForceClientUpdateRequest)` | Public Method | Triggers a forced update for a specific list of machine IDs |
| `processAll()` | Public Method | Triggers a forced update across all registered machines |
| `processMachines(List<String>)` | Private Method | Iterates machine IDs and delegates to `processMachine` |
| `processMachine(String)` | Private Method | Fetches latest `OpenFrameClientConfiguration` and publishes update via NATS; returns `PROCESSED` or `FAILED` status per machine |
| `validateMachineIds(List<String>)` | Private Method | Guards against empty machine ID lists |
| `buildResponseItem(String, ForceAgentStatus)` | Private Method | Constructs a `ForceClientUpdateResponseItem` with machine ID and status |

## Key Dependencies

- **`OpenFrameClientConfigurationRepository`** — Retrieves the most recent client configuration (`findFirstByOrderByCreatedAtDesc`)
- **`OpenFrameClientUpdatePublisher`** — Publishes update commands over NATS
- **`MachineRepository`** — Fetches all registered machines for bulk operations

## Usage Example

```java
// Force update for specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(List.of("machine-001", "machine-002"));

ForceClientUpdateResponse response = forceClientUpdateService.process(request);
response.getItems().forEach(item ->
    log.info("Machine: {}, Status: {}", item.getMachineId(), item.getStatus())
);

// Force update for ALL registered machines
ForceClientUpdateResponse allResponse = forceClientUpdateService.processAll();
```

Each response item carries a `ForceAgentStatus` of either `PROCESSED` or `FAILED`, allowing callers to audit per-machine update outcomes without interrupting the batch.
