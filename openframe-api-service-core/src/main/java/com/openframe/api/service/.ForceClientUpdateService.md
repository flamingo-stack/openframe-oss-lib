Service that orchestrates forced client configuration updates across OpenFrame machines by publishing the latest client configuration to specified machines or all machines in the system.

## Key Components

- **`process(ForceClientUpdateRequest)`** - Forces client updates for specific machines listed in the request
- **`processAll()`** - Forces client updates for all machines in the system
- **`processMachines(List<String>)`** - Internal method to process a list of machine IDs
- **`processMachine(String)`** - Handles individual machine update processing and error handling
- **Dependencies:**
  - `OpenFrameClientConfigurationRepository` - Retrieves latest client configuration
  - `OpenFrameClientUpdatePublisher` - Publishes configuration updates
  - `MachineRepository` - Manages machine data

## Usage Example

```java
@Autowired
private ForceClientUpdateService forceUpdateService;

// Force update specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(Arrays.asList("machine-001", "machine-002"));
ForceClientUpdateResponse response = forceUpdateService.process(request);

// Force update all machines
ForceClientUpdateResponse allResponse = forceUpdateService.processAll();

// Check results
response.getItems().forEach(item -> {
    System.out.println("Machine: " + item.getMachineId() + 
                      ", Status: " + item.getStatus());
});
```

The service retrieves the latest client configuration and publishes it to target machines, returning status information for each machine processed (PROCESSED or FAILED).
