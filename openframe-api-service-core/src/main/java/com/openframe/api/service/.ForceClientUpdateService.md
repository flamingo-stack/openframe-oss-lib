Service responsible for forcing client configuration updates on OpenFrame devices, supporting both targeted machine updates and bulk updates across all registered machines.

## Key Components

- **process(ForceClientUpdateRequest)** - Forces updates on specific machines identified by machine IDs
- **processAll()** - Forces updates on all registered machines in the system
- **processMachines(List<String>)** - Internal method to process a list of machine IDs
- **processMachine(String)** - Handles update publishing for individual machines with error handling
- **validateMachineIds(List<String>)** - Validates that machine ID list is not empty
- **buildResponseItem(String, ForceAgentStatus)** - Creates response items with machine ID and processing status

## Usage Example

```java
@Autowired
private ForceClientUpdateService forceUpdateService;

// Force update on specific machines
ForceClientUpdateRequest request = new ForceClientUpdateRequest();
request.setMachineId(Arrays.asList("machine-001", "machine-002"));
ForceClientUpdateResponse response = forceUpdateService.process(request);

// Force update on all machines
ForceClientUpdateResponse allResponse = forceUpdateService.processAll();

// Check results
response.getItems().forEach(item -> {
    log.info("Machine {} status: {}", item.getMachineId(), item.getStatus());
});
```

The service retrieves the latest client configuration from the database and publishes it to the specified machines via the `OpenFrameClientUpdatePublisher`. Each machine's processing result is tracked with `ForceAgentStatus` (PROCESSED or FAILED) and returned in the response.
