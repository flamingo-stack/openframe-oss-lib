Provides email domain validation functionality by checking domain existence in the system and filtering out generic public email providers like Gmail, Yahoo, and Outlook.

## Key Components

- **validateExists(List<String>)** - Validates that domains don't already exist in the system
- **validateGenericPublicDomain(List<String>)** - Throws exception if any domains are generic public providers
- **isGenericPublicDomain(String)** - Checks if a single domain is a generic public provider
- **GENERIC_PUBLIC_DOMAINS** - Comprehensive set of common email providers from various countries
- **DomainExistenceValidator** - Injected dependency for checking domain existence

## Usage Example

```java
@Autowired
private DomainValidationService domainValidationService;

public void processDomains(List<String> domains) {
    // Check if domains already exist in system
    domainValidationService.validateExists(domains);
    
    // Prevent registration of generic email providers
    domainValidationService.validateGenericPublicDomain(domains);
    
    // Check individual domain
    if (domainValidationService.isGenericPublicDomain("gmail.com")) {
        // Handle generic domain case
    }
}
```

The service supports international domains including Chinese (qq.com), Korean (naver.com), German (web.de), and privacy-focused providers (protonmail.com, tutanota.com).
