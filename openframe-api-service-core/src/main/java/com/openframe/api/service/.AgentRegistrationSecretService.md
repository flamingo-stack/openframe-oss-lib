Manages the lifecycle of agent registration secrets used to authenticate agents joining the OpenFrame platform, including retrieval, listing, and rotation of encrypted secrets.

## Key Components

| Method | Description |
|--------|-------------|
| `getActiveSecret()` | Fetches the currently active secret, decrypting it before returning |
| `getAllSecrets()` | Returns all secrets (active and inactive) with decrypted keys |
| `generateNewSecret()` | Rotates the secret: deactivates the current one, generates and persists a new encrypted secret |
| `deactivateExisting()` | (private) Marks the current active secret as inactive and triggers post-processing |

**Key Dependencies:**

| Dependency | Role |
|-----------|------|
| `AgentRegistrationSecretRepository` | MongoDB/data persistence for secrets |
| `EncryptionService` | Encrypts secrets at rest; decrypts on retrieval |
| `AgentRegistrationSecretGenerator` | Produces new cryptographically generated secret values |
| `AgentRegistrationSecretProcessor` | Handles post-generation and post-deactivation side effects |

## Usage Example

```java
// Retrieve the currently active registration secret
AgentRegistrationSecretResponse active = agentRegistrationSecretService.getActiveSecret();
System.out.println("Active key: " + active.getKey());

// Rotate to a new secret (deactivates current, generates new)
AgentRegistrationSecretResponse newSecret = agentRegistrationSecretService.generateNewSecret();
System.out.println("New key ID: " + newSecret.getId());

// List all secrets (audit/history)
List<AgentRegistrationSecretResponse> all = agentRegistrationSecretService.getAllSecrets();
```

> **Note:** Secret keys are stored encrypted via `EncryptionService` and are only decrypted at the service layer — raw keys are never persisted in plaintext. Throws `AgentRegistrationSecretNotFoundException` if no active secret exists.
