GraphQL data fetcher for the OpenFrame Knowledge Base module, handling all queries and mutations related to KB items (articles and folders), tags, and file attachments via the DGS (Domain Graph Service) framework.

## Key Components

| Component | Description |
|-----------|-------------|
| `knowledgeBaseItems` | Paginated query for KB items with filtering and cursor-based pagination |
| `knowledgeBaseItem` | Fetches a single KB item by Relay global ID |
| `knowledgeBaseTags` | Retrieves tags, optionally scoped to a folder subtree or filtered by archive status |
| `knowledgeBaseFolderTree` / `knowledgeBaseArticleTree` | Returns full folder/article trees for UI pickers |
| `archivedArticles` | Paginated query for archived articles with optional tag and search filters |
| `createFolder` / `renameFolder` / `deleteFolder` | Folder lifecycle mutations |
| `createArticle` / `updateArticle` | Article CRUD mutations, stamping the current authenticated user |
| `publishArticle` / `unpublishArticle` | Article publication state transitions |
| `archiveArticle` / `unarchiveArticle` | Article archive lifecycle with optional folder re-targeting |
| `moveToFolder` | Moves any KB item into a different folder |
| `addTagToKnowledgeBaseItem` / `removeTagFromKnowledgeBaseItem` | Tag association mutations |
| `knowledgeBaseAttachmentDownloadUrl` | Generates a pre-signed download URL for a KB attachment |
| `createKnowledgeBaseTempAttachmentUploadUrl` | Creates a pre-signed upload URL for a temporary attachment |
| `deleteKnowledgeBaseTempAttachment` | Removes a temporary attachment by ID |

All ID arguments are decoded from Relay global IDs before being forwarded to services.

## Usage Example

```java
// Query - paginated KB items with filter
query {
  knowledgeBaseItems(filter: { folderId: "S2bvd2xlZGdlQmFzZUl0ZW06YWJj" }, first: 10) {
    totalCount
    edges {
      node { id name type }
    }
  }
}

// Mutation - create a new article
mutation {
  createArticle(input: {
    name: "VPN Troubleshooting Guide"
    folderId: "S2bvd2xlZGdlQmFzZUl0ZW06YWJj"
    content: "<p>Steps to resolve VPN issues...</p>"
    tagIds: []
  }) {
    id
    name
    status
  }
}

// Mutation - generate a temp attachment upload URL
mutation {
  createKnowledgeBaseTempAttachmentUploadUrl(input: {
    fileName: "network-diagram.png"
    contentType: "image/png"
    fileSize: 204800
  }) {
    uploadUrl
    attachmentId
  }
}
```
