This GraphQL data fetcher handles organization-related queries using the Netflix DGS framework, providing endpoints for retrieving organizations with filtering, pagination, and search capabilities.

## Key Components

- **`organizations()`** - Fetches paginated and filtered list of organizations with optional search
- **`organization()`** - Retrieves a single organization by ID
- **`organizationByOrganizationId()`** - Retrieves a single organization by organization ID
- **Dependencies**: `OrganizationService`, `OrganizationQueryService`, `GraphQLOrganizationMapper`

## Usage Example

```java
// GraphQL query for paginated organizations with filtering
query {
  organizations(
    filter: { status: "ACTIVE" },
    pagination: { first: 10, after: "cursor123" },
    search: "tech"
  ) {
    edges {
      node {
        id
        name
        organizationId
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
    totalCount
  }
}

// Query single organization by ID
query {
  organization(id: "org123") {
    id
    name
    organizationId
  }
}
```

The data fetcher uses validation annotations to ensure input parameters are valid, maps DTOs using the organization mapper, and leverages service layers for data retrieval while maintaining proper logging for debugging purposes.
