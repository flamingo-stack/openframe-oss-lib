GraphQL DGS data fetcher for managing item assignments in the OpenFrame platform, handling queries and mutations for assigning/unassigning items to targets such as organizations, devices, tickets, and knowledge articles.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `assignedItemCounts` | `@DgsQuery` | Returns assignment counts grouped by target type for a given item ID |
| `assignedItems` | `@DgsQuery` | Returns paginated, filterable list of assignments for an item with cursor-based pagination |
| `assignItem` | `@DgsMutation` | Creates a new assignment linking an item to a target |
| `unassignItem` | `@DgsMutation` | Removes a specific assignment between an item and a target |
| `unassignAllByType` | `@DgsMutation` | Removes all assignments of a given target type from an item |
| `itemAssignmentNodeId` | `@DgsData` | Resolves the Relay global ID for `ItemAssignment` nodes |
| `resolveTarget` | `@DgsData` | Async resolver that batches target entity loading via DataLoaders (`Organization`, `Machine`, `Ticket`, `KnowledgeBaseItem`) |

All IDs are decoded from Relay global IDs before being passed to the service layer.

## Usage Example

```java
// GraphQL query — fetch assignment counts for an item
query {
  assignedItemCounts(itemId: "SXRlbUFzc2lnbm1lbnQ6MTIz") {
    targetType
    count
  }
}

// GraphQL mutation — assign an item to a device
mutation {
  assignItem(
    itemId: "S25vd2xlZGdlQmFzZUl0ZW06NDU2"
    itemType: KNOWLEDGE_ARTICLE
    targetType: DEVICE
    targetId: "TWFjaGluZTo3ODk="
  ) {
    id
    targetType
    target {
      ... on Machine {
        hostname
      }
    }
  }
}

// GraphQL mutation — remove all organization assignments from an item
mutation {
  unassignAllByType(
    itemId: "S25vd2xlZGdlQmFzZUl0ZW06NDU2"
    targetType: ORGANIZATION
  )
}
```
