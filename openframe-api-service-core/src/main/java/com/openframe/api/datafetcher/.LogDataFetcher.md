A GraphQL data fetcher component that handles log-related queries for audit and logging functionality, providing filtering, pagination, and detail retrieval capabilities.

## Key Components

- **LogDataFetcher**: Main DGS component class with three GraphQL query resolvers
- **logFilters()**: Returns available filter options based on provided criteria
- **logs()**: Retrieves paginated log events with filtering and search capabilities
- **logDetails()**: Fetches detailed information for a specific log event by composite key

## Usage Example

```java
// This component is automatically registered with DGS and handles GraphQL queries

// Example GraphQL queries this component resolves:

// Get available filter options
query {
  logFilters(filter: { toolTypes: ["JENKINS", "GITHUB"] }) {
    toolTypes
    eventTypes
    dateRange {
      start
      end
    }
  }
}

// Query logs with pagination and search
query {
  logs(
    filter: { toolTypes: ["JENKINS"] },
    pagination: { first: 10, after: "cursor123" },
    search: "error"
  ) {
    edges {
      node {
        id
        eventType
        timestamp
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
  }
}

// Get specific log details
query {
  logDetails(
    ingestDay: "2024-01-15",
    toolType: "JENKINS",
    eventType: "BUILD_FAILED",
    timestamp: "2024-01-15T10:30:00Z",
    toolEventId: "build-123"
  ) {
    details
    metadata
  }
}
```
