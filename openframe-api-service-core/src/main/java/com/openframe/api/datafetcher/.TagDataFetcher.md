GraphQL DGS data fetcher for tag management operations. Exposes queries and mutations for creating, reading, updating, and deleting tags across tenant-wide device resources, with Relay-compliant global ID encoding.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `tagNodeId` | `@DgsData` | Converts raw tag ID to Relay global ID format |
| `tags` | `@DgsQuery` | Returns all tags for the current tenant |
| `tagValueOptions` | `@DgsQuery` | Returns distinct value options for a given tag key |
| `tagKeySuggestions` | `@DgsQuery` | Autocomplete search over tag keys with optional limit |
| `tagValueSuggestions` | `@DgsQuery` | Autocomplete search over tag values for a specific key |
| `createTag` | `@DgsMutation` | Creates a new tag with key, entity type, description, and color |
| `updateTag` | `@DgsMutation` | Updates an existing tag by Relay global ID |
| `deleteTag` | `@DgsMutation` | Deletes a tag by Relay global ID, returns `true` on success |

## Usage Example

```java
// GraphQL query — fetch all tags
query {
  tags {
    id
    key
    description
    color
  }
}

// GraphQL mutation — create a tag
mutation {
  createTag(
    key: "environment"
    entityType: "DEVICE"
    description: "Deployment environment"
    color: "#FFC008"
  ) {
    id
    key
  }
}

// GraphQL mutation — update a tag using Relay global ID
mutation {
  updateTag(
    id: "VGFnOjY0YWJjMTIz"
    key: "env"
    color: "#FF5733"
  ) {
    id
    key
  }
}

// GraphQL query — autocomplete tag key suggestions
query {
  tagKeySuggestions(search: "env", limit: 5) {
    id
    key
  }
}
```

> **Note:** All IDs exchanged via mutations use Relay global ID encoding (`Tag:<rawId>` → base64). The `entityType` argument on `createTag` must match a valid `TagEntityType` enum value.
