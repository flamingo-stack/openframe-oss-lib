GraphQL Data Fetcher that provides query endpoints and data loaders for device management operations within the OpenFrame MSP platform. Implements Netflix DGS GraphQL framework for efficient device data retrieval with filtering, pagination, and related entity resolution.

## Key Components

- **Query Resolvers**: `devices()`, `device()`, and `deviceFilters()` for primary device data access
- **Field Resolvers**: `tags()`, `toolConnections()`, `installedAgents()`, and `organization()` for nested Machine data
- **Services**: DeviceService for core operations, DeviceFilterService for filter logic
- **Data Loaders**: Optimized batch loading for related entities to prevent N+1 query problems
- **Mappers**: GraphQLDeviceMapper for DTO transformations and pagination handling

## Usage Example

```java
// GraphQL query example that would trigger these resolvers
query GetDevices($filter: DeviceFilterInput, $pagination: CursorPaginationInput) {
  devices(filter: $filter, pagination: $pagination, search: "server") {
    edges {
      node {
        id
        machineId
        name
        tags {
          name
          color
        }
        organization {
          name
        }
        installedAgents {
          agentType
          version
        }
      }
    }
    totalCount
  }
}

// Individual device lookup
query GetDevice($machineId: String!) {
  device(machineId: $machineId) {
    id
    name
    toolConnections {
      toolType
      status
    }
  }
}
```

The data fetcher leverages DataLoader pattern for efficient batch loading and supports comprehensive device filtering, search, and sorting capabilities through the OpenFrame device management system.
