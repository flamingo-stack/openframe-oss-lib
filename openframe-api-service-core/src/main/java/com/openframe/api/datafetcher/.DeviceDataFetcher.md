A GraphQL data fetcher that provides query operations and field resolvers for device-related data in the OpenFrame API, using Netflix DGS framework with efficient DataLoader patterns for related entity fetching.

## Key Components

### Query Resolvers
- **`deviceFilters()`** - Returns available filter options for device queries
- **`devices()`** - Paginated device search with filtering and text search capabilities
- **`device()`** - Single device lookup by machine ID

### Field Resolvers
- **`tags()`** - Loads associated tags for a machine using DataLoader
- **`toolConnections()`** - Fetches tool connections for a machine via DataLoader
- **`installedAgents()`** - Retrieves installed agents for a machine using DataLoader
- **`organization()`** - Resolves the parent organization with null-safe handling

### Dependencies
- **`DeviceService`** - Core device business logic and data access
- **`DeviceFilterService`** - Handles device filtering operations
- **`GraphQLDeviceMapper`** - Maps between GraphQL DTOs and domain objects

## Usage Example

```java
// GraphQL Query Examples:

// Get paginated devices with filtering
query GetDevices($filter: DeviceFilterInput, $pagination: CursorPaginationInput) {
  devices(filter: $filter, pagination: $pagination, search: "server") {
    edges {
      node {
        machineId
        name
        organization {
          name
        }
        tags {
          name
        }
        toolConnections {
          toolId
        }
      }
    }
    totalCount
  }
}

// Get single device
query GetDevice($id: String!) {
  device(machineId: $id) {
    machineId
    name
    installedAgents {
      version
    }
  }
}
```

The DataLoader pattern ensures efficient N+1 query prevention when loading related entities across multiple machines in a single request.
