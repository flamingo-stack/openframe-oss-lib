GraphQL data fetcher component that handles event-related queries and mutations using Netflix DGS framework.

## Key Components

**Queries:**
- `events()` - Retrieves paginated events with filtering and search capabilities
- `eventById()` - Fetches a single event by ID
- `eventFilters()` - Returns available filter options for events

**Mutations:**
- `createEvent()` - Creates a new event with timestamp
- `updateEvent()` - Updates an existing event by ID

**Dependencies:**
- `EventService` - Core business logic for event operations
- `GraphQLEventMapper` - Maps between GraphQL DTOs and domain objects

## Usage Example

```java
// Query events with pagination and filtering
GenericConnection<GenericEdge<Event>> events = eventDataFetcher.events(
    EventFilterInput.builder().type("USER_ACTION").build(),
    CursorPaginationInput.builder().first(10).build(),
    "search term"
);

// Create a new event
Event newEvent = eventDataFetcher.createEvent(
    CreateEventInput.builder()
        .userId("user123")
        .type("LOGIN")
        .data(Map.of("ip", "192.168.1.1"))
        .build()
);

// Get single event
Event event = eventDataFetcher.eventById("event-id-123");
```

The component uses cursor-based pagination for efficient data retrieval and includes comprehensive logging for debugging purposes.
