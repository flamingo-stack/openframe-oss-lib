GraphQL data fetcher component that handles tool-related queries, providing endpoints for searching integrated tools and retrieving available filter options.

## Key Components

- **`integratedTools`** - Query method that retrieves a filtered and searchable list of tools
- **`toolFilters`** - Query method that returns available filter options for tool queries
- **Dependencies**: `ToolService` for business logic, `GraphQLToolMapper` for data transformation

## Usage Example

```java
// GraphQL queries that this data fetcher handles:

// Query for tools with filters and search
query GetTools($filter: ToolFilterInput, $search: String) {
    integratedTools(filter: $filter, search: $search) {
        tools {
            id
            name
            description
        }
        totalCount
    }
}

// Query for available filter options
query GetToolFilters {
    toolFilters {
        categories
        types
        statuses
    }
}
```

The component uses Netflix DGS annotations to expose GraphQL endpoints, validates input parameters, and delegates business logic to the `ToolService`. All operations include debug logging for monitoring query execution.
