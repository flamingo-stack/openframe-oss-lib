GraphQL data fetcher component that handles tool-related queries for the OpenFrame API, providing endpoints for retrieving integrated tools with filtering, searching, and sorting capabilities.

## Key Components

- **integratedTools()** - Main query resolver that fetches tools with optional filtering, search, and sorting parameters
- **toolFilters()** - Query resolver that returns available filter options for tools
- **ToolService** - Injected service for business logic operations
- **GraphQLToolMapper** - Mapper for converting GraphQL inputs to service layer DTOs

## Usage Example

```java
// The GraphQL queries this data fetcher handles:

// Query for integrated tools with filters
query GetIntegratedTools($filter: ToolFilterInput, $search: String, $sort: SortInput) {
  integratedTools(filter: $filter, search: $search, sort: $sort) {
    tools {
      id
      name
      category
      status
    }
    totalCount
    pageInfo {
      hasNextPage
      hasPreviousPage
    }
  }
}

// Query for available tool filter options
query GetToolFilters {
  toolFilters {
    categories
    statuses
    tags
  }
}
```

The data fetcher uses Netflix DGS annotations to expose GraphQL endpoints, with automatic validation of input arguments and structured logging for debugging tool queries.
