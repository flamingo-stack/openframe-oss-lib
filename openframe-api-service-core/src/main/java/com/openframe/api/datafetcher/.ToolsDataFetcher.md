A Netflix DGS GraphQL data fetcher component that provides queries for retrieving integrated tools with filtering and search capabilities.

## Key Components

- **integratedTools()** - Main query resolver that fetches filtered and searchable tool listings
- **toolFilters()** - Query resolver that returns available filter options for tools
- **ToolService** - Injected service for business logic operations
- **GraphQLToolMapper** - Injected mapper for converting GraphQL DTOs to service layer objects

## Usage Example

This data fetcher automatically exposes GraphQL queries through the DGS framework:

```java
// GraphQL query examples that this fetcher handles:

// Query for filtered tools with search
query GetTools($filter: ToolFilterInput, $search: String) {
  integratedTools(filter: $filter, search: $search) {
    tools {
      id
      name
      description
    }
    totalCount
  }
}

// Query for available filter options
query GetFilters {
  toolFilters {
    categories
    types
    statuses
  }
}
```

The component uses Spring validation (`@Valid`) on input arguments and includes comprehensive logging for debugging GraphQL operations. The separation of concerns is maintained by delegating business logic to `ToolService` and data mapping to `GraphQLToolMapper`.
