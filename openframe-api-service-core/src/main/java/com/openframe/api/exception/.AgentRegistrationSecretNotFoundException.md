A custom runtime exception thrown when an agent registration secret cannot be found during the authentication process.

## Key Components

- **AgentRegistrationSecretNotFoundException** - Custom exception class extending `RuntimeException`
- **errorCode** - Final field storing the specific error code associated with the exception
- **Constructor** - Takes an error code and message to initialize the exception

## Usage Example

```java
// Throwing the exception when secret validation fails
public void validateAgentSecret(String secretId) {
    AgentSecret secret = secretRepository.findById(secretId);
    if (secret == null) {
        throw new AgentRegistrationSecretNotFoundException(
            "AGENT_SECRET_001", 
            "Agent registration secret not found for ID: " + secretId
        );
    }
}

// Catching and handling the exception
try {
    agentService.registerAgent(agentId, secretId);
} catch (AgentRegistrationSecretNotFoundException ex) {
    logger.error("Registration failed with code: {}, message: {}", 
        ex.getErrorCode(), ex.getMessage());
    return ResponseEntity.status(HttpStatus.NOT_FOUND)
        .body(new ErrorResponse(ex.getErrorCode(), ex.getMessage()));
}
```
