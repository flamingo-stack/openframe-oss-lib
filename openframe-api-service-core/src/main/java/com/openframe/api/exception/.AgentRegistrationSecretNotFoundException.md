Custom exception thrown when an agent registration secret cannot be found, extending the base `NotFoundException` with an error code and message.

## Key Components

| Component | Description |
|-----------|-------------|
| `AgentRegistrationSecretNotFoundException` | Extends `NotFoundException` to signal a missing agent registration secret |
| `ErrorCode` | Enum/code passed to the parent to categorize the error |
| `message` | Human-readable description of the not-found condition |

## Usage Example

```java
throw new AgentRegistrationSecretNotFoundException(
    ErrorCode.AGENT_SECRET_NOT_FOUND,
    "No registration secret found for agent ID: " + agentId
);
```
