A custom runtime exception thrown when an agent registration secret cannot be found during the registration process.

## Key Components

- **AgentRegistrationSecretNotFoundException**: Custom runtime exception class
- **errorCode**: String field containing the specific error code for this exception
- **Constructor**: Accepts both error code and message parameters

## Usage Example

```java
// Throwing the exception when a registration secret is not found
if (registrationSecret == null) {
    throw new AgentRegistrationSecretNotFoundException(
        "AGENT_SECRET_NOT_FOUND", 
        "Registration secret for agent ID: " + agentId + " was not found"
    );
}

// Catching and handling the exception
try {
    validateAgentRegistrationSecret(secretId);
} catch (AgentRegistrationSecretNotFoundException ex) {
    logger.error("Agent registration failed: {} - {}", 
        ex.getErrorCode(), ex.getMessage());
    return ResponseEntity.status(HttpStatus.NOT_FOUND)
        .body("Agent registration secret not found");
}
```

This exception extends `RuntimeException` making it an unchecked exception, and uses Lombok's `@Getter` annotation to automatically generate getter methods for the `errorCode` field. The error code provides additional context beyond the exception message for more specific error handling and logging.
