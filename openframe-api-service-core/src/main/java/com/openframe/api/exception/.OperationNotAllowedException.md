A custom exception for signaling that a requested operation is forbidden, wrapping the `OPERATION_NOT_ALLOWED` error code from the OpenFrame core exception framework.

## Key Components

| Element | Description |
|---|---|
| `OperationNotAllowedException` | Extends `ForbiddenException` to represent HTTP 403-style operation restrictions |
| `ErrorCode.OPERATION_NOT_ALLOWED` | Core error code automatically applied to all instances |
| `OperationNotAllowedException(String message)` | Single constructor accepting a descriptive message for the forbidden operation |

## Usage Example

```java
// Throw when a user attempts a restricted action
public void deleteOrganization(String orgId, User currentUser) {
    if (!currentUser.isAdmin()) {
        throw new OperationNotAllowedException(
            "Only administrators are allowed to delete an organization."
        );
    }
    organizationService.delete(orgId);
}
```

> The exception inherits HTTP response mapping from `ForbiddenException`, so it will automatically translate to a `403 Forbidden` response with the `OPERATION_NOT_ALLOWED` error code in the API response body.
