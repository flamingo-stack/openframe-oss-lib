Spring configuration class that defines application-wide beans for the API module, primarily focused on security configuration.

## Key Components

- **`passwordEncoder()`** - Bean method that creates a BCryptPasswordEncoder instance for secure password hashing and verification

## Usage Example

```java
@Service
public class UserService {
    
    private final PasswordEncoder passwordEncoder;
    
    public UserService(PasswordEncoder passwordEncoder) {
        this.passwordEncoder = passwordEncoder;
    }
    
    public String hashPassword(String rawPassword) {
        return passwordEncoder.encode(rawPassword);
    }
    
    public boolean verifyPassword(String rawPassword, String hashedPassword) {
        return passwordEncoder.matches(rawPassword, hashedPassword);
    }
}
```

The configuration automatically makes the `PasswordEncoder` bean available for dependency injection throughout the Spring application context. The BCryptPasswordEncoder provides strong password hashing using the bcrypt algorithm with salt generation.
