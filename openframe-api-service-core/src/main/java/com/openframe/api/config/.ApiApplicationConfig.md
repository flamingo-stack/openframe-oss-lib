A Spring Boot configuration class that provides essential application-wide beans and settings for the OpenFrame API. This configuration currently focuses on security-related bean definitions.

## Key Components

- **@Configuration**: Marks this class as a Spring configuration class containing bean definitions
- **@RequiredArgsConstructor**: Lombok annotation for automatic constructor generation with required fields
- **@Slf4j**: Lombok annotation providing logging capabilities
- **passwordEncoder()**: Bean method that creates and configures a BCrypt password encoder

## Usage Example

```java
@Service
@RequiredArgsConstructor
public class UserService {
    
    private final PasswordEncoder passwordEncoder;
    
    public void createUser(String rawPassword) {
        // The BCryptPasswordEncoder bean is automatically injected
        String encodedPassword = passwordEncoder.encode(rawPassword);
        // Save user with encoded password
    }
    
    public boolean validatePassword(String rawPassword, String encodedPassword) {
        return passwordEncoder.matches(rawPassword, encodedPassword);
    }
}
```

The configuration automatically makes the BCrypt password encoder available throughout the application via Spring's dependency injection. BCrypt is chosen for its adaptive nature and built-in salt generation, providing secure password hashing with configurable computational cost.
