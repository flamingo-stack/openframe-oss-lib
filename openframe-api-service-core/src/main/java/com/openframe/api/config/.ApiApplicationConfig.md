Spring Boot configuration class that sets up essential application beans for the API layer, specifically providing password encryption functionality.

## Key Components

- **ApiApplicationConfig**: Main configuration class annotated with `@Configuration`
- **passwordEncoder()**: Bean method that provides BCrypt password encoding

## Usage Example

```java
// Spring will automatically inject the password encoder where needed
@Service
public class UserService {
    
    private final PasswordEncoder passwordEncoder;
    
    public UserService(PasswordEncoder passwordEncoder) {
        this.passwordEncoder = passwordEncoder;
    }
    
    public String encodePassword(String rawPassword) {
        return passwordEncoder.encode(rawPassword);
    }
    
    public boolean verifyPassword(String rawPassword, String encodedPassword) {
        return passwordEncoder.matches(rawPassword, encodedPassword);
    }
}
```

The configuration uses BCrypt for secure password hashing, which is automatically available throughout the application context. The `@RequiredArgsConstructor` and `@Slf4j` annotations from Lombok provide constructor injection and logging capabilities respectively.
