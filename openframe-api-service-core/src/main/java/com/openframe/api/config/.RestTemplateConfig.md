Spring Boot configuration class that provides a RestTemplate bean for making HTTP requests to external APIs and services.

## Key Components

- **`RestTemplateConfig`** - Configuration class that defines REST client beans
- **`restTemplate()`** - Bean factory method that creates a default RestTemplate instance

## Usage Example

The RestTemplate bean can be automatically injected into other components:

```java
@Service
public class ExternalApiService {
    
    private final RestTemplate restTemplate;
    
    public ExternalApiService(RestTemplate restTemplate) {
        this.restTemplate = restTemplate;
    }
    
    public String fetchData(String url) {
        return restTemplate.getForObject(url, String.class);
    }
    
    public void postData(String url, Object request) {
        restTemplate.postForObject(url, request, String.class);
    }
}
```

This configuration provides a basic RestTemplate with default settings. For production use, consider customizing with connection timeouts, error handlers, and interceptors.
