Spring configuration class that provides a RestTemplate bean for making HTTP requests in the application.

## Key Components

- **RestTemplateConfig**: Configuration class annotated with `@Configuration`
- **restTemplate()**: Bean method that creates and configures a RestTemplate instance

## Usage Example

Once configured, the RestTemplate can be injected and used throughout the application:

```java
@Service
public class ApiService {
    
    @Autowired
    private RestTemplate restTemplate;
    
    public String fetchData(String url) {
        return restTemplate.getForObject(url, String.class);
    }
    
    public ResponseEntity<User> createUser(User user) {
        return restTemplate.postForEntity("/api/users", user, User.class);
    }
}
```

The RestTemplate bean is automatically managed by Spring's IoC container and can be dependency-injected wherever HTTP client functionality is needed. This centralized configuration allows for easy customization of the RestTemplate (such as adding interceptors, timeout settings, or error handlers) in a single location.
