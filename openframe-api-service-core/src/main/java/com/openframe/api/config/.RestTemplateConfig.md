Configuration class that provides a Spring Bean for RestTemplate, enabling HTTP client functionality throughout the application.

## Key Components

- **RestTemplateConfig**: Spring configuration class
- **restTemplate()**: Bean factory method that creates and configures a RestTemplate instance

## Usage Example

The RestTemplate bean can be automatically injected into any Spring-managed component:

```java
@Service
public class ExternalApiService {
    
    @Autowired
    private RestTemplate restTemplate;
    
    public String fetchUserData(Long userId) {
        String url = "https://api.example.com/users/" + userId;
        return restTemplate.getForObject(url, String.class);
    }
    
    public void createUser(User user) {
        String url = "https://api.example.com/users";
        restTemplate.postForObject(url, user, User.class);
    }
}
```

This configuration enables HTTP operations like GET, POST, PUT, and DELETE requests to external APIs. The RestTemplate is thread-safe and can be shared across multiple components, making it ideal for microservice communication and third-party API integration.
