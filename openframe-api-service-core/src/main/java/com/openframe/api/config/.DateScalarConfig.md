A custom GraphQL scalar configuration that handles date values in yyyy-MM-dd format, providing automatic conversion between LocalDate objects and string representations.

## Key Components

- **`@DgsScalar(name = "Date")`** - Registers this as a GraphQL Date scalar type
- **`serialize()`** - Converts LocalDate objects to yyyy-MM-dd formatted strings for GraphQL responses
- **`parseValue()`** - Parses date strings from GraphQL variables into LocalDate objects
- **`parseLiteral()`** - Parses date string literals from GraphQL queries into LocalDate objects
- **`DATE_FORMATTER`** - Static formatter enforcing yyyy-MM-dd pattern

## Usage Example

```java
// In your GraphQL schema
type Event {
  id: ID!
  name: String!
  eventDate: Date  # Uses the custom Date scalar
}

// In your resolver
@DgsQuery
public Event getEvent(@InputArgument String id) {
    return Event.builder()
        .id(id)
        .name("Conference 2024")
        .eventDate(LocalDate.of(2024, 3, 15))  // Automatically serialized to "2024-03-15"
        .build();
}

// GraphQL query example
query {
  event(id: "123") {
    name
    eventDate  # Returns "2024-03-15"
  }
}
```

The scalar automatically validates date format and throws appropriate exceptions for invalid inputs, ensuring type safety across your GraphQL API.
