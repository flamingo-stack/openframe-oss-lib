This file defines a custom GraphQL scalar type for handling Java `Instant` objects in a Netflix DGS GraphQL API, providing bidirectional conversion between `Instant` objects and ISO-8601 formatted strings.

## Key Components

- **InstantScalarConfig**: Main class implementing `Coercing<Instant, String>` interface
- **serialize()**: Converts `Instant` objects to ISO-8601 string format for GraphQL responses
- **parseValue()**: Parses string input variables into `Instant` objects
- **parseLiteral()**: Parses string literals from GraphQL queries into `Instant` objects
- **@DgsScalar(name = "Instant")**: Registers the scalar type with GraphQL schema
- **@Component**: Makes it a Spring-managed bean

## Usage Example

Once registered, you can use the `Instant` scalar in your GraphQL schema and resolvers:

```java
// GraphQL Schema
type Query {
    getEventsByTime(startTime: Instant!, endTime: Instant!): [Event]
}

type Event {
    id: ID!
    timestamp: Instant!
    name: String!
}

// Resolver usage
@DgsQuery
public List<Event> getEventsByTime(
    @InputArgument Instant startTime,
    @InputArgument Instant endTime
) {
    return eventService.findByTimeRange(startTime, endTime);
}
```

GraphQL queries can use ISO-8601 formatted strings:
```graphql
query {
    getEventsByTime(
        startTime: "2023-10-15T10:30:00Z",
        endTime: "2023-10-15T18:00:00Z"
    ) {
        id
        timestamp
        name
    }
}
```
