Configures a custom GraphQL scalar type for handling Java `Instant` objects in DGS GraphQL operations. This component enables seamless serialization and deserialization between `Instant` objects and ISO-8601 string representations.

## Key Components

- **InstantScalarConfig** - Spring component implementing `Coercing<Instant, String>` interface
- **serialize()** - Converts `Instant` objects to ISO-8601 strings for GraphQL responses
- **parseValue()** - Parses string variables into `Instant` objects for GraphQL operations
- **parseLiteral()** - Handles literal string values in GraphQL queries/mutations

## Usage Example

```java
// GraphQL schema definition
scalar Instant

// Java entity with Instant field
public class Event {
    private Instant timestamp;
    // getters/setters
}

// GraphQL query
query {
    events {
        timestamp  # Returns: "2023-12-01T10:30:00Z"
    }
}

// GraphQL mutation with variable
mutation createEvent($timestamp: Instant!) {
    createEvent(timestamp: $timestamp) {
        id
    }
}

// Variables
{
    "timestamp": "2023-12-01T10:30:00Z"
}
```

The scalar automatically handles the conversion between Java `Instant` objects in your data model and ISO-8601 string representations in GraphQL operations, providing type safety and proper error handling for invalid date formats.
