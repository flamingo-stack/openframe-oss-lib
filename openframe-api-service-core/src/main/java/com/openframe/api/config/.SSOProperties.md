Configuration class for managing Single Sign-On (SSO) provider settings through Spring Boot's configuration properties system.

## Key Components

- **SSOProperties**: Main configuration class that binds SSO-related properties from application configuration files
- **providers**: List of `SSOProviderInfo` objects containing SSO provider configurations
- **@ConfigurationProperties**: Maps properties with "sso" prefix from configuration files
- **@Configuration**: Marks this as a Spring configuration component

## Usage Example

```java
// In application.yml or application.properties
// application.yml:
sso:
  providers:
    - name: "google"
      clientId: "your-google-client-id"
      clientSecret: "your-google-secret"
    - name: "github"
      clientId: "your-github-client-id"
      clientSecret: "your-github-secret"

// Injecting and using the configuration
@Service
public class SSOService {
    
    @Autowired
    private SSOProperties ssoProperties;
    
    public void configureSSOProviders() {
        List<SSOProviderInfo> providers = ssoProperties.getProviders();
        providers.forEach(provider -> {
            // Configure each SSO provider
            System.out.println("Configuring: " + provider.getName());
        });
    }
}
```
