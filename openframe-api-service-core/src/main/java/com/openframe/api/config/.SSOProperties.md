Configuration properties class that manages Single Sign-On (SSO) provider settings by binding external configuration properties to Java objects.

## Key Components

- **SSOProperties**: Main configuration class annotated with `@ConfigurationProperties(prefix = "sso")`
- **providers**: List of `SSOProviderInfo` objects containing SSO provider details
- Spring Boot auto-configuration support through `@Configuration` and Lombok `@Data`

## Usage Example

**Application configuration (application.yml):**
```yaml
sso:
  providers:
    - name: "Google"
      clientId: "google-client-id"
      clientSecret: "google-secret"
    - name: "GitHub"
      clientId: "github-client-id" 
      clientSecret: "github-secret"
```

**Injecting the properties:**
```java
@Service
public class SSOService {
    
    private final SSOProperties ssoProperties;
    
    public SSOService(SSOProperties ssoProperties) {
        this.ssoProperties = ssoProperties;
    }
    
    public List<SSOProviderInfo> getAvailableProviders() {
        return ssoProperties.getProviders();
    }
}
```

The class automatically maps configuration properties with the "sso" prefix to the `providers` field, enabling centralized SSO provider management.
