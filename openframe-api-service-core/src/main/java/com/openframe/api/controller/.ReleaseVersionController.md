REST controller that provides endpoints for retrieving application release version information. Handles HTTP requests for version data and returns structured responses with version details.

## Key Components

- **ReleaseVersionController**: Main controller class with `/release-version` base mapping
- **getReleaseVersion()**: GET endpoint that retrieves current release version information
- **ReleaseVersionQueryService**: Injected service for version data retrieval
- **ReleaseVersionResponse**: DTO for structured version response data

## Usage Example

```java
// GET request to retrieve release version
// Endpoint: GET /release-version

// Successful response (200 OK):
{
    "id": 1,
    "version": "1.2.3",
    "createdAt": "2023-10-15T10:30:00Z",
    "updatedAt": "2023-10-15T10:30:00Z"
}

// When no version found (404 Not Found):
// Empty response body with 404 status

// Service integration example:
@Autowired
private ReleaseVersionQueryService queryService;

// The controller handles Optional responses automatically,
// converting empty Optional to 404 status
```

The controller uses Spring Boot annotations for dependency injection and logging, returning appropriate HTTP status codes based on data availability.
