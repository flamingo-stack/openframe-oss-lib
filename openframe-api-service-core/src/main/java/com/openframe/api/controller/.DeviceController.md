REST controller that handles device management operations for the OpenFrame API, specifically focused on updating device status information.

## Key Components

- **DeviceController**: Main REST controller class annotated with `@RestController`
- **updateDeviceStatus()**: PATCH endpoint that updates a device's status by machine ID
- **DeviceService**: Injected service dependency for handling device business logic
- **UpdateDeviceStatusRequest**: DTO for capturing status update request data

## Usage Example

```java
// PATCH request to update device status
// URL: /devices/{machineId}
// Method: PATCH
// Body: { "status": "ACTIVE" }

// Example using RestTemplate
UpdateDeviceStatusRequest request = new UpdateDeviceStatusRequest("OFFLINE");
restTemplate.patchForObject(
    "/devices/machine-001", 
    request, 
    Void.class
);

// Example curl command
// curl -X PATCH http://localhost:8080/devices/machine-001 \
//   -H "Content-Type: application/json" \
//   -d '{"status":"MAINTENANCE"}'
```

The controller returns HTTP 204 (No Content) on successful updates and logs all status change operations for monitoring purposes.
