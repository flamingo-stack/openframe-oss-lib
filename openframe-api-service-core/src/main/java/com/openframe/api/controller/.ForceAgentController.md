REST controller that provides endpoints for forcing various operations on tool agents and clients, including installation, updates, and reinstallation across single or multiple targets.

## Key Components

- **ForceAgentController**: Main REST controller class with `/force` base mapping
- **Tool Agent Operations**: Install, update, and reinstall tool agents on specific machines or all machines
- **Client Operations**: Force client updates on specific targets or all clients
- **Service Dependencies**: 
  - `ForceToolInstallationService` - handles tool agent installations
  - `ForceClientUpdateService` - manages client updates
  - `ForceToolAgentUpdateService` - processes tool agent updates

## Usage Example

```java
// Force tool installation on specific machines
ForceToolInstallationRequest request = new ForceToolInstallationRequest();
request.setToolAgentId("agent123");
request.setMachineIds(Arrays.asList("machine1", "machine2"));
ForceToolAgentInstallationResponse response = controller.forceToolInstallation(request);

// Force client update on all clients
ForceClientUpdateResponse updateResponse = controller.forceClientUpdateAll();

// Force tool reinstallation with specific parameters
ForceToolReinstallationRequest reinstallRequest = new ForceToolReinstallationRequest();
reinstallRequest.setToolAgentId("agent123");
reinstallRequest.setMachineIds(Arrays.asList("machine1"));
ForceToolAgentInstallationResponse reinstallResponse = controller.forceToolReinstallation(reinstallRequest);
```
