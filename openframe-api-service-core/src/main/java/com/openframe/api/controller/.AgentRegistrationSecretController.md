REST controller that manages agent registration secrets for OpenFrame agents, providing endpoints to retrieve active secrets, list all secrets, and generate new registration keys.

## Key Components

- **AgentRegistrationSecretController** - Main REST controller class handling agent registration secret operations
- **AgentRegistrationSecretService** - Injected service handling the business logic for secret management
- **AgentRegistrationSecretResponse** - DTO class for returning secret data in API responses

### API Endpoints

- `GET /agent/registration-secret/active` - Retrieves the currently active registration secret
- `GET /agent/registration-secret` - Returns a list of all registration secrets
- `POST /agent/registration-secret/generate` - Creates and returns a new registration secret

## Usage Example

```java
// Service implementation example
@Service
public class AgentRegistrationSecretServiceImpl implements AgentRegistrationSecretService {
    
    @Override
    public AgentRegistrationSecretResponse getActiveSecret() {
        // Return the currently active secret for agent registration
        return AgentRegistrationSecretResponse.builder()
            .secretKey("abc123...")
            .isActive(true)
            .createdAt(LocalDateTime.now())
            .build();
    }
    
    @Override
    public AgentRegistrationSecretResponse generateNewSecret() {
        // Generate new secret and mark as active
        String newSecret = UUID.randomUUID().toString();
        // Save to database and return response
        return saveAndReturnSecret(newSecret);
    }
}
```

The controller uses standard Spring Boot annotations for REST functionality and includes logging for secret generation operations. All endpoints return structured responses using the `AgentRegistrationSecretResponse` DTO.
