A REST controller that manages agent registration secrets for secure agent onboarding and authentication within the OpenFrame system.

## Key Components

- **AgentRegistrationSecretController**: Main REST controller exposing endpoints for agent registration secret management
- **AgentRegistrationSecretService**: Injected service handling business logic for secret operations
- **AgentRegistrationSecretResponse**: DTO for returning secret information to clients

### Endpoints
- `GET /agent/registration-secret/active` - Retrieves the currently active registration secret
- `GET /agent/registration-secret` - Returns all registration secrets
- `POST /agent/registration-secret/generate` - Creates a new registration secret

## Usage Example

```java
// Inject the controller or call endpoints via HTTP client
@Autowired
private AgentRegistrationSecretController controller;

// Get the active secret for agent registration
AgentRegistrationSecretResponse activeSecret = controller.getActiveSecret();

// Generate a new registration secret
AgentRegistrationSecretResponse newSecret = controller.generateNewSecret();

// Retrieve all secrets for administrative purposes
List<AgentRegistrationSecretResponse> allSecrets = controller.getAllSecrets();
```

The controller uses Spring's `@RestController` with proper HTTP status codes and logging for monitoring secret generation activities.
