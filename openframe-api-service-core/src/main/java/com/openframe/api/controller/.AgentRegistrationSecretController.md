REST controller that manages agent registration secrets for secure agent onboarding, providing endpoints to retrieve active secrets, list all secrets, and generate new ones.

## Key Components

- **AgentRegistrationSecretController** - Main REST controller class
- **getActiveSecret()** - GET `/agent/registration-secret/active` - Returns the currently active registration secret
- **getAllSecrets()** - GET `/agent/registration-secret` - Retrieves all registration secrets
- **generateNewSecret()** - POST `/agent/registration-secret/generate` - Creates a new registration secret

## Usage Example

```java
// Inject the controller or call endpoints via HTTP client
@Autowired
private AgentRegistrationSecretController secretController;

// Get the active secret for agent registration
AgentRegistrationSecretResponse activeSecret = secretController.getActiveSecret();

// Retrieve all existing secrets
List<AgentRegistrationSecretResponse> allSecrets = secretController.getAllSecrets();

// Generate a new registration secret
AgentRegistrationSecretResponse newSecret = secretController.generateNewSecret();
```

The controller delegates business logic to `AgentRegistrationSecretService` and returns `AgentRegistrationSecretResponse` DTOs. All endpoints are mapped under the `/agent/registration-secret` base path, with the generation endpoint returning HTTP 201 status on success.
