REST controller that provides a simple health check endpoint for monitoring application status and availability.

## Key Components

- **HealthController** - Spring Boot REST controller class
- **health()** - GET endpoint method that returns application health status
- **Logger** - SLF4J logger for recording health check requests

## Usage Example

The health endpoint can be accessed via HTTP GET request:

```java
// The endpoint is automatically mapped to:
GET /health

// Returns:
// Status: 200 OK
// Body: "OK"
```

For monitoring or load balancer integration:

```java
// Example usage in a monitoring service
public boolean isApplicationHealthy(String baseUrl) {
    try {
        ResponseEntity<String> response = restTemplate.getForEntity(
            baseUrl + "/health", String.class);
        return response.getStatusCode().is2xxSuccessful() && 
               "OK".equals(response.getBody());
    } catch (Exception e) {
        return false;
    }
}
```

This controller follows standard health check patterns, logging each request for monitoring purposes and returning a simple "OK" response with HTTP 200 status when the application is running normally.
