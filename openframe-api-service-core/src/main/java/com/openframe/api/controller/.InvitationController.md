REST controller that manages user invitations, providing endpoints for creating, listing, revoking, and resending invitations within the OpenFrame platform.

## Key Components

- **InvitationController**: Main REST controller class mapped to `/invitations` endpoint
- **createInvitation()**: POST endpoint for creating new invitations with validation
- **listInvitations()**: GET endpoint for paginated invitation retrieval
- **revokeInvitation()**: DELETE endpoint for removing invitations by ID
- **resendInvitation()**: POST endpoint for renewing/resending existing invitations

## Usage Example

```java
// Creating a new invitation
@PostMapping
public InvitationResponse createInvitation(@Valid @RequestBody CreateInvitationRequest request) {
    return invitationService.createInvitation(request);
}

// Listing invitations with pagination
@GetMapping
public InvitationPageResponse listInvitations(
    @RequestParam(defaultValue = "0") int page,
    @RequestParam(defaultValue = "20") int size
) {
    return invitationService.listInvitations(page, size);
}

// Revoking an invitation
@DeleteMapping("/{id}")
public void revokeInvitation(@PathVariable String id) {
    invitationService.revokeInvitation(id);
}

// Resending an invitation
@PostMapping("/{id}/resend")
public InvitationResponse resendInvitation(@PathVariable String id) {
    return invitationService.renewInvitation(id);
}
```

The controller uses dependency injection with `InvitationService` and follows REST conventions with appropriate HTTP status codes for each operation.
