REST controller for managing user invitations in the OpenFrame API. Provides endpoints for creating, listing, and revoking invitations with validation and pagination support.

## Key Components

- **InvitationController** - Main REST controller class annotated with `@RestController`
- **InvitationService** - Injected service handling business logic for invitation operations
- **createInvitation()** - POST endpoint for creating new invitations
- **listInvitations()** - GET endpoint for retrieving paginated invitation lists
- **revokeInvitation()** - DELETE endpoint for removing invitations by ID

## Usage Example

```java
// Create a new invitation
@PostMapping
@ResponseStatus(HttpStatus.CREATED)
public InvitationResponse createInvitation(@Valid @RequestBody CreateInvitationRequest request) {
    return invitationService.createInvitation(request);
}

// List invitations with pagination
@GetMapping
public InvitationPageResponse listInvitations(
    @RequestParam(defaultValue = "0") int page,
    @RequestParam(defaultValue = "20") int size
) {
    return invitationService.listInvitations(page, size);
}

// Revoke an invitation
@DeleteMapping("/{id}")
@ResponseStatus(HttpStatus.NO_CONTENT)
public void revokeInvitation(@PathVariable String id) {
    invitationService.revokeInvitation(id);
}
```

The controller uses Spring Boot annotations for dependency injection, validation, and HTTP status codes. All endpoints are mapped under the `/invitations` base path with appropriate HTTP methods and status responses.
