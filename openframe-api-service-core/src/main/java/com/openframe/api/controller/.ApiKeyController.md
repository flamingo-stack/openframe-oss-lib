REST controller that manages API key operations for authenticated users, providing full CRUD functionality plus key regeneration capabilities.

## Key Components

- **ApiKeyController** - Main REST controller class handling API key management endpoints
- **GET /api-keys** - Retrieves all API keys for the authenticated user
- **POST /api-keys** - Creates a new API key with validation
- **GET /api-keys/{keyId}** - Fetches a specific API key by ID
- **PUT /api-keys/{keyId}** - Updates an existing API key
- **DELETE /api-keys/{keyId}** - Deletes an API key
- **POST /api-keys/{keyId}/regenerate** - Regenerates an existing API key

## Usage Example

```java
@Autowired
private TestRestTemplate restTemplate;

// Create a new API key
CreateApiKeyRequest createRequest = new CreateApiKeyRequest("My API Key", "Description");
CreateApiKeyResponse created = restTemplate.postForObject("/api-keys", createRequest, CreateApiKeyResponse.class);

// Get all API keys
List<ApiKeyResponse> apiKeys = restTemplate.getForObject("/api-keys", List.class);

// Update an API key
UpdateApiKeyRequest updateRequest = new UpdateApiKeyRequest("Updated Name");
restTemplate.put("/api-keys/{keyId}", updateRequest, created.keyId());

// Regenerate API key
CreateApiKeyResponse regenerated = restTemplate.postForObject("/api-keys/{keyId}/regenerate", null, CreateApiKeyResponse.class, created.keyId());
```
