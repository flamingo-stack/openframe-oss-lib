A REST API controller that handles organization management operations (create, update, delete) for internal API access, with read operations separated to external APIs.

## Key Components

**Endpoints:**
- `POST /organizations` - Creates new organizations
- `PUT /organizations/{id}` - Updates existing organizations  
- `DELETE /organizations/{id}` - Deletes organizations (with conflict handling)

**Dependencies:**
- `OrganizationCommandService` - Business logic for organization operations
- `OrganizationMapper` - Converts between entities and DTOs
- `CreateOrganizationRequest/UpdateOrganizationRequest` - Input DTOs
- `OrganizationResponse` - Output DTO

**Exception Handling:**
- Maps `IllegalArgumentException` to 404 Not Found
- Maps `OrganizationHasMachinesException` to 409 Conflict

## Usage Example

```java
// Creating an organization
CreateOrganizationRequest createRequest = new CreateOrganizationRequest("Acme Corp", "Manufacturing");
OrganizationResponse created = organizationController.createOrganization(createRequest);

// Updating an organization
UpdateOrganizationRequest updateRequest = new UpdateOrganizationRequest("Acme Corporation", "Tech");
OrganizationResponse updated = organizationController.updateOrganization("org-123", updateRequest);

// Deleting an organization
organizationController.deleteOrganization("org-123"); // Returns 204 No Content
```
