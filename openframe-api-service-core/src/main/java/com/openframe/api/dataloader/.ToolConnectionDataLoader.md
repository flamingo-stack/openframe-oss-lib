A GraphQL DataLoader that efficiently batch-loads tool connections for multiple machines to prevent N+1 query problems.

## Key Components

- **ToolConnectionDataLoader**: Main class implementing `BatchLoader<String, List<ToolConnection>>`
- **load()**: Asynchronous method that batch-loads tool connections for multiple machine IDs
- **ToolConnectionService**: Injected service handling the actual data retrieval
- **@DgsDataLoader**: Netflix DGS annotation registering this as a GraphQL data loader

## Usage Example

```java
// The DataLoader is automatically registered and can be used in GraphQL resolvers
@DgsComponent
public class MachineDataFetcher {
    
    @DgsData(parentType = "Machine")
    public CompletableFuture<List<ToolConnection>> toolConnections(
            DgsDataFetchingEnvironment dfe) {
        
        Machine machine = dfe.getSource();
        DataLoader<String, List<ToolConnection>> dataLoader = 
            dfe.getDataLoader("toolConnectionDataLoader");
        
        return dataLoader.load(machine.getId());
    }
}
```

This DataLoader batches multiple individual requests for tool connections into a single service call, significantly improving performance when loading connections for multiple machines in a single GraphQL query.
