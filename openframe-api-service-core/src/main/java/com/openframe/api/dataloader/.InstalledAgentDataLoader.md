A GraphQL data loader that efficiently batches and loads installed agents for multiple machines using Netflix DGS framework.

## Key Components

- **InstalledAgentDataLoader** - Main class implementing `BatchLoader` for batch loading installed agents
- **load()** - Asynchronous method that fetches installed agents for a list of machine IDs
- **InstalledAgentService** - Injected service dependency for data retrieval
- **@DgsDataLoader** - Netflix DGS annotation registering this as a data loader named "installedAgentDataLoader"

## Usage Example

```java
// The data loader is automatically registered with DGS and can be used in GraphQL resolvers
@DgsComponent
public class MachineResolver {
    
    @DgsData(parentType = "Machine", field = "installedAgents")
    public CompletableFuture<List<InstalledAgent>> getInstalledAgents(
            DgsDataFetchingEnvironment dfe) {
        
        Machine machine = dfe.getSource();
        DataLoader<String, List<InstalledAgent>> dataLoader = 
            dfe.getDataLoader("installedAgentDataLoader");
            
        return dataLoader.load(machine.getId());
    }
}
```

This data loader prevents the N+1 query problem by batching multiple machine ID requests into a single service call, improving GraphQL query performance when fetching installed agents for multiple machines.
