A Spring component that handles mapping between invitation DTOs and entity objects, providing conversion logic for user invitation workflows with automatic expiration handling.

## Key Components

- **toEntity()** - Converts `CreateInvitationRequest` to `Invitation` entity with auto-generated ID, normalized email, TTL-based expiration, and default ADMIN role
- **toResponse()** - Converts `Invitation` entity to `InvitationResponse` DTO with real-time expiration status checking
- **@Value ttl** - Configurable invitation time-to-live (defaults to 24 hours)

## Usage Example

```java
@Autowired
private InvitationMapper mapper;

// Create invitation entity from request
CreateInvitationRequest request = new CreateInvitationRequest();
request.setEmail("user@example.com");
request.setRoles(List.of(Role.USER));

Invitation invitation = mapper.toEntity(request);
// Results in entity with normalized email, PENDING status, 24h expiration

// Convert entity to response DTO
InvitationResponse response = mapper.toResponse(invitation);
// Automatically detects if invitation has expired and sets status accordingly
```

The mapper automatically normalizes email addresses to lowercase, generates UUIDs for new invitations, and dynamically calculates expiration status based on current time versus the stored expiration timestamp.
