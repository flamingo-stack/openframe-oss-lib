A Spring component that provides mapping functionality between GraphQL input/output DTOs and internal data structures for event-related operations.

## Key Components

- **`toEventFilterOptions`** - Converts GraphQL `EventFilterInput` to internal `EventFilterOptions` with user IDs, event types, and date range filtering
- **`toCursorPaginationCriteria`** - Maps `CursorPaginationInput` to internal pagination criteria with limit and cursor parameters
- **`toEventConnection`** - Transforms query results into GraphQL-compliant connection format with edges and page info

## Usage Example

```java
@Service
public class EventService {
    
    @Autowired
    private GraphQLEventMapper mapper;
    
    public GenericConnection<GenericEdge<Event>> getEvents(
            EventFilterInput filter, 
            CursorPaginationInput pagination) {
        
        // Convert GraphQL inputs to internal DTOs
        EventFilterOptions filterOptions = mapper.toEventFilterOptions(filter);
        CursorPaginationCriteria paginationCriteria = mapper.toCursorPaginationCriteria(pagination);
        
        // Query events using internal DTOs
        GenericQueryResult<Event> queryResult = eventRepository.findEvents(filterOptions, paginationCriteria);
        
        // Convert result to GraphQL connection format
        return mapper.toEventConnection(queryResult);
    }
}
```

The mapper handles null inputs gracefully by returning default builder instances and ensures proper GraphQL relay-style pagination with cursor-based edges.
