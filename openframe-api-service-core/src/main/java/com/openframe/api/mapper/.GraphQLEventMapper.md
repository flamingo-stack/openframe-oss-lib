Transforms GraphQL input types into internal DTOs and converts query results back to GraphQL-compatible connection formats for event-related operations.

## Key Components

**Main Methods:**
- `toEventFilterOptions()` - Converts GraphQL event filter input to internal filter options
- `toCursorPaginationCriteria()` - Maps pagination input to internal pagination criteria
- `toEventConnection()` - Transforms query results into GraphQL connection format with edges and page info

**Dependencies:**
- Event DTOs and filter objects
- Generic connection/edge types for GraphQL pagination
- Spring stereotype component for dependency injection

## Usage Example

```java
@Autowired
private GraphQLEventMapper eventMapper;

// Convert GraphQL input to internal filter
EventFilterInput filterInput = EventFilterInput.builder()
    .userIds(List.of("user1", "user2"))
    .eventTypes(List.of("LOGIN", "LOGOUT"))
    .build();
EventFilterOptions options = eventMapper.toEventFilterOptions(filterInput);

// Convert pagination input
CursorPaginationInput pageInput = CursorPaginationInput.builder()
    .limit(10)
    .cursor("cursor123")
    .build();
CursorPaginationCriteria criteria = eventMapper.toCursorPaginationCriteria(pageInput);

// Transform query results for GraphQL response
GenericQueryResult<Event> queryResult = eventService.getEvents(options, criteria);
GenericConnection<GenericEdge<Event>> connection = eventMapper.toEventConnection(queryResult);
```
