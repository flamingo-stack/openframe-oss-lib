Maps GraphQL assignment-related data structures to API response types, handling cursor pagination and item count transformations for the OpenFrame assignment domain.

## Key Components

| Method | Description |
|--------|-------------|
| `toCursorPaginationCriteria` | Converts `ConnectionArgs` into `CursorPaginationCriteria` for paginated queries |
| `toAssignedItemCounts` | Transforms a `Map<AssignmentTargetType, Long>` into a list of `AssignedItemCount` DTOs |
| `toAssignmentConnection` | Converts a `CountedGenericQueryResult<ItemAssignment>` into a cursor-based `CountedGenericConnection` with encoded cursors and page info |

## Usage Example

```java
@Autowired
private GraphQLAssignmentMapper assignmentMapper;

// Convert connection args to pagination criteria
CursorPaginationCriteria criteria = assignmentMapper.toCursorPaginationCriteria(connectionArgs);

// Map raw counts by target type to DTO list
Map<AssignmentTargetType, Long> rawCounts = assignmentService.getCounts();
List<AssignedItemCount> counts = assignmentMapper.toAssignedItemCounts(rawCounts);

// Build a paginated GraphQL connection from a query result
CountedGenericQueryResult<ItemAssignment> queryResult = assignmentService.findAssignments(criteria);
CountedGenericConnection<GenericEdge<ItemAssignment>> connection =
    assignmentMapper.toAssignmentConnection(queryResult);
```

> Each `ItemAssignment` node in the connection is wrapped in a `GenericEdge` with a Base64-encoded cursor derived from the item's ID, enabling standard GraphQL cursor-based pagination.
