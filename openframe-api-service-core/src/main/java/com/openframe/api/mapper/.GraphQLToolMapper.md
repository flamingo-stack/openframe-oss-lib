A Spring component that handles conversion between GraphQL input DTOs and internal filter options for tool-related operations.

## Key Components

- **GraphQLToolMapper** - Main mapper component annotated with `@Component` for Spring dependency injection
- **toToolFilterOptions()** - Converts `ToolFilterInput` GraphQLDTO to internal `ToolFilterOptions` object

## Usage Example

```java
@Autowired
private GraphQLToolMapper toolMapper;

// Convert GraphQL input to internal filter options
public List<Tool> getFilteredTools(ToolFilterInput filterInput) {
    ToolFilterOptions options = toolMapper.toToolFilterOptions(filterInput);
    return toolService.findToolsWithFilter(options);
}

// Handle null input gracefully
ToolFilterInput nullInput = null;
ToolFilterOptions emptyOptions = toolMapper.toToolFilterOptions(nullInput);
// Returns empty builder with default values
```

The mapper uses the builder pattern to construct `ToolFilterOptions` objects and handles null input by returning an empty builder instance. This provides a clean separation between GraphQL API concerns and internal business logic while ensuring type safety during the conversion process.
