A Spring component that maps GraphQL tool filter input DTOs to internal domain objects for filtering tools in the API layer.

## Key Components

- **GraphQLToolMapper** - Spring component providing mapping functionality between GraphQL and domain layers
- **toToolFilterOptions()** - Converts `ToolFilterInput` from GraphQL to `ToolFilterOptions` domain object with null safety

## Usage Example

```java
@Autowired
private GraphQLToolMapper mapper;

public List<Tool> getFilteredTools(ToolFilterInput filterInput) {
    // Convert GraphQL input to domain filter options
    ToolFilterOptions filterOptions = mapper.toToolFilterOptions(filterInput);
    
    // Use the converted options for tool filtering
    return toolService.findToolsByFilter(filterOptions);
}

// Handles null input gracefully
ToolFilterOptions emptyFilter = mapper.toToolFilterOptions(null);
// Returns: ToolFilterOptions with all fields null/default
```

The mapper provides clean separation between the GraphQL API layer and internal domain models, ensuring that changes to either layer don't directly impact the other. It includes null safety by returning an empty builder when input is null.
