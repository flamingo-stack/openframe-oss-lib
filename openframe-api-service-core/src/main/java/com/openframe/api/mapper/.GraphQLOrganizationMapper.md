This mapper class converts between GraphQL input types and internal DTOs for organization-related operations, facilitating the translation layer between the GraphQL API and the internal data processing layer.

## Key Components

- `toFilterOptions()` - Converts GraphQL filter input to internal OrganizationFilterOptions
- `toCursorPaginationCriteria()` - Transforms GraphQL pagination input to internal pagination criteria
- `toOrganizationConnection()` - Converts query results to GraphQL connection format with edges and page info

## Usage Example

```java
@Autowired
private GraphQLOrganizationMapper mapper;

// Convert filter input
OrganizationFilterInput filterInput = new OrganizationFilterInput();
filterInput.setCategory("TECH");
filterInput.setMinEmployees(50);
OrganizationFilterOptions options = mapper.toFilterOptions(filterInput);

// Convert pagination input
CursorPaginationInput paginationInput = new CursorPaginationInput();
paginationInput.setLimit(10);
paginationInput.setCursor("cursor123");
CursorPaginationCriteria criteria = mapper.toCursorPaginationCriteria(paginationInput);

// Convert query result to GraphQL connection
CountedGenericQueryResult<Organization> result = organizationService.findOrganizations(options, criteria);
CountedGenericConnection<GenericEdge<Organization>> connection = mapper.toOrganizationConnection(result);
```
