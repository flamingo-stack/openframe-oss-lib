A Spring component that maps between GraphQL organization inputs and internal DTOs, handling filtering, pagination, and result conversion for organization queries.

## Key Components

**Main Class:**
- `GraphQLOrganizationMapper` - Spring `@Component` for GraphQL to DTO transformations

**Core Mapping Methods:**
- `toFilterOptions()` - Converts `OrganizationFilterInput` to `OrganizationFilterOptions`
- `toCursorPaginationCriteria()` - Transforms `CursorPaginationInput` to `CursorPaginationCriteria`
- `toOrganizationConnection()` - Converts query results to GraphQL connection format with edges and pagination info

## Usage Example

```java
@Autowired
private GraphQLOrganizationMapper mapper;

// Convert GraphQL filter input
OrganizationFilterInput filterInput = // ... from GraphQL request
OrganizationFilterOptions filterOptions = mapper.toFilterOptions(filterInput);

// Convert pagination input
CursorPaginationInput paginationInput = // ... from GraphQL request
CursorPaginationCriteria criteria = mapper.toCursorPaginationCriteria(paginationInput);

// Convert query results to GraphQL connection
CountedGenericQueryResult<Organization> queryResult = organizationService.findOrganizations(criteria, filterOptions);
CountedGenericConnection<GenericEdge<Organization>> connection = mapper.toOrganizationConnection(queryResult);
```

The mapper handles null inputs gracefully and creates proper GraphQL edge structures with cursors based on organization IDs for efficient pagination.
