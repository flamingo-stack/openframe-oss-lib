A Spring component that converts between GraphQL input types and internal DTOs for audit log operations, providing mapping functionality for filters, pagination, and connection results.

## Key Components

- **`toLogFilterOptions()`** - Converts GraphQL `LogFilterInput` to internal `LogFilterOptions` DTO with date ranges, event types, severities, and device filtering
- **`toCursorPaginationCriteria()`** - Maps GraphQL pagination input to internal cursor-based pagination criteria
- **`toLogConnection()`** - Transforms query results into GraphQL connection format with edges, nodes, and page info
- **`createLogCursor()`** - Generates unique cursor strings combining timestamp and tool event ID for pagination

## Usage Example

```java
@Autowired
private GraphQLLogMapper logMapper;

// Convert GraphQL filter input to internal DTO
LogFilterInput graphqlInput = LogFilterInput.builder()
    .startDate(Instant.now().minus(7, ChronoUnit.DAYS))
    .eventTypes(List.of("ERROR", "WARNING"))
    .deviceId("device-123")
    .build();

LogFilterOptions filterOptions = logMapper.toLogFilterOptions(graphqlInput);

// Convert pagination input
CursorPaginationInput paginationInput = CursorPaginationInput.builder()
    .limit(20)
    .cursor("1640995200000_event-456")
    .build();

CursorPaginationCriteria criteria = logMapper.toCursorPaginationCriteria(paginationInput);

// Convert query results to GraphQL connection
GenericQueryResult<LogEvent> queryResult = logService.findLogs(filterOptions, criteria);
GenericConnection<GenericEdge<LogEvent>> connection = logMapper.toLogConnection(queryResult);
```
