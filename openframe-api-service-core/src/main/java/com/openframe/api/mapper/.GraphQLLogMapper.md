Maps between GraphQL input/output types and internal data structures for log-related operations. Handles conversion of filter inputs, pagination criteria, and query results for log events in a GraphQL API.

## Key Components

- **toLogFilterOptions()** - Converts GraphQL log filter input to internal filter options
- **toCursorPaginationCriteria()** - Maps GraphQL pagination input to internal pagination criteria  
- **toLogConnection()** - Transforms query results into GraphQL connection format with edges and page info
- **createLogCursor()** - Generates unique cursor strings for log events using timestamp and event ID

## Usage Example

```java
@Autowired
private GraphQLLogMapper logMapper;

// Map filter input from GraphQL request
LogFilterInput filterInput = // ... from GraphQL request
LogFilterOptions options = logMapper.toLogFilterOptions(filterInput);

// Convert pagination input
CursorPaginationInput paginationInput = // ... from GraphQL request  
CursorPaginationCriteria criteria = logMapper.toCursorPaginationCriteria(paginationInput);

// Transform query results for GraphQL response
GenericQueryResult<LogEvent> queryResult = logService.findLogs(options, criteria);
GenericConnection<GenericEdge<LogEvent>> connection = logMapper.toLogConnection(queryResult);
```
