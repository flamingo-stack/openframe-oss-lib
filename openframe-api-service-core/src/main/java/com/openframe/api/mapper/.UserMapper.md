A Spring component that maps User entities to UserResponse DTOs, converting database objects to client-safe response formats while transforming enum collections to string representations.

## Key Components

- **`UserMapper`** - Spring-managed mapper component for user data transformation
- **`toResponse(User)`** - Converts User entity to UserResponse DTO with enum-to-string mappings

## Usage Example

```java
@RestController
public class UserController {
    
    @Autowired
    private UserMapper userMapper;
    
    @GetMapping("/users/{id}")
    public UserResponse getUser(@PathVariable String id) {
        User user = userService.findById(id);
        return userMapper.toResponse(user);
    }
    
    @GetMapping("/users")
    public List<UserResponse> getAllUsers() {
        return userService.findAll()
            .stream()
            .map(userMapper::toResponse)
            .toList();
    }
}
```

The mapper automatically converts role enums to string names and status enums to their string representations, ensuring clean JSON serialization for API responses. All entity fields including timestamps are preserved in the response DTO.
