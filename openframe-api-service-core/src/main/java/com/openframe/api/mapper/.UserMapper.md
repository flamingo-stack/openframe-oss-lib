A Spring component that maps User entity objects to UserResponse DTOs, providing a clean separation between internal data models and API response formats.

## Key Components

- **`UserMapper`** - Spring component class for entity-to-DTO mapping
- **`toResponse(User entity)`** - Converts User entity to UserResponse DTO, mapping all fields including roles and status enums to their string representations

## Usage Example

```java
@Autowired
private UserMapper userMapper;

public UserResponse getUserById(String userId) {
    User user = userRepository.findById(userId);
    return userMapper.toResponse(user);
}

// Example transformation
User user = User.builder()
    .id("123")
    .email("john@example.com")
    .firstName("John")
    .lastName("Doe")
    .roles(List.of(Role.USER, Role.ADMIN))
    .status(UserStatus.ACTIVE)
    .build();

UserResponse response = userMapper.toResponse(user);
// response.getRoles() returns ["USER", "ADMIN"]
// response.getStatus() returns "ACTIVE"
```

This mapper ensures consistent API responses while keeping sensitive entity details encapsulated within the domain layer.
