A Spring component that converts User entity objects to UserResponse DTOs for API responses, handling role and status enum transformations.

## Key Components

- **UserMapper** - Spring-managed mapper component with `@Component` annotation
- **toResponse()** - Transforms User domain entities into client-safe UserResponse DTOs
- **Role/Status Mapping** - Converts enum collections to string lists for JSON serialization

## Usage Example

```java
@RestController
public class UserController {
    
    @Autowired
    private UserMapper userMapper;
    
    @GetMapping("/users/{id}")
    public ResponseEntity<UserResponse> getUser(@PathVariable String id) {
        User user = userService.findById(id);
        UserResponse response = userMapper.toResponse(user);
        return ResponseEntity.ok(response);
    }
    
    @GetMapping("/users")
    public List<UserResponse> getAllUsers() {
        List<User> users = userService.findAll();
        return users.stream()
                .map(userMapper::toResponse)
                .collect(Collectors.toList());
    }
}
```

The mapper ensures sensitive User entity data is properly transformed into a safe API response format, converting complex objects like role enums into simple string representations while preserving all necessary user information for client consumption.
