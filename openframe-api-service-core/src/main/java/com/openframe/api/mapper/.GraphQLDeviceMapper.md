Maps between GraphQL input/output types and internal data structures for device-related operations. Handles filtering, pagination, and connection transformations for the GraphQL device API.

## Key Components

- `toDeviceFilterOptions()` - Converts GraphQL filter input to internal filter options
- `toCursorPaginationCriteria()` - Maps GraphQL pagination input to internal pagination criteria  
- `toDeviceConnection()` - Transforms query results into GraphQL connection format with edges and page info

## Usage Example

```java
@Autowired
private GraphQLDeviceMapper mapper;

// Convert GraphQL filter input
DeviceFilterInput filterInput = DeviceFilterInput.builder()
    .statuses(Arrays.asList("ACTIVE", "INACTIVE"))
    .deviceTypes(Arrays.asList("MOBILE", "DESKTOP"))
    .build();
DeviceFilterOptions options = mapper.toDeviceFilterOptions(filterInput);

// Convert pagination input
CursorPaginationInput paginationInput = CursorPaginationInput.builder()
    .limit(10)
    .cursor("cursor123")
    .build();
CursorPaginationCriteria criteria = mapper.toCursorPaginationCriteria(paginationInput);

// Transform query results to GraphQL connection
CountedGenericQueryResult<Machine> queryResult = deviceService.findDevices(options, criteria);
CountedGenericConnection<GenericEdge<Machine>> connection = mapper.toDeviceConnection(queryResult);
```
