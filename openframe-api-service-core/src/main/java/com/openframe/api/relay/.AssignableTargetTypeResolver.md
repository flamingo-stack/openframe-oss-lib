Resolves the concrete GraphQL type name for the `AssignableTarget` union type, enabling the DGS framework to correctly identify polymorphic objects at runtime.

## Key Components

| Element | Description |
|---|---|
| `@DgsComponent` | Registers this class as a Netflix DGS component |
| `@DgsTypeResolver(name = "AssignableTarget")` | Binds the method to the `AssignableTarget` GraphQL union type |
| `resolveAssignableTarget(Object target)` | Maps a Java object instance to its corresponding GraphQL type name string |

**Supported union members:**

| Java Class | GraphQL Type |
|---|---|
| `Organization` | `"Organization"` |
| `Machine` | `"Machine"` |
| `Ticket` | `"Ticket"` |
| `KnowledgeBaseItem` | `"KnowledgeBaseItem"` |

## Usage Example

This resolver is invoked automatically by the DGS framework when a field returns an `AssignableTarget` union. No manual wiring is required.

```java
// GraphQL schema (conceptual)
// union AssignableTarget = Organization | Machine | Ticket | KnowledgeBaseItem

// DGS calls resolveAssignableTarget internally when resolving:
// { assignableTarget { ... on Machine { id hostname } } }

Object target = machineRepository.findById(id);
// Framework calls: resolver.resolveAssignableTarget(target) → "Machine"
```

If an unrecognized type is passed, an `IllegalArgumentException` is thrown:

```java
// Throws: IllegalArgumentException: Unknown AssignableTarget type: com.example.UnknownType
resolver.resolveAssignableTarget(new UnknownType());
```

> **Note:** When adding new members to the `AssignableTarget` union in the GraphQL schema, a corresponding `instanceof` branch must be added here to avoid runtime exceptions.
