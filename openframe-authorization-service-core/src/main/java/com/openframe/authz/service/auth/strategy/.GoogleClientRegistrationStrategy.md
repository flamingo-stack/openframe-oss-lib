A Spring component that implements Google-specific OAuth2 client registration strategy by extending the base OIDC registration functionality.

## Key Components

- **GoogleClientRegistrationStrategy** - Main strategy class for Google OAuth2 client registration
- **providerId()** - Returns the Google provider identifier constant
- **props()** - Provides Google-specific OIDC configuration properties
- **GoogleSSOProperties** - Configuration properties specific to Google SSO integration
- **BaseOidcClientRegistrationStrategy** - Parent class providing common OIDC registration logic

## Usage Example

```java
@Autowired
private GoogleClientRegistrationStrategy googleStrategy;

// The strategy is automatically registered and used by the OAuth2 framework
// Provider ID is used to identify this strategy
String providerId = googleStrategy.providerId(); // Returns "google"

// Configuration is accessed through the strategy
AbstractOidcProviderProperties config = googleStrategy.props();
```

The strategy pattern allows the application to handle Google OAuth2 client registration with provider-specific configuration while leveraging common OIDC functionality from the base class. This component is automatically discovered by Spring and registered for handling Google SSO authentication flows.
