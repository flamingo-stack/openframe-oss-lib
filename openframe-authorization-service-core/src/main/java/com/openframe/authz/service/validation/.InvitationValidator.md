A Spring service that validates authentication invitations to ensure they are in an acceptable state for processing.

## Key Components

- **loadAndEnsureAcceptable(String invitationId)** - Retrieves an invitation by ID and validates its acceptability
- **ensureAcceptable(AuthInvitation inv)** - Validates that an invitation is pending and not expired
- **AuthInvitationRepository** - Injected repository dependency for invitation data access

## Usage Example

```java
@Autowired
private InvitationValidator invitationValidator;

// Validate and load invitation by ID
try {
    AuthInvitation invitation = invitationValidator.loadAndEnsureAcceptable("inv-123");
    // Process valid invitation
} catch (IllegalArgumentException e) {
    // Handle invitation not found
} catch (IllegalStateException e) {
    // Handle expired or already used invitation
}

// Validate existing invitation object
AuthInvitation invitation = getInvitation();
try {
    invitationValidator.ensureAcceptable(invitation);
    // Invitation is valid for processing
} catch (IllegalStateException e) {
    // Handle validation failure
}
```

The validator throws `IllegalArgumentException` for missing invitations and `IllegalStateException` for invitations that are not pending or have expired.
