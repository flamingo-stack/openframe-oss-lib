A Spring service that validates authentication invitations to ensure they are in an acceptable state for processing.

## Key Components

- **loadAndEnsureAcceptable()** - Loads an invitation by ID from the repository and validates it's acceptable
- **ensureAcceptable()** - Validates that an invitation has PENDING status and hasn't expired
- **AuthInvitationRepository** - Injected dependency for accessing invitation data

## Usage Example

```java
@Autowired
private InvitationValidator invitationValidator;

// Load and validate invitation by ID
try {
    AuthInvitation invitation = invitationValidator.loadAndEnsureAcceptable("inv-123");
    // Process valid invitation
    processInvitation(invitation);
} catch (IllegalArgumentException e) {
    // Handle invitation not found
} catch (IllegalStateException e) {
    // Handle expired or already used invitation
}

// Validate existing invitation object
AuthInvitation invitation = getInvitation();
try {
    invitationValidator.ensureAcceptable(invitation);
    // Invitation is valid for acceptance
} catch (IllegalStateException e) {
    // Handle invalid invitation state
}
```

The validator throws `IllegalArgumentException` for non-existent invitations and `IllegalStateException` for invitations that are expired or not in PENDING status.
