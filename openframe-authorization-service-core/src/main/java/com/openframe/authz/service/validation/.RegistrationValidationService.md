This service validates tenant registration data, specifically ensuring domain availability during the registration process.

## Key Components

- **`ensureTenantDomainAvailable(String domain)`** - Validates that a tenant domain is not already in use by checking against the database

## Usage Example

```java
@Autowired
private RegistrationValidationService validationService;

public void registerNewTenant(TenantRegistrationRequest request) {
    // Validate domain availability before creating tenant
    validationService.ensureTenantDomainAvailable(request.getDomain());
    
    // Proceed with tenant creation
    tenantService.createTenant(request);
}
```

The service performs case-insensitive domain validation by converting the input to lowercase before checking existence. If a domain is already taken, it throws an `IllegalArgumentException` to prevent duplicate tenant domains in the system.
