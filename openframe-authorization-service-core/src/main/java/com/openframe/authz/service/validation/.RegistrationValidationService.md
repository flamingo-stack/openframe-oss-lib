Service for validating tenant registration requests, ensuring domain uniqueness and preventing duplicate tenant creation.

## Key Components

- **ensureTenantDomainAvailable()** - Validates that a tenant domain is not already registered in the system
- **TenantRepository** - Injected dependency for querying existing tenant data
- Domain normalization using lowercase conversion with ROOT locale

## Usage Example

```java
@Autowired
private RegistrationValidationService validationService;

public void registerNewTenant(String tenantDomain) {
    // Validate domain availability before creating tenant
    validationService.ensureTenantDomainAvailable("MyCompany.com");
    
    // Proceed with tenant creation if validation passes
    // Will throw IllegalArgumentException if domain exists
}
```

The service performs case-insensitive domain validation by converting inputs to lowercase using the ROOT locale, ensuring consistent domain checking regardless of input casing.
