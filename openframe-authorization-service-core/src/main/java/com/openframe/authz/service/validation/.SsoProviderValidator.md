Service class that validates SSO provider configurations and normalizes provider names for authentication operations.

## Key Components

- **normalizeProvider()** - Standardizes provider names by trimming whitespace and converting to lowercase
- **ensureProviderConfiguredForTenant()** - Validates that a specific SSO provider is configured and available for a given tenant
- **ensureProviderConfiguredForOnboarding()** - Convenience method to validate provider configuration for the onboarding tenant
- **SSOConfigService** - Injected dependency that provides access to SSO configuration data

## Usage Example

```java
@Autowired
private SsoProviderValidator validator;

// Normalize provider name before validation
String provider = validator.normalizeProvider("  GOOGLE  "); // returns "google"

// Validate provider is configured for a specific tenant
try {
    validator.ensureProviderConfiguredForTenant("tenant-123", "google");
    // Provider is valid, proceed with authentication
} catch (IllegalArgumentException e) {
    // Provider not configured for this tenant
}

// Validate provider for onboarding flow
validator.ensureProviderConfiguredForOnboarding("github");
```

The validator ensures SSO providers are properly configured before allowing authentication attempts, preventing runtime errors and providing consistent provider name handling across the authentication system.
