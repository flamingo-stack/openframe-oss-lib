Validates and normalizes SSO provider configurations for tenant-specific authentication flows. Ensures that requested SSO providers are properly configured before allowing authentication attempts.

## Key Components

- `normalizeProvider(String provider)` - Normalizes provider names to lowercase and trims whitespace
- `ensureProviderConfiguredForTenant(String tenantId, String provider)` - Validates that a provider is configured for a specific tenant
- `ensureProviderConfiguredForOnboarding(String provider)` - Validates provider configuration for onboarding tenant

## Usage Example

```java
@Autowired
private SsoProviderValidator validator;

public void authenticateUser(String tenantId, String provider) {
    // Normalize the provider name
    String normalizedProvider = validator.normalizeProvider(provider);
    
    // Validate provider is configured for tenant
    validator.ensureProviderConfiguredForTenant(tenantId, normalizedProvider);
    
    // Proceed with authentication...
}

public void onboardUser(String provider) {
    String normalizedProvider = validator.normalizeProvider("  GOOGLE  ");
    
    // Validate for onboarding flow
    validator.ensureProviderConfiguredForOnboarding(normalizedProvider);
    
    // Continue onboarding process...
}
```

The validator throws `IllegalArgumentException` when a provider is not configured for the specified tenant, ensuring authentication attempts only proceed with properly configured SSO providers.
