Handles SSO invitation acceptance flow by validating invitations and generating authorization data for OAuth2 redirects. This service orchestrates the initial step of accepting SSO invitations by creating secure cookies and redirect URLs.

## Key Components

- **startAccept()** - Main method that processes SSO invitation acceptance requests
- **COOKIE_TTL_SECONDS** - Cookie expiration constant (600 seconds)
- **Dependencies**:
  - `InvitationValidator` - Validates invitation existence and acceptability
  - `SsoProviderValidator` - Validates and normalizes SSO providers
  - `SsoCookieCodec` - Encodes secure cookie payloads

## Usage Example

```java
@Autowired
private SsoInvitationService ssoInvitationService;

public void acceptInvitation() {
    SsoInvitationAcceptRequest request = new SsoInvitationAcceptRequest();
    request.setInvitationId("inv-123");
    request.setProvider("google");
    request.setSwitchTenant(true);
    request.setRedirectTo("/dashboard");
    
    SsoAuthorizeData authData = ssoInvitationService.startAccept(request);
    
    // Use the returned data for OAuth2 flow
    String cookieToken = authData.token();
    String redirectUrl = authData.redirectPath(); // "/oauth2/authorization/google?tenant=tenant-id"
    String oauthState = authData.state();
}
```

The service creates a secure cookie containing invitation details and generates the appropriate OAuth2 authorization URL for the specified provider and tenant.
