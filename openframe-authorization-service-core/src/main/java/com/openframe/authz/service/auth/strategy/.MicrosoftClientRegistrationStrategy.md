A Spring component that implements Microsoft-specific OAuth 2.0 client registration strategy by extending the base OIDC registration functionality with Microsoft SSO configuration properties.

## Key Components

- **MicrosoftClientRegistrationStrategy** - Main strategy class extending `BaseOidcClientRegistrationStrategy`
- **providerId()** - Returns the Microsoft provider identifier constant
- **props()** - Provides Microsoft-specific OIDC configuration properties
- **microsoftProps** - Injected Microsoft SSO configuration properties

## Usage Example

This strategy is automatically registered with Spring's dependency injection and used by the authentication framework:

```java
// The strategy is automatically discovered and used when Microsoft SSO is configured
@Autowired
private List<ClientRegistrationStrategy> strategies;

// Framework usage - strategy selection by provider ID
ClientRegistrationStrategy microsoftStrategy = strategies.stream()
    .filter(s -> s.providerId().equals(MicrosoftSSOProperties.MICROSOFT))
    .findFirst()
    .orElseThrow();

// The strategy leverages the injected Microsoft properties
ClientRegistration registration = microsoftStrategy.buildClientRegistration();
```

The strategy integrates with the broader SSO configuration system, automatically applying Microsoft-specific settings like tenant configuration, scopes, and endpoint URLs when building OAuth 2.0 client registrations for Microsoft Azure AD authentication.
