Interface that defines the contract for default SSO provider configuration, allowing services to retrieve essential OAuth/OIDC client credentials and provider identification.

## Key Components

- **`providerId()`** - Returns the unique identifier for the SSO provider
- **`getDefaultClientId()`** - Retrieves the default OAuth client ID for the provider
- **`getDefaultClientSecret()`** - Retrieves the default OAuth client secret for the provider

## Usage Example

```java
// Implementation example
public class GoogleProviderConfig implements DefaultProviderConfig {
    
    @Override
    public String providerId() {
        return "google";
    }
    
    @Override
    public String getDefaultClientId() {
        return "your-google-client-id.googleusercontent.com";
    }
    
    @Override
    public String getDefaultClientSecret() {
        return "your-google-client-secret";
    }
}

// Usage in service
public class SSOService {
    
    public void configureProvider(DefaultProviderConfig config) {
        String provider = config.providerId();
        String clientId = config.getDefaultClientId();
        String clientSecret = config.getDefaultClientSecret();
        
        // Configure OAuth client with credentials
        setupOAuthClient(provider, clientId, clientSecret);
    }
}
```

This interface provides a clean abstraction for managing SSO provider configurations, enabling type-safe access to OAuth credentials while supporting multiple provider implementations.
