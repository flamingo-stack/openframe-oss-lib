This interface defines a contract for providing default configuration settings for SSO (Single Sign-On) authentication providers.

## Key Components

- **`providerId()`** - Returns the unique identifier for the SSO provider
- **`getDefaultClientId()`** - Returns the default client ID for OAuth/OIDC authentication
- **`getDefaultClientSecret()`** - Returns the default client secret for OAuth/OIDC authentication

## Usage Example

```java
public class GoogleProviderConfig implements DefaultProviderConfig {
    
    @Override
    public String providerId() {
        return "google";
    }
    
    @Override
    public String getDefaultClientId() {
        return "default-google-client-id";
    }
    
    @Override
    public String getDefaultClientSecret() {
        return "default-google-client-secret";
    }
}

// Usage in service configuration
DefaultProviderConfig config = new GoogleProviderConfig();
String providerId = config.providerId();
String clientId = config.getDefaultClientId();
```

This interface is typically implemented by provider-specific configuration classes to supply default OAuth/OIDC credentials for different SSO providers like Google, Microsoft, or custom identity providers.
