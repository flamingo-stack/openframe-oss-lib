A Spring service that manages tenant operations in a multi-tenant architecture system, providing methods for tenant creation, validation, and retrieval.

## Key Components

- **TenantService** - Main service class for tenant management operations
- **createTenant()** - Creates new tenants with domain validation and default settings
- **existByDomain()** - Checks if a tenant exists for a given domain
- **findById()** - Retrieves tenant by unique identifier
- **findFirst()** - Gets the first available tenant from repository
- **save()** - Persists tenant changes to the database

## Usage Example

```java
@Autowired
private TenantService tenantService;

// Create a new tenant
Tenant newTenant = tenantService.createTenant("Acme Corp", "acme.example.com");

// Check if domain is already taken
boolean domainExists = tenantService.existByDomain("acme.example.com");

// Find tenant by ID
Optional<Tenant> tenant = tenantService.findById("tenant-123");

// Get first available tenant
Optional<Tenant> firstTenant = tenantService.findFirst();

// Save tenant updates
tenant.ifPresent(t -> {
    t.setStatus(TenantStatus.SUSPENDED);
    tenantService.save(t);
});
```

The service automatically validates domain availability during tenant creation, sets up default configurations (FREE plan, ACTIVE status), and provides transactional safety for all operations.
