Service class responsible for managing tenant operations in OpenFrame's multi-tenant authentication and authorization system. Handles tenant creation, domain validation, and basic CRUD operations for tenant management.

## Key Components

- **createTenant()** - Creates new tenant with generated ID, validates domain availability, sets default status (ACTIVE) and plan (FREE)
- **existByDomain()** - Checks if a tenant exists with the specified domain
- **findById()** - Retrieves tenant by unique identifier
- **findFirst()** - Gets the first available tenant (utility method)
- **save()** - Persists tenant entity changes

**Dependencies:**
- `TenantRepository` - Data access layer for tenant persistence
- `RegistrationValidationService` - Domain validation and registration checks

## Usage Example

```java
@Autowired
private TenantService tenantService;

// Create a new tenant
Tenant tenant = tenantService.createTenant("Acme Corp", "acme.com");

// Check if domain exists
boolean domainExists = tenantService.existByDomain("acme.com");

// Find tenant by ID
Optional<Tenant> tenant = tenantService.findById("tenant-123");

// Update tenant
if (tenant.isPresent()) {
    Tenant updatedTenant = tenant.get();
    updatedTenant.setPlan(TenantPlan.PREMIUM);
    tenantService.save(updatedTenant);
}
```

The service uses transactional boundaries and includes comprehensive logging for tenant lifecycle operations. All tenant creation includes automatic domain uniqueness validation through the registration validation service.
