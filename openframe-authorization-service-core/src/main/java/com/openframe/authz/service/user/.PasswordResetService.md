A Spring service that handles secure password reset functionality using Redis for token storage and email notifications.

## Key Components

- **createResetToken(String email)** - Generates and stores a time-limited reset token for valid user emails
- **resetPassword(String token, String newPassword)** - Validates reset tokens and updates user passwords
- **Redis Integration** - Uses RedisTemplate for secure token storage with configurable TTL
- **Email Service** - Integrates with EmailService to send reset tokens to users
- **Security Features** - Token validation, expiration handling, and user verification

## Usage Example

```java
@Autowired
private PasswordResetService passwordResetService;

// Request password reset
passwordResetService.createResetToken("user@example.com");

// Reset password with token
try {
    passwordResetService.resetPassword("abc123token", "newSecurePassword");
} catch (IllegalArgumentException e) {
    // Handle invalid/expired token
    log.error("Password reset failed: {}", e.getMessage());
}
```

## Configuration

The service uses configurable TTL for reset tokens:

```java
# application.properties
openframe.password-reset.ttlMinutes=30  # Default: 30 minutes
```

Reset tokens are stored in Redis with the prefix `pwdreset:` and automatically expire after the configured duration. The service includes security measures like silent failure for non-existent emails and automatic token cleanup after successful password resets.
