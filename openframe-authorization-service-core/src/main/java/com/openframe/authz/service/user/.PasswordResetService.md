Service that handles secure password reset functionality using token-based validation. It generates temporary reset tokens stored in Redis and coordinates email notifications for password recovery workflows.

## Key Components

- **`createResetToken(String email)`** - Generates a secure reset token, stores it in Redis with TTL, and sends reset email
- **`resetPassword(String token, String newPassword)`** - Validates reset token, updates user password, and cleans up token
- **Redis integration** - Uses RedisTemplate for temporary token storage with configurable expiration
- **Email service integration** - Sends password reset emails via EmailService
- **Token utilities** - Leverages ResetTokenUtil for secure token generation

## Usage Example

```java
@Autowired
private PasswordResetService passwordResetService;

// Initiate password reset
passwordResetService.createResetToken("user@example.com");

// Complete password reset with token from email
try {
    passwordResetService.resetPassword("reset-token-123", "newSecurePassword");
} catch (IllegalArgumentException e) {
    // Handle invalid/expired token
}
```

**Configuration:**
```properties
openframe.password-reset.ttlMinutes=30
```

The service provides secure password reset with automatic token expiration, email validation, and cleanup of used tokens to prevent replay attacks.
