A Spring component that implements default provider configuration for Google SSO authentication, managing client credentials and provider identification.

## Key Components

- **GoogleDefaultProviderConfig** - Main component class implementing `DefaultProviderConfig`
- **GoogleSSOProperties** - Injected dependency containing Google SSO configuration properties
- **providerId()** - Returns the Google provider identifier constant
- **getDefaultClientId()** - Retrieves default Google OAuth client ID
- **getDefaultClientSecret()** - Retrieves default Google OAuth client secret

## Usage Example

```java
@Autowired
private GoogleDefaultProviderConfig googleConfig;

public void configureGoogleAuth() {
    String providerId = googleConfig.providerId(); // Returns "GOOGLE"
    String clientId = googleConfig.getDefaultClientId();
    String clientSecret = googleConfig.getDefaultClientSecret();
    
    // Use credentials to configure OAuth flow
    OAuthProvider provider = OAuthProvider.builder()
        .providerId(providerId)
        .clientId(clientId)
        .clientSecret(clientSecret)
        .build();
}
```

This component serves as a configuration adapter, abstracting Google-specific SSO properties behind a common interface for OAuth provider setup in the authentication service.
