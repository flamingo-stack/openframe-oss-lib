A Spring component that provides Google-specific default configuration values for SSO authentication, implementing the DefaultProviderConfig interface to supply Google OAuth client credentials.

## Key Components

- **GoogleDefaultProviderConfig** - Main configuration class implementing `DefaultProviderConfig`
- **providerId()** - Returns the Google provider identifier constant
- **getDefaultClientId()** - Retrieves default Google OAuth client ID
- **getDefaultClientSecret()** - Retrieves default Google OAuth client secret
- **GoogleSSOProperties** - Injected dependency containing Google SSO configuration properties

## Usage Example

```java
@Autowired
private GoogleDefaultProviderConfig googleConfig;

public void setupGoogleAuth() {
    String providerId = googleConfig.providerId(); // Returns "GOOGLE"
    String clientId = googleConfig.getDefaultClientId();
    String clientSecret = googleConfig.getDefaultClientSecret();
    
    // Use credentials to configure Google OAuth flow
    configureOAuthProvider(providerId, clientId, clientSecret);
}
```

This component acts as a bridge between the Google SSO properties configuration and the default provider configuration system, enabling consistent access to Google OAuth credentials across the authentication service.
