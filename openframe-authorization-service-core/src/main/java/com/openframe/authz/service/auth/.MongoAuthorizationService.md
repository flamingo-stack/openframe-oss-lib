A MongoDB-based implementation of Spring Security OAuth2's `OAuth2AuthorizationService` that handles persistence of OAuth2 authorization data including authorization codes, access tokens, and refresh tokens.

## Key Components

- **MongoAuthorizationService**: Main service class implementing `OAuth2AuthorizationService`
- **save()**: Persists OAuth2 authorization with detailed PKCE parameter logging
- **remove()**: Deletes authorization by ID
- **findById()**: Retrieves authorization by unique identifier
- **findByToken()**: Searches for authorization by token value and type (supports all OAuth2 token types)
- **MongoAuthorizationMapper**: Utility for converting between domain objects and MongoDB entities

## Usage Example

```java
@Autowired
private MongoAuthorizationService authorizationService;

// Save an OAuth2 authorization
OAuth2Authorization authorization = // ... create authorization
authorizationService.save(authorization);

// Find by authorization code
OAuth2Authorization found = authorizationService.findByToken(
    "auth_code_value", 
    new OAuth2TokenType("code")
);

// Find by access token
OAuth2Authorization byAccessToken = authorizationService.findByToken(
    "access_token_value", 
    OAuth2TokenType.ACCESS_TOKEN
);

// Remove authorization
authorizationService.remove(authorization);
```

The service includes comprehensive PKCE (Proof Key for Code Exchange) parameter debugging and supports flexible token lookup with automatic fallback across different token types when no specific type is provided.
