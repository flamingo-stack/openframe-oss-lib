A MongoDB-based implementation of Spring Security's OAuth2AuthorizationService that provides persistent storage for OAuth2 authorization data with PKCE (Proof Key for Code Exchange) support.

## Key Components

- **MongoAuthorizationService**: Main service class implementing `OAuth2AuthorizationService`
- **save()**: Persists OAuth2 authorization data to MongoDB with PKCE parameter logging
- **remove()**: Deletes authorization records by ID
- **findById()**: Retrieves authorization by unique identifier
- **findByToken()**: Flexible token lookup supporting multiple token types (access, refresh, authorization code, state)
- **MongoAuthorizationMapper**: Utility for converting between domain objects and MongoDB entities
- **PKCE Debug Logging**: Comprehensive logging for troubleshooting PKCE flow issues

## Usage Example

```java
@Autowired
private MongoAuthorizationService authorizationService;

// Save a new authorization
OAuth2Authorization authorization = OAuth2Authorization.builder()
    .id("auth-123")
    .registeredClientId("client-1")
    .principalName("user@example.com")
    .build();

authorizationService.save(authorization);

// Find by authorization ID
OAuth2Authorization found = authorizationService.findById("auth-123");

// Find by different token types
OAuth2Authorization byAccessToken = authorizationService
    .findByToken("access-token-value", OAuth2TokenType.ACCESS_TOKEN);

OAuth2Authorization byAuthCode = authorizationService
    .findByToken("auth-code-value", new OAuth2TokenType("code"));

// Remove authorization
authorizationService.remove(authorization);
```

The service includes extensive debug logging for PKCE parameters throughout the OAuth2 flow, making it easier to troubleshoot authorization code exchange issues.
