A Spring service that handles the complete tenant registration workflow, including pre/post processing hooks and validation of tenant domain and user email uniqueness.

## Key Components

- **registerTenant(TenantRegistrationRequest)** - Main registration method that orchestrates the entire tenant creation process
- **Dependencies**: 
  - `UserService` - User management operations
  - `TenantService` - Tenant CRUD operations
  - `OrganizationService` - Organization management
  - `RegistrationProcessor` - Custom pre/post processing hooks

## Usage Example

```java
@Autowired
private TenantRegistrationService registrationService;

public void registerNewTenant() {
    TenantRegistrationRequest request = TenantRegistrationRequest.builder()
        .tenantName("Acme Corp")
        .tenantDomain("acme")
        .email("admin@acme.com")
        .firstName("John")
        .lastName("Doe")
        .password("securePassword")
        .build();
    
    try {
        Tenant tenant = registrationService.registerTenant(request);
        log.info("Successfully registered tenant: {}", tenant.getId());
    } catch (IllegalArgumentException e) {
        log.error("Registration failed: {}", e.getMessage());
    }
}
```

The service performs validation to ensure tenant domains and user emails are unique, creates the tenant with an owner user having OWNER role, establishes the ownership relationship, creates a default organization, and executes custom processing hooks before and after registration.
