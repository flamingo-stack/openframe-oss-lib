Handles the complete tenant registration workflow, including tenant creation, owner user setup, and related organizational data initialization.

## Key Components

- **`registerTenant(TenantRegistrationRequest)`** - Main registration method that orchestrates the entire tenant creation process
- **Dependencies:**
  - `UserService` - Manages user creation and validation
  - `TenantService` - Handles tenant entity operations
  - `OrganizationService` - Creates default organizational structure
  - `RegistrationProcessor` - Provides pre/post-processing hooks

## Usage Example

```java
@Autowired
private TenantRegistrationService registrationService;

public void createNewTenant() {
    TenantRegistrationRequest request = TenantRegistrationRequest.builder()
        .tenantName("Acme Corp")
        .tenantDomain("acme")
        .email("admin@acme.com")
        .firstName("John")
        .lastName("Doe")
        .password("securePassword123")
        .build();
    
    try {
        Tenant newTenant = registrationService.registerTenant(request);
        log.info("Tenant registered successfully: {}", newTenant.getId());
    } catch (IllegalArgumentException e) {
        log.error("Registration failed: {}", e.getMessage());
    }
}
```

The service validates uniqueness of both tenant domain and user email, creates the tenant with an OWNER user, establishes default organization structure, and provides extension points for custom registration logic.
