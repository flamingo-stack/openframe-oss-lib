Abstract base class that provides common OIDC client registration logic for different OAuth2/OIDC providers in a multi-tenant environment. It builds Spring Security OAuth2 client registrations using provider-specific properties and tenant-specific SSO configurations.

## Key Components

- **BaseOidcClientRegistrationStrategy** - Abstract base class implementing `ClientRegistrationStrategy`
- **buildClient(String tenantId)** - Main method that constructs a complete OAuth2 client registration
- **props()** - Abstract method for subclasses to provide provider-specific OIDC properties
- **ssoConfigService** - Injected service for retrieving tenant SSO configurations
- **capitalize(String s)** - Utility method for formatting provider names

## Usage Example

```java
// Extend the base class for a specific OIDC provider
public class GoogleOidcStrategy extends BaseOidcClientRegistrationStrategy {
    
    private final GoogleOidcProperties googleProps;
    
    public GoogleOidcStrategy(SSOConfigService ssoConfigService, 
                             GoogleOidcProperties googleProps) {
        super(ssoConfigService);
        this.googleProps = googleProps;
    }
    
    @Override
    protected AbstractOidcProviderProperties props() {
        return googleProps;
    }
    
    @Override
    public String providerId() {
        return "google";
    }
}

// Use the strategy to build client registration
ClientRegistration registration = strategy.buildClient("tenant-123");
// Results in a fully configured OAuth2 client with:
// - Client ID/secret from tenant SSO config
// - Authorization/token/userinfo URLs from provider properties
// - Standard OIDC scopes and authentication method
// - Tenant-specific client name: "Google (tenant-123)"
```
