A Spring component that provides default configuration values for Microsoft OAuth/OIDC authentication provider integration.

## Key Components

- **MicrosoftDefaultProviderConfig** - Main configuration class implementing `DefaultProviderConfig`
- **providerId()** - Returns the Microsoft provider identifier constant
- **getDefaultClientId()** - Retrieves default OAuth client ID from properties
- **getDefaultClientSecret()** - Retrieves default OAuth client secret from properties
- **MicrosoftSSOProperties** - Injected dependency containing Microsoft-specific configuration

## Usage Example

```java
@Autowired
private MicrosoftDefaultProviderConfig microsoftConfig;

// Get provider information
String providerId = microsoftConfig.providerId(); // Returns "microsoft"
String clientId = microsoftConfig.getDefaultClientId();
String clientSecret = microsoftConfig.getDefaultClientSecret();

// Use in OAuth flow configuration
OAuthClientRegistration registration = OAuthClientRegistration.builder()
    .registrationId(providerId)
    .clientId(clientId)
    .clientSecret(clientSecret)
    .build();
```

This component acts as a bridge between the Microsoft SSO properties configuration and the authentication service, providing a standardized interface for accessing default Microsoft OAuth credentials and settings.
