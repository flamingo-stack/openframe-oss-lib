A Spring component that provides default Microsoft SSO configuration by implementing the DefaultProviderConfig interface for Microsoft OAuth/OIDC authentication.

## Key Components

- **MicrosoftDefaultProviderConfig** - Main configuration class that implements `DefaultProviderConfig`
- **providerId()** - Returns the Microsoft provider identifier constant
- **getDefaultClientId()** - Retrieves the default Microsoft OAuth client ID
- **getDefaultClientSecret()** - Retrieves the default Microsoft OAuth client secret

## Usage Example

This component is automatically registered by Spring and can be injected where Microsoft SSO configuration is needed:

```java
@Service
public class SSOService {
    
    private final MicrosoftDefaultProviderConfig microsoftConfig;
    
    public SSOService(MicrosoftDefaultProviderConfig microsoftConfig) {
        this.microsoftConfig = microsoftConfig;
    }
    
    public void configureMicrosoftSSO() {
        String providerId = microsoftConfig.providerId(); // "microsoft"
        String clientId = microsoftConfig.getDefaultClientId();
        String clientSecret = microsoftConfig.getDefaultClientSecret();
        
        // Configure Microsoft OAuth client...
    }
}
```

The configuration values are sourced from `MicrosoftSSOProperties`, which typically loads from application properties or environment variables.
