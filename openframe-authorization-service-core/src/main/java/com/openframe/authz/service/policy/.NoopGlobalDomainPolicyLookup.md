A no-operation implementation of `GlobalDomainPolicyLookup` that serves as a default fallback when no other domain policy lookup is configured. This component always returns empty results, effectively disabling domain-based tenant resolution.

## Key Components

- **NoopGlobalDomainPolicyLookup**: Main component class implementing `GlobalDomainPolicyLookup`
- **findTenantIdByDomainIfAutoAllowed()**: Returns `Optional.empty()` for any domain lookup
- **@ConditionalOnMissingBean**: Auto-configures only when no other `GlobalDomainPolicyLookup` bean exists

## Usage Example

This component is automatically registered by Spring Boot when no custom domain policy lookup is provided:

```java
// When injected, this will be the NoopGlobalDomainPolicyLookup
@Autowired
private GlobalDomainPolicyLookup domainPolicyLookup;

public void checkDomain(String domain) {
    Optional<String> tenantId = domainPolicyLookup.findTenantIdByDomainIfAutoAllowed(domain);
    // tenantId will always be Optional.empty() with this implementation
    if (tenantId.isEmpty()) {
        // Handle case where no tenant mapping exists
    }
}
```

To override this default behavior, simply provide your own `GlobalDomainPolicyLookup` bean implementation.
