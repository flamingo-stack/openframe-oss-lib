A no-operation implementation of `GlobalDomainPolicyLookup` that serves as a fallback component when no actual domain policy lookup service is configured. This component always returns empty results for domain-based tenant lookups.

## Key Components

- **NoopGlobalDomainPolicyLookup**: Spring component implementing `GlobalDomainPolicyLookup` interface
- **@ConditionalOnMissingBean**: Auto-configures only when no other `GlobalDomainPolicyLookup` bean exists (excluding itself)
- **findTenantIdByDomainIfAutoAllowed()**: Returns `Optional.empty()` for all domain lookups

## Usage Example

This component is automatically registered by Spring Boot when no other domain policy lookup implementation is available:

```java
// Spring will automatically wire this when no other implementation exists
@Autowired
private GlobalDomainPolicyLookup domainLookup;

public void handleDomainRequest(String domain) {
    Optional<String> tenantId = domainLookup.findTenantIdByDomainIfAutoAllowed(domain);
    if (tenantId.isEmpty()) {
        // With NoopGlobalDomainPolicyLookup, this will always execute
        handleUnknownDomain(domain);
    }
}

// To provide actual functionality, create your own implementation:
@Component
public class DatabaseGlobalDomainPolicyLookup implements GlobalDomainPolicyLookup {
    @Override
    public Optional<String> findTenantIdByDomainIfAutoAllowed(String domain) {
        // Your actual lookup logic here
        return tenantRepository.findByDomain(domain);
    }
}
```
