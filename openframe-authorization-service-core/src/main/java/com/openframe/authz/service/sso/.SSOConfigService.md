Manages SSO (Single Sign-On) configuration retrieval and processing for multi-tenant applications with fallback to default configurations.

## Key Components

- **getSSOConfig()** - Retrieves active SSO configuration for specific tenant and provider
- **getEffectiveSSOConfig()** - Gets tenant-specific config with fallback to global defaults
- **getActiveForTenant()** - Returns all active SSO configurations for a tenant
- **getEffectiveProvidersForTenant()** - Lists available SSO providers (tenant + defaults)
- **findAutoProvisionByDomain()** - Finds SSO config for auto-provisioning by email domain
- **getDecryptedClientSecret()** - Decrypts stored client secrets

## Usage Example

```java
@Autowired
private SSOConfigService ssoConfigService;

// Get effective SSO configuration with fallback
Optional<SSOConfig> config = ssoConfigService.getEffectiveSSOConfig("tenant123", "google");

if (config.isPresent()) {
    String clientId = config.get().getClientId();
    String clientSecret = ssoConfigService.getDecryptedClientSecret(config.get());
}

// Get all available providers for tenant
List<String> providers = ssoConfigService.getEffectiveProvidersForTenant("tenant123");

// Find auto-provisioning config by email domain
Optional<SSOPerTenantConfig> autoConfig = 
    ssoConfigService.findAutoProvisionByDomain("company.com");
```
