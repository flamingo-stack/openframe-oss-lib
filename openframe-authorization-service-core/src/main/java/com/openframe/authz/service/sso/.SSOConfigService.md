A service that manages Single Sign-On (SSO) configuration for multi-tenant applications, providing tenant-specific and global fallback SSO provider configurations.

## Key Components

- **getSSOConfig()** - Retrieves active SSO configuration for a specific tenant and provider
- **getEffectiveSSOConfig()** - Gets tenant-specific config with fallback to global properties-based defaults
- **getActiveForTenant()** - Returns all active SSO configurations for a tenant
- **getEffectiveProvidersForTenant()** - Lists all available SSO providers (tenant + default) for a tenant
- **findAutoProvisionByDomain()** - Finds SSO config that supports auto-provisioning for email domains
- **getDecryptedClientSecret()** - Decrypts stored client secrets using the encryption service

## Usage Example

```java
@Autowired
private SSOConfigService ssoConfigService;

// Get SSO config for tenant and provider
Optional<SSOPerTenantConfig> config = ssoConfigService.getSSOConfig("tenant123", "google");

// Get effective config with fallback to defaults
Optional<SSOConfig> effectiveConfig = ssoConfigService.getEffectiveSSOConfig("tenant123", "github");

// Get all available providers for a tenant
List<String> providers = ssoConfigService.getEffectiveProvidersForTenant("tenant123");

// Find auto-provisioning config by email domain
Optional<SSOPerTenantConfig> autoConfig = ssoConfigService.findAutoProvisionByDomain("company.com");

// Decrypt client secret for use
if (effectiveConfig.isPresent()) {
    String secret = ssoConfigService.getDecryptedClientSecret(effectiveConfig.get());
}
```
