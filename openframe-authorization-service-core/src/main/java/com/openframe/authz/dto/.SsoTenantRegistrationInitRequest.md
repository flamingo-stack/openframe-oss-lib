A data transfer object that encapsulates the initial request payload for starting SSO-based tenant registration processes.

## Key Components

- **SsoTenantRegistrationInitRequest**: Main DTO class with validation annotations
- **email**: Required validated email address for the registration
- **accessCode**: Optional access code for registration authorization
- **tenantName**: Required organization name with pattern validation (2-100 characters)
- **tenantDomain**: Validated tenant domain identifier
- **provider**: Required SSO provider specification
- **redirectTo**: Optional redirect URL after registration completion

## Usage Example

```java
// Create registration request
SsoTenantRegistrationInitRequest request = new SsoTenantRegistrationInitRequest(
    "admin@company.com",
    "ABC123",
    "Acme Corporation",
    "acme",
    "google",
    "https://app.example.com/dashboard"
);

// Or using builder pattern with Lombok
SsoTenantRegistrationInitRequest request = new SsoTenantRegistrationInitRequest();
request.setEmail("admin@company.com");
request.setTenantName("Acme Corporation");
request.setTenantDomain("acme");
request.setProvider("google");
request.setRedirectTo("https://app.example.com/dashboard");

// Validation will automatically enforce:
// - Valid email format
// - Organization name pattern (letters, numbers, common punctuation)
// - Required fields are not blank
// - Tenant domain format compliance
```
