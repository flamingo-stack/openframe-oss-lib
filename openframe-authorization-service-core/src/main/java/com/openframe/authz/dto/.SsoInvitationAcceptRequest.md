Data Transfer Object for handling SSO invitation acceptance requests in the authorization system.

## Key Components

- **invitationId** - Required identifier for the invitation being accepted
- **provider** - Required SSO provider name (e.g., "google", "microsoft")
- **switchTenant** - Optional flag to indicate tenant switching during acceptance
- **redirectTo** - Optional URL for post-acceptance redirection

## Usage Example

```java
// Create invitation acceptance request
SsoInvitationAcceptRequest request = new SsoInvitationAcceptRequest();
request.setInvitationId("inv-123456");
request.setProvider("google");
request.setSwitchTenant(true);
request.setRedirectTo("/dashboard");

// Or using constructor
SsoInvitationAcceptRequest request = new SsoInvitationAcceptRequest(
    "inv-123456", 
    "google", 
    true, 
    "/dashboard"
);

// Validation will ensure invitationId and provider are not blank
```

The class uses Lombok annotations for automatic getter/setter generation and Jakarta validation to enforce required fields. Typically used in REST controllers to capture invitation acceptance parameters from client requests.
