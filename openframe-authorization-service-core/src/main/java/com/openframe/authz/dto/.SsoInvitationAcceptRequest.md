A data transfer object (DTO) representing the request payload for accepting SSO invitations in the authorization service.

## Key Components

- **invitationId** - Required unique identifier for the invitation being accepted
- **provider** - Required SSO provider name (e.g., "google", "microsoft", "okta")
- **switchTenant** - Optional boolean flag to indicate if user should switch to the invited tenant
- **redirectTo** - Optional URL for post-acceptance redirection

## Usage Example

```java
// Create an SSO invitation acceptance request
SsoInvitationAcceptRequest request = new SsoInvitationAcceptRequest(
    "inv_123456789",
    "google",
    true,
    "/dashboard"
);

// Or using builder pattern with Lombok
SsoInvitationAcceptRequest request = new SsoInvitationAcceptRequest();
request.setInvitationId("inv_123456789");
request.setProvider("google");
request.setSwitchTenant(false);
request.setRedirectTo("/profile");

// Validation ensures invitationId and provider are not blank
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and standard object methods. Jakarta validation ensures required fields are properly validated during request processing.
