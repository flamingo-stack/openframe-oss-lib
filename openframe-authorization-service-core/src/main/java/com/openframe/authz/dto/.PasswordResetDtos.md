Contains DTO classes for password reset operations in the authentication system, providing request and confirmation data structures with comprehensive validation rules.

## Key Components

- **ResetRequest** - Inner static class for initiating password reset requests
- **ResetConfirm** - Inner static class for confirming password reset with new password
- **@ValidEmail** - Custom validation annotation for email format validation
- **@Pattern** - Complex regex validation for strong password requirements

## Usage Example

```java
// Create password reset request
PasswordResetDtos.ResetRequest resetRequest = new PasswordResetDtos.ResetRequest();
resetRequest.setEmail("user@example.com");

// Create password reset confirmation
PasswordResetDtos.ResetConfirm resetConfirm = new PasswordResetDtos.ResetConfirm();
resetConfirm.setToken("reset-token-12345");
resetConfirm.setNewPassword("SecurePass123!");

// Validation automatically enforces:
// - Valid email format
// - Non-blank token
// - Password minimum 8 characters
// - Password contains uppercase, lowercase, digit, and special character
```

The DTOs enforce strong security requirements including email validation and complex password policies with uppercase, lowercase, numeric, and special character requirements.
