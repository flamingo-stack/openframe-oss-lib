A data transfer object (DTO) that represents a user registration request initiated through an invitation, extending the base CoreUserRequest with invitation-specific fields.

## Key Components

- **InvitationRegistrationRequest** - Main DTO class extending CoreUserRequest
- **invitationId** - Required string field identifying the invitation being accepted
- **switchTenant** - Optional boolean flag indicating whether to switch tenant context
- Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create invitation registration request using builder pattern
InvitationRegistrationRequest request = InvitationRegistrationRequest.builder()
    .invitationId("inv-12345")
    .switchTenant(true)
    // Additional CoreUserRequest fields would be set here
    .build();

// Or using constructor
InvitationRegistrationRequest request = new InvitationRegistrationRequest(
    "inv-12345", 
    true
);

// Access fields
String invitationId = request.getInvitationId();
Boolean shouldSwitch = request.getSwitchTenant();
```

The class leverages Jakarta validation to ensure the invitation ID is provided, and uses Lombok's SuperBuilder to support inheritance from CoreUserRequest while maintaining builder pattern functionality.
