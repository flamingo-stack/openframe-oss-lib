A simple response DTO that encapsulates the results of a tenant domain availability check, indicating whether a requested domain is available and providing alternative suggestions if not.

## Key Components

- **isAvailable** - Boolean flag indicating if the requested tenant domain is available
- **suggestedUrl** - List of alternative domain suggestions when the requested domain is unavailable
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern
- **Jackson annotation** - Excludes null fields from JSON serialization

## Usage Example

```java
// Creating a response when domain is available
TenantAvailabilityResponse availableResponse = TenantAvailabilityResponse.builder()
    .isAvailable(true)
    .build();

// Creating a response when domain is unavailable with suggestions
TenantAvailabilityResponse unavailableResponse = TenantAvailabilityResponse.builder()
    .isAvailable(false)
    .suggestedUrl(List.of("acme-corp1.example.com", "acme-corp2.example.com"))
    .build();

// Typical usage in a REST controller
@GetMapping("/tenant/availability")
public TenantAvailabilityResponse checkAvailability(@RequestParam String domain) {
    return tenantService.checkDomainAvailability(domain);
}
```
