A response DTO that encapsulates the results of checking tenant domain availability, including availability status and suggested alternative URLs when a domain is unavailable.

## Key Components

- **`isAvailable`** - Boolean flag indicating whether the requested tenant domain is available for use
- **`suggestedUrl`** - List of alternative domain suggestions provided when the requested domain is not available
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern
- **`@JsonInclude(NON_NULL)`** - Excludes null fields from JSON serialization to reduce payload size

## Usage Example

```java
// Creating a response for an available domain
TenantAvailabilityResponse availableResponse = TenantAvailabilityResponse.builder()
    .isAvailable(true)
    .build();

// Creating a response for an unavailable domain with suggestions
TenantAvailabilityResponse unavailableResponse = TenantAvailabilityResponse.builder()
    .isAvailable(false)
    .suggestedUrl(List.of("mycompany-corp", "mycompany-inc", "mycompany-llc"))
    .build();

// Checking availability in a service
if (response.isAvailable()) {
    // Proceed with tenant creation
    createTenant(requestedDomain);
} else {
    // Show suggestions to user
    displayAlternatives(response.getSuggestedUrl());
}
```
