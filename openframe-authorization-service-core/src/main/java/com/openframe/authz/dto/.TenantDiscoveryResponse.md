A Data Transfer Object (DTO) that encapsulates the response data for tenant discovery operations, providing information about available tenants and authentication providers for a given email address.

## Key Components

- **email** - The email address used for tenant discovery
- **hasExistingAccounts** - Boolean flag indicating if the email has existing accounts
- **tenantId** - Identifier of the discovered tenant
- **authProviders** - List of available authentication provider names

## Usage Example

```java
// Creating a tenant discovery response
TenantDiscoveryResponse response = TenantDiscoveryResponse.builder()
    .email("user@example.com")
    .hasExistingAccounts(true)
    .tenantId("tenant-123")
    .authProviders(List.of("google", "microsoft", "saml"))
    .build();

// Accessing response data
if (response.isHasExistingAccounts()) {
    String tenantId = response.getTenantId();
    List<String> providers = response.getAuthProviders();
    // Process available authentication options
}

// JSON serialization automatically handles property mapping
// hasExistingAccounts -> "has_existing_accounts"
// tenantId -> "tenant_id" 
// authProviders -> "auth_providers"
```
