A response DTO that provides tenant discovery information based on an email address, including available authentication providers and existing account status.

## Key Components

- **email** - The email address used for tenant discovery
- **hasExistingAccounts** - Boolean flag indicating if accounts exist for this email
- **tenantId** - The discovered tenant identifier
- **authProviders** - List of available authentication provider names for the tenant

## Usage Example

```java
// Building a tenant discovery response
TenantDiscoveryResponse response = TenantDiscoveryResponse.builder()
    .email("user@example.com")
    .hasExistingAccounts(true)
    .tenantId("tenant-123")
    .authProviders(List.of("google", "saml", "local"))
    .build();

// Accessing response data
if (response.isHasExistingAccounts()) {
    String tenant = response.getTenantId();
    List<String> providers = response.getAuthProviders();
    // Route user to appropriate auth flow
}
```

The class uses Jackson annotations for JSON serialization with snake_case field names (`has_existing_accounts`, `tenant_id`, `auth_providers`) while maintaining camelCase in Java code through Lombok's automatic getter/setter generation.
