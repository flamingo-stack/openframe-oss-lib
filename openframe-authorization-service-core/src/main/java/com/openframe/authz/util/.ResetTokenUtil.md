A utility class for generating cryptographically secure reset tokens using URL-safe Base64 encoding without padding.

## Key Components

- **`generateResetToken()`** - Static method that generates a secure 32-byte random token encoded as a URL-safe Base64 string
- **`secureRandom`** - Static SecureRandom instance for cryptographically strong random number generation
- **`base64Url`** - Static Base64.Encoder configured for URL-safe encoding without padding

## Usage Example

```java
// Generate a secure reset token for password recovery
String resetToken = ResetTokenUtil.generateResetToken();
System.out.println("Reset token: " + resetToken);
// Output: Reset token: XyZ123AbC456DeF789... (43 characters, URL-safe)

// Use in password reset flow
public void initiatePasswordReset(String email) {
    String token = ResetTokenUtil.generateResetToken();
    // Store token in database with expiration
    passwordResetService.createResetRequest(email, token);
    // Send token via email
    emailService.sendResetLink(email, token);
}

// Validate reset token
public boolean isValidResetToken(String providedToken) {
    // Compare with stored token from database
    return storedToken.equals(providedToken) && !isExpired();
}
```

The utility generates 256-bit (32-byte) tokens that are URL-safe and suitable for use in password reset links, account verification, or other security-sensitive operations requiring unpredictable tokens.
