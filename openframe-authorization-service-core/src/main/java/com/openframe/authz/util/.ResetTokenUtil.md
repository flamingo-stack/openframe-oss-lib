A utility class that generates cryptographically secure reset tokens for password reset functionality and similar authentication operations.

## Key Components

- **`generateResetToken()`** - Static method that creates a URL-safe, base64-encoded random token
- **`SecureRandom`** - Cryptographically strong random number generator instance
- **`Base64.Encoder`** - URL-safe base64 encoder without padding for clean token format

## Usage Example

```java
// Generate a secure reset token for password reset emails
String resetToken = ResetTokenUtil.generateResetToken();
System.out.println(resetToken); // Example output: "k7B2xN9mP1qR8sT4vW6yZ3aC5dF7gH9j"

// Typical usage in a service class
public class PasswordResetService {
    public void initiatePasswordReset(String email) {
        String token = ResetTokenUtil.generateResetToken();
        // Store token in database with expiration
        // Send token via email to user
        sendResetEmail(email, token);
    }
}
```

The utility generates 256-bit random tokens that are URL-safe and suitable for use in reset links, API tokens, or any scenario requiring secure random identifiers.
