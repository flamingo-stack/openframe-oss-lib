Utility class for extracting and normalizing user identity fields from OIDC token claims, with special handling for Azure Active Directory (AAD) organizational accounts.

## Key Components

| Method | Description |
|--------|-------------|
| `resolveEmail(OidcUser)` | Resolves an email identifier from claims using a priority fallback chain: `email` → `preferred_username` → `upn` → `unique_name` |
| `stringClaim(Object)` | Returns a claim value as a non-blank `String`, or `null` |
| `resolvePictureUrl(OidcUser)` | Resolves the user's profile picture URL from the standard OIDC `picture` claim (Google only; returns `null` for Microsoft) |

## Usage Example

```java
@Service
public class UserProfileService {

    public UserProfile buildProfile(OidcUser oidcUser) {
        String email = OidcUserUtils.resolveEmail(oidcUser);
        String pictureUrl = OidcUserUtils.resolvePictureUrl(oidcUser);

        return UserProfile.builder()
                .email(email)
                .avatarUrl(pictureUrl)
                .build();
    }
}
```

> **Note:** Microsoft AAD tokens do not include a `picture` claim in the id_token. Retrieving a profile photo requires a separate call to the Microsoft Graph API (`/me/photo/$value`). `resolvePictureUrl` will return `null` for AAD users.
