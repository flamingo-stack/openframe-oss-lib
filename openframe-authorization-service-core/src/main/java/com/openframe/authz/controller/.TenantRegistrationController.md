REST controller that handles tenant registration workflows, supporting both direct JSON registration and SSO-based registration flows with proper session management.

## Key Components

- **`register()`** - Endpoint for direct tenant registration via JSON payload
- **`startSsoRegistration()`** - Initiates SSO registration flow with OAuth2 redirect
- **`clearAuthenticationState()`** - Cleans up existing sessions and cookies before SSO flow
- **`buildSsoRegistrationCookie()`** - Creates secure HTTP-only cookies for SSO state tracking

## Usage Example

```java
// Direct tenant registration
@Autowired
private TenantRegistrationController controller;

// Register tenant directly
TenantRegistrationRequest request = new TenantRegistrationRequest();
request.setTenantName("example-corp");
Tenant newTenant = controller.register(request);

// SSO registration flow (typically called via HTTP)
// GET /oauth/register/sso?domain=example.com&provider=google
// - Clears existing auth state
// - Sets secure SSO registration cookie
// - Redirects to OAuth2 authorization endpoint
```

The controller integrates with `TenantRegistrationService` for direct registration and `SsoTenantRegistrationService` for OAuth2-based flows, ensuring clean session management and secure cookie handling throughout the registration process.
