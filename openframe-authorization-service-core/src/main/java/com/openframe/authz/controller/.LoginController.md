A Spring Boot controller that handles login page routing and user authentication flow for the OpenFrame multi-tenant authorization system.

## Key Components

- **`@Controller`** - Spring MVC controller annotation for web request handling
- **`login()` method** - Handles GET requests to `/login` with error and logout message support
- **`index()` method** - Serves the root path (`/`) with a welcome message
- **Model attributes** - Passes error messages, logout confirmations, and system info to views

## Usage Example

```java
// The controller automatically handles these URL patterns:

// Basic login page access
// GET /login -> returns "login" view

// Login with error parameter (after failed authentication)
// GET /login?error -> adds "Invalid credentials" message to model

// Login after successful logout
// GET /login?logout -> adds "Logged out successfully" message to model

// Root path access
// GET / -> returns "index" view with welcome message

// Example of how Spring Security might redirect to this controller:
// After failed login attempt, Spring Security redirects to /login?error
// After logout, Spring Security redirects to /login?logout
```

The controller integrates with Spring Security's default authentication flow, handling both successful logout confirmations and authentication error messages through URL parameters.
