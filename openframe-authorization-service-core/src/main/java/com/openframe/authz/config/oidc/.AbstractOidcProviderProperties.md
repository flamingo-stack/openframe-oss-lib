This abstract base class provides common configuration properties and URL resolution logic for OIDC (OpenID Connect) providers in an authorization system.

## Key Components

**Properties:**
- `registrationRedirectUri` / `loginRedirectUri` - OAuth2 callback URLs
- `defaultClientId` / `defaultClientSecret` - Fallback credentials when tenant-specific config is unavailable
- OIDC endpoint URLs: `authorizationUrl`, `tokenUrl`, `userInfoUrl`, `issuerUri`, `jwkSetUri`
- `scopes` - List of requested OAuth2 scopes

**Methods:**
- `effectiveAuthorizationUrl()` - Returns the authorization endpoint URL for a given tenant
- `effectiveTokenUrl()` - Returns the token endpoint URL for a given tenant  
- `effectiveJwkSetUri()` - Returns the JWK Set URI for a given tenant
- `effectiveIssuerUri()` - Returns the issuer URI for a given tenant

## Usage Example

```java
// Extend the abstract class for a specific OIDC provider
public class AzureAdProviderProperties extends AbstractOidcProviderProperties {
    // Provider-specific implementation
}

// Configure properties
AzureAdProviderProperties props = new AzureAdProviderProperties();
props.setAuthorizationUrl("https://login.microsoftonline.com/common/oauth2/v2.0/authorize");
props.setDefaultClientId("your-client-id");
props.setScopes(List.of("openid", "profile", "email"));

// Get effective URLs for tenant-specific operations
String authUrl = props.effectiveAuthorizationUrl("tenant-123");
String tokenUrl = props.effectiveTokenUrl("tenant-123");
```
