Configuration properties class that defines Google-specific SSO settings for OpenID Connect authentication, extending the base OIDC provider properties with Google-specific configurations.

## Key Components

- **GoogleSSOProperties**: Main configuration class annotated with `@ConfigurationProperties` for binding Google SSO settings
- **GOOGLE**: String constant defining the provider identifier
- **Inheritance**: Extends `AbstractOidcProviderProperties` to inherit common OIDC configuration properties

## Usage Example

```java
// Application properties configuration
# application.yml
openframe:
  sso:
    google:
      client-id: your-google-client-id
      client-secret: your-google-client-secret
      redirect-uri: https://yourapp.com/oauth2/callback/google

// Injecting the properties in a service
@Service
public class GoogleAuthService {
    
    private final GoogleSSOProperties googleProperties;
    
    public GoogleAuthService(GoogleSSOProperties googleProperties) {
        this.googleProperties = googleProperties;
    }
    
    public void configureGoogleAuth() {
        String provider = GoogleSSOProperties.GOOGLE;
        // Access inherited properties like clientId, clientSecret, etc.
        String clientId = googleProperties.getClientId();
    }
}
```
