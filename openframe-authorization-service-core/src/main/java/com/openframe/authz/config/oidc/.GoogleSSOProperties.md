Configuration properties class for Google OAuth2/OpenID Connect Single Sign-On integration within the OpenFrame authorization system.

## Key Components

- **GoogleSSOProperties**: Extends `AbstractOidcProviderProperties` to inherit common OIDC configuration properties
- **GOOGLE constant**: String identifier for the Google SSO provider
- **@ConfigurationProperties**: Binds to `openframe.sso.google.*` configuration properties
- **@Component**: Registers as a Spring-managed bean for dependency injection

## Usage Example

```java
// Application configuration (application.yml)
openframe:
  sso:
    google:
      client-id: your-google-client-id
      client-secret: your-google-client-secret
      redirect-uri: https://yourapp.com/auth/callback
      scopes: openid,profile,email

// Injecting and using the properties
@Service
public class GoogleAuthService {
    
    private final GoogleSSOProperties googleProperties;
    
    public GoogleAuthService(GoogleSSOProperties googleProperties) {
        this.googleProperties = googleProperties;
    }
    
    public String getProviderName() {
        return GoogleSSOProperties.GOOGLE; // Returns "google"
    }
    
    public String getClientId() {
        return googleProperties.getClientId();
    }
}
```

This properties class enables type-safe configuration binding for Google SSO settings, inheriting standard OIDC properties like client credentials, scopes, and endpoints from its parent class.
