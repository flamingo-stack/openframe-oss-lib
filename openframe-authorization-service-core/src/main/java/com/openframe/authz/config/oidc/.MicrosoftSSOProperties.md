Configuration properties class for Microsoft Azure AD OAuth2/OIDC authentication, supporting both single-tenant and multi-tenant scenarios with dynamic endpoint resolution.

## Key Components

- **MicrosoftSSOProperties** - Main configuration class extending `AbstractOidcProviderProperties`
- **Common Endpoint Properties** - Optional multi-tenant endpoint URLs for shared Azure AD scenarios
- **resolveEndpoint()** - Private helper method for tenant-specific URL resolution
- **Effective URL Methods** - Override methods providing dynamic endpoint resolution based on tenant ID

## Usage Example

```java
// Configuration in application.yml
openframe:
  sso:
    microsoft:
      client-id: "your-app-id"
      client-secret: "your-secret"
      # Tenant-specific endpoints
      authorization-url: "https://login.microsoftonline.com/{msTenantId}/oauth2/v2.0/authorize"
      token-url: "https://login.microsoftonline.com/{msTenantId}/oauth2/v2.0/token"
      # Multi-tenant fallback endpoints
      common-authorization-url: "https://login.microsoftonline.com/common/oauth2/v2.0/authorize"
      common-token-url: "https://login.microsoftonline.com/common/oauth2/v2.0/token"

// Programmatic usage
@Autowired
private MicrosoftSSOProperties microsoftProperties;

// Resolve tenant-specific endpoint
String authUrl = microsoftProperties.effectiveAuthorizationUrl("tenant-123");

// Use common endpoint when tenant ID is null/blank
String commonAuthUrl = microsoftProperties.effectiveAuthorizationUrl(null);
```

The class automatically resolves between tenant-specific URLs (with `{msTenantId}` placeholder) and common multi-tenant endpoints based on whether a tenant ID is provided.
