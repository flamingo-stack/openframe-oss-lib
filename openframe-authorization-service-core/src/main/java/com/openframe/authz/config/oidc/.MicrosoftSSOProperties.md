Configuration properties class for Microsoft Azure AD OpenID Connect integration, providing tenant-specific and multi-tenant endpoint resolution.

## Key Components

- **MicrosoftSSOProperties**: Main configuration class extending `AbstractOidcProviderProperties`
- **MICROSOFT**: Static identifier constant for the provider
- **Common endpoint properties**: Multi-tenant endpoint URLs (commonAuthorizationUrl, commonTokenUrl, etc.)
- **resolveEndpoint()**: Private method for tenant-specific URL resolution
- **effective*() methods**: Public methods returning tenant-specific or common endpoints

## Usage Example

```java
// Configuration in application.yml
openframe:
  sso:
    microsoft:
      client-id: "your-client-id"
      client-secret: "your-client-secret"
      authorization-url: "https://login.microsoftonline.com/{msTenantId}/oauth2/v2.0/authorize"
      token-url: "https://login.microsoftonline.com/{msTenantId}/oauth2/v2.0/token"
      common-authorization-url: "https://login.microsoftonline.com/common/oauth2/v2.0/authorize"
      common-token-url: "https://login.microsoftonline.com/common/oauth2/v2.0/token"

// Usage in service
@Autowired
private MicrosoftSSOProperties microsoftProps;

public void configureEndpoints(String tenantId) {
    String authUrl = microsoftProps.effectiveAuthorizationUrl(tenantId);
    String tokenUrl = microsoftProps.effectiveTokenUrl(tenantId);
    // URLs automatically resolve to tenant-specific or common endpoints
}
```
