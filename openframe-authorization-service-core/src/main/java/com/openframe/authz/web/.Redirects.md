A utility class providing convenient methods for issuing HTTP redirects with proper status codes and URL building in Spring web applications.

## Key Components

- **`seeOther(response, relativePath)`** - Issues 303 See Other redirect to a relative path
- **`found(response, relativePath)`** - Issues 302 Found redirect to a relative path  
- **`foundAtRoot(response, relativePath)`** - Issues 302 redirect building URL against server root, ignoring context path
- **`buildAbsolute(relativePath)`** - Converts relative path to absolute URL using current context path
- **`redirect(response, status, relativePath)`** - Generic redirect method with customizable status code

## Usage Example

```java
@Controller
public class AuthController {
    
    @PostMapping("/login")
    public void handleLogin(HttpServletResponse response) {
        // Redirect after successful login using 303 See Other
        Redirects.seeOther(response, "/dashboard");
    }
    
    @GetMapping("/legacy-url")
    public void handleLegacyUrl(HttpServletResponse response) {
        // Permanent redirect using 302 Found
        Redirects.found(response, "/new-url");
    }
    
    @PostMapping("/oauth/callback")
    public void handleOAuthCallback(HttpServletResponse response) {
        // Redirect to root-level OAuth path, ignoring context path
        Redirects.foundAtRoot(response, "/oauth/success");
    }
}
```

The class automatically builds absolute URLs from relative paths using Spring's `ServletUriComponentsBuilder`, ensuring proper URL resolution within the application context.
