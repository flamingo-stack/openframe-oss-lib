A utility class for handling HTTP redirects with proper status codes and absolute URL resolution.

## Key Components

- **`seeOther(HttpServletResponse, String)`** - Static method that performs a 303 See Other redirect to a relative path, automatically converting it to an absolute URL using the current context path

## Usage Example

```java
import com.openframe.authz.web.Redirects;
import jakarta.servlet.http.HttpServletResponse;

@Controller
public class AuthController {
    
    @PostMapping("/login")
    public void handleLogin(HttpServletResponse response) {
        // Authenticate user...
        
        // Redirect to dashboard after successful login
        Redirects.seeOther(response, "/dashboard");
        // This converts "/dashboard" to "http://localhost:8080/myapp/dashboard"
    }
    
    @PostMapping("/logout")
    public void handleLogout(HttpServletResponse response) {
        // Clear session...
        
        // Redirect to home page
        Redirects.seeOther(response, "/");
    }
}
```

The method automatically resolves relative paths to absolute URLs using Spring's `ServletUriComponentsBuilder` and sets the appropriate HTTP 303 status code, which indicates the response can be found at a different URI and should be retrieved using GET.
