A simple record class representing the payload structure for SSO invitation cookies used in authentication workflows.

## Key Components

- **SsoInviteCookiePayload**: Immutable record containing all necessary data for SSO invitation cookie validation and processing
- **Fields**:
  - `s` - Session or signature identifier
  - `invitationId` - Unique identifier for the invitation
  - `switchTenant` - Boolean flag indicating if tenant switching is required
  - `provider` - Authentication provider name
  - `redirectTo` - Post-authentication redirect URL
  - `iat` - Issued at timestamp (Unix epoch)
  - `exp` - Expiration timestamp (Unix epoch)

## Usage Example

```java
// Creating an SSO invite cookie payload
var payload = new SsoInviteCookiePayload(
    "session_abc123",
    "inv_456789",
    true,
    "google",
    "/dashboard",
    System.currentTimeMillis() / 1000,
    (System.currentTimeMillis() / 1000) + 3600 // 1 hour expiry
);

// Accessing payload data
String inviteId = payload.invitationId();
boolean needsTenantSwitch = payload.switchTenant();
long expirationTime = payload.exp();

// Checking if cookie is expired
boolean isExpired = System.currentTimeMillis() / 1000 > payload.exp();
```

This record provides type-safe access to SSO invitation cookie data and leverages Java's record features for automatic generation of constructors, getters, equals, hashCode, and toString methods.
