Handles SSO authentication flow for invited users by processing invitation cookies and completing user registration. This handler extracts invitation details from encrypted cookies, creates user accounts using OIDC authentication data, and redirects to the target tenant.

## Key Components

- **`cookieName()`** - Returns the SSO invite cookie name constant
- **`handle()`** - Main flow processing method that decodes invitation cookies, extracts user info from OIDC claims, registers the user, and performs redirect
- **`requireOidcUser()`** - Validates authentication contains OIDC user principal
- **`requireEmail()`** - Ensures email is available from OIDC user
- **`clearCookie()`/`clearSession()`** - Security cleanup methods
- **`buildRedirect()`** - Constructs redirect URL with tenant parameter

## Usage Example

```java
@Autowired
private InviteSsoHandler inviteSsoHandler;

// Used automatically in SSO flow when invite cookie is present
public void processSsoLogin(HttpServletRequest request, 
                           HttpServletResponse response, 
                           Authentication auth) {
    if (inviteSsoHandler.cookieName().equals(cookieName)) {
        inviteSsoHandler.handle(request, response, auth);
        // User is registered and redirected to target tenant
    }
}
```

The handler automatically extracts first/last names from OIDC claims, generates a random password, and clears authentication state after successful registration.
