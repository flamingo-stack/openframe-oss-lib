Defines the `SsoFlowHandler` interface, which provides a contract and shared default behaviors for handling SSO (Single Sign-On) OAuth/OIDC authentication flows in the OpenFrame authorization service.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `cookieName()` | Abstract method | Returns the name of the SSO flow tracking cookie |
| `resolveCookie()` | Default method | Finds the SSO flow cookie from the incoming request, returns `null` if absent |
| `requireCookie()` | Default method | Like `resolveCookie()` but throws `IllegalStateException` if the cookie is missing (expired session) |
| `isActivated()` | Default method | Returns `true` if the SSO flow cookie is present on the request |
| `handle()` | Abstract method | Core SSO flow logic — implemented per flow type |
| `requireOidcUser()` | Default method | Extracts the `OidcUser` principal from the `Authentication` object, throws if type is unexpected |
| `requireEmail()` | Default method | Extracts and lowercases the verified email from the OIDC user, throws if blank |
| `resolveNames()` | Default method | Resolves given/family name from OIDC claims, falling back to splitting `fullName` |
| `clearFlowCookieAndRedirect()` | Default method | Clears the flow cookie and redirects to `/oauth/continue?tenantId=...` |

## Usage Example

```java
public class InviteFlowHandler implements SsoFlowHandler {

    @Override
    public String cookieName() {
        return "sso_invite_flow";
    }

    @Override
    public void handle(HttpServletRequest request,
                       HttpServletResponse response,
                       Authentication authentication) throws Exception {

        Cookie flowCookie = requireCookie(request);
        OidcUser oidcUser = requireOidcUser(authentication);

        String email = requireEmail(oidcUser);
        String[] names = resolveNames(oidcUser);
        String tenantId = flowCookie.getValue();

        // ... process invite acceptance ...

        clearFlowCookieAndRedirect(response, flowCookie, tenantId, null);
    }
}
```

> **Note:** The `redirectTo` parameter in `clearFlowCookieAndRedirect()` is not yet wired — a TODO marks it for future local debugging support.
