This utility class defines constant values used for SSO (Single Sign-On) registration and onboarding processes within the authentication and authorization system.

## Key Components

- **COOKIE_SSO_REG** - Cookie name for SSO registration tracking (`"of_sso_reg"`)
- **COOKIE_SSO_INVITE** - Cookie name for SSO invitation handling (`"of_sso_invite"`)
- **ONBOARDING_TENANT_ID** - Predefined tenant identifier for SSO onboarding processes (`"sso-onboarding"`)

## Usage Example

```java
import com.openframe.authz.security.SsoRegistrationConstants;

// Set SSO registration cookie
Cookie registrationCookie = new Cookie(
    SsoRegistrationConstants.COOKIE_SSO_REG, 
    registrationToken
);

// Handle SSO invitation
String inviteCookieName = SsoRegistrationConstants.COOKIE_SSO_INVITE;
Cookie inviteCookie = request.getCookies()
    .stream()
    .filter(cookie -> cookie.getName().equals(inviteCookieName))
    .findFirst()
    .orElse(null);

// Check if using onboarding tenant
if (SsoRegistrationConstants.ONBOARDING_TENANT_ID.equals(tenantId)) {
    // Handle onboarding flow
    processOnboardingUser(user);
}
```

The class follows the utility pattern with a private constructor to prevent instantiation, ensuring all constants are accessed statically.
