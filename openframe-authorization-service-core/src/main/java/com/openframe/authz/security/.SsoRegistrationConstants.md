This utility class defines constants used for SSO (Single Sign-On) registration and onboarding processes in the OpenFrame authorization security module.

## Key Components

- **COOKIE_SSO_REG**: Cookie name for SSO registration tracking (`"of_sso_reg"`)
- **COOKIE_SSO_INVITE**: Cookie name for SSO invitation handling (`"of_sso_invite"`)
- **ONBOARDING_TENANT_ID**: Special tenant identifier for SSO onboarding (`"sso-onboarding"`)

## Usage Example

```java
import com.openframe.authz.security.SsoRegistrationConstants;

// Setting SSO registration cookie
Cookie regCookie = new Cookie(SsoRegistrationConstants.COOKIE_SSO_REG, "registration-token");
response.addCookie(regCookie);

// Checking for SSO invitation cookie
String inviteToken = request.getCookieValue(SsoRegistrationConstants.COOKIE_SSO_INVITE);

// Using onboarding tenant ID for tenant isolation
if (SsoRegistrationConstants.ONBOARDING_TENANT_ID.equals(currentTenantId)) {
    // Handle onboarding-specific logic
    processOnboardingUser(user);
}
```

The class follows the utility pattern with a private constructor to prevent instantiation, ensuring these constants are accessed statically throughout the application.
