A secure cookie encoder/decoder service for SSO (Single Sign-On) operations that uses HMAC-SHA256 signatures to ensure cookie integrity and authenticity.

## Key Components

- **encode()** - Generic method to serialize and sign any payload object
- **encodeTenant()/encodeInvite()** - Specialized encoders for tenant registration and invite cookies
- **decodeTenant()/decodeInvite()** - Specialized decoders with automatic expiration validation
- **decode()** - Generic decoder with signature verification and expiration checking
- **constantTimeEquals()** - Timing-attack resistant signature comparison

## Usage Example

```java
@Autowired
private SsoCookieCodec cookieCodec;

// Encode a tenant registration cookie
SsoTenantRegCookiePayload payload = new SsoTenantRegCookiePayload(/* data */);
String encodedCookie = cookieCodec.encodeTenant(payload);

// Set in HTTP response
response.addCookie(new Cookie("sso-tenant", encodedCookie));

// Decode and validate cookie
String cookieValue = getCookieValue(request, "sso-tenant");
Optional<SsoTenantRegCookiePayload> decoded = cookieCodec.decodeTenant(cookieValue);

if (decoded.isPresent()) {
    // Valid, non-expired cookie
    SsoTenantRegCookiePayload data = decoded.get();
    // Process registration data
} else {
    // Invalid, tampered, or expired cookie
    handleInvalidCookie();
}
```

The service automatically validates HMAC signatures and checks expiration timestamps (via reflection on `exp()` method) during decoding, returning empty Optional for invalid cookies.
