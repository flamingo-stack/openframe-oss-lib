A record class that represents the payload structure for SSO (Single Sign-On) tenant registration cookies, encapsulating authentication and tenant-specific information used during the SSO registration flow.

## Key Components

- **s** - Session or security identifier string
- **tenantName** - The name of the tenant being registered
- **tenantDomain** - The domain associated with the tenant
- **provider** - The SSO provider being used for authentication
- **redirectTo** - URL to redirect to after successful registration
- **accessCode** - Access code for the SSO flow
- **iat** - Issued at timestamp (Unix timestamp)
- **exp** - Expiration timestamp (Unix timestamp)

## Usage Example

```java
// Creating a new SSO tenant registration payload
SsoTenantRegCookiePayload payload = new SsoTenantRegCookiePayload(
    "session123",
    "acme-corp",
    "acme.com",
    "okta",
    "https://app.acme.com/dashboard",
    "auth_code_xyz789",
    System.currentTimeMillis() / 1000, // Current time in seconds
    (System.currentTimeMillis() / 1000) + 3600 // Expires in 1 hour
);

// Accessing payload data
String tenantDomain = payload.tenantDomain();
boolean isExpired = payload.exp() < (System.currentTimeMillis() / 1000);
```
