A record class that represents the payload structure for SSO (Single Sign-On) tenant registration cookies, containing authentication and tenant information needed during the registration process.

## Key Components

- **s**: Session or security identifier string
- **tenantName**: Name of the tenant being registered
- **tenantDomain**: Domain associated with the tenant
- **provider**: SSO provider identifier (e.g., "google", "microsoft")
- **redirectTo**: URL to redirect to after successful registration
- **accessCode**: Temporary access code for authentication
- **iat**: Issued at timestamp (Unix epoch)
- **exp**: Expiration timestamp (Unix epoch)

## Usage Example

```java
// Create a new SSO tenant registration cookie payload
SsoTenantRegCookiePayload payload = new SsoTenantRegCookiePayload(
    "session-123",
    "acme-corp",
    "acme.example.com",
    "google",
    "/dashboard",
    "temp-access-456",
    System.currentTimeMillis() / 1000,
    (System.currentTimeMillis() / 1000) + 3600 // 1 hour expiration
);

// Access payload data
String tenantName = payload.tenantName();
boolean isExpired = payload.exp() < (System.currentTimeMillis() / 1000);
```

This immutable record is typically used when encoding/decoding JWT tokens or cookies that manage the SSO tenant registration flow, ensuring all necessary context is preserved throughout the authentication process.
