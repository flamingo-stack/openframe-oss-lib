Handles SSO-based tenant registration flow by processing authentication callbacks and creating new tenant accounts.

## Key Components

- **`TenantRegSsoHandler`** - Main handler component implementing `SsoFlowHandler`
- **`handle()`** - Core method that processes SSO authentication and registers new tenant
- **`cookieName()`** - Returns the SSO registration cookie identifier
- **Dependencies:**
  - `SsoCookieCodec` - Decodes SSO registration cookies
  - `TenantRegistrationService` - Handles tenant creation logic

## Usage Example

```java
@Component
public class SsoAuthenticationSuccessHandler {
    
    private final TenantRegSsoHandler tenantRegHandler;
    
    public void onAuthenticationSuccess(HttpServletRequest request, 
                                      HttpServletResponse response,
                                      Authentication authentication) {
        // Check if this is a tenant registration flow
        Cookie ssoCookie = findCookie(request, tenantRegHandler.cookieName());
        if (ssoCookie != null) {
            // Process tenant registration with SSO user data
            tenantRegHandler.handle(request, response, authentication);
        }
    }
}
```

The handler extracts user information from the OIDC authentication, decodes the registration context from cookies, and creates a new tenant account with the authenticated user as the admin. It automatically generates a random password since authentication is handled via SSO, and ensures proper case formatting for tenant domains.
