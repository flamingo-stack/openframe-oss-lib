A record class that encapsulates RSA key pair material and their PEM string representations for authentication purposes.

## Key Components

- **`RSAPublicKey publicKey`** - The RSA public key object
- **`RSAPrivateKey privateKey`** - The RSA private key object  
- **`String publicPem`** - PEM-encoded representation of the public key
- **`String privatePem`** - PEM-encoded representation of the private key

## Usage Example

```java
// Typically created by a key generation service
AuthenticationKeyPair keyPair = new AuthenticationKeyPair(
    rsaPublicKey,
    rsaPrivateKey,
    "-----BEGIN PUBLIC KEY-----\n...",
    "-----BEGIN PRIVATE KEY-----\n..."
);

// Access the key objects for cryptographic operations
RSAPublicKey pubKey = keyPair.publicKey();
RSAPrivateKey privKey = keyPair.privateKey();

// Access PEM strings for storage or transmission
String publicKeyPem = keyPair.publicPem();
String privateKeyPem = keyPair.privatePem();
```

This immutable record provides a convenient way to bundle both the Java key objects and their string representations, making it easy to work with RSA keys in different contexts within an authentication system.
