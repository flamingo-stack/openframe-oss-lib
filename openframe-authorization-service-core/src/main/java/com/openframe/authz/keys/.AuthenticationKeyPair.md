A record class that encapsulates RSA key pair components for authentication purposes, including both the key objects and their PEM-encoded string representations.

## Key Components

- **RSA Key Objects**: `publicKey` and `privateKey` - The actual RSA key instances for cryptographic operations
- **PEM Encodings**: `publicPem` and `privatePem` - Base64-encoded string representations for storage and transmission
- **Immutable Design**: Record structure ensures thread-safe, immutable key pair data

## Usage Example

```java
// Creating an AuthenticationKeyPair (typically from a key generator)
AuthenticationKeyPair keyPair = new AuthenticationKeyPair(
    rsaPublicKey,
    rsaPrivateKey,
    "-----BEGIN PUBLIC KEY-----\nMIIBIjANBgkqhkiG...",
    "-----BEGIN PRIVATE KEY-----\nMIIEvQIBADANBgkqhk..."
);

// Accessing key components
RSAPublicKey publicKey = keyPair.publicKey();
String publicPem = keyPair.publicPem();

// Using for JWT signing or verification
String token = jwtService.sign(payload, keyPair.privateKey());
boolean valid = jwtService.verify(token, keyPair.publicKey());

// Storing PEM strings in configuration or database
config.setPublicKey(keyPair.publicPem());
```

This value object is commonly used in authentication systems where both programmatic access to RSA keys and serializable PEM formats are needed for different components of the security infrastructure.
