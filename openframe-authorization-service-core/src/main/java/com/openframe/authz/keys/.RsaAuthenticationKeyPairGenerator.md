Spring component that generates 2048-bit RSA key pairs for authentication purposes. Implements the AuthenticationKeyPairGenerator interface to provide RSA-specific key generation functionality with PEM format conversion.

## Key Components

- **RsaAuthenticationKeyPairGenerator**: Main component implementing AuthenticationKeyPairGenerator
- **generate()**: Creates a new 2048-bit RSA key pair with both binary and PEM representations
- **@Component**: Spring-managed singleton for dependency injection

## Usage Example

```java
// Inject the generator (Spring will autowire)
@Autowired
private RsaAuthenticationKeyPairGenerator keyGenerator;

// Generate a new RSA key pair
AuthenticationKeyPair keyPair = keyGenerator.generate();

// Access the generated keys
RSAPublicKey publicKey = keyPair.getPublicKey();
RSAPrivateKey privateKey = keyPair.getPrivateKey();

// Get PEM formatted strings for storage/transmission
String publicPem = keyPair.getPublicPem();
String privatePem = keyPair.getPrivatePem();

// Example usage in JWT token signing setup
JwtEncoder jwtEncoder = NimbusJwtEncoder.withJwkSet(
    new JWKSet(new RSAKey.Builder(publicKey).privateKey(privateKey).build())
).build();
```

The generator uses Java's built-in cryptography APIs and converts the raw keys to PEM format through the PemUtil helper class, making them suitable for storage and interoperability with other systems.
