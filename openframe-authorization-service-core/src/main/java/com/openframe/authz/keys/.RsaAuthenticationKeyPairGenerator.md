Generates 2048-bit RSA key pairs for authentication purposes, implementing the AuthenticationKeyPairGenerator interface as a Spring component.

## Key Components

- **RsaAuthenticationKeyPairGenerator**: Main class that implements `AuthenticationKeyPairGenerator`
- **generate()**: Creates a new RSA key pair with both raw keys and PEM-formatted strings
- Uses Java's built-in `KeyPairGenerator` with RSA algorithm
- Leverages `PemUtil` for PEM format conversion

## Usage Example

```java
@Autowired
private RsaAuthenticationKeyPairGenerator keyGenerator;

public void createAuthenticationKeys() {
    // Generate new RSA key pair
    AuthenticationKeyPair keyPair = keyGenerator.generate();
    
    // Access the raw keys
    RSAPublicKey publicKey = keyPair.getPublicKey();
    RSAPrivateKey privateKey = keyPair.getPrivateKey();
    
    // Access PEM-formatted strings
    String publicPem = keyPair.getPublicKeyPem();
    String privatePem = keyPair.getPrivateKeyPem();
    
    // Use keys for JWT signing, authentication, etc.
}
```

The generator produces 2048-bit RSA keys suitable for cryptographic operations like JWT token signing and verification. Each generated key pair includes both the native Java key objects and their PEM string representations for easy serialization and storage.
