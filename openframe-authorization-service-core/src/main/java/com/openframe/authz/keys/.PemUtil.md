A utility class for converting between RSA keys and PEM (Privacy-Enhanced Mail) format strings, providing bidirectional parsing and serialization capabilities.

## Key Components

- **`parsePublicKey(String pem)`** - Converts PEM-formatted public key string to RSAPublicKey object
- **`parsePrivateKey(String pem)`** - Converts PEM-formatted private key string to RSAPrivateKey object  
- **`toPublicPem(RSAPublicKey key)`** - Serializes RSAPublicKey to PEM format string
- **`toPrivatePem(RSAPrivateKey key)`** - Serializes RSAPrivateKey to PEM format string
- **`wrap(String s)`** - Internal helper that formats base64 strings with 64-character line breaks

## Usage Example

```java
// Parse PEM strings into key objects
String publicPem = "-----BEGIN PUBLIC KEY-----\nMIIBIjANBgkq...-----END PUBLIC KEY-----";
String privatePem = "-----BEGIN PRIVATE KEY-----\nMIIEvQIBADAN...-----END PRIVATE KEY-----";

RSAPublicKey publicKey = PemUtil.parsePublicKey(publicPem);
RSAPrivateKey privateKey = PemUtil.parsePrivateKey(privatePem);

// Convert key objects back to PEM format
String publicPemOutput = PemUtil.toPublicPem(publicKey);
String privatePemOutput = PemUtil.toPrivatePem(privateKey);
```

The utility handles standard PEM headers/footers and base64 encoding/decoding automatically, throwing `IllegalStateException` for malformed inputs.
