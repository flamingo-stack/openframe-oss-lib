Utility class for parsing and formatting RSA keys in PEM format, providing bidirectional conversion between PEM-encoded strings and Java RSA key objects.

## Key Components

- **`parsePublicKey(String pem)`** - Converts PEM-formatted public key string to RSAPublicKey object
- **`parsePrivateKey(String pem)`** - Converts PEM-formatted private key string to RSAPrivateKey object  
- **`toPublicPem(RSAPublicKey key)`** - Converts RSAPublicKey object to PEM-formatted string
- **`toPrivatePem(RSAPrivateKey key)`** - Converts RSAPrivateKey object to PEM-formatted string
- **`wrap(String s)`** - Private utility method that formats base64 strings with 64-character line wrapping

## Usage Example

```java
// Parse PEM string to RSA key objects
String publicPem = "-----BEGIN PUBLIC KEY-----\nMIIBIjANBgkqhkiG9w0B...\n-----END PUBLIC KEY-----";
RSAPublicKey publicKey = PemUtil.parsePublicKey(publicPem);

String privatePem = "-----BEGIN PRIVATE KEY-----\nMIIEvQIBADANBgkqhkiG...\n-----END PRIVATE KEY-----";
RSAPrivateKey privateKey = PemUtil.parsePrivateKey(privatePem);

// Convert RSA key objects back to PEM format
String publicPemOutput = PemUtil.toPublicPem(publicKey);
String privatePemOutput = PemUtil.toPrivatePem(privateKey);
```

The utility handles PEM header/footer removal, whitespace cleanup, Base64 decoding/encoding, and proper line wrapping for standard PEM format compliance. All parsing methods throw `IllegalStateException` for invalid key data.
