Configures reactive MongoDB repository support for the OpenFrame data layer by enabling Spring Data's reactive Mongo repositories.

## Key Components

| Element | Description |
|---|---|
| `@Configuration` | Marks this class as a Spring configuration source |
| `@EnableReactiveMongoRepositories` | Activates reactive MongoDB repository scanning |
| `basePackages` | Scans `com.openframe.data.reactive.repository` for reactive repository interfaces |

## Usage Example

```java
// Reactive repository interfaces placed under the scanned package are auto-detected
package com.openframe.data.reactive.repository;

import org.springframework.data.mongodb.repository.ReactiveMongoRepository;
import reactor.core.publisher.Flux;

public interface DeviceReactiveRepository
        extends ReactiveMongoRepository<Device, String> {

    Flux<Device> findByTenantId(String tenantId);
}
```

> No manual wiring is required. Once `MongoReactiveConfig` is loaded by the Spring context, all `ReactiveMongoRepository` interfaces under the configured base package are registered as beans and ready for injection.
