Represents the response model returned after a Fleet MDM setup operation, containing the authentication token issued upon successful setup.

## Key Components

| Element | Type | Description |
|--------|------|-------------|
| `token` | `String` | Authentication token returned from the Fleet MDM setup endpoint |

**Annotations:**
- `@Getter` / `@Setter` — Lombok-generated accessors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently ignores unrecognized JSON fields during deserialization

## Usage Example

```java
SetupResponse response = fleetMdmClient.setup(setupRequest);

String authToken = response.getToken();

// Use the token for subsequent authenticated requests
httpClient.setAuthHeader("Bearer " + authToken);
```
