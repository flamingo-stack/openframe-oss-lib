Represents a host assignment reference in the Fleet MDM model, holding the unique identifier of an assigned host device.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id` | `Long` | Unique identifier of the assigned host, mapped from JSON property `"id"` |

**Annotations:**
- `@Getter` / `@Setter` — Lombok-generated accessors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently ignores unrecognized JSON fields during deserialization

## Usage Example

```java
ObjectMapper mapper = new ObjectMapper();
String json = "{\"id\": 42}";

AssignedHost host = mapper.readValue(json, AssignedHost.class);
System.out.println(host.getId()); // 42

host.setId(99L);
```
