Represents the response model returned by the Fleet MDM login API, containing the authentication token and available teams for the authenticated user.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `token` | `String` | Authentication token returned upon successful login |
| `availableTeams` | `List<Object>` | List of teams accessible to the authenticated user, mapped from `available_teams` JSON field |

## Usage Example

```java
// Deserializing a Fleet MDM login response
ObjectMapper mapper = new ObjectMapper();
LoginResponse response = mapper.readValue(jsonString, LoginResponse.class);

String token = response.getToken();
List<Object> teams = response.getAvailableTeams();

// Use token for subsequent API calls
System.out.println("Auth Token: " + token);
System.out.println("Available Teams: " + teams);
```

## Notes

- Annotated with `@JsonIgnoreProperties(ignoreUnknown = true)` to gracefully handle additional fields returned by the API without throwing deserialization errors.
- The `available_teams` JSON field is mapped to `availableTeams` via `@JsonProperty`.
- Uses Lombok `@Getter` and `@Setter` to auto-generate accessors, keeping the model concise.
