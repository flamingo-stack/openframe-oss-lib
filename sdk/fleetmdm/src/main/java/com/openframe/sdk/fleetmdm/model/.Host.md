A Jackson-annotated POJO that represents a managed device (host) returned by the Fleet MDM API, mapping JSON snake_case fields to Java camelCase properties.

## Key Components

| Field Group | Properties |
|-------------|-----------|
| **Identity** | `id`, `hostname`, `uuid`, `osqueryHostId`, `computerName`, `displayName`, `displayText` |
| **Platform / OS** | `platform`, `platformLike`, `osVersion`, `build`, `codeName` |
| **Hardware** | `cpuType`, `cpuBrand`, `cpuPhysicalCores`, `cpuLogicalCores`, `memory`, `hardwareVendor`, `hardwareModel`, `hardwareSerial` |
| **Network** | `primaryIp`, `primaryMac`, `publicIp` |
| **Disk** | `gigsDiskSpaceAvailable`, `gigsTotalDiskSpace`, `percentDiskSpaceAvailable` |
| **Agent versions** | `osqueryVersion`, `orbitVersion`, `fleetDesktopVersion` |
| **Timestamps** | `createdAt`, `updatedAt`, `seenTime`, `lastEnrolledAt`, `lastRestartedAt`, `softwareUpdatedAt`, `detailUpdatedAt`, `labelUpdatedAt`, `policyUpdatedAt` |
| **Team / Policy** | `teamId`, `teamName`, `status`, `packStats`, `scriptsEnabled` |

**Annotations:**
- `@JsonIgnoreProperties(ignoreUnknown = true)` — tolerates extra Fleet API fields without throwing
- `@JsonProperty(...)` — handles snake_case → camelCase mapping for all multi-word fields

## Usage Example

```java
// Deserialize a Fleet MDM API response
ObjectMapper mapper = new ObjectMapper();
Host host = mapper.readValue(jsonResponse, Host.class);

// Access device details
System.out.println(host.getHostname());       // e.g. "macbook-pro-01"
System.out.println(host.getPlatform());       // e.g. "darwin"
System.out.println(host.getOsVersion());      // e.g. "macOS 14.4.1"
System.out.println(host.getPrimaryIp());      // e.g. "192.168.1.42"
System.out.println(host.getStatus());         // e.g. "online"

// Check disk space
if (host.getGigsDiskSpaceAvailable() < 10.0) {
    System.out.println("Low disk space on: " + host.getDisplayName());
}
```
