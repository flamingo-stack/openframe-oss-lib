A comprehensive data model class representing a host device in the Fleet MDM system, with detailed hardware, software, and network information along with JSON serialization support.

## Key Components

**Main Class:**
- `Host` - POJO model with Jackson annotations for JSON mapping
- `@JsonIgnoreProperties(ignoreUnknown = true)` - Ignores unknown JSON fields during deserialization
- `@JsonProperty` annotations - Maps Java field names to JSON property names

**Key Properties:**
- **Identity**: `id`, `hostname`, `uuid`, `platform`
- **Hardware**: CPU details, memory, disk space, vendor information
- **Network**: Primary/public IP addresses, MAC address
- **Software**: OS version, osquery/orbit versions, Fleet desktop version
- **Timestamps**: Creation, updates, enrollment, and last seen times
- **Team Management**: `teamId`, `teamName` for organizational grouping
- **Status**: Current host status and display information

## Usage Example

```java
// Deserializing from JSON response
ObjectMapper mapper = new ObjectMapper();
Host host = mapper.readValue(jsonResponse, Host.class);

// Accessing host information
System.out.println("Host: " + host.getHostname());
System.out.println("Platform: " + host.getPlatform());
System.out.println("CPU: " + host.getCpuBrand());
System.out.println("Memory: " + host.getMemory() + " bytes");

// Working with team assignment
if (host.getTeamId() != null) {
    System.out.println("Team: " + host.getTeamName());
}

// Checking host status and versions
System.out.println("Status: " + host.getStatus());
System.out.println("osquery version: " + host.getOsqueryVersion());
```
