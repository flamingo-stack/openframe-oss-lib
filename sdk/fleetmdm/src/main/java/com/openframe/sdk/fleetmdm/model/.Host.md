A comprehensive data model representing a host device managed by Fleet MDM, containing detailed system information, hardware specifications, and management metadata.

## Key Components

**Main Class:**
- `Host` - POJO model with Jackson annotations for JSON serialization/deserialization

**Core Properties:**
- Basic identifiers: `id`, `hostname`, `uuid`, `platform`
- Timestamps: `createdAt`, `updatedAt`, `seenTime`, `lastEnrolledAt`
- System info: `osVersion`, `build`, `uptime`, `memory`
- Hardware details: `cpuBrand`, `hardwareVendor`, `hardwareModel`, `hardwareSerial`
- Network info: `primaryIp`, `primaryMac`, `publicIp`
- Fleet management: `teamId`, `teamName`, `osqueryVersion`, `scriptsEnabled`
- Storage metrics: `gigsDiskSpaceAvailable`, `percentDiskSpaceAvailable`

**Annotations:**
- `@JsonIgnoreProperties(ignoreUnknown = true)` - Ignores unknown JSON fields
- `@JsonProperty` - Maps JSON field names to Java properties

## Usage Example

```java
// Deserialize from JSON response
ObjectMapper mapper = new ObjectMapper();
Host host = mapper.readValue(jsonResponse, Host.class);

// Access host information
System.out.println("Host: " + host.getHostname());
System.out.println("Platform: " + host.getPlatform());
System.out.println("Team: " + host.getTeamName());
System.out.println("CPU: " + host.getCpuBrand());
System.out.println("Available Storage: " + host.getGigsDiskSpaceAvailable() + " GB");

// Check host status
if (host.getScriptsEnabled()) {
    System.out.println("Scripts are enabled on this host");
}
```
