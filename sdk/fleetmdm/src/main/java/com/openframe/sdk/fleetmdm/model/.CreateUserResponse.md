Represents the response model returned after successfully creating a user in the Fleet MDM system, containing the authentication token for the newly created user.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `token` | `String` | Authentication token issued upon user creation |

## Usage Example

```java
CreateUserResponse response = fleetMdmClient.createUser(createUserRequest);
String authToken = response.getToken();

// Use the token for subsequent authenticated requests
httpHeaders.setBearerAuth(authToken);
```

## Notes

- Annotated with `@JsonIgnoreProperties(ignoreUnknown = true)` to safely handle additional fields returned by the Fleet MDM API without breaking deserialization
- Lombok `@Getter` / `@Setter` annotations auto-generate accessor methods, keeping the model concise
