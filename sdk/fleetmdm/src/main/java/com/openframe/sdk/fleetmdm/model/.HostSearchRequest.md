A data transfer object that encapsulates search parameters for querying hosts in the Fleet MDM system, providing pagination, sorting, and filtering capabilities.

## Key Components

- **Fields**: `query` (search term), `page` (pagination offset), `perPage` (results per page), `orderKey` (sort field), `orderDirection` (sort direction)
- **Constructors**: Default constructor with sensible defaults (page=0, perPage=100), query-only constructor, and full parameter constructor
- **Getters/Setters**: Standard JavaBean accessor methods for all fields
- **Default Values**: Automatic initialization of pagination parameters to prevent null pointer exceptions

## Usage Example

```java
// Basic search with default pagination
HostSearchRequest basicSearch = new HostSearchRequest("laptop-001");

// Search with custom pagination
HostSearchRequest pagedSearch = new HostSearchRequest("windows", 1, 50);

// Search with sorting
HostSearchRequest sortedSearch = new HostSearchRequest();
sortedSearch.setQuery("active");
sortedSearch.setOrderKey("hostname");
sortedSearch.setOrderDirection("asc");
sortedSearch.setPage(0);
sortedSearch.setPerPage(25);

// Use with Fleet MDM API
FleetMDMClient client = new FleetMDMClient();
List<Host> results = client.searchHosts(sortedSearch);
```

This class follows standard JavaBean conventions and provides flexible construction patterns for different search scenarios, from simple queries to complex paginated and sorted requests.
