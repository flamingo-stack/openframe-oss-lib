A data model class that encapsulates search parameters for querying hosts in the Fleet MDM system. It provides pagination, sorting, and filtering capabilities with sensible default values.

## Key Components

- **Constructors**: Multiple constructors for flexible initialization with default pagination values (page=0, perPage=100)
- **Search Parameters**:
  - `query` - Search filter string
  - `page` - Page number for pagination
  - `perPage` - Number of results per page
  - `orderKey` - Field name for sorting
  - `orderDirection` - Sort direction (ASC/DESC)
- **Getters/Setters**: Standard JavaBean accessors for all properties

## Usage Example

```java
// Basic search with defaults (page=0, perPage=100)
HostSearchRequest basicSearch = new HostSearchRequest("hostname:server*");

// Search with custom pagination
HostSearchRequest pagedSearch = new HostSearchRequest("status:active", 1, 50);

// Full configuration with sorting
HostSearchRequest fullSearch = new HostSearchRequest();
fullSearch.setQuery("os:linux");
fullSearch.setPage(2);
fullSearch.setPerPage(25);
fullSearch.setOrderKey("hostname");
fullSearch.setOrderDirection("ASC");
```
