A model class representing the result of Fleet MDM query execution via osquery, encapsulating query response data, metadata, and execution status.

## Key Components

- **Properties**: `hostId`, `rows`, `error`, `status`, `query`, `executedAt`
- **Helper Methods**: 
  - `isSuccess()` - checks if query executed without errors
  - `getRowCount()` - returns number of result rows
- **JSON Mapping**: Uses Jackson annotations for serialization/deserialization

## Usage Example

```java
// Process query results
QueryResult result = fleetApiClient.executeQuery(queryId);

if (result.isSuccess()) {
    System.out.println("Query executed on host: " + result.getHostId());
    System.out.println("Found " + result.getRowCount() + " rows");
    
    // Process each row of data
    for (Map<String, Object> row : result.getRows()) {
        String processName = (String) row.get("name");
        Integer pid = (Integer) row.get("pid");
        System.out.println("Process: " + processName + " (PID: " + pid + ")");
    }
} else {
    System.err.println("Query failed: " + result.getError());
}

// Check execution metadata
System.out.println("Executed at: " + result.getExecutedAt());
System.out.println("Status: " + result.getStatus());
```
