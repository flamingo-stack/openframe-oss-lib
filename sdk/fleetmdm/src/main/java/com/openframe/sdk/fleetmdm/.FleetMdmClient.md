A Java client library for interacting with the Fleet MDM REST API, providing methods to manage hosts, execute queries, and retrieve configuration data.

## Key Components

**Main Class:**
- `FleetMdmClient` - Primary client for Fleet MDM API operations

**Core Methods:**
- `getHostById(long id)` - Retrieves a specific host by ID
- `searchHosts(HostSearchRequest)` - Searches for hosts with filtering and pagination
- `searchHosts(String query)` - Simple host search by query string
- `runQuery(long hostId, String query)` - Executes osquery SQL statements on hosts
- `getEnrollSecret()` - Retrieves the Fleet enrollment secret
- `getQueryById(long id)` - Fetches a specific query by ID

**Configuration:**
- Constructor accepts base URL and API token for authentication
- Built-in HTTP client with configurable timeouts
- Thread-safe JSON processing with Jackson ObjectMapper

## Usage Example

```java
// Initialize client
FleetMdmClient client = new FleetMdmClient("https://fleet.example.com", "your-api-token");

// Get a specific host
Host host = client.getHostById(123);

// Search for hosts
List<Host> hosts = client.searchHosts("hostname:web-server");

// Search with pagination
HostSearchRequest request = new HostSearchRequest("status:online", 0, 50);
List<Host> pagedHosts = client.searchHosts(request);

// Execute osquery on a host
QueryResult result = client.runQuery(123, "SELECT * FROM processes LIMIT 10;");

// Get enrollment secret
String secret = client.getEnrollSecret();
```
