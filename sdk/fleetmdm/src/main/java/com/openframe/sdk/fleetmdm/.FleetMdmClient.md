A Java SDK client for interacting with Fleet MDM (Mobile Device Management) REST API endpoints, providing methods for host management, query execution, and authentication.

## Key Components

### Main Class
- **FleetMdmClient** - Primary client class with configurable HTTP client and authentication

### Core Methods
- `getHostById(long id)` - Retrieve a specific host by ID
- `searchHosts(HostSearchRequest)` - Search hosts with flexible query parameters  
- `searchHosts(String query)` - Simple host search with default pagination
- `runQuery(long hostId, String query)` - Execute osquery SQL statements on hosts
- `getEnrollSecret()` - Retrieve Fleet MDM enrollment secret
- `getQueryById(long id)` - Fetch saved queries by ID

### Exception Handling
- **FleetMdmApiException** - API-specific errors with status codes
- **FleetMdmException** - General SDK exceptions

## Usage Example

```java
// Initialize client with Fleet MDM server
FleetMdmClient client = new FleetMdmClient(
    "https://fleet.example.com", 
    "your-api-token"
);

// Search for hosts
List<Host> hosts = client.searchHosts("hostname:web-server");

// Get specific host
Host host = client.getHostById(12345L);

// Execute osquery on host
QueryResult result = client.runQuery(12345L, 
    "SELECT name, version FROM programs WHERE name LIKE 'Chrome%'");

// Get enrollment secret
String secret = client.getEnrollSecret();

// Advanced search with pagination
HostSearchRequest searchRequest = new HostSearchRequest("ubuntu", 0, 50);
List<Host> pagedHosts = client.searchHosts(searchRequest);
```

The client handles authentication, error responses, JSON serialization, and provides comprehensive Fleet MDM functionality for device management workflows.
