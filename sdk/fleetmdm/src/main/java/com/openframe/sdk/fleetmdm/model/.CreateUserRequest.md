Represents the request payload for creating a new user in the Fleet MDM system, mapping Java fields to their corresponding JSON properties.

## Key Components

| Field | Type | JSON Property | Description |
|-------|------|---------------|-------------|
| `name` | `String` | `name` | Display name of the user |
| `email` | `String` | `email` | User's email address |
| `password` | `String` | `password` | User's password |
| `globalRole` | `String` | `global_role` | Assigned global role |
| `apiOnly` | `boolean` | `api_only` | Restricts user to API access only |

## Usage Example

```java
CreateUserRequest request = new CreateUserRequest();
request.setName("Jane Doe");
request.setEmail("jane.doe@example.com");
request.setPassword("securePassword123");
request.setGlobalRole("admin");
request.setApiOnly(false);
```

> Lombok `@Getter` and `@Setter` annotations auto-generate accessors. `@JsonProperty` ensures correct snake_case serialization when submitting to the Fleet MDM API.
