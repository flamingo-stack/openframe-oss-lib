A model class representing Fleet MDM queries, including both saved and scheduled queries with comprehensive metadata, authoring information, execution settings, and performance statistics.

## Key Components

- **Query** - Main model class with Jackson JSON mapping annotations
- **QueryStats** - Nested class containing performance metrics (execution times, total runs)
- **isScheduled()** - Helper method to determine if a query runs on a schedule
- **Standard getters/setters** - For all query properties including metadata, configuration, and statistics

## Usage Example

```java
// Create and configure a new query
Query query = new Query();
query.setName("System Info Query");
query.setQuery("SELECT * FROM system_info;");
query.setInterval(3600); // 1 hour
query.setAutomationsEnabled(true);
query.setPlatform("linux");
query.setObserverCanRun(false);

// Check if query is scheduled
if (query.isScheduled()) {
    System.out.println("Query runs every " + query.getInterval() + " seconds");
}

// Access performance statistics
Query.QueryStats stats = query.getStats();
if (stats != null) {
    System.out.println("Total executions: " + stats.getTotalExecutions());
    System.out.println("P95 user time: " + stats.getUserTimeP95());
}

// Get author information
System.out.println("Created by: " + query.getAuthorName() + 
                   " (" + query.getAuthorEmail() + ")");
```
