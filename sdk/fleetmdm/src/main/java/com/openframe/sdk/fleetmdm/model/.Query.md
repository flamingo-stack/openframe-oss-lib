A Jackson-annotated POJO representing a saved or scheduled osquery query from the Fleet MDM API, including scheduling configuration, authorship metadata, and execution statistics.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id`, `name`, `description`, `query` | Core fields | Query identity and SQL content |
| `interval`, `automationsEnabled` | Scheduling fields | Control scheduled execution |
| `platform`, `minOsqueryVersion` | Compatibility fields | Target platform constraints |
| `stats` | `QueryStats` | Nested performance statistics (p50/p95 CPU times, total executions) |
| `hostsIncludeAny` | `List<AssignedHost>` | Host targeting filter |
| `isScheduled()` | Helper method | Returns `true` if interval > 0 and automations are enabled |

### `QueryStats` (nested class)

Captures runtime performance metrics for the query:

| Field | Description |
|-------|-------------|
| `userTimeP50` / `userTimeP95` | User CPU time percentiles |
| `systemTimeP50` / `systemTimeP95` | System CPU time percentiles |
| `totalExecutions` | Cumulative execution count |

## Usage Example

```java
// Deserialize a Fleet MDM query response
ObjectMapper mapper = new ObjectMapper();
Query query = mapper.readValue(jsonResponse, Query.class);

// Check if the query runs on a schedule
if (query.isScheduled()) {
    System.out.println("Runs every " + query.getInterval() + " seconds on " + query.getPlatform());
}

// Access performance stats
Query.QueryStats stats = query.getStats();
if (stats != null) {
    System.out.println("P95 user time: " + stats.getUserTimeP95() + "ms");
    System.out.println("Total executions: " + stats.getTotalExecutions());
}

// Filter to team-scoped queries
if (query.getTeamId() != null) {
    System.out.println("Team query: " + query.getTeamName());
}
```
