A Java model class representing a Fleet MDM query object with comprehensive metadata including authorship, team assignment, scheduling configuration, and performance statistics.

## Key Components

- **Query**: Main model class with Jackson annotations for JSON serialization/deserialization
- **QueryStats**: Nested static class containing performance metrics (execution times, total runs)
- **Core Properties**: ID, name, description, SQL query text, timestamps
- **Author Information**: Author ID, name, and email address
- **Team Management**: Team ID and name for organizational grouping
- **Scheduling Configuration**: Interval, automations enabled, platform targeting
- **Execution Settings**: Observer permissions, logging configuration, data retention
- **Helper Methods**: `isScheduled()` utility method for checking active scheduling

## Usage Example

```java
// Create a new query
Query query = new Query();
query.setName("Security Audit");
query.setDescription("Monitor login attempts");
query.setQuery("SELECT * FROM logged_in_users;");
query.setInterval(3600); // Run every hour
query.setAutomationsEnabled(true);
query.setPlatform("darwin,linux,windows");

// Check if query is scheduled
if (query.isScheduled()) {
    System.out.println("Query runs every " + query.getInterval() + " seconds");
}

// Access performance statistics
Query.QueryStats stats = query.getStats();
if (stats != null) {
    System.out.println("Total executions: " + stats.getTotalExecutions());
    System.out.println("Average user time: " + stats.getUserTimeP50() + "ms");
}
```
