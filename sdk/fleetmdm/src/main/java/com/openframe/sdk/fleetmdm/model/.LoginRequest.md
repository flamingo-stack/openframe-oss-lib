A simple data transfer object (DTO) representing the payload required to authenticate with the Fleet MDM API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `email` | `String` | User's email address for authentication |
| `password` | `String` | User's password for authentication |

Lombok annotations `@Getter` and `@Setter` auto-generate all accessor methods at compile time, keeping the class minimal.

## Usage Example

```java
LoginRequest loginRequest = new LoginRequest();
loginRequest.setEmail("admin@example.com");
loginRequest.setPassword("securePassword123");

// Pass to your Fleet MDM authentication client
fleetMdmClient.login(loginRequest);
```
