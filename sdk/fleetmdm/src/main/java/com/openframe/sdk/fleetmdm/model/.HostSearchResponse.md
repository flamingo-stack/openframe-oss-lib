A JSON response model that wraps host search results from Fleet MDM API calls, including pagination and search metadata.

## Key Components

- **hosts** - List of `Host` objects returned from the search
- **page** - Current page number for pagination
- **perPage** - Number of results per page
- **orderKey** - Field used for sorting results
- **orderDirection** - Sort direction (ascending/descending)
- **query** - Original search query string

## Usage Example

```java
// Deserialize API response
ObjectMapper mapper = new ObjectMapper();
HostSearchResponse response = mapper.readValue(jsonString, HostSearchResponse.class);

// Access search results
List<Host> hosts = response.getHosts();
System.out.println("Found " + hosts.size() + " hosts");

// Check pagination info
System.out.println("Page: " + response.getPage());
System.out.println("Per page: " + response.getPerPage());

// Access search metadata
System.out.println("Query: " + response.getQuery());
System.out.println("Ordered by: " + response.getOrderKey() + 
                   " " + response.getOrderDirection());
```

The class uses Jackson annotations for JSON deserialization and ignores unknown properties for API compatibility.
