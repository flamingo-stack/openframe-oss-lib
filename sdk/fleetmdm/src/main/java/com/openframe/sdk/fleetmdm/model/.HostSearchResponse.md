Response wrapper class for paginated host search results from the Fleet MDM API. This model encapsulates the search results along with pagination metadata and query parameters used for the search operation.

## Key Components

- **hosts** - List of Host objects representing the search results
- **page** - Current page number in the paginated results
- **perPage** - Number of results per page
- **orderKey** - Field used for sorting the results
- **orderDirection** - Sort direction (ascending/descending)
- **query** - Original search query string used

## Usage Example

```java
// Deserializing API response
ObjectMapper mapper = new ObjectMapper();
HostSearchResponse response = mapper.readValue(jsonResponse, HostSearchResponse.class);

// Accessing search results
List<Host> hosts = response.getHosts();
System.out.println("Found " + hosts.size() + " hosts on page " + response.getPage());

// Checking pagination info
if (response.getPerPage() != null) {
    System.out.println("Results per page: " + response.getPerPage());
}

// Getting search metadata
String originalQuery = response.getQuery();
String sortField = response.getOrderKey();
String sortDirection = response.getOrderDirection();
```

The class uses Jackson annotations for JSON deserialization and includes `@JsonIgnoreProperties(ignoreUnknown = true)` to handle API response evolution gracefully.
