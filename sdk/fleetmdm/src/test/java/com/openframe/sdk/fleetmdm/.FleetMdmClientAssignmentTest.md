Unit test suite for `FleetMdmClient` covering live-query execution, host assignment mutations, and delete operations using mocked HTTP interactions.

## Key Components

| Component | Description |
|-----------|-------------|
| `FleetMdmClientAssignmentTest` | JUnit 5 test class validating async Fleet MDM client methods |
| `mockHttpClient` | Mockito-mocked `HttpClient` injected into `FleetMdmClient` for request interception |
| `mockAsyncResponse()` | Helper that stubs `sendAsync` to return a completed `CompletableFuture` with a configurable status code and body |

## Tests Covered

| Test Method | Validates |
|-------------|-----------|
| `runLiveQueryAsync_targetsExpectedPathAndBody` | POST to `/api/v1/fleet/queries/run`, response deserialization to `LiveQueryCampaign` |
| `runLiveQueryAsync_rejectsRequestWithoutQueryOrId` | Validation fails fast when query/id is missing |
| `addQueryHostsAsync_returnsAddedCount` | POST to `/api/v1/fleet/queries/{id}/hosts`, parses `added` count |
| `removePolicyHostsAsync_usesDeleteWithBody` | DELETE with body to `/api/v1/fleet/policies/{id}/hosts` |
| `deleteScheduledQueryAsync_usesDeleteOnIdPath` | DELETE to `/api/v1/fleet/queries/id/{id}` |
| `deletePolicyAsync_usesBatchDeletePost` | POST to `/api/latest/fleet/policies/delete` for batch delete |
| `hostListMustNotBeEmpty` | Empty host list rejected exceptionally |

## Usage Example

```java
// Typical test pattern: mock HTTP, invoke client method, assert HTTP shape + return value
@Test
void runLiveQueryAsync_targetsExpectedPathAndBody() throws Exception {
    mockAsyncResponse(200, "{\"campaign\":{\"id\":42,\"query\":\"SELECT 1\",\"status\":\"running\"}}");

    RunLiveQueryRequest request = new RunLiveQueryRequest();
    request.setQuery("SELECT 1");
    request.setSelected(new RunLiveQueryRequest.Selected(List.of(7L), null, null));

    LiveQueryCampaign campaign = client.runLiveQueryAsync(request).get();

    assertEquals(42L, campaign.getId());

    ArgumentCaptor<HttpRequest> captor = ArgumentCaptor.forClass(HttpRequest.class);
    verify(mockHttpClient).sendAsync(captor.capture(), any());
    assertTrue(captor.getValue().uri().toString().endsWith("/api/v1/fleet/queries/run"));
    assertEquals("POST", captor.getValue().method());
}
```

> All tests use `ArgumentCaptor` to inspect the outbound `HttpRequest`, verifying both the target URI path and HTTP method without making real network calls.
