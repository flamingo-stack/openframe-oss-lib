Represents a single automated task entry from the Tactical RMM API, mapping JSON response fields to a Java object with tolerant deserialization.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id` | `Integer` | Unique identifier of the automated task |
| `name` | `String` | Display name of the automated task |
| `getId()` / `setId()` | Method | Accessor for task ID |
| `getName()` / `setName()` | Method | Accessor for task name |
| `toString()` | Method | Human-readable representation of the task item |

**Annotations:**
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Safely ignores extra fields returned by the API without throwing errors
- `@JsonProperty` — Explicitly maps JSON keys `id` and `name` to their respective fields

## Usage Example

```java
// Deserialized automatically by Jackson from a Tactical RMM API response
ObjectMapper mapper = new ObjectMapper();
AutomatedTaskItem task = mapper.readValue(jsonString, AutomatedTaskItem.class);

System.out.println(task.getId());    // e.g. 42
System.out.println(task.getName());  // e.g. "Daily Disk Cleanup"
System.out.println(task);            // AutomatedTaskItem{id=42, name='Daily Disk Cleanup'}

// Manually constructing an instance
AutomatedTaskItem task = new AutomatedTaskItem();
task.setId(42);
task.setName("Daily Disk Cleanup");
```

> This model is part of the OpenFrame SDK integration with [Tactical RMM](https://docs.tacticalrmm.com), an open-source RMM platform used within the Flamingo MSP stack.
