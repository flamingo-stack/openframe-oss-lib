Represents a single action within a Tactical RMM automated task, mapping JSON task action data to a Java object with script execution configuration.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `type` | `String` | Action type identifier |
| `name` | `String` | Display name of the action |
| `script` | `Integer` | Script ID to execute |
| `timeout` | `Integer` | Execution timeout in seconds |
| `runAsUser` | `Boolean` | Whether to run the script as the logged-in user (`run_as_user`) |
| `scriptArgs` | `List<String>` | Arguments passed to the script (`script_args`) |
| `envVars` | `List<String>` | Environment variables for script execution (`env_vars`) |

## Usage Example

```java
// Deserializing a TaskAction from JSON (e.g., via Tactical RMM API response)
ObjectMapper mapper = new ObjectMapper();
String json = """
    {
        "type": "script",
        "name": "Disk Cleanup",
        "script": 42,
        "timeout": 120,
        "run_as_user": false,
        "script_args": ["--verbose", "--dry-run"],
        "env_vars": ["LOG_LEVEL=info"]
    }
    """;

TaskAction action = mapper.readValue(json, TaskAction.class);
System.out.println(action.getName());    // "Disk Cleanup"
System.out.println(action.getScript()); // 42
System.out.println(action.getScriptArgs()); // [--verbose, --dry-run]
```

> Unknown JSON fields are safely ignored via `@JsonIgnoreProperties(ignoreUnknown = true)`, ensuring forward compatibility with newer Tactical RMM API versions.
