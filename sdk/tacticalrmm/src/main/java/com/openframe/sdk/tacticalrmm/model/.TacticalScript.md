Represents a script object from the TacticalRMM API, mapping JSON response fields to a Java model using Jackson and Lombok.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `Integer` | Unique script identifier |
| `name` | `String` | Display name of the script |
| `description` | `String` | Human-readable script description |
| `shell` | `String` | Shell interpreter (e.g., `powershell`, `cmd`, `bash`) |
| `args` | `List<String>` | Default argument list |
| `category` | `String` | Script category/grouping |
| `scriptBody` | `String` | Actual script content (`script_body`) |
| `defaultTimeout` | `Integer` | Execution timeout in seconds (`default_timeout`) |
| `supportedPlatforms` | `List<String>` | Target OS platforms (`supported_platforms`) |
| `runAsUser` | `Boolean` | Whether to run as logged-in user (`run_as_user`) |
| `envVars` | `List<String>` | Environment variables to inject (`env_vars`) |

**Annotations:**
- `@Data` / `@NoArgsConstructor` — Lombok-generated getters, setters, and default constructor
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra fields from API responses
- `@JsonProperty` — Maps snake_case JSON keys to camelCase Java fields

## Usage Example

```java
// Deserialize from TacticalRMM API response
ObjectMapper mapper = new ObjectMapper();
TacticalScript script = mapper.readValue(apiResponse, TacticalScript.class);

// Access script details
System.out.println(script.getName());           // "Restart Spooler"
System.out.println(script.getShell());          // "powershell"
System.out.println(script.getScriptBody());     // "Restart-Service Spooler"
System.out.println(script.getDefaultTimeout()); // 30
System.out.println(script.getSupportedPlatforms()); // ["windows"]
```
