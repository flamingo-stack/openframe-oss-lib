A data model class representing agent information returned by Tactical RMM API responses. This POJO uses Jackson annotations for JSON serialization/deserialization and provides standard getter/setter methods.

## Key Components

- **@JsonIgnoreProperties** - Ignores unknown JSON properties during deserialization
- **@JsonProperty** annotations - Maps JSON field names to Java properties
- **agentId** - Unique identifier for the agent
- **platform** - Agent's platform information
- **operatingSystem** - Operating system details
- **hostname** - Agent's hostname
- Standard getter/setter methods and toString() implementation

## Usage Example

```java
// Deserializing from JSON response
ObjectMapper mapper = new ObjectMapper();
String jsonResponse = "{"
    + "\"agent_id\": \"abc123\","
    + "\"plat\": \"windows\","
    + "\"operating_system\": \"Windows 10\","
    + "\"hostname\": \"workstation-01\""
    + "}";

AgentInfo agent = mapper.readValue(jsonResponse, AgentInfo.class);
System.out.println(agent.getHostname()); // Output: workstation-01

// Creating and using programmatically
AgentInfo newAgent = new AgentInfo();
newAgent.setAgentId("xyz789");
newAgent.setPlatform("linux");
newAgent.setOperatingSystem("Ubuntu 20.04");
newAgent.setHostname("server-01");

System.out.println(newAgent.toString());
```
