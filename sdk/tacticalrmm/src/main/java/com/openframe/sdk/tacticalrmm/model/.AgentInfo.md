Data model class that represents agent information returned from Tactical RMM API responses. This POJO uses Jackson annotations for JSON serialization/deserialization and ignores unknown properties for API compatibility.

## Key Components

- **agentId** - Unique identifier for the Tactical RMM agent
- **platform** - Platform type (mapped from "plat" JSON field)
- **operatingSystem** - OS information (mapped from "operating_system" JSON field)
- **hostname** - Network hostname of the agent
- **Jackson Annotations** - `@JsonProperty` for field mapping and `@JsonIgnoreProperties` for unknown field handling
- **Standard Getters/Setters** - Full property access methods
- **toString()** - String representation for debugging and logging

## Usage Example

```java
// Deserialize from JSON response
ObjectMapper mapper = new ObjectMapper();
String jsonResponse = "{"
    + "\"agent_id\": \"12345\","
    + "\"plat\": \"windows\","
    + "\"operating_system\": \"Windows 10\","
    + "\"hostname\": \"DESKTOP-ABC123\""
    + "}";

AgentInfo agent = mapper.readValue(jsonResponse, AgentInfo.class);

// Access agent information
System.out.println("Agent ID: " + agent.getAgentId());
System.out.println("Platform: " + agent.getPlatform());
System.out.println("OS: " + agent.getOperatingSystem());
System.out.println("Hostname: " + agent.getHostname());

// Create new instance
AgentInfo newAgent = new AgentInfo();
newAgent.setAgentId("67890");
newAgent.setPlatform("linux");
newAgent.setOperatingSystem("Ubuntu 20.04");
newAgent.setHostname("server-01");
```
