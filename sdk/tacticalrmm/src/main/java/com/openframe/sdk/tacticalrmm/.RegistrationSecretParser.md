A utility class that extracts authentication secrets from Tactical RMM agent installation command strings using regex pattern matching.

## Key Components

- **`parse(String command)`** - Static method that parses a Tactical RMM install command and extracts the value following the `--auth` flag
- **Pattern matching logic** - Uses two regex patterns to handle both quoted and unquoted authentication tokens
- **Quote handling** - Automatically strips surrounding quotes from extracted values

## Usage Example

```java
// Parse a typical Tactical RMM installation command
String command = "tacticalagent-v2.9.0-windows-amd64.exe /VERYSILENT " +
    "&& \"C:\\Program Files\\TacticalAgent\\tacticalrmm.exe\" -m install " +
    "--api http://tactical-nginx.example.com:8000 --client-id 1 --site-id 1 " +
    "--agent-type server --auth 15ec17179f1c4db7082386ed631d4bbb2c687b9ad9673d777220d781dd20e6d9";

String secret = RegistrationSecretParser.parse(command);
// Returns: "15ec17179f1c4db7082386ed631d4bbb2c687b9ad9673d777220d781dd20e6d9"

// Handle quoted values
String quotedCommand = "tacticalrmm.exe --auth \"abc123def456\"";
String quotedSecret = RegistrationSecretParser.parse(quotedCommand);
// Returns: "abc123def456"

// Handle invalid input
String invalidSecret = RegistrationSecretParser.parse(null);
// Returns: ""
```

The parser uses case-insensitive matching and supports both quoted and unquoted authentication tokens, making it robust for various command line formats.
