A utility class that extracts authentication secrets from Tactical RMM agent installation command strings using regex pattern matching.

## Key Components

- **`parse(String command)`** - Static method that extracts the registration secret from a Tactical RMM install command by locating the `--auth` parameter and returning its value

## Usage Example

```java
// Extract secret from a Tactical RMM installation command
String installCommand = "tacticalagent-v2.9.0-windows-amd64.exe /VERYSILENT && " +
    "tacticalrmm.exe -m install --api http://tactical.example.com:8000 " +
    "--client-id 1 --site-id 1 --agent-type server " +
    "--auth 15ec17179f1c4db7082386ed631d4bbb2c687b9ad9673d777220d781dd20e6d9";

String secret = RegistrationSecretParser.parse(installCommand);
// Returns: "15ec17179f1c4db7082386ed631d4bbb2c687b9ad9673d777220d781dd20e6d9"

// Handles quoted values and edge cases
String quotedCommand = "tacticalrmm.exe --auth \"secret123\"";
String quotedSecret = RegistrationSecretParser.parse(quotedCommand);
// Returns: "secret123"

// Returns empty string for invalid input
String invalid = RegistrationSecretParser.parse("no auth flag here");
// Returns: ""
```

The parser handles case-insensitive matching, quoted values, and gracefully returns empty strings for malformed or missing commands.
