A simple data model representing a script item in Tactical RMM API responses, containing basic script identification information.

## Key Components

- **ScriptListItem** - Main model class with Jackson JSON annotations
- **id** - Integer field for unique script identifier
- **name** - String field for script display name
- **@JsonIgnoreProperties** - Annotation to handle unknown JSON properties gracefully
- **Standard getters/setters** - Bean-style property access methods
- **toString()** - Formatted string representation for debugging

## Usage Example

```java
// Deserializing from JSON response
ObjectMapper mapper = new ObjectMapper();
String json = "{\"id\": 123, \"name\": \"System Health Check\", \"extra_field\": \"ignored\"}";
ScriptListItem script = mapper.readValue(json, ScriptListItem.class);

// Accessing properties
Integer scriptId = script.getId();        // 123
String scriptName = script.getName();     // "System Health Check"

// Creating programmatically
ScriptListItem newScript = new ScriptListItem();
newScript.setId(456);
newScript.setName("Backup Script");
System.out.println(newScript); // ScriptListItem{id=456, name='Backup Script'}
```

The class uses Jackson annotations to map JSON fields and ignores unknown properties, making it resilient to API changes while providing clean access to script metadata.
