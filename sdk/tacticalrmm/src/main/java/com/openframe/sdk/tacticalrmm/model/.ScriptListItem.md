A data model class representing a script item in Tactical RMM API responses, containing basic script identification information.

## Key Components

- **ScriptListItem**: Main model class with Jackson annotations for JSON serialization
- **id**: Integer field for unique script identifier
- **name**: String field for script display name
- **@JsonIgnoreProperties**: Ignores unknown JSON properties during deserialization
- **toString()**: Provides formatted string representation of the object

## Usage Example

```java
import com.openframe.sdk.tacticalrmm.model.ScriptListItem;
import com.fasterxml.jackson.databind.ObjectMapper;

// Deserialize from JSON response
ObjectMapper mapper = new ObjectMapper();
String jsonResponse = "{\"id\": 123, \"name\": \"System Cleanup\"}";
ScriptListItem script = mapper.readValue(jsonResponse, ScriptListItem.class);

// Access script properties
Integer scriptId = script.getId();
String scriptName = script.getName();
System.out.println(script.toString()); // Output: ScriptListItem{id=123, name='System Cleanup'}

// Create new instance
ScriptListItem newScript = new ScriptListItem();
newScript.setId(456);
newScript.setName("Backup Script");
```
