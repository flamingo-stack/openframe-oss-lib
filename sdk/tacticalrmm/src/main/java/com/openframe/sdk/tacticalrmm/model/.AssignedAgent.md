A simple model class representing an agent assigned to a resource in the Tactical RMM integration, containing the agent's unique identifier.

## Key Components

| Component | Description |
|-----------|-------------|
| `agentId` | The unique identifier of the assigned agent, mapped from the `agent_id` JSON field |
| `@JsonIgnoreProperties` | Silently ignores any unknown JSON fields during deserialization |
| `@Data` | Lombok annotation generating getters, setters, `equals`, `hashCode`, and `toString` |
| `@NoArgsConstructor` | Lombok annotation generating a no-argument constructor |

## Usage Example

```java
// Deserializing from JSON
String json = "{\"agent_id\": \"abc-123-xyz\"}";
ObjectMapper mapper = new ObjectMapper();
AssignedAgent agent = mapper.readValue(json, AssignedAgent.class);

// Accessing the agent ID
String agentId = agent.getAgentId(); // "abc-123-xyz"

// Manual construction
AssignedAgent assignedAgent = new AssignedAgent();
assignedAgent.setAgentId("abc-123-xyz");
```
