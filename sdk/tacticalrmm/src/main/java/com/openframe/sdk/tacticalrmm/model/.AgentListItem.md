A lightweight data model representing a simplified agent entry from Tactical RMM API responses when the detail parameter is set to false. This class provides essential agent information for list views and caching operations.

## Key Components

- **Core Fields**: `id`, `agentId`, `hostname`, `site`, `client` - Essential agent identification properties
- **JSON Mapping**: Jackson annotations for seamless API response deserialization
- **Backward Compatibility**: `getPk()` method provides alias for legacy cache service integration
- **Robust Parsing**: `@JsonIgnoreProperties` annotation handles API response variations gracefully

## Usage Example

```java
// Deserialize from Tactical RMM API response
ObjectMapper mapper = new ObjectMapper();
AgentListItem agent = mapper.readValue(jsonResponse, AgentListItem.class);

// Access agent properties
System.out.println("Agent: " + agent.getHostname());
System.out.println("Site: " + agent.getSite());
System.out.println("Client: " + agent.getClient());

// Use with legacy cache service
Integer cacheKey = agent.getPk(); // Returns same as getId()

// Convert to string for logging
System.out.println(agent.toString());
```

This model is typically used when retrieving agent lists from the Tactical RMM API with minimal detail requirements, optimizing network payload and processing performance for overview displays.
