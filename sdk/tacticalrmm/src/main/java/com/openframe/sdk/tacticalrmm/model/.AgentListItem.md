A data model class representing a simplified agent entry from Tactical RMM API responses when `detail=false` is used. This class provides essential agent identification and organizational information.

## Key Components

- **Properties**: `id`, `agentId`, `hostname`, `site`, `client` - core agent identification fields
- **getPk()** - Alias method for `getId()` providing backward compatibility with cache service
- **Standard getters/setters** - Full property access methods
- **toString()** - Formatted string representation for debugging
- **@JsonIgnoreProperties** - Handles unknown JSON fields gracefully
- **@JsonProperty** - Maps JSON field names to Java properties

## Usage Example

```java
// Deserializing from JSON response
ObjectMapper mapper = new ObjectMapper();
AgentListItem agent = mapper.readValue(jsonResponse, AgentListItem.class);

// Accessing agent information
System.out.println("Agent: " + agent.getHostname());
System.out.println("Site: " + agent.getSite());
System.out.println("Client: " + agent.getClient());

// Using compatibility method
Integer pk = agent.getPk(); // Same as getId()

// Working with collections
List<AgentListItem> agents = mapper.readValue(
    jsonArray, 
    new TypeReference<List<AgentListItem>>() {}
);
```
