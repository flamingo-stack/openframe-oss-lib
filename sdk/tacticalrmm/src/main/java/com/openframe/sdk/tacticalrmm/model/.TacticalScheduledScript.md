Represents a scheduled script/task in the TacticalRMM system, mapping JSON API responses to a Java model with scheduling configuration, platform support, and agent assignment details.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` / `name` / `enabled` | `Integer` / `String` / `boolean` | Core task identity and state |
| `taskType` | `String` | Scheduling type (e.g., daily, weekly, one-time) |
| `runTimeDate` | `String` | Scheduled execution timestamp |
| `dailyInterval` / `weeklyInterval` | `Integer` | Recurrence interval for daily/weekly tasks |
| `runTimeBitWeekdays` | `Integer` | Bitmask representing scheduled days of the week |
| `taskSupportedPlatforms` | `List<String>` | Platforms the task can run on (e.g., `windows`, `linux`) |
| `actionsCount` / `agentsCount` | `Integer` | Summary counts for actions and assigned agents |
| `managedTaskId` | `Integer` | Reference to a managed task entity |
| `actions` | `List<TaskAction>` | List of actions executed by this task |
| `agents` | `List<AssignedAgent>` | Agents this scheduled script is assigned to |

**Annotations:** `@Data` (Lombok getters/setters), `@NoArgsConstructor`, `@JsonIgnoreProperties(ignoreUnknown = true)` for forward-compatible deserialization.

## Usage Example

```java
// Deserialize a TacticalRMM API response
ObjectMapper mapper = new ObjectMapper();
TacticalScheduledScript script = mapper.readValue(jsonResponse, TacticalScheduledScript.class);

// Check if enabled and inspect scheduling
if (script.isEnabled()) {
    System.out.println("Task: " + script.getName());
    System.out.println("Type: " + script.getTaskType());
    System.out.println("Platforms: " + script.getTaskSupportedPlatforms());
    System.out.println("Assigned agents: " + script.getAgentsCount());

    for (TaskAction action : script.getActions()) {
        // process each action
    }
}
```
