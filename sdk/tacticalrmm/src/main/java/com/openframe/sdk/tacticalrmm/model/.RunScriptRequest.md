Request body model for the TacticalRMM `POST /agents/<agent_id>/runscript/` endpoint, used to execute a saved script on a remote agent with configurable output delivery and execution options.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `script` | `Integer` | ID of the saved script to run |
| `output` | `String` | Output mode: `"wait"`, `"email"`, or `"collector"` |
| `args` | `List<String>` | Command-line arguments passed to the script |
| `runAsUser` | `Boolean` | Execute script under the logged-in user context |
| `envVars` | `List<String>` | Environment variables (format: `"KEY=VALUE"`) |
| `timeout` | `Integer` | Execution timeout in seconds |
| `runOnServer` | `Boolean` | Run on the RMM server instead of the agent |
| `emailMode` | `String` | Required when `output = "email"` |
| `emails` | `List<String>` | Recipient list when `output = "email"` |
| `customField` | `Integer` | Target custom field ID when `output = "collector"` |
| `saveAllOutput` | `Boolean` | Persist full script output to the database |

> `@JsonInclude(NON_NULL)` ensures only non-null fields are serialized, keeping the request payload minimal.

## Usage Example

```java
// Wait for inline output
RunScriptRequest request = new RunScriptRequest();
request.setScript(42);
request.setOutput("wait");
request.setTimeout(60);
request.setArgs(List.of("--verbose", "--dry-run"));
request.setEnvVars(List.of("DEBUG=true"));

// Email output mode
RunScriptRequest emailRequest = new RunScriptRequest();
emailRequest.setScript(42);
emailRequest.setOutput("email");
emailRequest.setEmailMode("always");
emailRequest.setEmails(List.of("admin@example.com"));

// Collector mode — store result in a custom field
RunScriptRequest collectorRequest = new RunScriptRequest();
collectorRequest.setScript(42);
collectorRequest.setOutput("collector");
collectorRequest.setCustomField(7);
collectorRequest.setSaveAllOutput(true);
```
