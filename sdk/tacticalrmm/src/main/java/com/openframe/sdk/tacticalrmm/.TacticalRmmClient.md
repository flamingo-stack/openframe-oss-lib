A Java client library for interacting with the Tactical RMM API, providing methods for agent management, command execution, and script operations. This client handles HTTP communication, JSON serialization, and error handling for common Tactical RMM operations.

## Key Components

- **TacticalRmmClient**: Main client class with HTTP client and JSON mapper
- **getInstallationSecret()**: Retrieves agent registration secrets
- **runCommand()**: Executes shell commands on remote agents (with overloaded variants)
- **getAgentInfo()**: Fetches detailed information about specific agents
- **getAllAgents()**: Lists all agents with minimal information
- **getAllScripts()**: Retrieves available scripts (method appears truncated)

## Usage Example

```java
// Initialize the client
TacticalRmmClient client = new TacticalRmmClient();

// Get agent registration secret
AgentRegistrationSecretRequest secretRequest = new AgentRegistrationSecretRequest();
String secret = client.getInstallationSecret(
    "https://your-tactical-server.com", 
    "your-api-key", 
    secretRequest
);

// Execute command on an agent
CommandResult result = client.runCommand(
    "https://your-tactical-server.com",
    "your-api-key",
    "agent-123",
    "powershell",
    "Get-Process"
);

// Get agent information
AgentInfo agentInfo = client.getAgentInfo(
    "https://your-tactical-server.com",
    "your-api-key", 
    "agent-123"
);

// List all agents
List<AgentListItem> agents = client.getAllAgents(
    "https://your-tactical-server.com",
    "your-api-key"
);
```
