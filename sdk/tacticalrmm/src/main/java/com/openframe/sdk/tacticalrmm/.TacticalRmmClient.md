A Java client for interacting with the Tactical RMM API, providing methods to manage agents, execute commands, and handle scripts across remote monitoring and management operations.

## Key Components

- **`getInstallationSecret()`** - Retrieves agent registration secrets for new installations
- **`runCommand()`** - Executes shell commands on specific agents with timeout and shell selection
- **`getAgentInfo()`** - Fetches detailed agent information including platform and OS details
- **`getAllAgents()`** - Returns a list of all registered agents with basic information
- **`getAllScripts()`** - Retrieves available scripts (method appears incomplete in provided code)
- **Exception handling** - Custom exceptions for API errors and general failures
- **HTTP client** - Built-in HTTP client with configurable timeouts and JSON processing

## Usage Example

```java
TacticalRmmClient client = new TacticalRmmClient();

// Get agent registration secret
AgentRegistrationSecretRequest request = new AgentRegistrationSecretRequest();
String secret = client.getInstallationSecret(
    "https://rmm.example.com", 
    "your-api-key", 
    request
);

// Execute command on agent
CommandResult result = client.runCommand(
    "https://rmm.example.com",
    "your-api-key", 
    "agent-123",
    "powershell",
    "Get-ComputerInfo",
    60,
    false
);

// Get all agents
List<AgentListItem> agents = client.getAllAgents(
    "https://rmm.example.com", 
    "your-api-key"
);

// Get specific agent details
AgentInfo agentInfo = client.getAgentInfo(
    "https://rmm.example.com",
    "your-api-key", 
    "agent-123"
);
```

The client includes comprehensive parameter validation, proper error handling with custom exceptions, and supports multiple shell types (cmd, powershell, bash) for cross-platform command execution.
