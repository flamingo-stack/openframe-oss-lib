Data model class for creating script resources in Tactical RMM through the API. This POJO encapsulates all the necessary configuration parameters for script creation requests.

## Key Components

- **CreateScriptRequest** - Main request model class with Jackson JSON annotations
- **Core Properties**:
  - `name` - Script identifier
  - `shell` - Execution shell (bash, powershell, etc.)
  - `scriptBody` - The actual script content
  - `defaultTimeout` - Execution timeout in seconds
- **Configuration Properties**:
  - `args` - Command line arguments
  - `envVars` - Environment variables
  - `runAsUser` - User execution context flag
  - `supportedPlatforms` - Compatible OS platforms
  - `category` - Script categorization
  - `description` - Script documentation

## Usage Example

```java
CreateScriptRequest request = new CreateScriptRequest();
request.setName("disk-cleanup");
request.setShell("bash");
request.setScriptBody("#!/bin/bash\ndf -h");
request.setDefaultTimeout(30);
request.setDescription("Check disk usage");
request.setSupportedPlatforms(Arrays.asList("linux", "darwin"));
request.setCategory("maintenance");
request.setArgs(Arrays.asList("-v", "--verbose"));
request.setEnvVars(Arrays.asList("LOG_LEVEL=INFO"));

// Use with TacticalRMM SDK to create script
tacticalRmmClient.createScript(request);
```
