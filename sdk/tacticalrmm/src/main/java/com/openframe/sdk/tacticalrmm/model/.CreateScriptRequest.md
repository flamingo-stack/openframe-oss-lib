A data transfer object that encapsulates all the necessary parameters for creating a new script in Tactical RMM through the API. This model uses Jackson annotations for JSON serialization and includes comprehensive script metadata and execution configuration.

## Key Components

- **Core Properties**: `name`, `shell`, `scriptBody` - Essential script identification and content
- **Execution Configuration**: `defaultTimeout`, `runAsUser`, `args`, `envVars` - Runtime behavior settings
- **Metadata**: `description`, `category`, `supportedPlatforms` - Organizational and compatibility information
- **Jackson Annotations**: All fields use `@JsonProperty` for proper API serialization
- **Standard Methods**: Full getter/setter methods and `toString()` implementation

## Usage Example

```java
CreateScriptRequest request = new CreateScriptRequest();
request.setName("System Health Check");
request.setShell("powershell");
request.setScriptBody("Get-ComputerInfo | Select-Object TotalPhysicalMemory, CsProcessors");
request.setDefaultTimeout(300);
request.setDescription("Checks system memory and CPU information");
request.setCategory("monitoring");
request.setSupportedPlatforms(Arrays.asList("windows"));
request.setRunAsUser(false);
request.setArgs(Arrays.asList("-ExecutionPolicy", "Bypass"));
request.setEnvVars(Arrays.asList("PATH=/usr/local/bin:$PATH"));

// Use with Tactical RMM API client
ScriptResponse response = tacticalRmmClient.createScript(request);
```
