A data model class that represents the result of executing a command on a TacticalRMM agent, containing execution details and output information.

## Key Components

- **agentId**: Unique identifier of the agent that executed the command
- **stdout**: Standard output captured from the command execution
- **timeout**: Command timeout value in seconds
- **shell**: Shell type used for command execution (e.g., "cmd", "powershell", "bash")
- **command**: The actual command string that was executed

All properties include standard getter and setter methods following JavaBean conventions.

## Usage Example

```java
// Creating a command result after execution
CommandResult result = new CommandResult();
result.setAgentId("agent-123");
result.setCommand("systeminfo");
result.setShell("cmd");
result.setTimeout(30);
result.setStdout("Host Name: DESKTOP-ABC123\nOS Name: Windows 10...");

// Accessing command execution details
String output = result.getStdout();
String executingAgent = result.getAgentId();
Integer executionTimeout = result.getTimeout();

// Typical usage in command execution workflow
if (result.getStdout() != null && !result.getStdout().isEmpty()) {
    System.out.println("Command '" + result.getCommand() + "' executed successfully");
    System.out.println("Output: " + result.getStdout());
} else {
    System.out.println("No output received from command execution");
}
```
