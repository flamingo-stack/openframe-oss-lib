A data model class that represents the result of a command execution in the TacticalRMM system, containing command details and output information.

## Key Components

- **CommandResult** - Main model class with getter/setter methods for:
  - `agentId` - Identifier of the agent that executed the command
  - `stdout` - Standard output from the command execution
  - `timeout` - Command timeout duration in seconds
  - `shell` - Shell type used for command execution
  - `command` - The actual command that was executed

## Usage Example

```java
// Creating a command result object
CommandResult result = new CommandResult();
result.setAgentId("agent-123");
result.setCommand("ls -la");
result.setShell("bash");
result.setTimeout(30);
result.setStdout("total 4\ndrwxr-xr-x 2 user user 4096 Jan 1 12:00 .");

// Reading command result data
String output = result.getStdout();
String executedCommand = result.getCommand();
Integer timeoutValue = result.getTimeout();

// Typically used in API responses or database operations
System.out.println("Agent " + result.getAgentId() + 
                   " executed: " + result.getCommand());
```
