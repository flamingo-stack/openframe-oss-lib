A data model class for creating agent registration secret requests in the Tactical RMM SDK. This class encapsulates configuration parameters needed to register new agents with specific installation methods, client/site assignments, and system capabilities.

## Key Components

- **Properties**: Configuration fields including `installMethod`, `client`, `site`, `expires`, `agentType`, `power`, `rdp`, `ping`, `goarch`, `api`, `fileName`, and `platform`
- **JSON Serialization**: Uses Jackson annotations for proper JSON property mapping
- **Getter/Setter Methods**: Standard accessor methods for all properties
- **toString()**: String representation method for debugging and logging

## Usage Example

```java
// Create a new agent registration request
AgentRegistrationSecretRequest request = new AgentRegistrationSecretRequest();

// Configure the agent registration
request.setInstallMethod("manual");
request.setClient(1);
request.setSite(5);
request.setExpires(3600); // 1 hour
request.setAgentType("server");
request.setPower(1); // Enable power management
request.setRdp(1); // Enable RDP
request.setPing(1); // Enable ping monitoring
request.setGoarch("amd64");
request.setApi("v4");
request.setFileName("agent-installer.exe");
request.setPlatform("windows");

// The object will be automatically serialized to JSON when sent via HTTP client
System.out.println(request.toString());
```
