Data model class for creating agent registration secrets in Tactical RMM. This class encapsulates all the configuration parameters needed to generate installation credentials for new RMM agents.

## Key Components

- **Installation Configuration**: `installMethod`, `agentType`, `platform`, `goarch` - Define how and where the agent will be installed
- **Organization Structure**: `client`, `site` - Specify the client and site hierarchy for agent placement
- **Security Settings**: `expires` - Controls how long the registration secret remains valid
- **Agent Capabilities**: `power`, `rdp`, `ping` - Configure which monitoring and control features are enabled
- **Runtime Parameters**: `api`, `fileName` - Specify API endpoint and installer filename
- **JSON Serialization**: Uses Jackson annotations for seamless API communication

## Usage Example

```java
// Create a new agent registration request
AgentRegistrationSecretRequest request = new AgentRegistrationSecretRequest();

// Configure basic installation settings
request.setInstallMethod("manual");
request.setAgentType("server");
request.setPlatform("windows");
request.setGoarch("amd64");

// Set organizational hierarchy
request.setClient(1);
request.setSite(5);

// Configure agent capabilities
request.setPower(1); // Enable power management
request.setRdp(1);   // Enable RDP access
request.setPing(1);  // Enable ping monitoring

// Set expiration (in hours)
request.setExpires(24);

// Configure API and filename
request.setApi("https://rmm.example.com");
request.setFileName("agent-installer.exe");

// Send request to generate registration secret
String registrationSecret = tacticalRmmClient.generateAgentSecret(request);
```
