Utility class for converting strings into URL-friendly slugs using configurable slug generation rules.

## Key Components

- **`SLUGIFY`** - Pre-configured Slugify instance with lowercase conversion and hyphen separators
- **`toSlug(String input)`** - Static method that converts any string into a URL-safe slug format

## Usage Example

```java
import com.openframe.core.util.SlugUtil;

// Convert regular text to slug
String slug1 = SlugUtil.toSlug("Hello World");
// Returns: "hello-world"

String slug2 = SlugUtil.toSlug("Special Characters! @#$");
// Returns: "special-characters"

// Handle null input gracefully
String slug3 = SlugUtil.toSlug(null);
// Returns: "org"

// Use for creating URL paths
String title = "My Amazing Blog Post";
String urlPath = "/posts/" + SlugUtil.toSlug(title);
// Results in: "/posts/my-amazing-blog-post"
```

The utility automatically handles null inputs by defaulting to "org" and ensures consistent slug formatting with lowercase letters and hyphen separators, making it ideal for generating URL-friendly identifiers from user input or content titles.
