A utility class for converting strings into URL-friendly slug format using the GitHub Slugify library. This class provides a centralized way to create consistent, lowercase, hyphen-separated slugs throughout the application.

## Key Components

- **`SLUGIFY`** - Static configured Slugify instance with lowercase conversion and hyphen separators
- **`toSlug(String input)`** - Static method that converts any string to a URL-safe slug format
- **Default fallback** - Returns "org" when input is null

## Usage Example

```java
import com.openframe.core.util.SlugUtil;

public class Example {
    public void demonstrateSlugGeneration() {
        // Convert regular text to slug
        String slug1 = SlugUtil.toSlug("Hello World!");
        // Result: "hello-world"
        
        // Handle special characters and spaces
        String slug2 = SlugUtil.toSlug("My Awesome Project 2024");
        // Result: "my-awesome-project-2024"
        
        // Handle null input safely
        String slug3 = SlugUtil.toSlug(null);
        // Result: "org"
        
        // Use in URL generation
        String organizationName = "Tech Company Inc.";
        String urlPath = "/organizations/" + SlugUtil.toSlug(organizationName);
        // Result: "/organizations/tech-company-inc"
    }
}
```

The utility is thread-safe and efficient for high-frequency slug generation operations, making it ideal for creating SEO-friendly URLs, file names, and identifiers from user input or dynamic content.
