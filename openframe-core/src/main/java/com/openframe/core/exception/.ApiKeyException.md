A custom runtime exception class specifically designed for handling failures related to API key operations within the OpenFrame application.

## Key Components

- **ApiKeyException** - Custom exception extending RuntimeException
- **Constructor with message** - Creates exception with descriptive error message
- **Constructor with message and cause** - Creates exception with message and underlying cause for exception chaining

## Usage Example

```java
// Throwing when API key is invalid
public void validateApiKey(String apiKey) {
    if (apiKey == null || apiKey.isEmpty()) {
        throw new ApiKeyException("API key cannot be null or empty");
    }
    
    if (!isValidFormat(apiKey)) {
        throw new ApiKeyException("Invalid API key format");
    }
}

// Exception chaining when wrapping other exceptions
public void authenticateWithApiKey(String apiKey) {
    try {
        // Some authentication logic that might fail
        externalAuthService.authenticate(apiKey);
    } catch (AuthenticationException e) {
        throw new ApiKeyException("API key authentication failed", e);
    }
}

// Catching and handling
try {
    validateApiKey(userProvidedKey);
} catch (ApiKeyException e) {
    logger.error("API key validation failed: " + e.getMessage());
    return errorResponse("Invalid API key provided");
}
```
