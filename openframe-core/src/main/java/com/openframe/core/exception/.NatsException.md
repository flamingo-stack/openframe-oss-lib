A custom runtime exception class for handling NATS messaging system errors within the OpenFrame Core framework.

## Key Components

- **NatsException**: Custom runtime exception extending `RuntimeException`
- **Constructor overloads**: Two constructors supporting message-only and message-with-cause scenarios

## Usage Example

```java
// Throwing a simple NATS exception
throw new NatsException("Failed to connect to NATS server");

// Throwing with underlying cause
try {
    // NATS operation that might fail
    connection.publish(subject, data);
} catch (IOException e) {
    throw new NatsException("Failed to publish message to subject: " + subject, e);
}

// Catching and handling
try {
    natsService.processMessage();
} catch (NatsException e) {
    logger.error("NATS operation failed: " + e.getMessage(), e);
    // Handle gracefully or re-throw
}
```

This exception provides a standardized way to handle NATS-related errors throughout the application, allowing for consistent error handling and debugging of messaging operations.
