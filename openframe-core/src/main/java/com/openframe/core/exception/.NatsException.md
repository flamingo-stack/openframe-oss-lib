A custom runtime exception class for handling NATS messaging system errors within the OpenFrame core framework.

## Key Components

- **NatsException** - Runtime exception specifically designed for NATS-related errors
- **Constructor overloads** - Support for both simple message and message with cause scenarios
- **Runtime inheritance** - Extends `RuntimeException` for unchecked exception handling

## Usage Example

```java
// Throwing a simple NATS exception
if (connection == null) {
    throw new NatsException("Failed to establish NATS connection");
}

// Throwing with underlying cause
try {
    natsConnection.publish(subject, data);
} catch (IOException e) {
    throw new NatsException("Failed to publish message to NATS", e);
}

// Handling in service layer
try {
    natsService.processMessage(payload);
} catch (NatsException e) {
    logger.error("NATS operation failed: {}", e.getMessage(), e);
    // Handle NATS-specific error logic
}
```

This exception class provides a clean way to distinguish NATS-specific errors from other system exceptions, allowing for targeted error handling and better debugging in distributed messaging scenarios.
