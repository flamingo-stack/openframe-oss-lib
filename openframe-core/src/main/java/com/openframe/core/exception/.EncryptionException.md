A custom runtime exception for handling encryption-related errors in the OpenFrame application, providing structured error reporting with error codes and detailed messaging.

## Key Components

- **EncryptionException** - Main exception class extending `RuntimeException`
- **errorCode** - String field for categorizing different types of encryption errors
- **Constructors** - Two overloaded constructors for different error scenarios

## Usage Example

```java
// Throwing an encryption exception with error code
public void encryptData(String data) throws EncryptionException {
    try {
        // Encryption logic here
        performEncryption(data);
    } catch (Exception e) {
        throw new EncryptionException("ENC001", "Failed to encrypt sensitive data", e);
    }
}

// Catching and handling the exception
try {
    encryptionService.encryptData(userData);
} catch (EncryptionException e) {
    logger.error("Encryption failed with code: {} - {}", 
                e.getErrorCode(), e.getMessage());
    // Handle specific error codes
    if ("ENC001".equals(e.getErrorCode())) {
        // Handle encryption failure
    }
}
```

The exception provides both human-readable messages and machine-readable error codes for precise error handling and logging in encryption operations.
