A custom runtime exception thrown when API key lookup operations fail to find a specified key.

## Key Components

**ApiKeyNotFoundException** - Runtime exception class with three constructor overloads:
- `ApiKeyNotFoundException(String keyId)` - Basic constructor for missing API key by ID
- `ApiKeyNotFoundException(String keyId, String userId)` - Constructor for user-specific API key lookup failures
- `ApiKeyNotFoundException(String message, Throwable cause)` - Generic constructor with custom message and underlying cause

## Usage Example

```java
// Basic API key not found
throw new ApiKeyNotFoundException("ak_123456789");

// User-specific API key not found
throw new ApiKeyNotFoundException("ak_123456789", "user_abc");

// With underlying cause
try {
    // Database lookup logic
} catch (SQLException e) {
    throw new ApiKeyNotFoundException("Database error during API key lookup", e);
}

// Catching the exception
try {
    apiKeyService.validateKey(keyId);
} catch (ApiKeyNotFoundException e) {
    logger.error("API key validation failed: {}", e.getMessage());
    return ResponseEntity.status(401).body("Invalid API key");
}
```
