A runtime exception thrown when an API key cannot be found in the system. This exception provides specific error messages for different scenarios involving missing API keys.

## Key Components

- **ApiKeyNotFoundException** - Main exception class extending `RuntimeException`
- **Constructor overloads** - Three constructors for different error scenarios:
  - Simple key ID lookup failure
  - User-specific key lookup failure  
  - Generic message with cause chaining

## Usage Example

```java
// Throwing when API key doesn't exist
public ApiKey findApiKey(String keyId) {
    ApiKey key = repository.findById(keyId);
    if (key == null) {
        throw new ApiKeyNotFoundException(keyId);
    }
    return key;
}

// Throwing for user-specific key lookup
public ApiKey getUserApiKey(String keyId, String userId) {
    ApiKey key = repository.findByKeyIdAndUserId(keyId, userId);
    if (key == null) {
        throw new ApiKeyNotFoundException(keyId, userId);
    }
    return key;
}

// Catching the exception
try {
    ApiKey key = apiService.findApiKey("key123");
} catch (ApiKeyNotFoundException e) {
    logger.error("Key lookup failed: {}", e.getMessage());
    return ResponseEntity.notFound().build();
}
```
