A Spring service that provides AES-256-CBC encryption/decryption capabilities with HMAC authentication using Spring Security Crypto. Designed for securing sensitive data like OAuth client secrets with configurable password and salt values.

## Key Components

- **EncryptionService**: Main service class with constructor injection of encryption credentials
- **encrypt(String)**: Encrypts plain text using AES-256-CBC + HMAC
- **decrypt(String)**: Decrypts encrypted text back to plain text
- **encryptClientSecret(String)**: OAuth-specific encryption method with debug logging
- **decryptClientSecret(String)**: OAuth-specific decryption method with debug logging

## Usage Example

```java
@Autowired
private EncryptionService encryptionService;

// Basic encryption/decryption
String plainText = "sensitive-data";
String encrypted = encryptionService.encrypt(plainText);
String decrypted = encryptionService.decrypt(encrypted);

// OAuth client secret handling
String clientSecret = "my-oauth-client-secret";
String encryptedSecret = encryptionService.encryptClientSecret(clientSecret);
String originalSecret = encryptionService.decryptClientSecret(encryptedSecret);
```

**Configuration Requirements:**
```properties
security.encryption.password=your-32-character-minimum-password
security.encryption.salt=your-16-character-hex-salt
```

The service validates input parameters and throws `EncryptionException` for operation failures and `IllegalArgumentException` for invalid inputs.
