A Spring service that provides AES-256-CBC encryption capabilities for securing sensitive data, particularly OAuth client secrets, using Spring Security Crypto.

## Key Components

- **EncryptionService**: Main service class with Spring Security Crypto integration
- **Constructor**: Initializes TextEncryptor with password and salt validation (32+ chars password, 16+ chars salt)
- **encrypt(String)**: Encrypts plain text using AES-256-CBC with HMAC
- **decrypt(String)**: Decrypts encrypted text back to plain text
- **encryptClientSecret(String)**: Specialized method for OAuth client secret encryption
- **decryptClientSecret(String)**: Specialized method for OAuth client secret decryption

## Usage Example

```java
@Autowired
private EncryptionService encryptionService;

// Encrypt sensitive data
String plainSecret = "my-oauth-client-secret";
String encrypted = encryptionService.encryptClientSecret(plainSecret);

// Decrypt when needed
String decrypted = encryptionService.decryptClientSecret(encrypted);

// General purpose encryption
String sensitiveData = "user-private-info";
String encryptedData = encryptionService.encrypt(sensitiveData);
String originalData = encryptionService.decrypt(encryptedData);
```

**Configuration Requirements:**
```properties
security.encryption.password=your-32-character-minimum-password-here
security.encryption.salt=your-16-character-hex-salt
```

The service validates input parameters, provides comprehensive error handling with custom exceptions, and includes specialized methods for OAuth client secret management with debug logging.
