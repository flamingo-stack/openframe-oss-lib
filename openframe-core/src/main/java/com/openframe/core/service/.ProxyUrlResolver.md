A Spring service that resolves proxy URLs for integrated tools, handling URL transformation and routing based on environment profiles.

## Key Components

- **`resolve()`** - Main method that transforms original request URIs into target URIs for proxying to integrated tools
- **`isLocalProfile()`** - Helper method to determine if the application is running in local development mode
- **Environment-aware routing** - Uses localhost for local profiles, actual hostnames for other environments
- **Path manipulation** - Extracts and transforms URL paths by removing prefixes and preserving query parameters

## Usage Example

```java
@Autowired
private ProxyUrlResolver proxyUrlResolver;

// Resolve a proxy URL for a tool
URI originalUri = URI.create("/tools/jupyter/notebooks/example.ipynb?param=value");
String toolUrl = "http://jupyter-server.com:8888";
String toolPort = "8888";

URI targetUri = proxyUrlResolver.resolve(
    "jupyter", 
    toolUrl, 
    toolPort, 
    originalUri, 
    "/tools"
);
// Result: http://localhost:8888/notebooks/example.ipynb?param=value (in local profile)
// Result: http://jupyter-server.com:8888/notebooks/example.ipynb?param=value (in other profiles)

// Works with agent tools too
URI agentUri = URI.create("/tools/agent/my-agent/api/status");
URI agentTarget = proxyUrlResolver.resolve(
    "my-agent", 
    "http://agent-service:3000", 
    "3000", 
    agentUri, 
    "/tools/agent"
);
```
