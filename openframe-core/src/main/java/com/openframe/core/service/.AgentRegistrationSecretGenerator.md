A Spring component that generates cryptographically secure random secrets for agent registration using Base64 URL-safe encoding.

## Key Components

- **`generate()`** - Creates a 32-character URL-safe Base64 encoded secret using SecureRandom
- **`secureRandom`** - SecureRandom instance for cryptographically strong random number generation
- **`KEY_LENGTH`** - Constant defining the target length (32 characters) of the generated secret

## Usage Example

```java
@Service
public class AgentService {
    
    private final AgentRegistrationSecretGenerator secretGenerator;
    
    public String createNewAgent() {
        // Generate a secure registration secret for new agent
        String registrationSecret = secretGenerator.generate();
        
        // Use the secret for agent registration
        Agent agent = new Agent();
        agent.setRegistrationSecret(registrationSecret);
        
        return registrationSecret; // Returns ~32 character URL-safe string
    }
}
```

The generator produces URL-safe Base64 strings (using `-` and `_` instead of `+` and `/`) making them suitable for use in URLs, tokens, and API keys. Each generated secret is cryptographically random and approximately 32 characters long.
