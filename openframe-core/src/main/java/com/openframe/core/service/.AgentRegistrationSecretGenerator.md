A Spring component that generates cryptographically secure random secrets for agent registration processes using Base64 URL-safe encoding.

## Key Components

- **`generate()`** - Creates a secure random secret string using `SecureRandom` and Base64 URL-safe encoding
- **`KEY_LENGTH`** - Constant defining the target length (32 characters) for generated secrets
- **`secureRandom`** - Instance of `SecureRandom` for cryptographically strong random number generation

## Usage Example

```java
@Service
public class AgentService {
    
    private final AgentRegistrationSecretGenerator secretGenerator;
    
    public AgentService(AgentRegistrationSecretGenerator secretGenerator) {
        this.secretGenerator = secretGenerator;
    }
    
    public Agent createNewAgent(String agentName) {
        String registrationSecret = secretGenerator.generate();
        
        Agent agent = new Agent();
        agent.setName(agentName);
        agent.setRegistrationSecret(registrationSecret);
        
        return agentRepository.save(agent);
    }
}
```

The generator produces URL-safe Base64 encoded strings that are suitable for use in web APIs, query parameters, and registration tokens while maintaining cryptographic security.
