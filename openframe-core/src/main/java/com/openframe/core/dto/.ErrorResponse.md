A standardized data transfer object for representing error responses in the OpenFrame application, providing consistent error messaging across the API.

## Key Components

- **ErrorResponse class**: Main DTO with `code` and `message` fields
- **Lombok annotations**: `@Data`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic getter/setter generation
- **Jackson annotation**: `@JsonInclude(NON_NULL)` to exclude null fields from JSON serialization
- **Convenience constructor**: Single-parameter constructor accepting only a message

## Usage Example

```java
// Create error response with just a message
ErrorResponse basicError = new ErrorResponse("User not found");

// Create error response with both code and message
ErrorResponse detailedError = new ErrorResponse("USER_404", "User not found");

// In a REST controller
@RestController
public class UserController {
    
    @ExceptionHandler(UserNotFoundException.class)
    public ResponseEntity<ErrorResponse> handleUserNotFound(UserNotFoundException ex) {
        ErrorResponse error = new ErrorResponse("USER_NOT_FOUND", ex.getMessage());
        return ResponseEntity.status(404).body(error);
    }
}
```

When serialized to JSON, the response automatically excludes null fields, ensuring clean API responses.
