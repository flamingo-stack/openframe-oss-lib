# CountedGenericQueryResult Sub-Module Documentation

## Overview
The `CountedGenericQueryResult` sub-module extends the `GenericQueryResult` to include a count of filtered results, enhancing the querying capabilities of the OpenFrame API.

## Core Functionality
- **Filtered Count**: Provides the number of results after applying filters.

## Key Fields
- `filteredCount`: int

## Usage
This sub-module is particularly useful in scenarios where understanding the number of filtered results is crucial, such as in paginated data retrieval.
