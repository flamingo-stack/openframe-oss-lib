# DeviceFilterOption Sub-Module Documentation

## Overview
The `DeviceFilterOption` sub-module is part of the device filtering DTOs in the OpenFrame API library. It provides options for filtering devices based on specific criteria.

## Core Functionality
- **Value-Label Pairs**: Represents the filter option as a value-label pair.
- **Count**: Number of items matching the filter.

## Key Fields
- `value`: String
- `label`: String
- `count`: Integer

## Usage
This DTO is used to define and apply filters to device data, allowing for more targeted queries.
