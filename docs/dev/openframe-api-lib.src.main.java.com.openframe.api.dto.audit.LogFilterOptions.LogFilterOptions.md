# Log Filter Options

## Overview
The `LogFilterOptions` class provides filtering options for log entries, allowing for date and type-based filtering.

## Properties
- `LocalDate startDate`: The start date for filtering logs.
- `LocalDate endDate`: The end date for filtering logs.
- `List<String> eventTypes`: The types of events to filter.
- `List<String> toolTypes`: The types of tools to filter.
- `List<String> severities`: The severities of logs to filter.
- `List<String> organizationIds`: The organization IDs to filter logs by.
- `String deviceId`: The device ID to filter logs.
