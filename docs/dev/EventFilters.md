# EventFilters Documentation

## Overview
The `EventFilters` class defines filters for events based on user IDs and event types.

## Core Components
- **userIds**: A list of user IDs for filtering events.
- **eventTypes**: A list of event types for filtering.

## Methods
- `getUserIds()`: Returns the list of user IDs.
- `setUserIds(List<String> userIds)`: Sets the list of user IDs.
- `getEventTypes()`: Returns the list of event types.
- `setEventTypes(List<String> eventTypes)`: Sets the list of event types.
