# QueryStats Documentation

## Overview
The `QueryStats` class is a nested class within the `Query` class that holds statistical data related to the execution of the query.

## Properties
- **userTimeP50**: 50th percentile of user execution time.
- **userTimeP95**: 95th percentile of user execution time.
- **systemTimeP50**: 50th percentile of system execution time.
- **systemTimeP95**: 95th percentile of system execution time.
- **totalExecutions**: Total number of times the query has been executed.
