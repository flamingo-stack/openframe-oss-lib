# Force Tool Agent Update Response Documentation

## Overview
The `ForceToolAgentUpdateResponse` class encapsulates the response structure for tool agent updates, containing a list of update response items.

## Core Components
- **Items**: A list of `ForceToolAgentUpdateResponseItem` objects representing the updates for each tool agent.

## Example Usage
```java
ForceToolAgentUpdateResponse response = new ForceToolAgentUpdateResponse();
response.setItems(new ArrayList<>());
```
