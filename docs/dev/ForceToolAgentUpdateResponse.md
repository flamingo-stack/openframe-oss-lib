# ForceToolAgentUpdateResponse Documentation

## Overview
The `ForceToolAgentUpdateResponse` class is a data transfer object that contains a list of items related to tool agent updates.

### Core Components
- **Items**: A list of `ForceToolAgentUpdateResponseItem` objects that represent the details of each tool agent update.

### Example Usage
```java
ForceToolAgentUpdateResponse response = new ForceToolAgentUpdateResponse();
response.setItems(myResponseItems);
```
