# LogFilterOptions Documentation

## Overview
The `LogFilterOptions` class provides options for filtering logs based on various criteria such as date ranges, event types, and organization IDs.

## Properties
- **startDate**: The start date for filtering logs.
- **endDate**: The end date for filtering logs.
- **eventTypes**: A list of event types to filter logs.
- **toolTypes**: A list of tool types to filter logs.
- **severities**: A list of severities to filter logs.
- **organizationIds**: A list of organization IDs to filter logs.
- **deviceId**: The ID of the device to filter logs.
