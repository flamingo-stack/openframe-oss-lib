# LogFilterOptions Documentation

## Overview
The `LogFilterOptions` class provides options for filtering log events based on various criteria such as date range and event types.

## Core Components
- **startDate**: The start date for filtering log events.
- **endDate**: The end date for filtering log events.
- **eventTypes**: List of event types to filter.
- **toolTypes**: List of tool types to filter.
- **severities**: List of severity levels to filter.
- **organizationIds**: List of organization IDs to filter.
- **deviceId**: Identifier for the device to filter.
