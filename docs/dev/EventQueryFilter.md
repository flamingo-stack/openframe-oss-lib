# EventQueryFilter Documentation

## Overview
The `EventQueryFilter` class is used to filter events based on various criteria such as user IDs, event types, and date ranges.

## Properties
- **userIds**: List of user IDs to filter events.
- **eventTypes**: List of event types to filter.
- **startDate**: The start date for filtering events.
- **endDate**: The end date for filtering events.

## Methods
- Getters and setters for each property are provided by Lombok annotations.
