# ForceToolInstallationRequest Documentation

## Overview
The `ForceToolInstallationRequest` class encapsulates the request for installing tools on specified machines.

## Core Components
- **machineIds**: A list of machine IDs where the tool should be installed.
- **toolAgentId**: The ID of the tool agent to be installed.

## Methods
- `getMachineIds()`: Returns the list of machine IDs.
- `setMachineIds(List<String> machineIds)`: Sets the list of machine IDs.
- `getToolAgentId()`: Returns the tool agent ID.
- `setToolAgentId(String toolAgentId)`: Sets the tool agent ID.
