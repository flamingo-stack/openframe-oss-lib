# OrganizationFilterOptions Documentation

## Overview
The `OrganizationFilterOptions` class is a data transfer object (DTO) that encapsulates the options for filtering organizations based on various criteria.

## Properties
- `category`: The category of the organization.
- `minEmployees`: The minimum number of employees in the organization.
- `maxEmployees`: The maximum number of employees in the organization.
- `hasActiveContract`: Indicates whether the organization has an active contract.
