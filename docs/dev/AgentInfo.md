# AgentInfo Documentation

## Overview
The `AgentInfo` class represents the agent information model used in Tactical RMM API responses. It includes properties that provide essential details about the agent.

## Properties
- **agentId**: Unique identifier for the agent.
- **platform**: The platform on which the agent is running.
- **operatingSystem**: The operating system of the agent.
- **hostname**: The hostname of the agent.

## Methods
- `getAgentId()`: Returns the agent ID.
- `setAgentId(String agentId)`: Sets the agent ID.
- `getPlatform()`: Returns the platform.
- `setPlatform(String platform)`: Sets the platform.
- `getOperatingSystem()`: Returns the operating system.
- `setOperatingSystem(String operatingSystem)`: Sets the operating system.
- `getHostname()`: Returns the hostname.
- `setHostname(String hostname)`: Sets the hostname.
- `toString()`: Returns a string representation of the agent information.
