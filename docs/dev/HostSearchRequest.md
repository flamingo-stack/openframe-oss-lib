# Host Search Request Documentation

## Overview
The `HostSearchRequest` class encapsulates the parameters required for searching hosts. It includes pagination and ordering options to facilitate efficient querying.

## Core Components
- **Query**: The search term used to find hosts.
- **Page**: The current page number for pagination (default is 0).
- **Per Page**: The number of results per page (default is 100).
- **Order Key**: The key by which results should be ordered.
- **Order Direction**: The direction of the order (ascending or descending).

## Usage
This class is essential for host management operations, allowing users to search and retrieve host information based on specific criteria.
