# HostSearchRequest Documentation

## Overview
The `HostSearchRequest` class represents the parameters for searching hosts within the Fleet MDM system.

## Properties
- **query**: The search query string.
- **page**: The page number for pagination.
- **perPage**: The number of results per page.
- **orderKey**: The key by which to order the results.
- **orderDirection**: The direction of the ordering (ascending or descending).
