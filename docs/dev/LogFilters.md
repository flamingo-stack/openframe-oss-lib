# LogFilters Documentation

## Overview
The `LogFilters` class is designed to filter log events based on various criteria such as tool types, event types, severities, and organizations.

## Properties
- **toolTypes**: A list of tool types to filter logs.
- **eventTypes**: A list of event types to filter logs.
- **severities**: A list of severities to filter logs.
- **organizations**: A list of organization filter options.
