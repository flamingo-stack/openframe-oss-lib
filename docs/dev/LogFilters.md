# LogFilters Documentation

## Overview
The `LogFilters` class provides a structure for filtering log entries based on various criteria such as tool types, event types, severities, and organizations.

## Core Components
- **toolTypes**: A list of tool types to filter logs.
- **eventTypes**: A list of event types to filter logs.
- **severities**: A list of severities to filter logs.
- **organizations**: A list of `OrganizationFilterOption` objects to filter logs by organization.
