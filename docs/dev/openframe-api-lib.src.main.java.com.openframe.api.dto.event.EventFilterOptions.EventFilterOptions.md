# Event Filter Options

## Overview
The `EventFilterOptions` class allows filtering of events based on user IDs, event types, and date ranges.

## Properties
- `List<String> userIds`: The user IDs to filter events by.
- `List<String> eventTypes`: The types of events to filter.
- `LocalDate startDate`: The start date for filtering events.
- `LocalDate endDate`: The end date for filtering events.
