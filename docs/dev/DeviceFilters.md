# DeviceFilters Documentation

## Overview
The `DeviceFilters` class represents a collection of filters that can be applied to devices, allowing for more granular control over device management.

## Core Components
- **statuses**: List of device filter options based on status.
- **deviceTypes**: List of device filter options based on type.
- **osTypes**: List of device filter options based on operating system type.
- **organizationIds**: List of organization IDs for filtering devices.
- **tags**: List of tag filter options for devices.
- **filteredCount**: Count of devices that match the applied filters.
