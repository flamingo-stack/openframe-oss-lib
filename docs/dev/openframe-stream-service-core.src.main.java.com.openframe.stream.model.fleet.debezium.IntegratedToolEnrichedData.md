# Integrated Tool Enriched Data Documentation

## Overview
The `IntegratedToolEnrichedData` class encapsulates enriched data related to tools, including machine and organization identifiers.

## Core Components
- **machineId**: Identifier for the machine.
- **hostname**: Hostname of the machine.
- **organizationId**: Identifier for the organization.
- **organizationName**: Name of the organization.
- **userId**: Identifier for the user associated with the tool.
