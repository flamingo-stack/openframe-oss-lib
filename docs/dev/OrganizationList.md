# OrganizationList Documentation

## Overview
The `OrganizationList` class is a Data Transfer Object (DTO) that encapsulates a list of organizations. It is used to structure the response for organization-related queries.

## Core Components
- **Field**: `organizations`
  - **Type**: `List<Organization>`
  - **Description**: A list containing organization objects.

## Usage
This DTO is typically used in API responses to provide a structured list of organizations to the client.
