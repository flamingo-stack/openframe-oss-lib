# EventFilterOptions Documentation

## Overview
The `EventFilterOptions` class is a data transfer object (DTO) that encapsulates the options for filtering events based on user IDs, event types, and a date range.

## Properties
- `userIds`: A list of user IDs to filter events.
- `eventTypes`: A list of event types to filter.
- `startDate`: The start date for filtering events.
- `endDate`: The end date for filtering events.

## Usage
This class is typically used in scenarios where events need to be filtered based on specific criteria, allowing for more targeted data retrieval.
