# ForceToolAgentInstallationResponseItem Documentation

## Overview
`ForceToolAgentInstallationResponseItem` represents the response item for tool agent installations. It contains information about the installation status of a tool on a specific machine.

## Core Components
- **machineId**: The ID of the machine where the tool is installed.
- **toolAgentId**: The ID of the tool agent that was installed.
- **status**: The status of the installation, represented by `ForceAgentStatus`.
