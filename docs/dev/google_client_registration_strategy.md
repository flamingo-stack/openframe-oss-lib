# GoogleClientRegistrationStrategy Documentation

## Overview
The `GoogleClientRegistrationStrategy` class implements the client registration strategy for Google OAuth. It extends the `BaseOidcClientRegistrationStrategy` and provides specific properties for Google SSO.

## Core Components
- **googleProps**: Holds the Google SSO properties.

## Methods
- `providerId()`: Returns the provider ID for Google.
- `props()`: Returns the properties specific to Google SSO.
