# AgentRegistrationSecretRequest Documentation

## Overview
The `AgentRegistrationSecretRequest` class represents a request for agent registration, containing various parameters necessary for the installation process.

## Properties
- `installMethod`: The method used for installation.
- `client`: The client ID associated with the request.
- `site`: The site ID associated with the request.
- `expires`: The expiration time for the request.
- `agentType`: The type of agent being registered.
- `power`: Power settings for the agent.
- `rdp`: RDP settings for the agent.
- `ping`: Ping settings for the agent.
- `goarch`: Architecture settings for the agent.
- `api`: API settings for the agent.
- `fileName`: The name of the file associated with the agent.
- `platform`: The platform for which the agent is being registered.
