# AgentRegistrationSecretRequest Documentation

## Overview
`AgentRegistrationSecretRequest` contains the necessary details for registering an agent, including installation methods and client information.

## Core Components
- **installMethod**: The method used for installation.
- **client**: The client ID associated with the agent.
- **site**: The site ID where the agent is registered.
- **expires**: The expiration time for the registration.
- **agentType**: The type of agent being registered.
- **power**: Power settings for the agent.
- **rdp**: Remote Desktop Protocol settings.
- **ping**: Ping settings for the agent.
- **goarch**: Architecture of the agent.
- **api**: API settings for the agent.
- **fileName**: The name of the file associated with the agent.
- **platform**: The platform on which the agent is running.
