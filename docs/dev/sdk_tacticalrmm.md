# ScriptListItem Documentation

## Overview
The `ScriptListItem` class represents a script item in the Tactical RMM API responses. It contains properties for the script's ID and name.

## Core Components
- **id**: An integer representing the unique identifier of the script.
- **name**: A string representing the name of the script.

## Methods
- `getId()`: Returns the ID of the script.
- `setId(Integer id)`: Sets the ID of the script.
- `getName()`: Returns the name of the script.
- `setName(String name)`: Sets the name of the script.
- `toString()`: Returns a string representation of the script list item.
