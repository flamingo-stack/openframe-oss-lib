A specialized exception thrown when a user lookup fails, extending `NotFoundException` with a predefined `USER_NOT_FOUND` error code.

## Key Components

| Component | Description |
|-----------|-------------|
| `UserNotFoundException` | Extends `NotFoundException`; signals that a user with a given ID could not be located |
| `ErrorCode.USER_NOT_FOUND` | Standardized error code passed to the parent exception for consistent error handling |

## Usage Example

```java
// Throw when a user cannot be found by ID
User user = userRepository.findById(userId)
    .orElseThrow(() -> new UserNotFoundException(userId));
```
