A simple filter model used to query tools by their attributes, supporting filtering by enabled state, type, category, and platform category.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `enabled` | `Boolean` | Filter tools by their enabled/disabled state |
| `type` | `String` | Filter by tool type |
| `category` | `String` | Filter by tool category |
| `platformCategory` | `String` | Filter by platform-specific category |

## Usage Example

```java
// Build a filter for enabled tools in a specific category
ToolQueryFilter filter = ToolQueryFilter.builder()
    .enabled(true)
    .type("automation")
    .category("network")
    .platformCategory("msp")
    .build();

// Or use no-args constructor with setters
ToolQueryFilter filter = new ToolQueryFilter();
filter.setEnabled(true);
filter.setCategory("monitoring");
```
