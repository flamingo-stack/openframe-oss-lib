Enumeration defining the high-level categories used to classify and route notifications within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `DASHBOARD` | Notifications related to dashboard activity |
| `CUSTOMERS` | Customer/client-related notifications |
| `DEVICES` | Device management notifications |
| `SCRIPTS` | Script execution notifications |
| `MONITORING` | System monitoring alerts |
| `SOFTWARE` | Software deployment/update notifications |
| `LOGS` | Log-related notifications |
| `TICKETS` | Helpdesk ticket notifications |
| `MINGO` | Mingo AI assistant notifications |
| `GENERIC` | Uncategorized or general-purpose notifications |

## Usage Example

```java
Notification notification = new Notification();
notification.setCategory(NotificationCategory.TICKETS);

// Filter notifications by category
List<Notification> ticketNotifications = notifications.stream()
    .filter(n -> n.getCategory() == NotificationCategory.TICKETS)
    .collect(Collectors.toList());

// Switch-based routing
switch (notification.getCategory()) {
    case MONITORING:
        alertOpsTeam(notification);
        break;
    case TICKETS:
        routeToHelpdesk(notification);
        break;
    default:
        handleGeneric(notification);
}
```
