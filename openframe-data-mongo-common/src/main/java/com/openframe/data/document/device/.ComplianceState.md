Enum representing the compliance status of a device within the OpenFrame platform, used to track and communicate device policy adherence states.

## Key Components

| Value | Description |
|---|---|
| `COMPLIANT` | Device meets all required policies and standards |
| `NON_COMPLIANT` | Device violates one or more policies |
| `IN_REMEDIATION` | Device is actively undergoing remediation to achieve compliance |
| `UNKNOWN` | Compliance status cannot be determined |

## Usage Example

```java
ComplianceState state = device.getComplianceState();

switch (state) {
    case COMPLIANT:
        // No action required
        break;
    case NON_COMPLIANT:
        triggerAlert(device);
        break;
    case IN_REMEDIATION:
        monitorProgress(device);
        break;
    case UNKNOWN:
        scheduleComplianceScan(device);
        break;
}
```
