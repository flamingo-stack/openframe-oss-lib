Enumeration defining the supported remote management and monitoring tool integrations available within the OpenFrame platform.

## Key Components

| Constant | Description |
|---|---|
| `MESHCENTRAL` | MeshCentral open-source remote monitoring and management platform |
| `OPENFRAME_RMM` | OpenFrame native remote monitoring and management solution |
| `FLEET_MDM` | Fleet open-source MDM/endpoint management solution |

## Usage Example

```java
ToolType toolType = ToolType.MESHCENTRAL;

switch (toolType) {
    case MESHCENTRAL   -> connectToMeshCentral();
    case OPENFRAME_RMM -> connectToOpenFrameRMM();
    case FLEET_MDM     -> connectToFleetMDM();
}
```
