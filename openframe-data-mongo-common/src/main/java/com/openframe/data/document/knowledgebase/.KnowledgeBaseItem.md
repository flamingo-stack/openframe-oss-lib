MongoDB document entity representing a single item in the OpenFrame knowledge base, supporting hierarchical content organization with articles, categories, and other typed entries.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | MongoDB document identifier (`@Id`) |
| `type` | `KnowledgeBaseItemType` | Indexed enum classifying the item (article, category, etc.) |
| `parentId` | `String` | Indexed reference to parent item, enabling tree hierarchy |
| `slug` | `String` | URL-friendly identifier for routing |
| `sortOrder` | `Integer` | Controls display ordering within a parent |
| `content` | `String` | Full body content of the item |
| `summary` | `String` | Short description or excerpt |
| `status` | `KnowledgeBaseArticleStatus` | Indexed lifecycle state (draft, published, archived, etc.) |
| `publishedAt` | `Instant` | Timestamp when item was published |
| `createdAt` / `updatedAt` | `Instant` | Auto-managed audit timestamps via `@CreatedDate` / `@LastModifiedDate` |

### Compound Indexes

| Index Name | Fields | Purpose |
|------------|--------|---------|
| `parent_type_sort` | `parentId`, `type`, `sortOrder` | Efficient child listing with ordered results |
| `type_status` | `type`, `status` | Fast filtering by type and publication state |

## Usage Example

```java
KnowledgeBaseItem article = KnowledgeBaseItem.builder()
    .type(KnowledgeBaseItemType.ARTICLE)
    .name("How to Reset a Password")
    .parentId("cat-account-management")
    .slug("how-to-reset-password")
    .sortOrder(1)
    .content("Step-by-step guide to resetting a user password...")
    .summary("Quick guide for password resets.")
    .status(KnowledgeBaseArticleStatus.PUBLISHED)
    .publishedAt(Instant.now())
    .createdBy("technician-42")
    .build();

mongoTemplate.save(article);
```
