Enum defining the supported shell interpreters available for executing scripts via the RMM agent.

## Key Components

| Value | Description |
|-------|-------------|
| `POWERSHELL` | Windows PowerShell interpreter |
| `CMD` | Windows Command Prompt |
| `BASH` | Unix/Linux Bash shell |
| `PYTHON` | Python interpreter |
| `NUSHELL` | Nu shell interpreter |
| `SHELL` | Generic POSIX shell |

## Usage Example

```java
Script script = new Script();
script.setShell(ScriptShell.POWERSHELL);
script.setContent("Get-Process | Select-Object Name, CPU");

// Branch execution logic based on the assigned interpreter
switch (script.getShell()) {
    case POWERSHELL -> agentDispatcher.runPowerShell(script);
    case BASH       -> agentDispatcher.runBash(script);
    case PYTHON     -> agentDispatcher.runPython(script);
    default         -> agentDispatcher.runGeneric(script);
}
```

## Notes

- Values intentionally mirror the shell identifiers exposed by **Tactical RMM**, allowing existing scripts to be migrated without modifying their shebangs or interpreter declarations.
- New interpreter entries should only be added once corresponding agent-side execution support is implemented.
