Defines the possible item types that can be associated with an assignment in the OpenFrame platform.

## Key Components

| Enum Constant | Description |
|---|---|
| `TICKET` | Represents a support ticket assignment item |
| `KNOWLEDGE_ARTICLE` | Represents a knowledge base article assignment item |

## Usage Example

```java
AssignmentItem item = new AssignmentItem();
item.setType(AssignmentItemType.TICKET);

if (item.getType() == AssignmentItemType.KNOWLEDGE_ARTICLE) {
    // handle knowledge article logic
}
```
