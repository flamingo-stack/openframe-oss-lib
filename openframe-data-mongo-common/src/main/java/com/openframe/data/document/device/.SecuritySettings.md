Data class representing security configuration and status settings for a device.

## Key Components

| Field | Type | Description |
|---|---|---|
| `firewallEnabled` | `boolean` | Indicates whether the device firewall is active |
| `antivirusEnabled` | `boolean` | Indicates whether antivirus protection is active |
| `encryptionStatus` | `String` | Current encryption state of the device |
| `lastSecurityScan` | `String` | Timestamp or label of the most recent security scan |
| `securityPolicies` | `Map<String, String>` | Key-value pairs representing applied security policies |

Uses Lombok's `@Data` annotation to auto-generate getters, setters, `equals`, `hashCode`, and `toString`.

## Usage Example

```java
SecuritySettings settings = new SecuritySettings();
settings.setFirewallEnabled(true);
settings.setAntivirusEnabled(true);
settings.setEncryptionStatus("AES-256");
settings.setLastSecurityScan("2024-01-15T10:30:00Z");

Map<String, String> policies = new HashMap<>();
policies.put("passwordPolicy", "complex");
policies.put("sessionTimeout", "900");
settings.setSecurityPolicies(policies);
```
