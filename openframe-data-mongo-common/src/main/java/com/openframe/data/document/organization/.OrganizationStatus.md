Enum representing the lifecycle status of an organization within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `ACTIVE` | Organization is operational and accessible |
| `ARCHIVED` | Organization has been archived but data is retained |
| `DELETED` | Organization has been marked for deletion |

## Usage Example

```java
Organization org = organizationRepository.findById(orgId);

if (org.getStatus() == OrganizationStatus.ACTIVE) {
    // Process active organization
} else if (org.getStatus() == OrganizationStatus.ARCHIVED) {
    // Handle archived state
}
```
