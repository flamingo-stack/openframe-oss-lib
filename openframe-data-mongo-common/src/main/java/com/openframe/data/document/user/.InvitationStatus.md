Enum representing the lifecycle states of a user invitation within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `PENDING` | Invitation has been sent but not yet acted upon |
| `ACCEPTED` | Invitation was accepted by the recipient |
| `REVOKED` | Invitation was cancelled before acceptance |

## Usage Example

```java
InvitationStatus status = InvitationStatus.PENDING;

switch (status) {
    case PENDING  -> sendReminderEmail();
    case ACCEPTED -> activateUserAccount();
    case REVOKED  -> notifyAdminOfRevocation();
}
```
