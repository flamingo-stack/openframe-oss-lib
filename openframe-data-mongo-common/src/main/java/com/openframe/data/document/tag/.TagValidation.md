Utility class providing shared validation rules for tag keys and values used across the OpenFrame document tagging system.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `MAX_LENGTH` | `int` (64) | Maximum allowed length for both keys and values |
| `PATTERN` | `Pattern` | Regex enforcing alphanumeric start, followed by `[a-zA-Z0-9_-]` characters |
| `validateKey(String)` | `static void` | Validates a tag key — throws `IllegalArgumentException` if null, empty, too long, or pattern mismatch |
| `validateValue(String, String)` | `static void` | Validates a single tag value against the same rules, scoped to a key for error messaging |
| `validateValues(List<String>, String)` | `static void` | Iterates a list of values and delegates to `validateValue`; null list is a no-op |

## Validation Rules

- Must **not** be null or empty
- Must **start** with an alphanumeric character (`[a-zA-Z0-9]`)
- Subsequent characters may be alphanumeric, underscore (`_`), or hyphen (`-`)
- Maximum **64 characters** in length

## Usage Example

```java
// Validate a tag key
TagValidation.validateKey("environment");        // ✅ passes
TagValidation.validateKey("env-prod_1");         // ✅ passes
TagValidation.validateKey("_invalid");           // ❌ throws IllegalArgumentException

// Validate a single value
TagValidation.validateValue("production", "environment"); // ✅ passes
TagValidation.validateValue("", "environment");           // ❌ throws

// Validate a list of values
List<String> values = List.of("prod", "staging", "dev");
TagValidation.validateValues(values, "environment");      // ✅ all pass

// Null list is safely ignored
TagValidation.validateValues(null, "environment");        // ✅ no-op
```
