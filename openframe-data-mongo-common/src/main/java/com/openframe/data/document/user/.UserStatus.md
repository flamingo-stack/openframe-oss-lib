Enum defining the possible lifecycle states for authorization server users.

## Key Components

| Value | Description |
|-------|-------------|
| `ACTIVE` | User account is active and operational |
| `DELETED` | User account has been soft-deleted |

## Usage Example

```java
UserStatus status = user.getStatus();

if (status == UserStatus.ACTIVE) {
    // proceed with authentication
} else if (status == UserStatus.DELETED) {
    // reject login or redirect
}
```
