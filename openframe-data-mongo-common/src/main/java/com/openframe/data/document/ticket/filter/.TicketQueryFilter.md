A filter object for querying tickets, supporting multi-value filtering across statuses, organizations, assignees, labels, devices, creation sources, and date ranges.

## Key Components

| Field | Type | Description |
|---|---|---|
| `statuses` | `List<TicketStatus>` | Filter by one or more ticket statuses |
| `organizationIds` | `List<String>` | Filter by organization IDs |
| `assigneeIds` | `List<String>` | Filter by assignee IDs |
| `labelIds` | `List<String>` | Filter by label IDs |
| `deviceIds` | `List<String>` | Filter by associated device IDs |
| `creationSources` | `List<TicketCreationSource>` | Filter by how tickets were created |
| `createdAtFrom` | `Instant` | Start of creation date range |
| `createdAtTo` | `Instant` | End of creation date range |

Built with Lombok (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) following the same pattern as `MachineQueryFilter`.

## Usage Example

```java
TicketQueryFilter filter = TicketQueryFilter.builder()
    .statuses(List.of(TicketStatus.OPEN, TicketStatus.IN_PROGRESS))
    .organizationIds(List.of("org-123", "org-456"))
    .assigneeIds(List.of("user-789"))
    .creationSources(List.of(TicketCreationSource.EMAIL))
    .createdAtFrom(Instant.parse("2024-01-01T00:00:00Z"))
    .createdAtTo(Instant.parse("2024-12-31T23:59:59Z"))
    .build();

List<Ticket> tickets = ticketRepository.findAll(filter);
```
