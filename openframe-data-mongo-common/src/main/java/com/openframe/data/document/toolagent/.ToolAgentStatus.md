Enum representing the operational status of a Tool Agent, indicating whether it is active or inactive within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `ENABLED` | The Tool Agent is active and available for use |
| `DISABLED` | The Tool Agent is inactive and will not process requests |

## Usage Example

```java
ToolAgent agent = toolAgentRepository.findById(agentId);

if (agent.getStatus() == ToolAgentStatus.ENABLED) {
    agent.execute(payload);
} else {
    throw new IllegalStateException("Tool Agent is disabled and cannot process requests.");
}
```
