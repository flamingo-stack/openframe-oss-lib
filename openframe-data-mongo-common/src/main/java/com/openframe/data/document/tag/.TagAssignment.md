MongoDB document entity representing the assignment of a tag to a specific entity (device, ticket, etc.), stored in the `tag_assignments` collection with a unique compound index enforcing one assignment per entity-tag-type combination.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | MongoDB document `_id` |
| `entityId` | `String` | ID of the tagged entity |
| `tagId` | `String` | Reference to the associated `Tag` document |
| `entityType` | `TagEntityType` | Enum discriminating the entity kind (e.g., device, ticket) |
| `values` | `List<String>` | Per-entity values for key-value tags; empty for label-only tags |
| `taggedAt` | `Instant` | Timestamp of when the tag was assigned |
| `taggedBy` | `String` | User or system that performed the assignment |

**Compound Index:** `entity_tag_idx` on `(entityId, tagId, entityType)` — enforces uniqueness so the same tag cannot be assigned twice to the same entity.

## Usage Example

```java
// Assign a key-value tag to a device
TagAssignment assignment = TagAssignment.builder()
        .entityId("device-abc-123")
        .tagId("tag-site-456")
        .entityType(TagEntityType.DEVICE)
        .values(List.of("site1", "site2"))
        .taggedAt(Instant.now())
        .taggedBy("technician-user-id")
        .build();

mongoTemplate.save(assignment);

// Assign a simple label tag (no values)
TagAssignment labelAssignment = TagAssignment.builder()
        .entityId("ticket-789")
        .tagId("tag-urgent-001")
        .entityType(TagEntityType.TICKET)
        .values(List.of())
        .taggedAt(Instant.now())
        .taggedBy("mingo-ai")
        .build();
```
