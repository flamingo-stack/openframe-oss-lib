Enumeration of supported device categories used to classify managed endpoints within the OpenFrame platform.

## Key Components

| Value | Description |
|---|---|
| `DESKTOP` | Standard desktop workstations |
| `LAPTOP` | Portable laptop computers |
| `SERVER` | Physical server machines |
| `MOBILE_DEVICE` | Smartphones and handheld devices |
| `TABLET` | Tablet computers |
| `NETWORK_DEVICE` | Routers, switches, firewalls, and similar networking hardware |
| `IOT_DEVICE` | Internet of Things endpoints |
| `VIRTUAL_MACHINE` | Virtualized compute instances |
| `CONTAINER_HOST` | Hosts running containerized workloads |
| `OTHER` | Unclassified or custom device types |

## Usage Example

```java
Device device = new Device();
device.setType(DeviceType.SERVER);

if (device.getType() == DeviceType.VIRTUAL_MACHINE) {
    // apply virtualization-specific monitoring policies
}

// Filter devices by category
List<Device> networkDevices = devices.stream()
    .filter(d -> d.getType() == DeviceType.NETWORK_DEVICE)
    .collect(Collectors.toList());
```
