MongoDB document entity representing a file attachment linked to a knowledge base item, stored in the `knowledge_base_item_attachments` collection.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Auto-generated MongoDB document ID (`@Id`) |
| `itemId` | `String` | Indexed reference to the parent knowledge base item |
| `fileName` | `String` | Original name of the uploaded file |
| `storagePath` | `String` | Path to the file in the storage backend |
| `fileSize` | `Long` | Size of the file in bytes |
| `contentType` | `String` | MIME type of the file (e.g., `application/pdf`) |
| `uploadedBy` | `String` | Identifier of the user who uploaded the file |
| `createdAt` | `Instant` | Auto-populated timestamp via `@CreatedDate` |

## Usage Example

```java
KnowledgeBaseItemAttachment attachment = KnowledgeBaseItemAttachment.builder()
    .itemId("kb-item-001")
    .fileName("network-diagram.pdf")
    .storagePath("/storage/kb/attachments/network-diagram.pdf")
    .fileSize(204800L)
    .contentType("application/pdf")
    .uploadedBy("technician-42")
    .build();
```

> The `itemId` field is indexed for efficient lookup of all attachments belonging to a specific knowledge base item. The `createdAt` field is automatically set by Spring Data MongoDB's auditing mechanism — no manual assignment needed.
