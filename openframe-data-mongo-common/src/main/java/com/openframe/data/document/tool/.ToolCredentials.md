Simple data container for storing authentication credentials associated with a tool integration, supporting both username/password and API key authentication methods.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `username` | `String` | Username for credential-based auth |
| `password` | `String` | Password for credential-based auth |
| `apiKey` | `ToolApiKey` | API key object for token-based auth |

> Lombok's `@Data` annotation auto-generates getters, setters, `equals()`, `hashCode()`, and `toString()` at compile time.

## Usage Example

```java
ToolCredentials credentials = new ToolCredentials();

// Username/password auth
credentials.setUsername("admin");
credentials.setPassword("s3cur3P@ss");

// API key auth
ToolApiKey apiKey = new ToolApiKey();
apiKey.setValue("sk-abc123");
credentials.setApiKey(apiKey);

// Typically embedded within a Tool document
Tool tool = new Tool();
tool.setCredentials(credentials);
```
