MongoDB document entity representing an integrated third-party tool within a tenant's OpenFrame environment, mapping tool metadata, credentials, layer configuration, and monitoring settings to the `integrated_tools` collection.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id` | `String` | MongoDB document identifier (`@Id`) |
| `tenantId` | `String` | Tenant scope identifier (implements `TenantScoped`) |
| `key` | `String` | Unique tool key per tenant (compound index with `tenantId`) |
| `name` / `description` / `icon` | `String` | Display metadata |
| `toolUrls` | `List<ToolUrl>` | Endpoint URLs associated with the tool |
| `type` / `toolType` / `category` / `platformCategory` | `String` | Classification fields |
| `enabled` | `boolean` | Activation state of the integration |
| `credentials` | `ToolCredentials` | Auth/credential configuration |
| `layer` / `layerOrder` / `layerColor` | `String` / `Integer` | UI layer rendering metadata |
| `metricsPath` / `healthCheckEndpoint` | `String` | Monitoring targets |
| `healthCheckInterval` / `connectionTimeout` / `readTimeout` | `Integer` | Polling and timeout settings (ms) |
| `allowedEndpoints` | `String[]` | Permitted proxy/routing endpoints |
| `debeziumConnectors` | `Object[]` | CDC connector definitions |

## Usage Example

```java
IntegratedTool tool = IntegratedTool.builder()
    .tenantId("tenant-abc")
    .key("zabbix")
    .name("Zabbix Monitoring")
    .type("monitoring")
    .toolType("zabbix")
    .category("infrastructure")
    .enabled(true)
    .healthCheckEndpoint("/api/v1/health")
    .healthCheckInterval(30)
    .connectionTimeout(5000)
    .readTimeout(10000)
    .credentials(new ToolCredentials("admin", "secret"))
    .build();
```

> The compound index `tenant_key_idx` on `{tenantId, key}` enforces uniqueness of tool keys per tenant while `sparse = true` excludes documents where either field is null.
