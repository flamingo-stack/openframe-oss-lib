MongoDB document entity representing a tag definition used to categorize and label entities within a tenant's OpenFrame environment.

## Key Components

| Component | Type | Description |
|---|---|---|
| `id` | `String` | MongoDB document identifier (`@Id`) |
| `key` | `String` | Unique tag identifier per entity type (e.g., `"site"`, `"org_type"`) |
| `description` | `String` | Human-readable description of the tag |
| `color` | `String` | Optional display color for UI rendering |
| `values` | `List<String>` | Predefined allowed values; `null` = free-text, non-empty = dropdown |
| `entityType` | `TagEntityType` | The entity type this tag applies to |
| `createdAt` | `Instant` | Timestamp of tag creation |
| `createdBy` | `String` | Identifier of the user who created the tag |

### Compound Index

A unique compound index (`key_entity_idx`) is enforced on `key` + `entityType`, ensuring tag keys are unique per entity type across the tenant.

### `values` Field Behavior

| State | UI Behavior |
|---|---|
| `null` | Free-text input or simple label |
| Empty `List` | Values explicitly cleared |
| Non-empty `List` | Renders as dropdown or checkbox selection |

## Usage Example

```java
Tag siteTag = Tag.builder()
    .key("site")
    .description("Physical site location")
    .color("#FFC008")
    .values(List.of("CHICAGO", "DALLAS", "SCHOOL1"))
    .entityType(TagEntityType.DEVICE)
    .createdAt(Instant.now())
    .createdBy("user-123")
    .build();
```
