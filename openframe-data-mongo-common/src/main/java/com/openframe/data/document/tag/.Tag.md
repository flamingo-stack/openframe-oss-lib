MongoDB document entity representing a tenant-scoped tag definition, used to categorize and label entities within the OpenFrame platform.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id` | `String` | MongoDB document `_id` |
| `tenantId` | `String` | Tenant isolation identifier (from `TenantScoped`) |
| `key` | `String` | Tag key identifier (e.g., `"site"`, `"org_type"`) |
| `description` | `String` | Human-readable tag description |
| `color` | `String` | Optional display color |
| `values` | `List<String>` | Predefined allowed values; `null` = free-text, non-empty = dropdown |
| `entityType` | `TagEntityType` | The entity type this tag applies to |
| `createdAt` | `Instant` | Creation timestamp |
| `createdBy` | `String` | Creator reference |

**Compound Index:** `tenant_key_entity_idx` enforces uniqueness of `(tenantId, key, entityType)` — a tag key is unique per entity type within a tenant.

**`values` field behavior:**
- `null` → simple label or free-text input
- `[]` (empty list) → values explicitly cleared
- `["A", "B"]` → renders as dropdown/checkbox selector in UI

## Usage Example

```java
Tag siteTag = Tag.builder()
    .tenantId("tenant-123")
    .key("site")
    .description("Physical site location")
    .color("#FFC008")
    .values(List.of("CHICAGO", "NEW_YORK", "DALLAS"))
    .entityType(TagEntityType.DEVICE)
    .createdAt(Instant.now())
    .createdBy("user-456")
    .build();
```
