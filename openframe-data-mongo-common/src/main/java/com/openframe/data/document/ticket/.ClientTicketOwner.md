Represents a client-type ticket owner in the OpenFrame ticketing system, extending the base `TicketOwner` with a machine identifier.

## Key Components

| Element | Description |
|---|---|
| `machineId` | Identifier of the client machine associated with the ticket |
| `type` | Automatically set to `TicketOwnerType.CLIENT` on construction |
| Extends | `TicketOwner` base class |

## Usage Example

```java
// Create a client ticket owner linked to a specific machine
ClientTicketOwner owner = new ClientTicketOwner("machine-abc-123");

// Access inherited and extended fields
TicketOwnerType type = owner.getType();       // TicketOwnerType.CLIENT
String machineId = owner.getMachineId();      // "machine-abc-123"

// Default no-args constructor (type must be set manually if needed)
ClientTicketOwner emptyOwner = new ClientTicketOwner();
```
