Configuration document for an integrated tool agent, defining deployment metadata, session behavior, command lifecycle arguments, and asset management for a managed tool agent within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for this configuration |
| `toolId` | `String` | Reference to the associated tool |
| `releaseVersion` | `boolean` | Flags whether this is a release version |
| `version` | `String` | Version string of the agent |
| `sessionType` | `SessionType` | Defines the session behavior type |
| `downloadConfigurations` | `List<DownloadConfiguration>` | Sources and settings for downloading agent artifacts |
| `assets` | `List<ToolAgentAsset>` | Static assets bundled with the agent |
| `installationCommandArgs` | `List<String>` | CLI args used during agent installation |
| `runCommandArgs` | `List<String>` | CLI args used to start the agent |
| `agentToolIdCommandArgs` | `List<String>` | CLI args for identifying the agent tool at runtime |
| `uninstallationCommandArgs` | `List<String>` | CLI args used during agent removal |
| `status` | `ToolAgentStatus` | Current lifecycle status of the agent |

## Usage Example

```java
IntegratedToolAgentConfiguration config = new IntegratedToolAgentConfiguration();
config.setId("agent-001");
config.setToolId("tool-ninja-rmm");
config.setReleaseVersion(true);
config.setVersion("2.1.0");
config.setSessionType(SessionType.PERSISTENT);
config.setStatus(ToolAgentStatus.ACTIVE);

config.setInstallationCommandArgs(List.of("--silent", "--target=/opt/agent"));
config.setRunCommandArgs(List.of("--mode=daemon", "--log-level=info"));
config.setUninstallationCommandArgs(List.of("--purge", "--silent"));

config.setDownloadConfigurations(List.of(downloadConfig));
config.setAssets(List.of(agentAsset));
```
