A data model representing the health metrics and status information for a monitored device.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `cpuUsage` | `double` | Current CPU utilization percentage |
| `memoryUsage` | `double` | Current memory utilization percentage |
| `diskUsage` | `double` | Current disk utilization percentage |
| `activeProcesses` | `List<String>` | Names of currently running processes |
| `alerts` | `List<Alert>` | Active alerts associated with the device |

> Uses Lombok's `@Data` annotation to auto-generate getters, setters, `equals`, `hashCode`, and `toString`.

## Usage Example

```java
DeviceHealth health = new DeviceHealth();
health.setCpuUsage(72.5);
health.setMemoryUsage(58.3);
health.setDiskUsage(45.0);
health.setActiveProcesses(List.of("nginx", "postgres", "java"));

Alert alert = new Alert();
// configure alert...
health.setAlerts(List.of(alert));

double cpu = health.getCpuUsage(); // 72.5
```
