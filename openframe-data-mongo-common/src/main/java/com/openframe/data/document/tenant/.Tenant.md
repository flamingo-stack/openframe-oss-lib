Represents a tenant entity in OpenFrame's multi-tenant architecture, where each tenant corresponds to an organization or company using the platform. Extends `CoreTenant` with ownership, status, and audit timestamp fields.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `ownerId` | `String` | ID of the first registered user, designated as the tenant owner |
| `status` | `TenantStatus` | Lifecycle status of the tenant; defaults to `ACTIVE` |
| `createdAt` | `LocalDateTime` | Auto-populated creation timestamp via `@CreatedDate` |
| `updatedAt` | `LocalDateTime` | Auto-populated modification timestamp via `@LastModifiedDate` |
| `generateTenantId()` | `static String` | Generates a new UUID-based tenant identifier |
| `isActive()` | `boolean` | Returns `true` if the tenant status is `ACTIVE` |

## Usage Example

```java
// Create a new tenant
Tenant tenant = Tenant.builder()
    .ownerId("user-123")
    .status(TenantStatus.ACTIVE)
    .build();

// Generate a unique tenant ID
String tenantId = Tenant.generateTenantId();

// Check tenant status before processing
if (tenant.isActive()) {
    // proceed with tenant operations
}

// Clone and update an existing tenant
Tenant updated = tenant.toBuilder()
    .status(TenantStatus.SUSPENDED)
    .build();
```

> **Note:** `@SuperBuilder` is used to support builder inheritance from `CoreTenant`. Audit timestamps (`createdAt`, `updatedAt`) are managed automatically by Spring Data — no manual assignment is required.
