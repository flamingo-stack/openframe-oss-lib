A simple configuration data class that holds local filename metadata, pairing a filename with its associated operating system identifier.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `filename` | `String` | The local file's name or path |
| `os` | `String` | The operating system associated with the file |

> Lombok's `@Data` annotation auto-generates `getters`, `setters`, `equals`, `hashCode`, and `toString` at compile time — no boilerplate required.

## Usage Example

```java
LocalFilenameConfiguration config = new LocalFilenameConfiguration();
config.setFilename("agent-config.json");
config.setOs("linux");

String file = config.getFilename(); // "agent-config.json"
String os   = config.getOs();       // "linux"
```
