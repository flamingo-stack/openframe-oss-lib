Defines the recipient classification for notifications within the OpenFrame notification system, distinguishing between human users and automated machine consumers.

## Key Components

| Value | Description |
|-------|-------------|
| `USER` | Notification targets a human user |
| `MACHINE` | Notification targets an automated system or machine process |

## Usage Example

```java
NotificationDocument notification = new NotificationDocument();
notification.setRecipientType(RecipientType.USER);

// Route based on recipient type
if (notification.getRecipientType() == RecipientType.MACHINE) {
    // Handle automated/system notification
} else {
    // Handle user-facing notification
}
```
