Defines a filter object for querying users by email pattern, name pattern, and account status.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `emailRegex` | `String` | Regex pattern to match against user email addresses |
| `nameRegex` | `String` | Regex pattern to match against user names |
| `status` | `UserStatus` | Filter by user account status (e.g., active, disabled) |

## Usage Example

```java
// Filter active users whose email contains "@acme.com"
UserQueryFilter filter = UserQueryFilter.builder()
    .emailRegex(".*@acme\\.com")
    .status(UserStatus.ACTIVE)
    .build();

// Filter by name pattern only
UserQueryFilter nameFilter = UserQueryFilter.builder()
    .nameRegex("^John.*")
    .build();
```

> All fields are optional — omitted fields are ignored during query execution, allowing partial or combined filter criteria.
