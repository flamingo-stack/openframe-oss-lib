Enum defining the severity levels available for notifications within the OpenFrame notification system.

## Key Components

| Value | Description |
|-------|-------------|
| `INFO` | Informational notifications with no urgency |
| `WARNING` | Notifications requiring attention but not critical |
| `DANGER` | Critical notifications requiring immediate action |

## Usage Example

```java
Notification notification = new Notification();
notification.setSeverity(NotificationSeverity.DANGER);
```
