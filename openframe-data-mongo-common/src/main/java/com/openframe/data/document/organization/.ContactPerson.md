Represents a contact person embedded within an organization document in the OpenFrame data model. This is a MongoDB embedded document (not a standalone collection) used to store key contact details for organizational records.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `contactName` | `String` | Full name of the contact person |
| `title` | `String` | Job title or position |
| `phone` | `String` | Phone number |
| `email` | `String` | Email address |

Lombok annotations (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Builder pattern
ContactPerson contact = ContactPerson.builder()
    .contactName("Jane Smith")
    .title("IT Manager")
    .phone("+1-555-123-4567")
    .email("jane.smith@example.com")
    .build();

// Embedded within an organization document
Organization org = Organization.builder()
    .name("Acme Corp")
    .primaryContact(contact)
    .build();
```
