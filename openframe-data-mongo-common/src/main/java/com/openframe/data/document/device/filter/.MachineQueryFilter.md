A filter model for querying machine/device records, encapsulating multiple optional filter criteria that can be applied when searching or listing devices in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|---|---|---|
| `statuses` | `List<String>` | Filter by device status values (e.g., online, offline) |
| `deviceTypes` | `List<String>` | Filter by device type classifications |
| `osTypes` | `List<String>` | Filter by operating system types |
| `organizationIds` | `List<String>` | Filter by organization/tenant identifiers |

Lombok annotations `@Data`, `@NoArgsConstructor`, and `@AllArgsConstructor` auto-generate getters, setters, and constructors.

## Usage Example

```java
// Build a filter for active Windows workstations in a specific org
MachineQueryFilter filter = new MachineQueryFilter(
    List.of("online"),
    List.of("workstation"),
    List.of("windows"),
    List.of("org-123", "org-456")
);

// Or use the no-args constructor and set selectively
MachineQueryFilter filter = new MachineQueryFilter();
filter.setOsTypes(List.of("linux", "macos"));
filter.setOrganizationIds(List.of("org-789"));
```
