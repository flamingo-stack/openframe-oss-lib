Defines the lifecycle status values for tenants within the OpenFrame authorization server.

## Key Components

| Value | Description |
|-------|-------------|
| `ACTIVE` | Tenant is operational and authorized |
| `INACTIVE` | Tenant exists but is currently disabled |
| `DELETED` | Tenant has been soft-deleted |

## Usage Example

```java
TenantDocument tenant = tenantRepository.findById(tenantId);

if (tenant.getStatus() == TenantStatus.ACTIVE) {
    // Allow access
} else if (tenant.getStatus() == TenantStatus.INACTIVE) {
    throw new TenantDisabledException("Tenant is inactive");
} else if (tenant.getStatus() == TenantStatus.DELETED) {
    throw new TenantNotFoundException("Tenant no longer exists");
}
```
