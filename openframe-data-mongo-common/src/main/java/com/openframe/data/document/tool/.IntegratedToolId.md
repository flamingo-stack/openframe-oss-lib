Enum representing the unique string identifiers for integrated MSP tools supported by the OpenFrame platform.

## Key Components

| Member | Value | Description |
|--------|-------|-------------|
| `FLEET_SERVER_ID` | `"fleetmdm-server"` | Identifier for the Fleet MDM server integration |

## Usage Example

```java
String toolId = IntegratedToolId.FLEET_SERVER_ID.getValue();
// toolId == "fleetmdm-server"

if (tool.getId().equals(IntegratedToolId.FLEET_SERVER_ID.getValue())) {
    // handle Fleet MDM logic
}
```

## Notes

- Uses Lombok `@Getter` to auto-generate `getValue()` for each enum constant.
- String values are stable identifiers used to reference tool configurations across the OpenFrame platform (e.g., persisted in documents or matched in routing logic).
