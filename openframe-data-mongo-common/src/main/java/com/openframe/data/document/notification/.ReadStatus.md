Enum representing the read/lifecycle state of a notification document.

## Key Components

| Value | Description |
|-------|-------------|
| `UNREAD` | Notification has not yet been viewed by the recipient |
| `READ` | Notification has been viewed by the recipient |
| `DELETED` | Notification has been soft-deleted |

## Usage Example

```java
NotificationDocument notification = notificationRepository.findById(id);

if (notification.getReadStatus() == ReadStatus.UNREAD) {
    notification.setReadStatus(ReadStatus.READ);
    notificationRepository.save(notification);
}

// Soft-delete a notification
notification.setReadStatus(ReadStatus.DELETED);
```
