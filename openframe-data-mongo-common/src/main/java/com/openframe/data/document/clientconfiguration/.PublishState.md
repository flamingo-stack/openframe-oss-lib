Represents the publication state of a client configuration document, tracking whether it has been successfully published and how many publish attempts have been made.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `published` | `boolean` | Indicates if the configuration has been successfully published |
| `attempts` | `int` | Number of publish attempts made |
| `pending()` | Static factory | Creates an initial unpublished state with zero attempts |
| `nonPublished(current)` | Static factory | Creates a failed-publish state, incrementing the attempt counter from the previous state |
| `published()` | Static factory | Creates a successful publish state, resetting attempts to zero |

## Usage Example

```java
// Initial state before any publish attempt
PublishState state = PublishState.pending();
// published=false, attempts=0

// After a failed publish attempt
state = PublishState.nonPublished(state);
// published=false, attempts=1

// Retry after another failure
state = PublishState.nonPublished(state);
// published=false, attempts=2

// Successfully published
state = PublishState.published();
// published=true, attempts=0

// Safe to pass null as current state
PublishState fromNull = PublishState.nonPublished(null);
// published=false, attempts=1
```
