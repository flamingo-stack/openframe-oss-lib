Enum representing the possible lifecycle and operational states of a device within the OpenFrame platform.

## Key Components

| Constant | Description |
|---|---|
| `ACTIVE` | Device is operational and in use |
| `PENDING` | Device is awaiting provisioning or approval |
| `INACTIVE` | Device is registered but not currently in use |
| `MAINTENANCE` | Device is undergoing maintenance |
| `DECOMMISSIONED` | Device has been formally retired from service |
| `ONLINE` | Device is connected and reachable |
| `OFFLINE` | Device is unreachable or disconnected |
| `DELETED` | Device has been removed |
| `ARCHIVED` | Device record is retained for historical purposes |

## Usage Example

```java
Device device = deviceRepository.findById(deviceId);

if (device.getStatus() == DeviceStatus.OFFLINE) {
    alertService.notify("Device is offline: " + device.getName());
}

device.setStatus(DeviceStatus.MAINTENANCE);
deviceRepository.save(device);
```
