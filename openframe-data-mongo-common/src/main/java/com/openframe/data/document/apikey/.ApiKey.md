MongoDB document entity representing an API key with tenant scoping, permission management, and lifecycle state tracking.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `keyId` | `String` | Primary identifier (`@Id`) for the API key document |
| `tenantId` | `String` | Indexed tenant reference (implements `TenantScoped`) |
| `hashedKey` | `String` | Stored hash of the raw API key secret |
| `userId` | `String` | Indexed owner/creator of the key |
| `scopes` / `roles` | `List<String>` | Permission grants (TODO: enforcement pending) |
| `enabled` | `boolean` | Manual activation flag, defaults to `true` |
| `expiresAt` | `Instant` | Optional expiration timestamp |
| `isExpired()` | `boolean` | Returns `true` if `expiresAt` is set and in the past |
| `isActive()` | `boolean` | Returns `true` only when `enabled && !isExpired()` |

## Usage Example

```java
// Create a new API key document
ApiKey apiKey = ApiKey.builder()
    .tenantId("tenant-abc")
    .userId("user-123")
    .hashedKey(hashService.hash(rawSecret))
    .name("CI Integration Key")
    .description("Used by GitHub Actions pipeline")
    .scopes(List.of("tickets:read", "tickets:write"))
    .roles(List.of("technician"))
    .enabled(true)
    .createdAt(Instant.now())
    .expiresAt(Instant.now().plus(365, ChronoUnit.DAYS))
    .build();

// Lifecycle checks
if (apiKey.isActive()) {
    // proceed with authenticated request
}

if (apiKey.isExpired()) {
    // reject or prompt renewal
}
```

> **Note:** `hashedKey` stores only the hash — the raw key is never persisted. Scope and role enforcement is marked TODO and not yet implemented.
