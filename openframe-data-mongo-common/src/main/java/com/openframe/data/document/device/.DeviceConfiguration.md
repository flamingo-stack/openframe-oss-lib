Represents the configuration state of a managed device, encapsulating settings, software inventory, network configuration, and security policies as a structured document.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `settings` | `Map<String, String>` | General device key-value configuration settings |
| `installedSoftware` | `List<String>` | Inventory of software installed on the device |
| `networkConfig` | `Map<String, String>` | Network-related configuration parameters |
| `security` | `SecuritySettings` | Nested security policy and settings object |

> Uses Lombok `@Data` for automatic generation of getters, setters, `equals`, `hashCode`, and `toString`.

## Usage Example

```java
DeviceConfiguration config = new DeviceConfiguration();

config.setSettings(Map.of(
    "os", "Windows 11",
    "timezone", "UTC-5"
));

config.setInstalledSoftware(List.of(
    "Microsoft Office 365",
    "Cisco AnyConnect",
    "CrowdStrike Falcon"
));

config.setNetworkConfig(Map.of(
    "ipAddress", "192.168.1.100",
    "gateway",   "192.168.1.1"
));

SecuritySettings sec = new SecuritySettings();
// configure sec...
config.setSecurity(sec);
```
