Enum representing the possible connection states for a tool or data source within the OpenFrame platform.

## Key Components

| Value | Description |
|---|---|
| `CONNECTED` | The tool or data source is successfully connected and operational |
| `DISCONNECTED` | The tool or data source is not currently connected |
| `ERROR` | The connection encountered an error state |

## Usage Example

```java
ConnectionStatus status = ConnectionStatus.CONNECTED;

switch (status) {
    case CONNECTED:
        // Handle active connection
        break;
    case DISCONNECTED:
        // Attempt reconnection
        break;
    case ERROR:
        // Log and handle error state
        break;
}
```
