Enum representing the lifecycle states of an integrated tool installation task within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `PROCESSING` | The installation task is currently in progress |
| `PROCESSED` | The installation task has completed successfully |
| `FAILED` | The installation task encountered an error and did not complete |

## Usage Example

```java
IntegratedToolInstallationTaskStatus status = IntegratedToolInstallationTaskStatus.PROCESSING;

switch (status) {
    case PROCESSING:
        // Handle in-progress installation
        break;
    case PROCESSED:
        // Handle successful completion
        break;
    case FAILED:
        // Handle installation failure
        break;
}
```
