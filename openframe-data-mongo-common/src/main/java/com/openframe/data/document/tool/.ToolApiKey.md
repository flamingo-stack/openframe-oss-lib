Represents a tool-level API key configuration, encapsulating the key value, its type, and an associated name.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `key` | `String` | The raw API key value |
| `type` | `APIKeyType` | Enum classifying the API key (e.g., header, bearer) |
| `keyName` | `String` | Human-readable label or parameter name for the key |

## Usage Example

```java
ToolApiKey toolApiKey = new ToolApiKey();
toolApiKey.setKey("sk-abc123");
toolApiKey.setType(APIKeyType.BEARER);
toolApiKey.setKeyName("Authorization");
```
