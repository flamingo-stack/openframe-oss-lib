Enum representing the origin channel through which a ticket was created, automatically populated by the system based on the creation flow.

## Key Components

| Constant | Description |
|---|---|
| `FAE_FORM` | Ticket submitted via the Fae client-facing form |
| `FAE_DIALOG` | Ticket created through a Fae conversational dialog |
| `ADMIN_DASHBOARD` | Ticket created directly from the admin dashboard |

## Usage Example

```java
Ticket ticket = new Ticket();
ticket.setCreationSource(TicketCreationSource.FAE_FORM);

// Filter tickets by source
if (ticket.getCreationSource() == TicketCreationSource.ADMIN_DASHBOARD) {
    // apply admin-specific handling
}
```
