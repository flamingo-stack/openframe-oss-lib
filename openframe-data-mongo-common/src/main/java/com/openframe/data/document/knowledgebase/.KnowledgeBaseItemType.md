An enum defining the two possible content types within the OpenFrame knowledge base system.

## Key Components

| Value | Description |
|-------|-------------|
| `FOLDER` | Represents a container/directory for organizing knowledge base content |
| `ARTICLE` | Represents an individual knowledge base document or article |

## Usage Example

```java
KnowledgeBaseItem item = new KnowledgeBaseItem();
item.setType(KnowledgeBaseItemType.ARTICLE);

if (item.getType() == KnowledgeBaseItemType.FOLDER) {
    // handle folder logic
} else {
    // handle article rendering
}
```
