Enum representing the security compliance status of a device within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `COMPLIANT` | Device meets all security policy requirements |
| `NON_COMPLIANT` | Device violates one or more security policies |
| `UNKNOWN` | Device compliance status cannot be determined |

## Usage Example

```java
Device device = deviceRepository.findById(deviceId);

if (device.getSecurityState() == SecurityState.COMPLIANT) {
    log.info("Device {} passed security check", deviceId);
} else if (device.getSecurityState() == SecurityState.NON_COMPLIANT) {
    alertService.triggerComplianceAlert(device);
} else {
    log.warn("Security state unknown for device {}", deviceId);
}
```
