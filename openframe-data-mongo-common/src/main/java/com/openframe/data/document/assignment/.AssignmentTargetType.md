Enum defining the possible entity types that can be targeted by an assignment operation within the OpenFrame data model.

## Key Components

| Value | Description |
|---|---|
| `ORGANIZATION` | Targets an organization entity |
| `DEVICE` | Targets a device entity |
| `TICKET` | Targets a support ticket entity |
| `KNOWLEDGE_ARTICLE` | Targets a knowledge base article entity |

## Usage Example

```java
AssignmentDocument assignment = new AssignmentDocument();
assignment.setTargetType(AssignmentTargetType.TICKET);

if (assignment.getTargetType() == AssignmentTargetType.DEVICE) {
    // handle device-specific assignment logic
}
```
