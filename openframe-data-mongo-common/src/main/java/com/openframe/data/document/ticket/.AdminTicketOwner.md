Represents a ticket owner of type `ADMIN`, extending the base `TicketOwner` class with a specific user identifier.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `userId` | `String` | The unique identifier of the admin user who owns the ticket |
| `type` | `TicketOwnerType` | Inherited field, automatically set to `TicketOwnerType.ADMIN` on construction |

## Usage Example

```java
// Create an admin ticket owner with a specific user ID
AdminTicketOwner owner = new AdminTicketOwner("admin-user-123");

// Access inherited and specific fields
TicketOwnerType type = owner.getType();   // TicketOwnerType.ADMIN
String userId = owner.getUserId();         // "admin-user-123"

// No-arg constructor (type must be set manually if used)
AdminTicketOwner emptyOwner = new AdminTicketOwner();
```

## Notes

- Extends `TicketOwner` and uses Lombok's `@EqualsAndHashCode(callSuper = true)` to include parent fields in equality checks
- The `type` field is always initialized to `TicketOwnerType.ADMIN` when using the parameterized constructor, ensuring type safety across the ticket ownership model
- The `@NoArgsConstructor` annotation supports deserialization frameworks such as Jackson or Hibernate
