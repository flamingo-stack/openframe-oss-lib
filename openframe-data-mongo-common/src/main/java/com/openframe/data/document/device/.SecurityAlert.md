A plain data model representing a security alert associated with a device, used for storing and transferring security event information within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the alert |
| `severity` | `String` | Severity level of the alert (e.g., `LOW`, `HIGH`, `CRITICAL`) |
| `description` | `String` | Human-readable description of the security event |
| `detectedAt` | `Instant` | Timestamp when the alert was first detected |
| `resolved` | `boolean` | Whether the alert has been resolved |
| `resolvedAt` | `Instant` | Timestamp when the alert was resolved |
| `resolution` | `String` | Description of how the alert was resolved |

Annotated with Lombok `@Data`, which auto-generates getters, setters, `equals()`, `hashCode()`, and `toString()` at compile time.

## Usage Example

```java
SecurityAlert alert = new SecurityAlert();
alert.setId("alert-001");
alert.setSeverity("HIGH");
alert.setDescription("Unauthorized SSH login attempt detected");
alert.setDetectedAt(Instant.now());
alert.setResolved(false);

// Later, when resolved:
alert.setResolved(true);
alert.setResolvedAt(Instant.now());
alert.setResolution("Blocked IP and rotated SSH keys");
```
