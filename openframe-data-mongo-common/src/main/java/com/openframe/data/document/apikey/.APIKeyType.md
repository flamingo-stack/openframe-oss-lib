Enum representing the authentication method used when attaching an API key to outbound requests.

## Key Components

| Value | Description |
|---|---|
| `HEADER` | API key is sent as a custom HTTP header |
| `BEARER_TOKEN` | API key is sent as an `Authorization: Bearer` token |
| `NONE` | No API key authentication is applied |

## Usage Example

```java
APIKeyDocument apiKey = new APIKeyDocument();
apiKey.setType(APIKeyType.BEARER_TOKEN);

switch (apiKey.getType()) {
    case HEADER -> request.addHeader("X-API-Key", apiKey.getValue());
    case BEARER_TOKEN -> request.addHeader("Authorization", "Bearer " + apiKey.getValue());
    case NONE -> { /* no auth applied */ }
}
```
