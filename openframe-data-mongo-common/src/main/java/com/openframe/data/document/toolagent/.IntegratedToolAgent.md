MongoDB document entity representing a tenant-scoped integrated tool agent configuration, mapping agent lifecycle commands, assets, and deployment state within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | MongoDB document identifier (`@Id`) |
| `tenantId` | `String` | Tenant isolation key (part of compound unique index with `key`) |
| `key` | `String` | Unique agent identifier per tenant |
| `documentVersion` | `Long` | Optimistic locking version (`@Version`) |
| `toolId` | `String` | Reference to the parent tool |
| `releaseVersion` | `boolean` | Flags whether this is a release-grade version |
| `sessionType` | `SessionType` | Defines the agent session behavior |
| `downloadConfigurations` | `List<DownloadConfiguration>` | Download sources and options for the agent |
| `assets` | `List<ToolAgentAsset>` | Associated binary/resource assets |
| `installationCommandArgs` | `List<String>` | CLI args for agent installation |
| `runCommandArgs` | `List<String>` | CLI args to start the agent |
| `agentToolIdCommandArgs` | `List<String>` | CLI args for tool identity injection |
| `uninstallationCommandArgs` | `List<String>` | CLI args for agent removal |
| `status` | `ToolAgentStatus` | Current runtime/deployment status |
| `publishState` | `PublishState` | Publication lifecycle state |

## Usage Example

```java
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setTenantId("tenant-abc");
agent.setKey("rmm-agent-v2");
agent.setToolId("tool-001");
agent.setVersion("2.1.0");
agent.setReleaseVersion(true);
agent.setSessionType(SessionType.PERSISTENT);
agent.setInstallationCommandArgs(List.of("--silent", "--target=/opt/agent"));
agent.setRunCommandArgs(List.of("--daemon", "--port=9090"));
agent.setUninstallationCommandArgs(List.of("--purge"));
agent.setStatus(ToolAgentStatus.ACTIVE);
agent.setPublishState(PublishState.PUBLISHED);

mongoTemplate.save(agent);
```

> The compound index on `tenantId` + `key` enforces uniqueness per tenant while `sparse = true` allows null keys without index conflicts.
