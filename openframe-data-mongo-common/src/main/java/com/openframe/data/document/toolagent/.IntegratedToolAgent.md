Represents a MongoDB document entity for a versioned, integrated tool agent within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | MongoDB document identifier (`@Id`) |
| `documentVersion` | `Long` | Optimistic locking version (`@Version`) |
| `toolId` | `String` | Reference to the associated tool |
| `releaseVersion` | `boolean` | Flags whether this is a release version |
| `version` | `String` | Version string for the agent |
| `sessionType` | `SessionType` | Defines the session behavior type |
| `downloadConfigurations` | `List<DownloadConfiguration>` | Download settings for the agent |
| `assets` | `List<ToolAgentAsset>` | Binary or resource assets bundled with the agent |
| `installationCommandArgs` | `List<String>` | CLI args used during agent installation |
| `runCommandArgs` | `List<String>` | CLI args used to run the agent |
| `agentToolIdCommandArgs` | `List<String>` | CLI args for agent tool identification |
| `uninstallationCommandArgs` | `List<String>` | CLI args used during agent removal |
| `status` | `ToolAgentStatus` | Current lifecycle status of the agent |
| `publishState` | `PublishState` | Publication state (e.g. draft, published) |

## Usage Example

```java
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setToolId("tool-abc-123");
agent.setVersion("1.2.0");
agent.setReleaseVersion(true);
agent.setSessionType(SessionType.PERSISTENT);
agent.setStatus(ToolAgentStatus.ACTIVE);
agent.setPublishState(PublishState.PUBLISHED);

agent.setInstallationCommandArgs(List.of("--install", "--silent"));
agent.setRunCommandArgs(List.of("--start", "--config=/etc/agent.conf"));
agent.setUninstallationCommandArgs(List.of("--uninstall", "--purge"));

// Persist via Spring Data MongoDB repository
integratedToolAgentRepository.save(agent);
```

> Stored in the `integrated_tool_agents` MongoDB collection. Uses Spring Data's `@Version` for optimistic concurrency control, preventing conflicting concurrent updates to the same agent document.
