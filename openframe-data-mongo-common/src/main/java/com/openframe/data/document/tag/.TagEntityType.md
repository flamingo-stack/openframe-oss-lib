Enum representing the types of entities that can be tagged within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `DEVICE` | Tags applied to managed devices |
| `TICKET` | Tags applied to support tickets |
| `KNOWLEDGE_ARTICLE` | Tags applied to knowledge base articles |

## Usage Example

```java
Tag tag = new Tag();
tag.setEntityType(TagEntityType.DEVICE);

// Filter tags by entity type
List<Tag> deviceTags = tagRepository.findByEntityType(TagEntityType.DEVICE);
```
