Enum defining the supported operating-system platforms for RMM script execution targeting.

## Key Components

| Value | Description |
|-------|-------------|
| `WINDOWS` | Microsoft Windows environments |
| `LINUX` | Linux-based operating systems |
| `MACOS` | Apple macOS environments |

## Usage Example

```java
Script script = new Script();
script.setPlatform(ScriptPlatform.LINUX);

// Filter scripts by platform
List<Script> linuxScripts = scripts.stream()
    .filter(s -> s.getPlatform() == ScriptPlatform.LINUX)
    .toList();
```

> **Note:** Platform targeting is intentionally coarse-grained. Version-specific or distro-specific logic should be handled within the script body or via runtime arguments rather than additional enum values.
