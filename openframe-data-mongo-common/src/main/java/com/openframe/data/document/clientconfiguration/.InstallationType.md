Enum representing the supported installation types for a client configuration, used to distinguish how an OpenFrame component is deployed or run.

## Key Components

| Value | Description |
|-------|-------------|
| `STANDARD` | Default installation mode |
| `GUI_APP` | Deployed as a graphical user interface application |
| `SERVICE` | Deployed as a background service |

## Usage Example

```java
ClientConfiguration config = new ClientConfiguration();
config.setInstallationType(InstallationType.SERVICE);

if (config.getInstallationType() == InstallationType.GUI_APP) {
    // launch GUI-specific initialization
}
```
