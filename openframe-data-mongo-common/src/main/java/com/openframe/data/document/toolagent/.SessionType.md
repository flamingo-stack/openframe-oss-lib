Brief enumeration defining the supported session classification types for tool agent documents within the OpenFrame data layer.

## Key Components

| Constant | Description |
|----------|-------------|
| `SERVICE` | Represents an automated service-level session |
| `CONSOLE` | Represents a console/terminal-initiated session |
| `USER` | Represents an end-user initiated session |

## Usage Example

```java
SessionType sessionType = SessionType.USER;

switch (sessionType) {
    case SERVICE -> handleServiceSession();
    case CONSOLE -> handleConsoleSession();
    case USER    -> handleUserSession();
}
```
