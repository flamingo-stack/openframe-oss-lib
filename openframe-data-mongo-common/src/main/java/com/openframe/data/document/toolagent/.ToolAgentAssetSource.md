Defines the possible asset source locations for a Tool Agent, representing where tool-related assets can be retrieved from.

## Key Components

| Enum Constant | Description |
|---------------|-------------|
| `ARTIFACTORY` | Assets sourced from an Artifactory repository |
| `TOOL_API` | Assets sourced via a Tool API endpoint |
| `GITHUB` | Assets sourced from a GitHub repository |

## Usage Example

```java
ToolAgentAssetSource source = ToolAgentAssetSource.GITHUB;

switch (source) {
    case ARTIFACTORY -> fetchFromArtifactory();
    case TOOL_API    -> fetchFromToolApi();
    case GITHUB      -> fetchFromGitHub();
}
```
