A simple data model representing a URL endpoint for a tool integration, encapsulating connection details including address, port, and endpoint type.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `url` | `String` | The base URL or hostname of the tool endpoint |
| `port` | `String` | The port number for the connection |
| `type` | `ToolUrlType` | Enum classifying the URL type (e.g., API, web, etc.) |

Lombok annotations provide boilerplate-free construction:
- `@Data` — getters, setters, `equals`, `hashCode`, `toString`
- `@Builder` — fluent builder pattern
- `@NoArgsConstructor` / `@AllArgsConstructor` — default and full constructors

## Usage Example

```java
// Builder pattern
ToolUrl toolUrl = ToolUrl.builder()
    .url("https://helpdesk.example.com")
    .port("8443")
    .type(ToolUrlType.API)
    .build();

// All-args constructor
ToolUrl toolUrl = new ToolUrl("https://helpdesk.example.com", "8443", ToolUrlType.WEB);

// Getter access
String endpoint = toolUrl.getUrl();
ToolUrlType urlType = toolUrl.getType();
```
