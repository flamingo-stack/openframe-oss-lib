Generic repository interface for API key data access operations, supporting both blocking and reactive implementations through parameterized return types.

## Key Components

| Element | Description |
|---|---|
| `BaseApiKeyRepository<T, B, L, ID>` | Generic interface with type parameters for blocking (`Optional`, `List`) or reactive (`Mono`, `Flux`) implementations |
| `findByIdAndUserId` | Retrieves a single API key by its ID and owning user ID |
| `findByUserId` | Retrieves all API keys associated with a given user |
| `findExpiredKeys` | Retrieves all keys whose expiry precedes the provided `Instant` |

### Type Parameters

| Parameter | Blocking | Reactive |
|---|---|---|
| `T` | `Optional<ApiKey>` | `Mono<ApiKey>` |
| `B` | `boolean` | `Mono<Boolean>` |
| `L` | `List<ApiKey>` | `Flux<ApiKey>` |
| `ID` | `String` | `String` |

## Usage Example

```java
// Blocking implementation
public class JpaApiKeyRepository
    implements BaseApiKeyRepository<Optional<ApiKey>, Boolean, List<ApiKey>, String> {

    @Override
    public Optional<ApiKey> findByIdAndUserId(String keyId, String userId) {
        return jpaRepo.findByIdAndUserId(keyId, userId);
    }

    @Override
    public List<ApiKey> findByUserId(String userId) {
        return jpaRepo.findAllByUserId(userId);
    }

    @Override
    public List<ApiKey> findExpiredKeys(Instant currentTime) {
        return jpaRepo.findByExpiresAtBefore(currentTime);
    }
}

// Reactive implementation
public class ReactiveApiKeyRepository
    implements BaseApiKeyRepository<Mono<ApiKey>, Mono<Boolean>, Flux<ApiKey>, String> {

    @Override
    public Mono<ApiKey> findByIdAndUserId(String keyId, String userId) {
        return reactiveRepo.findByIdAndUserId(keyId, userId);
    }
}
```

> **Note:** The `B` (boolean) type parameter is reserved for future write operations (e.g., soft-delete, revocation). Current read-only methods only use `T`, `L`, and `ID`.
