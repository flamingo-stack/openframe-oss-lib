Service interface for handling repository events related to machines, tags, and tag assignments, publishing resulting state changes as `MachinePinotMessage` Kafka messages for downstream Pinot ingestion.

## Key Components

| Method | Trigger | Description |
|--------|---------|-------------|
| `processMachineSave` | Single machine saved | Fetches all tags for the machine and publishes to Kafka |
| `processMachineSaveAll` | Batch machine save | Iterates collection, publishes per-machine |
| `processTagAssignmentSave` | Single assignment saved | Resolves machine + tag, publishes updated machine state |
| `processTagAssignmentSaveAll` | Batch assignment save | Processes each assignment, publishes affected machines |
| `processTagSave` | Single tag saved | Only re-publishes when tag key changes |
| `processTagSaveAll` | Batch tag save | Publishes affected machines per tag change |
| `processTagAssignmentDelete` | Single assignment deleted | **Must be called before delete** — resolves `machineId` first |
| `processTagAssignmentDeleteByTagId` | Bulk delete by tag | **Must be called before delete** — re-syncs all affected machines |

> ⚠️ The two delete methods must be invoked **before** the actual deletion occurs. The pre-delete call is required to resolve which `machineId`(s) are affected before the relationship is removed from the database.

## Usage Example

```java
@Service
public class MachineTagEventServiceImpl implements MachineTagEventService {

    @Override
    public void processMachineSave(Machine machine) {
        // Fetch all tags assigned to this machine
        List<Tag> tags = tagRepository.findByMachineId(machine.getId());
        MachinePinotMessage message = buildMessage(machine, tags);
        kafkaTemplate.send(MACHINE_TOPIC, message);
    }

    @Override
    public void processTagAssignmentDelete(String machineId, String tagId) {
        // Called BEFORE delete — machineId still resolvable
        Machine machine = machineRepository.findById(machineId).orElseThrow();
        List<Tag> remainingTags = tagRepository.findByMachineIdExcluding(machineId, tagId);
        MachinePinotMessage message = buildMessage(machine, remainingTags);
        kafkaTemplate.send(MACHINE_TOPIC, message);
    }
}

// Wiring into a repository event listener
@Component
public class TagAssignmentListener {

    private final MachineTagEventService eventService;

    public void onBeforeDelete(TagAssignment assignment) {
        // Pre-delete hook: resolve machine before relationship is removed
        eventService.processTagAssignmentDelete(
            assignment.getMachineId(),
            assignment.getTagId()
        );
    }
}
```
