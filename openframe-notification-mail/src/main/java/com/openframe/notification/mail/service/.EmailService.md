A service interface that defines contract methods for sending different types of email notifications in the OpenFrame notification system.

## Key Components

- **`sendInvitationEmail(String, String)`** - Sends invitation emails with an invitation ID
- **`sendPasswordResetEmail(String, String)`** - Sends password reset emails with a reset token

## Usage Example

```java
@Service
public class EmailServiceImpl implements EmailService {
    
    @Override
    public void sendInvitationEmail(String toEmail, String invitationId) {
        // Implementation for sending invitation emails
        // Use invitationId to generate invitation link
    }
    
    @Override
    public void sendPasswordResetEmail(String toEmail, String resetToken) {
        // Implementation for sending password reset emails  
        // Use resetToken to generate reset link
    }
}

// Usage in controller or service
@Autowired
private EmailService emailService;

public void inviteUser(String email, String inviteId) {
    emailService.sendInvitationEmail(email, inviteId);
}

public void resetUserPassword(String email, String token) {
    emailService.sendPasswordResetEmail(email, token);
}
```
