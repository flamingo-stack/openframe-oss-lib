A service interface for email notification functionality within the OpenFrame platform, providing standardized methods for sending various types of system emails.

## Key Components

- **sendInvitationEmail()** - Sends invitation emails to new users with invitation ID
- **sendPasswordResetEmail()** - Sends password reset emails containing reset tokens
- **sendEmailVerificationEmail()** - Sends email verification messages with verification tokens

## Usage Example

```java
@Service
public class EmailServiceImpl implements EmailService {
    
    @Override
    public void sendInvitationEmail(String toEmail, String invitationId) {
        // Implementation for sending invitation emails
        emailTemplate.send(toEmail, "invitation.html", 
            Map.of("invitationId", invitationId));
    }
    
    @Override
    public void sendPasswordResetEmail(String toEmail, String resetToken) {
        // Implementation for password reset emails
        emailTemplate.send(toEmail, "password-reset.html", 
            Map.of("resetToken", resetToken));
    }
    
    @Override
    public void sendEmailVerificationEmail(String toEmail, String verifyToken) {
        // Implementation for email verification
        emailTemplate.send(toEmail, "verify-email.html", 
            Map.of("verifyToken", verifyToken));
    }
}

// Usage in controllers or other services
@Autowired
private EmailService emailService;

public void inviteUser(String email, String inviteId) {
    emailService.sendInvitationEmail(email, inviteId);
}
```

This interface follows the standard service pattern for dependency injection and can be implemented with various email providers (SMTP, SendGrid, etc.).
