An interface defining email notification services for user authentication and invitation workflows.

## Key Components

- **`sendInvitationEmail(String toEmail, String invitationId)`** - Sends invitation emails to new users with their invitation identifier
- **`sendPasswordResetEmail(String toEmail, String resetToken)`** - Sends password reset emails containing secure reset tokens

## Usage Example

```java
@Service
public class EmailServiceImpl implements EmailService {
    
    @Override
    public void sendInvitationEmail(String toEmail, String invitationId) {
        // Implementation for sending invitation emails
        emailClient.send(toEmail, "Welcome!", generateInvitationContent(invitationId));
    }
    
    @Override
    public void sendPasswordResetEmail(String toEmail, String resetToken) {
        // Implementation for sending reset emails
        emailClient.send(toEmail, "Reset Password", generateResetContent(resetToken));
    }
}

// Usage in controller or service
@Autowired
private EmailService emailService;

public void inviteUser(String email, String invitationId) {
    emailService.sendInvitationEmail(email, invitationId);
}

public void requestPasswordReset(String email, String token) {
    emailService.sendPasswordResetEmail(email, token);
}
```
