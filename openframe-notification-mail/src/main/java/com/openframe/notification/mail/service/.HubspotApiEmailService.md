Spring service implementation that sends transactional emails through HubSpot's Single Send API, supporting invitation and password reset email templates with dynamic link generation.

## Key Components

- **HubspotApiEmailService**: Main service class implementing `EmailService` interface
- **sendInvitationEmail()**: Sends workspace invitation emails with personalized links
- **sendPasswordResetEmail()**: Sends password reset emails with secure tokens
- **sendWithTemplate()**: Private method handling HubSpot API communication with template-based emails
- **WebClient**: Reactive HTTP client for HubSpot API integration
- **Configuration Properties**: Externalized email templates, URLs, and API credentials

## Usage Example

```java
@Configuration
@ConfigurationProperties(prefix = "openframe.mail")
public class MailConfig {
    // Required properties for HubSpot integration
    private String provider = "hubspot-api";
    private String from = "noreply@company.com";
    
    @NestedConfigurationProperty
    private HubspotConfig hubspot = new HubspotConfig();
    
    public static class HubspotConfig {
        private String accessToken;
        private String baseUrl = "https://api.hubapi.com";
        private String invitationEmailId;
        private String resetEmailId;
    }
}

@Service
public class UserService {
    private final EmailService emailService;
    
    public void inviteUser(String email, String invitationId) {
        emailService.sendInvitationEmail(email, invitationId);
    }
    
    public void resetPassword(String email, String resetToken) {
        emailService.sendPasswordResetEmail(email, resetToken);
    }
}
```

The service automatically activates when `openframe.mail.provider=hubspot-api` and uses reactive WebClient for non-blocking API calls with comprehensive error handling and logging.
