Email service implementation that integrates with HubSpot's Marketing API to send transactional emails using predefined templates for user invitations, password resets, and email verification.

## Key Components

- **`@ConditionalOnProperty`** - Only activated when `openframe.mail.provider=hubspot-api`
- **`sendInvitationEmail()`** - Sends workspace invitation emails with invitation links
- **`sendPasswordResetEmail()`** - Sends password reset emails with reset tokens
- **`sendEmailVerificationEmail()`** - Sends email verification with verification tokens
- **`sendWithTemplate()`** - Private method handling HubSpot API calls with template rendering
- **WebClient integration** - Reactive HTTP client for HubSpot Marketing v4 API

## Usage Example

```java
@Autowired
private EmailService emailService;

// Send invitation email
emailService.sendInvitationEmail("user@example.com", "invite-123");

// Send password reset
emailService.sendPasswordResetEmail("user@example.com", "reset-token-456");

// Send email verification
emailService.sendEmailVerificationEmail("user@example.com", "verify-token-789");
```

Configuration required:
```yaml
openframe:
  mail:
    provider: hubspot-api
    from: noreply@flamingo.run
    hubspot:
      access-token: ${HUBSPOT_ACCESS_TOKEN}
      base-url: https://api.hubapi.com
      invitation-email-id: "template-id-1"
      reset-email-id: "template-id-2"
      verify-email-id: "template-id-3"
```
