A Spring service that implements email functionality using HubSpot's transactional email API. This service sends invitation and password reset emails through HubSpot's marketing API with custom templates.

## Key Components

**Main Class:**
- `HubspotApiEmailService` - Implements `EmailService` interface with HubSpot API integration

**Public Methods:**
- `sendInvitationEmail(String toEmail, String invitationId)` - Sends workspace invitation emails
- `sendPasswordResetEmail(String toEmail, String resetToken)` - Sends password reset emails

**Configuration Properties:**
- `openframe.mail.hubspot.access-token` - HubSpot API authentication token
- `openframe.mail.hubspot.base-url` - HubSpot API base URL
- `openframe.mail.hubspot.invitation-email-id` - Template ID for invitations
- `openframe.mail.hubspot.reset-email-id` - Template ID for password resets

## Usage Example

```java
@Autowired
private HubspotApiEmailService emailService;

// Send invitation email
emailService.sendInvitationEmail("user@example.com", "invitation-123");

// Send password reset email
emailService.sendPasswordResetEmail("user@example.com", "reset-token-456");
```

**Configuration:**
```properties
openframe.mail.provider=hubspot-api
openframe.mail.hubspot.access-token=your-hubspot-token
openframe.mail.hubspot.base-url=https://api.hubapi.com
openframe.mail.hubspot.invitation-email-id=template-id-1
openframe.mail.hubspot.reset-email-id=template-id-2
```
