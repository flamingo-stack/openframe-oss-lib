A Spring service implementation that provides SMTP-based email functionality for user invitations and password reset notifications.

## Key Components

- **SmtpEmailService** - Main service class implementing `EmailService` interface
- **sendInvitationEmail()** - Sends user invitation emails with registration links
- **sendPasswordResetEmail()** - Sends password reset emails with reset tokens
- **sendPlainText()** - Private helper method for sending plain text emails
- **JavaMailSender** - Injected Spring mail sender for SMTP operations

## Usage Example

```java
@Autowired
private SmtpEmailService emailService;

// Send invitation email
emailService.sendInvitationEmail("user@example.com", "invitation-123");

// Send password reset email  
emailService.sendPasswordResetEmail("user@example.com", "reset-token-456");
```

**Configuration Requirements:**
```properties
# Enable SMTP provider (default)
openframe.mail.provider=smtp

# Configure link templates
openframe.invitations.link-template=https://app.com/invite/{id}
openframe.password-reset.link-template=https://app.com/reset/{token}
```

The service is conditionally activated when the mail provider is set to "smtp" (or by default) and automatically logs successful email deliveries. It uses configurable link templates to generate dynamic URLs for invitations and password resets.
