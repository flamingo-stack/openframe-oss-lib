SMTP-based email service implementation that sends plain text emails for user invitations and password resets using Spring's JavaMailSender. This service is conditionally enabled when the mail provider is configured as "smtp" and provides basic email functionality for the OpenFrame platform.

## Key Components

- **SmtpEmailService** - Main service class implementing the `EmailService` interface
- **sendInvitationEmail()** - Sends invitation emails with registration links
- **sendPasswordResetEmail()** - Sends password reset emails with secure tokens
- **sendEmailVerificationEmail()** - Throws UnsupportedOperationException (requires HubSpot provider)
- **sendPlainText()** - Private utility method for sending simple text emails

## Configuration

The service is conditionally activated based on configuration:

```java
@ConditionalOnProperty(prefix = "openframe.mail", name = "provider", havingValue = "smtp", matchIfMissing = true)
```

Required configuration properties:
- `openframe.invitations.link-template` - Template URL for invitation links (with `{id}` placeholder)
- `openframe.password-reset.link-template` - Template URL for password reset links (with `{token}` placeholder)

## Usage Example

```java
@Autowired
private EmailService emailService;

// Send invitation email
emailService.sendInvitationEmail("user@example.com", "inv-123");

// Send password reset email  
emailService.sendPasswordResetEmail("user@example.com", "reset-token-456");

// Note: Email verification requires HubSpot provider
// emailService.sendEmailVerificationEmail(); // Throws UnsupportedOperationException
```

The service automatically replaces placeholders in link templates and logs successful email sends for monitoring purposes.
