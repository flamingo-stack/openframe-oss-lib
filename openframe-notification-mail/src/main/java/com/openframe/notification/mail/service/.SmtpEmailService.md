A Spring service that implements email functionality using SMTP for OpenFrame application notifications, specifically handling invitation and password reset emails.

## Key Components

**Main Class:**
- `SmtpEmailService` - SMTP-based implementation of the `EmailService` interface

**Dependencies:**
- `JavaMailSender` - Spring's email sending abstraction
- `linkTemplate` - Configurable invitation link template
- `resetLinkTemplate` - Configurable password reset link template

**Public Methods:**
- `sendInvitationEmail(String, String)` - Sends invitation emails with registration links
- `sendPasswordResetEmail(String, String)` - Sends password reset emails with reset tokens

**Configuration:**
- Conditionally enabled when `openframe.mail.provider=smtp` (default)
- Uses Spring Boot's auto-configuration for SMTP settings

## Usage Example

```java
@Autowired
private SmtpEmailService emailService;

// Send invitation
emailService.sendInvitationEmail("user@example.com", "inv-123");

// Send password reset
emailService.sendPasswordResetEmail("user@example.com", "reset-token-456");
```

**Required Configuration:**
```properties
# SMTP settings (Spring Boot auto-config)
spring.mail.host=smtp.gmail.com
spring.mail.port=587
spring.mail.username=your-email@gmail.com
spring.mail.password=your-password

# OpenFrame templates
openframe.invitations.link-template=https://app.com/invite/{id}
openframe.password-reset.link-template=https://app.com/reset/{token}
```

The service automatically logs successful email deliveries and uses plain text formatting for maximum compatibility across email clients.
