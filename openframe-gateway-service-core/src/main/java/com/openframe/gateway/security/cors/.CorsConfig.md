A Spring Boot configuration class that sets up Cross-Origin Resource Sharing (CORS) for the OpenFrame Gateway, enabling controlled cross-origin requests in a reactive web environment.

## Key Components

- **`@ConditionalOnProperty`** - Conditionally enables CORS based on `openframe.gateway.disable-cors` property (enabled by default)
- **`corsConfiguration()`** - Creates a `CorsConfiguration` bean populated from Spring Cloud Gateway global CORS properties
- **`corsWebFilter()`** - Builds a reactive `CorsWebFilter` that applies CORS rules to all endpoints (`/**`)

## Usage Example

Configure CORS settings in your application properties:

```java
# Enable/disable CORS (default: enabled)
openframe.gateway.disable-cors=false

# Configure CORS policies
spring.cloud.gateway.globalcors.cors-configurations.[/**].allowed-origins=*
spring.cloud.gateway.globalcors.cors-configurations.[/**].allowed-methods=GET,POST,PUT,DELETE
spring.cloud.gateway.globalcors.cors-configurations.[/**].allowed-headers=*
spring.cloud.gateway.globalcors.cors-configurations.[/**].allow-credentials=true
```

The configuration automatically registers the CORS filter for all routes, allowing the gateway to handle preflight requests and apply the specified CORS policies to incoming requests from web browsers.
