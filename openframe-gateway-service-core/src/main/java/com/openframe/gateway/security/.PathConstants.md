Defines path prefix constants used across the OpenFrame Gateway security and routing layer to standardize endpoint URL segments.

## Key Components

| Constant | Value | Description |
|---|---|---|
| `CLIENTS_PREFIX` | `/clients` | Client-facing endpoint prefix |
| `API_PREFIX` | `/api` | General API endpoint prefix |
| `TOOLS_PREFIX` | `/tools` | Tools endpoint prefix |
| `WS_TOOLS_PREFIX` | `/ws/tools` | WebSocket tools endpoint prefix |
| `CHAT_PREFIX` | `/chat` | Chat endpoint prefix |
| `GUIDE_PREFIX` | `/guide` | Guide endpoint prefix |

## Usage Example

```java
import com.openframe.gateway.security.PathConstants;

// Use in security configuration to permit or restrict routes
http.authorizeRequests()
    .antMatchers(PathConstants.API_PREFIX + "/**").authenticated()
    .antMatchers(PathConstants.CLIENTS_PREFIX + "/**").hasRole("CLIENT")
    .antMatchers(PathConstants.WS_TOOLS_PREFIX + "/**").authenticated()
    .anyRequest().permitAll();

// Use in route matching
String toolsRoute = PathConstants.TOOLS_PREFIX + "/execute";
String chatRoute  = PathConstants.CHAT_PREFIX + "/session";
```

> **Note:** `PathConstants` is a non-instantiable utility class (private constructor). Always reference constants statically via `PathConstants.CONSTANT_NAME`.
