A utility class that defines standardized path prefixes for different endpoint categories in the OpenFrame Gateway security module. This class provides centralized constants to ensure consistent URL routing across the application.

## Key Components

- **CLIENTS_PREFIX** - Path prefix for client-related endpoints (`/clients`)
- **DASHBOARD_PREFIX** - Path prefix for dashboard API endpoints (`/api`)
- **TOOLS_PREFIX** - Path prefix for tools-related endpoints (`/tools`)
- **WS_TOOLS_PREFIX** - Path prefix for WebSocket tools endpoints (`/ws/tools`)

## Usage Example

```java
// Using path constants in controller mappings
@RestController
@RequestMapping(PathConstants.DASHBOARD_PREFIX)
public class DashboardController {
    // Maps to /api/status
    @GetMapping("/status")
    public ResponseEntity<String> getStatus() {
        return ResponseEntity.ok("OK");
    }
}

// Using in WebSocket configuration
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        // Maps to /ws/tools/terminal
        registry.addHandler(terminalHandler, PathConstants.WS_TOOLS_PREFIX + "/terminal");
    }
}

// Building URLs programmatically
String clientsEndpoint = PathConstants.CLIENTS_PREFIX + "/list";
String toolsEndpoint = PathConstants.TOOLS_PREFIX + "/docker";
```
