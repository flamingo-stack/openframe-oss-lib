Defines path prefix constants used across the OpenFrame Gateway for routing and security configuration.

## Key Components

| Constant | Value | Purpose |
|---|---|---|
| `CLIENTS_PREFIX` | `/clients` | Client-facing endpoint routes |
| `API_PREFIX` | `/api` | General API routes |
| `TOOLS_PREFIX` | `/tools` | Tool/integration endpoints |
| `WS_TOOLS_PREFIX` | `/ws/tools` | WebSocket tool endpoints |
| `CHAT_PREFIX` | `/chat` | Chat/messaging routes |
| `CONTENT_PREFIX` | `/content` | Content delivery routes |

## Usage Example

```java
import com.openframe.gateway.security.PathConstants;

// Use in security filter or route configuration
http.authorizeRequests()
    .antMatchers(PathConstants.API_PREFIX + "/**").authenticated()
    .antMatchers(PathConstants.CLIENTS_PREFIX + "/**").hasRole("CLIENT")
    .antMatchers(PathConstants.WS_TOOLS_PREFIX + "/**").authenticated();

// Use in route predicate
RouteDefinition route = new RouteDefinition();
route.setPredicates(List.of(
    new PredicateDefinition("Path=" + PathConstants.TOOLS_PREFIX + "/**")
));
```

> **Note:** `PathConstants` is a non-instantiable utility class. All constants are `public static final` and should be referenced statically throughout the gateway security and routing layers.
