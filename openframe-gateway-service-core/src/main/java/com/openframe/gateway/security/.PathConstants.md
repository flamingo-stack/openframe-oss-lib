Defines centralized path prefix constants used across the OpenFrame Gateway for routing and security configuration.

## Key Components

| Constant | Value | Purpose |
|---|---|---|
| `CLIENTS_PREFIX` | `/clients` | Client-facing endpoint routes |
| `API_PREFIX` | `/api` | REST API endpoint routes |
| `TOOLS_PREFIX` | `/tools` | Tool/integration endpoint routes |
| `WS_TOOLS_PREFIX` | `/ws/tools` | WebSocket tool endpoint routes |
| `CHAT_PREFIX` | `/chat` | Chat/messaging endpoint routes |
| `CONTENT_PREFIX` | `/content` | Content delivery endpoint routes |

## Usage Example

```java
import com.openframe.gateway.security.PathConstants;

@Configuration
public class SecurityConfig {

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http.authorizeHttpRequests(auth -> auth
            .requestMatchers(PathConstants.CLIENTS_PREFIX + "/**").hasRole("CLIENT")
            .requestMatchers(PathConstants.API_PREFIX + "/**").authenticated()
            .requestMatchers(PathConstants.WS_TOOLS_PREFIX + "/**").authenticated()
            .anyRequest().permitAll()
        );
        return http.build();
    }
}
```

> `PathConstants` is a non-instantiable utility class — all fields are `static final`. Reference constants directly via the class name rather than instantiating it.
