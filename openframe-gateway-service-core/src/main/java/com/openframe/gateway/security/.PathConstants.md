Utility class containing path constants for the OpenFrame Gateway application, defining standardized URL prefixes for different endpoint categories.

## Key Components

- **CLIENTS_PREFIX** - Path prefix for client-related endpoints (`/clients`)
- **API_PREFIX** - Path prefix for general API endpoints (`/api`)
- **TOOLS_PREFIX** - Path prefix for tool-related endpoints (`/tools`)
- **WS_TOOLS_PREFIX** - Path prefix for WebSocket tool endpoints (`/ws/tools`)
- **CHAT_PREFIX** - Path prefix for chat-related endpoints (`/chat`)

All constants are defined as `public static final` strings to ensure they are immutable and accessible throughout the application without instantiation.

## Usage Example

```java
@RestController
@RequestMapping(PathConstants.API_PREFIX)
public class ApiController {
    
    @GetMapping("/status")
    public ResponseEntity<String> getStatus() {
        // This endpoint will be available at /api/status
        return ResponseEntity.ok("Gateway is running");
    }
}

@Component
public class SecurityConfig {
    
    public void configurePathMatchers(HttpSecurity http) {
        http.authorizeRequests()
            .antMatchers(PathConstants.CLIENTS_PREFIX + "/**")
            .hasRole("CLIENT")
            .antMatchers(PathConstants.TOOLS_PREFIX + "/**")
            .hasRole("USER");
    }
}
```

This class follows the utility class pattern with a private constructor to prevent instantiation, ensuring path constants remain consistent across the Gateway application's routing and security configurations.
