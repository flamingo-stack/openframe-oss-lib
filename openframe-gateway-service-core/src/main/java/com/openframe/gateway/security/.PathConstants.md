This utility class defines standardized path prefixes used throughout the Gateway application for routing and endpoint organization.

## Key Components

- **CLIENTS_PREFIX** - Path prefix for client-related endpoints (`/clients`)
- **DASHBOARD_PREFIX** - Path prefix for API dashboard endpoints (`/api`)
- **TOOLS_PREFIX** - Path prefix for tool-related HTTP endpoints (`/tools`)
- **WS_TOOLS_PREFIX** - Path prefix for WebSocket tool endpoints (`/ws/tools`)

## Usage Example

```java
// Building endpoint URLs
String clientEndpoint = PathConstants.CLIENTS_PREFIX + "/status";
// Results in: "/clients/status"

String dashboardEndpoint = PathConstants.DASHBOARD_PREFIX + "/metrics";
// Results in: "/api/metrics"

// In Spring controllers
@RestController
@RequestMapping(PathConstants.DASHBOARD_PREFIX)
public class DashboardController {
    
    @GetMapping("/health")
    public ResponseEntity<String> getHealth() {
        // Handles requests to /api/health
        return ResponseEntity.ok("healthy");
    }
}

// WebSocket configuration
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {
    
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        registry.addHandler(toolsHandler, PathConstants.WS_TOOLS_PREFIX + "/**");
    }
}
```
