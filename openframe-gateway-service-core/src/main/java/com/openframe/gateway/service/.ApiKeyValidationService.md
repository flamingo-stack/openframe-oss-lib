A reactive Spring service that handles API key validation with format parsing, credential verification, and statistics tracking.

## Key Components

- **validateApiKey()** - Main validation method that parses API keys in format `ak_{keyId}.sk_{secret}` and verifies against database
- **recordSuccessfulRequest()** - Records successful API key usage for analytics
- **recordFailedRequest()** - Records failed API key attempts for monitoring
- **ParsedApiKey** - Internal record class for parsing API key components
- **ApiKeyValidationResult** - Result wrapper containing validation status and API key data

## Usage Example

```java
@Autowired
private ApiKeyValidationService validationService;

public Mono<String> authenticateRequest(String apiKey) {
    return validationService.validateApiKey(apiKey)
        .flatMap(result -> {
            if (result.isValid()) {
                validationService.recordSuccessfulRequest(result.getApiKey().getId());
                return processRequest(result.getApiKey());
            } else {
                return Mono.error(new UnauthorizedException(result.getErrorMessage()));
            }
        });
}

// Example API key format
String validApiKey = "ak_1234567890abcdef.sk_secretkey123456789";
```

The service expects API keys in the specific format `ak_{keyId}.sk_{secret}` and performs comprehensive validation including format checking, database lookup, password verification, and status validation.
