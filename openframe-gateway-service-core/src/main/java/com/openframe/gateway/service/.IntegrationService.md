A reactive service that manages external tool integration testing and connection validation for the OpenFrame gateway system.

## Key Components

- **testIntegrationConnection()** - Tests connectivity to integrated tools by validating tool status and making health check requests
- **adjustUrl()** - Modifies URLs for local development by redirecting to localhost while preserving path and query parameters
- **isLocalProfile()** - Detects if the application is running in local development mode based on Spring profiles

## Usage Example

```java
@RestController
public class IntegrationController {
    
    private final IntegrationService integrationService;
    
    @GetMapping("/integrations/{toolId}/test")
    public Mono<ResponseEntity<String>> testConnection(@PathVariable String toolId) {
        return integrationService.testIntegrationConnection(toolId)
            .map(response -> ResponseEntity.ok(response))
            .onErrorResume(ResponseStatusException.class, 
                ex -> Mono.just(ResponseEntity.status(ex.getStatusCode())
                    .body(ex.getReason())));
    }
}
```

The service validates that tools exist and are enabled before attempting connections. In local development, it automatically redirects API calls to localhost for easier testing. Health check responses are logged for monitoring integration status.
