A Spring service that acts as a reactive HTTP proxy for integrated tools, forwarding API and agent requests to external endpoints with authentication and error handling.

## Key Components

**Main Methods:**
- `proxyApiRequest()` - Proxies API requests to integrated tools with authentication headers
- `proxyAgentRequest()` - Proxies agent-specific requests with tool authorization
- `buildApiRequestHeaders()` - Constructs headers with API key authentication (header, bearer token, or none)
- `buildAgentRequestHeaders()` - Builds headers for agent requests using tool authorization
- `proxy()` - Core proxy method handling WebClient configuration and request forwarding

**Key Features:**
- Reactive HTTP client with SSL configuration and timeouts
- Multiple authentication types (API key, bearer token)
- Tool enablement validation
- URL resolution and request transformation
- Comprehensive error handling and logging

## Usage Example

```java
@Autowired
private RestProxyService restProxyService;

// Proxy API request to external tool
Mono<ResponseEntity<String>> response = restProxyService
    .proxyApiRequest("tool-123", serverRequest, requestBody);

// Proxy agent request with tool authorization
Mono<ResponseEntity<String>> agentResponse = restProxyService
    .proxyAgentRequest("tool-456", serverRequest, agentData);

response.subscribe(result -> {
    if (result.getStatusCode().is2xxSuccessful()) {
        // Handle successful proxy response
    }
});
```

The service automatically handles authentication, SSL configuration, and forwards responses with appropriate status codes and error handling.
