Reactive Spring service that proxies HTTP requests from the OpenFrame gateway to downstream integrated tools, handling authentication header injection, SSL configuration, and error mapping.

## Key Components

| Component | Description |
|-----------|-------------|
| `proxyApiRequest` | Proxies REST API calls to a tool using API-key-based authentication headers |
| `proxyAgentRequest` | Proxies agent-scoped requests, forwarding `Tool-Authorization` headers from the inbound request |
| `buildApiRequestHeaders` | Constructs standard JSON headers plus resolved API key headers for direct API calls |
| `buildAgentRequestHeaders` | Extracts and forwards agent auth headers from the original `ServerHttpRequest` |
| `proxy` | Core dispatch method: builds a per-request `WebClient` with a 16 MB codec buffer, 60 s timeout, and curl logging |
| `buildHttpClient` | Configures a Reactor Netty `HttpClient` with SSL bypass (`InsecureTrustManagerFactory`), TCP tuning, and enlarged header decoder limits |
| `processErrorResponse` | Converts 4xx/5xx upstream responses into `WebClientResponseException` for uniform downstream handling |
| `buildErrorResponse` | Maps exceptions back to typed `ResponseEntity<String>` responses |

## Usage Example

```java
// Injected via Spring context
@Autowired
RestProxyService restProxyService;

// Forward an API request to an integrated tool
Mono<ResponseEntity<String>> result = restProxyService.proxyApiRequest(
    "zabbix",          // toolId matching IntegratedTool key
    serverHttpRequest, // reactive ServerHttpRequest from gateway
    requestBody        // raw JSON string body (may be null/empty)
);

// Forward an agent request (e.g. Tactical RMM agent polling)
Mono<ResponseEntity<String>> agentResult = restProxyService.proxyAgentRequest(
    "tacticalrmm",
    serverHttpRequest,
    requestBody
);
```

> **Note:** SSL certificate validation is currently disabled via `InsecureTrustManagerFactory`. This is intentional for self-hosted tool integrations but should be replaced with proper trust store configuration before production hardening.
