A utility class that provides standardized logging message constants for rate limiting operations in the API gateway.

## Key Components

- **LOG_RATE_LIMIT_CHECK** - Constant for logging rate limit validation messages
- **LOG_RATE_LIMIT_STATUS** - Constant for logging rate limit status retrieval messages

## Usage Example

```java
import static com.openframe.gateway.constants.RateLimitConstants.*;

public class RateLimitService {
    private static final Logger logger = LoggerFactory.getLogger(RateLimitService.class);
    
    public boolean checkRateLimit(String apiKey) {
        logger.info(LOG_RATE_LIMIT_CHECK, apiKey);
        // Rate limit validation logic
        return true;
    }
    
    public RateLimitStatus getRateLimitStatus(String apiKey) {
        logger.info(LOG_RATE_LIMIT_STATUS, apiKey);
        // Status retrieval logic
        return new RateLimitStatus();
    }
}
```

This constants class follows the utility class pattern with a private constructor to prevent instantiation, ensuring consistent logging messages across the rate limiting subsystem.
