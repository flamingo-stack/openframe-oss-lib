Defines logging message constants used throughout the rate limiting system for consistent API key monitoring and debugging.

## Key Components

- **LOG_RATE_LIMIT_CHECK**: Standard logging message for rate limit validation operations
- **LOG_RATE_LIMIT_STATUS**: Standard logging message for rate limit status retrieval operations

## Usage Example

```java
import com.openframe.gateway.constants.RateLimitConstants;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

public class RateLimitService {
    private static final Logger logger = LoggerFactory.getLogger(RateLimitService.class);
    
    public boolean checkRateLimit(String apiKey) {
        logger.debug(RateLimitConstants.LOG_RATE_LIMIT_CHECK, apiKey);
        // Rate limiting logic here
        return true;
    }
    
    public RateLimitStatus getRateLimitStatus(String apiKey) {
        logger.info(RateLimitConstants.LOG_RATE_LIMIT_STATUS, apiKey);
        // Status retrieval logic here
        return new RateLimitStatus();
    }
}
```

This utility class follows the constants pattern with a private constructor to prevent instantiation and provides centralized logging messages for rate limiting operations using SLF4J placeholder syntax.
