A specialized WebSocket proxy URL filter for tool agent endpoints that extends the base tool WebSocket filter functionality.

## Key Components

- **ToolAgentWebSocketProxyUrlFilter** - Main component class extending `ToolWebSocketProxyUrlFilter`
- **getRequestToolId()** - Extracts tool ID from the request path (4th segment)
- **getEndpointPrefix()** - Returns the agent-specific WebSocket endpoint prefix

## Usage Example

```java
@Configuration
public class WebSocketConfig {
    
    @Autowired
    private ToolAgentWebSocketProxyUrlFilter agentFilter;
    
    public void configureWebSocket() {
        // Filter automatically processes paths like:
        // /ws/tools/agent/{toolId}/connect
        
        // The filter extracts toolId from position 4 in the path
        String toolId = agentFilter.getRequestToolId("/ws/tools/agent/my-tool-123/connect");
        // Returns: "my-tool-123"
        
        // Uses TOOLS_AGENT_WS_ENDPOINT_PREFIX for routing
        String prefix = agentFilter.getEndpointPrefix();
    }
}
```

This filter is specifically designed for tool agent WebSocket connections, parsing tool IDs from agent-specific URL patterns and applying the appropriate endpoint prefix for proper routing within the gateway.
