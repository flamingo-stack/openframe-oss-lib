A Spring component that extracts and parses JWT token claims from HTTP request headers without token signature validation.

## Key Components

- **`getExpiration(ServerWebExchange exchange)`** - Extracts the expiration timestamp from a JWT token in the request
- **`getClaims(ServerWebExchange exchange)`** - Private method that parses JWT claims from the Authorization header

## Usage Example

```java
@RestController
@RequiredArgsConstructor
public class TokenController {
    
    private final RequestJwt–°laimsReader claimsReader;
    
    @GetMapping("/token-info")
    public ResponseEntity<String> getTokenExpiration(ServerWebExchange exchange) {
        try {
            Instant expiration = claimsReader.getExpiration(exchange);
            return ResponseEntity.ok("Token expires at: " + expiration);
        } catch (IllegalStateException e) {
            return ResponseEntity.badRequest().body("Invalid token: " + e.getMessage());
        }
    }
}
```

The component expects requests with `Authorization: Bearer <jwt-token>` headers and throws `IllegalStateException` for missing or malformed authorization headers. Note that this implementation only parses the claims portion of the JWT without validating the token signature.
