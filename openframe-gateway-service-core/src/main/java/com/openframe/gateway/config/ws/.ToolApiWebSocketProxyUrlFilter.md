A specialized WebSocket proxy URL filter that handles tool API endpoints by extracting tool IDs from URL paths and applying the appropriate endpoint prefix.

## Key Components

- **ToolApiWebSocketProxyUrlFilter** - Main component class extending `ToolWebSocketProxyUrlFilter`
- **getRequestToolId()** - Extracts tool ID from the 4th path segment (index 3)
- **getEndpointPrefix()** - Returns the tools API WebSocket endpoint prefix constant

## Usage Example

```java
// The filter is automatically registered as a Spring component
// and handles WebSocket requests matching the tools API pattern

// Example URL path processing:
// Input path: "/gateway/tools/api/my-tool-id/websocket"
// getRequestToolId() returns: "my-tool-id"
// getEndpointPrefix() returns: TOOLS_API_WS_ENDPOINT_PREFIX

// Constructor usage in configuration:
@Configuration
public class WebSocketConfig {
    
    @Bean
    public ToolApiWebSocketProxyUrlFilter toolApiFilter(
            ReactiveIntegratedToolRepository toolRepository,
            ToolUrlService toolUrlService,
            ProxyUrlResolver proxyUrlResolver) {
        return new ToolApiWebSocketProxyUrlFilter(
            toolRepository, 
            toolUrlService, 
            proxyUrlResolver
        );
    }
}
```
