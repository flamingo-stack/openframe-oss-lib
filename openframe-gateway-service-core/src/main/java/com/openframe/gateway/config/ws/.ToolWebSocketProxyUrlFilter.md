Reactive Spring Cloud Gateway filter that resolves and proxies WebSocket connections to integrated tools by mapping request paths to upstream tool URIs.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `ORIGINAL_AUTHORIZATION_ATTR` | `String` constant | Exchange attribute key for preserving the original `Authorization` header before upstream mutation |
| `filter()` | `GatewayFilter` override | Core pipeline method — extracts the tool ID from the path, validates the tool, resolves the upstream WebSocket URI, and forwards the exchange |
| `getTool()` | private `Mono<IntegratedTool>` | Fetches the tool from the repository and guards against missing or disabled tools |
| `mutateExchange()` | protected hook | Optional override point for subclasses to modify the exchange (e.g. strip or replace headers) before forwarding |
| `getRequestToolId()` | abstract | Subclass-defined path parser that extracts the tool identifier from the request URI |
| `getEndpointPrefix()` | abstract | Subclass-defined prefix passed to the upstream resolver to qualify the target WebSocket endpoint |
| `getOrder()` | `Ordered` override | Runs one step after `RouteToRequestUrlFilter` so the resolved URL is set correctly in the gateway chain |

## Usage Example

```java
@Component
public class ZabbixWebSocketFilter extends ToolWebSocketProxyUrlFilter {

    public ZabbixWebSocketFilter(
            ReactiveIntegratedToolRepository toolRepository,
            ToolUpstreamResolverRegistry upstreamRegistry) {
        super(toolRepository, upstreamRegistry);
    }

    @Override
    protected String getRequestToolId(String path) {
        // e.g. /ws/tools/zabbix/events -> "zabbix"
        String[] segments = path.split("/");
        return segments[3];
    }

    @Override
    protected String getEndpointPrefix() {
        return "/zabbix";
    }

    // Optional: strip the gateway auth header before forwarding
    @Override
    protected ServerWebExchange mutateExchange(ServerWebExchange exchange, IntegratedTool tool) {
        ServerHttpRequest mutated = exchange.getRequest().mutate()
                .headers(h -> h.remove(HttpHeaders.AUTHORIZATION))
                .build();
        return exchange.mutate().request(mutated).build();
    }
}
```

> **Note:** If the tool is not found or is disabled, the filter short-circuits with an `IllegalArgumentException`, which should be mapped to an appropriate HTTP error via a global exception handler.
