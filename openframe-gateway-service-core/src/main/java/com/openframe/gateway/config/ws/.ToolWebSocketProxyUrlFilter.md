Abstract Spring Cloud Gateway filter that resolves and proxies WebSocket connections to integrated tools by dynamically determining the upstream WebSocket URI based on the tool configuration and request path.

## Key Components

| Component | Description |
|-----------|-------------|
| `ToolWebSocketProxyUrlFilter` | Abstract `GatewayFilter` base class for WebSocket proxy URL resolution |
| `ORIGINAL_AUTHORIZATION_ATTR` | Exchange attribute key for preserving the original `Authorization` header downstream |
| `filter()` | Core filter logic: extracts tool ID from path, validates the tool, resolves the upstream WebSocket URI, and stores it as `GATEWAY_REQUEST_URL_ATTR` |
| `getTool()` | Looks up the `IntegratedTool` by ID, throwing if not found or disabled |
| `mutateExchange()` | Overridable hook for subclasses to further mutate the exchange before forwarding |
| `getRequestToolId()` | Abstract — subclasses extract the tool ID from the request path |
| `getEndpointPrefix()` | Abstract — subclasses provide the endpoint prefix used during upstream WebSocket URI resolution |
| `getOrder()` | Runs one step after `RouteToRequestUrlFilter` to override the resolved URL |

## Usage Example

```java
@Component
public class MyToolWebSocketFilter extends ToolWebSocketProxyUrlFilter {

    public MyToolWebSocketFilter(
            ReactiveIntegratedToolRepository toolRepository,
            ToolUpstreamResolverRegistry upstreamRegistry) {
        super(toolRepository, upstreamRegistry);
    }

    @Override
    protected String getRequestToolId(String path) {
        // e.g. /ws/tools/{toolId}/...
        return path.split("/")[3];
    }

    @Override
    protected String getEndpointPrefix() {
        return "/ws";
    }

    @Override
    protected ServerWebExchange mutateExchange(ServerWebExchange exchange, IntegratedTool tool) {
        // Optionally strip or replace auth headers before forwarding
        return exchange.mutate()
                .request(r -> r.headers(h -> h.remove(HttpHeaders.AUTHORIZATION)))
                .build();
    }
}
```
