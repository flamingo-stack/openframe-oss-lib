Reactive Spring Cloud Gateway filter that intercepts WebSocket proxy requests for integrated tools, resolves the upstream WebSocket URI, and forwards the mutated exchange through the filter chain.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `ORIGINAL_AUTHORIZATION_ATTR` | `String` constant | Exchange attribute key used to preserve the original `Authorization` header before upstream mutation |
| `getOrder()` | Method | Sets filter precedence to `ROUTE_TO_URL_FILTER_ORDER + 1`, ensuring it runs immediately after route URL resolution |
| `filter()` | Method | Core reactive pipeline — extracts tool ID from path, validates the tool, resolves the WebSocket upstream URI, stores it in `GATEWAY_REQUEST_URL_ATTR`, and delegates to the chain |
| `getTool()` | Method (private) | Looks up the `IntegratedTool` by key and enforces enabled-state validation, emitting errors on missing or disabled tools |
| `mutateExchange()` | Method (protected) | Hook for subclasses to modify the `ServerWebExchange` (e.g. rewrite headers) before forwarding; returns the original exchange by default |
| `getRequestToolId()` | Abstract method | Subclass-defined path parsing to extract the tool identifier |
| `getEndpointPrefix()` | Abstract method | Subclass-defined prefix passed to the upstream resolver for WebSocket endpoint construction |

## Usage Example

```java
@Component
public class ZabbixWebSocketFilter extends ToolWebSocketProxyUrlFilter {

    public ZabbixWebSocketFilter(
            ReactiveIntegratedToolRepository toolRepository,
            ToolUpstreamResolverRegistry upstreamRegistry) {
        super(toolRepository, upstreamRegistry);
    }

    @Override
    protected String getRequestToolId(String path) {
        // e.g. /ws/zabbix/... -> "zabbix"
        return path.split("/")[2];
    }

    @Override
    protected String getEndpointPrefix() {
        return "/zabbix";
    }

    @Override
    protected ServerWebExchange mutateExchange(ServerWebExchange exchange, IntegratedTool tool) {
        // Strip or replace auth headers before forwarding upstream
        ServerHttpRequest mutated = exchange.getRequest().mutate()
                .header(HttpHeaders.AUTHORIZATION, "Bearer " + tool.getApiKey())
                .build();
        return exchange.mutate().request(mutated).build();
    }
}
```

> **Note:** The original `Authorization` header is preserved under `ORIGINAL_AUTHORIZATION_ATTR` on the exchange attributes, allowing downstream filters or logging to inspect the caller's credentials independently of whatever header is sent upstream.
