MongoDB repository interface for managing `ItemAssignment` documents, providing CRUD operations and targeted deletion capabilities for item-to-entity assignment records.

## Key Components

| Element | Description |
|---|---|
| `ItemAssignmentRepository` | Spring Data MongoDB repository interface for `ItemAssignment` documents |
| `deleteByItemIdAndTargetTypeAndTargetId` | Removes a specific assignment by item ID, target type, and target entity ID |
| `deleteByItemIdAndTargetType` | Removes all assignments for an item ID filtered by target type |

Extends both `MongoRepository<ItemAssignment, String>` and `CustomItemAssignmentRepository`, combining standard CRUD with custom query logic.

## Usage Example

```java
// Delete a specific assignment for a user target
itemAssignmentRepository.deleteByItemIdAndTargetTypeAndTargetId(
    "item-123",
    AssignmentTargetType.USER,
    "user-456"
);

// Delete all assignments for an item assigned to groups
itemAssignmentRepository.deleteByItemIdAndTargetType(
    "item-123",
    AssignmentTargetType.GROUP
);
```
