Implements `CustomMachineRepository` using `MongoTemplate` to provide cursor-based pagination, dynamic query building, and machine counting for the `Machine` collection in MongoDB.

## Key Components

| Method | Description |
|---|---|
| `findMachinesWithCursor` | Retrieves a paginated list of `Machine` documents using cursor-based navigation with configurable sort field and direction |
| `applyCursorCriteria` | Builds `$or` criteria to correctly paginate on non-`_id` sort fields, handling ties via a compound `(sortField, _id)` comparison |
| `getSortFieldValue` | Extracts the value of a sort field from a `Machine` instance via a switch expression |
| `buildDeviceQuery` | Constructs a `Query` from a `MachineQueryFilter` (status, device type, OS type, organization) and an optional full-text search regex against multiple fields |
| `countMachines` | Returns the total count of documents matching a pre-built `Query` |
| `isSortableField` | Validates that a requested sort field is in the allowed `SORTABLE_FIELDS` allowlist |
| `getDefaultSortField` | Returns `_id` as the fallback sort field |

## Usage Example

```java
// Build a filtered query
MachineQueryFilter filter = MachineQueryFilter.builder()
    .statuses(List.of("ONLINE"))
    .osTypes(List.of("WINDOWS"))
    .organizationIds(List.of("org_abc123"))
    .build();

Query query = repository.buildDeviceQuery(filter, "workstation");

// Count matching machines
long total = repository.countMachines(query);

// Fetch first page (no cursor)
List<Machine> firstPage = repository.findMachinesWithCursor(
    query, null, 25, "lastSeen", "DESC"
);

// Fetch next page using last document's ID as cursor
String cursor = firstPage.get(firstPage.size() - 1).getId().toString();
List<Machine> nextPage = repository.findMachinesWithCursor(
    query, cursor, 25, "lastSeen", "DESC"
);
```

> **Note:** Cursor-based pagination avoids offset drift on live data. When paginating on a non-unique sort field (e.g., `status`, `lastSeen`), ties are broken by `_id` to ensure stable, gap-free page traversal.
