MongoDB repository interface for managing `AgentRegistrationSecret` documents, providing lookup and existence-check operations for agent registration secrets.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `findByActiveTrue()` | Query Method | Returns the currently active registration secret, if one exists |
| `existsAny()` | `@Query` Method | Returns `true` if **any** secret document exists in the collection |

## Usage Example

```java
@Service
public class AgentRegistrationService {

    private final AgentRegistrationSecretRepository secretRepository;

    public AgentRegistrationService(AgentRegistrationSecretRepository secretRepository) {
        this.secretRepository = secretRepository;
    }

    public AgentRegistrationSecret getActiveSecret() {
        // Check if any secrets exist before fetching
        if (!secretRepository.existsAny()) {
            throw new IllegalStateException("No registration secrets configured");
        }

        return secretRepository.findByActiveTrue()
                .orElseThrow(() -> new IllegalStateException("No active registration secret found"));
    }
}
```

## Notes

- Extends `MongoRepository<AgentRegistrationSecret, String>`, inheriting standard CRUD operations with `String` as the document ID type.
- `findByActiveTrue()` returns an `Optional`, making it safe to handle cases where no active secret is set.
- `existsAny()` uses a match-all query (`{}`) with `exists = true`, providing a lightweight check without loading document data — useful for first-time setup detection.
