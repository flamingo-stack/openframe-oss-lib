MongoDB repository interface for managing `TenantKey` documents, providing lookup and counting operations scoped to active keys for a specific tenant.

## Key Components

| Member | Description |
|--------|-------------|
| `findFirstByTenantIdAndActiveTrue(String tenantId)` | Retrieves the first active `TenantKey` for the given tenant, returning an `Optional` |
| `countByTenantIdAndActiveTrue(String tenantId)` | Returns the total number of active keys associated with a tenant ID |

## Usage Example

```java
@Service
public class TenantKeyService {

    private final TenantKeyRepository tenantKeyRepository;

    public TenantKeyService(TenantKeyRepository tenantKeyRepository) {
        this.tenantKeyRepository = tenantKeyRepository;
    }

    public Optional<TenantKey> getActiveTenantKey(String tenantId) {
        return tenantKeyRepository.findFirstByTenantIdAndActiveTrue(tenantId);
    }

    public boolean hasActiveKeys(String tenantId) {
        return tenantKeyRepository.countByTenantIdAndActiveTrue(tenantId) > 0;
    }
}
```
