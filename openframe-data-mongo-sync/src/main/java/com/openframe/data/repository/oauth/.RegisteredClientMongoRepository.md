MongoDB repository interface for persisting and querying OAuth2 registered clients, extending Spring Data's `MongoRepository` with a custom lookup method by client ID.

## Key Components

| Component | Description |
|-----------|-------------|
| `RegisteredClientMongoRepository` | Spring Data MongoDB repository interface for `MongoRegisteredClient` documents |
| `findByClientId(String clientId)` | Custom derived query method to look up a registered OAuth2 client by its `clientId` field |

## Usage Example

```java
@Service
public class OAuth2ClientService {

    private final RegisteredClientMongoRepository repository;

    public OAuth2ClientService(RegisteredClientMongoRepository repository) {
        this.repository = repository;
    }

    public MongoRegisteredClient getClientById(String clientId) {
        return repository.findByClientId(clientId)
                .orElseThrow(() -> new ClientNotFoundException("Client not found: " + clientId));
    }

    public MongoRegisteredClient saveClient(MongoRegisteredClient client) {
        return repository.save(client);
    }
}
```

> **Note:** The primary key type is `String`, aligning with MongoDB's default `_id` field. The `findByClientId` method targets the OAuth2 `client_id` field, which is distinct from the document's internal `_id`.
