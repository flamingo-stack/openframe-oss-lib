MongoDB repository interface for managing `FeFeatureFlags` documents, providing standard CRUD operations via Spring Data MongoDB.

## Key Components

| Component | Description |
|-----------|-------------|
| `FeFeatureFlagsRepository` | Spring Data `MongoRepository` for `FeFeatureFlags` documents, keyed by `String` ID |

## Usage Example

```java
@Service
public class FeatureFlagService {

    private final FeFeatureFlagsRepository featureFlagsRepository;

    public FeatureFlagService(FeFeatureFlagsRepository featureFlagsRepository) {
        this.featureFlagsRepository = featureFlagsRepository;
    }

    public FeFeatureFlags getFlags(String id) {
        return featureFlagsRepository.findById(id)
                .orElseThrow(() -> new RuntimeException("Feature flags not found"));
    }

    public FeFeatureFlags saveFlags(FeFeatureFlags flags) {
        return featureFlagsRepository.save(flags);
    }

    public void deleteFlags(String id) {
        featureFlagsRepository.deleteById(id);
    }
}
```

Inherits all standard Spring Data methods such as `save()`, `findById()`, `findAll()`, and `deleteById()` with no additional query methods required.
