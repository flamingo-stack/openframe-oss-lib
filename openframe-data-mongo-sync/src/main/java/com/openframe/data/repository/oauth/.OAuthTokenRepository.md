Spring Data MongoDB repository interface for managing `OAuthToken` documents, providing lookup operations by access and refresh tokens.

## Key Components

| Element | Description |
|---|---|
| `findByAccessToken(String)` | Retrieves an `OAuthToken` by its access token value |
| `findByRefreshToken(String)` | Retrieves an `OAuthToken` by its refresh token value |

Both methods return `Optional<OAuthToken>` to safely handle cases where no matching token exists.

## Usage Example

```java
@Service
public class OAuthTokenService {

    private final OAuthTokenRepository tokenRepository;

    public OAuthTokenService(OAuthTokenRepository tokenRepository) {
        this.tokenRepository = tokenRepository;
    }

    public OAuthToken validateAccessToken(String accessToken) {
        return tokenRepository.findByAccessToken(accessToken)
                .orElseThrow(() -> new TokenNotFoundException("Invalid access token"));
    }

    public OAuthToken rotateToken(String refreshToken) {
        return tokenRepository.findByRefreshToken(refreshToken)
                .orElseThrow(() -> new TokenNotFoundException("Invalid refresh token"));
    }
}
```
