Spring Data MongoDB repository interface for persisting and retrieving `ApiKeyStats` documents, providing full CRUD and query operations out of the box via `MongoRepository`.

## Key Components

| Element | Description |
|---|---|
| `ApiKeyStatsMongoRepository` | Repository interface extending `MongoRepository` |
| `ApiKeyStats` | MongoDB document entity being managed |
| `String` | Type of the document's `_id` field |

## Usage Example

```java
@Service
public class ApiKeyStatsService {

    private final ApiKeyStatsMongoRepository apiKeyStatsRepository;

    public ApiKeyStatsService(ApiKeyStatsMongoRepository apiKeyStatsRepository) {
        this.apiKeyStatsRepository = apiKeyStatsRepository;
    }

    public ApiKeyStats save(ApiKeyStats stats) {
        return apiKeyStatsRepository.save(stats);
    }

    public Optional<ApiKeyStats> findById(String id) {
        return apiKeyStatsRepository.findById(id);
    }

    public List<ApiKeyStats> findAll() {
        return apiKeyStatsRepository.findAll();
    }
}
```

## Notes

- Inherits standard `MongoRepository` methods: `save()`, `findById()`, `findAll()`, `deleteById()`, etc.
- Custom query methods can be added by declaring method signatures following Spring Data naming conventions (e.g., `findByApiKeyId(String apiKeyId)`).
- Bean is auto-registered via `@Repository` and Spring Data's repository scanning.
