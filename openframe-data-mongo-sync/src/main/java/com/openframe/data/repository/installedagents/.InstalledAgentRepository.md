Spring Data MongoDB repository interface for managing `InstalledAgent` documents, providing CRUD operations and custom query methods for agent lookup by machine identity and type.

## Key Components

| Method | Description |
|--------|-------------|
| `findByMachineId` | Retrieves all agents installed on a specific machine |
| `findByMachineIdIn` | Batch lookup of agents across multiple machines |
| `findByMachineIdAndAgentType` | Finds a unique agent by machine ID and agent type combination |

## Usage Example

```java
@Service
public class AgentService {

    private final InstalledAgentRepository agentRepository;

    // Find all agents on a specific machine
    List<InstalledAgent> agents = agentRepository.findByMachineId("machine-001");

    // Batch lookup across multiple machines
    List<InstalledAgent> multiAgents = agentRepository
        .findByMachineIdIn(List.of("machine-001", "machine-002"));

    // Find a specific agent type on a machine
    Optional<InstalledAgent> agent = agentRepository
        .findByMachineIdAndAgentType("machine-001", "monitoring");
}
```

## Notes

- Extends `MongoRepository<InstalledAgent, String>` — inherits full CRUD via Spring Data
- `findByMachineIdAndAgentType` returns `Optional` to safely handle cases where the agent type is not installed on the target machine
- The `findByMachineIdIn` batch method is suited for fleet-wide agent inventory queries across multiple managed endpoints
