MongoDB repository interface for managing `TagAssignment` documents, providing CRUD and query operations for tag-to-entity associations filtered by entity type, tag ID, and tag values.

## Key Components

| Method | Description |
|--------|-------------|
| `findByEntityType` | Retrieves all assignments for a given entity type |
| `findByEntityIdAndEntityType` | Looks up assignments for a specific entity and type |
| `findByTagId` / `findByTagIdIn` | Fetches assignments by one or multiple tag IDs |
| `findByEntityIdInAndEntityType` | Bulk lookup of assignments across multiple entity IDs |
| `findByTagIdInAndEntityType` | Filters assignments by multiple tags and entity type |
| `findByEntityIdAndTagIdAndEntityType` | Exact match lookup returning an `Optional` |
| `findByEntityIdInAndTagIdInAndEntityType` | Intersection query across entity IDs and tag IDs |
| `deleteByEntityIdAndTagIdAndEntityType` | Removes a specific tag assignment |
| `deleteByEntityIdAndEntityType` | Removes all tag assignments for an entity |
| `deleteByTagId` | Removes all assignments linked to a tag |
| `findByValuesContainingAnyAndEntityType` | Custom query matching assignments whose `values` array contains any of the provided values |
| `findByTagIdInAndValuesContainingAnyAndEntityType` | Combines tag ID filtering with value array matching |

## Usage Example

```java
// Find all tag assignments for a specific ticket
List<TagAssignment> assignments = tagAssignmentRepository
    .findByEntityIdAndEntityType(ticketId, TagEntityType.TICKET);

// Bulk fetch assignments for multiple entities
List<TagAssignment> bulkAssignments = tagAssignmentRepository
    .findByEntityIdInAndEntityType(List.of(id1, id2, id3), TagEntityType.DEVICE);

// Find assignments by tag values (custom @Query)
List<TagAssignment> valueMatches = tagAssignmentRepository
    .findByValuesContainingAnyAndEntityType(
        List.of("critical", "high-priority"),
        TagEntityType.TICKET
    );

// Remove a specific tag from an entity
tagAssignmentRepository
    .deleteByEntityIdAndTagIdAndEntityType(entityId, tagId, TagEntityType.TICKET);
```
