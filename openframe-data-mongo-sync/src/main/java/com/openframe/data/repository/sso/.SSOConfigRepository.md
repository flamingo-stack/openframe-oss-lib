Spring Data MongoDB repository interface for managing `SSOConfig` documents, providing lookup operations by SSO provider and enabled status.

## Key Components

| Method | Return Type | Description |
|--------|-------------|-------------|
| `findByProvider(String provider)` | `Optional<SSOConfig>` | Retrieves a single SSO configuration by provider name (e.g., `"google"`, `"azure"`) |
| `findByEnabledTrue()` | `List<SSOConfig>` | Returns all SSO configurations where the `enabled` flag is `true` |

Inherits full CRUD operations from `MongoRepository<SSOConfig, String>`.

## Usage Example

```java
@Service
public class SSOService {

    private final SSOConfigRepository ssoConfigRepository;

    public SSOService(SSOConfigRepository ssoConfigRepository) {
        this.ssoConfigRepository = ssoConfigRepository;
    }

    public SSOConfig getProviderConfig(String provider) {
        return ssoConfigRepository.findByProvider(provider)
                .orElseThrow(() -> new IllegalArgumentException("SSO provider not found: " + provider));
    }

    public List<SSOConfig> getActiveProviders() {
        return ssoConfigRepository.findByEnabledTrue();
    }
}
```
