Spring Data MongoDB repository interface for managing `TicketAttachment` documents, providing CRUD operations and ticket-scoped queries for the OpenFrame platform.

## Key Components

| Method | Description |
|--------|-------------|
| `findByTicketId(String ticketId)` | Retrieves all attachments associated with a specific ticket |
| `findByTicketIdIn(List<String> ticketIds)` | Batch-fetches attachments across multiple tickets |
| `deleteByTicketId(String ticketId)` | Removes all attachments linked to a given ticket |

Inherits full CRUD operations from `MongoRepository<TicketAttachment, String>`.

## Usage Example

```java
// Fetch all attachments for a single ticket
List<TicketAttachment> attachments = ticketAttachmentRepository
    .findByTicketId("ticket-123");

// Batch fetch attachments for multiple tickets
List<String> ticketIds = List.of("ticket-123", "ticket-456");
List<TicketAttachment> batchAttachments = ticketAttachmentRepository
    .findByTicketIdIn(ticketIds);

// Delete all attachments when a ticket is removed
ticketAttachmentRepository.deleteByTicketId("ticket-123");
```

> **Note:** `deleteByTicketId` should be called during ticket deletion workflows to maintain referential integrity, since MongoDB does not enforce cascading deletes natively.
