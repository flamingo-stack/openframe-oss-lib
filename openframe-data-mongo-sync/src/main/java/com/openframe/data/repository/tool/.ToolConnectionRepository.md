MongoDB repository interface for managing `ToolConnection` documents, providing query methods to look up tool connections by machine, agent tool, and tool type.

## Key Components

| Method | Description |
|--------|-------------|
| `findByMachineId` | Retrieves all connections for a specific machine |
| `findByMachineIdIn` | Batch lookup of connections across multiple machines |
| `findFirstByAgentToolIdOrderByConnectedAtDesc` | Returns the most recent connection for a given agent tool |
| `findByMachineIdAndToolType` | Finds a unique connection by machine and tool type combination |
| `findByToolType` | Lists all connections of a specific tool type |
| `findByAgentToolIdInAndToolType` | Batch lookup filtered by agent tool IDs and tool type |

## Usage Example

```java
// Get the latest connection for an agent tool
Optional<ToolConnection> latest = toolConnectionRepository
    .findFirstByAgentToolIdOrderByConnectedAtDesc("agent-tool-123");

// Find all RMM connections across a set of agents
List<ToolConnection> rmmConnections = toolConnectionRepository
    .findByAgentToolIdInAndToolType(agentToolIds, ToolType.RMM);

// Lookup a specific machine's PSA connection
Optional<ToolConnection> psaConn = toolConnectionRepository
    .findByMachineIdAndToolType("machine-456", ToolType.PSA);
```
