A lightweight projection record used to hold aggregated notification counts grouped by category, typically returned from MongoDB aggregation queries.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `category` | `NotificationCategory` | The notification category, mapped from MongoDB's `_id` aggregation field |
| `count` | `long` | The total number of notifications for the given category |

## Usage Example

```java
// Typical use in a MongoDB aggregation pipeline returning CategoryCount projections
List<CategoryCount> results = mongoTemplate.aggregate(
    Aggregation.newAggregation(
        Aggregation.group("category").count().as("count")
    ),
    "notifications",
    CategoryCount.class
).getMappedResults();

results.forEach(cc ->
    System.out.println(cc.category() + ": " + cc.count())
);
```

> **Note:** The `@Field("_id")` annotation maps the MongoDB aggregation group key (`_id`) to the `category` field, which is standard when using `$group` stages in aggregation pipelines.
