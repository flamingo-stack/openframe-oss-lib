Spring Data MongoDB repository interface for managing `KnowledgeBaseItem` documents, providing standard CRUD operations alongside custom query methods for filtering and retrieving knowledge base entries.

## Key Components

| Method | Description |
|--------|-------------|
| `findByParentIdAndType` | Retrieves items matching a specific parent ID and item type |
| `findByParentId` | Retrieves all items belonging to a given parent |
| `findByIdIn` | Fetches a batch of items by a list of IDs |
| `findByTypeOrderByNameAsc` | Returns all items of a given type, sorted alphabetically by name |

**Extended Interfaces:**
- `MongoRepository<KnowledgeBaseItem, String>` — standard MongoDB CRUD and pagination
- `CustomKnowledgeBaseItemRepository` — project-specific custom query logic

## Usage Example

```java
@Service
public class KnowledgeBaseService {

    private final KnowledgeBaseItemRepository repository;

    public KnowledgeBaseService(KnowledgeBaseItemRepository repository) {
        this.repository = repository;
    }

    public List<KnowledgeBaseItem> getArticlesByCategory(String categoryId) {
        return repository.findByParentIdAndType(categoryId, KnowledgeBaseItemType.ARTICLE);
    }

    public List<KnowledgeBaseItem> getAllFoldersSorted() {
        return repository.findByTypeOrderByNameAsc(KnowledgeBaseItemType.FOLDER);
    }

    public List<KnowledgeBaseItem> fetchSelected(List<String> ids) {
        return repository.findByIdIn(ids);
    }
}
```
