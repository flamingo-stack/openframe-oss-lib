MongoDB repository interface for managing `OAuthClient` documents, providing lookup and existence-check operations by `clientId` and `machineId`.

## Key Components

| Method | Description |
|--------|-------------|
| `findByClientId(String clientId)` | Retrieves an `OAuthClient` by its OAuth client identifier |
| `findByMachineId(String machineId)` | Retrieves an `OAuthClient` by its associated machine identifier |
| `existsByMachineId(String machineId)` | Returns `true` if an `OAuthClient` exists for the given machine identifier |

Inherits full CRUD operations from `MongoRepository<OAuthClient, String>`.

## Usage Example

```java
@Service
public class OAuthClientService {

    private final OAuthClientRepository repository;

    public OAuthClientService(OAuthClientRepository repository) {
        this.repository = repository;
    }

    public OAuthClient getByClientId(String clientId) {
        return repository.findByClientId(clientId)
                .orElseThrow(() -> new EntityNotFoundException("Client not found: " + clientId));
    }

    public boolean isMachineRegistered(String machineId) {
        return repository.existsByMachineId(machineId);
    }

    public OAuthClient getByMachineId(String machineId) {
        return repository.findByMachineId(machineId)
                .orElseThrow(() -> new EntityNotFoundException("Machine not found: " + machineId));
    }
}
```
