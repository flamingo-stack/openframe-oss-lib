Spring Data MongoDB repository interface for managing `OAuthClient` documents, providing standard CRUD operations plus custom lookup methods by client ID and machine ID.

## Key Components

| Method | Return Type | Description |
|--------|-------------|-------------|
| `findByClientId(String clientId)` | `Optional<OAuthClient>` | Retrieves an OAuth client by its unique client ID |
| `existsByMachineId(String machineId)` | `boolean` | Checks whether an OAuth client is registered for a given machine ID |

Inherits all standard MongoDB CRUD operations from `MongoRepository<OAuthClient, String>`.

## Usage Example

```java
@Service
public class OAuthClientService {

    private final OAuthClientRepository oAuthClientRepository;

    public OAuthClientService(OAuthClientRepository oAuthClientRepository) {
        this.oAuthClientRepository = oAuthClientRepository;
    }

    public OAuthClient getClient(String clientId) {
        return oAuthClientRepository.findByClientId(clientId)
                .orElseThrow(() -> new EntityNotFoundException("OAuth client not found: " + clientId));
    }

    public boolean isMachineRegistered(String machineId) {
        return oAuthClientRepository.existsByMachineId(machineId);
    }
}
```
