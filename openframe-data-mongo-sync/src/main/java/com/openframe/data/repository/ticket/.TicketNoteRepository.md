Spring Data MongoDB repository interface for managing `TicketNote` documents, providing CRUD operations and custom query methods scoped to ticket associations.

## Key Components

| Method | Description |
|--------|-------------|
| `findByTicketId(String ticketId)` | Retrieves all notes associated with a specific ticket |
| `findByTicketId(String ticketId, Sort sort)` | Retrieves notes for a ticket with configurable sort ordering |
| `findByTicketIdIn(List<String> ticketIds)` | Batch-fetches notes across multiple ticket IDs |
| `deleteByTicketId(String ticketId)` | Removes all notes belonging to a specific ticket |

## Usage Example

```java
// Fetch notes for a ticket, sorted by creation date descending
Sort sort = Sort.by(Sort.Direction.DESC, "createdAt");
List<TicketNote> notes = ticketNoteRepository.findByTicketId("ticket-123", sort);

// Batch fetch notes for multiple tickets
List<String> ticketIds = List.of("ticket-123", "ticket-456");
List<TicketNote> allNotes = ticketNoteRepository.findByTicketIdIn(ticketIds);

// Cascade delete all notes when a ticket is removed
ticketNoteRepository.deleteByTicketId("ticket-123");
```

> **Note:** The `deleteByTicketId` method is useful for cascading cleanup when a parent ticket is deleted, ensuring no orphaned notes remain in the MongoDB collection.
