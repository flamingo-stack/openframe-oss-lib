Custom MongoDB query interface for ticket data access, providing advanced filtering, cursor-based pagination, bulk operations, and analytics beyond standard Spring Data repository capabilities.

## Key Components

| Method | Description |
|--------|-------------|
| `buildTicketQuery(...)` | Constructs a MongoDB `Query` from a `TicketQueryFilter`, with optional search, ID restriction, and machine owner filtering. Overloaded default for filter-only use. |
| `findTicketsWithCursor(...)` | Retrieves a paginated list of tickets using cursor-based pagination with configurable sort field and direction. |
| `countTickets(Query)` | Returns the total count of tickets matching a given query. |
| `countTicketsByStatus()` | Returns a `Map<TicketStatus, Long>` aggregating ticket counts grouped by status. |
| `getTotalCount()` | Returns the overall ticket count in the collection. |
| `getAverageResolutionTimeMs()` | Returns an `Optional<Long>` representing the average ticket resolution time in milliseconds. |
| `updateStatusBulk(...)` | Bulk-updates tickets from one status to another, returning the number of modified documents. |
| `updateTitle(...)` | Updates the title of a specific ticket by ID. |
| `isSortableField(String)` | Validates whether a given field name is allowed for sorting. |
| `getDefaultSortField()` | Returns the default field used for ticket sorting. |

## Usage Example

```java
// Build a filtered query and paginate with cursor
Query query = ticketRepository.buildTicketQuery(
    filter, "network issue", List.of("t1", "t2"), "machine-42"
);

List<Ticket> page = ticketRepository.findTicketsWithCursor(
    query, lastCursorId, 25, "createdAt", "DESC"
);

// Aggregate ticket counts by status
Map<TicketStatus, Long> statusCounts = ticketRepository.countTicketsByStatus();

// Bulk-close all open tickets
int updated = ticketRepository.updateStatusBulk(TicketStatus.OPEN, TicketStatus.CLOSED);

// Fetch average resolution time
ticketRepository.getAverageResolutionTimeMs()
    .ifPresent(ms -> log.info("Avg resolution: {}ms", ms));
```
