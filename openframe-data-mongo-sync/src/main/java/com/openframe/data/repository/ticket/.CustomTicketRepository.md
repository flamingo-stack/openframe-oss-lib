Custom repository interface extending ticket data access with advanced querying, filtering, cursor-based pagination, and bulk operation capabilities for MongoDB-backed ticket management.

## Key Components

| Method | Description |
|--------|-------------|
| `buildTicketQuery(filter)` | Default overload — builds a `Query` from a `TicketQueryFilter` with no search, restrictions, or owner constraints |
| `buildTicketQuery(filter, search, restrictToTicketIds, ownerMachineId)` | Full query builder supporting text search, ticket ID allowlist, and machine owner scoping |
| `findTicketsWithCursor(query, cursor, limit, sortField, sortDirection)` | Cursor-based paginated ticket retrieval with configurable sort |
| `countTickets(query)` | Returns total count matching a given `Query` |
| `countTicketsByStatus()` | Aggregates ticket counts grouped by `TicketStatus` |
| `countTicketsByStatusKind()` | Aggregates ticket counts grouped by `TicketStatusKind` |
| `countTicketsByStatusId()` | Aggregates ticket counts keyed by raw status ID string |
| `getTotalCount()` | Returns the total number of tickets in the collection |
| `getAverageResolutionTimeMs()` | Computes average time-to-resolution in milliseconds (may be absent) |
| `updateStatusBulk(fromStatus, toStatus)` | Bulk-transitions all tickets from one status to another; returns affected count |
| `reassignTicketsToStatus(fromStatusId, toStatusId, toKind)` | Reassigns tickets between status IDs and updates the status kind |
| `updateTitle(ticketId, title)` | Updates the title of a single ticket by ID |
| `isSortableField(field)` | Validates whether a field name is eligible for sorting |
| `getDefaultSortField()` | Returns the default sort field used when none is specified |

## Usage Example

```java
// Build a filtered query with search and pagination
Query query = ticketRepository.buildTicketQuery(
    filter,
    "network outage",
    List.of("TKT-001", "TKT-002"),
    "machine-abc123"
);

// Paginate using a cursor
List<Ticket> page = ticketRepository.findTicketsWithCursor(
    query, "cursor_xyz", 25, "createdAt", "DESC"
);

// Bulk status transition (e.g., auto-close stale open tickets)
int affected = ticketRepository.updateStatusBulk(openStatus, closedStatus);

// Aggregate counts for a dashboard
Map<TicketStatusKind, Long> kindCounts = ticketRepository.countTicketsByStatusKind();
```
