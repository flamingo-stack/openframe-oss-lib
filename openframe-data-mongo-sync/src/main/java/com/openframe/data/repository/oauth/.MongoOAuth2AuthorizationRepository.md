MongoDB repository interface for persisting and querying OAuth2 authorization records, extending Spring Data's `MongoRepository` for CRUD operations on `MongoOAuth2Authorization` documents.

## Key Components

| Method | Description |
|--------|-------------|
| `findByState` | Looks up an authorization by its OAuth2 state parameter (used during authorization code flow) |
| `findByAuthorizationCodeValue` | Retrieves an authorization by its authorization code |
| `findByAccessTokenValue` | Retrieves an authorization by its access token |
| `findByRefreshTokenValue` | Retrieves an authorization by its refresh token |

## Usage Example

```java
@Service
public class OAuth2AuthorizationService {

    private final MongoOAuth2AuthorizationRepository authorizationRepository;

    public OAuth2AuthorizationService(MongoOAuth2AuthorizationRepository authorizationRepository) {
        this.authorizationRepository = authorizationRepository;
    }

    public MongoOAuth2Authorization findByAccessToken(String token) {
        return authorizationRepository.findByAccessTokenValue(token)
                .orElseThrow(() -> new OAuth2AuthenticationException("Token not found"));
    }

    public MongoOAuth2Authorization findByAuthCode(String code) {
        return authorizationRepository.findByAuthorizationCodeValue(code)
                .orElseThrow(() -> new OAuth2AuthenticationException("Authorization code not found"));
    }
}
```

> All query methods return `Optional<MongoOAuth2Authorization>`, ensuring null-safe handling when a matching token or state does not exist in the store.
