Spring Data MongoDB repository interface for managing `OpenFrameClientConfiguration` documents, providing tenant-aware configuration lookup and retrieval of the most recent configuration entry.

## Key Components

| Member | Return Type | Description |
|--------|-------------|-------------|
| `findFirstByOrderByCreatedAtDesc()` | `Optional<OpenFrameClientConfiguration>` | Retrieves the most recently created client configuration document |
| `findByTenantId(String tenantId)` | `Optional<OpenFrameClientConfiguration>` | Looks up a client configuration by its associated tenant ID |

Extends `MongoRepository<OpenFrameClientConfiguration, String>`, inheriting standard CRUD operations with `String` as the document ID type.

## Usage Example

```java
@Service
public class ClientConfigService {

    private final OpenFrameClientConfigurationRepository repository;

    public ClientConfigService(OpenFrameClientConfigurationRepository repository) {
        this.repository = repository;
    }

    public OpenFrameClientConfiguration getConfigForTenant(String tenantId) {
        return repository.findByTenantId(tenantId)
            .orElseThrow(() -> new ConfigNotFoundException("No config found for tenant: " + tenantId));
    }

    public OpenFrameClientConfiguration getLatestConfig() {
        return repository.findFirstByOrderByCreatedAtDesc()
            .orElseThrow(() -> new ConfigNotFoundException("No client configuration exists"));
    }
}
```
