A simple Java record that pairs a `Notification` document with its corresponding `ReadStatus`, used to represent a notification alongside its read/unread state in a single immutable data object.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `notification` | `Notification` | The notification document entity |
| `status` | `ReadStatus` | The read/unread status associated with the notification |

## Usage Example

```java
Notification notification = notificationRepository.findById(id);
ReadStatus status = readStatusRepository.findByNotificationId(id);

NotificationWithStatus notificationWithStatus = new NotificationWithStatus(notification, status);

// Access components via auto-generated accessors
Notification n = notificationWithStatus.notification();
ReadStatus s = notificationWithStatus.status();
```
