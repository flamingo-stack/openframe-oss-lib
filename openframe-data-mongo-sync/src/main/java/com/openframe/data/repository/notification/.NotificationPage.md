A simple record that wraps a paginated list of `NotificationWithStatus` items, serving as a container for notification query results.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `items` | `List<NotificationWithStatus>` | The list of notifications with their associated status |
| `of(...)` | `static factory method` | Convenience constructor for creating a `NotificationPage` instance |

## Usage Example

```java
List<NotificationWithStatus> notifications = notificationRepository.findAll();
NotificationPage page = NotificationPage.of(notifications);

// Access items
page.items().forEach(n -> System.out.println(n.status()));
```
