MongoDB repository interface for `ReleaseVersion` document operations, providing tenant-scoped access to the single release version record stored per tenant.

## Key Components

| Method | Description |
|--------|-------------|
| `findFirstByOrderByCreatedAtAsc()` | Retrieves the earliest release version document ordered by creation date |
| `findFirstBy()` | Returns the single release version for the current tenant; automatically scoped by `TenantAwareMongoTemplate` |

## Usage Example

```java
@Service
public class ReleaseVersionService {

    private final ReleaseVersionRepository releaseVersionRepository;

    public ReleaseVersionService(ReleaseVersionRepository releaseVersionRepository) {
        this.releaseVersionRepository = releaseVersionRepository;
    }

    public ReleaseVersion getCurrentVersion() {
        return releaseVersionRepository.findFirstBy()
                .orElseThrow(() -> new IllegalStateException("No release version found for tenant"));
    }

    public ReleaseVersion getEarliestVersion() {
        return releaseVersionRepository.findFirstByOrderByCreatedAtAsc()
                .orElseThrow(() -> new IllegalStateException("No release version records found"));
    }
}
```

## Notes

- Extends `MongoRepository<ReleaseVersion, String>` — inherits standard CRUD operations
- Designed to maintain a **single document per tenant**; `TenantAwareMongoTemplate` handles automatic tenant scoping on `findFirstBy()`
- Both methods return `Optional<ReleaseVersion>` — callers should handle the empty case explicitly
