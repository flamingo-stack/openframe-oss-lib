Custom repository interface for querying `ItemAssignment` documents with advanced filtering, cursor-based pagination, and aggregation support beyond standard Spring Data capabilities.

## Key Components

| Method | Description |
|--------|-------------|
| `findAssignmentsWithCursor` | Fetches paginated assignments for an item using cursor-based navigation with optional search, sort field, and direction filters |
| `countAssignments` | Returns the total count of assignments matching item ID, target type, and search criteria |
| `countByItemIdGroupedByTargetType` | Aggregates assignment counts grouped by `AssignmentTargetType` for a given item |
| `isSortableField` | Validates whether a given field name is eligible for sorting |
| `getDefaultSortField` | Returns the default field used for sorting when none is specified |

## Usage Example

```java
// Inject via Spring - implementation provided by CustomItemAssignmentRepositoryImpl
@Autowired
private CustomItemAssignmentRepository customRepo;

// Fetch first page of USER-type assignments for an item
List<ItemAssignment> page = customRepo.findAssignmentsWithCursor(
    "item-123",
    AssignmentTargetType.USER,
    "john",         // search term
    "createdAt",    // sort field
    "DESC",         // sort direction
    null,           // no cursor = first page
    20              // page size
);

// Count all assignments grouped by target type
Map<AssignmentTargetType, Long> counts =
    customRepo.countByItemIdGroupedByTargetType("item-123");
// e.g., { USER=12, GROUP=4, ROLE=1 }

// Validate sort input before querying
if (customRepo.isSortableField(requestedSort)) {
    // safe to use requestedSort
} else {
    requestedSort = customRepo.getDefaultSortField();
}
```
