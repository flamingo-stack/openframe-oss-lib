MongoDB repository interface for managing `ConnectorAlert` documents, providing query methods to retrieve active (unresolved) connector alerts.

## Key Components

| Method | Description |
|--------|-------------|
| `findByConnectorNameAndResolvedFalse(String connectorName)` | Retrieves a single active alert for a specific connector by name |
| `findByResolvedFalse()` | Returns all currently unresolved alerts across all connectors |

Inherits full CRUD operations from `MongoRepository<ConnectorAlert, String>`.

## Usage Example

```java
@Service
public class ConnectorAlertService {

    private final ConnectorAlertRepository alertRepository;

    public ConnectorAlertService(ConnectorAlertRepository alertRepository) {
        this.alertRepository = alertRepository;
    }

    // Check if a specific connector has an active alert
    public Optional<ConnectorAlert> getActiveAlert(String connectorName) {
        return alertRepository.findByConnectorNameAndResolvedFalse(connectorName);
    }

    // Fetch all unresolved alerts for dashboard display
    public List<ConnectorAlert> getAllActiveAlerts() {
        return alertRepository.findByResolvedFalse();
    }
}
```
