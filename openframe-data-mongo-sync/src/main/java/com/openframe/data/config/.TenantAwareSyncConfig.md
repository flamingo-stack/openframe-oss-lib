Activates MongoDB repository scanning with tenant isolation support, enabled only when both `spring.data.mongodb.enabled` and `openframe.tenant-isolation.enabled` are set to `true`.

## Key Components

- **`@ConditionalOnExpression`** — Guards configuration activation using a SpEL expression that requires both MongoDB and tenant isolation to be explicitly enabled
- **`@EnableMongoRepositories`** — Scans `com.openframe.data.repository` for Spring Data MongoDB repository interfaces when conditions are met

## Usage Example

```java
// application.yml
spring:
  data:
    mongodb:
      enabled: true
      uri: mongodb://localhost:27017/openframe

openframe:
  tenant-isolation:
    enabled: true
```

When both flags are `true`, Spring bootstraps all MongoDB repositories under `com.openframe.data.repository` with tenant-aware context. If either flag is `false`, the configuration is skipped entirely — preventing repository registration in non-multi-tenant or non-MongoDB deployments.
