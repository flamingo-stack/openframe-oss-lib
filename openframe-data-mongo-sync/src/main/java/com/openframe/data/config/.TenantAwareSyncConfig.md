Conditionally enables MongoDB repository scanning when both MongoDB and tenant isolation are active in the OpenFrame platform.

## Key Components

- **`@ConditionalOnExpression`** — Activates this configuration only when both `spring.data.mongodb.enabled` and `openframe.tenant-isolation.enabled` resolve to `true`
- **`@EnableMongoRepositories`** — Scans `com.openframe.data.repository` for MongoDB repository interfaces when the condition is met

## Usage Example

```java
// application.yml — enable both flags to activate this config
spring:
  data:
    mongodb:
      enabled: true
      uri: mongodb://localhost:27017/openframe

openframe:
  tenant-isolation:
    enabled: true
```

When either flag is `false`, the MongoDB repositories under `com.openframe.data.repository` are not registered, preventing tenant-scoped data access from being wired into the application context.
