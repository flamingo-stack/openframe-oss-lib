Provides GCS (Google Cloud Storage) presigned URL generation and basic file operations using Application Default Credentials (ADC) and V4 signatures.

## Key Components

| Method | Description |
|--------|-------------|
| `generateUploadUrl` | Creates a V4-signed PUT URL for client-side file uploads |
| `generateDownloadUrl` | Creates a V4-signed GET URL for temporary file access |
| `deleteFile` | Deletes a blob from the configured GCS bucket |
| `moveFile` | Copies a blob to a new path then deletes the source |
| `withPrefix` | Prepends the configured prefix to object paths when enabled |

**Configuration properties:**

| Property | Description |
|----------|-------------|
| `storage.s3.bucket` | Target GCS bucket name |
| `storage.s3.prefix` | Optional path prefix for all objects |
| `storage.s3.prefix-enabled` | Enables/disables prefix prepending |
| `storage.s3.disabled` | Must be `false` to activate this service (via `@ConditionalOnProperty`) |

> **IAM Requirement:** The service account must have the `roles/iam.serviceAccountTokenCreator` role to sign URLs via ADC.

## Usage Example

```java
// Inject the service
@Autowired
private GcsPresignedUrlService gcsPresignedUrlService;

// Generate a 15-minute upload URL
String uploadUrl = gcsPresignedUrlService.generateUploadUrl(
    "uploads/document.pdf",
    "application/pdf",
    Duration.ofMinutes(15)
);

// Generate a 1-hour download URL
String downloadUrl = gcsPresignedUrlService.generateDownloadUrl(
    "uploads/document.pdf",
    Duration.ofHours(1)
);

// Move a file from temp to permanent storage
gcsPresignedUrlService.moveFile(
    "temp/document.pdf",
    "archive/document.pdf"
);
```
