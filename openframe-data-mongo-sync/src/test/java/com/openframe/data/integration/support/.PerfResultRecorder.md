Utility class for collecting and rendering performance benchmark results as structured Markdown reports. It accumulates rows of key-value metrics across named sections and writes a timestamped `.md` file with tables, environment details, and summary headlines.

## Key Components

| Member | Description |
|---|---|
| `PerfResultRecorder(String suiteName)` | Constructor — sets the report title used in the filename and heading |
| `withMeta(key, value)` | Adds a run parameter entry (e.g. batch size, iterations) to the "Dataset and run parameters" section |
| `withEnvironment(key, value)` | Adds an environment entry (e.g. JVM version, host) to the "Environment" section |
| `withOverview(String narrative)` | Sets a free-text narrative block rendered directly below the title |
| `describe(section, narrative)` | Attaches a descriptive paragraph to a named section |
| `record(section, keysAndValues...)` | Appends a metric row to a named section and logs it via SLF4J |
| `headline(columnsHeader, keysAndValues...)` | Adds a row to the top-level "Summary" table |
| `writeMarkdown(Path baseDir)` | Renders and writes the report to a timestamped `.md` file under `baseDir` |
| `render()` | Internal builder that assembles the full Markdown string |
| `renderTable(out, rows)` | Static helper that writes a GFM-compatible pipe table from a list of row maps |

## Usage Example

```java
PerfResultRecorder recorder = new PerfResultRecorder("ingestion-perf")
    .withEnvironment("JVM", System.getProperty("java.version"))
    .withMeta("Records", 100_000)
    .withMeta("Threads", 8)
    .withOverview("Measures end-to-end ingestion throughput under concurrent load.")
    .describe("Batch Insert", "Each row represents one batch of 1 000 records.");

// Record per-run metrics
recorder.record("Batch Insert",
    "Batch #", 1,
    "Duration ms", 342,
    "Records/s", 2923);

// Add a headline summary row
recorder.headline("Scenario | Duration ms | Records/s",
    "Scenario", "Batch Insert",
    "Duration ms", 342,
    "Records/s", 2923);

// Write the report
recorder.writeMarkdown(Path.of("build/perf-results"));
```

The output file (e.g. `build/perf-results/ingestion-perf-2024-06-01T12-00-00-000Z.md`) contains an Environment section, a Dataset section, a Summary table, and one table per named section.
