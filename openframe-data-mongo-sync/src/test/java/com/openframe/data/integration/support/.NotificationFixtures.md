Provides pre-built `Notification` test fixtures for use in integration tests, offering reusable factory methods that construct valid `Notification` objects with `GenericContext`.

## Key Components

| Method | Description |
|--------|-------------|
| `basic()` | Returns a default `Notification` with a `"welcome"` type and a hardcoded JSON payload |
| `basic(String type, String payload)` | Returns a parameterized `Notification` using the provided type and JSON payload string |

## Usage Example

```java
// Default welcome notification
Notification notification = NotificationFixtures.basic();

// Custom type and payload
Notification custom = NotificationFixtures.basic(
    "alert",
    "{\"message\":\"Disk usage at 90%\"}"
);
```

> Both methods set `createdAt` to `Instant.now()` and use `GenericContext` as the context type. This class is not instantiable — use its static factory methods directly in test setup or assertion helpers.
