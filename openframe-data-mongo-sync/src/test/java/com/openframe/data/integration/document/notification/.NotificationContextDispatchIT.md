Integration test verifying that `Notification` documents with polymorphic context objects are correctly persisted to and deserialized from MongoDB, using `type` as the sole discriminator field (no `_class` leakage).

## Key Components

| Component | Role |
|-----------|------|
| `NotificationContextDispatchIT` | Main test class extending `BaseMongoIntegrationTest` |
| `NotificationRepository` | Spring Data repository under test for CRUD operations |
| `MongoTemplate` | Used for raw document inspection and collection teardown |
| `TestExtendedContext` | Concrete `NotificationContext` subclass with extra fields (`extraField`, `extraNumber`) |
| `GenericContext` | Fallback context type carrying an opaque JSON payload string |

## Test Cases

**`given_extended_context_when_reread_then_dispatches_to_concrete_subclass`**
Saves a `Notification` with a `TestExtendedContext`, re-reads by ID, and asserts the context deserializes back to the correct concrete subclass with all extra fields intact.

**`given_generic_context_when_reread_then_round_trips_as_generic`**
Saves a `Notification` with a `GenericContext` carrying a raw JSON payload, re-reads it, and asserts the type string and payload survive the round-trip.

**`given_saved_notification_when_inspecting_raw_doc_then_context_has_no_class_field`**
Queries the raw MongoDB document directly via `MongoTemplate` and asserts the embedded `context` subdocument contains a `type` field but **no** `_class` field, confirming the discriminator strategy is clean.

## Usage Example

```java
// Typical pattern exercised by these tests:
Notification notification = Notification.builder()
    .title("My Event")
    .createdAt(Instant.now())
    .context(TestExtendedContext.builder()
        .extraField("value")
        .extraNumber(7)
        .build())
    .build();

Notification saved = repository.save(notification);
Optional<Notification> found = repository.findById(saved.getId());

TestExtendedContext ctx = (TestExtendedContext) found.get().getContext();
assertThat(ctx.getExtraField()).isEqualTo("value");
```

> **Run condition:** Tests are guarded by `@EnabledIfSystemProperty(named = "integration.tests", matches = "true")` and require a live MongoDB instance via `BaseMongoIntegrationTest`.
