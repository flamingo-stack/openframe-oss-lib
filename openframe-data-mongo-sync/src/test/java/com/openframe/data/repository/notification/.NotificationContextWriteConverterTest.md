Unit test suite for `NotificationContextWriteConverter`, validating the conversion of `NotificationContext` objects into BSON `Document` instances for MongoDB persistence.

## Key Components

| Test Method | Purpose |
|---|---|
| `given_null_source_when_converting_then_returns_null` | Verifies null input returns null without invoking Jackson |
| `given_generic_context_when_converting_then_document_contains_type_and_payload` | Confirms a populated `GenericContext` produces a BSON document with correct `type` and `payload` fields |
| `given_jackson_failure_when_converting_then_throws_with_type_in_message` | Ensures serialization failures surface as `IllegalStateException` with the discriminator type embedded in the message |

## Coverage Summary

```text
Null guard         → short-circuits without ObjectMapper invocation
Happy path         → discriminator + payload fields present in output Document
Error path         → JsonProcessingException wrapped into IllegalStateException
                     with type name for log diagnosability
```

## Usage Example

```java
// Production wiring (Spring Data MongoDB converter registration)
ObjectMapper objectMapper = new ObjectMapper();
NotificationContextWriteConverter converter =
        new NotificationContextWriteConverter(objectMapper);

GenericContext ctx = GenericContext.builder()
        .type("ALERT")
        .payload("{\"message\":\"Disk usage at 90%\"}")
        .build();

Document bson = converter.convert(ctx);
// bson.getString("type")    → "ALERT"
// bson.getString("payload") → "{\"message\":\"Disk usage at 90%\"}"
```

## Design Notes

- Uses **Mockito BDD-style** stubbing (`given(...).willThrow(...)`) for the error path, keeping mock setup expressive and readable
- The `IllegalStateException` message deliberately includes the discriminator type so production log searches can pinpoint which context variant triggered the failure
- Jackson is **not invoked** on null input, preventing unnecessary mock interactions and reflecting the converter's early-exit contract
