A Spring component that listens to NATS JetStream for installed agent messages on the `machine.*.installed-agent` subject, processes them via `InstalledAgentService`, and manages durable consumer lifecycle with explicit acknowledgment.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `subscribeToInstalledAgents()` | `@EventListener` | Bootstraps JetStream push subscription on `ApplicationReadyEvent` |
| `buildConsumerConfig()` | `private` | Creates or updates a durable NATS consumer; handles 404 (consumer not found) gracefully |
| `handleMessage(Message)` | `private` | Deserializes `InstalledAgentMessage`, extracts `machineId` from subject, delegates to service, and acks on success |
| `isLastAttempt(long)` | `private` | Returns `true` when `deliveredCount` equals `MAX_DELIVER` (50), signaling the final redelivery attempt |
| `cleanup()` | `@PreDestroy` | Unsubscribes and drains the dispatcher with a 5-second grace period on shutdown |

**Stream constants:**

| Constant | Value |
|----------|-------|
| `STREAM_NAME` | `INSTALLED_AGENTS` |
| `SUBJECT` | `machine.*.installed-agent` |
| `CONSUMER_NAME` | `installed-agent-processor-v1` |
| `MAX_DELIVER` | `50` |
| `ACK_WAIT` | `30s` |

## Usage Example

```java
// Bean is auto-registered by Spring; no manual wiring needed.
// On ApplicationReadyEvent, the listener self-subscribes:

// Incoming NATS subject example:
// machine.abc-123.installed-agent

// Payload (InstalledAgentMessage JSON):
// { "agentType": "endpoint-protection", "version": "2.1.0" }

// On success  → message.ack()
// On failure  → message left unacked for redelivery (up to MAX_DELIVER=50 attempts)
// On shutdown → subscription.unsubscribe() + dispatcher.drain(5s)
```

> **Redelivery behavior:** If processing throws an exception, the message is intentionally left unacknowledged. NATS will redeliver it after the 30-second `ACK_WAIT` window. On the 50th delivery, `isLastAttempt` returns `true`, allowing `InstalledAgentService` to apply final-attempt logic (e.g., dead-letter handling).
