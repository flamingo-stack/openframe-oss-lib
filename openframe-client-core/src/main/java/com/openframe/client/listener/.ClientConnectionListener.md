Handles NATS messaging events for client machine connection and disconnection status updates. This listener processes JSON messages and updates machine status through the MachineStatusService.

## Key Components

- **machineConnectedConsumer()** - Bean that processes machine connection events and updates status to online
- **machineDisconnectionConsumer()** - Bean that processes machine disconnection events and updates status to offline
- **ObjectMapper** - Deserializes JSON messages into ClientConnectionEvent objects
- **MachineStatusService** - Service dependency for updating machine status

## Usage Example

```java
// The beans are automatically registered with Spring's messaging framework
// Messages are consumed automatically when published to the corresponding topics

// Example ClientConnectionEvent JSON structure:
{
  "client": {
    "name": "machine-001"
  },
  "timestamp": "2023-12-01T10:30:00Z"
}

// Manual service usage (if needed):
@Autowired
private MachineStatusService machineStatusService;

public void updateMachineStatus() {
    machineStatusService.updateToOnline("machine-001", Instant.now());
    machineStatusService.updateToOffline("machine-001", Instant.now());
}
```
