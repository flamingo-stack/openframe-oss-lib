Configuration class that defines a password encoder bean for secure password hashing in the OpenFrame client application using BCrypt algorithm.

## Key Components

- **`passwordEncoder()`** - Bean method that creates and configures a BCryptPasswordEncoder instance for password hashing and verification

## Usage Example

```java
// The PasswordEncoder bean can be injected and used throughout the application
@Service
public class UserService {
    
    @Autowired
    private PasswordEncoder passwordEncoder;
    
    public void createUser(String rawPassword) {
        // Encode password before storing
        String encodedPassword = passwordEncoder.encode(rawPassword);
        // Save user with encoded password
    }
    
    public boolean validatePassword(String rawPassword, String encodedPassword) {
        // Verify password against stored hash
        return passwordEncoder.matches(rawPassword, encodedPassword);
    }
}
```

This configuration enables secure password handling across the application by providing a centralized BCrypt encoder that automatically generates salted hashes and can verify passwords against existing hashes.
