This Spring configuration class provides a centralized bean definition for password encoding functionality using BCrypt hashing algorithm for secure password storage.

## Key Components

- **`PasswordEncoderConfig`** - Spring configuration class that defines password encoding beans
- **`passwordEncoder()`** - Bean method that creates and returns a BCryptPasswordEncoder instance

## Usage Example

The password encoder bean can be injected into other Spring components for secure password handling:

```java
@Service
public class UserService {
    
    private final PasswordEncoder passwordEncoder;
    
    public UserService(PasswordEncoder passwordEncoder) {
        this.passwordEncoder = passwordEncoder;
    }
    
    public void createUser(String rawPassword) {
        // Encode password before storing
        String encodedPassword = passwordEncoder.encode(rawPassword);
        // Save to database...
    }
    
    public boolean validatePassword(String rawPassword, String storedPassword) {
        // Verify password against stored hash
        return passwordEncoder.matches(rawPassword, storedPassword);
    }
}
```

The BCrypt algorithm automatically handles salt generation and provides strong protection against rainbow table attacks, making it ideal for production password security requirements.
