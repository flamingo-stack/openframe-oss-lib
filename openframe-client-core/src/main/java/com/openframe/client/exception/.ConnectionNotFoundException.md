Custom exception for missing connection resources in the OpenFrame client layer, extending the core `NotFoundException` type.

## Key Components

- **`ConnectionNotFoundException`** — Signals that a requested connection could not be found; wraps a human-readable message passed to the parent `NotFoundException`.

## Usage Example

```java
Connection conn = connectionRepository.findById(id)
    .orElseThrow(() -> new ConnectionNotFoundException(
        "Connection not found with id: " + id
    ));
```
