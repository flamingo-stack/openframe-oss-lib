A custom runtime exception thrown when a required connection cannot be found or established in the OpenFrame client.

## Key Components

- **ConnectionNotFoundException**: A runtime exception class that extends `RuntimeException`
- **Constructor**: Takes a string message parameter to provide context about the connection failure

## Usage Example

```java
// Throwing the exception when a database connection is not found
public Connection getConnection(String connectionId) {
    Connection conn = connectionPool.get(connectionId);
    if (conn == null) {
        throw new ConnectionNotFoundException(
            "Connection with ID '" + connectionId + "' not found"
        );
    }
    return conn;
}

// Catching and handling the exception
try {
    Connection conn = connectionManager.getConnection("db-primary");
    // Use connection...
} catch (ConnectionNotFoundException e) {
    logger.error("Failed to obtain connection: " + e.getMessage());
    // Implement fallback logic or retry mechanism
}
```

This exception is typically used in connection management scenarios where a specific connection identifier or connection pool reference cannot be resolved, allowing for proper error handling and debugging in distributed systems or database connectivity layers.
