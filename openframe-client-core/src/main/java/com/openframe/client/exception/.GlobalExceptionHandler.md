A centralized exception handler that intercepts and processes all exceptions thrown across the Spring Boot application, converting them into standardized HTTP error responses with appropriate status codes and error messages.

## Key Components

- **@ControllerAdvice**: Global exception handling annotation that applies to all controllers
- **Exception Handlers**: Methods for specific exception types including:
  - `AccessDeniedException` → 401 Unauthorized
  - `MethodArgumentNotValidException` → 400 Bad Request with validation details
  - Custom exceptions (`InvalidAgentIdException`, `ConnectionNotFoundException`, etc.)
  - Generic `Exception` → 500 Internal Server Error
- **ErrorResponse**: Standardized response format with error codes and messages
- **Logging**: All exceptions are logged with appropriate error levels

## Usage Example

```java
// When a validation error occurs in a controller:
@PostMapping("/agents")
public ResponseEntity<Agent> createAgent(@Valid @RequestBody AgentRequest request) {
    // If validation fails, GlobalExceptionHandler automatically catches
    // MethodArgumentNotValidException and returns:
    // HTTP 400 with {"errorCode": "bad_request", "message": "Field errors..."}
}

// Custom exception handling:
public void validateAgent(String agentId) {
    if (!isValidId(agentId)) {
        throw new InvalidAgentIdException("Agent ID must be valid UUID");
        // Automatically returns HTTP 400 with appropriate error response
    }
}
```
