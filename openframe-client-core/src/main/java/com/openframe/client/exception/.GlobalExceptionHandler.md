A centralized exception handler for the OpenFrame client application that intercepts and standardizes error responses across all controllers, providing consistent error formatting and logging.

## Key Components

- **@ControllerAdvice**: Global exception handling for all controllers
- **Exception Handlers**: Specialized methods for different exception types
  - `handleAccessDeniedException()` - 401 Unauthorized responses
  - `handleValidationException()` - 400 Bad Request with validation details
  - `handleConnectionNotFoundException()` - 404 Not Found for missing resources
  - `handleDuplicateConnectionException()` - 409 Conflict for duplicate resources
  - `handleException()` - Catch-all for unexpected errors
- **ErrorResponse**: Standardized error response format with error codes
- **Logging**: SLF4J logging for all exceptions with appropriate levels

## Usage Example

```java
// Exception handlers automatically intercept thrown exceptions
@RestController
public class AgentController {
    
    @PostMapping("/agents")
    public ResponseEntity<Agent> createAgent(@Valid @RequestBody AgentRequest request) {
        // If validation fails, MethodArgumentNotValidException is thrown
        // GlobalExceptionHandler.handleValidationException() returns:
        // HTTP 400: {"errorCode": "bad_request", "message": "Field validation errors"}
        
        if (invalidAgentId(request.getAgentId())) {
            throw new InvalidAgentIdException("Invalid agent ID format");
            // Returns HTTP 400: {"errorCode": "bad_request", "message": "Invalid agent ID format"}
        }
        
        return ResponseEntity.ok(agentService.create(request));
    }
}

// Custom exceptions are mapped to appropriate HTTP status codes
throw new ConnectionNotFoundException("Connection not found");
// Returns HTTP 404: {"errorCode": "not_found", "message": "Connection not found"}

throw new DuplicateConnectionException("Connection already exists");
// Returns HTTP 409: {"errorCode": "conflict", "message": "Connection already exists"}
```
