A custom runtime exception thrown when an invalid or malformed agent ID is encountered during client operations.

## Key Components

- **InvalidAgentIdException**: A runtime exception class that extends `RuntimeException`
- **Constructor**: Accepts a string message parameter to provide specific error details

## Usage Example

```java
// Throwing the exception when validating an agent ID
public void validateAgentId(String agentId) {
    if (agentId == null || agentId.trim().isEmpty()) {
        throw new InvalidAgentIdException("Agent ID cannot be null or empty");
    }
    
    if (!agentId.matches("^[A-Z0-9]{8}$")) {
        throw new InvalidAgentIdException("Invalid agent ID format: " + agentId);
    }
}

// Catching and handling the exception
try {
    validateAgentId("INVALID_ID");
} catch (InvalidAgentIdException e) {
    System.err.println("Agent validation failed: " + e.getMessage());
    // Handle the error appropriately
}
```

This exception is typically used in agent management systems where specific agent ID formats or validation rules must be enforced, providing clear error messaging when validation fails.
