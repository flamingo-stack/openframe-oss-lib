A runtime exception thrown when an invalid agent identifier is encountered during OpenFrame client operations.

## Key Components

- **InvalidAgentIdException**: Custom runtime exception class that extends `RuntimeException`
- **Constructor**: Single parameterized constructor accepting an error message string

## Usage Example

```java
// Throwing the exception when agent ID validation fails
public void validateAgentId(String agentId) {
    if (agentId == null || agentId.trim().isEmpty()) {
        throw new InvalidAgentIdException("Agent ID cannot be null or empty");
    }
    
    if (!agentId.matches("^[a-zA-Z0-9-_]+$")) {
        throw new InvalidAgentIdException("Invalid agent ID format: " + agentId);
    }
}

// Catching and handling the exception
try {
    client.connectToAgent(agentId);
} catch (InvalidAgentIdException e) {
    logger.error("Failed to connect: " + e.getMessage());
    // Handle invalid agent ID scenario
}
```

This exception is part of the OpenFrame client's error handling system, providing specific feedback when agent identification fails validation or lookup operations.
