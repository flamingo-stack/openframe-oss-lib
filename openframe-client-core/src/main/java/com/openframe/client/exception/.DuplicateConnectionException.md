A custom runtime exception thrown when attempting to create a duplicate connection in the OpenFrame client system.

## Key Components

- **DuplicateConnectionException** - Custom runtime exception class that extends `RuntimeException`
- **Constructor** - Single parameterized constructor accepting an error message

## Usage Example

```java
// Throwing the exception when a duplicate connection is detected
if (connectionExists(clientId)) {
    throw new DuplicateConnectionException(
        "Connection already exists for client: " + clientId
    );
}

// Catching the exception in connection management
try {
    connectionManager.createConnection(clientId);
} catch (DuplicateConnectionException e) {
    logger.error("Failed to create connection: " + e.getMessage());
    // Handle duplicate connection scenario
}
```

This exception is typically used in connection pooling or client management scenarios where only one active connection per client is allowed, helping to enforce connection uniqueness constraints at the application level.
