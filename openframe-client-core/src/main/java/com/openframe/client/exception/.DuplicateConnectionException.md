A custom runtime exception thrown when attempting to create a connection that already exists in the OpenFrame client system.

## Key Components

- **DuplicateConnectionException**: A runtime exception class that extends `RuntimeException`
- **Constructor**: Takes a string message parameter to describe the specific duplicate connection error

## Usage Example

```java
// Throwing the exception when a duplicate connection is detected
public void createConnection(String connectionId) {
    if (connectionExists(connectionId)) {
        throw new DuplicateConnectionException(
            "Connection with ID '" + connectionId + "' already exists"
        );
    }
    // Proceed with connection creation
}

// Catching the exception
try {
    connectionManager.createConnection("conn-123");
} catch (DuplicateConnectionException e) {
    logger.error("Failed to create connection: " + e.getMessage());
    // Handle duplicate connection scenario
}
```

This exception is typically used in connection management scenarios where the system needs to prevent multiple connections with the same identifier or when connection pooling constraints are violated.
