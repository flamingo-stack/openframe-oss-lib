A custom runtime exception thrown when agent registration secret validation fails during the authentication process.

## Key Components

- **AgentRegistrationSecretValidationErrorException**: Custom runtime exception class that extends `RuntimeException`
- **Constructor**: Single constructor accepting a string message parameter

## Usage Example

```java
// Throwing the exception during secret validation
public void validateRegistrationSecret(String providedSecret, String expectedSecret) {
    if (!providedSecret.equals(expectedSecret)) {
        throw new AgentRegistrationSecretValidationErrorException(
            "Invalid registration secret provided for agent authentication"
        );
    }
}

// Catching and handling the exception
try {
    agentService.registerAgent(agentId, registrationSecret);
} catch (AgentRegistrationSecretValidationErrorException e) {
    logger.error("Agent registration failed: " + e.getMessage());
    return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
        .body("Registration secret validation failed");
}
```

This exception is typically used in agent registration workflows where secret-based authentication is required, providing clear error handling for invalid or mismatched registration secrets.
