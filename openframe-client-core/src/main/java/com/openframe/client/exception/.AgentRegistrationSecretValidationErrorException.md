A custom runtime exception thrown when agent registration secret validation fails during the authentication process.

## Key Components

- **AgentRegistrationSecretValidationErrorException** - Runtime exception class that extends `RuntimeException`
- **Constructor** - Single constructor accepting an error message string

## Usage Example

```java
// Thrown when agent registration secret is invalid
if (!isValidRegistrationSecret(providedSecret)) {
    throw new AgentRegistrationSecretValidationErrorException(
        "Invalid registration secret provided for agent: " + agentId
    );
}

// Catching the exception
try {
    validateAgentRegistration(secret);
} catch (AgentRegistrationSecretValidationErrorException e) {
    logger.error("Agent registration failed: {}", e.getMessage());
    // Handle validation failure
}
```

This exception is part of OpenFrame's client exception hierarchy and is typically thrown during agent registration processes when the provided authentication secret doesn't match expected values or fails validation rules.
