Custom exception thrown when an agent registration secret fails validation, indicating an unauthorized access attempt during the agent registration process.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `AgentRegistrationSecretValidationErrorException` | Class | Extends `UnauthorizedException` to signal invalid agent registration secrets |
| `AgentRegistrationSecretValidationErrorException(String message)` | Constructor | Accepts a descriptive error message passed up to the parent `UnauthorizedException` |

## Usage Example

```java
public void validateRegistrationSecret(String secret) {
    if (!isValid(secret)) {
        throw new AgentRegistrationSecretValidationErrorException(
            "Agent registration secret is invalid or has expired."
        );
    }
}
```

> This exception integrates with OpenFrame's authorization layer via `UnauthorizedException`, ensuring invalid registration secrets are handled consistently across the platform's security pipeline.
