A custom runtime exception thrown when agent registration secret validation fails, providing structured error handling with error codes for client applications.

## Key Components

- **AgentRegistrationSecretValidationException**: Custom runtime exception class
- **errorCode**: String field containing the specific error code for the validation failure
- **Constructor**: Takes error code and message parameters for detailed error reporting
- **@Getter**: Lombok annotation providing automatic getter method for the errorCode field

## Usage Example

```java
// Throwing the exception during validation
public void validateRegistrationSecret(String secret) {
    if (secret == null || secret.isEmpty()) {
        throw new AgentRegistrationSecretValidationException(
            "EMPTY_SECRET", 
            "Registration secret cannot be empty"
        );
    }
    
    if (!isValidSecret(secret)) {
        throw new AgentRegistrationSecretValidationException(
            "INVALID_SECRET", 
            "Registration secret format is invalid"
        );
    }
}

// Catching and handling the exception
try {
    agentService.registerAgent(secret);
} catch (AgentRegistrationSecretValidationException e) {
    String errorCode = e.getErrorCode();
    String message = e.getMessage();
    logger.error("Registration failed: {} - {}", errorCode, message);
}
```
