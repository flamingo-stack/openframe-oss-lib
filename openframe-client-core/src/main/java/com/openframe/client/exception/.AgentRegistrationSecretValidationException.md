A custom runtime exception thrown when agent registration secret validation fails, providing both error codes and descriptive messages for debugging and error handling.

## Key Components

- **AgentRegistrationSecretValidationException**: Main exception class extending `RuntimeException`
- **errorCode**: String field containing a specific error code for programmatic handling
- **message**: Inherited message field from parent `RuntimeException` class
- **@Getter**: Lombok annotation providing automatic getter method for `errorCode`

## Usage Example

```java
// Throwing the exception with error code and message
if (!isValidSecret(registrationSecret)) {
    throw new AgentRegistrationSecretValidationException(
        "INVALID_SECRET", 
        "The provided registration secret is invalid or expired"
    );
}

// Catching and handling the exception
try {
    validateAgentRegistration(secret);
} catch (AgentRegistrationSecretValidationException ex) {
    String errorCode = ex.getErrorCode(); // "INVALID_SECRET"
    String message = ex.getMessage();     // "The provided registration secret..."
    
    // Handle based on error code
    switch (errorCode) {
        case "INVALID_SECRET":
            // Log security event and return 401
            break;
        case "EXPIRED_SECRET":
            // Prompt for new registration
            break;
    }
}
```
