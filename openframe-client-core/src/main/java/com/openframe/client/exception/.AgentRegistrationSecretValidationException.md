Signals an authorization failure when an agent registration secret fails validation, extending `UnauthorizedException` with a specific error code and message.

## Key Components

| Element | Description |
|---|---|
| `AgentRegistrationSecretValidationException` | Custom exception class for invalid agent registration secrets |
| `UnauthorizedException` | Parent class providing HTTP 401-style exception behavior |
| `ErrorCode` | Enum/object used to categorize the specific error type |

## Usage Example

```java
// Throw when an agent provides an invalid or mismatched registration secret
if (!secretService.isValid(providedSecret)) {
    throw new AgentRegistrationSecretValidationException(
        ErrorCode.INVALID_AGENT_SECRET,
        "Agent registration secret does not match the expected value."
    );
}
```

> This exception is typically raised during agent onboarding flows in the OpenFrame platform, where each agent must present a valid secret before being registered. Catching this at the controller or global exception handler level will return an unauthorized response to the caller.
