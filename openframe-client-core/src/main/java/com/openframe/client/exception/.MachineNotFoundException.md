A custom runtime exception thrown when a machine resource cannot be found in the OpenFrame client system.

## Key Components

- **MachineNotFoundException** - Runtime exception class that extends `RuntimeException`
- **Constructor** - Single parameterized constructor accepting an error message

## Usage Example

```java
// Throwing the exception when a machine is not found
public Machine findMachine(String machineId) {
    Machine machine = repository.findById(machineId);
    if (machine == null) {
        throw new MachineNotFoundException("Machine with ID " + machineId + " not found");
    }
    return machine;
}

// Catching and handling the exception
try {
    Machine machine = machineService.findMachine("SERVER-001");
} catch (MachineNotFoundException e) {
    logger.error("Failed to locate machine: " + e.getMessage());
    // Handle the missing machine scenario
}
```

This exception follows standard Java exception patterns and is typically used in service layers or repositories when machine lookups fail, providing clear error messaging for debugging and user feedback.
