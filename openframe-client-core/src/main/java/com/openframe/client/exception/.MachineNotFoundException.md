Custom exception thrown when a requested machine resource cannot be found, extending the core `NotFoundException` for machine-specific 404-style error handling.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `MachineNotFoundException` | Class | Extends `NotFoundException` for machine lookup failures |
| `MachineNotFoundException(String message)` | Constructor | Accepts a descriptive error message passed to the parent exception |

## Usage Example

```java
// Throw when a machine lookup fails
public Machine getMachineById(String machineId) {
    return machineRepository.findById(machineId)
        .orElseThrow(() -> new MachineNotFoundException(
            "Machine not found with ID: " + machineId
        ));
}

// Catch specifically for machine-related errors
try {
    Machine machine = machineService.getMachineById("machine-123");
} catch (MachineNotFoundException e) {
    log.error("Machine lookup failed: {}", e.getMessage());
}
```
