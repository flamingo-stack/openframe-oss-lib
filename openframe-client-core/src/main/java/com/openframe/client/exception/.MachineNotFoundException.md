A custom runtime exception thrown when a requested machine cannot be found in the OpenFrame client system.

## Key Components

- **MachineNotFoundException** - Extends `RuntimeException` to indicate a machine lookup failure
- **Constructor** - Accepts a descriptive error message for debugging and logging

## Usage Example

```java
// Throwing the exception when machine lookup fails
public Machine findMachine(String machineId) {
    Machine machine = machineRepository.findById(machineId);
    if (machine == null) {
        throw new MachineNotFoundException("Machine with ID '" + machineId + "' not found");
    }
    return machine;
}

// Catching and handling the exception
try {
    Machine machine = machineService.findMachine("invalid-id");
} catch (MachineNotFoundException e) {
    logger.error("Failed to locate machine: " + e.getMessage());
    // Handle gracefully - show user-friendly error, return default, etc.
}
```

This exception follows standard Java exception patterns and is typically used in service layers to signal when machine operations fail due to missing resources.
