REST controller for OAuth2 authentication of OpenFrame agents, handling token issuance and refresh operations with comprehensive error handling.

## Key Components

- **AgentAuthController**: Main REST controller class annotated with `@RestController`
- **getClientToken()**: POST endpoint at `/oauth/token` for token operations
- **AgentAuthService**: Injected service handling the business logic for token management
- **AgentTokenResponse**: DTO for structured token response data

## Usage Example

```java
// Example token request (client credentials grant)
POST /oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=client_credentials
&client_id=agent_123
&client_secret=secret_456

// Example refresh token request
POST /oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=refresh_token
&refresh_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

// Successful response (200 OK)
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600
}

// Error response (401 Unauthorized)
{
  "message": "Invalid client credentials"
}
```

The controller supports both client credentials and refresh token grant types, returning appropriate HTTP status codes and structured error responses for authentication failures or server errors.
