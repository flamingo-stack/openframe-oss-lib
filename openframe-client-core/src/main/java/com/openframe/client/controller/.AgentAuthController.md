REST controller that handles OAuth 2.0 token operations for agent authentication, providing endpoints for token issuance and refresh functionality.

## Key Components

- **AgentAuthController**: Main REST controller class managing OAuth token endpoints
- **getClientToken()**: POST endpoint at `/oauth/token` for issuing and refreshing agent tokens
- **AgentAuthService**: Injected service handling the core authentication logic
- **AgentTokenResponse**: DTO for structured token response data

## Usage Example

```java
// Token request with client credentials
POST /oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=client_credentials&client_id=my-agent&client_secret=secret123

// Refresh token request
POST /oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=refresh_token&refresh_token=abc123xyz

// Success response (200 OK)
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "Bearer",
  "expires_in": 3600
}

// Error response (401 Unauthorized)
{
  "message": "Invalid client credentials"
}
```

The controller supports standard OAuth 2.0 flows including client credentials and refresh token grants, with comprehensive error handling for authentication failures and server errors.
