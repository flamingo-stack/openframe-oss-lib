REST controller that handles agent registration endpoints for the OpenFrame client application.

## Key Components

- **AgentController**: Main REST controller class annotated with `@RestController` and mapped to `/api/agents`
- **register()**: POST endpoint at `/register` that processes agent registration requests
- **AgentRegistrationService**: Injected service dependency that handles the registration business logic
- **DTOs**: Uses `AgentRegistrationRequest` and `AgentRegistrationResponse` for request/response mapping

## Usage Example

```java
// Example registration request
POST /api/agents/register
Headers: X-Initial-Key: your-initial-key
Content-Type: application/json

{
  "agentName": "MyAgent",
  "capabilities": ["processing", "analysis"]
}

// Example response
{
  "agentId": "agent-123",
  "token": "jwt-token",
  "status": "REGISTERED"
}
```

The controller requires an initial key via the `X-Initial-Key` header for authentication and validates the request body using Jakarta validation annotations. All registration logic is delegated to the injected `AgentRegistrationService`.
