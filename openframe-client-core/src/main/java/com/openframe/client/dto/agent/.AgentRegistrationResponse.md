A data transfer object that encapsulates the response returned when an agent successfully registers with the OpenFrame system.

## Key Components

- **AgentRegistrationResponse**: Main DTO class with three core fields
  - `machineId`: Unique identifier for the registered machine
  - `clientId`: OAuth client identifier for authentication
  - `clientSecret`: OAuth client secret for secure communication
- **Lombok annotations**: `@Data`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, and constructors

## Usage Example

```java
// Creating a response object
AgentRegistrationResponse response = new AgentRegistrationResponse(
    "machine-001", 
    "client-abc123", 
    "secret-xyz789"
);

// Accessing response data
String machineId = response.getMachineId();
String clientId = response.getClientId();
String clientSecret = response.getClientSecret();

// Used in service layer
@Service
public class AgentService {
    public AgentRegistrationResponse registerAgent(AgentRequest request) {
        // Registration logic
        return new AgentRegistrationResponse(
            generateMachineId(),
            generateClientId(),
            generateClientSecret()
        );
    }
}
```
