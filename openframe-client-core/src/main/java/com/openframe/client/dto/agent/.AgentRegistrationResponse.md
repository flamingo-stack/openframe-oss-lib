A data transfer object representing the response returned when an agent successfully registers with the OpenFrame system, containing essential authentication credentials.

## Key Components

- **machineId**: Unique identifier for the registered machine/agent
- **clientId**: OAuth2 client identifier for API authentication
- **clientSecret**: OAuth2 client secret for secure API access
- **Lombok annotations**: `@Data`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Typical usage in an agent registration service
public AgentRegistrationResponse registerAgent(AgentRegistrationRequest request) {
    // Registration logic here...
    
    return new AgentRegistrationResponse(
        "machine-12345",
        "client-abc123",
        "secret-xyz789"
    );
}

// Client-side usage after registration
AgentRegistrationResponse response = registrationService.register(request);
String machineId = response.getMachineId();
String clientId = response.getClientId();
String clientSecret = response.getClientSecret();

// Store credentials for future API calls
authConfig.setClientCredentials(clientId, clientSecret);
```

This DTO is typically used in REST API responses where agents need to receive their authentication credentials after successful registration with the OpenFrame platform.
