A data transfer object (DTO) that encapsulates all necessary information required for registering an agent device with the OpenFrame system. This class serves as the request payload for agent registration endpoints.

## Key Components

- **Core identification fields**: `hostname`, `organizationId` for basic device identity
- **Network information**: `ip`, `macAddress`, `osUuid`, `agentVersion`, `status` for network and version tracking
- **Hardware details**: `displayName`, `serialNumber`, `manufacturer`, `model` for physical device identification
- **Operating system data**: `type`, `osType`, `osVersion`, `osBuild`, `timezone` for system compatibility and configuration
- **Lombok @Data annotation**: Automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
AgentRegistrationRequest request = new AgentRegistrationRequest();
request.setHostname("workstation-001");
request.setOrganizationId("org-12345");
request.setIp("192.168.1.100");
request.setMacAddress("00:11:22:33:44:55");
request.setAgentVersion("2.1.0");
request.setStatus(DeviceStatus.ACTIVE);
request.setType(DeviceType.DESKTOP);
request.setOsType("Windows");
request.setOsVersion("10.0.19044");
request.setTimezone("America/New_York");

// Send to registration endpoint
agentService.registerAgent(request);
```
