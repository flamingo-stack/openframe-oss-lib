A data transfer object (DTO) that represents the response structure for agent authentication tokens in the OpenFrame client library.

## Key Components

- **accessToken** - The JWT access token for API authentication
- **refreshToken** - Token used to refresh expired access tokens
- **tokenType** - The type of token (typically "Bearer")
- **expiresIn** - Token expiration time in seconds from issuance

## Usage Example

```java
// Typically returned from authentication service
AgentTokenResponse tokenResponse = authService.authenticateAgent(credentials);

// Extract token information
String token = tokenResponse.getAccessToken();
String type = tokenResponse.getTokenType();
long expiry = tokenResponse.getExpiresIn();

// Use in API client headers
String authHeader = tokenResponse.getTokenType() + " " + tokenResponse.getAccessToken();

// Handle token refresh when needed
if (isTokenExpired(tokenResponse.getExpiresIn())) {
    AgentTokenResponse newToken = authService.refreshToken(tokenResponse.getRefreshToken());
}
```

This DTO uses Lombok annotations for automatic generation of getters, setters, constructors, and other boilerplate methods, making it a clean data container for OAuth-style token responses.
