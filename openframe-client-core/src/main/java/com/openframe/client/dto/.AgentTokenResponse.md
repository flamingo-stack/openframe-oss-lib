A data transfer object representing an authentication token response for agents, containing access credentials and metadata.

## Key Components

- **accessToken** - The primary authentication token for API requests
- **refreshToken** - Token used to refresh expired access tokens
- **tokenType** - Type of token (typically "Bearer")
- **expiresIn** - Token expiration time in seconds

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and utility methods.

## Usage Example

```java
// Creating a token response
AgentTokenResponse tokenResponse = new AgentTokenResponse(
    "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "def50200e3b0c44298fc1c149afbf4c8996fb9...",
    "Bearer",
    3600L
);

// Using with default constructor
AgentTokenResponse response = new AgentTokenResponse();
response.setAccessToken("access-token-value");
response.setTokenType("Bearer");
response.setExpiresIn(3600L);

// Accessing token information
String token = response.getAccessToken();
boolean isExpired = System.currentTimeMillis() > response.getExpiresIn() * 1000;
```
