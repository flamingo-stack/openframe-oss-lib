Represents a tag to be created and assigned to a device during agent registration. If the specified tag key does not exist in the organization, it is automatically created as a `CUSTOM` type.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `key` | `String` | Tag key name (e.g., `"site"`). Required — must not be blank. |
| `values` | `List<String>` | List of values to assign to the tag (e.g., `["CHICAGO"]`). |

## Usage Example

```java
AgentRegistrationTagInput tag = AgentRegistrationTagInput.builder()
    .key("site")
    .values(List.of("CHICAGO"))
    .build();
```

Typically used as part of an agent registration payload to categorize devices by location, environment, or other custom attributes automatically at enrollment time.
