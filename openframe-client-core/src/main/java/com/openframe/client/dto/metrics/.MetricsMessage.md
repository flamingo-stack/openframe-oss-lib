A data transfer object (DTO) that represents system metrics data with machine identification and performance measurements.

## Key Components

- **MetricsMessage**: Main DTO class with Lombok `@Data` annotation for automatic getter/setter generation
- **machineId**: String identifier for the source machine
- **cpu**: Double value representing CPU utilization percentage
- **memory**: Double value representing memory usage percentage  
- **timestamp**: Instant object capturing when the metrics were recorded

## Usage Example

```java
// Creating a new metrics message
MetricsMessage metrics = new MetricsMessage();
metrics.setMachineId("server-001");
metrics.setCpu(75.5);
metrics.setMemory(62.3);
metrics.setTimestamp(Instant.now());

// Accessing metrics data
String machine = metrics.getMachineId();
double cpuUsage = metrics.getCpu();
double memoryUsage = metrics.getMemory();
Instant recordedAt = metrics.getTimestamp();

// Typical usage in monitoring systems
if (metrics.getCpu() > 80.0) {
    alertService.sendHighCpuAlert(metrics.getMachineId());
}
```

This DTO is commonly used in monitoring and metrics collection systems to transfer performance data between services or store metrics in databases.
