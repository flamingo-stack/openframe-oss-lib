A data transfer object (DTO) that encapsulates the required parameters for creating a new OAuth2 client in the OpenFrame system.

## Key Components

- **grantTypes**: Array of OAuth2 grant types that the client is authorized to use (e.g., "authorization_code", "client_credentials")
- **scopes**: Array of permission scopes that define what resources the client can access
- **@Data annotation**: Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a new client request for a web application
CreateClientRequest request = new CreateClientRequest();
request.setGrantTypes(new String[]{"authorization_code", "refresh_token"});
request.setScopes(new String[]{"read", "write", "profile"});

// Use with a service or controller
ClientService clientService = new ClientService();
Client newClient = clientService.createClient(request);

// Or in a REST controller
@PostMapping("/clients")
public ResponseEntity<Client> createClient(@RequestBody CreateClientRequest request) {
    return ResponseEntity.ok(clientService.createClient(request));
}
```
