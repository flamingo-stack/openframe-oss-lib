A data transfer object (DTO) used for creating new OAuth2 clients in the OpenFrame authentication system.

## Key Components

- **grantTypes**: Array of OAuth2 grant types supported by the client (e.g., "authorization_code", "client_credentials")
- **scopes**: Array of permission scopes the client can request (e.g., "read", "write", "admin")

## Usage Example

```java
// Create a new client request for a web application
CreateClientRequest request = new CreateClientRequest();
request.setGrantTypes(new String[]{"authorization_code", "refresh_token"});
request.setScopes(new String[]{"read", "write", "profile"});

// Create a service client with limited permissions
CreateClientRequest serviceRequest = new CreateClientRequest();
serviceRequest.setGrantTypes(new String[]{"client_credentials"});
serviceRequest.setScopes(new String[]{"api:read"});
```

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, equals, hashCode, and toString methods. This DTO is typically used in REST API endpoints for client registration and management operations.
