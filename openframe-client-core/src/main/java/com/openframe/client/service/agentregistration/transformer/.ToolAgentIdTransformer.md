Interface for transforming tool agent identifiers based on tool type and retry attempts.

## Key Components

- **`getToolType()`** - Returns the specific `ToolType` this transformer handles
- **`transform(String agentToolId, boolean lastAttempt)`** - Transforms the agent tool ID, with different behavior based on whether this is the final retry attempt

## Usage Example

```java
public class CustomToolTransformer implements ToolAgentIdTransformer {
    
    @Override
    public ToolType getToolType() {
        return ToolType.API_TOOL;
    }
    
    @Override
    public String transform(String agentToolId, boolean lastAttempt) {
        if (lastAttempt) {
            // Final attempt - use fallback format
            return "fallback_" + agentToolId;
        }
        // Regular transformation
        return "transformed_" + agentToolId.toLowerCase();
    }
}

// Registration usage
ToolAgentIdTransformer transformer = new CustomToolTransformer();
String transformedId = transformer.transform("AGENT_001", false);
// Result: "transformed_agent_001"
```

This interface enables tool-specific ID transformation strategies during agent registration, allowing different handling patterns for initial attempts versus final retry attempts.
