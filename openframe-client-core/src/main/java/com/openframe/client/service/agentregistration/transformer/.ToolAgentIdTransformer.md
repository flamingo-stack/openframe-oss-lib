A transformer interface for converting agent tool IDs into different formats based on the tool type and attempt status.

## Key Components

- **`getToolType()`** - Returns the `ToolType` that this transformer handles
- **`transform(String agentToolId, boolean lastAttempt)`** - Converts an agent tool ID to a transformed format, with behavior potentially varying based on whether this is the final attempt

## Usage Example

```java
public class CustomToolTransformer implements ToolAgentIdTransformer {
    
    @Override
    public ToolType getToolType() {
        return ToolType.DATABASE;
    }
    
    @Override
    public String transform(String agentToolId, boolean lastAttempt) {
        if (lastAttempt) {
            return "final_" + agentToolId.toUpperCase();
        }
        return "retry_" + agentToolId.toLowerCase();
    }
}

// Usage in service
ToolAgentIdTransformer transformer = new CustomToolTransformer();
String transformedId = transformer.transform("db_tool_123", false);
// Returns: "retry_db_tool_123"
```

This interface enables polymorphic handling of different tool types, allowing each implementation to apply specific ID transformation logic based on the tool type and retry context.
