A Spring component that extracts machine IDs from NATS message subjects following a specific topic pattern. It validates the subject format and returns the machine identifier for message routing or processing.

## Key Components

- **`extract(String subject)`** - Main method that parses NATS subjects and extracts machine IDs
- **Subject validation** - Ensures proper format: `machine.{machineId}.{suffix}`
- **Error handling** - Throws `IllegalArgumentException` for invalid formats or empty values
- **Logging** - Debug-level logging for successful extractions

## Usage Example

```java
@Autowired
private NatsTopicMachineIdExtractor extractor;

// Valid subject extraction
String subject = "machine.server001.status";
String machineId = extractor.extract(subject); // Returns "server001"

// Another valid example
String alertSubject = "machine.prod-db-01.alert";
String dbMachineId = extractor.extract(alertSubject); // Returns "prod-db-01"

// Invalid subjects will throw IllegalArgumentException
try {
    extractor.extract("invalid.format"); // Throws exception
} catch (IllegalArgumentException e) {
    log.error("Invalid subject format: {}", e.getMessage());
}
```

The component expects NATS subjects to follow the pattern `machine.{machineId}.{anything}` where the machine ID is always in the second position after splitting by dots.
