A Spring component that generates cryptographically secure random secrets for agent registration using Base64 URL-safe encoding.

## Key Components

- **`generate()`** - Generates a 32-byte random secret encoded as a Base64 URL-safe string without padding
- **`secureRandom`** - SecureRandom instance for cryptographically strong random number generation

## Usage Example

```java
@Autowired
private AgentSecretGenerator secretGenerator;

public void registerAgent() {
    // Generate a secure secret for agent authentication
    String agentSecret = secretGenerator.generate();
    
    // Example output: "dGVzdFNlY3JldEZvckFnZW50UmVnaXN0cmF0aW9u"
    System.out.println("Generated secret: " + agentSecret);
    
    // Use the secret for agent registration
    agentRegistrationService.registerWithSecret(agentSecret);
}
```

The generated secrets are 256-bit random values (32 bytes) encoded as URL-safe Base64 strings, making them suitable for use in URLs, HTTP headers, or as secure tokens for agent authentication and registration processes.
