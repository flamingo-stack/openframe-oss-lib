A Spring component that generates cryptographically secure secrets for agent registration using URL-safe Base64 encoding.

## Key Components

- **`AgentSecretGenerator`** - Spring component for generating secure random secrets
- **`generate()`** - Creates a 32-byte random secret encoded as URL-safe Base64 string
- **`SecureRandom`** - Cryptographically strong random number generator instance

## Usage Example

```java
@Autowired
private AgentSecretGenerator secretGenerator;

public void registerNewAgent() {
    // Generate a secure secret for the new agent
    String agentSecret = secretGenerator.generate();
    
    // The secret will be a URL-safe Base64 string like:
    // "kQw8J9X2mP4nL6vR8tY1uE5wQ7zA3sD9fG2hK4jM6pN"
    
    // Use the secret for agent authentication/registration
    agentRegistrationService.createAgent(agentSecret);
}
```

The generator produces 256-bit (32-byte) secrets that are cryptographically secure and URL-safe, making them suitable for use in web APIs, tokens, and secure agent identification without requiring URL encoding.
