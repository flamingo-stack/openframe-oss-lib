This service handles the installation of integrated tool agents on registered machines by coordinating between agent retrieval and tool installation processes.

## Key Components

- **`process(String machineId)`** - Main method that retrieves all enabled integrated tool agents and triggers their installation on the specified machine
- **`integratedToolAgentService`** - Dependency for retrieving enabled tool agents
- **`toolInstallationService`** - Dependency for processing tool installations

## Usage Example

```java
@Autowired
private AgentRegistrationToolInstallationService installationService;

// Install all enabled tool agents on a specific machine
public void setupMachineTools(String machineId) {
    installationService.process(machineId);
}

// Example usage during agent registration workflow
public void registerNewAgent(String machineId) {
    // ... other registration steps
    
    // Install required tools on the newly registered machine
    installationService.process(machineId);
    
    log.info("Tool installation initiated for machine: {}", machineId);
}
```

The service acts as an orchestrator in the agent registration workflow, ensuring that all enabled integrated tool agents are properly installed on target machines through a simple, automated process.
