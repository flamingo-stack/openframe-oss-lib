Interface that defines post-processing hooks for agent registration operations. Provides extensibility points for custom logic after agent registration completion.

## Key Components

- **`AgentRegistrationProcessor`** - Main interface for registration processing hooks
- **`postProcessAgentRegistration()`** - Default no-op method for post-registration processing

## Usage Example

```java
@Component
public class CustomAgentProcessor implements AgentRegistrationProcessor {
    
    @Override
    public void postProcessAgentRegistration(Machine machine, AgentRegistrationRequest request) {
        // Log registration
        log.info("Agent registered: {} on machine {}", 
                 request.getAgentId(), machine.getHostname());
        
        // Send notification
        notificationService.sendAgentRegisteredNotification(machine);
        
        // Update metrics
        metricsService.incrementAgentRegistrations();
    }
}

// Usage in registration service
@Service
public class AgentRegistrationService {
    
    @Autowired
    private List<AgentRegistrationProcessor> processors;
    
    public void registerAgent(AgentRegistrationRequest request) {
        Machine machine = performRegistration(request);
        
        // Trigger post-processing hooks
        processors.forEach(processor -> 
            processor.postProcessAgentRegistration(machine, request));
    }
}
```
