Handles OAuth refresh token processing to generate new access tokens for authenticated agents, implementing token rotation and validation checks.

## Key Components

- **handle()** - Main method that processes refresh tokens and returns new token pairs
- **validateExpiration()** - Ensures the refresh token hasn't expired
- **validateRefreshCount()** - Prevents excessive token refreshing beyond configured limits
- **AccessTokenGenerator** - Generates new access tokens for authenticated clients
- **RefreshTokenGenerator** - Creates new refresh tokens with incremented refresh counts

## Usage Example

```java
@Service
public class AuthService {
    
    @Autowired
    private RefreshTokenHandler refreshTokenHandler;
    
    public AgentTokenResponse refreshUserToken(String refreshToken) {
        try {
            // Process the refresh token and get new tokens
            AgentTokenResponse response = refreshTokenHandler.handle(refreshToken);
            
            log.info("Successfully refreshed token for client");
            return response;
            
        } catch (IllegalArgumentException e) {
            log.error("Token refresh failed: {}", e.getMessage());
            throw new UnauthorizedException("Invalid refresh token");
        }
    }
}
```

The handler implements OAuth2 token rotation best practices by generating both a new access token and refresh token on each refresh, while enforcing expiration times and maximum refresh counts to prevent token abuse.
