Transforms MeshCentral agent IDs by adding a standardized node prefix for consistent identification across the system.

## Key Components

- **`MeshCentralAgentIdTransformer`** - Spring component implementing `ToolAgentIdTransformer` interface
- **`getToolType()`** - Returns `ToolType.MESHCENTRAL` to identify the supported tool type
- **`transform(String agentToolId, boolean __)`** - Adds "node//" prefix to agent IDs
- **`NODE_PREFIX`** - Constant defining the "node//" prefix used for transformation

## Usage Example

```java
@Autowired
private MeshCentralAgentIdTransformer transformer;

// Transform a MeshCentral agent ID
String originalId = "ABC123-DEF456";
String transformedId = transformer.transform(originalId, false);
// Result: "node//ABC123-DEF456"

// Check supported tool type
ToolType supportedType = transformer.getToolType();
// Returns: ToolType.MESHCENTRAL

// Handle blank input gracefully
String blankResult = transformer.transform("", false);
// Returns: "" (with warning logged)
```

The transformer ensures MeshCentral agent IDs follow a consistent format by prepending the node prefix, with built-in validation and logging for monitoring transformation operations.
