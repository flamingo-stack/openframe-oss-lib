A Spring component that generates OAuth2 access tokens for OpenFrame clients using JWT with configurable expiration and standardized claims.

## Key Components

- **`generate(OAuthClient, String)`** - Main method that creates a JWT access token for the given OAuth client and grant type
- **`buildClaims(OAuthClient, String)`** - Private helper that constructs the JWT claims set with client-specific information
- **`expirationSeconds`** - Configurable token lifetime injected from application properties
- **`TOKEN_TYPE`** - Static constant defining "Bearer" token type
- **`jwtService`** - Injected JWT service for token generation and signing

## Usage Example

```java
@Service
public class AuthService {
    
    private final AccessTokenGenerator tokenGenerator;
    
    public AuthResponse authenticateClient(OAuthClient client, String grantType) {
        // Generate access token for the authenticated client
        String accessToken = tokenGenerator.generate(client, grantType);
        
        return AuthResponse.builder()
            .accessToken(accessToken)
            .tokenType("Bearer")
            .expiresIn(tokenGenerator.getExpirationSeconds())
            .build();
    }
}
```

The generated tokens include standard JWT claims (issuer, subject, issued/expiry times) plus custom claims for machine ID, grant type, and client roles, making them suitable for both authentication and authorization in the OpenFrame ecosystem.
