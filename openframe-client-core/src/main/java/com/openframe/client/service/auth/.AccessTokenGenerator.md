A Spring component responsible for generating JWT-based OAuth2 access tokens for authenticated clients. It creates tokens with configurable expiration times and includes client-specific claims.

## Key Components

- **`generate(OAuthClient client, String grantType)`** - Main method that creates a JWT access token for the given OAuth client and grant type
- **`buildClaims(OAuthClient client, String grantType)`** - Private method that constructs JWT claims including issuer, expiration, subject, and custom claims
- **`expirationSeconds`** - Configurable token expiration time injected from application properties
- **`TOKEN_TYPE`** - Constant defining the token type as "Bearer"

## Usage Example

```java
@Autowired
private AccessTokenGenerator tokenGenerator;

// Generate access token for OAuth client
OAuthClient client = // ... obtain client
String grantType = "client_credentials";

String accessToken = tokenGenerator.generate(client, grantType);

// Check token expiration configuration
int expiration = tokenGenerator.getExpirationSeconds();
```

The generated tokens include standard JWT claims (issuer, subject, expiration) plus custom claims for machine ID, grant type, and user roles. Token expiration is configured via the `security.oauth2.token.access.expiration-seconds` property.
