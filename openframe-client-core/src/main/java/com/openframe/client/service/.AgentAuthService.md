Handles OAuth2 authentication for agents by routing token requests to appropriate grant type handlers based on the provided grant type parameter.

## Key Components

- **`issueClientToken()`** - Main authentication method that validates clients and delegates to specific grant handlers
- **`CLIENT_CREDENTIALS_GRANT_TYPE`** - Constant for client credentials flow ("client_credentials")
- **`REFRESH_TOKEN_GRANT_TYPE`** - Constant for refresh token flow ("refresh_token")
- **`ClientCredentialsHandler`** - Injected handler for client credentials authentication
- **`RefreshTokenHandler`** - Injected handler for refresh token authentication

## Usage Example

```java
@Autowired
private AgentAuthService authService;

// Client credentials flow
AgentTokenResponse response = authService.issueClientToken(
    AgentAuthService.CLIENT_CREDENTIALS_GRANT_TYPE,
    null, // no refresh token needed
    "client123",
    "secret456"
);

// Refresh token flow  
AgentTokenResponse refreshedResponse = authService.issueClientToken(
    AgentAuthService.REFRESH_TOKEN_GRANT_TYPE,
    "existing_refresh_token",
    null, // client credentials not needed
    null
);
```

The service uses a switch statement to route requests to the appropriate handler and throws `IllegalArgumentException` for unsupported grant types.
