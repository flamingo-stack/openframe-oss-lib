Service class that handles OAuth 2.0 authentication flows for agents, supporting both client credentials and refresh token grant types.

## Key Components

- **CLIENT_CREDENTIALS_GRANT_TYPE** - Constant for "client_credentials" flow
- **REFRESH_TOKEN_GRANT_TYPE** - Constant for "refresh_token" flow  
- **clientCredentialsHandler** - Handler for client credentials authentication
- **refreshTokenHandler** - Handler for refresh token authentication
- **issueClientToken()** - Main method that routes authentication requests to appropriate handlers

## Usage Example

```java
@Autowired
private AgentAuthService agentAuthService;

// Client credentials flow
AgentTokenResponse response = agentAuthService.issueClientToken(
    AgentAuthService.CLIENT_CREDENTIALS_GRANT_TYPE,
    null, // no refresh token needed
    "my-client-id",
    "my-client-secret"
);

// Refresh token flow
AgentTokenResponse refreshResponse = agentAuthService.issueClientToken(
    AgentAuthService.REFRESH_TOKEN_GRANT_TYPE,
    "existing-refresh-token",
    null, // client ID not needed for refresh
    null  // client secret not needed for refresh
);
```

The service uses a switch statement to delegate to specialized handlers based on the grant type, throwing an `IllegalArgumentException` for unsupported grant types. All authentication attempts are logged for debugging purposes.
