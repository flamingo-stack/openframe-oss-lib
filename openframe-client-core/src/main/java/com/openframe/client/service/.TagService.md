A Spring service that manages tags and their associations with machines, providing CRUD operations for tags and many-to-many relationship management through machine tags.

## Key Components

- **Tag Management**: Create, read, update, and delete tag entities with organization-based filtering
- **Machine-Tag Association**: Add and remove tag associations for specific machines with audit tracking
- **Repository Dependencies**: Utilizes `TagRepository` and `MachineTagRepository` for data persistence
- **Relationship Queries**: Retrieve tags for machines and machine IDs for specific tags

## Usage Example

```java
@Autowired
private TagService tagService;

// Create a new tag
Tag newTag = new Tag();
newTag.setName("Production");
newTag.setOrganizationId("org-123");
Tag savedTag = tagService.createTag(newTag);

// Get all tags for an organization
List<Tag> orgTags = tagService.getAllTags("org-123");

// Associate tag with a machine
tagService.addTagToMachine("machine-456", savedTag.getId(), "user-789");

// Get all tags for a specific machine
List<Tag> machineTags = tagService.getTagsForMachine("machine-456");

// Remove tag association
tagService.removeTagFromMachine("machine-456", savedTag.getId());

// Delete tag (removes all machine associations first)
tagService.deleteTag(savedTag.getId());
```

The service automatically handles timestamp creation and ensures referential integrity by cleaning up machine tag associations before deleting tags.
