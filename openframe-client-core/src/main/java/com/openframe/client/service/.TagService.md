A Spring service class that manages tags and their associations with machines, providing CRUD operations and relationship management between tags and machines through a many-to-many relationship.

## Key Components

- **getAllTags(String organizationId)** - Retrieves all tags for a specific organization
- **createTag(Tag tag)** - Creates a new tag with timestamp
- **getTagById(String id)** - Fetches a single tag by ID
- **updateTag(Tag tag)** - Updates an existing tag
- **deleteTag(String id)** - Deletes a tag and all its machine associations
- **addTagToMachine()** - Associates a tag with a machine
- **removeTagFromMachine()** - Removes tag-machine association
- **getTagsForMachine(String machineId)** - Gets all tags assigned to a machine
- **getMachineIdsForTag(String tagId)** - Gets all machine IDs associated with a tag

## Usage Example

```java
@Autowired
private TagService tagService;

// Create and manage tags
Tag newTag = new Tag();
newTag.setName("Production");
newTag.setOrganizationId("org123");
Tag savedTag = tagService.createTag(newTag);

// Associate tag with machine
tagService.addTagToMachine("machine456", savedTag.getId(), "user123");

// Retrieve machine's tags
List<Tag> machineTags = tagService.getTagsForMachine("machine456");

// Clean up - removes all associations
tagService.deleteTag(savedTag.getId());
```
