Delegates agent tool ID transformation to the appropriate `ToolAgentIdTransformer` implementation based on the given `ToolType`.

## Key Components

| Element | Description |
|---|---|
| `transformers` | Injected list of all `ToolAgentIdTransformer` implementations available in the Spring context |
| `transform(ToolType, String, String, boolean)` | Finds the matching transformer for the given `ToolType` and applies it; falls back to the original `agentToolId` if no match is found |

## Usage Example

```java
@Service
@RequiredArgsConstructor
public class AgentRegistrationService {

    private final ToolAgentIdTransformerService toolAgentIdTransformerService;

    public void registerAgent(ToolType toolType, String machineId, String rawAgentToolId) {
        String resolvedId = toolAgentIdTransformerService.transform(
            toolType,
            machineId,
            rawAgentToolId,
            false // not the last attempt
        );

        // proceed with resolvedId
    }
}
```

> **Note:** If no `ToolAgentIdTransformer` is registered for the provided `ToolType`, the service returns the original `agentToolId` unchanged, ensuring safe fallback behavior during agent registration flows.
