A Spring service that manages the transformation of agent tool IDs using type-specific transformers through a strategy pattern implementation.

## Key Components

- **ToolAgentIdTransformerService** - Main service class that coordinates tool ID transformations
- **transformers** - Injected list of `ToolAgentIdTransformer` implementations for different tool types
- **transform()** - Core method that finds and applies the appropriate transformer based on tool type

## Usage Example

```java
@Autowired
private ToolAgentIdTransformerService transformerService;

public void processAgentTool() {
    // Transform an agent tool ID for a specific tool type
    String originalId = "agent-tool-123";
    ToolType toolType = ToolType.API_ENDPOINT;
    boolean isLastAttempt = false;
    
    String transformedId = transformerService.transform(
        toolType, 
        originalId, 
        isLastAttempt
    );
    
    // Use the transformed ID for further processing
    log.info("Transformed {} to {}", originalId, transformedId);
}
```

The service uses dependency injection to automatically discover all `ToolAgentIdTransformer` implementations and selects the appropriate one based on the tool type. If no matching transformer is found, it returns the original agent tool ID unchanged. The `lastAttempt` flag provides additional context to transformers for handling retry scenarios.
