A Spring service that transforms agent tool IDs based on tool type using a strategy pattern with registered transformers.

## Key Components

- **ToolAgentIdTransformerService** - Main service class that orchestrates ID transformation
- **transform()** - Core method that finds the appropriate transformer for a given ToolType and applies the transformation
- **transformers** - Injected list of ToolAgentIdTransformer implementations

## Usage Example

```java
@Autowired
private ToolAgentIdTransformerService transformerService;

public void processAgentTool() {
    String originalId = "agent-tool-123";
    ToolType toolType = ToolType.DATABASE;
    
    // Transform the agent tool ID
    String transformedId = transformerService.transform(
        toolType, 
        originalId, 
        false  // not last attempt
    );
    
    // Use transformed ID for further processing
    processWithTransformedId(transformedId);
    
    // On retry/last attempt
    String finalId = transformerService.transform(
        toolType, 
        originalId, 
        true   // last attempt
    );
}
```

The service uses Spring's dependency injection to automatically discover all ToolAgentIdTransformer implementations and routes transformation requests to the appropriate transformer based on the tool type. If no matching transformer is found, the original agent tool ID is returned unchanged.
