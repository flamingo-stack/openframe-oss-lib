Handles OAuth 2.0 client credentials grant flow authentication by validating client credentials and generating access and refresh tokens for authenticated API access.

## Key Components

- **handle()** - Main authentication method that processes client credentials and returns token response
- **validateClientSecret()** - Validates provided client secret against stored encrypted secret using password encoder
- **OAuthClientRepository** - Data access layer for OAuth client management
- **AccessTokenGenerator/RefreshTokenGenerator** - Token generation services for creating JWT tokens
- **AgentTokenResponse** - Response DTO containing access token, refresh token, and expiration details

## Usage Example

```java
@Service
public class AuthenticationService {
    
    @Autowired
    private ClientCredentialsHandler credentialsHandler;
    
    public AgentTokenResponse authenticateClient(String clientId, String clientSecret) {
        try {
            // Handle client credentials authentication
            AgentTokenResponse tokenResponse = credentialsHandler.handle(clientId, clientSecret);
            
            // Token response contains:
            // - Access token (JWT)
            // - Refresh token
            // - Token type ("Bearer")
            // - Expiration time in seconds
            
            return tokenResponse;
        } catch (IllegalArgumentException e) {
            // Handle invalid client credentials
            throw new AuthenticationException("Authentication failed", e);
        }
    }
}
```

The handler follows the OAuth 2.0 client credentials grant specification, requiring valid client ID and secret to generate tokens for machine-to-machine authentication scenarios.
