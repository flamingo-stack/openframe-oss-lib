Handles OAuth 2.0 client credentials grant flow for agent authentication, validating client credentials and generating access/refresh token pairs.

## Key Components

- **`handle(String clientId, String clientSecret)`** - Main method that processes client credentials and returns token response
- **`validateClientSecret(OAuthClient client, String clientSecret)`** - Private method for secure client secret validation using password encoder
- **Dependencies**: 
  - `OAuthClientRepository` - Client data access
  - `PasswordEncoder` - Secure password validation
  - `AccessTokenGenerator` - JWT access token creation
  - `RefreshTokenGenerator` - Refresh token generation

## Usage Example

```java
@Autowired
private ClientCredentialsHandler handler;

public void authenticateClient() {
    try {
        AgentTokenResponse response = handler.handle("my-client-id", "my-client-secret");
        
        String accessToken = response.getAccessToken();
        String refreshToken = response.getRefreshToken();
        long expiresIn = response.getExpiresIn();
        
        // Use tokens for API access
        log.info("Authentication successful, token expires in {} seconds", expiresIn);
        
    } catch (IllegalArgumentException e) {
        log.error("Authentication failed: {}", e.getMessage());
        // Handle invalid credentials
    }
}
```

The handler implements secure credential validation and follows OAuth 2.0 standards for the client credentials grant type, commonly used for machine-to-machine authentication.
