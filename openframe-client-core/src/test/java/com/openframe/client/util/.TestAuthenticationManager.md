A test implementation of Spring Security's AuthenticationManager that always throws a BadCredentialsException, primarily used for testing authentication failure scenarios.

## Key Components

- **TestAuthenticationManager** - Implementation of `AuthenticationManager` interface
- **authenticate()** - Method that unconditionally throws `BadCredentialsException`

## Usage Example

```java
// Use in test configuration
@TestConfiguration
public class TestSecurityConfig {
    
    @Bean
    @Primary
    public AuthenticationManager testAuthenticationManager() {
        return new TestAuthenticationManager();
    }
}

// Or inject directly in tests
@Test
public void testAuthenticationFailure() {
    AuthenticationManager authManager = new TestAuthenticationManager();
    
    UsernamePasswordAuthenticationToken token = 
        new UsernamePasswordAuthenticationToken("user", "password");
    
    assertThrows(BadCredentialsException.class, () -> {
        authManager.authenticate(token);
    });
}
```

This utility class is valuable for unit testing authentication workflows where you need to simulate authentication failures without setting up complex authentication providers.
