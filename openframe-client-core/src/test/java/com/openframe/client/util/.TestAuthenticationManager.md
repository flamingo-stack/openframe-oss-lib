A test implementation of Spring Security's AuthenticationManager that always rejects authentication attempts with a BadCredentialsException. This utility class is designed for testing scenarios where authentication failure behavior needs to be verified.

## Key Components

- **TestAuthenticationManager**: Implements `AuthenticationManager` interface
- **authenticate()**: Always throws `BadCredentialsException` regardless of input credentials

## Usage Example

```java
// In test configuration
@TestConfiguration
public class TestSecurityConfig {
    
    @Bean
    @Primary
    public AuthenticationManager testAuthenticationManager() {
        return new TestAuthenticationManager();
    }
}

// In unit tests
@Test
public void testAuthenticationFailure() {
    TestAuthenticationManager authManager = new TestAuthenticationManager();
    UsernamePasswordAuthenticationToken token = 
        new UsernamePasswordAuthenticationToken("user", "password");
    
    assertThrows(BadCredentialsException.class, 
        () -> authManager.authenticate(token));
}

// Testing authentication failure handling
@Test
public void testLoginControllerWithFailedAuth() {
    // Configure test to use TestAuthenticationManager
    // Verify proper error handling when authentication fails
}
```

This class is particularly useful for testing authentication error flows, security configurations, and ensuring proper exception handling in authentication-dependent components.
