Unit test suite for `MachineTagEventAspect`, verifying that repository save and delete operations are correctly intercepted and delegated to `MachineTagEventService` with proper ordering, error resilience, and tag ID presence checks.

## Key Components

| Test Method | Purpose |
|---|---|
| `testMachineSaveAspect` | Verifies `afterMachineSave` delegates to `processMachineSave` |
| `testMachineSaveAllAspect` | Verifies `afterMachineSaveAll` delegates to `processMachineSaveAll` |
| `testTagAssignmentSaveAspect` | Verifies `afterTagAssignmentSave` delegates to `processTagAssignmentSave` |
| `testAroundTagSave_ExistingTag` | Confirms `processTagSave` is called only when tag has an ID |
| `testAroundTagSave_NewTag` | Confirms `processTagSave` is skipped for tags without an ID |
| `testAroundTagSaveAll_MixedTags` | Only existing (ID-bearing) tags trigger `processTagSave` |
| `testTagAssignmentDelete_DelegatesToServiceBeforeProceed` | Service is invoked **before** `proceed()` on delete |
| `testTagAssignmentDelete_ProceedsEvenWhenServiceThrows` | Delete proceeds even if the service throws |
| `testErrorHandling_MachineSave` | Aspect swallows service errors without propagating |
| `testErrorHandling_ProceedingJoinPointThrowsException` | Exceptions from `proceed()` are re-thrown |

## Usage Example

```java
// Instantiating the aspect under test (mirrors @BeforeEach setup)
MachineTagEventAspect aspect = new MachineTagEventAspect(machineTagEventService);

// Testing save interception for an existing tag
Tag tag = new Tag();
tag.setId("tag-1");
tag.setKey("environment");
tag.setColor("#FFC008");

when(proceedingJoinPoint.proceed()).thenReturn(tag);
Object result = aspect.aroundTagSave(proceedingJoinPoint, tag);

verify(machineTagEventService, times(1)).processTagSave(tag);
assertEquals(tag, result);

// Testing delete ordering: service must fire before repository proceed
aspect.aroundTagAssignmentDelete(proceedingJoinPoint, "machine-1", "tag-1", TagEntityType.DEVICE);

var inOrder = inOrder(machineTagEventService, proceedingJoinPoint);
inOrder.verify(machineTagEventService).processTagAssignmentDelete("machine-1", "tag-1");
inOrder.verify(proceedingJoinPoint).proceed();
```
