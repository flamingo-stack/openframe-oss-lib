Unit test class for verifying the behavior of `MachineTagEventAspect`, an AspectJ component that intercepts repository operations for machines, machine tags, and tags to trigger event processing.

## Key Components

- **Test Methods for After Advice**: Tests `afterMachineSave`, `afterMachineSaveAll`, `afterMachineTagSave`, and `afterMachineTagSaveAll` methods
- **Test Methods for Around Advice**: Tests `aroundTagSave` and `aroundTagSaveAll` methods with different tag scenarios
- **Error Handling Tests**: Verifies aspect behavior when services throw exceptions or invalid data is provided
- **Mock Dependencies**: Uses `@Mock` annotations for `MachineTagEventService` and `ProceedingJoinPoint`

## Usage Example

```java
@Test
void testMachineSaveAspect() {
    // Arrange
    Machine machine = new Machine();
    machine.setId("machine-1");
    machine.setHostname("test-machine");

    // Act - directly call the aspect method
    machineTagEventAspect.afterMachineSave(null, machine, machine);

    // Assert
    verify(machineTagEventService, times(1)).processMachineSave(machine);
}

@Test
void testAroundTagSave_ExistingTag() throws Throwable {
    // Arrange
    Tag tag = new Tag();
    tag.setId("tag-1");
    when(proceedingJoinPoint.proceed()).thenReturn(tag);

    // Act
    Object result = machineTagEventAspect.aroundTagSave(proceedingJoinPoint, tag);

    // Assert
    verify(machineTagEventService, times(1)).processTagSave(tag);
    assertEquals(tag, result);
}
```
