Configuration data class for Apache Pinot table setup, holding metadata references for schema and table configuration files used in the OpenFrame management layer.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | Identifier for the Pinot configuration entry |
| `schemaFile` | `String` | Path or reference to the Pinot schema file |
| `tableRealtimeConfigFile` | `String` | Path or reference to the realtime table config file |
| `tableOfflineConfigFile` | `String` | Path or reference to the offline table config file |

## Usage Example

```java
// Instantiate via all-args constructor
PinotConfig config = new PinotConfig(
    "events-table",
    "schema/events_schema.json",
    "config/events_realtime.json",
    "config/events_offline.json"
);

// Or use the no-args constructor with setters (Lombok @Data)
PinotConfig config = new PinotConfig();
config.setName("events-table");
config.setSchemaFile("schema/events_schema.json");
config.setTableRealtimeConfigFile("config/events_realtime.json");
config.setTableOfflineConfigFile("config/events_offline.json");

// Typically consumed via Spring @ConfigurationProperties binding
String schema = config.getSchemaFile();
```

> **Note:** Lombok annotations (`@Data`, `@AllArgsConstructor`, `@NoArgsConstructor`) auto-generate getters, setters, `equals`, `hashCode`, `toString`, and both constructors at compile time.
