Manages the lifecycle of Debezium connectors by communicating with the Kafka Connect REST API — handling creation, updates, versioned recreation on config drift, deletion, and status/config retrieval.

## Key Components

| Method | Description |
|--------|-------------|
| `createOrUpdateDebeziumConnector` | Entry point that processes an array of connector specs, routing each to the appropriate create/update/recreate path |
| `applyConnectorSpec` | Core dispatch logic — determines whether to create, update in-place (identity strategy), or recreate on drift (versioned strategy) |
| `recreateOnConfigDrift` | Compares running config against incoming spec; creates a new versioned connector and removes stale ones when drift is detected |
| `createConnector` | POSTs a new connector definition to Kafka Connect |
| `deleteConnector` | DELETEs a connector, tolerating 404 responses |
| `listConnectors` | Returns all registered connector names; returns empty list on transient failure |
| `getConnectorStatus` | Fetches `ConnectorStatus` for a named connector |
| `getConnectorConfig` | Fetches the live deployed config map for a named connector |
| `configsEquivalent` | Normalises and compares two config maps (stringifying values, ignoring `name` key) for semantic equality |

## Usage Example

```java
// Inject the service
@Autowired
private DebeziumService debeziumService;

// Create or update connectors from a spec array (e.g. loaded from MongoDB)
Object[] specs = toolRepository.findDebeziumConnectorSpecs();
debeziumService.createOrUpdateDebeziumConnector(specs);

// Manually create a connector with a resolved name
Map<String, Object> config = new HashMap<>();
config.put("connector.class", "io.debezium.connector.postgresql.PostgresConnector");
config.put("database.hostname", "postgres-host");
debeziumService.createConnector("my-connector-v2", config);

// Check connector status
ConnectorStatus status = debeziumService.getConnectorStatus("my-connector-v2");

// List all active connectors
List<String> connectors = debeziumService.listConnectors();
```

## Configuration Properties

| Property | Default | Description |
|----------|---------|-------------|
| `openframe.debezium.base-url` | *(required)* | Kafka Connect REST API base URL |
| `openframe.debezium.connect.connect-timeout` | `5s` | HTTP connection timeout |
| `openframe.debezium.connect.read-timeout` | `30s` | HTTP read timeout |
