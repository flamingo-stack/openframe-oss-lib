Manages Debezium connector lifecycle via the Kafka Connect REST API, handling creation, versioned recreation on config drift, updates, deletion, and status retrieval for CDC connectors in the OpenFrame platform.

## Key Components

| Member | Description |
|--------|-------------|
| `createOrUpdateDebeziumConnector(Object[])` | Entry point — processes an array of connector specs and applies create/update logic per spec |
| `applyConnectorSpec(Map)` | Routes each spec to the correct strategy: initial creation, identity update, or versioned recreation |
| `createInitialVersion(...)` | Creates a connector for the first time using the name strategy's resolved name |
| `tryUpdateExistingIdentityConnector(...)` | PUT-updates config in-place when the identity naming strategy is in use |
| `recreateOnConfigDrift(...)` | Detects config changes for versioned connectors, rate-gates recreation via `RecreationTracker`, creates new version, and prunes stale ones |
| `createConnector(String, Map)` | POST to `/connectors` with the effective connector name and config payload |
| `deleteConnector(String)` | DELETE `/connectors/{name}`, tolerates 404 |
| `listConnectors()` | GET `/connectors` — returns empty list on transient failure |
| `getConnectorStatus(String)` | GET `/connectors/{name}/status` — returns `null` on 404 |
| `getConnectorConfig(String)` | GET `/connectors/{name}/config` — returns `null` on 404 or unreachable |
| `configsEquivalent(Map, Map)` | Semantic equality check that normalizes all values to strings and ignores the `name` key |

## Usage Example

```java
// Spring-managed; inject and call after tools are resolved
@Autowired
private DebeziumService debeziumService;

// Apply a connector spec array sourced from an IntegratedTool document
Object[] specs = integratedTool.getDebeziumConnectors();
debeziumService.createOrUpdateDebeziumConnector(specs);

// Directly create a named connector
Map<String, Object> config = new HashMap<>();
config.put("connector.class", "io.debezium.connector.mongodb.MongoDbConnector");
config.put("mongodb.connection.string", "mongodb://localhost:27017");
config.put("topic.prefix", "openframe");
debeziumService.createConnector("openframe-mongo-v1", config);

// Check connector health
ConnectorStatus status = debeziumService.getConnectorStatus("openframe-mongo-v1");
if (status == null) {
    log.warn("Connector not found or Kafka Connect unreachable");
}

// Remove a stale versioned connector
debeziumService.deleteConnector("openframe-mongo-v0");
```
