DTO representing the health status of a Debezium connector, including its tasks. Provides helper methods to detect failures and extract error traces from the Kafka Connect status API response.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `name` | `String` | Connector name |
| `connector` | `Connector` | Top-level connector state (e.g. `RUNNING`, `FAILED`) |
| `tasks` | `List<TaskStatus>` | List of individual task statuses |
| `isConnectorFailed()` | Method | Returns `true` if the connector state is `FAILED` |
| `hasFailedTasks()` | Method | Returns `true` if any task has a `FAILED` state |
| `hasFailures()` | Method | Combines connector and task failure checks |
| `getFailedTasks()` | Method | Filters and returns only failed `TaskStatus` entries |
| `getFailureTraces()` | Method | Collects non-blank stack traces from all failed tasks |
| `getFirstFailureTrace()` | Method | Returns the first line of the first available failure trace |
| `Connector` | Inner class | Holds `state` and `workerId` for the connector |
| `TaskStatus` | Inner class | Holds `id`, `state`, `workerId`, `trace`, and failure helpers |

## Usage Example

```java
// Deserialize from Kafka Connect REST API response
ConnectorStatus status = objectMapper.readValue(apiResponse, ConnectorStatus.class);

if (status.hasFailures()) {
    if (status.isConnectorFailed()) {
        log.error("Connector {} is in FAILED state", status.getName());
    }

    for (ConnectorStatus.TaskStatus task : status.getFailedTasks()) {
        log.error("Task {} failed — {}", task.getId(), task.firstTraceLine());
    }

    // Get a concise summary for alerting
    String summary = status.getFirstFailureTrace();
    alertingService.notify(status.getName(), summary);
}
```
