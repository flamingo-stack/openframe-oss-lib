Defines a four-level severity classification enum used to categorize the urgency or impact of events, alerts, and log entries within the OpenFrame data model.

## Key Components

| Value | Description |
|-------|-------------|
| `INFO` | Informational events requiring no action |
| `WARNING` | Potential issues warranting attention |
| `ERROR` | Failures that impact functionality |
| `CRITICAL` | Severe failures requiring immediate response |

## Usage Example

```java
import com.openframe.data.cassandra.model.enums.Severity;

// Assign severity to an alert entity
AlertEntity alert = new AlertEntity();
alert.setSeverity(Severity.CRITICAL);

// Filter or branch logic by severity level
if (alert.getSeverity() == Severity.ERROR || alert.getSeverity() == Severity.CRITICAL) {
    notificationService.triggerPage(alert);
}
```
