A custom unchecked exception for signaling tenant-related errors within the Cassandra data layer of the OpenFrame platform.

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `CassandraTenantException(String message)` | Constructor | Creates exception with a descriptive error message |
| `CassandraTenantException(String message, Throwable cause)` | Constructor | Creates exception with a message and an underlying cause for exception chaining |

## Usage Example

```java
// Throw when a tenant context is missing or invalid
if (tenantId == null) {
    throw new CassandraTenantException("Tenant ID must not be null for Cassandra query execution");
}

// Wrap a lower-level exception with tenant context
try {
    resolveTenantKeyspace(tenantId);
} catch (Exception e) {
    throw new CassandraTenantException("Failed to resolve keyspace for tenant: " + tenantId, e);
}
```

> This exception extends `RuntimeException`, so it does not require explicit `throws` declarations. It is intended to surface multi-tenant isolation violations or misconfiguration errors at the Cassandra template layer in OpenFrame's data infrastructure.
