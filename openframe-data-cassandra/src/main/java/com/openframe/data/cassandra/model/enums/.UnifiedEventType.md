Enum defining all possible event types across the OpenFrame platform, each associated with a severity level and human-readable summary for audit logging and event tracking.

## Key Components

### Fields (via Lombok `@Getter`)
| Field | Type | Description |
|-------|------|-------------|
| `severity` | `Severity` | Event severity classification (`INFO`, `WARNING`, `ERROR`, `CRITICAL`) |
| `summary` | `String` | Human-readable description of the event |

### Event Categories

| Category | Examples |
|----------|---------|
| Authentication | `LOGIN`, `LOGOUT`, `LOGIN_FAILED`, `PASSWORD_CHANGED` |
| Device Management | `DEVICE_ONLINE`, `DEVICE_OFFLINE`, `DEVICE_REGISTERED` |
| User Management | `USER_CREATED`, `USER_UPDATED`, `USER_ROLE_CHANGED` |
| Scripts & Tasks | `SCRIPT_EXECUTED`, `TASK_FAILED`, `COMMAND_RUN_FINISHED` |
| Remote Access | `REMOTE_SESSION_START`, `REMOTE_RECORDING_COMPLETED` |
| Security | `HOST_WIPED` *(CRITICAL)*, `HOST_LOCKED` *(WARNING)* |
| MDM | `MDM_ENROLLED`, `MDM_UNENROLLED`, `MDM_ENABLED` |
| Monitoring | `ALERT_TRIGGERED`, `MONITORING_CHECK_FAILED` |
| Batch & Integrations | `BATCH_OPERATION_STARTED`, `INTEGRATION_ADDED` |

## Usage Example

```java
// Retrieving severity and summary from an event type
UnifiedEventType eventType = UnifiedEventType.SCRIPT_FAILED;

Severity severity = eventType.getSeverity(); // Severity.ERROR
String summary   = eventType.getSummary();   // "Script execution failed"

// Typical use in audit log creation
AuditEvent audit = AuditEvent.builder()
    .eventType(UnifiedEventType.HOST_WIPED)
    .severity(UnifiedEventType.HOST_WIPED.getSeverity()) // CRITICAL
    .message(UnifiedEventType.HOST_WIPED.getSummary())   // "Host wiped"
    .timestamp(Instant.now())
    .build();

// Filtering high-severity events
boolean isHighSeverity = eventType.getSeverity() == Severity.ERROR
    || eventType.getSeverity() == Severity.CRITICAL;
```
