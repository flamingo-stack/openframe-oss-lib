JUnit 5 integration test suite that validates policy creation and device assignment workflows in the OpenFrame monitoring system.

## Key Components

| Component | Description |
|-----------|-------------|
| `testAddPolicy()` | Creates an empty monitoring policy and verifies all fields (id, name, description, query) are persisted correctly |
| `testSelectPolicyDevice()` | Creates a Windows version policy, locates an online Windows device with active agents, assigns it to the policy via fleet ID, and confirms host inclusion |
| `BaseTest` | Parent class providing shared test setup/teardown |
| `MonitoringApi` | API client for policy CRUD and device selection operations |
| `DeviceApi` | API client for querying and retrieving device details |
| `MonitoringGenerator` | Factory methods (`emptyPolicy()`, `windowsVersionPolicy()`, `policyHost()`) for test data generation |
| `DeviceGenerator` | Filter helpers (`osAndStatusDevicesFilter()`, `getFleetId()`) for device queries |

## Usage Example

```java
// Create a basic policy and assert it was saved
CreatePolicyRequest request = emptyPolicy();
Policy policy = MonitoringApi.createPolicy(request);
assertThat(policy.getId()).isNotNull();

// Assign an online Windows device to a policy
List<Machine> devices = DeviceApi.getDevices(
    osAndStatusDevicesFilter("WINDOWS", DeviceStatus.ONLINE)
);
Machine device = DeviceApi.getDevice(devices.getFirst().getMachineId());
Integer fleetId = Integer.valueOf(getFleetId(device));

MonitoringApi.selectPolicyDevices(policy.getId(), fleetId);
Policy updated = MonitoringApi.getPolicy(policy.getId());
assertThat(updated.getHostsIncludeAny()).contains(policyHost(device));
```

## Test Tags

All tests are tagged `device`, allowing selective execution:

```bash
mvn test -Dgroups="device"
```
