Abstract base class for Playwright-based UI tests, providing shared browser lifecycle management, per-test browser context setup with tracing, automatic login via `UILoginFlow`, and failure artifact capture (trace zip, screenshot, HTML snapshot).

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `playwright` / `browser` | `static` fields | Shared Playwright instance and Chromium browser, created once per test class via `@BeforeAll`/`@AfterAll` |
| `context` / `page` | Instance fields | Fresh `BrowserContext` and `Page` created per test via `@BeforeEach` |
| `navigationSidebar` | Instance field | Returned by `UILoginFlow.login()` after automatic authentication in `@BeforeEach` |
| `afterTest` | `@RegisterExtension` | `AfterTestExecutionCallback` that triggers `captureArtifacts()` after every test |
| `beforeEachFail` | `@RegisterExtension` | `LifecycleMethodExecutionExceptionHandler` that captures artifacts if `@BeforeEach` itself throws |
| `captureArtifacts()` | Private method | On failure, writes a Playwright trace zip, full-page PNG screenshot, and HTML snapshot to `/tmp/playwright-failures/` |

## Usage Example

```java
class MyFeatureUITest extends BaseUITest {

    @Test
    void userCanViewDashboard() {
        // `page` and `navigationSidebar` are ready after automatic login
        navigationSidebar.goToDashboard();

        page.waitForSelector(".dashboard-widget");
        assertThat(page.locator(".dashboard-widget")).isVisible();
    }

    @Test
    void userCanNavigateToTickets() {
        navigationSidebar.goToTickets();

        assertThat(page.locator("h1")).containsText("Tickets");
    }
}
```

On test failure, artifacts are saved as:

```text
/tmp/playwright-failures/
  ├── userCanViewDashboard-1718123456789.trace.zip
  ├── userCanViewDashboard-1718123456789.html
  └── userCanViewDashboard-1718123456789.png
```

> **Note:** Browser credentials are resolved via `UserConfig.getUser()` — ensure the test user is configured in the environment before running UI tests.
