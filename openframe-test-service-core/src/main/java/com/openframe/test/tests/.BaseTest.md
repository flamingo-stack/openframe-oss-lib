Base class for all test classes in the OpenFrame test suite, ensuring a clean authentication state before each test class executes.

## Key Components

- **`setup()` (`@BeforeAll`)** – Static lifecycle method that clears stored authentication cookies via `AuthHelper.clearCookies()` before any tests in the subclass run.

## Usage Example

```java
// Extend BaseTest to inherit automatic cookie cleanup
public class DashboardTest extends BaseTest {

    @Test
    public void testDashboardLoads() {
        // Auth state is already cleared before this test class runs
        // Proceed with a fresh session
    }
}
```

> **Note:** All test classes in the OpenFrame suite should extend `BaseTest` to guarantee a consistent, unauthenticated starting state, preventing session bleed between test classes.
