JUnit 5 integration test suite validating CRUD operations and query functionality for the OpenFrame Knowledge Base API, covering folders, articles, tags, and item lifecycle management.

## Key Components

| Test Method | Tag(s) | Description |
|---|---|---|
| `testGetRootFolders` | `saas`, `read` | Verifies listing of root-level folders with required fields |
| `testGetArticlesInFolder` | `saas`, `read` | Validates article retrieval within a folder and filtered count |
| `testGetAllTags` | `saas`, `read` | Asserts at least one tag exists with valid `id` and `key` |
| `testCreateDraftArticle` | `saas` | Creates a folder, then a tagged draft article, validates all fields |
| `testPublishArticle` | `saas` | Creates a draft and transitions it to `PUBLISHED` status |
| `testUpdateArticle` | `saas` | Updates name, content, summary and verifies `updatedAt` changes |
| `testCreateAndDeleteFolder` | `saas` | Creates a root folder and confirms successful deletion |
| `testRenameFolder` | `saas` | Renames a folder and verifies the new name persists |
| `testMoveToFolder` | `saas` | Moves a child folder under a parent and checks `parentId` |
| `testGetKnowledgeBaseItemById` | `saas` | Fetches both a folder and article by ID and validates types |

## Usage Example

```java
// The suite is currently @Disabled — enable per test to run selectively:
@Tag("saas")
@Test
@DisplayName("Create draft article")
public void testCreateDraftArticle() {
    KnowledgeBaseItem folder = KnowledgeBaseApi.createFolder(randomFolderName(), null);
    CreateArticleInput input = draftArticle(folder.getId(), List.of(tagId));
    KnowledgeBaseItem created = KnowledgeBaseApi.createArticle(input);

    assertThat(created.getStatus()).isEqualTo(KnowledgeBaseArticleStatus.DRAFT);
    assertThat(created.getPublishedAt()).isNull();
}
```

> **Note:** The class is annotated `@Disabled` at the class level. Remove or override this annotation to activate tests. Tests are tagged `saas` (all) and `read` (non-mutating queries), enabling selective execution via JUnit tag filters.
