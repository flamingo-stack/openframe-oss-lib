JUnit 5 integration test suite for the OpenFrame Knowledge Base API, covering CRUD operations and folder/article management via `KnowledgeBaseApi`.

## Key Components

| Test Method | Tag(s) | Description |
|---|---|---|
| `testGetRootFolders` | `saas`, `read` | Validates root folder listing — checks id, name, type, parentId, timestamps |
| `testGetArticlesInFolder` | `saas`, `read` | Finds a folder containing articles and validates article fields + filtered count |
| `testGetAllTags` | `saas`, `read` | Asserts at least one tag exists with non-blank id and key |
| `testCreateDraftArticle` | `saas` | Creates a folder, attaches a tag, creates a draft article, validates all fields |
| `testPublishArticle` | `saas` | Creates a draft then publishes it, verifies status transitions to `PUBLISHED` |
| `testUpdateArticle` | `saas` | Creates an article, updates name/content/summary, verifies field changes and `updatedAt` mutation |
| `testCreateAndDeleteFolder` | `saas` | Creates a root folder and immediately deletes it |
| `testRenameFolder` | `saas` | Creates and renames a folder, verifies new name |
| `testMoveToFolder` | `saas` | Creates parent/child folders, moves child under parent, validates `parentId` |
| `testGetKnowledgeBaseItemById` | `saas` | Creates a folder + article, fetches by id (truncated in source) |

## Usage Example

```java
// Run only read-only tests against SaaS environment
@Tag("saas")
@Tag("read")
@Test
@DisplayName("Get root folders")
public void testGetRootFolders() {
    List<KnowledgeBaseItem> folders = KnowledgeBaseApi.getKnowledgeBaseItems(
        rootFoldersFilter(), null, 100
    );

    assertThat(folders).isNotEmpty();
    assertThat(folders).allSatisfy(folder -> {
        assertThat(folder.getType()).isEqualTo(KnowledgeBaseItemType.FOLDER);
        assertThat(folder.getParentId()).isNull(); // root = no parent
    });
}

// Create and publish an article
KnowledgeBaseItem folder = KnowledgeBaseApi.createFolder(randomFolderName(), null);
KnowledgeBaseItem draft = KnowledgeBaseApi.createArticle(draftArticle(folder.getId(), List.of()));
KnowledgeBaseItem published = KnowledgeBaseApi.publishArticle(draft.getId());
assertThat(published.getStatus()).isEqualTo(KnowledgeBaseArticleStatus.PUBLISHED);
```

> **Note:** All tests extend `BaseTest` and are tagged `saas`. Read-only tests additionally carry the `read` tag, enabling selective CI execution. Test data is generated via `KnowledgeBaseGenerator` static factory methods such as `randomFolderName()` and `draftArticle()`.
