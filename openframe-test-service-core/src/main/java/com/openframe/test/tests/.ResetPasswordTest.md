Test class for password reset functionality with comprehensive validation of both successful reset flows and error scenarios.

## Key Components

- **testResetPassword()** - Complete password reset flow test that initiates reset, retrieves token from Redis, confirms reset with new password, and verifies authentication
- **testResetPasswordWithInvalidToken()** - Negative test case validating proper error handling when an invalid reset token is provided
- **UnauthorizedTest** - Base class providing test infrastructure for unauthenticated operations
- **Redis** integration for token retrieval and validation
- **AuthGenerator** for creating test data and expected error responses

## Usage Example

```java
// Run the complete password reset test suite
@Tag("reset")
public class ResetPasswordTest extends UnauthorizedTest {
    
    // Test successful password reset flow
    @Test
    public void testResetPassword() {
        // Get user and initiate reset
        User user = UserConfig.getUser();
        UserApi.resetPassword(user);
        
        // Retrieve token from Redis cache
        String token = Redis.getResetToken(user.getEmail());
        
        // Confirm reset with new password
        ResetConfirmRequest request = AuthGenerator.resetConfirmRequest(token);
        UserApi.confirmReset(request);
        
        // Verify user is authenticated with new credentials
        user.setPassword(request.getNewPassword());
        assertThat(UserApi.me().isAuthenticated()).isTrue();
    }
}
```

The test validates the complete password reset workflow from initiation through Redis token storage to successful password change and authentication verification.
