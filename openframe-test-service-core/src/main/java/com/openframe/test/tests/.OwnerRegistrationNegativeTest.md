A JUnit test class that validates negative scenarios for owner user registration in the OpenFrame platform, ensuring proper error handling when registration fails.

## Key Components

- **OwnerRegistrationNegativeTest** - Main test class extending `UnauthorizedTest` for testing registration failures
- **testRegistrationClosed()** - Verifies that registration is blocked when the registration system is closed
- **testRegisterExistingUser()** - Ensures duplicate email registration is properly rejected with appropriate error response

## Usage Example

```java
// Test execution via Maven or Gradle
mvn test -Dtest=OwnerRegistrationNegativeTest

// Individual test execution
mvn test -Dtest=OwnerRegistrationNegativeTest#testRegistrationClosed

// Running only OSS-tagged tests
mvn test -Dgroups="oss"
```

The test class uses AssertJ for fluent assertions and leverages test data generators to create realistic scenarios. Each test retrieves an existing user from the database, attempts registration with that user's credentials, and validates that the appropriate error response is returned. This ensures the registration API properly handles edge cases and provides meaningful error messages to clients attempting invalid registrations.
