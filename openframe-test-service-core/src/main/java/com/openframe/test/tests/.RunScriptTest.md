JUnit 5 integration test suite validating script execution and scheduling workflows against the OpenFrame platform's Script and Device APIs.

## Key Components

| Component | Description |
|-----------|-------------|
| `testBulkRun()` | Verifies that a speed test script can be dispatched immediately to an online Windows device via its Tactical agent |
| `testScheduleScript()` | Creates a new script schedule and assigns an online Windows device, asserting that agents and task results are registered |
| `testScheduleExecution()` | Retrieves the most recent script schedule's execution history and validates result fields (execution time, last run date, agent ID/hostname, status) |

## Usage Example

```java
// Bulk run a script against an online Windows device
List<Machine> devices = DeviceApi.getDevices(
    osAndStatusDevicesFilter("WINDOWS", DeviceStatus.ONLINE)
);
Machine device = DeviceApi.getDevice(devices.getFirst().getMachineId());
String tacticalId = getTacticalId(device);

RunScriptRequest runScriptRequest = runSpeedTestScriptRequest(tacticalId);
String response = ScriptApi.runScript(runScriptRequest);
assertThat(response).contains("will now be run");

// Schedule a script and assign a device
CreateScriptScheduleRequest scheduleRequest = createOnlineCheckSchedule();
ScriptSchedule schedule = ScriptApi.createScriptSchedule(scheduleRequest);
ScheduleAssignDeviceResponse assignResponse =
    ScriptApi.scheduleAssignDevice(schedule.getId(), tacticalId);
assertThat(assignResponse.getAgentsCount()).isNotZero();
```

## Test Tags

| Tag | Purpose |
|-----|---------|
| `device` | Tests requiring a live, online Windows device with a Tactical agent |
| `scheduled` | Tests validating previously created schedule execution history |
