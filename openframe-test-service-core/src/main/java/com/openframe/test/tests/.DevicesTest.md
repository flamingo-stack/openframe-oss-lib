Comprehensive JUnit test suite for validating device management functionality in the OpenFrame platform, including device filtering, searching, archiving, and deletion operations.

## Key Components

- **`DevicesTest`** - Main test class extending `AuthorizedTest` for authenticated device operations
- **Device Filter Tests** - Validates device filtering capabilities by status, OS type, and organization
- **Device Retrieval Tests** - Tests getting devices by machine ID and searching by hostname
- **Device Lifecycle Tests** - Tests archiving and deletion of offline devices
- **Monitor Tests** - Tagged tests for continuous monitoring of device functionality

## Usage Example

```java
// Test device filtering functionality
@Test
@DisplayName("Verify device filters")
public void testGetDeviceFilters() {
    DeviceFilters filters = DeviceApi.getDeviceFilters();
    assertThat(filters).isNotNull();
    assertThat(filters.getStatuses()).isNotEmpty();
    assertThat(filters.getFilteredCount()).isNotZero();
}

// Test device retrieval by machine ID
@Test
@DisplayName("Get device by machineId")
public void testGetDevice() {
    List<Machine> devices = DeviceApi.getDevices(DeviceGenerator.onlineDevicesFilter());
    assertThat(devices).as("Expected at least one ONLINE device").isNotEmpty();
    Machine device = DeviceApi.getDevice(devices.getFirst().getMachineId());
    assertThat(device).isNotNull();
}
```

The test suite uses `@Tag("monitor")` for automated monitoring tests and `@Tag("shared")` for shared test scenarios. All tests leverage the `DeviceApi` for device operations and `DeviceGenerator` for creating test filters, with comprehensive assertions using AssertJ.
