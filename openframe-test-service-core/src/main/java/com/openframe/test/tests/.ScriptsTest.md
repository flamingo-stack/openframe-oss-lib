This test class provides comprehensive integration testing for script management operations in the OpenFrame platform, covering CRUD operations and validation of script data integrity.

## Key Components

- **`testListScripts()`** - Validates script listing functionality and ensures all returned scripts have required fields (ID, name, description)
- **`testGetScript()`** - Tests individual script retrieval by ID and verifies script body content is populated
- **`testAddScript()`** - Creates new scripts using generated test data and validates the API response
- **`testEditScript()`** - Updates existing script descriptions and confirms changes are persisted correctly
- **`ScriptApi`** - Static API client for script operations (list, get, add, edit)
- **`ScriptGenerator`** - Test data factory for creating script objects and expected responses

## Usage Example

```java
// Run all script tests
@Test
public void runScriptTests() {
    // List and validate scripts
    List<Script> scripts = ScriptApi.listScripts();
    assertThat(scripts).isNotEmpty();
    
    // Get specific script with full details
    Script detailedScript = ScriptApi.getScript(scripts.get(0).getId());
    assertThat(detailedScript.getScriptBody()).isNotEmpty();
    
    // Create new script
    Script newScript = ScriptGenerator.addScriptRequest();
    String addResponse = ScriptApi.addScript(newScript);
    
    // Update script
    newScript.setDescription("Modified description");
    String editResponse = ScriptApi.editScript(newScript);
}
```

The test class extends `AuthorizedTest` for authenticated API access and uses AssertJ for fluent assertions. Tests are tagged for categorization (`shared`, `monitor`) and include descriptive display names for clear test reporting.
