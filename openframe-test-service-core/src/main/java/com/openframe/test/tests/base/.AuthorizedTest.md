Base abstract test class that provides authentication setup and database connection management for OpenFrame test suites requiring authorized access.

## Key Components

- **Abstract test class**: Must be extended by concrete test classes
- **@BeforeAll config()**: Configures RestAssured settings and opens MongoDB connection
- **@AfterAll close()**: Cleans up by closing MongoDB connection
- **Static lifecycle methods**: Ensures proper test environment setup/teardown

## Usage Example

```java
// Extend AuthorizedTest for tests requiring authentication
public class UserApiTest extends AuthorizedTest {
    
    @Test
    void testGetUserProfile() {
        given()
            .when()
                .get("/api/users/profile")
            .then()
                .statusCode(200)
                .body("username", notNullValue());
    }
    
    @Test
    void testCreateUser() {
        User newUser = new User("testuser", "test@example.com");
        
        given()
            .contentType(ContentType.JSON)
            .body(newUser)
            .when()
                .post("/api/users")
            .then()
                .statusCode(201);
    }
}
```

This class ensures that all extending test classes automatically have RestAssured configured for API testing and MongoDB connections properly managed throughout the test lifecycle.
