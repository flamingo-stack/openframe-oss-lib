Provides a REST API client method for discovering the tenant ID associated with a given user's email address via the authentication service.

## Key Components

| Component | Description |
|-----------|-------------|
| `TenantApi` | Static utility class wrapping the tenant discovery endpoint |
| `discoverTenant(User user)` | Sends a GET request to `sas/tenant/discover` and returns the resolved `tenant_id` |
| `DISCOVER` | Constant holding the tenant discovery endpoint path |

## Usage Example

```java
User user = new User("admin@example.com", "password123");
String tenantId = TenantApi.discoverTenant(user);
System.out.println("Tenant ID: " + tenantId);
```

## Notes

- Uses `RestAssured` under the hood with a pre-configured auth flow request spec from `RequestSpecHelper`
- Base URI is resolved dynamically via `EnvironmentConfig.getAuthUrl()`
- Asserts a `200` HTTP status code before extracting the `tenant_id` from the JSON response — any non-200 response will fail the test immediately
