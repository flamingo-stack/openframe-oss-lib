A GraphQL API client class for the OpenFrame Knowledge Base module, providing static helper methods for all knowledge base operations used in automated testing via REST Assured.

## Key Components

| Method | Description |
|--------|-------------|
| `getKnowledgeBaseItem(String id)` | Fetches a single KB item by ID |
| `getKnowledgeBaseFolderTree()` | Retrieves the full folder hierarchy |
| `getKnowledgeBaseArticleTree()` | Retrieves the full article hierarchy |
| `getKnowledgeBaseItems(...)` | Lists items with filter, search, and pagination |
| `getKnowledgeBaseItemsFilteredCount(...)` | Returns the filtered total count of items |
| `getKnowledgeBaseTags(String folderId)` | Lists tags scoped to a folder |
| `createArticle(CreateArticleInput)` | Creates a new article |
| `publishArticle(String articleId)` | Publishes a draft article |
| `updateArticle(UpdateArticleInput)` | Updates article content or metadata |
| `createFolder(String name, String parentId)` | Creates a new folder |
| `renameFolder(String folderId, String name)` | Renames an existing folder |
| `moveToFolder(String itemId, String parentId)` | Moves an item to a different folder |
| `deleteFolder(DeleteFolderInput)` | Deletes a folder, returns boolean success |
| `archiveArticle(String articleId)` | Archives an article |
| `unarchiveArticle(String articleId, String parentId)` | Restores an archived article |
| `getArchivedArticles(...)` | Lists archived articles with search/tag filters |
| `addTagToItem(String itemId, String tagId)` | Attaches a tag to a KB item |
| `removeTagFromItem(String itemId, String tagId)` | Detaches a tag from a KB item |

## Usage Example

```java
// Fetch a knowledge base item by ID
KnowledgeBaseItem item = KnowledgeBaseApi.getKnowledgeBaseItem("article-123");

// Create a new article inside a folder
CreateArticleInput input = new CreateArticleInput("My Article", "folder-456");
KnowledgeBaseItem created = KnowledgeBaseApi.createArticle(input);

// Publish the article
KnowledgeBaseItem published = KnowledgeBaseApi.publishArticle(created.getId());

// Search items with a filter
KnowledgeBaseFilterInput filter = new KnowledgeBaseFilterInput("ARTICLE");
List<KnowledgeBaseItem> items = KnowledgeBaseApi.getKnowledgeBaseItems(filter, "setup guide", 20);

// Tag management
KnowledgeBaseApi.addTagToItem(created.getId(), "tag-789");
KnowledgeBaseApi.removeTagFromItem(created.getId(), "tag-789");

// Archive and restore
KnowledgeBaseApi.archiveArticle(created.getId());
KnowledgeBaseApi.unarchiveArticle(created.getId(), "folder-456");
```
