Provides REST-Assured-based API methods for executing each step of the OpenFrame OAuth2/PKCE authentication flow, from initiating login through token exchange, refresh, and logout.

## Key Components

| Method | Description |
|---|---|
| `startOAuthFlow` | Initiates the OAuth login redirect with tenant ID and dashboard redirect URL |
| `initiateAuthorization` | Begins the `/oauth2/authorize` flow with PKCE parameters |
| `submitCredentials` | POSTs username/password to the SAS login endpoint using session cookie |
| `getAuthorizationCode` | Re-requests the authorization endpoint with a `continue` flag to obtain the auth code |
| `exchangeCodeForTokens` | Exchanges the authorization code and state via the OAuth callback endpoint |
| `refresh` | Overloaded — refreshes tokens using `refresh_token` cookie, with or without `tenantId` |
| `attemptRefresh` | Raw refresh attempt returning the full `Response` for assertion testing |
| `logout` | Overloaded — clears the session via `oauth/logout`, with or without `tenantId` |
| `getAuthQueryParams` | Private helper that builds PKCE query parameters (`code_challenge`, `S256`, scopes, etc.) |

## Usage Example

```java
// Step-by-step OAuth2 PKCE flow
AuthParts authParts = AuthPartsBuilder.build(user);

Response loginRedirect = AuthApi.startOAuthFlow(authParts);
authParts.setCookies(loginRedirect.getCookies());

Response authResponse = AuthApi.initiateAuthorization(authParts);
authParts.setCookies(authResponse.getCookies());

Response credentialsResponse = AuthApi.submitCredentials(authParts);
authParts.setCookies(credentialsResponse.getCookies());

Response codeResponse = AuthApi.getAuthorizationCode(authParts);
authParts.setAuthorizationCode(extractCode(codeResponse));

Response tokenResponse = AuthApi.exchangeCodeForTokens(authParts);
Map<String, String> sessionCookies = tokenResponse.getCookies();

// Refresh and logout
Map<String, String> refreshed = AuthApi.refresh(user.getTenantId(), sessionCookies);
AuthApi.logout(user.getTenantId(), refreshed);
```
