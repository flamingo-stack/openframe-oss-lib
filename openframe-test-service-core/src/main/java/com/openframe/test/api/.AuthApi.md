A REST API wrapper class for handling OAuth2 authentication flows in the OpenFrame testing framework. This utility provides methods to execute the complete OAuth2 authorization code flow with PKCE, including login, token refresh, and logout operations.

## Key Components

- **OAuth Flow Methods**: `startOAuthFlow()`, `initiateAuthorization()`, `submitCredentials()`, `getAuthorizationCode()`, `exchangeCodeForTokens()` - Complete OAuth2 authorization code flow
- **Token Management**: `refresh()` - Refresh access tokens using refresh tokens (multiple overloads)  
- **Session Management**: `logout()` - Terminate user sessions and clear cookies
- **Helper Methods**: `getAuthQueryParams()` - Builds standard OAuth2 query parameters with PKCE
- **Constants**: OAuth endpoints, client ID, and redirect URIs for consistent API calls

## Usage Example

```java
// Start OAuth flow for a tenant
AuthParts authParts = new AuthParts("tenant123", "user@example.com", "password");
Response loginResponse = AuthApi.startOAuthFlow(authParts);

// Complete authorization flow
Response authResponse = AuthApi.initiateAuthorization(authParts);
Response credentialsResponse = AuthApi.submitCredentials(authParts);
Response codeResponse = AuthApi.getAuthorizationCode(authParts);
Response tokenResponse = AuthApi.exchangeCodeForTokens(authParts);

// Refresh tokens when needed
Map<String, String> newCookies = AuthApi.refresh(tenant, currentCookies);

// Logout user
Map<String, String> clearedCookies = AuthApi.logout(tenant, currentCookies);
```

The class uses RestAssured for HTTP requests and integrates with the OpenFrame authentication system, supporting both tenant-specific and tenant-agnostic operations through method overloading.
