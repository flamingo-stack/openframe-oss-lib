API client for querying log data via GraphQL, providing methods to retrieve log filters, paginated log events, and detailed log information using RestAssured for HTTP interactions.

## Key Components

| Method | Description |
|--------|-------------|
| `getLogFilters()` | Fetches available filter options for log queries, returns `LogFilters` |
| `getLogs()` | Retrieves all log events without filters, returns `List<LogEvent>` |
| `getLogs(LogFilterInput)` | Retrieves log events matching a specific `LogFilterInput`, returns `List<LogEvent>` |
| `searchLogs(String)` | Retrieves log events matching a search string, returns `List<LogEvent>` |
| `getLogDetails(LogEvent)` | Fetches full details for a specific log event using its identifying fields, returns `LogDetails` |

## Usage Example

```java
// Fetch available log filters
LogFilters filters = LogsApi.getLogFilters();

// Retrieve all logs
List<LogEvent> allLogs = LogsApi.getLogs();

// Retrieve logs with a filter applied
LogFilterInput filter = new LogFilterInput();
filter.setToolType("N-central");
List<LogEvent> filteredLogs = LogsApi.getLogs(filter);

// Search logs by keyword
List<LogEvent> searchResults = LogsApi.searchLogs("authentication failed");

// Fetch details for a specific log event
LogEvent event = allLogs.get(0);
LogDetails details = LogsApi.getLogDetails(event);
```

> **Note:** All methods require a valid authorized session. Authentication is handled automatically via `getAuthorizedSpec()`. GraphQL queries are sourced from `LogQueries` constants and responses are validated against the `graphqlSuccess()` response spec.
