A GraphQL API client for managing log-related operations in OpenFrame's test automation framework. This class provides methods to retrieve log filters, fetch logs with various filtering options, search logs, and get detailed log information.

## Key Components

- **getLogFilters()** - Retrieves available log filters configuration
- **getLogs()** - Fetches all logs without filters
- **getLogs(LogFilterInput filter)** - Retrieves logs with specific filtering criteria
- **searchLogs(String search)** - Searches logs by text query
- **getLogDetails(LogEvent logEvent)** - Gets detailed information for a specific log event

All methods use GraphQL queries defined in the `LogQueries` class and return strongly-typed DTOs for log data structures.

## Usage Example

```java
// Get available log filters
LogFilters filters = LogsApi.getLogFilters();

// Retrieve all logs
List<LogEvent> allLogs = LogsApi.getLogs();

// Filter logs by specific criteria
LogFilterInput filter = new LogFilterInput();
filter.setToolType("authentication");
List<LogEvent> filteredLogs = LogsApi.getLogs(filter);

// Search logs by text
List<LogEvent> searchResults = LogsApi.searchLogs("error");

// Get detailed information for a specific log event
LogEvent logEvent = allLogs.get(0);
LogDetails details = LogsApi.getLogDetails(logEvent);
```
