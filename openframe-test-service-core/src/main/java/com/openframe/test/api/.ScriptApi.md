GraphQL API client for OpenFrame script management, providing methods to list, fetch, create, update, delete, and archive scripts within the OpenFrame/Flamingo test framework.

## Key Components

| Method | Description |
|--------|-------------|
| `listScripts()` | Retrieves a list of scripts via GraphQL |
| `getScript(String id)` | Fetches a single script by ID |
| `createScript(Script script)` | Creates a new script |
| `updateScript(Script script)` | Updates an existing script by ID |
| `deleteScript(String id)` | Deletes a script by ID |
| `archiveScript(String id)` | Archives a script by ID |

## Usage Example

```java
// List available scripts
List<Script> scripts = ScriptApi.listScripts();

// Fetch and update a specific script
Script script = ScriptApi.getScript("script-42");
script.setName("Updated Script Name");
ScriptApi.updateScript(script);

// Create a new script
Script created = ScriptApi.createScript(newScript);

// Archive or delete a script
ScriptApi.archiveScript(created.getId());
```

> **Note:** All methods use an authorized GraphQL request spec via `getAuthorizedSpec()` and expect HTTP `200` responses.
