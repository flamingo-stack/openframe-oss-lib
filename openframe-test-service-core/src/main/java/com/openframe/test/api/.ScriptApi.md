REST API client for interacting with Tactical RMM script management endpoints, providing methods to create, edit, run, and schedule scripts on managed agents within the OpenFrame/Flamingo test framework.

## Key Components

| Method | Description |
|--------|-------------|
| `listScripts()` | Retrieves up to 9 scripts from the scripts list |
| `getScript(Integer id)` | Fetches a single script by ID |
| `addScript(Script script)` | Creates a new script |
| `editScript(Script script)` | Updates an existing script by ID |
| `runScript(RunScriptRequest request)` | Executes a bulk script run against agents |
| `createScriptSchedule(CreateScriptScheduleRequest request)` | Creates a new script schedule |
| `scheduleAssignDevice(Integer scheduleId, String... agents)` | Assigns agents to a script schedule |
| `getScheduleExecutionHistory(Integer scheduleId)` | Retrieves execution history (limit 50) for a schedule |
| `getScriptSchedules()` | Lists all script schedules |

## API Endpoints

```text
GET/POST   tools/tactical-rmm/scripts/
GET/PUT    tools/tactical-rmm/scripts/{id}/
POST       tools/tactical-rmm/agents/actions/bulk/
GET/POST   tools/tactical-rmm/script-schedules/
PUT        tools/tactical-rmm/script-schedules/{id}/agents/
GET        tools/tactical-rmm/script-schedules/{id}/history/
```

## Usage Example

```java
// List available scripts (returns up to 9)
List<Script> scripts = ScriptApi.listScripts();

// Fetch and edit a specific script
Script script = ScriptApi.getScript(42);
script.setName("Updated Script Name");
ScriptApi.editScript(script);

// Run a script against multiple agents
RunScriptRequest runRequest = new RunScriptRequest();
// configure runRequest...
ScriptApi.runScript(runRequest);

// Create a schedule and assign agents
CreateScriptScheduleRequest scheduleRequest = new CreateScriptScheduleRequest();
ScriptSchedule schedule = ScriptApi.createScriptSchedule(scheduleRequest);

ScriptApi.scheduleAssignDevice(schedule.getId(), "agent-001", "agent-002");

// Review execution history
ScheduleExecutionHistory history = ScriptApi.getScheduleExecutionHistory(schedule.getId());
```

> **Note:** All methods use an authorized request spec via `getAuthorizedSpec()` and expect HTTP `200` responses. `listScripts()` returns `null` if fewer than 10 scripts exist.
