API client for managing scripts in the OpenFrame tactical RMM system. Provides methods to list, retrieve, create, and update scripts via REST endpoints.

## Key Components

- **`SCRIPTS`** - Base endpoint path for scripts collection (`tools/tactical-rmm/scripts/`)
- **`SCRIPT`** - Individual script endpoint path with ID parameter (`{id}/`)
- **`listScripts()`** - Retrieves all scripts as a list
- **`getScript(Integer id)`** - Fetches a specific script by ID
- **`editScript(Script script)`** - Updates an existing script
- **`addScript(Script script)`** - Creates a new script

## Usage Example

```java
// List all available scripts
List<Script> scripts = ScriptApi.listScripts();

// Get a specific script by ID
Script script = ScriptApi.getScript(123);

// Create a new script
Script newScript = new Script();
newScript.setName("Backup Script");
newScript.setContent("#!/bin/bash\necho 'Running backup'");
String createResponse = ScriptApi.addScript(newScript);

// Update an existing script
script.setContent("#!/bin/bash\necho 'Updated backup script'");
String updateResponse = ScriptApi.editScript(script);
```

All methods use authorized request specifications and return appropriate response types - collections/objects for read operations, strings for write operations. The API expects 200 status codes for successful operations.
