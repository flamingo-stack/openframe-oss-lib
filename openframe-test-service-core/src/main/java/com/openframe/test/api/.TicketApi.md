A test API client for interacting with the OpenFrame ticket management system via GraphQL mutations and queries, used in automated test suites.

## Key Components

| Method | Description |
|--------|-------------|
| `getTickets()` | Fetches a paginated, filtered list of tickets |
| `getTicket()` | Retrieves a single ticket by ID |
| `createTicket()` | Creates a new ticket from a `CreateTicketInput` |
| `reorderTicket()` | Reorders a ticket using `ReorderTicketInput` |
| `getTicketStatuses()` | Lists the lifecycle status definitions (`ticketStatuses`) |
| `createTicketStatus()` | Creates a custom status definition |
| `deleteTicketStatus()` | Deletes a custom status definition (optional replacement status id for in-use statuses); returns `boolean` |
| `attemptDeleteTicketStatusErrors()` | Deletes a status expected to be rejected (e.g. a system status); returns the top-level GraphQL `errors` |
| `resolveSystemStatusId()` | Resolves the id of the system status definition for a lifecycle `kind` (e.g. `RESOLVED`, `ARCHIVED`) |
| `transitionTicket()` | Transitions a ticket to a status definition id; throws on `userErrors` |
| `attemptTransitionTicketErrors()` | Transitions a ticket expected to be rejected; returns the top-level GraphQL `errors` (invalid transitions surface there, not in `userErrors`) |
| `getTicketLabels()` | Retrieves all available ticket labels |

**Private helpers:**

- `mutateTicket()` — Shared mutation executor; automatically asserts no `userErrors` are returned and extracts the resulting `Ticket` object.

## Usage Example

```java
// Create a new ticket
CreateTicketInput input = CreateTicketInput.builder()
        .subject("Printer not working")
        .build();
Ticket created = TicketApi.createTicket(input);

// Fetch tickets with filter and pagination
TicketFilterInput filter = TicketFilterInput.builder().status("OPEN").build();
CursorPaginationInput pagination = CursorPaginationInput.builder().first(10).build();
TicketConnection tickets = TicketApi.getTickets(filter, pagination, "printer");

// Resolve a ticket by transitioning it to the RESOLVED system status
String resolvedStatusId = TicketApi.resolveSystemStatusId("RESOLVED");
Ticket resolved = TicketApi.transitionTicket(created.getId(), resolvedStatusId);

// Attempt a transition expected to be rejected and inspect the top-level GraphQL errors
String archivedStatusId = TicketApi.resolveSystemStatusId("ARCHIVED");
List<GraphqlError> errors = TicketApi.attemptTransitionTicketErrors(created.getId(), archivedStatusId);
if (!errors.isEmpty()) {
    System.out.println("Expected error code: " + errors.get(0).getExtensions().get("code"));
}
```
