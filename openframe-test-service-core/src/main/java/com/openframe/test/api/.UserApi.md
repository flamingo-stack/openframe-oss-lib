REST API client utility class for user-related operations in the OpenFrame test suite. Provides methods for user authentication, profile management, and password reset functionality using RestAssured.

## Key Components

- **me()** - Retrieves current user profile information
- **deleteUser(String userId)** - Deletes a user account by ID
- **resetPassword(User user)** - Initiates password reset flow for a user
- **confirmReset(ResetConfirmRequest request)** - Confirms password reset with new credentials
- **attemptConfirmReset(ResetConfirmRequest request)** - Tests invalid password reset scenarios

## API Endpoints

- `api/me` - User profile endpoint
- `api/users/{userId}` - User management operations
- `sas/password-reset/request` - Password reset initiation
- `sas/password-reset/confirm` - Password reset confirmation

## Usage Example

```java
// Get current user information
MeResponse userProfile = UserApi.me();
System.out.println("User: " + userProfile.getName());

// Delete a user account
int statusCode = UserApi.deleteUser("user-123");

// Reset password flow
User user = new User();
user.setEmail("user@example.com");
UserApi.resetPassword(user);

// Confirm password reset
ResetConfirmRequest resetRequest = new ResetConfirmRequest();
resetRequest.setToken("reset-token");
resetRequest.setNewPassword("newPassword123");
UserApi.confirmReset(resetRequest);

// Test invalid reset (returns error response)
ErrorResponse error = UserApi.attemptConfirmReset(invalidRequest);
```

The class uses both authorized and unauthorized request specifications depending on the operation, with proper HTTP status code validation for each endpoint.
