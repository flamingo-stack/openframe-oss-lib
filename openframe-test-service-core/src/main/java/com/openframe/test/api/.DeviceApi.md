Provides a REST and GraphQL API client for interacting with device-related endpoints in the OpenFrame test suite, supporting CRUD operations, pagination, filtering, and multi-tool integrations (Fleet MDM, Tactical RMM, MeshCentral).

## Key Components

| Method | Description |
|--------|-------------|
| `getDeviceHostnames(filter)` | Fetches a list of device hostnames matching the given filter via GraphQL |
| `getDeviceIds(filter)` | Returns machine IDs for devices matching the filter |
| `getAllDeviceIds(filter, first)` | Paginates through all matching devices and collects machine IDs |
| `getDevice(machineId)` | Retrieves full device details by machine ID |
| `getAnyDevice(filters...)` | Returns the first available device matching any of the provided filters |
| `getDevices(filter)` | Returns a list of `Machine` objects matching the filter |
| `getAllDevices(filter, first)` | Paginates through all devices and extracts Fleet MDM agent IDs |
| `updateDeviceStatus(device, status)` | PATCHes a device's status (e.g., `ARCHIVED`, `DELETED`) |
| `archiveDevice(device)` / `deleteDevice(device)` | Convenience wrappers around `updateDeviceStatus` |
| `searchDevice(filter, search)` | Searches for a device using a text query and filter |
| `getTacticalInfo(tacticalId)` | Fetches agent details from Tactical RMM |
| `getMeshInfo(meshId)` | Retrieves device status from MeshCentral |
| `getFleetInfo(fleetId)` / `getFleetOsVersion(fleetId)` | Queries Fleet MDM for host details or OS version |
| `getDeviceFilters()` / `getDeviceFilters(filter)` | Retrieves available device filter options |

## Usage Example

```java
// Retrieve devices matching a specific filter
DeviceFilterInput filter = new DeviceFilterInput();
filter.setOs("Windows");

List<Machine> devices = DeviceApi.getDevices(filter);
Machine first = devices.getFirst();

// Archive a device
DeviceApi.archiveDevice(first);

// Fetch cross-tool info for the same device
String tacticalId = first.getTacticalId();
TacticalAgent agent = DeviceApi.getTacticalInfo(tacticalId);

// Paginate through all device IDs
List<String> allIds = DeviceApi.getAllDeviceIds(filter, 50);

// Get Fleet MDM OS version
String osVersion = DeviceApi.getFleetOsVersion(first.getFleetId());
```
