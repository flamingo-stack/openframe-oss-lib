A comprehensive API wrapper class for device management operations in OpenFrame's test automation framework, providing methods to query, filter, update, and manage device lifecycle through both GraphQL and REST endpoints.

## Key Components

**Static Methods:**
- `getDeviceHostnames(DeviceFilterInput)` - Retrieves filtered list of device hostnames via GraphQL
- `getDeviceIds(DeviceFilterInput)` - Fetches device machine IDs with applied filters
- `getDevice(String)` - Gets complete device information by machine ID
- `getDevices(DeviceFilterInput)` - Returns filtered list of Machine objects
- `updateDeviceStatus(Machine, DeviceStatus)` - Updates device status via REST PATCH
- `archiveDevice(Machine)` - Sets device status to ARCHIVED
- `deleteDevice(Machine)` - Sets device status to DELETED
- `searchDevice(DeviceFilterInput, String)` - Performs device search with filters
- `getDeviceFilters()` - Retrieves available device filter options

**Dependencies:**
- Uses RestAssured for HTTP requests with authorized specifications
- Leverages GraphQL queries from `DeviceQueries` constants
- Integrates with DTO classes for type-safe data handling

## Usage Example

```java
// Get devices by filter
DeviceFilterInput filter = new DeviceFilterInput();
filter.setStatus(DeviceStatus.ACTIVE);

List<Machine> devices = DeviceApi.getDevices(filter);
List<String> hostnames = DeviceApi.getDeviceHostnames(filter);

// Get specific device
Machine device = DeviceApi.getDevice("machine-123");

// Update device lifecycle
DeviceApi.archiveDevice(device);
DeviceApi.deleteDevice(device);

// Search with filters
Machine found = DeviceApi.searchDevice(filter, "server-name");

// Get available filters
DeviceFilters filters = DeviceApi.getDeviceFilters();
```
