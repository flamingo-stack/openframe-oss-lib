A test API client that provides methods for interacting with organization endpoints in the OpenFrame platform, supporting both GraphQL queries and REST operations for comprehensive organization management testing.

## Key Components

**Static Methods:**
- `getOrganizationNames()` - Retrieves all organization names via GraphQL
- `getOrganizationIds()` - Fetches all organization IDs via GraphQL  
- `retrieveOrganization(String id)` - Gets detailed organization data by ID
- `createOrganization(CreateOrganizationRequest request)` - Creates new organization via REST POST
- `updateOrganization(String id, CreateOrganizationRequest request)` - Updates existing organization via REST PUT
- `deleteOrganization(Organization organization)` - Removes organization via REST DELETE

**Key Features:**
- Combines GraphQL queries for data retrieval with REST endpoints for CRUD operations
- Uses `RequestSpecHelper` for consistent authorization across all requests
- Automatic JSON response mapping to `Organization` objects
- Built-in status code validation for REST operations

## Usage Example

```java
// Get all organization names for test data setup
List<String> orgNames = OrganizationApi.getOrganizationNames();

// Create a new organization for testing
CreateOrganizationRequest request = new CreateOrganizationRequest();
request.setName("Test Organization");
Organization newOrg = OrganizationApi.createOrganization(request);

// Retrieve detailed organization data
Organization fullOrg = OrganizationApi.retrieveOrganization(newOrg.getId());

// Update organization details
request.setName("Updated Test Organization");
Organization updatedOrg = OrganizationApi.updateOrganization(newOrg.getId(), request);

// Clean up test data
OrganizationApi.deleteOrganization(updatedOrg);
```
