Provides a test API client for managing organizations in the OpenFrame platform, supporting CRUD operations and status management via both REST and GraphQL endpoints.

## Key Components

| Method | Type | Description |
|--------|------|-------------|
| `listOrganizations()` | GraphQL | Fetches all organizations using cursor-based pagination (100 per page) |
| `getOrganizations(boolean isDefault)` | GraphQL | Filters organizations by their default status |
| `retrieveOrganizationByOrganizationId(String organizationId)` | GraphQL | Fetches a single organization by its ID |
| `createOrganization(CreateOrganizationRequest request)` | REST POST | Creates a new organization, expects HTTP 201 |
| `updateOrganization(String id, CreateOrganizationRequest request)` | REST PUT | Updates an existing organization, expects HTTP 200 |
| `archiveOrganization(Organization organization)` | REST PATCH | Archives an organization by setting status to `ARCHIVED`, expects HTTP 204 |

## Usage Example

```java
// List all default organizations
List<Organization> defaultOrgs = OrganizationApi.getOrganizations(true);

// Create a new organization
CreateOrganizationRequest request = new CreateOrganizationRequest("Acme Corp", ...);
Organization created = OrganizationApi.createOrganization(request);

// Retrieve by organization ID
Organization org = OrganizationApi.retrieveOrganizationByOrganizationId("org-uuid-123");

// Update an organization
Organization updated = OrganizationApi.updateOrganization(created.getId(), request);

// Archive an organization
OrganizationApi.archiveOrganization(org);
```

> All methods use `RequestSpecHelper.getAuthorizedSpec()` for authenticated requests. GraphQL calls are validated with the `graphqlSuccess()` response spec.
