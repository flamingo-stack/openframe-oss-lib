Provides REST-Assured-based API client methods for user registration against the OpenFrame SAS OAuth registration endpoint.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `REGISTER` | `String` | Static endpoint URL built from `EnvironmentConfig.getRegistrationUrl()` + `sas/oauth/register` |
| `registerUser()` | `static method` | Posts a registration request and asserts HTTP `200`, returning a `UserRegistrationResponse` |
| `attemptRegistration()` | `static method` | Posts a registration request and asserts HTTP `400`, returning an `ErrorResponse` (for negative test scenarios) |

## Usage Example

```java
// Successful registration
UserRegistrationRequest newUser = new UserRegistrationRequest("john@example.com", "SecureP@ss1");
UserRegistrationResponse registered = RegistrationApi.registerUser(newUser);

// Negative test — duplicate or invalid registration
UserRegistrationRequest badUser = new UserRegistrationRequest("", "");
ErrorResponse error = RegistrationApi.attemptRegistration(badUser);
assertThat(error.getMessage()).isNotEmpty();
```

> **Note:** Both methods use `relaxedHTTPSValidation()` to bypass SSL certificate checks in test environments. `registerUser` additionally enables `log().ifValidationFails()` for easier debugging on assertion failures.
