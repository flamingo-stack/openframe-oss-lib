REST API wrapper for user registration functionality in automated testing, providing methods to register users and handle registration errors using RestAssured.

## Key Components

- **`registerUser(UserRegistrationRequest)`** - Successfully registers a new user and returns registration response with 200 status
- **`attemptRegistration(UserRegistrationRequest)`** - Attempts registration expecting failure and returns error response with 400 status  
- **`REGISTER`** - Constant defining the registration endpoint path (`sas/oauth/register`)

## Usage Example

```java
// Successful user registration
UserRegistrationRequest validUser = new UserRegistrationRequest("john@example.com", "password123");
UserRegistrationResponse response = RegistrationApi.registerUser(validUser);

// Test invalid registration attempt
UserRegistrationRequest invalidUser = new UserRegistrationRequest("", "weak");
ErrorResponse errorResponse = RegistrationApi.attemptRegistration(invalidUser);

// Use in test assertions
assertEquals("Registration successful", response.getMessage());
assertEquals("VALIDATION_ERROR", errorResponse.getErrorCode());
```

The API uses unauthorized request specifications and automatically handles JSON content type serialization. This design supports both positive and negative test scenarios for user registration workflows.
