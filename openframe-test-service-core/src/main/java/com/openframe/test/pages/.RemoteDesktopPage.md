Page Object Model class for the Remote Desktop session page in OpenFrame's Playwright-based UI test suite. It encapsulates selectors and interactions for noVNC/MeshCentral canvas-rendered desktop sessions.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `REMOTE_DESKTOP_CANVAS` | Selector | Targets the noVNC `<canvas>` element via `tabindex='0'` |
| `DEVICE_NAME_HEADING` | Selector | `main h1` — device name in the session header |
| `DEVICE_SUBTITLE` | Selector | `main h1 + p` — subtitle (e.g. `"Desktop • Default"`) |
| `ACTIONS_BUTTON` | Selector | Radix UI dropdown trigger with `aria-haspopup="menu"` |
| `SETTINGS_BUTTON` | Selector | Plain button without `aria-haspopup` |
| `waitForCanvasVisible(int)` | Method | Waits until canvas is in DOM and visible; returns `this` for chaining |
| `waitForDesktop()` | Method | Polls via `waitForCondition` until canvas pixels are non-blank |
| `canvasIsNotBlank()` | Method | Evaluates center-pixel RGB via `getImageData` to confirm live stream |
| `getCanvasWidth()` / `getCanvasHeight()` | Methods | Read DOM `width`/`height` attributes; return `0` if null |
| `goBackToDevice()` | Method | Clicks "Back to Device" and returns a `DeviceDetailsPage` instance |

## Usage Example

```java
// Navigate to a remote desktop session
RemoteDesktopPage rdp = new RemoteDesktopPage(page);

// Wait for the canvas to appear and stream to start
rdp.waitForCanvasVisible(15_000)
   .waitForDesktop();

// Assert session metadata
assertThat(rdp.getDeviceName()).isEqualTo("WORKSTATION-01");
assertThat(rdp.getDeviceSubtitle()).contains("Desktop");

// Assert canvas dimensions match expected noVNC defaults
assertThat(rdp.getCanvasWidth()).isEqualTo(1024);
assertThat(rdp.getCanvasHeight()).isEqualTo(768);

// Navigate back to device details
DeviceDetailsPage details = rdp.goBackToDevice();
```

> **Note:** `canvasIsNotBlank()` samples the center pixel of the canvas via JavaScript `getImageData`. A fully black frame returns `false`, so `waitForDesktop()` will keep polling until actual stream content is rendered.
