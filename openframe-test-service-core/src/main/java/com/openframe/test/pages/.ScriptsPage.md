Page Object Model class for the OpenFrame Scripts section, providing Playwright-based locators and actions for both the **Scripts List** and **Scripts Schedules** tabs at `/scripts/`.

## Key Components

### Constants
- `URL` — Scripts List tab endpoint
- `URL_SCHEDULES` — Scripts Schedules tab endpoint (`?tab=schedules`)

### Navigation Methods
| Method | Description |
|---|---|
| `navigate()` | Loads the Scripts List tab and waits for network idle |
| `navigateToSchedules()` | Loads the Scripts Schedules tab and waits for network idle |

### Tab Bar
| Method | Description |
|---|---|
| `clickScriptsListTab()` | Clicks the Scripts List tab, waits for URL without `tab=` |
| `clickSchedulesTab()` | Clicks the Schedules tab, waits for `tab=schedules` in URL |
| `getActiveTabLabel()` | Returns the visible label of the currently active tab |

### Scripts List
- `scriptRows()` — All rendered script row cards
- `scriptRowByName(name)` — Row scoped by visible script name
- `rowMenuButton(scriptName)` — ⋯ context menu trigger for a named row
- `rowRunButton(scriptName)` — ▷ Run button for a named row
- `ctxEditScript()` / `ctxScriptDetails()` — Context menu item locators

### Filter Panel
Locators for Shell Type (`cmd`, `powershell`, `python`, `shell`), OS (`macOS`, `Linux`, `Windows`), Added By (`Tactical`), and Category (`Custom`, `DEPRECATED`, `OpenFrame`) checkboxes, plus `filterApplyButton()` / `filterResetButton()`.

### Scripts Schedules
- `scheduleRows()` — All rendered schedule rows
- `scheduleRowByName(name)` — Row scoped by visible schedule name
- `scheduleEditButton(scheduleName)` — Edit button for a named schedule row

## Usage Example

```java
ScriptsPage scripts = new ScriptsPage(page);

// Navigate and verify active tab
scripts.navigate();
assertThat(scripts.pageHeading()).isVisible();
assertEquals("Scripts List", scripts.getActiveTabLabel());

// Search and check results
scripts.searchInput().fill("Restart Service");
assertThat(scripts.resultsCount()).isVisible();

// Open context menu for a specific script
scripts.rowMenuButton("Restart Service").click();
scripts.ctxEditScript().click();

// Apply a Shell Type filter
scripts.openFiltersButton().click();
scripts.filterPowershellCheckbox().check();
scripts.filterApplyButton().click();

// Switch to Schedules tab
scripts.clickSchedulesTab();
scripts.scheduleEditButton("Nightly Backup").click();
```
