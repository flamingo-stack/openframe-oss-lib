Page Object Model class for the Device Details page (`/devices/details/{id}/`), providing structured access to device information, status, metadata fields, action controls, tabs, and agent cards via Playwright locators.

## Key Components

| Method | Description |
|--------|-------------|
| `getDeviceName()` | Returns the device name from the `h1` heading |
| `getDeviceStatus()` | Returns connectivity status badge text (`ONLINE`, `OFFLINE`, `ARCHIVED`) |
| `getLastUpdatedText()` | Returns the last-updated timestamp; returns empty string if absent |
| `getInfoField(String label)` | Looks up a metadata field value by label (e.g. `"Hostname"`, `"Serial Number"`) |
| `clickBackToDevices()` | Clicks the Back navigation button |
| `openRemoteDesktop()` | Navigates to the remote-desktop sub-route and returns a `RemoteDesktopPage` |
| `openFileManager()` | Opens the File Manager via the more-actions menu and returns a `FileManagerPage` |
| `openRemoteShellCmd()` | Opens the Remote Shell dropdown and selects `CMD` |
| `openRemoteShellPowerShell()` | Opens the Remote Shell dropdown, selects `PowerShell`, and returns a `RemoteShellPage` |
| `clickRunScript()` | Opens more-actions and clicks `Run Script` |
| `clickArchiveDevice()` | Opens more-actions and clicks `Archive Device` |
| `clickDeleteDevice()` | Opens more-actions and clicks `Delete Device` |
| `clickTab(String tabName)` | Clicks a named tab (`Hardware`, `Network`, `Security`, etc.) |
| `isTabActive(String tabName)` | Returns `true` if the specified tab is currently active |
| `getAgentStatus(String agentName)` | Returns the status of a named agent card (`ONLINE`/`OFFLINE`) |
| `isLoaded()` | Returns `true` when both device name and status are non-empty |

## Usage Example

```java
// Navigate to a device detail page
page.navigate("/devices/details/abc123/");
DeviceDetailsPage detailsPage = new DeviceDetailsPage(page);

// Assert the page has loaded
assertTrue(detailsPage.isLoaded());

// Read header info
String name   = detailsPage.getDeviceName();   // e.g. "VM115982"
String status = detailsPage.getDeviceStatus(); // e.g. "ONLINE"

// Read a metadata field from the info card
String hostname = detailsPage.getInfoField("Hostname"); // e.g. "vm115982"

// Navigate to tabs and check agent status
detailsPage.clickTab("Agents");
String agentStatus = detailsPage.getAgentStatus("MeshCentral"); // "ONLINE"

// Open File Manager via the more-actions menu
FileManagerPage fm = detailsPage.openFileManager();

// Open a PowerShell remote shell session
RemoteShellPage shell = detailsPage.openRemoteShellPowerShell();
```
