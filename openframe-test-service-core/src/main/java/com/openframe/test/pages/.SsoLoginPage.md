Page Object for the OpenFrame SSO login form at `https://openframe.build/sas/login`, providing locators and fluent actions for Playwright-based end-to-end tests.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `URL` | Constant | Static URL for the SSO login page |
| `formHeading()` | Locator | The "OpenFrame Single Sign-On" H2 heading |
| `emailInput()` | Locator | `#username` field |
| `passwordInput()` | Locator | `#password` field |
| `submitButton()` | Locator | `#submitBtn` submit button |
| `forgotPasswordButton()` | Locator | `#forgotPasswordBtn` button |
| `togglePasswordButton()` | Locator | `#passwordToggle` eye icon |
| `isLoginFailed()` | Query | Returns `true` if still on `/sas/login` after submission |
| `getEmailValue()` | Query | Reads retained email field value after failed login |
| `login(email, password)` | Action | Fluent shortcut — fills both fields and submits |
| `clickSubmit()` | Action | Clicks submit, waits for URL navigation, returns `DashboardPage` |

## Usage Example

```java
// Navigate to SSO page
page.navigate(SsoLoginPage.URL);
SsoLoginPage ssoPage = new SsoLoginPage(page);

// Happy path — successful login
DashboardPage dashboard = ssoPage.login("user@example.com", "s3cur3Pass!");

// Unhappy path — check for failure and inspect retained email value
DashboardPage result = ssoPage.login("bad@example.com", "wrongpass");
if (ssoPage.isLoginFailed()) {
    String retained = ssoPage.getEmailValue(); // "bad@example.com"
    System.out.println("Login failed. Email retained: " + retained);
}

// Step-by-step alternative
ssoPage.enterEmail("user@example.com")
       .enterPassword("s3cur3Pass!")
       .clickSubmit();
```

## Behavior Notes

- **Success**: POST redirects to `test-qa.openframe.build/dashboard/` → `DashboardPage` is returned
- **Failure**: Browser stays at `/sas/login`; field values are retained but no inline error is rendered — use `isLoginFailed()` to detect this state
- `clickSubmit()` waits up to **15 seconds** for URL navigation before timing out
