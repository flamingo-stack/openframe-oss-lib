Page Object Model implementation for the Run Script page (`/scripts/details/{id}/run/`), providing Playwright-based interactions for configuring and executing scripts against target devices.

## Key Components

### Locators
| Field | Description |
|-------|-------------|
| `backToScriptDetailsButton` | Navigation button returning to script details |
| `runScriptButton` | Submit button (enabled only when ≥1 device selected) |
| `timeoutInput` | Numeric input for execution timeout (default: 90s) |
| `addScriptArgumentButton` | Appends a new script argument row |
| `addEnvironmentVarButton` | Appends a new environment variable row |
| `deviceSearchInput` | Filters the device picker grid by name |
| `organizationFilterInput` | Combobox for filtering devices by organization |
| `deviceOptions` | All selectable device cards (`[role="option"]`) |

### Methods
| Method | Description |
|--------|-------------|
| `getScriptMetadataField(labelText)` | Reads metadata values (Shell Type, Category, etc.) via XPath sibling traversal |
| `setScriptArgument(index, key, value)` | Sets key/value for a script argument row by 0-based index |
| `setEnvironmentVar(index, key, value)` | Sets key/value for an env var row, offset-aware of script arg count |
| `deleteEnvironmentVar(index)` | Deletes env var using shared delete buttons, offset by arg count |
| `toggleDeviceByName(name)` | Clicks a device card to select/deselect it |
| `isDeviceSelected(name)` | Checks `aria-selected` attribute on a device card |
| `getSelectedDeviceNames()` | Returns names of all currently selected devices |

## Usage Example

```java
RunScriptPage runPage = new RunScriptPage(page);

// Configure timeout
runPage.setTimeout(120);

// Set script arguments
runPage.setScriptArgument(0, "--output", "/tmp/result.txt");

// Add and configure an environment variable
runPage.addEnvironmentVar();
runPage.setEnvironmentVar(0, "LOG_LEVEL", "DEBUG");

// Filter and select a target device
runPage.searchForDevice("WIN-SERVER-01");
runPage.toggleDeviceByName("WIN-SERVER-01");

// Verify selection before running
assert runPage.isDeviceSelected("WIN-SERVER-01");
assert runPage.getSelectedDeviceNames().contains("WIN-SERVER-01");
```

> **Note:** Environment variable index offsets are automatically calculated relative to script argument count — no manual offset management needed when calling `setEnvironmentVar` or `deleteEnvironmentVar`.
