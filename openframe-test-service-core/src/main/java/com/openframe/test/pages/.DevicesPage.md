Page Object Model for the Devices list page (`/devices/`), providing locator methods and actions for all interactive elements including device rows, search, view toggles, filter panel, and context menus using Playwright.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `URL_FRAGMENT` | Constant | URL path `/devices/` |
| `DevicesPage(Page)` | Constructor | Initializes POM with a Playwright `Page` instance |
| `pageHeading()` | Locator | Main `<h1>` heading |
| `tableViewButton()` / `gridViewButton()` | Locators | View toggle buttons (table active by default) |
| `addDeviceButton()` | Locator | "Add Device" submit button (desktop variant) |
| `searchInput()` | Locator | Search bar with placeholder "Search for Devices" |
| `openFiltersButton()` | Locator | "Filter Tags" button that opens the Sort & Filter modal |
| `resultsCount()` | Locator | "N results" span in the column header row |
| `deviceRows()` | Locator | All rendered device row cards |
| `deviceRowByName(String)` | Locator | First device row whose visible text contains the given name |
| `rowMenuButtonByName(String)` | Locator | Three-dot "More actions" button inside a named device row |
| `filterPanel()` | Locator | `role="dialog"` Sort and Filter modal |
| `filterResetButton()` / `filterApplyButton()` | Locators | Reset/Apply actions inside the filter panel |
| `filterTagKeyCheckbox(String)` | Locator | Checkbox for a specific tag key in the "Tag Keys" section |
| `activeFilterChip(String)` | Locator | Active filter chip by `"key:value"` text (e.g. `"purpose:auto_test"`) |

## Usage Example

```java
// Initialize page object
DevicesPage devicesPage = new DevicesPage(page);
page.navigate(baseUrl + DevicesPage.URL_FRAGMENT);
page.waitForLoadState(LoadState.NETWORKIDLE);

// Assert page loaded
assertThat(devicesPage.pageHeading()).hasText("Devices");

// Search for a specific device
devicesPage.searchInput().fill("workstation-01");

// Check result count and click into a row
assertThat(devicesPage.resultsCount()).hasText("1 results");
devicesPage.deviceRowByName("workstation-01").click();

// Open context menu for a device
devicesPage.rowMenuButtonByName("workstation-01").click();

// Apply a tag filter
devicesPage.openFiltersButton().click();
devicesPage.filterTagKeyCheckbox("purpose").click();
devicesPage.filterApplyButton().click();
assertThat(devicesPage.activeFilterChip("purpose:auto_test")).isVisible();
```
