Page Object Model implementation for the **New Device** page (`/devices/new/`) in the OpenFrame Playwright test suite. Encapsulates all UI interactions for device installation command generation, organization/platform selection, and antivirus exclusion path management.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `getSelectedOrganization()` | Method | Returns the currently selected org name from the selector button |
| `selectOrganization(String)` | Method | Opens dropdown, searches, and clicks the matching org option |
| `searchOrganizations(String)` | Method | Returns all org names matching a search query |
| `selectPlatform(String)` | Method | Clicks a platform option (`"Windows"` or `"macOS"`) |
| `getSelectedPlatform()` | Method | Returns the active platform via JS evaluation of `bg-ods-accent` class |
| `isPlatformAvailable(String)` | Method | Checks if a platform is enabled (not `opacity-50`/`cursor-not-allowed`) |
| `getDeviceAddCommand()` | Method | Returns the full generated install command string |
| `clickCopyCommand()` | Method | Clicks "Copy Command" button |
| `clickRunOnCurrentMachine()` | Method | Clicks "Run on Current Machine" button |
| `addArgument(String)` | Method | Types an argument into the tag input and presses Enter |
| `getExclusionPaths()` | Method | Returns all antivirus exclusion paths as a `List<String>` |
| `copyExclusionPath(int)` | Method | Clicks the copy button for a path at a given 0-based index |
| `getExclusionPathCount()` | Method | Returns the total number of exclusion paths displayed |

## Usage Example

```java
// Navigate to the New Device page via the Devices list
DevicesPage devicesPage = new DevicesPage(page);
devicesPage.clickAddDevice();

NewDevicePage newDevicePage = new NewDevicePage(page);

// Select organization and platform
newDevicePage.selectOrganization("Acme Corp");
newDevicePage.selectPlatform("Windows");

// Retrieve and copy the generated install command
String command = newDevicePage.getDeviceAddCommand();
newDevicePage.clickCopyCommand();

// Add optional CLI arguments
newDevicePage.addArgument("--silent");
newDevicePage.addArgument("--no-restart");

// Verify antivirus exclusion paths
int pathCount = newDevicePage.getExclusionPathCount();
String firstPath = newDevicePage.getExclusionPath(0);
// e.g. "C:\Program Files\OpenFrame"

// Navigate back
newDevicePage.clickBackToDevices();
```

> **Note:** Linux is shown as a platform option but is currently in "coming soon" state. Use `isPlatformAvailable("Linux")` to assert its disabled state in tests.
