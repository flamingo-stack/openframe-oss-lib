Page Object Model implementation for the File Manager page in OpenFrame's Playwright test suite, providing programmatic access to file browsing, navigation, and file operation UI components.

## Key Components

| Method / Field | Description |
|---|---|
| `getPageHeading()` | Returns the "File Manager" heading text |
| `getDeviceName()` | Returns the device name subtitle |
| `clickBackToDevice()` | Navigates back to the device details page |
| `getBreadcrumbSegments()` | Returns ordered list of breadcrumb path labels |
| `getCurrentFolder()` | Returns the last breadcrumb segment (current directory) |
| `clickBreadcrumb(String)` | Navigates to a specific breadcrumb level by label |
| `search(String)` / `clearSearch()` | Filters the file table via the search input |
| `clickNewFolder()` / `clickCopy()` / `clickCut()` / `clickPaste()` | Toolbar action methods |
| `uploadFile(String)` | Triggers upload via hidden `<input type="file">` without native dialog |
| `getRowCount()` / `getAllRowNames()` | Inspects visible table rows |
| `getRowName(int)` / `getRowSize(int)` / `getRowEditedDate(int)` | Per-row column accessors (0-based index) |
| `isRowChecked(int)` / `clickRowCheckbox(int)` | Row selection state and toggle |
| `enterFolder(int)` / `enterFolderByName(String)` | Navigates into a folder by index or name |
| `openRowContextMenu(int)` / `openRowContextMenuByName(String)` | Opens the ⋯ context menu (Copy / Rename / Cut / Delete) |
| `clickContextMenuItem(String)` | Clicks a context menu item by label after menu is open |

## Usage Example

```java
// Navigate to File Manager from DeviceDetailsPage
FileManagerPage fileManager = deviceDetails.clickManageFiles();

// Assert page loaded correctly
assert fileManager.getPageHeading().equals("File Manager");
assert fileManager.getDeviceName().equals("vm114267");

// Navigate into a folder using breadcrumbs
fileManager.enterFolderByName("Program Files");
assert fileManager.getCurrentFolder().equals("Program Files");

// Search and inspect results
fileManager.search("config");
int results = fileManager.getRowCount();
String firstName = fileManager.getRowName(0);

// Select a file and copy it via context menu
fileManager.openRowContextMenuByName("config.json");
fileManager.clickContextMenuItem("Copy");

// Upload a local file (bypasses native file picker)
fileManager.uploadFile("/tmp/test-upload.txt");

// Navigate back via breadcrumb
fileManager.clickBreadcrumb("Root");
```

> **Note:** URL pattern for this page is `/devices/details/{id}/file-manager/`. The table is rendered as a custom div grid (no `<table>` element), so row selectors target `.bg-ods-bg.rounded-lg > div:nth-child(2) > div` directly.
