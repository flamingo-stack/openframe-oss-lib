Page Object Model implementation for the OpenFrame Monitoring page (`/monitoring/`), providing Playwright-based locators and fluent API methods for both the Policies and Queries tabs.

## Key Components

### URLs
| Constant | Path |
|---|---|
| `BASE_URL` | `/monitoring/` |
| `POLICIES_URL` | `/monitoring/` |
| `QUERIES_URL` | `/monitoring/?tab=queries` |

### Locator Groups
- **Tab navigation** — `tabPolicies`, `tabQueries`
- **Policies header** — heading, add button (text + icon variants), search input
- **Metric cards** — Total Policies, Compliance Rate (ratio + percent), Failed Policies, Updated timestamp
- **Policy rows** — `policyRows` targets `div[class*='h-[68px]']` inner flex rows; columns mapped as: `[0]` Name · `[1]` Severity · `[2]` Platform · `[3]` Status · `[4]` Actions · `[5]` Link
- **Queries tab** — heading, add button (text + icon variants), search input, empty state

### Key Methods

| Method | Returns | Description |
|---|---|---|
| `navigate()` | `MonitoringPage` | Navigate to base monitoring URL |
| `navigateToQueriesTab()` | `MonitoringPage` | Navigate directly via query param |
| `clickPoliciesTab()` / `clickQueriesTab()` | `MonitoringPage` | Tab switching via click |
| `searchPolicies(query)` | `MonitoringPage` | Fill the policies search input |
| `getPolicyStatusByName(name)` | `String` | Lookup status badge by policy name |
| `clickMoreActionsForPolicy(rowIndex)` | `MonitoringPage` | Open `…` menu for a row |
| `getTotalPoliciesCount()` | `int` | Parsed integer from Total Policies card |
| `getComplianceRateRatio()` | `String` | e.g. `"1/1"` |
| `getComplianceRatePercent()` | `String` | e.g. `"(100%)"` |
| `getFailedPoliciesCount()` | `int` | Parsed integer from Failed Policies card |
| `getUpdatedTimestamp()` | `String` | e.g. `"3 minutes ago"` |

All action methods return `this` for fluent chaining.

## Usage Example

```java
MonitoringPage monitoring = new MonitoringPage(page);

// Navigate and verify policies tab
monitoring.navigate();
System.out.println(monitoring.getPoliciesHeadingText()); // "Policies"
System.out.println(monitoring.getTotalPoliciesCount());  // e.g. 3
System.out.println(monitoring.getComplianceRateRatio()); // e.g. "2/3"

// Search and inspect a specific policy
monitoring.searchPolicies("Windows Baseline")
          .clickMoreActionsForPolicy(0);

// Check status by name
String status = monitoring.getPolicyStatusByName("test policy");
// Returns "Compliant" or "Non-Compliant"

// Switch to Queries tab
monitoring.clickQueriesTab()
          .searchQueries("disk usage");
```

> **Note:** `getPolicyStatusByName` uses `:scope > div` (direct children only) to reliably target column index `[3]` (Status), avoiding off-by-one issues caused by nested wrapper divs.
