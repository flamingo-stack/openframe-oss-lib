Page Object Model representing the left navigation sidebar present on every authenticated OpenFrame page, providing locators, state queries, navigation actions, and collapse/expand controls via Playwright.

## Key Components

### Locators
| Method | Returns |
|--------|---------|
| `sidebar()` | Root `<aside>` element |
| `dashboardNavItem()` / `devicesNavItem()` / etc. | Primary nav button locators |
| `settingsNavItem()` | Secondary nav button locator |
| `activeNavItem()` | Currently active nav button (`aria-current="page"`) |
| `hideMenuButton()` | "Hide Menu" collapse toggle |

### State Queries
| Method | Description |
|--------|-------------|
| `isExpanded()` | `true` when sidebar shows labels (224 px wide) |
| `isCollapsed()` | `true` when sidebar is icon-only (56 px wide) |
| `getActiveNavItemLabel()` | Returns active item's `aria-label` (e.g. `"Devices"`) |
| `isNavItemActive(Locator)` | Checks `aria-current="page"` on a given nav button |

### Navigation Actions
Each `goTo*()` method clicks the nav item and waits for the URL to settle. `goToDevices()` and `goToMonitoring()` additionally wait for page load conditions and return typed page objects (`DevicesPage`, `MonitoringPage`).

### Collapse/Expand
`collapse()` clicks "Hide Menu" and waits for the button to disappear; no-ops if already collapsed.

## Usage Example

```java
NavigationSidebar sidebar = new NavigationSidebar(page);

// Check sidebar state
boolean expanded = sidebar.isExpanded();

// Navigate to Devices and get typed page object
DevicesPage devicesPage = sidebar.goToDevices();

// Navigate to Monitoring
MonitoringPage monitoringPage = sidebar.goToMonitoring();

// Verify active item
String activeLabel = sidebar.getActiveNavItemLabel(); // "Devices"
boolean isActive = sidebar.isNavItemActive(sidebar.devicesNavItem()); // true

// Collapse sidebar
sidebar.collapse();
```
