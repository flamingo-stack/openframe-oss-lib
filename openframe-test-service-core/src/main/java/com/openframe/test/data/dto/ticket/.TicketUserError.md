A simple DTO representing a user-facing validation or field-level error associated with a ticket operation.

## Key Components

| Element | Description |
|--------|-------------|
| `field` | The name of the field that triggered the error |
| `message` | A human-readable description of the error |
| `@JsonIgnoreProperties` | Safely ignores unknown JSON fields during deserialization |
| `@Builder` | Enables fluent builder pattern for object construction |

## Usage Example

```java
// Build a TicketUserError using the builder pattern
TicketUserError error = TicketUserError.builder()
    .field("title")
    .message("Title must not be blank")
    .build();

// Or deserialize from a JSON response
// { "field": "priority", "message": "Invalid priority value" }
TicketUserError deserialized = objectMapper.readValue(json, TicketUserError.class);

System.out.println(error.getField());   // "title"
System.out.println(error.getMessage()); // "Title must not be blank"
```
