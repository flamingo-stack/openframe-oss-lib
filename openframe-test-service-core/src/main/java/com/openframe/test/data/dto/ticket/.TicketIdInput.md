A simple Data Transfer Object (DTO) used to encapsulate a ticket identifier for input operations within the OpenFrame test data layer.

## Key Components

| Element | Type | Description |
|---|---|---|
| `id` | `String` | The unique identifier of a ticket |
| `@Data` | Lombok | Generates getters, setters, `equals`, `hashCode`, and `toString` |
| `@Builder` | Lombok | Enables the builder pattern for object construction |
| `@NoArgsConstructor` / `@AllArgsConstructor` | Lombok | Generates both default and full constructors |
| `@JsonInclude(NON_NULL)` | Jackson | Excludes `null` fields from JSON serialization |

## Usage Example

```java
// Using the builder pattern
TicketIdInput input = TicketIdInput.builder()
    .id("TICKET-1234")
    .build();

// Using the all-args constructor
TicketIdInput input = new TicketIdInput("TICKET-1234");

// Using the no-args constructor with setter
TicketIdInput input = new TicketIdInput();
input.setId("TICKET-1234");

// Passing to a service or API call
ticketService.getTicketById(input);
```

> **Note:** When serialized to JSON with a `null` id, the `id` field will be omitted entirely due to `@JsonInclude(NON_NULL)`. This is useful when constructing partial payloads in test scenarios.
