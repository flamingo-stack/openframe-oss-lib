A data transfer object representing a filtering option for device tags, containing the filter value, display label, and occurrence count.

## Key Components

- **value** - The actual filter value used in queries
- **label** - Human-readable display text for the filter option
- **count** - Number of items matching this filter option

## Usage Example

```java
// Creating a tag filter option
TagFilterOption option = TagFilterOption.builder()
    .value("production")
    .label("Production Environment")
    .count(42)
    .build();

// Using in a filter list
List<TagFilterOption> filterOptions = Arrays.asList(
    TagFilterOption.builder()
        .value("prod")
        .label("Production")
        .count(25)
        .build(),
    TagFilterOption.builder()
        .value("staging")
        .label("Staging")
        .count(8)
        .build()
);

// Accessing properties
String filterValue = option.getValue();      // "production"
String displayText = option.getLabel();     // "Production Environment"
Integer itemCount = option.getCount();      // 42
```

This DTO is typically used in filtering interfaces where users can see available tag options along with how many devices match each filter criteria.
