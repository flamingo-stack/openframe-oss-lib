A simple DTO representing a filter option for organizations, containing an identifier and display name. Used for populating organization-based filter dropdowns or selection lists in log/reporting contexts.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the organization |
| `name` | `String` | Display name of the organization |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently ignores unrecognized JSON fields during deserialization

## Usage Example

```java
// Builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// All-args constructor
OrganizationFilterOption option = new OrganizationFilterOption("org-123", "Acme Corporation");

// Deserialization (unknown fields ignored)
String json = "{\"id\":\"org-123\",\"name\":\"Acme\",\"extraField\":\"ignored\"}";
OrganizationFilterOption option = objectMapper.readValue(json, OrganizationFilterOption.class);
```
