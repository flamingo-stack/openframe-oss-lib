A simple data transfer object (DTO) representing a filter option for organizations in the test data logging system, providing id and name fields for organization selection filters.

## Key Components

- **OrganizationFilterOption**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **id**: String field representing the unique identifier of the organization
- **name**: String field representing the display name of the organization

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using all-args constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "Tech Solutions Inc");

// Access fields using generated getters
String orgId = option.getId();
String orgName = option.getName();

// Modify using generated setters
option.setName("Updated Organization Name");
```

This DTO is typically used in filtering interfaces where users can select organizations from dropdown lists or search filters, providing both a technical identifier and human-readable display name for each organization option.
