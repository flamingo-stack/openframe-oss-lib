Data Transfer Object (DTO) for encapsulating all parameters required to execute a script via the OpenFrame script execution API.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `mode` | `String` | Execution mode (e.g., `agents`, `site`, `client`) |
| `target` | `String` | Target identifier for the script run |
| `monType` / `osType` | `String` | Monitor type and OS type filters |
| `cmd` / `shell` | `String` | Command string and shell interpreter |
| `customShell` | `String` | Custom shell path (`custom_shell`) |
| `customField` | `String` | Custom field reference (`custom_field`) |
| `collectorAllOutput` | `Boolean` | Capture full output (`collector_all_output`) |
| `saveToAgentNote` | `Boolean` | Persist output as agent note (`save_to_agent_note`) |
| `patchMode` | `String` | Patch execution mode |
| `offlineAgents` | `Boolean` | Include offline agents in the run |
| `client` / `site` | `String` | Scope the run to a specific client or site |
| `agents` | `List<String>` | Explicit list of agent IDs to target |
| `script` | `Integer` | Script ID to execute |
| `timeout` | `Integer` | Execution timeout in seconds |
| `args` | `List<String>` | Positional arguments passed to the script |
| `envVars` | `List<String>` | Environment variables (`env_vars`) |
| `runAsUser` | `Boolean` | Execute in user context (`run_as_user`) |

## Usage Example

```java
RunScriptRequest request = RunScriptRequest.builder()
    .mode("agents")
    .script(42)
    .agents(List.of("agent-001", "agent-002"))
    .shell("powershell")
    .args(List.of("-Verbose"))
    .envVars(List.of("LOG_LEVEL=debug"))
    .timeout(120)
    .runAsUser(false)
    .saveToAgentNote(true)
    .collectorAllOutput(true)
    .build();
```

> **Note:** Unknown JSON properties are silently ignored via `@JsonIgnoreProperties(ignoreUnknown = true)`, making this DTO forward-compatible with newer API versions.
