A simple Data Transfer Object (DTO) representing an image attached to a ticket, encapsulating a single image URL field.

## Key Components

| Element | Description |
|---|---|
| `imageUrl` | String field holding the URL pointing to the ticket image |
| `@Data` | Lombok annotation generating getters, setters, `equals`, `hashCode`, and `toString` |
| `@Builder` | Enables the builder pattern for object construction |
| `@NoArgsConstructor` / `@AllArgsConstructor` | Generates default and full constructors |
| `@JsonIgnoreProperties(ignoreUnknown = true)` | Silently ignores unrecognized JSON fields during deserialization |

## Usage Example

```java
// Builder pattern
TicketImage image = TicketImage.builder()
    .imageUrl("https://cdn.example.com/tickets/screenshot.png")
    .build();

// No-args constructor with setter
TicketImage image = new TicketImage();
image.setImageUrl("https://cdn.example.com/tickets/screenshot.png");

// All-args constructor
TicketImage image = new TicketImage("https://cdn.example.com/tickets/screenshot.png");

// Access the URL
String url = image.getImageUrl();
```
