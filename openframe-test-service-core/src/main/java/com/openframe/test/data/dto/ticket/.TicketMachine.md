A lightweight DTO representing the association between a support ticket and a machine/device within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the ticket-machine record |
| `machineId` | `String` | Identifier of the associated machine/device |
| `hostname` | `String` | Hostname of the associated machine |
| `organizationId` | `String` | Identifier of the owning organization |

**Lombok Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Builder pattern
TicketMachine machine = TicketMachine.builder()
    .id("tm-001")
    .machineId("mch-abc123")
    .hostname("workstation-42")
    .organizationId("org-xyz")
    .build();

// Deserialization (unknown fields are safely ignored)
String json = """
    {
      "id": "tm-001",
      "machineId": "mch-abc123",
      "hostname": "workstation-42",
      "organizationId": "org-xyz",
      "extraField": "ignored"
    }
    """;
TicketMachine machine = objectMapper.readValue(json, TicketMachine.class);
```
