A Data Transfer Object (DTO) representing the request payload for creating a new policy in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | The name of the policy to be created |
| `description` | `String` | A human-readable description of the policy |
| `query` | `String` | The query expression defining the policy logic |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for fluent object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Using the Builder pattern
CreatePolicyRequest request = CreatePolicyRequest.builder()
    .name("Block Unauthorized Access")
    .description("Denies requests from unrecognized IP ranges")
    .query("source.ip NOT IN trusted_ranges")
    .build();

// Using the All-Args Constructor
CreatePolicyRequest request = new CreatePolicyRequest(
    "Block Unauthorized Access",
    "Denies requests from unrecognized IP ranges",
    "source.ip NOT IN trusted_ranges"
);
```
