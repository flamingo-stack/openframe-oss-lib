A data transfer object (DTO) representing a file attachment associated with a support ticket in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the attachment |
| `ticketId` | `String` | Reference to the parent ticket |
| `fileName` | `String` | Original name of the uploaded file |
| `contentType` | `String` | MIME type of the file |
| `fileSize` | `Long` | Size of the file in bytes |
| `uploadedAt` | `Instant` | Timestamp of when the file was uploaded |
| `uploadedBy` | `String` | Identifier of the user who uploaded the file |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the fluent builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Builder pattern
TicketAttachment attachment = TicketAttachment.builder()
    .id("att-001")
    .ticketId("tkt-123")
    .fileName("screenshot.png")
    .contentType("image/png")
    .fileSize(204800L)
    .uploadedAt(Instant.now())
    .uploadedBy("user-456")
    .build();

// Access fields
String file = attachment.getFileName();
Long size = attachment.getFileSize();
```
