A DTO (Data Transfer Object) representing available filter options for device queries, containing categorized lists of filterable attributes and a count of matching results.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `statuses` | `List<DeviceFilterOption>` | Available device status filter options |
| `deviceTypes` | `List<DeviceFilterOption>` | Available device type filter options |
| `osTypes` | `List<DeviceFilterOption>` | Available OS type filter options |
| `organizationIds` | `List<DeviceFilterOption>` | Available organization filter options |
| `tags` | `List<TagFilterOption>` | Available tag filter options |
| `filteredCount` | `Integer` | Total count of devices matching applied filters |

**Lombok Annotations:**
- `@Data` — generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — enables builder pattern instantiation
- `@NoArgsConstructor` / `@AllArgsConstructor` — generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — safely ignores unrecognized JSON fields during deserialization

## Usage Example

```java
// Builder pattern
DeviceFilters filters = DeviceFilters.builder()
    .statuses(List.of(new DeviceFilterOption("online", "Online")))
    .deviceTypes(List.of(new DeviceFilterOption("workstation", "Workstation")))
    .osTypes(List.of(new DeviceFilterOption("windows", "Windows")))
    .organizationIds(List.of(new DeviceFilterOption("org-123", "Acme Corp")))
    .tags(List.of(new TagFilterOption("critical", "Critical")))
    .filteredCount(42)
    .build();

// Access filter options
List<DeviceFilterOption> availableStatuses = filters.getStatuses();
Integer matchCount = filters.getFilteredCount();
```
