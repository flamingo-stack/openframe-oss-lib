Data Transfer Object (DTO) representing a script entity used in test data operations, mapping JSON API responses to a strongly-typed Java object.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `Integer` | Unique script identifier |
| `name` | `String` | Script display name |
| `description` | `String` | Script description |
| `shell` | `String` | Shell interpreter (e.g., `bash`, `powershell`) |
| `args` | `List<String>` | Script arguments |
| `scriptType` | `String` | Type classification (`script_type`) |
| `scriptBody` | `String` | Raw script content (`script_body`) |
| `scriptHash` | `String` | Integrity hash (`script_hash`) |
| `defaultTimeout` | `Integer` | Execution timeout in seconds (`default_timeout`) |
| `supportedPlatforms` | `List<String>` | Compatible OS platforms (`supported_platforms`) |
| `runAsUser` | `Boolean` | Whether to run under user context (`run_as_user`) |
| `envVars` | `List<String>` | Environment variable bindings (`env_vars`) |
| `hidden` | `Boolean` | Visibility flag |
| `favorite` | `Boolean` | Favorited by technician |

## Usage Example

```java
// Deserialize from API response
Script script = objectMapper.readValue(responseBody, Script.class);

// Build via Lombok builder
Script script = Script.builder()
    .id(42)
    .name("Disk Cleanup")
    .shell("powershell")
    .scriptType("automation")
    .scriptBody("Get-Disk | Clear-Disk")
    .defaultTimeout(300)
    .supportedPlatforms(List.of("windows"))
    .runAsUser(false)
    .hidden(false)
    .build();

// Access deserialized fields
System.out.println(script.getScriptBody());
System.out.println(script.getSupportedPlatforms());
```

> **Note:** `@JsonIgnoreProperties(ignoreUnknown = true)` ensures forward compatibility — unknown fields from the API are safely ignored during deserialization.
