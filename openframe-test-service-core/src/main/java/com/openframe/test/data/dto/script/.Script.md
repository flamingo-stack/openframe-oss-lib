A data transfer object (DTO) representing an executable script with metadata for automation and execution configuration in the OpenFrame MSP platform.

## Key Components

- **Core Properties**: `id`, `name`, `description`, `shell`, `args`, `category`
- **Script Content**: `scriptBody`, `scriptHash`, `syntax`, `filename`
- **Execution Settings**: `defaultTimeout`, `runAsUser`, `envVars`, `supportedPlatforms`
- **Classification**: `scriptType`, `favorite`, `hidden`
- **Lombok Annotations**: Automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create a new PowerShell script for Windows automation
Script script = Script.builder()
    .name("System Health Check")
    .description("Monitors CPU, memory, and disk usage")
    .shell("powershell")
    .scriptType("diagnostic")
    .scriptBody("Get-WmiObject -Class Win32_Processor | Select-Object LoadPercentage")
    .defaultTimeout(30)
    .category("monitoring")
    .supportedPlatforms(List.of("windows"))
    .runAsUser(false)
    .favorite(true)
    .hidden(false)
    .build();

// Serialize to JSON for API transport
ObjectMapper mapper = new ObjectMapper();
String json = mapper.writeValueAsString(script);
```
