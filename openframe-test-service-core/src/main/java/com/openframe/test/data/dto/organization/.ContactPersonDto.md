Data Transfer Object representing a contact person's information within an organization context, used for serializing and deserializing contact data in API responses and requests.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `contactName` | `String` | Full name of the contact person |
| `title` | `String` | Job title or role of the contact |
| `phone` | `String` | Contact phone number |
| `email` | `String` | Contact email address |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern instantiation
- `@AllArgsConstructor` / `@NoArgsConstructor` — Generates both full and no-arg constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently ignores unrecognized JSON fields during deserialization

## Usage Example

```java
// Builder pattern
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("Jane Doe")
    .title("IT Manager")
    .phone("+1-555-123-4567")
    .email("jane.doe@company.com")
    .build();

// No-arg constructor with setters
ContactPersonDto contact = new ContactPersonDto();
contact.setContactName("Jane Doe");
contact.setEmail("jane.doe@company.com");

// All-args constructor
ContactPersonDto contact = new ContactPersonDto(
    "Jane Doe", "IT Manager", "+1-555-123-4567", "jane.doe@company.com"
);
```
