A data transfer object (DTO) that encapsulates contact person information for organization-related operations in the OpenFrame platform.

## Key Components

- **contactName** - The full name of the contact person
- **title** - Job title or position within the organization  
- **phone** - Contact phone number
- **email** - Email address for communication

## Usage Example

```java
// Create a new contact person
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("IT Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Use in organization context
OrganizationDto organization = OrganizationDto.builder()
    .name("Acme Corp")
    .primaryContact(contact)
    .build();

// Access contact information
String contactEmail = contact.getEmail();
String contactTitle = contact.getTitle();
```

The class uses Lombok annotations for automatic generation of constructors, getters, setters, and builder pattern implementation, making it ideal for API responses and data mapping operations within the MSP platform's organization management system.
