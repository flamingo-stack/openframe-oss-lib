A DTO (Data Transfer Object) representing filter criteria for querying log entries, supporting multi-value filtering across tool types, event types, severities, and organizations.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolTypes` | `List<String>` | Filter by tool/integration type |
| `eventTypes` | `List<String>` | Filter by log event category |
| `severities` | `List<String>` | Filter by severity level (e.g., INFO, WARN, ERROR) |
| `organizations` | `List<OrganizationFilterOption>` | Filter by one or more organizations |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra fields during deserialization

## Usage Example

```java
// Build a filter for error-level logs from a specific tool
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("RMM", "PSA"))
    .eventTypes(List.of("ALERT", "INCIDENT"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizations(List.of(
        OrganizationFilterOption.builder()
            .id("org-123")
            .name("Acme Corp")
            .build()
    ))
    .build();

// Pass to log query service
List<LogEntry> logs = logService.queryLogs(filters);
```
