A data transfer object (DTO) used to represent the input payload for reordering a ticket relative to other tickets in a list or board.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | The ID of the ticket to be reordered |
| `afterTicketId` | `String` | The ID of the ticket this ticket should be placed after |
| `beforeTicketId` | `String` | The ID of the ticket this ticket should be placed before |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonInclude(NON_NULL)` — Omits `null` fields during JSON serialization

## Usage Example

```java
// Place ticket between two existing tickets
ReorderTicketInput input = ReorderTicketInput.builder()
    .id("ticket-123")
    .afterTicketId("ticket-100")
    .beforeTicketId("ticket-150")
    .build();

// Move ticket to after a specific ticket only
ReorderTicketInput input = ReorderTicketInput.builder()
    .id("ticket-123")
    .afterTicketId("ticket-100")
    .build();
```

> Either `afterTicketId` or `beforeTicketId` can be omitted (serialized as `null` and excluded from JSON output), allowing flexible positioning — move after, before, or between two tickets.
