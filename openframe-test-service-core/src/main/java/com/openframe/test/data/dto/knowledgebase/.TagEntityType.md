Enum defining the types of entities that can be associated with tags in the OpenFrame knowledge base system.

## Key Components

| Value | Description |
|---|---|
| `DEVICE` | Tag applied to a device entity |
| `TICKET` | Tag applied to a support ticket entity |
| `KNOWLEDGE_ARTICLE` | Tag applied to a knowledge base article entity |

## Usage Example

```java
TagEntityType entityType = TagEntityType.TICKET;

switch (entityType) {
    case DEVICE:
        // Handle device tag
        break;
    case TICKET:
        // Handle ticket tag
        break;
    case KNOWLEDGE_ARTICLE:
        // Handle knowledge article tag
        break;
}
```
