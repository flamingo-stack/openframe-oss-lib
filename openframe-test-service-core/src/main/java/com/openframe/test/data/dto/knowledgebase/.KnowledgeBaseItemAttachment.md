A Data Transfer Object (DTO) representing a file attachment associated with a knowledge base item, used for deserializing attachment metadata from API responses in test contexts.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier of the attachment |
| `fileName` | `String` | Name of the attached file |
| `fileSize` | `Long` | Size of the file in bytes |
| `contentType` | `String` | MIME type of the attachment |
| `createdAt` | `String` | Timestamp of when the attachment was created |

## Key Annotations

- **`@Data`** — Generates getters, setters, `equals`, `hashCode`, and `toString`
- **`@Builder`** — Enables fluent builder pattern for object construction
- **`@NoArgsConstructor` / `@AllArgsConstructor`** — Generates both default and full constructors
- **`@JsonIgnoreProperties(ignoreUnknown = true)`** — Tolerates extra fields in JSON responses without throwing errors

## Usage Example

```java
// Builder pattern
KnowledgeBaseItemAttachment attachment = KnowledgeBaseItemAttachment.builder()
    .id("att-001")
    .fileName("network-diagram.pdf")
    .fileSize(204800L)
    .contentType("application/pdf")
    .createdAt("2024-01-15T10:30:00Z")
    .build();

// Deserialization via Jackson
KnowledgeBaseItemAttachment attachment = objectMapper
    .readValue(jsonResponse, KnowledgeBaseItemAttachment.class);

// Assertion in test
assertThat(attachment.getFileName()).isEqualTo("network-diagram.pdf");
assertThat(attachment.getFileSize()).isGreaterThan(0L);
```
