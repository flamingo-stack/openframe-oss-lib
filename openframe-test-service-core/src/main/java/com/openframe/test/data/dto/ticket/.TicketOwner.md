Data Transfer Object (DTO) representing the owner of a ticket, which can be either a machine or a user entity within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `type` | `String` | Identifies the owner type (e.g., machine or user) |
| `machineId` | `String` | Unique identifier for a machine owner |
| `machine` | `TicketMachine` | Nested machine details object |
| `userId` | `String` | Unique identifier for a user owner |
| `user` | `TicketUser` | Nested user details object |

## Key Behaviors

- **`@JsonIgnoreProperties(ignoreUnknown = true)`** — Safely deserializes JSON payloads that may contain extra fields not defined in this DTO, preventing failures on API response changes.
- **Lombok annotations** (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) — Auto-generates getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Build a user-owned ticket owner
TicketOwner owner = TicketOwner.builder()
    .type("user")
    .userId("usr_123")
    .user(TicketUser.builder().name("John Doe").build())
    .build();

// Build a machine-owned ticket owner
TicketOwner machineOwner = TicketOwner.builder()
    .type("machine")
    .machineId("mch_456")
    .machine(TicketMachine.builder().hostname("workstation-01").build())
    .build();
```
