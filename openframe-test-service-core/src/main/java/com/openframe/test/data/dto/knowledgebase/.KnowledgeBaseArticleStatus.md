Enum representing the lifecycle states of a knowledge base article within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `DRAFT` | Article is in progress and not yet visible to end users |
| `PUBLISHED` | Article is live and accessible to clients/technicians |
| `ARCHIVED` | Article has been retired and is no longer active |

## Usage Example

```java
KnowledgeBaseArticleDto article = new KnowledgeBaseArticleDto();
article.setStatus(KnowledgeBaseArticleStatus.PUBLISHED);

if (article.getStatus() == KnowledgeBaseArticleStatus.DRAFT) {
    // Prevent client-facing display
}
```
