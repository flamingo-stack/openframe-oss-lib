A data transfer object (DTO) representing the author of a knowledge base article, containing identity and contact information along with an associated profile image.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the author |
| `firstName` | `String` | Author's first name |
| `lastName` | `String` | Author's last name |
| `email` | `String` | Author's email address |
| `image` | `KnowledgeBaseImage` | Author's associated profile image object |

**Lombok Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently ignores unknown JSON fields during deserialization

## Usage Example

```java
// Builder pattern
KnowledgeBaseAuthor author = KnowledgeBaseAuthor.builder()
    .id("author-123")
    .firstName("Jane")
    .lastName("Doe")
    .email("jane.doe@example.com")
    .image(KnowledgeBaseImage.builder().url("https://example.com/avatar.png").build())
    .build();

// Access fields
String fullName = author.getFirstName() + " " + author.getLastName();
String contact  = author.getEmail();
```

> **Note:** This DTO is designed for test data scenarios within the OpenFrame platform. The `@JsonIgnoreProperties` annotation ensures forward compatibility when the API returns additional author fields not yet mapped in this class.
