A data transfer object that encapsulates all the essential components needed for authentication flows, including OAuth2 PKCE parameters and session management data.

## Key Components

- **Basic Auth Fields**: `email`, `password` for credential-based authentication
- **Multi-tenant Support**: `tenantId` for tenant-specific authentication
- **OAuth2 PKCE Flow**: `state`, `codeChallenge`, `codeVerifier` for secure authorization
- **Session Management**: `cookies` map for maintaining authentication state
- **Authorization Flow**: `authorizationCode` for OAuth2 code exchange

## Usage Example

```java
// Building authentication data for OAuth2 PKCE flow
AuthParts authData = AuthParts.builder()
    .email("user@example.com")
    .tenantId("tenant-123")
    .state("random-state-value")
    .codeChallenge("challenge-string")
    .codeVerifier("verifier-string")
    .cookies(Map.of("session", "abc123", "csrf", "xyz789"))
    .build();

// Building for basic credential authentication
AuthParts basicAuth = AuthParts.builder()
    .email("user@domain.com")
    .password("securePassword")
    .tenantId("tenant-456")
    .build();

// Accessing authentication components
String userEmail = authData.getEmail();
String challenge = authData.getCodeChallenge();
Map<String, String> sessionCookies = authData.getCookies();
```
