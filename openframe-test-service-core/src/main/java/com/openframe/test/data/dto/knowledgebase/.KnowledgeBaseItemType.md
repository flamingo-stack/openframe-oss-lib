Enum representing the two possible content types for knowledge base items within the OpenFrame platform.

## Key Components

| Value | Description |
|-------|-------------|
| `FOLDER` | Represents a container/directory node in the knowledge base hierarchy |
| `ARTICLE` | Represents a leaf-level content document in the knowledge base |

## Usage Example

```java
KnowledgeBaseItemType itemType = KnowledgeBaseItemType.ARTICLE;

if (itemType == KnowledgeBaseItemType.FOLDER) {
    // handle folder navigation
} else {
    // render article content
}
```
