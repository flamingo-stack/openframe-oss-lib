A DTO (Data Transfer Object) that represents the response payload returned after a successful user registration in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the registered user/entity |
| `name` | `String` | Name associated with the registration |
| `domain` | `String` | Domain linked to the account |
| `ownerId` | `String` | ID of the account owner |
| `status` | `String` | Current registration status |
| `plan` | `String` | Subscription plan assigned |
| `hubspotId` | `String` | Linked HubSpot CRM identifier |
| `createdAt` | `String` | Timestamp of account creation |
| `updatedAt` | `String` | Timestamp of last update |
| `active` | `Boolean` | Whether the account is currently active |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Safely ignores extra fields in the JSON response, preventing deserialization failures

## Usage Example

```java
// Deserializing a registration API response
UserRegistrationResponse response = objectMapper.readValue(jsonResponse, UserRegistrationResponse.class);

// Using the builder pattern in tests
UserRegistrationResponse mockResponse = UserRegistrationResponse.builder()
    .id("usr_123")
    .name("Acme Corp")
    .domain("acme.flamingo.run")
    .ownerId("owner_456")
    .status("ACTIVE")
    .plan("PRO")
    .active(true)
    .createdAt("2024-01-15T10:00:00Z")
    .build();
```

> **Note:** This class lives in the `test` package, indicating it is primarily used in integration or API tests to validate registration endpoint responses.
