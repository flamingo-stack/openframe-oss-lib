Utility class for generating test data related to user invitations in the OpenFrame test suite. Provides static factory methods for creating invitation request objects and conflict response payloads using the DataFaker library.

## Key Components

| Method | Description |
|---|---|
| `newUserInvitationRequest()` | Generates an `InvitationRequest` with a random fake email address |
| `existingUserInvitationRequest(AuthUser)` | Generates an `InvitationRequest` using an existing authenticated user's email |
| `acceptInvitationRequest(Invitation)` | Builds an `AcceptInvitationRequest` with fake name data and a configured password |
| `alreadyAcceptedResponse()` | Returns a conflict response for already used or revoked invitations |
| `invitationRevokedResponse()` | Alias for `alreadyAcceptedResponse()` |
| `userAlreadyExistsResponse(AuthUser)` | Returns a conflict response indicating a duplicate user in the tenant |

## Usage Example

```java
// Generate a new invitation request for a fresh user
InvitationRequest request = InvitationGenerator.newUserInvitationRequest();

// Generate an invitation request for an existing user
InvitationRequest existingRequest = InvitationGenerator.existingUserInvitationRequest(authUser);

// Accept an invitation with fake user details
AcceptInvitationRequest acceptRequest = InvitationGenerator.acceptInvitationRequest(invitation);

// Assert conflict responses
InvitationConflictResponse conflictResponse = InvitationGenerator.userAlreadyExistsResponse(authUser);
// conflictResponse.getMessage() → "User with email user@example.com already exists in tenant"
```
