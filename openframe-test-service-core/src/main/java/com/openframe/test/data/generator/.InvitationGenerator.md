A utility class that generates test data for invitation-related operations in OpenFrame's test suite, using the DataFaker library to create realistic mock data for various invitation scenarios.

## Key Components

- **newUserInvitationRequest()** - Creates invitation requests with random email addresses for new users
- **existingUserInvitationRequest()** - Generates invitation requests using existing user email addresses
- **acceptInvitationRequest()** - Builds invitation acceptance requests with random names and default password
- **Conflict Response Generators** - Creates standardized error responses for various conflict scenarios:
  - `alreadyAcceptedResponse()` - For already used/revoked invitations
  - `invitationRevokedResponse()` - Alias for revoked invitation conflicts
  - `userAlreadyExistsResponse()` - For duplicate user email conflicts

## Usage Example

```java
// Generate invitation for new user
InvitationRequest newUserRequest = InvitationGenerator.newUserInvitationRequest();

// Create invitation using existing user's email
AuthUser existingUser = UserGenerator.createAuthUser();
InvitationRequest existingUserRequest = InvitationGenerator.existingUserInvitationRequest(existingUser);

// Build acceptance request for an invitation
Invitation invitation = createTestInvitation();
AcceptInvitationRequest acceptRequest = InvitationGenerator.acceptInvitationRequest(invitation);

// Generate conflict responses for test scenarios
InvitationConflictResponse conflictResponse = InvitationGenerator.userAlreadyExistsResponse(existingUser);
```
