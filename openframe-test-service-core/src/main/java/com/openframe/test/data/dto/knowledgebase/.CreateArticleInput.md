Data Transfer Object (DTO) for creating a knowledge base article, encapsulating all required and optional fields for article creation requests.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | Article title |
| `parentId` | `String` | Parent article/folder ID for hierarchy |
| `content` | `String` | Article body content |
| `summary` | `String` | Brief article description |
| `status` | `KnowledgeBaseArticleStatus` | Publication status enum |
| `tagIds` | `List<String>` | Associated tag identifiers |
| `assignedOrganizationIds` | `List<String>` | Linked organization IDs |
| `assignedDeviceIds` | `List<String>` | Linked device IDs |
| `assignedTicketIds` | `List<String>` | Linked ticket IDs |
| `assignedKnowledgeArticleIds` | `List<String>` | Related article IDs |

**Annotations:**
- `@JsonInclude(NON_NULL)` — omits null fields from serialization
- `@Builder` — enables fluent builder pattern
- `@Data` — generates getters, setters, `equals`, and `hashCode`

## Usage Example

```java
CreateArticleInput article = CreateArticleInput.builder()
    .name("VPN Troubleshooting Guide")
    .content("## Overview\nSteps to resolve common VPN issues...")
    .summary("Quick reference for VPN connectivity problems")
    .status(KnowledgeBaseArticleStatus.PUBLISHED)
    .tagIds(List.of("tag-vpn", "tag-network"))
    .assignedOrganizationIds(List.of("org-abc123"))
    .assignedTicketIds(List.of("ticket-001", "ticket-002"))
    .build();
```
