A data transfer object (DTO) representing software installed on a fleet device, including its name and associated vulnerabilities.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `name` | `String` | The name of the software package |
| `vulnerabilities` | `List<FleetVulnerability>` | List of known vulnerabilities associated with the software |

**Lombok Annotations:**
- `@Data` ‚Äî Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` ‚Äî Enables builder pattern instantiation
- `@NoArgsConstructor` / `@AllArgsConstructor` ‚Äî Generates no-arg and all-arg constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` ‚Äî Silently ignores unmapped JSON fields during deserialization

## Usage Example

```java
// Builder pattern
FleetSoftware software = FleetSoftware.builder()
    .name("OpenSSL 1.0.2")
    .vulnerabilities(List.of(
        FleetVulnerability.builder()
            .cveId("CVE-2023-1234")
            .severity("HIGH")
            .build()
    ))
    .build();

// Accessing fields
String softwareName = software.getName();
List<FleetVulnerability> vulns = software.getVulnerabilities();

// Deserialization (unknown JSON fields are ignored)
FleetSoftware parsed = objectMapper.readValue(json, FleetSoftware.class);
```
