A DTO (Data Transfer Object) representing a tag entity within the Knowledge Base module, used for deserializing tag-related API responses in test data contexts.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `id` | `String` | Unique identifier for the tag |
| `key` | `String` | Tag key/name used for lookup |
| `description` | `String` | Human-readable description of the tag |
| `color` | `String` | Display color associated with the tag |
| `values` | `List<String>` | List of possible values for the tag |
| `entityType` | `TagEntityType` | Enum defining the entity this tag belongs to |
| `createdAt` | `String` | Timestamp of tag creation |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra fields in JSON payloads

## Usage Example

```java
// Builder pattern
KnowledgeBaseTag tag = KnowledgeBaseTag.builder()
    .id("tag-001")
    .key("priority")
    .description("Issue priority level")
    .color("#FF0000")
    .values(List.of("low", "medium", "high"))
    .entityType(TagEntityType.ARTICLE)
    .createdAt("2024-01-15T10:00:00Z")
    .build();

// Deserialization via Jackson
KnowledgeBaseTag tag = objectMapper.readValue(jsonResponse, KnowledgeBaseTag.class);

// Accessing fields
String tagKey = tag.getKey();
List<String> allowedValues = tag.getValues();
```
