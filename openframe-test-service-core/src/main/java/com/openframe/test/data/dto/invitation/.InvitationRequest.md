A data transfer object (DTO) that encapsulates invitation request data, specifically containing the email address of the user to be invited to the OpenFrame platform.

## Key Components

- **email** - String field containing the email address of the invitee
- **Lombok annotations** - Automatically generates getters, setters, constructors, and builder pattern methods
- **@Data** - Provides toString, equals, hashCode, and accessor methods
- **@Builder** - Enables fluent builder pattern for object creation
- **@NoArgsConstructor/@AllArgsConstructor** - Generates default and full parameter constructors

## Usage Example

```java
// Creating invitation request using builder pattern
InvitationRequest request = InvitationRequest.builder()
    .email("user@example.com")
    .build();

// Creating with constructor
InvitationRequest request = new InvitationRequest("user@example.com");

// Using in REST controller
@PostMapping("/invitations")
public ResponseEntity<String> sendInvitation(@RequestBody InvitationRequest request) {
    invitationService.sendInvite(request.getEmail());
    return ResponseEntity.ok("Invitation sent");
}

// Validation example
if (request.getEmail() != null && !request.getEmail().isEmpty()) {
    // Process invitation
}
```

This DTO is typically used in REST API endpoints for handling invitation-related requests, providing a clean contract for client-server communication when inviting users to join the OpenFrame platform.
