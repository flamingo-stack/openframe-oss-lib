A simple Data Transfer Object (DTO) representing the request payload for sending an invitation, containing the recipient's email address.

## Key Components

| Element | Type | Description |
|---|---|---|
| `email` | `String` | The email address of the invitation recipient |

**Lombok Annotations:**
- `@Data` ‚Äî Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` ‚Äî Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` ‚Äî Generates both default and full constructors
- `@JsonInclude(NON_NULL)` ‚Äî Excludes `null` fields from JSON serialization

## Usage Example

```java
// Using the builder pattern
InvitationRequest request = InvitationRequest.builder()
    .email("technician@example.com")
    .build();

// Using the all-args constructor
InvitationRequest request = new InvitationRequest("technician@example.com");

// Using setter (via @Data)
InvitationRequest request = new InvitationRequest();
request.setEmail("technician@example.com");
```
