An enumeration defining the various types of devices that can be managed within the OpenFrame system, providing standardized device categorization for MSP operations.

## Key Components

- **DESKTOP** - Traditional desktop computers
- **LAPTOP** - Portable laptop computers  
- **SERVER** - Server hardware and systems
- **MOBILE_DEVICE** - Smartphones and mobile devices
- **TABLET** - Tablet devices and similar form factors
- **NETWORK_DEVICE** - Routers, switches, and networking equipment
- **IOT_DEVICE** - Internet of Things and smart devices
- **VIRTUAL_MACHINE** - Virtualized computing instances
- **CONTAINER_HOST** - Systems running containerized workloads
- **OTHER** - Fallback category for unspecified device types
