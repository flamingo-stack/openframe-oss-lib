Defines the possible actions that can be performed on children of a folder in the OpenFrame knowledge base module.

## Key Components

- **`MOVE`** – Reassigns child items to another folder
- **`ARCHIVE`** – Archives all child items within the folder

## Usage Example

```java
FolderChildrenAction action = FolderChildrenAction.MOVE;

switch (action) {
    case MOVE:
        // Reassign children to a new parent folder
        break;
    case ARCHIVE:
        // Archive all children of the folder
        break;
}
```
