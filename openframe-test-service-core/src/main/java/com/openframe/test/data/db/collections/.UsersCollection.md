A static utility class that provides convenient methods for querying user documents from the MongoDB "users" collection, with support for filtering by ID, status, and role.

## Key Components

- **findUser()** - Returns the first user document found in the collection
- **findUser(String id)** - Finds a user by their unique ID
- **findUser(UserStatus status)** - Finds an admin user with the specified status
- **findUser(UserRole role)** - Finds an active user with the specified role
- **findUser(String field, String value)** - Generic field-based user lookup
- **findUser(UserStatus status, UserRole role)** - Finds a user matching both status and role criteria

## Usage Example

```java
import com.openframe.test.data.db.collections.UsersCollection;
import com.openframe.test.data.dto.invitation.UserRole;
import com.openframe.test.data.dto.invitation.UserStatus;

// Find any user
AuthUser anyUser = UsersCollection.findUser();

// Find user by ID
AuthUser userById = UsersCollection.findUser("507f1f77bcf86cd799439011");

// Find active admin user
AuthUser activeAdmin = UsersCollection.findUser(UserStatus.ACTIVE, UserRole.ADMIN);

// Find user by email
AuthUser userByEmail = UsersCollection.findUser("email", "user@example.com");

// Find active user with specific role
AuthUser activeUser = UsersCollection.findUser(UserRole.USER);
```

The class leverages MongoDB's filtering capabilities through the `Filters` API and integrates with the OpenFrame test data infrastructure for seamless user data retrieval in testing scenarios.
