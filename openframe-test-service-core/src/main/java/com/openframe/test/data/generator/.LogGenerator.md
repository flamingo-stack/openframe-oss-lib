A utility class for generating test data objects, specifically for creating log filter inputs used in testing scenarios.

## Key Components

- **severityAndToolFilter()** - Static method that creates a `LogFilterInput` object with specified severity level and tool type filters

## Usage Example

```java
// Create a filter for error logs from monitoring tools
LogFilterInput errorFilter = LogGenerator.severityAndToolFilter("ERROR", "MONITORING");

// Create a filter for warning logs from security tools  
LogFilterInput warningFilter = LogGenerator.severityAndToolFilter("WARNING", "SECURITY");

// Use in test scenarios
@Test
public void testLogFiltering() {
    LogFilterInput filter = LogGenerator.severityAndToolFilter("INFO", "APPLICATION");
    
    // Apply filter to log service
    List<LogEntry> filteredLogs = logService.filterLogs(filter);
    
    assertThat(filteredLogs).isNotEmpty();
}
```

The class follows the builder pattern through the `LogFilterInput.builder()` method, making it easy to construct filter objects with specific criteria for log severity levels and tool types in test environments.
