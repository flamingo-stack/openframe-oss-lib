Utility class for generating test data inputs related to log filtering and search operations.

## Key Components

- **`severityFilter(String severity)`** — Static factory method that constructs a `LogFilterInput` object pre-configured with a single severity level, useful for building filter payloads in log-related tests.
- **`searchTerm(String summary)`** — Static helper that extracts the first word from a given summary string, simulating a basic search term for log query tests.

## Usage Example

```java
// Generate a filter input for a specific severity level
LogFilterInput criticalFilter = LogGenerator.severityFilter("CRITICAL");

// Extract a search term from a log summary
String term = LogGenerator.searchTerm("Disk usage exceeded threshold");
// term == "Disk"
```
