A Data Transfer Object (DTO) representing a label/tag that can be applied to a ticket in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the label |
| `key` | `String` | Short label key/name |
| `description` | `String` | Human-readable description of the label |
| `color` | `String` | Display color associated with the label |
| `createdAt` | `Instant` | Timestamp of label creation |
| `createdBy` | `String` | Identifier of the user who created the label |

**Lombok Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Safely deserializes JSON payloads with extra/unknown fields

## Usage Example

```java
// Builder pattern
TicketLabel label = TicketLabel.builder()
    .id("label-001")
    .key("urgent")
    .description("Requires immediate attention")
    .color("#FF0000")
    .createdAt(Instant.now())
    .createdBy("user-42")
    .build();

// Access fields
System.out.println(label.getKey());        // "urgent"
System.out.println(label.getColor());      // "#FF0000"
System.out.println(label.getCreatedBy());  // "user-42"
```
