A data transfer object (DTO) representing a device node within a mesh network, used for deserializing mesh device state from JSON API responses.

## Key Components

| Field | Type | Description |
|---|---|---|
| `nodeId` | `String` | Unique identifier for the mesh network node |
| `online` | `boolean` | Current connectivity status of the device |
| `lastConnectTime` | `long` | Unix timestamp of the last successful connection |
| `lastConnectAddr` | `String` | Network address used during the last connection |

## Annotations

- **`@Data`** — Generates getters, setters, `equals`, `hashCode`, and `toString`
- **`@Builder`** — Enables the builder pattern for object construction
- **`@NoArgsConstructor` / `@AllArgsConstructor`** — Generates both default and full constructors
- **`@JsonIgnoreProperties(ignoreUnknown = true)`** — Silently ignores unrecognized JSON fields during deserialization

## Usage Example

```java
// Builder pattern
MeshDevice device = MeshDevice.builder()
    .nodeId("node-abc123")
    .online(true)
    .lastConnectTime(1718000000L)
    .lastConnectAddr("192.168.1.42")
    .build();

// Check device status
if (device.isOnline()) {
    System.out.println("Node " + device.getNodeId()
        + " connected from " + device.getLastConnectAddr());
}
```
