A data transfer object (DTO) representing a machine/device in the OpenFrame MSP platform test suite, containing comprehensive device information including identification, status, specifications, and connected tools.

## Key Components

- **Identity Fields**: `id`, `machineId`, `hostname`, `displayName` for device identification
- **Network Information**: `ip`, `macAddress` for network connectivity details
- **System Specifications**: `osType`, `osVersion`, `osBuild`, `manufacturer`, `model` for hardware/software details
- **Status Tracking**: `status` (DeviceStatus enum), `lastSeen`, `agentVersion` for monitoring
- **Relationships**: `tags`, `toolConnections`, `installedAgents` collections for associated data
- **Metadata**: `organizationId`, `timezone`, `registeredAt`, `updatedAt` for organizational context

## Usage Example

```java
Machine machine = Machine.builder()
    .machineId("WKSTN-001")
    .ip("192.168.1.100")
    .hostname("workstation-01")
    .displayName("John's Workstation")
    .osType("Windows")
    .osVersion("11")
    .status(DeviceStatus.ONLINE)
    .organizationId("org-123")
    .type(DeviceType.WORKSTATION)
    .agentVersion("2.1.0")
    .build();

// Access device information
String deviceInfo = machine.getHostname() + " (" + machine.getIp() + ")";
boolean isOnline = machine.getStatus() == DeviceStatus.ONLINE;
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it ideal for test data creation and API responses in the OpenFrame platform.
