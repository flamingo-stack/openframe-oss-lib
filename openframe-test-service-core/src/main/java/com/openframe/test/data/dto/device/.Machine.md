A DTO (Data Transfer Object) representing a managed machine/device entity within the OpenFrame platform, used for deserializing device data from API responses in test contexts.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` / `machineId` | `String` | Unique identifiers for the machine |
| `ip` / `macAddress` | `String` | Network identifiers |
| `status` | `DeviceStatus` | Current device status enum |
| `type` | `DeviceType` | Device category enum |
| `tags` | `List<Tag>` | Associated device tags |
| `toolConnections` | `List<ToolConnection>` | Connected MSP tool integrations |
| `installedAgents` | `List<InstalledAgent>` | Agents deployed on the machine |

**Lombok Annotations:**
- `@Data` — generates getters, setters, `equals`, `hashCode`, `toString`
- `@Builder` — enables fluent builder pattern
- `@NoArgsConstructor` / `@AllArgsConstructor` — generates both constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — safely ignores unmapped JSON fields

## Usage Example

```java
// Build a Machine instance in a test
Machine machine = Machine.builder()
    .id("abc-123")
    .machineId("machine-456")
    .hostname("workstation-01")
    .displayName("Dev Workstation")
    .ip("192.168.1.100")
    .osType("Windows")
    .osVersion("11")
    .status(DeviceStatus.ONLINE)
    .type(DeviceType.WORKSTATION)
    .organizationId("org-789")
    .tags(List.of(new Tag("production")))
    .installedAgents(List.of(new InstalledAgent("openframe-agent", "1.2.0")))
    .build();

// Deserialized automatically from API response JSON via Jackson
Machine parsed = objectMapper.readValue(jsonResponse, Machine.class);
```
