An enumeration defining the possible status values for tenant entities in the OpenFrame system.

## Key Components

- **ACTIVE** - Represents an active tenant that can access system resources
- **INACTIVE** - Represents an inactive tenant with restricted or no access

## Usage Example

```java
// Setting tenant status during creation
Tenant tenant = new Tenant();
tenant.setStatus(TenantStatus.ACTIVE);

// Checking tenant status in business logic
if (tenant.getStatus() == TenantStatus.ACTIVE) {
    // Allow tenant operations
    processRequest(tenant);
} else {
    // Handle inactive tenant
    throw new TenantInactiveException("Tenant is not active");
}

// Using in service layer filtering
public List<Tenant> getActiveTenants() {
    return tenantRepository.findByStatus(TenantStatus.ACTIVE);
}
```

This enum provides type-safe status management for tenant lifecycle operations, ensuring consistent status handling across the multi-tenant architecture.
