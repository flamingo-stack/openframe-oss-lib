DTO representing a single knowledge base item (article or category) returned from the OpenFrame knowledge base API, used in test data deserialization.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier of the item |
| `type` | `KnowledgeBaseItemType` | Enum indicating article or category |
| `parentId` | `String` | Reference to parent category |
| `status` | `KnowledgeBaseArticleStatus` | Publication state (draft, published, etc.) |
| `author` | `KnowledgeBaseAuthor` | Nested author details |
| `tags` | `List<KnowledgeBaseTag>` | Associated tags |
| `attachments` | `List<KnowledgeBaseItemAttachment>` | File attachments |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates unknown fields during deserialization

## Usage Example

```java
// Deserialize from API response
KnowledgeBaseItem item = objectMapper.readValue(responseBody, KnowledgeBaseItem.class);

// Build manually in tests
KnowledgeBaseItem item = KnowledgeBaseItem.builder()
    .id("kb-001")
    .type(KnowledgeBaseItemType.ARTICLE)
    .name("VPN Setup Guide")
    .status(KnowledgeBaseArticleStatus.PUBLISHED)
    .author(KnowledgeBaseAuthor.builder().name("Jane Doe").build())
    .tags(List.of(new KnowledgeBaseTag("networking")))
    .build();

// Access fields
String title = item.getName();
KnowledgeBaseItemType type = item.getType();
```
