A data transfer object (DTO) that represents standardized error responses from the OpenFrame API, following the observed structure of actual error responses.

## Key Components

- **code** - String field containing the error type identifier (e.g., "BAD_REQUEST", "VALIDATION_ERROR")
- **message** - String field with the human-readable error description
- **Lombok annotations** - `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating an error response using builder pattern
ErrorResponse errorResponse = ErrorResponse.builder()
    .code("BAD_REQUEST")
    .message("Registration is closed for this organization")
    .build();

// Creating an error response using constructor
ErrorResponse error = new ErrorResponse("VALIDATION_ERROR", "Invalid email format");

// Accessing fields
String errorCode = errorResponse.getCode();
String errorMessage = errorResponse.getMessage();
```

This DTO is designed to match the actual API response structure observed in OpenFrame error responses, ensuring consistent error handling across test scenarios and API interactions.
