A DTO (Data Transfer Object) representing a conflict response returned when an invitation operation encounters a conflict (e.g., HTTP 409), containing an error code and descriptive message.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `code` | `String` | Machine-readable error code identifying the conflict type |
| `message` | `String` | Human-readable description of the conflict |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@AllArgsConstructor` / `@NoArgsConstructor` — Generates both full and no-arg constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Deserialize from a 409 Conflict API response
InvitationConflictResponse conflict = objectMapper.readValue(responseBody,
    InvitationConflictResponse.class);

System.out.println(conflict.getCode());    // e.g., "INVITATION_ALREADY_EXISTS"
System.out.println(conflict.getMessage()); // e.g., "An invitation for this email already exists"

// Build manually in tests
InvitationConflictResponse mockConflict = InvitationConflictResponse.builder()
    .code("INVITATION_ALREADY_EXISTS")
    .message("An invitation for this email already exists")
    .build();
```

> **Note:** This class is part of the `com.openframe.test` package, indicating it is used within the [OpenFrame](https://openframe.ai) test suite to assert conflict scenarios in invitation-related API flows.
