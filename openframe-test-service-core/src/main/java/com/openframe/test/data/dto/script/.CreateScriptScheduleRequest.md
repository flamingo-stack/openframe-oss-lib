A DTO representing the request payload for creating a scheduled script task in the OpenFrame platform.

## Key Components

**`CreateScriptScheduleRequest`** — Top-level request object containing:

| Field | Type | JSON Key |
|---|---|---|
| `name` | `String` | `name` |
| `taskType` | `String` | `task_type` |
| `runTimeDate` | `String` | `run_time_date` |
| `taskSupportedPlatforms` | `List<String>` | `task_supported_platforms` |
| `enabled` | `Boolean` | `enabled` |
| `dailyInterval` | `Integer` | `daily_interval` |
| `actions` | `List<Action>` | `actions` |

**`Action`** — Nested static class defining individual script actions within the schedule:

| Field | Type | JSON Key |
|---|---|---|
| `type` | `String` | `type` |
| `script` | `Integer` | `script` |
| `name` | `String` | `name` |
| `timeout` | `Integer` | `timeout` |
| `scriptArgs` | `List<String>` | `script_args` |
| `envVars` | `List<String>` | `env_vars` |

## Usage Example

```java
CreateScriptScheduleRequest request = CreateScriptScheduleRequest.builder()
    .name("Nightly Cleanup")
    .taskType("recurring")
    .runTimeDate("2024-01-15T02:00:00")
    .taskSupportedPlatforms(List.of("windows", "linux"))
    .enabled(true)
    .dailyInterval(1)
    .actions(List.of(
        CreateScriptScheduleRequest.Action.builder()
            .type("script")
            .script(42)
            .name("cleanup-script")
            .timeout(300)
            .scriptArgs(List.of("--verbose", "--dry-run"))
            .envVars(List.of("ENV=production"))
            .build()
    ))
    .build();
```

> Unknown JSON properties are silently ignored via `@JsonIgnoreProperties(ignoreUnknown = true)`, making the DTO resilient to API response changes.
