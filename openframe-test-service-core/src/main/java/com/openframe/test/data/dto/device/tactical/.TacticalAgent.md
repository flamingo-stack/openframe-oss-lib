A simple DTO (Data Transfer Object) representing a Tactical RMM agent, used for deserializing agent data in test contexts. It captures the agent's status field while safely ignoring any unknown JSON properties.

## Key Components

| Element | Description |
|---|---|
| `status` | String field representing the current state of the Tactical RMM agent |
| `@JsonIgnoreProperties(ignoreUnknown = true)` | Prevents deserialization failures when the API response contains additional fields |
| `@Builder` | Enables fluent builder pattern for object construction |
| `@Data` | Generates getters, setters, `equals`, `hashCode`, and `toString` via Lombok |

## Usage Example

```java
// Deserialize from JSON response
TacticalAgent agent = objectMapper.readValue(jsonResponse, TacticalAgent.class);
String currentStatus = agent.getStatus();

// Build via builder pattern
TacticalAgent agent = TacticalAgent.builder()
    .status("online")
    .build();
```
