Data Transfer Object (DTO) representing an authenticated user within the OpenFrame platform, encapsulating identity, authentication state, and role-based access information.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique user identifier |
| `tenantId` | `String` | Associated tenant for multi-tenancy support |
| `passwordHash` | `String` | Hashed user password |
| `emailVerified` | `Boolean` | Email verification status |
| `loginProvider` | `String` | Authentication provider (e.g., local, OAuth) |
| `email` | `String` | User email address |
| `firstName` / `lastName` | `String` | User display name |
| `roles` | `List<UserRole>` | Assigned roles for authorization |
| `status` | `UserStatus` | Current account status |
| `image` | `Object` | User profile image |
| `createdAt` / `updatedAt` | `LocalDateTime` | Audit timestamps |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Provides both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Safely ignores unmapped JSON fields during deserialization

## Usage Example

```java
AuthUser user = AuthUser.builder()
    .id("usr_abc123")
    .tenantId("tenant_xyz")
    .email("tech@flamingo.run")
    .firstName("Mingo")
    .lastName("AI")
    .emailVerified(true)
    .loginProvider("local")
    .status(UserStatus.ACTIVE)
    .roles(List.of(UserRole.TECHNICIAN))
    .createdAt(LocalDateTime.now())
    .build();

String email = user.getEmail();
List<UserRole> roles = user.getRoles();
```
