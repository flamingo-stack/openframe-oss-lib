A data transfer object representing a filter option for device filtering operations, containing the option value, display label, and associated count.

## Key Components

- **`value`** - The actual filter value used internally
- **`label`** - Human-readable display text for the filter option
- **`count`** - Number of items associated with this filter option

## Usage Example

```java
// Create a device filter option for status filtering
DeviceFilterOption statusOption = DeviceFilterOption.builder()
    .value("active")
    .label("Active Devices")
    .count(45)
    .build();

// Use in filter dropdown
List<DeviceFilterOption> statusFilters = Arrays.asList(
    DeviceFilterOption.builder()
        .value("active")
        .label("Active")
        .count(120)
        .build(),
    DeviceFilterOption.builder()
        .value("inactive")
        .label("Inactive")
        .count(23)
        .build()
);

// Access properties
String filterValue = statusOption.getValue();    // "active"
String displayText = statusOption.getLabel();   // "Active Devices"
Integer itemCount = statusOption.getCount();    // 45
```

This DTO is typically used in UI components to populate filter dropdowns with options showing both the display name and count of matching items.
