A data transfer object (DTO) for handling password reset confirmation requests, containing the reset token and new password provided by users during the password reset process.

## Key Components

- **token** - The password reset token sent to the user's email or generated during the reset initiation
- **newPassword** - The new password the user wants to set for their account
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a reset confirmation request using builder pattern
ResetConfirmRequest request = ResetConfirmRequest.builder()
    .token("eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .newPassword("newSecurePassword123!")
    .build();

// Using in a REST controller
@PostMapping("/auth/reset/confirm")
public ResponseEntity<?> confirmPasswordReset(@RequestBody ResetConfirmRequest request) {
    userService.confirmPasswordReset(request.getToken(), request.getNewPassword());
    return ResponseEntity.ok().build();
}

// Accessing fields
String resetToken = request.getToken();
String password = request.getNewPassword();
```

This DTO is typically used in REST endpoints where users submit their password reset token along with their desired new password to complete the password reset workflow.
