A simple DTO (Data Transfer Object) used to carry the password reset confirmation payload, containing a reset token and the new desired password.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `token` | `String` | The password reset token issued during the reset request flow |
| `newPassword` | `String` | The new password the user wishes to set |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@AllArgsConstructor` / `@NoArgsConstructor` — Generates both full and empty constructors
- `@JsonInclude(NON_NULL)` — Omits `null` fields during JSON serialization

## Usage Example

```java
// Using the builder pattern
ResetConfirmRequest request = ResetConfirmRequest.builder()
    .token("eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .newPassword("MyNewSecureP@ssword123")
    .build();

// Using the all-args constructor
ResetConfirmRequest request = new ResetConfirmRequest(
    "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "MyNewSecureP@ssword123"
);

// Deserializing from JSON (null fields are excluded in serialization)
// {
//   "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
//   "newPassword": "MyNewSecureP@ssword123"
// }
```

> **Note:** This DTO is part of the `com.openframe.test.data.dto.user` package, typically used in test scenarios to validate the password reset confirmation endpoint in the OpenFrame platform.
