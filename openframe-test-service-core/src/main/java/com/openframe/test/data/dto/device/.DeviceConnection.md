A DTO (Data Transfer Object) representing a paginated/filtered connection of device edges, typically used in GraphQL-style API responses.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `edges` | `List<DeviceEdge>` | List of device edge objects in the connection |
| `filteredCount` | `Integer` | Total count of devices after filtering |

**Lombok Annotations:**
- `@Data` — generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — safely ignores unknown JSON fields during deserialization

## Usage Example

```java
// Builder pattern
DeviceConnection connection = DeviceConnection.builder()
    .edges(List.of(new DeviceEdge(), new DeviceEdge()))
    .filteredCount(2)
    .build();

// Access fields
List<DeviceEdge> edges = connection.getEdges();
Integer count = connection.getFilteredCount();

// Deserialization (unknown JSON fields are safely ignored)
DeviceConnection fromJson = objectMapper.readValue(jsonString, DeviceConnection.class);
```
