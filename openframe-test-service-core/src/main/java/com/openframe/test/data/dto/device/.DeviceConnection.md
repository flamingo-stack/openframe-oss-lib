A data transfer object representing a paginated collection of device connections with filtering capabilities in the OpenFrame MSP platform.

## Key Components

- **edges**: List of `DeviceEdge` objects containing the actual device connection data
- **filteredCount**: Total number of devices after applying filters (useful for pagination)
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Building a device connection response
DeviceConnection deviceConnection = DeviceConnection.builder()
    .edges(List.of(
        DeviceEdge.builder()
            .node(device1)
            .cursor("cursor1")
            .build(),
        DeviceEdge.builder()
            .node(device2)
            .cursor("cursor2")
            .build()
    ))
    .filteredCount(42)
    .build();

// Accessing filtered results
List<DeviceEdge> deviceEdges = deviceConnection.getEdges();
Integer totalFiltered = deviceConnection.getFilteredCount();

// Typical GraphQL connection pattern usage
return DeviceConnection.builder()
    .edges(pagedDevices)
    .filteredCount(totalMatchingDevices)
    .build();
```

This DTO follows the GraphQL connection pattern commonly used in OpenFrame's API layer for handling paginated device queries with cursor-based navigation and filtering metadata.
