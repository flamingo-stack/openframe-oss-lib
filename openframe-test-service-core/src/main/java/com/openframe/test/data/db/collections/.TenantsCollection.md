MongoDB collection wrapper for tenant data operations, providing convenient methods to query tenant documents by domain or other fields.

## Key Components

- **`findTenantByDomain(String domain)`** - Finds a tenant by their domain name
- **`findTenant(String field, String value)`** - Generic method to find a tenant by any field-value pair
- Uses MongoDB `Filters.eq()` for exact match queries
- Integrates with the shared `MongoDB.getCollection()` utility

## Usage Example

```java
// Find tenant by domain
Tenant tenant = TenantsCollection.findTenantByDomain("example.com");

// Find tenant by custom field
Tenant tenantById = TenantsCollection.findTenant("_id", "tenant123");
Tenant tenantByName = TenantsCollection.findTenant("name", "Acme Corp");

// Handle null results when tenant not found
if (tenant != null) {
    System.out.println("Found tenant: " + tenant.getName());
} else {
    System.out.println("Tenant not found");
}
```

This collection class follows the repository pattern for tenant data access, abstracting MongoDB operations behind simple static methods. The `findTenant` method provides flexibility for querying by any field, while `findTenantByDomain` offers a convenient shortcut for the most common use case.
