A data transfer object representing a connection between a device tool and the OpenFrame system, tracking connection state and metadata for MSP tool integrations.

## Key Components

- **Connection Identifiers**: `id`, `machineId`, and `agentToolId` for unique identification
- **Tool Metadata**: `toolType` specifies the category of connected tool
- **Connection Status**: `status` field using `ConnectionStatus` enum to track current state
- **Metadata Storage**: `metadata` field for storing tool-specific configuration or information
- **Timestamp Tracking**: Connection lifecycle timestamps (`connectedAt`, `lastSyncAt`, `disconnectedAt`)

## Usage Example

```java
// Creating a new tool connection
ToolConnection connection = ToolConnection.builder()
    .id("conn_123")
    .machineId("machine_456")
    .toolType("monitoring")
    .agentToolId("agent_tool_789")
    .status(ConnectionStatus.ACTIVE)
    .metadata("{\"endpoint\": \"https://api.example.com\"}")
    .connectedAt("2024-01-15T10:30:00Z")
    .lastSyncAt("2024-01-15T11:00:00Z")
    .build();

// Updating connection status
connection.setStatus(ConnectionStatus.DISCONNECTED);
connection.setDisconnectedAt("2024-01-15T12:00:00Z");
```

This DTO enables OpenFrame to track and manage connections to various MSP tools, providing visibility into tool availability and synchronization status across the unified platform.
