Data Transfer Object (DTO) representing a connection between a device/machine and an external tool within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the tool connection |
| `machineId` | `String` | ID of the machine associated with the connection |
| `toolType` | `String` | Type/category of the tool being connected |
| `agentToolId` | `String` | Identifier of the agent-side tool instance |
| `status` | `ConnectionStatus` | Current connection state (enum) |
| `metadata` | `String` | Additional connection metadata (JSON or plain string) |
| `connectedAt` | `String` | Timestamp when the connection was established |
| `lastSyncAt` | `String` | Timestamp of the last synchronization |
| `disconnectedAt` | `String` | Timestamp when the connection was terminated |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates unknown JSON fields during deserialization

## Usage Example

```java
ToolConnection connection = ToolConnection.builder()
    .id("conn-001")
    .machineId("machine-abc")
    .toolType("RMM")
    .agentToolId("agent-tool-xyz")
    .status(ConnectionStatus.CONNECTED)
    .metadata("{\"version\": \"1.0\"}")
    .connectedAt("2024-01-15T10:00:00Z")
    .lastSyncAt("2024-01-15T12:00:00Z")
    .build();
```

> **Note:** `disconnectedAt` will remain `null` for active connections and is only populated when a tool is explicitly disconnected from the device.
