A DTO (Data Transfer Object) representing paginated execution history for scheduled scripts, used for deserializing API responses in test data handling.

## Key Components

### `ScheduleExecutionHistory` (Root Class)
Paginated wrapper containing execution history records.

| Field | Type | Description |
|-------|------|-------------|
| `total` | `Integer` | Total number of records available |
| `limit` | `Integer` | Page size limit |
| `offset` | `Integer` | Pagination offset |
| `results` | `List<Result>` | List of individual execution records |

### `ScheduleExecutionHistory.Result` (Nested Class)
Represents a single script execution record.

| Field | JSON Key | Description |
|-------|----------|-------------|
| `id` | `id` | Execution record identifier |
| `agentId` | `agent_id` | ID of the agent that ran the script |
| `agentHostname` | `agent_hostname` | Hostname of the executing agent |
| `agentPlatform` | `agent_platform` | OS/platform of the agent |
| `organization` | `organization` | Owning organization |
| `retcode` | `retcode` | Process return/exit code |
| `stdout` | `stdout` | Standard output from execution |
| `stderr` | `stderr` | Standard error from execution |
| `executionTime` | `execution_time` | Duration of execution |
| `lastRun` | `last_run` | Timestamp of last run |
| `status` | `status` | Execution status |
| `syncStatus` | `sync_status` | Synchronization status |

## Usage Example

```java
// Deserializing an API response
ObjectMapper mapper = new ObjectMapper();
ScheduleExecutionHistory history = mapper.readValue(apiResponse, ScheduleExecutionHistory.class);

// Accessing paginated results
System.out.println("Total records: " + history.getTotal());

for (ScheduleExecutionHistory.Result result : history.getResults()) {
    System.out.println("Agent: " + result.getAgentHostname());
    System.out.println("Status: " + result.getStatus());
    System.out.println("Exit Code: " + result.getRetcode());
}

// Builder pattern usage
ScheduleExecutionHistory.Result result = ScheduleExecutionHistory.Result.builder()
    .agentId("agent-123")
    .agentHostname("workstation-01")
    .status("completed")
    .retcode(0)
    .build();
```
