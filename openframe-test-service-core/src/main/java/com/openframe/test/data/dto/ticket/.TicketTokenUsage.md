A data transfer object (DTO) representing AI token usage metrics associated with a ticket's chat interaction in the OpenFrame platform.

## Key Components

| Field | Type | Description |
|---|---|---|
| `chatType` | `String` | The type/category of chat interaction |
| `inputTokensSize` | `Long` | Number of tokens in the input/prompt |
| `outputTokensSize` | `Long` | Number of tokens in the generated response |
| `totalTokensSize` | `Long` | Combined total of input and output tokens |
| `contextSize` | `Long` | Size of the context window used |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Build via fluent builder
TicketTokenUsage usage = TicketTokenUsage.builder()
    .chatType("MINGO_AI")
    .inputTokensSize(512L)
    .outputTokensSize(256L)
    .totalTokensSize(768L)
    .contextSize(4096L)
    .build();

// Access metrics
Long total = usage.getTotalTokensSize();
System.out.println("Total tokens used: " + total);
```
