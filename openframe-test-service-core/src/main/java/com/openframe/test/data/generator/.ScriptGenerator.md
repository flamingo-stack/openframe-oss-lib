Utility class that generates test data and mock responses for Script-related operations in the OpenFrame platform testing framework.

## Key Components

- **`addScriptRequest()`** - Static method that creates a pre-configured Script object with sample data for testing script creation scenarios
- **`addScriptResponse(Script)`** - Static method that generates a formatted success message for script addition operations
- **`editScriptResponse(Script)`** - Static method that generates a formatted success message for script editing operations

## Usage Example

```java
// Generate test script data
Script testScript = ScriptGenerator.addScriptRequest();

// Create mock responses for API operations
String addResponse = ScriptGenerator.addScriptResponse(testScript);
// Returns: "Dir was added!"

String editResponse = ScriptGenerator.editScriptResponse(testScript);
// Returns: "Dir was edited!"

// Use in test scenarios
@Test
public void testScriptCreation() {
    Script script = ScriptGenerator.addScriptRequest();
    // Verify script properties
    assertEquals("Dir", script.getName());
    assertEquals("powershell", script.getShell());
    assertEquals(List.of("windows"), script.getSupportedPlatforms());
}
```

The generated script includes realistic test data like PowerShell commands, Windows platform support, environment variables, and typical script metadata used in MSP automation workflows.
