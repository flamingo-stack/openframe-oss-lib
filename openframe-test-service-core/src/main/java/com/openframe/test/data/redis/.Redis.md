Redis utility class for retrieving password reset tokens from a Redis cache using the Jedis client library. This class scans Redis keys to find password reset tokens associated with specific email addresses.

## Key Components

- **getResetToken(String email)**: Static method that scans Redis keys matching a password reset pattern and returns the token associated with the provided email address
- **KEY**: Constant defining the Redis key pattern `"of:{default}:pwdreset:*"` used for matching password reset entries
- **Redis scanning logic**: Uses `SCAN` command with pagination to efficiently search through keys without blocking Redis

## Usage Example

```java
// Retrieve a password reset token for a user
String userEmail = "user@example.com";
String resetToken = Redis.getResetToken(userEmail);

if (resetToken != null) {
    System.out.println("Reset token found: " + resetToken);
    // Use the token for password reset validation
} else {
    System.out.println("No reset token found for email: " + userEmail);
}
```

The method connects to Redis using configuration from `RedisConfig.getRedisUri()`, scans keys matching the password reset pattern, and returns the token portion of the key when a matching email is found. The connection is properly closed after the operation to prevent resource leaks.
