A data transfer object (DTO) that represents an edge in a paginated device graph, containing a device node and its pagination cursor for GraphQL-style pagination.

## Key Components

- **`Machine node`** - The actual device/machine object at this edge
- **`cursor`** - String identifier used for cursor-based pagination navigation
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Building a device edge for pagination response
DeviceEdge edge = DeviceEdge.builder()
    .node(machine)
    .cursor("eyJpZCI6MTIzfQ==")
    .build();

// Using in a paginated device list
List<DeviceEdge> edges = deviceService.getDevices(first, after)
    .stream()
    .map(machine -> DeviceEdge.builder()
        .node(machine)
        .cursor(encodeCursor(machine.getId()))
        .build())
    .collect(Collectors.toList());

// Extracting cursor for next page request
String lastCursor = edges.get(edges.size() - 1).getCursor();
```

This pattern follows GraphQL Relay specification for cursor-based pagination, where each edge contains both the data item (node) and a cursor for efficient pagination through large device datasets.
