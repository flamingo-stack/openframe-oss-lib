A GraphQL-style edge wrapper DTO that pairs a `Machine` node with its pagination cursor, used in paginated device query responses.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `node` | `Machine` | The device/machine entity payload |
| `cursor` | `String` | Opaque pagination cursor for this edge |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra fields in API responses without throwing errors

## Usage Example

```java
// Deserializing a paginated device response
DeviceEdge edge = DeviceEdge.builder()
    .node(machine)
    .cursor("eyJpZCI6IjEyMzQifQ==")
    .build();

// Accessing the machine and cursor
Machine device = edge.getNode();
String nextCursor = edge.getCursor();
```
