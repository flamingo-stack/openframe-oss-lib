A data transfer object (DTO) that represents the structure for user registration API requests in OpenFrame's test suite.

## Key Components

**Main Class:**
- `UserRegistrationRequest` - DTO with user and tenant registration fields

**Lombok Annotations:**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Creates fluent builder pattern with `toBuilder` support
- `@NoArgsConstructor` - Generates default constructor
- `@AllArgsConstructor` - Generates constructor with all fields

**Fields:**
- `email` - User's email address
- `firstName` - User's first name
- `lastName` - User's last name
- `password` - User's password
- `tenantName` - Organization/tenant name
- `tenantDomain` - Tenant's domain identifier

## Usage Example

```java
// Create registration request using builder pattern
UserRegistrationRequest request = UserRegistrationRequest.builder()
    .email("john.doe@example.com")
    .firstName("John")
    .lastName("Doe")
    .password("SecurePassword123")
    .tenantName("Acme Corp")
    .tenantDomain("acme")
    .build();

// Modify existing request
UserRegistrationRequest updatedRequest = request.toBuilder()
    .email("john.doe@acme.com")
    .build();

// Use in test scenarios
@Test
void testUserRegistration() {
    UserRegistrationRequest testUser = UserRegistrationRequest.builder()
        .email("test@openframe.ai")
        .firstName("Test")
        .lastName("User")
        .password("TestPass123")
        .tenantName("Test Tenant")
        .tenantDomain("test")
        .build();
    
    // Use in API test calls
    mockMvc.perform(post("/api/users/register")
        .contentType(MediaType.APPLICATION_JSON)
        .content(objectMapper.writeValueAsString(testUser)));
}
```
