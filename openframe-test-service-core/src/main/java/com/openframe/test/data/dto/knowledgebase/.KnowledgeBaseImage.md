A simple DTO representing an image entry within the Flamingo/OpenFrame knowledge base, holding the image's URL and a hash for identification or deduplication.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `imageUrl` | `String` | The URL pointing to the knowledge base image resource |
| `hash` | `String` | A hash value used for image identification or deduplication |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Safely ignores unknown JSON fields during deserialization

## Usage Example

```java
// Builder pattern
KnowledgeBaseImage image = KnowledgeBaseImage.builder()
    .imageUrl("https://cdn.example.com/kb/image123.png")
    .hash("d41d8cd98f00b204e9800998ecf8427e")
    .build();

// All-args constructor
KnowledgeBaseImage image = new KnowledgeBaseImage(
    "https://cdn.example.com/kb/image123.png",
    "d41d8cd98f00b204e9800998ecf8427e"
);

// Accessing fields
String url = image.getImageUrl();
String hash = image.getHash();
```
