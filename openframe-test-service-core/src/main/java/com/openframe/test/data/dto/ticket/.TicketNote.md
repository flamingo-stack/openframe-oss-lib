A data transfer object (DTO) representing a note attached to a support ticket, used for deserializing ticket note data in test contexts.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the note |
| `ticketId` | `String` | Reference to the parent ticket |
| `content` | `String` | The note's text body |
| `authorId` | `String` | Identifier of the note's author |
| `author` | `TicketUser` | Nested author user object |
| `authorImage` | `TicketImage` | Author's profile image data |
| `createdAt` | `Instant` | Timestamp of note creation |
| `updatedAt` | `Instant` | Timestamp of last modification |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Silently drops unrecognized JSON fields during deserialization

## Usage Example

```java
// Builder pattern
TicketNote note = TicketNote.builder()
    .id("note-001")
    .ticketId("ticket-42")
    .content("Issue reproduced on version 3.2.1")
    .authorId("user-99")
    .author(new TicketUser("user-99", "Jane Doe"))
    .createdAt(Instant.now())
    .build();

// Deserialization via Jackson
TicketNote note = objectMapper.readValue(jsonString, TicketNote.class);

// Accessing fields
String noteContent = note.getContent();
Instant created = note.getCreatedAt();
```
