A data transfer object (DTO) that represents the response structure for user authentication status requests, typically used in "me" or current user endpoints.

## Key Components

- **User user** - The authenticated user object containing user details
- **boolean authenticated** - Flag indicating whether the user is currently authenticated
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a response for an authenticated user
MeResponse authenticatedResponse = MeResponse.builder()
    .user(currentUser)
    .authenticated(true)
    .build();

// Creating a response for an unauthenticated request
MeResponse unauthenticatedResponse = MeResponse.builder()
    .user(null)
    .authenticated(false)
    .build();

// In a REST controller
@GetMapping("/api/me")
public ResponseEntity<MeResponse> getCurrentUser(Authentication auth) {
    if (auth != null && auth.isAuthenticated()) {
        User user = userService.findByUsername(auth.getName());
        return ResponseEntity.ok(
            MeResponse.builder()
                .user(user)
                .authenticated(true)
                .build()
        );
    }
    
    return ResponseEntity.ok(
        MeResponse.builder()
            .authenticated(false)
            .build()
    );
}
```

This DTO follows common REST API patterns for user session validation and provides a consistent structure for frontend applications to determine authentication state and retrieve current user information.
