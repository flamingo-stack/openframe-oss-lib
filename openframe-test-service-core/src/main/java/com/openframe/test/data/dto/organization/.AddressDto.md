A Data Transfer Object (DTO) representing a physical address, used within the organization domain of the OpenFrame test data layer.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `street1` | `String` | Primary street address line |
| `street2` | `String` | Secondary address line (suite, unit, etc.) |
| `city` | `String` | City name |
| `state` | `String` | State or province |
| `postalCode` | `String` | ZIP or postal code |
| `country` | `String` | Country name or code |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main St")
    .street2("Suite 400")
    .city("Austin")
    .state("TX")
    .postalCode("78701")
    .country("US")
    .build();

// All-args constructor
AddressDto address = new AddressDto(
    "123 Main St", "Suite 400", "Austin", "TX", "78701", "US"
);

// No-args + setters
AddressDto address = new AddressDto();
address.setCity("Austin");
address.setState("TX");
```
