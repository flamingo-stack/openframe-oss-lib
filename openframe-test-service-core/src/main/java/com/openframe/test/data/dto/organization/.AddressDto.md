A data transfer object (DTO) that represents address information for organizations within the OpenFrame MSP platform, using Lombok annotations for automatic code generation.

## Key Components

- **AddressDto**: Main DTO class with standard address fields
- **Lombok Annotations**: 
  - `@Data` - Generates getters, setters, toString, equals, and hashCode
  - `@Builder` - Provides builder pattern implementation
  - `@NoArgsConstructor` - Creates no-argument constructor
  - `@AllArgsConstructor` - Creates constructor with all fields
- **Address Fields**: Standard postal address components (street1, street2, city, state, postalCode, country)

## Usage Example

```java
// Create address using builder pattern
AddressDto address = AddressDto.builder()
    .street1("123 Main Street")
    .street2("Suite 100")
    .city("San Francisco")
    .state("CA")
    .postalCode("94105")
    .country("USA")
    .build();

// Create empty address and populate
AddressDto address2 = new AddressDto();
address2.setStreet1("456 Oak Avenue");
address2.setCity("Austin");
address2.setState("TX");

// Use in organization context
OrganizationDto org = OrganizationDto.builder()
    .name("Tech Company")
    .address(address)
    .build();
```

This DTO is commonly used in organization management, client registration, and MSP service delivery contexts where physical addresses need to be captured and transferred between system layers.
