A DTO (Data Transfer Object) representing a log event record used for testing purposes within the OpenFrame platform, capturing telemetry and audit data across tools and devices.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolEventId` | `String` | Unique identifier for the tool event |
| `eventType` | `String` | Category or type of the event |
| `ingestDay` | `String` | Date the event was ingested |
| `toolType` | `String` | The source tool that generated the event |
| `severity` | `String` | Event severity level (e.g., INFO, WARN, ERROR) |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Hostname of the originating machine |
| `organizationId` | `String` | Tenant/organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `summary` | `String` | Brief description of the event |
| `timestamp` | `String` | ISO timestamp of when the event occurred |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonIgnoreProperties(ignoreUnknown = true)` — tolerant deserialization for forward-compatible payloads

## Usage Example

```java
LogEvent event = LogEvent.builder()
    .toolEventId("evt-001")
    .eventType("ALERT")
    .severity("HIGH")
    .hostname("workstation-42")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .toolType("RMM")
    .userId("user-789")
    .deviceId("device-456")
    .summary("Disk usage exceeded 90%")
    .timestamp("2024-01-15T10:30:00Z")
    .ingestDay("2024-01-15")
    .build();
```
