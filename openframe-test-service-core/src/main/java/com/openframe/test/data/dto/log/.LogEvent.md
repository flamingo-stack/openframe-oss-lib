A data transfer object (DTO) representing a log event in the OpenFrame test data system, designed to capture standardized logging information across different tools and systems.

## Key Components

- **Data Fields**: Comprehensive set of properties including event identification (`toolEventId`, `eventType`), metadata (`ingestDay`, `toolType`, `severity`), entity references (`userId`, `deviceId`, `hostname`), organizational context (`organizationId`, `organizationName`), and event details (`summary`, `timestamp`)
- **Lombok Annotations**: Uses `@Data` for getters/setters, `@Builder` for fluent object creation, and constructor annotations for flexible instantiation

## Usage Example

```java
// Create a log event using builder pattern
LogEvent logEvent = LogEvent.builder()
    .toolEventId("evt_123456")
    .eventType("SECURITY_ALERT")
    .ingestDay("2024-01-15")
    .toolType("ENDPOINT_DETECTION")
    .severity("HIGH")
    .userId("user_789")
    .deviceId("device_456")
    .hostname("workstation-01")
    .organizationId("org_123")
    .organizationName("Acme Corp")
    .summary("Malware detected on endpoint")
    .timestamp("2024-01-15T10:30:00Z")
    .build();

// Access properties
String eventId = logEvent.getToolEventId();
String severity = logEvent.getSeverity();
```

This DTO provides a standardized structure for log events in testing scenarios, supporting various security and monitoring tools within the OpenFrame platform.
