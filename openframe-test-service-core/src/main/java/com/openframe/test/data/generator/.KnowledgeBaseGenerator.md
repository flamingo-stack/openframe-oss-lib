Utility class for generating test data related to knowledge base operations, including filter inputs and article creation payloads using randomized fake data.

## Key Components

| Method | Description |
|--------|-------------|
| `rootFoldersFilter()` | Creates a `KnowledgeBaseFilterInput` scoped to root-level folders |
| `articlesInFolderFilter(String folderId)` | Creates a `KnowledgeBaseFilterInput` for articles within a specific folder |
| `draftArticle(String parentFolderId, List<String> tagIds)` | Builds a `CreateArticleInput` with randomized name, content, and summary in `DRAFT` status |
| `randomFolderName()` | Returns a randomized folder name prefixed with `"Test Folder"` |

## Usage Example

```java
// Generate a filter for root-level folders
KnowledgeBaseFilterInput rootFilter = KnowledgeBaseGenerator.rootFoldersFilter();

// Generate a filter for articles inside a specific folder
KnowledgeBaseFilterInput articleFilter = KnowledgeBaseGenerator.articlesInFolderFilter("folder-123");

// Generate a draft article with tags under a parent folder
List<String> tagIds = List.of("tag-1", "tag-2");
CreateArticleInput draftArticle = KnowledgeBaseGenerator.draftArticle("folder-123", tagIds);

// Generate a random folder name for test setup
String folderName = KnowledgeBaseGenerator.randomFolderName();
```

> All string content (article name, body, summary) is generated via [DataFaker](https://www.datafaker.net/), ensuring unique and realistic test data on each invocation. Prefix `"Test Article"` / `"Test Folder"` allows easy identification and cleanup of generated test records.
