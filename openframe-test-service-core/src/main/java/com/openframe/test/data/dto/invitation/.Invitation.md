A data transfer object representing user invitations within the OpenFrame system, handling invitation lifecycle management and role assignments.

## Key Components

- **Core Fields**: `id`, `email` - Primary identifier and target email address
- **Role Management**: `roles` - List of `UserRole` objects defining permissions
- **Status Tracking**: `status` - Current invitation state (defaults to `PENDING`)
- **Timestamp Fields**: `expiresAt`, `createdAt`, `updatedAt` - Lifecycle tracking
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Create a new invitation with builder pattern
Invitation invitation = Invitation.builder()
    .id("inv-123")
    .email("user@example.com")
    .roles(List.of(UserRole.ADMIN, UserRole.USER))
    .expiresAt(Instant.now().plusSeconds(7 * 24 * 3600)) // 7 days
    .createdAt(Instant.now())
    .build();

// Default values are automatically applied
System.out.println(invitation.getStatus()); // PENDING
System.out.println(invitation.getRoles().isEmpty()); // false (initialized as ArrayList)

// Update invitation status
invitation.setStatus(InvitationStatus.ACCEPTED);
invitation.setUpdatedAt(Instant.now());
```

The class leverages Lombok for clean, maintainable code while providing builder pattern support for flexible object construction with sensible defaults.
