A Data Transfer Object (DTO) representing a device tag entity used in test data operations within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the tag |
| `name` | `String` | Display name of the tag |
| `description` | `String` | Optional description of the tag |
| `color` | `String` | Visual color associated with the tag |
| `organizationId` | `String` | ID of the owning organization |
| `createdAt` | `String` | Timestamp of tag creation |
| `createdBy` | `String` | Identifier of the creator |

**Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables the builder pattern for object construction
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates default and full constructors
- `@JsonIgnoreProperties(ignoreUnknown = true)` — Tolerates extra JSON fields during deserialization

## Usage Example

```java
// Using the builder pattern
Tag tag = Tag.builder()
    .id("tag-001")
    .name("Critical")
    .description("High priority devices")
    .color("#FF0000")
    .organizationId("org-123")
    .createdAt("2024-01-15T10:00:00Z")
    .createdBy("user-456")
    .build();

// Using the all-args constructor
Tag tag = new Tag(
    "tag-001", "Critical", "High priority devices",
    "#FF0000", "org-123", "2024-01-15T10:00:00Z", "user-456"
);

// Accessing fields
String tagName = tag.getName();
String orgId = tag.getOrganizationId();
```
