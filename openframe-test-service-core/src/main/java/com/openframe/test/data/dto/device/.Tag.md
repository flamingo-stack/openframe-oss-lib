A data transfer object representing device tags within the OpenFrame test suite, providing a standardized structure for tag-related operations in device management.

## Key Components

- **Tag Entity Fields**: Core tag attributes including ID, name, description, and visual color
- **Organizational Context**: Links tags to specific organizations through `organizationId`
- **Audit Trail**: Tracks creation metadata with `createdAt` and `createdBy` fields
- **Lombok Annotations**: Automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create a new tag using builder pattern
Tag deviceTag = Tag.builder()
    .id("tag-001")
    .name("Production Server")
    .description("Critical production infrastructure")
    .color("#FF5733")
    .organizationId("org-123")
    .createdAt("2024-01-15T10:30:00Z")
    .createdBy("admin@company.com")
    .build();

// Access tag properties
String tagName = deviceTag.getName();
String tagColor = deviceTag.getColor();

// Update tag description
deviceTag.setDescription("Updated production server tag");
```

This DTO enables consistent tag data handling across the OpenFrame device management system, supporting features like device categorization, filtering, and organizational separation.
