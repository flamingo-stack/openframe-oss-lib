A MongoDB document entity representing organizations within the OpenFrame test data system, containing comprehensive business information including contact details, revenue metrics, and contract lifecycle management.

## Key Components

- **Entity Fields**: Complete organization profile with `id`, `organizationId`, `name`, `category`, and `numberOfEmployees`
- **Business Metrics**: Revenue tracking through `monthlyRevenue` and employee count
- **Contact Management**: Embedded `ContactInformationDto` for communication details
- **Contract Lifecycle**: Date tracking with `contractStartDate` and `contractEndDate`
- **Audit Trail**: Timestamp fields for `createdAt`, `updatedAt`, and soft deletion via `deleted`/`deletedAt`
- **MongoDB Integration**: BSON annotations for proper ObjectId handling and document mapping

## Usage Example

```java
// Create a new organization
Organization org = Organization.builder()
    .organizationId("ORG-001")
    .name("Acme Corporation")
    .category("Technology")
    .numberOfEmployees(150)
    .websiteUrl("https://acme.com")
    .monthlyRevenue("$50000")
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2024, 12, 31))
    .isDefault(false)
    .createdAt(Instant.now())
    .deleted(false)
    .build();

// Access organization details
String companyName = org.getName();
Integer teamSize = org.getNumberOfEmployees();
LocalDate contractEnd = org.getContractEndDate();
```

The class uses Lombok annotations for automatic getter/setter generation and builder pattern support, making it ideal for test data creation and MongoDB document operations within the OpenFrame testing framework.
