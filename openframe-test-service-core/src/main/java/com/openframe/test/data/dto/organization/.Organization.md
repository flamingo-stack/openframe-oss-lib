A MongoDB document DTO representing a company or business entity within the OpenFrame platform, capturing organizational metadata including contract details, revenue, and contact information.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | MongoDB ObjectId (BSON primary key) |
| `organizationId` | `String` | Application-level organization identifier |
| `name` | `String` | Organization display name |
| `category` | `String` | Business category classification |
| `numberOfEmployees` | `Integer` | Headcount |
| `contactInformation` | `ContactInformationDto` | Nested contact details object |
| `monthlyRevenue` | `String` | Revenue figure |
| `contractStartDate` / `contractEndDate` | `LocalDate` | Contract period bounds |
| `isDefault` | `Boolean` | Flags the default organization |
| `status` / `statusChangedAt` | `String` / `Instant` | Lifecycle state tracking |
| `createdAt` / `updatedAt` | `Instant` | Audit timestamps |

**Annotations used:**
- `@BsonId` + `@BsonRepresentation(BsonType.OBJECT_ID)` — maps `id` to MongoDB `_id`
- `@JsonIgnoreProperties(ignoreUnknown = true)` — tolerates extra JSON fields during deserialization
- Lombok `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` — generates boilerplate

## Usage Example

```java
Organization org = Organization.builder()
    .organizationId("org-001")
    .name("Acme IT Solutions")
    .category("MSP")
    .numberOfEmployees(50)
    .monthlyRevenue("25000")
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2025, 1, 1))
    .isDefault(true)
    .status("ACTIVE")
    .createdAt(Instant.now())
    .build();
```
