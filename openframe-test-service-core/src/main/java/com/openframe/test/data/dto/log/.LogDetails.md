A data transfer object (DTO) representing the structure of a log entry, used for deserializing and carrying log event data in test contexts.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolEventId` | `String` | Unique identifier for the tool event |
| `eventType` | `String` | Category or type of the event |
| `ingestDay` | `String` | Date the log was ingested |
| `toolType` | `String` | Type of tool that generated the log |
| `severity` | `String` | Log severity level (e.g., INFO, ERROR) |
| `userId` | `String` | Associated user identifier |
| `deviceId` | `String` | Associated device identifier |
| `hostname` | `String` | Hostname of the source machine |
| `organizationId` | `String` | Tenant/organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `message` | `String` | Primary log message |
| `timestamp` | `String` | Time the event occurred |
| `details` | `String` | Additional event detail payload |

**Lombok annotations:** `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` auto-generate getters, setters, builder pattern, and constructors.

**`@JsonIgnoreProperties(ignoreUnknown = true)`** ensures safe deserialization even when the source payload contains extra fields.

## Usage Example

```java
// Build via Lombok builder
LogDetails log = LogDetails.builder()
    .toolEventId("evt-001")
    .eventType("ALERT")
    .severity("HIGH")
    .hostname("workstation-42")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .message("Disk usage exceeded 90%")
    .timestamp("2024-01-15T10:30:00Z")
    .build();

// Or deserialize from JSON (unknown fields safely ignored)
LogDetails log = objectMapper.readValue(jsonString, LogDetails.class);
```
