A data transfer object (DTO) representing log event details in the OpenFrame test data structure, designed to encapsulate comprehensive logging information for audit and monitoring purposes.

## Key Components

- **Core Identifiers**: `toolEventId`, `eventType`, `toolType` for event classification
- **Temporal Data**: `timestamp`, `ingestDay` for time-based tracking
- **Entity References**: `userId`, `deviceId`, `hostname`, `organizationId`, `organizationName` for context
- **Log Content**: `message`, `details`, `severity` for actual log information
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate reduction

## Usage Example

```java
// Creating a log details record
LogDetails logEntry = LogDetails.builder()
    .toolEventId("evt_123456")
    .eventType("SECURITY_ALERT")
    .toolType("FIREWALL")
    .severity("HIGH")
    .userId("user_789")
    .deviceId("dev_456")
    .hostname("prod-server-01")
    .organizationId("org_123")
    .organizationName("Acme Corp")
    .message("Suspicious network activity detected")
    .timestamp("2024-01-15T10:30:00Z")
    .ingestDay("2024-01-15")
    .details("Multiple failed login attempts from IP 192.168.1.100")
    .build();

// Accessing properties
String severity = logEntry.getSeverity();
String orgName = logEntry.getOrganizationName();
```

The class serves as a standardized container for log data across the OpenFrame testing framework, enabling consistent handling of security events, system logs, and audit trails.
