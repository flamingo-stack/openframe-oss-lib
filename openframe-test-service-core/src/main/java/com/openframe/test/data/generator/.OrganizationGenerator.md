A test data generator utility that creates mock organization data for testing purposes, combining predefined business information with faker-generated contact details.

## Key Components

- **createOrganizationRequest()** - Generates a complete organization creation request with "Tech Solutions Inc" as the base company
- **updateOrganizationRequest()** - Creates an updated version of the organization with modified details for testing updates
- **contactInformation()** - Builds contact information with conditional mailing address logic
- **address()** - Generates randomized address data using Faker library
- **contactPerson()** - Creates fake contact person details with CEO title

## Usage Example

```java
// Create organization with same mailing/physical address
CreateOrganizationRequest newOrg = OrganizationGenerator.createOrganizationRequest(true);

// Create organization with different addresses
CreateOrganizationRequest orgWithSeparateAddresses = OrganizationGenerator.createOrganizationRequest(false);

// Generate update data for testing modifications
CreateOrganizationRequest updatedOrg = OrganizationGenerator.updateOrganizationRequest(true);

// Example of generated data structure
System.out.println(newOrg.getName()); // "Tech Solutions Inc"
System.out.println(newOrg.getNumberOfEmployees()); // 25
System.out.println(newOrg.getContactInformation().getContacts().get(0).getContactName()); // Random name
```

The generator provides consistent test data for organization creation and update scenarios, with the flexibility to test both unified and separate mailing address configurations.
