Enumeration defining subscription tiers available for tenant organizations in the OpenFrame platform. This enum is used to categorize tenants based on their subscription level and associated feature access.

## Key Components

- **FREE** - Basic tier with limited features for trial users
- **BASIC** - Entry-level paid subscription with standard features
- **PREMIUM** - Advanced tier with enhanced capabilities and higher limits
- **ENTERPRISE** - Full-featured tier for large organizations with custom requirements

## Usage Example

```java
// Setting tenant plan during registration
public class TenantService {
    public Tenant createTenant(String name, TenantPlan plan) {
        Tenant tenant = new Tenant();
        tenant.setName(name);
        tenant.setPlan(plan);
        
        // Apply plan-specific configurations
        switch (plan) {
            case FREE:
                tenant.setMaxUsers(5);
                break;
            case BASIC:
                tenant.setMaxUsers(25);
                break;
            case PREMIUM:
                tenant.setMaxUsers(100);
                break;
            case ENTERPRISE:
                tenant.setMaxUsers(-1); // unlimited
                break;
        }
        
        return tenantRepository.save(tenant);
    }
    
    // Checking plan privileges
    public boolean canAccessAdvancedFeatures(Tenant tenant) {
        return tenant.getPlan() == TenantPlan.PREMIUM || 
               tenant.getPlan() == TenantPlan.ENTERPRISE;
    }
}
```
