A data transfer object (DTO) representing the input payload for deleting a knowledge base folder, with options to handle child content during deletion.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | The unique identifier of the folder to delete |
| `childrenAction` | `FolderChildrenAction` | Enum defining how to handle child items (e.g., delete or move) |
| `moveTargetFolderId` | `String` | Target folder ID when children are moved rather than deleted |

**Annotations:**
- `@Data` / `@Builder` / `@NoArgsConstructor` / `@AllArgsConstructor` — Lombok-generated boilerplate
- `@JsonInclude(NON_NULL)` — Omits null fields from JSON serialization

## Usage Example

```java
// Delete folder and move children to another folder
DeleteFolderInput input = DeleteFolderInput.builder()
    .id("folder-123")
    .childrenAction(FolderChildrenAction.MOVE)
    .moveTargetFolderId("folder-456")
    .build();

// Delete folder and remove all children
DeleteFolderInput input = DeleteFolderInput.builder()
    .id("folder-123")
    .childrenAction(FolderChildrenAction.DELETE)
    .build();
```

> **Note:** `moveTargetFolderId` is only required when `childrenAction` is set to `MOVE`. Since `@JsonInclude(NON_NULL)` is applied, omitted fields will not appear in the serialized JSON payload.
