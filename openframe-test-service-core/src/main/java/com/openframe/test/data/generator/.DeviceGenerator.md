Utility class for generating `DeviceFilterInput` test data objects and extracting tool-specific agent IDs from `Machine` device instances.

## Key Components

| Method | Description |
|--------|-------------|
| `onlineDevicesFilter()` | Returns a filter for `ONLINE` devices |
| `offlineDevicesFilter()` | Returns a filter for `OFFLINE` devices |
| `osDevicesFilter(String os)` | Returns a filter scoped to a specific OS type |
| `listedStatusesDevicesFilter()` | Returns a filter covering all 7 known device statuses |
| `statDevicesFilter()` | Returns a filter for the 5 operational statuses (excludes `DECOMMISSIONED`/`PENDING`) |
| `osAndStatusDevicesFilter(String os, DeviceStatus...)` | Returns a combined OS + status filter |
| `statusDevicesFilter(DeviceStatus...)` | Base builder method for any status combination |
| `getFleetId(Machine)` | Extracts the `FLEET_MDM` agent tool ID from a device's tool connections |
| `getMeshId(Machine)` | Extracts the `MESHCENTRAL` agent tool ID |

## Usage Example

```java
// Build a filter for online Windows devices
DeviceFilterInput filter = DeviceGenerator.osAndStatusDevicesFilter(
    "WINDOWS",
    DeviceStatus.ONLINE
);

// Build a filter for all operational statuses
DeviceFilterInput statFilter = DeviceGenerator.statDevicesFilter();

// Extract tool-specific IDs from a device
Machine device = ...; // retrieved from API
String fleetId    = DeviceGenerator.getFleetId(device);
String meshId     = DeviceGenerator.getMeshId(device);
```
