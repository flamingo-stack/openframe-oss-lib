Data Transfer Object for encapsulating an organization's contact information, including contact persons and address details.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `contacts` | `List<ContactPersonDto>` | List of contact persons associated with the organization |
| `physicalAddress` | `AddressDto` | Physical/street address of the organization |
| `mailingAddress` | `AddressDto` | Mailing address of the organization |
| `mailingAddressSameAsPhysical` | `Boolean` | Flag indicating if mailing address matches physical address |

## Annotations

- **`@Data`** – Generates getters, setters, `equals`, `hashCode`, and `toString`
- **`@Builder`** – Enables the builder pattern for object construction
- **`@NoArgsConstructor` / `@AllArgsConstructor`** – Generates default and full constructors
- **`@JsonIgnoreProperties(ignoreUnknown = true)`** – Silently ignores unmapped JSON fields during deserialization

## Usage Example

```java
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("Jane Doe")
            .email("jane.doe@example.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("Austin")
        .state("TX")
        .build()
    )
    .mailingAddressSameAsPhysical(true)
    .build();
```
