A Data Transfer Object (DTO) that encapsulates contact information for an organization, including contact persons and address details with flexible physical/mailing address configuration.

## Key Components

- **contacts** - List of `ContactPersonDto` objects representing individuals associated with the organization
- **physicalAddress** - `AddressDto` representing the organization's physical location
- **mailingAddress** - `AddressDto` representing the organization's mailing address
- **mailingAddressSameAsPhysical** - Boolean flag indicating whether mailing and physical addresses are identical

## Usage Example

```java
// Create contact information for an organization
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("John Doe")
            .email("john@company.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("New York")
        .build())
    .mailingAddressSameAsPhysical(true)
    .build();

// Check if separate mailing address is needed
if (!contactInfo.getMailingAddressSameAsPhysical()) {
    contactInfo.setMailingAddress(separateMailingAddress);
}
```

The DTO uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern implementation, making it ideal for transferring contact data between application layers.
