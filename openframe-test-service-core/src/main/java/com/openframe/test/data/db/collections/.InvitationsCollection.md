A MongoDB collection helper class that provides query methods for invitation records, supporting lookups by email address and invitation status.

## Key Components

- **findInvitation(String email)** - Finds the first invitation record matching the specified email address
- **findInvitation(InvitationStatus status)** - Finds the first invitation record with the given status (e.g., PENDING, ACCEPTED, REJECTED)

## Usage Example

```java
// Find invitation by email
Invitation userInvitation = InvitationsCollection.findInvitation("user@example.com");

// Find invitation by status
Invitation pendingInvitation = InvitationsCollection.findInvitation(InvitationStatus.PENDING);

// Check if invitation exists
if (userInvitation != null) {
    System.out.println("Found invitation for: " + userInvitation.getEmail());
} else {
    System.out.println("No invitation found for the specified email");
}
```

The class uses MongoDB's `Filters.eq()` for exact matching and returns the first matching document. Both methods may return `null` if no matching invitation is found. This collection helper is typically used in test scenarios to verify invitation creation and status changes within the OpenFrame platform.
