Provides MongoDB data access methods for querying the `invitations` collection, supporting lookups by email address or invitation status.

## Key Components

- **`findInvitation(String email)`** — Retrieves the first `Invitation` document matching the given email address.
- **`findInvitation(InvitationStatus status)`** — Retrieves the first `Invitation` document matching the given `InvitationStatus` enum value.

Both methods automatically close the MongoDB connection after each query via `closeConnection()`.

## Usage Example

```java
// Find an invitation by email
Invitation invite = InvitationsCollection.findInvitation("user@example.com");

// Find the first invitation with a specific status
Invitation pendingInvite = InvitationsCollection.findInvitation(InvitationStatus.PENDING);

if (pendingInvite != null) {
    System.out.println("Found invitation for: " + pendingInvite.getEmail());
}
```

> **Note:** Both methods return `null` if no matching document is found. Always perform a null check before accessing the returned `Invitation` object.
