Data Transfer Object (DTO) representing a scheduled script task, used for deserializing script schedule data from JSON API responses.

## Key Components

| Field | Type | JSON Property | Description |
|-------|------|--------------|-------------|
| `id` | `Integer` | `id` | Unique schedule identifier |
| `name` | `String` | `name` | Schedule display name |
| `taskType` | `String` | `task_type` | Type of scheduled task |
| `runTimeDate` | `String` | `run_time_date` | Scheduled execution date/time |
| `enabled` | `Boolean` | `enabled` | Whether the schedule is active |
| `taskSupportedPlatforms` | `List<String>` | `task_supported_platforms` | Platforms the task supports |
| `actionsCount` | `Integer` | `actions_count` | Number of actions in the schedule |
| `agentsCount` | `Integer` | `agents_count` | Number of agents assigned |

## Annotations

- **`@JsonIgnoreProperties(ignoreUnknown = true)`** — Tolerates extra fields in API responses without throwing errors
- **`@JsonProperty`** — Maps snake_case JSON fields to camelCase Java fields
- **Lombok** (`@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor`) — Auto-generates getters, setters, builder pattern, and constructors

## Usage Example

```java
// Deserialize from JSON response
ScriptSchedule schedule = objectMapper.readValue(jsonResponse, ScriptSchedule.class);

// Build programmatically using Lombok builder
ScriptSchedule schedule = ScriptSchedule.builder()
    .id(42)
    .name("Nightly Patch Run")
    .taskType("patch")
    .runTimeDate("2024-01-15T02:00:00Z")
    .enabled(true)
    .taskSupportedPlatforms(List.of("windows", "linux"))
    .actionsCount(5)
    .agentsCount(20)
    .build();
```
