A utility class that generates authentication-related test data and components for OAuth2 PKCE (Proof Key for Code Exchange) flows and password reset operations.

## Key Components

**Static Factory Methods:**
- `resetConfirmRequest(String token)` - Creates password reset confirmation request objects
- `invalidTokenResponse()` - Generates standardized invalid token error responses
- `clearedCookies()` - Returns map of cleared authentication cookies
- `generateAuthParts(User user, Tenant tenant)` - Creates complete OAuth2 PKCE authentication components

**Private Helper Methods:**
- `generateCodeVerifier()` - Creates cryptographically secure PKCE code verifier
- `generateCodeChallenge(String codeVerifier)` - Generates SHA-256 based code challenge
- `generateState()` - Produces random state parameter for OAuth2 security

## Usage Example

```java
// Generate OAuth2 PKCE authentication components
User testUser = UserGenerator.createTestUser();
Tenant testTenant = TenantGenerator.createTestTenant();
AuthParts authParts = AuthGenerator.generateAuthParts(testUser, testTenant);

// Create password reset request
String resetToken = "abc123-reset-token";
ResetConfirmRequest resetRequest = AuthGenerator.resetConfirmRequest(resetToken);

// Generate error response for invalid tokens
ErrorResponse errorResponse = AuthGenerator.invalidTokenResponse();

// Clear authentication cookies after logout
Map<String, String> clearedCookies = AuthGenerator.clearedCookies();
```

The class implements OAuth2 PKCE security standards using SHA-256 hashing and secure random number generation, making it suitable for testing authentication flows that require cryptographically secure components.
