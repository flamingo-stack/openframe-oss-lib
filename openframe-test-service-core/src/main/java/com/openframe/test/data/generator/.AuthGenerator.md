Utility class for generating authentication test data, including PKCE OAuth2 flow components, password reset requests, and cookie state helpers.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `NEW_PASSWORD` | Constant | Default password used in reset confirmation tests (`Password124!`) |
| `resetConfirmRequest(String token)` | Static Method | Builds a `ResetConfirmRequest` with the given token and default new password |
| `invalidTokenResponse()` | Static Method | Returns a pre-built `ErrorResponse` representing an invalid/expired reset token |
| `clearedCookies()` | Static Method | Returns a map of emptied auth cookies (`access_token`, `refresh_token`, `JSESSIONID`) |
| `generateAuthParts(User, String tenantId)` | Static Method | Assembles a full `AuthParts` object with PKCE verifier/challenge and random state |
| `generateCodeVerifier()` | Private | Generates a cryptographically secure 32-byte Base64URL-encoded PKCE code verifier |
| `generateCodeChallenge(String)` | Private | Derives a SHA-256 Base64URL-encoded PKCE code challenge from the verifier |
| `generateState()` | Private | Generates a cryptographically secure 16-byte Base64URL-encoded OAuth2 state parameter |

## Usage Example

```java
// Generate full PKCE auth parts for a test user
User testUser = UserGenerator.defaultUser();
AuthParts authParts = AuthGenerator.generateAuthParts(testUser, "tenant-123");

// Use in OAuth2 authorization request
String codeChallenge = authParts.getCodeChallenge(); // SHA-256 of verifier
String state = authParts.getState();                 // CSRF protection token

// Build a password reset confirmation
ResetConfirmRequest resetRequest = AuthGenerator.resetConfirmRequest("some-reset-token");

// Assert logout clears cookies
Map<String, String> expected = AuthGenerator.clearedCookies();
assertThat(responseCookies).containsAllEntriesOf(expected);
```

> **Note:** PKCE values are generated using `SecureRandom` and comply with [RFC 7636](https://datatracker.ietf.org/doc/html/rfc7636) — safe for use in OAuth2 authorization code flow tests.
