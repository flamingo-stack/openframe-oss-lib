DTO representing a Fleet/MDM policy object with its metadata, author information, host compliance counts, and associated hosts — used for deserializing policy API responses in test data pipelines.

## Key Components

### `Policy` (root class)
| Field | Type | Description |
|---|---|---|
| `id` | `Integer` | Unique policy identifier |
| `name` | `String` | Policy display name |
| `query` | `String` | SQL query defining the policy check |
| `critical` | `Boolean` | Flags policy as critical severity |
| `authorId` / `authorName` / `authorEmail` | `Integer` / `String` / `String` | Policy author details |
| `teamId` | `Integer` | Associated team scope |
| `platform` | `String` | Target platform (e.g., `darwin`, `windows`) |
| `hostsIncludeAny` | `List<Host>` | Hosts targeted by this policy |
| `calendarEventsEnabled` | `Boolean` | Whether calendar event integration is active |
| `conditionalAccessEnabled` | `Boolean` | Whether conditional access enforcement is active |
| `passingHostCount` / `failingHostCount` | `Integer` | Compliance host counts |
| `createdAt` / `updatedAt` / `hostCountUpdatedAt` | `String` | Audit timestamps |

### `Policy.Host` (nested static class)
Lightweight host reference embedded in policy scope.

| Field | Type | Description |
|---|---|---|
| `id` | `Integer` | Host identifier |
| `hostname` | `String` | Host display name |

## Usage Example

```java
// Deserialize from API response JSON
ObjectMapper mapper = new ObjectMapper();
Policy policy = mapper.readValue(jsonResponse, Policy.class);

System.out.println(policy.getName());           // "Antivirus enabled"
System.out.println(policy.getFailingHostCount()); // 3

// Build programmatically in tests
Policy policy = Policy.builder()
    .id(42)
    .name("Disk Encryption")
    .critical(true)
    .platform("darwin")
    .passingHostCount(10)
    .failingHostCount(2)
    .hostsIncludeAny(List.of(
        Policy.Host.builder().id(1).hostname("mac-01.local").build()
    ))
    .build();
```
