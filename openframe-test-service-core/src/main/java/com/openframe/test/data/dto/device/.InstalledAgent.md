A data transfer object (DTO) representing an installed agent on a device within the OpenFrame platform, containing agent metadata and tracking information.

## Key Components

- **id** - Unique identifier for the installed agent instance
- **machineId** - Identifier linking the agent to its host machine/device
- **agentType** - Type classification of the agent (e.g., monitoring, security, etc.)
- **version** - Current version number of the installed agent
- **createdAt** - Timestamp when the agent was first installed
- **updatedAt** - Timestamp of the last agent update or modification

## Usage Example

```java
// Create a new installed agent record
InstalledAgent agent = InstalledAgent.builder()
    .id("agent-123")
    .machineId("machine-456")
    .agentType("monitoring")
    .version("2.1.0")
    .createdAt("2024-01-15T10:30:00Z")
    .updatedAt("2024-01-15T10:30:00Z")
    .build();

// Update agent version
InstalledAgent updatedAgent = agent.toBuilder()
    .version("2.1.1")
    .updatedAt("2024-01-20T14:45:00Z")
    .build();

// Access agent properties
String agentInfo = String.format("Agent %s v%s on machine %s", 
    agent.getAgentType(), agent.getVersion(), agent.getMachineId());
```

This DTO is typically used for tracking agent deployments, version management, and device inventory within the MSP platform's device management system.
