A Data Transfer Object (DTO) representing an installed monitoring/management agent on a device within the OpenFrame platform.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the installed agent record |
| `machineId` | `String` | Identifier of the machine/device the agent is installed on |
| `agentType` | `String` | Type/category of the agent (e.g., RMM, monitoring) |
| `version` | `String` | Installed agent version string |
| `createdAt` | `String` | Timestamp of when the agent record was created |
| `updatedAt` | `String` | Timestamp of the last update to the agent record |

## Annotations

- **`@Data`** — Generates getters, setters, `equals`, `hashCode`, and `toString`
- **`@Builder`** — Enables fluent builder pattern for object construction
- **`@NoArgsConstructor` / `@AllArgsConstructor`** — Generates both default and full constructors
- **`@JsonIgnoreProperties(ignoreUnknown = true)`** — Safely deserializes JSON payloads with extra fields

## Usage Example

```java
// Build via Lombok builder
InstalledAgent agent = InstalledAgent.builder()
    .id("agent-001")
    .machineId("machine-abc")
    .agentType("RMM")
    .version("3.2.1")
    .createdAt("2024-01-15T10:00:00Z")
    .updatedAt("2024-06-01T08:30:00Z")
    .build();

// Deserialize from API response
InstalledAgent agent = objectMapper.readValue(jsonResponse, InstalledAgent.class);
```

> This DTO is part of the `com.openframe.test.data.dto.device` package, used in test layers to validate device agent data returned by the OpenFrame API.
