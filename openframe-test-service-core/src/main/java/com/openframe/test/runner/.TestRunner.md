A JUnit 5 test execution utility that programmatically runs tests from a specified package with optional tag-based filtering and returns execution summaries.

## Key Components

- **TestRunner class**: Main test execution orchestrator
- **runTests(String... tags)**: Executes tests from the configured package with tag filtering
- **runTests(LauncherDiscoveryRequest)**: Executes tests using a custom discovery request
- **LauncherDiscoveryRequest**: JUnit 5 test discovery configuration
- **SummaryGeneratingListener**: Collects test execution statistics and results

## Usage Example

```java
// Create a test runner for a specific package
TestRunner runner = new TestRunner("com.openframe.api.tests");

// Run all tests with specific tags
TestExecutionSummary summary = runner.runTests("integration", "smoke");

// Check results
System.out.println("Tests run: " + summary.getTestsFoundCount());
System.out.println("Failures: " + summary.getTestsFailedCount());
System.out.println("Success rate: " + 
    (summary.getTestsSucceededCount() * 100.0 / summary.getTestsFoundCount()) + "%");

// Run with custom discovery request
LauncherDiscoveryRequest customRequest = LauncherDiscoveryRequestBuilder.request()
    .selectors(selectPackage("com.openframe.core"))
    .filters(includeTags("unit"))
    .build();
    
TestExecutionSummary customSummary = runner.runTests(customRequest);
```

The runner provides programmatic control over test execution, making it useful for custom test orchestration, CI/CD integration, or building test reporting dashboards.
