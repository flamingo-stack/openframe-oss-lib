A simple data model representing a test entity with a display name and associated tags, used within the OpenFrame test runner infrastructure.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `displayName` | `String` | Human-readable name for the test |
| `tags` | `Set<String>` | Collection of tags used to categorize or filter tests |

**Lombok Annotations:**
- `@Data` — Generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — Enables fluent builder pattern instantiation
- `@NoArgsConstructor` / `@AllArgsConstructor` — Generates both default and full constructors

## Usage Example

```java
// Using the builder pattern
Test test = Test.builder()
    .displayName("Network Connectivity Check")
    .tags(Set.of("network", "smoke", "critical"))
    .build();

// Using the all-args constructor
Test test2 = new Test("Disk Health Check", Set.of("storage", "regression"));

// Accessing fields (via Lombok @Data)
String name = test.getDisplayName();
Set<String> tags = test.getTags();
```
