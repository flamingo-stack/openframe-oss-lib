A utility class providing helper methods for string manipulation and URL processing operations within the OpenFrame testing framework.

## Key Components

- **`extractQueryParam(String url, String paramName)`** - Static method that extracts a specific query parameter value from a URL string

## Usage Example

```java
// Extract parameter from URL
String url = "https://api.example.com/search?query=openframe&limit=10&sort=date";
String queryValue = StringUtils.extractQueryParam(url, "query");
// Returns: "openframe"

String limitValue = StringUtils.extractQueryParam(url, "limit");
// Returns: "10"

// Handle missing parameters
String missingParam = StringUtils.extractQueryParam(url, "filter");
// Returns: null

// Handle malformed URLs
String invalidUrl = "https://api.example.com/search";
String result = StringUtils.extractQueryParam(invalidUrl, "query");
// Returns: null (no query string present)
```

The utility handles edge cases including null URLs, URLs without query strings, and malformed parameter pairs. It splits query parameters on "&" delimiters and key-value pairs on "=" delimiters, returning the first matching parameter value or null if not found.
