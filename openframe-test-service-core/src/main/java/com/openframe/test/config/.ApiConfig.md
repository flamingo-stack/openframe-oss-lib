A configuration utility class that provides dynamic API base URL resolution for test environments, supporting both system properties and environment variables with fallback to localhost.

## Key Components

- **DEFAULT_BASE_URL** - Default localhost URL constant (`https://localhost/`)
- **GRAPHQL** - GraphQL endpoint path constant (`api/graphql`)
- **getBaseUrl()** - Static method that resolves the base URL from multiple sources with priority order
- **baseUrl** - Private static field for caching the resolved URL

## Configuration Priority

The URL resolution follows this priority order:
1. System property: `api.base.url`
2. Environment variable: `API_BASE_URL`
3. Default fallback: `https://localhost/`

## Usage Example

```java
// Get the configured base URL
String apiBase = ApiConfig.getBaseUrl();

// Build GraphQL endpoint
String graphqlEndpoint = ApiConfig.getBaseUrl() + ApiConfig.GRAPHQL;

// Set via system property (highest priority)
System.setProperty("api.base.url", "https://api.staging.openframe.ai/");

// Set via environment variable
// export API_BASE_URL=https://api.prod.openframe.ai/

// URL automatically normalized with trailing slash
ApiConfig.getBaseUrl(); // Returns "https://api.staging.openframe.ai/"
```

The class automatically ensures URLs end with a trailing slash and caches the resolved value for performance. Debug logging is enabled via Lombok's `@Slf4j` annotation.
