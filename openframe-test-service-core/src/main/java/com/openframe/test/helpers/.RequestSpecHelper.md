Utility class for building and configuring RestAssured `RequestSpecification` instances used across OpenFrame API tests, with thread-safe base URL and cookie management, retry-on-connect logic, and SLF4J-bridged request logging.

## Key Components

| Member | Description |
|--------|-------------|
| `setBaseUrl(String)` / `getBaseUrl()` | Thread-local base URL override, falls back to `EnvironmentConfig.getBaseUrl()` |
| `setCookies(Map)` / `getCookies()` / `clearCookies()` | Thread-local cookie store; auto-fetches via `AuthHelper` if unset |
| `getAuthorizedSpec()` | Returns a `RequestSpecification` pre-loaded with session cookies |
| `getUnAuthorizedSpec()` | Returns a `RequestSpecification` with no authentication |
| `getAuthFlowRequestSpec()` | Spec targeting the auth/identity base URL (e.g. for login flows) |
| `graphqlSuccess()` | `ResponseSpecification` asserting HTTP 200 and no `errors` field in the body |
| `retryingHttpClientConfig()` | Configures a 10 s connect / 32 s socket timeout with up to 2 connect retries |
| `createRetryingHttpClient()` | Retries only on `ConnectException` / `ConnectTimeoutException` — not on read timeouts |
| `Slf4jOutputStream` (inner) | Bridges RestAssured's `PrintStream` logging to SLF4J line-by-line |

## Usage Example

```java
// Authorized JSON request
given()
    .spec(RequestSpecHelper.getAuthorizedSpec())
    .body(payload)
.when()
    .post("/api/resource")
.then()
    .statusCode(201);

// GraphQL request with success assertion
given()
    .spec(RequestSpecHelper.getAuthorizedSpec())
    .body(graphqlQuery)
.when()
    .post("/graphql")
.then()
    .spec(RequestSpecHelper.graphqlSuccess());

// Override base URL per-thread (e.g. in parallel test suites)
RequestSpecHelper.setBaseUrl("https://staging.openframe.ai");

// Auth flow (login endpoint)
given()
    .spec(RequestSpecHelper.getAuthFlowRequestSpec())
    .formParam("username", user)
    .formParam("password", pass)
.when()
    .post("/auth/login");
```

> **Threading:** `baseUrl` and `cookies` are stored in `ThreadLocal` fields, making this class safe for parallel test execution without cross-thread contamination.
