Utility class for building and managing REST Assured `RequestSpecification` objects in the OpenFrame test framework, providing thread-safe session management, SSL configuration, and SLF4J-integrated request logging.

## Key Components

| Method/Field | Description |
|---|---|
| `getAuthorizedSpec()` | Builds a request spec with session cookies attached for authenticated API calls |
| `getUnAuthorizedSpec()` | Builds a base request spec without authentication cookies |
| `getAuthFlowRequestSpec()` | Builds a request spec targeting the auth URL (for login/token flows) |
| `graphqlSuccess()` | Returns a `ResponseSpecification` asserting HTTP 200 with no GraphQL `errors` field |
| `setCookies()` / `getCookies()` | Thread-local cookie store; lazily fetches from `AuthHelper` if unset |
| `setBaseUrl()` / `getBaseUrl()` | Thread-local base URL override; falls back to `EnvironmentConfig.getBaseUrl()` |
| `setEnableLogging()` | Toggles request logging via the SLF4J stream |
| `Slf4jOutputStream` | Inner class that bridges REST Assured's `PrintStream` output to SLF4J `Logger` |

All specs are pre-configured with a **32-second timeout**, relaxed HTTPS validation, and JSON content type.

## Usage Example

```java
// Authenticated request
given()
    .spec(RequestSpecHelper.getAuthorizedSpec())
    .body(graphqlQuery)
.when()
    .post("/graphql")
.then()
    .spec(RequestSpecHelper.graphqlSuccess());

// Override base URL per thread (e.g., in parallel tests)
RequestSpecHelper.setBaseUrl("https://staging.openframe.ai");

// Unauthenticated request
given()
    .spec(RequestSpecHelper.getUnAuthorizedSpec())
.when()
    .get("/public/health");

// Auth flow (login endpoint)
given()
    .spec(RequestSpecHelper.getAuthFlowRequestSpec())
    .formParam("username", user)
    .formParam("password", pass)
.when()
    .post("/auth/login");
```
