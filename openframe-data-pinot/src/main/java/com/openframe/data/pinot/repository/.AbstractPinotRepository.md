Base abstract class for Apache Pinot-backed repositories in the OpenFrame data layer. Centralizes query execution, `ResultSet` iteration, and error handling so concrete subclasses focus solely on domain-specific row mapping logic.

## Key Components

| Method | Description |
|--------|-------------|
| `executeQuery(query, mapper)` | Generic query executor; accepts a curried mapper `ResultSet → (rowIndex → T)` and streams results into a `List<T>` |
| `executeSingleColumnQuery(query)` | Convenience wrapper returning non-null, non-empty strings from a single-column result |
| `executeKeyCountQuery(query)` | Convenience wrapper for `(value, count)` two-column queries; returns an ordered `LinkedHashMap<String, Integer>` |
| `executeCountQuery(query)` | Executes a `COUNT(*)` style query and returns the integer result |
| `buildColumnIndexMap(resultSet)` | Builds a `columnName → index` lookup map, enabling name-based column access in row mappers |

All methods wrap exceptions in `PinotQueryException` and log both debug (query text) and error (failure) events via SLF4J.

## Usage Example

```java
@Repository
public class DeviceRepository extends AbstractPinotRepository {

    public DeviceRepository(Connection pinotConnection) {
        super(pinotConnection);
    }

    public List<Device> findByTenant(String tenantId) {
        String query = "SELECT device_id, hostname, status "
                     + "FROM devices WHERE tenant_id = '" + tenantId + "'";

        return executeQuery(query, resultSet -> {
            // Build column map once per ResultSet
            Map<String, Integer> cols = buildColumnIndexMap(resultSet);
            return rowIndex -> new Device(
                resultSet.getString(rowIndex, cols.get("device_id")),
                resultSet.getString(rowIndex, cols.get("hostname")),
                resultSet.getString(rowIndex, cols.get("status"))
            );
        });
    }

    public int countByTenant(String tenantId) {
        String query = "SELECT COUNT(*) FROM devices WHERE tenant_id = '" + tenantId + "'";
        return executeCountQuery(query);
    }

    public Map<String, Integer> statusBreakdown(String tenantId) {
        String query = "SELECT status, COUNT(*) FROM devices "
                     + "WHERE tenant_id = '" + tenantId + "' GROUP BY status";
        return executeKeyCountQuery(query);
    }
}
```
