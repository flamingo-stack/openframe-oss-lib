Apache Pinot repository implementation for querying IT event logs within the OpenFrame platform, supporting multi-tenant filtering, cursor-based pagination, full-text search, and dynamic facet/option retrieval.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `SORTABLE_COLUMNS` | `List<String>` | Allowlist of fields that may be used as sort keys |
| `DEFAULT_SORT_COLUMN` | `String` | Falls back to `eventTimestamp` when no sort is specified |
| `PRIMARY_KEY_FIELD` | `String` | `toolEventId` — used as tiebreaker in cursor pagination |
| `logsTable` | `String` | Configurable via `pinot.tables.logs.name` (default: `logs`) |
| `findLogs(...)` | Method | Returns paginated `LogProjection` list with optional filters |
| `searchLogs(...)` | Method | Extends `findLogs` with relevance-based full-text search |
| `getEventTypeOptions(...)` | Method | Returns distinct `eventType` values matching current filters |
| `getSeverityOptions(...)` | Method | Returns distinct `severity` values matching current filters |
| `getToolTypeOptions(...)` | Method | Returns distinct `toolType` values matching current filters |
| `getAvailableDateRanges(...)` | Method | Returns distinct `ingestDay` values for date picker population |
| `getOrganizationOptions(...)` | Method | Returns `OrganizationOption` pairs (`id` + `name`), nulls filtered |
| `isSortableField(String)` | Method | Validates a field against the sortable column allowlist |
| `executeLogQuery(String)` | Private | Maps raw Pinot result rows to `LogProjection` objects |

## Usage Example

```java
// Paginated log retrieval with filters
List<LogProjection> logs = pinotClientLogRepository.findLogs(
    "tenant-123",
    LocalDate.of(2024, 1, 1),
    LocalDate.of(2024, 1, 31),
    List.of("NinjaRMM", "ConnectWise"),  // toolTypes
    List.of("ALERT"),                    // eventTypes
    List.of("HIGH", "CRITICAL"),         // severities
    List.of("org-456"),                  // organizationIds
    "device-789",                        // deviceId
    null,                                // cursor (first page)
    50,                                  // limit
    "eventTimestamp",                    // sortField
    "DESC"                               // sortDirection
);

// Full-text search across log summaries
List<LogProjection> results = pinotClientLogRepository.searchLogs(
    "tenant-123",
    LocalDate.now().minusDays(7),
    LocalDate.now(),
    List.of(), List.of(), List.of(), List.of(),
    null, "disk failure", null, 25, "eventTimestamp", "DESC"
);

// Populate filter dropdowns
List<String> severities = pinotClientLogRepository.getSeverityOptions(
    "tenant-123", startDate, endDate,
    List.of("NinjaRMM"), List.of(), List.of(), List.of()
);
```
