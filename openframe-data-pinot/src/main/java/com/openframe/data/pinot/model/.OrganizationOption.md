A simple data model representing a selectable organization option, typically used for dropdowns or filtered queries within the OpenFrame/Pinot data layer.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `id` | `String` | Unique identifier for the organization |
| `name` | `String` | Human-readable display name of the organization |

**Lombok Annotations:**
- `@Data` — generates getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — enables the builder pattern for construction
- `@AllArgsConstructor` / `@NoArgsConstructor` — generates both full and no-arg constructors

## Usage Example

```java
// Builder pattern
OrganizationOption option = OrganizationOption.builder()
    .id("org-123")
    .name("Acme IT Services")
    .build();

// No-arg constructor with setters
OrganizationOption option = new OrganizationOption();
option.setId("org-456");
option.setName("Blue Sky MSP");

// All-args constructor
OrganizationOption option = new OrganizationOption("org-789", "TechCorp");
```
