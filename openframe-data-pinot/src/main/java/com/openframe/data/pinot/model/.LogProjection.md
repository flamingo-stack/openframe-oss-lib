A Lombok-annotated projection model representing a flattened view of a log event retrieved from Apache Pinot, used within the OpenFrame data pipeline.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `toolEventId` | `String` | Unique identifier for the tool event |
| `toolType` | `String` | Type of tool that generated the event |
| `eventType` | `String` | Classification of the event |
| `severity` | `String` | Severity level of the log entry |
| `userId` / `deviceId` | `String` | Actor identifiers associated with the event |
| `hostname` | `String` | Source host of the event |
| `organizationId` / `organizationName` | `String` | Tenant/org context |
| `summary` | `String` | Human-readable event description |
| `eventTimestamp` | `Instant` | UTC timestamp of when the event occurred |
| `ingestDay` | `String` | Partitioning field used for Pinot segment routing |

Lombok annotations (`@Data`, `@Builder`, `@AllArgsConstructor`, `@NoArgsConstructor`) auto-generate getters, setters, builder pattern, and constructors.

## Usage Example

```java
// Build a projection manually (e.g. in tests or mappers)
LogProjection projection = LogProjection.builder()
    .toolEventId("evt-001")
    .toolType("RMM")
    .eventType("ALERT")
    .severity("HIGH")
    .userId("usr-42")
    .deviceId("dev-99")
    .hostname("workstation-01")
    .organizationId("org-123")
    .organizationName("Acme Corp")
    .summary("Disk usage exceeded 90%")
    .eventTimestamp(Instant.now())
    .ingestDay("2025-01-15")
    .build();

String severity = projection.getSeverity(); // "HIGH"
```
