Custom unchecked exception for signaling failures during Apache Pinot query execution within the OpenFrame data layer.

## Key Components

| Component | Description |
|-----------|-------------|
| `PinotQueryException(String message)` | Constructs exception with a descriptive error message |
| `PinotQueryException(String message, Throwable cause)` | Constructs exception wrapping an underlying cause for full stack trace propagation |

## Usage Example

```java
// Throw with message only
throw new PinotQueryException("Failed to execute Pinot query: invalid table name");

// Throw wrapping a caught exception
try {
    pinotConnection.executeQuery(sql);
} catch (Exception e) {
    throw new PinotQueryException("Pinot query execution failed", e);
}

// Catch in calling code
try {
    repository.findMetrics(filter);
} catch (PinotQueryException e) {
    log.error("Query layer error: {}", e.getMessage(), e.getCause());
}
```
