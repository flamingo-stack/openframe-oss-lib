Unit test suite for `PinotClientDeviceRepository`, validating query construction, faceted filtering logic, result mapping, tenant isolation, and SQL injection protection against an Apache Pinot data source.

## Key Components

| Nested Class | Responsibility |
|---|---|
| `TenantIdValidation` | Asserts null/blank tenant throws `PinotQueryException`; verifies all facet queries embed the `tenantId` filter |
| `FacetQueryStructure` | Validates generated SQL structure (SELECT fields, GROUP BY, ORDER BY, DELETED exclusion) |
| `ExcludeFieldBehavior` | Confirms faceted search skips filtering on its own dimension while applying all others |
| `CountAndMapping` | Verifies `getFilteredDeviceCount` reads the correct cell and facet methods map rows to `Map<String, Integer>` |
| `SqlInjection` | Ensures single quotes in `tenantId` are escaped before interpolation into SQL |

## Usage Example

```java
// Run the full test class with Maven
// mvn test -Dtest=PinotClientDeviceRepositoryTest

// Example: verifying facet query excludes its own dimension
@Test
void deviceTypeFacetSkipsDeviceTypeFilter() {
    whenQueryReturnsEmptyRows();
    repository.getDeviceTypeFilterOptions(
        TENANT_ID,
        List.of(),
        List.of("LAPTOP"),   // user-selected deviceType
        List.of("WINDOWS"),  // user-selected osType
        List.of(), List.of(), List.of()
    );
    String query = captureQuery();
    assertTrue(query.contains("osType = 'WINDOWS'"));   // applied
    assertFalse(query.contains("deviceType = 'LAPTOP'")); // excluded (own dimension)
}
```

## Design Notes

- Uses **Mockito** (`@Mock`, `ArgumentCaptor`) to intercept and inspect raw SQL strings sent to the Pinot `Connection`
- Helper `whenQueryReturnsEmptyRows()` stubs the full `Connection → ResultSetGroup → ResultSet` call chain for structure-only tests
- `setField()` uses reflection to inject the `devicesTable` property, avoiding a full Spring context
- The `DELETED` exclusion (`status != 'DELETED'`) is treated as a hard system constraint separate from user-supplied status filters
