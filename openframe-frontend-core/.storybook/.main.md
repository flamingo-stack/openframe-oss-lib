This file configures Storybook for a Next.js application using Vite as the build tool, with telemetry disabled for privacy.

## Key Components

- **config**: Main Storybook configuration object of type `StorybookConfig`
- **stories**: Pattern matching for story files across the src directory
- **framework**: Specifies Next.js + Vite integration
- **core.disableTelemetry**: Privacy setting to disable analytics collection

## Configuration Details

The configuration sets up:

- **Story Discovery**: Automatically finds story files with extensions `.stories.js|jsx|mjs|ts|tsx` in the `src` directory and subdirectories
- **Framework Integration**: Uses `@storybook/nextjs-vite` for optimal Next.js component rendering with Vite's fast build system
- **No Addons**: Clean setup without additional Storybook plugins
- **Privacy**: Telemetry disabled to prevent usage data collection

## Usage Example

```typescript
// This configuration enables Storybook to automatically discover files like:
// src/components/Button/Button.stories.tsx
// src/pages/Dashboard/Dashboard.stories.ts
// src/utils/helpers/formatters.stories.jsx

// Example story file that would be discovered:
export default {
  title: 'Components/Button',
  component: Button,
};

export const Primary = {
  args: {
    primary: true,
    label: 'Button',
  },
};
```

This minimal configuration provides a solid foundation for component development and testing in a Next.js project using Storybook's component-driven development workflow.
