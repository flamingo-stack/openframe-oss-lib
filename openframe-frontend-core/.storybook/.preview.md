Storybook preview configuration file that sets up global parameters and styling for the component story environment.

## Key Components

- **preview**: Main configuration object exported for Storybook
- **parameters.controls.matchers**: Pattern-based control type detection for story args
- **Global CSS import**: Loads the main application stylesheet for consistent styling

## Usage Example

This file is automatically loaded by Storybook and doesn't require direct imports. The configuration enables:

```typescript
// Automatic color picker for props matching the pattern
export const MyStory = {
  args: {
    backgroundColor: '#ff0000', // Will show color picker
    textColor: '#000000',       // Will show color picker  
    createdDate: new Date(),    // Will show date picker
  }
}
```

The matchers automatically detect:
- **Color controls**: Any prop ending with "background" or "color" 
- **Date controls**: Any prop ending with "Date"

This eliminates the need to manually specify control types in individual stories, providing a better developer experience when building and testing UI components.
