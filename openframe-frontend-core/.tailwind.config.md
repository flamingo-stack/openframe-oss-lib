Tailwind CSS configuration file for a design system implementation that extends the default configuration with custom colors, typography, and animations. Integrates shadcn/ui compatibility alongside a comprehensive Open Design System (ODS) color palette using CSS custom properties.

## Key Components

- **Content paths**: Scans all TypeScript/JavaScript files in the `src` directory
- **Container configuration**: Centered layout with responsive breakpoints up to 1400px
- **Color system**: Dual compatibility layer supporting both shadcn/ui and ODS color tokens
- **Typography**: Custom font families (DM Sans, Azeret Mono) with semantic naming
- **Animations**: Accordion expand/collapse keyframes and custom radius calculations
- **Plugin integration**: Includes `tailwindcss-animate` for enhanced animations

## Usage Example

```typescript
// Using ODS color tokens in components
<div className="bg-ods-bg border border-ods-border text-ods-text-primary">
  <button className="bg-ods-accent hover:bg-ods-accent-hover text-ods-text-on-accent">
    Primary Action
  </button>
</div>

// Alternative flat structure usage
<div className="bg-bg-primary border-border-primary text-text-primary">
  <span className="text-accent-primary hover:text-accent-hover">Link</span>
</div>

// Custom fonts and animations
<h1 className="font-heading animate-accordion-down">Heading</h1>
<p className="font-body text-ods-text-secondary">Body text</p>
```
