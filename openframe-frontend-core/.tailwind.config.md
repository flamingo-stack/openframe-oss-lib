A comprehensive Tailwind CSS configuration that implements the Open Design System (ODS) color palette alongside shadcn/ui compatibility, featuring extensive theming support through CSS custom properties.

## Key Components

- **ODS Color System**: Complete implementation of the Open Design System colors with both flat and nested naming conventions
- **shadcn/ui Compatibility**: Maintains all standard shadcn/ui color tokens for component compatibility  
- **Custom Typography**: DM Sans for body text and Azeret Mono for headings/monospace
- **Responsive Container**: Centered container with custom breakpoints and padding
- **Animation Support**: Accordion animations and tailwindcss-animate plugin integration

## Usage Example

```typescript
// Using ODS colors in components
<div className="bg-ods-bg border border-ods-border text-ods-text-primary">
  <button className="bg-ods-accent hover:bg-ods-accent-hover text-ods-text-on-accent">
    Click me
  </button>
</div>

// Using flat naming convention
<div className="bg-bg-primary border-border-primary text-text-primary">
  <span className="text-accent-primary">Highlighted text</span>
</div>

// Legacy nested structure still supported
<div className="bg-ods-bg border border-ods-border">
  <p className="text-ods-text-primary">Primary text</p>
  <p className="text-ods-text-secondary">Secondary text</p>
</div>
```

The configuration provides multiple naming patterns for the same colors, ensuring flexibility while maintaining consistency across the design system. All colors reference CSS custom properties for dynamic theming support.
