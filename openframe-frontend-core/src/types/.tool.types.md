This file provides centralized type definitions and utilities for tool types used across the platform, serving as a single source of truth for tool-related type information.

## Key Components

- **`ToolType`** - Union type defining all supported tool types (TACTICAL_RMM, FLEET_MDM, MESHCENTRAL, etc.)
- **`toolLabels`** - Record mapping tool types to human-readable display labels
- **`getToolLabel()`** - Utility function to retrieve display labels for tool types with fallback

## Usage Example

```typescript
import { ToolType, getToolLabel, toolLabels } from './tool.types'

// Type-safe tool type usage
const toolType: ToolType = 'TACTICAL_RMM'

// Get display label
const displayName = getToolLabel('FLEET_MDM') // Returns: "Fleet"

// Access all labels
const allLabels = toolLabels
console.log(allLabels.MESHCENTRAL) // "MeshCentral"

// Use in component props
interface ToolBadgeProps {
  type: ToolType
}

function ToolBadge({ type }: ToolBadgeProps) {
  return <span>{getToolLabel(type)}</span>
}
```

This centralized approach ensures consistent tool type handling and prevents type mismatches across components like ToolBadge and ToolIcon.
