Customer interview management types for video testimonials and case studies, including AI-powered processing capabilities for transcription, summarization, and video clip extraction.

## Key Components

### Core Interfaces
- **`CustomerInterview`** - Main entity with video content, AI processing status, and SEO metadata
- **`VideoTeaser`** - Short promotional video clips with thumbnail and visibility controls
- **`VideoClip`** - AI-generated video segments with confidence scoring and timestamps
- **`Speaker`** - Identified speakers in video transcripts with confidence levels

### Data Management
- **`CreateCustomerInterviewData`** - Required fields for creating new interviews
- **`UpdateCustomerInterviewData`** - Partial update interface for modifications
- **`CustomerInterviewFilters`** - Query filtering by platform, industry, status, and search terms

### AI Processing
The interface tracks AI processing states for:
- Transcript generation (AssemblyAI integration)
- Summary extraction with bullet points and gist
- Video clip identification (TwelveLabs integration)
- Speaker identification with confidence scoring

## Usage Example

```typescript
// Create a new customer interview
const newInterview: CreateCustomerInterviewData = {
  title: "MSP Success Story: 50% Cost Reduction",
  slug: "msp-cost-reduction-story",
  summary: "How **TechFlow MSP** reduced operational costs by 50% using OpenFrame",
  user_id: "customer-uuid",
  main_video_url: "https://videos.example.com/interview.mp4",
  status: "draft",
  author_id: "author-uuid",
  platforms: ["azure", "aws"],
  teasers: [{
    url: "https://videos.example.com/teaser.mp4",
    title: "Quick Preview",
    published: true,
    source: "manual"
  }]
};

// Filter interviews by platform and status
const filters: CustomerInterviewFilters = {
  platform: "azure",
  status: "completed",
  featured: true,
  limit: 10
};
```
