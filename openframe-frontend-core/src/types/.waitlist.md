This file defines TypeScript interfaces for managing waitlist functionality, including comprehensive tracking, HubSpot integration, and cohort management capabilities.

## Key Components

- **`WaitlistEntry`** - Complete waitlist entry interface with user data, tracking fields, HubSpot integration metadata, and cohort relationship data
- **`CreateWaitlistEntry`** - Input interface for creating new waitlist entries with required and optional fields
- **`WaitlistResponse`** - Standard API response wrapper for waitlist operations

## Usage Example

```typescript
// Creating a new waitlist entry
const newEntry: CreateWaitlistEntry = {
  email: "user@example.com",
  name: "John Doe",
  company: "Acme Corp",
  platform_id: "platform-123",
  utm_source: "google",
  utm_campaign: "launch"
};

// API response handling
const response: WaitlistResponse = await createWaitlistEntry(newEntry);
if (response.success && response.data) {
  console.log(`Entry created with ID: ${response.data.id}`);
  
  // Access HubSpot integration data
  if (response.data.hubspot_contact_id) {
    console.log(`HubSpot contact: ${response.data.hubspot_contact_url}`);
  }
  
  // Check cohort membership
  if (response.data.cohort) {
    console.log(`Assigned to cohort: ${response.data.cohort.cohort_name}`);
  }
}
```
