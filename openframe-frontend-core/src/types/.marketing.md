This file defines comprehensive TypeScript interfaces for a Marketing AI system that manages campaigns, content generation, media creation, and social media publishing.

## Key Components

### Core Campaign Types
- **MarketingCampaign** - Main campaign entity with content source configuration
- **CreateCampaignInput/UpdateCampaignInput** - Campaign creation and modification interfaces

### AI Content Management
- **AIContent** - AI-generated content (blog posts, social posts, ads, emails) with approval workflow
- **ContentType/ContentStatus** - Enums for content classification and state management
- **URLInjectionPreference** - Controls how URLs are embedded in published content

### AI Media Generation
- **AIMedia** - AI-generated media assets with multi-provider support
- **MediaProvider** - Supports DALL-E, Midjourney, Figma exports, and video generation services
- **MediaDimensions** - Standardized media sizing and aspect ratio handling

### Social Media Integration
- **SocialAccount** - OAuth-enabled social platform connections
- **ScheduledPost** - Automated posting with retry logic and status tracking

### Workflow Management
- **ContentApproval** - Review and approval system for AI-generated content
- **AIUsage** - Cost and token tracking across AI services
