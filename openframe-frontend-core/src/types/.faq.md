This TypeScript interface defines the structure for FAQ (Frequently Asked Questions) data objects used throughout the application for managing help content and user support information.

## Key Components

- **`Faq`** - Main interface defining the complete structure of an FAQ item with metadata fields

### Interface Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | `number` | Unique identifier for the FAQ item |
| `question` | `string` | The question text displayed to users |
| `answer` | `string` | The answer/response content |
| `section` | `string \| null` | Optional category or section grouping |
| `display_order` | `number` | Sort order for displaying FAQs |
| `is_active` | `boolean` | Visibility flag for published/draft state |
| `created_at` | `string` | ISO timestamp of creation |
| `updated_at` | `string` | ISO timestamp of last modification |

## Usage Example

```typescript
import { Faq } from './faq';

// Example FAQ object
const sampleFaq: Faq = {
  id: 1,
  question: "How do I reset my password?",
  answer: "Click the 'Forgot Password' link on the login page and follow the instructions.",
  section: "Account Management",
  display_order: 1,
  is_active: true,
  created_at: "2024-01-15T10:30:00Z",
  updated_at: "2024-01-15T10:30:00Z"
};

// Usage in component
function displayFaqs(faqs: Faq[]) {
  return faqs
    .filter(faq => faq.is_active)
    .sort((a, b) => a.display_order - b.display_order);
}
```
