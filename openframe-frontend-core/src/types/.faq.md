A TypeScript interface definition for FAQ (Frequently Asked Questions) data structure used in applications that manage help documentation or customer support content.

## Key Components

- **Faq interface**: Defines the complete structure for FAQ items with fields for content, organization, and metadata
- **Core fields**: `question`, `answer`, and `section` for content organization
- **Management fields**: `display_order` for sorting, `is_active` for visibility control
- **Audit fields**: `created_at` and `updated_at` timestamps for tracking changes

## Usage Example

```typescript
import { Faq } from './faq';

// Creating a new FAQ item
const newFaq: Faq = {
  id: 1,
  question: "How do I reset my password?",
  answer: "Click the 'Forgot Password' link on the login page...",
  section: "Account Management",
  display_order: 1,
  is_active: true,
  created_at: "2024-01-15T10:30:00Z",
  updated_at: "2024-01-15T10:30:00Z"
};

// Using in a function
function filterActiveFaqs(faqs: Faq[]): Faq[] {
  return faqs
    .filter(faq => faq.is_active)
    .sort((a, b) => a.display_order - b.display_order);
}
```
