A centralized type system that serves as the single source of truth for all shell/script type information across the platform, based on Tactical RMM supported shell types.

## Key Components

- **`ShellType`** - Union type defining all supported shell types (PowerShell, CMD, Bash, Python, Nushell, Deno, Shell)
- **`ShellTypeDefinition`** - Interface describing shell metadata including ID, label, value, and React icon
- **`SHELL_TYPES`** - Complete array of shell definitions with icons and labels
- **`shellLabels`** - Record mapping shell types to display labels
- **`getShellLabel()`** - Utility function to get display label for a shell type

## Usage Example

```typescript
import { ShellType, SHELL_TYPES, getShellLabel } from './shell.types'

// Use shell type in component
const shellType: ShellType = 'POWERSHELL'
const displayName = getShellLabel(shellType) // "PowerShell"

// Render shell selector dropdown
const ShellSelector = () => (
  <select>
    {SHELL_TYPES.map(shell => (
      <option key={shell.id} value={shell.value}>
        {shell.label}
      </option>
    ))}
  </select>
)

// Find shell definition
const pythonShell = SHELL_TYPES.find(s => s.id === 'PYTHON')
console.log(pythonShell?.label) // "Python"
```
