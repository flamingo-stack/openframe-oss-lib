Provides centralized type definitions and utilities for shell/script types across the platform. Serves as the single source of truth for shell type information based on Tactical RMM supported types.

## Key Components

- **`ShellType`** - Union type defining all supported shell types (PowerShell, CMD, Bash, Python, etc.)
- **`ShellTypeDefinition`** - Interface for shell metadata including id, label, value, and icon
- **`SHELL_TYPES`** - Complete array of all shell type definitions with icons and labels
- **`shellLabels`** - Record mapping shell types to display labels
- **`getShellLabel()`** - Helper function to get display label for a shell type

## Usage Example

```typescript
import { ShellType, SHELL_TYPES, getShellLabel } from './shell.types'

// Use the shell type union
const currentShell: ShellType = 'POWERSHELL'

// Get display label
const label = getShellLabel('PYTHON') // Returns "Python"

// Render shell dropdown options
const shellOptions = SHELL_TYPES.map(shell => ({
  value: shell.value,
  label: shell.label,
  icon: shell.icon
}))

// Type-safe shell validation
function isValidShell(shell: string): shell is ShellType {
  return shell in shellLabels
}
```
