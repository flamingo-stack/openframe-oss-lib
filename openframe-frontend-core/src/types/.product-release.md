Defines all TypeScript types, interfaces, and constants for the Flamingo product release system, modeling release metadata, changelog entries, media assets, external integrations, and filtering options.

## Key Components

### Interfaces

| Interface | Purpose |
|-----------|---------|
| `ProductRelease` | Core entity with versioning, changelog arrays, video/media fields, AI confidence scores, SEO, and hydrated relations |
| `ChangelogEntry` | Single changelog item with optional per-entry `visibility` flag (`public` \| `internal`) |
| `ReleaseMedia` | Attached media assets (image, video, screenshot, demo) |
| `ProductReleaseGitHubLink` | Links to GitHub release tags |
| `KnowledgeBaseLink` | References to KB article paths |
| `ClickUpRoadmapTask` / `ClickUpDeliveryTask` | ClickUp task associations from roadmap and delivery lists |
| `CreateProductReleaseData` | Input shape for creating a release |
| `UpdateProductReleaseData` | Partial of create shape for updates |
| `ProductReleaseFilters` | Query filter options (platform, tags, type, status, pagination) |
| `ProductReleaseListResponse` | Paginated list response wrapper |
| `GitHubRelease` | GitHub release shape used in the release wizard |

### Constants

| Export | Purpose |
|--------|---------|
| `releaseTypeOptions` | Dropdown options for `major`, `minor`, `patch`, `beta`, `alpha` with colors |
| `releaseStatusOptions` | Dropdown options for `alpha`, `beta`, `stable`, `deprecated` with colors |
| `changelogLabels` | Human-readable labels for changelog JSONB categories |
| `SEMVER_REGEX` | Validates semantic versioning strings (e.g., `2.1.0-beta+build.1`) |

## Usage Example

```typescript
import type { ProductRelease, ChangelogEntry, ProductReleaseFilters } from './product-release'
import { releaseTypeOptions, SEMVER_REGEX } from './product-release'

// Validate a version string
const isValid = SEMVER_REGEX.test('2.1.0-beta') // true

// Type a changelog entry with visibility control
const entry: ChangelogEntry = {
  title: 'Added OAuth2 SSO support',
  description: 'Supports Google and Microsoft providers',
  visibility: 'public'
}

// Build a filter query
const filters: ProductReleaseFilters = {
  platform: 'openframe',
  release_type: ['major', 'minor'],
  release_status: ['stable'],
  has_breaking_changes: false,
  limit: 20,
  offset: 0
}

// Access type metadata for UI
const typeLabel = releaseTypeOptions.find(o => o.value === 'major')?.label
// → "Major Release"
```
