Defines TypeScript interfaces and types for managing product releases with comprehensive changelog tracking, media attachments, and external integrations.

## Key Components

**Core Interfaces:**
- `ProductRelease` - Main release entity with version info, changelog categories, and publishing status
- `ChangelogEntry` - Individual changelog items with title and description
- `CreateProductReleaseData`/`UpdateProductReleaseData` - Data transfer objects for CRUD operations
- `ProductReleaseFilters` - Query filtering options for release lists

**Media & External Links:**
- `ReleaseMedia` - Images, videos, screenshots, and demos
- `ProductReleaseGitHubLink` - GitHub release integrations
- `KnowledgeBaseLink` - Documentation references
- `ClickUpRoadmapTask`/`ClickUpDeliveryTask` - Project management integrations

**Configuration:**
- `releaseTypeOptions` - Predefined release types (major, minor, patch, etc.)
- `releaseStatusOptions` - Release stability levels (alpha, beta, stable, deprecated)
- `SEMVER_REGEX` - Semantic versioning validation pattern
