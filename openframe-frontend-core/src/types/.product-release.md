Defines all TypeScript types, interfaces, and constants for the product release domain, covering the full lifecycle from creation to publishing including changelog entries, media, external integrations, and AI-generated content fields.

## Key Components

### Interfaces

| Interface | Purpose |
|-----------|---------|
| `ProductRelease` | Core entity with full release data — versioning, changelog, video/AI fields, SEO, author, and relations |
| `ChangelogEntry` | Single changelog item with optional per-entry `visibility` flag (supports internal entries for investor updates) |
| `ReleaseMedia` | Attached media assets (`image`, `video`, `screenshot`, `demo`) |
| `CreateProductReleaseData` | Input shape for creating a release |
| `UpdateProductReleaseData` | Partial of `CreateProductReleaseData` for updates |
| `ProductReleaseFilters` | Query filter bag for list endpoints |
| `ProductReleaseListResponse` | Paginated list response wrapper |
| `GitHubRelease` | GitHub release shape used by the release wizard |
| `ProductReleaseGitHubLink` / `KnowledgeBaseLink` / `ClickUpRoadmapTask` / `ClickUpDeliveryTask` | One-to-many external integration link records |

### Constants

| Export | Purpose |
|--------|---------|
| `releaseTypeOptions` | Dropdown metadata for `major`, `minor`, `patch`, `beta`, `alpha` |
| `releaseStatusOptions` | Dropdown metadata for `alpha`, `beta`, `stable`, `deprecated` |
| `changelogLabels` | Human-readable labels for the four changelog JSONB categories |
| `SEMVER_REGEX` | Validates semantic versioning strings (e.g. `2.1.0-beta+build.1`) |

## Usage Example

```typescript
import type {
  ProductRelease,
  CreateProductReleaseData,
  ProductReleaseFilters,
  ChangelogEntry
} from './product-release'
import { releaseTypeOptions, SEMVER_REGEX } from './product-release'

// Validate version string before submission
const version = '2.1.0-beta'
if (!SEMVER_REGEX.test(version)) {
  throw new Error('Invalid semantic version')
}

// Build changelog entries
const features: ChangelogEntry[] = [
  { title: 'Dark mode support', description: 'System-wide dark theme', visibility: 'public' },
  { title: 'Internal perf refactor', visibility: 'internal' }
]

// Create payload
const payload: CreateProductReleaseData = {
  title: 'Flamingo 2.1 Beta',
  slug: 'flamingo-2-1-beta',
  version,
  release_type: 'beta',
  release_status: 'beta',
  release_date: '2025-01-15',
  status: 'draft',
  author_id: 'user-uuid',
  platforms: ['platform-uuid-1'],
  tags: [1, 4],
  features_added: features
}

// Filter releases for display
const filters: ProductReleaseFilters = {
  release_type: ['major', 'minor'],
  release_status: ['stable'],
  has_breaking_changes: false,
  limit: 20,
  offset: 0
}

// Render dropdown options
releaseTypeOptions.forEach(opt =>
  console.log(`${opt.label} (${opt.color}): ${opt.description}`)
)
```
