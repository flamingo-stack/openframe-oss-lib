This file defines TypeScript types for a comprehensive announcement system that supports multi-platform messaging with customizable icons, colors, and call-to-action buttons.

## Key Components

**Core Interfaces:**
- `Announcement` - Main database model with platform association, styling, and CTA configuration
- `AnnouncementAsset` - File attachments for announcements
- `CreateAnnouncementData`/`UpdateAnnouncementData` - API request payloads
- `AnnouncementFormData` - Form handling with file uploads

**Configuration:**
- `PLATFORM_CONFIGS` - Platform-specific branding (OpenFrame, OpenMSP, Flamingo)
- `AVAILABLE_SVG_ICONS` - Predefined icon options
- `IconType` - Support for SVG or PNG icons

**API Types:**
- `AnnouncementListResponse` - Paginated announcement listings
- `GetAnnouncementsOptions` - Query parameters with filtering and sorting

## Usage Example

```typescript
// Creating an announcement
const newAnnouncement: CreateAnnouncementData = {
  title: "New Feature Launch",
  description: "Check out our latest update!",
  background_color: "var(--ods-open-yellow-base)",
  icon_type: "svg",
  icon_svg_name: "rocket",
  platform_id: "openframe",
  is_active: true,
  cta_enabled: true,
  cta_text: "Learn More",
  cta_url: "https://example.com",
  cta_target: "_blank",
  created_by: "user123"
};

// Querying announcements
const options: GetAnnouncementsOptions = {
  platform: "openframe",
  is_active: true,
  page: 1,
  pageSize: 10
};
```
