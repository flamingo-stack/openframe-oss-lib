TypeScript type definitions for the announcement system, providing comprehensive interfaces for managing platform-specific announcements with customizable styling, call-to-action buttons, and asset management.

## Key Components

- **Core Types**: `Announcement`, `AnnouncementAsset` - Database models for announcements and their associated files
- **API Types**: `CreateAnnouncementData`, `UpdateAnnouncementData`, `AnnouncementListResponse` - Request/response interfaces
- **Form Types**: `AnnouncementFormData`, `GetAnnouncementsOptions` - UI form and query parameter structures
- **Configuration**: `PLATFORM_CONFIGS`, `AVAILABLE_SVG_ICONS` - Platform and icon definitions
- **Component Props**: `AnnouncementBarProps`, `AnnouncementFormProps` - React component interfaces

## Usage Example

```typescript
import type { Announcement, CreateAnnouncementData, GetAnnouncementsOptions } from './announcement';

// Creating a new announcement
const newAnnouncement: CreateAnnouncementData = {
  title: "New Feature Released",
  description: "Check out our latest updates",
  background_color: "#ff6b35",
  icon_type: "svg",
  icon_svg_name: "rocket",
  platform_id: "openframe",
  is_active: true,
  cta_enabled: true,
  cta_text: "Learn More",
  cta_url: "https://example.com/features",
  cta_target: "_blank",
  created_by: "user-123"
};

// Querying announcements with filters
const queryOptions: GetAnnouncementsOptions = {
  platform: "openframe",
  is_active: true,
  page: 1,
  pageSize: 10,
  sortBy: "created_at",
  sortOrder: "desc"
};
```
