TypeScript type definitions for the announcement system that handles platform-wide notifications with customizable styling, icons, and call-to-action buttons. Provides comprehensive types for database models, API operations, form handling, and UI components.

## Key Components

- **Core Models**: `Announcement`, `AnnouncementAsset` - Database entities with full metadata
- **API Types**: `CreateAnnouncementData`, `UpdateAnnouncementData`, `AnnouncementListResponse` - Request/response interfaces
- **Form Types**: `AnnouncementFormData`, `GetAnnouncementsOptions` - Form handling and query parameters
- **UI Props**: `AnnouncementBarProps`, `AnnouncementFormProps`, `AnnouncementListProps` - Component interfaces
- **Configuration**: `AVAILABLE_SVG_ICONS`, `IconType`, `AvailableSvgIcon` - Icon and styling options
- **Utility Types**: `AnnouncementFilters`, `AnnouncementStats`, `AnnouncementValidation` - Helper interfaces

## Usage Example

```typescript
// Creating a new announcement
const newAnnouncement: CreateAnnouncementData = {
  title: "New Feature Release",
  description: "Check out our latest OpenFrame integration!",
  background_color: "#FFC008",
  icon_type: "svg",
  icon_svg_name: "rocket",
  platform_id: "openframe-platform-id",
  is_active: true,
  cta_enabled: true,
  cta_text: "Learn More",
  cta_url: "https://flamingo.run/openframe",
  cta_target: "_blank",
  created_by: "user-123"
};

// Querying announcements with filters
const options: GetAnnouncementsOptions = {
  platform: "openframe",
  is_active: true,
  page: 1,
  pageSize: 10,
  sortBy: "created_at",
  sortOrder: "desc"
};
```
