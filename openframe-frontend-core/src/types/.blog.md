Comprehensive TypeScript type definitions for a multi-platform blog system with SEO capabilities, media management, and content organization features.

## Key Components

**Core Types:**
- `Platform` - Supported platforms (openmsp, openframe, flamingo, universal)
- `BlogStatus` - Post status states (draft, published, scheduled, archived)
- `BlogPost` - Main blog post entity with SEO fields and relationships
- `BlogCategory`/`BlogTag` - Content organization structures

**Data Management:**
- `CreateBlogPostData`/`UpdateBlogPostData` - Form data interfaces
- `GetBlogPostsOptions` - Query filtering and pagination
- `BlogPostStats` - Dashboard analytics data
- `BlogPagination` - Paginated results with metadata

**SEO & Analytics:**
- `SEOAnalysisResult` - Content optimization scores and suggestions
- `BlogSEOAnalytics` - Historical SEO performance data
- `BlogMediaAsset` - File management for blog content

**Platform Configuration:**
- `PLATFORM_CONFIGS` - Platform display settings with CSS variables
- `PlatformConfig` - Individual platform configuration structure

## Usage Example

```typescript
// Creating a new blog post
const newPost: CreateBlogPostData = {
  title: "My Blog Post",
  slug: "my-blog-post",
  content: "Post content here...",
  author_id: "user-123",
  status: "published",
  platforms: ["openmsp", "universal"],
  categories: [1, 2],
  tags: [5, 8]
};

// Querying posts with filters
const options: GetBlogPostsOptions = {
  platform: "openmsp",
  status: "published",
  page: 1,
  pageSize: 10,
  sortBy: "published_at",
  sortOrder: "desc"
};
```
