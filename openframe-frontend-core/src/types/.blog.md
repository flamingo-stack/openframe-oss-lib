Comprehensive TypeScript definitions for the Flamingo blog system, including database models, API interfaces, and SEO analytics for multi-platform content management.

## Key Components

- **Core Models**: `BlogPost`, `BlogCategory`, `BlogTag` - Main database entities with normalized relationships
- **Platform Integration**: `BlogPostPlatform` - Links posts to different platforms with featured content support
- **Media Management**: `BlogMediaAsset` - File uploads, storage paths, and metadata
- **SEO Features**: `BlogSEOAnalytics`, `SEOAnalysisResult` - Content optimization and scoring
- **Form Data**: `CreateBlogPostData`, `UpdateBlogPostData` - Structured input validation
- **Query Options**: `GetBlogPostsOptions`, `BlogSearchParams` - Flexible content filtering
- **Pagination**: `BlogPagination`, `BlogPostSummary` - Optimized list views
- **Dashboard**: `BlogPostStats` - Analytics and performance metrics

## Usage Example

```typescript
import { BlogPost, CreateBlogPostData, GetBlogPostsOptions } from './blog';

// Create a new blog post
const newPost: CreateBlogPostData = {
  title: "OpenFrame AI Platform Overview",
  slug: "openframe-ai-platform-overview",
  content: "Comprehensive guide to OpenFrame...",
  author_id: "user123",
  status: "published",
  platforms: ["flamingo", "docs"],
  categories: [1, 3],
  tags: [5, 8, 12]
};

// Query published posts with filtering
const queryOptions: GetBlogPostsOptions = {
  platform: "flamingo",
  status: "published",
  page: 1,
  pageSize: 10,
  sortBy: "published_at",
  sortOrder: "desc"
};
```
