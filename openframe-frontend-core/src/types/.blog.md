This file defines TypeScript interfaces and types for a multi-platform blog system that supports content management across different platforms (OpenMSP, OpenFrame, Flamingo) with comprehensive SEO, categorization, and analytics features.

## Key Components

**Core Types:**
- `Platform` - Platform identifiers ('openmsp', 'openframe', 'flamingo', etc.)
- `BlogStatus` - Post status states ('draft', 'published', 'scheduled', 'archived')
- `BlogPost` - Main blog post interface with SEO fields, relationships, and metadata

**Content Management:**
- `BlogAuthor` - Author profile with basic information
- `BlogCategory` / `BlogTag` - Content organization with slugs and descriptions
- `BlogMediaAsset` - File attachments with upload metadata

**Platform Integration:**
- `BlogPostPlatform` - Links posts to specific platforms with featured status
- `PLATFORM_CONFIGS` - Configuration array with display names and styling

**API Interfaces:**
- `CreateBlogPostData` / `UpdateBlogPostData` - Form submission types
- `GetBlogPostsOptions` - Query parameters with filtering and pagination
- `BlogPagination` - Paginated response structure

**Analytics & SEO:**
- `BlogSEOAnalytics` - SEO scoring and analysis results
- `BlogPostStats` - Dashboard statistics and metrics
- `SEOAnalysisResult` - Comprehensive SEO evaluation

## Usage Example

```typescript
// Create a new blog post
const newPost: CreateBlogPostData = {
  title: "Getting Started with OpenFrame",
  slug: "getting-started-openframe",
  content: "# Introduction\nThis guide covers...",
  author_id: "user_123",
  status: "published",
  platforms: ["openframe", "universal"],
  featured_platform: "openframe",
  categories: [1, 2],
  tags: [5, 8, 12]
};

// Query posts with filters
const queryOptions: GetBlogPostsOptions = {
  platform: "openframe",
  status: "published",
  page: 1,
  pageSize: 10,
  sortBy: "published_at",
  sortOrder: "desc"
};

// Platform configuration
const openframeConfig = PLATFORM_CONFIGS.find(p => p.name === 'openframe');
console.log(openframeConfig?.display_name); // "OpenFrame"
```
