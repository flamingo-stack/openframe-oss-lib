This file defines TypeScript interfaces and types for AI-powered customer interview video processing, including transcription, summarization, and video clip extraction features. It serves as a comprehensive type system for the customer interview AI pipeline.

## Key Components

- **AIProcessingState** - Tracks processing status for transcript, summary, and clips
- **TranscriptionResult** - Contains formatted transcripts, summaries, and metadata from AssemblyAI
- **VideoClipsResult** - Handles video clip extraction results from TwelveLabs API
- **VideoProcessingOptions** - Configuration options for AI processing features
- **AIStatusResponse** - Real-time status updates with progress tracking
- **Component Props Types** - Props interfaces for React components like TranscriptViewer and VideoClipCard
- **VideoAIConfig** - Configuration for AssemblyAI and TwelveLabs integrations

## Usage Example

```typescript
// Define processing options
const options: VideoProcessingOptions = {
  processTranscription: true,
  processSummary: true,
  processClips: true,
  clipQueries: ["key insights", "pain points"],
  enableSpeakerDiarization: true,
  language: 'en'
}

// Handle processing results
const result: VideoProcessingResult = await processVideo(options)
if (result.transcription) {
  console.log('Summary:', result.transcription.summary_gist)
  console.log('Confidence:', result.transcription.confidence_transcript)
}

// Track processing status
const status: AIStatusResponse = await getProcessingStatus(jobId)
console.log('Overall progress:', status.overallProgress)
```

The types support a complete AI workflow from video upload through transcription, summarization, and intelligent clip extraction with comprehensive error handling and progress tracking.
