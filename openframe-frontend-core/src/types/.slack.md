TypeScript interfaces and types for implementing a Slack-style community chat system, defining the complete data structures for users, channels, messages, and API interactions.

## Key Components

**Core Entities:**
- `SlackUser` - User profile with online status and metadata
- `SlackChannel` - Channel configuration with privacy and activity tracking
- `SlackMessage` - Message structure with reactions and threading support
- `SlackCommunityStats` - Community-wide activity metrics

**Data Management:**
- `SlackCommunityData` - Complete community state container
- `SlackApiResponse` - Standardized API response wrapper
- `UseSlackCommunityReturn` - React hook interface for community management

**Interactive Features:**
- `SlackTypingIndicator` - Real-time typing status
- `SlackReaction` - Message reaction system
- `SlackAttachment` - File and media attachments
- `SlackMessageSendRequest` - Message sending payload

## Usage Example

```typescript
import { SlackCommunityData, SlackMessage, UseSlackCommunityReturn } from './slack';

// Using the community hook interface
const { 
  data, 
  sendMessage, 
  switchChannel, 
  activeChannelId 
}: UseSlackCommunityReturn = useSlackCommunity();

// Sending a message
await sendMessage("Hello community!", "general-channel-id");

// Accessing channel data
const activeChannel = data?.channels.find(ch => ch.id === activeChannelId);
const unreadCount = activeChannel?.unreadCount || 0;
```
