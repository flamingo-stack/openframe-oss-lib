Defines TypeScript interfaces for category-related components and data structures in an e-commerce or product catalog system. These interfaces standardize the props and data formats for category cards and category data management.

## Key Components

- **CategoryCardProps** - Props interface for basic category card components with product/subcategory counts
- **RealCategoryCardProps** - Enhanced props interface with vendor data and loading states
- **CategoryData** - Data structure interface for category collections with loading and error states

## Usage Example

```typescript
import { CategoryCardProps, RealCategoryCardProps, CategoryData } from './category';

// Basic category card
const categoryCard: CategoryCardProps = {
  category: {
    id: '1',
    name: 'Electronics',
    slug: 'electronics',
    description: 'Electronic devices and accessories',
    iconUrls: ['icon1.png', 'icon2.png'],
    subcategory_count: 5,
    productCount: 150
  },
  vendorCount: 25,
  className: 'category-card'
};

// Enhanced category card with loading states
const realCategoryCard: RealCategoryCardProps = {
  category: {
    id: '2',
    name: 'Clothing',
    slug: 'clothing',
    icon_url: 'clothing-icon.png'
  },
  vendors: [],
  isLoading: false,
  isLoadingVendorCount: true
};

// Category data collection
const categoryData: CategoryData = {
  data: [/* category objects */],
  isLoading: false,
  error: null,
  vendors: [],
  loading: false
};
```
