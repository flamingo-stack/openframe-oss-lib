A comprehensive type definition file that provides TypeScript interfaces for all navigation-related components in the OpenFrame UI system, including headers, mobile navigation, sidebars, and user configurations.

## Key Components

- **`NavigationItem`** - Base interface for navigation items with support for icons, badges, dropdowns, and custom elements
- **`HeaderConfig`** - Configuration for header components including logo, navigation positioning, and mobile behavior
- **`MobileNavConfig`** - Configuration for mobile navigation panels with sectioned items and footer support
- **`SlidingSidebarConfig`** - Configuration for sliding sidebar overlays with position control
- **`NavigationSidebarConfig`** - Configuration for persistent navigation sidebars with minimization support
- **`NavigationSidebarItem`** - Specialized navigation item for sidebar components with section grouping
- **`UnifiedSidebarUser`** - User profile information interface for authenticated navigation states

## Usage Example

```typescript
import { NavigationItem, HeaderConfig } from './navigation'

const navigationItems: NavigationItem[] = [
  {
    id: 'home',
    label: 'Home',
    href: '/',
    icon: <HomeIcon />,
    isActive: true
  },
  {
    id: 'products',
    label: 'Products',
    type: 'dropdown',
    children: [
      { id: 'web', label: 'Web Apps', href: '/web' },
      { id: 'mobile', label: 'Mobile Apps', href: '/mobile' }
    ]
  }
]

const headerConfig: HeaderConfig = {
  logo: { element: <Logo />, href: '/' },
  navigation: { items: navigationItems, position: 'center' },
  mobile: { enabled: true },
  backgroundColor: 'bg-ods-card'
}
```
