Type definitions for configuring navigation components across a React application, providing standardized interfaces for headers, sidebars, and mobile navigation.

## Key Components

- **NavigationItem** - Base interface for all navigation items with support for icons, badges, dropdowns, and custom elements
- **HeaderConfig** - Configuration for main application header including logo, navigation items, actions, and mobile behavior
- **MobileNavConfig** - Settings for mobile navigation panels with sectioned items and custom footer
- **SlidingSidebarConfig** - Configuration for slide-out sidebar navigation with positioning options
- **NavigationSidebarConfig** - Settings for persistent sidebar navigation with minimize functionality
- **NavigationSidebarItem** - Specialized navigation item for sidebar components with section grouping

## Usage Example

```typescript
import { NavigationItem, HeaderConfig } from './navigation'

// Basic navigation items
const navItems: NavigationItem[] = [
  {
    id: 'home',
    label: 'Home',
    href: '/',
    icon: <HomeIcon />,
    isActive: true
  },
  {
    id: 'products',
    label: 'Products',
    type: 'dropdown',
    children: [
      { id: 'web', label: 'Web Apps', href: '/web' },
      { id: 'mobile', label: 'Mobile Apps', href: '/mobile' }
    ]
  }
]

// Header configuration
const headerConfig: HeaderConfig = {
  logo: {
    element: <Logo />,
    href: '/'
  },
  navigation: {
    items: navItems,
    position: 'center'
  },
  actions: {
    right: [<SearchButton />, <UserMenu />]
  },
  mobile: {
    enabled: true,
    isOpen: false
  },
  backgroundColor: 'bg-ods-card'
}
```
