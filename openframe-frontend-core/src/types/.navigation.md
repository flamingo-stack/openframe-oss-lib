Defines shared TypeScript interfaces for all navigation-related components, providing a unified type contract for headers, sidebars, and mobile navigation panels.

## Key Components

| Interface | Purpose |
|-----------|---------|
| `NavigationItem` | Base interface for any nav item — supports links, dropdowns, badges, icons, and custom elements |
| `HeaderConfig` | Full configuration for the top header including logo, nav items, actions, and mobile behavior |
| `MobileNavConfig` | Configuration for grouped mobile navigation panels with optional footer |
| `SlidingSidebarConfig` | Configuration for animated slide-in sidebars (left or right) |
| `NavigationSidebarConfig` | Configuration for collapsible vertical sidebars with minimize support |
| `NavigationSidebarItem` | Sidebar-specific nav item with section placement (`primary` / `secondary`) |
| `UnifiedSidebarUser` | User identity shape (name, email, avatar, role) consumed by unified sidebar components |

## Usage Example

```typescript
import { NavigationItem, HeaderConfig } from './navigation'

const navItems: NavigationItem[] = [
  {
    id: 'dashboard',
    label: 'Dashboard',
    href: '/dashboard',
    type: 'single',
    isActive: true,
  },
  {
    id: 'tools',
    label: 'Tools',
    type: 'dropdown',
    children: [
      { id: 'tickets', label: 'Tickets', href: '/tickets' },
      { id: 'assets', label: 'Assets', href: '/assets' },
    ],
  },
]

const headerConfig: HeaderConfig = {
  logo: {
    element: <FlamingoLogo />,
    href: '/',
  },
  navigation: {
    items: navItems,
    position: 'center',
  },
  autoHide: true,
  backgroundColor: 'bg-ods-card',
}
```
