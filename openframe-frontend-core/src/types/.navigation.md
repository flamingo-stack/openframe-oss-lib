This file defines TypeScript interfaces for a comprehensive navigation system, providing type definitions for headers, sidebars, mobile navigation, and various navigation item configurations.

## Key Components

- **`NavigationItem`** - Base interface for navigation items with support for icons, badges, dropdowns, and custom elements
- **`HeaderConfig`** - Configuration for header components including logo, navigation items, actions, and mobile behavior
- **`MobileNavConfig`** - Configuration for mobile navigation panels with sectioned items and footer content
- **`SlidingSidebarConfig`** - Configuration for sliding sidebar components with position and state management
- **`NavigationSidebarConfig`** - Configuration for main navigation sidebars with minimization support
- **`NavigationSidebarItem`** - Specialized navigation item interface for sidebar components with sections

## Usage Example

```typescript
import { HeaderConfig, NavigationItem } from './navigation'

// Define navigation items
const navItems: NavigationItem[] = [
  {
    id: 'home',
    label: 'Home',
    href: '/',
    icon: <HomeIcon />,
    isActive: true
  },
  {
    id: 'products',
    label: 'Products',
    type: 'dropdown',
    children: [
      { id: 'product1', label: 'Product 1', href: '/products/1' }
    ]
  }
]

// Configure header
const headerConfig: HeaderConfig = {
  logo: {
    element: <Logo />,
    href: '/'
  },
  navigation: {
    items: navItems,
    position: 'center'
  },
  mobile: {
    enabled: true
  }
}
```
