This file defines TypeScript types for user-related data structures in the application, covering user profiles, favorites, activities, and preferences.

## Key Components

- **`UserProfile`** - Complete user profile with personal/professional info and role-based permissions
- **`UserFavorite`** - User's favorited vendors with timestamps
- **`UserActivity`** - Tracks user interactions and behaviors with detailed activity types
- **`UserPreferences`** - User settings for theme and notification preferences

## Usage Example

```typescript
import { UserProfile, UserActivity, UserPreferences } from './user'

// Create a user profile
const user: UserProfile = {
  id: "user_123",
  full_name: "John Doe",
  email: "john@company.com",
  role: "user",
  job_title: "Software Engineer",
  department: "Engineering",
  // ... other fields
}

// Track user activity
const activity: UserActivity = {
  id: 1,
  user_id: user.id,
  vendor_id: 456,
  activity_type: "favorited",
  created_at: new Date().toISOString()
}

// Set user preferences
const preferences: UserPreferences = {
  id: "pref_123",
  user_id: user.id,
  theme: "dark",
  email_notifications: true,
  created_at: new Date().toISOString(),
  updated_at: new Date().toISOString()
}
```
