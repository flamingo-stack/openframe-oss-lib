This file defines TypeScript types and interfaces for The Miami Cyber Gang (TMCG) community platform, providing comprehensive data structures for member profiles, social links, API responses, and React component props.

## Key Components

**Core Data Types:**
- `TMCGMember` - Complete member profile with roles, social links, and metadata
- `TMCGMemberSocialLink` - Social media platform connection details
- `TMCGMembersResponse`/`TMCGMemberResponse` - API response wrappers

**React Component Props:**
- `TMCGMemberCardProps` - Props for individual member display cards
- `TMCGMembersGridProps` - Props for member grid layouts with loading states
- `TMCGSocialLinksProps` - Props for social media link components

**Hook Types:**
- `UseTMCGMembersResult`/`UseTMCGMemberResult` - Return types for data fetching hooks

**Constants:**
- `TMCG_ROLES` - Available member roles (founder, organizer, member, etc.)
- `TMCG_ROLE_DISPLAY_NAMES` - Human-readable role labels
- `TMCG_SOCIAL_PLATFORMS` - Supported social media platforms

## Usage Example

```typescript
import { TMCGMember, TMCG_ROLES, UseTMCGMembersResult } from './tmcg'

// Type-safe member data
const member: TMCGMember = {
  id: 'user123',
  full_name: 'John Doe',
  avatar_url: 'https://example.com/avatar.jpg',
  company: 'Tech Corp',
  tmcg_roles: [TMCG_ROLES.ORGANIZER, TMCG_ROLES.SPEAKER],
  social_links: [
    { platform: 'linkedin', url: 'https://linkedin.com/in/johndoe' }
  ],
  created_at: '2023-01-01T00:00:00Z',
  updated_at: '2023-01-01T00:00:00Z'
}

// Hook usage
const { members, isLoading, error }: UseTMCGMembersResult = useTMCGMembers()
```
