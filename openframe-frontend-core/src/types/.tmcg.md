This file defines TypeScript types, interfaces, and constants for managing TMCG (The Miami Cyber Gang) members and their associated data structures.

## Key Components

**Core Interfaces:**
- `TMCGMember` - Complete member profile with personal info, roles, and social links
- `TMCGMemberSocialLink` - Social media platform links with optional usernames
- `TMCGMembersResponse`/`TMCGMemberResponse` - API response wrappers

**Component Props:**
- `TMCGMemberCardProps` - Props for member display cards
- `TMCGMembersGridProps` - Props for member grid layouts
- `TMCGSocialLinksProps` - Props for social links components

**Hook Types:**
- `UseTMCGMembersResult`/`UseTMCGMemberResult` - Return types for React hooks

**Constants:**
- `TMCG_ROLES` - Available member roles (founder, organizer, member, etc.)
- `TMCG_ROLE_DISPLAY_NAMES` - Human-readable role names
- `TMCG_SOCIAL_PLATFORMS` - Supported social media platforms

## Usage Example

```typescript
import { TMCGMember, TMCG_ROLES, TMCGMemberCardProps } from './tmcg'

const member: TMCGMember = {
  id: '123',
  full_name: 'John Doe',
  tmcg_roles: [TMCG_ROLES.ORGANIZER, TMCG_ROLES.SPEAKER],
  social_links: [
    { platform: 'linkedin', url: 'https://linkedin.com/in/johndoe' }
  ],
  created_at: '2024-01-01T00:00:00Z',
  updated_at: '2024-01-01T00:00:00Z'
}

const cardProps: TMCGMemberCardProps = {
  member,
  compact: true,
  className: 'member-card'
}
```
