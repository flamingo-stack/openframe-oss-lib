Defines unified platform types and interfaces used across Flamingo's multi-platform architecture for announcements, blog posts, and platform configuration management.

## Key Components

- **PlatformName** - Union type defining supported platform identifiers (`'openmsp' | 'tmcg' | 'flamingo' | 'flamingo-teaser' | 'universal' | 'admin-hub' | 'openframe'`)
- **PlatformRecord** - Database entity interface representing platform records with metadata
- **PlatformConfig** - API configuration interface with display properties and styling defaults
- **PlatformOption** - Simplified interface for dropdown/filter UI components
- **PlatformStats** - Interface for platform analytics and content counting
- **PlatformFilter** - Type for filtering operations including 'all' option

## Usage Example

```typescript
import { PlatformName, PlatformConfig, PlatformStats } from './platform';

// Define platform configuration
const config: PlatformConfig = {
  id: 'uuid-123',
  value: 'flamingo',
  label: 'Flamingo MSP',
  name: 'flamingo',
  display_name: 'Flamingo MSP',
  default_color: '#FFC008',
  default_icon: 'flamingo-icon',
  description: 'AI-powered MSP platform'
};

// Filter by platform
const filterPlatform: PlatformName = 'openframe';

// Platform statistics
const stats: PlatformStats = {
  platform: 'flamingo',
  display_name: 'Flamingo MSP',
  announcement_count: 15,
  blog_post_count: 8,
  is_active: true
};
```
