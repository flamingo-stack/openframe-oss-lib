Defines unified platform types and interfaces used across announcements, blog posts, and platform configuration throughout the application.

## Key Components

- **`PlatformName`** - Union type defining all supported platform identifiers
- **`PlatformRecord`** - Database model interface for the platforms table
- **`PlatformConfig`** - API configuration interface with display properties and defaults
- **`PlatformOption`** - Simplified interface for dropdown/filter components
- **`PlatformFilter`** - Filter type that extends PlatformName with 'all' option
- **`PlatformStats`** - Interface for platform usage statistics with content counts

## Usage Example

```typescript
import { PlatformName, PlatformConfig, PlatformFilter } from './platform';

// Type-safe platform selection
const currentPlatform: PlatformName = 'openmsp';

// Filter configuration
const filterOptions: PlatformFilter[] = ['all', 'tmcg', 'flamingo'];

// Platform dropdown setup
const platformConfig: PlatformConfig = {
  value: 'tmcg',
  label: 'TMCG Platform',
  name: 'tmcg',
  display_name: 'TMCG Platform',
  default_color: '#blue-500',
  default_icon: 'platform-icon',
  description: 'TMCG platform configuration'
};
```
