TypeScript definitions for unified platform types used across the Flamingo ecosystem, including OpenMSP, TMCG, and various hub configurations.

## Key Components

- **PlatformName**: Union type defining all supported platform identifiers including `openmsp`, `tmcg`, `flamingo`, and various hub platforms
- **PlatformRecord**: Database model interface representing platform entries with metadata like display names, descriptions, and status flags
- **PlatformConfig**: API configuration interface extending platform records with UI-specific properties like colors and icons
- **PlatformOption**: Simplified interface for dropdown/filter components requiring only value-label pairs
- **PlatformStats**: Analytics interface tracking announcement and blog post counts per platform
