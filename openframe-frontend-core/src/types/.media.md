This file defines the core `Media` interface for handling different types of media assets in a TypeScript application, providing a unified structure for both images and videos.

## Key Components

- **Media interface**: A comprehensive type definition with properties for:
  - Basic identification (`id`, `type`, `src`)
  - Content metadata (`alt`, `title`, `caption`)
  - Visual properties (`width`, `height`, `thumbnail`)
  - Video-specific fields (`poster`, `duration`)

## Usage Example

```typescript
import { Media } from './media';

// Image media example
const imageMedia: Media = {
  id: 'img-001',
  type: 'image',
  src: '/images/hero.jpg',
  alt: 'Hero banner image',
  width: 1200,
  height: 600,
  thumbnail: '/images/hero-thumb.jpg'
};

// Video media example
const videoMedia: Media = {
  id: 'vid-001',
  type: 'video',
  src: '/videos/intro.mp4',
  title: 'Introduction Video',
  poster: '/images/video-poster.jpg',
  duration: 120,
  width: 1920,
  height: 1080
};

// Use in a media gallery
const mediaGallery: Media[] = [imageMedia, videoMedia];
```
