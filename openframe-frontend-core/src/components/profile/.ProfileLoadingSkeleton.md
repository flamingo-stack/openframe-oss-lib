A React component that displays an animated loading skeleton for user profile sections, providing visual feedback while profile data is being fetched.

## Key Components

- **ProfileLoadingSkeleton**: Main component that renders placeholder elements with pulse animations
- **ProfileLoadingSkeletonProps**: Interface defining optional className prop for styling customization
- **Header Section**: Circular avatar placeholder with name and subtitle skeletons
- **Content Section**: Multiple text line placeholders of varying widths

## Usage Example

```typescript
import { ProfileLoadingSkeleton } from './ProfileLoadingSkeleton'

// Basic usage
function UserProfile() {
  const { data: user, isLoading } = useUser()
  
  if (isLoading) {
    return <ProfileLoadingSkeleton />
  }
  
  return <UserProfileContent user={user} />
}

// With custom styling
function ProfileCard() {
  return (
    <ProfileLoadingSkeleton 
      className="bg-white p-6 rounded-lg shadow-md" 
    />
  )
}
```

The component uses Tailwind CSS classes with `animate-pulse` for smooth loading animations and leverages a utility function `cn` for conditional class merging. Perfect for maintaining UI consistency during data loading states.
