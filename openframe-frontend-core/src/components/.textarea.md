A reusable textarea component built with React and forwardRef support, featuring consistent styling and mobile-optimized behavior to prevent unwanted zoom on focus.

## Key Components

- **`Textarea`** - Main textarea component with forwardRef support
- **`TextareaProps`** - TypeScript interface extending native textarea HTML attributes
- **Mobile optimization** - Prevents zoom on iOS devices with 16px minimum font size

## Usage Example

```typescript
import { Textarea } from "./components/textarea"

// Basic usage
<Textarea placeholder="Enter your message..." />

// With custom styling
<Textarea 
  className="min-h-[120px]" 
  placeholder="Description"
  value={description}
  onChange={(e) => setDescription(e.target.value)}
/>

// With ref access
const textareaRef = useRef<HTMLTextAreaElement>(null)

<Textarea 
  ref={textareaRef}
  onFocus={() => textareaRef.current?.select()}
/>
```

The component includes default styling for borders, focus states, and disabled states, while maintaining full compatibility with native textarea attributes and events.
