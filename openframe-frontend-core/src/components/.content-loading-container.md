A React component that provides a unified loading container for content areas with smooth transitions between loading and loaded states.

## Key Components

- **`ContentLoadingContainer`** - Main component that wraps content and handles loading states
- **`useContentLoading`** - Custom hook providing utilities for managing loading states
- **`ContentLoadingContainerProps`** - Interface defining component properties including skeleton component, loading state, and transition settings

## Usage Example

```typescript
import { ContentLoadingContainer, useContentLoading } from './content-loading-container'

function VendorPage() {
  const [isLoading, setIsLoading] = useState(true)
  const { getSkeletonCount, getMinHeight } = useContentLoading(isLoading)

  return (
    <ContentLoadingContainer
      isLoading={isLoading}
      skeletonComponent={
        <CardSkeletonGrid 
          variant="vendor" 
          count={getSkeletonCount('vendor')} 
        />
      }
      minHeight={getMinHeight('vendor')}
      transitionDuration={300}
    >
      <VendorGrid vendors={vendors} />
    </ContentLoadingContainer>
  )
}
```

The component prevents layout jumps by maintaining consistent dimensions, shows skeleton overlays during loading, and provides accessibility features with proper ARIA attributes. It supports customizable transition durations, loading opacity, and minimum heights for different content types.
