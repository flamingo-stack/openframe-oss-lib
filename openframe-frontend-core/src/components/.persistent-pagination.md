A React component that maintains pagination visibility during loading states while providing visual feedback about disabled interactions.

## Key Components

- **`PersistentPagination`** - Main wrapper component that keeps pagination visible but disabled during loading
- **`usePaginationLoading`** - Hook for managing pagination loading states and accessibility attributes
- **`PersistentPaginationWrapper`** - Enhanced pagination component with built-in loading state management
- **`PersistentPaginationProps`** - TypeScript interface defining component properties

## Usage Example

```typescript
import { PersistentPagination } from './persistent-pagination'

// Basic usage with existing pagination component
<PersistentPagination
  isLoading={isLoadingData}
  currentPage={currentPage}
  totalPages={totalPages}
  disabledOpacity={0.5}
>
  <Pagination 
    currentPage={currentPage} 
    totalPages={totalPages}
    onPageChange={handlePageChange}
  />
</PersistentPagination>

// Using the wrapper component
<PersistentPaginationWrapper
  isLoading={isLoading}
  currentPage={page}
  totalPages={total}
  onPageChange={setPage}
  variant="vendor"
/>

// Using the hook for custom implementations
const { shouldShowPagination, paginationProps } = usePaginationLoading(
  isLoading, 
  currentPage, 
  totalPages
)
```

The component ensures consistent layout by always rendering pagination, using opacity and pointer-events to indicate loading states while maintaining accessibility with screen reader announcements.
