A customizable React SVG component that renders a clock/time icon with configurable styling properties.

## Key Components

- **`CustomTimeIconProps`** - Interface defining optional props for styling and sizing
- **`CustomTimeIcon`** - React functional component that renders an SVG clock icon

## Usage Example

```typescript
import { CustomTimeIcon } from './custom-time-icon';

// Basic usage with defaults
<CustomTimeIcon />

// Customized with different size and color
<CustomTimeIcon 
  width={32} 
  height={32} 
  fill="#3b82f6" 
  className="my-time-icon" 
/>

// With additional SVG props
<CustomTimeIcon 
  fill="#ef4444" 
  onClick={handleClick}
  aria-label="Time indicator"
/>
```

The component uses default values of 24px dimensions and gray fill color (#888888). All props are optional and the component spreads additional props to the underlying SVG element, making it highly flexible for various use cases. The icon depicts a clock face with hands pointing to approximately 3:30.
