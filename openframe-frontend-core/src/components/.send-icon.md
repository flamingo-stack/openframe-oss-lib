A customizable React SVG component that renders a send/paper plane icon, commonly used for submit buttons or messaging interfaces.

## Key Components

- **SendIconProps**: Interface defining optional `className` and `fill` props
- **SendIcon**: React functional component that renders an SVG send icon with customizable styling and color

## Usage Example

```typescript
import { SendIcon } from './send-icon';

// Basic usage with defaults
<SendIcon />

// Custom styling and color
<SendIcon 
  className="w-8 h-8 hover:opacity-70" 
  fill="#3B82F6" 
/>

// In a button component
<button className="flex items-center gap-2">
  Send Message
  <SendIcon className="w-5 h-5" fill="white" />
</button>
```

The component uses default values of `"w-6 h-6"` for className (Tailwind CSS sizing) and `"#1A1A1A"` (dark gray) for fill color. The SVG viewBox is set to 24x24 and includes two path elements that create the paper plane icon shape.
