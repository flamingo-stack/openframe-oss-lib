A customizable waitlist button component that extends the base Button component with additional styling props for colors and loading states.

## Key Components

- **`JoinWaitlistButtonProps`** - Interface extending `ButtonHTMLAttributes` with custom color and loading properties
- **`JoinWaitlistButton`** - Forward ref component that renders a styled button with customizable appearance
- **Color Props** - `lowerPathColor`, `upperPathColor`, `buttonBackgroundColor`, `buttonTextColor` for theming
- **Loading State** - `loading` prop for handling async operations

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button-stub';

// Basic usage
<JoinWaitlistButton onClick={handleJoinWaitlist}>
  Join Our Beta
</JoinWaitlistButton>

// With custom styling
<JoinWaitlistButton
  buttonBackgroundColor="#007bff"
  buttonTextColor="#ffffff"
  lowerPathColor="#0056b3"
  upperPathColor="#66b3ff"
  loading={isSubmitting}
  disabled={isSubmitting}
  onClick={handleSubmit}
>
  {isSubmitting ? 'Joining...' : 'Join Waitlist'}
</JoinWaitlistButton>

// With ref
const buttonRef = useRef<HTMLButtonElement>(null);
<JoinWaitlistButton ref={buttonRef} className="custom-class" />
```
