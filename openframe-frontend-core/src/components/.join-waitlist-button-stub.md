A React component that renders a customizable waitlist signup button with theme support and flexible styling options.

## Key Components

- **`JoinWaitlistButtonProps`** - Interface extending button HTML attributes with custom styling props for colors and loading state
- **`JoinWaitlistButton`** - Forward ref component that wraps a base Button component with waitlist-specific defaults
- **Custom styling props** - `lowerPathColor`, `upperPathColor`, `buttonBackgroundColor`, `buttonTextColor` for theme customization
- **Loading state support** - Optional `loading` prop for handling async waitlist operations

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button-stub';

// Basic usage with default text
<JoinWaitlistButton onClick={handleWaitlistSignup} />

// Customized styling and content
<JoinWaitlistButton
  buttonBackgroundColor="#3b82f6"
  buttonTextColor="#ffffff"
  loading={isSubmitting}
  disabled={isSubmitting}
  onClick={handleWaitlistSignup}
>
  {isSubmitting ? 'Joining...' : 'Get Early Access'}
</JoinWaitlistButton>

// With custom theme colors
<JoinWaitlistButton
  lowerPathColor="#ef4444"
  upperPathColor="#dc2626"
  buttonBackgroundColor="#991b1b"
  className="w-full"
>
  Reserve Your Spot
</JoinWaitlistButton>
```

The component maintains full button functionality while providing specialized styling options for waitlist scenarios. It defaults to "Join Waitlist" text when no children are provided and forwards all standard button props and refs.
