A React component that renders the Claude AI assistant logo as a customizable SVG icon.

## Key Components

- **ClaudeIcon**: Main functional component that returns an SVG element with the Claude logo path
- **ClaudeIconProps**: TypeScript interface extending React's SVG props for type safety
- Uses "use client" directive for Next.js client-side rendering

## Usage Example

```typescript
import { ClaudeIcon } from './claude-icon';

// Basic usage
<ClaudeIcon />

// With custom styling
<ClaudeIcon 
  className="w-6 h-6 text-blue-500" 
  style={{ color: '#1e40af' }}
/>

// With event handlers
<ClaudeIcon 
  onClick={() => console.log('Claude icon clicked')}
  onMouseEnter={() => setHovered(true)}
/>

// Custom size (inherits from font-size by default)
<div style={{ fontSize: '2rem' }}>
  <ClaudeIcon />
</div>
```

The component accepts all standard SVG props including className, style, onClick, and accessibility attributes. The icon scales with the parent's font-size and includes proper aria-label for accessibility.
