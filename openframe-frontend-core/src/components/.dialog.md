A comprehensive dialog component system built with React that provides a modal overlay with backdrop blur, close functionality, and structured content areas.

## Key Components

- **Dialog**: Main container component that controls visibility with `open` prop and backdrop click-to-close
- **DialogContent**: Centered modal content area with built-in close button (X icon)
- **DialogTrigger**: Button wrapper for opening the dialog
- **DialogClose**: Button wrapper for closing actions
- **DialogHeader/Footer**: Layout components for structured dialog content
- **DialogTitle/Description**: Text components with semantic styling
- **DialogOverlay**: Backdrop overlay component
- **DialogPortal**: Portal wrapper (currently pass-through)

## Usage Example

```typescript
import { useState } from 'react'
import { Dialog, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogTrigger } from './dialog'

function MyComponent() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <Dialog open={isOpen} onOpenChange={setIsOpen}>
      <DialogTrigger onClick={() => setIsOpen(true)}>
        Open Dialog
      </DialogTrigger>
      
      <DialogContent onCloseClick={() => setIsOpen(false)}>
        <DialogHeader>
          <DialogTitle>Confirm Action</DialogTitle>
          <DialogDescription>
            Are you sure you want to proceed?
          </DialogDescription>
        </DialogHeader>
      </DialogContent>
    </Dialog>
  )
}
```
