A flexible React component wrapper that adds interactive behaviors like hover, focus, and press states with visual feedback including ripple effects and animations.

## Key Components

**Main Components:**
- `InteractiveWrapper` - Core wrapper component with configurable interactive states
- `InteractiveOptions` - Configuration interface for enabling different interaction types
- `RippleEffect` - Visual feedback component for click interactions
- `InteractivePresets` - Pre-configured components (Card, Button, NavItem, Input, Toggle)

**Key Features:**
- Polymorphic rendering via `as` prop
- Accessibility enhancements with ARIA attributes
- State management integration via `useInteractiveState` hook
- Loading and disabled state handling
- Customizable ripple effects

## Usage Example

```typescript
import { InteractiveWrapper, InteractivePresets } from './interactive-wrapper';

// Basic wrapper with custom interactions
<InteractiveWrapper
  interactive={{
    enableHover: true,
    enableRipple: true,
    enableAnimations: true
  }}
  showRipple={true}
  focusable={true}
  onStateChange={(state) => console.log(state)}
>
  <div>Interactive content</div>
</InteractiveWrapper>

// Using pre-configured components
<InteractivePresets.Button onClick={handleClick}>
  Click me
</InteractivePresets.Button>

<InteractivePresets.Card className="p-6">
  <h3>Interactive Card</h3>
</InteractivePresets.Card>

<InteractivePresets.Toggle 
  checked={isToggled} 
  onClick={() => setIsToggled(!isToggled)} 
/>
```
