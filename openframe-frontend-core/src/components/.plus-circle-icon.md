A React component that renders an SVG plus circle icon with customizable styling and color options.

## Key Components

- **PlusCircleIcon**: Main component function that renders a circular plus icon
- **PlusCircleIconProps**: TypeScript interface extending SVGProps with custom properties
  - `className?`: Optional CSS class name
  - `whiteOverlay?`: Boolean to toggle white fill color (defaults to false)
  - `iconSize?`: Number to set width/height (defaults to 24px)

## Usage Example

```typescript
import { PlusCircleIcon } from './plus-circle-icon'

// Basic usage with default green color
<PlusCircleIcon />

// Custom size and styling
<PlusCircleIcon 
  iconSize={32} 
  className="cursor-pointer hover:opacity-80" 
/>

// White overlay variant
<PlusCircleIcon 
  whiteOverlay={true}
  iconSize={20}
/>

// With additional SVG props
<PlusCircleIcon 
  onClick={() => console.log('Add item')}
  aria-label="Add new item"
  role="button"
/>
```

The icon displays in green (#5EA62E) by default, but can be switched to white using the `whiteOverlay` prop. All standard SVG element props are supported through prop spreading.
