A React component for displaying MSP (Managed Service Provider) information with an avatar and name in a clean, horizontal layout.

## Key Components

- **MSPDisplay** - Main component that renders an MSP's logo and name
- **MSPDisplayProps** - TypeScript interface defining component props:
  - `name` (required) - MSP name to display
  - `logoUrl` (optional) - URL for MSP logo image
  - `size` (optional) - Avatar size in pixels, defaults to 40px
  - `className` (optional) - Additional CSS classes

## Usage Example

```typescript
import { MSPDisplay } from './msp-display';

// Basic usage with just a name
<MSPDisplay name="TechCorp Solutions" />

// With logo and custom sizing
<MSPDisplay 
  name="SecureNet MSP"
  logoUrl="https://example.com/logo.png"
  size={60}
/>

// With custom styling
<MSPDisplay 
  name="CloudFirst Partners"
  logoUrl="/assets/cloudFirst-logo.png"
  className="border rounded-lg p-4"
/>
```

The component uses a square avatar that shows the MSP logo when available, falling back to initials from the name. The layout is responsive with truncated text to handle long MSP names gracefully. The `SquareAvatar` component handles the logo display and fallback logic internally.
