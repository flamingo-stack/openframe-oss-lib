A React component that displays an MSP (Managed Service Provider) with their logo avatar and name in a horizontal layout.

## Key Components

- **MSPDisplay**: Main component that renders an MSP's logo and name
- **MSPDisplayProps**: Interface defining component props including name, logoUrl, size, and styling options
- **SquareAvatar**: Imported component used to display the MSP logo with fallback to name initials

## Usage Example

```typescript
import { MSPDisplay } from './msp-display';

function MSPList() {
  return (
    <div className="space-y-4">
      {/* MSP with logo */}
      <MSPDisplay 
        name="TechCorp Solutions" 
        logoUrl="https://example.com/logo.png"
        size={48}
      />
      
      {/* MSP without logo (shows initials) */}
      <MSPDisplay 
        name="Digital Services Inc"
        logoUrl={null}
        className="bg-gray-100 p-2 rounded"
      />
      
      {/* Default size (40px) */}
      <MSPDisplay name="CloudTech MSP" />
    </div>
  );
}
```

The component automatically truncates long names and provides consistent spacing between the avatar and text. When no logo is provided, the SquareAvatar component will display the MSP's initials as a fallback.
