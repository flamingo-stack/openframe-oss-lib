A React component that renders an SVG icon representing an empty or placeholder vendor state. The icon displays a rounded square with a subtle pattern fill using base64-encoded image data.

## Key Components

- **EmptyVendorIcon**: Main React functional component that renders the SVG icon
- **EmptyVendorIconProps**: TypeScript interface defining optional `className` prop for styling
- **SVG Pattern**: Uses SVG `<pattern>` and `<image>` elements with base64 data for the icon's visual content

## Usage Example

```typescript
import { EmptyVendorIcon } from './empty-vendor-icon';

// Basic usage
function VendorCard() {
  return (
    <div className="vendor-placeholder">
      <EmptyVendorIcon />
      <p>No vendor selected</p>
    </div>
  );
}

// With custom styling
function StyledVendorIcon() {
  return (
    <EmptyVendorIcon className="w-8 h-8 opacity-50 hover:opacity-75" />
  );
}

// In a list of vendors
function VendorList({ vendors }) {
  return (
    <div className="vendor-grid">
      {vendors.length === 0 && (
        <EmptyVendorIcon className="text-gray-400" />
      )}
    </div>
  );
}
```

The component is designed to be flexible with styling through the optional `className` prop while maintaining a consistent 24x24 viewBox for predictable sizing.
