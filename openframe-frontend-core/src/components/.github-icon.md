A React component that renders the GitHub logo as an SVG icon with customizable dimensions and styling.

## Key Components

- **GitHubIcon**: Main functional component that renders a GitHub logo SVG
- **GitHubIconProps**: Interface defining optional props for customization
  - `className`: CSS class string for styling
  - `width`: Icon width in pixels (default: 15)
  - `height`: Icon height in pixels (default: 14)

## Usage Example

```typescript
import { GitHubIcon } from './github-icon';

// Basic usage with defaults
<GitHubIcon />

// Custom dimensions
<GitHubIcon width={24} height={24} />

// With styling
<GitHubIcon 
  className="text-blue-500 hover:text-blue-700" 
  width={20} 
  height={20} 
/>

// In a button
<button className="flex items-center gap-2">
  <GitHubIcon width={16} height={16} />
  View on GitHub
</button>
```

The icon uses `fill="currentColor"` to inherit the text color from its parent container, making it easy to style with CSS color utilities.
