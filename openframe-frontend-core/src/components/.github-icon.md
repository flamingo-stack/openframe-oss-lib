A React component that renders the GitHub logo as an SVG icon with customizable styling and dimensions.

## Key Components

- **GitHubIconProps** - Interface defining optional props for className, width, and height
- **GitHubIcon** - Main React functional component that renders the GitHub logo SVG

## Usage Example

```typescript
import { GitHubIcon } from './github-icon';

// Basic usage with default size (15x14)
<GitHubIcon />

// Custom size and styling
<GitHubIcon 
  width={24} 
  height={24} 
  className="text-gray-600 hover:text-gray-800" 
/>

// In a button or link
<a href="https://github.com/username" className="flex items-center gap-2">
  <GitHubIcon width={20} height={20} />
  View on GitHub
</a>
```

The component uses `currentColor` for the fill, making it easy to control the icon color through CSS text color properties. Default dimensions are 15x14 pixels, matching the original viewBox proportions.
