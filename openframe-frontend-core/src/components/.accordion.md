A styled accordion component built on top of Radix UI primitives, providing collapsible content sections with smooth animations and customizable styling.

## Key Components

- **`Accordion`** - Root container component that manages accordion state
- **`AccordionItem`** - Individual accordion section wrapper with border styling
- **`AccordionTrigger`** - Clickable header with chevron icon that toggles content visibility
- **`AccordionContent`** - Collapsible content area with slide animations

## Usage Example

```typescript
import { Accordion, AccordionContent, AccordionItem, AccordionTrigger } from "./accordion"

export function FAQ() {
  return (
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">
        <AccordionTrigger>What is your return policy?</AccordionTrigger>
        <AccordionContent>
          We offer a 30-day return policy for all items in original condition.
        </AccordionContent>
      </AccordionItem>
      
      <AccordionItem value="item-2">
        <AccordionTrigger>How long does shipping take?</AccordionTrigger>
        <AccordionContent>
          Standard shipping takes 3-5 business days within the continental US.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

The component includes built-in hover effects, rotating chevron animations, and smooth expand/collapse transitions using custom CSS animations.
