A React component that renders the Microsoft logo as an SVG icon with customizable dimensions and styling.

## Key Components

- **`MicrosoftIcon`** - Main React component that renders the Microsoft four-square logo
- **`MsIconProps`** - TypeScript interface defining optional props for width, height, and className
- **SVG paths** - Four colored squares representing Microsoft's brand colors (orange, green, blue, yellow)

## Usage Example

```typescript
import { MicrosoftIcon } from './ms-icon'

// Default usage (24x24px)
<MicrosoftIcon />

// Custom size
<MicrosoftIcon width={32} height={32} />

// With custom styling
<MicrosoftIcon 
  width="48" 
  height="48" 
  className="hover:opacity-80 transition-opacity" 
/>

// In a button
<button className="flex items-center gap-2">
  <MicrosoftIcon width={16} height={16} />
  Sign in with Microsoft
</button>
```

The component uses fixed SVG dimensions internally but accepts width/height props for container sizing. The `className` prop allows for additional CSS styling and integrates well with utility-first CSS frameworks like Tailwind CSS.
