A reusable React component that displays category information in a styled card format with icons, counts, and navigation.

## Key Components

- **CategoryCard**: Main functional component that renders a category information card
- **CategoryCardProps**: TypeScript interface defining the component's props:
  - `name`: Category display name
  - `description`: Category description text
  - `categoryCount`: Number of subcategories
  - `productCount`: Number of products in category
  - `icons`: Optional array of React nodes for visual representation

## Usage Example

```typescript
import { CategoryCard } from './category-card';
import { DatabaseIcon, ServerIcon } from 'lucide-react';

// Basic usage
<CategoryCard
  name="Databases"
  description="Modern database solutions for scalable applications"
  categoryCount={5}
  productCount={23}
/>

// With custom icons
<CategoryCard
  name="Infrastructure"
  description="Cloud infrastructure and deployment tools"
  categoryCount={8}
  productCount={45}
  icons={[
    <DatabaseIcon key="db" className="w-6 h-6 text-white" />,
    <ServerIcon key="server" className="w-6 h-6 text-white" />
  ]}
/>
```

The component features a dark theme with hover effects, responsive layout, and automatically fills empty icon slots with placeholder elements when fewer than 10 icons are provided. The card includes a clickable arrow button for navigation and displays category/product counts in a subtitle format.
