A React component that displays category information in a card format with icons, metadata, and a navigation button.

## Key Components

**CategoryCard** - Main component that renders a category card with:
- `name` - Category title
- `description` - Category description text
- `categoryCount` - Number of subcategories
- `productCount` - Number of products in category
- `icons` - Optional array of React nodes for category icons

## Usage Example

```typescript
import { CategoryCard } from './category-card';
import { ShoppingBag, Laptop } from 'lucide-react';

function CategoryList() {
  const techIcons = [
    <Laptop key="laptop" size={20} />,
    <ShoppingBag key="bag" size={20} />
  ];

  return (
    <CategoryCard
      name="Electronics"
      description="Latest gadgets and electronic devices for modern living"
      categoryCount={12}
      productCount={150}
      icons={techIcons}
    />
  );
}
```

The component features a dark theme with hover effects on the navigation button. Icons are displayed in a horizontal grid (defaults to 10 placeholder slots if none provided), with category stats and description below.
