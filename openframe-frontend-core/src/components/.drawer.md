A React drawer component library that wraps the Vaul primitive to provide a customizable bottom sheet/drawer interface. It includes pre-styled components with default styling and accessibility features.

## Key Components

- **Drawer** - Root wrapper component with optional background scaling
- **DrawerTrigger** - Button/element that opens the drawer
- **DrawerContent** - Main drawer panel with bottom sheet styling and drag handle
- **DrawerOverlay** - Semi-transparent backdrop overlay
- **DrawerHeader/Footer** - Structured layout sections for content organization
- **DrawerTitle/Description** - Accessible heading and description elements
- **DrawerClose** - Element to close the drawer
- **DrawerPortal** - Portal component for rendering outside normal DOM tree

## Usage Example

```typescript
import {
  Drawer,
  DrawerContent,
  DrawerHeader,
  DrawerTitle,
  DrawerTrigger,
  DrawerFooter,
  DrawerClose
} from "./drawer"

function MyDrawer() {
  return (
    <Drawer>
      <DrawerTrigger asChild>
        <button>Open Drawer</button>
      </DrawerTrigger>
      <DrawerContent>
        <DrawerHeader>
          <DrawerTitle>Settings</DrawerTitle>
        </DrawerHeader>
        <div className="p-4">
          Drawer content goes here
        </div>
        <DrawerFooter>
          <DrawerClose asChild>
            <button>Close</button>
          </DrawerClose>
        </DrawerFooter>
      </DrawerContent>
    </Drawer>
  )
}
```
