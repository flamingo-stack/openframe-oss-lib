A styled wrapper component built on top of Radix UI's AlertDialog primitive that provides accessible, customizable alert dialog components with pre-configured styling and animations.

## Key Components

- **AlertDialog** - Root container component
- **AlertDialogTrigger** - Button/element that opens the dialog
- **AlertDialogContent** - Main dialog container with overlay and animations
- **AlertDialogHeader/Footer** - Layout components for dialog structure
- **AlertDialogTitle/Description** - Text content components
- **AlertDialogAction/Cancel** - Action buttons with button styling

## Usage Example

```typescript
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "./alert-dialog"

function DeleteConfirmDialog() {
  return (
    <AlertDialog>
      <AlertDialogTrigger asChild>
        <button>Delete Item</button>
      </AlertDialogTrigger>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>Are you sure?</AlertDialogTitle>
          <AlertDialogDescription>
            This action cannot be undone. This will permanently delete the item.
          </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogCancel>Cancel</AlertDialogCancel>
          <AlertDialogAction onClick={handleDelete}>Delete</AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  )
}
```

All components are fully accessible and include smooth fade/zoom animations for opening and closing states.
