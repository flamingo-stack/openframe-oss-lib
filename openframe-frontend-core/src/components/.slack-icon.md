A React component that renders the official Slack logo as a customizable SVG icon with the brand's characteristic four-colored design.

## Key Components

- **SlackIcon**: Main functional component that renders the Slack logo SVG
- **SlackIconProps**: TypeScript interface defining optional props for width, height, and CSS className
- **SVG paths**: Four colored path elements representing Slack's distinctive logo segments in brand colors

## Usage Example

```typescript
import { SlackIcon } from './slack-icon'

// Default 24x24 icon
<SlackIcon />

// Custom size
<SlackIcon width={32} height={32} />

// With custom styling
<SlackIcon 
  width="48" 
  height="48" 
  className="hover:opacity-80 transition-opacity" 
/>

// In a button
<button className="flex items-center gap-2">
  <SlackIcon width={20} height={20} />
  Connect Slack
</button>
```

The component maintains the original Slack brand colors (#E01E5A, #36C5F0, #2EB67D, #ECB22E) and provides a clean, scalable icon suitable for integration buttons, navigation menus, or any UI requiring the Slack logo.
