Scrollable log entry list component that renders a collection of `LogEntry` items with severity indicators, timestamps, and optional tool icons, connected by a vertical timeline line.

## Key Components

- **`LogsList`** — Main exported component (forwarded ref) that renders the log container with overflow scrolling, loading state, and empty state handling
- **`LogCard`** — Internal card component for individual log entries with click/keyboard handlers, severity dot, formatted timestamp, and optional tool icon
- **`formatTimestamp`** — Utility that formats a `string | Date` into `YYYY/MM/DD,HH:mm` display format

## Props (`LogsListProps`)

| Prop | Default | Description |
|------|---------|-------------|
| `logs` | — | Array of `LogEntry` items to display |
| `maxHeight` | `'400px'` | Container max height; use `'100%'` for full parent height |
| `showConnector` | `true` | Renders vertical timeline connector between entries |
| `onLogClick` | — | Callback fired with the clicked `LogEntry` |
| `loading` | `false` | Shows a loading placeholder |
| `emptyMessage` | `'No logs to display'` | Text shown when `logs` is empty |
| `className` | — | Additional CSS classes for the container |

## Usage Example

```typescript
import { LogsList } from './logs-list'
import { LogEntry } from '../types/logs.types'

const logs: LogEntry[] = [
  {
    id: '1',
    title: 'Agent task completed',
    severity: 'info',
    timestamp: new Date(),
    toolType: 'bash',
  },
  {
    id: '2',
    title: 'Connection timeout detected',
    severity: 'error',
    timestamp: '2024-01-15T10:30:00Z',
  },
]

export function ActivityPanel() {
  return (
    <LogsList
      logs={logs}
      maxHeight="500px"
      showConnector
      onLogClick={(log) => console.log('Selected:', log.id)}
    />
  )
}
```
