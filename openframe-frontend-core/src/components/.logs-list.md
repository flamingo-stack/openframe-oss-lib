Scrollable log entry list component that renders a collection of `LogEntry` items with severity indicators, timestamps, tool icons, and optional timeline connectors.

## Key Components

- **`LogsList`** — Primary exported component (forward-ref). Accepts an array of log entries and renders them in a scrollable dark-themed container.
- **`LogCard`** — Internal card component for a single log entry. Displays a severity dot, title, formatted timestamp, optional tool icon, and a vertical connector line between entries.
- **`formatTimestamp`** — Helper that normalizes `string | Date` timestamps to `YYYY/MM/DD,HH:MM` format.

## Props (`LogsListProps`)

| Prop | Default | Description |
|------|---------|-------------|
| `logs` | — | Array of `LogEntry` objects to display |
| `maxHeight` | `'400px'` | Container max height; use `'100%'` for full-height flex layout |
| `showConnector` | `true` | Renders vertical timeline line between entries |
| `onLogClick` | — | Callback fired with the clicked `LogEntry` |
| `loading` | `false` | Shows a loading placeholder when `true` |
| `emptyMessage` | `'No logs to display'` | Text shown when `logs` is empty |
| `className` | — | Additional Tailwind classes for the container |

## Usage Example

```typescript
import { LogsList } from './logs-list'
import { LogEntry } from '../types/logs.types'

const logs: LogEntry[] = [
  {
    id: '1',
    title: 'Agent connected',
    timestamp: new Date(),
    severity: 'info',
    toolType: 'ssh',
  },
  {
    id: '2',
    title: 'Script execution failed',
    timestamp: '2024-01-15T14:32:00Z',
    severity: 'error',
  },
]

export function ActivityPanel() {
  return (
    <LogsList
      logs={logs}
      maxHeight="300px"
      showConnector
      onLogClick={(log) => console.log('Selected:', log.id)}
    />
  )
}
```
