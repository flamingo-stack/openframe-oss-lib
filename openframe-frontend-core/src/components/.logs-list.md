A React component that renders a scrollable list of log entries with visual connectors, severity indicators, and interactive functionality.

## Key Components

- **`LogsList`** - Main component that displays a list of log entries with scrolling, loading states, and empty states
- **`LogCard`** - Individual log entry component with severity dot, timestamp, title, and optional tool icon
- **`formatTimestamp`** - Utility function that formats dates into `YYYY/MM/DD,HH:MM` format

## Usage Example

```typescript
import { LogsList } from './logs-list'
import { LogEntry } from '../types/logs.types'

const logs: LogEntry[] = [
  {
    id: '1',
    title: 'API request completed',
    severity: 'info',
    timestamp: new Date(),
    toolType: 'api'
  },
  {
    id: '2', 
    title: 'Error processing data',
    severity: 'error',
    timestamp: '2024-01-15T10:30:00Z'
  }
]

function App() {
  return (
    <LogsList
      logs={logs}
      maxHeight="500px"
      showConnector={true}
      onLogClick={(log) => console.log('Clicked:', log.title)}
      loading={false}
      emptyMessage="No logs available"
    />
  )
}
```

The component supports keyboard navigation, displays loading states, handles empty states, and includes visual connectors between log entries for better readability.
