A React SVG icon component that renders an organizations/groups icon with customizable styling and color properties.

## Key Components

- **OrganizationsIconProps**: Interface extending React SVG props with optional `color` property
- **OrganizationsIcon**: Functional component that renders a 24x24 SVG icon representing organizations/groups

## Usage Example

```typescript
import { OrganizationsIcon } from './organizations-icon'

// Basic usage with default white color
<OrganizationsIcon />

// Custom color and styling
<OrganizationsIcon 
  color="#3b82f6" 
  className="w-6 h-6" 
/>

// With click handler and custom props
<OrganizationsIcon 
  color="currentColor"
  onClick={() => console.log('Organizations clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The icon displays a card-like container with user profile elements, commonly used to represent organizations, teams, or user groups in applications. The component accepts all standard SVG props and defaults to white color if none is specified.
