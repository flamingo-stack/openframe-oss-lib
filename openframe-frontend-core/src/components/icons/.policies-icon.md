A React SVG icon component that renders a folder with a security shield symbol, commonly used to represent policies, security settings, or protected documents.

## Key Components

- **PoliciesIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with an optional `color` prop
- **PoliciesIcon**: Functional component that renders an SVG icon with customizable styling and color

## Usage Example

```typescript
import { PoliciesIcon } from './policies-icon'

// Basic usage with default white color
<PoliciesIcon />

// Custom color and className
<PoliciesIcon 
  color="#3b82f6" 
  className="w-6 h-6" 
/>

// With additional SVG props
<PoliciesIcon 
  color="currentColor"
  width={32}
  height={32}
  onClick={() => console.log('Policies clicked')}
/>
```

The icon is built with a 24x24 viewBox and includes two main path elements: a folder outline and a security shield overlay. The `color` prop defaults to 'white' and controls the fill color of both paths. All standard SVG props are supported through the spread operator.
