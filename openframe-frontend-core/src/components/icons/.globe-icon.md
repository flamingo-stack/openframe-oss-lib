A customizable globe/world icon component built with SVG that provides a clean, scalable representation of the Earth with meridians and equator lines.

## Key Components

- **GlobeIcon** - Main React functional component that renders an SVG globe icon
- **GlobeIconProps** - TypeScript interface defining the component's props:
  - `color` - Optional stroke color (defaults to 'white')
  - `className` - Optional CSS class for styling

## Usage Example

```typescript
import { GlobeIcon } from './globe-icon';

// Basic usage with default white color
<GlobeIcon />

// Custom color and styling
<GlobeIcon 
  color="#3b82f6" 
  className="w-6 h-6 hover:scale-110 transition-transform" 
/>

// Dark theme variant
<GlobeIcon 
  color="#1f2937" 
  className="globe-dark" 
/>
```

The component uses a 24x24 viewBox with stroke-based rendering, making it ideal for icons in navigation bars, global settings buttons, or internationalization indicators. The SVG structure includes a main circle, curved meridian line, and horizontal equator for a recognizable globe representation.
