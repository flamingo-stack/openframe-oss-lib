A React component that renders a customizable globe/world SVG icon with configurable color and styling options.

## Key Components

- **GlobeIcon** - Main functional component that renders an SVG globe icon
- **GlobeIconProps** - TypeScript interface defining component props:
  - `color?: string` - Icon stroke color (defaults to 'white')
  - `className?: string` - Optional CSS classes for styling

## Usage Example

```typescript
import { GlobeIcon } from './globe-icon';

// Basic usage with default white color
<GlobeIcon />

// Custom color
<GlobeIcon color="#3b82f6" />

// With custom styling
<GlobeIcon 
  color="currentColor" 
  className="w-6 h-6 hover:scale-110 transition-transform" 
/>

// In a navigation menu
<button className="flex items-center gap-2">
  <GlobeIcon color="#64748b" className="w-5 h-5" />
  Language
</button>
```

The component uses a 24x24 viewBox SVG with three path elements to create a simple globe representation: an outer circle, a vertical ellipse for longitude lines, and a horizontal line for the equator.
