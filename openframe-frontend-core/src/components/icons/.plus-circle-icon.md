A React SVG icon component that renders a circular plus/add button with customizable styling and color options.

## Key Components

- **PlusCircleIcon**: Main functional component that renders an SVG plus circle icon
- **PlusCircleIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with additional props:
  - `className?: string` - CSS class for styling
  - `whiteOverlay?: boolean` - Toggles between white and green (#5EA62E) fill colors
  - `iconSize?: number` - Controls width/height dimensions (default: 24px)

## Usage Example

```typescript
import { PlusCircleIcon } from './plus-circle-icon'

// Basic usage with default green color
<PlusCircleIcon />

// White overlay version with custom size
<PlusCircleIcon whiteOverlay={true} iconSize={32} />

// With custom styling and click handler
<PlusCircleIcon 
  className="hover:scale-110 cursor-pointer"
  iconSize={20}
  onClick={() => console.log('Add clicked')}
/>

// All SVG props are supported
<PlusCircleIcon 
  whiteOverlay={false}
  iconSize={28}
  role="button"
  aria-label="Add item"
/>
```

The component uses a fixed 24x24 viewBox and scales via the `iconSize` prop, maintaining crisp rendering at any size. The `whiteOverlay` prop switches between white and the default green theme color.
