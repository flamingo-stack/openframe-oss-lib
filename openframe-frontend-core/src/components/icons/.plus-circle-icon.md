A React SVG icon component that renders a circular plus/add button with customizable styling and color options.

## Key Components

- **PlusCircleIcon**: Main component function that renders the SVG icon
- **PlusCircleIconProps**: TypeScript interface extending SVG props with custom options:
  - `className?`: Optional CSS class name
  - `whiteOverlay?`: Boolean to switch between green (#5EA62E) and white colors
  - `iconSize?`: Number to control width/height (default: 24px)

## Usage Example

```typescript
import { PlusCircleIcon } from './plus-circle-icon'

// Basic usage with default green color
<PlusCircleIcon />

// Custom size and white overlay
<PlusCircleIcon 
  iconSize={32} 
  whiteOverlay={true}
  className="my-icon"
/>

// With click handler and additional SVG props
<PlusCircleIcon 
  onClick={() => console.log('Add clicked')}
  style={{ cursor: 'pointer' }}
  aria-label="Add item"
/>
```

The component accepts all standard SVG element props through the spread operator, making it highly flexible for various use cases like buttons, form elements, or decorative icons.
