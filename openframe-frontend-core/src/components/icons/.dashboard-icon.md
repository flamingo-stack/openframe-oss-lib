A React component that renders an SVG dashboard/speedometer icon with customizable styling properties.

## Key Components

- **DashboardIcon**: Main functional component that renders a dashboard/speedometer SVG icon
- **DashboardIconProps**: TypeScript interface defining optional props for className, size, and color customization

## Usage Example

```typescript
import { DashboardIcon } from './dashboard-icon'

// Basic usage with defaults (20px size, white color)
<DashboardIcon />

// Customized with specific styling
<DashboardIcon 
  className="dashboard-nav-icon" 
  size={24} 
  color="#3B82F6" 
/>

// In a navigation menu
<button className="nav-item">
  <DashboardIcon size={16} color="currentColor" />
  <span>Dashboard</span>
</button>
```

The component uses a fixed 24x24 viewBox regardless of the size prop, ensuring consistent aspect ratio. The SVG depicts a dashboard/speedometer with a needle indicator and circular gauge, making it ideal for dashboard navigation or analytics interfaces.
