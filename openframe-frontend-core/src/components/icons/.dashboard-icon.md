A React component that renders a customizable dashboard speedometer/gauge icon as an SVG element.

## Key Components

- **DashboardIcon**: Main functional component that renders a dashboard speedometer icon
- **DashboardIconProps**: TypeScript interface defining optional props:
  - `className`: CSS class names for styling
  - `size`: Icon size (defaults to 20px)
  - `color`: Fill color (defaults to 'white')

## Usage Example

```typescript
import { DashboardIcon } from './dashboard-icon'

// Basic usage with defaults
<DashboardIcon />

// Customized icon
<DashboardIcon 
  className="dashboard-indicator" 
  size={32} 
  color="#3b82f6" 
/>

// In a navigation menu
<button className="nav-item">
  <DashboardIcon size={24} color="currentColor" />
  Dashboard
</button>
```

The component renders a fixed 24x24 viewBox SVG regardless of the size prop, ensuring consistent proportions. The icon features a speedometer design with a needle pointer and circular gauge, commonly used for dashboard navigation or analytics interfaces.
