A React component that renders an SVG icon of a robot with customizable styling and color properties.

## Key Components

- **AiRobotIcon**: Main React functional component that renders the robot SVG
- **AiRobotIconProps**: TypeScript interface defining optional `className` and `color` props
- **SVG Elements**: Robot illustration with antenna, body, eyes, and side indicators

## Usage Example

```typescript
import { AiRobotIcon } from './ai-robot-icon'

// Basic usage with default pink color
<AiRobotIcon />

// Custom styling and color
<AiRobotIcon 
  className="w-16 h-16 mx-auto" 
  color="#3b82f6" 
/>

// In a button or card
<button className="p-4 border rounded-lg">
  <AiRobotIcon 
    className="mb-2" 
    color="#10b981" 
  />
  <span>AI Assistant</span>
</button>
```

The icon uses stroke-based rendering with configurable color (defaults to `#f357bb`) and accepts standard className props for responsive sizing and positioning. The SVG maintains a fixed 200x200 viewBox for consistent scaling.
