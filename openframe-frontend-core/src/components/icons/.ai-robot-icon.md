A React SVG icon component that renders a stylized AI robot with customizable color and styling.

## Key Components

- **AiRobotIcon**: Main React functional component that renders an SVG robot icon
- **AiRobotIconProps**: TypeScript interface defining optional `className` and `color` props
- **SVG Path**: Single path element defining the robot's outline with head antenna, rectangular body, circular eyes, and side arms

## Usage Example

```typescript
import { AiRobotIcon } from './ai-robot-icon'

// Basic usage with default pink color
<AiRobotIcon />

// Custom styling and color
<AiRobotIcon 
  className="w-16 h-16 hover:opacity-80" 
  color="#00ff00" 
/>

// In a button or interactive element
<button className="p-2 rounded-lg bg-gray-100">
  <AiRobotIcon className="w-8 h-8" color="#333" />
</button>
```

The icon uses a 200x200 viewBox with stroke-based rendering, making it scalable and suitable for various UI contexts. The default magenta color (`#f357bb`) can be overridden, and additional CSS classes can be applied for responsive sizing and hover effects.
