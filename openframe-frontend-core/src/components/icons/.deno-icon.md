A **deprecated** React SVG icon component rendering the Deno logo, marked for replacement by `icons-v2-generated`.

## Key Components

- **`DenoIconProps`** *(deprecated)* — Interface with optional `size` (number), `className` (string), and `color` (string) props
- **`DenoIcon`** *(deprecated)* — Functional component rendering an inline SVG of the Deno logo

## Usage Example

```typescript
import { DenoIcon } from './deno-icon'

// Default usage (16px, white)
<DenoIcon />

// Custom size and color
<DenoIcon size={32} color="#000000" className="my-icon" />
```

## Notes

> ⚠️ **Deprecated** — Both `DenoIconProps` and `DenoIcon` are marked `@deprecated`. Use icons from `icons-v2-generated` instead.

Default prop values:

| Prop | Default |
|------|---------|
| `size` | `16` |
| `className` | `''` |
| `color` | `'white'` |
