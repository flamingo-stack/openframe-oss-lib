A React SVG icon component that renders the official Deno logo as a scalable vector graphic with customizable size, styling, and color properties.

## Key Components

- **DenoIconProps**: TypeScript interface defining component props
  - `size?: number` - Icon dimensions (default: 16px)
  - `className?: string` - CSS class names for styling
  - `color?: string` - Fill color for the SVG paths (default: 'white')

- **DenoIcon**: React functional component that renders the Deno dinosaur logo
  - Uses inline SVG with proper viewBox for scalability
  - Includes the complete Deno logo path data
  - Supports customization via props

## Usage Example

```typescript
import { DenoIcon } from './deno-icon'

// Basic usage with defaults (16px, white)
<DenoIcon />

// Custom size and color
<DenoIcon size={32} color="#000000" />

// With custom styling
<DenoIcon 
  size={24} 
  color="currentColor" 
  className="hover:opacity-80 transition-opacity" 
/>

// Large icon for headers
<DenoIcon size={48} color="#40E0D0" />
```

The component is optimized for accessibility and performance, using a single SVG element with the official Deno branding colors and proportions. The `displayName` is set for better debugging in React DevTools.
