A React component that renders the official Deno logo as an inline SVG icon with customizable size, styling, and color properties.

## Key Components

- **`DenoIconProps`** - Interface defining optional props for size, className, and color
- **`DenoIcon`** - React functional component that renders the Deno logo SVG
- Default props: `size={16}`, `className=""`, `color="white"`

## Usage Example

```typescript
import { DenoIcon } from './deno-icon'

// Basic usage with defaults
<DenoIcon />

// Customized icon
<DenoIcon 
  size={32} 
  className="mr-2 hover:opacity-80" 
  color="#000000" 
/>

// Large branded icon
<DenoIcon 
  size={64} 
  color="#40E0D0" 
  className="logo-icon" 
/>
```

The component uses the official Deno dinosaur logo path data and accepts standard React props for flexible integration into different UI contexts. The SVG maintains its aspect ratio and scales proportionally based on the size prop.
