A React component that renders a dashed circle icon as an SVG element. The circle features a customizable dashed stroke pattern and supports both outlined and filled display modes.

## Key Components

- **ShapeCircleDashIconProps**: Interface defining component props including optional className, size, color, and fill mode
- **ShapeCircleDashIcon**: Main functional component that renders a dashed circle SVG with configurable styling

## Usage Example

```typescript
import { ShapeCircleDashIcon } from './shape-circle-dash-icon'

// Basic usage with defaults (24px, white, outlined)
<ShapeCircleDashIcon />

// Custom styled dashed circle
<ShapeCircleDashIcon 
  size={32}
  color="#3b82f6"
  className="my-icon"
/>

// Filled dashed circle
<ShapeCircleDashIcon 
  size={40}
  color="#ef4444"
  full={true}
/>

// Large circle with custom styling
<ShapeCircleDashIcon 
  size={64}
  color="#10b981"
  className="hover:opacity-80 transition-opacity"
/>
```

The component uses a `4 2` dash pattern (4 units dash, 2 units gap) with rounded line caps for a clean appearance. The `full` prop controls whether the circle is filled or outlined.
