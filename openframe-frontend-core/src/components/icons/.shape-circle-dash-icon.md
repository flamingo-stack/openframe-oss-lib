A React icon component that renders a dashed circle SVG with customizable styling and fill options.

## Key Components

- **ShapeCircleDashIcon** - Main functional component that renders a dashed circle SVG
- **ShapeCircleDashIconProps** - TypeScript interface defining component props:
  - `className` - Optional CSS class string
  - `size` - Icon dimensions in pixels (default: 24)
  - `color` - Stroke and fill color (default: 'white')
  - `full` - Boolean to toggle filled vs outlined style (default: false)

## Usage Example

```typescript
import { ShapeCircleDashIcon } from './shape-circle-dash-icon'

// Basic usage with defaults (24px, white, outlined)
<ShapeCircleDashIcon />

// Customized icon
<ShapeCircleDashIcon
  size={32}
  color="#3B82F6"
  full={true}
  className="my-icon"
/>

// Outlined version with custom styling
<ShapeCircleDashIcon
  size={16}
  color="currentColor"
  className="text-gray-600"
/>
```

The component uses a 4-2 dash pattern with rounded line caps, creating a subtle dashed circle effect. The `full` prop controls whether the circle appears as a filled shape or just the dashed outline.
