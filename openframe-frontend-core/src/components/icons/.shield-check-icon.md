A React component that renders a shield with checkmark icon, commonly used to indicate security, verification, or protection status in UI designs.

## Key Components

- **ShieldCheckIcon**: Main functional component that renders an SVG shield icon with a checkmark
- **ShieldCheckIconProps**: TypeScript interface defining customizable props for styling and dimensions

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | CSS classes for styling |
| `size` | `number` | `24` | Width and height in pixels |
| `color` | `string` | `'white'` | Stroke color for the icon |
| `strokeWidth` | `number` | `1.5` | Line thickness of the strokes |

## Usage Example

```typescript
import { ShieldCheckIcon } from './shield-check-icon'

// Basic usage
<ShieldCheckIcon />

// Customized appearance
<ShieldCheckIcon 
  size={32}
  color="green"
  strokeWidth={2}
  className="security-badge"
/>

// In a security indicator component
function SecurityStatus({ isVerified }: { isVerified: boolean }) {
  return (
    <div className="flex items-center gap-2">
      <ShieldCheckIcon 
        color={isVerified ? 'green' : 'gray'}
        size={20}
      />
      <span>{isVerified ? 'Verified' : 'Unverified'}</span>
    </div>
  )
}
```

The component uses SVG paths to create a shield outline with an internal checkmark, making it perfect for security badges, verification indicators, and trust symbols.
