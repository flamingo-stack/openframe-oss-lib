A React component that renders an SVG shield icon with a checkmark, commonly used to indicate security, protection, or verified status.

## Key Components

- **ShieldCheckIcon** - Main component that renders the shield with checkmark SVG
- **ShieldCheckIconProps** - TypeScript interface defining component props:
  - `className` - Optional CSS class string
  - `size` - Icon dimensions (default: 24px)
  - `color` - Stroke color (default: 'white')
  - `strokeWidth` - Line thickness (default: 1.5)

## Usage Example

```typescript
import { ShieldCheckIcon } from './shield-check-icon'

// Basic usage
<ShieldCheckIcon />

// Customized appearance
<ShieldCheckIcon 
  size={32}
  color="#10b981"
  strokeWidth={2}
  className="security-badge"
/>

// In a security status component
<div className="flex items-center gap-2">
  <ShieldCheckIcon size={20} color="green" />
  <span>Verified Account</span>
</div>
```

The component uses inline SVG for crisp rendering at any size and provides flexible styling options through props. The shield outline and checkmark are rendered as separate paths with consistent stroke properties.
