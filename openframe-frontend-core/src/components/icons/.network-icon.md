A React SVG icon component that renders a network/hierarchy diagram symbol with customizable styling and color options.

## Key Components

- **`NetworkIconProps`**: Interface extending `React.SVGProps<SVGSVGElement>` with optional `color` prop
- **`NetworkIcon`**: React functional component that renders an SVG network icon with interconnected nodes

## Usage Example

```typescript
import { NetworkIcon } from './network-icon'

// Basic usage with default styling
<NetworkIcon />

// Custom color and size
<NetworkIcon 
  color="#3b82f6" 
  className="h-8 w-8"
/>

// With additional SVG props
<NetworkIcon 
  color="currentColor"
  onClick={() => console.log('Network clicked')}
  aria-label="Network diagram"
/>

// In a button component
<button className="flex items-center gap-2">
  <NetworkIcon color="#10b981" className="h-5 w-5" />
  Connect Network
</button>
```

The component defaults to a white fill color and `h-6 w-6` dimensions, but accepts any standard SVG props for full customization.
