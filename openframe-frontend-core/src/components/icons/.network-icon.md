A React SVG icon component that renders a network topology diagram with interconnected nodes and pathways.

## Key Components

- **NetworkIconProps**: Interface extending React SVG props with optional `color` prop
- **NetworkIcon**: Functional component rendering a network/connectivity icon as inline SVG

## Usage Example

```typescript
import { NetworkIcon } from './network-icon'

// Basic usage with defaults (white color, h-6 w-6 size)
<NetworkIcon />

// Custom styling
<NetworkIcon 
  color="blue" 
  className="h-8 w-8" 
  onClick={() => console.log('Network clicked')}
/>

// With additional SVG props
<NetworkIcon 
  color="#10B981"
  className="h-12 w-12 hover:scale-110 transition-transform"
  aria-label="Network topology"
/>
```

The icon displays a network diagram with central hub and connected nodes, making it ideal for representing network connectivity, system architecture, or data flow visualizations. The component accepts standard SVG props for full customization while providing sensible defaults.
