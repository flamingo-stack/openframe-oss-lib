A React component that renders the WhatsApp logo as an SVG icon with customizable styling properties.

## Key Components

- **WhatsAppIcon**: Main functional component that renders the WhatsApp SVG icon
- **WhatsAppIconProps**: TypeScript interface defining optional props for customization:
  - `color`: Fill color (default: 'white')
  - `className`: CSS classes for styling
  - `size`: Width and height in pixels (default: 24)

## Usage Example

```typescript
import { WhatsAppIcon } from './whatsapp-icon';

// Basic usage with defaults
<WhatsAppIcon />

// Customized with green color and larger size
<WhatsAppIcon 
  color="#25D366" 
  size={32} 
  className="hover:opacity-80" 
/>

// For use in buttons or links
<button className="flex items-center gap-2">
  <WhatsAppIcon color="#25D366" size={20} />
  Contact us on WhatsApp
</button>
```

The component uses the official WhatsApp logo path data and maintains proper aspect ratio while allowing flexible integration into different UI contexts.
