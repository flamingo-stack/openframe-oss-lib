A customizable React SVG component that renders the WhatsApp brand icon with configurable styling properties.

## Key Components

- **WhatsAppIcon**: React functional component that renders an SVG WhatsApp logo
- **WhatsAppIconProps**: TypeScript interface defining optional props for color, className, and size customization

## Usage Example

```typescript
import { WhatsAppIcon } from './whatsapp-icon';

// Default usage (white icon, 24px size)
<WhatsAppIcon />

// Custom green color and larger size
<WhatsAppIcon color="#25D366" size={32} />

// With CSS classes and custom styling
<WhatsAppIcon 
  className="hover:scale-110 transition-transform" 
  color="#128C7E" 
  size={20} 
/>
```

The component uses the official WhatsApp logo path and provides sensible defaults: white color, 24px size, and no additional CSS classes. The SVG maintains proper aspect ratio and scales cleanly at any size.
