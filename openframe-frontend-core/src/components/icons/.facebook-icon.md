A React component that renders a customizable Facebook icon using SVG, with configurable color and styling properties.

## Key Components

- **FacebookIcon**: Main component function that renders the Facebook logo SVG
- **FacebookIconProps**: TypeScript interface defining component props
  - `color?: string` - Stroke color (defaults to 'white')
  - `className?: string` - CSS classes for styling

## Usage Example

```typescript
import { FacebookIcon } from './facebook-icon';

// Default white icon
<FacebookIcon />

// Custom colored icon with CSS classes
<FacebookIcon 
  color="#1877F2" 
  className="hover:opacity-80 transition-opacity" 
/>

// Icon for dark theme
<FacebookIcon 
  color="#000000" 
  className="w-6 h-6" 
/>
```

The component uses a fixed 24x24 viewBox with stroke-based rendering, making it suitable for various UI contexts like social media links, sharing buttons, or navigation elements.
