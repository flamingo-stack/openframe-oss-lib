A customizable Facebook icon component that renders an SVG icon with configurable styling options.

## Key Components

- **`FacebookIcon`** - React functional component that renders a Facebook logo SVG
- **`FacebookIconProps`** - TypeScript interface defining component props:
  - `color?: string` - Stroke color (defaults to 'white')
  - `className?: string` - CSS class name for styling

## Usage Example

```typescript
import { FacebookIcon } from './facebook-icon';

// Basic usage with default white color
<FacebookIcon />

// Custom color
<FacebookIcon color="#1877F2" />

// With custom styling
<FacebookIcon 
  color="#4267B2" 
  className="hover:opacity-80 cursor-pointer" 
/>

// In a social media links section
<div className="flex space-x-4">
  <FacebookIcon color="#1877F2" className="w-6 h-6" />
  {/* other social icons */}
</div>
```

The component uses a 24x24 viewBox with stroke-based rendering, making it scalable and easily themeable. The SVG includes proper accessibility attributes and follows modern React patterns with TypeScript support.
