Renders a customizable star SVG icon as a React component with configurable dimensions, fill color, and className support.

## Key Components

### `CustomStarIconProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | CSS class for the SVG element |
| `fill` | `string` | `"#888888"` | Fill color for the star path |
| `width` | `number` | `24` | SVG width in pixels |
| `height` | `number` | `24` | SVG height in pixels |

### `CustomStarIcon`
A functional React component rendering a 24×24 viewBox outlined star SVG path. Accepts additional SVG props via spread.

> **Deprecated:** Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { CustomStarIcon } from './custom-star-icon';

// Default gray star
<CustomStarIcon />

// Custom colored and sized star
<CustomStarIcon
  fill="#FFC008"
  width={32}
  height={32}
  className="star-rating-icon"
/>
```
