A customizable SVG star icon component built for React applications with configurable styling and size properties.

## Key Components

**`CustomStarIconProps`** - Interface defining the component's optional props:
- `className` - CSS class for styling
- `fill` - Icon color (default: "#888888")
- `width` & `height` - Icon dimensions (default: 24px)

**`CustomStarIcon`** - React functional component that renders an outlined star SVG with customizable properties and spreads additional props.

## Usage Example

```typescript
import { CustomStarIcon } from './custom-star-icon';

// Basic usage with defaults
<CustomStarIcon />

// Customized star icon
<CustomStarIcon 
  fill="#FFD700" 
  width={32} 
  height={32}
  className="rating-star"
/>

// With additional SVG props
<CustomStarIcon 
  fill="#FF6B6B"
  onClick={handleStarClick}
  style={{ cursor: 'pointer' }}
/>
```

The component uses a 24x24 viewBox and renders an outlined star path, making it ideal for rating systems, favorites, or decorative elements. All unused props are spread to the underlying SVG element for maximum flexibility.
