A customizable star icon component built as an SVG React element with configurable styling properties.

## Key Components

**CustomStarIconProps Interface**
- `className?: string` - Optional CSS class for styling
- `fill?: string` - Star color (defaults to "#888888")
- `width?: number` - Icon width in pixels (defaults to 24)
- `height?: number` - Icon height in pixels (defaults to 24)

**CustomStarIcon Component**
- Renders an outlined star SVG with 24x24 viewBox
- Supports all standard SVG props via spread operator
- Uses a detailed path element for crisp star outline
- Responsive design with configurable dimensions

## Usage Example

```typescript
import { CustomStarIcon } from './custom-star-icon';

// Basic usage with defaults (gray, 24x24)
<CustomStarIcon />

// Customized star icon
<CustomStarIcon 
  fill="#FFD700" 
  width={32} 
  height={32}
  className="star-favorite"
/>

// With additional SVG properties
<CustomStarIcon 
  fill="#FF6B6B"
  onClick={handleStarClick}
  style={{ cursor: 'pointer' }}
/>
```

The component is ideal for rating systems, favorites indicators, or decorative elements where a consistent star shape is needed across different sizes and colors.
