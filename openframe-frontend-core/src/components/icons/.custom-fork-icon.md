A customizable SVG fork icon component for React applications, designed to represent Git fork operations or branching concepts.

## Key Components

- **CustomForkIconProps**: Interface defining the component's props including optional className, fill color, and dimensions
- **CustomForkIcon**: Main functional component that renders an SVG fork icon with customizable styling and size

## Usage Example

```typescript
import { CustomForkIcon } from './custom-fork-icon';

// Basic usage with defaults
<CustomForkIcon />

// Customized appearance
<CustomForkIcon 
  className="my-icon"
  fill="#4a90e2"
  width={32}
  height={32}
/>

// With additional props
<CustomForkIcon 
  fill="#ff6b6b"
  onClick={handleFork}
  style={{ cursor: 'pointer' }}
/>
```

The component uses a 24x24 viewBox and defaults to a gray fill color (#888888). It supports all standard SVG props through the spread operator, making it flexible for various use cases like buttons, navigation elements, or decorative icons in version control interfaces.
