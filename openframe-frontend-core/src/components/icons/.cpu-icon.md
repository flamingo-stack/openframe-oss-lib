A reusable React component that renders an SVG icon representing a CPU or processor chip. The icon features a square chip design with connection pins extending from all sides.

## Key Components

- **CpuIconProps**: Interface extending React's SVGProps for type safety and prop forwarding
- **CpuIcon**: Functional component that renders the CPU icon with customizable styling

## Usage Example

```typescript
import { CpuIcon } from './cpu-icon'

// Basic usage with default styling (24x24, white color)
<CpuIcon />

// Custom size and color
<CpuIcon className="h-8 w-8" color="#3b82f6" />

// With additional SVG props
<CpuIcon 
  className="h-10 w-10" 
  color="currentColor"
  onClick={() => console.log('CPU clicked')}
  aria-label="CPU processor"
/>

// Using with CSS classes for responsive sizing
<CpuIcon className="h-4 w-4 md:h-6 md:w-6 text-gray-700" color="currentColor" />
```

The component accepts all standard SVG properties through prop spreading, making it highly flexible for different use cases while maintaining consistent visual design.
