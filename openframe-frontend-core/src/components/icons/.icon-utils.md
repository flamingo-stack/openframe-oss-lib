A utility function that provides centralized icon rendering for a React application, mapping string identifiers to their corresponding icon components from both Lucide React and custom logo components.

## Key Components

- **`renderSvgIcon(name, props)`** - Main function that takes an icon name string and optional SVG props, returning the appropriate React icon component
- **Icon mapping** - Internal record mapping icon names to their respective component renderers
- **Fallback handling** - Returns Megaphone icon when an unknown name is provided
- **Lucide icons** - Standard UI icons (megaphone, bell, info, star, rocket, package, globe)
- **Custom logos** - Platform-specific logo components (OpenFrame, OpenMSP, Flamingo, Miami Cyber Gang)

## Usage Example

```typescript
import { renderSvgIcon } from './icon-utils';

// Render a Lucide icon
const notificationIcon = renderSvgIcon('bell', { 
  size: 24, 
  className: 'text-blue-500' 
});

// Render a custom logo
const platformLogo = renderSvgIcon('openframe-logo', { 
  width: 32, 
  height: 32 
});

// Fallback to megaphone for unknown icons
const fallbackIcon = renderSvgIcon('unknown-icon', { 
  className: 'icon-fallback' 
});

// Use in JSX
<div className="icon-container">
  {renderSvgIcon('star', { size: 16 })}
  {renderSvgIcon('flamingo-logo')}
</div>
```
