A React SVG component that renders the face-only version of the Miami Cyber Gang logo, featuring a stylized cyberpunk character with customizable colors and size.

## Key Components

- **MiamiCyberGangLogoFaceOnly**: Main React functional component that renders an inline SVG
- **MiamiCyberGangLogoFaceOnlyProps**: TypeScript interface defining component props
- **SVG Paths**: Multiple path elements creating the logo design with face, eyes, and distinctive cyber aesthetic

## Props

- `size?: number` - Controls width and height (default: 64px)
- `originalPink?: string` - Accent color with CSS custom property fallback
- `originalWhite?: string` - Text/detail color with CSS custom property fallback  
- `className?: string` - Additional CSS classes for styling

## Usage Example

```typescript
import { MiamiCyberGangLogoFaceOnly } from './miami-cyber-gang-logo-face-only';

// Basic usage
<MiamiCyberGangLogoFaceOnly />

// Custom size and colors
<MiamiCyberGangLogoFaceOnly 
  size={128}
  originalPink="#FF1493"
  originalWhite="#FFFFFF"
  className="logo-animation"
/>

// Using with design system
<MiamiCyberGangLogoFaceOnly 
  size={96}
  className="header-logo"
/>
```

The component automatically integrates with the ODS (design system) color tokens while providing fallback values, making it suitable for both branded applications and standalone usage.
