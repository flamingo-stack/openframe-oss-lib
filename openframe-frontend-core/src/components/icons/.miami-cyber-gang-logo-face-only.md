A React SVG component that renders the face-only version of the Miami Cyber Gang logo with customizable colors and sizing.

## Key Components

- **MiamiCyberGangLogoFaceOnly** - Main React functional component that renders an SVG logo
- **MiamiCyberGangLogoFaceOnlyProps** - TypeScript interface defining component props:
  - `size` - Controls logo dimensions (default: 64px)
  - `originalPink` - Accent color with CSS variable fallback
  - `originalWhite` - Text/detail color with CSS variable fallback
  - `className` - Additional CSS classes

## Usage Example

```typescript
import { MiamiCyberGangLogoFaceOnly } from './miami-cyber-gang-logo-face-only';

// Basic usage with defaults
<MiamiCyberGangLogoFaceOnly />

// Custom size and colors
<MiamiCyberGangLogoFaceOnly 
  size={128}
  originalPink="#FF6B9D"
  originalWhite="#FFFFFF"
  className="my-logo"
/>

// Using CSS variables (recommended)
<MiamiCyberGangLogoFaceOnly 
  size={96}
  className="themed-logo"
/>
```

The component integrates with design systems by defaulting to CSS custom properties (`--ods-accent`, `--ods-text-on-accent`) while allowing color overrides when needed.
