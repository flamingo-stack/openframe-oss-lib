A reusable React SVG icon component that renders an image/photo icon with customizable styling and color options.

## Key Components

- **`ImageIconProps`** - TypeScript interface defining optional className and color properties
- **`ImageIcon`** - React functional component that renders an SVG icon representing an image/photo

## Usage Example

```typescript
import { ImageIcon } from './image-icon';

// Basic usage with default styling
<ImageIcon />

// Custom size and color
<ImageIcon 
  className="w-8 h-8" 
  color="#3B82F6" 
/>

// Using with Tailwind classes
<ImageIcon 
  className="w-12 h-12 text-gray-500 hover:text-gray-700" 
/>

// In a button or container
<button className="p-2 border rounded">
  <ImageIcon className="w-5 h-5" />
  Upload Image
</button>
```

The component features a rectangular frame with rounded corners, a small circle representing the sun/light source, and a diagonal line representing a mountain landscape - creating the universal symbol for images/photos. The SVG uses `currentColor` by default, allowing it to inherit text color from parent elements, or accepts a custom color prop for specific styling needs.
