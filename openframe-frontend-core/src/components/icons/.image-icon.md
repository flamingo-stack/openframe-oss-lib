A React SVG icon component that renders an image/picture icon with customizable styling and color properties.

## Key Components

- **ImageIcon**: Functional React component that renders an SVG image icon
- **ImageIconProps**: TypeScript interface defining optional `className` and `color` props
- **SVG Elements**: Rectangle frame, circle (representing sun/light), and path (representing mountains/landscape)

## Usage Example

```typescript
import { ImageIcon } from './image-icon';

// Basic usage with default styling
<ImageIcon />

// Custom size and color
<ImageIcon 
  className="w-8 h-8" 
  color="#3B82F6" 
/>

// Using with Tailwind classes
<ImageIcon 
  className="w-12 h-12 text-gray-600" 
/>

// In a button or container
<button className="p-2 hover:bg-gray-100">
  <ImageIcon className="w-5 h-5" color="currentColor" />
</button>
```

The component defaults to `w-6 h-6` sizing and uses `currentColor` for the stroke and fill, making it easy to integrate with existing color schemes and responsive design systems.
