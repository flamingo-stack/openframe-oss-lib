A React component that renders a Telegram icon using SVG with customizable styling and color properties.

## Key Components

- **TelegramIcon**: Main functional component that renders the Telegram icon
- **TelegramIconProps**: TypeScript interface defining the component's props
  - `color?: string` - Stroke color for the icon (defaults to 'white')
  - `className?: string` - Optional CSS class name for styling

## Usage Example

```typescript
import { TelegramIcon } from './telegram-icon';

// Basic usage with default white color
<TelegramIcon />

// Custom color
<TelegramIcon color="#0088cc" />

// With custom styling
<TelegramIcon 
  color="#229ED9" 
  className="telegram-icon hover:opacity-80" 
/>

// In a button component
<button className="flex items-center gap-2">
  <TelegramIcon color="#ffffff" className="w-5 h-5" />
  Contact on Telegram
</button>
```

The component uses Lucide-style SVG structure with a 24x24 viewBox and renders the iconic paper airplane shape representing Telegram's messaging platform. The icon maintains consistent stroke properties and supports responsive sizing through CSS classes.
