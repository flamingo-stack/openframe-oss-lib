Renders a Telegram-style SVG icon as a React component with configurable stroke color and CSS class support.

## Key Components

- **`TelegramIconProps`** — Interface with optional `color` (stroke color, defaults to `'white'`) and `className` (CSS class) props.
- **`TelegramIcon`** — ⚠️ **Deprecated.** Functional component returning a 24×24 SVG icon. Migrate to `icons-v2-generated` instead.

## Usage Example

```typescript
import { TelegramIcon } from './telegram-icon';

// Default white icon
<TelegramIcon />

// Custom color and class
<TelegramIcon color="#229ED9" className="my-icon" />
```

> **Deprecated:** Use icons from `icons-v2-generated` instead of this component.
