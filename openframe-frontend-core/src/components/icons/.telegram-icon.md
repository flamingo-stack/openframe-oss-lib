A React component that renders a Telegram messenger icon as an SVG element with customizable styling options.

## Key Components

- **TelegramIcon**: React functional component that renders a Telegram icon
- **TelegramIconProps**: TypeScript interface defining optional `color` and `className` props
- **Default styling**: 24x24 viewport, white stroke color, 2px stroke width

## Usage Example

```typescript
import { TelegramIcon } from './telegram-icon';

// Basic usage with default white color
<TelegramIcon />

// Custom color
<TelegramIcon color="#0088cc" />

// With custom CSS classes
<TelegramIcon 
  className="w-8 h-8 hover:opacity-80" 
  color="#229ED9" 
/>

// In a button component
<button className="flex items-center gap-2">
  <TelegramIcon color="currentColor" />
  Contact us on Telegram
</button>
```

The icon uses stroke-based SVG paths to create the distinctive Telegram paper plane symbol, making it suitable for both light and dark themes when combined with appropriate color values.
