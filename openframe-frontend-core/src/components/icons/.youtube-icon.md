A React component that renders an SVG YouTube icon with customizable styling and color props.

## Key Components

- **`YouTubeIcon`** - Main component export that renders a scalable YouTube play button icon
- **`YouTubeIconProps`** - TypeScript interface defining optional `color` and `className` props

## Usage Example

```typescript
import { YouTubeIcon } from './youtube-icon';

// Basic usage with default white color
<YouTubeIcon />

// Custom color and styling
<YouTubeIcon 
  color="#ff0000" 
  className="w-6 h-6 hover:scale-110" 
/>

// Integration in a video thumbnail
<div className="relative">
  <img src="thumbnail.jpg" alt="Video" />
  <div className="absolute inset-0 flex items-center justify-center">
    <YouTubeIcon color="red" className="w-12 h-12" />
  </div>
</div>
```

The icon uses SVG paths to create the classic YouTube rounded rectangle with a centered play triangle. The `stroke` attribute controls the outline color while maintaining the transparent fill for versatile styling options.
