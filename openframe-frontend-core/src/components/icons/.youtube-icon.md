A React component that renders a customizable YouTube icon as an SVG element with configurable styling properties.

## Key Components

- **YouTubeIcon**: Main component function that renders the YouTube play button icon
- **YouTubeIconProps**: TypeScript interface defining optional `color` and `className` props
- Default props: `color` defaults to 'white', `className` is optional

## Usage Example

```typescript
import { YouTubeIcon } from './youtube-icon';

// Basic usage with default white color
<YouTubeIcon />

// Custom color and styling
<YouTubeIcon 
  color="#ff0000" 
  className="w-8 h-8 hover:opacity-80" 
/>

// With Tailwind classes
<YouTubeIcon 
  color="currentColor" 
  className="text-red-600 w-6 h-6" 
/>
```

The component uses a 24x24 viewBox SVG with stroke-based rendering, making it scalable and easily customizable through CSS classes or the color prop.
