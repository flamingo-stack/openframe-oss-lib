A React SVG icon component that renders a users/people icon with customizable styling and color properties.

## Key Components

- **`UsersIconProps`** - TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional `color` prop
- **`UsersIcon`** - React functional component that renders an SVG icon showing multiple user silhouettes

## Usage Example

```typescript
import { UsersIcon } from './users-icon';

// Basic usage with defaults (white color, h-6 w-6 size)
<UsersIcon />

// Custom color and styling
<UsersIcon 
  color="#3B82F6" 
  className="h-8 w-8" 
/>

// With additional SVG props
<UsersIcon 
  color="currentColor"
  className="h-5 w-5"
  onClick={() => console.log('Users clicked')}
  aria-label="View users"
/>
```

The component defaults to a 24x24 viewBox with white stroke color and `h-6 w-6` Tailwind classes. All standard SVG props are supported through prop spreading, making it highly flexible for different use cases.
