A collection of custom SVG icon components for OpenFrame AI features, providing consistent visual elements with customizable styling through Tailwind CSS classes.

## Key Components

- **NaturalLanguageIcon** - Chat bubble icon representing natural language processing capabilities
- **AutomatedDiagnosticsIcon** - Gear with settings icon for automated system diagnostics
- **StatusUpdatesIcon** - Bell with notification dot for status update notifications
- **SmartEscalationIcon** - Bidirectional arrows indicating escalation workflows
- **ScriptGenerationIcon** - Terminal/code window icon for script generation features
- **PatchPolicyIcon** - Branching flow diagram representing patch management policies
- **BulkOperationsIcon** - Checklist icon for bulk operation capabilities
- **AuditLoggingIcon** - Document with checkmark for audit logging functionality

## Usage Example

```typescript
import { NaturalLanguageIcon, AutomatedDiagnosticsIcon } from './openframe-ai-icons';

// Default size (w-12 h-12)
<NaturalLanguageIcon />

// Custom size
<AutomatedDiagnosticsIcon className="w-8 h-8 text-blue-500" />

// In a feature grid
<div className="grid grid-cols-2 gap-4">
  <div className="flex items-center space-x-2">
    <NaturalLanguageIcon className="w-6 h-6" />
    <span>Natural Language Processing</span>
  </div>
  <div className="flex items-center space-x-2">
    <AutomatedDiagnosticsIcon className="w-6 h-6" />
    <span>Automated Diagnostics</span>
  </div>
</div>
```

All icons use a consistent golden yellow stroke color (#FFC008) and accept optional `className` props for responsive sizing and additional styling.
