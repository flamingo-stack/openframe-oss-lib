This file exports a collection of SVG icon components for OpenFrame AI features, providing a consistent visual language for AI-powered IT operations tools.

## Key Components

- **NaturalLanguageIcon** - Speech bubble icon representing natural language processing capabilities
- **AutomatedDiagnosticsIcon** - Gear with center circle for automated system diagnostics
- **StatusUpdatesIcon** - Bell with notification indicator for real-time status updates
- **SmartEscalationIcon** - Arrows indicating escalation paths and routing
- **ScriptGenerationIcon** - Terminal/code window representing automated script creation
- **PatchPolicyIcon** - Flowchart diagram for patch management policies
- **BulkOperationsIcon** - Checklist showing multiple operations management
- **AuditLoggingIcon** - Document with checkmark for compliance and logging

## Usage Example

```typescript
import { NaturalLanguageIcon, AutomatedDiagnosticsIcon } from './openframe-ai-icons';

// Use with default sizing (w-12 h-12)
<NaturalLanguageIcon />

// Customize with Tailwind classes
<AutomatedDiagnosticsIcon className="w-8 h-8 text-blue-500" />

// In a feature list
const features = [
  { icon: <NaturalLanguageIcon className="w-6 h-6" />, title: "Natural Language Queries" },
  { icon: <AutomatedDiagnosticsIcon className="w-6 h-6" />, title: "Smart Diagnostics" }
];
```

All icons are styled with `#FFC008` (golden yellow) stroke color and accept an optional `className` prop for customization.
