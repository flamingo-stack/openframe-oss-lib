A reusable React component that renders a customizable magnifying glass search icon as an SVG element.

## Key Components

- **`SearchIcon`**: Main functional component that renders the search icon
- **`SearchIconProps`**: TypeScript interface defining component props:
  - `color`: Icon fill color (default: '#888888')
  - `width`: Icon width in pixels (default: 22)
  - `height`: Icon height in pixels (default: 22)
  - `className`: CSS class name for styling

## Usage Example

```typescript
import { SearchIcon } from './search-icon';

// Basic usage with defaults
<SearchIcon />

// Customized icon
<SearchIcon 
  color="#3b82f6"
  width={24}
  height={24}
  className="hover:opacity-75"
/>

// In a search input
<div className="relative">
  <input type="text" placeholder="Search..." />
  <SearchIcon 
    className="absolute right-3 top-1/2 transform -translate-y-1/2"
    color="#6b7280"
  />
</div>
```

The component features a magnifying glass design with a circular lens and angled handle, making it ideal for search interfaces, input fields, and navigation elements.
