A reusable React SVG component that renders a search icon with customizable styling properties.

## Key Components

- **SearchIcon**: Main functional component that renders an SVG search icon
- **SearchIconProps**: TypeScript interface defining customizable props (color, dimensions, className)

## Usage Example

```typescript
import { SearchIcon } from './search-icon';

function SearchBar() {
  return (
    <div className="search-container">
      {/* Default icon */}
      <SearchIcon />
      
      {/* Customized icon */}
      <SearchIcon 
        width={24}
        height={24}
        color="#0070f3"
        className="search-input-icon"
      />
    </div>
  );
}
```

The component provides sensible defaults (22x22 pixels, gray color) and accepts optional props for customization. The SVG uses a magnifying glass design with proper viewBox scaling and accessible markup.
