A React SVG component that renders the Flamingo brand logo with customizable styling and accessibility features.

## Key Components

- **FlamingoLogo**: Main component that renders an SVG flamingo logo
- **FlamingoLogoProps**: TypeScript interface extending SVG props with custom options
- **Unique ID generation**: Uses React.useId() for accessibility and avoiding ID conflicts

## Props

- `className?: string` - CSS classes for styling
- `fill?: string` - Fill color (defaults to CSS custom property `--ods-flamingo-pink-base`)
- `...props` - All standard SVG element props are supported

## Usage Example

```typescript
import { FlamingoLogo } from './flamingo-logo';

// Basic usage with default pink color
<FlamingoLogo />

// Custom styling and color
<FlamingoLogo 
  className="w-8 h-8" 
  fill="#ff6b9d" 
  aria-label="Flamingo logo"
/>

// Responsive logo with custom props
<FlamingoLogo 
  className="logo-header" 
  width={60} 
  height={60}
  role="img"
/>
```

The component generates unique IDs for the SVG path to prevent conflicts when multiple logos are rendered on the same page, and follows accessibility best practices by accepting ARIA attributes through the spread props.
