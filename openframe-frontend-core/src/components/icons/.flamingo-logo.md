A React SVG component that renders the Flamingo logo with customizable styling and accessibility features.

## Key Components

- **FlamingoLogoProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional `className` and `fill` properties
- **FlamingoLogo**: Main functional component that renders an SVG flamingo logo with default pink styling
- **uniqueId**: Generated using `React.useId()` to ensure unique element IDs for accessibility

## Usage Example

```typescript
import { FlamingoLogo } from './flamingo-logo';

// Basic usage with default pink color
<FlamingoLogo />

// Custom styling with className and fill color
<FlamingoLogo 
  className="logo-header" 
  fill="#ff6b9d" 
  width={60}
  height={60}
/>

// With additional SVG props
<FlamingoLogo 
  fill="currentColor"
  onClick={() => console.log('Logo clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component uses CSS custom properties by default (`--ods-flamingo-pink-base`) and accepts all standard SVG element props through prop spreading.
