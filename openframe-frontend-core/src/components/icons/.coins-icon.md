A React functional component that renders an SVG icon depicting two overlapping coins, commonly used to represent currency, payments, or financial features.

## Key Components

- **CoinsIcon** - Main React component that renders the SVG icon
- **CoinsIconProps** - TypeScript interface defining component props:
  - `color` - SVG stroke color (default: 'white')
  - `className` - CSS class for styling
  - `width` - Icon width in pixels (default: 20)
  - `height` - Icon height in pixels (default: 20)

## Usage Example

```typescript
import { CoinsIcon } from './coins-icon';

// Basic usage with defaults
<CoinsIcon />

// Customized appearance
<CoinsIcon 
  color="#3B82F6" 
  width={24} 
  height={24}
  className="hover:opacity-80 transition-opacity"
/>

// In a button or navigation element
<button className="flex items-center gap-2">
  <CoinsIcon color="#10B981" width={16} height={16} />
  Wallet Balance
</button>
```

The icon features two circular coins with decorative curved elements, using stroke-based rendering for clean scalability. All visual properties are customizable through props, making it suitable for various UI contexts like navigation menus, dashboards, or payment interfaces.
