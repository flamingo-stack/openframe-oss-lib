A React functional component that renders an SVG icon depicting two overlapping coins, commonly used in financial or gaming applications to represent currency or rewards.

## Key Components

- **CoinsIcon**: Main React functional component that renders the coins SVG
- **CoinsIconProps**: TypeScript interface defining optional props for customization
  - `color`: SVG stroke color (default: 'white')
  - `className`: CSS class name for styling
  - `width`/`height`: Icon dimensions (default: 20x20)

## Usage Example

```typescript
import { CoinsIcon } from './coins-icon';

// Basic usage with defaults
<CoinsIcon />

// Customized appearance
<CoinsIcon 
  color="#FFD700" 
  width={32} 
  height={32}
  className="my-coins-icon"
/>

// In a button or card
<button className="flex items-center gap-2">
  <CoinsIcon color="currentColor" width={16} height={16} />
  Collect Coins
</button>
```

The icon features two circular outlines representing coins with small curved elements suggesting depth and overlap, making it ideal for indicating monetary values, points, or collectible items in user interfaces.
