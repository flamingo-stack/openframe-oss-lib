A customizable React SVG icon component that renders the GetApp logo with configurable colors and dimensions.

## Key Components

- **GetAppIconProps**: Interface defining component props including dimensions, styling, and six customizable color properties
- **GetAppIcon**: Main component function that renders an SVG with multiple colored paths forming the GetApp logo
- **Default Colors**: Teal/cyan color palette (#009C9C, #40E3E3, #1ABAB8, #87F0F0, #17CFCF, #424A52)

## Usage Example

```typescript
import { GetAppIcon } from './getapp-icon';

// Basic usage with defaults
<GetAppIcon />

// Customized size and colors
<GetAppIcon 
  width={48}
  height={48}
  className="my-icon"
  firstColor="#FF0000"
  secondColor="#00FF00"
  thirdColor="#0000FF"
/>

// Responsive sizing
<GetAppIcon 
  width="100%" 
  height="auto"
  className="responsive-icon"
/>
```

The component renders an intricate geometric logo with six distinct colored sections, each controllable via color props. The SVG uses a fixed viewBox (10 260 170 90) optimized for the logo's proportions, making it scalable while maintaining visual integrity.
