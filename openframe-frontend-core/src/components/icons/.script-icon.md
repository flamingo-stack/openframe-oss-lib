A React SVG icon component that renders a script or code brackets symbol with customizable styling properties.

## Key Components

- **ScriptIcon**: Main functional component that renders an SVG icon
- **Props**:
  - `className`: Optional CSS classes (defaults to "w-5 h-5")
  - `size`: Icon dimensions in pixels (defaults to 22)
  - `color`: Fill color (defaults to 'white')

## Usage Example

```typescript
import { ScriptIcon } from './script-icon'

// Basic usage with defaults
<ScriptIcon />

// Customized icon
<ScriptIcon 
  size={32} 
  color="#3b82f6" 
  className="mr-2 text-blue-500" 
/>

// In a button
<button className="flex items-center">
  <ScriptIcon size={16} color="currentColor" className="mr-1" />
  Run Script
</button>
```

The icon displays curly braces/brackets typically used to represent code, scripts, or programming-related functionality. The SVG uses two path elements to create the left and right bracket shapes with rounded corners and connecting elements.
