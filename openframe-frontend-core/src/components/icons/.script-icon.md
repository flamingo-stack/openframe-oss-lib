Brief description of the file's purpose: Renders a deprecated SVG script/code bracket icon component for use in React applications.

## Key Components

- **`ScriptIcon`** — A React functional component that renders an SVG icon resembling curly braces `{ }`, commonly used to represent scripts or code blocks.

> ⚠️ **Deprecated:** This component is marked as deprecated. Use icons from `icons-v2-generated` instead.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `"w-5 h-5"` | Tailwind/CSS class for sizing and styling |
| `size` | `number` | `22` | SVG width and height in pixels |
| `color` | `string` | `"white"` | SVG fill color |

## Usage Example

```typescript
import { ScriptIcon } from './script-icon'

// Default usage
<ScriptIcon />

// Custom size and color
<ScriptIcon size={32} color="#FFC008" className="w-8 h-8" />
```

## Migration

Replace usages with the equivalent icon from the `icons-v2-generated` package:

```typescript
// ❌ Deprecated
import { ScriptIcon } from './script-icon'

// ✅ Preferred
import { ScriptIcon } from '@/components/icons-v2-generated'
```
