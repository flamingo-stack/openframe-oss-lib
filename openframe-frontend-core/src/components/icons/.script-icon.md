A React functional component that renders an SVG icon representing curly braces, commonly used to indicate script or code content.

## Key Components

- **ScriptIcon**: The main React component that renders an SVG with curly brace paths
- **Props interface**: Accepts `className`, `size`, and `color` props for customization
- **SVG paths**: Two path elements that create left and right curly brace shapes

## Usage Example

```typescript
import { ScriptIcon } from './script-icon'

// Basic usage with defaults
<ScriptIcon />

// Custom sizing and styling
<ScriptIcon 
  size={32} 
  color="blue" 
  className="w-8 h-8 mr-2" 
/>

// In a button or navigation item
<button className="flex items-center">
  <ScriptIcon size={16} color="currentColor" />
  <span>Scripts</span>
</button>
```

The component provides sensible defaults (22px size, white color, w-5 h-5 classes) while allowing full customization through props. The SVG uses a 24x22 viewBox and renders symmetrical curly braces that scale cleanly at any size.
