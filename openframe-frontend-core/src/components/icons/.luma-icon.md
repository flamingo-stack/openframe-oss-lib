A React component that renders the Luma brand icon as an SVG element with customizable size and styling properties.

## Key Components

- **LumaIcon**: Main component that renders a star-shaped SVG icon
- **LumaIconProps**: Interface extending SVGProps with optional className and size properties
- **Props**: Accepts `className`, `size` (default: 24), `color` (default: 'white'), and all standard SVG attributes

## Usage Example

```typescript
import { LumaIcon } from './luma-icon';

// Basic usage with defaults
<LumaIcon />

// Customized size and color
<LumaIcon size={48} color="#3b82f6" />

// With custom styling
<LumaIcon 
  className="hover:opacity-80 transition-opacity" 
  size={32}
  color="currentColor"
/>

// With additional SVG props
<LumaIcon 
  size={40}
  onClick={() => console.log('Icon clicked')}
  role="button"
  tabIndex={0}
/>
```

The component maintains a fixed aspect ratio (133:134) and scales proportionally based on the size prop. All unused props are forwarded to the underlying SVG element for maximum flexibility.
