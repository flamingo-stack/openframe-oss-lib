A React component that renders an SVG icon displaying a placeholder image for vendors when no specific vendor icon is available.

## Key Components

- **EmptyVendorIcon**: React functional component that renders a 24x24px SVG with an embedded placeholder image
- **EmptyVendorIconProps**: Interface defining optional `className` prop for styling customization

## Usage Example

```typescript
import { EmptyVendorIcon } from './empty-vendor-icon';

// Basic usage
function VendorCard() {
  return (
    <div>
      <EmptyVendorIcon />
      <span>Unknown Vendor</span>
    </div>
  );
}

// With custom styling
function StyledVendorIcon() {
  return (
    <EmptyVendorIcon 
      className="w-8 h-8 opacity-50 grayscale" 
    />
  );
}

// In a vendor list fallback
function VendorLogo({ vendor }) {
  return vendor.icon ? (
    <img src={vendor.icon} alt={vendor.name} />
  ) : (
    <EmptyVendorIcon className="vendor-placeholder" />
  );
}
```

The component uses an embedded base64-encoded PNG image within an SVG pattern, making it self-contained without external dependencies. The rounded rectangle design provides a clean fallback appearance for missing vendor logos.
