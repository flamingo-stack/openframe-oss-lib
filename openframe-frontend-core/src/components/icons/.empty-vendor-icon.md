A React component that renders a placeholder SVG icon for vendors without custom icons. The component displays a generic patterned square icon with rounded corners.

## Key Components

- **EmptyVendorIcon** - Main functional component that renders an SVG placeholder icon
- **EmptyVendorIconProps** - TypeScript interface defining optional className prop for styling customization

## Usage Example

```typescript
import { EmptyVendorIcon } from './empty-vendor-icon';

// Basic usage
<EmptyVendorIcon />

// With custom styling
<EmptyVendorIcon className="w-8 h-8 text-gray-400" />

// In a vendor list where some items lack icons
{vendors.map(vendor => (
  <div key={vendor.id}>
    {vendor.icon ? (
      <img src={vendor.icon} alt={vendor.name} />
    ) : (
      <EmptyVendorIcon className="vendor-placeholder" />
    )}
    <span>{vendor.name}</span>
  </div>
))}
```

The SVG is embedded with a base64-encoded image pattern that creates a subtle textured appearance. The icon maintains a 24x24 viewBox and accepts standard CSS classes for responsive sizing and theming.
