A deprecated SVG thumbs-down icon component with configurable size and color props.

## Key Components

- **`ThumbsDownIconProps`** — Interface with optional `className` (default: `'w-6 h-6'`) and `color` (default: `#5EFAF0`) props
- **`ThumbsDownIcon`** — Deprecated functional component rendering a 23×23 SVG thumbs-down icon

> ⚠️ **Deprecated:** Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { ThumbsDownIcon } from './thumbs-down-icon';

// Default usage
<ThumbsDownIcon />

// Custom size and color
<ThumbsDownIcon className="w-8 h-8" color="#FF0000" />
```
