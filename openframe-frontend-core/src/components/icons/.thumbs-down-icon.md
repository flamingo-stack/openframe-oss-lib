A React SVG icon component that renders a thumbs down symbol with customizable styling and color properties.

## Key Components

- **ThumbsDownIconProps** - Interface defining optional `className` and `color` props
- **ThumbsDownIcon** - Functional component that renders an SVG thumbs down icon with default dimensions (w-6 h-6) and cyan color (#5EFAF0)

## Usage Example

```typescript
import { ThumbsDownIcon } from './thumbs-down-icon';

// Basic usage with defaults
<ThumbsDownIcon />

// Custom styling and color
<ThumbsDownIcon 
  className="w-8 h-8" 
  color="#ff0000" 
/>

// For feedback systems
<button onClick={handleDislike}>
  <ThumbsDownIcon className="w-5 h-5" color="#dc2626" />
  Dislike
</button>
```

The component uses a 23x23 viewBox with two path elements to create the thumbs down shape, making it suitable for user feedback interfaces, rating systems, or general UI interactions where negative feedback indication is needed.
