A React component that renders a customizable thumbs-down SVG icon, commonly used for negative feedback or dislike functionality.

## Key Components

- **ThumbsDownIconProps**: Interface defining optional `className` and `color` props
- **ThumbsDownIcon**: Functional component that renders an SVG thumbs-down icon with customizable styling

## Usage Example

```typescript
import { ThumbsDownIcon } from './thumbs-down-icon';

// Default styling (w-6 h-6, cyan color)
<ThumbsDownIcon />

// Custom size and color
<ThumbsDownIcon 
  className="w-8 h-8" 
  color="#ff0000" 
/>

// In a feedback component
const FeedbackButtons = () => (
  <div className="flex gap-2">
    <button onClick={handleDislike}>
      <ThumbsDownIcon className="w-5 h-5" color="#ef4444" />
    </button>
  </div>
);
```

The component uses Tailwind CSS classes by default and accepts any valid CSS color value for customization. The SVG is optimized with a 23x23 viewBox and uses fill paths for crisp rendering at any size.
