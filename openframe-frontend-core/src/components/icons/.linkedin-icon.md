A reusable React component that renders a LinkedIn icon as an SVG element with customizable styling and size properties.

## Key Components

- **`LinkedInIcon`** - Main component that renders the LinkedIn logo SVG
- **`LinkedInIconProps`** - Interface extending SVG props with optional `className`, `size`, and `color` properties

## Usage Example

```typescript
import { LinkedInIcon } from './linkedin-icon';

// Basic usage with default props
<LinkedInIcon />

// Custom size and styling
<LinkedInIcon 
  size={32} 
  color="#0077B5" 
  className="hover:opacity-80 transition-opacity" 
/>

// With click handler
<LinkedInIcon 
  size={28}
  onClick={() => window.open('https://linkedin.com/profile')}
  className="cursor-pointer"
/>
```

The component defaults to 24px size with white color and accepts all standard SVG element props through spread operators, making it flexible for various use cases like buttons, navigation menus, or social media links.
