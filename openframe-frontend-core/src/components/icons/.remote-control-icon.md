A React component that renders a customizable SVG remote control icon with two interlocking circular shapes.

## Key Components

- **RemoteControlIcon**: React functional component that accepts optional props for styling and appearance
  - `className`: CSS classes for styling (default: "w-5 h-5")
  - `size`: Icon dimensions in pixels (default: 22)
  - `color`: Fill color for the icon (default: 'white')

## Usage Example

```typescript
import { RemoteControlIcon } from './remote-control-icon'

// Basic usage with defaults
<RemoteControlIcon />

// Customized appearance
<RemoteControlIcon 
  size={32} 
  color="#3B82F6" 
  className="w-8 h-8 hover:opacity-80" 
/>

// In a button component
<button className="p-2 bg-blue-500 rounded">
  <RemoteControlIcon color="white" size={20} />
  Connect Device
</button>
```

The icon features two interlocking circular paths that visually represent a remote connection or pairing concept, making it suitable for UI elements related to device control, wireless connections, or remote access functionality.
