A React component that renders an SVG icon representing a remote control or wireless connection symbol, featuring a circular chain-like design with customizable styling properties.

## Key Components

- **RemoteControlIcon**: Main functional component that renders an SVG icon
- **Props interface**: Accepts `className`, `size`, and `color` properties for customization
- **SVG paths**: Two path elements creating a chain-link or remote control visual

## Usage Example

```typescript
import { RemoteControlIcon } from './remote-control-icon'

// Default usage (white, 22px)
<RemoteControlIcon />

// Custom styling
<RemoteControlIcon 
  className="w-8 h-8 text-blue-500" 
  size={32} 
  color="currentColor" 
/>

// Different colors and sizes
<RemoteControlIcon size={16} color="#FF6B6B" />
<RemoteControlIcon className="hover:opacity-80" color="black" />
```

The component uses default values of 22px size, white color, and Tailwind CSS classes for dimensions. The `color` prop directly sets the SVG fill, while `className` allows for additional styling including hover states and responsive classes.
