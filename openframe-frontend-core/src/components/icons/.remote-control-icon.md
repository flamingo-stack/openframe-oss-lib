SVG icon component for representing remote control functionality, marked as deprecated in favor of the newer `icons-v2-generated` icon library.

## Key Components

**`RemoteControlIcon`** — A React functional component that renders a remote control SVG icon with two path elements forming a circular/rotated square shape with a small accent mark.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `"w-5 h-5"` | Tailwind/CSS class for sizing |
| `size` | `number` | `22` | Width and height in pixels |
| `color` | `string` | `"white"` | SVG fill color |

## Usage Example

```typescript
import { RemoteControlIcon } from './remote-control-icon'

// Default usage
<RemoteControlIcon />

// Custom size and color
<RemoteControlIcon size={32} color="#FFC008" className="w-8 h-8" />
```

> **⚠️ Deprecated:** This component is deprecated. Use icons from `icons-v2-generated` instead.
