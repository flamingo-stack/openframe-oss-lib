A React SVG component that renders the X (formerly Twitter) logo with customizable styling properties.

## Key Components

- **XLogo**: Main functional component that renders an SVG version of the X logo
- **XLogoProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional className

## Usage Example

```typescript
import { XLogo } from './x-logo';

// Basic usage with default white color
<XLogo />

// Custom styling with className and color override
<XLogo 
  className="hover:opacity-80 cursor-pointer" 
  color="#1DA1F2" 
  width={32} 
  height={32}
/>

// With click handler and custom props
<XLogo 
  className="social-icon"
  onClick={() => window.open('https://x.com')}
  style={{ transition: 'opacity 0.2s' }}
/>
```

The component accepts all standard SVG props and defaults to 24x24 pixels with white fill color. The logo consists of two path elements that form the distinctive X symbol.
