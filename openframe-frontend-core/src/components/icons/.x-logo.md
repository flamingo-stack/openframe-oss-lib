This React component renders the X (formerly Twitter) logo as an SVG icon with customizable properties and styling.

## Key Components

- **XLogo**: Main component that renders the X logo SVG
- **XLogoProps**: Interface extending SVG props with optional className
- **SVG paths**: Two path elements defining the X logo geometry

## Usage Example

```typescript
import { XLogo } from './x-logo';

// Basic usage with default white color
function SocialLinks() {
  return (
    <div className="social-icons">
      <XLogo />
    </div>
  );
}

// Custom styling and color
function Header() {
  return (
    <nav>
      <XLogo 
        className="w-8 h-8 hover:opacity-80" 
        color="#1DA1F2" 
        width={32}
        height={32}
      />
    </nav>
  );
}

// With click handler
function ShareButton() {
  return (
    <button onClick={() => window.open('https://x.com/share')}>
      <XLogo className="mr-2" />
      Share on X
    </button>
  );
}
```

The component accepts all standard SVG properties through props spreading, making it highly flexible for different use cases while maintaining a consistent 24x24 default size and white fill color.
