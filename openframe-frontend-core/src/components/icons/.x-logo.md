SVG icon component rendering the X (formerly Twitter) logo, marked as deprecated in favor of the `icons-v2-generated` icon library.

## Key Components

- **`XLogoProps`** — Extends `React.SVGProps<SVGSVGElement>` with an optional `className` prop, allowing full SVG attribute passthrough
- **`XLogo`** — Deprecated functional component rendering a 24×24 X logo SVG with two filled paths

## Usage Example

```typescript
import { XLogo } from './x-logo';

// Basic usage
<XLogo />

// With custom class and color
<XLogo className="social-icon" color="black" />

// With SVG passthrough props
<XLogo width={32} height={32} aria-label="X (Twitter)" />
```

> **Deprecated:** Use icons from `icons-v2-generated` instead of this component.
