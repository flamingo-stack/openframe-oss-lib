A React SVG icon component that renders the Python programming language logo with customizable styling options.

## Key Components

- **PythonIconProps**: TypeScript interface defining optional props for size, className, and color customization
- **PythonIcon**: React functional component that renders an SVG representation of the Python logo
- **displayName**: Set to 'PythonIcon' for debugging purposes

## Usage Example

```typescript
import { PythonIcon } from './python-icon'

// Default usage (16px, white color)
<PythonIcon />

// Custom size and color
<PythonIcon size={32} color="#3776ab" />

// With additional CSS classes
<PythonIcon 
  size={24} 
  className="hover:opacity-80 transition-opacity" 
  color="currentColor" 
/>

// In a list of tech stack icons
<div className="flex gap-2">
  <PythonIcon size={20} color="#3776ab" />
  <span>Python</span>
</div>
```

The component uses SVG paths to create the iconic Python logo shape, making it scalable and lightweight for use in tech stacks, skill lists, or anywhere a Python identifier is needed.
