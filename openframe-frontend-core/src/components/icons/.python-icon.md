A React component that renders a scalable Python logo SVG icon with customizable size, color, and styling properties.

## Key Components

- **PythonIconProps**: Interface defining the component's props
  - `size?: number` - Icon dimensions (default: 16px)
  - `className?: string` - Additional CSS classes
  - `color?: string` - Fill color (default: 'white')
- **PythonIcon**: Main functional component that renders the Python logo as an SVG
- **displayName**: Set to 'PythonIcon' for debugging purposes

## Usage Example

```typescript
import { PythonIcon } from './python-icon'

// Basic usage with defaults
<PythonIcon />

// Customized icon
<PythonIcon 
  size={24} 
  color="#3776ab" 
  className="programming-language-icon" 
/>

// Large icon with custom styling
<PythonIcon 
  size={48} 
  color="currentColor" 
  className="tech-stack-logo hover:opacity-80" 
/>
```

The component renders the official Python logo with its characteristic interlocking snake design. The SVG is optimized with a 1024x1024 viewBox and uses fill-based coloring for easy customization. Perfect for tech stacks, programming language indicators, or anywhere you need a recognizable Python representation.
