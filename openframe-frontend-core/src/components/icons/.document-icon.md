A React component that renders a customizable document icon as an SVG element, featuring a document with folded corner and text lines.

## Key Components

- **`DocumentIconProps`** - Interface extending `React.SVGProps<SVGSVGElement>` with an optional `color` prop
- **`DocumentIcon`** - Main component function that renders the document SVG icon
- **Props**: `color` (defaults to 'white'), `className`, and all standard SVG props

## Usage Example

```typescript
import { DocumentIcon } from './document-icon'

// Basic usage with default white color
<DocumentIcon />

// Custom color and styling
<DocumentIcon 
  color="#333333" 
  className="w-6 h-6" 
  width={32} 
  height={32}
/>

// With click handler
<DocumentIcon 
  color="blue" 
  onClick={() => console.log('Document clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component renders a 24x24 SVG document icon with a folded top-right corner and two horizontal lines representing text content. All standard SVG props are supported through prop spreading.
