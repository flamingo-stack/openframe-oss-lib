A React component that renders an SVG archive/box icon with customizable styling and color properties.

## Key Components

- **ArchiveIconProps**: Interface extending React.SVGProps with optional `color` prop
- **ArchiveIcon**: Functional component that renders a 24x24 SVG archive icon with default styling

## Usage Example

```typescript
import { ArchiveIcon } from './archive-icon';

// Basic usage with defaults (white color, h-6 w-6 size)
<ArchiveIcon />

// Custom color
<ArchiveIcon color="blue" />

// Custom styling and props
<ArchiveIcon 
  className="h-8 w-8" 
  color="#333" 
  onClick={handleClick}
/>

// With additional SVG props
<ArchiveIcon 
  color="red"
  style={{ cursor: 'pointer' }}
  aria-label="Archive item"
/>
```

The component accepts all standard SVG props and provides sensible defaults for quick implementation in UI libraries like Tailwind CSS.
