A React component that renders the Microsoft logo as an SVG icon with customizable dimensions and styling.

## Key Components

- **`MicrosoftIcon`** - Main component that renders the four-square Microsoft logo
- **`MsIconProps`** - TypeScript interface defining optional props for width, height, and className
- **SVG paths** - Four colored squares representing the Microsoft brand colors (orange, green, blue, yellow)

## Usage Example

```typescript
import { MicrosoftIcon } from './ms-icon'

// Default usage (24x24px)
<MicrosoftIcon />

// Custom size
<MicrosoftIcon width={32} height={32} />

// With custom styling
<MicrosoftIcon 
  width="48" 
  height="48" 
  className="hover:opacity-80 transition-opacity" 
/>

// In a button
<button className="flex items-center gap-2">
  <MicrosoftIcon width={20} height={20} />
  Sign in with Microsoft
</button>
```

The component uses fixed SVG paths with the official Microsoft brand colors and maintains aspect ratio regardless of the provided dimensions. The `className` prop allows for additional styling and CSS utilities.
