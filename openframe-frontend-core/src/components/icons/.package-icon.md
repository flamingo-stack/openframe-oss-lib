A reusable React SVG icon component that renders a package/box icon with customizable styling and color properties.

## Key Components

- **PackageIconProps**: TypeScript interface extending React's SVGProps for type safety
- **PackageIcon**: Functional component that renders an SVG package icon with default sizing and styling

## Usage Example

```typescript
import { PackageIcon } from './package-icon'

// Basic usage with defaults (24x24px, white color)
<PackageIcon />

// Custom styling and color
<PackageIcon 
  className="h-8 w-8" 
  color="blue" 
/>

// With additional SVG props
<PackageIcon 
  className="h-4 w-4" 
  color="#333" 
  onClick={handleClick}
  role="button"
  aria-label="Package icon"
/>
```

The component accepts all standard SVG props through prop spreading, making it highly flexible for different use cases. The default dimensions are 24x24 pixels with a white stroke color, but these can be easily overridden through the `className` and `color` props respectively.
