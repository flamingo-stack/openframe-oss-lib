Deprecated SVG Instagram icon React component with configurable stroke color and className props.

## Key Components

- **`InstagramIconProps`** — Interface with optional `color` (default: `'white'`) and `className` string props
- **`InstagramIcon`** — ⚠️ Deprecated functional component rendering a 24×24 SVG Instagram icon using stroke-based styling

> **Deprecated:** Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { InstagramIcon } from './instagram-icon';

// Default white icon
<InstagramIcon />

// Custom color and class
<InstagramIcon color="#E1306C" className="social-icon" />
```
