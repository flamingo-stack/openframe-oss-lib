A React component that renders an SVG Instagram icon with customizable styling and color options.

## Key Components

- **`InstagramIcon`** - Main functional component that renders the Instagram logo SVG
- **`InstagramIconProps`** - TypeScript interface defining component props:
  - `color?: string` - Sets the stroke color (defaults to 'white')
  - `className?: string` - Applies custom CSS classes

## Usage Example

```typescript
import { InstagramIcon } from './instagram-icon';

// Basic usage with default white color
<InstagramIcon />

// Custom color and styling
<InstagramIcon 
  color="#E4405F" 
  className="hover:scale-110 transition-transform" 
/>

// In a social media links section
<div className="flex gap-4">
  <InstagramIcon color="#000" className="w-6 h-6" />
  <InstagramIcon color="currentColor" className="text-blue-500" />
</div>
```

The component uses a 24x24 viewBox and includes the characteristic Instagram camera icon with rounded square frame, circular lens, and viewfinder dot. The SVG is designed with stroke-based rendering for clean scaling at any size.
