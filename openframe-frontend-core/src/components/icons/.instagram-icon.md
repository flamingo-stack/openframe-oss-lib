A React component that renders an Instagram icon as an SVG element with customizable styling properties.

## Key Components

- **InstagramIcon**: React functional component that renders the Instagram logo as an SVG
- **InstagramIconProps**: TypeScript interface defining optional `color` and `className` props

## Usage Example

```typescript
import { InstagramIcon } from './instagram-icon';

// Basic usage with default white color
<InstagramIcon />

// With custom color
<InstagramIcon color="#E4405F" />

// With CSS classes for styling
<InstagramIcon 
  className="hover:scale-110 transition-transform" 
  color="#833AB4" 
/>

// In a social media links section
<div className="flex space-x-4">
  <InstagramIcon 
    color="currentColor" 
    className="w-6 h-6 text-gray-600 hover:text-pink-500" 
  />
</div>
```

The component uses a 24x24 viewBox and renders the classic Instagram camera icon with a rounded square border, circular lens, and dot flash indicator. The SVG is optimized for crisp rendering at various sizes and supports standard stroke-based styling.
