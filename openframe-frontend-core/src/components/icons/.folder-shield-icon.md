A React component that renders an SVG icon combining a folder with a security shield, typically used to represent secured or protected directories and file systems.

## Key Components

- **FolderShieldIcon**: Main React functional component that renders the combined folder-shield SVG icon
- **FolderShieldIconProps**: TypeScript interface defining optional props for customization (className, size, color)

## Usage Example

```typescript
import { FolderShieldIcon } from './folder-shield-icon'

// Basic usage with defaults (24px, white color)
<FolderShieldIcon />

// Customized with different size and color
<FolderShieldIcon 
  size={32} 
  color="#3B82F6" 
  className="hover:opacity-80" 
/>

// Large icon for headers or prominent displays
<FolderShieldIcon 
  size={48} 
  color="currentColor" 
  className="text-green-600" 
/>
```

The icon features a folder outline with an overlaid shield symbol, making it ideal for UI elements representing secure folders, encrypted directories, or protected file access areas. The SVG is scalable and customizable through props, with sensible defaults for general use cases.
