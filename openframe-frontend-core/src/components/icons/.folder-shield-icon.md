A React component that renders an SVG icon depicting a folder with a security shield overlay, commonly used for representing protected or secure directories.

## Key Components

- **FolderShieldIcon**: Main React functional component that renders the SVG icon
- **FolderShieldIconProps**: TypeScript interface defining component props:
  - `className` - Optional CSS class name for styling
  - `size` - Icon dimensions in pixels (default: 24)
  - `color` - Stroke color for the icon paths (default: 'white')

## Usage Example

```typescript
import { FolderShieldIcon } from './folder-shield-icon';

// Basic usage with defaults
<FolderShieldIcon />

// Customized icon
<FolderShieldIcon 
  size={32}
  color="#3b82f6"
  className="my-folder-icon"
/>

// In a component
const SecureFolder = () => (
  <div className="folder-item">
    <FolderShieldIcon size={20} color="currentColor" />
    <span>Protected Documents</span>
  </div>
);
```

The SVG uses a 65x64 viewBox and consists of two stroke paths - one for the folder outline and another for the shield shape. The icon scales proportionally and accepts any valid CSS color value.
