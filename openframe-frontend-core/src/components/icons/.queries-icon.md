A React SVG icon component that renders a queries or bracket-style icon with customizable styling properties.

## Key Components

- **QueriesIconProps** - Interface extending React's SVGProps with an optional `color` prop
- **QueriesIcon** - Functional component that renders an SVG with bracket-like shapes and dots

## Usage Example

```typescript
import { QueriesIcon } from './queries-icon'

// Basic usage with default white color
<QueriesIcon />

// Custom color and className
<QueriesIcon 
  color="#3b82f6" 
  className="w-6 h-6" 
/>

// With additional SVG props
<QueriesIcon 
  color="currentColor"
  width={32}
  height={32}
  onClick={handleClick}
/>
```

The icon features a symmetrical bracket design with three circular elements in the center, making it suitable for representing database queries, data connections, or API endpoints. The component accepts standard SVG props and defaults to 24x24 dimensions with white fill color.
