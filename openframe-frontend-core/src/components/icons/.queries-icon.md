Deprecated SVG icon component rendering a "queries" symbol (curly braces with dots) at 24×24px.

## Key Components

- **`QueriesIconProps`** — Extends `React.SVGProps<SVGSVGElement>` with an optional `color` prop (defaults to `'white'`)
- **`QueriesIcon`** — Functional component that renders a 24×24 SVG queries icon

> ⚠️ **Deprecated:** Both the interface and component are marked as deprecated. Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { QueriesIcon } from './queries-icon'

// Basic usage
<QueriesIcon />

// With custom color and className
<QueriesIcon color="#FFC008" className="my-icon" />

// With inherited SVG props
<QueriesIcon color="black" width={32} height={32} aria-label="Queries" />
```
