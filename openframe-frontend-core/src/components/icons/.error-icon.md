A React SVG component that renders a red error icon, designed for displaying error states throughout the application.

## Key Components

- **ErrorIcon**: Main component that renders an SVG error icon
- **Props**: Extends `React.SVGProps<SVGSVGElement>` with optional `className` and `fill` props
- **Default styling**: Red fill color (`#FF6B6B`) and inline-block display

## Usage Example

```typescript
import { ErrorIcon } from './error-icon';

// Basic usage
<ErrorIcon />

// With custom size and styling
<ErrorIcon className="h-8 w-8 text-red-500" />

// With custom fill color
<ErrorIcon fill="#DC2626" className="h-6 w-6" />

// In error states
{isError && (
  <div className="flex items-center gap-2">
    <ErrorIcon className="h-5 w-5" />
    <span>Something went wrong</span>
  </div>
)}
```

The component automatically merges CSS classes using the `cn` utility and accepts all standard SVG props for maximum flexibility.
