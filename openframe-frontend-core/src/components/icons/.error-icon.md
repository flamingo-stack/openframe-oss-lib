A React SVG component that renders an error icon with customizable styling and fill color for displaying error states in the UI.

## Key Components

- **ErrorIcon**: Main component that renders an SVG error icon
- **Props**: Extends `React.SVGProps<SVGSVGElement>` with custom `fill` prop
- **Default styling**: Red fill color (`#FF6B6B`) and inline-block display
- **Utility integration**: Uses `cn` utility for className merging

## Usage Example

```typescript
import { ErrorIcon } from './error-icon';

// Basic usage with default red color
<ErrorIcon className="h-6 w-6" />

// Custom color and size
<ErrorIcon 
  className="h-12 w-12" 
  fill="#DC2626" 
/>

// With additional SVG props
<ErrorIcon 
  className="h-8 w-8"
  onClick={handleClick}
  role="button"
  aria-label="Error indicator"
/>
```

The component accepts all standard SVG properties and automatically applies responsive sizing through className props. The default fill color provides good contrast for error states, but can be customized for different design systems.
