A React component that renders an SVG community/group icon with customizable styling properties.

## Key Components

- **CommunityIcon**: Main React functional component that renders a community SVG icon
- **CommunityIconProps**: TypeScript interface defining optional props for customization
  - `color`: Fill color for the icon (default: 'white')
  - `className`: CSS class name for styling
  - `width`: Icon width in pixels (default: 24)
  - `height`: Icon height in pixels (default: 24)

## Usage Example

```typescript
import { CommunityIcon } from './community-icon';

// Basic usage with defaults
<CommunityIcon />

// Customized icon
<CommunityIcon 
  color="#3B82F6" 
  width={32} 
  height={32}
  className="hover:opacity-75"
/>

// In a button component
<button className="flex items-center gap-2">
  <CommunityIcon color="#059669" width={20} height={20} />
  Join Community
</button>
```

The component uses a fixed 24x24 viewBox and scales accordingly based on the provided dimensions. The SVG depicts multiple user figures representing a community or group, making it suitable for social features, team sections, or community-related UI elements.
