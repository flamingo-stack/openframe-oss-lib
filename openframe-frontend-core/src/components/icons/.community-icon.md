A React component that renders an SVG community icon representing people/users, commonly used for community-related features and navigation.

## Key Components

- **CommunityIcon**: React functional component that renders a scalable SVG icon
- **CommunityIconProps**: TypeScript interface defining customizable properties
- **SVG Path Elements**: Two path elements creating a community/users icon design

## Usage Example

```typescript
import { CommunityIcon } from './community-icon';

// Basic usage with defaults
<CommunityIcon />

// Customized appearance
<CommunityIcon 
  width={32} 
  height={32} 
  color="#3B82F6" 
  className="hover:opacity-80" 
/>

// Large icon for headers
<CommunityIcon 
  width={48} 
  height={48} 
  color="currentColor" 
/>
```

The component accepts optional props for `width` (24px default), `height` (24px default), `color` (white default), and `className` for additional CSS styling. The SVG uses a 24x24 viewBox and scales proportionally based on the provided dimensions.
