A React SVG icon component that renders an information circle symbol (a circle with an "i" inside) for displaying informational content or help indicators.

## Key Components

- **InfoCircleIcon** - Main functional component that renders the SVG icon
- **InfoCircleIconProps** - TypeScript interface defining component props including styling options
- **Flexible sizing system** - Supports both unified `size` prop or separate `width`/`height` props

## Usage Example

```typescript
import { InfoCircleIcon } from './info-circle-icon';

function HelpTooltip() {
  return (
    <div className="help-container">
      {/* Basic usage with default size (24px) */}
      <InfoCircleIcon />
      
      {/* Custom size and color */}
      <InfoCircleIcon 
        size={20} 
        color="#3B82F6" 
        className="ml-2" 
      />
      
      {/* Separate width and height */}
      <InfoCircleIcon 
        width={16} 
        height={16} 
        color="currentColor" 
      />
    </div>
  );
}
```

The component uses stroke-based SVG paths with customizable color (defaults to `currentColor` for CSS inheritance) and includes proper accessibility attributes. The sizing logic prioritizes the `size` prop, falling back to individual `width`/`height` props, and defaults to 24px when no dimensions are specified.
