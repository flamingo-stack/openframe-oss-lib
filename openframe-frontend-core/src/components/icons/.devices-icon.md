A React SVG icon component that renders a devices/multi-screen icon with customizable size, color, and styling properties.

## Key Components

- **DevicesIconProps**: Interface extending `React.SVGProps<SVGSVGElement>` with optional `size` and `color` props
- **DevicesIcon**: Functional component that renders an SVG icon depicting multiple connected devices (monitor and stand)

## Usage Example

```typescript
import { DevicesIcon } from './devices-icon'

// Basic usage
<DevicesIcon />

// With custom size and color
<DevicesIcon size={32} color="#3b82f6" />

// With CSS classes and custom styling
<DevicesIcon 
  className="hover:opacity-80" 
  style={{ margin: '8px' }}
  size={24}
/>

// With additional SVG props
<DevicesIcon 
  size={28}
  onClick={() => console.log('Devices clicked')}
  aria-label="Connected devices"
/>
```

The component accepts all standard SVG element properties through props spreading, making it highly flexible for various use cases. The default size is 20px and the icon uses the current text color when no color is specified.
