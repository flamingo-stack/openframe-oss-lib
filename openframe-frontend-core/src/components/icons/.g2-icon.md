A React SVG icon component that renders the G2 logo with customizable styling properties.

## Key Components

- **G2Icon** - Main functional component that renders an SVG representation of the G2 logo
- **G2IconProps** - TypeScript interface defining optional props for width, height, className, and color

## Usage Example

```typescript
import { G2Icon } from './g2-icon';

// Default usage (large size, red color)
<G2Icon />

// Custom size and styling
<G2Icon 
  width={100} 
  height={100} 
  className="my-icon" 
  color="#3b82f6" 
/>

// Responsive with CSS classes
<G2Icon 
  width="100%" 
  height="auto" 
  className="w-8 h-8 text-blue-500" 
/>
```

The component defaults to a large 2500x2500 size with the signature G2 red color (`#ff492c`), making it suitable for high-resolution displays. All props are optional, allowing for flexible integration across different UI contexts.
