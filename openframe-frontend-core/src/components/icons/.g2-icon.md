A React component that renders the G2 logo as a scalable SVG icon with customizable properties.

## Key Components

- **G2Icon** - Main functional component that renders the G2 logo SVG
- **G2IconProps** - TypeScript interface defining component props:
  - `width` - Icon width (default: 2500)
  - `height` - Icon height (default: 2500) 
  - `className` - CSS class name for styling
  - `color` - Fill color for the SVG paths (default: '#ff492c')

## Usage Example

```typescript
import { G2Icon } from './g2-icon';

// Basic usage with defaults
<G2Icon />

// Customized icon
<G2Icon 
  width={100}
  height={100}
  color="#0066cc"
  className="logo-icon"
/>

// Responsive sizing
<G2Icon 
  width="100%"
  height="auto"
  className="responsive-logo"
/>
```

The component uses the G2 brand's signature orange color by default and maintains the original SVG viewBox for proper scaling across different sizes.
