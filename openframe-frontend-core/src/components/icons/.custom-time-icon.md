A customizable clock/time SVG icon component built with React and TypeScript, featuring a circular clock face with hour and minute hands.

## Key Components

- **`CustomTimeIconProps`** - TypeScript interface defining optional props for styling and sizing
- **`CustomTimeIcon`** - React functional component that renders an SVG clock icon with customizable appearance

## Usage Example

```typescript
import { CustomTimeIcon } from './custom-time-icon';

// Basic usage with default gray color and 24px size
<CustomTimeIcon />

// Customized with blue color and larger size
<CustomTimeIcon 
  fill="#3b82f6" 
  width={32} 
  height={32} 
  className="my-clock-icon" 
/>

// With additional props passed through
<CustomTimeIcon 
  fill="#ef4444" 
  onClick={handleTimeClick}
  aria-label="Current time"
/>
```

The icon displays a clean clock face with the hands positioned at approximately 3:00. All props are optional, with sensible defaults of 24px dimensions and a neutral gray color (#888888). The component spreads additional props to the SVG element for maximum flexibility.
