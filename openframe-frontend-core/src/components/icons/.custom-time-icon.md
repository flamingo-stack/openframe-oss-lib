A customizable SVG clock/time icon component with configurable styling properties and default gray appearance.

## Key Components

- **`CustomTimeIconProps`** - Interface defining component props including className, fill color, and dimensions
- **`CustomTimeIcon`** - Main React component exporting a clock icon with hour hand pointing to approximately 12:30

## Usage Example

```typescript
import { CustomTimeIcon } from './custom-time-icon';

// Basic usage with defaults
<CustomTimeIcon />

// Customized appearance
<CustomTimeIcon 
  className="my-clock-icon"
  fill="#3b82f6"
  width={32}
  height={32}
/>

// With additional props
<CustomTimeIcon 
  fill="#ef4444"
  onClick={handleTimeClick}
  aria-label="Current time"
/>
```

The component accepts standard SVG element props through spread operators, making it fully compatible with event handlers and accessibility attributes. Default styling provides a 24x24px icon with gray (#888888) fill color.
