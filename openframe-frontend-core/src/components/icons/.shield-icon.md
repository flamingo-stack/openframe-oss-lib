A React SVG component that renders a shield icon with a checkmark, commonly used to represent security, protection, or verification states in applications.

## Key Components

- **ShieldIconProps**: Interface extending React's SVGProps for type safety
- **ShieldIcon**: Main functional component that renders the shield SVG with customizable styling

## Usage Example

```typescript
import { ShieldIcon } from './shield-icon'

// Basic usage with defaults
<ShieldIcon />

// Custom styling
<ShieldIcon 
  className="h-8 w-8" 
  color="green" 
/>

// With additional SVG props
<ShieldIcon 
  className="h-4 w-4"
  color="#3B82F6"
  onClick={() => console.log('Shield clicked')}
  aria-label="Security verified"
/>

// In a security status component
function SecurityStatus({ isVerified }: { isVerified: boolean }) {
  return (
    <div className="flex items-center gap-2">
      <ShieldIcon 
        className="h-5 w-5" 
        color={isVerified ? "green" : "gray"} 
      />
      <span>{isVerified ? "Verified" : "Pending"}</span>
    </div>
  )
}
```

The component accepts all standard SVG props and provides sensible defaults for size (24x24) and color (white).
