A React SVG component that renders the "OpenFrame" text logo as scalable vector graphics with customizable styling.

## Key Components

- **OpenFrameText**: Main React component that renders the OpenFrame logo text
- **OpenFrameTextProps**: TypeScript interface extending SVGProps with additional customization options
- **textColor**: Prop for customizing the text color (defaults to dark gray `#1A1A1A`)
- **className**: Standard CSS class name prop for styling

## Usage Example

```typescript
import { OpenFrameText } from './openframe-text';

// Basic usage with default styling
<OpenFrameText />

// Custom text color
<OpenFrameText textColor="#FF0000" />

// With custom class and additional SVG props
<OpenFrameText 
  className="logo" 
  textColor="#0066CC"
  width={174}
  height={30}
/>

// Responsive sizing while maintaining aspect ratio
<OpenFrameText 
  className="w-full h-auto max-w-xs" 
  textColor="currentColor"
/>
```

The component preserves the original 87x15 viewBox aspect ratio and accepts all standard SVG element properties, making it flexible for various layout needs while maintaining crisp rendering at any size.
