A React SVG component that renders the "OpenFrame" logo text as scalable vector graphics with customizable text color.

## Key Components

**OpenFrameText** - Main component that renders the OpenFrame logo text as an SVG element with the following features:
- Extends all standard SVG props for full flexibility
- Customizable text color via `textColor` prop (defaults to `#1A1A1A`)
- Fixed dimensions (87x15) with responsive scaling via `viewBox`
- CSS custom property support for advanced styling
- Individual letter paths for crisp rendering at any scale

## Usage Example

```typescript
import { OpenFrameText } from './openframe-text';

// Basic usage with default styling
<OpenFrameText />

// With custom text color
<OpenFrameText textColor="#FF6B35" />

// With additional SVG props and custom styling
<OpenFrameText 
  className="logo-text"
  textColor="#2563EB"
  width={174}
  height={30}
  style={{ filter: 'drop-shadow(2px 2px 4px rgba(0,0,0,0.1))' }}
/>

// Using currentColor for theme integration
<OpenFrameText 
  textColor="currentColor"
  className="text-primary"
/>
```

The component maintains crisp text rendering at any size while providing full control over color and standard SVG properties like dimensions, classes, and CSS transforms.
