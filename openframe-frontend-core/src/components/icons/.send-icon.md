A React component that renders a paper airplane send icon as an SVG, commonly used for submit buttons and messaging interfaces.

## Key Components

- **SendIconProps**: Interface defining optional `className` and `fill` props for customization
- **SendIcon**: Functional component that renders a stylized paper airplane SVG icon

## Usage Example

```typescript
import { SendIcon } from './send-icon';

// Default usage
<SendIcon />

// Custom styling
<SendIcon 
  className="w-8 h-8" 
  fill="#3B82F6" 
/>

// In a button
<button className="flex items-center gap-2">
  Send Message
  <SendIcon className="w-4 h-4" fill="white" />
</button>
```

The component uses a 24x24 viewBox with customizable dimensions via className (defaults to `w-6 h-6`) and fill color (defaults to dark gray `#1A1A1A`). The SVG paths create a modern paper airplane design suitable for send/submit actions in forms and messaging UIs.
