A React component that renders an SVG send/paper plane icon with customizable styling and fill color.

## Key Components

- **`SendIcon`** - Main React functional component that renders the send icon
- **`SendIconProps`** - TypeScript interface defining optional `className` and `fill` props
- Default props: `className="w-6 h-6"` (Tailwind CSS classes) and `fill="#1A1A1A"` (dark gray)

## Usage Example

```typescript
import { SendIcon } from './send-icon';

// Basic usage with defaults
<SendIcon />

// Custom styling and color
<SendIcon 
  className="w-8 h-8" 
  fill="#3B82F6" 
/>

// In a button component
<button className="flex items-center gap-2">
  Send Message
  <SendIcon className="w-4 h-4" fill="white" />
</button>
```

The icon uses a 24x24 viewBox and includes two path elements that create a paper plane design. The component is fully customizable through className for sizing/positioning and fill for color changes.
