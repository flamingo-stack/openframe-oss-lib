SVG icon component rendering a thumbs-up symbol, deprecated in favor of the `icons-v2-generated` icon set.

## Key Components

- **`ThumbsUpIconProps`** — Interface accepting optional `className` (default: `'w-6 h-6'`) and `color` (default: `#F357BB`) props
- **`ThumbsUpIcon`** — Deprecated functional component rendering a 23×23 SVG thumbs-up icon

## Usage Example

```typescript
import { ThumbsUpIcon } from './thumbs-up-icon';

// Default usage
<ThumbsUpIcon />

// Custom size and color
<ThumbsUpIcon className="w-8 h-8" color="#FF0000" />
```

> **Deprecated:** Use icons from `icons-v2-generated` instead.
