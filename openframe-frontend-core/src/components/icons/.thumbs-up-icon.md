A React SVG icon component that renders a thumbs-up symbol with customizable styling and color properties.

## Key Components

- **ThumbsUpIconProps**: Interface defining optional `className` and `color` props
- **ThumbsUpIcon**: Functional component that renders an SVG thumbs-up icon with default pink color (`#F357BB`) and `w-6 h-6` dimensions

## Usage Example

```typescript
import { ThumbsUpIcon } from './thumbs-up-icon';

// Default usage with pink color
<ThumbsUpIcon />

// Custom styling and color
<ThumbsUpIcon 
  className="w-8 h-8" 
  color="#3B82F6" 
/>

// Only custom size
<ThumbsUpIcon className="w-4 h-4" />

// Only custom color
<ThumbsUpIcon color="#10B981" />
```

The component uses inline SVG with two path elements to create the thumbs-up shape, making it scalable and customizable while maintaining crisp rendering at any size.
