A React SVG icon component that renders a thumbs-up symbol with customizable styling and color properties.

## Key Components

- **ThumbsUpIconProps**: Interface defining optional `className` and `color` props
- **ThumbsUpIcon**: Functional component that renders an SVG thumbs-up icon with default dimensions and pink color

## Usage Example

```typescript
import { ThumbsUpIcon } from './thumbs-up-icon';

// Basic usage with defaults
<ThumbsUpIcon />

// Custom styling and color
<ThumbsUpIcon 
  className="w-8 h-8" 
  color="#4CAF50" 
/>

// In a button component
<button className="flex items-center gap-2">
  <ThumbsUpIcon className="w-5 h-5" color="#fff" />
  Like
</button>
```

The component defaults to a 6x6 Tailwind CSS size (`w-6 h-6`) and uses a pink color (`#F357BB`). The SVG viewBox is fixed at 23x23 units, but the rendered size adapts to the provided className dimensions.
