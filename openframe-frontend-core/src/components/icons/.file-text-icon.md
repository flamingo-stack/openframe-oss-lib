A reusable React SVG icon component that renders a file/document icon with text lines, designed for use in file-related UI elements.

## Key Components

- **`FileTextIconProps`** - TypeScript interface extending React's SVGProps for type safety
- **`FileTextIcon`** - Main functional component that renders the SVG icon with customizable styling

## Usage Example

```typescript
import { FileTextIcon } from './file-text-icon'

// Basic usage with defaults
<FileTextIcon />

// Custom styling
<FileTextIcon 
  className="h-8 w-8" 
  color="blue" 
/>

// With additional SVG props
<FileTextIcon 
  className="h-4 w-4"
  color="#333"
  onClick={() => console.log('File clicked')}
  aria-label="Text document"
/>

// In a file list component
function FileItem({ filename }: { filename: string }) {
  return (
    <div className="flex items-center gap-2">
      <FileTextIcon className="h-5 w-5" color="gray" />
      <span>{filename}</span>
    </div>
  )
}
```

The component accepts standard SVG props and provides sensible defaults (24px size, white color) while allowing full customization through className and color props.
