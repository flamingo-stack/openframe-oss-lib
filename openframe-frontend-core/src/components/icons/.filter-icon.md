A deprecated React functional component that renders a 16×16 SVG filter/funnel icon with three horizontal lines of decreasing width.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `FilterIcon` | `React.FC<FilterIconProps>` | SVG filter icon component |
| `FilterIconProps` | `interface` | Optional `className` and `size` props |

> **⚠️ Deprecated:** Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { FilterIcon } from './filter-icon'

// Basic usage
<FilterIcon />

// With custom className
<FilterIcon className="text-gray-500" />
```

## Notes

- Props `className` (default: `''`) and `size` (default: `20`) are accepted but **not applied** to the SVG — the icon renders at a fixed `16×16` viewport regardless of the `size` prop.
- Icon color is hardcoded to `#888888`.
- Prefer `icons-v2-generated` for new implementations.
