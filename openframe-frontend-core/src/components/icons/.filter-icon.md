A React component that renders an SVG filter icon with three horizontal lines of decreasing width, commonly used in UI filtering interfaces.

## Key Components

- **FilterIcon**: React functional component that renders a filter icon SVG
- **FilterIconProps**: TypeScript interface defining optional `className` and `size` props
- **SVG paths**: Two path elements creating the visual filter icon with rounded rectangles

## Usage Example

```typescript
import { FilterIcon } from './filter-icon'

// Basic usage
<FilterIcon />

// With custom styling
<FilterIcon className="text-blue-500 hover:text-blue-700" />

// With custom size (note: size prop is defined but not applied to SVG)
<FilterIcon size={24} className="w-6 h-6" />

// In a filter button
<button className="flex items-center gap-2">
  <FilterIcon className="opacity-60" />
  Filter Results
</button>
```

**Note**: The `size` prop is currently unused in the implementation. The SVG has fixed dimensions of 16x16. To make the icon responsive to the `size` prop, the SVG's `width` and `height` attributes should reference the `size` parameter instead of being hardcoded.
