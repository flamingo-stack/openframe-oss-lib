A React functional component that renders an SVG filter icon with three horizontal lines of decreasing width, commonly used in UI for filtering or sorting functionality.

## Key Components

- **FilterIcon**: Main React component that renders the filter SVG icon
- **FilterIconProps**: TypeScript interface defining optional `className` and `size` props
- Fixed 16x16 viewBox with gray (#888888) colored paths

## Usage Example

```typescript
import { FilterIcon } from './filter-icon'

// Basic usage
<FilterIcon />

// With custom styling
<FilterIcon 
  className="text-blue-500 hover:text-blue-700" 
  size={24} 
/>

// In a button component
<button className="flex items-center gap-2">
  <FilterIcon size={16} />
  Filter Results
</button>
```

Note: The `size` prop is defined in the interface but not currently applied to the SVG element - the icon maintains a fixed 16x16 pixel size regardless of the size prop value.
