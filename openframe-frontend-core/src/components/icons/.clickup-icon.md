SVG icon component rendering the ClickUp brand logo with customizable dimensions and styling.

## Key Components

**`ClickUpIconProps`** ‚Äî Interface defining optional props:
- `width` ‚Äî Icon width (default: `24`)
- `height` ‚Äî Icon height (default: `24`)
- `className` ‚Äî Additional CSS classes (default: `''`)

**`ClickUpIcon`** ‚Äî Named export rendering a two-path SVG with ClickUp's signature gradient colors (purple-to-cyan bottom arc, pink-to-yellow top chevron).

## Usage Example

```typescript
import { ClickUpIcon } from './clickup-icon'

// Default 24√ó24
<ClickUpIcon />

// Custom size with Tailwind class
<ClickUpIcon width={40} height={40} className="opacity-80" />

// String-based responsive size
<ClickUpIcon width="100%" height="auto" />
```
