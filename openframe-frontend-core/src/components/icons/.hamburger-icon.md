A React component that renders an SVG hamburger menu icon with three horizontal lines, commonly used for mobile navigation toggles.

## Key Components

- **HamburgerIcon**: Main functional component that renders the hamburger icon
- **HamburgerIconProps**: TypeScript interface defining optional props for customization
  - `width`: Icon width (default: 16px)
  - `height`: Icon height (default: 16px)
  - `className`: CSS class for styling

## Usage Example

```typescript
import { HamburgerIcon } from './hamburger-icon';

// Basic usage with defaults
<HamburgerIcon />

// Customized size and styling
<HamburgerIcon 
  width={24} 
  height={24} 
  className="text-blue-500 cursor-pointer" 
/>

// In a mobile menu button
<button onClick={toggleMenu} className="md:hidden">
  <HamburgerIcon width={20} height={20} className="hover:opacity-80" />
</button>
```

The icon uses a fixed white fill color (`#FAFAFA`) and maintains a 16x16 viewBox regardless of the specified dimensions, ensuring consistent proportions when scaled.
