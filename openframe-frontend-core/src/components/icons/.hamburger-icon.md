A React functional component that renders a customizable hamburger menu icon as an SVG element with three horizontal lines.

## Key Components

- **HamburgerIcon**: Main component that renders the hamburger menu icon
- **HamburgerIconProps**: TypeScript interface defining optional props for customization
  - `width`: Icon width in pixels (default: 16)
  - `height`: Icon height in pixels (default: 16)  
  - `className`: CSS class name for styling

## Usage Example

```typescript
import { HamburgerIcon } from './hamburger-icon';

// Basic usage with defaults
<HamburgerIcon />

// Custom size and styling
<HamburgerIcon 
  width={24} 
  height={24} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a mobile menu button
<button className="md:hidden" onClick={toggleMenu}>
  <HamburgerIcon width={20} height={20} className="transition-colors" />
</button>
```

The icon uses a fixed light color (`#FAFAFA`) and maintains a 16x16 viewBox regardless of the actual dimensions, ensuring consistent proportions when scaled.
