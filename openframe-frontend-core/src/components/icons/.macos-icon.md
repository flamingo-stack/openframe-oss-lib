A React component that renders the Apple macOS logo as an inline SVG icon with customizable styling and color options.

## Key Components

- **MacOSIconProps** - TypeScript interface extending `React.SVGProps<SVGSVGElement>` with an optional `color` prop
- **MacOSIcon** - Functional component that renders a 16x16 SVG of the Apple logo with two path elements representing the iconic apple shape and leaf

## Usage Example

```typescript
import { MacOSIcon } from './macos-icon'

// Basic usage with default white color
<MacOSIcon />

// Custom color and styling
<MacOSIcon 
  color="#000000" 
  className="w-8 h-8" 
  width="32" 
  height="32" 
/>

// With click handler and additional SVG props
<MacOSIcon 
  color="#007AFF"
  onClick={() => console.log('Apple icon clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component accepts all standard SVG props through spreading, making it highly flexible for different use cases while maintaining the recognizable Apple logo design. The default color is white, but can be customized via the `color` prop to match your application's theme.
