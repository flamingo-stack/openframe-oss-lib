A React component that renders the Apple macOS logo as an SVG icon with customizable styling and color options.

## Key Components

- **`MacOSIconProps`** - Interface extending SVG properties with optional `color` prop
- **`MacOSIcon`** - Functional component rendering the Apple logo SVG with customizable color (defaults to white)

## Usage Example

```typescript
import { MacOSIcon } from './macos-icon';

// Basic usage with default white color
<MacOSIcon />

// Custom color and styling
<MacOSIcon 
  color="#007AFF" 
  className="w-8 h-8" 
  onClick={handleClick}
/>

// With additional SVG properties
<MacOSIcon 
  color="currentColor"
  width={24}
  height={24}
  aria-label="macOS"
/>
```

The component accepts all standard SVG properties through prop spreading, making it highly flexible for different use cases while maintaining the iconic Apple logo shape.
