A React SVG icon component that renders an edit/pencil icon commonly used for profile editing functionality.

## Key Components

- **EditProfileIcon**: Main functional component that renders an SVG edit icon
- **EditProfileIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional className
- **SVG paths**: Two path elements creating a document with pencil overlay design

## Usage Example

```typescript
import { EditProfileIcon } from './edit-profile-icon';

// Basic usage
<EditProfileIcon />

// With custom styling
<EditProfileIcon 
  className="w-6 h-6 text-blue-500 hover:text-blue-700" 
  onClick={handleEditClick}
/>

// With additional SVG properties
<EditProfileIcon 
  width={32}
  height={32}
  className="cursor-pointer"
  aria-label="Edit profile"
/>
```

The component accepts all standard SVG props and forwards them to the underlying `<svg>` element, making it highly flexible for different styling and interaction needs. The icon uses a 24x24 viewBox and fills with `#FAFAFA` color by default.
