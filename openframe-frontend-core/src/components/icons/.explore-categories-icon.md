A React icon component that renders an SVG graphic for exploring categories functionality, commonly used in navigation or menu interfaces.

## Key Components

- **ExploreCategoriesIconProps**: TypeScript interface defining optional props for customization
- **ExploreCategoriesIcon**: Functional React component that renders the SVG icon

**Props:**
- `color` - Fill color for the SVG (default: 'white')
- `className` - CSS classes for styling (default: 'w-6 h-6')
- `width` - Explicit width override
- `height` - Explicit height override

## Usage Example

```typescript
import { ExploreCategoriesIcon } from './explore-categories-icon';

// Basic usage with defaults
<ExploreCategoriesIcon />

// Custom styling and color
<ExploreCategoriesIcon 
  className="w-8 h-8" 
  color="#3B82F6" 
/>

// Explicit dimensions
<ExploreCategoriesIcon 
  width={32}
  height={32}
  color="currentColor"
/>

// In a button component
<button className="flex items-center gap-2">
  <ExploreCategoriesIcon color="#059669" />
  <span>Explore Categories</span>
</button>
```

The SVG features a complex path design with viewBox "0 0 21 21" and includes layered geometric shapes that form a recognizable categories/exploration icon.
