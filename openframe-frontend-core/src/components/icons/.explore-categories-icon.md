Deprecated SVG icon component rendering an "explore categories" symbol as an inline React SVG element.

## Key Components

### `ExploreCategoriesIconProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | SVG fill color |
| `className` | `string` | `'w-6 h-6'` | CSS class names |
| `width` | `number` | — | Explicit SVG width |
| `height` | `number` | — | Explicit SVG height |

### `ExploreCategoriesIcon`
A functional React component that renders a 21×21 viewBox SVG icon representing an "explore categories" action (two-path composition of a bookmark/tag shape).

## Usage Example

```typescript
import { ExploreCategoriesIcon } from './explore-categories-icon';

// Default usage (white, 24×24 via Tailwind)
<ExploreCategoriesIcon />

// Custom color and size via className
<ExploreCategoriesIcon color="#FFC008" className="w-8 h-8" />

// Explicit pixel dimensions
<ExploreCategoriesIcon color="black" width={32} height={32} />
```

> ⚠️ **Deprecated** — This component is marked `@deprecated`. Use the equivalent icon from `icons-v2-generated` instead.
