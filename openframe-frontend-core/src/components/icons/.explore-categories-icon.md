React SVG icon component that renders an "explore categories" interface icon with customizable styling properties.

## Key Components

**ExploreCategoriesIconProps Interface:**
- `color?: string` - Fill color for the SVG paths (default: 'white')
- `className?: string` - CSS classes for styling (default: 'w-6 h-6')
- `width?: number` - Explicit width override
- `height?: number` - Explicit height override

**ExploreCategoriesIcon Function:**
- Main component that renders the SVG icon
- Uses a 21x21 viewBox with two path elements
- Represents a categorization/folder interface element

## Usage Example

```typescript
import { ExploreCategoriesIcon } from './explore-categories-icon';

// Basic usage with defaults
<ExploreCategoriesIcon />

// Custom styling and colors
<ExploreCategoriesIcon 
  color="#3B82F6" 
  className="w-8 h-8" 
/>

// Explicit dimensions
<ExploreCategoriesIcon 
  width={32} 
  height={32} 
  color="#059669" 
/>

// In a button component
<button className="flex items-center gap-2">
  <ExploreCategoriesIcon color="#6B7280" />
  Explore Categories
</button>
```
