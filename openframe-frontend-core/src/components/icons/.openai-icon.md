A React component that renders the OpenAI logo as an SVG icon with customizable styling properties.

## Key Components

- **OpenAiIcon**: Main functional component that renders the OpenAI logo
- **OpenAiIconProps**: TypeScript interface defining optional props:
  - `className`: CSS class for styling
  - `size`: Icon dimensions (default: 24px)
  - `color`: Fill and stroke color (default: 'white')

## Usage Example

```typescript
import { OpenAiIcon } from './openai-icon'

// Basic usage with defaults
<OpenAiIcon />

// Customized icon
<OpenAiIcon 
  size={48} 
  color="#10a37f" 
  className="ml-2 hover:opacity-80" 
/>

// Large icon for headers
<OpenAiIcon size={64} color="black" />
```

The component uses a 320x320 viewBox and scales proportionally based on the `size` prop. The SVG path contains the complete OpenAI logo geometry with both fill and stroke using the same color value.
