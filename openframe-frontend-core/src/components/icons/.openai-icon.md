A React component that renders the OpenAI logo as an SVG icon with customizable styling properties.

## Key Components

- **OpenAiIconProps** - Interface defining the component's props:
  - `className` - Optional CSS class for styling
  - `size` - Icon dimensions in pixels (default: 24)
  - `color` - Fill and stroke color (default: 'white')

- **OpenAiIcon** - Main functional component that renders the OpenAI logo SVG

## Usage Example

```typescript
import { OpenAiIcon } from './openai-icon'

// Basic usage with defaults
<OpenAiIcon />

// Customized icon
<OpenAiIcon 
  size={32} 
  color="#10a37f" 
  className="ml-2" 
/>

// Large icon with custom styling
<OpenAiIcon 
  size={64} 
  color="currentColor" 
  className="hover:opacity-80 transition-opacity" 
/>
```

The component uses a 320x320 viewBox and scales proportionally based on the `size` prop. The SVG path contains the official OpenAI logo design with both fill and stroke properties accepting the same color value for consistent appearance.
