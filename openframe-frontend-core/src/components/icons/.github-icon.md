A customizable React component that renders the GitHub logo as an SVG icon with configurable styling properties.

## Key Components

- **GitHubIcon**: Main React functional component that renders the GitHub logo SVG
- **GitHubIconProps**: TypeScript interface defining optional props for customization
  - `color`: Fill color (default: 'white')
  - `className`: CSS class name for styling
  - `width`: Icon width in pixels (default: 15)
  - `height`: Icon height in pixels (default: 14)

## Usage Example

```typescript
import { GitHubIcon } from './github-icon';

// Basic usage with defaults
<GitHubIcon />

// Customized appearance
<GitHubIcon 
  color="#000000"
  width={24}
  height={24}
  className="hover:opacity-80"
/>

// In a button or link
<a href="https://github.com/user/repo">
  <GitHubIcon color="#333" width={20} height={20} />
</a>
```

The component uses a fixed viewBox of "0 0 15 14" to maintain proper proportions while allowing flexible sizing through the width and height props.
