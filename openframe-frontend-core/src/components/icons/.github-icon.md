A React component that renders a customizable GitHub logo SVG icon with configurable styling and dimensions.

## Key Components

- **GitHubIconProps** - Interface defining optional props for customization
- **GitHubIcon** - Main functional component that renders the GitHub SVG logo

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | Fill color for the icon |
| `className` | `string` | `""` | CSS classes to apply |
| `width` | `number` | `15` | Icon width in pixels |
| `height` | `number` | `14` | Icon height in pixels |

## Usage Example

```typescript
import { GitHubIcon } from './github-icon';

// Basic usage with defaults
<GitHubIcon />

// Customized styling
<GitHubIcon 
  color="black" 
  width={24} 
  height={24} 
  className="hover:opacity-80 transition-opacity" 
/>

// Dark theme variant
<GitHubIcon color="#24292e" width={20} height={20} />
```

The component uses a fixed viewBox of "0 0 15 14" while allowing dynamic width/height scaling, making it suitable for responsive designs and various UI contexts.
