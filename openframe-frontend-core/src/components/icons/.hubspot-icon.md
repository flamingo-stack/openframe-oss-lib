A React SVG icon component that renders the HubSpot brand logo with customizable styling and properties.

## Key Components

- **HubspotIcon**: Main component that renders the HubSpot logo as an SVG
- **Props**: Accepts all standard SVG props plus custom `fill` prop (defaults to HubSpot orange `#FF7A59`)
- **Styling**: Uses `cn` utility for conditional class merging with default inline-block display

## Usage Example

```typescript
import { HubspotIcon } from './components/icons/hubspot-icon';

// Basic usage with default orange color
<HubspotIcon className="h-6 w-6" />

// Custom size and color
<HubspotIcon 
  className="h-8 w-8" 
  fill="#000000" 
/>

// With additional SVG props
<HubspotIcon 
  className="h-4 w-4"
  onClick={handleClick}
  role="button"
  aria-label="HubSpot integration"
/>
```

The component automatically applies the HubSpot brand color and includes proper viewBox dimensions for crisp rendering at any size.
