A React SVG component that renders the HubSpot brand logo icon with customizable styling and fill color.

## Key Components

- **HubspotIcon** - Main SVG component that renders the HubSpot logo
- **Props Interface** - Extends `React.SVGProps<SVGSVGElement>` with custom `fill` prop
- **Default Styling** - Uses HubSpot's brand orange color (`#FF7A59`) by default
- **Class Name Utility** - Integrates with `cn` utility for conditional styling

## Usage Example

```typescript
import { HubspotIcon } from './components/icons/hubspot-icon';

// Basic usage with default orange color
<HubspotIcon className="h-6 w-6" />

// Custom color and size
<HubspotIcon 
  className="h-8 w-8" 
  fill="#333333" 
/>

// With additional SVG props
<HubspotIcon 
  className="h-4 w-4" 
  onClick={handleClick}
  title="HubSpot Integration"
/>
```

The component automatically applies `inline-block` display and accepts all standard SVG properties, making it flexible for use in buttons, navigation items, or anywhere the HubSpot brand logo is needed.
