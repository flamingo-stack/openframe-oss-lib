A React component that renders a PowerShell terminal icon as an SVG element with customizable styling properties.

## Key Components

- **`PowerShellIconProps`** - TypeScript interface defining optional props for size, CSS class, and color
- **`PowerShellIcon`** - Functional React component that renders the PowerShell logo SVG
- **Default props** - Size defaults to 16px, empty className, and white color

## Usage Example

```typescript
import { PowerShellIcon } from './powershell-icon'

// Basic usage with defaults (16px, white)
<PowerShellIcon />

// Custom sizing and styling
<PowerShellIcon 
  size={24} 
  className="icon-button" 
  color="#0078d4" 
/>

// Large blue icon for headers
<PowerShellIcon 
  size={48} 
  color="#012456" 
  className="header-icon" 
/>
```

The component uses a scalable SVG with the official PowerShell logo design, featuring the characteristic "PS" prompt symbol with angled brackets. The `viewBox` is fixed at "0 0 128 128" while the actual dimensions scale with the `size` prop.
