A React component that renders a customizable PowerShell logo as an SVG icon. The component provides configurable size, styling, and color options for use in UI applications.

## Key Components

- **PowerShellIcon**: Main React functional component that renders the PowerShell logo SVG
- **PowerShellIconProps**: TypeScript interface defining component props:
  - `size?: number` - Icon dimensions (default: 16px)
  - `className?: string` - CSS class name for styling
  - `color?: string` - Fill color for the icon (default: 'white')

## Usage Example

```typescript
import { PowerShellIcon } from './powershell-icon'

// Basic usage with defaults
<PowerShellIcon />

// Customized icon
<PowerShellIcon 
  size={24}
  color="#0078D4"
  className="my-icon-class"
/>

// Large blue PowerShell icon
<PowerShellIcon 
  size={48}
  color="blue"
/>
```

The component uses the PowerShell brand colors and styling, maintaining the recognizable terminal prompt symbol and "PS" text elements in the logo design.
