A deprecated React component that renders a PowerShell icon as an inline SVG.

## Key Components

### `PowerShellIconProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `16` | Width and height of the SVG in pixels |
| `className` | `string` | `''` | Additional CSS classes |
| `color` | `string` | `'white'` | Fill color for the icon path |

### `PowerShellIcon`
A functional React component rendering a `128×128` viewBox PowerShell SVG icon, scaled via the `size` prop.

> ⚠️ **Deprecated** — This component and its props interface are deprecated. Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { PowerShellIcon } from './powershell-icon'

// Basic usage
<PowerShellIcon />

// Custom size and color
<PowerShellIcon size={32} color="#0078D4" className="my-icon" />
```
