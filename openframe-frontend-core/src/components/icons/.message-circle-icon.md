A React SVG component that renders a message circle icon with three dots, typically used for chat, messaging, or conversation features.

## Key Components

- **`MessageCircleIconProps`** - Interface extending React's SVGProps for type safety
- **`MessageCircleIcon`** - React functional component that renders the SVG icon

## Usage Example

```typescript
import { MessageCircleIcon } from './message-circle-icon'

// Basic usage with defaults (24x24, white stroke)
<MessageCircleIcon />

// Custom styling
<MessageCircleIcon 
  className="h-8 w-8" 
  color="blue" 
/>

// With additional SVG props
<MessageCircleIcon 
  className="h-5 w-5"
  color="#333"
  onClick={() => console.log('Message clicked')}
  role="button"
  aria-label="Open messages"
/>
```

The component accepts all standard SVG props and provides sensible defaults: `h-6 w-6` for className and `white` for stroke color. The icon features a speech bubble outline with three dots inside, making it perfect for messaging interfaces, chat buttons, or conversation indicators.
