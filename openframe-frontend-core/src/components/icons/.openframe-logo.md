A React SVG component that renders the OpenFrame logo with customizable colors and standard SVG properties.

## Key Components

- **OpenFrameLogo**: Main component that renders an SVG logo with a distinctive L-shaped frame design
- **Props Interface**: Extends `React.SVGProps<SVGSVGElement>` with optional color customization:
  - `className`: CSS class for styling
  - `lowerPathColor`: Color for the white squares/blocks (defaults to '#ffffff')
  - `upperPathColor`: Color for the main frame/border (defaults to '#1A1A1A')

## Usage Example

```typescript
import { OpenFrameLogo } from './openframe-logo';

// Basic usage with default colors
<OpenFrameLogo className="w-8 h-8" />

// Custom colors
<OpenFrameLogo 
  className="w-12 h-12"
  lowerPathColor="#00ff00"
  upperPathColor="#333333"
/>

// With additional SVG props
<OpenFrameLogo 
  className="logo"
  width={64}
  height={64}
  style={{ marginRight: '10px' }}
/>
```

The logo features a 32x32 viewBox with an L-shaped frame containing four white squares positioned in the top-right, bottom-left, bottom-right, and bottom-center areas. The design uses CSS custom properties for color theming and maintains accessibility through currentColor fallbacks.
