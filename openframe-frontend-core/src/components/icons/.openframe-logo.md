A React SVG component that renders the OpenFrame logo with customizable colors and standard SVG properties support.

## Key Components

- **OpenFrameLogo**: Main React functional component that renders an SVG logo
- **Props Interface**: Accepts `className`, `lowerPathColor`, `upperPathColor`, and all standard SVG element props
- **SVG Structure**: Contains two main visual elements - white squares (lower path) and a black frame border (upper path)

## Usage Example

```typescript
import { OpenFrameLogo } from './openframe-logo';

// Basic usage
<OpenFrameLogo />

// With custom colors and styling
<OpenFrameLogo 
  className="my-logo"
  lowerPathColor="#ff0000"
  upperPathColor="#0000ff"
  width={64}
  height={64}
/>

// With additional SVG properties
<OpenFrameLogo 
  onClick={handleClick}
  style={{ cursor: 'pointer' }}
  aria-label="OpenFrame Logo"
/>
```

The component uses a 32x32 viewBox and defaults to white squares (`#ffffff`) with a dark frame (`#1A1A1A`). The logo maintains its aspect ratio and can be easily customized through props while supporting all standard SVG element interactions and styling.
