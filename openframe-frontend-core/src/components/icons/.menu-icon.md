A React component that renders a customizable menu icon using Lucide React's Menu icon with configurable styling and dimensions.

## Key Components

- **MenuIcon**: Main functional component that wraps the Lucide Menu icon
- **MenuIconProps**: TypeScript interface defining optional props for styling and sizing
- **Props**:
  - `className`: Optional CSS class string for styling
  - `width`: Icon width in pixels (default: 24)
  - `height`: Icon height in pixels (default: 24)

## Usage Example

```typescript
import { MenuIcon } from './menu-icon';

// Basic usage with defaults
<MenuIcon />

// Custom sizing
<MenuIcon width={32} height={32} />

// With custom styling
<MenuIcon className="text-blue-500 hover:text-blue-700" />

// Full customization
<MenuIcon 
  className="menu-button" 
  width={20} 
  height={20} 
/>
```

The component provides a clean abstraction over the Lucide Menu icon, making it reusable across the application with consistent default sizing while allowing customization when needed.
