A React component that renders an SVG icon representing vendors or supplier management, featuring customizable styling and dimensions.

## Key Components

- **VendorsIcon**: Main React functional component that renders a vendors/supplier SVG icon
- **VendorsIconProps**: TypeScript interface defining optional props for customization
  - `color`: Fill color for the icon (default: 'white')
  - `className`: CSS class name for styling
  - `width`/`height`: Icon dimensions (default: 24x24)

## Usage Example

```typescript
import { VendorsIcon } from './vendors-icon';

// Basic usage with defaults
<VendorsIcon />

// Customized icon
<VendorsIcon 
  color="#3B82F6" 
  width={32} 
  height={32}
  className="vendors-icon" 
/>

// In a button or navigation
<button className="icon-button">
  <VendorsIcon color="currentColor" />
  <span>Vendors</span>
</button>
```

The icon displays a symbolic representation of vendor/supplier management with clean lines and is fully scalable as an SVG. All styling properties are optional with sensible defaults.
