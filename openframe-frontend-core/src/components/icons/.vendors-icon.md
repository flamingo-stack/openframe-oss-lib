A React component that renders an SVG icon representing vendors or business partnerships, featuring a customizable vector graphic with dynamic styling properties.

## Key Components

- **VendorsIcon**: Main functional component that renders an SVG icon
- **VendorsIconProps**: TypeScript interface defining component props:
  - `color`: Fill color for the SVG paths (default: 'white')
  - `className`: CSS class name for styling
  - `width`: Icon width in pixels (default: 24)
  - `height`: Icon height in pixels (default: 24)

The component renders a complex SVG with two main path elements that create a vendors/partnership themed icon, commonly used in business applications for representing vendor management or partnership features.

## Usage Example

```typescript
import { VendorsIcon } from './vendors-icon';

// Basic usage with defaults
<VendorsIcon />

// Customized styling
<VendorsIcon 
  color="#007bff" 
  width={32} 
  height={32}
  className="vendor-menu-icon"
/>

// In a button or navigation
<button className="nav-item">
  <VendorsIcon color="#333" width={20} height={20} />
  <span>Vendors</span>
</button>
```

The icon maintains its aspect ratio and provides consistent theming through the configurable color prop, making it suitable for various UI contexts where vendor-related functionality needs visual representation.
