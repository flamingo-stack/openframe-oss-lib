A React component that renders the Miami Cyber Gang logo as an SVG with customizable colors and theming support.

## Key Components

- **MiamiCyberGangLogo**: Main React functional component that renders the logo SVG
- **MiamiCyberGangLogoProps**: TypeScript interface defining component props including size, colors, and styling options

## Usage Example

```typescript
import { MiamiCyberGangLogo } from './miami-cyber-gang-logo';

// Basic usage with default styling
<MiamiCyberGangLogo />

// Custom size and colors
<MiamiCyberGangLogo 
  size={120}
  originalBlack="#000000"
  originalPink="#FF1493" 
  originalWhite="#FFFFFF"
  className="my-logo"
/>

// Using CSS custom properties (recommended)
<MiamiCyberGangLogo 
  size={100}
  className="themed-logo"
/>
```

The component features a detailed circular logo design with text elements and decorative shapes. It uses CSS custom properties by default (`--ods-text-primary`, `--ods-accent`, `--ods-text-on-accent`) for automatic theming support, while allowing manual color overrides. The SVG scales proportionally based on the `size` prop while maintaining its 120x120 viewBox aspect ratio.
