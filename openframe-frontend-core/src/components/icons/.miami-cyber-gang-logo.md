A React component that renders an SVG logo for "Miami Cyber Gang" with customizable colors and size. The logo features a circular design with cyber-themed graphics and text elements in a Miami-inspired color palette.

## Key Components

- **MiamiCyberGangLogo**: Main React functional component that renders the SVG logo
- **MiamiCyberGangLogoProps**: TypeScript interface defining component props including size, color customization, and className

### Props
- `size`: Logo dimensions (default: 80px)
- `originalBlack`: Main circle color (defaults to ODS text primary)
- `originalPink`: Accent color (defaults to ODS accent color)
- `originalWhite`: Text/detail color (defaults to ODS text-on-accent)
- `className`: Additional CSS classes

## Usage Example

```typescript
import { MiamiCyberGangLogo } from './miami-cyber-gang-logo';

// Basic usage
<MiamiCyberGangLogo />

// Customized size and colors
<MiamiCyberGangLogo 
  size={120}
  originalBlack="#000000"
  originalPink="#FF1493"
  originalWhite="#FFFFFF"
  className="my-logo"
/>

// Using CSS custom properties
<MiamiCyberGangLogo 
  size={100}
  className="themed-logo"
/>
```

The component uses CSS custom properties by default, making it compatible with design system theming while allowing for explicit color overrides when needed.
