A customizable React SVG component that renders an external link icon, commonly used to indicate links that open in new tabs or navigate to external websites.

## Key Components

- **CustomExternalLinkIconProps**: TypeScript interface defining component props including optional styling and sizing properties
- **CustomExternalLinkIcon**: The main React functional component that renders an SVG external link icon with customizable appearance

## Usage Example

```typescript
import { CustomExternalLinkIcon } from './custom-external-link-icon';

// Basic usage with default styling
<CustomExternalLinkIcon />

// Customized with different size and color
<CustomExternalLinkIcon 
  width={24} 
  height={24} 
  fill="#0066cc" 
  className="external-link-icon" 
/>

// In a link component
<a href="https://example.com" target="_blank" rel="noopener">
  Visit External Site
  <CustomExternalLinkIcon width={16} height={16} fill="currentColor" />
</a>
```

The component accepts standard SVG props through spread operator, making it fully customizable. Default values provide a gray icon at 21x22 pixels, suitable for most use cases where an external link indicator is needed.
