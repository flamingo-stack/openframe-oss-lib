A React SVG component that renders a customizable external link icon, commonly used to indicate links that open in new windows or navigate to external websites.

## Key Components

- **CustomExternalLinkIconProps**: Interface defining the component's props including styling and sizing options
- **CustomExternalLinkIcon**: Functional component that renders an SVG external link icon with customizable properties

## Usage Example

```typescript
import { CustomExternalLinkIcon } from './custom-external-link-icon';

// Basic usage with defaults
<CustomExternalLinkIcon />

// Customized styling
<CustomExternalLinkIcon 
  className="external-link-icon" 
  fill="#007bff" 
  width={24} 
  height={24} 
/>

// In a link component
<a href="https://example.com" target="_blank">
  Visit site <CustomExternalLinkIcon width={16} height={16} />
</a>
```

The component accepts standard SVG props through prop spreading and provides sensible defaults: gray fill color (#888888) and 21x22px dimensions. The SVG paths create a classic external link symbol with an arrow pointing diagonally outward from a square.
