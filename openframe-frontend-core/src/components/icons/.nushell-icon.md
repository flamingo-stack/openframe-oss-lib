A React component that renders the Nushell shell logo as an SVG icon with customizable size, styling, and color properties.

## Key Components

- **`NushellIconProps`** - TypeScript interface defining optional props for size, className, and color
- **`NushellIcon`** - React functional component that renders the Nushell logo SVG
- **Default values** - Size defaults to 16px, color to white, with empty className

## Usage Example

```typescript
import { NushellIcon } from './nushell-icon'

// Basic usage with defaults (16px, white)
<NushellIcon />

// Custom size and color
<NushellIcon size={24} color="#4a90e2" />

// With custom styling
<NushellIcon 
  size={32} 
  className="hover:opacity-80 transition-opacity" 
  color="currentColor" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <NushellIcon size={20} />
  Open Nushell
</button>
```

The component uses a 24x24 viewBox and scales proportionally based on the size prop. The SVG path renders the distinctive "nu>" prompt symbol that represents the Nushell command-line interface.
