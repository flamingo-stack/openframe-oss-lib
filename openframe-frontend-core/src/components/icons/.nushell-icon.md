A React component that renders the Nushell shell logo as an SVG icon with customizable size, styling, and color properties.

## Key Components

- **`NushellIconProps`** - Interface defining component props: `size`, `className`, and `color`
- **`NushellIcon`** - Main React functional component that renders the Nushell logo SVG

## Usage Example

```typescript
import { NushellIcon } from './nushell-icon'

// Basic usage with defaults (16px, white color)
<NushellIcon />

// Custom size and color
<NushellIcon size={24} color="#4A90E2" />

// With custom styling
<NushellIcon 
  size={32} 
  className="shell-icon hover:opacity-80" 
  color="currentColor" 
/>
```

The component provides sensible defaults: 16px size, white color, and empty className. The SVG maintains its aspect ratio and can be styled with CSS classes for hover effects, animations, or layout positioning.
