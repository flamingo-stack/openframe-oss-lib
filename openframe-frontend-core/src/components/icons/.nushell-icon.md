Renders the Nushell shell logo as a configurable SVG icon component. This component is **deprecated** in favor of the `icons-v2-generated` icon set.

## Key Components

- **`NushellIconProps`** — Interface for size, className, and color customization
- **`NushellIcon`** — React functional component rendering the Nushell SVG path at a configurable size and color

## Usage Example

```typescript
import { NushellIcon } from './nushell-icon'

// Default usage
<NushellIcon />

// Custom size and color
<NushellIcon size={24} color="#FFC008" className="my-icon" />
```

> **⚠️ Deprecated:** Use icons from `icons-v2-generated` instead of this component.
