A customizable React SVG icon component that renders an open-source symbol featuring angled brackets with a diagonal slash, commonly used to represent coding or open-source development.

## Key Components

- **OpenSourceIcon**: Main React functional component that renders the SVG icon
- **OpenSourceIconProps**: TypeScript interface defining customizable properties:
  - `color` - Fill color for the icon (default: 'white')
  - `className` - CSS classes for styling
  - `width` - Icon width in pixels (default: 19)
  - `height` - Icon height in pixels (default: 18)

## Usage Example

```typescript
import { OpenSourceIcon } from './open-source-icon';

// Basic usage with defaults
<OpenSourceIcon />

// Customized with dark theme
<OpenSourceIcon 
  color="#333333" 
  className="hover:opacity-80" 
  width={24} 
  height={24} 
/>

// As a button icon
<button className="flex items-center gap-2">
  <OpenSourceIcon color="#10B981" width={16} height={16} />
  View Source
</button>
```

The SVG uses three path elements to create the bracket-slash design and maintains proper aspect ratio scaling. All props are optional with sensible defaults for immediate use.
