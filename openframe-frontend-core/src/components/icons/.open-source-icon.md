A React component that renders an SVG icon representing open source/code, featuring angle brackets and a forward slash symbol.

## Key Components

- **OpenSourceIcon**: Main React functional component that renders a customizable SVG icon
- **OpenSourceIconProps**: TypeScript interface defining optional props for styling and sizing
- **SVG paths**: Two path elements creating the visual representation of code brackets and slash

## Usage Example

```typescript
import { OpenSourceIcon } from './open-source-icon';

// Basic usage with defaults
<OpenSourceIcon />

// Customized with specific styling
<OpenSourceIcon 
  color="#007acc"
  width={24}
  height={22}
  className="code-icon"
/>

// In a button or link
<button className="open-source-btn">
  <OpenSourceIcon color="currentColor" />
  View Source Code
</button>
```

The component defaults to white color with 19x18 pixel dimensions, making it suitable for navigation bars, buttons, or any UI element indicating open source code availability.
