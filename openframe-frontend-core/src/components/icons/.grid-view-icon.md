A React SVG icon component that renders a 2x2 grid layout symbol, commonly used in UI interfaces for switching to grid view mode.

## Key Components

- **GridViewIcon**: Default exported React functional component that renders an SVG grid icon
- **Props interface**: Accepts optional `className`, `size`, and `color` properties for customization

## Usage Example

```typescript
import { GridViewIcon } from './grid-view-icon'

// Basic usage with defaults (22px, white fill)
<GridViewIcon />

// Custom styling
<GridViewIcon 
  size={32} 
  color="black" 
  className="hover:opacity-75 transition-opacity" 
/>

// In a button component
<button className="flex items-center gap-2">
  <GridViewIcon size={16} color="currentColor" />
  Grid View
</button>
```

The component provides sensible defaults (22px size, white color, basic width/height classes) while allowing full customization through props. The SVG uses two path elements to create a four-square grid pattern with rounded corners.
