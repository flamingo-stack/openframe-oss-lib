A React component that renders a customizable grid view icon using SVG, commonly used for toggling between grid and list layouts in user interfaces.

## Key Components

- **GridViewIcon**: Main functional component that renders a 2x2 grid icon
- **Props interface**: Accepts `className`, `size`, and `color` for customization
- **SVG paths**: Two path elements that create the four-square grid pattern

## Usage Example

```typescript
import { GridViewIcon } from './grid-view-icon'

// Default usage (white icon, 22px, w-5 h-5 classes)
<GridViewIcon />

// Custom size and color
<GridViewIcon size={24} color="#333" />

// Custom styling with Tailwind classes
<GridViewIcon 
  className="w-6 h-6 hover:opacity-80" 
  color="currentColor"
/>

// In a button for layout switching
<button className="p-2 rounded">
  <GridViewIcon size={20} color="#6b7280" />
</button>
```

The component uses a 21x22 viewBox and creates a symmetrical 2x2 grid pattern. The dual path structure ensures proper rendering across different browsers and provides visual consistency for grid/tile view toggles in applications.
