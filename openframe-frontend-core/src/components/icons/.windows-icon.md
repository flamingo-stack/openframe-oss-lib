A React SVG component that renders the Windows logo icon with customizable styling properties.

## Key Components

- **`WindowsIconProps`** - Interface extending React's SVGProps for type-safe prop passing
- **`WindowsIcon`** - Main component function that renders the Windows four-pane logo as an SVG

## Usage Example

```typescript
import { WindowsIcon } from './windows-icon'

// Basic usage with default white color
<WindowsIcon />

// With custom styling
<WindowsIcon 
  className="my-icon" 
  color="blue" 
  width={24} 
  height={24} 
/>

// As a clickable button icon
<button onClick={handleClick}>
  <WindowsIcon color="#0078d4" className="mr-2" />
  Open Windows Menu
</button>
```

The component accepts all standard SVG props and defaults to a 16x16 viewBox with white fill color. The `color` prop controls the fill color of the Windows logo paths, making it easy to match your design system's color scheme.
