A React component that renders an X (close) icon as an SVG, typically used for close buttons in modals, notifications, or other dismissible UI elements.

## Key Components

- **XIcon**: Main component that renders a circular X icon with customizable properties
- **XIconProps**: TypeScript interface defining the component's props including color, dimensions, and CSS class

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | Fill color for the icon (currently not applied to paths) |
| `width` | `number` | `16` | Width of the SVG in pixels |
| `height` | `number` | `16` | Height of the SVG in pixels |
| `className` | `string` | `''` | Additional CSS classes to apply |

## Usage Example

```typescript
import { XIcon } from './x-icon';

// Basic usage with defaults
<XIcon />

// Customized icon
<XIcon 
  width={24} 
  height={24} 
  className="hover:opacity-75 cursor-pointer"
  color="red"
/>

// In a close button
<button onClick={handleClose} className="absolute top-2 right-2">
  <XIcon width={20} height={20} />
</button>
```

**Note**: The `color` prop is defined but not currently used in the SVG paths, which are hardcoded to `#FAFAFA` (light gray). The paths would need to use `fill={color}` instead of the hardcoded hex values to make the color prop functional.
