A React component that renders an SVG close (X) icon with customizable styling properties.

## Key Components

- **XIcon** - Main functional component that renders an SVG X/close icon
- **XIconProps** - TypeScript interface defining optional props for customization
- Props include `color`, `width`, `height`, and `className` with sensible defaults

## Usage Example

```typescript
import { XIcon } from './x-icon';

// Basic usage with defaults (16x16, white color)
<XIcon />

// Customized icon
<XIcon 
  width={24} 
  height={24} 
  color="red" 
  className="close-button" 
/>

// In a modal or dialog close button
<button onClick={handleClose} className="modal-close">
  <XIcon width={20} height={20} color="#333" />
</button>
```

The icon features a circular outline with an X symbol inside, making it ideal for close buttons, modal dismissals, or general "remove/cancel" actions. The SVG is scalable and maintains crisp edges at any size.
