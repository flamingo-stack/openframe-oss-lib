A React SVG component that renders the official Google logo with proper brand colors and scalable vector graphics.

## Key Components

- **GoogleLogo**: React functional component that renders the Google "G" logo as an inline SVG
- Accepts standard SVG props and optional `className` for styling
- Uses authentic Google brand colors (#4285F4, #34A853, #FBBC05, #EB4335)
- Fixed 24x24px default size with scalable viewBox

## Usage Example

```typescript
import { GoogleLogo } from './google-logo';

// Basic usage
<GoogleLogo />

// With custom styling
<GoogleLogo 
  className="w-8 h-8" 
  width={32} 
  height={32} 
/>

// With additional SVG props
<GoogleLogo 
  className="hover:opacity-80 transition-opacity"
  onClick={() => handleGoogleSignIn()}
  role="button"
  aria-label="Sign in with Google"
/>
```

The component spreads all SVG props, making it fully customizable while maintaining the authentic Google logo appearance. Perfect for authentication buttons, branding elements, or any UI that needs the recognizable Google logo icon.
