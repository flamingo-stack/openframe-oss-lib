A React SVG component that renders the official Google logo using the brand's standard colors and design.

## Key Components

- **GoogleLogo**: Main React functional component that renders the Google "G" logo
- Accepts standard SVG props and optional `className` for styling
- Fixed 24x24 pixel dimensions with scalable SVG viewBox
- Uses Google's official brand colors (#4285F4, #34A853, #FBBC05, #EB4335)

## Usage Example

```typescript
import { GoogleLogo } from './google-logo';

// Basic usage
<GoogleLogo />

// With custom styling
<GoogleLogo className="w-8 h-8" />

// With additional SVG props
<GoogleLogo 
  className="hover:opacity-80" 
  onClick={handleGoogleSignIn}
  aria-label="Sign in with Google"
/>

// Custom size override
<GoogleLogo 
  width="32" 
  height="32" 
  className="cursor-pointer"
/>
```

The component is commonly used in authentication flows, social login buttons, and anywhere the Google brand needs to be displayed with consistent visual identity.
