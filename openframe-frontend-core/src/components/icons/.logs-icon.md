A React SVG icon component that renders a logs/document icon with customizable styling properties.

## Key Components

- **LogsIcon**: Main functional component that renders an SVG icon
- **LogsIconProps**: TypeScript interface defining optional props:
  - `className`: CSS class for styling
  - `size`: Icon dimensions (default: 24px)
  - `color`: Fill color (default: 'white')

## Usage Example

```typescript
import { LogsIcon } from './logs-icon'

// Basic usage with defaults
<LogsIcon />

// Customized icon
<LogsIcon 
  size={32} 
  color="#3b82f6" 
  className="hover:opacity-80" 
/>

// In a button component
<button className="flex items-center gap-2">
  <LogsIcon size={16} color="currentColor" />
  View Logs
</button>
```

The icon displays a document/logs symbol with bullet points and lines, making it suitable for log viewers, documentation sections, or file management interfaces.
