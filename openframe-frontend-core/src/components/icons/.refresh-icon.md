A customizable SVG refresh/reload icon component for React applications.

## Key Components

- **RefreshIconProps**: Interface defining optional props for customization
  - `className?: string` - CSS classes for styling
  - `size?: number` - Icon dimensions (default: 22px)
  - `color?: string` - Icon color (default: '#888888')

- **RefreshIcon**: Functional React component that renders a circular arrow refresh icon

## Usage Example

```typescript
import { RefreshIcon } from './refresh-icon';

// Basic usage with defaults
<RefreshIcon />

// Customized icon
<RefreshIcon 
  size={24} 
  color="#007bff" 
  className="refresh-button" 
/>

// In a button component
<button onClick={handleRefresh}>
  <RefreshIcon size={16} color="#ffffff" />
  Refresh Data
</button>
```

The component renders as an SVG with two curved arrow paths forming a circular refresh symbol, commonly used in UI elements for reload, refresh, or sync actions.
