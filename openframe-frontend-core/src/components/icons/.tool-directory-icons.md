Collection of SVG icons for different tool categories in a directory interface, including security operations, network operations, IT services, operations, and general exploration.

## Key Components

- **SOCIcon** - Security Operations Center icon with shield and checkmark design
- **NOCIcon** - Network Operations Center icon featuring server/monitor representation
- **ITIcon** - IT services icon with stacked server/hardware visualization
- **OPSIcon** - Operations icon with folder and bookmark elements
- **ExploreCategoriesIcon** - General exploration icon with key and folder combination

All components share a common `IconProps` interface for consistent styling and theming.

## Usage Example

```typescript
import { SOCIcon, NOCIcon, ITIcon, OPSIcon, ExploreCategoriesIcon } from './tool-directory-icons';

// Basic usage with default styles
<SOCIcon />

// Custom styling
<NOCIcon className="w-8 h-8" fill="#3B82F6" />

// In a directory listing
const categories = [
  { name: 'Security Operations', icon: SOCIcon },
  { name: 'Network Operations', icon: NOCIcon },
  { name: 'IT Services', icon: ITIcon },
  { name: 'Operations', icon: OPSIcon },
  { name: 'Explore All', icon: ExploreCategoriesIcon }
];

{categories.map(({ name, icon: Icon }) => (
  <div key={name}>
    <Icon className="w-6 h-6 mr-2" fill="currentColor" />
    <span>{name}</span>
  </div>
))}
```
