A collection of React SVG icon components for different IT service categories, including SOC, NOC, IT, Operations, and category exploration.

## Key Components

- **SOCIcon** - Security Operations Center icon with shield and checkmark design
- **NOCIcon** - Network Operations Center icon with server/monitor representation
- **ITIcon** - IT services icon with server rack design and indicator lights
- **OPSIcon** - Operations icon with folder and bookmark elements
- **ExploreCategoriesIcon** - Category exploration icon with key and folder motifs

All components accept `IconProps` interface with optional `className` and `fill` properties.

## Usage Example

```typescript
import { SOCIcon, NOCIcon, ITIcon, OPSIcon, ExploreCategoriesIcon } from './tool-directory-icons';

// Default usage with white fill and w-6 h-6 size
<SOCIcon />

// Custom styling
<NOCIcon className="w-8 h-8" fill="#3B82F6" />

// Dynamic styling
<ITIcon 
  className={`w-5 h-5 ${isActive ? 'text-blue-500' : 'text-gray-400'}`} 
  fill="currentColor" 
/>

// Operations and exploration icons
<OPSIcon className="w-10 h-10" fill="#10B981" />
<ExploreCategoriesIcon className="w-4 h-4" fill="#6B7280" />
```
