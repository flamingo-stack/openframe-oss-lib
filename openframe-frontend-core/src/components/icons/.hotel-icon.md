A React SVG icon component that renders a hotel/building icon with customizable styling properties.

## Key Components

- **HotelIconProps** - Interface defining optional props for className, size, and color customization
- **HotelIcon** - Main functional component that renders an SVG hotel icon with default styling

## Usage Example

```typescript
import { HotelIcon } from './hotel-icon'

// Basic usage with defaults (24px, white color)
<HotelIcon />

// Customized size and color
<HotelIcon size={32} color="#333" />

// With CSS classes for additional styling
<HotelIcon 
  className="hover:opacity-80 transition-opacity" 
  size={48} 
  color="currentColor" 
/>

// In a button or interactive element
<button className="flex items-center gap-2">
  <HotelIcon size={20} color="white" />
  Hotels
</button>
```

The component uses a 65x64 viewBox SVG with a detailed hotel building path. All props have sensible defaults: 24px size, white color, and empty className. The icon scales proportionally and accepts any valid CSS color value.
