A React component that renders an SVG hotel icon with customizable styling properties.

## Key Components

- **HotelIcon**: Main functional component that renders a hotel building SVG
- **HotelIconProps**: TypeScript interface defining optional props:
  - `className`: CSS class string for styling
  - `size`: Icon dimensions in pixels (default: 24)
  - `color`: Fill color for the icon (default: 'white')

## Usage Example

```typescript
import { HotelIcon } from './hotel-icon'

// Basic usage with defaults
<HotelIcon />

// Customized icon
<HotelIcon 
  size={32} 
  color="#333" 
  className="hotel-building-icon" 
/>

// Large icon for headers
<HotelIcon size={48} color="blue" />
```

The component renders a detailed hotel building outline with a door and windows, making it suitable for hotel booking interfaces, travel applications, or accommodation listings.
