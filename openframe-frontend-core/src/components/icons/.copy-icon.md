A reusable React SVG icon component that renders a copy/duplicate symbol with customizable styling properties.

## Key Components

- **CopyIconProps**: Interface defining optional props for className, size, and color customization
- **CopyIcon**: Main component that renders an SVG copy icon with two overlapping rectangles representing the copy action

## Usage Example

```typescript
import { CopyIcon } from './copy-icon';

// Basic usage with default 24px size and gray color
<CopyIcon />

// Customized with larger size and blue color
<CopyIcon 
  size={32} 
  color="#0066cc" 
  className="copy-button-icon" 
/>

// In a copy button component
<button onClick={handleCopy}>
  <CopyIcon size={16} color="currentColor" />
  Copy to clipboard
</button>
```

The component uses a 24x24 viewBox and scales proportionally based on the size prop. The SVG paths create a visual representation of two overlapping document rectangles, universally recognized as the copy/duplicate icon.
