A React component that renders an SVG copy icon with customizable styling properties.

## Key Components

- **CopyIconProps** - Interface defining optional props for className, size, and color customization
- **CopyIcon** - Functional component that renders a copy/duplicate icon using SVG paths

## Usage Example

```typescript
import { CopyIcon } from './copy-icon';

// Basic usage with defaults (24px, gray color)
<CopyIcon />

// Customized icon
<CopyIcon 
  size={16} 
  color="#007acc" 
  className="hover:opacity-80" 
/>

// Large icon for buttons
<button className="flex items-center gap-2">
  <CopyIcon size={20} color="currentColor" />
  Copy to clipboard
</button>
```

The component provides sensible defaults (24px size, gray color) while allowing full customization through props. The SVG uses two path elements to create a layered copy icon design that scales cleanly at any size.
