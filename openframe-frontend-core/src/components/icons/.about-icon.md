A React component that renders a customizable information/question mark icon using SVG, typically used for "About" sections or help indicators.

## Key Components

- **AboutIcon**: Main functional component that renders an SVG icon
- **AboutIconProps**: TypeScript interface defining optional props for customization:
  - `width` - Icon width in pixels (default: 24)
  - `height` - Icon height in pixels (default: 24) 
  - `className` - CSS classes for styling
  - `color` - Fill color for the icon (default: 'white')

The component renders a circular icon with an information symbol (question mark with dot) commonly used in user interfaces to indicate help or additional information.

## Usage Example

```typescript
import { AboutIcon } from './about-icon';

// Basic usage with defaults
<AboutIcon />

// Customized size and color
<AboutIcon 
  width={32} 
  height={32} 
  color="#007bff" 
  className="hover:opacity-80" 
/>

// Large blue icon for help section
<AboutIcon 
  width={48} 
  height={48} 
  color="#1e40af" 
  className="cursor-pointer transition-colors" 
/>
```

The icon uses a 24x24 viewBox and scales proportionally based on the provided width and height props, making it suitable for various UI contexts from navigation bars to help tooltips.
