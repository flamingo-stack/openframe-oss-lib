A customizable React SVG component that renders an information/about icon with a circular design and question mark symbol.

## Key Components

- **AboutIcon**: Main functional component that renders an SVG information icon
- **AboutIconProps**: TypeScript interface defining optional props for customization
  - `width`: Icon width (default: 24px)
  - `height`: Icon height (default: 24px) 
  - `className`: CSS classes for styling
  - `color`: Fill color (default: 'white')

## Usage Example

```typescript
import { AboutIcon } from './about-icon';

// Basic usage with defaults
<AboutIcon />

// Customized icon
<AboutIcon 
  width={32}
  height={32}
  color="#007bff"
  className="hover:opacity-75 transition-opacity"
/>

// In a button or interactive element
<button className="flex items-center gap-2">
  <AboutIcon width={16} height={16} color="currentColor" />
  About
</button>
```

The component provides a consistent information icon that can be easily integrated into buttons, navigation menus, or help sections while maintaining full control over size, styling, and color.
