A React SVG icon component that renders a "refresh" or "automation" symbol with two curved arrows forming a circular pattern, commonly used to represent automated processes or refresh actions.

## Key Components

- **AutomateEverythingIcon**: Main export - a functional React component that renders an SVG icon
- **Props Interface**: Extends standard SVG props with optional `className` for styling flexibility
- **SVG Paths**: Two curved arrow paths that create a circular refresh/automation symbol in golden yellow (#FFC008)

## Usage Example

```typescript
import { AutomateEverythingIcon } from './automate-everything-icon';

// Basic usage
<AutomateEverythingIcon />

// With custom styling
<AutomateEverythingIcon 
  className="w-6 h-6 text-blue-500" 
  onClick={handleRefresh}
/>

// With custom size and props
<AutomateEverythingIcon 
  width={32} 
  height={32} 
  className="hover:opacity-80 cursor-pointer"
  aria-label="Refresh data"
/>
```

The icon has a default size of 24x24 pixels and uses a golden yellow fill color. All standard SVG props are supported, making it highly customizable for different use cases like buttons, navigation elements, or status indicators.
