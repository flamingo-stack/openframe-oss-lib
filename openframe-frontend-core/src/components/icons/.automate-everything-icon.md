A React component that renders an SVG icon representing automation/refresh functionality with curved arrow paths in a golden color scheme.

## Key Components

- **AutomateEverythingIcon**: Main React functional component that renders an SVG icon
- **Props interface**: Extends `React.SVGProps<SVGSVGElement>` with optional `className` prop
- **SVG paths**: Two curved arrow paths forming a refresh/automation symbol

## Usage Example

```typescript
import { AutomateEverythingIcon } from './automate-everything-icon';

// Basic usage
<AutomateEverythingIcon />

// With custom styling
<AutomateEverythingIcon 
  className="w-6 h-6" 
  onClick={() => handleRefresh()}
/>

// With custom size and color override
<AutomateEverythingIcon 
  width={32} 
  height={32}
  style={{ fill: '#FF6B6B' }}
/>
```

The icon is designed at 24x24px with a golden fill color (#FFC008) and can be customized through standard SVG props including size, className, and event handlers.
