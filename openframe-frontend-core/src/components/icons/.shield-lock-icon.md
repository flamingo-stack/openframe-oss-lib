A React SVG icon component that renders a shield with lock symbol, commonly used to represent security, protection, or authentication features.

## Key Components

- **ShieldLockIconProps** - Interface defining component properties with optional styling customization
- **ShieldLockIcon** - Main functional component that renders the shield lock SVG with configurable appearance

## Usage Example

```typescript
import { ShieldLockIcon } from './shield-lock-icon'

// Basic usage with defaults (24px, white)
<ShieldLockIcon />

// Customized appearance
<ShieldLockIcon 
  size={32} 
  color="#3b82f6" 
  className="security-icon hover:opacity-80" 
/>

// In a security feature card
<div className="flex items-center gap-2">
  <ShieldLockIcon size={20} color="green" />
  <span>Secure Authentication</span>
</div>
```

The icon features a shield outline with an internal padlock design, rendered as a stroke-only SVG that scales cleanly at any size. Default styling uses white color at 24px, but all visual properties are customizable through props.
