A customizable React SVG icon component that renders a shield with a lock symbol, typically used to represent security or protection features.

## Key Components

- **ShieldLockIcon**: Main functional component that renders an SVG shield with lock icon
- **ShieldLockIconProps**: TypeScript interface defining optional props for customization
  - `className`: CSS class for styling
  - `size`: Icon dimensions (default: 24px)
  - `color`: Stroke color (default: 'white')

## Usage Example

```typescript
import { ShieldLockIcon } from './shield-lock-icon'

// Basic usage
<ShieldLockIcon />

// Customized icon
<ShieldLockIcon 
  size={32} 
  color="#3B82F6" 
  className="security-icon" 
/>

// In a security feature section
<div className="feature-card">
  <ShieldLockIcon size={48} color="#10B981" />
  <h3>Secure Authentication</h3>
  <p>Your data is protected with enterprise-grade security</p>
</div>
```

The component uses stroke-based SVG rendering, making it ideal for icons that need to match different design themes through color customization.
