A React component that renders an SVG icon depicting a "jump in" action with a monitor/screen and download arrow, crossed out with a diagonal line to indicate restriction or unavailability.

## Key Components

- **JumpInIcon**: Main functional component that renders the SVG icon
- **JumpInIconProps**: TypeScript interface defining component props
  - `size?: number` - Icon dimensions (default: 48px)
  - `color?: string` - Stroke color (default: 'white')  
  - `className?: string` - Additional CSS classes

## Usage Example

```typescript
import { JumpInIcon } from './jump-in-icon';

// Basic usage with defaults
<JumpInIcon />

// Customized icon
<JumpInIcon 
  size={32} 
  color="#ff0000" 
  className="my-custom-class" 
/>

// In a button or interactive element
<button className="action-btn">
  <JumpInIcon size={24} color="currentColor" />
  Jump In Disabled
</button>
```

The icon features a monitor with a download arrow below it, all crossed out by a diagonal line, making it ideal for indicating disabled or restricted access to streaming, downloading, or "jumping into" content.
