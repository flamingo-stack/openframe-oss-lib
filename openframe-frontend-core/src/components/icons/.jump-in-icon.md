A deprecated React SVG icon component rendering a "Jump In" icon (chart with a diagonal slash/cancel overlay) with configurable size, color, and className props.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `JumpInIcon` | `React.FC<JumpInIconProps>` | SVG icon component — **deprecated** |
| `JumpInIconProps` | `interface` | Props interface for the component |

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `48` | Width and height of the SVG in pixels |
| `color` | `string` | `'white'` | Stroke color of the icon paths |
| `className` | `string` | `''` | Additional CSS class names |

## Usage Example

```typescript
import { JumpInIcon } from './jump-in-icon';

// Default usage
<JumpInIcon />

// Custom size and color
<JumpInIcon size={32} color="#FFC008" className="my-icon" />
```

> ⚠️ **Deprecated:** This component is deprecated. Use icons from `icons-v2-generated` instead.
