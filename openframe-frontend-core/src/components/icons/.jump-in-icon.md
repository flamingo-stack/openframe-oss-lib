React SVG icon component that renders a "jump in" interface symbol with customizable styling properties.

## Key Components

- **JumpInIcon**: Main functional component that renders an SVG icon
- **JumpInIconProps**: TypeScript interface defining component props:
  - `size`: Optional number for icon dimensions (default: 48px)
  - `color`: Optional string for stroke color (default: 'white')  
  - `className`: Optional string for additional CSS classes

## Usage Example

```typescript
import { JumpInIcon } from './jump-in-icon';

// Basic usage with defaults
<JumpInIcon />

// Customized icon
<JumpInIcon 
  size={64} 
  color="#3B82F6" 
  className="hover:opacity-80 transition-opacity" 
/>

// Small icon for buttons
<JumpInIcon size={24} color="currentColor" />
```

The icon features a computer/display symbol with an arrow pointing downward and is enclosed in a circle with a diagonal line through it, commonly used to represent "jump in" or "enter" actions in user interfaces. The SVG uses a 80x80 viewBox and scales proportionally based on the size prop.
