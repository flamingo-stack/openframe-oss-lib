A collection of SVG icon components for vendor showcase features, providing customizable icons with consistent styling and props interface.

## Key Components

- **VendorShowcaseMainIcon** - Main showcase icon (200x200) with pink accent color
- **TransparentTrustedIcon** - Trust/verification icon with customizable size and color
- **ModularHellIcon** - Modular architecture icon featuring grid/block design
- **RapidInnovationIcon** - Innovation/speed icon with lightning bolt styling  
- **LowerTCOIcon** - Cost optimization icon with circular/orbital design

## Usage Example

```typescript
import { 
  VendorShowcaseMainIcon, 
  TransparentTrustedIcon,
  ModularHellIcon 
} from './vendor-showcase-icons';

// Basic usage
<VendorShowcaseMainIcon className="mx-auto" />

// Customized icons
<TransparentTrustedIcon 
  size={64} 
  color="#00FF00" 
  className="inline-block" 
/>

<ModularHellIcon 
  size={24} 
  color="#FF5733"
  className="mr-2" 
/>

// Feature showcase grid
<div className="grid grid-cols-2 gap-4">
  <RapidInnovationIcon size={48} color="#F357BB" />
  <LowerTCOIcon size={48} color="#5EFAF0" />
</div>
```

All icons accept optional `className`, `size`, and `color` props for flexible styling and theming.
