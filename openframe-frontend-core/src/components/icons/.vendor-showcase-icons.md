A React component library providing SVG icons for vendor showcase features, including a main showcase icon and four feature-specific icons with customizable styling.

## Key Components

- **VendorShowcaseMainIcon** - Main showcase icon (200x200px) with fixed pink styling
- **TransparentTrustedIcon** - Trust/transparency feature icon with customizable color and size
- **ModularHellIcon** - Modular architecture feature icon with grid pattern design
- **RapidInnovationIcon** - Innovation speed feature icon with lightning bolt motif
- **LowerTCOIcon** - Cost optimization feature icon with circular arrow design

## Usage Example

```typescript
import { 
  VendorShowcaseMainIcon, 
  TransparentTrustedIcon,
  ModularHellIcon 
} from './vendor-showcase-icons';

// Main showcase icon (fixed styling)
<VendorShowcaseMainIcon className="hero-icon" />

// Feature icons with custom styling
<TransparentTrustedIcon 
  className="feature-icon" 
  color="#00FF00" 
  size={48} 
/>

<ModularHellIcon 
  className="grid-icon" 
  size={32} 
  color="#FF0000" 
/>
```

All feature icons accept optional `className`, `size` (number), and `color` (string) props for flexible styling integration.
