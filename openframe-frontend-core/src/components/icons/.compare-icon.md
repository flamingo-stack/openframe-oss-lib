A React component that renders an SVG icon depicting a comparison or balance scale symbol, commonly used for product comparison features.

## Key Components

- **CompareIcon**: Main React functional component that renders the comparison scale SVG
- **CompareIconProps**: Interface defining optional props for customization
  - `color`: Fill color for the icon (default: 'white')
  - `className`: CSS classes for styling
  - `width`/`height`: Icon dimensions (default: 24px)

## Usage Example

```typescript
import { CompareIcon } from './compare-icon';

// Basic usage
<CompareIcon />

// Customized icon
<CompareIcon 
  color="#007bff" 
  width={32} 
  height={32} 
  className="compare-btn-icon"
/>

// In a button component
<button className="compare-button">
  <CompareIcon color="currentColor" width={16} height={16} />
  Compare Products
</button>
```

The component features a balance scale design with a vertical center post and two weighing pans, making it ideal for comparison functionality in e-commerce applications or data comparison interfaces.
