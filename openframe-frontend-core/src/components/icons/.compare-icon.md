A React functional component that renders a customizable SVG icon representing a comparison or balance scale symbol.

## Key Components

- **CompareIcon**: Main React component that renders an SVG icon with a central vertical line and two balanced scales
- **CompareIconProps**: TypeScript interface defining component props including color, className, width, and height
- Default props: 24x24 dimensions, white fill color, empty className

## Usage Example

```typescript
import { CompareIcon } from './compare-icon';

// Basic usage with defaults
<CompareIcon />

// Customized with different props
<CompareIcon 
  width={32} 
  height={32} 
  color="#3b82f6" 
  className="hover:opacity-80" 
/>

// In a button or interactive element
<button className="flex items-center gap-2">
  <CompareIcon width={16} height={16} color="currentColor" />
  Compare Items
</button>
```

The SVG uses a 24x24 viewBox and consists of two main path elements: a central vertical line with circular top element, and two symmetrical scale/balance icons on either side. All visual properties are customizable through props while maintaining the icon's aspect ratio and visual consistency.
