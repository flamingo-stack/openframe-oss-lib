A React component that renders the Trustpilot star icon as an SVG with customizable styling and colors.

## Key Components

- **TrustpilotIcon**: Main React functional component that renders the Trustpilot star logo
- **TrustpilotIconProps**: TypeScript interface defining optional props for customization
- **SVG paths**: Two path elements creating the star shape with primary and accent colors

## Usage Example

```typescript
import { TrustpilotIcon } from './trustpilot-icon';

// Basic usage with defaults
<TrustpilotIcon />

// Customized size and colors
<TrustpilotIcon 
  width={100}
  height={100}
  className="my-trustpilot-icon"
  firstColor="#ff6b35"
  secondColor="#cc4125"
/>

// Responsive sizing
<TrustpilotIcon 
  width="100%"
  height="auto"
  className="w-8 h-8"
/>
```

The component accepts optional props for dimensions (width/height), CSS classes, and two colors that control the star's appearance. The default colors match Trustpilot's brand green (#00b67a primary, #005128 accent). The SVG maintains its aspect ratio and scales appropriately based on the provided dimensions.
