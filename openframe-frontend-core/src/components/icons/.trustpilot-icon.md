A React component that renders the Trustpilot star logo as an SVG icon with customizable styling and colors.

## Key Components

- **TrustpilotIcon** - Main component that renders the Trustpilot star SVG
- **TrustpilotIconProps** - TypeScript interface defining component props including dimensions, styling, and color customization

## Usage Example

```typescript
import { TrustpilotIcon } from './trustpilot-icon';

// Basic usage with default colors
<TrustpilotIcon width={100} height={100} />

// Customized with different colors and CSS class
<TrustpilotIcon 
  width="50px"
  height="50px"
  className="my-icon"
  firstColor="#ff6b35"
  secondColor="#2c1810"
/>

// Responsive usage
<TrustpilotIcon 
  width="100%" 
  height="auto"
  className="responsive-icon"
/>
```

The component uses Trustpilot's signature green colors by default (`#00b67a` and `#005128`) but allows full customization of dimensions and colors through props.
