A React component that renders an SVG sun icon with customizable styling properties for theme switching and UI elements.

## Key Components

- **SunIcon**: React functional component that renders a detailed sun SVG icon
- **SunIconProps**: TypeScript interface defining optional props for customization
- **SVG Elements**: Two path elements creating the sun's center circle and radiating rays

## Usage Example

```typescript
import { SunIcon } from './sun-icon';

// Basic usage with defaults (24x24, white)
<SunIcon />

// Customized for dark mode toggle
<SunIcon 
  width={32} 
  height={32} 
  color="#f59e0b" 
  className="hover:scale-110 transition-transform" 
/>

// Small icon for status indicators
<SunIcon width={16} height={16} color="currentColor" />
```

The component provides sensible defaults (24x24 pixels, white color) while allowing full customization through props. The SVG uses a viewBox of "0 0 24 24" ensuring consistent scaling regardless of the specified width/height. The icon consists of a central circle with radiating rays, making it ideal for light mode indicators, weather displays, or theme toggle buttons.
