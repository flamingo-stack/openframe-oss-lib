A customizable React SVG component that renders a sun icon, commonly used for light theme toggles or weather displays.

## Key Components

- **SunIcon**: Main React functional component that renders an SVG sun icon
- **SunIconProps**: TypeScript interface defining optional props for customization
  - `color`: Fill color (default: 'white')
  - `className`: CSS class for styling
  - `width`/`height`: Dimensions (default: 24px)

## Usage Example

```typescript
import { SunIcon } from './sun-icon';

// Basic usage
<SunIcon />

// Customized sun icon
<SunIcon 
  color="orange" 
  width={32} 
  height={32}
  className="theme-toggle-icon"
/>

// In a theme toggle button
<button onClick={toggleTheme}>
  <SunIcon color="currentColor" width={20} height={20} />
</button>
```

The component features a detailed sun design with radiating rays and accepts standard SVG props for easy integration into design systems and theming contexts.
