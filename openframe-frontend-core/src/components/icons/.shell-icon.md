A deprecated shell/terminal icon SVG component for React.

## Key Components

- **`ShellIcon`** — A React SVG icon that renders a terminal/shell window symbol with a prompt indicator and cursor line.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `"w-5 h-5"` | CSS class for the SVG element |
| `size` | `number` | `22` | Width and height in pixels |
| `color` | `string` | `"white"` | Stroke color of the icon paths |

## Usage Example

```typescript
import { ShellIcon } from './shell-icon'

// Default usage
<ShellIcon />

// Custom size and color
<ShellIcon size={32} color="#FFC008" className="w-8 h-8" />
```

> ⚠️ **Deprecated:** This component is deprecated. Use icons from `icons-v2-generated` instead.
