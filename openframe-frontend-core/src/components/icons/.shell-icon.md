A React component that renders an SVG icon representing a terminal or shell interface, commonly used in developer tools and command-line applications.

## Key Components

- **ShellIcon**: React functional component that renders a terminal/shell SVG icon
- **Props**:
  - `className`: Optional CSS classes (defaults to "w-5 h-5")
  - `size`: Icon dimensions in pixels (defaults to 22)
  - `color`: Stroke color for the SVG paths (defaults to 'white')

## Usage Example

```typescript
import { ShellIcon } from './shell-icon'

// Basic usage with defaults
<ShellIcon />

// Custom styling
<ShellIcon 
  className="w-8 h-8" 
  size={32} 
  color="blue" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <ShellIcon size={18} color="currentColor" />
  Terminal
</button>
```

The icon features a rectangular terminal window outline with a command prompt indicator (">") and cursor line, making it perfect for CLI tools, code editors, or developer interfaces.
