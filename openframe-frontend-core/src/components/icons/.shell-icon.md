A React component that renders a terminal shell icon as an SVG, commonly used in development and command-line interfaces.

## Key Components

- **ShellIcon**: React functional component that renders a terminal shell SVG icon
- **Props interface**: Accepts `className`, `size`, and `color` parameters for customization
- **SVG path**: Terminal window outline with command prompt indicator

## Usage Example

```typescript
import { ShellIcon } from './shell-icon'

// Basic usage with defaults
<ShellIcon />

// Customized with different size and color
<ShellIcon 
  size={32} 
  color="black" 
  className="w-8 h-8" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <ShellIcon size={16} color="currentColor" />
  Terminal
</button>
```

The component uses a transparent fill with a colored stroke to create a clean, outline-style terminal icon. Default styling provides a 22px white icon with Tailwind CSS classes for responsive sizing.
