A React component that renders a customizable bot/robot SVG icon with configurable color and styling properties.

## Key Components

- **`BotIconProps`** - Interface extending SVG props with optional `color` parameter
- **`BotIcon`** - Main component function that renders the robot SVG icon

## Usage Example

```typescript
import { BotIcon } from './bot-icon'

// Basic usage with default styling
<BotIcon />

// Custom color and size
<BotIcon 
  color="#3b82f6" 
  className="h-8 w-8" 
/>

// With additional SVG properties
<BotIcon 
  color="currentColor"
  className="h-10 w-10"
  onClick={() => console.log('Bot clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component defaults to a 24x24px white icon with `h-6 w-6` classes. The `color` prop controls both stroke and fill colors, and all standard SVG properties can be passed through via props spreading.
