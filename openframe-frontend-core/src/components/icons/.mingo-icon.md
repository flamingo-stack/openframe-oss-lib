A React SVG icon component that renders a custom "Mingo" icon with configurable color and standard SVG properties.

## Key Components

- **`MingoIconProps`** - TypeScript interface extending `React.SVGProps<SVGSVGElement>` with an optional `color` prop
- **`MingoIcon`** - React functional component that renders an SVG icon with two circles and geometric paths, defaulting to white color

## Usage Example

```typescript
import { MingoIcon } from './mingo-icon'

// Basic usage with default white color
<MingoIcon />

// Custom color and className
<MingoIcon color="#3b82f6" className="w-6 h-6" />

// With additional SVG properties
<MingoIcon 
  color="currentColor" 
  width={32} 
  height={32}
  onClick={() => console.log('clicked')}
/>

// Using with Tailwind classes
<MingoIcon 
  color="rgb(239 68 68)" 
  className="hover:scale-110 transition-transform cursor-pointer" 
/>
```

The icon consists of two circular elements and rectangular shapes forming a distinctive logo design. All SVG props are forwarded to the underlying `<svg>` element, making it fully customizable for styling and event handling.
