A React component that renders the official Slack logo as an SVG icon with customizable dimensions and colors. The component creates Slack's distinctive four-colored interlocking shape pattern with proper scaling and styling support.

## Key Components

- **SlackIcon**: Main functional component that renders the Slack logo SVG
- **SlackIconProps**: TypeScript interface defining component props including dimensions, CSS classes, and color override options
- **Color Variables**: Four brand-specific colors (red, blue, green, yellow) that can be overridden with a single `injectedColor` prop

## Usage Example

```typescript
import { SlackIcon } from './slack-icon'

// Basic usage with default 24x24 size
<SlackIcon />

// Custom size and styling
<SlackIcon 
  width={32} 
  height={32} 
  className="hover:opacity-80" 
/>

// Override all colors with a single color
<SlackIcon 
  width="48" 
  height="48" 
  injectedColor="#666666" 
/>

// Use in a navigation or social media section
<div className="flex gap-4">
  <SlackIcon width={20} height={20} className="text-gray-600" />
  <span>Join our Slack</span>
</div>
```

The component maintains the official Slack brand colors by default but allows for monochrome variations when needed. The SVG is optimized for crisp rendering at any size with proper viewBox scaling.
