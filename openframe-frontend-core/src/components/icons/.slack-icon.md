A customizable React SVG component that renders the Slack logo icon with configurable dimensions, styling, and color options.

## Key Components

- **SlackIcon**: Main functional component that renders the Slack logo as an SVG
- **SlackIconProps**: TypeScript interface defining component props including width, height, className, and injectedColor
- **Color variables**: Pre-defined Slack brand colors (red, blue, green, yellow) with override capability

## Usage Example

```typescript
import { SlackIcon } from './slack-icon'

// Basic usage with default size (24x24)
<SlackIcon />

// Custom size and styling
<SlackIcon 
  width={32} 
  height={32} 
  className="hover:opacity-80" 
/>

// Monochrome version using injected color
<SlackIcon 
  width="100%" 
  height="auto"
  injectedColor="#666666" 
/>

// Large icon with custom CSS classes
<SlackIcon 
  width={64}
  height={64}
  className="text-blue-500 drop-shadow-lg"
/>
```

The component maintains the authentic Slack logo design with four colored sections representing the brand's signature aesthetic, while providing flexibility for different use cases and design requirements.
