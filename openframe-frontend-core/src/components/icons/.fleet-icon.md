A React component that renders a customizable fleet/grid icon using SVG circles arranged in a specific pattern.

## Key Components

- **FleetIconProps**: Interface extending SVG props with optional `size`, `color`, and `className` properties
- **FleetIcon**: Forwardable React component that renders an SVG with 6 circles in a fleet/grid layout

## Usage Example

```typescript
import { FleetIcon } from './components/fleet-icon'

// Basic usage with default props
<FleetIcon />

// Customized fleet icon
<FleetIcon 
  size={24} 
  color="#3b82f6" 
  className="hover:opacity-80" 
/>

// With additional SVG props
<FleetIcon 
  size={32}
  color="currentColor"
  onClick={() => console.log('Fleet clicked')}
  aria-label="Fleet view"
/>

// Using ref
const fleetRef = useRef<SVGSVGElement>(null)
<FleetIcon ref={fleetRef} size={20} />
```

The component renders 6 circles in a grid pattern (3x2 layout with one missing position) and accepts all standard SVG properties while providing convenient props for common customizations.
