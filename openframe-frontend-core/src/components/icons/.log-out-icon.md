A customizable React component that renders an SVG log out icon with configurable size, color, and styling properties.

## Key Components

- **LogOutIcon** - Main React component that renders a log out/sign out icon
- **LogOutIconProps** - TypeScript interface defining component props:
  - `color` - Icon stroke color (default: 'white')
  - `className` - CSS classes for styling
  - `size` - Icon dimensions in pixels (default: 20)

## Usage Example

```typescript
import { LogOutIcon } from './log-out-icon'

// Basic usage with defaults
<LogOutIcon />

// Customized icon
<LogOutIcon 
  size={24} 
  color="#333333" 
  className="hover:opacity-75 cursor-pointer" 
/>

// In a logout button
<button onClick={handleLogout}>
  <LogOutIcon size={16} color="currentColor" />
  Sign Out
</button>
```

The component renders a clean, minimalist log out icon featuring a door with an arrow pointing right, making it perfect for authentication interfaces and navigation menus.
