A React component that renders a customizable logout/sign-out SVG icon with configurable styling properties. The icon features a door with an arrow pointing outward, commonly used for logout functionality in user interfaces.

## Key Components

- **LogOutIcon**: Main functional component that renders the SVG logout icon
- **LogOutIconProps**: TypeScript interface defining the component's props:
  - `color?: string` - Icon stroke color (defaults to 'white')
  - `className?: string` - CSS classes for styling (defaults to empty string)
  - `size?: number` - Icon dimensions in pixels (defaults to 20)

## Usage Example

```typescript
import { LogOutIcon } from './log-out-icon'

// Basic usage with defaults
<LogOutIcon />

// Customized icon
<LogOutIcon 
  size={24} 
  color="#ff4444" 
  className="hover:opacity-75 transition-opacity cursor-pointer" 
/>

// In a button
<button onClick={handleLogout} className="flex items-center gap-2">
  <LogOutIcon size={16} color="currentColor" />
  Sign Out
</button>
```

The component uses a 24x24 viewBox and renders as a scalable SVG with stroke-based graphics for clean rendering at any size. All props are optional with sensible defaults for immediate use.
