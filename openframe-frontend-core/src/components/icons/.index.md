A comprehensive icon library that exports SVG icon components organized by functional categories for a web application interface.

## Key Components

**Logo Components**: Brand logos including `OpenmspLogo`, `OpenFrameLogo`, `FlamingoLogo`, `ElestioLogo`, and partner brands

**Navigation Icons**: UI navigation elements like `MenuIcon`, `SearchIcon`, `FilterIcon`, `DashboardIcon`, `SettingsIcon`

**Feature Icons**: Functional icons for specific features such as `CompareIcon`, `VendorsIcon`, `CommunityIcon`, `RemoteControlIcon`

**State Icons**: UI state indicators including `CheckCircleIcon`, `PlusCircleIcon`, `ThumbsUpIcon`, `ErrorIcon`

**Social/Brand Icons**: Third-party service icons like `GitHubIcon`, `SlackIcon`, `GoogleLogo`, `LinkedInIcon`

**System Icons**: Operating system and tool icons including `WindowsIcon`, `MacOSIcon`, `LinuxIcon`, `PowerShellIcon`

**Specialized Icon Sets**: Grouped exports for vendor showcase, AI features, tool directory, and device types

## Usage Example

```typescript
import { 
  SearchIcon, 
  GitHubIcon, 
  CheckCircleIcon,
  OpenmspLogo 
} from './icons';

function Header() {
  return (
    <div>
      <OpenmspLogo className="h-8 w-8" />
      <SearchIcon className="text-gray-500" />
      <GitHubIcon size={24} />
      <CheckCircleIcon className="text-green-500" />
    </div>
  );
}
```
