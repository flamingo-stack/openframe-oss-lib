This file serves as the main export barrel for a deprecated icon library in the OpenFrame frontend system, providing access to over 150+ custom SVG icons organized by functional categories.

## Key Components

**Logo Icons**: Brand logos including OpenmspLogo, OpenFrameLogo, FlamingoLogo, ElestioLogo, and partner brands

**Navigation Icons**: UI navigation elements like MenuIcon, SearchIcon, FilterIcon, HamburgerIcon, and feature-specific icons

**UI State Icons**: Interactive state indicators including CheckCircleIcon, ErrorIcon, ThumbsUpIcon, RefreshIcon, and theme toggles (SunIcon/MoonIcon)

**Social & Brand Icons**: Platform icons for GitHub, Slack, Microsoft, Google, Reddit, LinkedIn, and other social platforms

**System & Tool Icons**: Technical icons for operating systems (WindowsIcon, LinuxIcon, MacOSIcon), shell types (PowerShellIcon, BashIcon), and MSP tools

**OpenFrame AI Icons**: Specialized icons for AI features like NaturalLanguageIcon, AutomatedDiagnosticsIcon, and SmartEscalationIcon

## Usage Example

```typescript
import { 
  OpenFrameLogo, 
  SearchIcon, 
  GitHubIcon,
  CheckCircleIcon 
} from '@flamingo-stack/openframe-frontend-core/components/icons';

// Use in React component
<div>
  <OpenFrameLogo />
  <SearchIcon />
  <GitHubIcon />
  <CheckCircleIcon />
</div>
```

**⚠️ Deprecation Notice**: This module is deprecated in favor of `icons-v2` which offers 1600+ icons with better TypeScript support and smaller bundle sizes. Migrate to the new module for future projects.
