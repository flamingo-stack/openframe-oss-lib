This file serves as the main export barrel for the OpenFrame frontend icon system, providing access to 100+ SVG icon components organized by functional categories.

## Key Components

**Deprecated Module Notice**: This entire icon module is deprecated in favor of `@flamingo-stack/openframe-frontend-core/components/icons-v2` which offers 1600+ icons with improved TypeScript support and smaller bundle size.

**Icon Categories**:
- **Logo icons**: Brand logos for OpenMSP, OpenFrame, Flamingo, partner companies
- **Navigation icons**: UI elements for menus, search, filters, user actions
- **Feature icons**: OpenFrame-specific functionality (AI agents, automation, diagnostics)
- **Platform icons**: Social media, development tools, operating systems
- **UI state icons**: Status indicators, actions, theme toggles
- **Tool directory icons**: MSP tool categories and types

**Special Exports**:
- `icon-utils`: Utility functions for icon manipulation
- `*-icons`: Grouped icon collections for specific features
- OS-specific icons for Windows, macOS, Linux
- Shell type icons for PowerShell, Bash, Python, etc.

## Usage Example

```typescript
// Deprecated usage (still functional)
import { 
  OpenFrameLogo, 
  SearchIcon, 
  MingoIcon,
  WindowsIcon 
} from '@flamingo-stack/openframe-frontend-core/components/icons';

// Recommended migration to icons-v2
import { 
  CopyIcon,
  SearchIcon 
} from '@flamingo-stack/openframe-frontend-core/components/icons-v2';

// Component usage
<SearchIcon className="w-4 h-4" />
<OpenFrameLogo className="h-8" />
```
