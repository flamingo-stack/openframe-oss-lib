A React SVG icon component that renders a stylized license/scales of justice icon with customizable styling properties.

## Key Components

- **`CustomLicenseIcon`** - Main SVG icon component with configurable dimensions and styling
- **`CustomLicenseIconProps`** - TypeScript interface defining component props including optional className, fill color, width, and height

## Usage Example

```typescript
import { CustomLicenseIcon } from './custom-license-icon';

// Basic usage with defaults (24x24px, gray fill)
<CustomLicenseIcon />

// Customized styling
<CustomLicenseIcon 
  className="license-icon"
  fill="#007bff"
  width={32}
  height={32}
/>

// With additional props
<CustomLicenseIcon 
  fill="#000"
  onClick={() => console.log('License clicked')}
  title="License information"
/>
```

The component uses SVG paths to render a detailed license/legal scales icon and accepts standard HTML attributes through prop spreading, making it flexible for various use cases in legal or compliance-related interfaces.
