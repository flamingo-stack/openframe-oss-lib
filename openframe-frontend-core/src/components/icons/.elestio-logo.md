A React SVG component that renders the official Elestio logo with collision-safe unique IDs to prevent conflicts when multiple instances are used on the same page.

## Key Components

- **ElestioLogo**: Main component that renders the Elestio logo as an SVG
- **uniqueId generation**: Uses `React.useId()` to create unique identifiers for SVG elements
- **Collision prevention**: Ensures mask and gradient IDs don't conflict between multiple logo instances
- **Customizable props**: Accepts standard SVG props and optional className for styling

## Usage Example

```typescript
import { ElestioLogo } from './elestio-logo';

// Basic usage
<ElestioLogo />

// With custom styling
<ElestioLogo className="w-8 h-8" />

// With additional SVG props
<ElestioLogo 
  className="logo" 
  width={50} 
  height={48} 
  aria-label="Elestio company logo"
/>

// Multiple instances on same page (safe due to unique IDs)
<div>
  <ElestioLogo className="header-logo" />
  <ElestioLogo className="footer-logo" />
</div>
```

The component features a complex gradient design with multiple colored segments (red, orange, purple, blue, yellow) that form the distinctive Elestio "e" symbol. The unique ID system prevents SVG definition conflicts that commonly occur when the same SVG with internal references is rendered multiple times.
