React component that renders the official Elestio logo as an SVG with dynamic gradient colors and unique IDs to prevent rendering conflicts when multiple instances appear on the same page.

## Key Components

- **ElestioLogo**: Main component that accepts standard SVG props and optional className
- **Unique ID generation**: Uses React's `useId()` hook to create collision-free element IDs
- **SVG elements**: Complex path definitions with masks, gradients, and color stops
- **Responsive design**: Fixed dimensions (25x24) with viewBox for scalability

## Usage Example

```typescript
import { ElestioLogo } from './elestio-logo';

// Basic usage
<ElestioLogo />

// With custom styling
<ElestioLogo 
  className="w-8 h-8" 
  style={{ color: 'red' }}
/>

// With SVG props
<ElestioLogo 
  width={50} 
  height={48} 
  onClick={() => console.log('Logo clicked')}
/>

// Multiple instances (safe from ID collisions)
<div>
  <ElestioLogo className="header-logo" />
  <ElestioLogo className="footer-logo" />
</div>
```

The component automatically handles ID conflicts by generating unique identifiers for each instance, making it safe to use multiple logos on the same page without visual glitches.
