A React functional component that renders a customizable moon-shaped SVG icon, commonly used for dark mode toggles or theme switching interfaces.

## Key Components

- **MoonIcon**: Main React component that renders the moon SVG
- **MoonIconProps**: TypeScript interface defining optional props for customization
  - `color`: SVG fill color (default: 'white')
  - `className`: CSS class for styling
  - `width`: Icon width in pixels (default: 23)
  - `height`: Icon height in pixels (default: 24)

## Usage Example

```typescript
import { MoonIcon } from './moon-icon';

// Basic usage with defaults
<MoonIcon />

// Customized moon icon
<MoonIcon 
  color="#333" 
  width={32} 
  height={32}
  className="dark-mode-toggle"
/>

// In a theme toggle button
<button onClick={toggleDarkMode}>
  <MoonIcon color="currentColor" className="transition-colors" />
</button>
```

The component uses a crescent moon design with a fixed viewBox of "0 0 23 24" and includes proper TypeScript typing for all props. The SVG scales proportionally when width/height are modified while maintaining the original aspect ratio.
