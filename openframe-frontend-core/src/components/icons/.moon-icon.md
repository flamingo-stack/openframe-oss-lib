A React functional component that renders an SVG moon icon with customizable styling properties.

## Key Components

- **MoonIcon** - Main exported React component that renders a crescent moon SVG
- **MoonIconProps** - TypeScript interface defining the component's props:
  - `color` - SVG fill color (default: 'white')
  - `className` - CSS class for styling
  - `width` - Icon width in pixels (default: 23)
  - `height` - Icon height in pixels (default: 24)

## Usage Example

```typescript
import { MoonIcon } from './moon-icon';

// Basic usage with defaults
<MoonIcon />

// Customized moon icon
<MoonIcon 
  color="#1a1a1a" 
  width={32} 
  height={32} 
  className="dark-mode-toggle" 
/>

// Dark theme toggle button
<button onClick={toggleDarkMode}>
  <MoonIcon color="currentColor" className="transition-colors" />
</button>
```

The component uses a fixed viewBox of "0 0 23 24" to maintain consistent proportions regardless of the specified dimensions, making it ideal for theme toggles, navigation icons, or any UI element requiring a moon symbol.
