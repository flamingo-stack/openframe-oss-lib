SVG moon icon React component used for UI theming or dark mode indicators.

## Key Components

### `MoonIcon`
**Status: ⚠️ Deprecated** — Use icons from `icons-v2-generated` instead.

A functional React component that renders a crescent moon SVG icon with configurable appearance.

**Props (`MoonIconProps`):**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | SVG fill color |
| `className` | `string` | `''` | Additional CSS classes |
| `width` | `number` | `23` | Icon width in pixels |
| `height` | `number` | `24` | Icon height in pixels |

## Usage Example

```typescript
import { MoonIcon } from './moon-icon';

// Default usage
<MoonIcon />

// Custom color and size
<MoonIcon color="#FFC008" width={32} height={32} />

// With Tailwind class
<MoonIcon className="opacity-80 hover:opacity-100" color="currentColor" />
```

## Migration Note

This component is deprecated. Replace with the equivalent from the `icons-v2-generated` package:

```typescript
// ❌ Deprecated
import { MoonIcon } from './moon-icon';

// ✅ Preferred
import { MoonIcon } from '@/components/icons-v2-generated';
```
