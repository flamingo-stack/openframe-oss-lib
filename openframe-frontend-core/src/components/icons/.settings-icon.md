A React component that renders a customizable settings gear icon as an SVG element.

## Key Components

- **SettingsIcon**: Main component function that renders an octagonal settings icon
- **SettingsIconProps**: TypeScript interface defining optional props:
  - `className`: CSS class string for styling
  - `size`: Number for icon dimensions (default: 20)
  - `color`: String for fill color (default: 'white')

## Usage Example

```typescript
import { SettingsIcon } from './settings-icon'

// Basic usage with defaults
<SettingsIcon />

// Customized with different size and color
<SettingsIcon 
  size={32} 
  color="#333333" 
  className="settings-btn-icon"
/>

// In a button component
<button className="settings-button">
  <SettingsIcon size={24} color="currentColor" />
  Settings
</button>
```

The component renders a gear icon within an octagonal border, making it ideal for settings buttons, configuration panels, or navigation elements. The SVG is scalable and the color can be dynamically controlled through props.
