A React component that renders a customizable settings icon as an SVG with a hexagonal outer shape and circular center element.

## Key Components

- **SettingsIconProps**: Interface defining optional props for styling customization
- **SettingsIcon**: Main component function that renders the SVG icon
- **SVG Elements**: Two path elements creating the hexagonal border and inner circle

## Usage Example

```typescript
import { SettingsIcon } from './settings-icon'

// Basic usage with defaults (white color, size 20)
<SettingsIcon />

// Customized with different color and CSS class
<SettingsIcon 
  className="hover:opacity-80 cursor-pointer" 
  color="#3b82f6" 
/>

// Larger size for buttons or headers
<SettingsIcon 
  size={32}
  color="currentColor" 
  className="text-gray-600"
/>
```

The component accepts optional `className`, `size`, and `color` props with sensible defaults. The SVG uses a fixed 24x22 viewBox regardless of the size prop, ensuring consistent proportions. The `fill` attribute uses the `color` prop value, defaulting to white, making it suitable for dark backgrounds or can be set to `"currentColor"` to inherit from parent text color.
