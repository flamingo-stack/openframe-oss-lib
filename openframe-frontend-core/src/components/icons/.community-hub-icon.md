A React component that renders a customizable community hub icon as an SVG, featuring interconnected people and connection nodes representing a collaborative community network.

## Key Components

**CommunityHubIcon**: Main functional component that accepts customization props
- `color` (optional): Fill color for the icon paths (default: 'white')
- `className` (optional): CSS classes for styling and positioning
- `width` (optional): Icon width in pixels (default: 19)
- `height` (optional): Icon height in pixels (default: 16)

**SVG Structure**: Two-path design representing community connectivity
- Connection nodes and network links
- Central figure with surrounding community elements
- Scalable vector format with 19x16 viewBox

## Usage Example

```typescript
import { CommunityHubIcon } from './community-hub-icon';

// Basic usage with defaults
<CommunityHubIcon />

// Customized appearance
<CommunityHubIcon 
  color="#3B82F6" 
  width={24} 
  height={20} 
  className="hover:opacity-75 transition-opacity" 
/>

// Navigation or button integration
<button className="flex items-center gap-2">
  <CommunityHubIcon color="currentColor" />
  <span>Community Hub</span>
</button>
```

The icon is optimized for navigation menus, buttons, and UI elements where community or social features need visual representation.
