A customizable SVG icon component representing a community hub or user group, built with React and TypeScript for flexible integration into web applications.

## Key Components

**CommunityHubIcon** - Main React functional component that renders an SVG icon with two distinct visual elements: group connectivity indicators and a central user profile figure.

**CommunityHubIconProps** - TypeScript interface defining optional props:
- `color` - Icon fill color (default: 'white')
- `className` - CSS classes for styling
- `width` - Icon width in pixels (default: 19)
- `height` - Icon height in pixels (default: 16)

## Usage Example

```typescript
import { CommunityHubIcon } from './community-hub-icon';

// Basic usage with defaults
<CommunityHubIcon />

// Customized styling and colors
<CommunityHubIcon 
  color="#3B82F6" 
  width={24} 
  height={20}
  className="hover:opacity-80 transition-opacity"
/>

// In a navigation menu
<button className="flex items-center gap-2">
  <CommunityHubIcon color="currentColor" width={16} height={14} />
  <span>Community</span>
</button>
```

The component maintains its original aspect ratio through the fixed viewBox while allowing flexible sizing through width/height props.
