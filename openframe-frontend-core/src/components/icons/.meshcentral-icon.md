A React component that renders the MeshCentral logo as an SVG icon with customizable styling properties.

## Key Components

- **MeshCentralIcon**: Main React functional component that renders the MeshCentral logo
- **MeshCentralIconProps**: TypeScript interface defining component props
  - `className?`: Optional CSS class name for styling
  - `size?`: Icon dimensions (number or string, defaults to 16)
  - `color?`: Fill color for the SVG paths (defaults to 'white')

## Usage Example

```typescript
import { MeshCentralIcon } from './meshcentral-icon';

// Basic usage with defaults
<MeshCentralIcon />

// Customized icon
<MeshCentralIcon 
  size={24} 
  color="#007acc" 
  className="logo-icon" 
/>

// Responsive icon with CSS units
<MeshCentralIcon 
  size="2rem" 
  color="currentColor" 
  className="navbar-logo" 
/>
```

The component renders a detailed SVG representation of the MeshCentral logo with multiple path elements. It's designed to be scalable and reusable across different parts of an application where the MeshCentral branding is needed.
