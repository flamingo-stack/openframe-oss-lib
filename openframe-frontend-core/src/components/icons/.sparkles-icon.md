A React functional component that renders a customizable sparkles/magic icon using SVG graphics.

## Key Components

- **SparklesIcon**: Main React component that renders an SVG sparkles icon
- **SparklesIconProps**: TypeScript interface defining component props
  - `className`: Optional CSS class for styling
  - `color`: Icon color (defaults to "currentColor")
  - `size`: Icon dimensions in pixels (defaults to 20)

## Usage Example

```typescript
import { SparklesIcon } from './sparkles-icon'

// Basic usage
<SparklesIcon />

// With custom styling
<SparklesIcon 
  size={32}
  color="#FFD700"
  className="hover:scale-110 transition-transform"
/>

// Inheriting parent text color
<div className="text-blue-500">
  <SparklesIcon /> {/* Will be blue */}
</div>
```

The icon features a central four-pointed star with a smaller plus symbol in the top-right corner, making it perfect for indicating magic, enhancement, or special features in your application.
