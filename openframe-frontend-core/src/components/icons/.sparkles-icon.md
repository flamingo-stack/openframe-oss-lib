A reusable React SVG icon component that renders a sparkles/magic icon with customizable appearance properties.

## Key Components

- **`SparklesIcon`** - Main functional component that renders an SVG sparkles icon
- **`SparklesIconProps`** - TypeScript interface defining component props:
  - `className` - Optional CSS class names for styling
  - `color` - Icon color (defaults to `currentColor`)
  - `size` - Icon dimensions in pixels (defaults to 20)

## Usage Example

```typescript
import { SparklesIcon } from './sparkles-icon'

// Basic usage with defaults
<SparklesIcon />

// Customized with props
<SparklesIcon 
  size={32}
  color="#ff6b6b"
  className="animate-pulse"
/>

// Using with Tailwind classes
<SparklesIcon 
  size={24}
  className="text-purple-500 hover:text-purple-700 transition-colors"
/>
```

The component renders an SVG with a main sparkle shape and a small cross accent, making it perfect for indicating magical features, AI assistance, or special functionality in user interfaces.
