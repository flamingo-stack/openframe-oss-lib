A customizable SVG icon component that renders a donut/doughnut shape with configurable styling and color properties.

## Key Components

- **DonutIcon**: React functional component that renders an SVG donut icon
- **Props**:
  - `className` (optional): CSS class name for styling
  - `color` (optional): Fill color for the icon paths (defaults to 'white')

## Usage Example

```typescript
import { DonutIcon } from './donut-icon';

// Basic usage with default white color
<DonutIcon />

// With custom color
<DonutIcon color="#3B82F6" />

// With custom styling
<DonutIcon 
  className="w-8 h-8 hover:opacity-80" 
  color="currentColor" 
/>

// In a button component
<button className="flex items-center gap-2">
  <DonutIcon color="#10B981" />
  <span>Add Donut</span>
</button>
```

The icon maintains a fixed 24x24 viewBox and scales appropriately when CSS dimensions are applied through the className prop.
