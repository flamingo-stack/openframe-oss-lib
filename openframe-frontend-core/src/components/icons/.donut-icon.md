A React component that renders a customizable donut/loading spinner icon as an SVG graphic.

## Key Components

- **DonutIcon**: Default export function component that renders an SVG donut icon
- **Props Interface**: Accepts optional `className` and `color` props for styling customization
- **SVG Path Elements**: Two path elements creating the donut shape with inner and outer rings

## Usage Example

```typescript
import { DonutIcon } from './donut-icon';

// Basic usage with default white color
<DonutIcon />

// With custom color
<DonutIcon color="#3b82f6" />

// With custom styling
<DonutIcon 
  className="w-8 h-8 animate-spin" 
  color="currentColor" 
/>

// As a loading indicator
<div className="flex items-center gap-2">
  <DonutIcon className="animate-spin" color="#10b981" />
  <span>Loading...</span>
</div>
```

The icon renders as a 24x24 SVG with a donut/ring shape, commonly used for loading states, progress indicators, or decorative elements. The component accepts standard HTML props and can be easily styled with CSS classes or inline styles.
