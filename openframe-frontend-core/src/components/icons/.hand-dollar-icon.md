A React functional component that renders a customizable hand holding dollar sign SVG icon with configurable styling properties.

## Key Components

- **HandDollarIcon**: Main React component that renders an SVG icon
- **HandDollarIconProps**: TypeScript interface defining component props
  - `className`: Optional CSS class for styling
  - `width`: Icon width in pixels (default: 24)
  - `height`: Icon height in pixels (default: 24)
  - `color`: Stroke color (default: CSS variable `--ods-flamingo-cyan-base`)

The component features a hand holding/offering money symbol with customizable dimensions and styling, commonly used for financial or payment-related UI elements.

## Usage Example

```typescript
import { HandDollarIcon } from './hand-dollar-icon';

function PaymentSection() {
  return (
    <div>
      {/* Default styling */}
      <HandDollarIcon />
      
      {/* Custom size and color */}
      <HandDollarIcon 
        width={32} 
        height={32} 
        color="#008080" 
      />
      
      {/* With custom CSS class */}
      <HandDollarIcon 
        className="payment-icon" 
        width={20}
        height={20}
      />
    </div>
  );
}
```

The icon uses stroke-based rendering and maintains a 24x24 viewBox for consistent scaling across different sizes.
