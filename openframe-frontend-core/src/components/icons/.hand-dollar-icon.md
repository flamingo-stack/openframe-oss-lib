A React component that renders an SVG icon depicting a hand holding or receiving money, typically used in financial or payment-related interfaces.

## Key Components

- **HandDollarIcon**: Main React functional component that renders the SVG icon
- **HandDollarIconProps**: TypeScript interface defining component props including styling and size options

## Usage Example

```typescript
import { HandDollarIcon } from './hand-dollar-icon';

// Basic usage with default styling
<HandDollarIcon />

// Customized with specific dimensions and color
<HandDollarIcon 
  width={32}
  height={32}
  color="#00A651"
  className="payment-icon"
/>

// Using CSS variables for theming
<HandDollarIcon 
  color="var(--primary-color)"
  width={20}
  height={20}
/>
```

The component accepts optional props for `className`, `width`, `height`, and `color`. Default values are 24px for dimensions and a CSS variable `--ods-flamingo-cyan-base` for color. The SVG uses stroke-based styling and maintains a 24x24 viewBox for consistent scaling.
