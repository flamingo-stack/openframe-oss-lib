A React functional component that renders an SVG bar chart icon, typically used to represent financial data or margin crisis scenarios.

## Key Components

- **`MarginCrisisIcon`**: The main React functional component that renders a customizable SVG bar chart
- **`MarginCrisisIconProps`**: TypeScript interface defining the component's props including styling and color options

## Props

- `className?`: Optional CSS class for styling
- `width?`: Icon width (default: 148)
- `height?`: Icon height (default: 148)
- `strokeColor?`: Color for the bar outlines (default: '#F357BB')

## Usage Example

```typescript
import { MarginCrisisIcon } from './margin-crisis-icon';

// Basic usage
<MarginCrisisIcon />

// Customized with props
<MarginCrisisIcon 
  width={100}
  height={100}
  strokeColor="#FF0000"
  className="my-chart-icon"
/>

// Responsive sizing
<MarginCrisisIcon 
  width="100%"
  height="auto"
  strokeColor="currentColor"
/>
```

The icon features three bars of different heights representing data visualization, making it suitable for financial dashboards, analytics interfaces, or margin-related UI components.
