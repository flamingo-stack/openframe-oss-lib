A React component that renders a customizable bar chart icon with three bars of different heights, typically used to represent financial data or margin crisis scenarios.

## Key Components

- **MarginCrisisIconProps** - Interface defining optional customization props including className, dimensions, and stroke color
- **MarginCrisisIcon** - Main functional component that renders an SVG bar chart with three ascending bars

## Usage Example

```typescript
import { MarginCrisisIcon } from './margin-crisis-icon';

// Default usage
<MarginCrisisIcon />

// Customized with different size and color
<MarginCrisisIcon 
  width={100}
  height={100}
  strokeColor="#FF6B6B"
  className="margin-icon"
/>

// Responsive sizing
<MarginCrisisIcon 
  width="100%"
  height="auto"
  strokeColor="currentColor"
/>
```

The component features three vertical bars of increasing height (left to right), created using SVG path elements with rounded corners. Default dimensions are 148x148px with a magenta stroke color (#F357BB). All props are optional, allowing for flexible styling and integration into different UI contexts.
