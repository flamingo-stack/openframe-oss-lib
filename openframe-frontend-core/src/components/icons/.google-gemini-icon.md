A React component that renders the Google Gemini AI icon as an SVG with customizable styling and color properties.

## Key Components

- **GoogleGeminiIconProps** - Interface defining component props with optional className, size, and color
- **GoogleGeminiIcon** - Main functional component that renders a scalable SVG icon with filters and gradients

The component features a distinctive sparkle/star shape with gradient fills and multiple blur filters to create depth and visual appeal.

## Usage Example

```typescript
import { GoogleGeminiIcon } from './google-gemini-icon'

// Basic usage with defaults (24px, white color)
<GoogleGeminiIcon />

// Custom size and color
<GoogleGeminiIcon 
  size={32} 
  color="#4893FC" 
  className="ml-2" 
/>

// Large icon with custom styling
<GoogleGeminiIcon 
  size={48} 
  color="currentColor" 
  className="text-blue-500 hover:text-blue-600" 
/>
```

The icon maintains its aspect ratio and quality at any size, making it suitable for buttons, navigation elements, or branding components in applications using Google's Gemini AI services.
