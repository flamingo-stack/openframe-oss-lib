A React SVG icon component that renders the Google Gemini logo with customizable styling properties. The component features a star-shaped gradient design with multiple filtered elements creating a glowing effect.

## Key Components

- **GoogleGeminiIconProps**: Interface defining optional props for className, size, and color customization
- **GoogleGeminiIcon**: Main functional component that renders the SVG icon with default values

## Usage Example

```typescript
import { GoogleGeminiIcon } from './google-gemini-icon'

// Basic usage with defaults (24px, white color)
<GoogleGeminiIcon />

// Custom size and color
<GoogleGeminiIcon 
  size={32} 
  color="#4893FC" 
  className="my-icon-class" 
/>

// Large icon with custom styling
<GoogleGeminiIcon 
  size={48} 
  color="currentColor" 
  className="hover:opacity-80 transition-opacity" 
/>
```

The icon maintains its aspect ratio and uses SVG filters for visual effects. The `color` prop affects the filtered elements while the main star shape uses a built-in gradient from blue to purple.
