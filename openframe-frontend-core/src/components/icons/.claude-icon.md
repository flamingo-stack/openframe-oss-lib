A React component that renders the Claude AI assistant logo as an SVG icon with customizable properties.

## Key Components

- **ClaudeIcon**: Main component function that renders the Claude logo SVG
- **ClaudeIconProps**: TypeScript interface extending React's SVGProps for type safety
- SVG with branded orange fill color (#D97757) and accessibility features

## Usage Example

```typescript
import { ClaudeIcon } from './claude-icon';

// Basic usage
<ClaudeIcon />

// With custom size and styling
<ClaudeIcon 
  width="2em" 
  height="2em" 
  className="text-blue-500" 
/>

// With event handlers
<ClaudeIcon 
  onClick={() => console.log('Claude icon clicked')}
  style={{ cursor: 'pointer' }}
/>

// Custom color override
<ClaudeIcon fill="#333333" />
```

The component accepts all standard SVG properties through props spreading, making it highly customizable while maintaining the original Claude branding. The SVG includes proper accessibility labeling and responsive sizing based on the current font size (1em by default).
