A React SVG icon component that renders a user profile icon with an edit/pencil overlay, commonly used in user interfaces for user account management.

## Key Components

- **UserIcon**: Main component that renders an SVG user icon with customizable styling
- **UserIconProps**: TypeScript interface defining component props including dimensions, color, and CSS classes

## Usage Example

```typescript
import { UserIcon } from './user-icon'

function UserProfile() {
  return (
    <div>
      {/* Default usage */}
      <UserIcon />
      
      {/* Custom size and color */}
      <UserIcon 
        width={32} 
        height={32} 
        color="#3b82f6"
        className="hover:scale-110 transition-transform"
      />
      
      {/* Responsive sizing */}
      <UserIcon 
        width="100%" 
        height="auto"
        className="max-w-6 text-gray-600"
      />
    </div>
  )
}
```

The component defaults to 24x23 dimensions with white fill color and accepts standard HTML attributes for flexible styling. The SVG includes two main paths: a user silhouette and an edit pencil icon overlay, making it ideal for user account edit buttons or profile management interfaces.
