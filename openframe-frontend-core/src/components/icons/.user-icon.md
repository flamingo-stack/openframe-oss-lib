A React component that renders a user icon SVG with an edit/pencil overlay, commonly used for user profile editing interfaces.

## Key Components

- **`UserIcon`**: Main functional component that renders the SVG icon
- **`UserIconProps`**: TypeScript interface defining component props:
  - `color?: string` - Fill color for the icon (default: 'white')
  - `width?: number | string` - Icon width (default: 24)
  - `height?: number | string` - Icon height (default: 23)
  - `className?: string` - CSS classes (default: 'text-current')

## Usage Example

```typescript
import { UserIcon } from './user-icon'

// Basic usage with defaults
<UserIcon />

// Customized styling
<UserIcon 
  width={32}
  height={32}
  color="#3b82f6"
  className="hover:opacity-80"
/>

// Responsive sizing
<UserIcon 
  width="100%"
  height="auto"
  color="currentColor"
  className="w-6 h-6 text-gray-700"
/>
```

The icon combines a user profile silhouette with an edit pencil, making it ideal for profile editing buttons or user management interfaces.
