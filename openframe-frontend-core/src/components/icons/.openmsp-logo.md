A customizable React SVG component that renders the OpenMSP logo with configurable colors for different design elements including bubbles and shapes.

## Key Components

- **OpenmspLogoProps**: Interface extending `React.SVGProps<SVGSVGElement>` with custom color properties
- **OpenmspLogo**: Main component function that renders the SVG logo with customizable styling
- **Color Props**: `frontBubbleColor`, `innerFrontBubbleColor`, `backBubbleColor` for theme customization
- **Standard SVG Props**: Supports all standard SVG attributes through prop spreading

## Usage Example

```typescript
import { OpenmspLogo } from './openmsp-logo';

// Basic usage with defaults
<OpenmspLogo className="w-24 h-24" />

// Customized colors
<OpenmspLogo 
  frontBubbleColor="#007ACC"
  innerFrontBubbleColor="#E3F2FD"
  backBubbleColor="#F5F5F5"
  className="logo-header"
/>

// With additional SVG props
<OpenmspLogo 
  width={120}
  height={120}
  frontBubbleColor="#FF6B35"
  onClick={() => console.log('Logo clicked')}
/>
```

The component renders a complex layered logo design with overlapping rectangular and circular elements, perfect for branding applications where consistent visual identity with theme flexibility is needed.
