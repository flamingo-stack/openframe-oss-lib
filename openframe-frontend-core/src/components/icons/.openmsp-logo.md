A React component that renders the Openmsp logo as a customizable SVG with configurable colors for different visual elements.

## Key Components

- **OpenmspLogo**: Main React functional component that renders the logo SVG
- **OpenmspLogoProps**: TypeScript interface extending SVGProps with custom color properties
- **Color customization props**:
  - `frontBubbleColor`: Color for the front bubble elements (default: black)
  - `innerFrontBubbleColor`: Color for the inner front bubble (default: white)  
  - `backBubbleColor`: Color for the back bubble (default: white)
  - `className`: CSS class for styling

## Usage Example

```typescript
import { OpenmspLogo } from './openmsp-logo';

// Basic usage with default colors
<OpenmspLogo />

// Customized with brand colors
<OpenmspLogo 
  className="w-24 h-24"
  frontBubbleColor="#3B82F6"
  innerFrontBubbleColor="#EFF6FF"
  backBubbleColor="#1E40AF"
/>

// With additional SVG props
<OpenmspLogo 
  width={48}
  height={48}
  frontBubbleColor="#059669"
  onClick={() => console.log('Logo clicked')}
/>
```

The component maintains a fixed viewBox of "0 0 97 96" while allowing responsive sizing through className or direct SVG properties.
