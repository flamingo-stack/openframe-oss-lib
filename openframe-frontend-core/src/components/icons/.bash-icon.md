A React SVG icon component that renders a Bash/shell terminal symbol with customizable styling properties.

## Key Components

- **BashIconProps** - TypeScript interface defining the component's props:
  - `size?: number` - Icon dimensions (default: 16px)
  - `className?: string` - CSS classes for styling
  - `color?: string` - Fill color (default: 'white')

- **BashIcon** - Main React functional component that renders an SVG icon representing a Bash terminal or hashtag symbol

## Usage Example

```typescript
import { BashIcon } from './bash-icon'

// Basic usage with defaults
<BashIcon />

// Customized icon
<BashIcon 
  size={24} 
  color="#00ff00" 
  className="terminal-icon" 
/>

// Large icon for headers
<BashIcon 
  size={32} 
  color="currentColor" 
  className="text-blue-500" 
/>
```

The component uses a 24x24 viewBox and scales proportionally based on the `size` prop. The SVG path creates a hashtag-like symbol commonly associated with shell/terminal interfaces. The `displayName` is set for better debugging in React DevTools.
