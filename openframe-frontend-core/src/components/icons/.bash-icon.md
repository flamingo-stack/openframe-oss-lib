A React component that renders a customizable Bash/hashtag icon as an SVG element.

## Key Components

- **BashIconProps**: Interface defining component props (`size`, `className`, `color`)
- **BashIcon**: React functional component that renders a hashtag symbol SVG
- **displayName**: Component display name set for debugging purposes

## Usage Example

```typescript
import { BashIcon } from './bash-icon'

// Basic usage with default props (16px, white color)
<BashIcon />

// Customized icon
<BashIcon 
  size={24} 
  color="#007acc" 
  className="my-icon-class" 
/>

// Large blue icon for headers
<BashIcon size={32} color="blue" />
```

The component accepts optional props for size (defaults to 16px), color (defaults to white), and CSS className for styling. The SVG uses a 24x24 viewBox and scales based on the size prop while maintaining aspect ratio.
