React component that renders an SVG icon depicting a shield with a key lock symbol, commonly used for security and authentication features.

## Key Components

- **ShieldKeyIcon**: Main functional component that renders a customizable shield-lock SVG icon
- **ShieldKeyIconProps**: TypeScript interface defining component props for styling and sizing

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes for styling |
| `size` | `number` | `24` | Width and height of the icon in pixels |
| `color` | `string` | `'white'` | Fill color of the SVG paths |

## Usage Example

```typescript
import { ShieldKeyIcon } from './shield-key-icon'

// Basic usage with defaults
<ShieldKeyIcon />

// Customized icon
<ShieldKeyIcon 
  size={32} 
  color="#3b82f6" 
  className="hover:opacity-80" 
/>

// Large security badge
<ShieldKeyIcon 
  size={48} 
  color="currentColor" 
  className="text-green-500" 
/>
```

The component features a detailed shield outline with an embedded lock mechanism, making it ideal for security dashboards, authentication forms, or privacy-related UI elements. The SVG uses a 24x24 viewBox and scales proportionally based on the size prop.
