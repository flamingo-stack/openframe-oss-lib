A React component that renders an SVG icon depicting a shield with a key/lock symbol, commonly used for security-related UI elements.

## Key Components

- **`ShieldKeyIcon`** - Main React functional component that renders the shield-key SVG
- **`ShieldKeyIconProps`** - TypeScript interface defining component props:
  - `className?` - Optional CSS class string
  - `size?` - Icon dimensions (default: 24px)
  - `color?` - Fill color (default: 'white')

## Usage Example

```typescript
import { ShieldKeyIcon } from './shield-key-icon'

// Basic usage with defaults
<ShieldKeyIcon />

// Custom styling
<ShieldKeyIcon 
  size={32} 
  color="#3b82f6" 
  className="security-icon" 
/>

// In a security context
<button className="security-btn">
  <ShieldKeyIcon size={20} color="currentColor" />
  Enable 2FA
</button>
```

The component uses a 24x24 viewBox and scales proportionally. The SVG contains two path elements representing the shield outline and inner key/lock mechanism, making it ideal for authentication, security settings, and protection-related interfaces.
