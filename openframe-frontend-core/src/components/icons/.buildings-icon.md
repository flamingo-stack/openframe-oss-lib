A customizable React SVG icon component that renders a buildings/cityscape icon with configurable styling properties.

## Key Components

**BuildingsIcon** - Main React functional component that renders an SVG icon
- **Props Interface**: `BuildingsIconProps` with optional className, size, and color properties
- **Default Values**: 24px size, white color, empty className
- **SVG Path**: Complex path data creating a buildings/cityscape silhouette

## Usage Example

```typescript
import { BuildingsIcon } from './buildings-icon'

// Basic usage with defaults
<BuildingsIcon />

// Customized styling
<BuildingsIcon 
  size={32} 
  color="#3B82F6" 
  className="hover:opacity-75 transition-opacity" 
/>

// Large blue buildings icon
<BuildingsIcon size={48} color="blue" />
```

The component uses a 64x64 viewBox and scales proportionally based on the size prop, making it suitable for various UI contexts like navigation menus, dashboards, or property-related interfaces.
