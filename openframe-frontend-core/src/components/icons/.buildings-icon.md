A React component that renders a customizable buildings/cityscape SVG icon with configurable size, color, and styling options.

## Key Components

- **BuildingsIcon**: Main functional component that renders an SVG icon
- **BuildingsIconProps**: TypeScript interface defining component props
  - `className?`: Optional CSS class string for styling
  - `size?`: Optional number for width/height (default: 24px)
  - `color?`: Optional string for fill color (default: 'white')

## Usage Example

```typescript
import { BuildingsIcon } from './buildings-icon'

// Basic usage with defaults
<BuildingsIcon />

// Customized with different size and color
<BuildingsIcon 
  size={32} 
  color="#007bff" 
  className="my-buildings-icon" 
/>

// Large icon with custom styling
<BuildingsIcon 
  size={48} 
  color="currentColor" 
  className="hover:opacity-80 transition-opacity" 
/>
```

The icon uses a 64x64 viewBox and features a detailed path representing multiple buildings in a cityscape silhouette. The SVG is optimized for scalability and maintains crisp edges at any size.
