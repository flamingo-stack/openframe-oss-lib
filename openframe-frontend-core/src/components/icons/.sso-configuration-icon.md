A React SVG icon component that renders a Single Sign-On (SSO) configuration symbol, featuring a shield with dots and a lock element to represent secure authentication settings.

## Key Components

- **SSOConfigurationIcon**: Main functional component that renders an SVG icon
- **SSOConfigurationIconProps**: TypeScript interface defining optional props for customization
  - `className`: CSS class for styling
  - `size`: Icon dimensions (default: 24px)
  - `color`: Fill color (default: 'white')

## Usage Example

```typescript
import { SSOConfigurationIcon } from './sso-configuration-icon'

// Basic usage with defaults
<SSOConfigurationIcon />

// Customized icon
<SSOConfigurationIcon 
  size={32}
  color="#3b82f6"
  className="hover:opacity-80 transition-opacity"
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <SSOConfigurationIcon size={16} color="currentColor" />
  SSO Settings
</button>
```

The component is fully customizable and follows standard SVG prop conventions, making it suitable for use in authentication interfaces, admin panels, and configuration screens.
