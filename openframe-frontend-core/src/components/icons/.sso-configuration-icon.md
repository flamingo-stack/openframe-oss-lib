A React component that renders an SVG icon representing SSO (Single Sign-On) configuration, featuring a security lock symbol combined with user authentication indicators.

## Key Components

- **SSOConfigurationIcon**: Main functional component that renders a customizable SSO configuration SVG icon
- **SSOConfigurationIconProps**: TypeScript interface defining the component's optional props (className, size, color)

## Usage Example

```typescript
import { SSOConfigurationIcon } from './sso-configuration-icon'

// Basic usage with default props
<SSOConfigurationIcon />

// Customized with specific styling
<SSOConfigurationIcon 
  size={32}
  color="#007bff"
  className="sso-icon"
/>

// In a button or menu item
<button className="config-button">
  <SSOConfigurationIcon size={20} color="currentColor" />
  Configure SSO
</button>
```

The component provides flexible styling options with sensible defaults (24px size, white color) and accepts standard HTML attributes through the className prop. The SVG uses a viewBox for scalability and includes detailed path elements that form a lock icon combined with user authentication symbols, making it ideal for SSO configuration interfaces.
