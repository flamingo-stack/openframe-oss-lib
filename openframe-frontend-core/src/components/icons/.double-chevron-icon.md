A React SVG icon component that renders double chevron arrows with configurable direction, size, and color properties.

## Key Components

- **DoubleChevronIcon**: Main functional component that renders a double chevron SVG icon
- **DoubleChevronIconProps**: TypeScript interface defining component props
- **DoubleChevronDirection**: Type union for directional variants ('left' | 'right' | 'up' | 'down')
- **rotationMap**: Record mapping directions to Tailwind CSS rotation classes

## Key Features

- **Directional Control**: Supports left, right, up, and down orientations via CSS transforms
- **Customizable Styling**: Configurable size, color, and additional CSS classes
- **Smooth Transitions**: Built-in 200ms transition animations for direction changes
- **Accessibility**: Semantic SVG structure with proper viewBox and fill properties

## Usage Example

```typescript
import { DoubleChevronIcon } from './components/icons/double-chevron-icon'

function NavigationButtons() {
  return (
    <div className="flex gap-2">
      {/* Default left-pointing chevron */}
      <DoubleChevronIcon />
      
      {/* Custom sized right-pointing chevron */}
      <DoubleChevronIcon 
        direction="right" 
        size={32} 
        color="#3b82f6" 
      />
      
      {/* Up arrow with custom styling */}
      <DoubleChevronIcon 
        direction="up"
        className="hover:scale-110"
        color="#ef4444"
      />
    </div>
  )
}
```
