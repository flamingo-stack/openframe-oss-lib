A React SVG icon component that renders a double chevron (two arrow-like shapes) pointing in configurable directions with rotation-based orientation.

## Key Components

- **DoubleChevronIcon**: Main component that renders the SVG icon
- **DoubleChevronIconProps**: TypeScript interface defining component props
- **DoubleChevronDirection**: Type union for arrow directions ('left' | 'right' | 'up' | 'down')
- **rotationMap**: Configuration object mapping directions to Tailwind CSS rotation classes

## Usage Example

```typescript
import { DoubleChevronIcon } from './double-chevron-icon'

// Basic usage with default left direction
<DoubleChevronIcon />

// Customized icon with different direction and styling
<DoubleChevronIcon
  direction="right"
  size={32}
  color="#3b82f6"
  className="hover:opacity-80"
/>

// Use in navigation components
<button className="flex items-center gap-2">
  <DoubleChevronIcon direction="up" size={16} />
  Collapse
</button>
```

The component uses CSS transforms for directional changes and includes smooth transition animations. The SVG maintains its aspect ratio while allowing custom sizing through the `size` prop.
