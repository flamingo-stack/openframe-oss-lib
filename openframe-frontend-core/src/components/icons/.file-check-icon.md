A React SVG icon component that renders a file document with a checkmark, indicating a verified or completed file status.

## Key Components

- **FileCheckIconProps** - TypeScript interface extending React's SVGProps for type safety
- **FileCheckIcon** - Main functional component that renders the SVG icon with customizable styling

## Usage Example

```typescript
import { FileCheckIcon } from './file-check-icon'

// Basic usage with defaults (24x24px, white color)
<FileCheckIcon />

// Custom styling
<FileCheckIcon 
  className="h-8 w-8" 
  color="#10b981" 
/>

// With additional SVG properties
<FileCheckIcon 
  className="h-10 w-10"
  color="currentColor"
  onClick={handleClick}
  aria-label="Verified file"
/>
```

The component accepts standard SVG props through prop spreading, making it highly flexible for different use cases. Default dimensions are 24x24 pixels with white stroke color, but both can be easily overridden through the `className` and `color` props respectively.
