A React SVG icon component that renders a file document with a checkmark, typically used to indicate completed, verified, or approved file states.

## Key Components

- **`FileCheckIconProps`** - Interface extending `React.SVGProps<SVGSVGElement>` for type-safe prop passing
- **`FileCheckIcon`** - Main component function that renders the SVG icon with customizable styling

## Usage Example

```typescript
import { FileCheckIcon } from './file-check-icon'

// Basic usage with default styling
<FileCheckIcon />

// Custom size and color
<FileCheckIcon 
  className="h-8 w-8" 
  color="green" 
/>

// With additional SVG props
<FileCheckIcon 
  className="h-4 w-4"
  color="#10B981"
  onClick={() => console.log('File verified!')}
  style={{ cursor: 'pointer' }}
/>

// In a status indicator
<div className="flex items-center gap-2">
  <FileCheckIcon className="h-5 w-5" color="emerald" />
  <span>Document verified</span>
</div>
```

The component provides sensible defaults (24x24px, white color) while allowing full customization through className, color props, and standard SVG attributes. The icon features a document outline with a folded corner and an internal checkmark symbol.
