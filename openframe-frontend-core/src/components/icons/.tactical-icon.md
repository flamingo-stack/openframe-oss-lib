A React functional component that renders a tactical/military-style icon as an SVG element with customizable size, styling, and color properties.

## Key Components

- **TacticalIcon**: React functional component that renders an SVG tactical symbol
- **Props**:
  - `size`: Optional number for width/height (default: 16px)
  - `className`: Optional CSS class string for styling
  - `color`: Optional fill color string (default: 'white')

## Usage Example

```typescript
import { TacticalIcon } from './tactical-icon';

// Basic usage with defaults
<TacticalIcon />

// Customized with larger size and different color
<TacticalIcon size={32} color="#007acc" />

// With custom styling
<TacticalIcon 
  size={24} 
  className="tactical-symbol rotate-45" 
  color="red" 
/>

// In a button component
<button className="icon-button">
  <TacticalIcon size={20} color="currentColor" />
  Deploy
</button>
```

The component provides a scalable vector icon suitable for tactical interfaces, military applications, or strategic planning tools. The SVG maintains crisp rendering at any size and supports standard CSS styling through the className prop.
