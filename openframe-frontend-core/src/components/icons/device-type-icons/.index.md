A module that provides device-type-specific icons and utilities for rendering appropriate device icons based on device types.

## Key Components

- **DesktopIcon** - Icon component for desktop devices
- **LaptopIcon** - Icon component for laptop/mobile devices  
- **ServerIcon** - Icon component for server devices
- **getDeviceTypeIcon** - Utility function that returns the appropriate icon component based on device type
- **DeviceType** - Type definition for supported device types
- **DeviceTypeIconProps** - Props interface for device icon components

## Usage Example

```typescript
import { 
  DesktopIcon, 
  LaptopIcon, 
  ServerIcon, 
  getDeviceTypeIcon,
  type DeviceType 
} from './index';

// Direct icon usage
const MyComponent = () => (
  <div>
    <DesktopIcon />
    <LaptopIcon />
    <ServerIcon />
  </div>
);

// Dynamic icon selection
const DeviceDisplay = ({ deviceType }: { deviceType: DeviceType }) => {
  const IconComponent = getDeviceTypeIcon(deviceType);
  return <IconComponent />;
};

// Usage with different device types
<DeviceDisplay deviceType="desktop" />
<DeviceDisplay deviceType="laptop" />
<DeviceDisplay deviceType="server" />
```
