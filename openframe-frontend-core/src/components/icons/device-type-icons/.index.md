A module that provides device-type icons and utilities for displaying appropriate icons based on device categories.

## Key Components

- **Icon Components**: `DesktopIcon`, `LaptopIcon`, `ServerIcon` - Individual React components for each device type
- **Icon Utility**: `getDeviceTypeIcon` - Function that returns the appropriate icon component based on device type
- **Types**: `DeviceType`, `DeviceTypeIconProps` - TypeScript types for device categories and icon component props

## Usage Example

```typescript
import { 
  DesktopIcon, 
  LaptopIcon, 
  getDeviceTypeIcon,
  type DeviceType 
} from './index';

// Use individual icons directly
<DesktopIcon className="w-6 h-6" />
<LaptopIcon className="w-6 h-6" />

// Use utility function to get icon based on device type
const deviceType: DeviceType = 'laptop';
const IconComponent = getDeviceTypeIcon(deviceType);
<IconComponent className="w-6 h-6" />

// Dynamic icon selection
const devices = ['desktop', 'laptop', 'server'];
devices.map(type => {
  const Icon = getDeviceTypeIcon(type as DeviceType);
  return <Icon key={type} className="device-icon" />;
});
```

This module centralizes device icon management, making it easy to maintain consistent device representations across an application while providing both direct icon access and dynamic selection capabilities.
