A utility function that returns the appropriate React icon component based on device type, with support for customizable styling properties.

## Key Components

- **`DeviceType`** - Type union defining supported device types: 'desktop', 'laptop', 'mobile', 'tablet', 'server'
- **`DeviceTypeIconProps`** - Interface for icon styling options (className, color, size)
- **`getDeviceTypeIcon()`** - Main function that maps device types to their corresponding icon components

## Usage Example

```typescript
import { getDeviceTypeIcon } from './get-device-type-icon'

// Basic usage
const desktopIcon = getDeviceTypeIcon('desktop')

// With custom styling
const styledIcon = getDeviceTypeIcon('laptop', {
  className: 'my-icon',
  color: '#007bff',
  size: 24
})

// Fallback for unknown types
const fallbackIcon = getDeviceTypeIcon('unknown') // Returns DevicesIcon

// In a React component
function DeviceList({ devices }) {
  return (
    <div>
      {devices.map(device => (
        <div key={device.id}>
          {getDeviceTypeIcon(device.type, { size: 20 })}
          {device.name}
        </div>
      ))}
    </div>
  )
}
```

The function uses a switch statement to return specific icons for desktop, laptop, and server types, while mobile, tablet, and any unrecognized types default to a generic `DevicesIcon`.
