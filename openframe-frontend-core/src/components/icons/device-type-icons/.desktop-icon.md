A React SVG component that renders a desktop monitor icon with customizable styling and color properties.

## Key Components

- **`DesktopIconProps`**: Interface extending React's SVG props with an optional `color` property
- **`DesktopIcon`**: Functional component that renders a 24x24 desktop monitor SVG icon with a monitor screen and stand

## Usage Example

```typescript
import { DesktopIcon } from './desktop-icon'

// Basic usage with default white color
<DesktopIcon />

// Custom color and styling
<DesktopIcon 
  color="#3b82f6" 
  className="w-8 h-8" 
  onClick={() => console.log('Desktop clicked')}
/>

// With additional SVG properties
<DesktopIcon 
  color="currentColor"
  width={32}
  height={32}
  aria-label="Desktop computer"
/>
```

The component accepts all standard SVG element properties and defaults to white color. The icon consists of two main paths: a monitor stand/base and the main screen bezel, making it suitable for desktop applications, system interfaces, or navigation elements.
