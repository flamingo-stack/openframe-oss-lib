A React component that renders an SVG desktop monitor icon with customizable color and styling properties.

## Key Components

- **`DesktopIconProps`** - TypeScript interface extending standard SVG props with an optional `color` parameter
- **`DesktopIcon`** - Functional React component that renders a 24x24px desktop monitor SVG icon

## Usage Example

```typescript
import { DesktopIcon } from './desktop-icon'

// Basic usage with default white color
<DesktopIcon />

// Custom color and className
<DesktopIcon 
  color="#3b82f6" 
  className="w-6 h-6" 
/>

// With additional SVG props
<DesktopIcon 
  color="currentColor"
  width={32}
  height={32}
  onClick={() => console.log('Desktop clicked')}
/>
```

The icon depicts a desktop monitor with a stand and uses two SVG paths - one for the monitor stand/base and another for the screen bezel. The `color` prop defaults to white and controls the fill color of both paths, while all standard SVG properties are passed through via the spread operator.
