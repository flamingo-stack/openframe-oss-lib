A React component that renders an SVG laptop icon with customizable styling and color properties.

## Key Components

- **`LaptopIconProps`** - Interface extending React SVG props with an optional `color` parameter
- **`LaptopIcon`** - Functional component that renders a 24x24 pixel laptop SVG icon

## Usage Example

```typescript
import { LaptopIcon } from './laptop-icon'

// Basic usage with default white color
<LaptopIcon />

// Custom color and className
<LaptopIcon 
  color="#3B82F6" 
  className="w-8 h-8" 
/>

// With additional SVG props
<LaptopIcon 
  color="currentColor"
  width={32}
  height={32}
  onClick={handleClick}
/>
```

The component accepts all standard SVG element properties through props spreading, making it highly flexible for different use cases. The default color is white, but can be overridden via the `color` prop. The icon depicts a laptop computer with a distinctive screen and base design, suitable for UI elements indicating computer-related functionality.
