A customizable SVG laptop icon component that renders a laptop computer graphic with configurable styling and color properties.

## Key Components

- **LaptopIconProps**: Interface extending React SVGProps with optional color property
- **LaptopIcon**: React functional component that renders an SVG laptop icon
- **Default props**: Sets white as the default color value

## Usage Example

```typescript
import { LaptopIcon } from './laptop-icon'

// Basic usage with default white color
<LaptopIcon />

// Custom color
<LaptopIcon color="#3b82f6" />

// With additional styling
<LaptopIcon 
  className="w-8 h-8" 
  color="currentColor"
  onClick={handleClick}
/>

// Custom size and styling
<LaptopIcon 
  width={32}
  height={32}
  color="#ef4444"
  className="hover:opacity-80"
/>
```

The component accepts all standard SVG properties through props spreading, making it flexible for various use cases like buttons, navigation, or decorative elements.
