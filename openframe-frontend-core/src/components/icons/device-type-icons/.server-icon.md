A React SVG icon component that renders a server/database icon with two stacked rectangular units, commonly used to represent server infrastructure or database systems.

## Key Components

- **ServerIconProps**: Interface extending React's SVGProps with an optional `color` prop
- **ServerIcon**: Functional component that renders a server icon SVG with customizable styling

## Usage Example

```typescript
import { ServerIcon } from './server-icon'

// Basic usage with default white color
<ServerIcon />

// Custom color and className
<ServerIcon color="#3b82f6" className="w-6 h-6" />

// With additional SVG props
<ServerIcon 
  color="currentColor" 
  width={32} 
  height={32}
  className="server-icon"
  onClick={() => console.log('Server clicked')}
/>
```

The icon features a 24x24 viewBox displaying two server units with connection ports/indicators on each unit. The `color` prop defaults to 'white' and controls the fill color of all path elements. All standard SVG props are supported through prop spreading, making it highly customizable for different use cases.
