A customizable React SVG icon component that renders a server/database icon with two stacked rectangular server units and indicator lights.

## Key Components

- **`ServerIconProps`** - TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional `color` prop
- **`ServerIcon`** - Main functional component that renders the server SVG icon
- **Props**: Accepts all standard SVG props plus custom `color` (defaults to 'white')

## Usage Example

```typescript
import { ServerIcon } from './server-icon';

// Basic usage with default white color
<ServerIcon />

// Custom color and size
<ServerIcon 
  color="#3B82F6" 
  width={32} 
  height={32}
  className="my-server-icon"
/>

// With click handler
<ServerIcon 
  color="#10B981"
  onClick={() => console.log('Server clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The icon displays two horizontally stacked server units with connection ports and status indicators, making it ideal for representing servers, databases, or network infrastructure in user interfaces.
