A React component that renders the Figma logo as an SVG icon with customizable styling and color properties.

## Key Components

- **FigmaIconProps** - Interface defining optional `className` and `color` props
- **FigmaIcon** - Main component export that renders an SVG representation of the Figma logo

## Usage Example

```typescript
import { FigmaIcon } from './figma-icon';

// Basic usage with defaults
<FigmaIcon />

// Custom styling and color
<FigmaIcon 
  className="w-8 h-8" 
  color="#F24E1E" 
/>

// In a button or link
<button className="flex items-center gap-2">
  <FigmaIcon className="w-5 h-5" color="#ffffff" />
  Open in Figma
</button>
```

The component uses Tailwind CSS classes by default (`w-6 h-6`) and renders in black (`#000000`) unless overridden. The SVG maintains proper aspect ratio and scales based on the provided className dimensions.
