A React component that renders a customizable command/terminal icon using SVG, featuring a rounded rectangle with a command prompt symbol inside.

## Key Components

- **CmdIconProps**: Interface defining optional props for size, className, and color customization
- **CmdIcon**: Main React functional component that renders the SVG icon
- **displayName**: Set to 'CmdIcon' for debugging purposes

## Usage Example

```typescript
import { CmdIcon } from './cmd-icon'

// Basic usage with defaults (16px, white color)
<CmdIcon />

// Customized size and color
<CmdIcon size={24} color="#007acc" />

// With custom styling
<CmdIcon 
  size={32} 
  color="#ff6b6b" 
  className="hover:opacity-80 transition-opacity" 
/>

// In a button component
<button className="flex items-center gap-2">
  <CmdIcon size={20} color="currentColor" />
  Open Terminal
</button>
```

The component defaults to 16px size with white color, making it suitable for toolbars, buttons, or navigation elements where a terminal/command interface indicator is needed.
