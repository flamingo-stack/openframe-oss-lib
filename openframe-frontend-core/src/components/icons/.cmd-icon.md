A React component that renders a command line/terminal icon using SVG, featuring a bordered rectangle with a greater-than symbol representing a command prompt.

## Key Components

- **CmdIcon**: Main React functional component that renders the terminal icon
- **CmdIconProps**: TypeScript interface defining the component's props:
  - `size`: Controls icon dimensions (default: 16px)
  - `className`: CSS class for styling
  - `color`: Icon color (default: 'white')

## Usage Example

```typescript
import { CmdIcon } from './cmd-icon'

// Basic usage with defaults
<CmdIcon />

// Customized icon
<CmdIcon 
  size={24} 
  color="#000000" 
  className="terminal-icon" 
/>

// Large blue icon for headers
<CmdIcon 
  size={32} 
  color="#0066cc" 
  className="mb-2" 
/>
```

The component creates a 24x24 viewBox SVG that scales to the specified size, with a stroked rectangle border and a monospace ">" character centered inside to represent a command prompt interface.
