A React SVG icon component that renders a warning/alert triangle with an exclamation mark, commonly used for displaying alerts, warnings, or error states in user interfaces.

## Key Components

- **`AlertTriangleIconProps`** - TypeScript interface extending `React.SVGProps<SVGSVGElement>` for type safety
- **`AlertTriangleIcon`** - Functional React component that renders the triangle warning icon
- **Default props** - `className` defaults to "h-6 w-6", `color` defaults to "white"

## Usage Example

```typescript
import { AlertTriangleIcon } from './alert-triangle-icon'

// Basic usage with defaults
<AlertTriangleIcon />

// Custom styling and color
<AlertTriangleIcon 
  className="h-8 w-8" 
  color="red" 
/>

// With additional SVG props
<AlertTriangleIcon 
  className="h-4 w-4"
  color="#ff6b6b"
  onClick={() => console.log('Warning clicked')}
  role="img"
  aria-label="Warning"
/>

// In an alert component
<div className="alert">
  <AlertTriangleIcon className="h-5 w-5" color="orange" />
  <span>Please review your input</span>
</div>
```

The component accepts all standard SVG element properties and provides a flexible, reusable warning icon for React applications.
