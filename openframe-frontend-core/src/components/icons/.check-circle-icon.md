A React SVG icon component that renders a check mark inside a circle, commonly used to indicate success, completion, or validation states.

## Key Components

- **CheckCircleIcon**: Main functional component that renders the SVG icon
- **CheckCircleIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional `className` prop
- **SVG Paths**: Two path elements creating the circle outline and inner checkmark

## Usage Example

```typescript
import { CheckCircleIcon } from './check-circle-icon'

// Basic usage with default styling
<CheckCircleIcon />

// With custom color and className
<CheckCircleIcon 
  color="#10B981" 
  className="w-6 h-6" 
/>

// With additional SVG props
<CheckCircleIcon 
  width={32}
  height={32}
  onClick={handleClick}
  aria-label="Task completed"
/>

// In a success message component
function SuccessMessage() {
  return (
    <div className="flex items-center gap-2">
      <CheckCircleIcon color="#059669" />
      <span>Operation completed successfully</span>
    </div>
  )
}
```

The component accepts all standard SVG props and defaults to a 24x24 viewBox with `#1A1A1A` color. The icon maintains aspect ratio and can be easily styled with CSS classes or inline props.
