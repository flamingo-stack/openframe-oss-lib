A React SVG icon component that renders a users/group icon with customizable styling properties.

## Key Components

- **UsersGroupIcon**: Main functional component that renders an SVG icon
- **UsersGroupIconProps**: TypeScript interface defining component props
  - `className?: string` - Optional CSS class names
  - `size?: number` - Icon dimensions (default: 24px)
  - `color?: string` - Fill color (default: 'white')

## Usage Example

```typescript
import { UsersGroupIcon } from './users-group-icon';

// Basic usage with defaults
<UsersGroupIcon />

// Customized icon
<UsersGroupIcon 
  size={32} 
  color="#007bff" 
  className="mr-2 hover:opacity-80" 
/>

// In a button
<button className="flex items-center">
  <UsersGroupIcon size={20} color="currentColor" />
  <span className="ml-2">Manage Users</span>
</button>
```

The component provides a clean, reusable way to display user group iconography throughout an application with consistent styling and easy customization options.
