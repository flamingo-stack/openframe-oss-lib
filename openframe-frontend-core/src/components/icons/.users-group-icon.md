A React SVG icon component that renders a users group/team icon with customizable appearance properties.

## Key Components

- **UsersGroupIcon**: Main React functional component that renders an SVG icon
- **UsersGroupIconProps**: TypeScript interface defining optional props:
  - `className` - CSS class string for styling
  - `size` - Icon dimensions in pixels (default: 24)
  - `color` - Fill color for the icon (default: 'white')

## Usage Example

```typescript
import { UsersGroupIcon } from './users-group-icon';

// Basic usage with defaults
<UsersGroupIcon />

// Custom size and color
<UsersGroupIcon size={32} color="#333" />

// With additional CSS classes
<UsersGroupIcon 
  className="team-icon hover:opacity-80" 
  size={20} 
  color="currentColor" 
/>

// Responsive sizing
<UsersGroupIcon 
  size={window.innerWidth < 768 ? 16 : 24}
  color="var(--primary-color)"
/>
```

The component uses a 24x24 viewBox and contains vector paths that create a multi-user icon design. It's ideal for UI elements representing teams, groups, or collaborative features in applications.
