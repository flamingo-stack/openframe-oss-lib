A React component that renders an SVG icon representing a vendor directory with a search magnifying glass overlay, commonly used in business/vendor management interfaces.

## Key Components

- **VendorDirectoryIcon**: Main React functional component that renders the SVG icon
- **VendorDirectoryIconProps**: TypeScript interface defining optional props for customization
- **Props**: `color`, `className`, `width`, `height` with sensible defaults

## Usage Example

```typescript
import { VendorDirectoryIcon } from './vendor-directory-icon';

// Basic usage with defaults (20x20, white color)
<VendorDirectoryIcon />

// Customized icon
<VendorDirectoryIcon 
  width={24} 
  height={24} 
  color="#3B82F6" 
  className="hover:opacity-80" 
/>

// In a button component
<button className="flex items-center gap-2">
  <VendorDirectoryIcon color="currentColor" width={16} height={16} />
  Browse Vendors
</button>
```

The icon features a box/package symbol combined with a magnifying glass, making it ideal for vendor search, directory browsing, or supplier management functionality in applications.
