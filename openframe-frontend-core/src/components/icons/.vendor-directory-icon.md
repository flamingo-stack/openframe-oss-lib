A deprecated SVG icon component representing a vendor directory with a search/magnifier overlay.

## Key Components

**`VendorDirectoryIcon`** — A React functional component rendering an inline SVG icon composed of two paths: a 3D package/directory shape and a circular search indicator.

**`VendorDirectoryIconProps`** interface:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | Fill color for both SVG paths |
| `className` | `string` | `""` | CSS class for the SVG element |
| `width` | `number` | `20` | SVG width in pixels |
| `height` | `number` | `20` | SVG height in pixels |

> **Deprecated:** Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { VendorDirectoryIcon } from './vendor-directory-icon';

// Basic usage
<VendorDirectoryIcon />

// Custom size and color
<VendorDirectoryIcon
  color="#FFC008"
  width={24}
  height={24}
  className="my-icon"
/>
```
