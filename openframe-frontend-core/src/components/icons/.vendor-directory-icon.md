A React SVG icon component that renders a vendor directory/search icon, featuring a 3D package/box with a magnifying glass overlay to represent searching through vendor directories.

## Key Components

- **VendorDirectoryIcon**: Main functional component that renders the SVG icon
- **VendorDirectoryIconProps**: TypeScript interface defining customizable props including color, dimensions, and CSS classes

## Usage Example

```typescript
import { VendorDirectoryIcon } from './vendor-directory-icon';

// Basic usage with defaults (white, 20x20px)
<VendorDirectoryIcon />

// Customized appearance
<VendorDirectoryIcon 
  color="#3B82F6" 
  width={32} 
  height={32} 
  className="hover:opacity-80 transition-opacity" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <VendorDirectoryIcon color="currentColor" />
  <span>Search Vendors</span>
</button>
```

The icon uses a fixed 20x20 viewBox that scales proportionally with the width/height props. The SVG contains two main path elements: a 3D isometric package/directory icon and a magnifying glass overlay, making it perfect for vendor search or directory browsing functionality in applications.
