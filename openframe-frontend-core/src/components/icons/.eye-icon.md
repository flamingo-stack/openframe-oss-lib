Renders an eye or eye-off SVG icon, typically used for toggling password visibility in form fields.

## Key Components

**`EyeIcon`** — `React.FC<EyeIconProps>`
> ⚠️ **Deprecated:** Use icons from `icons-v2-generated` instead.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `size` | `number` | `24` | Width and height in pixels |
| `color` | `string` | `'white'` | SVG stroke color |
| `off` | `boolean` | `false` | When `true`, renders the eye-off (hidden) variant |

## Usage Example

```typescript
import { EyeIcon } from './eye-icon'

// Standard eye (visible state)
<EyeIcon size={20} color="#000" />

// Eye-off (hidden state)
<EyeIcon off size={20} color="#000" />

// Password toggle button
const [show, setShow] = useState(false)

<button onClick={() => setShow(prev => !prev)}>
  <EyeIcon off={!show} size={18} color="gray" />
</button>
```
