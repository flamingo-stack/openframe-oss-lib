A React component that renders an eye icon, with support for toggling between visible and hidden states (eye/eye-off icons).

## Key Components

- **`EyeIcon`** - Main React functional component that renders SVG eye icons
- **`EyeIconProps`** - TypeScript interface defining component props:
  - `className` - Optional CSS classes
  - `size` - Icon dimensions (default: 24px)
  - `color` - Stroke color (default: 'white')
  - `off` - Toggle between eye and eye-off states (default: false)

## Usage Example

```typescript
import { EyeIcon } from './eye-icon'

// Basic usage
<EyeIcon />

// Password visibility toggle
const [showPassword, setShowPassword] = useState(false)

<button onClick={() => setShowPassword(!showPassword)}>
  <EyeIcon 
    off={showPassword} 
    size={20} 
    color="gray" 
    className="cursor-pointer"
  />
</button>

// Custom styling
<EyeIcon 
  size={32} 
  color="#3b82f6" 
  className="hover:opacity-80"
/>
```

The component is commonly used in forms for password visibility toggles, where the `off` prop controls whether to show the standard eye (visible) or crossed-out eye (hidden) icon.
