A React component that renders an eye icon with support for both visible and hidden states, commonly used for password visibility toggles.

## Key Components

- **EyeIcon**: Main functional component that renders SVG eye icons
- **EyeIconProps**: TypeScript interface defining component props
- **Conditional rendering**: Switches between eye and eye-off icons based on the `off` prop

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes for styling |
| `size` | `number` | `24` | Icon dimensions in pixels |
| `color` | `string` | `'white'` | SVG stroke color |
| `off` | `boolean` | `false` | When true, shows eye-off (hidden) state |

## Usage Example

```typescript
import { EyeIcon } from './eye-icon'

// Basic eye icon
<EyeIcon />

// Eye-off icon with custom styling
<EyeIcon 
  off={true} 
  size={20} 
  color="#666666" 
  className="cursor-pointer hover:opacity-80" 
/>

// Password visibility toggle
const [showPassword, setShowPassword] = useState(false)

<button onClick={() => setShowPassword(!showPassword)}>
  <EyeIcon off={!showPassword} color="#374151" />
</button>
```

The component uses inline SVG elements for optimal performance and customization, making it ideal for form controls and UI elements that need to indicate visibility states.
