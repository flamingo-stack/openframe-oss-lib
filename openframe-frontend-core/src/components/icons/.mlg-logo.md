A React component that renders the MLG logo as a customizable SVG graphic.

## Key Components

- **`MlgLogo`** - Main React functional component that renders an SVG logo
- **`MlgLogoProps`** - TypeScript interface defining optional `color` and `size` props

## Usage Example

```typescript
import { MlgLogo } from './mlg-logo';

// Default usage (240px, white color)
<MlgLogo />

// Custom size and color
<MlgLogo size={120} color="#ff6b35" />

// Large logo with dark color
<MlgLogo size={400} color="#2c3e50" />
```

The component provides a scalable vector logo with default dimensions of 240px and white color. Both properties can be overridden via props to match different design requirements. The SVG maintains its aspect ratio and visual quality at any size.
