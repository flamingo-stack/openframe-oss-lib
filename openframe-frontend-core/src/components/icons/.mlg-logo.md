A React component that renders the MLG (Major League Gaming) logo as an SVG with customizable size and color properties.

## Key Components

- **MlgLogoProps** - Interface defining optional `color` and `size` props
- **MlgLogo** - Main component that renders the MLG logo SVG
- **SVG paths** - Two path elements that form the complete MLG logo design

## Usage Example

```typescript
import { MlgLogo } from './mlg-logo';

// Default logo (240px, white)
<MlgLogo />

// Custom sized logo
<MlgLogo size={120} />

// Custom colored logo
<MlgLogo color="#ff6b35" />

// Fully customized logo
<MlgLogo size={300} color="#1a1a1a" />
```

The component uses a fixed viewBox (0 0 1080 1081.4) that scales proportionally based on the `size` prop. The `color` prop controls the fill color of both SVG paths. Default values are 240px size and white color, making it suitable for dark backgrounds or branded applications.
