Renders the MLG (Managed Lifecycle Gateway) brand logo as an inline SVG React component with configurable size and color.

## Key Components

### `MlgLogoProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'white'` | Fill color for the SVG paths |
| `size` | `number` | `240` | Width and height in pixels |

### `MlgLogo`
Deprecated SVG logo component. Renders a static SVG at a `1080×1081.4` viewBox, scaling uniformly via the `size` prop.

> ⚠️ **Deprecated** — Use icons from `icons-v2-generated` instead.

## Usage Example

```typescript
import { MlgLogo } from './mlg-logo';

// Default white logo at 240px
<MlgLogo />

// Custom color and size
<MlgLogo color="#FFC008" size={120} />

// Dark background usage
<MlgLogo color="black" size={48} />
```
