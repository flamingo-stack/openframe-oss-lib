A React SVG icon component that displays a stylized "reclaim profits" graphic with a golden/yellow color scheme, featuring circular and hand-like elements.

## Key Components

- **ReclaimProfitsIcon** - Main React functional component that renders an SVG icon
- Accepts standard SVG props and an optional `className` prop
- Fixed 24x24 pixel dimensions with scalable viewBox
- Uses `#FFC008` (golden yellow) fill color for all path elements

## Usage Example

```typescript
import { ReclaimProfitsIcon } from './reclaim-profits-icon';

// Basic usage
<ReclaimProfitsIcon />

// With custom styling
<ReclaimProfitsIcon 
  className="w-8 h-8 hover:opacity-80" 
  onClick={() => console.log('Icon clicked')}
/>

// With additional SVG props
<ReclaimProfitsIcon 
  width={32}
  height={32}
  aria-label="Reclaim profits"
/>
```

The component spreads all props to the underlying SVG element, making it fully compatible with standard SVG attributes and event handlers while maintaining the original 24x24 aspect ratio through the viewBox.
