A React SVG icon component that renders a minus sign inside a circle, commonly used for removal or decrement actions in user interfaces.

## Key Components

- **MinusCircleIcon**: Main functional component that renders the SVG icon
- **MinusCircleIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional `className` prop
- **SVG Elements**: Two path elements creating the circle outline and horizontal minus line with default gray fill (`#888888`)

## Usage Example

```typescript
import { MinusCircleIcon } from './minus-circle-icon'

// Basic usage
<MinusCircleIcon />

// With custom styling
<MinusCircleIcon 
  className="text-red-500 hover:text-red-700" 
  width={32} 
  height={32}
/>

// As a button with click handler
<button onClick={handleRemove}>
  <MinusCircleIcon className="w-5 h-5" />
  Remove Item
</button>

// With custom fill color
<MinusCircleIcon 
  fill="currentColor"
  className="text-blue-600"
/>
```

The component accepts all standard SVG props and spreads them to the underlying `<svg>` element, making it highly customizable for different use cases like form controls, list item removal, or quantity decrements.
