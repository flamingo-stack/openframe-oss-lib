A React component that renders a minus sign inside a circle as an SVG icon, commonly used for remove or decrement actions in user interfaces.

## Key Components

- **MinusCircleIconProps** - Interface extending `React.SVGProps<SVGSVGElement>` with optional className
- **MinusCircleIcon** - Main export function that renders the SVG icon
- **SVG Structure** - 24x24 viewBox with circle outline and centered minus line paths

## Usage Example

```typescript
import { MinusCircleIcon } from './minus-circle-icon'

// Basic usage
<MinusCircleIcon />

// With custom styling
<MinusCircleIcon 
  className="text-red-500 hover:text-red-700" 
  width={20} 
  height={20} 
/>

// As a button for removing items
<button onClick={handleRemove}>
  <MinusCircleIcon className="w-5 h-5" />
  Remove Item
</button>

// With additional SVG props
<MinusCircleIcon 
  fill="currentColor"
  stroke="none"
  aria-label="Remove"
/>
```

The component accepts all standard SVG props through prop spreading, making it highly customizable. The default size is 24x24 pixels with a gray fill color (`#888888`), but these can be overridden through className or direct SVG props.
