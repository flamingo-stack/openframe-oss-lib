A React SVG component that renders the Linux Tux penguin logo as a scalable icon with customizable styling properties.

## Key Components

- **LinuxIconProps**: TypeScript interface extending standard SVG element props
- **LinuxIcon**: Functional component that renders the Linux penguin mascot SVG

## Usage Example

```typescript
import { LinuxIcon } from './linux-icon'

// Basic usage with default white color
<LinuxIcon />

// Custom styling
<LinuxIcon 
  className="my-linux-icon" 
  color="#0066cc" 
  width={24} 
  height={24} 
/>

// With click handler
<LinuxIcon 
  color="black"
  onClick={() => console.log('Linux icon clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component accepts all standard SVG props and defaults to 16x16 pixels with white fill color. The `color` prop controls the fill color of the penguin silhouette, while additional styling can be applied through `className` or other SVG attributes.
