A React SVG icon component that renders the Linux penguin (Tux) logo with customizable styling properties.

## Key Components

- **LinuxIconProps**: Interface extending React's SVGProps for type-safe SVG attributes
- **LinuxIcon**: Functional component that renders the Linux mascot as an inline SVG

## Usage Example

```typescript
import { LinuxIcon } from './linux-icon'

// Basic usage with default white color
<LinuxIcon />

// Custom styling
<LinuxIcon 
  color="#000000" 
  className="w-8 h-8" 
  onClick={handleClick}
/>

// With custom dimensions
<LinuxIcon 
  width={24} 
  height={24} 
  color="#1a73e8"
/>
```

The component accepts all standard SVG props including `className`, `onClick`, `width`, `height`, and styling attributes. The `color` prop defaults to 'white' and controls the fill color of the penguin silhouette. The viewBox is set to maintain the icon's aspect ratio across different sizes.
