A React component that renders an SVG icon representing a table view with grid lines and dividers.

## Key Components

- **TableViewIcon** - React functional component that renders a customizable table/grid icon
- **Props interface** - Accepts `className`, `size`, and `color` parameters for styling customization

## Usage Example

```typescript
import { TableViewIcon } from './table-view-icon'

// Basic usage with defaults
<TableViewIcon />

// Custom styling
<TableViewIcon 
  className="w-6 h-6 hover:opacity-75" 
  size={24} 
  color="#3b82f6" 
/>

// In a button
<button className="flex items-center gap-2">
  <TableViewIcon size={16} color="currentColor" />
  Table View
</button>
```

The component uses default values of `w-5 h-5` for className, `22` for size, and `'white'` for color. The SVG maintains its aspect ratio and includes proper accessibility attributes through the viewBox and xmlns properties.
