A React component that renders an SVG table view icon with customizable size, color, and styling properties.

## Key Components

- **TableViewIcon**: A functional React component that exports a customizable table grid icon
- **Props interface**: Accepts optional `className`, `size`, and `color` props with sensible defaults
- **SVG elements**: Contains two path elements that form a table/grid visualization

## Usage Example

```typescript
import { TableViewIcon } from './table-view-icon'

// Basic usage with defaults
<TableViewIcon />

// Custom styling
<TableViewIcon 
  className="w-6 h-6" 
  size={24} 
  color="#3B82F6" 
/>

// In a button or interactive element
<button className="flex items-center gap-2">
  <TableViewIcon size={18} color="currentColor" />
  Switch to Table View
</button>
```

The component is designed for UI elements where users can toggle between different data visualization modes, commonly used in data tables, spreadsheet applications, or dashboard interfaces. The icon visually represents a table grid structure and accepts standard styling props for integration with design systems.
