SVG icon component for the "Pilot" brand/logo mark, rendering the stylized "pilot" wordmark as inline SVG.

## Key Components

**`PilotIconProps`** — Interface defining optional props:
- `className` — CSS classes for sizing/layout (defaults to `'w-6 h-6'`)
- `color` — Fill color for the SVG paths (defaults to `'var(--ods-system-greys-white)'`)

**`PilotIcon`** — Functional component exporting the Pilot logo as an inline SVG with a `156.8 x 83` viewBox.

## Usage Example

```typescript
// Default white icon at 24x24px
<PilotIcon />

// Custom size and color
<PilotIcon className="w-10 h-10" color="#FFC008" />

// Inheriting color from parent via currentColor
<PilotIcon color="currentColor" className="w-8 h-8 text-gray-900" />
```
