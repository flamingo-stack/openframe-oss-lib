A React component that renders the Capterra logo as a customizable SVG icon with configurable dimensions and colors.

## Key Components

- **CapterraIcon**: Main React functional component that renders the Capterra logo
- **CapterraIconProps**: TypeScript interface defining optional props for customization
- **SVG Elements**: Four colored path elements that form the geometric Capterra logo design

## Usage Example

```typescript
import { CapterraIcon } from './capterra-icon';

// Basic usage with defaults
<CapterraIcon />

// Customized size and colors
<CapterraIcon 
  width={100}
  height={100}
  className="logo-icon"
  firstColor="#custom1"
  secondColor="#custom2"
  thirdColor="#custom3"
  fourthColor="#custom4"
/>

// Responsive sizing
<CapterraIcon 
  width="100%"
  height="auto"
  className="responsive-logo"
/>
```

The component uses default orange, blue, dark blue, and red colors that match Capterra's brand identity, with a default size of 2500x2500 pixels. All props are optional, making it flexible for various use cases while maintaining brand consistency.
