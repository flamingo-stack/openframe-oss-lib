A React component that renders the Capterra logo as a customizable SVG icon with configurable colors and dimensions.

## Key Components

- **CapterraIcon**: Main React functional component that renders the Capterra logo
- **CapterraIconProps**: TypeScript interface defining component props including dimensions and color customization options

## Usage Example

```typescript
import { CapterraIcon } from './capterra-icon';

// Basic usage with default styling
<CapterraIcon />

// Custom size and colors
<CapterraIcon 
  width={100}
  height={100}
  className="logo-icon"
  firstColor="#ff0000"
  secondColor="#00ff00"
  thirdColor="#0000ff"
  fourthColor="#ffff00"
/>

// Responsive sizing
<CapterraIcon 
  width="100%"
  height="auto"
  className="responsive-logo"
/>
```

The component accepts optional props for width/height (defaults to 2500x2500), CSS classes, and four customizable colors that correspond to different sections of the logo. The SVG uses a fixed viewBox and path elements to create the distinctive Capterra geometric design.
