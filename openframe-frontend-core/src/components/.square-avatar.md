A reusable React component that displays a square avatar with rounded edges, supporting both image display and initials fallback functionality.

## Key Components

- **`SquareAvatar`** - Main component that renders either an image or initials based on props
- **`SquareAvatarProps`** - TypeScript interface defining component props including image source, fallback name, size, and styling options

## Usage Example

```typescript
import { SquareAvatar } from './square-avatar';

// With image
<SquareAvatar 
  src="https://example.com/profile.jpg"
  fallbackName="John Doe"
  size={64}
/>

// Fallback to initials
<SquareAvatar 
  fallbackName="Jane Smith"
  size={48}
  className="border-2 border-blue-500"
/>

// Full width with aspect ratio
<SquareAvatar 
  src="https://example.com/avatar.png"
  fallbackName="Company Name"
  fullWidth
/>
```

The component automatically generates up to 2 initials from the fallback name, uses image proxying for external URLs, and provides flexible sizing options. It falls back gracefully to initials when no image is provided or fails to load.
