A React component that renders an SVG icon representing a user with an edit/pencil overlay, commonly used for user profile editing or account management interfaces.

## Key Components

- **UserIcon**: Main component that renders a customizable user icon SVG
- **UserIconProps**: Interface defining optional props for width, height, and CSS className
- Default dimensions: 24x23 pixels with "text-current" className for CSS color inheritance

## Usage Example

```typescript
import { UserIcon } from './user-icon'

// Basic usage with defaults
<UserIcon />

// Custom sizing
<UserIcon width={32} height={32} />

// With custom styling
<UserIcon 
  width="2rem" 
  height="2rem" 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a button
<button className="flex items-center gap-2">
  <UserIcon width={20} height={20} />
  Edit Profile
</button>
```

The icon uses `currentColor` for the fill, making it easy to style with CSS text color properties. The SVG contains two paths: one depicting a user silhouette and another showing an edit pencil overlay.
