A React component that renders a customizable SVG donut icon with optional CSS class styling.

## Key Components

- **DonutIcon**: Main React functional component that renders the donut SVG
- **className prop**: Optional string for applying custom CSS classes
- **SVG paths**: Two path elements that create the donut shape - inner circle cutout and outer ring

## Usage Example

```typescript
import { DonutIcon } from './donut-icon';

// Basic usage
<DonutIcon />

// With custom styling
<DonutIcon className="text-blue-500 w-8 h-8" />

// In a button or container
<button className="flex items-center gap-2">
  <DonutIcon className="text-white" />
  Settings
</button>
```

The icon uses `currentColor` for the fill, making it inherit the text color of its parent element. The default size is 24x24 pixels with a viewBox that scales proportionally when resized via CSS classes.
