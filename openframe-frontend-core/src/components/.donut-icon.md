A React component that renders an SVG icon depicting a donut or torus shape with customizable styling.

## Key Components

- **DonutIcon**: Function component that accepts an optional `className` prop for custom styling
- **SVG paths**: Two path elements that create the donut shape - one for the inner details and one for the outer ring

## Usage Example

```typescript
import { DonutIcon } from './donut-icon';

// Basic usage
<DonutIcon />

// With custom styling
<DonutIcon className="w-8 h-8 text-blue-500" />

// In a button
<button className="flex items-center gap-2">
  <DonutIcon className="text-current" />
  Statistics
</button>
```

The icon uses `currentColor` for the fill, allowing it to inherit the text color from its parent element. The default size is 24x24 pixels, but can be overridden with CSS classes. Perfect for dashboard interfaces, analytics sections, or anywhere a circular chart representation is needed.
