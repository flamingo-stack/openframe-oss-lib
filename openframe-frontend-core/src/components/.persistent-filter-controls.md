A React component system that provides loading state management for filter controls, keeping them visible but visually disabled during data fetching operations.

## Key Components

- **PersistentFilterControls** - Main wrapper component that manages loading states with opacity changes and pointer event handling
- **PersistentSearchContainer** - Search-specific wrapper that keeps search interactive during loading
- **PersistentSidebar** - Desktop sidebar wrapper with sticky positioning
- **PersistentMobileDropdown** - Mobile-optimized dropdown wrapper with responsive visibility

## Usage Example

```typescript
// Basic filter controls wrapper
<PersistentFilterControls isLoading={isLoadingData}>
  <SearchInput />
  <CategoryFilter />
  <PriceRangeFilter />
</PersistentFilterControls>

// Specialized wrappers for different layouts
<PersistentSearchContainer isLoading={isLoading}>
  <SearchInput placeholder="Search vendors..." />
</PersistentSearchContainer>

<PersistentSidebar isLoading={isLoading}>
  <CategoryFilterSidebar />
</PersistentSidebar>

<PersistentMobileDropdown isLoading={isLoading}>
  <MobileFilterMenu />
</PersistentMobileDropdown>
```

The component provides smooth transitions, accessibility support with ARIA attributes, and customizable opacity/interaction settings during loading states.
