A React component library that provides persistent, accessible filter controls with loading state management and visual feedback for improved user experience during data fetching.

## Key Components

- **`PersistentFilterControls`** - Main wrapper component that manages loading states with visual feedback
- **`PersistentSearchContainer`** - Enhanced search wrapper that keeps search interactive during loading
- **`PersistentSidebar`** - Desktop sidebar wrapper with sticky positioning
- **`PersistentMobileDropdown`** - Mobile-specific dropdown wrapper with responsive visibility

## Usage Example

```typescript
// Basic filter controls with loading state
<PersistentFilterControls isLoading={isLoadingData}>
  <SearchInput />
  <CategoryFilters />
  <SortControls />
</PersistentFilterControls>

// Search container that stays interactive during loading
<PersistentSearchContainer isLoading={isLoading}>
  <SearchInput placeholder="Search vendors..." />
</PersistentSearchContainer>

// Responsive sidebar with custom styling
<PersistentSidebar 
  isLoading={isLoading}
  className="w-64 space-y-4"
>
  <CategoryFilter />
  <PriceRangeFilter />
</PersistentSidebar>

// Mobile dropdown with automatic responsive hiding
<PersistentMobileDropdown isLoading={isLoading}>
  <FilterDropdown />
</PersistentMobileDropdown>
```

The components provide accessibility features including ARIA attributes, screen reader announcements, and keyboard navigation preservation during loading states. They use customizable opacity transitions and pointer event management to create smooth user experiences.
