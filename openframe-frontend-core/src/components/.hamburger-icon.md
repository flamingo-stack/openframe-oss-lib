A reusable React component that renders a hamburger menu icon as an SVG with three horizontal lines. The component is designed for use in mobile navigation menus and responsive layouts.

## Key Components

- `HamburgerIcon` - Main functional component that renders the SVG icon
- `HamburgerIconProps` - TypeScript interface defining optional props for customization
  - `width` - Icon width in pixels (default: 16)
  - `height` - Icon height in pixels (default: 16)
  - `className` - CSS class for styling (default: empty string)

## Usage Example

```typescript
import { HamburgerIcon } from './hamburger-icon';

// Basic usage with defaults
<HamburgerIcon />

// Custom size and styling
<HamburgerIcon 
  width={24} 
  height={24} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a mobile menu button
<button 
  onClick={toggleMobileMenu}
  className="md:hidden p-2"
>
  <HamburgerIcon width={20} height={20} />
</button>
```

The icon uses a fixed white fill color (`#FAFAFA`) and maintains a 16x16 viewBox regardless of the rendered size, ensuring consistent proportions across different dimensions.
