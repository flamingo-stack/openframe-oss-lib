A React component that renders a customizable external link icon as an SVG element, typically used to indicate links that open in new windows or navigate to external websites.

## Key Components

- **CustomExternalLinkIconProps**: Interface defining optional props for customization
- **CustomExternalLinkIcon**: Main React functional component that renders the SVG icon

## Usage Example

```typescript
import { CustomExternalLinkIcon } from './custom-external-link-icon';

// Basic usage with defaults
<CustomExternalLinkIcon />

// Customized appearance
<CustomExternalLinkIcon 
  className="external-link-icon"
  fill="#0066cc"
  width={24}
  height={24}
/>

// In a link component
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  Visit Site <CustomExternalLinkIcon fill="#666" width={16} height={16} />
</a>
```

The component uses default values (gray fill `#888888`, 21x22px dimensions) and spreads additional props to the SVG element for maximum flexibility. The icon displays an arrow pointing up-right from a square, following standard external link iconography.
