A customizable star icon component that renders an SVG with configurable styling properties for React applications.

## Key Components

- **`CustomStarIcon`** - Main component that renders a star-shaped SVG icon
- **`CustomStarIconProps`** - TypeScript interface defining component props:
  - `className` - Optional CSS class names
  - `fill` - Star color (defaults to "#888888")
  - `width` - Icon width in pixels (defaults to 24)
  - `height` - Icon height in pixels (defaults to 24)

## Usage Example

```typescript
import { CustomStarIcon } from './custom-star-icon';

// Basic usage with defaults
<CustomStarIcon />

// Customized star icon
<CustomStarIcon 
  className="star-icon" 
  fill="#FFD700" 
  width={32} 
  height={32} 
/>

// In a rating component
const StarRating = ({ rating }: { rating: number }) => (
  <div className="flex">
    {[1, 2, 3, 4, 5].map(star => (
      <CustomStarIcon
        key={star}
        fill={star <= rating ? "#FFD700" : "#E5E5E5"}
        width={20}
        height={20}
      />
    ))}
  </div>
);
```

The component spreads additional props to the SVG element, allowing for event handlers and other HTML attributes.
