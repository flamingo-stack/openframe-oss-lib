React context provider that manages dynamic theming state and provides theme controls throughout the application component tree.

## Key Components

- **`DynamicThemeProvider`** - Context provider component that wraps the app with theme state
- **`useDynamicTheme`** - Hook to access theme context (theme object, dark mode toggle, update functions)
- **`DynamicThemeContextType`** - Interface defining theme context structure with theme data and control methods

## Usage Example

```typescript
// Wrap your app with the provider
import { DynamicThemeProvider } from './dynamic-theme-provider';

function App() {
  return (
    <DynamicThemeProvider>
      <YourAppComponents />
    </DynamicThemeProvider>
  );
}

// Use the theme hook in components
import { useDynamicTheme } from './dynamic-theme-provider';

function ThemeControls() {
  const { theme, isDark, updateTheme, toggleDark } = useDynamicTheme();

  return (
    <div style={{ backgroundColor: theme.background }}>
      <button onClick={toggleDark}>
        Switch to {isDark ? 'light' : 'dark'} mode
      </button>
      <button onClick={() => updateTheme({ primary: '#ff6b35' })}>
        Update theme
      </button>
    </div>
  );
}
```
