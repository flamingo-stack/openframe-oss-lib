A React component library that provides a fully-featured navigation menu system built on top of Radix UI primitives. It includes styled components for creating dropdown menus, submenus, and navigation bars with smooth animations and accessibility features.

## Key Components

- **NavigationMenu** - Root container component that wraps the entire navigation structure
- **NavigationMenuList** - Container for menu items, renders as a horizontal flex list
- **NavigationMenuItem** - Individual menu item wrapper (re-exported from Radix)
- **NavigationMenuTrigger** - Clickable trigger button with chevron icon for dropdown menus
- **NavigationMenuContent** - Collapsible content area that appears when trigger is activated
- **NavigationMenuLink** - Navigation link component (re-exported from Radix)
- **NavigationMenuViewport** - Portal container for dropdown content positioning
- **NavigationMenuIndicator** - Visual indicator showing active menu state
- **navigationMenuTriggerStyle** - CVA-based styling function for trigger buttons

## Usage Example

```typescript
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
} from "./navigation-menu"

function MainNavigation() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Products</NavigationMenuTrigger>
          <NavigationMenuContent>
            <NavigationMenuLink href="/laptops">Laptops</NavigationMenuLink>
            <NavigationMenuLink href="/phones">Phones</NavigationMenuLink>
          </NavigationMenuContent>
        </NavigationMenuItem>
        <NavigationMenuItem>
          <NavigationMenuLink href="/about">About</NavigationMenuLink>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  )
}
```
