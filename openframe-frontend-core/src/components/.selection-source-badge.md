A React component that displays a badge indicating the source of vendor selection (AI, manual, or placeholder) using the VendorTag component with appropriate styling.

## Key Components

- **SelectionSourceBadge**: Main component that renders a source indicator badge
- **SelectionSourceType**: Type union defining valid source types ('ai', 'manual', 'placeholder')
- **SelectionSourceBadgeProps**: Interface defining component props including source, className, and hidden state

## Usage Example

```typescript
import { SelectionSourceBadge } from './selection-source-badge';

function VendorList() {
  return (
    <div>
      {/* Show AI-selected vendor badge */}
      <SelectionSourceBadge source="ai" />
      
      {/* Show manually selected vendor badge */}
      <SelectionSourceBadge source="manual" />
      
      {/* Hidden badge for conditional display */}
      <SelectionSourceBadge 
        source="placeholder" 
        hidden={!showPlaceholders}
      />
    </div>
  );
}
```

The component wraps the VendorTag component with a small size variant and handles null source values gracefully by returning null. It follows OpenMSP design tokens for consistent styling across the application.
