A React component that displays a small badge indicating the source of a vendor selection (AI, manual, or placeholder).

## Key Components

- **`SelectionSourceBadge`** - Main component that renders a source indicator badge
- **`SelectionSourceType`** - Type union defining valid source types: `'ai' | 'manual' | 'placeholder'`
- **`SelectionSourceBadgeProps`** - Interface defining component props including source, className, and hidden state

## Usage Example

```typescript
import { SelectionSourceBadge } from './selection-source-badge';

// Display AI selection badge
<SelectionSourceBadge source="ai" />

// Display manual selection badge with custom styling
<SelectionSourceBadge 
  source="manual" 
  className="ml-2"
/>

// Hide badge conditionally
<SelectionSourceBadge 
  source="placeholder" 
  hidden={isLoading}
/>
```

The component wraps the `VendorTag` component and automatically handles styling based on the source type, following OpenMSP design tokens. Returns `null` when no source is provided.
