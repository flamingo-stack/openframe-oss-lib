A React component that renders a call-to-action section for joining the OpenFrame waitlist, featuring the OpenFrame logo, compelling copy, and intelligent navigation handling.

## Key Components

- **JoinWaitlistCTA**: Main component with customizable styling and behavior
- **OpenFrameLogo**: Branded logo with customizable colors
- **ResponsiveIconsBlock**: Grid of technology icons in the bottom section
- **JoinWaitlistButton**: Styled button with loading state support

## Props Interface

```typescript
interface JoinWaitlistCTAProps {
  showBottomText?: boolean;        // Controls bottom headline visibility
  onJoin?: () => void;            // Custom handler for CTA button
  loading?: boolean;              // External loading state control
  className?: string;             // Additional CSS classes
}
```

## Usage Example

```typescript
import JoinWaitlistCTA from './join-waitlist-cta';

// Basic usage with default navigation
<JoinWaitlistCTA />

// Custom handler with loading state
<JoinWaitlistCTA 
  onJoin={() => handleCustomJoin()}
  loading={isSubmitting}
  showBottomText={false}
  className="mt-8"
/>

// Hide bottom text and add custom styling
<JoinWaitlistCTA 
  showBottomText={false}
  className="my-16 px-4"
/>
```

The component includes smart navigation logic that either scrolls to an existing form on the waitlist page or navigates to `/waitlist#waitlist-form`, automatically focusing the email input for better UX.
