A comprehensive call-to-action component for OpenFrame's waitlist signup, featuring intelligent navigation handling and customizable loading states.

## Key Components

- **JoinWaitlistCTA**: Main React component that renders a styled CTA section with OpenFrame branding
- **defaultHandler**: Smart navigation logic that either scrolls to existing form or navigates to waitlist page
- **JoinWaitlistCTAProps**: TypeScript interface defining component configuration options
- **Responsive Design**: Mobile-first layout with adaptive typography and spacing

## Usage Example

```typescript
import JoinWaitlistCTA from './join-waitlist-cta';

// Basic usage with default navigation
function HomePage() {
  return <JoinWaitlistCTA />;
}

// Custom handler with loading state
function CustomPage() {
  const [loading, setLoading] = useState(false);
  
  const handleJoin = async () => {
    setLoading(true);
    await submitToAPI();
    setLoading(false);
  };

  return (
    <JoinWaitlistCTA 
      onJoin={handleJoin}
      loading={loading}
      showBottomText={false}
      className="mt-8"
    />
  );
}
```

The component automatically detects if the user is already on the waitlist page and provides smooth scrolling to the form, otherwise it navigates to `/waitlist#waitlist-form`.
