A React component wrapper for the Lucide React Menu icon with customizable styling and dimensions.

## Key Components

- **`MenuIcon`** - React functional component that renders a hamburger menu icon
- **`MenuIconProps`** - TypeScript interface defining optional props for customization

## Usage Example

```typescript
import { MenuIcon } from './menu-icon';

// Basic usage with defaults (24x24px)
<MenuIcon />

// Custom size and styling
<MenuIcon 
  width={32} 
  height={32} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a navigation header
<button className="p-2 rounded-md">
  <MenuIcon className="text-gray-600" width={20} height={20} />
</button>
```

The component provides sensible defaults (24px dimensions, empty className) while allowing full customization of the underlying Lucide Menu icon through props.
