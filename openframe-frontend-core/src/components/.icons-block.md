A responsive icon grid component that creates a decorative background pattern with randomly distributed icons arranged in a 2-row grid that adapts to screen width.

## Key Components

- **ResponsiveIconsBlock**: Main component that renders a responsive grid of icons
- **IconsBlockProps**: Interface defining optional `loading` prop for placeholder state
- **availableIcons**: Static array containing all available icon components from various sources
- **Icon mapping**: Maps lucide-react icons (Sun, Moon, CheckCircle, etc.) to local constants

## Usage Example

```typescript
import { ResponsiveIconsBlock } from './icons-block';

// Basic usage with auto-loading state
<ResponsiveIconsBlock />

// Force loading placeholder
<ResponsiveIconsBlock loading={true} />

// In a layout component
function Header() {
  return (
    <div className="relative">
      <ResponsiveIconsBlock />
      <div className="absolute inset-0 z-20">
        {/* Your header content */}
      </div>
    </div>
  );
}
```

The component automatically calculates grid columns based on viewport width, prevents adjacent icon duplication using a deterministic algorithm, and provides a grayscale visual treatment. Icons are rendered at 56px cell size with responsive scaling and include proper accessibility attributes.
