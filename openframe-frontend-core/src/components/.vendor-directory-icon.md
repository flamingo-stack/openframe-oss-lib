A React component that renders an SVG icon representing a vendor directory with a search magnifying glass overlay.

## Key Components

- **VendorDirectoryIcon**: React functional component that renders a customizable SVG icon
- **VendorDirectoryIconProps**: TypeScript interface defining optional props for styling and sizing
- **SVG Elements**: Two path elements creating a package/directory icon with a search magnifying glass

## Usage Example

```typescript
import { VendorDirectoryIcon } from './vendor-directory-icon';

// Basic usage with default size (20x20)
<VendorDirectoryIcon />

// Custom sizing and styling
<VendorDirectoryIcon 
  width={24} 
  height={24} 
  className="text-blue-500" 
/>

// In a button or navigation element
<button className="flex items-center gap-2">
  <VendorDirectoryIcon width={16} height={16} />
  Vendor Directory
</button>
```

The icon uses `currentColor` for fill, making it easy to style with CSS text color properties. The component accepts optional width, height, and className props for customization while providing sensible defaults.
