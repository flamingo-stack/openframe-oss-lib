A React component that renders the Flamingo logo as a customizable SVG icon with configurable size, color, and styling options.

## Key Components

- **FlamingoLogo**: Main React functional component that renders the flamingo SVG
- **FlamingoLogoProps**: TypeScript interface extending SVGProps with additional customization options
  - `className`: Optional CSS class name
  - `size`: Logo dimensions (default: 120px)
  - `color`: Fill color (default: CSS variable for flamingo pink)

## Usage Example

```typescript
import { FlamingoLogo } from './flamingo-logo';

// Basic usage with defaults
<FlamingoLogo />

// Customized logo
<FlamingoLogo 
  size={80}
  color="#ff69b4"
  className="my-logo"
/>

// With additional SVG props
<FlamingoLogo 
  size={60}
  onClick={() => console.log('Logo clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component uses React.useId() for unique identifiers and spreads additional props to the underlying SVG element, making it fully compatible with standard SVG attributes and event handlers.
