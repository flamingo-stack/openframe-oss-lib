Wrapper components for creating resizable panel layouts using react-resizable-panels with custom styling and optional visual handles.

## Key Components

- **ResizablePanelGroup**: Container component that manages a group of resizable panels with flex layout styling
- **ResizablePanel**: Individual panel component that can be resized within a group
- **ResizableHandle**: Drag handle between panels with optional visual grip indicator

## Usage Example

```typescript
import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from './resizable'

function Layout() {
  return (
    <ResizablePanelGroup direction="horizontal" className="h-screen">
      <ResizablePanel defaultSize={25} minSize={20}>
        <div>Sidebar content</div>
      </ResizablePanel>
      
      <ResizableHandle withHandle />
      
      <ResizablePanel defaultSize={75}>
        <div>Main content area</div>
      </ResizablePanel>
    </ResizablePanelGroup>
  )
}

// Vertical layout
<ResizablePanelGroup direction="vertical">
  <ResizablePanel>Top panel</ResizablePanel>
  <ResizableHandle withHandle />
  <ResizablePanel>Bottom panel</ResizablePanel>
</ResizablePanelGroup>
```

The components support both horizontal and vertical layouts with responsive styling that automatically adjusts for different orientations. The `withHandle` prop adds a visual grip icon for better UX.
