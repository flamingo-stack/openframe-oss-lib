A React SVG component that renders the OpenMSP logo with customizable color properties and standard SVG attributes.

## Key Components

- **OpenmspLogo**: Main React component that renders an SVG logo
- **OpenmspLogoProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with color customization options
- **Color Properties**: 
  - `frontBubbleColor` - Color of the front bubble elements (default: black)
  - `innerFrontBubbleColor` - Color of the inner front bubble (default: white)
  - `backBubbleColor` - Color of the back bubble (default: white)
  - `className` - Optional CSS class name

## Usage Example

```typescript
import { OpenmspLogo } from './openmsp-logo';

// Basic usage with default colors
<OpenmspLogo />

// Custom colors
<OpenmspLogo 
  frontBubbleColor="#3B82F6" 
  innerFrontBubbleColor="#EFF6FF"
  backBubbleColor="#F3F4F6"
  className="w-24 h-24"
/>

// With additional SVG props
<OpenmspLogo 
  width={120}
  height={120}
  onClick={() => console.log('Logo clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The component maintains a fixed viewBox while allowing customization through props and standard SVG attributes.
