A configurable, responsive footer component for React applications that supports flexible layouts, social media integration, and custom content injection through a comprehensive configuration system.

## Key Components

**Interfaces:**
- `FooterLink` - Link structure with href and label
- `FooterSection` - Section containing title and array of links  
- `FooterConfig` - Main configuration object with layout options, social links, and custom content slots
- `FooterProps` - Component props accepting config and optional custom link renderer

**Components:**
- `Footer` - Main export that validates config and renders the footer
- `UniversalFooter` - Internal component handling the actual rendering logic
- `NavLinkSkeleton` - Loading state component for navigation links

**Key Features:**
- Responsive grid layout (2/4 columns)
- Optional social media integration via `SocialIconRow`
- Flexible content positioning (move description left/right)
- Custom component injection slots
- Configurable background colors and styling
- Suspense boundaries with skeleton loading states

## Usage Example

```typescript
const footerConfig: FooterConfig = {
  name: "MyApp",
  legalName: "MyApp Inc.",
  description: "Building the future of web applications",
  sections: [
    {
      title: "Product",
      links: [
        { href: "/features", label: "Features" },
        { href: "/pricing", label: "Pricing" }
      ]
    }
  ],
  social: {
    github: "https://github.com/myapp",
    twitter: "https://twitter.com/myapp"
  },
  backgroundColor: "bg-ods-bg-card"
}

<Footer config={footerConfig} />
```
