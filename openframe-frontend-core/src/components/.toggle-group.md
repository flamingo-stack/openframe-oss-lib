A React component that provides a grouped toggle interface built on Radix UI primitives, allowing multiple toggle buttons to share styling variants and behavior context.

## Key Components

- **`ToggleGroup`** - Container component that wraps toggle items and provides shared context for variant and size styling
- **`ToggleGroupItem`** - Individual toggle button within the group that inherits styling from the group context
- **`ToggleGroupContext`** - React context for sharing variant props between parent and child components

## Usage Example

```typescript
import { ToggleGroup, ToggleGroupItem } from "./toggle-group"

function FilterButtons() {
  return (
    <ToggleGroup type="multiple" variant="outline" size="sm">
      <ToggleGroupItem value="all">All</ToggleGroupItem>
      <ToggleGroupItem value="active">Active</ToggleGroupItem>
      <ToggleGroupItem value="inactive">Inactive</ToggleGroupItem>
    </ToggleGroup>
  )
}

// Single selection mode
function ViewToggle() {
  return (
    <ToggleGroup type="single" defaultValue="grid">
      <ToggleGroupItem value="list">List</ToggleGroupItem>
      <ToggleGroupItem value="grid">Grid</ToggleGroupItem>
    </ToggleGroup>
  )
}
```

The component supports both single and multiple selection modes through Radix UI's toggle group primitive, with automatic styling inheritance from parent to child components.
