Renders a vendor logo icon with configurable size, background style, and fallback initials — designed for consistent vendor branding across cards, tables, and dropdowns.

## Key Components

**`VendorIconProps`** — Interface defining:
- `vendor` — Vendor object extending `VendorWithMedia` with `title`, optional `id`, `slug`, and `logo`
- `size` — One of `xs | sm | md | lg | l | xl` (default: `md`)
- `showBackground` — Toggles rounded container with border (default: `true`)
- `backgroundStyle` — Container theme: `dark | light | white` (default: `dark`)
- `className` — Additional Tailwind classes for the container

**`VendorIcon`** — Main exported component that:
1. Resolves the logo URL via `getVendorLogo`
2. Proxies the image URL via `getProxiedImageUrl`
3. Renders an `<Image>` if a logo is available, or falls back to a 2-character uppercase abbreviation of `vendor.title`

## Usage Example

```typescript
import { VendorIcon } from './vendor-icon'

// With background (default)
<VendorIcon vendor={vendor} size="lg" backgroundStyle="white" />

// Logo only, no container background
<VendorIcon vendor={vendor} size="sm" showBackground={false} />

// In a comparison table with custom class
<VendorIcon
  vendor={vendor}
  size="xl"
  backgroundStyle="light"
  className="mx-auto"
/>

// Fallback renders initials when logo is unavailable
// e.g. vendor.title = "Acme Corp" → displays "AC"
```

## Size Reference

| Prop | Container | Image |
|------|-----------|-------|
| `xs` | 24×24 | 16×16 |
| `sm` | 32×32 | 20×20 |
| `md` | 40×40 | 32×32 |
| `lg` | 48×48 | 40×40 |
| `xl` | 64×64 | 40×40 |
