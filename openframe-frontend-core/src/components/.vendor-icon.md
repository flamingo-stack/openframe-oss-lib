Renders a vendor logo icon with configurable size, background style, and fallback initials display for use across the platform.

## Key Components

**`VendorIcon`** — Primary exported component that displays a vendor's logo image or a 2-character initials fallback.

**`VendorIconProps`** — Interface defining:
- `vendor` — Vendor object with optional `logo`, `slug`, `id`, and required `title`
- `size` — One of `xs | sm | md | lg | l | xl` (default: `md`)
- `showBackground` — Toggles a rounded container with border (default: `true`)
- `backgroundStyle` — One of `dark | light | white` (default: `dark`)
- `className` — Additional CSS classes

**Size maps** — Two lookup tables (`sizeClasses`, `imageSizeMap`) translate size tokens into Tailwind classes and pixel dimensions.

**Background variants** — `backgroundClasses` maps style tokens to ODS design-system utility classes.

## Usage Example

```typescript
import { VendorIcon } from './vendor-icon'

// Standard usage with background
<VendorIcon
  vendor={{ id: 1, title: 'Acme Corp', slug: 'acme', logo: '/logos/acme.png' }}
  size="md"
  backgroundStyle="dark"
/>

// Compact icon without background (e.g. inside a dropdown)
<VendorIcon
  vendor={{ title: 'Zabbix' }}
  size="sm"
  showBackground={false}
  className="mr-2"
/>

// White card style for light backgrounds
<VendorIcon
  vendor={{ title: 'Grafana', logo: null }}
  size="lg"
  backgroundStyle="white"
/>
// Falls back to "GR" initials when logo is null
```
