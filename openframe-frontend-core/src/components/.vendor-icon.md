A reusable React component that displays vendor logos with consistent styling and sizing across the platform, featuring fallback text when logos are unavailable.

## Key Components

- **`VendorIcon`** - Main component that renders vendor logos with configurable size, background, and styling
- **`VendorIconProps`** - TypeScript interface defining component props including vendor data, size options, and styling preferences
- **`sizeClasses`** - Tailwind CSS class mappings for different icon sizes (xs through xl)
- **`imageSizeMap`** - Pixel dimensions for Next.js Image component optimization
- **`backgroundClasses`** - Styling options for different background themes (dark, light, white)

## Usage Example

```typescript
import { VendorIcon } from './vendor-icon'

// Basic usage with medium size
<VendorIcon 
  vendor={{ 
    id: 1, 
    title: "OpenAI", 
    logo: "https://example.com/openai-logo.png" 
  }} 
/>

// Large icon with white background
<VendorIcon 
  vendor={vendorData}
  size="xl"
  backgroundStyle="white"
  showBackground={true}
  className="shadow-md"
/>

// Small icon without background for inline use
<VendorIcon 
  vendor={vendorData}
  size="sm"
  showBackground={false}
/>
```

The component automatically handles image proxying, provides fallback text using vendor initials when logos are missing, and maintains consistent styling across different use cases like vendor cards, comparison tables, and dropdown menus.
