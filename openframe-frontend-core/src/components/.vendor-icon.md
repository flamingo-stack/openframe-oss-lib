A reusable React component for displaying vendor logos with consistent styling and sizing across the platform.

## Key Components

- **VendorIcon**: Main component that renders vendor logos with configurable size, background, and styling options
- **VendorIconProps**: TypeScript interface defining component props including vendor data, size variants, and styling options
- **Size variants**: Six predefined sizes (xs, sm, md, lg, l, xl) with corresponding CSS classes and image dimensions
- **Background styles**: Three background options (dark, light, white) with appropriate styling
- **Fallback rendering**: Displays vendor initials when logo is unavailable

## Usage Example

```typescript
import { VendorIcon } from './vendor-icon'

// Basic usage
<VendorIcon vendor={vendorData} />

// With custom size and background
<VendorIcon 
  vendor={vendorData}
  size="lg"
  backgroundStyle="white"
  showBackground={true}
/>

// Without background for inline usage
<VendorIcon 
  vendor={vendorData}
  size="sm"
  showBackground={false}
  className="mr-2"
/>
```

The component handles image proxying, provides accessible alt text, and gracefully falls back to vendor initials when logos aren't available.
