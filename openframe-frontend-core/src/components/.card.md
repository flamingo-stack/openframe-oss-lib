A collection of reusable React card components providing a consistent interface for displaying structured content with header, title, description, content, and footer sections, plus a specialized horizontal card variant.

## Key Components

- `Card` - Main container with rounded borders and shadow
- `CardHeader` - Top section with vertical spacing for titles/descriptions
- `CardTitle` - Large, semibold heading text
- `CardDescription` - Smaller, muted descriptive text
- `CardContent` - Main content area with padding
- `CardFooter` - Bottom section for actions/controls
- `CardHorizontal` - Specialized horizontal layout with icon, title, and description

## Usage Example

```typescript
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, CardHorizontal } from './card'

// Standard vertical card
<Card>
  <CardHeader>
    <CardTitle>Project Overview</CardTitle>
    <CardDescription>Track your project progress</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Your project content goes here...</p>
  </CardContent>
  <CardFooter>
    <button>View Details</button>
  </CardFooter>
</Card>

// Horizontal card for categories
<CardHorizontal
  icon={<IconComponent />}
  title="Development"
  description="Build amazing applications"
  borderLeft={true}
/>
```
