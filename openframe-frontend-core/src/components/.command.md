A comprehensive command palette component built on top of cmdk with Radix UI integration, providing a searchable interface for commands and actions with keyboard navigation support.

## Key Components

- **Command** - Root command palette container with overflow handling
- **CommandDialog** - Modal wrapper that displays the command palette in a dialog
- **CommandInput** - Search input with integrated search icon and styling
- **CommandList** - Scrollable container for command results with max height constraint
- **CommandEmpty** - Placeholder component shown when no results are found
- **CommandGroup** - Logical grouping container for related commands
- **CommandItem** - Individual selectable command with hover and selection states
- **CommandSeparator** - Visual divider between command groups
- **CommandShortcut** - Display component for keyboard shortcuts
