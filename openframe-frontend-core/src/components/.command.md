A comprehensive command palette component built on top of cmdk with styled UI components for building searchable command interfaces in React applications.

## Key Components

- **Command** - Main container component wrapping CommandPrimitive with custom styling
- **CommandDialog** - Modal dialog version of the command palette using Dialog components
- **CommandInput** - Search input field with integrated search icon
- **CommandList** - Scrollable container for command items with overflow handling
- **CommandGroup** - Groups related commands with optional headings
- **CommandItem** - Individual selectable command entries with hover/selection states
- **CommandEmpty** - Placeholder component shown when no commands match the search
- **CommandSeparator** - Visual divider between command groups
- **CommandShortcut** - Displays keyboard shortcuts aligned to the right

## Usage Example

```typescript
import { 
  Command, 
  CommandDialog, 
  CommandInput, 
  CommandList, 
  CommandGroup, 
  CommandItem 
} from "./command"

function SearchDialog({ open, onOpenChange }: { open: boolean, onOpenChange: (open: boolean) => void }) {
  return (
    <CommandDialog open={open} onOpenChange={onOpenChange}>
      <CommandInput placeholder="Type a command or search..." />
      <CommandList>
        <CommandGroup heading="Suggestions">
          <CommandItem onSelect={() => console.log("Calendar selected")}>
            Calendar
          </CommandItem>
          <CommandItem onSelect={() => console.log("Search Emoji")}>
            Search Emoji
          </CommandItem>
        </CommandGroup>
      </CommandList>
    </CommandDialog>
  )
}
```
