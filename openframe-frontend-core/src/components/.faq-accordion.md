A React component that creates an accessible FAQ accordion interface with smooth animations and keyboard navigation support.

## Key Components

**FaqAccordion**: Main accordion component that renders a list of expandable FAQ items
- `FaqItem`: Interface defining the structure for FAQ data (id, question, answer)
- `FaqAccordionProps`: Props interface supporting items array and optional default open states
- `useMeasuredHeight`: Custom hook for smooth height animations using actual content measurements

## Usage Example

```typescript
import { FaqAccordion, FaqItem } from './faq-accordion'

const faqData: FaqItem[] = [
  {
    id: 1,
    question: "How do I get started?",
    answer: "Simply sign up for an account and follow our onboarding guide."
  },
  {
    id: 2,
    question: "What payment methods do you accept?",
    answer: "We accept all major credit cards, PayPal, and bank transfers."
  }
]

export default function FAQPage() {
  return (
    <FaqAccordion 
      items={faqData}
      defaultOpenIds={[1]} // First item open by default
    />
  )
}
```

The component features smooth expand/collapse animations, hover effects, keyboard accessibility (Enter/Space), and maintains open state using a Set for efficient lookups. Styling uses Tailwind CSS with custom design tokens.
