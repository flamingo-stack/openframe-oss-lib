A React component that renders appropriate icons for different tool types used in logging and monitoring systems.

## Key Components

- **ToolIcon**: Forward-ref React component that displays tool-specific icons
- **renderIcon()**: Internal method that maps tool types to their corresponding icon components
- **Supported Tools**: Fleet MDM, MeshCentral, Tactical RMM, and OpenFrame Chat/Client
- **Props**: `toolType` (string), `size` (number, default 16), `className` (optional styling)

## Usage Example

```typescript
import { ToolIcon } from './tool-icon'

// Basic usage with different tool types
<ToolIcon toolType="FLEET" size={24} />
<ToolIcon toolType="MESHCENTRAL" size={20} />
<ToolIcon toolType="TACTICAL_RMM" className="text-blue-500" />

// With ref for direct DOM access
const iconRef = useRef<HTMLDivElement>(null)
<ToolIcon 
  ref={iconRef}
  toolType="OPENFRAME_CHAT" 
  size={32}
  className="hover:opacity-80"
/>
```

The component normalizes tool type strings to uppercase and handles various naming conventions (e.g., "FLEET_MDM", "FLEET-MDM") while providing a consistent 16px default size and accessible labeling.
