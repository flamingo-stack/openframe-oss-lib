A flexible React component that renders specific tool icons based on the provided tool type, with support for various MSP tools and OpenFrame variants.

## Key Components

- **ToolIcon**: Forwarded ref component that displays icons for different MSP tools
- **renderIcon()**: Internal function that maps tool types to their corresponding icon components
- **Tool Type Mapping**: Supports Fleet MDM, MeshCentral, Tactical RMM, and OpenFrame variants
- **Flexible Sizing**: Configurable icon size with 16px default
- **Accessibility**: Includes aria-label for screen readers

## Usage Example

```typescript
import { ToolIcon } from './tool-icon'

// Basic usage with different tool types
<ToolIcon toolType="FLEET" />
<ToolIcon toolType="MESHCENTRAL" size={24} />
<ToolIcon toolType="TACTICAL_RMM" size={32} className="text-blue-500" />

// OpenFrame variants
<ToolIcon toolType="OPENFRAME" />
<ToolIcon toolType="OPENFRAME_CHAT" />
<ToolIcon toolType="OPENFRAME_CLIENT" />

// With custom styling
<ToolIcon 
  toolType="FLEET_MDM" 
  size={20}
  className="hover:opacity-80 transition-opacity"
/>

// Fallback for unknown tools (renders empty)
<ToolIcon toolType="UNKNOWN_TOOL" />
```

The component normalizes tool type strings to uppercase and handles various naming conventions (e.g., "FLEET_MDM", "FLEET-MDM") while providing a consistent 16x16px container with customizable sizing and styling options.
