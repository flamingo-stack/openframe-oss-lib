A React component that renders tool-specific icons based on the provided tool type, supporting various management and monitoring platforms with consistent sizing and styling.

## Key Components

- **ToolIcon**: Forwardable React component that displays appropriate icons for different tool types
- **renderIcon()**: Internal function that maps tool types to their corresponding icon components
- **Tool type mapping**: Supports Fleet MDM, MeshCentral, Tactical RMM, and OpenFrame platforms
- **Flexible sizing**: Configurable icon size with default 16px dimensions

## Usage Example

```typescript
import { ToolIcon } from './components/tool-icon'

// Basic usage with different tool types
function LogEntry() {
  return (
    <div className="log-entry">
      <ToolIcon toolType="FLEET_MDM" size={24} />
      <ToolIcon toolType="MESHCENTRAL" />
      <ToolIcon toolType="TACTICAL_RMM" className="mr-2" />
      <ToolIcon toolType="OPENFRAME_CHAT" size={20} />
    </div>
  )
}

// With ref forwarding
function InteractiveLog() {
  const iconRef = React.useRef<HTMLDivElement>(null)
  
  return (
    <ToolIcon 
      ref={iconRef}
      toolType="FLEET"
      size={32}
      className="hover:opacity-80 cursor-pointer"
    />
  )
}
```

The component normalizes tool type strings to uppercase and handles various naming conventions (e.g., "FLEET", "FLEET_MDM", "FLEET-MDM" all map to the Fleet icon). It renders an empty fragment for unrecognized tool types and provides consistent accessibility labeling.
