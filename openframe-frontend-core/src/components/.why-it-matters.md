A React client component that renders a styled "Why It Matters" section displaying four numbered value propositions for the Flamingo/OpenFrame platform.

## Key Components

- **`WhyItMattersItem`** — Internal list item component rendering a numbered entry with title and description. Accepts `number`, `title`, `description`, and optional `isLast` props to conditionally suppress the bottom border on the final item.
- **`WhyItMatters`** — Default export. Renders the full section with a hardcoded list of four value propositions: Cut Costs, Stay in Control, Build What You Need, and Scale Freely.

## Usage Example

```typescript
import WhyItMatters from '@/components/why-it-matters';

// Drop into any page layout — no props required
export default function LandingPage() {
  return (
    <main>
      <WhyItMatters />
    </main>
  );
}
```

To extend or customize the value propositions, update the `items` array inside `WhyItMatters`:

```typescript
const items = [
  {
    number: "1.",
    title: "Your Custom Title",
    description: "Your custom description text",
  },
  // ...
];
```

## Notes

- Marked as `"use client"` — runs on the client side in Next.js App Router.
- Styled exclusively with Tailwind CSS using design tokens (`ods-bg`, `ods-card`, `ods-accent`, `ods-text-primary`, `ods-border`) — ensure these are defined in your Tailwind config.
- Layout is responsive: items stack vertically on mobile (`flex-col`) and switch to horizontal on `md+` breakpoints (`flex-row`).
