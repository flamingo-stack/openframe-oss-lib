React component that renders a "Why It Matters" section with a numbered list of benefits, featuring hover effects and responsive design for showcasing key value propositions.

## Key Components

- **WhyItMatters** - Main section component that displays a centered title and numbered list of benefits
- **WhyItMattersItem** - Individual list item component with number, title, and description
- **WhyItMattersItemProps** - TypeScript interface defining props for list items (number, title, description, isLast)

## Usage Example

```typescript
import WhyItMatters from './why-it-matters';

function HomePage() {
  return (
    <div>
      <WhyItMatters />
    </div>
  );
}
```

The component automatically renders four predefined benefits (Cut Costs, Stay in Control, Build What You Need, Scale Freely) in a styled card layout. Each item features:
- Numbered list format with accent coloring
- Hover effects for better interactivity
- Responsive design (mobile/desktop layouts)
- Custom typography using Azeret Mono and DM Sans fonts
- Dark theme with custom CSS variables (ods-*)

The `isLast` prop conditionally removes bottom borders from the final list item for clean visual presentation.
