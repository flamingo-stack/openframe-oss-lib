A React component that renders a customizable sun SVG icon with optional styling and sizing properties.

## Key Components

- **SunIcon**: React functional component that renders an SVG sun icon
- **SunIconProps**: TypeScript interface defining optional props for className, width, and height
- Default dimensions: 24x24 pixels
- Uses `currentColor` fill for easy theming

## Usage Example

```typescript
import { SunIcon } from './sun-icon';

// Basic usage with defaults
<SunIcon />

// Custom size and styling
<SunIcon 
  width={32} 
  height={32} 
  className="text-yellow-500 hover:text-yellow-600" 
/>

// With CSS classes for theming
<SunIcon className="dark-mode-sun" />
```

The component is ideal for theme toggles, weather displays, or any UI requiring a sun symbol. The SVG uses `currentColor` which inherits the text color from parent elements, making it easy to style with CSS or utility classes.
