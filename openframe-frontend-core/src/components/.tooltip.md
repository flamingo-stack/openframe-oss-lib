A React tooltip component library providing both compound component patterns and a simple all-in-one tooltip with intelligent positioning and viewport awareness.

## Key Components

- **`Tooltip`** - Main component with automatic positioning, viewport boundaries, and arrow indicators
- **`TooltipProvider`** - Context provider for compound tooltip components
- **`TooltipRoot`** - State management wrapper for compound tooltips
- **`TooltipTrigger`** - Interactive element that shows/hides tooltip on hover/focus
- **`TooltipContent`** - Tooltip content container with portal rendering

## Usage Example

```typescript
import { Tooltip, TooltipRoot, TooltipTrigger, TooltipContent } from './tooltip'

// Simple tooltip
<Tooltip 
  content="This is a helpful tooltip" 
  side="top" 
  align="center"
>
  <button>Hover me</button>
</Tooltip>

// Compound component pattern
<TooltipRoot>
  <TooltipTrigger>
    <span>Complex trigger</span>
  </TooltipTrigger>
  <TooltipContent side="bottom">
    <div>Rich content with <strong>formatting</strong></div>
  </TooltipContent>
</TooltipRoot>
```

The main `Tooltip` component handles positioning calculations, viewport boundaries, and supports special text formatting including `[YELLOW]` tags for highlighted content. It renders via React portals for proper z-index layering.
