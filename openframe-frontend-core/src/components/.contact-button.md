A client-side React component that provides a reusable button for navigating to the contact page. Uses programmatic navigation to redirect users when clicked.

## Key Components

- **ContactButton** - Main component that renders a clickable button
- **ContactButtonProps** - TypeScript interface defining component props:
  - `children`: React nodes to display inside the button
  - `className`: Optional CSS classes for styling

## Usage Example

```typescript
import { ContactButton } from './contact-button';

// Basic usage
<ContactButton>Get in Touch</ContactButton>

// With custom styling
<ContactButton className="bg-blue-500 text-white px-4 py-2 rounded">
  Contact Us
</ContactButton>

// With icon and text
<ContactButton className="flex items-center gap-2">
  <MailIcon />
  Send Message
</ContactButton>
```

The component handles click events by programmatically navigating to `/contact` using `window.location.href`, making it suitable for scenarios where you need direct page navigation rather than client-side routing.
