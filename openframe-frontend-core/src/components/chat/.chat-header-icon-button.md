Square 56×56 icon button component used in the chat panel's top-navigation bar, serving as a styled cell for actions like archive, restore, close, and the overflow menu trigger.

## Key Components

- **`ChatHeaderIconButton`** — A `forwardRef` button component that renders a fixed-size (`size-14`) icon cell with a left border divider, hover background, and focus ring styles. Fully compatible as a Radix UI `asChild` trigger via prop spread.
- **`ChatHeaderIconButtonProps`** — Extends `React.ButtonHTMLAttributes<HTMLButtonElement>` with no additional props.

## Usage Example

```typescript
import { ChatHeaderIconButton } from './chat-header-icon-button'
import { X } from 'lucide-react'

// Basic close button
<ChatHeaderIconButton onClick={handleClose} aria-label="Close chat">
  <X className="size-5" />
</ChatHeaderIconButton>

// As a Radix DropdownMenu trigger
import * as DropdownMenu from '@radix-ui/react-dropdown-menu'

<DropdownMenu.Trigger asChild>
  <ChatHeaderIconButton aria-label="More options">
    <MoreHorizontal className="size-5" />
  </ChatHeaderIconButton>
</DropdownMenu.Trigger>
```

## Notes

- Default `type` is `'button'` to prevent accidental form submission.
- Icon color stays `ods-text-secondary` on hover; only the background transitions to `ods-bg-hover`.
- Accessibility: pass `aria-label` when the button has no visible text label.
