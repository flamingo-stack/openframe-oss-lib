A 56×56 square icon button component for the chat panel's top navigation bar, rendering action cells (archive, restore, close, menu trigger) separated by a 1px left border divider.

## Key Components

- **`ChatHeaderIconButton`** — A `forwardRef` button component that wraps icon children in a fixed-size, accessible square cell with hover and focus-visible states.
- **`ChatHeaderIconButtonProps`** — Extends `React.ButtonHTMLAttributes<HTMLButtonElement>` with no additional props, allowing full native button prop passthrough.

## Usage Example

```typescript
import { ChatHeaderIconButton } from './chat-header-icon-button'
import { X, Archive } from 'lucide-react'

// Standalone usage
<ChatHeaderIconButton onClick={handleClose} aria-label="Close chat">
  <X size={20} />
</ChatHeaderIconButton>

// As a Radix UI asChild trigger
import * as DropdownMenu from '@radix-ui/react-dropdown-menu'

<DropdownMenu.Trigger asChild>
  <ChatHeaderIconButton aria-label="More options">
    <MoreHorizontal size={20} />
  </ChatHeaderIconButton>
</DropdownMenu.Trigger>
```

## Styling Notes

| Token | Purpose |
|---|---|
| `size-14` | Fixed 56×56px cell |
| `border-l border-ods-border` | 1px divider between header cells |
| `text-ods-text-secondary` | Icon colour (unchanged on hover) |
| `hover:bg-ods-bg-hover` | Background-only hover state |
| `focus-visible:ring-2 ring-ods-accent` | Keyboard focus ring |
