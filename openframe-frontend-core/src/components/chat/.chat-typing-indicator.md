A React component that displays a typing indicator for chat interfaces, showing "Assistant is typing" text with an animated cursor.

## Key Components

- **ChatTypingIndicator**: A forward-ref component that renders a typing indicator with customizable size
- **ChatTypingIndicatorProps**: Interface extending HTMLDivElement attributes with optional size prop
- **Size variants**: Small (sm), medium (md), and large (lg) cursor sizes

## Usage Example

```typescript

      />
    </div>
  )
}
```

The component uses CSS animations (`animate-pulse-cursor`) and design system tokens (`ods-text-secondary`) for consistent styling across the application.
