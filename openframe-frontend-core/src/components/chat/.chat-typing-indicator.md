A React component that displays an animated typing indicator for chat interfaces, featuring three pulsing dots with customizable sizing and optional text display.

## Key Components

- **ChatTypingIndicator** - Main component with forwardRef support
- **ChatTypingIndicatorProps** - Interface extending HTMLDivElement attributes with:
  - `size` - Controls dot and container dimensions ('sm' | 'md' | 'lg')
  - `showText` - Toggles "Assistant is typing" text display
  - `dotClassName` - Custom styling for individual dots
- **dotPulse animation** - CSS keyframes for sequential dot scaling and opacity effects

## Usage Example

```typescript
import { ChatTypingIndicator } from "./chat-typing-indicator"

// Basic usage
<ChatTypingIndicator />

// With text and custom size
<ChatTypingIndicator 
  size="lg" 
  showText={true}
  className="my-4"
/>

// Custom dot styling
<ChatTypingIndicator
  dotClassName="bg-blue-500"
  size="sm"
/>
```

The component creates three animated dots that pulse in sequence with a 200ms delay between each, providing a smooth visual indication that an AI assistant or user is actively typing in a chat interface.
