A React component that displays a visual typing indicator for chat interfaces, showing when an assistant or user is currently typing a message.

## Key Components

- **`ChatTypingIndicator`** - Main React component that renders a "typing" message with an animated cursor
- **`ChatTypingIndicatorProps`** - Interface extending HTML div attributes with optional size prop
- **Size variants** - Three size options: `sm`, `md` (default), and `lg`

## Usage Example

```typescript
import { ChatTypingIndicator } from './components/chat-typing-indicator'

// Basic usage with default medium size
<ChatTypingIndicator />

// With custom size
<ChatTypingIndicator size="lg" />

// With additional styling
<ChatTypingIndicator 
  size="sm" 
  className="my-4 px-3" 
/>
```

The component displays "Assistant is typing" text alongside an animated pulsing cursor. The cursor size scales based on the `size` prop, while maintaining consistent styling through the design system's color tokens (`ods-text-secondary`) and custom animations (`animate-pulse-cursor`). It forwards refs and accepts all standard HTML div attributes for maximum flexibility.
