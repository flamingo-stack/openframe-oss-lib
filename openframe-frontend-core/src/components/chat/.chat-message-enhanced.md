A flexible React component for displaying enhanced chat messages with support for text, tool execution displays, approval requests, and rich media content.

## Key Components

- **ChatMessageEnhanced**: Main forwardRef component that renders chat messages with avatar, content segments, and metadata
- **MessageSegment**: Union type defining different content types (`text`, `tool_execution`, `approval_request`)
- **MessageContent**: Type alias for either string or array of message segments
- **ChatMessageEnhancedProps**: Interface extending HTMLDivElement attributes with chat-specific props

## Usage Example

```typescript
import { ChatMessageEnhanced } from './chat-message-enhanced'

// Simple text message
<ChatMessageEnhanced
  role="assistant"
  content="Hello! How can I help you today?"
  name="Fae"
  timestamp={new Date()}
  showAvatar={true}
/>

// Multi-segment message with tool execution
<ChatMessageEnhanced
  role="assistant"
  content={[
    { type: 'text', text: 'Let me search for that information.' },
    { 
      type: 'tool_execution', 
      data: {
        integratedToolType: 'search',
        toolFunction: 'webSearch',
        input: { query: 'React hooks' },
        output: 'Found 42 results'
      }
    }
  ]}
  isTyping={false}
/>

// Approval request message
<ChatMessageEnhanced
  role="assistant"
  content={[{
    type: 'approval_request',
    data: { message: 'Execute system command?', details: 'rm -rf temp/' },
    status: 'pending',
    onApprove: (id) => console.log('Approved:', id),
    onReject: (id) => console.log('Rejected:', id)
  }]}
/>
```

The component automatically handles avatar display, message formatting, tool expansion states, and responsive styling based on the message role and content type.
