A React component that renders enhanced chat messages with support for multiple content types, AI assistant branding, and expandable tool execution displays in the Flamingo platform.

## Key Components

- **ChatMessageEnhanced** - Main forwardRef component that renders chat messages with role-based styling and avatars
- **MemoizedChatMessageEnhanced** - Performance-optimized memoized version exported as the main component
- **normalizeContent()** - Utility function that converts string content to structured MessageSegment arrays
- **getAvatarProps()** - Generates avatar configuration with role-specific styling (Mingo vs Fae AI assistants)
- **toggleToolExpanded()** - State management for collapsible tool execution displays

## Usage Example

```typescript
import { ChatMessageEnhanced } from './chat-message-enhanced'

// Basic text message
<ChatMessageEnhanced
  role="user"
  content="How can I deploy my application?"
  name="John Doe"
  timestamp={new Date()}
/>

// AI assistant message with tool execution
<ChatMessageEnhanced
  role="assistant"
  assistantType="mingo"
  content={[
    { type: 'text', text: 'I\'ll help you deploy your app.' },
    { 
      type: 'tool_execution', 
      data: {
        integratedToolType: 'kubernetes',
        toolFunction: 'deploy',
        status: 'completed'
      }
    }
  ]}
  isTyping={false}
/>

// Error message
<ChatMessageEnhanced
  role="error"
  content={[{
    type: 'error',
    title: 'Deployment Failed',
    details: 'Invalid configuration detected'
  }]}
/>
```

The component supports text, tool execution, approval requests, and error message segments with role-based theming and avatar customization for Mingo (cyan) and Fae (pink) AI assistants.
