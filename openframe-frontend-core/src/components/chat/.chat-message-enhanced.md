A React component for rendering enhanced chat messages with support for multiple content types, avatars, and interactive elements in the Flamingo OpenFrame chat interface.

## Key Components

- **ChatMessageEnhanced** - Main forwardRef component that renders chat messages with role-based styling
- **normalizeContent()** - Utility function that converts string content to MessageSegment array format
- **MemoizedChatMessageEnhanced** - Performance-optimized memoized version with custom comparison logic

## Core Features

- **Multi-format content support** - Handles text, tool executions, and approval requests
- **Role-based styling** - Different appearances for user, assistant (Mingo/Fae), and error messages
- **Interactive tool displays** - Expandable/collapsible tool execution results
- **Avatar system** - Configurable avatars with gradient backgrounds for different assistant types
- **Typing indicators** - Shows typing animation when `isTyping` is true
- **Performance optimization** - Memoized to prevent unnecessary re-renders

## Usage Example

```typescript
import { ChatMessageEnhanced } from './chat-message-enhanced'

// Basic text message
<ChatMessageEnhanced
  role="assistant"
  content="Hello! How can I help you today?"
  assistantType="mingo"
  timestamp={new Date()}
/>

// Complex message with tool execution
<ChatMessageEnhanced
  role="assistant"
  content={[
    { type: 'text', text: 'Let me check that for you...' },
    { 
      type: 'tool_execution', 
      data: { 
        integratedToolType: 'api',
        toolFunction: 'searchDatabase',
        result: { status: 'success' }
      }
    }
  ]}
  assistantType="fae"
  showAvatar={true}
/>

// User message with typing indicator
<ChatMessageEnhanced
  role="user"
  content=""
  isTyping={true}
  name="John Doe"
/>
```
