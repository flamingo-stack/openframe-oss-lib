A full-bleed accent banner component indicating that the current chat session is in Guide Mode — a read-only, exploratory mode powered by Mingo AI.

## Key Components

- **`GuideModeBanner`** — Main export. Renders a yellow accent strip with a leading `CompassIcon`, a truncated label, and an optional trailing `QuestionCircleIcon` with a tooltip.
- **`GuideModeBannerProps`** — Interface with three optional props:
  - `label` — Banner text (uppercased via `text-h5` style), defaults to `"Guide Mode Chat"`
  - `tooltip` — Tooltip content explaining Guide Mode limitations; pass `null` to hide the help icon entirely
  - `className` — Additional CSS class overrides

## Usage Example

```typescript
// Default usage
<GuideModeBanner />

// Custom label and tooltip
<GuideModeBanner
  label="Exploration Mode"
  tooltip="This session is read-only and will not modify any devices."
/>

// Hide the help icon
<GuideModeBanner tooltip={null} />

// With className override
<GuideModeBanner className="rounded-t-md" />
```

## Notes

- Designed for placement directly below the chat header (Figma node `7532:328222`)
- Guide Mode is intentionally non-destructive: Mingo will answer how-to questions and guide configuration but **will not** touch devices, run scripts, or perform actions
- The tooltip aligns `bottom/end` with a max-width of `280px` and `0ms` delay for immediate display
- Fully accessible: the help button includes `aria-label="About Guide Mode"` and visible focus ring styles
