A full-bleed accent banner component that indicates when the chat interface is operating in Guide Mode, displaying a compass icon, label, and optional help tooltip.

## Key Components

- **`GuideModeBanner`** — Main exported component rendering the yellow accent strip
- **`GuideModeBannerProps`** — Interface with three optional props: `label`, `tooltip`, and `className`
- **Default label** — `'Guide Mode Chat'` (rendered uppercase via `text-h5` style)
- **Default tooltip** — Explains Guide Mode's read-only, exploration-only constraints (no device actions or script execution)

## Usage Example

```typescript
// Default usage — uses built-in label and tooltip
<GuideModeBanner />

// Custom label and tooltip
<GuideModeBanner
  label="Explore Mode"
  tooltip="This session is for exploration only."
/>

// Hide the help icon entirely
<GuideModeBanner tooltip={null} />

// With additional className
<GuideModeBanner className="border-b border-ods-border" />
```

## Behavior Notes

- **Tooltip** is suppressed when `tooltip` prop is explicitly set to `null`
- **`TooltipProvider`** uses `delayDuration={0}` for immediate display
- Tooltip is anchored `bottom/end` with a max-width of `280px`
- The help button includes `aria-label="About Guide Mode"` and a visible `focus-visible` ring for accessibility
- Layout is a fixed `h-8` flex row with `shrink-0` on icons to prevent compression of the label
