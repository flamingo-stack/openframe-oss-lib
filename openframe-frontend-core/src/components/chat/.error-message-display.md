A React component that displays error messages with an optional expandable details section, featuring a red alert design with an AlertCircle icon.

## Key Components

- **ErrorMessageDisplay** - Forwardable React component that renders error messages with consistent styling
- **Expandable Details** - Toggle functionality to show/hide additional error information
- **Icon Integration** - Uses `AlertCircle` and `ChevronDown` from Lucide React for visual indicators
- **Accessibility Support** - Includes proper ARIA attributes for screen readers

## Usage Example

```typescript
import { ErrorMessageDisplay } from './error-message-display'

// Basic error message
<ErrorMessageDisplay 
  title="Connection failed" 
/>

// Error with expandable details
<ErrorMessageDisplay 
  title="Authentication Error"
  details="Invalid credentials provided. Please check your username and password."
  className="mb-4"
/>

// With ref for programmatic access
const errorRef = useRef<HTMLDivElement>(null)

<ErrorMessageDisplay 
  ref={errorRef}
  title="Server Error"
  details="The server encountered an unexpected error. Please try again later."
/>
```

The component uses CSS custom properties for theming (`--ods-attention-red-error-secondary`, `--ods-error`) and applies consistent spacing, typography, and hover states. The details section expands with a smooth rotation animation on the chevron icon.
