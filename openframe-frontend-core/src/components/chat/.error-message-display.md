A React component that displays error messages with an expandable details section, designed for user-friendly error communication in the Flamingo platform.

## Key Components

- **ErrorMessageDisplay**: A forwardRef component that renders error messages with:
  - Alert icon and styled error container
  - Expandable/collapsible details section
  - Accessibility features (ARIA labels, keyboard navigation)
  - Custom styling using CSS variables and Tailwind classes

## Usage Example

```typescript
import { ErrorMessageDisplay } from "./error-message-display"

// Basic error message
<ErrorMessageDisplay 
  title="Authentication failed"
/>

// Error with expandable details
<ErrorMessageDisplay 
  title="Connection timeout"
  details="The server failed to respond within 30 seconds. Please check your network connection and try again."
  className="mb-4"
/>

// With ref for programmatic access
const errorRef = useRef<HTMLDivElement>(null)

<ErrorMessageDisplay 
  ref={errorRef}
  title="Validation error"
  details="Email format is invalid. Please enter a valid email address."
/>
```

The component features a red error theme with white text, an AlertCircle icon from Lucide React, and smooth expand/collapse animations. When details are provided, users can click the chevron button to toggle the detailed error information.
