A React component for displaying tool execution status and results with expandable details in a chat or activity feed interface.

## Key Components

- **`ToolExecutionMessage`** - Interface defining tool execution data structure
- **`ToolExecutionDisplayProps`** - Component props extending HTML div attributes
- **`ToolExecutionDisplay`** - Main component with collapsible view of tool execution progress

## Usage Example

```typescript
import { ToolExecutionDisplay } from './tool-execution-display'

// Executing tool message
const executingMessage = {
  type: "EXECUTING_TOOL" as const,
  integratedToolType: "web_search",
  toolFunction: "searchWeb",
  parameters: { query: "latest AI news", limit: 10 }
}

// Completed tool message
const completedMessage = {
  type: "EXECUTED_TOOL" as const,
  integratedToolType: "web_search", 
  toolFunction: "searchWeb",
  parameters: { query: "latest AI news" },
  result: "Found 5 relevant articles...",
  success: true
}

function ChatInterface() {
  const [expanded, setExpanded] = useState(false)

  return (
    <div>
      <ToolExecutionDisplay 
        message={executingMessage}
        isExpanded={expanded}
        onToggleExpand={() => setExpanded(!expanded)}
      />
    </div>
  )
}
```

The component automatically shows loading states, success/error indicators, and formats parameters and results in an organized, expandable layout.
