A React component that displays the execution status and details of integrated tools, with collapsible sections showing parameters and results.

## Key Components

- **ToolExecutionDisplay** - Main forwardRef component that renders tool execution information
- **handleToggle** - Function to manage expand/collapse state (controlled or uncontrolled)
- **formatToolName** - Utility to format tool type names by replacing underscores with spaces
- **formatParameters** - Utility to format parameter objects for display, with truncation for long strings

## Usage Example

```typescript
import { ToolExecutionDisplay } from './tool-execution-display'

// Executing tool message
const executingMessage = {
  type: "EXECUTING_TOOL",
  integratedToolType: "OPENFRAME_API",
  toolFunction: "createUser",
  parameters: { name: "John", email: "john@example.com" }
}

// Executed tool message
const executedMessage = {
  type: "EXECUTED_TOOL", 
  integratedToolType: "OPENFRAME_API",
  toolFunction: "createUser",
  parameters: { name: "John", email: "john@example.com" },
  result: "User created successfully",
  success: true
}

// Usage with controlled expansion
<ToolExecutionDisplay 
  message={executingMessage}
  isExpanded={false}
  onToggleExpand={() => setExpanded(!expanded)}
/>

// Usage with uncontrolled expansion
<ToolExecutionDisplay message={executedMessage} />
```

The component displays different visual states (loading spinner, success/error icons) based on execution status, shows parameters during execution, and displays results after completion with appropriate success/error styling.
