A React component that displays tool execution status and details with expandable content, showing parameters, results, and visual status indicators for AI tool operations.

## Key Components

- **ToolExecutionDisplay** - Main component that renders tool execution information with collapsible details
- **ToolExecutionMessage** - Interface defining the structure of tool execution data
- **ToolExecutionDisplayProps** - Props interface extending HTML div attributes

## Usage Example

```typescript
import { ToolExecutionDisplay } from "./tool-execution-display"

// Executing tool example
const executingMessage = {
  type: "EXECUTING_TOOL" as const,
  integratedToolType: "file_system",
  toolFunction: "readFile",
  parameters: { path: "/home/user/document.txt" }
}

// Executed tool example  
const executedMessage = {
  type: "EXECUTED_TOOL" as const,
  integratedToolType: "web_search", 
  toolFunction: "search",
  parameters: { query: "React hooks" },
  result: "Found 25 results for React hooks",
  success: true
}

export default function ToolDemo() {
  return (
    <div className="space-y-4">
      <ToolExecutionDisplay 
        message={executingMessage}
        isExpanded={true}
      />
      
      <ToolExecutionDisplay
        message={executedMessage}
        onToggleExpand={() => console.log("Toggled")}
      />
    </div>
  )
}
```

The component automatically handles formatting of tool names, parameter display with truncation for long values, and shows appropriate status icons (loading spinner, success checkmark, or error X) based on execution state.
