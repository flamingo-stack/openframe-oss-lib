A React component that displays tool execution status and details in an expandable card interface, supporting both executing and completed tool states with visual status indicators.

## Key Components

- **ToolExecutionDisplay**: Main component that renders tool execution information with collapsible details
- **ToolExecutionMessage**: Interface defining the structure for tool execution data
- **ToolExecutionDisplayProps**: Props interface extending standard HTML div attributes

Key features:
- Expandable/collapsible interface with chevron indicators
- Real-time status visualization (loading, success, error states)
- Parameter display for both executing and completed tools
- Result display with success/error styling
- Tool icon integration and formatted tool names

## Usage Example

```typescript
import { ToolExecutionDisplay, ToolExecutionMessage } from './tool-execution-display'

// For an executing tool
const executingMessage: ToolExecutionMessage = {
  type: "EXECUTING_TOOL",
  integratedToolType: "DOCKER_COMPOSE",
  toolFunction: "start_containers",
  parameters: {
    service: "web",
    environment: "production"
  }
}

// For a completed tool
const completedMessage: ToolExecutionMessage = {
  type: "EXECUTED_TOOL",
  integratedToolType: "KUBERNETES",
  toolFunction: "deploy_application",
  parameters: { namespace: "default" },
  result: "Deployment successful",
  success: true
}

function ToolExecutionList() {
  const [expanded, setExpanded] = useState(false)
  
  return (
    <div className="space-y-4">
      <ToolExecutionDisplay 
        message={executingMessage}
        isExpanded={expanded}
        onToggleExpand={() => setExpanded(!expanded)}
      />
      <ToolExecutionDisplay message={completedMessage} />
    </div>
  )
}
```
