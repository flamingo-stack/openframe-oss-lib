Compound layout components for building a full-screen chat interface, including a header with avatar, connection status, ticket info, and configurable footer/content regions.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `ChatContainer` | `forwardRef` | Root full-screen wrapper with ODS background and responsive padding |
| `ChatHeader` | `forwardRef` | Header with avatar, user name/title, connection indicator, ticket info, and action buttons |
| `ChatContent` | `forwardRef` | Flex-grow middle region for message lists |
| `ChatFooter` | `forwardRef` | Sticky footer wrapper with optional `fullWidth` or legacy `contentClassName` layout control |
| `ConnectionIndicator` | `FC` (internal) | Animated dot showing `connected` / `connecting` / `disconnected` state using ODS attention tokens |

### `ChatHeader` Notable Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `connectionStatus` | `'connected' \| 'connecting' \| 'disconnected'` | `'disconnected'` | Drives the `ConnectionIndicator` dot color |
| `ticketInfo` | `object` | — | Renders a bordered ticket row with title, meta, and `TicketStatusTag` |
| `fullWidth` | `boolean` | `false` | Drops `max-w-ods-content-narrow` for side-panel layouts |
| `bare` | `boolean` | `false` | Removes card border/background from the header shell |
| `onBack` | `() => void` | — | Renders a back chevron button absolutely positioned to the left |
| `headerActions` | `ReactNode` | — | Custom action slot appended after built-in buttons |

## Usage Example

```typescript
import {
  ChatContainer,
  ChatHeader,
  ChatContent,
  ChatFooter,
} from "@/components/chat/chat-container"

export function MyChatPanel() {
  return (
    <ChatContainer>
      <ChatHeader
        userName="Fae"
        userTitle="AI Assistant"
        connectionStatus="connected"
        serverUrl="mingo.flamingo.run"
        showNewChat
        onNewChat={() => console.log("new chat")}
        onClose={() => console.log("close")}
        ticketInfo={{
          title: "Printer offline — 3rd floor",
          meta: "TKT-1042 · Assigned to Grace",
          status: "in_progress",
          statusName: "In Progress",
        }}
      />
      <ChatContent>
        {/* message list goes here */}
      </ChatContent>
      <ChatFooter fullWidth>
        {/* input bar goes here */}
      </ChatFooter>
    </ChatContainer>
  )
}
```

> **Layout note:** Pass `fullWidth` to both `ChatHeader` and `ChatFooter` when embedding the chat inside a side panel to prevent the default `max-w-ods-content-narrow` (600 px) column from floating in wider containers.
