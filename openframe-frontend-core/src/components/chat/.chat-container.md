React component library for building chat interfaces with a clean, card-based layout that includes connection status indicators, user avatars, and proper responsive design.

## Key Components

- **`ChatContainer`** - Main wrapper component providing full-screen chat layout with ODS theming
- **`ChatHeader`** - Header section displaying user info, server connection status, and optional new chat button
- **`ChatContent`** - Flexible content area for chat messages with proper overflow handling
- **`ChatFooter`** - Footer section for input controls and actions
- **`ConnectionIndicator`** - Visual indicator showing real-time connection status with color coding

## Usage Example

```typescript
import { ChatContainer, ChatHeader, ChatContent, ChatFooter } from './chat-container'

function ChatApp() {
  return (
    <ChatContainer>
      <ChatHeader
        userName="Grace 'Fae' Meadows"
        userTitle="Your Personal Assistant"
        serverUrl="localhost:3000"
        connectionStatus="connected"
        showNewChat={true}
        onNewChat={() => console.log('New chat started')}
      />
      
      <ChatContent>
        {/* Chat messages go here */}
        <div>Your chat messages...</div>
      </ChatContent>
      
      <ChatFooter>
        {/* Input controls go here */}
        <input placeholder="Type a message..." />
      </ChatFooter>
    </ChatContainer>
  )
}
```

The components use ODS (design system) classes for consistent theming and include accessibility features like ARIA labels and proper semantic structure.
