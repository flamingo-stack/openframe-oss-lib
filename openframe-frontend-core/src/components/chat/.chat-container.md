A comprehensive chat interface component system that provides the layout structure and connection status indicators for Flamingo's AI chat experience.

## Key Components

- **ChatContainer** - Main wrapper component with full-screen layout and OpenFrame design system styling
- **ChatHeader** - Header section displaying user info, connection status, and optional new chat button
- **ChatContent** - Flexible content area for chat messages with proper overflow handling
- **ChatFooter** - Footer section for input controls with responsive width constraints
- **ConnectionIndicator** - Visual status indicator showing real-time connection state (connected/connecting/disconnected)

## Usage Example

```typescript
import { ChatContainer, ChatHeader, ChatContent, ChatFooter } from './chat-container'

function ChatInterface() {
  return (
    <ChatContainer>
      <ChatHeader
        userName="Grace 'Fae' Meadows"
        userTitle="Your Personal Assistant"
        connectionStatus="connected"
        serverUrl="api.flamingo.run"
        showNewChat={true}
        onNewChat={() => console.log('New chat started')}
      />
      <ChatContent>
        {/* Chat messages go here */}
      </ChatContent>
      <ChatFooter>
        {/* Input controls go here */}
      </ChatFooter>
    </ChatContainer>
  )
}
```

The component uses OpenFrame Design System (ODS) tokens for consistent theming and includes accessibility features like proper ARIA labels and semantic structure.
