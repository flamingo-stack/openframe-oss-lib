A comprehensive React component library for building chat interfaces with layout components, connection status indicators, and user avatars.

## Key Components

- **ChatContainer** - Main full-screen wrapper with responsive padding and theming
- **ChatHeader** - Header component with user avatar, connection status, and optional new chat button
- **ChatContent** - Flexible content area that fills available space
- **ChatFooter** - Fixed footer section with constrained width
- **ConnectionIndicator** - Status indicator showing connected/disconnected/connecting states

## Usage Example

```typescript
import { ChatContainer, ChatHeader, ChatContent, ChatFooter } from './chat-container'

function ChatApp() {
  return (
    <ChatContainer>
      <ChatHeader
        userName="Grace Meadows"
        userTitle="AI Assistant"
        connectionStatus="connected"
        serverUrl="localhost:8080"
        showNewChat={true}
        onNewChat={() => console.log('New chat')}
      />
      
      <ChatContent>
        {/* Your chat messages go here */}
      </ChatContent>
      
      <ChatFooter>
        {/* Input field and send button */}
      </ChatFooter>
    </ChatContainer>
  )
}
```

All components are forwardRef-enabled and accept standard HTML div props. The header includes a connection indicator that automatically styles based on status (green for connected, yellow pulsing for connecting, red for disconnected).
