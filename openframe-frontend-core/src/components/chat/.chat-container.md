React component system for building chat interfaces with connection status indicators and layout management.

## Key Components

- **ConnectionIndicator**: Visual status indicator with animated states (connected, connecting, disconnected)
- **ChatContainer**: Main wrapper component with responsive padding and full-height layout
- **ChatHeader**: Header section with user avatar, name, connection status, and optional new chat button
- **ChatContent**: Flexible content area that fills remaining vertical space
- **ChatFooter**: Fixed footer section with constrained width for input controls

## Usage Example

```typescript
import { ChatContainer, ChatHeader, ChatContent, ChatFooter } from './chat-container'

function ChatInterface() {
  return (
    <ChatContainer>
      <ChatHeader
        userName="Grace 'Fae' Meadows"
        userTitle="Your Personal Assistant"
        userAvatar="/avatar.png"
        connectionStatus="connected"
        serverUrl="api.flamingo.run"
        showNewChat={true}
        onNewChat={() => console.log('New chat')}
      />
      <ChatContent>
        {/* Chat messages go here */}
      </ChatContent>
      <ChatFooter>
        {/* Input controls go here */}
      </ChatFooter>
    </ChatContainer>
  )
}
```

The components use the OpenFrame Design System (ODS) for consistent theming and include responsive design patterns with proper accessibility attributes.
