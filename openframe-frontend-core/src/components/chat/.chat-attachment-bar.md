Renders the chat attachment UI primitives for the compose box: a chip strip showing staged file thumbnails above the textarea, and a `+` icon button below it for triggering file selection.

## Key Components

### Exports

| Export | Type | Description |
|--------|------|-------------|
| `ChatAttachmentChipStrip` | Component | Horizontal strip of file chips rendered above the textarea; returns `null` when no attachments are staged |
| `ChatAttachmentAddButton` | Component | Icon-only `+` button for the bottom-controls row; renders an invisible same-size placeholder when `attachmentsEnabled` is false to prevent layout shift |
| `StagedAttachment` | Interface | Shape consumed by the chip strip, mirroring `useChatAttachments` output |
| `ChatAttachmentSize` | Type | `'default' \| 'compact'` density toggle for narrow surfaces (e.g. ticket-drawer composer) |
| `CHAT_ATTACHMENT_MIME_TYPES` | Constant | Allowed MIME type allow-list (excludes `image/svg+xml` for XSS safety) |
| `CHAT_ATTACHMENT_CONCURRENT_UPLOADS_PER_USER` | Constant | Client-side cap of 5 concurrent uploads per user |

### Internal

- `AttachmentChip` — single chip within the strip; shows blob thumbnail (images), file-type initials (other), upload progress bar, filename, size/status sub-line, and a `×` remove button
- `useObjectUrl` — local hook that creates a `URL.createObjectURL` blob URL for instant image previews before upload completes

## Usage Example

```typescript
import {
  ChatAttachmentAddButton,
  ChatAttachmentChipStrip,
} from './chat-attachment-bar'

// Bottom-controls row
<ChatAttachmentAddButton
  attachmentsEnabled={identity.attachmentsEnabled}
  attachmentsCount={staged.length}
  onAddFiles={addFiles}
  disabled={isStreaming}
  size="compact"
/>

// Above textarea
<ChatAttachmentChipStrip
  attachments={staged}
  onRemove={removeFile}
  disabled={isStreaming}
  size="compact"
/>
```

> **Layout stability note:** `ChatAttachmentAddButton` always reserves its footprint (`h-6 w-6` compact / `h-7 w-7` default) via an invisible `<span>` placeholder until `attachmentsEnabled` resolves, preventing row-width jumps on async identity load.
