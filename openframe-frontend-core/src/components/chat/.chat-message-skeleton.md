Loading state skeleton components for chat message UI, mirroring the layout of `ChatMessageEnhanced` and `ChatMessageList` during data fetching.

## Key Components

### `ChatMessageSkeleton`
Single animated message row placeholder with:
- Inline avatar pulse (assistant messages only, controlled via `showAvatar` + `isUser`)
- Name + timestamp placeholder bars in the header row
- Three body lines using percentage widths (`full`, `11/12`, `3/5`) for responsive reflow

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Extra classes on the row wrapper |
| `showAvatar` | `boolean` | `true` | Renders the circular avatar placeholder |
| `isUser` | `boolean` | `false` | Suppresses avatar for user-side messages |

---

### `ChatMessageListSkeleton`
Full scrollable list of `ChatMessageSkeleton` rows, matching `ChatMessageList` layout semantics.

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Extra classes on the scroll container |
| `messageCount` | `number` | `6` | Number of skeleton rows to render |
| `showAvatars` | `boolean` | `true` | Passed down to each row |
| `fullWidth` | `boolean` | `false` | Drops `max-w-ods-content-narrow` to fill panel width |
| `contentClassName` | `string` | — | Extra classes on the inner content column |

Every 3rd message (`index % 3 === 0`) is treated as a user message, suppressing its avatar to simulate a realistic conversation pattern.

## Usage Example

```typescript
// Drop-in replacement while chat history loads
import { ChatMessageListSkeleton } from "@/components/chat/chat-message-skeleton"

function ChatPanel() {
  if (isLoading) {
    return (
      <ChatMessageListSkeleton
        messageCount={8}
        fullWidth
        showAvatars
      />
    )
  }
  return <ChatMessageList messages={messages} />
}

// Single row — useful inside virtualized lists or custom loading states
import { ChatMessageSkeleton } from "@/components/chat/chat-message-skeleton"

<ChatMessageSkeleton isUser={false} showAvatar />
```
