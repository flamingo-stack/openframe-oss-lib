A React component that provides animated skeleton placeholders for chat messages and message lists, supporting both user and AI assistant (Mingo/Fae) message types with contextual styling and avatars.

## Key Components

- **`ChatMessageSkeleton`** - Single message skeleton with avatar, name, timestamp, and content placeholders
- **`ChatMessageListSkeleton`** - Full chat message list with configurable message count and assistant types
- **Props interfaces** - `ChatMessageSkeletonProps` and `ChatMessageListSkeletonProps` for type safety

## Usage Example

```typescript
// Single message skeleton
<ChatMessageSkeleton 
  showAvatar={true}
  isUser={false}
  assistantType="mingo"
  className="my-4"
/>

// Full chat list skeleton (loading state)
<ChatMessageListSkeleton
  messageCount={8}
  showAvatars={true}
  assistantType="fae"
  contentClassName="px-6"
/>

// User message skeleton
<ChatMessageSkeleton 
  isUser={true}
  showAvatar={false}
/>
```

The component features gradient avatar placeholders (cyan for Mingo, pink for Fae), responsive layout with flex containers, and smooth scrolling for the message list. All skeleton elements use the `animate-pulse` class and themed colors from the OpenFrame design system (`ods-*` classes).
