A scrollable chat message list component with automatic stick-to-bottom behavior, prepend anchoring for infinite scroll, and streaming indicators.

## Key Components

- **`ChatMessageList`** ‚Äî Main `forwardRef` component rendering a virtualized-style message list with `useStickToBottom` for physics-based auto-scroll
- **`useStickToBottom`** ‚Äî External hook managing scroll-to-bottom with spring animation, user-escape detection, and ResizeObserver
- **`STREAMING_WORDS`** ‚Äî Flamingo-themed cycling phrases displayed during AI response streaming
- **`hasNonEmptyContent`** ‚Äî Utility checking if a `MessageContent` (string or `MessageSegment[]`) contains visible text
- **`AnchorWatcher` / `disposeAnchorWatcher`** ‚Äî Interface and teardown helper for the 2s prepend-settle ResizeObserver window
- **Force-stick effects** ‚Äî Handles dialog changes, first-load, and new user messages with instant scroll snaps
- **Prepend anchoring** (`useLayoutEffect`) ‚Äî Preserves viewport position when older messages load above the current reading position

## Usage Example

```typescript
import { ChatMessageList } from "./chat-message-list"
import type { ChatMessageListProps } from "./types"

const messages = [
  { id: "1", role: "user", content: "Hello!" },
  { id: "2", role: "assistant", content: "Hi there!" },
]

function ChatWindow() {
  const listRef = useRef<HTMLDivElement>(null)

  return (
    <ChatMessageList
      ref={listRef}
      messages={messages}
      dialogId="dialog-123"
      isLoading={false}
      isTyping={true}
      autoScroll={true}
      showAvatars={true}
      hasNextPage={true}
      isFetchingNextPage={false}
      onLoadMore={() => fetchOlderMessages()}
    />
  )
}
```

> **Scroll strategy:** The library handles streaming growth via spring physics. Explicit `scrollToBottom({ animation: 'instant' })` calls are reserved for intent transitions ‚Äî dialog switches, first message, and new user sends (including post-image-load follow-ups).
