A scroll-managed chat message list component that renders a list of chat messages with stick-to-bottom behavior, prepend anchoring for infinite scroll upward, and streaming indicators.

## Key Components

- **`ChatMessageList`** — Main `forwardRef` component wrapping `useStickToBottom` for auto-scroll management
- **`STREAMING_WORDS`** — Flamingo-themed cycling phrases shown during AI response streaming
- **`hasNonEmptyContent`** — Utility that checks if a `MessageContent` (string or `MessageSegment[]`) has visible text
- **`AnchorWatcher`** / **`disposeAnchorWatcher`** — Interface + teardown helper for the 2s resize-settle window used during top-anchor prepend
- **Scroll strategies:**
  - `useStickToBottom` (spring physics) — handles ongoing streaming
  - Force-stick via `scrollToBottom({ animation: 'instant' })` — dialog change, first load, new user message
  - `useLayoutEffect` prepend anchoring — preserves viewport position when older messages load above

## Usage Example

```typescript
import { ChatMessageList } from "@/components/chat"

<ChatMessageList
  ref={listRef}

  NavLinkAnchor={Link}
/>
```

## Scroll Behavior Notes

| Trigger | Scroll Type |
|---------|-------------|
| Dialog change | Instant snap |
| First message in empty chat | Instant snap |
| New user message | Instant snap + `<img>` load follow-up |
| Streaming assistant response | Spring physics (library-managed) |
| Load-older prepend | `scrollTop` delta correction via `useLayoutEffect` |
