A React component for displaying a scrollable list of chat messages with intelligent auto-scrolling behavior and support for different message types.

## Key Components

- **`ChatMessageList`** - Main forwardRef component that renders messages with auto-scroll
- **`Message`** - Interface defining message structure (id, role, content, timestamp, avatar)
- **`ChatMessageListProps`** - Props interface extending HTMLDivAttributes with chat-specific options
- **Auto-scroll logic** - Complex scroll management using ResizeObserver and MutationObserver
- **Message rendering** - Conditionally renders `ChatMessage` or `ChatMessageEnhanced` based on content type

## Usage Example

```typescript
import { ChatMessageList, Message } from './chat-message-list'

const messages: Message[] = [
  {
    id: '1',
    role: 'user',
    content: 'Hello!',
    timestamp: new Date(),
    avatar: '/user-avatar.png'
  },
  {
    id: '2', 
    role: 'assistant',
    content: 'Hi there! How can I help?',
    timestamp: new Date()
  }
]

function ChatContainer() {
  return (
    <ChatMessageList 
      messages={messages}
      isTyping={false}
      autoScroll={true}
      showAvatars={true}
      className="h-96"
    />
  )
}
```

The component automatically scrolls to bottom for new user messages and maintains scroll position for assistant responses, with sophisticated observers to handle dynamic content changes.
