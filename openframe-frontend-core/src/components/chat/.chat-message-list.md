A React component that renders a scrollable list of chat messages with automatic scrolling behavior and support for both basic and enhanced message types.

## Key Components

- **`Message` Interface**: Defines the structure for chat messages with id, role, content, timestamp, and optional avatar
- **`ChatMessageListProps` Interface**: Props including messages array, typing indicator, auto-scroll settings, and styling options
- **`ChatMessageList` Component**: Main forwardRef component that manages message rendering and scroll behavior
- **Auto-scroll Logic**: Complex scroll management using ResizeObserver and MutationObserver for smooth user experience
- **Message Type Detection**: Automatically chooses between `ChatMessage` and `ChatMessageEnhanced` based on content type

## Usage Example

```typescript
import { ChatMessageList, type Message } from './chat-message-list'

function ChatInterface() {
  const [messages, setMessages] = useState<Message[]>([
    {
      id: '1',
      role: 'user',
      content: 'Hello there!',
      timestamp: new Date(),
      name: 'John'
    },
    {
      id: '2',
      role: 'assistant',
      content: [{ type: 'text', text: 'Hi! How can I help you?' }],
      timestamp: new Date(),
      avatar: '/bot-avatar.png'
    }
  ])
  const [isTyping, setIsTyping] = useState(false)

  return (
    <ChatMessageList
      messages={messages}
      isTyping={isTyping}
      autoScroll={true}
      showAvatars={true}
      className="h-96 border rounded"
    />
  )
}
```

The component features intelligent scroll behavior that pins to bottom during assistant responses, smooth scrolling for user messages, and automatic cleanup of observers for optimal performance.
