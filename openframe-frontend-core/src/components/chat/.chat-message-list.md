A React component that renders and manages a scrollable list of chat messages with intelligent auto-scrolling behavior and support for different message types including enhanced content and pending approvals.

## Key Components

- **Message Interface**: Defines the structure for chat messages with role, content, timestamp, and optional avatar/assistant type
- **ChatMessageListProps**: Props interface supporting message list, typing indicator, auto-scroll, avatars, and pending approvals
- **Auto-scroll Management**: Complex scroll behavior that maintains position at bottom for new messages while respecting user scroll position
- **Observer Pattern**: Uses ResizeObserver and MutationObserver to handle dynamic content changes and maintain scroll position
- **Dual Message Components**: Automatically selects between `ChatMessage` and `ChatMessageEnhanced` based on content type

## Usage Example

```typescript
import { ChatMessageList, type Message } from './chat-message-list'

const messages: Message[] = [
  {
    id: '1',
    role: 'user',
    content: 'Hello, how can you help me?',
    timestamp: new Date()
  },
  {
    id: '2', 
    role: 'assistant',
    content: 'I can help you with various tasks!',
    timestamp: new Date(),
    assistantType: 'fae'
  }
]

function ChatInterface() {
  return (
    <ChatMessageList
      messages={messages}
      isTyping={false}
      autoScroll={true}
      showAvatars={true}
      assistantType="fae"
      className="h-96"
    />
  )
}
```

The component features sophisticated scroll management that keeps users at the bottom for new messages while preserving their scroll position when reviewing history, plus sticky pending approval sections for interactive workflows.
