A scrollable chat interface component that displays a list of chat messages with automatic scrolling behavior, skeleton loading states, and pending approval messages.

## Key Components

- **ChatMessageList**: Main forwardRef component that renders chat messages in a scrollable container
- **Auto-scroll logic**: Maintains scroll position at bottom when new messages arrive or user sends messages
- **Scroll detection**: Tracks user scroll behavior to determine when to disable auto-scrolling
- **Loading states**: Shows skeleton placeholder while messages are loading
- **Pending approvals**: Sticky section at bottom for approval-required messages

## Usage Example

```typescript
import { ChatMessageList } from './chat-message-list'
import { useRef } from 'react'

function ChatInterface() {
  const listRef = useRef<HTMLDivElement>(null)
  
  const messages = [
    {
      id: '1',
      role: 'user' as const,
      content: 'Hello!',
      timestamp: new Date(),
    },
    {
      id: '2', 
      role: 'assistant' as const,
      content: 'Hi there! How can I help?',
      timestamp: new Date(),
      assistantType: 'mingo' as const
    }
  ]

  return (
    <ChatMessageList
      ref={listRef}
      messages={messages}
      dialogId="chat-123"
      isLoading={false}
      isTyping={false}
      autoScroll={true}
      showAvatars={true}
      assistantType="mingo"
      pendingApprovals={[]}
      className="h-96"
    />
  )
}
```

The component handles complex scroll behavior including touch gestures, wheel events, and automatic positioning when dialogs change or new messages arrive.
