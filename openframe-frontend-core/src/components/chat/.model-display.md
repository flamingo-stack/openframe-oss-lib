A React component that displays AI model information with provider-specific icons and names, designed for showing model details in UI elements.

## Key Components

- **`ModelDisplay`** - Main forwardRef component that renders model info with icon and name
- **`ModelDisplayProps`** - Interface extending HTMLDivElement attributes with optional `provider`, `modelName`, and `displayName` props
- **`getProviderIcon()`** - Helper function that returns appropriate icon component based on provider string

## Usage Example

```typescript
import { ModelDisplay } from './components/ui/model-display';

// Basic usage with provider and model name
<ModelDisplay 
  provider="openai" 
  modelName="gpt-4-turbo"
/>

// With custom display name
<ModelDisplay 
  provider="anthropic" 
  modelName="claude-3-opus" 
  displayName="Claude 3 Opus"
/>

// With custom styling
<ModelDisplay 
  provider="google" 
  modelName="gemini-pro"
  className="text-lg font-bold"
/>
```

The component automatically matches provider names (case-insensitive) to display appropriate icons for Anthropic/Claude, OpenAI, and Google/Gemini models. It uses the `displayName` prop if provided, otherwise falls back to `modelName`.
