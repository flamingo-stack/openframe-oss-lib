A React component that displays AI model information with provider-specific icons and styling. This component renders the model name alongside an appropriate icon based on the provider (Anthropic/Claude, OpenAI, or Google/Gemini).

## Key Components

- **`ModelDisplay`** - Main forwardRef component that renders model info with icon
- **`ModelDisplayProps`** - Interface extending HTMLDivElement attributes with model-specific props
- **`getProviderIcon`** - Helper function that returns the appropriate provider icon based on provider string

## Usage Example

```typescript
import { ModelDisplay } from './components/ui/model-display';

function ChatInterface() {
  return (
    <div>
      {/* Display with provider and model name */}
      <ModelDisplay 
        provider="anthropic" 
        modelName="claude-3-sonnet" 
        className="mb-2"
      />
      
      {/* Display with custom display name */}
      <ModelDisplay 
        provider="openai" 
        modelName="gpt-4" 
        displayName="GPT-4 Turbo"
      />
      
      {/* Display with Google Gemini */}
      <ModelDisplay 
        provider="google-gemini" 
        modelName="gemini-pro"
      />
    </div>
  );
}
```

The component automatically maps provider strings to their corresponding icons and supports common provider name variations (e.g., "anthropic"/"claude", "google"/"gemini").
