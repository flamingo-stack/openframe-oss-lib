Mobile-optimized header component for the chat panel drawer, implementing the Figma node `7363:85532` layout with a back chevron, truncated `h2` title, and a collapsible `⋯` actions menu.

## Key Components

- **`ChatPanelHeaderMobileProps`** — Extends `ChatPanelHeaderProps` with an optional `className`; `onClose` is accepted for API parity but unused (drawer dismissal is handled externally)
- **`menuItems`** — Dynamically built array of `ActionsMenuItem` entries based on view state (`isArchivedView`, `showBack`) and available callbacks; the `⋯` trigger is hidden entirely when the array is empty
- **`ChatPanelHeaderMobile`** — Exported function component rendering the mobile header with bottom-border card surface, optional back chevron, and trailing actions menu

## Usage Example

```typescript
<ChatPanelHeaderMobile
  title="Support Chat — Acme Corp"
  showBack={true}
  backAriaLabel="Back to chat list"
  isArchivedView={false}
  onBack={handleBack}
  onRename={handleRename}
  onArchive={handleArchive}
  onOpenArchive={handleOpenArchive}
/>

// Archived view — shows Unarchive instead of Rename/Archive
<ChatPanelHeaderMobile
  title="Archived: Printer Issue"
  isArchivedView={true}
  onRestore={handleRestore}
/>
```

## Layout Notes

| Element | Behavior |
|---|---|
| Surface | `ods-card` + bottom border, `l` horizontal/bottom padding, `xl` top padding on title block |
| Title | `text-h2`, truncated with `title` tooltip |
| Back button | 44px outlined icon; shown only when `showBack={true}` |
| `⋯` menu | Hidden when no actions apply (e.g. archive list view) |
