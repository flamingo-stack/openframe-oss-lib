Accessible chat text-input component with slash-command autocomplete, imperative ref API, and send/stop controls for Flamingo's AI chat interfaces.

## Key Components

### `ChatInput` (default export)
`forwardRef` component accepting `ChatInputProps` and exposing `ChatInputRef`.

**Props (via `ChatInputProps`):**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onSend` | `(msg: string) => void \| Promise \| false` | — | Called on submit; returning `false` prevents draft clear |
| `onStop` | `() => void` | — | Cancels in-flight response |
| `sending` | `boolean` | `false` | Disables input while streaming |
| `awaitingResponse` | `boolean` | `false` | Shows typing indicator |
| `slashCommands` | `SlashCommandsConfig` | — | Enables `/cmd` autocomplete |
| `allowEmptySend` | `boolean` | `false` | Permits send with no text (e.g. file attachments) |
| `autoFocus` | `boolean` | `false` | Focuses textarea on mount |
| `disabled` | `boolean` | `false` | Fully disables input |

### `ChatInputRef` (imperative handle)

| Method | Description |
|--------|-------------|
| `focus()` | Focuses textarea (no scroll jump) |
| `blur()` | Blurs textarea |
| `clear()` | Clears draft text |
| `setValue(text)` | Pre-fills input, moves cursor to end |
| `setValueAndCursor(text, offset)` | Pre-fills input, positions cursor at `offset` |
| `submit(text)` | Programmatic send (success-gated clear) |
| `getValue()` | Returns current draft without re-render |

### Slash Command Autocomplete
Triggered by `/` prefix matching `SLASH_INPUT_TRIGGER` (`/^\/([a-z][a-z0-9-]*)?$/`). Fetches suggestions via `slashCommands.fetchCommands()` with 150ms debounce and `AbortController` cleanup. Keyboard navigation: `↑↓` cycle, `Tab`/`Enter` accept, `Esc` dismiss.

## Usage Example

```typescript
import { useRef } from "react"
import { ChatInput } from "@/components/chat/chat-input"
import type { ChatInputRef } from "@/components/chat/types"

export function MyChatPanel() {
  const inputRef = useRef<ChatInputRef>(null)

  const handleSend = async (message: string) => {
    const ok = await sendToAPI(message)
    if (!ok) return false // keeps draft on failure
  }

  // Pre-fill a slash command from a quick-action chip
  const handleQuickAction = () => {
    inputRef.current?.setValue("/diagnose ")
  }

  return (
    <ChatInput
      ref={inputRef}
      onSend={handleSend}
      onStop={() => abortStream()}
      sending={isStreaming}
      awaitingResponse={isWaiting}
      slashCommands={{ fetchCommands: fetchSlashCommands }}
      autoFocus
    />
  )
}
```
