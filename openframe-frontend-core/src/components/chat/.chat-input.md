A React component that provides a chat input interface with auto-resizing textarea, send functionality, and keyboard shortcuts. Features Enter-to-send behavior, loading states, and optional avatar spacing alignment.

## Key Components

- **ChatInput** - Main forwardRef component with textarea and send button
- **ChatInputProps** - Interface extending TextareaHTMLAttributes with chat-specific props:
  - `onSend` - Callback function triggered when message is sent
  - `sending` - Boolean to indicate loading state
  - `reserveAvatarOffset` - Adds spacing for avatar alignment
  - `disabled` - Disables input functionality

## Usage Example

```typescript
import { ChatInput } from './chat-input'

function ChatInterface() {
  const [isSending, setIsSending] = useState(false)
  
  const handleSend = async (message: string) => {
    setIsSending(true)
    try {
      await sendMessage(message)
    } finally {
      setIsSending(false)
    }
  }

  return (
    <ChatInput
      onSend={handleSend}
      sending={isSending}
      placeholder="Type your message..."
      reserveAvatarOffset={true}
    />
  )
}
```

The component automatically handles Enter key submission (Shift+Enter for line breaks), textarea auto-resizing, and form state management. The send button is disabled when empty, sending, or disabled.
