A React component that provides an auto-resizing textarea input with send functionality, commonly used for chat interfaces. It features keyboard shortcuts, loading states, and optional avatar spacing alignment.

## Key Components

- **ChatInput** - Main forwardRef component with textarea and send button
- **ChatInputProps** - Interface extending textarea props with chat-specific options
- **Auto-resize functionality** - Dynamically adjusts textarea height based on content
- **Send button** - Integrated button with disabled states and loading feedback
- **Avatar offset** - Optional spacing to align with chat message avatars

## Usage Example

```typescript
import { ChatInput } from './chat-input'
import { useRef } from 'react'

function ChatInterface() {
  const inputRef = useRef<HTMLTextAreaElement>(null)

  const handleSend = (message: string) => {
    console.log('Sending message:', message)
    // Process the message
  }

  return (
    <ChatInput
      ref={inputRef}
      onSend={handleSend}
      sending={false}
      placeholder="Type your message..."
      reserveAvatarOffset={true}
      disabled={false}
    />
  )
}
```

The component handles Enter key submission (Shift+Enter for new lines), automatically clears the input after sending, and maintains proper disabled states during message transmission. The textarea grows vertically with content up to a maximum height of 160px.
