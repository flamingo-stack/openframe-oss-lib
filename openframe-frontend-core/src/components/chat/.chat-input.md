A React component that provides an auto-resizing text input with send functionality for chat interfaces. It supports multiple states including active input, loading, and awaiting response with proper keyboard handling and accessibility features.

## Key Components

- **ChatInput** - Main forwardRef component with auto-expanding textarea
- **handleSubmit** - Processes message sending and resets input state
- **handleKeyDown** - Handles Enter key submission (Shift+Enter for new lines)
- **handleChange** - Auto-resizes textarea based on content height
- **Awaiting Response UI** - Special state showing typing indicator and status message

## Usage Example

```typescript
import { ChatInput } from './chat-input'
import { useRef } from 'react'

function ChatInterface() {
  const inputRef = useRef<HTMLTextAreaElement>(null)

  const handleSendMessage = (message: string) => {
    console.log('Sending:', message)
    // Process message sending logic
  }

  return (
    <ChatInput
      ref={inputRef}
      onSend={handleSendMessage}
      sending={false}
      awaitingResponse={false}
      placeholder="Type your message..."
      autoFocus={true}
      reserveAvatarOffset={true}
    />
  )
}
```

The component automatically handles textarea resizing (20px to 160px max height), disables input during sending states, and provides visual feedback through the typing indicator when awaiting responses. It uses Tailwind CSS with ODS (OpenFrame Design System) tokens for consistent styling.
