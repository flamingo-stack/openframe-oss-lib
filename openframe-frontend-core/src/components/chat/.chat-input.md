A React chat input component that provides an auto-resizing textarea with send functionality and loading states. It supports both standard input mode and a waiting state display for technician responses.

## Key Components

- **ChatInput** - Main forwardRef component with auto-resizing textarea and send button
- **ChatInputProps** - Interface extending textarea props with chat-specific options
- **Auto-resize logic** - Dynamic height adjustment based on content
- **Keyboard handling** - Enter to send (Shift+Enter for new line)
- **State management** - Handles sending, awaiting response, and disabled states
- **Layout options** - Configurable avatar offset spacing

## Usage Example

```typescript
import { ChatInput } from './components/chat/chat-input'
import { useRef } from 'react'

function ChatInterface() {
  const inputRef = useRef<HTMLTextAreaElement>(null)

  const handleSend = (message: string) => {
    console.log('Sending message:', message)
    // Handle message sending logic
  }

  return (
    <div className="chat-container">
      <ChatInput
        ref={inputRef}
        onSend={handleSend}
        sending={false}
        awaitingResponse={false}
        placeholder="Type your message..."
        reserveAvatarOffset={true}
        disabled={false}
      />
    </div>
  )
}
```

The component automatically handles form submission on Enter key press, prevents submission while sending or disabled, and provides visual feedback for different states including a typing indicator during awaiting response mode.
