A central export module that provides a unified interface for accessing all chat-related UI components in the application.

## Key Components

- **approval-request-message** - Component for displaying approval request messages
- **chat-container** - Main container component that wraps the entire chat interface
- **chat-input** - Input field component for user message entry
- **chat-message** - Basic message display component
- **chat-message-enhanced** - Advanced message component with additional features
- **chat-message-list** - Container component for displaying multiple chat messages
- **chat-quick-action** - Quick action buttons for common chat operations
- **chat-typing-indicator** - Visual indicator showing when someone is typing
- **tool-execution-display** - Component for showing tool execution status and results
- **model-display** - Component for displaying AI model information

## Usage Example

```typescript
// Import individual components
import { ChatContainer, ChatInput, ChatMessage } from './chat'

// Or import specific components as needed
import { ChatMessageEnhanced, ToolExecutionDisplay } from './chat'

// Use in a React component
function App() {
  return (
    <ChatContainer>
      <ChatMessageList>
        <ChatMessage content="Hello!" />
        <ChatMessageEnhanced content="Enhanced message" />
      </ChatMessageList>
      <ChatInput onSend={handleSend} />
    </ChatContainer>
  )
}
```

This barrel export pattern enables clean imports and better organization of chat-related components throughout the application.
