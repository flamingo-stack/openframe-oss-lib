This is an index file that serves as the main entry point for a chat component library, re-exporting all chat-related React components for easy importing.

## Key Components

- **chat-container** - Main wrapper component for the chat interface
- **chat-input** - Input field component for typing messages
- **chat-message** - Basic message display component
- **chat-message-enhanced** - Advanced message component with additional features
- **chat-message-list** - Container for displaying multiple chat messages
- **chat-quick-action** - Quick action buttons for common chat operations
- **chat-typing-indicator** - Visual indicator showing when someone is typing
- **tool-execution-display** - Component for showing tool/command execution results
- **model-display** - Component for displaying AI model information

## Usage Example

```typescript
// Import individual components
import { ChatContainer, ChatInput, ChatMessage } from './chat-components'

// Or import multiple components at once
import { 
  ChatContainer, 
  ChatMessageList, 
  ChatTypingIndicator 
} from './chat-components'

// Use in a React component
function MyChat() {
  return (
    <ChatContainer>
      <ChatMessageList />
      <ChatTypingIndicator />
      <ChatInput />
    </ChatContainer>
  )
}
```
