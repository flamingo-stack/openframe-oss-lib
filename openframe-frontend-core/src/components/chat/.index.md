A comprehensive export file that serves as the main entry point for the Flamingo chat system components and utilities.

## Key Components

This index file provides a centralized export system for the complete chat module, organizing exports into four main categories:

- **Chat Components**: Core UI components including `chat-container`, `chat-input`, `chat-message-enhanced`, `chat-message-list`, `chat-quick-action`, `chat-typing-indicator`, `tool-execution-display`, `model-display`, `chat-sidebar`, and specialized message components like `approval-request-message` and `error-message-display`
- **Types**: TypeScript type definitions and interfaces for chat-related data structures
- **Hooks**: Custom React hooks for chat functionality and state management
- **Utils**: Utility functions and helpers for chat operations

## Usage Example

```typescript
import {
  ChatContainer,
  ChatInput,
  ChatMessageList,
  useChatState,
  ChatMessage
} from '@/components/chat'

// Use imported components in your application
function MyApp() {
  const { messages, sendMessage } = useChatState()
  
  return (
    <ChatContainer>
      <ChatMessageList messages={messages} />
      <ChatInput onSend={sendMessage} />
    </ChatContainer>
  )
}
```

The `'use client'` directive indicates this module is designed for client-side rendering in Next.js applications, making all exported components available for browser-side interactivity.
