A central export module for all chat-related components, types, hooks, and utilities in the Flamingo platform's messaging system. This barrel export simplifies imports and provides a clean API surface for the chat functionality.

## Key Components

**Component Exports:**
- `approval-request-message` - Handles approval workflow messages
- `chat-container` - Main chat interface wrapper
- `chat-input` - Message input field with enhanced features
- `chat-message-enhanced` - Rich message display with formatting
- `chat-message-list` - Scrollable message history container
- `chat-quick-action` - Quick response buttons and actions
- `chat-typing-indicator` - Real-time typing status display
- `tool-execution-display` - Shows AI tool execution progress
- `model-display` - Displays active AI model information
- `chat-sidebar` - Chat navigation and settings panel

**Additional Exports:**
- `types` - TypeScript interfaces and type definitions
- `hooks` - React hooks for chat state management
- `utils` - Chat utility functions and helpers

## Usage Example

```typescript
// Import specific chat components
import { 
  ChatContainer, 
  ChatInput, 
  ChatMessageList,
  ChatTypingIndicator 
} from '@/components/chat'

// Import chat types and hooks
import { 
  ChatMessage, 
  useChat, 
  useChatHistory 
} from '@/components/chat'

// Use in a chat interface
export function ChatInterface() {
  const { messages, sendMessage } = useChat()
  
  return (
    <ChatContainer>
      <ChatMessageList messages={messages} />
      <ChatTypingIndicator />
      <ChatInput onSendMessage={sendMessage} />
    </ChatContainer>
  )
}
```
