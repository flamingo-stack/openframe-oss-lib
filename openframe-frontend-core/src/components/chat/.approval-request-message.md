A React component that displays approval requests with interactive approve/reject functionality for commands or actions requiring user authorization.

## Key Components

- **ApprovalRequestMessage** - Main forwarded ref component that renders approval cards with command display and action buttons
- **Status States** - Handles three states: 'pending' (interactive), 'approved', and 'rejected' (read-only with status tags)
- **Action Handlers** - `handleApprove` and `handleReject` functions with loading states and error handling
- **Command Display** - Formatted code block showing the command with optional icon and explanation text

## Usage Example

```typescript
import { ApprovalRequestMessage } from './approval-request-message'
import { Terminal } from 'lucide-react'

function MyComponent() {
  const approvalData = {
    requestId: 'req-123',
    command: 'sudo rm -rf /tmp/cache',
    explanation: 'This will clear the temporary cache directory',
    icon: <Terminal className="w-4 h-4" />
  }

  return (
    <ApprovalRequestMessage
      data={approvalData}
      status="pending"
      onApprove={async (requestId) => {
        await api.approveRequest(requestId)
      }}
      onReject={async (requestId) => {
        await api.rejectRequest(requestId)
      }}
    />
  )
}
```

The component automatically switches between interactive mode (with approve/reject buttons) and read-only mode (with status indicators) based on the approval status.
