Renders the Chat Archive page within the Mingo chat panel, displaying a date-grouped list of archived dialogs with responsive headers for mobile and desktop layouts.

## Key Components

**`ChatArchivePageProps`** — Interface defining the component's props:
- `dialogs` — Read-only array of `DialogItem` entries (grouped by Today / Yesterday / Older)
- `onSelectDialog(id)` — Callback to open a selected archived dialog
- `onBack` — Returns to the previous list view
- `onClose` — Closes the entire chat panel
- `isLoading` — Drives skeleton/loading state
- `hasMore` — Indicates additional pages available
- `onLoadMore` — Fetches the next page of archived dialogs

**`ChatArchivePage`** — Main exported component (Figma node `7361:427312`) rendering three possible content states:
1. **Populated** — Delegates to `MingoChatHistory` (row action menus are omitted)
2. **Loading skeleton** — Six animated `h-12` pulse rows to prevent layout shift
3. **Empty state** — Centred `BoxArchiveIcon` with instructional copy

## Usage Example

```typescript
import { ChatArchivePage } from './chat-archive-page'

function ChatPanel() {
  return (
    <ChatArchivePage
      dialogs={archivedDialogs}
      onSelectDialog={(id) => openDialog(id)}
      onBack={() => setView('list')}
      onClose={() => setPanelOpen(false)}
      isLoading={isFetching}
      hasMore={hasNextPage}
      onLoadMore={fetchNextPage}
    />
  )
}
```
