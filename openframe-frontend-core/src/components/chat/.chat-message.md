A React component for displaying individual chat messages with support for user/assistant/error roles, avatars, typing indicators, and timestamps.

## Key Components

- **ChatMessage**: Main component that renders a chat message bubble with configurable styling based on role
- **ChatMessageProps**: Interface defining props including `role`, `content`, `name`, `avatar`, `isTyping`, `timestamp`, and `showAvatar`
- **getAvatarProps()**: Internal helper function that generates avatar properties with fallback initials

## Usage Example

```typescript
import { ChatMessage } from './chat-message'

// Basic usage
<ChatMessage 
  role="assistant" 
  content="Hello! How can I help you today?" 
/>

// With avatar and timestamp
<ChatMessage
  role="user"
  content="What's the weather like?"
  name="John Doe"
  avatar="/user-avatar.jpg"
  timestamp={new Date()}
/>

// Typing indicator
<ChatMessage
  role="assistant"
  content=""
  isTyping={true}
/>

// Error message
<ChatMessage
  role="error"
  content="Failed to send message"
  showAvatar={false}
/>
```

The component automatically styles messages differently based on role - user messages appear right-aligned without background, while assistant messages have a card background and gradient avatar.
