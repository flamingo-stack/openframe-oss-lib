A React component that renders individual chat messages with support for different roles, avatars, typing indicators, and timestamps.

## Key Components

- **ChatMessageProps** - Interface defining component props including role, content, avatar, and display options
- **ChatMessage** - Main component that renders a chat message with flexible styling based on role (user/assistant/error)
- **getAvatarProps()** - Internal function that generates avatar properties with fallback initials and role-specific styling

## Features

- **Role-based styling** - Different visual treatments for user, assistant, and error messages
- **Avatar support** - Optional avatars with fallback to initials, user avatars are hidden by default
- **Typing indicator** - Shows animated typing indicator when `isTyping` is true
- **Timestamps** - Optional timestamp display with localized formatting
- **Error handling** - Special styling and text color for error messages
- **Responsive layout** - Flexbox-based layout with proper spacing and alignment

## Usage Example

```typescript
import { ChatMessage } from "./chat-message"

function ChatExample() {
  return (
    <div className="space-y-4">
      {/* User message */}
      <ChatMessage
        role="user"
        content="Hello, how can you help me today?"
        name="John"
        timestamp={new Date()}
      />
      
      {/* Assistant typing */}
      <ChatMessage
        role="assistant"
        content=""
        isTyping={true}
        avatar="/assistant-avatar.png"
      />
      
      {/* Error message */}
      <ChatMessage
        role="error"
        content="Failed to connect to server"
        showAvatar={false}
      />
    </div>
  )
}
```
