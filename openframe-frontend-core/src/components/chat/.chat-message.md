A React component that renders individual chat messages with support for different user roles, avatars, and AI assistant types (Mingo and Fae) used in Flamingo's AI chat interface.

## Key Components

- **`ChatMessageProps`** - Interface defining message properties including role, content, avatar, typing state, and assistant type
- **`ChatMessage`** - Forwardable React component that renders formatted chat messages
- **Role-based styling** - Different visual treatments for user, assistant, and error messages
- **Avatar system** - Displays user/assistant avatars with gradient backgrounds for AI assistants
- **Typing indicator** - Shows animated typing state when message is being composed

## Usage Example

```typescript
import { ChatMessage } from './chat-message'

// User message
<ChatMessage
  role="user"
  content="How can I configure OpenFrame?"
  name="John Doe"
  timestamp={new Date()}
/>

// Assistant response with Mingo AI
<ChatMessage
  role="assistant"
  content="I'll help you configure OpenFrame..."
  assistantType="mingo"
  isTyping={false}
  showAvatar={true}
/>

// Error message
<ChatMessage
  role="error"
  content="Failed to connect to the server"
/>

// Typing indicator
<ChatMessage
  role="assistant"
  content=""
  assistantType="fae"
  isTyping={true}
/>
```

The component automatically handles avatar colors (cyan for Mingo, pink for Fae), message alignment, and responsive styling with proper spacing and typography for the OpenFrame chat interface.
