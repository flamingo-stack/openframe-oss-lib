A responsive quick-action chip row component that dynamically collapses overflow chips into a "⋯" dropdown menu based on available container width.

## Key Components

**`ChatQuickActionRow`** — Main export. Renders a horizontal row of action chips, measuring available width to determine how many chips fit inline. Overflow chips collapse into an `ActionsMenuDropdown`. Accepts an optional `leading` slot that always stays pinned and never collapses.

**`ChipButton`** — Internal wrapper that renders a `Tag` chip as a focusable button with accessible focus ring styling.

**`QuickActionChip`** — Interface defining a single chip with `id`, `label`, optional `icon`, `variant` (`'primary'` for accent/yellow or `'outline'` for bordered), and `onSelect` callback.

**`ChatQuickActionRowProps`** — Props interface accepting `leading` (pinned ReactNode), `chips` (collapsible array), and `className`.

## Key Behaviors

- Uses `useAutoLimitTags` hook with off-screen measurement copies (via `measureRef`) to compute how many chips fit before overflow
- Hidden chips move into an `ActionsMenuDropdown` triggered by a `⋯` (`Ellipsis01Icon`) button
- Returns `null` when both `leading` and `chips` are empty

## Usage Example

```typescript
import { ChatQuickActionRow } from './chat-quick-action-row'
import { MessageIcon, SearchIcon } from '../icons'

<ChatQuickActionRow
  leading={<PinnedActionButton />}
  chips={[
    { id: 'new-chat', label: 'New Chat', icon: <MessageIcon />, onSelect: () => startChat() },
    { id: 'search', label: 'Search KB', icon: <SearchIcon />, variant: 'primary', onSelect: () => openSearch() },
    { id: 'escalate', label: 'Escalate Ticket', onSelect: () => escalate() },
  ]}
  className="px-2"
/>
```
