A responsive single-line chip row for chat quick-actions that automatically collapses overflow chips into a "⋯" dropdown menu based on available container width.

## Key Components

**`ChatQuickActionRow`** — Main export. Renders a row with an optional pinned `leading` slot and a set of collapsible `chips`. Uses `useAutoLimitTags` to measure available width off-screen, rendering as many chips inline as fit and moving the rest into an `ActionsMenuDropdown`.

**`ChipButton`** — Internal button wrapper around the `Tag` UI component, supporting `'primary'` (amber accent) and `'outline'` (bordered) variants with keyboard focus ring.

**Interfaces:**
- `QuickActionChip` — Shape of each chip: `id`, `label`, optional `icon`, `variant`, and `onSelect` callback.
- `ChatQuickActionRowProps` — Row props: optional `leading` slot, `chips` array, and `className`.

## Usage Example

```typescript
import { ChatQuickActionRow, QuickActionChip } from './chat-quick-action-row'
import { SparklesIcon } from '../icons'

const chips: QuickActionChip[] = [
  { id: 'new-ticket', label: 'New Ticket', onSelect: () => createTicket() },
  { id: 'run-script', label: 'Run Script', onSelect: () => runScript() },
  { id: 'view-assets', label: 'View Assets', onSelect: () => viewAssets() },
]

<ChatQuickActionRow
  leading={
    <button onClick={startGuide}>
      <Tag variant="primary" label="Start Guide Chat" icon={<SparklesIcon />} />
    </button>
  }
  chips={chips}
  className="px-3 py-2"
/>
```

The `leading` slot (e.g. Mingo's primary CTA) is always visible. Chips that don't fit the container width collapse into the `⋯` overflow menu automatically.
