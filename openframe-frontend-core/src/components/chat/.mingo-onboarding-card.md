A single onboarding/slash-command row card used in Mingo's chat empty-state list. Renders as an interactive `<button>` when a click handler is provided (and no actions exist), or as a static `<div>` with inline action buttons otherwise.

## Key Components

**`MingoOnboardingCard`** — Main component (Figma node `7363:205939`)
Displays an icon, title, optional slash-command label, description, and an optional row of action buttons. Stacks cleanly in a column container using `border-b` dividers, with `last:border-b-0` to trim the final card.

**`MingoOnboardingCardProps`** — Component props interface
| Prop | Type | Description |
|------|------|-------------|
| `icon` | `ReactNode` | 16×16 leading icon, inherits `ods-text-secondary` color |
| `title` | `ReactNode` | Card title (DM Sans Medium 14px, `ods-text-primary`) |
| `slashCommand` | `ReactNode` | Right-aligned label (e.g. `/roadmap`) |
| `description` | `ReactNode` | Optional subtitle below the title row |
| `actions` | `ReadonlyArray<MingoOnboardingCardAction>` | Outline chip buttons; disables card-level `onClick` |
| `onClick` | `() => void` | Makes card interactive when no `actions` present |
| `className` | `string` | Appended to root element |

**`MingoOnboardingCardAction`** — Action button descriptor
Each action exposes `id`, `label`, and `onClick` (receives `MouseEvent`); click events are auto-stopped from propagating to the card.

## Usage Example

```typescript
import { MingoOnboardingCard } from './mingo-onboarding-card'
import { SearchIcon } from '../icons'

// Interactive card (no actions)
<MingoOnboardingCard
  icon={<SearchIcon />}
  title="Search knowledge base"
  slashCommand="/search"
  description="Find articles, tickets, and docs"
  onClick={() => console.log('card clicked')}
/>

// Static card with inline action buttons
<MingoOnboardingCard
  title="Recent tickets"
  slashCommand="/roadmap"
  actions={[
    { id: 'recent', label: 'Recent', onClick: () => fetchRecent() },
    { id: 'search', label: 'Search', onClick: (e) => { e.stopPropagation(); openSearch() } },
  ]}
/>
```
