Defines and exports a single shared Tailwind CSS class string for chip-density action buttons used across the device density UI components.

## Key Components

- **`CHIP_ACTION_BUTTON_CLASS`** — A constant string of Tailwind utility classes providing consistent styling for compact action buttons (e.g., Recent, Search, Find, Display, Submit, Cancel) rendered inside chip card tertiary action rows and the `<DisplayChipAction>` popover.

## Usage Example

```typescript
import { CHIP_ACTION_BUTTON_CLASS } from './chip-action-class'

// Apply to a button element
<button className={CHIP_ACTION_BUTTON_CLASS}>
  Recent
</button>

// Extend with additional classes when needed
<button className={`${CHIP_ACTION_BUTTON_CLASS} my-extra-class`}>
  Search
</button>
```

## Design Notes

| Property | Value | Intent |
|----------|-------|--------|
| Font size | `text-[11px]` | Sub-UI-Kit density, fits chip action rows |
| Padding | `px-2 py-1` | Tight, below standard `Button size="small"` |
| Colors | `ods-text-secondary` / hover `ods-text-primary` | ODS token-based theming |
| Background | `bg-transparent` / hover `ods-card-hover` | Subtle interactive feedback |
| Focus ring | `focus-visible:ring-ods-focus` | Accessible keyboard navigation |

Centralizing this constant prevents Tailwind class chain drift across the empty-state chip and `<DisplayChipAction>` popover, ensuring byte-equivalent styling at both call sites.
