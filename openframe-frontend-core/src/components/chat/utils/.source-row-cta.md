Resolves the click-action contract (icon, href, targetPlatform, askability, chatRef) for source rows across chat-adjacent surfaces like chips, inline cards, and search results.

## Key Components

**`resolveSourceRowCTA(row, ctx)`** — Core pure function. Accepts a `SourceRowInput` and `SourceRowContext`, returns a `SourceRowCTA` with resolved icon, href, targetPlatform, askable flag, and chatRef.

**`sourceRowCtxFromRuntime(runtime, surface)`** — Factory that builds a `SourceRowContext` from a `ChatRuntime` instance and surface config, centralizing wiring of `currentPlatform`, `composeContentUrl`, and `docPlatformTargets`.

**`SourceRowInput`** — Row data shape: `sourceRepo`, `documentType`, `id`, `title`, `externalUrl`, `targetPlatform`, `path`.

**`SourceRowContext`** — Host-injected dependencies: `baseRoute`, `chipBasePlatform`, `currentPlatform`, `tableIdForDocumentType`, `composeContentUrl`, `docPlatformTargets`.

**`SourceRowCTA`** — Output contract: `icon`, `iconLabel`, `href` (safe, nullable), `targetPlatform`, `askable`, `chatRef`.

**`DOC_TABLE_TYPES`** — Constant tuple `['markdown', 'data_room_doc']` identifying doc-table rows that use path-based navigation instead of `externalUrl`.

## Usage Example

```typescript
import { resolveSourceRowCTA, sourceRowCtxFromRuntime } from './source-row-cta'

const ctx = sourceRowCtxFromRuntime(runtime, {
  baseRoute: '/knowledge-base',
  chipBasePlatform: 'flamingo',
})

const cta = resolveSourceRowCTA(
  {
    sourceRepo: 'blog-posts',
    documentType: 'blog_post',
    id: 'abc-123',
    title: 'Getting Started',
    externalUrl: 'https://example.com/posts/abc-123',
  },
  ctx,
)

// cta.href      → sanitized destination URL
// cta.icon      → React icon component
// cta.askable   → true (has id + documentType)
// cta.chatRef   → pre-built ChatRef for Ask handler
```

**href resolution priority:**
1. Doc-table row (no `externalUrl`) → `docPlatformTargets` → `chipBasePlatform` → `baseRoute` → `null`
2. Entity row with `composeContentUrl` + `externalUrl` → unified content-href seam
3. Entity row without composer → `externalUrl` verbatim (legacy)
