Utilities for parsing NATS message chunks into structured actions for AI tool execution workflows. This module provides functions to transform raw chunk data into typed action objects and utilities for identifying different chunk types.

## Key Components

- **`parseChunkToAction`** - Main parser function that converts raw NATS chunks into structured `ParsedChunkAction` objects
- **`isControlChunk`** - Identifies message start/end control chunks
- **`isErrorChunk`** - Detects error message chunks
- **`isMetadataChunk`** - Identifies AI model metadata chunks
- **`extractTextFromChunk`** - Extracts text content from text-type chunks

## Usage Example

```typescript
import { parseChunkToAction, isControlChunk, extractTextFromChunk } from './chunk-parser'

// Parse different chunk types
const textChunk = { type: 'TEXT', text: 'Hello world' }
const action = parseChunkToAction(textChunk)
// Returns: { action: 'text', text: 'Hello world' }

// Check for control messages
const startChunk = { type: 'MESSAGE_START' }
if (isControlChunk(startChunk)) {
  console.log('Processing message start')
}

// Extract text content
const text = extractTextFromChunk(textChunk)
// Returns: 'Hello world'

// Handle tool execution
const toolChunk = {
  type: 'EXECUTING_TOOL',
  integratedToolType: 'filesystem',
  toolFunction: 'readFile',
  parameters: { path: '/tmp/file.txt' }
}
const toolAction = parseChunkToAction(toolChunk)
// Returns structured tool execution action
```
