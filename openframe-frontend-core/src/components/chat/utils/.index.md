Centralized export module for chat message processing utilities, providing parsing, accumulation, and state management functionality for streaming message data.

## Key Components

**Chunk Processing**
- `parseChunkToAction` - Converts raw chunks into action objects
- `isControlChunk`, `isErrorChunk`, `isMetadataChunk` - Type guard functions for chunk classification
- `extractTextFromChunk` - Extracts text content from message chunks

**Message Accumulation**
- `MessageSegmentAccumulator` - Class for accumulating streaming message segments
- `createMessageSegmentAccumulator` - Factory function for creating accumulators
- `AccumulatorCallbacks` - Type definition for callback functions

**Historical Processing**
- `processHistoricalMessages` - Processes previously received messages
- `extractErrorMessages` - Extracts error information from message history
- `processHistoricalMessagesWithErrors` - Combined processing with error handling

**State Management**
- `extractIncompleteMessageState` - Retrieves state of partially received messages
