Barrel/index file for the `message-processing` utilities package, re-exporting all public APIs from the module's sub-files.

## Key Components

**Chunk Processing**
- `parseChunkToAction`, `isControlChunk`, `isErrorChunk`, `isMetadataChunk`, `extractTextFromChunk` — stream chunk classification and parsing

**Message Accumulation**
- `MessageSegmentAccumulator`, `createMessageSegmentAccumulator` — stateful accumulator for assembling streaming message segments

**Historical Messages**
- `processHistoricalMessages`, `extractErrorMessages`, `processHistoricalMessagesWithErrors` — replay and error extraction from past messages

**Navigation & Routing**
- `executeNavigation`, `executeNavigationImperative` — unified navigation primitive handling new-tab/same-tab, internal/external, embed/host, and fallback chains
- `handleChatNavClick`, `computeIsNewTab`, `buildAnchorProps`, `decideNewTab` — click-event helpers and anchor attribute builders

**Chat Proxy Auth**
- `getChatProxyAuth`, `setChatProxyAuth`, `clearChatProxyAuth`, `applyProxyAuth`, `chatAuthedFetch` — localStorage adapter and authenticated fetch wrapper

**Content Helpers**
- `flattenAssistantContent`, `parseScrollAnchor`, `buildAutoContinuationDirective` — pure content transformation utilities

**Slash Commands**
- `parseWireCommandOverride`, `sanitizeTitleForChat`, `buildDiscussAddendum` — command dispatch utilities

**Attachments**
- `formatChatAttachmentMarkdownForBubble`, `stripChatAttachmentMarkdown`, `buildChatAttachmentViewUrl` — chat attachment markdown formatting and parsing

**UI Primitives**
- `ICON_REGISTRY`, `getIconComponent`, `normalizeIconKey` — single source of truth for icon-name → React component lookup
- `COMPACT_CARD_*` constants, `safeHref` — compact card CSS class tokens and safe URL guard

## Usage Example

import {
  executeNavigation,
  parseChunkToAction,
  getChatProxyAuth,
  MessageSegmentAccumulator,
} from '@flamingo/message-processing'

// Navigate with unified primitive
executeNavigation({ href: '/tickets/123', event: clickEvent })

// Parse a streaming chunk
const action = parseChunkToAction(chunk)

// Accumulate streamed segments
const accumulator = new MessageSegmentAccumulator(callbacks)
accumulator.push(chunk)
