Barrel (re-export) index for the message processing utilities package, consolidating all chat-related helpers into a single import surface.

## Key Components

### Chunk Processing
- `parseChunkToAction`, `isControlChunk`, `isErrorChunk`, `isMetadataChunk`, `extractTextFromChunk` — parse and classify streaming response chunks

### Message Accumulation
- `MessageSegmentAccumulator` / `createMessageSegmentAccumulator` — stateful accumulator for assembling streamed message segments with lifecycle callbacks

### Historical Message Processing
- `processHistoricalMessages`, `extractErrorMessages`, `processHistoricalMessagesWithErrors` — utilities for replaying and transforming stored chat history

### Navigation & Routing
- `executeNavigation` / `executeNavigationImperative` — **single source of truth** for all chat navigation (new-tab/same-tab, internal/external, embed/host, runtime→router→window fallback)
- `handleChatNavClick`, `decideNewTab`, `computeIsNewTab`, `buildAnchorProps` — surface-level wrappers and anchor attribute builders
- `isCrossOriginUrl`, `stripSameOriginToPath`, `resolveExternalNavigation` — URL classification helpers

### Chat Proxy Auth
- `getChatProxyAuth`, `setChatProxyAuth`, `clearChatProxyAuth`, `applyProxyAuth` — localStorage adapter for proxy credentials
- `chatAuthedFetch` — fetch wrapper that injects proxy auth headers

### Content & Formatting
- `flattenAssistantContent`, `escapeMarkdownInline`, `formatChatAttachmentMarkdownForBubble`, `stripChatAttachmentMarkdown` — content normalization and attachment markdown formatting
- `buildAutoContinuationDirective`, `parseScrollAnchor` — chat wire protocol helpers

### UI / Style Registries
- `ICON_REGISTRY`, `getIconComponent`, `normalizeIconKey` — single source of truth for icon-name → React component lookup
- `COMPACT_CARD_*` constants, `safeHref` — shared CSS class tokens for compact card surfaces
- `getStatusColorScheme` — agent status → color scheme mapper

## Usage Example

```typescript
import {
  executeNavigation,
  MessageSegmentAccumulator,
  getIconComponent,
  chatAuthedFetch,
} from '@/lib/message-processing'

// Navigate from a chat chip
executeNavigation({
  href: '/tickets/123',
  event: clickEvent,
  runtimeMode: 'hub',
})

// Stream and accumulate a response
const accumulator = new MessageSegmentAccumulator({
  onSegmentComplete: (segment) => console.log(segment),
})

// Resolve an icon dynamically
const Icon = getIconComponent('check-circle')
```
