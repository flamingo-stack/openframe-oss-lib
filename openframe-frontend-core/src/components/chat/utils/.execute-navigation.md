Unified navigation execution primitive that handles all navigation actions across the library — resolving new-tab vs. same-tab behavior, internal vs. external routing, and runtime/router/window fallback chains in a single place.

## Key Components

### Interfaces

- **`NavClickEvent`** — Minimal structural mouse-event surface; allows chip buttons and tiles to invoke navigation without casting `React.MouseEvent`. Supports `button`, `metaKey`, `ctrlKey`, `shiftKey`, `altKey`.
- **`ExecuteNavigationClickArgs`** — Arguments for click-handler form: `event`, `runtime`, `href`, optional `path`, `targetPlatform`, and `fallbackNavigate`.
- **`ExecuteNavigationImperativeArgs`** — Arguments for imperative form (no event object): `runtime`, `href`, optional `targetPlatform`, and `fallbackNavigate`.

### Functions

- **`runNavigation`** *(internal)* — Core decision engine. Calls `computeIsNewTab`; if true, uses `runtime.navigation.openExternal` or `window.open`; if false, calls `runtime.navigation.navigate` and falls back to `fallbackNavigate(path)` or `window.location.assign`.
- **`executeNavigation`** — Click-handler form. Guards against empty `href`, modifier keys, and non-primary buttons before delegating to `runNavigation`. Returns `true` if it handled the event, `false` to let the browser handle it natively.
- **`executeNavigationImperative`** — Imperative form for buttons, list-row handlers, and post-action redirects. Skips all event/modifier logic and calls `runNavigation` directly.

## Usage Example

```typescript
// Click-handler form (anchor / chip / markdown link)
import { executeNavigation } from './execute-navigation'

function handleClick(event: React.MouseEvent) {
  const handled = executeNavigation({
    event,
    runtime,           // ChatRuntime | null
    href: 'https://example.com/tickets/42',
    path: '/tickets/42',
    targetPlatform: 'internal',
    fallbackNavigate: router.push,
  })
  if (!handled) {
    // Browser will handle natively (new tab via modifier key, etc.)
  }
}

// Imperative form (button action / post-form redirect)
import { executeNavigationImperative } from './execute-navigation'

executeNavigationImperative({
  runtime,
  href: '/dashboard',
  fallbackNavigate: router.push,
})
```
