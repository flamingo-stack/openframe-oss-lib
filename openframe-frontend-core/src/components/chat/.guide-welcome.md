A React client component that renders the Guide Mode chat empty state, featuring a centered greeting, scrollable onboarding content area with edge-fade affordances, and a pinned quick-action chip row with overflow handling.

## Key Components

**`GuideWelcome`** — Main export. Renders a vertically centered greeting (Mingo icon + title + subtitle), a shared scroll region for child content (slash-command list), and a pinned chip row above the composer.

**`GuideQuickAction`** — Interface for quick-action chips with `id`, `label`, and optional `prompt` fields.

**`GuideWelcomeProps`** — Full prop surface including `title`, `subtitle`, `quickActions`, `maxVisibleQuickActions` (default `3`), `onQuickAction`, `children`, and `className`.

**Scroll-fade logic** — A `ResizeObserver` + `onScroll` handler drives `scrollFade.top` / `scrollFade.bottom` state, toggling 48px gradient overlays at the scroll region edges.

**Overflow menu** — Quick actions beyond `maxVisibleQuickActions` collapse into an `ActionsMenuDropdown` triggered by an `Ellipsis01Icon` chip.

## Usage Example

```typescript
import { GuideWelcome, GuideQuickAction } from './guide-welcome'

const actions: GuideQuickAction[] = [
  { id: 'start', label: 'How to start', prompt: 'How do I get started with OpenFrame?' },
  { id: 'tickets', label: 'Open a ticket', prompt: 'Create a support ticket' },
  { id: 'docs', label: 'View docs' },
  { id: 'status', label: 'Check status' }, // overflows into "⋯" menu
]

export function GuideChatShell() {
  return (
    <GuideWelcome
      title="Guide Mode Chat"
      subtitle="Ask about OpenFrame docs or manage your tickets here."
      quickActions={actions}
      maxVisibleQuickActions={3}
      onQuickAction={(action) => seedComposer(action.prompt ?? action.label)}
    >
      <SlashCommandList />
    </GuideWelcome>
  )
}
```
