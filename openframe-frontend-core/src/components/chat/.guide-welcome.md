Guide-mode chat empty state component that renders a centered greeting, a scrollable content region with edge-fade affordances, and a pinned quick-action chip row above the composer.

## Key Components

**`GuideWelcome`** — Main component (Figma node `7532:328214`) mirroring `MingoWelcome` for guide-mode sessions. Features:
- Vertically centered `MingoIcon` greeting with configurable title/subtitle
- Shared scroll region for greeting + slash-command onboarding list (`children`), with top/bottom gradient fades that activate dynamically via `ResizeObserver`
- Pinned quick-action chip row: first `maxVisibleQuickActions` (default `3`) render as `Tag` chips; overflow actions collapse into an `ActionsMenuDropdown` triggered by an `⋯` chip

**`GuideQuickAction`** — Interface describing a chip: `id`, `label`, and optional `prompt` (seeded into the composer on click).

**`GuideWelcomeProps`** — Full props interface with overridable `title`, `subtitle`, `quickActions`, `maxVisibleQuickActions`, `onQuickAction`, `children`, and `className`.

## Usage Example

```typescript
import { GuideWelcome, GuideQuickAction } from './guide-welcome'

const actions: GuideQuickAction[] = [
  { id: 'tickets', label: 'View open tickets', prompt: 'Show my open tickets' },
  { id: 'docs',    label: 'Search docs',        prompt: 'Search OpenFrame docs for...' },
  { id: 'issues',  label: 'Known issues' },
  { id: 'start',   label: 'How to start' }, // overflows into ⋯ menu
]

function GuideChatPage() {
  return (
    <GuideWelcome
      title="Ask Mingo"
      subtitle="Temporary session — nothing is saved."
      quickActions={actions}
      maxVisibleQuickActions={3}
      onQuickAction={(action) => composer.seed(action.prompt ?? action.label)}
    >
      <SlashCommandList />
    </GuideWelcome>
  )
}
```
