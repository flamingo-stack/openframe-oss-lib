A React component that renders an interactive quick action button for chat interfaces, featuring hint animations and customizable click handlers.

## Key Components

- **`ChatQuickAction`** - The main component that renders a button with text and chevron icon
- **`ChatQuickActionProps`** - TypeScript interface defining component props including text, callbacks, and hint state
- **`handleClick`** - Internal click handler that manages hint interactions and action execution

## Usage Example

```typescript
import { ChatQuickAction } from './chat-quick-action'

function ChatInterface() {
  const [showHint, setShowHint] = React.useState(true)

  const handleQuickAction = (text: string) => {
    // Send the quick action text to chat
    sendMessage(text)
  }

  const stopHint = () => {
    setShowHint(false)
  }

  return (
    <div className="space-y-2">
      <ChatQuickAction
        text="Explain this code"
        onAction={handleQuickAction}
        isHintActive={showHint}
        onHintInteraction={stopHint}
      />
      <ChatQuickAction
        text="Generate documentation"
        onAction={handleQuickAction}
        disabled={false}
      />
    </div>
  )
}
```

The component supports hint animations via `isHintActive` prop, accessibility features including focus rings and keyboard navigation, and uses the design system's color tokens for consistent theming.
