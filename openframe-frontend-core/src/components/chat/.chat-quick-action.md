A client-side React component that renders clickable quick action buttons for chat interfaces, featuring hover effects, hint animations, and accessibility support.

## Key Components

- **`ChatQuickAction`** - Main component that renders an interactive button with text and chevron icon
- **`ChatQuickActionProps`** - Interface defining component props including text, callbacks, and hint functionality
- **Key props:**
  - `text` - Display text for the action
  - `onAction` - Callback fired with the text when clicked
  - `isHintActive` - Enables pulse animation to draw attention
  - `onHintInteraction` - Callback to stop hint animation on interaction

## Usage Example

```typescript
import { ChatQuickAction } from './chat-quick-action'

function ChatInterface() {
  const [showHint, setShowHint] = useState(true)

  return (
    <div className="space-y-2">
      <ChatQuickAction
        text="Explain this code"
        onAction={(text) => sendMessage(text)}
        isHintActive={showHint}
        onHintInteraction={() => setShowHint(false)}
      />
```
