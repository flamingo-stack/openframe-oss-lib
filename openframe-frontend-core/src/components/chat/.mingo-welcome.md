Renders the Mingo AI chat empty state for the OpenFrame panel, handling new-user greeting, feature grid, returning-user dialog history, loading/error states, and a dismissible onboarding promo notification.

## Key Components

### Exported Interfaces

| Interface | Purpose |
|-----------|---------|
| `MingoFeatureCard` | Single capability cell in the 2-up feature grid |
| `MingoWelcomePromo` | Config for the one-time "New to OpenFrame?" notification card |
| `MingoQuickAction` | Extra quick-action chip appended after "Start Guide Chat" |
| `MingoWelcomeProps` | Full props surface for the `MingoWelcome` component |

### `MingoWelcome` Component

The primary export. Manages five distinct render states:

- **New user** — greeting heading + `MingoIcon` + 2-up feature card grid
- **Returning user** — `dialogHistory` node replaces the greeting/grid
- **Loading** — renders `MingoChatHistorySkeleton` while the first history page fetches
- **Error** — shows a retry block when history fails to load and nothing is cached
- **Promo notification** — one-time "New to OpenFrame?" card, dismissible to `localStorage` or `sessionStorage`

## Usage Example

```typescript
import { MingoWelcome } from './mingo-welcome'
import { MingoChatHistory } from './mingo-chat-history'

// New user — show greeting + feature grid + guide promo
<MingoWelcome
  userName="Alex"
  onStartGuideChat={() => switchToGuideMode()}
/>

// Returning user — replace grid with history list
<MingoWelcome
  userName="Alex"
  hasExistingChats
  dialogHistory={<MingoChatHistory dialogs={dialogs} />}
  onStartGuideChat={() => switchToGuideMode()}
/>

// First load still in-flight
<MingoWelcome isLoadingHistory />

// Load failed — show retry button
<MingoWelcome
  loadError
  onRetry={() => refetchDialogs()}
/>

// Custom feature cards and promo
<MingoWelcome
  featureCards={[
    { id: 'custom', icon: <MyIcon size={16} />, text: 'Custom capability' },
  ]}
  promo={{ title: 'Welcome!', description: 'Get started here.' }}
  promoStorage="session"
/>
```

## Notes

- **SSR-safe** — storage is only accessed after mount via `useEffect`
- **Scroll fades** — `ResizeObserver` drives top/bottom gradient affordances on the greeting scroll region
- Promo dismissal is keyed by `promoStorageKey` (default: `mingo-welcome:promo-dismissed`) and persists across sessions unless `promoStorage="session"` is set
- All default copy (subtitle, feature cards, promo text) is OpenFrame-specific but fully overridable, keeping the component platform-agnostic
