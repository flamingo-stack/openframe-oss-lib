Manages dialog history state for the chat panel, encapsulating archive navigation, read-only archived conversation mode, and confirmation modal logic so the parent `EmbeddableChat` component remains a thin orchestration layer.

## Key Components

### `UseChatDialogManagerArgs`
Interface defining required and optional inputs:
- `dialogs` — active (non-archived) dialog list
- `activeDialogId`, `selectDialog`, `clearMessages` — conversation selection primitives
- `renameDialog`, `archiveDialog` — adapter-wired mutations
- `fetchArchivedDialogs?`, `unarchiveDialog?` — optional callbacks that gate archive/restore UI affordances

### `useChatDialogManager`
Primary hook returning grouped state and handlers:

| Group | Fields |
|---|---|
| Archive page | `archiveOpen`, `archivedDialogs`, `archivedCursor`, `archivedLoading`, `openArchive`, `closeArchive`, `loadArchivedPage`, `handleArchivedSelect` |
| Current-chat selection | `handleSelectDialog`, `isOpeningDialog` |
| Archived conversation | `isViewingArchived`, `handleBack`, `activeDialog` |
| Action modals | `renameTarget`, `archiveTarget`, `restoreTarget` + setters + confirm handlers |

## Usage Example

```typescript
const dialogManager = useChatDialogManager({
  dialogs,
  activeDialogId,
  selectDialog,
  clearMessages,
  renameDialog,
  archiveDialog,
  fetchArchivedDialogs,
  unarchiveDialog,
})

// Open the archive page
dialogManager.openArchive()

// Select a dialog from the current-chats list (immediate panel flip)
dialogManager.handleSelectDialog('dialog-id-123')

// Select from archive (enters read-only mode + disables composer)
dialogManager.handleArchivedSelect('archived-id-456')

// Trigger rename confirmation modal
dialogManager.setRenameTarget(someDialogItem)

// Back-chevron: returns to archive page if viewing archived, otherwise list
dialogManager.handleBack()
```

> **Note:** `isOpeningDialog` and `isViewingArchived` are synchronously set to avoid lag between panel flip and message load, ensuring immediate UI transitions regardless of data fetch timing.
