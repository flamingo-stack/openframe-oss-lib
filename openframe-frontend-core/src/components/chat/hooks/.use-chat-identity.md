Surfaces the `{ authTier, source, attachmentsEnabled, user }` identity bag for the current chat session, enabling UI gating on chat-side auth tiers without sending a message first.

## Key Components

**`ChatIdentityResponse`** — Wire-shape interface mirroring the hub's `/api/auth/identity` response. Contains `authTier` (`'session' | 'bearer-act-as' | 'anon'`), `source`, `attachmentsEnabled` (server-computed), and a nullable `user` object with name/email/avatar fields.

**`ChatIdentitySurface`** — Hook return type extending `ChatIdentityResponse` with an `isLoading` boolean for skeleton/spinner consumers. Never throws — errors and non-OK responses fall back silently to anon defaults.

**`useResolveChatIdentity(enabled)`** — Internal `useEffect` + `useState` resolver. Reads the identity URL from `useRequiredChatRuntime()` and refetches automatically when proxy credentials rotate (`proxyEmail` dep). Passing `enabled = false` short-circuits the fetch while still satisfying Rules of Hooks.

**`ChatIdentityProvider`** — Optional context provider. Lifts a single identity resolution over a React subtree so independent surfaces (e.g. a persistent chat widget + a tickets page in separate subtrees) share one `/identity` fetch instead of each self-fetching.

**`useChatIdentity()`** — Public hook. Reads from `ChatIdentityProvider` when present; falls back to self-resolution otherwise. No client-side cache by design — avoids stale identity across credential rotations.

## Usage Example

```typescript
// Standalone (self-fetching, hub default pattern)
function AttachmentButton() {
  const { attachmentsEnabled, isLoading } = useChatIdentity()
  if (isLoading) return <Skeleton />
  return attachmentsEnabled ? <UploadButton /> : null
}

// Shared resolution across independent subtrees (embed root)
function EmbedRoot() {
  return (
    <ChatRuntimeContext.Provider value={runtime}>
      <ChatIdentityProvider>
        <ChatWidget />
        <TicketsPage />
      </ChatIdentityProvider>
    </ChatRuntimeContext.Provider>
  )
}

// Gate on auth tier
function TicketCenterAuthed() {
  const { authTier, user } = useChatIdentity()
  if (authTier === 'anon' || !user?.email) return <SignInPrompt />
  return <TicketList userEmail={user.email} />
}
```
