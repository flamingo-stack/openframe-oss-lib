Responsive top-navigation header for the chat panel, rendering a back chevron + title and state-dependent right-hand actions (restore, ⋯ menu, archive entry, close) across list, active-conversation, archived-conversation, and guide views. Delegates to `ChatPanelHeaderMobile` on small screens and renders a fixed-height desktop bar on `md+`.

## Key Components

- **`ChatPanelHeader`** — Main export. Composes the full header bar with conditional rendering based on `showBack`, `isArchivedView`, and provided callbacks.
- **`ChatPanelHeaderProps`** — Interface defining all header configuration: navigation state, title, ARIA labels, and action callbacks (`onBack`, `onClose`, `onRestore`, `onRename`, `onArchive`, `onOpenArchive`).
- **`COMPACT_HEADER_BUTTON`** — Exported Tailwind class string for the borderless, hover-highlighted back-chevron button used on desktop.
- **`ChatPanelHeaderMobile`** — Separate mobile layout (hidden on `md+`), rendered via `flex md:hidden`.

## Conditional Right-Hand Actions

| Condition | Rendered action |
|---|---|
| `isArchivedView && onRestore` | Refresh/unarchive button |
| `showBack && !isArchivedView && menuItems.length > 0` | ⋯ dropdown (rename / archive) |
| `!showBack && onOpenArchive` | Clock/archive-entry button |
| Always | Close (✕) button |

## Usage Example

```typescript
<ChatPanelHeader
  showBack={true}
  title="Support Chat"
  backAriaLabel="Back to chats"
  isArchivedView={false}
  onBack={() => setView('list')}
  onClose={() => setPanelOpen(false)}
  onRename={() => openRenameDialog()}
  onArchive={() => archiveConversation(id)}
/>

// List view (no back, with archive entry):
<ChatPanelHeader
  title="Mingo AI"
  onBack={() => {}}
  onClose={() => setPanelOpen(false)}
  onOpenArchive={() => setView('archive')}
/>
```
