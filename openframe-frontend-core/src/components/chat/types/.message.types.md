Defines comprehensive type system for chat messages, including execution states, approval workflows, and content segments used throughout the Flamingo AI chat interface.

## Key Components

- **MESSAGE_TYPE** - Constant enum defining all supported message types (TEXT, EXECUTING_TOOL, APPROVAL_REQUEST, etc.)
- **MessageSegment** - Union type for different content segments (text, tool execution, approval requests, errors)
- **ToolExecutionData** - Interface for tool execution states with parameters and results
- **ApprovalRequestData/ApprovalResultData** - Types for command approval workflow
- **MessageData** - Union of all possible message data structures from GraphQL/API
- **HistoricalMessage** - Interface for persisted chat messages with metadata
- **ProcessedMessage/Message** - Final processed message interfaces for UI rendering

## Usage Example

```typescript
import { MESSAGE_TYPE, type MessageSegment, type ProcessedMessage } from './message.types'

// Create a tool execution message
const toolMessage: ProcessedMessage = {
  id: 'msg-123',
  role: 'assistant',
  content: [{
    type: 'tool_execution',
    data: {
      type: 'EXECUTING_TOOL',
      integratedToolType: 'docker',
      toolFunction: 'listContainers',
      parameters: { status: 'running' }
    }
  }],
  timestamp: new Date(),
  assistantType: 'mingo'
}

// Handle approval request
const approvalSegment: MessageSegment = {
  type: 'approval_request',
  data: {
    command: 'docker rm container-123',
    explanation: 'Remove unused container'
  },
  onApprove: async (requestId) => {
    console.log(`Approved: ${requestId}`)
  }
}
```
