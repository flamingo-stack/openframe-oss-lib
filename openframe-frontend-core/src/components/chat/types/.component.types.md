Component prop types for Flamingo's chat interface components, defining TypeScript interfaces for the entire chat system from containers to individual messages.

## Key Components

### **Core Chat Components**
- `ChatContainerProps` - Root chat container wrapper
- `ChatHeaderProps` - Chat header with user info and controls
- `ChatMessageListProps` - Message list with scrolling and typing indicators
- `ChatInputProps` - Message input with send functionality
- `ChatSidebarProps` - Dialog/conversation sidebar navigation

### **Message Components**
- `ChatMessageEnhancedProps` - Individual message display with rich content support
- `ApprovalRequestMessageProps` - Interactive approval messages for tool execution
- `ToolExecutionDisplayProps` - Tool execution status and results display
- `ErrorMessageDisplayProps` - Error message presentation

### **Utility Components**
- `ChatTypingIndicatorProps` - Animated typing indicators
- `ConnectionIndicatorProps` - Real-time connection status
- `ChatQuickActionProps` - Quick action buttons with hint animations
- `ModelDisplayProps` - AI model information display

### **Data Types**
- `DialogItem` - Conversation/dialog metadata
- `ChatInputRef` / `ChatMessageListRef` - Component reference interfaces

## Usage Example

```typescript
import type { 
  ChatContainerProps, 
  ChatMessageEnhancedProps,
  ApprovalRequestMessageProps 
} from './component.types'

// Chat container setup
const chatProps: ChatContainerProps = {
  className: "chat-wrapper",
  children: messageComponents
}

// Enhanced message with approval handling
const messageProps: ChatMessageEnhancedProps = {
  role: "assistant",
  content: messageSegments,
  assistantType: "mingo",
  onApprove: async (requestId) => await handleApproval(requestId),
  onReject: (requestId) => handleRejection(requestId)
}

// Approval request component
const approvalProps: ApprovalRequestMessageProps = {
  data: approvalRequestData,
  status: "pending",
  onApprove: handleApprove,
  onReject: handleReject
}
```
