TypeScript interface definitions for React chat component props, providing type safety for a comprehensive chat system with support for real-time messaging, tool execution, approvals, and dialog management.

## Key Components

**Core Chat Components:**
- `ChatContainerProps` - Main chat container wrapper
- `ChatHeaderProps` - Chat header with user info and controls
- `ChatMessageEnhancedProps` - Individual message display with approval/rejection handlers
- `ChatMessageListProps` - Message list container with auto-scroll and typing indicators
- `ChatInputProps` - Message input with send functionality and multi-line support

**Interactive Features:**
- `ApprovalRequestMessageProps` - Approval workflow for tool execution requests
- `ToolExecutionDisplayProps` - Expandable tool execution result display
- `ChatQuickActionProps` - Quick action buttons with hint animations
- `ChatTypingIndicatorProps` - Real-time typing status indicators

**Navigation & Management:**
- `ChatSidebarProps` - Dialog list sidebar with infinite scroll support
- `DialogListItemProps` - Individual dialog items with unread counts
- `ModelDisplayProps` - AI model information display
- `ConnectionIndicatorProps` - Real-time connection status

## Usage Example

```typescript
// Enhanced message component with approval handling
const messageProps: ChatMessageEnhancedProps = {
  role: 'assistant',
  content: segments,
  assistantType: 'mingo',
  timestamp: new Date(),
  onApprove: async (requestId) => {
    await handleToolApproval(requestId);
  },
  onReject: (requestId) => {
    handleToolRejection(requestId);
  }
};

// Chat input with ref for programmatic control
const inputRef = useRef<ChatInputRef>(null);

const inputProps: ChatInputProps = {
  onSend: (message) => sendMessage(message),
  sending: isSending,
  maxRows: 6,
  showSendButton: true
};

// Clear input after sending
inputRef.current?.clear();
```
