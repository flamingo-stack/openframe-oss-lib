TypeScript component prop interface definitions for a comprehensive chat UI system, providing type safety for React components used in AI-powered chat interfaces with approval workflows, tool execution displays, and dialog management.

## Key Components

**Core Chat Props**:
- `ChatContainerProps` - Main chat wrapper with HTML div attributes
- `ChatHeaderProps` - Header with user info, settings, and connection status
- `ChatMessageEnhancedProps` - Enhanced messages supporting segments, typing states, and approval callbacks
- `ChatMessageListProps` - Message list with scrolling, loading states, and approval handling
- `ChatInputProps` - Text input with send functionality and response states

**Specialized Component Props**:
- `ToolExecutionDisplayProps` - Expandable tool execution results display
- `ApprovalRequestMessageProps` - Interactive approval request with status handling
- `ConnectionIndicatorProps` - Real-time connection status indicator
- `ChatTypingIndicatorProps` - Animated typing indicator with customizable appearance
- `ChatQuickActionProps` - Action buttons with hint animations and loading states

**Navigation & Management**:
- `ChatSidebarProps` - Dialog list with infinite scroll and creation states
- `DialogListItemProps` - Individual dialog items with selection and unread counts
- `ModelDisplayProps` - AI model information display with provider details

## Usage Example

```typescript
import type { 
  ChatMessageEnhancedProps, 
  ChatInputProps, 
  ApprovalRequestMessageProps 
} from './component.types'

// Enhanced message with approval handling
const messageProps: ChatMessageEnhancedProps = {
  role: 'assistant',
  content: [
    { type: 'text', content: 'Here is the analysis:' },
    { 
      type: 'approval_request', 
      data: { requestId: 'req_123', action: 'execute_command' } 
    }
  ],
  assistantType: 'mingo',
  onApprove: async (requestId) => {
    await approveRequest(requestId)
  },
  onReject: (requestId) => {
    rejectRequest(requestId)
  }
}

// Chat input with send handling
const inputProps: ChatInputProps = {
  onSend: (message) => sendMessage(message),
  sending: false,
  awaitingResponse: true,
  maxRows: 4,
  placeholder: "Ask Mingo anything..."
}
```
