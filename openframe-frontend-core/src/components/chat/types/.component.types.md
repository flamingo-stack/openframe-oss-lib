Defines all TypeScript prop and configuration types used by the Flamingo chat UI component library, covering containers, headers, messages, inputs, and slash commands.

## Key Components

- **`NavLinkAnchorComponent`** — Component type for custom anchor rendering in markdown links, allowing host apps to inject their own routing logic
- **`ChatContainerProps`** — Base div wrapper props for the chat layout
- **`ChatHeaderProps`** — Full header configuration including user info, connection status, ticket metadata, and layout modifiers (`fullWidth`, `bare`)
- **`ChatHeaderTicketInfo`** — Ticket display metadata (title, status, lifecycle color/kind)
- **`ConnectionIndicatorProps`** — Connection status indicator props
- **`ChatMessageEnhancedProps`** — Rich message row props supporting roles, content segments, approval callbacks, inline entity cards (`chatRefs`, `renderEntityCard`), and custom anchor routing (`NavLinkAnchor`)
- **`ChatMessageListProps`** — Scrollable message list props with infinite scroll, avatar control, approval forwarding, and full-width layout
- **`ChatMessageListRef`** — Imperative ref handle for programmatic scroll control
- **`SlashCommandSummary`** — Autocomplete command descriptor including actions, icons, labels, and display ordering
- **`SlashCommandsProp`** — Slash command integration contract (`fetchCommands`, `resolveSourceIcon`)
- **`SlashCommandActionId`** — `'browse' | 'search' | 'find' | 'display'` action dispatch union

## Usage Example

```typescript
import type {
  ChatHeaderProps,
  ChatMessageEnhancedProps,
  NavLinkAnchorComponent,
  SlashCommandSummary,
} from './component.types'

const MyAnchor: NavLinkAnchorComponent = ({ href, className, children }) => (
  <a href={href} className={className}>{children}</a>
)

const headerProps: ChatHeaderProps = {
  userName: 'Mingo AI',
  connectionStatus: 'connected',
  fullWidth: true,
  bare: false,
  ticketInfo: {
    title: 'Ticket #1042',
    statusName: 'In Progress',
    statusColor: '#FFC008',
    statusKind: 'active',
  },
}

const messageProps: ChatMessageEnhancedProps = {
  role: 'assistant',
  content: 'How can I help?',
  NavLinkAnchor: MyAnchor,
  renderEntityCard: (ref) => <EntityCard ref={ref} />,
}
```
