Defines core constants, types, and utility functions for the Flamingo chat system, covering message roles, owner types, assistant identities, connection states, and SSE wire shapes.

## Key Components

| Export | Kind | Description |
|--------|------|-------------|
| `CHAT_TYPE` | Const | Discriminates between `CLIENT_CHAT` and `ADMIN_AI_CHAT` contexts |
| `OWNER_TYPE` | Const | Message ownership: `CLIENT`, `ADMIN`, or `ASSISTANT` |
| `MESSAGE_ROLE` | Const | LLM-style roles: `user`, `assistant`, `error`, `system` |
| `ASSISTANT_TYPE` | Const | AI assistant identity: `fae` (client-facing) or `mingo` (technician) |
| `AUTHOR_TYPE` | Const | Granular authorship including `user`, `admin`, `fae`, `mingo`, `system` |
| `APPROVAL_STATUS` | Const | Chat approval lifecycle: `pending`, `approved`, `rejected` |
| `CONNECTION_STATUS` | Const | WebSocket/SSE lifecycle states |
| `MessageOwner` | Interface | Owner metadata including optional user profile and avatar resolution |
| `buildChatRefKey` | Function | Builds canonical `docType:externalId` lookup keys for inline card markers |
| `isStructuredContent` | Function | Type guard — checks if content is a `MessageSegment[]` |
| `normalizeContent` | Function | Coerces `string \| MessageSegment[]` into a uniform segment array |
| `SSEEventData` | Union type | Wire shapes for `TEXT` and tool execution SSE events |
| `QuickAction` | Interface | Shape for quick-action chips rendered in the chat UI |

## Usage Example

```typescript
import {
  CHAT_TYPE,
  ASSISTANT_TYPE,
  APPROVAL_STATUS,
  CONNECTION_STATUS,
  MESSAGE_ROLE,
  buildChatRefKey,
  isStructuredContent,
  normalizeContent,
  type ChatType,
  type AssistantType,
  type SSEEventData,
} from './chat.types'

// Distinguish chat context
const chatType: ChatType = CHAT_TYPE.ADMIN // 'ADMIN_AI_CHAT'

// Route to the correct AI assistant
const assistant: AssistantType = ASSISTANT_TYPE.MINGO // technician-facing

// Build a card reference key for inline markers like [card://ticket:123]
const refKey = buildChatRefKey('ticket', '123') // 'ticket:123'

// Normalize raw LLM content into segments for the renderer
const segments = normalizeContent('Hello, how can I help?')
// → [{ type: 'text', text: 'Hello, how can I help?' }]

// Handle incoming SSE events
function handleSSE(event: SSEEventData) {
  if (event.type === 'TEXT') {
    console.log(event.text)
  } else if (event.type === 'EXECUTED_TOOL') {
    console.log(event.toolFunction, event.success)
  }
}

// Gate a message on approval state
if (status === APPROVAL_STATUS.PENDING) {
  showApprovalBanner()
}

// React to connection lifecycle
if (state === CONNECTION_STATUS.RECONNECTING) {
  showReconnectingIndicator()
}
```
