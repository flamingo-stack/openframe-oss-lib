Defines core constants, enums, interfaces, and utility functions for the Flamingo chat system, covering chat types, message roles, connection states, and SSE event shapes.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `CHAT_TYPE` | Const | Chat contexts: `CLIENT_CHAT`, `ADMIN_AI_CHAT` |
| `OWNER_TYPE` | Const | Message ownership: `CLIENT`, `ADMIN`, `ASSISTANT` |
| `MESSAGE_ROLE` | Const | Conversation roles: `user`, `assistant`, `error`, `system` |
| `ASSISTANT_TYPE` | Const | AI assistants: `fae` (client-facing) and `mingo` (technician) |
| `AUTHOR_TYPE` | Const | Extended authorship including `user`, `admin`, `fae`, `mingo`, `system` |
| `APPROVAL_STATUS` | Const | Workflow states: `pending`, `approved`, `rejected` |
| `CONNECTION_STATUS` | Const | WebSocket/SSE lifecycle: `connecting`, `connected`, `disconnected`, `reconnecting`, `closed`, `error` |
| `MessageOwner` | Interface | Owner identity with optional user profile and avatar resolution |
| `buildChatRefKey` | Function | Constructs canonical `docType:externalId` lookup keys for inline card markers |
| `isStructuredContent` | Function | Type guard distinguishing segment arrays from plain strings |
| `normalizeContent` | Function | Normalizes `string \| MessageSegment[]` into a uniform segment array |
| `SSEEventData` | Union type | Wire shapes for `TEXT` and tool execution SSE events |
| `QuickAction` | Interface | Quick-reply chip shape with `id` and `text` |

## Usage Example

```typescript
import {
  CHAT_TYPE,
  MESSAGE_ROLE,
  ASSISTANT_TYPE,
  CONNECTION_STATUS,
  APPROVAL_STATUS,
  buildChatRefKey,
  isStructuredContent,
  normalizeContent,
  type ChatType,
  type MessageOwner,
  type SSEEventData,
} from './chat.types'

// Distinguish chat context
const chatType: ChatType = CHAT_TYPE.ADMIN // 'ADMIN_AI_CHAT'

// Identify the active AI assistant
const assistant = ASSISTANT_TYPE.MINGO // technician-facing

// Build a card reference key for inline rendering
const refKey = buildChatRefKey('ticket', 'TKT-1042') // 'ticket:TKT-1042'

// Normalize incoming message content to segments
const segments = normalizeContent('Hello, how can I help?')
// → [{ type: 'text', text: 'Hello, how can I help?' }]

// Type-guard for structured content
if (isStructuredContent(segments)) {
  console.log('Segment count:', segments.length)
}

// Handle SSE events from the AI stream
function handleSSE(event: SSEEventData) {
  if (event.type === 'TEXT') {
    console.log('Stream text:', event.text)
  } else if (event.type === 'EXECUTED_TOOL') {
    console.log(`Tool "${event.toolFunction}" success:`, event.success)
  }
}

// Check connection lifecycle
const status = CONNECTION_STATUS.RECONNECTING

// Build a MessageOwner for an authenticated user
const owner: MessageOwner = {
  type: 'CLIENT',
  userId: 'usr_abc123',
  user: {
    id: 'usr_abc123',
    firstName: 'Jane',
    image: { imageUrl: '/avatars/jane.png' },
  },
}
```
