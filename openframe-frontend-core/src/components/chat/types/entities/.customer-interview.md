Re-exports Customer Interview entity types from the shared lib (`@/types/customer-interview`) into the chat card consumer barrel.

## Key Components

| Export | Description |
|--------|-------------|
| `CustomerInterview` | Canonical shape for a customer interview record |
| `CustomerInterviewConfig` | Configuration options for a customer interview |
| `CustomerInterviewFilters` | Filter parameters for querying interviews |
| `CustomerInterviewListResponse` | Paginated/list response wrapper for interviews |

## Usage Example

```typescript
import type {
  CustomerInterview,
  CustomerInterviewConfig,
  CustomerInterviewFilters,
  CustomerInterviewListResponse,
} from '@/entities/chat/customer-interview';

const filters: CustomerInterviewFilters = { status: 'completed' };

const response: CustomerInterviewListResponse = await fetchInterviews(filters);
```

> **Note:** This is a pure re-export barrel — no logic or competing type definitions live here. The canonical source of truth is `src/types/customer-interview.ts`.
