Barrel export file for the Tickets feature module, re-exporting all public components, hooks, and types related to ticket management and the Help Center surface.

## Key Components

**Components**
- `TicketCenter` — Main ticket management interface
- `TicketOpenForm` — Form for opening new tickets
- `TicketRow` — Individual ticket row renderer
- `TicketDetailDrawer` — Slide-out drawer for ticket details (exposes `TicketRef`)
- `TicketLinkedDeliveryCard` — Card showing delivery items linked to a ticket
- `TicketReplyComposer` — Composer UI for replying to tickets
- `HelpCenterList` — Full-page customer-facing ticket surface for OpenFrame's `/tickets` route; composable inside third-party `<PageShell>` wrappers
- `HelpCenterCard` — Individual card within the Help Center list
- `HelpCenterCreateForm` / `HelpCenterCreateFormSkeleton` — Ticket creation form with loading skeleton

**Hooks**
- `useTicketsList` — Fetches and manages the ticket list state
- `useTicketActions` — CRUD actions for tickets; exposes `mapTicketActionError` helper
- `useTicketEngagements` — Manages ticket engagement threads and file attachments

**Types & Constants**
- `TicketData`, `OptimisticTicket`, `AnyTicket` — Core ticket data shapes
- `TicketClickupSummary` — ClickUp integration summary type
- `TicketActionErrorCode` / `MappedTicketActionError` — Structured error handling
- `TICKET_TOAST_COPY` — Toast notification strings
- `isOptimistic` — Type guard for optimistic ticket entries

## Usage Example

```typescript
import {
  HelpCenterList,
  useTicketsList,
  useTicketActions,
  isOptimistic,
  type AnyTicket,
} from '@/features/tickets'

function MyTicketPage() {
  const { tickets } = useTicketsList()
  const { closeTicket } = useTicketActions()

  return (
    <HelpCenterList
      tickets={tickets}
      onClose={(ticket: AnyTicket) => {
        if (!isOptimistic(ticket)) closeTicket(ticket.id)
      }}
    />
  )
}
```
