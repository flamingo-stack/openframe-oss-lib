Barrel/index file that re-exports all public components, hooks, and types for the Tickets & Help Center feature module.

## Key Components

**UI Components**
- `TicketCenter` — primary ticket management container
- `TicketOpenForm` — form for opening new tickets
- `TicketRow` — single ticket list row
- `TicketDetailDrawer` — slide-out drawer showing full ticket detail (ref-forwarded via `TicketRef`)
- `TicketLinkedDeliveryCard` — card linking a ticket to a delivery item
- `TicketReplyComposer` — rich reply/comment input for ticket threads
- `HelpCenterList` — full-page customer-facing ticket surface used by OpenFrame's `/tickets` route; composes hero + search + filter chrome with an expandable card list
- `HelpCenterCard` — individual help center ticket card
- `HelpCenterCreateForm` / `HelpCenterCreateFormSkeleton` — creation form with loading skeleton

**Hooks**
- `useTicketsList` — fetches and manages the ticket list state
- `useTicketActions` — CRUD/action mutations; exposes `mapTicketActionError` helper and `UseTicketActionsReturn` type
- `useTicketEngagements` — loads ticket engagement history (messages, file attachments)

**Types & Utilities**
- `TicketData`, `OptimisticTicket`, `AnyTicket` — core ticket data shapes
- `TicketClickupSummary` — ClickUp-linked ticket metadata
- `TicketActionErrorCode` / `MappedTicketActionError` — structured error types
- `TICKET_TOAST_COPY` — toast notification strings (re-exported as alias)
- `isOptimistic` — type guard for optimistic ticket entries

## Usage Example

```typescript
import {
  HelpCenterList,
  useTicketActions,
  isOptimistic,
  type AnyTicket,
} from '@/features/tickets'

function TicketsPage() {
  const { closeTicket } = useTicketActions()

  const handleClose = (ticket: AnyTicket) => {
    if (!isOptimistic(ticket)) {
      closeTicket(ticket.id)
    }
  }

  return <HelpCenterList onTicketAction={handleClose} />
}
```
