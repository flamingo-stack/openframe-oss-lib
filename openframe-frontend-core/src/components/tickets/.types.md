Defines the complete type contract for the ticket management system, including wire shapes, cache slot structure, error codes, and UI constants shared across the ticket center, drawer, and action helpers.

## Key Components

**Interfaces**
- `TicketData` — Primary wire shape returned by `POST /api/chat/agent/find-ticket`; includes HubSpot fields, ClickUp link, assigned owner, and customer info
- `TicketAssignedOwner` — Resolved agent profile (name, email, avatar) for drawer header attribution
- `TicketClickupSummary` — Compact linked ClickUp task projection; aligns with `DeliveryItem` for shared rendering primitives
- `OptimisticTicket` — Extends `TicketData` with `_optimistic: true` for pre-server-roundtrip list prepending
- `TicketsCacheSlot` — Typed shape for `['tickets', …]` TanStack Query cache slots; prevents bare-array assumptions
- `MappedTicketActionError` — Normalized error envelope with routing flags (`supportSystemDown`, `removeRowFromCache`)

**Types**
- `AnyTicket` — Union of `TicketData | OptimisticTicket`
- `TicketActionErrorCode` — Discriminated string union of all known server error codes

**Functions**
- `isOptimistic(t)` — Type guard narrowing `AnyTicket` to `OptimisticTicket`

**Constants**
- `TICKET_TEXT_MAX_CHARS` — Client-side cap (`5000`) for ticket content
- `TICKET_LIVE_POLL_MS` — Shared refresh cadence (`8000`ms) for list + conversation polls
- `TOAST_COPY` — Centralized user-visible toast strings

## Usage Example

```typescript
import {
  AnyTicket,
  isOptimistic,
  TicketsCacheSlot,
  MappedTicketActionError,
  TICKET_LIVE_POLL_MS,
  TOAST_COPY,
} from './types'

// Type guard for optimistic rows
function renderTicket(ticket: AnyTicket) {
  if (isOptimistic(ticket)) return <SkeletonRow />
  return <TicketRow data={ticket} />
}

// Cache mutation typed through slot shape
queryClient.setQueriesData<TicketsCacheSlot>(
  { queryKey: ['tickets'] },
  (old) => ({
    ...old,
    tickets: old?.tickets?.map((t) =>
      t.id === updatedId ? { ...t, status: 'CLOSED' } : t
    ),
  })
)

// Live poll wired to shared constant
useTicketsList({ refetchInterval: TICKET_LIVE_POLL_MS })

// Toast on success
toast(TOAST_COPY.close_success)

// Error routing
function handleError(err: MappedTicketActionError) {
  if (err.supportSystemDown) showSystemDownBanner()
  if (err.removeRowFromCache) evictTicketRow(err)
  toast({ title: err.message })
}
```
