A comprehensive set of semantic table components built with React and styled with Tailwind CSS classes. These components provide a complete table structure with consistent styling and accessibility features.

## Key Components

- **Table**: Main table wrapper with responsive overflow handling
- **TableHeader**: Header section (`<thead>`) with bottom borders
- **TableBody**: Body section (`<tbody>`) with conditional borders
- **TableFooter**: Footer section (`<tfoot>`) with primary background styling
- **TableRow**: Individual row (`<tr>`) with hover states and selection styling
- **TableHead**: Header cell (`<th>`) with consistent padding and typography
- **TableCell**: Data cell (`<td>`) with standard padding and alignment
- **TableCaption**: Caption element with muted text styling

## Usage Example

```typescript
import { 
  Table, 
  TableHeader, 
  TableBody, 
  TableHead, 
  TableRow, 
  TableCell, 
  TableCaption 
} from "./components/ui/table"

function DataTable() {
  return (
    <Table>
      <TableCaption>A list of recent invoices</TableCaption>
      <TableHeader>
        <TableRow>
          <TableHead>Invoice</TableHead>
          <TableHead>Status</TableHead>
          <TableHead>Amount</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell>INV001</TableCell>
          <TableCell>Paid</TableCell>
          <TableCell>$250.00</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  )
}
```
