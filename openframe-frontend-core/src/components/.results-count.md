A React component that displays pagination information and result counts for search/listing pages, with support for dynamic result types and optional sorting indicators.

## Key Components

- **`ResultsCountProps`** - TypeScript interface defining component props including pagination parameters, total results, result type, and optional sorting message
- **`ResultsCount`** - Main functional component that calculates and renders the current page range and total count with proper singular/plural formatting

## Usage Example

```typescript
import { ResultsCount } from './results-count'

// Basic usage for vendor listings
<ResultsCount
  currentPage={2}
  pageSize={10}
  totalResults={45}
  resultType="vendors"
/>

// With sorting message for posts
<ResultsCount
  currentPage={1}
  pageSize={20}
  totalResults={3}
  resultType="posts"
  sortingMessage="sorted by date"
/>

// Example output: "Showing 11-20 of 45 vendors â€¢ sorted by date"
```

The component automatically handles edge cases like zero results (renders nothing), calculates proper start/end indices for the current page, and applies correct singular/plural forms based on the result type and count. The sorting message is displayed as an optional accent-colored indicator.
