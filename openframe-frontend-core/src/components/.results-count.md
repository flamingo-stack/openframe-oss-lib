A React component that displays pagination information showing the current range of results and total count for search/listing interfaces.

## Key Components

- **`ResultsCountProps`** - Interface defining component props including pagination state and result type
- **`ResultsCount`** - Main component that renders the results count display with optional sorting message

### Props

- `currentPage`: Current page number (1-based)
- `pageSize`: Number of items per page
- `totalResults`: Total number of available results
- `resultType`: Type of content being displayed ('vendors' or 'posts')
- `sortingMessage`: Optional message about current sorting method

## Usage Example

```typescript
import { ResultsCount } from './results-count'

// Basic usage for vendor listings
<ResultsCount
  currentPage={2}
  pageSize={20}
  totalResults={156}
  resultType="vendors"
/>
// Output: "Showing 21-40 of 156 vendors"

// With sorting message
<ResultsCount
  currentPage={1}
  pageSize={10}
  totalResults={45}
  resultType="posts"
  sortingMessage="sorted by date"
/>
// Output: "Showing 1-10 of 45 posts â€¢ sorted by date"
```

The component automatically handles singular/plural forms, calculates the correct result range for the current page, and returns `null` when there are no results to display.
