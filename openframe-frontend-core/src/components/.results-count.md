A React component that displays pagination information, showing the current range of results and total count with optional sorting context.

## Key Components

- **`ResultsCountProps`** - Interface defining component props including pagination parameters, result type, and optional sorting message
- **`ResultsCount`** - Main component that calculates and renders the "Showing X-Y of Z results" text with proper pluralization

## Usage Example

```typescript
import { ResultsCount } from './results-count'

function SearchResults() {
  return (
    <div>
      <ResultsCount
        currentPage={2}
        pageSize={10}
        totalResults={45}
        resultType="vendors"
        sortingMessage="sorted by relevance"
      />
      {/* Search results would go here */}
    </div>
  )
}

// Renders: "Showing 11-20 of 45 vendors â€¢ sorted by relevance"
```

The component automatically handles edge cases like single results (using singular form) and returns null when there are no results. It supports both "vendors" and "posts" result types with appropriate pluralization.
