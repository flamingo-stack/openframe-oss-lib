A React component system for creating multi-step progress indicators with customizable styling and state management. The component automatically tracks completion status and provides visual feedback for each step.

## Key Components

- **`Steps`** - Container component that manages step numbering and state propagation
- **`Step`** - Individual step component with title, description, and visual indicators
- **`StepsProps`** - Interface requiring `currentStep` number for state management
- **`StepProps`** - Interface for individual steps with title, optional description, and auto-injected state props

## Usage Example

```typescript
import { Steps, Step } from './components/steps'

function SetupWizard() {
  const [currentStep, setCurrentStep] = useState(2)

  return (
    <Steps currentStep={currentStep}>
      <Step 
        title="Account Setup" 
        description="Create your account and verify email"
      />
      <Step 
        title="Profile Information" 
        description="Add your personal details"
      />
      <Step 
        title="Preferences" 
        description="Configure your settings"
      />
    </Steps>
  )
}
```

The component automatically displays completed steps with checkmarks, highlights the current step, and connects steps with vertical lines. Each step receives `isActive`, `isCompleted`, and `stepNumber` props automatically based on the `currentStep` value.
