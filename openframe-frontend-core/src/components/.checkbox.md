A reusable checkbox component built on top of Radix UI's checkbox primitive with custom styling and a check icon indicator.

## Key Components

- **Checkbox**: Main component that renders a styled checkbox input with proper accessibility features
- **CheckboxPrimitive**: Radix UI's unstyled checkbox primitive used as the foundation
- **Check**: Lucide React icon displayed when checkbox is checked

## Usage Example

```typescript
import { Checkbox } from "./components/checkbox"

function MyForm() {
  const [isChecked, setIsChecked] = React.useState(false)

  return (
    <div className="flex items-center space-x-2">
      <Checkbox 
        id="terms" 
        checked={isChecked}
        onCheckedChange={setIsChecked}
      />
      <label htmlFor="terms" className="text-sm font-medium">
        Accept terms and conditions
      </label>
    </div>
  )
}

// With custom styling
<Checkbox 
  className="data-[state=checked]:bg-blue-600" 
  checked={true}
  disabled={false}
/>
```

The component supports all standard checkbox properties including `checked`, `onCheckedChange`, `disabled`, and custom CSS classes through the `className` prop.
