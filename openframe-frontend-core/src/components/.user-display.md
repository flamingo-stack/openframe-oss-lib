A reusable React component that displays a user's avatar alongside their name and optional subtitle text in a horizontal layout.

## Key Components

- **`UserDisplay`** - Main component that renders an avatar with user information
- **`UserDisplayProps`** - Interface defining component props including name, avatar URL, subtitle, size, and styling options
- **`SquareAvatar`** - Imported component for rendering user avatars
- **`cn`** - Utility function for conditional CSS class names

## Usage Example

```typescript
import { UserDisplay } from './user-display';

// Basic usage
<UserDisplay 
  name="John Doe" 
  avatarUrl="https://example.com/avatar.jpg" 
/>

// With subtitle and custom size
<UserDisplay 
  name="Jane Smith" 
  avatarUrl="https://example.com/avatar2.jpg"
  subtitle="2 hours ago"
  size={48}
  className="mb-4"
/>

// Without avatar (shows fallback)
<UserDisplay 
  name="Anonymous User" 
  subtitle="Just now"
/>
```

The component automatically handles text truncation for long names/subtitles and provides a fallback avatar when no image URL is provided. It's commonly used in comment headers, user lists, and other areas requiring consistent user identification display.
