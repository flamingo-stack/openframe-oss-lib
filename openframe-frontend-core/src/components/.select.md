A styled Select component built on top of Radix UI primitives with pre-configured Tailwind CSS styling and accessible design patterns.

## Key Components

- **Select** - Root container component
- **SelectTrigger** - Clickable button with chevron icon to open dropdown
- **SelectContent** - Dropdown container with portal rendering and animations
- **SelectItem** - Individual selectable option with check indicator
- **SelectValue** - Displays the selected value
- **SelectLabel** - Section headers within dropdown
- **SelectGroup** - Groups related options together
- **SelectSeparator** - Visual divider between option groups
- **SelectScrollUpButton/SelectScrollDownButton** - Navigation for long lists

## Usage Example

```typescript
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "./select"

function Example() {
  return (
    <Select>
      <SelectTrigger className="w-[180px]">
        <SelectValue placeholder="Select a fruit" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="apple">Apple</SelectItem>
        <SelectItem value="banana">Banana</SelectItem>
        <SelectItem value="orange">Orange</SelectItem>
      </SelectContent>
    </Select>
  )
}
```

All components are forwardRef-enabled and fully accessible, with keyboard navigation, screen reader support, and smooth animations included by default.
