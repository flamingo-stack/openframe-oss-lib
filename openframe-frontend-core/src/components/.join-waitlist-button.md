A specialized button component for waitlist signups that provides consistent branding across the application with customizable colors and loading states.

## Key Components

- **JoinWaitlistButtonProps**: Interface extending ButtonProps with custom styling options and loading state
- **JoinWaitlistButton**: Primary export - a forwardRef component that renders a branded button with OpenFrame logo
- **OpenFrameLogo**: Imported icon component displayed when not in loading state
- **Button**: Base UI component providing core button functionality

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button';

// Basic usage with default styling
<JoinWaitlistButton onClick={handleJoinWaitlist} />

// With loading state (hides logo, shows spinner)
<JoinWaitlistButton 
  loading={isSubmitting} 
  onClick={handleJoinWaitlist} 
/>

// With custom colors and text
<JoinWaitlistButton
  buttonBackgroundColor="#ff6b35"
  buttonTextColor="#ffffff"
  lowerPathColor="#ffffff"
  upperPathColor="#ff6b35"
  onClick={handleJoinWaitlist}
>
  Get Early Access
</JoinWaitlistButton>

// As a form submit button
<JoinWaitlistButton
  type="submit"
  loading={form.formState.isSubmitting}
  disabled={!form.formState.isValid}
/>
```

The component automatically uses "primary" variant and "touch" size from the base Button component, with the OpenFrame logo (5x5 on mobile, 6x6 on desktop) appearing on the left side when not loading.
