A specialized button component for waitlist signup functionality that displays an OpenFrame logo and provides consistent styling across the application.

## Key Components

- **JoinWaitlistButton**: Main component extending Button with OpenFrame branding
- **JoinWaitlistButtonProps**: Interface extending ButtonProps with custom styling options
- **OpenFrameLogo**: Left-aligned icon component that hides during loading states

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button';

// Basic usage
<JoinWaitlistButton onClick={handleJoinWaitlist} />

// With loading state
<JoinWaitlistButton 
  loading={isSubmitting} 
  onClick={handleJoinWaitlist} 
/>

// Custom styling
<JoinWaitlistButton
  buttonBackgroundColor="#ff6b35"
  buttonTextColor="#ffffff"
  lowerPathColor="#ffffff"
  upperPathColor="#000000"
  className="my-4"
>
  Get Early Access
</JoinWaitlistButton>

// Disabled state
<JoinWaitlistButton disabled>
  Waitlist Full
</JoinWaitlistButton>
```

The component automatically handles the primary variant and touch size styling, includes responsive logo sizing, and integrates the Button's built-in loading spinner functionality.
