A React SVG component that renders the official Google logo with authentic brand colors. The component accepts standard SVG props and optional className for styling customization.

## Key Components

- **GoogleLogo**: Main export component that renders the Google "G" logo as an inline SVG
- **Props**: Extends `React.SVGProps<SVGSVGElement>` with optional `className` string
- **SVG paths**: Four colored path elements representing Google's brand colors (blue, green, yellow, red)

## Usage Example

```typescript
import { GoogleLogo } from './google-logo';

// Basic usage
<GoogleLogo />

// With custom styling
<GoogleLogo className="w-8 h-8" />

// With additional SVG props
<GoogleLogo 
  className="hover:opacity-80 transition-opacity"
  onClick={handleGoogleSignIn}
  aria-label="Sign in with Google"
/>

// Custom size
<GoogleLogo width="32" height="32" />
```

The component maintains the authentic Google brand colors and proportions while providing flexibility for integration into various UI contexts through standard React props and className customization.
