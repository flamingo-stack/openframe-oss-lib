Provides skeleton loading components for various page layouts and sections in a React application. Creates consistent animated placeholders that match the actual component structures while content loads.

## Key Components

- **Page Structure Skeletons**: `AnnouncementBarSkeleton`, `HeaderSkeleton`, `HeroSkeleton` - Core layout elements
- **Navigation Skeletons**: `BreadcrumbSkeleton`, `CategorySidebarSkeleton` - Navigation and filtering components  
- **Search & Results**: `SearchContainerSkeleton`, `ResultsHeaderSkeleton` - Search interface placeholders
- **Layout Patterns**: `TwoColumnLayoutSkeleton`, `ArticleLayoutSkeleton`, `VendorDetailLayoutSkeleton` - Complete page structures
- **Base Components**: Uses `UnifiedSkeleton`, `TextSkeleton`, `MediaSkeleton`, `InteractiveSkeleton` for consistent styling

## Usage Example

```typescript
import { HeaderSkeleton, TwoColumnLayoutSkeleton, CategorySidebarSkeleton } from './page-layout-skeleton'

// Show loading state for entire page
function LoadingPage() {
  return (
    <>
      <HeaderSkeleton />
      <main className="container mx-auto px-4 py-8">
        <TwoColumnLayoutSkeleton 
          sidebarContent={<CategorySidebarSkeleton />}
          mainContent={<div>Custom content skeleton</div>}
        />
      </main>
    </>
  )
}

// Conditional loading for search results
function SearchPage({ isLoading }: { isLoading: boolean }) {
  if (isLoading) {
    return <SearchContainerSkeleton showFilters={true} />
  }
  return <SearchResults />
}
```
