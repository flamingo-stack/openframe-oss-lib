Loading placeholder UI that mirrors the exact layout and dimensions of an `OrganizationCard`, preventing layout shift during data fetches.

## Key Components

### `OrganizationCardSkeleton`
Single card skeleton with three toggleable regions:
- **Header** — 60×60px org logo placeholder + title/subtitle text bars
- **Description** — Fixed 48px height, two-line text shimmer (controlled by `showDescription`)
- **Footer** — Two stat items with icon+label pairs and a tag/badge block (controlled by `showFooter`)

### `OrganizationCardSkeletonGrid`
Renders a responsive grid of `OrganizationCardSkeleton` instances.

| Breakpoint | Columns |
|------------|---------|
| Mobile | 1 |
| `md` | 2 |
| `lg` | 3 |

### `OrganizationCardSkeletonProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Extra classes on the card root |
| `containerClassName` | `string` | — | Overrides card background/border |
| `showFooter` | `boolean` | `true` | Toggle footer stats area |
| `showDescription` | `boolean` | `true` | Toggle description area |

## Usage Example

```typescript
// Single skeleton while a card loads
<OrganizationCardSkeleton showDescription={false} />

// Full grid while a list fetches
<OrganizationCardSkeletonGrid
  count={9}
  showFooter={true}
  showDescription={true}
/>

// Custom container styling
<OrganizationCardSkeleton
  containerClassName="bg-neutral-900 border border-neutral-700"
  className="opacity-75"
/>
```
