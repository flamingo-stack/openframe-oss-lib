This file provides specialized skeleton components for common UI patterns, building on top of the unified skeleton system to create realistic loading states for complex content layouts.

## Key Components

- **ParagraphSkeleton** - Multi-line text blocks with natural line length variation
- **ListSkeleton** - Configurable list items with optional icons and actions
- **TableSkeleton** - Grid-based data tables with headers and rows
- **FormSkeleton** - Form layouts with labels, inputs, and buttons
- **NavigationSkeleton** - Menu items in horizontal or vertical layouts
- **ProfileSkeleton** - User profiles with avatar, bio, and stats
- **CommentSkeleton** - Threaded comments with ratings and replies
- **FeatureListSkeleton** - Feature lists with optional grouping and icons
- **TimelineSkeleton** - Activity feeds with timeline indicators
- **PricingSkeleton** - Pricing plan cards with features and CTAs

## Usage Example

```typescript
import { ParagraphSkeleton, TableSkeleton, ProfileSkeleton } from './content-skeleton'

// Article content loading
<ParagraphSkeleton lines={6} />

// Data table loading
<TableSkeleton rows={10} columns={5} />

// User profile loading with stats
<ProfileSkeleton showBio={true} showStats={true} />

// Comments section
<CommentSkeleton 
  showRating={true} 
  showReplies={true} 
/>

// Pricing page
<PricingSkeleton plans={3} />
```
