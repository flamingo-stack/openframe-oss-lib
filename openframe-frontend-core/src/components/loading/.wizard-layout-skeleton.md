A loading skeleton component that mimics the structure of a wizard/multi-step form interface while content is being loaded.

## Key Components

- **WizardLayoutSkeleton**: Main component that renders skeleton placeholders for wizard UI elements
- **WizardLayoutSkeletonProps**: Interface defining component props including step count and styling
- **UnifiedSkeleton**: Used for creating generic skeleton shapes (progress indicators, buttons)
- **TextSkeleton**: Used for text-specific skeleton placeholders (headings, body text)

## Usage Example

```typescript
import { WizardLayoutSkeleton } from './wizard-layout-skeleton'

// Basic usage with default 6 steps
function LoadingWizard() {
  return <WizardLayoutSkeleton />
}

// Custom step count and styling
function CustomLoadingWizard() {
  return (
    <WizardLayoutSkeleton 
      steps={4} 
      className="p-6 bg-gray-50" 
    />
  )
}

// Conditional rendering during data loading
function WizardContainer({ isLoading, children }) {
  return isLoading ? <WizardLayoutSkeleton steps={5} /> : children
}
```

The component includes accessibility features with proper ARIA labels and creates a realistic loading state that matches typical wizard layout patterns with progress indicators, header content, and navigation buttons.
