A loading skeleton placeholder component that mirrors the exact dimensions and styling of `OrganizationIcon`/`VendorIcon` to prevent layout shifts during data loading states.

## Key Components

**`OrganizationIconSkeleton`** — Main exported component that renders an animated pulse placeholder.

**`OrganizationIconSkeletonProps`** — Interface defining:
- `size` — Six size variants (`xs`→`xl`) matching `OrganizationIcon` pixel dimensions exactly
- `showBackground` — Toggles the container border/background wrapper (default: `true`)
- `backgroundStyle` — Three background themes: `dark`, `light`, `white` (default: `dark`)
- `className` — Additional Tailwind classes for overrides

**`sizeClasses`** — Internal map of size tokens to Tailwind dimension classes (`w-6 h-6` through `w-16 h-16`)

**`backgroundClasses`** — Internal map of background style tokens to ODS design token classes

## Usage Example

```typescript
// Default (md, dark background) — organizations table / dashboard
<OrganizationIconSkeleton />

// Devices table
<OrganizationIconSkeleton size="sm" />

// Detail view
<OrganizationIconSkeleton size="xl" />

// Light card background
<OrganizationIconSkeleton size="md" backgroundStyle="light" />

// No background container (bare skeleton)
<OrganizationIconSkeleton size="sm" showBackground={false} />
```

## Notes

- Renders with `role="status"` and `aria-label` for screen reader accessibility
- Inner pulse element is sized at 50% (`w-1/2 h-1/2`) of the container, matching icon inner content proportions
- Drop-in replacement wherever `OrganizationIcon` or `VendorIcon` appears in skeleton screens
