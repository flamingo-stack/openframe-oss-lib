A React skeleton component that provides loading placeholders for organization icons with exact visual parity to the actual OrganizationIcon component.

## Key Components

- **OrganizationIconSkeleton** - Main skeleton component with configurable size, background, and styling options
- **OrganizationIconSkeletonProps** - TypeScript interface defining component props
- **sizeClasses** - Predefined size variants (xs through xl) matching icon dimensions
- **backgroundClasses** - Background style variants (dark, light, white) for different themes

## Usage Example

```typescript
import { OrganizationIconSkeleton } from './components/organization-icon-skeleton'

// Basic usage with default medium size
<OrganizationIconSkeleton />

// Different sizes for various contexts
<OrganizationIconSkeleton size="sm" /> // For device tables
<OrganizationIconSkeleton size="xl" /> // For detail views

// Custom styling and background options
<OrganizationIconSkeleton 
  size="lg"
  backgroundStyle="white"
  className="mr-2"
/>

// Without background container
<OrganizationIconSkeleton 
  size="md"
  showBackground={false}
/>
```

The skeleton includes proper accessibility attributes and pulse animation to indicate loading state while preventing layout shifts during icon loading.
