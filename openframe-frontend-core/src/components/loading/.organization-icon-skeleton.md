Loading skeleton component that renders a placeholder matching the exact dimensions and styling of `OrganizationIcon`/`VendorIcon`, preventing layout shifts during data fetch.

## Key Components

**`OrganizationIconSkeletonProps`** — Interface defining component configuration:
- `size` — Six size variants (`xs`→`xl`) matching `OrganizationIcon` dimensions (24px–64px)
- `showBackground` — Toggles the rounded container wrapper (default: `true`)
- `backgroundStyle` — Visual theme variant: `dark`, `light`, or `white`
- `className` — Additional Tailwind/CSS classes for override

**`sizeClasses`** — Internal map of size tokens to Tailwind dimension classes

**`backgroundClasses`** — Internal map of background style tokens to Tailwind classes using ODS design tokens

**`OrganizationIconSkeleton`** — Main exported function component rendering an animated pulse placeholder

## Usage Example

```typescript
// Default — organizations table or dashboard
<OrganizationIconSkeleton size="md" />

// Devices table
<OrganizationIconSkeleton size="sm" />

// Detail view header
<OrganizationIconSkeleton size="xl" backgroundStyle="light" />

// Without background wrapper (inline contexts)
<OrganizationIconSkeleton size="sm" showBackground={false} />

// Light card style
<OrganizationIconSkeleton size="lg" backgroundStyle="white" className="mt-2" />
```

The inner element renders at 50% of the container size with `animate-pulse`, and the outer `div` includes `role="status"` and `aria-label` for screen reader accessibility.
