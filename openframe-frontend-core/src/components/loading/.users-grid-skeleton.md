A loading skeleton component for the users grid table, rendering animated placeholder rows while user data is being fetched.

## Key Components

**`UsersGridSkeleton`** — Default export component that renders a pulsing skeleton table with fixed column headers (`Name`, `Email`, `Role`, `Created`, `Last Sign-In`) and configurable placeholder rows.

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `number` | `10` | Number of skeleton rows to render |
| `className` | `string` | — | Additional CSS classes for the wrapper |

## Usage Example

```typescript
// Basic usage with default 10 rows
<UsersGridSkeleton />

// Custom row count while data loads
{isLoading ? (
  <UsersGridSkeleton rows={5} className="mt-4" />
) : (
  <UsersGrid users={users} />
)}
```

## Notes

- Uses `animate-pulse` (Tailwind) for the shimmer effect
- Each row renders 5 cells matching the 5 column headers
- Styled with ODS design tokens (`ods-border`, `ods-card`, `ods-text-primary`) ensuring consistency with the live users table
- `cn()` utility allows safe class merging for layout overrides
