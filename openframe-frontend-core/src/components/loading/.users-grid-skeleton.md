A loading skeleton component that renders an animated placeholder table for the users grid while data is being fetched.

## Key Components

- **`UsersGridSkeleton`** — Default export component that renders a pulsing table skeleton with fixed columns: `Name`, `Email`, `Role`, `Created`, and `Last Sign-In`

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `number` | `10` | Number of skeleton rows to render |
| `className` | `string` | — | Additional CSS classes for the wrapper |

## Usage Example

```typescript
// Basic usage with default 10 rows
<UsersGridSkeleton />

// Custom row count while data loads
{isLoading ? (
  <UsersGridSkeleton rows={5} className="mt-4" />
) : (
  <UsersGrid users={users} />
)}
```

## Notes

- Uses `animate-pulse` for the Tailwind CSS shimmer effect
- Applies ODS design tokens (`ods-border`, `ods-card`, `ods-text-primary`) for consistent dark-mode theming
- Each skeleton cell renders a full-width `h-4` bar — no column-specific width variation
- `cn()` utility merges class names, allowing style overrides via `className`
