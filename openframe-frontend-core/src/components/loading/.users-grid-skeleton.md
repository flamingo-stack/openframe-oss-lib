A loading skeleton component that displays a placeholder table structure while user data is being fetched, providing visual feedback during loading states.

## Key Components

- **`UsersGridSkeleton`** - Main component that renders an animated skeleton table
- **`Props` interface** - Defines optional `rows` (default: 10) and `className` properties
- **Table structure** - Renders skeleton rows with 5 columns matching a typical users grid layout

## Usage Example

```typescript
import { UsersGridSkeleton } from './components/skeletons/users-grid-skeleton'

function UsersPage() {
  const [users, setUsers] = useState(null)
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    fetchUsers().then(data => {
      setUsers(data)
      setLoading(false)
    })
  }, [])

  if (loading) {
    return <UsersGridSkeleton rows={15} className="mt-4" />
  }

  return <UsersTable data={users} />
}
```

The component displays animated placeholder rows with shimmer effects, maintaining the same visual structure as the actual users table (Name, Email, Role, Created, Last Sign-In columns) for a seamless loading experience.
