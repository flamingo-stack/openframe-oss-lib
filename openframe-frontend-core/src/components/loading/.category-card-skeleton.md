A loading skeleton component that mimics the structure and layout of a category card while content is being fetched.

## Key Components

**CategoryCardSkeleton** - Main component that renders a skeleton placeholder with:
- Row of 10 circular icon placeholders
- Text skeleton elements (heading and body lines)
- Arrow button placeholder
- Proper accessibility attributes

**Props:**
- `className?: string` - Optional CSS classes for styling customization
