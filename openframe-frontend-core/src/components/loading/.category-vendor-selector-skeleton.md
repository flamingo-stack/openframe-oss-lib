A React skeleton component that displays loading placeholders for category vendor selector sections while data is being fetched.

## Key Components

- **CategoryVendorSelectorSkeleton**: Main component that renders a structured loading state with header, subcategory titles, and vendor slot placeholders
- **CategoryVendorSelectorSkeletonProps**: Interface defining component props including customizable subcategory count and CSS classes
- **TextSkeleton & UnifiedSkeleton**: Imported skeleton primitives for consistent loading animations

## Usage Example

```typescript
import { CategoryVendorSelectorSkeleton } from './category-vendor-selector-skeleton'

// Basic usage with default 3 subcategories
function LoadingState() {
  return <CategoryVendorSelectorSkeleton />
}

// Custom subcategory count and styling
function CustomLoadingState() {
  return (
    <CategoryVendorSelectorSkeleton 
      subcategories={5}
      className="my-4 shadow-lg"
    />
  )
}

// Conditional rendering during data fetch
function CategoryPage() {
  const { data, loading } = useCategoryData()
  
  if (loading) {
    return <CategoryVendorSelectorSkeleton subcategories={4} />
  }
  
  return <CategoryVendorSelector data={data} />
}
```

The component uses a responsive grid layout (2 columns on mobile, 3 on desktop) and includes proper accessibility attributes for screen readers during loading states.
