A loading skeleton component that displays placeholder content for a category vendor selector interface with configurable subcategory sections.

## Key Components

- **`CategoryVendorSelectorSkeleton`**: Main component that renders a loading state with header and subcategory sections
- **`CategoryVendorSelectorSkeletonProps`**: Interface defining props including optional `subcategories` count and `className`
- **Skeleton Elements**: Uses `TextSkeleton` for headings/text and `UnifiedSkeleton` for vendor card placeholders

## Usage Example

```typescript
import { CategoryVendorSelectorSkeleton } from './category-vendor-selector-skeleton'

// Basic usage with default 3 subcategories
function LoadingState() {
  return <CategoryVendorSelectorSkeleton />
}

// Custom subcategory count with styling
function CustomLoadingState() {
  return (
    <CategoryVendorSelectorSkeleton 
      subcategories={5}
      className="shadow-lg"
    />
  )
}

// In a conditional render
function CategoryPage({ isLoading, data }) {
  if (isLoading) {
    return <CategoryVendorSelectorSkeleton subcategories={4} />
  }
  
  return <CategoryVendorSelector data={data} />
}
```

The component renders a card-style container with a header section (title and description skeletons) followed by the specified number of subcategory blocks, each containing a title and a 2-column grid of vendor placeholder cards.
