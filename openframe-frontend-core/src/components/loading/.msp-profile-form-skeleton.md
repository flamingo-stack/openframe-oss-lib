A skeleton loading component that displays placeholder content while the MSP (Managed Service Provider) profile form is loading, featuring a responsive two-column layout with logo upload area and configurable input fields.

## Key Components

- **MspProfileFormSkeleton** - Main skeleton component with responsive grid layout
- **SkeletonProps** - Interface defining optional className and configurable fields count
- **Left Column** - Logo uploader placeholder with label and media skeleton
- **Right Column** - Grid of label/input skeleton pairs based on fields prop

## Usage Example

```typescript
import { MspProfileFormSkeleton } from './msp-profile-form-skeleton'

// Basic usage with default 4 fields
function ProfilePage() {
  return (
    <div>
      {loading ? (
        <MspProfileFormSkeleton />
      ) : (
        <MspProfileForm />
      )}
    </div>
  )
}

// Custom field count and styling
function CustomForm() {
  return (
    <MspProfileFormSkeleton 
      fields={6}
      className="max-w-4xl mx-auto p-4"
    />
  )
}
```

The component automatically handles responsive breakpoints, collapsing from two columns to single column on mobile devices. It includes proper accessibility attributes and maintains visual consistency with the actual form layout.
