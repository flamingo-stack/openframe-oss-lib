A React component that displays an animated loading screen with a pulsing heart-like animation, progress indicator, and status message for OpenMSP report generation.

## Key Components

- **OpenmspHeartbeatLoader** - Main component that creates a full-screen loading interface with:
  - Animated OpenMSP logo with heartbeat-style pulsing animation
  - Progress percentage display
  - Customizable status label
  - Background processing notification
  - Accessibility support with ARIA labels

## Usage Example

```typescript
// Basic usage with default settings
<OpenmspHeartbeatLoader />

// With progress tracking
<OpenmspHeartbeatLoader 
  progress={45} 
  label="Analyzing data..." 
/>

// With custom styling and progress bar width
<OpenmspHeartbeatLoader 
  className="bg-gray-100" 
  progress={75}
  label="Generating final report..."
  barWidth="w-96"
/>

// In a report generation flow
function ReportGenerator() {
  const [progress, setProgress] = useState(0);
  const [status, setStatus] = useState("Initializing...");

  return (
    <OpenmspHeartbeatLoader 
      progress={progress}
      label={status}
    />
  );
}
```

The component features a distinctive double-beat heartbeat animation (lub-dub pattern) that cycles every 3.6 seconds, providing visual feedback during long-running background processes while allowing users to navigate away safely.
