A React component that displays a heartbeat-animated loading screen with the OpenMSP logo, progress indicator, and user-friendly messaging. The component features a custom CSS animation that mimics a double-beat heartbeat pattern applied to the logo.

## Key Components

- **OpenmspHeartbeatLoader**: Main component with customizable progress display and status messaging
- **Heartbeat Animation**: Inline CSS keyframes creating a realistic lub-dub heartbeat effect
- **OpenmspLogo**: Animated logo component with custom colors
- **Progress Display**: Shows percentage completion and current status label

## Usage Example

```typescript
import { OpenmspHeartbeatLoader } from './openmsp-heartbeat';

// Basic usage
<OpenmspHeartbeatLoader />

// With progress and custom label
<OpenmspHeartbeatLoader 
  progress={45} 
  label="Analyzing data..." 
  className="bg-gray-100"
/>

// Custom bar width and styling
<OpenmspHeartbeatLoader 
  progress={75}
  label="Generating report..."
  barWidth="w-96"
  className="min-h-full"
/>
```

The component automatically handles accessibility with ARIA labels and provides reassuring messaging that users can navigate away during processing. The heartbeat animation runs continuously with a 3.6-second cycle, creating an engaging visual feedback loop.
