A loading skeleton placeholder component for the Margin Report page, rendering animated gray blocks that mirror the layout of the full report while data is being fetched or access is gated.

## Key Components

### `MarginReportSkeleton`
Main export. Renders a full-page skeleton that mimics the Margin Report structure including:
- **Header** — title and subtitle placeholders
- **Summary cards** — 3-column grid of card blocks
- **MSP Profile & Report Info** — 2-column info card row
- **Vendor solution lists** — Commercial Stack vs Open-Source Stack with 5 vendor row placeholders each
- **Strategic Recommendations** — 3-column card grid
- **OpenFrame value section** — feature card grid with logo placeholder
- **CTA Overlay** — optional viewport-centered overlay with description and action content

### `MarginReportSkeletonProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `animate` | `boolean` | `true` | Toggles `animate-pulse` Tailwind animation |
| `description` | `React.ReactNode` | — | Explanatory text shown above overlay content |
| `overlayContent` | `React.ReactNode` | — | Node rendered centered over the skeleton (e.g. a CTA button or spinner) |

## Usage Example

```typescript
// Basic animated skeleton while report loads
<MarginReportSkeleton />

// Gated report with CTA overlay
<MarginReportSkeleton
  animate={true}
  description="Generate your free Margin Report to unlock savings insights."
  overlayContent={
    <button onClick={handleGenerate}>
      Generate Report
    </button>
  }
/>

// Static snapshot (no animation)
<MarginReportSkeleton animate={false} />
```
