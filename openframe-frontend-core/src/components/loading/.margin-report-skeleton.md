A React component that renders a skeleton loading state for a margin report page with optional overlay content and animation controls.

## Key Components

**MarginReportSkeleton** - Main skeleton component that displays placeholder elements for:
- Page header with title and description placeholders
- Summary cards grid (3 cards)
- MSP profile and report info sections
- Commercial and Open-Source vendor solution lists
- Strategic recommendations grid
- OpenFrame value proposition section
- Optional overlay with custom content

**MarginReportSkeletonProps** - Interface defining:
- `animate` - Controls pulse animation (default: true)
- `description` - Optional text shown above overlay content
- `overlayContent` - Custom React node displayed over skeleton

## Usage Example

```typescript
import { MarginReportSkeleton } from './margin-report-skeleton';

// Basic skeleton with animation
<MarginReportSkeleton />

// Skeleton with custom overlay content
<MarginReportSkeleton
  animate={true}
  description="Please wait while we generate your report..."
  overlayContent={
    <button className="px-6 py-2 bg-blue-500 text-white rounded">
      Generate Report
    </button>
  }
/>

// Static skeleton without animation
<MarginReportSkeleton animate={false} />
```

The component uses Tailwind CSS classes and follows a dark theme design pattern with responsive grid layouts that adapt to different screen sizes.
