A comprehensive skeleton loading component system providing consistent placeholder UI elements with accessibility support and responsive design.

## Key Components

- **`UnifiedSkeleton`** - Base skeleton component with customizable variants (default, text, circular, rectangular)
- **`TextSkeleton`** - Pre-configured text loading skeletons:
  - `Heading` - Large headings (h1, h2)
  - `Subheading` - Medium headings (h3, h4)  
  - `Body` - Body text content
  - `Caption` - Small text and metadata
- **`InteractiveSkeleton`** - UI element loading skeletons:
  - `Button` - Button placeholders with proper touch targets
  - `Input` - Form input field skeletons
  - `Chip` - Filter/tag placeholders
- **`MediaSkeleton`** - Image and icon loading skeletons:
  - `Avatar` - Circular profile images with size variants
  - `CardImage` - 16:9 aspect ratio image placeholders
  - `Icon` - Square icon placeholders with size variants

## Usage Example

```typescript
import { UnifiedSkeleton, TextSkeleton, MediaSkeleton } from './unified-skeleton'

// Basic skeleton
<UnifiedSkeleton variant="text" width={200} height={20} />

// Text skeletons
<TextSkeleton.Heading className="mb-4" />
<TextSkeleton.Body />

// Media skeletons  
<MediaSkeleton.Avatar size="lg" />
<MediaSkeleton.CardImage />

// Custom skeleton with no animation
<UnifiedSkeleton 
  variant="rectangular" 
  animate={false}
  aria-label="Loading chart data"
/>
```
