A React component that renders a customizable moon SVG icon, commonly used for dark mode toggles and theme switching interfaces.

## Key Components

- **MoonIcon** - Main functional component that renders the moon SVG
- **MoonIconProps** - TypeScript interface defining component props:
  - `className` - Optional CSS class string
  - `width` - Optional width (defaults to 23px)
  - `height` - Optional height (defaults to 24px)

## Usage Example

```typescript
import { MoonIcon } from './moon-icon';

// Basic usage with defaults
<MoonIcon />

// With custom styling and size
<MoonIcon 
  className="text-gray-600 hover:text-gray-800" 
  width={32} 
  height={32} 
/>

// In a dark mode toggle button
<button onClick={toggleDarkMode}>
  <MoonIcon className="w-5 h-5" />
</button>
```

The icon uses `currentColor` for the fill, making it easy to style with CSS text color properties. The SVG maintains its aspect ratio and scales proportionally when resized.
