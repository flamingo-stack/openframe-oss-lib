A React hover card component built on top of Radix UI primitives that displays contextual content when hovering over a trigger element. It provides styled wrapper components with smooth animations and customizable positioning.

## Key Components

- **`HoverCard`** - Root container component that manages hover state
- **`HoverCardTrigger`** - Element that triggers the hover card when hovered
- **`HoverCardContent`** - The content panel with built-in styling and animations

## Usage Example

```typescript
import { HoverCard, HoverCardTrigger, HoverCardContent } from "./hover-card"

function UserProfile() {
  return (
    <HoverCard>
      <HoverCardTrigger asChild>
        <button className="text-blue-500 underline">
          @username
        </button>
      </HoverCardTrigger>
      <HoverCardContent className="w-80">
        <div className="flex gap-4">
          <img src="/avatar.jpg" alt="Avatar" className="w-12 h-12 rounded-full" />
          <div>
            <h4 className="font-semibold">John Doe</h4>
            <p className="text-sm text-muted-foreground">
              Software developer passionate about React and TypeScript
            </p>
          </div>
        </div>
      </HoverCardContent>
    </HoverCard>
  )
}
```

The component includes smooth fade and zoom animations, customizable alignment, and follows accessibility best practices through Radix UI primitives.
