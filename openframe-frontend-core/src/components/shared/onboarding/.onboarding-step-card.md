A React component that renders an interactive card for onboarding steps with completion states, action buttons, and skip functionality.

## Key Components

- **OnboardingStepCard**: Main component that displays step information with dynamic UI based on completion status
- **OnboardingStepCardProps**: TypeScript interface defining step data structure and component props
- **handleAction**: Async function that manages action button clicks with loading state
- **handleSkip**: Function that handles skip button clicks

## Usage Example

```typescript
import { OnboardingStepCard } from './components/onboarding-step-card'

const step = {
  id: 'connect-wallet',
  title: 'Connect Your Wallet',
  description: 'Link your crypto wallet to get started',
  actionIcon: (color) => <WalletIcon color={color} />,
  actionText: 'Connect Wallet',
  completedText: 'View Wallet',
  onAction: async () => await connectWallet(),
  onSkip: () => skipStep('connect-wallet')
}

<OnboardingStepCard
  step={step}
  isCompleted={false}
  isSkipped={false}
  isCheckingCompletion={false}
  onAction={() => handleStepAction(step.id)}
  onSkip={() => handleSkipStep(step.id)}
/>
```

The component displays different states: loading skeletons during completion checks, success badges for completed steps, skip badges for skipped steps, or action/skip buttons for pending steps.
