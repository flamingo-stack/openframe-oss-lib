A React component that displays a guided onboarding walkthrough with step-by-step cards, automatic completion tracking, and persistent state management.

## Key Components

**OnboardingWalkthrough** - Main component that renders the complete onboarding experience
- `steps` - Array of onboarding step configurations
- `onDismiss` - Optional callback when walkthrough is dismissed
- `storageKey` - Key for localStorage persistence (default: 'openframe-onboarding-state')
- `completionStatus` - External completion status mapping for automatic step marking
- `isLoadingCompletion` - Loading state for completion checks

**Key Features:**
- Automatic step completion syncing from external hooks
- Race condition prevention with refs
- Responsive design (mobile-first)
- Skip/dismiss functionality
- Persistent state across sessions

## Usage Example

```typescript
import { OnboardingWalkthrough } from './onboarding-walkthrough'

const steps = [
  {
    id: 'setup-profile',
    title: 'Complete Your Profile',
    description: 'Add your basic information',
    onAction: async () => router.push('/profile'),
    checkComplete: async () => user?.profileComplete ?? false
  },
  {
    id: 'knowledge-base',
    title: 'Explore Knowledge Base',
    description: 'Learn about our features',
    onAction: async () => window.open('/docs'),
    // Auto-completes without verification needed
  }
]

function App() {
  const completionStatus = { 'setup-profile': true }
  
  return (
    <OnboardingWalkthrough
      steps={steps}
      completionStatus={completionStatus}
      onDismiss={() => console.log('Dismissed')}
    />
  )
}
```
