This file serves as the main entry point for the onboarding UI components and related hooks, providing a centralized export interface for consumer modules.

## Key Components

- **OnboardingWalkthrough** - Main walkthrough component for guided user onboarding flows
- **OnboardingStepCard** - Individual step card component for displaying onboarding instructions
- **useOnboardingState** - Custom hook for managing onboarding state and navigation
- **Type Exports** - TypeScript interfaces for component props and state management

## Usage Example

```typescript
import { 
  OnboardingWalkthrough, 
  OnboardingStepCard, 
  useOnboardingState,
  type OnboardingStepConfig 
} from './index'

// Using the onboarding system
function MyApp() {
  const onboardingState = useOnboardingState();
  
  const steps: OnboardingStepConfig[] = [
    { id: 'welcome', title: 'Welcome', content: 'Getting started...' },
    { id: 'features', title: 'Features', content: 'Explore features...' }
  ];

  return (
    <OnboardingWalkthrough steps={steps} state={onboardingState}>
      <OnboardingStepCard stepId="welcome" />
      <OnboardingStepCard stepId="features" />
    </OnboardingWalkthrough>
  );
}
```

This barrel export pattern simplifies imports and maintains a clean API surface for the onboarding module.
