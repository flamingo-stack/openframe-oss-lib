This file serves as a centralized export barrel for onboarding-related components and utilities, providing a clean API for implementing user onboarding flows in React applications.

## Key Components

- **OnboardingWalkthrough** - Main component for creating guided onboarding experiences
- **OnboardingStepCard** - Individual step component for displaying onboarding content
- **useOnboardingState** - React hook for managing onboarding state and navigation
- **Type Exports** - TypeScript interfaces for props and configuration objects

## Usage Example

```typescript
import { 
  OnboardingWalkthrough, 
  OnboardingStepCard,
  useOnboardingState,
  type OnboardingStepConfig 
} from './path/to/onboarding'

function App() {
  const steps: OnboardingStepConfig[] = [
    { id: 'welcome', title: 'Welcome', content: 'Get started!' },
    { id: 'features', title: 'Features', content: 'Explore features' }
  ]
  
  const onboarding = useOnboardingState(steps)

  return (
    <OnboardingWalkthrough {...onboarding}>
      <OnboardingStepCard 
        title="Welcome"
        content="Welcome to our app!"
        onNext={onboarding.nextStep}
      />
    </OnboardingWalkthrough>
  )
}
```
