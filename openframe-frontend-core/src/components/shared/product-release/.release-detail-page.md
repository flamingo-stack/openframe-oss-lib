A React component that renders a detailed view page for product releases with dynamic content sections and injectable dependencies.

## Key Components

**Main Export:**
- `ReleaseDetailPage` - Primary component that displays release information with metadata, changelog, and linked tasks

**Types & Interfaces:**
- `ReleaseDetailPageProps` - Main component props including injectable components and API endpoints
- `MarkdownRendererProps`, `RoadmapSectionProps`, `DeliverySectionProps` - Props for injectable components
- `UseReleaseResult` - Type for the release data fetching hook

**Features:**
- Responsive metadata grid showing release type, status, date, and author
- Image gallery with modal viewer
- YouTube video embedding
- Dynamic roadmap and delivery task fetching
- Breaking changes alerts and changelog sections
- External link integration (GitHub, documentation, migration guides)

## Usage Example

```typescript
import { ReleaseDetailPage } from './release-detail-page';
import { useRelease } from '../hooks/use-release';
import { MarkdownRenderer } from '../components/markdown';

function ReleasePage({ params }: { params: { slug: string } }) {
  return (
    <ReleaseDetailPage
      slug={params.slug}
      useRelease={useRelease}
      MarkdownRenderer={MarkdownRenderer}
      roadmapApiEndpoint="/api/roadmap"
      deliveryApiEndpoint="/api/delivery"
    />
  );
}
```

The component supports dependency injection for markdown rendering and task sections, making it adaptable to different application architectures while maintaining consistent UI presentation.
