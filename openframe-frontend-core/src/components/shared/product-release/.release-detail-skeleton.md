A React component that provides a skeleton loading state specifically for release detail pages, utilizing a reusable detail page skeleton with release-specific configuration.

## Key Components

- **ReleaseDetailSkeleton**: Main component that renders a skeleton placeholder for release detail pages
- **DetailPageSkeleton**: Imported base skeleton component that handles the actual skeleton rendering

## Usage Example

```typescript
import { ReleaseDetailSkeleton } from './release-detail-skeleton';

// Use in a release detail page while data is loading
function ReleaseDetailPage({ releaseId }: { releaseId: string }) {
  const { data: release, isLoading } = useRelease(releaseId);

  if (isLoading) {
    return <ReleaseDetailSkeleton />;
  }

  return (
    <div>
      {/* Actual release content */}
    </div>
  );
}

// Or in a Suspense fallback
<Suspense fallback={<ReleaseDetailSkeleton />}>
  <ReleaseDetail id="release-123" />
</Suspense>
```

The component is pre-configured with 4 metadata columns and an image gallery placeholder, making it ideal for music releases or similar content that typically displays multiple images and detailed metadata.
