A React component that renders a loading skeleton placeholder for product release cards, matching the layout structure of the actual card with animated placeholder elements.

## Key Components

- **ProductReleaseCardSkeleton**: Main skeleton component with responsive flexbox layout
- **ProductReleaseCardSkeletonProps**: Interface defining optional className prop
- **Skeleton Elements**: Placeholder bars for title (2 lines), description (3 lines), version info, and icon

## Usage Example

```typescript
import { ProductReleaseCardSkeleton } from './components/product-release-card-skeleton'

// Basic usage
<ProductReleaseCardSkeleton />

// With custom styling
<ProductReleaseCardSkeleton className="mb-4 shadow-lg" />

// Loading state in a list
{isLoading ? (
  <div className="space-y-4">
    {Array.from({ length: 3 }).map((_, index) => (
      <ProductReleaseCardSkeleton key={index} />
    ))}
  </div>
) : (
  productReleases.map(release => <ProductReleaseCard key={release.id} {...release} />)
)}
```

The component uses `animate-pulse` for subtle loading animation and maintains responsive behavior that matches the actual product release card layout on both mobile and desktop viewports.
