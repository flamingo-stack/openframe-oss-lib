Formats a full document path into a human-readable breadcrumb trail, showing only parent folder segments (excluding the filename).

## Key Components

- **`formatRelativePath(fullPath: string): string`** — Strips the `.md` extension, splits on `/`, excludes the last segment (filename), then capitalizes and replaces hyphens with spaces in each remaining segment, joining them with ` / `.

## Usage Example

```typescript
import { formatRelativePath } from './format-relative-path'

formatRelativePath('openframe-oss-tenant/architecture/api-controllers.md')
// → 'Openframe oss tenant / Architecture'

formatRelativePath('docs/getting-started.md')
// → 'Docs'

formatRelativePath('')
// → ''
```
