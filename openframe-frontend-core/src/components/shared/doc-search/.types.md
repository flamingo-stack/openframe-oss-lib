Single wire-shape contract for one document/entity search result returned by `/api/docs/search`.

## Key Components

### `DocSearchResult` interface

| Field | Type | Description |
|-------|------|-------------|
| `path` | `string` | Internal path identifier for the result |
| `name` | `string` | Display name of the document or folder |
| `type` | `'file' \| 'folder'` | Whether the result is a file or folder node |
| `snippet` | `string` | Preview text excerpt shown in search results |
| `matchType` | `'title' \| 'content'` | Whether the match was on title or body content |
| `documentType` | `string?` | Optional document category/type classifier |
| `externalUrl` | `string?` | External viewer URL when the doc lives outside the hub |
| `targetPlatform` | `string \| null \| undefined` | Platform owning `externalUrl`; used by `decideNewTab` to determine same-tab vs. new-tab navigation |
| `sourceRepo` | `string?` | Repository the document originates from |
| `entityId` | `string?` | Primary key value from `RagTableConfig`; enables chat panel drill-in for entity rows that have no public viewer URL |

## Usage Example

```typescript
import type { DocSearchResult } from './types'

// Consuming a search API response
const results: DocSearchResult[] = await fetch('/api/docs/search?q=firewall')
  .then(res => res.json())

results.forEach(result => {
  if (result.entityId) {
    // No public URL — open chat panel with entity drill-in
    openChatPanel({ entityIdFilter: result.entityId })
  } else if (result.externalUrl) {
    const newTab = decideNewTab(result.targetPlatform)
    navigate(result.externalUrl, { newTab })
  } else {
    navigate(result.path)
  }
})
```

> **Sync contract:** This interface must remain field-for-field identical with `lib/data/doc-search-utils.ts:DocSearchResult` on the hub side. Any field additions must be mirrored in both locations.
