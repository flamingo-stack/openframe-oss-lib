React hook for fetching markdown-backed legal documents (privacy policy, terms of service, etc.) from a hub API endpoint, with optional SSR/RSC `initialData` support.

## Key Components

### Interfaces

| Interface | Description |
|-----------|-------------|
| `LegalDocument` | Shape of the fetched document, mirroring the server-side `LegalDocument` type (avoids importing server types in client code) |
| `UseLegalDocsOptions` | Optional `initialData` (SSR pre-fetch) and `apiEndpoint` override |
| `UseLegalDocsReturn` | `{ data, isLoading, error, refetch }` |

### `useLegalDocs(docType, options?)`

| Param | Type | Description |
|-------|------|-------------|
| `docType` | `string` | Document identifier — drives the default endpoint path (`/api/legal/<docType>`). Common values: `'privacy'`, `'terms'` |
| `options.initialData` | `LegalDocument \| null` | Pre-fetched payload from server; skips initial client fetch when provided |
| `options.apiEndpoint` | `string` | Override the default endpoint (e.g. `/proxy/legal/privacy` for reverse-proxy embedders) |

**Behaviours:**
- Resets cached data when `docType` changes, preventing stale content flicker during sequential doc loads
- Skips fetch on mount when `initialData` is present
- Guards against tainted format-string injection (CodeQL `js/tainted-format-string`) by keeping `console.error` format string constant

## Usage Example

```typescript
import { useLegalDocs } from '@/hooks/use-legal-docs';

// Basic usage — fetches from /api/legal/privacy
function PrivacyPage() {
  const { data, isLoading, error, refetch } = useLegalDocs('privacy');

  if (isLoading) return <Spinner />;
  if (error) return <ErrorMessage message={error} onRetry={refetch} />;

  return <MarkdownRenderer content={data?.content} />;
}

// With SSR pre-fetch — skips client fetch on first render
function TermsPage({ initialData }: { initialData: LegalDocument }) {
  const { data } = useLegalDocs('terms', { initialData });

  return <MarkdownRenderer content={data?.content} />;
}

// Reverse-proxy embedder with custom endpoint
function EmbeddedPrivacy() {
  const { data } = useLegalDocs('privacy', {
    apiEndpoint: '/proxy/legal/privacy',
  });

  return <MarkdownRenderer content={data?.content} />;
}
```
