Unified page component for rendering markdown-backed legal documents (privacy policy, terms of service, etc.), replacing duplicate per-document hub components with a single configurable implementation.

## Key Components

### `LegalDocumentPage`
Main exported component that handles all legal document rendering states: loading skeleton, error display, empty state, and full markdown content.

### `LegalDocumentPageProps`
Configuration interface covering:
- `docType` — drives the default API endpoint `/api/legal/<docType>`
- `title`, `fallbackDescription`, `contactEmail` — document-specific copy
- `errorTitle`, `errorContactPrompt`, `emptyStateMessage` — user-facing error/empty strings
- `initialData` / `initialLastUpdatedLabel` — SSR hydration props
- `apiEndpoint` — optional override for reverse-proxy or alternate API paths
- `MarkdownRenderer` — injectable renderer component (defaults to `SimpleMarkdownRenderer`)
- `backButton` — back-nav config or `false` to suppress

### `LegalDocumentMarkdownRendererProps`
Interface contract for custom `MarkdownRenderer` components, accepting `content`, optional `sectionIds`, and `demoteMarkdownH1ToH2`.

## Usage Example

```typescript
// Privacy Policy page
<LegalDocumentPage
  docType="privacy"
  title="Privacy Policy"
  fallbackDescription="Our privacy policy and data protection practices."
  contactEmail="privacy@openframe.io"
  errorTitle="Unable to load privacy policy"
  errorContactPrompt="For privacy-related questions, please contact:"
  emptyStateMessage="Privacy policy content is not available at this time."
  initialData={ssrDocument}
  initialLastUpdatedLabel="January 1, 2025"
  backButton={{ label: 'Back to home', href: '/' }}
/>

// Terms of Service with custom renderer and hidden back button
<LegalDocumentPage
  docType="terms"
  title="Terms of Service"
  fallbackDescription="Our terms and conditions."
  contactEmail="legal@openframe.io"
  errorTitle="Unable to load terms of service"
  errorContactPrompt="For legal questions, please contact:"
  emptyStateMessage="Terms of service content is not available."
  MarkdownRenderer={RichMarkdownRenderer}
  backButton={false}
/>
```
