A React skeleton component that displays loading placeholders for detail pages while content is being fetched. It provides customizable skeleton elements for titles, metadata, images, and content sections.

## Key Components

- **`DetailPageSkeleton`** - Main skeleton component with configurable layout options
- **`DetailPageSkeletonProps`** - Interface defining component configuration options
  - `metadataColumns` - Number of metadata grid columns (default: 4)
  - `showImageGallery` - Toggle between horizontal gallery or featured image layout

## Usage Example

```typescript
import { DetailPageSkeleton } from './detail-page-skeleton';

// Basic usage with defaults
<DetailPageSkeleton />

// Customized for case studies (featured image instead of gallery)
<DetailPageSkeleton 
  showImageGallery={false}
  metadataColumns={3}
/>

// Product detail page with more metadata columns
<DetailPageSkeleton 
  showImageGallery={true}
  metadataColumns={6}
/>
```

The component renders animated skeleton placeholders including a title block, category tags, configurable metadata grid, optional image gallery or featured image, and multiple content sections. Perfect for maintaining layout consistency while data loads.
