A React component that renders a horizontal row of social media icon buttons, supporting multiple platforms with customizable styling and links.

## Key Components

- **`SocialIconRow`** - Main component that renders a flex row of social media buttons
- **`SocialLink`** - Interface defining platform, href, and optional label properties
- **`SocialIconRowProps`** - Component props including className, links array, and button variant
- **`renderSocialIcon()`** - Utility function that maps platform names to appropriate icons
- **`defaultLinks`** - Default social links for GitHub, LinkedIn, and Facebook

## Usage Example

```typescript
import { SocialIconRow } from './social-icon-row';

// Basic usage with default links
<SocialIconRow />

// Custom links and styling
<SocialIconRow 
  className="my-4"
  variant="ghost"
  links={[
    { platform: 'github', href: 'https://github.com/myorg', label: 'GitHub' },
    { platform: 'twitter', href: 'https://twitter.com/myhandle', label: 'Twitter' },
    { platform: 'linkedin', href: 'https://linkedin.com/in/myprofile' }
  ]}
/>

// Supports 13+ platforms including GitHub, Twitter/X, LinkedIn, 
// Facebook, Instagram, YouTube, WhatsApp, Telegram, and more
```

The component automatically handles platform name variations (e.g., 'twitter'/'x', 'youtube'/'yt') and falls back to a globe icon for unrecognized platforms.
