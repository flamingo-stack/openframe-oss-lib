A flexible React component for displaying a row of social media icons as interactive buttons with customizable links and styling variants.

## Key Components

- **`SocialIconRow`** - Main component that renders a row of social media icon buttons
- **`SocialLink`** - Interface defining platform, href, and optional label for each social link
- **`SocialIconRowProps`** - Props interface supporting custom className, links array, and button variants
- **`renderSocialIcon()`** - Helper function that maps platform names to appropriate icons with normalization
- **`defaultLinks`** - Pre-configured social links for GitHub, LinkedIn, and Facebook

## Usage Example

```typescript
import { SocialIconRow } from './social-icon-row';

// Using default links
function Footer() {
  return <SocialIconRow variant="ghost" className="max-w-sm" />;
}

// With custom links
function ProfileCard() {
  const customLinks = [
    { platform: 'github', href: 'https://github.com/username', label: 'My GitHub' },
    { platform: 'twitter', href: 'https://twitter.com/username', label: 'Follow me' },
    { platform: 'linkedin', href: 'https://linkedin.com/in/username' }
  ];

  return (
    <SocialIconRow 
      links={customLinks}
      variant="secondary"
      className="justify-center"
    />
  );
}
```

The component supports 13+ social platforms with intelligent platform name matching (e.g., 'twitter'/'x', 'youtube'/'yt') and falls back to a globe icon for unrecognized platforms. All links open in new tabs with proper accessibility attributes.
