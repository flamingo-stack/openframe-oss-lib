A React SVG icon component that renders an edit profile icon with a document and pencil design, extending standard SVG properties for flexible usage.

## Key Components

- **EditProfileIcon**: Main functional component that renders the SVG icon
- **EditProfileIconProps**: TypeScript interface extending React.SVGProps<SVGSVGElement> with optional className
- **SVG Paths**: Two path elements creating a document with edit pencil visual design

## Usage Example

```typescript
import { EditProfileIcon } from './edit-profile-icon';

// Basic usage
<EditProfileIcon />

// With custom styling
<EditProfileIcon className="text-blue-500 hover:text-blue-700" />

// With additional SVG properties
<EditProfileIcon 
  width={32} 
  height={32} 
  onClick={handleEdit}
  aria-label="Edit profile"
/>

// In a button
<button className="p-2 rounded-md">
  <EditProfileIcon className="w-5 h-5" />
  Edit Profile
</button>
```

The component uses a 24x24 viewBox with white (#FAFAFA) fill color and supports all standard SVG properties through prop spreading, making it highly customizable for different UI contexts.
