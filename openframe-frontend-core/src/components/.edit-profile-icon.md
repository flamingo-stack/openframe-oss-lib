A React SVG icon component that renders an edit/pencil icon typically used for profile editing functionality.

## Key Components

- **EditProfileIcon**: Main functional component that renders an SVG edit icon
- **EditProfileIconProps**: TypeScript interface extending `React.SVGProps<SVGSVGElement>` with optional className
- **SVG paths**: Two path elements creating a pencil/edit icon with document background

## Usage Example

```typescript
import { EditProfileIcon } from './edit-profile-icon';

// Basic usage
<EditProfileIcon />

// With custom styling
<EditProfileIcon className="w-6 h-6 text-blue-500" />

// With additional SVG properties
<EditProfileIcon 
  className="hover:opacity-80" 
  onClick={handleEditProfile}
  role="button"
  aria-label="Edit profile"
/>

// In a button component
<button className="p-2 rounded-full bg-gray-100">
  <EditProfileIcon className="w-5 h-5" />
</button>
```

The component accepts all standard SVG props and forwards them to the underlying SVG element, making it highly customizable while maintaining a consistent 24x24 viewBox with white (#FAFAFA) fill paths.
