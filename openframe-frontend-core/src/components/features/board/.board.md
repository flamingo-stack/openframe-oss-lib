Kanban-style drag-and-drop board component that manages multi-column ticket reordering and cross-column moves using `@dnd-kit/core`.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Board` | Component | Root board with full DnD context, column rendering, and custom scrollbar |
| `BoardProps` | Interface | Configuration props for columns, callbacks, and rendering slots |
| `locate` | Helper | Finds a ticket's column, index, and reference by ticket ID |
| `findIndexInColumn` | Helper | Returns a ticket's index within a specific column |

**Key behaviors:**
- `collisionDetection` — custom strategy: prefers ticket hits, falls back to column hits, then `closestCorners`
- `handleDragOver` — handles real-time cross-column moves with `dropDisabled` and `allowedFromColumns` guards
- `handleDragEnd` — emits a `BoardChange` with `afterTicketId`/`beforeTicketId` for ordered persistence
- `useBoardCollapse` — persists per-column collapsed state to storage via `collapseStorageKey`
- `useHorizontalScrollbar` — custom scrollbar track/thumb replacing native browser scrollbar

## Usage Example

```typescript
import { Board } from './board'
import type { BoardColumnDef, BoardChange } from './types'

const columns: BoardColumnDef[] = [
  { id: 'open', label: 'Open', tickets: [...] },
  { id: 'in-progress', label: 'In Progress', tickets: [...], allowedFromColumns: ['open'] },
  { id: 'done', label: 'Done', tickets: [...], dropDisabled: true },
]

function handleChange(change: BoardChange) {
  console.log(`Moved ticket ${change.ticketId} from ${change.fromColumnId} to ${change.toColumnId}`)
  // persist order using afterTicketId / beforeTicketId
}

<Board
  columns={columns}
  onChange={handleChange}
  onLoadMore={(columnId) => fetchMore(columnId)}
  onAddTicket={(columnId) => openCreateModal(columnId)}
  getTicketHref={(id) => `/tickets/${id}`}
  collapseStorageKey="board-collapse-state"
/>
```

> `isDraggingRef` prevents external `columns` prop updates from overwriting optimistic local state mid-drag, ensuring smooth UX without flicker.
