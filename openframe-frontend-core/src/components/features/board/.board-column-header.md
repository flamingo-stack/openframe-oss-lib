Renders the header for a kanban board column, supporting both expanded and collapsed states with ticket status display, ticket count, and optional action buttons.

## Key Components

### `BoardColumnHeaderProps`
| Prop | Type | Description |
|------|------|-------------|
| `column` | `BoardColumnDef` | Column definition including label, tickets, color, and status key |
| `collapsed` | `boolean` | Controls collapsed/expanded layout (default: `false`) |
| `onToggleCollapse` | `() => void` | Callback to toggle the collapsed state |
| `onAddTicket` | `() => void` | Optional callback to add a new ticket to the column |
| `onArchive` | `() => void` | Optional callback to archive resolved tickets |

### `BoardColumnHeader`
The main exported component. Renders two layouts:
- **Collapsed** — vertical layout with expand arrow, rotated status tag, and ticket count
- **Expanded** — horizontal layout with status tag, count, collapse arrow, and optional archive/add-ticket action buttons

Status tags use their canonical variant/icon for known statuses (e.g., `ACTIVE`, `RESOLVED`); custom status UUIDs fall back to the column's `color` prop.

## Usage Example

```typescript
import { BoardColumnHeader } from './board-column-header'

<BoardColumnHeader
  column={{
    id: 'in-progress',
    label: 'In Progress',
    tickets: [],
    total: 12,
    color: '#FFC008',
  }}
  collapsed={false}
  onToggleCollapse={() => setCollapsed(true)}
  onAddTicket={() => openNewTicketModal()}
  onArchive={() => archiveResolved()}
/>
```
