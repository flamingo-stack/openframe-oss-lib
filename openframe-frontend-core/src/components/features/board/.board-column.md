Renders a single kanban board column with drag-and-drop support, collapsible state, infinite scroll via IntersectionObserver, and ticket card rendering.

## Key Components

- **`BoardColumn`** — Main exported component. Renders the column shell with header, optional color tinting, collapse animation, and border-joining props for adjacent columns.
- **`ColumnBody`** — Internal component handling the droppable zone (`useDroppable`), `SortableContext` for ticket reordering, scrollable ticket list, and infinite scroll sentinel.
- **`SkeletonStack`** — Renders `n` `TicketCardSkeleton` placeholders during initial load.
- **`EmptyState`** — Shown when a column has no tickets; displays a tag icon with instructional text.

## Key Props (`BoardColumnProps`)

| Prop | Type | Description |
|------|------|-------------|
| `column` | `BoardColumnDef` | Column definition including tickets, color, and flags |
| `collapsed` | `boolean` | Collapses column to `w-14` with transition |
| `joinLeft` / `joinRight` | `boolean` | Removes border/radius for adjacent column joining |
| `onLoadMore` | `(columnId: string) => void` | Triggered by IntersectionObserver when sentinel enters viewport |
| `renderAssignSlot` | `(ticket: BoardTicket) => ReactNode` | Renders custom assignee UI per ticket |

## Usage Example

```typescript
<BoardColumn
  column={columnDef}
  collapsed={false}
  onToggleCollapse={() => setCollapsed(c => !c)}
  onAddTicket={(colId) => openNewTicketModal(colId)}
  getTicketHref={(ticketId) => `/tickets/${ticketId}`}
  renderAssignSlot={(ticket) => <AvatarPicker ticket={ticket} />}
  onLoadMore={(colId) => fetchNextPage(colId)}
  joinLeft
/>
```
