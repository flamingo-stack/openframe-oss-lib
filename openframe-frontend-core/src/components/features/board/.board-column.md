Renders a single kanban board column with drag-and-drop support, collapsible state, infinite scroll loading, and ticket card display.

## Key Components

- **`BoardColumn`** — Main exported component. Renders a collapsible column shell with a header, optional color tinting, and border-joining utilities (`joinLeft`/`joinRight`) for adjacent column layouts.
- **`ColumnBody`** — Internal component managing the droppable zone (`useDroppable`), sortable ticket list (`SortableContext`), and infinite scroll via `IntersectionObserver`.
- **`SkeletonStack`** — Renders `n` `TicketCardSkeleton` placeholders during initial load.
- **`EmptyState`** — Displays a tag icon and instructional text when a column has no tickets.

## Props (`BoardColumnProps`)

| Prop | Type | Description |
|------|------|-------------|
| `column` | `BoardColumnDef` | Column definition including tickets, color, and loading state |
| `collapsed` | `boolean` | Collapses column to `56px` width |
| `onToggleCollapse` | `() => void` | Toggle handler for collapse state |
| `onAddTicket` | `(columnId: string) => void` | Optional callback to add a ticket |
| `onArchive` | `(columnId: string) => void` | Optional archive callback (only when `column.archivable`) |
| `getTicketHref` | `(ticketId: string) => string` | Generates links for ticket cards |
| `renderAssignSlot` | `(ticket: BoardTicket) => ReactNode` | Renders a custom assignee UI per card |
| `onLoadMore` | `(columnId: string) => void` | Triggered by scroll sentinel for pagination |
| `loadMoreRootMargin` | `string` | `IntersectionObserver` root margin, defaults to `"200px 0px"` |
| `joinLeft` / `joinRight` | `boolean` | Removes border/radius on the joined side |

## Usage Example

```typescript
<BoardColumn
  column={myColumn}
  collapsed={false}
  onToggleCollapse={() => setCollapsed(prev => !prev)}
  onAddTicket={(columnId) => openNewTicketModal(columnId)}
  getTicketHref={(ticketId) => `/tickets/${ticketId}`}
  renderAssignSlot={(ticket) => <AvatarPicker ticket={ticket} />}
  onLoadMore={(columnId) => fetchNextPage(columnId)}
/>
```
