A React component for selecting operating system platforms with visual OS icons and support for disabled states with custom badges.

## Key Components

- **`OSPlatformSelector`** - Main component for platform selection with tab-style interface
- **`OSPlatformSelectorProps`** - Props interface defining configuration options
- **`PlatformSelectorOption`** - Interface for custom platform options with badge support

## Usage Example

```typescript
import { OSPlatformSelector } from '@flamingo/ui-kit/components/features'
import { useState } from 'react'

// Basic usage
const [platform, setPlatform] = useState<OSPlatformId>('darwin')

<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Select Platform"
  disabledPlatforms={['linux']}
/>

// Advanced usage with custom badges
<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Choose OS Platform"
  options={[
    { platformId: 'windows' },
    { platformId: 'darwin' },
    { 
      platformId: 'linux', 
      disabled: true, 
      badge: { text: 'Coming Soon', colorScheme: 'cyan' }
    }
  ]}
  className="my-4"
  iconSize="w-6 h-6"
/>
```

The component automatically builds platform options from `OS_TYPES` if no custom options are provided, supports responsive design, and integrates with the Flamingo Design System (ODS) styling.
