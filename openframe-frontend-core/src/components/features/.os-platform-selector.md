A React component for selecting operating system platforms with tab-style interface, OS icons, and support for disabled states and badges.

## Key Components

- **OSPlatformSelector**: Main component for platform selection with customizable options
- **PlatformSelectorOption**: Interface defining platform options with badges and disabled states
- **OSPlatformSelectorProps**: Props interface with support for custom options, disabled platforms, and styling

## Usage Example

```typescript
import { OSPlatformSelector } from './os-platform-selector'
import { useState } from 'react'

// Basic usage with disabled platforms
const [platform, setPlatform] = useState<OSPlatformId>('darwin')

<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Select Platform"
  disabledPlatforms={['linux']}
/>

// Advanced usage with custom options and badges
<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Choose OS"
  options={[
    { platformId: 'windows' },
    { platformId: 'darwin' },
    { 
      platformId: 'linux', 
      disabled: true, 
      badge: { text: 'Coming Soon', colorScheme: 'cyan' } 
    }
  ]}
  iconSize="w-6 h-6"
/>
```

The component features a responsive design with hover states, accessibility support, and consistent styling that integrates with the ODS design system.
