A React component that provides a tab-style selector for choosing between operating system platforms (Windows, macOS, Linux) with support for disabled states and custom badges.

## Key Components

- **OSPlatformSelector**: Main component that renders a horizontal tab selector with OS icons
- **PlatformSelectorOption**: Interface defining platform options with optional badges and disabled states
- **OSPlatformSelectorProps**: Component props including value, onChange handler, and styling options

## Usage Example

```typescript
import { OSPlatformSelector } from '@flamingo/ui-kit/components/features'
import { useState } from 'react'

// Basic usage with disabled platforms
const [platform, setPlatform] = useState<OSPlatformId>('darwin')

<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Select Platform"
  disabledPlatforms={['linux']}
/>

// Advanced usage with custom badges
<OSPlatformSelector
  value={platform}
  onValueChange={setPlatform}
  label="Choose Your OS"
  options={[
    { platformId: 'windows' },
    { platformId: 'darwin' },
    { 
      platformId: 'linux', 
      disabled: true, 
      badge: { text: 'Coming Soon', colorScheme: 'cyan' } 
    }
  ]}
  iconSize="w-6 h-6"
/>
```

The component features responsive design, ODS (OpenFrame Design System) styling, hover states, and accessibility support with proper cursor states for disabled options.
