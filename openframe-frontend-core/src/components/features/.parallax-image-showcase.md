A React component that creates animated image showcases with parallax effects based on scroll position and mouse movement, supporting multiple layout configurations.

## Key Components

- **ParallaxImageShowcase**: Main component with three layout modes (`non-boxed`, `boxed`, `grid`)
- **Animation Controls**: Configurable intensity settings for scroll and mouse-based transforms
- **Layout Options**: 
  - `non-boxed`: Overlapping images with z-index layering
  - `boxed`: Clean two-row grid with logo element support
  - `grid`: Simple 3-column responsive grid
- **Motion Effects**: Combines Framer Motion transforms for x/y translation and rotation

## Usage Example

```typescript
import { ParallaxImageShowcase } from './parallax-image-showcase'

const images = [
  { src: '/image1.jpg', alt: 'Description 1', position: 'left' as const },
  { src: '/image2.jpg', alt: 'Description 2', position: 'center' as const },
  { src: '/image3.jpg', alt: 'Description 3', position: 'right' as const }
]

// Grid layout with shadow effects
<ParallaxImageShowcase
  images={images}
  layout="grid"
  intensity={2}
  shadow={true}
  className="h-96"
/>

// Boxed layout with custom logo
<ParallaxImageShowcase
  images={images}
  layout="boxed"
  logoElement={<MyLogo />}
  intensity={0.5}
/>
```

The component automatically tracks global mouse movement and scroll position to create smooth parallax animations that respond to user interaction across the entire viewport.
