An interactive React component that creates animated image showcases with parallax effects, supporting three distinct layouts with scroll and mouse-based movement animations.

## Key Components

- **ParallaxImageShowcase**: Main component with three layout modes:
  - `non-boxed`: Stacked images with z-index layering and complex positioning
  - `boxed`: Clean two-row grid with logo support and image overlaps
  - `grid`: Simple responsive three-column grid layout
- **Animation System**: Combines scroll-based and global mouse tracking using Framer Motion
- **Responsive Design**: Adaptive layouts with mobile-friendly breakpoints

## Usage Example

```typescript
import { ParallaxImageShowcase } from './parallax-image-showcase'

// Grid layout with shadows
<ParallaxImageShowcase
  images={[
    { src: '/img1.jpg', alt: 'Screenshot 1', position: 'left' },
    { src: '/img2.jpg', alt: 'Screenshot 2', position: 'center' },
    { src: '/img3.jpg', alt: 'Screenshot 3', position: 'right' }
  ]}
  layout="grid"
  intensity={2}
  shadow={true}
  className="h-96"
/>

// Boxed layout with logo
<ParallaxImageShowcase
  images={images}
  layout="boxed"
  logoElement={<div>My Logo</div>}
  intensity={0.5}
/>

// Non-boxed layered layout
<ParallaxImageShowcase
  images={images}
  layout="non-boxed"
  intensity={1}
/>
```

The component features global mouse tracking, smooth spring animations, and configurable intensity levels for different visual effects. All layouts respond to both scroll position and mouse movement for immersive parallax experiences.
