A responsive dropdown component for multi-section filtering with checkboxes, radio buttons, and custom triggers. Supports mobile-first responsive design with automatic placement optimization.

## Key Components

- **FilterCheckbox**: Custom checkbox component with Flamingo branding
- **FiltersDropdown**: Main dropdown component with responsive positioning
- **FilterOption**: Interface for individual filter options with separator support
- **FilterSection**: Interface for grouped filter sections (checkbox, radio, select types)
- **FiltersDropdownProps**: Complete configuration interface with mobile responsiveness

## Usage Example

```typescript
import { FiltersDropdown, FilterSection } from './filters-dropdown'

const filterSections: FilterSection[] = [
  {
    id: 'status',
    title: 'Status',
    type: 'checkbox',
    allowSelectAll: true,
    options: [
      { id: 'active', label: 'Active', value: 'active' },
      { id: 'pending', label: 'Pending', value: 'pending' },
      { id: 'inactive', label: 'Inactive', value: 'inactive' }
    ]
  },
  {
    id: 'priority',
    title: 'Priority',
    type: 'radio',
    options: [
      { id: 'high', label: 'High', value: 'high' },
      { id: 'medium', label: 'Medium', value: 'medium' },
      { id: 'low', label: 'Low', value: 'low' }
    ]
  }
]

function MyComponent() {
  const [appliedFilters, setAppliedFilters] = useState({})

  const handleApplyFilters = (filters: Record<string, string[]>) => {
    setAppliedFilters(filters)
    // Apply filters to your data
  }

  return (
    <FiltersDropdown
      sections={filterSections}
      onApply={handleApplyFilters}
      onReset={() => setAppliedFilters({})}
      currentFilters={appliedFilters}
      triggerLabel="FILTER STATUS"
      placement="bottom-start"
      responsive={true}
    />
  )
}
```

The component features automatic viewport-aware positioning, mobile-responsive behavior with fixed positioning, click-outside and escape key handling, and state synchronization with external filter changes. It includes custom styling with Flamingo design tokens and supports both default and custom trigger elements.
