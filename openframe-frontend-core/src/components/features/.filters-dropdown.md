A client-side React dropdown component that provides advanced filtering capabilities with multiple filter types, responsive design, and intelligent positioning.

## Key Components

- **`FiltersDropdown`** - Main dropdown component with trigger and filter panels
- **`FilterCheckbox`** - Custom styled checkbox with yellow accent color
- **`FilterOption`** - Interface for individual filter options with separator support
- **`FilterSection`** - Interface for grouped filters (checkbox, radio, or select types)
- **`FiltersDropdownProps`** - Main component props with responsive and positioning options

## Usage Example

```typescript
import { FiltersDropdown, FilterSection } from './filters-dropdown'

const filterSections: FilterSection[] = [
  {
    id: 'status',
    title: 'Status',
    type: 'checkbox',
    allowSelectAll: true,
    options: [
      { id: 'active', label: 'Active', value: 'active' },
      { id: 'inactive', label: 'Inactive', value: 'inactive' }
    ]
  },
  {
    id: 'priority',
    title: 'Priority',
    type: 'radio',
    options: [
      { id: 'high', label: 'High', value: 'high' },
      { id: 'medium', label: 'Medium', value: 'medium' }
    ]
  }
]

const [appliedFilters, setAppliedFilters] = useState<Record<string, string[]>>({})

<FiltersDropdown
  sections={filterSections}
  onApply={setAppliedFilters}
  onReset={() => setAppliedFilters({})}
  currentFilters={appliedFilters}
  triggerLabel="FILTER TICKETS"
  placement="bottom-start"
  responsive={true}
/>
```

The component features intelligent positioning that adapts to viewport constraints, mobile-responsive behavior with full-width display on small screens, and maintains filter state synchronization with external state management.
