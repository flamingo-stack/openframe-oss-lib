A React component that provides a customizable dropdown interface for applying multiple filter criteria with support for checkboxes, radio buttons, and responsive mobile layouts.

## Key Components

- **FiltersDropdown** - Main dropdown component with trigger and filter panel
- **FilterCheckbox** - Custom styled checkbox component with checkmark animation
- **FilterOption** - Interface defining individual filter options with support for separators
- **FilterSection** - Interface for grouping related filters with different input types
- **FiltersDropdownProps** - Configuration interface with responsive and positioning options

## Usage Example

```typescript
import { FiltersDropdown, FilterSection } from './filters-dropdown'

const filterSections: FilterSection[] = [
  {
    id: 'status',
    title: 'Status',
    type: 'checkbox',
    allowSelectAll: true,
    options: [
      { id: 'active', label: 'Active', value: 'active' },
      { id: 'inactive', label: 'Inactive', value: 'inactive' }
    ]
  },
  {
    id: 'priority',
    title: 'Priority',
    type: 'radio',
    options: [
      { id: 'high', label: 'High', value: 'high' },
      { id: 'medium', label: 'Medium', value: 'medium' },
      { id: 'low', label: 'Low', value: 'low' }
    ]
  }
]

function MyComponent() {
  const [appliedFilters, setAppliedFilters] = useState<Record<string, string[]>>({})

  const handleApplyFilters = (filters: Record<string, string[]>) => {
    setAppliedFilters(filters)
    // Apply filters to your data
  }

  return (
    <FiltersDropdown
      sections={filterSections}
      onApply={handleApplyFilters}
      currentFilters={appliedFilters}
      triggerLabel="Filter Items"
      placement="bottom-start"
      responsive={true}
    />
  )
}
```

The component features intelligent positioning to prevent viewport overflow, mobile-responsive behavior, click-outside closing, and maintains filter state between opens. It supports custom trigger elements and provides visual indicators for active filter counts.
