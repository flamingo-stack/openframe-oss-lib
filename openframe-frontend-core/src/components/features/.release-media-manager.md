A React client component for managing release media attachments, supporting file upload, drag-to-reorder, preview, and metadata editing for images and videos.

## Key Components

### `ReleaseMediaItem` (interface)
Represents a single media entry with the following fields:

| Field | Type | Description |
|-------|------|-------------|
| `media_type` | `'image' \| 'video' \| 'screenshot' \| 'demo'` | Category of the media |
| `media_url` | `string` | Resolved URL after upload |
| `title` | `string?` | Optional display title |
| `description` | `string?` | Optional description |
| `display_order` | `number?` | Sort index |
| `_file` | `File?` | Temporary pre-upload file reference |
| `_uploading` | `boolean?` | Upload in-progress flag |

### `ReleaseMediaManager` (component)

**Props:**

| Prop | Type | Description |
|------|------|-------------|
| `media` | `ReleaseMediaItem[]` | Current media list (controlled) |
| `onChange` | `(media: ReleaseMediaItem[]) => void` | Fires on any mutation |
| `onUpload` | `(file, mediaType) => Promise<string>` | Handles actual file upload, returns URL |
| `className` | `string?` | Optional wrapper class |

**Internal behaviors:**
- Auto-detects `image/*` → `screenshot` and `video/*` → `demo` on file select
- Shows upload spinner while `onUpload` resolves; reverts on failure
- Drag-and-drop reordering updates `display_order` on all items

## Usage Example

```typescript
import { ReleaseMediaManager, ReleaseMediaItem } from './release-media-manager';

const [media, setMedia] = useState<ReleaseMediaItem[]>([]);

async function handleUpload(file: File, mediaType: string): Promise<string> {
  const form = new FormData();
  form.append('file', file);
  const res = await fetch('/api/releases/upload', { method: 'POST', body: form });
  const { url } = await res.json();
  return url;
}

<ReleaseMediaManager
  media={media}
  onChange={setMedia}
  onUpload={handleUpload}
/>
```

> **Note:** `onUpload` is the only side-effect hook — all state changes are surfaced through `onChange`, keeping the component fully controlled.
