A React component for managing media uploads and organization in release forms, providing drag-and-drop file upload, media type selection, and reorderable gallery display.

## Key Components

**Interfaces:**
- `ReleaseMediaItem` - Media item data structure with type, URL, metadata, and upload state
- `ReleaseMediaManagerProps` - Component props including media array, change handler, and upload callback

**Main Component:**
- `ReleaseMediaManager` - Complete media management interface with upload, preview, and organization features

**Key Features:**
- File upload with progress indication
- Drag-and-drop reordering
- Media type selection (image/video/screenshot/demo)
- Preview thumbnails for images and videos
- Editable titles and metadata

## Usage Example

```typescript
import { ReleaseMediaManager, ReleaseMediaItem } from './release-media-manager';

function ReleaseForm() {
  const [media, setMedia] = useState<ReleaseMediaItem[]>([]);

  const handleUpload = async (file: File, mediaType: string): Promise<string> => {
    // Upload file to storage service
    const formData = new FormData();
    formData.append('file', file);
    formData.append('type', mediaType);
    
    const response = await fetch('/api/upload', {
      method: 'POST',
      body: formData
    });
    
    const { url } = await response.json();
    return url;
  };

  return (
    <ReleaseMediaManager
      media={media}
      onChange={setMedia}
      onUpload={handleUpload}
      className="my-4"
    />
  );
}
```
