React component for managing media attachments on release entries, supporting file upload, drag-to-reorder, and per-item metadata editing for images and videos.

## Key Components

### `ReleaseMediaItem` (interface)
Represents a single media entry with the following fields:

| Field | Type | Description |
|-------|------|-------------|
| `media_type` | `'image' \| 'video' \| 'screenshot' \| 'demo'` | Category of the media |
| `media_url` | `string` | Resolved URL after upload |
| `title` | `string?` | Optional display title |
| `description` | `string?` | Optional description |
| `display_order` | `number?` | Sort index (auto-set on drag reorder) |
| `_file` | `File?` | Temporary local file pre-upload |
| `_uploading` | `boolean?` | Upload in-progress flag |

### `ReleaseMediaManager` (component)
The primary exported component. Accepts:

- **`media`** — current array of `ReleaseMediaItem`
- **`onChange`** — callback fired on any mutation (add, remove, reorder, field edit)
- **`onUpload`** — async handler that receives a `File` and `mediaType`, returns the uploaded URL
- **`className`** — optional CSS class override

Key internal behaviors:
- **File type detection** — images map to `screenshot`, videos map to `demo` on initial add
- **Optimistic upload** — item is appended immediately with `_uploading: true`; rolled back on error
- **Drag-and-drop reorder** — HTML5 drag events; `display_order` is normalized on drop
- **Media preview** — `<video>` element for `video`/`demo` types; Next.js `<Image>` for all others

## Usage Example

```typescript
const [mediaItems, setMediaItems] = useState<ReleaseMediaItem[]>([]);

async function handleUpload(
  file: File,
  mediaType: 'image' | 'video' | 'screenshot' | 'demo'
): Promise<string> {
  const formData = new FormData();
  formData.append('file', file);
  formData.append('type', mediaType);
  const res = await fetch('/api/releases/upload', { method: 'POST', body: formData });
  const { url } = await res.json();
  return url;
}

<ReleaseMediaManager
  media={mediaItems}
  onChange={setMediaItems}
  onUpload={handleUpload}
/>
```
