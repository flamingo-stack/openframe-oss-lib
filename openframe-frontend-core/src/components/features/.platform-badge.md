A React component that renders branded platform badges with customizable icons, colors, and sizes for different platforms like Flamingo, OpenMSP, and OpenFrame.

## Key Components

- **PlatformBadge**: Main component that renders a pill-shaped badge with platform icon and optional label
- **PlatformIcon**: Internal helper component that renders the appropriate logo based on platform type
- **sizeClasses**: Configuration object defining three size variants (xs, sm, md)
- **platformColors**: Platform-specific color schemes with background, border, and text colors
- **PlatformBadgeProps**: TypeScript interface defining component props including platform data, size, and display options

## Usage Example

```typescript
import { PlatformBadge } from './platform-badge';

// Basic usage with platform data
<PlatformBadge 
  platform={{
    id: 'flamingo-1',
    name: 'flamingo',
    display_name: 'Flamingo'
  }}
/>

// Customized badge without label
<PlatformBadge 
  platform={{
    id: 'openmsp-2',
    name: 'openmsp', 
    display_name: 'OpenMSP'
  }}
  size="md"
  showLabel={false}
  className="shadow-lg"
/>

// Badge for unknown platform (shows globe icon)
<PlatformBadge 
  platform={{
    id: 'custom-1',
    name: 'unknown-platform',
    display_name: 'Custom Platform'
  }}
  size="xs"
/>
```
