A React component that renders platform identification badges with branded colors, icons, and optional labels for different platforms in the ecosystem.

## Key Components

- **PlatformBadge** - Main component that renders a styled badge with platform icon and optional label
- **PlatformIcon** - Internal component that selects and renders the appropriate logo/icon based on platform
- **sizeClasses** - Configuration object defining three size variants (xs, sm, md) with corresponding spacing and icon dimensions
- **platformColors** - Color scheme definitions for each supported platform using brand-specific colors with transparency

## Usage Example

```typescript
import { PlatformBadge } from './platform-badge';

// Basic usage with platform object
<PlatformBadge 
  platform={{
    id: 'openframe-1',
    name: 'openframe',
    display_name: 'OpenFrame'
  }}
/>

// Different sizes and configurations
<PlatformBadge 
  platform={platformData}
  size="md"
  showLabel={false}
  className="shadow-lg"
/>

// Renders as null if no platform provided
<PlatformBadge platform={null} />
```

The component supports platforms like OpenMSP, Flamingo, OpenFrame, Admin Hub, TMCG, and includes a fallback Globe icon for unknown platforms. Each platform has its own branded color scheme and corresponding logo component.
