A reusable React component for creating interactive section navigation with customizable layouts, badges, and visual states for selection interfaces.

## Key Components

- **`SectionSelector`** - Main component that renders a collection of selectable sections
- **`SectionButton`** - Internal button component for individual section items
- **`SectionItem`** - Interface defining section properties including title, description, badges, and icons
- **`SectionSelectorProps`** - Configuration interface for layout options and event handlers

## Usage Example

```typescript
import { SectionSelector, SectionItem } from './section-selector'

const sections: SectionItem[] = [
  {
    id: 'setup',
    title: 'Initial Setup',
    description: 'Configure your environment',
    number: '1',
    badge: { text: 'Required', colorScheme: 'yellow' }
  },
  {
    id: 'deploy',
    title: 'Deployment',
    subtitle: 'Deploy to production',
    leftIcon: <DeployIcon />
  }
]

function MyComponent() {
  const [activeSection, setActiveSection] = useState('setup')

  return (
    <SectionSelector
      sections={sections}
      activeSection={activeSection}
      onSectionChange={setActiveSection}
      layout="vertical"
      buttonWidth="responsive"
      showDescription={true}
    />
  )
}
```

The component supports vertical and wrap layouts, optional numbering, status badges, icons, and responsive width configurations. Sections can be disabled and styled with custom CSS classes.
