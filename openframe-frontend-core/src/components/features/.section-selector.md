A React component for creating interactive section navigation with customizable layouts, badges, and responsive design capabilities.

## Key Components

- **SectionSelector** - Main component that renders a collection of navigational buttons
- **SectionButton** - Internal button component with consistent styling and layout logic
- **SectionItem** - TypeScript interface defining section data structure with optional badges and screenshots
- **SectionSelectorProps** - Configuration interface with layout, styling, and behavior options

## Usage Example

```typescript
import { SectionSelector, SectionItem } from './section-selector'

const sections: SectionItem[] = [
  {
    id: 'overview',
    title: 'Project Overview',
    subtitle: 'Introduction and goals',
    description: 'Learn about the project objectives and scope',
    number: '01',
    badge: {
      text: 'New',
      variant: 'button',
      colorScheme: 'green'
    }
  },
  {
    id: 'setup',
    title: 'Setup Guide',
    subtitle: 'Installation steps',
    disabled: false
  }
]

function Navigation() {
  const [activeSection, setActiveSection] = useState('overview')

  return (
    <SectionSelector
      sections={sections}
      activeSection={activeSection}
      onSectionChange={setActiveSection}
      layout="vertical"
      buttonWidth="responsive"
      showDescription={true}
    />
  )
}
```

The component supports vertical and wrap layouts, responsive width options, status badges, and comprehensive styling customization through className props.
