A React component for managing GitHub release links with add/remove functionality and automatic ordering.

## Key Components

- **`GitHubReleaseLink`** - Interface defining the structure for GitHub release entries with URL and optional display order
- **`GitHubReleasesManager`** - Main component that renders a list manager for GitHub release URLs
- **`GitHubReleasesManagerProps`** - Props interface accepting releases array, onChange callback, and optional className

## Usage Example

```typescript
import { GitHubReleasesManager, GitHubReleaseLink } from './github-releases-manager';

function ProjectForm() {
  const [releases, setReleases] = useState<GitHubReleaseLink[]>([
    { github_release_url: "https://github.com/org/repo/releases/tag/v1.0.0", display_order: 1 }
  ]);

  return (
    <GitHubReleasesManager
      releases={releases}
      onChange={setReleases}
      className="mt-4"
    />
  );
}
```

The component leverages `ArrayEntryManager` to provide a consistent interface for adding, removing, and reordering GitHub release links with URL validation and visual feedback.
