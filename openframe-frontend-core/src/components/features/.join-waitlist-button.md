A customizable "Join Waitlist" button component that provides a consistent interface for early access signup across the Flamingo platform. It combines the OpenFrame logo with styled button functionality and loading states.

## Key Components

- **JoinWaitlistButton** - Main component that renders a primary button with OpenFrame logo
- **JoinWaitlistButtonProps** - Interface extending ButtonProps with custom styling options
- **OpenFrameLogo** - Icon component displayed when not in loading state

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button';

// Basic usage
<JoinWaitlistButton onClick={handleJoinWaitlist} />

// With custom styling and loading state
<JoinWaitlistButton
  loading={isSubmitting}
  buttonBackgroundColor="#FF6B35"
  buttonTextColor="#FFFFFF"
  lowerPathColor="#FFFFFF"
  upperPathColor="#FF6B35"
  platform="homepage"
  onClick={handleSubmit}
>
  Join Beta Program
</JoinWaitlistButton>

// Custom text content
<JoinWaitlistButton>
  Get OpenFrame Access
</JoinWaitlistButton>
```

The component automatically handles the OpenFrame logo display (hidden during loading), uses the primary button variant with touch-friendly sizing, and inherits DM Sans bold typography. The default text is "Get Early Access" but can be overridden via children prop.
