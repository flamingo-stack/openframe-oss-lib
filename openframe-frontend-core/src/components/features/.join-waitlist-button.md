A specialized button component for joining waitlists across the application, featuring a consistent OpenFrame logo and customizable styling options.

## Key Components

- **JoinWaitlistButton**: Main component with forwardRef support
- **JoinWaitlistButtonProps**: Interface extending ButtonProps with custom styling and loading state options
- **OpenFrameLogo**: Integrated icon component with customizable colors

## Usage Example

```typescript
import { JoinWaitlistButton } from './join-waitlist-button';

// Basic usage
<JoinWaitlistButton onClick={handleJoinWaitlist} />

// With loading state
<JoinWaitlistButton 
  loading={isSubmitting} 
  onClick={handleJoinWaitlist} 
/>

// Custom styling
<JoinWaitlistButton
  buttonBackgroundColor="#ff6b35"
  buttonTextColor="#ffffff"
  lowerPathColor="#ffffff"
  upperPathColor="#000000"
  platform="mobile"
>
  Get Early Access
</JoinWaitlistButton>

// With custom class and disabled state
<JoinWaitlistButton
  className="my-custom-class"
  disabled={!isValid}
  onClick={handleSubmit}
/>
```

The component automatically handles the OpenFrame logo visibility (hidden during loading) and inherits the primary button styling with touch-friendly sizing.
