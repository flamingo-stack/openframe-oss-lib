A specialized button component for user waitlist registration that provides consistent styling and branding across the application with customizable visual properties.

## Key Components

- **JoinWaitlistButtonProps**: Interface extending base ButtonProps with loading state and color customization options
- **JoinWaitlistButton**: Forwardable React component that renders a primary-styled button with OpenFrame logo integration

## Key Features

- **Consistent Branding**: Uses primary variant with yellow background and DM Sans bold typography
- **Loading States**: Shows spinner when loading, hides logo icon during loading
- **Customizable Colors**: Supports custom background, text, and logo path colors
- **Responsive Icon**: OpenFrame logo scales appropriately on different screen sizes
- **Accessibility**: Proper forwardRef implementation for focus management

## Usage Example

```typescript
import { JoinWaitlistButton } from './components/join-waitlist-button';

// Basic usage
<JoinWaitlistButton onClick={handleJoinWaitlist} />

// With loading state
<JoinWaitlistButton 
  loading={isSubmitting} 
  disabled={isSubmitting}
/>

// Custom styling
<JoinWaitlistButton
  buttonBackgroundColor="#FF6B35"
  buttonTextColor="#FFFFFF"
  lowerPathColor="#FFFFFF"
  upperPathColor="#000000"
  platform="mobile"
>
  Join Beta
</JoinWaitlistButton>

// With custom className
<JoinWaitlistButton 
  className="w-full mt-4" 
  onClick={handleSignup}
/>
```
