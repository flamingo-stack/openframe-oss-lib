A React component that provides a specialized interface for managing ClickUp task links with display ordering capabilities, built on top of a generic array entry manager.

## Key Components

- **`ClickUpTaskLink`** - Interface defining the structure for ClickUp task entries with ID and optional display order
- **`ClickUpTasksManager`** - Main component that renders a task management interface
- **`ClickUpTasksManagerProps`** - Props interface supporting customizable titles, styling, and change handlers

## Usage Example

```typescript
import { ClickUpTasksManager, ClickUpTaskLink } from './clickup-tasks-manager';

function RoadmapEditor() {
  const [tasks, setTasks] = useState<ClickUpTaskLink[]>([
    { clickup_task_id: '86cq2uabk', display_order: 1 },
    { clickup_task_id: 'abc123def', display_order: 2 }
  ]);

  return (
    <ClickUpTasksManager
      tasks={tasks}
      onChange={setTasks}
      title="Sprint Tasks"
      className="mb-4"
    />
  );
}
```

The component leverages the `ArrayEntryManager` for consistent UI patterns while providing ClickUp-specific defaults like placeholder text, icons, and messaging. It accepts both string and ReactNode titles for flexible integration with badge systems or custom formatting.
