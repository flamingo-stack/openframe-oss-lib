Reusable configuration section component for highlight video generation settings, providing duration selection and subtitle burning controls in a unified horizontal layout.

## Key Components

**`HighlightConfigSection`** — Client-side React component used across `CustomerInterview` and `ProductRelease` entities to configure highlight video output.

**`HighlightConfigSectionProps`** — Interface defining:
- `targetDurationSeconds` — Current duration value (in seconds)
- `onTargetDurationChange` — Handler called with new duration in seconds
- `skipSubtitleBurning` — Boolean toggle for subtitle burn-in
- `onSkipSubtitleBurningChange` — Handler called with updated checkbox state
- `disabled?` — Disables all controls (default: `false`)
- `className?` — Optional Tailwind class override

## Usage Example

```typescript
import { HighlightConfigSection } from '@/components/highlight-config-section';

function MyForm() {
  const [duration, setDuration] = React.useState(180);
  const [skipSubs, setSkipSubs] = React.useState(false);

  return (
    <HighlightConfigSection
      targetDurationSeconds={duration}
      onTargetDurationChange={setDuration}
      skipSubtitleBurning={skipSubs}
      onSkipSubtitleBurningChange={setSkipSubs}
      disabled={false}
    />
  );
}
```

## Duration Options

| Value | Label |
|-------|-------|
| `60` | 1 minute |
| `120` | 2 minutes |
| `180` | 3 minutes *(Recommended)* |
