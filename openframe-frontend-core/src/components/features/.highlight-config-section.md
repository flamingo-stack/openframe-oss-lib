Reusable configuration section component for highlight video generation settings, providing duration selection and subtitle burning controls in a unified horizontal layout.

## Key Components

### `HighlightConfigSection`
A controlled form component rendering two configuration options side-by-side:
- **Target Duration** — A `Select` dropdown with preset values (1–5 minutes, defaulting to 3 minutes as recommended)
- **Skip Subtitle Burning** — A checkbox toggle to optionally bypass subtitle rendering

### `HighlightConfigSectionProps`
| Prop | Type | Description |
|------|------|-------------|
| `targetDurationSeconds` | `number` | Current duration value in seconds |
| `onTargetDurationChange` | `(seconds: number) => void` | Fires when duration selection changes |
| `skipSubtitleBurning` | `boolean` | Current subtitle skip state |
| `onSkipSubtitleBurningChange` | `(skip: boolean) => void` | Fires when checkbox toggled |
| `disabled` | `boolean?` | Disables all controls (default: `false`) |
| `className` | `string?` | Additional CSS classes for the wrapper |

## Usage Example

```typescript
import { HighlightConfigSection } from './highlight-config-section';

function HighlightEditor() {
  const [duration, setDuration] = useState(180); // 3 minutes default
  const [skipSubs, setSkipSubs] = useState(false);

  return (
    <HighlightConfigSection
      targetDurationSeconds={duration}
      onTargetDurationChange={setDuration}
      skipSubtitleBurning={skipSubs}
      onSkipSubtitleBurningChange={setSkipSubs}
      disabled={isProcessing}
    />
  );
}
```

> **Note:** This component is designed to be shared across `CustomerInterview` and `ProductRelease` entity workflows, ensuring consistent highlight configuration UX across both flows.
