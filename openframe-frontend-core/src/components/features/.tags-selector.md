A reusable React component for selecting tags with search functionality, inline chip display, and autocomplete dropdown. Designed for use across blog posts, case studies, and product releases with a unified interface.

## Key Components

- **TagsSelector**: Main component with search input, selected tag chips, and autocomplete dropdown
- **Tag**: Interface defining tag structure with id, name, and slug
- **TagsSelectorProps**: Configuration props including available tags, selection handlers, and customization options

## Usage Example

```typescript
import { TagsSelector, Tag } from './components/tags-selector';

const availableTags: Tag[] = [
  { id: 1, name: 'React', slug: 'react' },
  { id: 2, name: 'TypeScript', slug: 'typescript' },
  { id: 3, name: 'Next.js', slug: 'nextjs' }
];

function MyComponent() {
  const [selectedTagIds, setSelectedTagIds] = useState<number[]>([]);

  return (
    <TagsSelector
      availableTags={availableTags}
      selectedTagIds={selectedTagIds}
      onTagsChange={setSelectedTagIds}
      maxTags={5}
      placeholder="Search for tags..."
      className="w-full"
    />
  );
}
```

Features include search-as-you-type filtering, removable tag chips displayed inline, configurable tag limits (default 10), auto-opening dropdown on focus, and full keyboard accessibility.
