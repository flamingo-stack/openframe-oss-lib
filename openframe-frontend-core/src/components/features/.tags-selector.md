A unified tag selector component with search autocomplete and inline chip display, designed for content management across blog posts, case studies, and product releases.

## Key Components

- **TagsSelector**: Main component with search input containing selected tag chips
- **Tag Interface**: Defines tag structure with id, name, and slug
- **TagsSelectorProps**: Configuration props including available tags, selection state, and limits
- **Search & Filter Logic**: Filters available tags based on search query and current selection
- **Chip Management**: Inline tag chips with remove functionality inside the search input

## Usage Example

```typescript
import { TagsSelector } from './components/tags-selector';

const availableTags = [
  { id: 1, name: 'React', slug: 'react' },
  { id: 2, name: 'TypeScript', slug: 'typescript' },
  { id: 3, name: 'Next.js', slug: 'nextjs' }
];

function BlogEditor() {
  const [selectedTagIds, setSelectedTagIds] = useState<number[]>([]);

  return (
    <TagsSelector
      availableTags={availableTags}
      selectedTagIds={selectedTagIds}
      onTagsChange={setSelectedTagIds}
      maxTags={5}
      placeholder="Search and select tags..."
      className="w-full"
    />
  );
}
```

The component features a search input with selected tags displayed as removable chips inside the input field, an autocomplete dropdown showing filtered available tags, and configurable tag limits with visual feedback.
