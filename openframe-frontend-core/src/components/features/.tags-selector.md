A comprehensive React component for selecting multiple tags with search functionality and visual chip display inside the input field.

## Key Components

**TagsSelector** - Main component featuring:
- Search autocomplete with dropdown suggestions
- Selected tags displayed as removable chips inside the search input
- Configurable tag limits (default 10)
- Focus-triggered dropdown with all available tags
- Real-time filtering based on search query
- Disabled state support with visual feedback

**Interfaces**:
- `Tag` - Defines tag structure with id, name, and slug
- `TagsSelectorProps` - Component configuration including available tags, selection state, and UI options

## Usage Example

```typescript
import { TagsSelector } from './components/tags-selector';

const availableTags = [
  { id: 1, name: 'React', slug: 'react' },
  { id: 2, name: 'TypeScript', slug: 'typescript' },
  { id: 3, name: 'Next.js', slug: 'nextjs' }
];

function BlogEditor() {
  const [selectedTagIds, setSelectedTagIds] = useState<number[]>([]);

  return (
    <TagsSelector
      availableTags={availableTags}
      selectedTagIds={selectedTagIds}
      onTagsChange={setSelectedTagIds}
      maxTags={5}
      placeholder="Search and select tags..."
      className="w-full"
    />
  );
}
```

The component automatically handles tag addition/removal, search filtering, and displays a counter showing current selection against the maximum limit. Selected tags appear as yellow chips with remove buttons directly in the search input area.
