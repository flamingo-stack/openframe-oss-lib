A React component for selecting and managing tags with search autocomplete, inline chips, and configurable limits. Built for content management across blog posts, case studies, and product releases.

## Key Components

- **TagsSelector**: Main component with search input, selected tag chips, and dropdown autocomplete
- **Tag Interface**: Defines tag structure with id, name, and slug properties
- **TagsSelectorProps**: Configuration interface supporting tag limits, placeholders, and disabled states

## Usage Example

```typescript
import { TagsSelector, Tag } from './components/tags-selector';

const availableTags: Tag[] = [
  { id: 1, name: "React", slug: "react" },
  { id: 2, name: "TypeScript", slug: "typescript" },
  { id: 3, name: "Next.js", slug: "nextjs" }
];

function BlogPostEditor() {
  const [selectedTagIds, setSelectedTagIds] = useState<number[]>([]);

  return (
    <TagsSelector
      availableTags={availableTags}
      selectedTagIds={selectedTagIds}
      onTagsChange={setSelectedTagIds}
      maxTags={5}
      placeholder="Search blog tags..."
      className="mb-4"
    />
  );
}
```

The component features inline chip display within the search input, automatic dropdown opening on focus, removable tags with X buttons, and a tag counter with maximum limit warnings. Selected tags appear as yellow chips inside the search bar, with remaining space used for searching additional tags.
