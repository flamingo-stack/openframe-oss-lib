A React component that renders a list of authentication provider buttons with configurable layout and divider options.

## Key Components

- **`SSOConfigStatus`** - Interface defining provider configuration with enabled status and optional client ID
- **`AuthProvidersList`** - Main component that renders filtered provider buttons with error handling
- **`ProviderButton`** - Imported component for individual provider buttons

## Properties

- `enabledProviders` - Array of SSO provider configurations
- `onProviderClick` - Async callback for handling provider authentication
- `loading` - Optional loading state that disables all buttons
- `showDivider` - Toggle for divider display (default: true)
- `dividerText` - Custom divider text (default: "or")
- `orientation` - Layout direction: vertical or horizontal (default: vertical)

## Usage Example

```typescript
import { AuthProvidersList, SSOConfigStatus } from './auth-providers-list';

const providers: SSOConfigStatus[] = [
  { provider: 'google', enabled: true, clientId: 'google-client-id' },
  { provider: 'github', enabled: true },
  { provider: 'microsoft', enabled: false }
];

export function LoginForm() {
  const handleAuth = async (provider: string) => {
    // Implement your auth logic (OAuth redirect, etc.)
    window.location.href = `/auth/${provider}`;
  };

  return (
    <AuthProvidersList
      enabledProviders={providers}
      onProviderClick={handleAuth}
      orientation="horizontal"
      dividerText="or continue with"
    />
  );
}
```

The component automatically filters out disabled providers and includes error handling for failed authentication attempts.
