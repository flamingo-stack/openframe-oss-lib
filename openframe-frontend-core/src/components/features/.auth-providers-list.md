A React component that renders a list of authentication provider buttons for SSO (Single Sign-On) integration. It supports multiple providers with configurable layout and styling options.

## Key Components

- **`SSOConfigStatus`** - Interface defining provider configuration with enabled state and optional client ID
- **`AuthProvidersList`** - Main component that renders filtered, enabled authentication providers
- **`ProviderButton`** - Imported child component for individual provider buttons

## Usage Example

```typescript
import { AuthProvidersList, SSOConfigStatus } from './auth-providers-list';

const providers: SSOConfigStatus[] = [
  { provider: 'google', enabled: true, clientId: 'google-client-id' },
  { provider: 'microsoft', enabled: true },
  { provider: 'github', enabled: false }
];

function LoginForm() {
  const handleAuth = async (provider: string) => {
    // Implement your authentication logic
    await signInWithProvider(provider);
  };

  return (
    <AuthProvidersList 
      enabledProviders={providers}
      onProviderClick={handleAuth}
      loading={false}
      orientation="horizontal"
      dividerText="Sign in with"
    />
  );
}
```

The component automatically filters out disabled providers, handles async authentication callbacks with error logging, and supports both vertical and horizontal layouts with optional dividers.
