A React component for embedding YouTube videos with fallback handling, loading states, and error recovery. It uses ReactPlayer as the primary method but falls back to iframe embedding for better compatibility.

## Key Components

- **`YouTubeEmbed`** - Main component for embedding YouTube videos with configurable controls and styling
- **`extractYouTubeId`** - Utility function to extract video IDs from various YouTube URL formats
- **`YouTubeLinkParser`** - Component that automatically converts YouTube URLs to embeds in markdown contexts
- **`Play` & `Loader`** - SVG icon components for UI elements

## Usage Example

```typescript
import { YouTubeEmbed, extractYouTubeId, YouTubeLinkParser } from './youtube-embed';

// Basic embed
<YouTubeEmbed 
  videoId="dQw4w9WgXcQ" 
  title="Rick Astley - Never Gonna Give You Up"
/>

// With minimal controls
<YouTubeEmbed 
  videoId="dQw4w9WgXcQ"
  minimalControls={true}
  showMeta={false}
/>

// Extract video ID from URL
const videoId = extractYouTubeId("https://www.youtube.com/watch?v=dQw4w9WgXcQ");

// Auto-convert YouTube links in markdown
<YouTubeLinkParser href="https://youtu.be/dQw4w9WgXcQ">
  Watch this video
</YouTubeLinkParser>
```

The component handles loading states, error recovery, and provides a responsive 16:9 aspect ratio container with proper accessibility features.
