A unified React component for managing media galleries with upload, reordering, and deletion capabilities across different content types like events and product releases.

## Key Components

**MediaGalleryManager**: Main component providing complete media management functionality
- File upload with drag-and-drop support
- Visual media grid with thumbnails
- Drag-and-drop reordering with visual feedback
- Individual item deletion with confirmation states
- Responsive design for mobile and desktop

**MediaItem Interface**: TypeScript definition for media objects
- Support for images, videos, screenshots, and demos
- Optional metadata like titles, descriptions, and display order
- Upload state tracking for UI feedback

## Usage Example

```typescript
import { MediaGalleryManager, MediaItem } from './media-gallery-manager';

function MyComponent() {
  const [media, setMedia] = useState<MediaItem[]>([]);
  const [isUploading, setIsUploading] = useState(false);

  const handleUpload = async (file: File, mediaType: 'image' | 'video') => {
    setIsUploading(true);
    try {
      // Upload to your storage service
      const url = await uploadFile(file);
      return url;
    } finally {
      setIsUploading(false);
    }
  };

  return (
    <MediaGalleryManager
      media={media}
      onChange={setMedia}
      onUpload={handleUpload}
      isUploading={isUploading}
      modalTitle="Product Media"
    />
  );
}
```

The component handles automatic file type detection, maintains display order through drag-and-drop, and provides visual feedback for all user interactions including upload progress and deletion states.
