A React component that provides a complete media gallery management solution with drag-and-drop upload, reordering, and deletion capabilities for images and videos. Used across events, product releases, and any content requiring media galleries.

## Key Components

**Interfaces:**
- `MediaItem` - Defines media object structure with type, URL, metadata
- `MediaGalleryManagerProps` - Component configuration options

**Main Component:**
- `MediaGalleryManager` - Complete gallery manager with upload, display, reorder, and delete functionality

**Key Features:**
- File upload with automatic type detection (image/video)
- Drag-and-drop reordering with visual feedback
- Media deletion with loading states
- Responsive grid layout
- Upload progress indication

## Usage Example

```typescript
import { MediaGalleryManager, MediaItem } from './media-gallery-manager';

function EventForm() {
  const [media, setMedia] = useState<MediaItem[]>([]);

  const handleUpload = async (file: File, mediaType: 'image' | 'video'): Promise<string> => {
    const formData = new FormData();
    formData.append('file', file);
    
    const response = await fetch('/api/upload', {
      method: 'POST',
      body: formData
    });
    
    const { url } = await response.json();
    return url;
  };

  return (
    <MediaGalleryManager
      media={media}
      onChange={setMedia}
      onUpload={handleUpload}
      modalTitle="Event Gallery"
      className="my-6"
    />
  );
}
```
