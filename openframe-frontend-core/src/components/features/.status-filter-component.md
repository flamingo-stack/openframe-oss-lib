A reusable status filter component that renders a horizontal row of filter buttons for admin dashboards, automatically handling "All" selection and displaying optional item counts.

## Key Components

- **StatusOption** - Interface defining filter options with `value` and `label` properties
- **StatusFilterComponentProps** - Props interface including status selection handlers, options array, and optional count display
- **StatusFilterComponent** - Main component that renders filter buttons with active state styling and optional item count

## Usage Example

```typescript
import { StatusFilterComponent, StatusOption } from './status-filter-component';

const statusOptions: StatusOption[] = [
  { value: 'all', label: 'All' },
  { value: 'published', label: 'Published' },
  { value: 'draft', label: 'Draft' },
  { value: 'archived', label: 'Archived' }
];

function Dashboard() {
  const [selectedStatus, setSelectedStatus] = useState('all');

  return (
    <StatusFilterComponent
      selectedStatus={selectedStatus}
      onStatusChange={setSelectedStatus}
      statusOptions={statusOptions}
      showCount={true}
      count={42}
      className="mb-6"
    />
  );
}
```

The component automatically filters out duplicate "All" options and applies consistent styling with active/inactive states for each filter button.
