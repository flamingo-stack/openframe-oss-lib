A reusable status filter component for admin dashboards that displays filter buttons with an optional item count.

## Key Components

- **StatusFilterComponent**: Main React component that renders a horizontal row of status filter buttons
- **StatusOption**: Interface defining the structure for filter options with value and label
- **StatusFilterComponentProps**: Props interface supporting status selection, change handlers, options, and optional count display

The component automatically handles the "All" filter option separately and applies the ODS design system styling with responsive typography.

## Usage Example

```typescript
import { StatusFilterComponent, StatusOption } from './status-filter-component';

const statusOptions: StatusOption[] = [
  { value: 'published', label: 'Published' },
  { value: 'draft', label: 'Draft' },
  { value: 'archived', label: 'Archived' }
];

function BlogDashboard() {
  const [selectedStatus, setSelectedStatus] = useState('all');
  const [itemCount, setItemCount] = useState(42);

  return (
    <StatusFilterComponent
      selectedStatus={selectedStatus}
      onStatusChange={setSelectedStatus}
      statusOptions={statusOptions}
      showCount={true}
      count={itemCount}
      className="mb-6"
    />
  );
}
```

The component uses the Filter icon from Lucide React and integrates with the ODS design system for consistent styling across admin interfaces.
