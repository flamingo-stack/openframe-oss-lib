A React component that provides a user interface for managing a collection of knowledge base article links with ordering capabilities.

## Key Components

- **`KBLink` interface**: Defines the structure for knowledge base links with path and optional display order
- **`KnowledgeBaseLinksManager` component**: Main component that renders a list manager for KB articles
- **`KnowledgeBaseLinksManagerProps` interface**: Props interface for the main component

## Usage Example

```typescript
import { KnowledgeBaseLinksManager, KBLink } from './knowledge-base-links-manager';

function DocumentationSettings() {
  const [kbLinks, setKbLinks] = useState<KBLink[]>([
    { kb_article_path: '/api/authentication/api-keys.md', display_order: 1 },
    { kb_article_path: '/guides/getting-started.md', display_order: 2 }
  ]);

  return (
    <KnowledgeBaseLinksManager
      links={kbLinks}
      onChange={setKbLinks}
      className="mt-4"
    />
  );
}
```

The component wraps the `ArrayEntryManager` with KB-specific configuration, including a FileText icon, appropriate placeholder text, and messaging for empty states. Users can add, remove, and reorder knowledge base article references through the interface.
