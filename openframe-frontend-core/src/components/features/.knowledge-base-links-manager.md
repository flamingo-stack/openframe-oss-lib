A React component that provides a specialized interface for managing knowledge base article links with display order support.

## Key Components

- **`KBLink` interface**: Defines the structure for knowledge base links with relative paths and optional display order
- **`KnowledgeBaseLinksManager` component**: Main component that renders an array manager specifically for KB articles
- **Props interface**: Configures the component with links array, change handler, and optional styling

## Usage Example

```typescript
import { KnowledgeBaseLinksManager, KBLink } from './knowledge-base-links-manager';

function DocumentationEditor() {
  const [kbLinks, setKbLinks] = useState<KBLink[]>([
    { kb_article_path: '/api/authentication/api-keys.md', display_order: 1 },
    { kb_article_path: '/guides/getting-started.md', display_order: 2 }
  ]);

  return (
    <KnowledgeBaseLinksManager
      links={kbLinks}
      onChange={setKbLinks}
      className="mt-4"
    />
  );
}
```

The component leverages the `ArrayEntryManager` to provide a consistent UI for adding, editing, and reordering knowledge base article references. It includes a FileText icon and customized messaging specific to documentation management workflows.
