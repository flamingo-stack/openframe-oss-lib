Reusable React component for managing a dynamic list of changelog entries, supporting add, remove, expand/collapse, and optional per-entry public/internal visibility toggling.

## Key Components

**`ChangelogManager`** — Default export component accepting:

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Section label rendered above the entry list |
| `entries` | `ChangelogEntry[]` | Controlled list of changelog entries |
| `onChange` | `(entries: ChangelogEntry[]) => void` | Callback fired on any mutation |
| `expandAll` | `boolean` | When `true`, expands all entries (triggered after AI enrichment) |
| `showVisibilityToggle` | `boolean` | Renders an Eye/EyeOff toggle per entry (used for investor updates) |

**Internal state:**
- `expandedIndices` — `Set<number>` tracking which entries are expanded

**Key behaviors:**
- `addEntry` — Appends a blank entry and auto-expands it
- `removeEntry` — Removes an entry and re-indexes the expanded set
- `toggleVisibility` — Flips entry visibility between `'public'` and `'internal'`
- `updateEntry` — Patches a single field on a specific entry

## Usage Example

```typescript
import { ChangelogManager } from './changelog-manager';
import type { ChangelogEntry } from '../../types/product-release';

const [entries, setEntries] = useState<ChangelogEntry[]>([]);

// Product release usage (no visibility toggle)
<ChangelogManager
  title="What's New"
  entries={entries}
  onChange={setEntries}
/>

// Investor update usage (with visibility toggle + auto-expand after AI fill)
<ChangelogManager
  title="Highlights"
  entries={entries}
  onChange={setEntries}
  showVisibilityToggle
  expandAll={aiEnrichmentComplete}
/>
```
