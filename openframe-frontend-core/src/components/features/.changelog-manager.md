A reusable React component for managing a dynamic list of changelog entries with collapsible cards, optional per-entry visibility toggling, and support for bulk expansion (e.g., after AI enrichment).

## Key Components

**`ChangelogManager`** — Primary export. Renders a controlled list of `ChangelogEntry` items with add/remove/edit capabilities.

**Props (`ChangelogManagerProps`):**

| Prop | Type | Description |
|------|------|-------------|
| `title` | `string` | Section label displayed above the list |
| `entries` | `ChangelogEntry[]` | Controlled array of changelog entries |
| `onChange` | `(entries: ChangelogEntry[]) => void` | Called on any mutation |
| `expandAll` | `boolean` | Expands all cards (e.g., post-AI enrichment) |
| `showVisibilityToggle` | `boolean` | Enables public/internal toggle per entry (investor updates use case) |

**Internal state:**
- `expandedIndices` — `Set<number>` tracking which cards are open; automatically adjusted on add/remove to keep indices consistent.

## Usage Example

```typescript
import { ChangelogManager } from './changelog-manager';
import { useState } from 'react';
import type { ChangelogEntry } from '../../types/product-release';

export function ReleaseForm() {
  const [features, setFeatures] = useState<ChangelogEntry[]>([]);

  return (
    <ChangelogManager
      title="New Features"
      entries={features}
      onChange={setFeatures}
      expandAll={false}
      showVisibilityToggle={false}
    />
  );
}
```

For investor update forms, enable the visibility toggle:

```typescript
<ChangelogManager
  title="Investor Highlights"
  entries={highlights}
  onChange={setHighlights}
  showVisibilityToggle={true}
  expandAll={aiEnrichmentComplete}
/>
```

> **Note:** `showVisibilityToggle` defaults new entries to `'public'`. Omit it for standard product release forms to preserve their existing UX.
