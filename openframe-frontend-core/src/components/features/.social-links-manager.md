A React component for managing user social media links with platform selection, URL input, and dynamic icon display functionality.

## Key Components

- **SocialLinksManager**: Main component for adding, editing, and removing social links
- **SocialLink**: Interface defining platform, URL, and optional username
- **SocialPlatform**: Interface for platform configuration with icon mapping and display properties
- **iconMap**: Mapping object connecting platform names to their corresponding icons

## Usage Example

```typescript
import { SocialLinksManager } from './social-links-manager';

const platforms = [
  { id: '1', name: 'linkedin', display_name: 'LinkedIn', icon_name: 'linkedin', enabled: true },
  { id: '2', name: 'github', display_name: 'GitHub', icon_name: 'github', enabled: true }
];

function ProfileForm() {
  const [socialLinks, setSocialLinks] = useState([
    { platform: 'linkedin', url: 'https://linkedin.com/in/username' }
  ]);

  return (
    <SocialLinksManager
      links={socialLinks}
      onChange={setSocialLinks}
      platforms={platforms}
      className="my-4"
    />
  );
}
```

The component renders a dynamic form where users can select social platforms from a dropdown, enter URLs, and manage their social media presence. Each link displays the platform's icon and provides deletion functionality.
