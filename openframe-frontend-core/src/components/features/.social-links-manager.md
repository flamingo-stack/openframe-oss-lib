A React component for managing social media links with platform selection, URL input, and dynamic icon rendering.

## Key Components

- **SocialLinksManager**: Main component for adding, editing, and removing social media links
- **SocialLink**: Interface defining platform, URL, and optional username fields
- **SocialPlatform**: Interface for platform configuration with display names, icons, and URL patterns
- **iconMap**: Static mapping of platform names to their corresponding icon components

## Usage Example

```typescript
import { SocialLinksManager, SocialLink, SocialPlatform } from './social-links-manager';

const platforms: SocialPlatform[] = [
  {
    id: '1',
    name: 'linkedin',
    display_name: 'LinkedIn',
    icon_name: 'linkedin',
    placeholder: 'LinkedIn profile URL',
    enabled: true
  }
];

function ProfileEditor() {
  const [socialLinks, setSocialLinks] = useState<SocialLink[]>([]);

  return (
    <SocialLinksManager
      links={socialLinks}
      onChange={setSocialLinks}
      platforms={platforms}
      className="max-w-2xl"
    />
  );
}
```

The component provides a user-friendly interface with platform dropdowns, URL inputs, icon previews, and remove buttons. It supports dynamic platform configuration and includes built-in icons for popular social media platforms.
