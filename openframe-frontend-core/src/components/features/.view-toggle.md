A React component that provides a toggle switch for switching between grid and table view modes, built on Radix UI with ODS design tokens for accessibility and consistent theming.

## Key Components

- **ViewToggle** - Main component that renders a two-state toggle group
- **ViewMode** - Type union for 'grid' | 'table' view states
- **ViewToggleProps** - Interface defining component props including value, onValueChange, disabled, className, size, and aria-label

## Usage Example

```typescript
import { ViewToggle, ViewMode } from './components/view-toggle'
import { useState } from 'react'

function DataDisplay() {
  const [viewMode, setViewMode] = useState<ViewMode>('table')

  return (
    <div>
      <ViewToggle
        value={viewMode}
        onValueChange={setViewMode}
        size="default"
        className="mb-4"
        aria-label="Switch between grid and table view"
      />
      
      {viewMode === 'grid' ? (
        <GridComponent />
      ) : (
        <TableComponent />
      )}
    </div>
  )
}
```

The component features smooth transitions, proper keyboard navigation, and visual feedback with ODS color tokens. It only triggers `onValueChange` when a different option is selected, preventing unnecessary re-renders.
