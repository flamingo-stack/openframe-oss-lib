A React context provider that manages global loading states with an animated progress bar overlay.

## Key Components

- **LoadingProvider**: Context provider component that wraps the application and displays a fixed progress bar
- **useLoading**: Custom hook to access loading state and control functions
- **LoadingContextType**: Type definition for the context value containing `isLoading` boolean and `setIsLoading` function

## Features

- Animated progress bar with smooth transitions (10% â†’ 90% during loading, 100% on completion)
- Fixed positioning at the top of the viewport with high z-index
- Auto-cleanup with 500ms delay after loading completes
- Progress simulation using exponential easing for realistic feel

## Usage Example

```typescript
// App setup
function App() {
  return (
    <LoadingProvider>
      <YourAppContent />
    </LoadingProvider>
  )
}

// In any component
function DataFetcher() {
  const { isLoading, setIsLoading } = useLoading()
  
  const fetchData = async () => {
    setIsLoading(true)
    try {
      await fetch('/api/data')
    } finally {
      setIsLoading(false)
    }
  }
  
  return (
    <button onClick={fetchData} disabled={isLoading}>
      {isLoading ? 'Loading...' : 'Fetch Data'}
    </button>
  )
}
```
