A generic React component for managing arrays of objects with inline editing, dynamic addition/removal, and multi-line paste support for efficient bulk entry.

## Key Components

- **ArrayEntryManager**: Generic component that manages an array of objects with a specific field for editing
- **addItem()**: Adds new empty items to the array
- **removeItem()**: Removes items by index
- **updateItem()**: Updates the specified field value for an item
- **handlePaste()**: Intelligently handles multi-line clipboard data to create multiple entries

## Usage Example

```typescript
import { ArrayEntryManager } from './array-entry-manager';
import { Github } from 'lucide-react';

interface Release {
  github_release_url: string;
  version?: string;
}

function ReleaseManager() {
  const [releases, setReleases] = useState<Release[]>([]);

  return (
    <ArrayEntryManager
      title="GitHub Releases"
      items={releases}
      onChange={setReleases}
      fieldKey="github_release_url"
      placeholder="https://github.com/user/repo/releases/tag/v1.0.0"
      emptyMessage="No releases added yet"
      addButtonText="Add Release"
      icon={<Github className="h-4 w-4" />}
    />
  );
}
```

The component features smart paste handling that automatically splits newline-separated values into multiple entries, making bulk data entry efficient. It maintains consistent styling with the design system and provides accessible controls for all array operations.
