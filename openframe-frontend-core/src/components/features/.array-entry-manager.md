A React component for managing arrays of entries with add/edit/remove functionality, supporting both immediate and deferred save modes.

## Key Components

- **ArrayEntryManager** - Generic TypeScript component for managing array entries with configurable fields
- **Props Interface** - Comprehensive configuration options including custom renderers, save modes, and styling
- **State Management** - Handles both immediate updates and draft mode with dirty state tracking
- **Paste Handling** - Smart multi-line paste support for bulk entry addition

## Usage Example

```typescript
import { ArrayEntryManager } from './array-entry-manager';
import { Github } from 'lucide-react';

interface Release {
  github_release_url: string;
  version?: string;
}

function ReleaseManager() {
  const [releases, setReleases] = useState<Release[]>([]);
  const [isSaving, setIsSaving] = useState(false);

  const handleSave = async (items: Release[]) => {
    setIsSaving(true);
    // Save to API
    await saveReleases(items);
    setIsSaving(false);
  };

  return (
    <ArrayEntryManager
      title="GitHub Releases"
      items={releases}
      onChange={handleSave}
      fieldKey="github_release_url"
      placeholder="https://github.com/owner/repo/releases/tag/v1.0.0"
      emptyMessage="No releases configured"
      addButtonText="Add Release"
      icon={<Github className="h-5 w-5" />}
      requireSave={true}
      isSaving={isSaving}
      renderLabel={(item, index) => 
        item.version && <span className="badge">{item.version}</span>
      }
    />
  );
}
```

The component supports immediate onChange updates or deferred saves with dirty state tracking, multi-line paste for bulk additions, and custom label rendering for enhanced UX.
