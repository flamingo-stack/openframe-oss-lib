A reusable React component for managing dynamic arrays of objects with editable string fields, featuring add/remove functionality and multi-line paste support.

## Key Components

- **ArrayEntryManager**: Generic component that manages arrays of objects with a specified editable field
- **ArrayEntryManagerProps**: TypeScript interface defining component props including items array, change handler, and UI customization options
- **addItem()**: Adds new empty item to the array
- **removeItem()**: Removes item at specified index
- **updateItem()**: Updates field value for item at specified index
- **handlePaste()**: Handles multi-line clipboard paste to create multiple entries

## Usage Example

```typescript
interface ReleaseItem {
  github_release_url: string;
}

const [releases, setReleases] = useState<ReleaseItem[]>([]);

<ArrayEntryManager
  title="GitHub Releases"
  items={releases}
  onChange={setReleases}
  fieldKey="github_release_url"
  placeholder="Enter GitHub release URL"
  emptyMessage="No releases added yet"
  addButtonText="Add Release"
  icon={<GitBranch className="h-4 w-4" />}
/>
```

The component supports pasting multiple URLs separated by newlines, automatically creating separate entries for each line.
