A customizable button component for promoting OpenFrame with multiple visual themes and an integrated beta status badge.

## Key Components

**StartWithOpenFrameButton** - Main React component with forwardRef support
- **Props Interface**: `StartWithOpenFrameButtonProps` extending base ButtonProps
- **Visual Modes**: 5 color schemes (outline, yellow, pink, purple, cyan)
- **Size Options**: Small, medium, and large variants
- **Icon Integration**: OpenFrameLogo with dynamic colors
- **Status Badge**: Built-in "Beta" badge indicator

## Usage Example

```typescript
import { StartWithOpenFrameButton } from './start-with-openframe-button';

// Default outline mode
<StartWithOpenFrameButton 
  onClick={() => window.open('/openframe')}
>
  Start Free Trial
</StartWithOpenFrameButton>

// Yellow primary mode with large size
<StartWithOpenFrameButton 
  mode="yellow"
  buttonSize="lg"
  onClick={handleStartTrial}
>
  Get Started Now
</StartWithOpenFrameButton>

// Cyan mode with custom colors
<StartWithOpenFrameButton 
  mode="cyan"
  buttonBackgroundColor="#00D4FF"
  buttonTextColor="#000000"
  loading={isLoading}
>
  Launch OpenFrame
</StartWithOpenFrameButton>
```

The component automatically adjusts icon colors based on the selected mode to maintain proper contrast and visual hierarchy. Each mode follows Flamingo's design system color tokens for consistent branding.
