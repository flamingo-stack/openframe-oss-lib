A customizable "Start with OpenFrame" button component that displays the OpenFrame logo and a beta status badge, designed for use in headers and call-to-action sections.

## Key Components

- **StartWithOpenFrameButton** - Main button component with five visual modes (outline, yellow, pink, purple, cyan)
- **StartWithOpenFrameButtonProps** - TypeScript interface extending ButtonProps with mode, size, and color customization options
- **Mode variants** - Different color schemes for various UI contexts
- **OpenFrame logo integration** - Automatically includes branded logo with dynamic colors
- **Beta badge** - Built-in status indicator for beta features

## Usage Example

```typescript
import { StartWithOpenFrameButton } from './components/start-with-openframe-button';

// Default outline mode
<StartWithOpenFrameButton>
  Start Free Trial
</StartWithOpenFrameButton>

// Colored variants
<StartWithOpenFrameButton mode="yellow" buttonSize="lg">
  Get Started
</StartWithOpenFrameButton>

<StartWithOpenFrameButton 
  mode="cyan" 
  buttonBackgroundColor="#00D4AA"
  buttonTextColor="#000000"
  loading={isSubmitting}
>
  Start Trial
</StartWithOpenFrameButton>

// Custom content
<StartWithOpenFrameButton mode="pink">
  Begin Your Journey
</StartWithOpenFrameButton>
```

The component automatically handles icon colors, hover states, and loading states while maintaining consistent branding across different visual themes.
