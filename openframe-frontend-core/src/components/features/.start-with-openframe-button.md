A React component that renders a styled "Start with OpenFrame" button for the Flamingo header, featuring multiple visual modes and OpenFrame branding with a beta status badge.

## Key Components

- **StartWithOpenFrameButtonProps** - Interface extending ButtonProps with custom mode options and styling properties
- **StartWithOpenFrameButton** - Main forwardRef component with five visual modes: outline (default), yellow, pink, purple, and cyan
- **Mode-based styling** - Dynamic variant selection and CSS class application based on the selected mode
- **Icon integration** - OpenFrame logo on the left with dynamic color theming based on mode
- **Beta badge** - StatusBadge component displayed on the right side

## Usage Example

```typescript
import { StartWithOpenFrameButton } from './components/start-with-openframe-button';

// Basic usage with default outline mode
<StartWithOpenFrameButton 
  onClick={() => console.log('Starting trial')}
>
  Start Free Trial
</StartWithOpenFrameButton>

// Yellow mode with custom size
<StartWithOpenFrameButton 
  mode="yellow"
  buttonSize="lg"
  onClick={handleTrialStart}
>
  Get Started
</StartWithOpenFrameButton>

// Cyan mode with custom colors
<StartWithOpenFrameButton 
  mode="cyan"
  buttonBackgroundColor="#00BCD4"
  buttonTextColor="#FFFFFF"
  loading={isLoading}
/>

// Pink/Purple mode
<StartWithOpenFrameButton 
  mode="pink"
  buttonSize="sm"
  disabled={!canStart}
>
  Start Now
</StartWithOpenFrameButton>
```

The component automatically handles icon color theming, loading states, and maintains consistent styling patterns across all modes while supporting custom color overrides for the cyan variant.
