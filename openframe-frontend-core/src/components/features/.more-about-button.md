A React client component that renders a styled "Learn more" button with an arrow icon, designed to navigate users to additional information pages.

## Key Components

- **MoreAboutButtonProps** - Interface defining optional onClick handler, href URL, and CSS className
- **MoreAboutButton** - Main component that renders a large ghost-style button with OpenFrame branding colors
- **Default exports** - Both named and default exports for flexible importing

## Usage Example

```typescript
import { MoreAboutButton } from './more-about-button'

// Basic usage with default "/openframe" link
<MoreAboutButton />

// Custom destination and styling
<MoreAboutButton 
  href="/custom-page"
  className="mt-4"
/>

// With click handler (overrides href behavior)
<MoreAboutButton 
  onClick={() => console.log('Button clicked')}
  className="custom-styles"
/>
```

The component uses the OpenFrame design system's yellow color (`--ods-open-yellow-base`) and automatically opens external links in new tabs. It leverages a custom Button component with built-in support for right icons and various styling variants.
