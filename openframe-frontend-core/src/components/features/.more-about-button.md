A reusable "Learn more" button component with a right arrow icon, designed to navigate to informational pages or trigger custom actions.

## Key Components

- **`MoreAboutButton`** - Main React functional component that renders a styled button
- **`MoreAboutButtonProps`** - Interface defining optional props for customization
  - `onClick?: () => void` - Optional click handler function
  - `href?: string` - Navigation URL (defaults to "/openframe")
  - `className?: string` - Additional CSS classes for styling

## Usage Example

```typescript
import { MoreAboutButton } from './more-about-button'

// Basic usage with default href
<MoreAboutButton />

// Custom navigation
<MoreAboutButton href="/custom-page" />

// With custom styling
<MoreAboutButton 
  href="/features" 
  className="mt-4 mb-2" 
/>

// With click handler (note: onClick prop exists in interface but not implemented)
<MoreAboutButton onClick={() => console.log('clicked')} />
```

The component uses a ghost variant button with yellow text styling and opens links in new tabs when an href is provided. It leverages the Lucide React ArrowRight icon for visual enhancement.
