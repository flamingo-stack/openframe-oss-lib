React component that renders operating system type indicators with icons and labels using a Button wrapper. Automatically handles OS type normalization, unknown OS cases, and provides flexible display modes.

## Key Components

- **`OSTypeBadge`** - Main component displaying OS icon and label with customizable variants
- **`OSTypeIcon`** - Icon-only variant for compact displays
- **`OSTypeLabel`** - Label-only variant without icons
- **`OSTypeBadgeProps`** - Configuration interface with display options

## Usage Example

```typescript
import { OSTypeBadge, OSTypeIcon, OSTypeLabel } from './os-type-badge'

// Full badge with icon and label
<OSTypeBadge osType="windows" variant="primary" />

// Various OS types (automatically normalized)
<OSTypeBadge osType="Darwin" />      // Shows as "macOS"
<OSTypeBadge osType="Ubuntu" />      // Shows Linux icon

// Icon-only mode for compact layouts
<OSTypeIcon osType="windows" size="w-5 h-5" />

// Label-only for text-heavy interfaces
<OSTypeLabel osType="linux" className="font-medium" />

// Custom styling and alignment
<OSTypeBadge 
  osType="macos" 
  variant="outline"
  alignment="center"
  iconColor="#007AFF"
/>

// Handles unknown/missing OS gracefully
<OSTypeBadge osType="unknown" />     // Returns null
<OSTypeLabel osType="" />            // Shows "Unknown"
```

The component integrates with external `os.types` utilities for normalization and supports all Button component variants for consistent styling across the application.
