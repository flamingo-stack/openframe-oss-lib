A React component for displaying operating system information with icons and labels, featuring flexible display options and automatic OS type normalization.

## Key Components

- **OSTypeBadge** - Main component that renders OS type with icon and/or label using a Button wrapper
- **OSTypeIcon** - Convenience component for displaying only the OS icon
- **OSTypeLabel** - Convenience component for displaying only the OS label
- **OSTypeBadgeProps** - Interface defining component props with extensive customization options

## Usage Example

```typescript
import { OSTypeBadge, OSTypeIcon, OSTypeLabel } from './os-type-badge'

// Full badge with icon and label
<OSTypeBadge osType="windows" />
<OSTypeBadge osType="Darwin" variant="primary" />

// Icon only display
<OSTypeBadge osType="ubuntu" iconOnly />
<OSTypeIcon osType="linux" size="w-6 h-6" />

// Label only display
<OSTypeBadge osType="macos" labelOnly />
<OSTypeLabel osType="windows" className="font-bold" />

// Custom styling and alignment
<OSTypeBadge 
  osType="linux" 
  variant="outline" 
  alignment="center"
  iconColor="blue" 
/>
```

The component automatically handles OS type normalization, supports various display modes (icon-only, label-only, or both), and provides extensive styling customization through Button variants and CSS classes.
