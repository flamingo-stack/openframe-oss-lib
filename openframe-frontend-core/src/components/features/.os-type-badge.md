A React component for displaying operating system types with icons and labels, supporting flexible display modes and automatic OS type normalization.

## Key Components

- **OSTypeBadge** - Main component that displays OS type with icon and label
- **OSTypeIcon** - Convenience component for icon-only display
- **OSTypeLabel** - Convenience component for label-only display
- **OSTypeBadgeProps** - Interface defining component props including display modes and styling options

## Usage Example

```typescript
import { OSTypeBadge, OSTypeIcon, OSTypeLabel } from './os-type-badge'

// Full badge with icon and label
<OSTypeBadge osType="windows" />
<OSTypeBadge osType="Darwin" />
<OSTypeBadge osType="Ubuntu" />

// Icon only mode
<OSTypeBadge osType="linux" iconOnly />
<OSTypeIcon osType="macOS" size="w-6 h-6" />

// Label only mode
<OSTypeBadge osType="windows" labelOnly />
<OSTypeLabel osType="fedora" />

// With custom styling and right icon
<OSTypeBadge 
  osType="windows" 
  className="bg-blue-100" 
  iconColor="#0078d4"
  rightIcon={<ChevronRight />}
/>
```

The component automatically handles OS type normalization through utility functions, supports case-insensitive input, and gracefully handles unknown OS types. It uses responsive typography and includes proper TypeScript interfaces for type safety.
