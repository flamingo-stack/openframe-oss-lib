A client-side React component that renders a selectable button with optional icon, image, description, and tag elements, featuring visual selection states and accessibility support.

## Key Components

- **SelectButton** - Main forwardRef component with comprehensive selection UI
- **SelectButtonProps** - Interface defining all available props including visual and behavioral options
- **Accessibility features** - Includes proper ARIA attributes (`role="option"`, `aria-selected`) and focus management

## Usage Example

```typescript
import { SelectButton } from './select-button'
import { UserIcon } from '../icons'

// Basic selection button
<SelectButton
  title="Premium Plan"
  description="Advanced features included"
  selected={false}
  onClick={() => setSelected('premium')}
/>

// With icon and tag
<SelectButton
  title="Admin User"
  description="Full system access"
  selected={true}
  icon={<UserIcon />}
  tag="Recommended"
  tagVariant="solid"
  onClick={() => selectUserType('admin')}
/>

// With image
<SelectButton
  title="Profile Template"
  image={{
    src: "/templates/profile.jpg",
    alt: "Profile template preview"
  }}
  disabled={false}
  onClick={() => selectTemplate('profile')}
/>
```

The component supports responsive design with different sizing on mobile/desktop, visual selection states with custom CSS variables, and various content configurations including icons, images, descriptions, and tags.
