A reusable button component for selection interfaces that displays an item with optional icon/image, title, description, and visual selection state feedback.

## Key Components

- **`SelectButton`** - Main component with forward ref support for form integration
- **`SelectButtonProps`** - TypeScript interface defining all component properties including selection state, content options, and interaction handlers
- **`CheckCircleIcon`** - Visual indicator for selected state from the icons library

## Usage Example

```typescript
import { SelectButton } from './select-button'
import { UserIcon } from '../icons'

// Basic selection button
<SelectButton
  title="Premium Plan"
  description="Advanced features with priority support"
  selected={selectedPlan === 'premium'}
  onClick={() => setSelectedPlan('premium')}
/>

// With icon
<SelectButton
  title="Admin User"
  icon={<UserIcon />}
  selected={userType === 'admin'}
  onClick={() => setUserType('admin')}
/>

// With image
<SelectButton
  title="Avatar Option"
  image={{ src: '/avatar.jpg', alt: 'User avatar' }}
  selected={selectedAvatar === 'option1'}
  disabled={!canSelectAvatar}
  onClick={() => setSelectedAvatar('option1')}
/>
```

The component provides comprehensive accessibility support with ARIA attributes, responsive design with mobile/desktop variations, and integrated focus management for keyboard navigation.
