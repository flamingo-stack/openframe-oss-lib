A customizable button component that displays as a selectable option with support for icons, images, descriptions, and tags. Used for creating interactive selection lists or option grids with visual feedback for selected states.

## Key Components

- **SelectButton** - Main component with `forwardRef` support for external ref management
- **SelectButtonProps** - TypeScript interface defining all available props including styling and content options
- **Conditional rendering** - Smart display logic for icons, images, descriptions, and tags based on screen size

## Usage Example

```typescript
import { SelectButton } from './select-button'
import { UserIcon } from '../icons'

// Basic usage
<SelectButton
  title="Premium Plan"
  description="Advanced features included"
  selected={selectedPlan === 'premium'}
  onClick={() => setSelectedPlan('premium')}
/>

// With icon and tag
<SelectButton
  title="Team Workspace"
  description="Collaborate with your team"
  icon={<UserIcon />}
  tag="Popular"
  tagVariant="default"
  selected={workspace === 'team'}
  onClick={() => setWorkspace('team')}
/>

// With image
<SelectButton
  title="Custom Theme"
  image={{ src: "/theme-preview.jpg", alt: "Theme preview" }}
  disabled={!isPremium}
  onClick={handleThemeSelect}
/>
```

The component includes responsive design with different sizing for mobile/desktop, accessibility support with ARIA attributes, and visual states for hover, active, and selected interactions.
