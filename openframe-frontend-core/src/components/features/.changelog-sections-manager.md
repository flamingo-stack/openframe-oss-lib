A React component that organizes and manages changelog entries across four predefined sections: features, bug fixes, improvements, and breaking changes.

## Key Components

- **ChangelogSectionsManager**: Main component that renders multiple `ChangelogManager` instances for different changelog categories
- **ChangelogSectionsManagerProps**: Interface defining props including entry arrays, change handlers, styling options, and expand control

## Usage Example

```typescript
import { ChangelogSectionsManager } from './changelog-sections-manager';
import type { ChangelogEntry } from '../../types/product-release';

function ProductReleaseEditor() {
  const [features, setFeatures] = useState<ChangelogEntry[]>([]);
  const [bugs, setBugs] = useState<ChangelogEntry[]>([]);
  const [improvements, setImprovements] = useState<ChangelogEntry[]>([]);
  const [breaking, setBreaking] = useState<ChangelogEntry[]>([]);

  return (
    <ChangelogSectionsManager
      featuresAdded={features}
      bugsFixed={bugs}
      improvements={improvements}
      breakingChanges={breaking}
      onFeaturesAddedChange={setFeatures}
      onBugsFixedChange={setBugs}
      onImprovementsChange={setImprovements}
      onBreakingChangesChange={setBreaking}
      expandAll={true} // Useful after AI enrichment
      className="mt-4"
    />
  );
}
```

The component provides a structured layout for managing changelog entries with consistent styling and the ability to expand all sections simultaneously.
