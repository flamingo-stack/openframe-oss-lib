This index file serves as the main entry point for AI enrichment UI components, providing a centralized export interface for all AI-related form enhancement functionality.

## Key Components

- **AIEnrichButton** - Interactive button component for triggering AI enrichment processes
- **ConfidenceBadge** - Visual indicator showing AI confidence levels for enriched data
- **AIStatusIndicator** - Component displaying current AI processing status with multiple states
- **AIWarningsSection** - UI section for displaying AI-related warnings and alerts
- **AIEnrichSection** - Main container component for AI enrichment functionality with confidence tracking
- **AIRequiredBadge** - Badge component indicating fields that require AI processing

## Usage Example

```typescript
import { 
  AIEnrichButton, 
  ConfidenceBadge, 
  AIStatusIndicator,
  type AIStatus,
  type ConfidenceField 
} from './ai-enrich-components'

// Use AI enrichment components in a form
const MyForm = () => {
  const [status, setStatus] = useState<AIStatus>('idle')
  const confidenceData: ConfidenceField = { 
    value: 'enriched text', 
    confidence: 0.85 
  }

  return (
    <div>
      <AIEnrichButton onEnrich={() => setStatus('processing')} />
      <AIStatusIndicator status={status} />
      <ConfidenceBadge confidence={confidenceData.confidence} />
    </div>
  )
}
```
