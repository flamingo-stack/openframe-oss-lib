A customizable button component specifically designed for AI enrichment features, with built-in loading states and sparkles icon integration.

## Key Components

- **`AIEnrichButton`** - Main React component that renders an AI-themed action button
- **`AIEnrichButtonProps`** - TypeScript interface defining component props including onClick handler, loading states, and styling options
- **Size and variant mapping objects** - Internal mappings that translate component props to underlying Button component variants

## Usage Example

```typescript
import { AIEnrichButton } from './components/AIEnrichButton'

function MyComponent() {
  const [isEnriching, setIsEnriching] = useState(false)

  const handleEnrich = async () => {
    setIsEnriching(true)
    // Perform AI enrichment
    await enrichData()
    setIsEnriching(false)
  }

  return (
    <AIEnrichButton
      onClick={handleEnrich}
      loading={isEnriching}
      label="Enhance with AI"
      loadingLabel="Processing..."
      size="lg"
      variant="primary"
      className="my-custom-class"
    />
  )
}
```

The component automatically handles disabled states during loading and displays a sparkles icon when not in loading state, making it ideal for AI-powered features throughout your application.
