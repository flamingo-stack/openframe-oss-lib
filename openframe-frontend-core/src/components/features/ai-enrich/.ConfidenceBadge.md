A React component that displays confidence scores as styled badges with configurable labels, percentages, and visual styling based on confidence levels.

## Key Components

- **`ConfidenceBadge`** - Main component that renders a confidence score as a styled badge
- **`ConfidenceBadgeProps`** - Interface defining component props including confidence score, display options, and styling
- **`sizeStyles`** - Object containing Tailwind CSS classes for different badge sizes (sm, md, lg)

## Usage Example

```typescript
import { ConfidenceBadge } from './components/ConfidenceBadge'

function MyComponent() {
  return (
    <div>
      {/* Basic usage with confidence score */}
      <ConfidenceBadge confidence={85} />
      
      {/* Hide percentage, show only label */}
      <ConfidenceBadge 
        confidence={65} 
        showPercentage={false} 
        size="md"
      />
      
      {/* Custom styling */}
      <ConfidenceBadge 
        confidence={92} 
        size="lg"
        className="ml-2"
      />
      
      {/* Low confidence (may not render if level is 'none') */}
      <ConfidenceBadge confidence={15} />
    </div>
  )
}
```

The component automatically determines styling (colors, borders, backgrounds) based on the confidence value using imported helper functions, and returns `null` for confidence levels marked as 'none'.
