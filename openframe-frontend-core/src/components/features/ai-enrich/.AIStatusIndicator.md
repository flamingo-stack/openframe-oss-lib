A React component that displays the status of AI operations with optional confidence indicators and visual feedback.

## Key Components

- **`AIStatus`** - Type union for status states: 'idle' | 'loading' | 'success' | 'error'
- **`AIStatusIndicatorProps`** - Interface defining component props including status, confidence, message, and styling options
- **`AIStatusIndicator`** - Main component that renders status with sparkles icon, text, and optional confidence badge
- **`statusConfig`** - Configuration object mapping each status to its visual styling (colors, background, labels)

## Usage Example

```typescript
import { AIStatusIndicator } from './components/AIStatusIndicator'

// Basic usage with different statuses
<AIStatusIndicator status="loading" />
<AIStatusIndicator status="success" message="Data enriched successfully" />
<AIStatusIndicator status="error" message="Processing failed" />

// With confidence scoring
<AIStatusIndicator 
  status="success" 
  confidence={85}
  message="Profile enriched"
  showConfidence={true}
/>

// Custom styling
<AIStatusIndicator 
  status="idle"
  className="border rounded-md"
/>
```

The component automatically applies appropriate styling based on status, shows a pulsing animation during loading, and displays confidence scores as colored badges when available. Confidence styling is handled through imported utility functions.
