A React component that displays a visual indicator badge for fields required by AI enrichment features, featuring a sparkles icon and cyan styling.

## Key Components

- **AIRequiredBadge**: Main component that renders a rounded badge with sparkles icon
- **AIRequiredBadgeProps**: Interface defining optional className and size props
- **SparklesIcon**: Imported icon component for visual enhancement
- **cn**: Utility function for conditional className merging

## Usage Example

```typescript
import { AIRequiredBadge } from './components/AIRequiredBadge'

// Basic usage with default small size
<AIRequiredBadge />

// Medium size variant
<AIRequiredBadge size="md" />

// With custom styling
<AIRequiredBadge 
  size="sm" 
  className="absolute top-0 right-0" 
/>

// Typically used alongside form labels
<label>
  Product Description
  <AIRequiredBadge size="sm" />
</label>
```

The component uses ODS (design system) cyan color tokens and DM Sans font, providing consistent styling across the application. The badge automatically adjusts icon size and padding based on the selected size variant.
