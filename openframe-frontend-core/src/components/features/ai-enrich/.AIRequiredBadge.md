A React badge component that visually indicates fields required for AI enrichment, featuring a sparkles icon and ODS cyan styling with responsive sizing options.

## Key Components

- **AIRequiredBadge** - Main component that renders an inline badge with sparkles icon
- **AIRequiredBadgeProps** - Interface defining className and size props
- **SparklesIcon** - Imported icon component for visual enhancement
- **cn** - Utility function for conditional class name merging

## Usage Example

```typescript
import { AIRequiredBadge } from './components/ui/badges/AIRequiredBadge'

// Small inline badge (default)
<label>
  Description
  <AIRequiredBadge />
</label>

// Medium size variant
<div className="form-section">
  AI Content Field
  <AIRequiredBadge size="md" />
</div>

// With custom styling
<AIRequiredBadge 
  size="sm" 
  className="ml-4 opacity-80" 
/>
```

The component uses ODS (Open Design System) Flamingo cyan colors with 10% opacity background, includes a sparkles icon that scales with the badge size, and applies DM Sans font styling. The badge is designed to be placed inline next to form labels or field descriptions to clearly communicate AI functionality requirements.
