A comprehensive React component that provides a complete UI section for AI enrichment functionality, including status displays, required field validation, and confidence indicators.

## Key Components

- **AIEnrichSection** - Main component that orchestrates the entire AI enrichment experience
- **ConfidenceField** - Interface for displaying field-level confidence scores
- **AIRequiredField** - Interface for tracking and displaying missing required fields
- **AIEnrichSectionProps** - Comprehensive props interface with customizable labels, styling, and behavior options

## Usage Example

```typescript
import { AIEnrichSection } from './AIEnrichSection'

function MyForm() {
  const [enrichmentStatus, setEnrichmentStatus] = useState<'idle' | 'loading' | 'success' | 'error'>('idle')
  
  const requiredFields = [
    { key: 'email', label: 'Email', isFilled: !!formData.email },
    { key: 'version', label: 'Version', isFilled: !!formData.version }
  ]

  return (
    <AIEnrichSection
      onEnrich={() => performEnrichment()}
      status={enrichmentStatus}
      statusMessage="Successfully enriched with AI data"
      overallConfidence={85}
      requiredFields={requiredFields}
      warnings={['Some fields had low confidence scores']}
      onClear={() => setEnrichmentStatus('idle')}
      canEnrich={requiredFields.every(f => f.isFilled)}
    />
  )
}
```

The component handles loading states, displays missing required fields as badges, shows confidence scores, and provides a complete enrichment workflow with clear visual feedback.
