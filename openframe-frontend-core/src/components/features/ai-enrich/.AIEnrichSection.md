A React component that provides a comprehensive UI section for AI enrichment functionality, including status display, confidence metrics, warnings, and required field validation.

## Key Components

- **AIEnrichSection** - Main component managing AI enrichment workflow and display
- **ConfidenceField** - Interface for fields with confidence scoring
- **AIRequiredField** - Interface for tracking required field completion status
- **AIEnrichSectionProps** - Comprehensive props interface supporting various states and customization

## Usage Example

```typescript
import { AIEnrichSection, AIRequiredField } from './AIEnrichSection'

// Define required fields for validation
const requiredFields: AIRequiredField[] = [
  { key: 'email', label: 'Email', isFilled: !!formData.email },
  { key: 'version', label: 'Version', isFilled: !!formData.version }
]

// Use in component
<AIEnrichSection
  onEnrich={handleEnrichment}
  loading={isEnriching}
  canEnrich={allFieldsFilled}
  status="success"
  statusMessage="Package enriched successfully"
  overallConfidence={85}
  warnings={["Low confidence in description field"]}
  requiredFields={requiredFields}
  onClear={clearResults}
  title="Package AI Enrichment"
  buttonLabel="Enhance Package"
>
  <div>Additional enrichment results...</div>
</AIEnrichSection>
```

The component handles multiple states (idle, loading, success, error), displays unfilled required fields as badges when disabled, and provides flexible content areas for warnings and custom results display.
