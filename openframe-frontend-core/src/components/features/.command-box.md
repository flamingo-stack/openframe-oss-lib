A React component for displaying command text with optional action buttons in a styled container. Provides consistent UI patterns for showing terminal commands, scripts, or other code snippets with associated actions.

## Key Components

- **CommandBox**: Main component for rendering command display with actions
- **CommandBoxAction**: Interface defining button properties (label, onClick, variant, icon, etc.)
- **CommandBoxProps**: Props interface with command text, title, actions, and styling options

## Usage Example

```typescript
import { CommandBox } from './command-box'
import { Copy, Play } from 'lucide-react'

<CommandBox
  title="Installation Command"
  command="npm install @flamingo/ui-kit"
  maxLines={3}
  primaryAction={{
    label: 'Copy',
    onClick: () => navigator.clipboard.writeText(command),
    icon: <Copy className="w-4 h-4" />,
    variant: 'primary'
  }}
  secondaryAction={{
    label: 'Execute',
    onClick: handleExecute,
    icon: <Play className="w-4 h-4" />,
    variant: 'outline'
  }}
/>
```

The component features responsive design, monospace font for commands, line clamping support, and consistent ODS styling with primary/secondary action buttons.
