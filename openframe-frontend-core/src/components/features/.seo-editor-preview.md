A comprehensive React component that provides SEO metadata editing with real-time Open Graph social media preview functionality. This component handles SEO titles, descriptions, keywords, and OG images while automatically falling back to content data when SEO fields are empty.

## Key Components

- **SEOEditorPreview** - Main component with form fields and live preview
- **SEOEditorPreviewProps** - Interface defining all required and optional props
- **Image Upload Handler** - File upload functionality for OG images with loading states
- **Live Preview** - Real-time social media card preview showing how content appears when shared

## Usage Example

```typescript
import { SEOEditorPreview } from './seo-editor-preview';

function BlogEditor() {
  const [seo, setSeo] = useState({
    title: '',
    description: '',
    keywords: '',
    ogImageUrl: ''
  });

  const handleImageUpload = async (file: File): Promise<string> => {
    const formData = new FormData();
    formData.append('image', file);
    const response = await fetch('/api/upload', { 
      method: 'POST', 
      body: formData 
    });
    const { url } = await response.json();
    return url;
  };

  return (
    <SEOEditorPreview
      seoTitle={seo.title}
      seoDescription={seo.description}
      seoKeywords={seo.keywords}
      ogImageUrl={seo.ogImageUrl}
      title="Blog Post Title"
      summary="Blog post summary text"
      featuredImage="/featured-image.jpg"
      onSeoTitleChange={(value) => setSeo(prev => ({ ...prev, title: value }))}
      onSeoDescriptionChange={(value) => setSeo(prev => ({ ...prev, description: value }))}
      onSeoKeywordsChange={(value) => setSeo(prev => ({ ...prev, keywords: value }))}
      onOgImageUrlChange={(value) => setSeo(prev => ({ ...prev, ogImageUrl: value }))}
      onOgImageUpload={handleImageUpload}
      domain="example.com"
    />
  );
}
```

The component features a responsive two-column layout, intelligent fallback to content data, drag-and-drop image upload, and a live social media preview card that updates in real-time as users edit SEO fields.
