This file provides a comprehensive SEO editor component with real-time social media preview functionality for managing meta tags, Open Graph data, and image uploads in content management workflows.

## Key Components

- **SEOEditorPreview**: Main component for editing SEO meta tags with live preview
- **SEOEditorPreviewProps**: TypeScript interface defining all props including SEO fields, change handlers, and AI confidence scores
- **Image upload handling**: File upload with progress states and error handling
- **Live preview**: Real-time Open Graph card preview showing how content appears on social media
- **AI integration**: Support for displaying AI-generated content with confidence badges

## Usage Example

```typescript
import { SEOEditorPreview } from './seo-editor-preview';

function BlogEditor() {
  const [seoData, setSeoData] = useState({
    seoTitle: '',
    seoDescription: '',
    seoKeywords: '',
    ogImageUrl: ''
  });

  const handleImageUpload = async (file: File) => {
    const formData = new FormData();
    formData.append('image', file);
    const response = await fetch('/api/upload', { method: 'POST', body: formData });
    const { url } = await response.json();
    return url;
  };

  return (
    <SEOEditorPreview
      seoTitle={seoData.seoTitle}
      seoDescription={seoData.seoDescription}
      seoKeywords={seoData.seoKeywords}
      ogImageUrl={seoData.ogImageUrl}
      title="Blog Post Title"
      summary="Blog post summary"
      featuredImage="/featured.jpg"
      onSeoTitleChange={(value) => setSeoData(prev => ({ ...prev, seoTitle: value }))}
      onSeoDescriptionChange={(value) => setSeoData(prev => ({ ...prev, seoDescription: value }))}
      onSeoKeywordsChange={(value) => setSeoData(prev => ({ ...prev, seoKeywords: value }))}
      onOgImageUrlChange={(value) => setSeoData(prev => ({ ...prev, ogImageUrl: value }))}
      onOgImageUpload={handleImageUpload}
      aiConfidenceSeoTitle={0.85}
      domain="example.com"
    />
  );
}
```

The component automatically falls back to content fields (title, summary, featuredImage) when SEO fields are empty, displays AI confidence scores when available, and provides drag-and-drop image upload functionality with real-time preview updates.
