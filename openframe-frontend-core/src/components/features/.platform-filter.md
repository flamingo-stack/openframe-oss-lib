A React component that renders a filterable platform selector with customizable buttons and optional icons.

## Key Components

- **PlatformFilterComponent** - Main component that displays platform filter buttons
- **PlatformFilterComponentProps** - TypeScript interface defining component props
- **getSmallPlatformIcon** - Utility function for retrieving platform icons

## Usage Example

```typescript
import { PlatformFilterComponent } from './platform-filter';
import type { PlatformConfig } from '../../types/platform';

const platforms: PlatformConfig[] = [
  { value: 'ios', label: 'iOS' },
  { value: 'android', label: 'Android' },
  { value: 'web', label: 'Web' }
];

function GameFilter() {
  const [selectedPlatform, setSelectedPlatform] = useState('all');

  return (
    <PlatformFilterComponent
      selectedPlatform={selectedPlatform}
      onPlatformChange={setSelectedPlatform}
      platforms={platforms}
      showIcons={true}
      size="default"
      className="my-4"
    />
  );
}
```

The component automatically includes an "All Platforms" option and highlights the currently selected platform. Icons can be toggled via the `showIcons` prop, and button styling follows the DM Sans font family with responsive text sizing.
