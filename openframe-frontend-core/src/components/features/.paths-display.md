Displays a list of file/folder paths with optional copy-to-clipboard functionality, styled with ODS design tokens and monospace formatting.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `PathsDisplay` | Component | Main component rendering a styled list of paths with optional copy buttons |
| `PathsDisplayProps` | Interface | Props definition for the `PathsDisplay` component |
| `OPENFRAME_PATHS` | Constant | Pre-defined path sets for Windows, macOS, and Linux platforms |
| `getOpenFramePaths` | Function | Returns a mutable array of paths for a given platform key |
| `OpenFramePathsPlatform` | Type | Union type of `'windows' \| 'darwin' \| 'linux'` |

## Usage Example

```typescript
import { PathsDisplay, getOpenFramePaths } from '@flamingo/ui-kit/components/features'

// Basic usage with antivirus exclusion paths
<PathsDisplay
  paths={getOpenFramePaths('windows')}
  title="Antivirus Exclusions"
  description="Add these folders to your antivirus exclusions list"
  onCopyPath={(path) => {
    navigator.clipboard.writeText(path)
    toast({ title: 'Copied', description: 'Path copied to clipboard' })
  }}
/>

// Display-only, no copy buttons
<PathsDisplay
  paths={getOpenFramePaths('linux')}
  title="Linux Agent Paths"
  showCopyButtons={false}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `paths` | `string[]` | — | Array of paths to display |
| `onCopyPath` | `(path: string) => void` | — | Callback triggered on copy button click |
| `title` | `string` | — | Optional heading above the list |
| `description` | `string` | — | Optional subtext below the title |
| `showCopyButtons` | `boolean` | `true` | Toggle copy button visibility |
| `copyIconSize` | `string` | `'w-5 h-5'` | Tailwind size classes for the copy icon |
| `className` | `string` | — | Additional CSS classes for the container |

> **Note:** The component renders `null` when `paths` is empty or undefined. Copy buttons only render when both `showCopyButtons` is `true` **and** `onCopyPath` is provided.
