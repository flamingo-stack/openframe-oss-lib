A React component for displaying lists of file/folder paths with optional copy-to-clipboard functionality and consistent styling.

## Key Components

- **`PathsDisplay`** - Main component for rendering path lists with copy buttons
- **`PathsDisplayProps`** - TypeScript interface defining component props
- **`OPENFRAME_PATHS`** - Pre-defined path constants for Windows, macOS, and Linux platforms
- **`getOpenFramePaths()`** - Utility function to retrieve platform-specific paths
- **`OpenFramePathsPlatform`** - Type for supported platforms

## Usage Example

```typescript
import { PathsDisplay, getOpenFramePaths } from './paths-display'

// Basic usage
<PathsDisplay
  paths={['/usr/local/bin', '/opt/app']}
  title="Installation Paths"
  description="Required directories for the application"
  onCopyPath={(path) => navigator.clipboard.writeText(path)}
/>

// Using pre-defined paths
<PathsDisplay
  paths={getOpenFramePaths('windows')}
  title="Antivirus Exclusions"
  showCopyButtons={true}
  onCopyPath={(path) => {
    navigator.clipboard.writeText(path)
    console.log(`Copied: ${path}`)
  }}
/>

// Minimal configuration
<PathsDisplay
  paths={['C:\\Program Files\\App']}
  showCopyButtons={false}
/>
```
