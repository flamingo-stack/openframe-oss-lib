Barrel export file for the notifications module, re-exporting all public components, hooks, context providers, and TypeScript types from the notifications subsystem.

## Key Components

**Context & Hooks**
- `NotificationsProvider` — Context provider for managing notification state
- `useNotifications` — Hook to access notifications context (throws if no provider)
- `useOptionalNotifications` — Hook to access notifications context (returns `undefined` if no provider)

**UI Components**
- `NotificationDrawer` — Slide-out drawer displaying a list of notifications
- `NotificationTile` — Individual notification item/card
- `NotificationPopups` — Toast-style popup notifications with configurable position

**Types**
- `Notification`, `NotificationVariant`, `NotificationSeverity`, `AddNotificationInput`
- `NotificationsProviderProps`, `NotificationsActions`
- `NotificationDrawerProps`, `NotificationTileProps`, `NotificationPopupsProps`, `NotificationPopupsPosition`

## Usage Example

```typescript
import {
  NotificationsProvider,
  useNotifications,
  NotificationPopups,
  NotificationDrawer,
} from './notifications'
import type { AddNotificationInput } from './notifications'

function App() {
  return (
    <NotificationsProvider>
      <NotificationPopups position="top-right" />
      <NotificationDrawer />
      <MyPage />
    </NotificationsProvider>
  )
}

function MyPage() {
  const { addNotification } = useNotifications()

  const notify = () => {
    const input: AddNotificationInput = {
      title: 'Success',
      message: 'Operation completed.',
      variant: 'toast',
      severity: 'success',
    }
    addNotification(input)
  }

  return <button onClick={notify}>Notify</button>
}
```
