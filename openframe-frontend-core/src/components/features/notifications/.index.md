Barrel export file for the notifications module, re-exporting all public components, hooks, context utilities, and TypeScript types from the notifications subsystem.

## Key Components

- **`NotificationsProvider`** — Context provider that wraps the app/section requiring notification state
- **`useNotifications`** / **`useOptionalNotifications`** — Hooks for consuming notification context (required vs. optional access)
- **`NotificationDrawer`** — Slide-out drawer component for displaying a list of notifications
- **`NotificationTile`** — Individual notification item/card component
- **`NotificationPopups`** — Toast-style popup notifications with configurable positioning

## Exported Types

| Type | Description |
|------|-------------|
| `Notification` | Core notification data shape |
| `NotificationVariant` | Visual variant (e.g. info, warning) |
| `NotificationSeverity` | Severity level enum/union |
| `AddNotificationInput` | Input shape for creating a new notification |
| `NotificationsActions` | Dispatch actions available via context |
| `NotificationPopupsPosition` | Positioning options for popup notifications |

## Usage Example

```typescript
import {
  NotificationsProvider,
  useNotifications,
  NotificationDrawer,
  NotificationPopups,
} from './notifications'
import type { AddNotificationInput } from './notifications'

function App() {
  return (
    <NotificationsProvider>
      <NotificationPopups position="top-right" />
      <NotificationDrawer />
      <MyPage />
    </NotificationsProvider>
  )
}

function MyPage() {
  const { addNotification } = useNotifications()

  const notify = () => {
    const input: AddNotificationInput = {
      title: 'Task complete',
      variant: 'success',
      severity: 'low',
    }
    addNotification(input)
  }

  return <button onClick={notify}>Notify</button>
}
```
