A right-side drawer component for displaying and managing unread notifications, built on Radix UI Dialog primitives with infinite scroll support.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `NotificationDrawer` | Component | Root drawer component; consumes `NotificationsContext` and renders the full panel |
| `DrawerScrollList` | Internal | Scrollable list with `IntersectionObserver`-based infinite scroll sentinel |
| `DrawerLoadingTiles` | Internal | Animated skeleton tiles shown while loading more notifications |
| `EmptyState` | Internal | Shown when there are no unread notifications |
| `ShowNotificationsToggleRow` | Internal | Toggle switch to enable/disable pop-up notification banners |
| `NotificationsHistoryButton` | Internal | Outline button that triggers `onHistoryClick` and closes the drawer |

### `NotificationDrawerProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes for the drawer content |
| `liveDurationMs` | `number` | — | Passed to each `NotificationTile` for live countdown behavior |
| `loadMoreRootMargin` | `string` | `'200px'` | `rootMargin` for the `IntersectionObserver` that triggers pagination |

## Usage Example

```typescript
import { NotificationsProvider } from './notifications-context'
import { NotificationDrawer } from './notification-drawer'

export function AppShell() {
  return (
    <NotificationsProvider
      notifications={notifications}
      onMarkRead={handleMarkRead}
      onLoadMore={handleLoadMore}
      hasMore={hasMore}
    >
      {/* Drawer reads all state from context — no extra props required */}
      <NotificationDrawer
        liveDurationMs={30_000}
        loadMoreRootMargin="300px"
      />
    </NotificationsProvider>
  )
}
```

> **Note:** `NotificationDrawer` returns `null` when rendered outside a `NotificationsProvider`. Open/close state, `markRead`, `markAllRead`, and pagination are all driven by `useOptionalNotifications()`.
