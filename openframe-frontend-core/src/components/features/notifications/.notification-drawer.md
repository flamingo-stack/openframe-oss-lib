A right-side drawer component for displaying and managing unread notifications, built on Radix UI Dialog primitives with infinite scroll support.

## Key Components

- **`NotificationDrawer`** — Main exported component. Renders a slide-in drawer (right side) consuming the `NotificationsContext` via `useOptionalNotifications`. Returns `null` if context is unavailable.
- **`DrawerScrollList`** — Internal scrollable list with `IntersectionObserver`-based infinite scroll. Triggers `loadMore` when a sentinel `div` enters the viewport.
- **`DrawerLoadingTiles`** — Animated pulse skeleton tiles shown during paginated load.
- **`EmptyState`** — Displayed when there are no unread notifications.
- **`ShowNotificationsToggleRow`** — Toggle switch to enable/disable pop-up notification popups.
- **`NotificationsHistoryButton`** — Outline button that calls `onHistoryClick` and closes the drawer.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes for the drawer content |
| `liveDurationMs` | `number` | — | Forwarded to `NotificationTile` for live countdown display |
| `loadMoreRootMargin` | `string` | `"200px"` | `IntersectionObserver` root margin for triggering load-more |

## Usage Example

```typescript
import { NotificationDrawer } from './notification-drawer'
import { NotificationsProvider } from './notifications-context'

export function AppShell() {
  return (
    <NotificationsProvider
      notifications={notifications}
      onHistoryClick={() => router.push('/notifications')}
    >
      <NotificationDrawer
        liveDurationMs={30_000}
        loadMoreRootMargin="300px"
      />
    </NotificationsProvider>
  )
}
```

> **Note:** `NotificationDrawer` must be rendered inside a `NotificationsProvider`. If the context is absent, the component silently renders nothing — safe for conditional provider usage.
