Renders a fixed-position stack of transient notification pop-ups, automatically filtering to unread, unsettled notifications within a configurable live window.

## Key Components

**`NotificationPopups`** — Main exported component that reads from `NotificationsContext` and displays up to `maxVisible` live notifications as dismissible tiles.

**`NotificationPopupsPosition`** — Union type for the four supported anchor positions: `'top-right'`, `'top-left'`, `'bottom-right'`, `'bottom-left'`.

**`NotificationPopupsProps`** — Component props interface:

| Prop | Default | Description |
|------|---------|-------------|
| `liveDurationMs` | `4000` | How long (ms) a notification stays visible |
| `maxVisible` | `4` | Maximum simultaneous pop-ups |
| `position` | `'top-right'` | Screen anchor position |
| `hideWhenDrawerOpen` | `true` | Hides stack when the notification drawer is open |
| `className` | — | Additional CSS classes |

**`positionClasses`** — Internal map of position tokens to Tailwind utility classes, reversing flex direction for bottom-anchored positions.

## Usage Example

```typescript
import { NotificationPopups } from './notification-popups'
import { NotificationsProvider } from './notifications-context'

export function AppShell() {
  return (
    <NotificationsProvider>
      <main>{/* app content */}</main>
      <NotificationPopups
        position="bottom-right"
        liveDurationMs={5000}
        maxVisible={3}
        hideWhenDrawerOpen={true}
      />
    </NotificationsProvider>
  )
}
```

## Behaviour Notes

- Returns `null` if no `NotificationsContext` is present, `showPopups` is false, or the drawer is open with `hideWhenDrawerOpen` enabled.
- Clicking a tile body invokes `notification.onClick` (if set) and marks it read + settled; otherwise opens the drawer and marks it settled.
- Nested `<button>` elements (dismiss, check) stop click propagation via `closest('button')` guard, preventing double-handling.
- The container uses `aria-live="polite"` for accessible announcements; individual tiles are keyboard-accessible via `role="button"` + `tabIndex={0}`.
