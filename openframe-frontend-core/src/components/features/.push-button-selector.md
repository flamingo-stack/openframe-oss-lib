A React component that provides an interactive button-based selector for choosing between multiple options, supporting both single and multi-select modes with visual feedback and selection summaries.

## Key Components

- **`SelectableOption`** - Interface defining selectable items with id, name, description, icon, and color properties
- **`PushButtonSelector`** - Main component rendering clickable option cards with selection indicators
- **`toggleSelection`** - Function handling selection logic for both single and multi-select modes
- **`getSelectedOptions`** - Utility function filtering selected options for display

## Usage Example

```typescript
import { PushButtonSelector, SelectableOption } from './push-button-selector';
import { Star, Heart } from 'lucide-react';

const options: SelectableOption[] = [
  { id: '1', name: 'Premium', description: 'Best value option', icon: <Star /> },
  { id: '2', name: 'Standard', description: 'Basic features', icon: <Heart /> }
];

function MyComponent() {
  const [selected, setSelected] = useState<string[]>([]);

  return (
    <PushButtonSelector
      options={options}
      selectedIds={selected}
      onSelectionChange={setSelected}
      multiSelect={true}
      title="Choose Plan"
      selectionSummary={true}
      helpText="Select one or more options"
    />
  );
}
```

The component features hover effects, selection animations, validation warnings for empty selections, and an optional summary panel showing selected items as badges.
