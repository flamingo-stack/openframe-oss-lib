A React component that displays multiple OS type badges in a horizontal group layout, commonly used for showing supported platforms in scripts and multi-OS contexts.

## Key Components

- **`OSTypeBadgeGroup`** - Main component that renders multiple OS badges with optional display limits
- **`OSTypeBadgeGroupProps`** - Interface defining component props including OS types array, styling options, and display limits
- **`maxDisplay`** - Optional prop to limit visible badges with "+N more" overflow indicator

## Usage Example

```typescript
import { OSTypeBadgeGroup } from './components/os-type-badge-group'

// Basic usage with multiple OS types
<OSTypeBadgeGroup osTypes={['windows', 'darwin', 'linux']} />

// With display limit and custom styling
<OSTypeBadgeGroup 
  osTypes={['macos', 'linux', 'windows', 'freebsd']} 
  maxDisplay={3}
  iconSize="w-6 h-6"
  className="my-4"
/>

// Handles empty arrays gracefully
<OSTypeBadgeGroup osTypes={[]} /> // Returns null
```

The component automatically handles OS type aliases (e.g., 'darwin' for macOS) and displays icons only without labels. When `maxDisplay` is set and exceeded, remaining badges are collapsed into a "+N more" indicator.
