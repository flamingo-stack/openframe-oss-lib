A React component that displays multiple operating system type badges in a horizontal group, useful for showing supported platforms for scripts and applications.

## Key Components

- **OSTypeBadgeGroup**: Main component that renders a collection of OS badges
- **OSTypeBadgeGroupProps**: Interface defining component properties including OS types array, styling options, and display limits
- Integrates with **OSTypeBadge** component for individual badge rendering
- **maxDisplay** prop with overflow handling ("+N more" indicator)

## Usage Example

```typescript
import { OSTypeBadgeGroup } from './os-type-badge-group'

// Basic usage with multiple OS types
<OSTypeBadgeGroup osTypes={['windows', 'darwin', 'linux']} />

// With display limit and custom styling
<OSTypeBadgeGroup 
  osTypes={['macos', 'linux', 'windows', 'freebsd']} 
  maxDisplay={3}
  iconSize="w-6 h-6"
  className="my-4"
/>

// Handle empty arrays gracefully
<OSTypeBadgeGroup osTypes={[]} /> // Returns null
```

The component automatically handles OS type aliases (e.g., 'darwin' â†’ macOS), displays only icons without labels, and provides responsive wrapping with gap spacing. When the number of OS types exceeds `maxDisplay`, it shows a truncated list with a "+N more" indicator.
