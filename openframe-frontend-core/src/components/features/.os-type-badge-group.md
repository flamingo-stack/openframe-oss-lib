A React component that displays multiple OS type badges in a horizontal group, typically used for showing supported platforms across scripts and multi-OS applications.

## Key Components

**OSTypeBadgeGroup** - Main component that renders a collection of OS type badges
- `osTypes: string[]` - Array of OS type strings (handles case-insensitive input and aliases)
- `className?: string` - Optional additional CSS classes for styling
- `iconSize?: string` - Icon size class (defaults to 'w-5 h-5')
- `maxDisplay?: number` - Limits displayed badges and shows "+N more" for overflow

**Dependencies**
- `OSTypeBadge` - Individual badge component for single OS types
- `cn` - Utility for conditional CSS class names

## Usage Example

```typescript
import { OSTypeBadgeGroup } from './os-type-badge-group'

// Basic usage with multiple OS types
<OSTypeBadgeGroup osTypes={['windows', 'darwin', 'linux']} />

// With maximum display limit
<OSTypeBadgeGroup 
  osTypes={['windows', 'macos', 'linux', 'freebsd', 'openbsd']} 
  maxDisplay={3} 
/>

// Custom styling and icon size
<OSTypeBadgeGroup 
  osTypes={['macos', 'linux']} 
  className="justify-center"
  iconSize="w-6 h-6"
/>

// Handles empty arrays gracefully
<OSTypeBadgeGroup osTypes={[]} /> // Renders nothing
```

The component automatically wraps badges using flexbox, handles empty arrays by returning null, and displays a "+N more" indicator when the badge count exceeds the `maxDisplay` limit.
