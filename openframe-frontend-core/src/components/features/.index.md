Central barrel export file for the feature components library, re-exporting all public UI components, hooks, providers, and utilities used across the Flamingo/OpenFrame platform.

## Key Components

**Theme System**
- `DynamicThemeProvider`, `useDynamicTheme` — runtime theme switching
- `ThemeProvider`, `useTheme`, `useThemeToggle` — canonical ODS theme layer (wraps `next-themes`), headless by design; includes `THEME_STORAGE_KEY`, `THEME_ATTRIBUTE`, `DEFAULT_THEME` constants and related types

**Video Primitives**
- `video` — unified `<Video>` component replacing legacy `<VideoPlayer>` + `<YouTubeEmbed>`
- `entity-video-section`, `video-bites-display`, `video-ratio-tabs` — public detail-page wrappers and aspect-ratio grouping
- `video-clips-section`, `highlight-video-section`, `highlight-video-combined-section` — clip and highlight management
- `transcript-summary-editor`, `transcribe-summarize-section` — AI transcription/summarization editors

**UI Managers & Selectors**
- `array-entry-manager`, `changelog-manager`, `media-gallery-manager`, `social-links-manager` — CRUD-style list managers
- `filters-dropdown`, `platform-filter`, `status-filter-component`, `tags-selector` — filtering primitives
- `command-box`, `section-selector`, `select-button`, `push-button-selector` — input/selection controls

**Platform & Integration**
- `os-type-badge`, `os-type-badge-group`, `platform-badge` — OS/platform labeling
- `auth-providers-list`, `provider-button` — authentication UI
- `github-releases-manager`, `clickup-tasks-manager` — third-party integrations
- `ai-enrich`, `policy-configuration-panel` — AI enrichment controls
- `notifications`, `board`, `ticket-status-config-list` — MSP operational UI

## Usage Example

```typescript
import {
  ThemeProvider,
  useThemeToggle,
  Video,
  FiltersDropdown,
  CommandBox,
  AiEnrich,
} from '@your-org/feature-components'

function App() {
  return (
    <ThemeProvider defaultTheme="dark" attribute="class">
      <CommandBox />
      <Video src="https://example.com/video.mp4" />
    </ThemeProvider>
  )
}

function ThemeToggle() {
  const { toggle, theme } = useThemeToggle()
  return <button onClick={toggle}>Current: {theme}</button>
}
```
