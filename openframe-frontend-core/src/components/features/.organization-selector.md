A React component that provides a dropdown selector for choosing organizations with icons, image preloading, and loading states. Built on top of a Select UI component with customizable styling and organization limits.

## Key Components

**OrganizationOption Interface**
- Defines organization data structure with id, name, imageUrl, and isDefault properties

**OrganizationSelectorProps Interface** 
- Comprehensive props including organizations array, selection handlers, styling options, and loading states

**OrganizationSelector Component**
- Main dropdown component with image preloading via `useBatchImages` hook
- Integrates `OrganizationIcon` component for consistent organization branding
- Supports loading skeletons and customizable appearance

## Usage Example

```typescript
import { OrganizationSelector } from './organization-selector'

const organizations = [
  { id: '1', organizationId: 'org-1', name: 'Acme Corp', imageUrl: '/logo1.png' },
  { id: '2', organizationId: 'org-2', name: 'Tech Inc', isDefault: true }
]

function MyComponent() {
  const [selectedOrgId, setSelectedOrgId] = useState('')

  return (
    <OrganizationSelector
      organizations={organizations}
      value={selectedOrgId}
      onValueChange={setSelectedOrgId}
      label="Select Organization"
      placeholder="Choose organization"
      iconSize="md"
      maxItems={50}
    />
  )
}
```
