A React component that provides a unified interface for editing SEO metadata and Open Graph settings with live preview capabilities.

## Key Components

- **OGEditorPreview** - Main component that renders SEO form fields alongside OG image upload and preview
- **OGEditorPreviewProps** - TypeScript interface defining component props including SEO fields, change handlers, and injectable components
- Form fields for SEO title, description, keywords, and OG image URL
- Responsive grid layout with proper spacing and theming
- Support for disabled state and custom styling

## Usage Example

```typescript
import { OGEditorPreview } from './og-editor-preview';
import { OGImageUpload } from './og-image-upload';
import { OGPreview } from './og-preview';

function BlogEditor() {
  const [seoTitle, setSeoTitle] = useState('');
  const [seoDescription, setSeoDescription] = useState('');
  const [seoKeywords, setSeoKeywords] = useState('');

  return (
    <OGEditorPreview
      seoTitle={seoTitle}
      seoDescription={seoDescription}
      seoKeywords={seoKeywords}
      onSeoTitleChange={setSeoTitle}
      onSeoDescriptionChange={setSeoDescription}
      onSeoKeywordsChange={setSeoKeywords}
      OGImageUploadComponent={<OGImageUpload />}
      OGPreviewComponent={<OGPreview title={seoTitle} />}
      title="Blog Post Title"
      summary="Post summary"
      featuredImage="/image.jpg"
    />
  );
}
```
