A unified React component for editing SEO metadata and Open Graph data with live preview functionality. This component provides a consistent interface for managing meta titles, descriptions, keywords, and OG images across different content types.

## Key Components

- **OGEditorPreview** - Main component that renders the complete SEO/OG editing interface
- **OGEditorPreviewProps** - TypeScript interface defining all required props including SEO fields, auto-populate sources, change handlers, and render prop components
- Responsive grid layout with form inputs for SEO metadata
- Integration points for custom OG image upload and preview components

## Usage Example

```typescript
import { OGEditorPreview } from './components/og-editor-preview';
import { OGImageUpload } from './components/og-image-upload';
import { OGPreviewCard } from './components/og-preview-card';

function BlogEditor() {
  const [seoData, setSeoData] = useState({
    title: '',
    description: '',
    keywords: ''
  });

  return (
    <OGEditorPreview
      seoTitle={seoData.title}
      seoDescription={seoData.description}
      seoKeywords={seoData.keywords}
      ogImageUrl=""
      title="Blog Post Title"
      summary="Auto-populated summary"
      featuredImage=""
      onSeoTitleChange={(title) => setSeoData(prev => ({ ...prev, title }))}
      onSeoDescriptionChange={(description) => setSeoData(prev => ({ ...prev, description }))}
      onSeoKeywordsChange={(keywords) => setSeoData(prev => ({ ...prev, keywords }))}
      onOgImageUrlChange={(url) => console.log('OG image changed:', url)}
      OGImageUploadComponent={<OGImageUpload />}
      OGPreviewComponent={<OGPreviewCard />}
    />
  );
}
```
