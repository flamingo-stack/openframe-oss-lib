A React component for displaying organization logos with automatic fallback to initials, supporting both pre-fetched blob URLs and authenticated image fetching. Provides visual parity with VendorIcon styling and multiple size variants.

## Key Components

- **OrganizationIcon**: Main component with dual-mode image support (blob URLs vs API fetching)
- **OrganizationIconProps**: TypeScript interface defining all configuration options
- **Size variants**: Six predefined sizes (xs, sm, md, lg, l, xl) with corresponding pixel dimensions
- **Background styles**: Three variants (dark, light, white) for different UI contexts
- **useAuthenticatedImage hook**: Handles secure image fetching with loading states

## Usage Example

```typescript
// With pre-fetched blob URL
<OrganizationIcon
  imageUrl="blob:http://localhost:3000/..."
  organizationName="Acme Corp"
  size="md"
  preFetched={true}
/>

// With automatic authenticated fetching
<OrganizationIcon
  imageUrl="/api/organizations/123/image"
  organizationName="Tech Solutions"
  size="xl"
  refreshKey={organization.imageVersion}
  backgroundStyle="light"
/>

// Without background for inline use
<OrganizationIcon
  imageUrl={null}
  organizationName="New Company"
  size="sm"
  showBackground={false}
/>
```

The component automatically falls back to displaying 2-letter initials when no image is available, with built-in loading states and zero layout shifts.
