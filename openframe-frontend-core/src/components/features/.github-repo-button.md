A specialized button component that creates a pre-configured GitHub repository link with consistent styling and GitHub icon. It extends the base Button component with GitHub-specific defaults and constraints.

## Key Components

- **GithubRepoButtonProps**: Interface extending ButtonProps with restricted variants and GitHub-specific defaults
- **GithubRepoButton**: Main component function that renders a GitHub-styled button with icon
- **Default Props**: Pre-configured with GitHub icon, outline variant, and default repository URL

## Usage Example

```typescript
import { GithubRepoButton } from './github-repo-button';

// Basic usage with defaults
<GithubRepoButton />

// Custom text and repository
<GithubRepoButton 
  href="https://github.com/myorg/myrepo"
>
  View Source Code
</GithubRepoButton>

// Different sizes
<GithubRepoButton size="sm">
  Star on GitHub
</GithubRepoButton>

<GithubRepoButton size="lg">
  Contribute
</GithubRepoButton>

// With additional props
<GithubRepoButton 
  className="mt-4"
  disabled={false}
  onClick={() => console.log('GitHub button clicked')}
>
  Fork Repository
</GithubRepoButton>
```

The component automatically opens links in new tabs and includes the GitHub icon, making it ideal for documentation sites and project showcases.
