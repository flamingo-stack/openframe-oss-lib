A reusable React component that renders a GitHub repository button with consistent styling and behavior, pre-configured to link to the Flamingo Stack OpenFrame repository.

## Key Components

- **GithubRepoButton** - Main component that renders a styled button with GitHub icon
- **GithubRepoButtonProps** - TypeScript interface extending ButtonProps with custom overrides
- **Button** - Underlying UI button component from the design system
- **GitHubIcon** - GitHub logo icon component

## Usage Example

```typescript
import { GithubRepoButton } from './github-repo-button';

// Basic usage with default text and repository
<GithubRepoButton />

// Custom text and size
<GithubRepoButton size="lg">
  View Source Code
</GithubRepoButton>

// Custom repository URL
<GithubRepoButton 
  href="https://github.com/your-org/your-repo"
  className="mt-4"
>
  Fork on GitHub
</GithubRepoButton>

// Small variant for compact layouts
<GithubRepoButton size="sm">
  GitHub
</GithubRepoButton>
```

The component automatically opens links in a new tab, uses an outline variant styling, and includes the GitHub icon on the left. It defaults to the OpenFrame repository but accepts custom URLs and styling overrides.
