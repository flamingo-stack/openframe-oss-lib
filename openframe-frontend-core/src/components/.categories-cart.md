A React component that renders an interactive category card displaying vendor icons, category information, and statistics with hover animations and responsive design.

## Key Components

- **CategoriesCart**: Main component that renders a clickable category card
- **VendorIcon**: Child component for displaying vendor logos in a grid layout
- **Link**: Next.js navigation component for routing to category pages
- **ChevronRight**: Lucide icon for the call-to-action arrow button

## Props Interface

The component accepts `RealCategoryCardProps` with the following properties:

- `category` - Category object with name, slug, and description
- `vendors` - Array of vendor objects for icon display
- `vendorCount` - Total number of products in the category
- `subcategoryCount` - Number of subcategories
- `isLoading` - Loading state for skeleton UI
- `className` - Additional CSS classes

## Usage Example

```typescript
import { CategoriesCart } from './categories-cart';

// Basic usage with category data
<CategoriesCart
  category={{
    name: "AI & Machine Learning",
    slug: "ai-ml",
    description: "Advanced AI tools and platforms"
  }}
  vendors={vendorArray}
  vendorCount={150}
  subcategoryCount={8}
  isLoading={false}
/>

// Loading state
<CategoriesCart
  category={category}
  vendors={[]}
  vendorCount={0}
  subcategoryCount={0}
  isLoading={true}
/>
```

The component features gradient fade effects on vendor icons, skeleton loading states, and hover animations that highlight the category with accent colors and transform the arrow button.
