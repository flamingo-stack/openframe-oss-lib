A React component library that provides a styled popover interface built on top of Radix UI primitives with custom styling and animations.

## Key Components

- **`Popover`** - Root container component that manages popover state
- **`PopoverTrigger`** - Button or element that triggers the popover to open/close
- **`PopoverContent`** - The actual popover content container with styling and animations

## Usage Example

```typescript
import { Popover, PopoverTrigger, PopoverContent } from "./popover"

function MyComponent() {
  return (
    <Popover>
      <PopoverTrigger asChild>
        <button>Open Popover</button>
      </PopoverTrigger>
      <PopoverContent>
        <div className="space-y-2">
          <h4 className="font-medium">Settings</h4>
          <p className="text-sm text-muted-foreground">
            Configure your preferences here.
          </p>
        </div>
      </PopoverContent>
    </Popover>
  )
}
```

The `PopoverContent` comes with built-in styling including rounded borders, shadows, and smooth enter/exit animations. It defaults to center alignment with a 4px offset from the trigger element. The styling uses CSS custom properties for theming (`bg-popover`, `text-popover-foreground`) and includes responsive animations based on the popover's position.
