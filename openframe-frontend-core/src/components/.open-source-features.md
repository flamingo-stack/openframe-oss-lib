React component that displays a section highlighting OpenFrame's open-source features and philosophy through a grid of feature cards with icons, titles, and descriptions.

## Key Components

### OpenSourceFeatures
- **Type**: React functional component
- **Purpose**: Main section component displaying open-source benefits
- **Features**: Responsive grid layout with bold accent text styling

### FeatureCard
- **Type**: React functional component with props interface
- **Props**: `icon` (React.ReactNode), `title` (string), `description` (string)
- **Features**: Hover animations, consistent card styling, icon containers

### Feature Data
- **Built on FOSS**: Emphasizes transparency and community-driven development
- **Own Your Stack**: Highlights cost savings and customization
- **Modular by Design**: Showcases adaptability and extensibility
- **Community-Powered**: Focuses on collaborative development model

## Usage Example

```typescript
import OpenSourceFeatures from './open-source-features';

function HomePage() {
  return (
    <div>
      <HeroSection />
      <OpenSourceFeatures />
      <AdditionalSections />
    </div>
  );
}

// Or use the individual card component
import { FeatureCard } from './open-source-features';
import { Shield } from 'lucide-react';

<FeatureCard
  icon={<Shield className="w-6 h-6" />}
  title="Security First"
  description="Open source means security through transparency."
/>
```

The component uses Tailwind CSS with custom ODS (OpenFrame Design System) tokens and includes responsive breakpoints for mobile, tablet, and desktop layouts.
