A client-side React section component that renders a responsive grid of four feature cards highlighting OpenFrame's open-source philosophy.

## Key Components

- **`FeatureCard`** — Reusable card component accepting an `icon`, `title`, and `description` prop. Renders an icon container with a hover state transition and a text block.
- **`FeatureCardProps`** — TypeScript interface defining the card's prop shape (`icon: React.ReactNode`, `title: string`, `description: string`).
- **`OpenSourceFeatures`** — Default export. Defines the four feature entries (`Built on FOSS`, `Own Your Stack`, `Modular by Design`, `Community-Powered`) and renders them in a 1→2→4 column responsive grid.

## Usage Example

```typescript
import OpenSourceFeatures from '@/components/open-source-features';

export default function LandingPage() {
  return (
    <main>
      <OpenSourceFeatures />
    </main>
  );
}
```

To add or modify a feature card, extend the `features` array inside `OpenSourceFeatures`:

```typescript
const features = [
  {
    icon: <Terminal className="w-6 h-6" />,
    title: "Built on FOSS",
    description: "No black boxes. No hidden fees."
  },
  // Add new entries here
];
```

## Notes

- Uses Lucide icons (`Terminal`, `DollarSign`, `Network`, `Users`) sized at `w-6 h-6`.
- Styled with ODS design tokens (`ods-card`, `ods-border`, `ods-accent`, etc.) — requires the ODS Tailwind theme to be configured.
- Marked `"use client"` — Next.js App Router client component.
