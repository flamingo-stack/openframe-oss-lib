A responsive header component that supports auto-hide on scroll, dropdown navigation menus, and configurable branding with mobile menu integration.

## Key Components

- **Header**: Main component accepting `HeaderProps` with configuration and platform
- **HeaderProps**: Interface defining config object and optional platform string
- **HeaderConfig**: Re-exported type for header configuration
- **renderNavigationItem**: Internal function handling different navigation item types (links, dropdowns, custom elements)

## Usage Example

```typescript
import { Header, HeaderConfig } from './header'

const headerConfig: HeaderConfig = {
  logo: {
    href: '/',
    element: <img src="/logo.png" alt="Logo" />
  },
  navigation: {
    items: [
      { id: 'home', label: 'Home', href: '/' },
      { 
        id: 'products', 
        label: 'Products',
        children: [
          { id: 'product1', label: 'Product 1', href: '/product1' }
        ]
      }
    ]
  },
  actions: {
    right: <button>Sign In</button>
  },
  autoHide: true,
  mobile: { 
    enabled: true, 
    onToggle: () => setMobileMenuOpen(!mobileMenuOpen) 
  }
}

<Header config={headerConfig} platform="web" />
```

The component features scroll-based auto-hide functionality, keyboard navigation support, click-outside dropdown closing, and responsive design with mobile menu integration.
