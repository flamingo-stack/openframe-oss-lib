A responsive React header component that provides configurable navigation, dropdown menus, auto-hide functionality, and mobile support for web applications.

## Key Components

- **`Header`** - Main header component with sticky positioning and scroll-based visibility
- **`HeaderProps`** - Interface defining configuration options including navigation, actions, and styling
- **Dropdown Management** - State-driven dropdown system with click-outside and escape key handling
- **Navigation Rendering** - Flexible navigation item renderer supporting links, dropdowns, and custom elements
- **Scroll Behavior** - Optional auto-hide functionality based on scroll direction
- **Mobile Support** - Configurable mobile menu toggle with responsive design

## Usage Example

```typescript
import { Header, HeaderConfig } from './header'
import { Logo } from './logo'
import { UserMenu } from './user-menu'

const headerConfig: HeaderConfig = {
  logo: {
    href: '/',
    element: <Logo />
  },
  navigation: {
    position: 'center',
    items: [
      {
        id: 'docs',
        label: 'Documentation',
        href: '/docs',
        isActive: true
      },
      {
        id: 'community',
        label: 'Community',
        children: [
          { id: 'discord', label: 'Discord', href: '/discord' },
          { id: 'github', label: 'GitHub', href: '/github', isExternal: true }
        ]
      }
    ]
  },
  actions: {
    right: [<UserMenu key="user" />]
  },
  autoHide: true,
  mobile: {
    enabled: true,
    onToggle: () => setMobileMenuOpen(!isMobileMenuOpen)
  }
}

function App() {
  return <Header config={headerConfig} platform="web" />
}
```

The component features automatic dropdown closing on outside clicks, smooth scroll-based hiding, and comprehensive keyboard navigation support.
