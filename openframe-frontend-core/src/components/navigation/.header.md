A responsive header component for React/Next.js applications that provides auto-hide scrolling behavior, dropdown navigation, and mobile menu support with customizable styling.

## Key Components

**Main Export:**
- `Header` - Primary header component with navigation and mobile support
- `HeaderProps` - Interface for component props
- `HeaderConfig` - Re-exported type for header configuration

**Core Features:**
- Auto-hide on scroll (configurable)
- Dropdown navigation menus with click-outside handling
- Mobile responsive design with hamburger menu
- Keyboard navigation (Escape key support)
- Active state management for navigation items

**Internal Functions:**
- `renderNavigationItem()` - Renders different navigation item types (links, dropdowns, custom elements)
- Scroll event handling for auto-hide functionality
- Dropdown state management with refs for DOM manipulation

## Usage Example

```typescript
import { Header, HeaderConfig } from './components/header'
import { Logo } from './components/logo'
import { Button } from './components/ui/button'

const headerConfig: HeaderConfig = {
  logo: {
    href: '/',
    element: <Logo />
  },
  navigation: {
    position: 'center',
    items: [
      {
        id: 'products',
        label: 'Products',
        children: [
          { id: 'openframe', label: 'OpenFrame', href: '/openframe' },
          { id: 'flamingo', label: 'Flamingo MSP', href: '/msp' }
        ]
      },
      {
        id: 'docs',
        label: 'Documentation',
        href: '/docs'
      }
    ]
  },
  actions: {
    right: <Button variant="primary">Get Started</Button>
  },
  autoHide: true,
  mobile: {
    enabled: true,
    onToggle: () => setMobileMenuOpen(!mobileMenuOpen)
  }
}

export default function Layout() {
  return <Header config={headerConfig} platform="openframe" />
}
```
