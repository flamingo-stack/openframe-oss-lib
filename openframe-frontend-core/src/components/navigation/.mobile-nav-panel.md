A responsive mobile navigation panel component that displays a modal overlay with navigation sections, items, and optional footer actions.

## Key Components

- **MobileNavPanel**: Main component that renders a modal navigation panel with backdrop
- **MobileNavPanelProps**: Interface defining `isOpen` boolean and `config` object props
- **renderNavigationItem**: Internal function that handles different navigation item types (links, buttons, custom elements)

Key features include:
- Body scroll prevention when open
- Escape key handling for accessibility
- Responsive positioning and sizing
- Scrollable content area with fixed header/footer
- Support for navigation sections with optional titles

## Usage Example

```typescript
import { MobileNavPanel } from './mobile-nav-panel'
import { MobileNavConfig } from '../types/navigation'

const navConfig: MobileNavConfig = {
  onClose: () => setIsOpen(false),
  sections: [
    {
      title: "Navigation",
      items: [
        { id: "home", label: "Home", href: "/", icon: <HomeIcon /> },
        { id: "profile", label: "Profile", onClick: handleProfile, badge: "2" }
      ]
    }
  ],
  footer: <Button>Sign Out</Button>
}

<MobileNavPanel isOpen={isMenuOpen} config={navConfig} />
```
