A React component that provides a global search input field with a search icon, designed for header navigation areas in the OpenFrame platform.

## Key Components

- **HeaderGlobalSearch**: Main functional component that renders a search form with an input field and search icon
- **HeaderGlobalSearchProps**: TypeScript interface defining component props including value, onChange, onSubmit callbacks, and styling options
- **Internal state management**: Handles both controlled and uncontrolled input modes depending on whether onChange prop is provided

## Usage Example

```typescript
import { HeaderGlobalSearch } from './header-global-search'

// Controlled mode with external state
function App() {
  const [searchQuery, setSearchQuery] = useState('')

  const handleSearch = (query: string) => {
    console.log('Searching for:', query)
    // Perform search logic
  }

  return (
    <HeaderGlobalSearch
      value={searchQuery}
      onChange={setSearchQuery}
      onSubmit={handleSearch}
      placeholder="Search across all modules..."
      className="max-w-md"
    />
  )
}

// Uncontrolled mode with internal state
function SimpleSearch() {
  return (
    <HeaderGlobalSearch
      onSubmit={(query) => performSearch(query)}
      placeholder="Quick search..."
    />
  )
}
```

The component supports both Enter key and form submission for triggering searches, uses OpenFrame design system classes (ods-*), and automatically manages focus states and responsive styling.
