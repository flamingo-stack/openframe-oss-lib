A React component that creates an animated sliding sidebar navigation with collapsible menu items, dynamic header height detection, and smooth animations using Framer Motion.

## Key Components

- **SlidingSidebar** - Main component that renders a slide-in navigation sidebar
- **SlidingSidebarProps** - Interface defining the component's configuration props
- **renderMenuItem** - Internal method that recursively renders navigation items with proper nesting
- **Dynamic header height calculation** - Automatically adjusts sidebar positioning based on header and announcement bar heights
- **Expandable menu items** - Support for collapsible navigation sections with smooth animations

## Usage Example

```typescript
import { SlidingSidebar } from './components/sliding-sidebar'
import { SlidingSidebarConfig } from './types/navigation'

function App() {
  const [sidebarOpen, setSidebarOpen] = useState(false)
  
  const sidebarConfig: SlidingSidebarConfig = {
    isOpen: sidebarOpen,
    onClose: () => setSidebarOpen(false),
    position: 'left',
    items: [
      {
        id: 'dashboard',
        label: 'Dashboard',
        href: '/dashboard',
        icon: <DashboardIcon />,
        isActive: true
      },
      {
        id: 'docs',
        label: 'Documentation',
        children: [
          {
            id: 'getting-started',
            label: 'Getting Started',
            href: '/docs/getting-started'
          }
        ]
      }
    ],
    footer: <div>Footer content</div>
  }

  return (
    <SlidingSidebar config={sidebarConfig} />
  )
}
```

The component automatically handles header height detection, provides smooth slide animations, supports nested navigation items with expand/collapse functionality, and includes an overlay backdrop for mobile-friendly interaction.
