A sliding sidebar component that provides animated navigation with collapsible menu items and dynamic header height calculation. It supports both left and right positioning with smooth animations powered by Framer Motion.

## Key Components

- **SlidingSidebar**: Main component that renders an animated sidebar overlay
- **SlidingSidebarProps**: Interface defining component props with navigation config
- **Dynamic Header Height**: Automatically calculates header and announcement bar heights
- **Menu Item Rendering**: Recursive function handling nested navigation items with expand/collapse
- **Animation Support**: Smooth slide transitions and collapsible item animations

## Usage Example

```typescript
import { SlidingSidebar } from './sliding-sidebar'

const sidebarConfig = {
  isOpen: true,
  position: 'left',
  onClose: () => setSidebarOpen(false),
  items: [
    {
      id: 'home',
      label: 'Home',
      href: '/',
      icon: <HomeIcon />,
      isActive: true
    },
    {
      id: 'products',
      label: 'Products',
      children: [
        { id: 'web', label: 'Web Apps', href: '/web' },
        { id: 'mobile', label: 'Mobile', href: '/mobile' }
      ]
    }
  ],
  footer: <div>Footer content</div>
}

<SlidingSidebar config={sidebarConfig} />
```
