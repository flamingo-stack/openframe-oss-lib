A responsive navigation sidebar component that adapts to different screen sizes with collapsible functionality and mobile overlay support.

## Key Components

- **NavigationSidebar**: Main component that renders a collapsible sidebar with navigation items
- **NavigationSidebarProps**: Interface defining the configuration prop
- **renderNavigationItem**: Renders individual navigation items with active states and badges
- **renderSidebarContent**: Renders the complete sidebar content structure
- **Constants**: `MINIMIZED_WIDTH` (56px) and `EXPANDED_WIDTH` (224px) for responsive sizing

## Usage Example

```typescript
import { NavigationSidebar } from './navigation-sidebar'
import { HomeIcon, SettingsIcon } from '../icons'

const sidebarConfig = {
  items: [
    {
      id: 'home',
      label: 'Home',
      icon: <HomeIcon />,
      path: '/',
      isActive: true
    },
    {
      id: 'settings',
      label: 'Settings',
      icon: <SettingsIcon />,
      path: '/settings',
      section: 'secondary',
      badge: '2'
    }
  ],
  onNavigate: (path: string) => router.push(path),
  onToggleMinimized: () => console.log('Sidebar toggled')
}

function App() {
  return (
    <NavigationSidebar config={sidebarConfig} />
  )
}
```

The component automatically handles minimization state persistence via localStorage, responsive behavior across screen sizes, and provides accessibility features including ARIA labels and keyboard navigation support.
