A responsive navigation sidebar component that provides collapsible menu functionality with mobile overlay support and smooth animations.

## Key Components

- **NavigationSidebar**: Main component that renders a collapsible sidebar with logo, navigation items, and toggle controls
- **NavigationSidebarProps**: Interface defining the required configuration object
- **renderNavigationItem**: Internal function that renders individual navigation items with active states and badges
- **sidebarContent**: Internal function that generates the complete sidebar layout for both desktop and mobile views

## Usage Example

```typescript
import { NavigationSidebar } from './navigation-sidebar'
import { HomeIcon, SettingsIcon } from './icons'

const config = {
  minimized: false,
  items: [
    {
      id: 'home',
      label: 'Home',
      icon: <HomeIcon />,
      path: '/home',
      isActive: true
    },
    {
      id: 'settings',
      label: 'Settings',
      icon: <SettingsIcon />,
      section: 'secondary',
      onClick: () => console.log('Settings clicked'),
      badge: '2'
    }
  ],
  onNavigate: (path) => router.push(path),
  onToggleMinimized: () => console.log('Sidebar toggled'),
  footer: <div>Custom footer content</div>
}

function App() {
  return (
    <div className="flex h-screen">
      <NavigationSidebar config={config} />
      <main className="flex-1">
        {/* Main content */}
      </main>
    </div>
  )
}
```

The component automatically handles mobile responsiveness, state persistence via localStorage, and provides smooth transitions between minimized and expanded states.
