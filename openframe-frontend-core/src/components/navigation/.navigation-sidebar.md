A responsive navigation sidebar component that supports desktop and mobile views with collapsible functionality and persistent state management.

## Key Components

### Main Export
- **NavigationSidebar** - Primary sidebar component with responsive behavior
- **NavigationSidebarProps** - TypeScript interface defining component props

### Key Features
- **Responsive Design** - Automatically adapts between desktop and mobile layouts
- **Collapsible State** - Desktop sidebar can be minimized/expanded with localStorage persistence
- **Mobile Overlay** - Full-screen overlay menu for mobile devices
- **Active State Management** - Visual indicators for active navigation items
- **Smooth Animations** - Text fade transitions and hover effects

### State Management
- **useLocalStorage** - Persists minimized state across sessions
- **Mobile Detection** - Responsive breakpoint handling at 768px
- **Animation Timing** - Coordinated text visibility delays to prevent flickering

## Usage Example

```typescript
import { NavigationSidebar } from './components/navigation-sidebar'
import { HomeIcon, SettingsIcon } from './icons'

const sidebarConfig = {
  minimized: false,
  items: [
    {
      id: 'home',
      label: 'Home',
      icon: <HomeIcon />,
      path: '/home',
      isActive: true
    },
    {
      id: 'settings',
      label: 'Settings',
      icon: <SettingsIcon />,
      section: 'secondary',
      onClick: () => console.log('Settings clicked')
    }
  ],
  onNavigate: (path: string) => router.push(path),
  onToggleMinimized: () => console.log('Sidebar toggled'),
  footer: <div>Custom footer content</div>
}

function App() {
  return (
    <div className="flex h-screen">
      <NavigationSidebar config={sidebarConfig} />
      <main className="flex-1">
        {/* Main content */}
      </main>
    </div>
  )
}
```
