A responsive application layout component that provides a structured shell with navigation sidebar, header, and mobile menu functionality for Flamingo Stack applications.

## Key Components

- **`AppLayout`** - Main layout component that orchestrates the entire application shell
- **`AppLayoutProps`** - TypeScript interface defining all layout configuration options
- **Mobile menu state management** - Handles toggle and close functionality for mobile navigation
- **Responsive design** - Combines desktop sidebar with mobile burger menu
- **Suspense integration** - Built-in loading state handling for async content

## Usage Example

```typescript
import { AppLayout } from './app-layout'
import { NavigationSidebarConfig } from '../../types/navigation'

const sidebarConfig: NavigationSidebarConfig = {
  sections: [
    {
      title: 'Dashboard',
      items: [
        { label: 'Overview', href: '/dashboard', icon: 'dashboard' }
      ]
    }
  ]
}

function MyApp({ children }: { children: React.ReactNode }) {
  return (
    <AppLayout
      sidebarConfig={sidebarConfig}
      headerProps={{
        title: 'Flamingo MSP',
        user: { name: 'John Doe', avatar: '/avatar.jpg' }
      }}
      mobileBurgerMenuProps={{
        className: 'custom-mobile-menu'
      }}
      loadingFallback={<div>Loading...</div>}
      mainClassName="custom-main-styles"
    >
      {children}
    </AppLayout>
  )
}
```

The component uses Tailwind CSS classes for styling and implements a flex-based layout with overflow handling for proper scrolling behavior on both desktop and mobile devices.
