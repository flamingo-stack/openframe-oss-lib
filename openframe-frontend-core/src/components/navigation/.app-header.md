A customizable application header component that provides search functionality, organization filtering, notifications, theme switching, and user menu capabilities.

## Key Components

- **`AppHeader`** - Main header component with configurable features
- **`AppHeaderProps`** - Interface defining all available configuration options including search, organizations, notifications, theme switcher, and user settings

## Usage Example

```typescript
import { AppHeader } from './components/app-header'

function App() {
  const organizations = [
    { id: '1', name: 'Acme Corp' },
    { id: '2', name: 'Tech Solutions' }
  ]

  return (
    <AppHeader
      showSearch
      onSearch={(query) => console.log('Searching:', query)}
      showOrganizations
      organizations={organizations}
      selectedOrgId="1"
      onOrgChange={(id) => setSelectedOrg(id)}
      showNotifications
      unreadCount={5}
      showThemeSwitcher
      showUser
      userName="John Doe"
      userEmail="john@example.com"
      userAvatarUrl="/avatar.jpg"
      onProfile={() => navigate('/profile')}
      onLogout={() => signOut()}
    />
  )
}
```

The component is highly modular - each feature can be enabled/disabled independently using boolean props, making it suitable for different application contexts and user permission levels.
