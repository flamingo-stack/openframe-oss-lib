A flexible, customizable application header component built with React that provides common header functionality including search, organization filtering, notifications, theme switching, and user management.

## Key Components

### Main Export
- **`AppHeader`** - Main header component with configurable features
- **`AppHeaderProps`** - TypeScript interface defining all configuration options

### Core Features
- **Global Search** - Optional search input with form submission handling
- **Organization Filter** - Dropdown for switching between organizations
- **Notifications** - Bell icon with unread count badge
- **Theme Switcher** - Toggle button for theme management
- **User Menu** - User avatar, profile, and logout functionality

## Usage Example

```typescript
import { AppHeader } from './components/app-header'

function App() {
  const [searchQuery, setSearchQuery] = useState('')
  const [selectedOrg, setSelectedOrg] = useState('')
  
  return (
    <AppHeader
      showSearch={true}
      onSearch={setSearchQuery}
      showOrganizations={true}
      organizations={[
        { id: '1', name: 'Acme Corp' },
        { id: '2', name: 'Tech Startup' }
      ]}
      selectedOrgId={selectedOrg}
      onOrgChange={setSelectedOrg}
      showNotifications={true}
      unreadCount={5}
      showThemeSwitcher={true}
      showUser={true}
      userName="John Doe"
      userEmail="john@example.com"
      userAvatarUrl="/avatar.jpg"
      onProfile={() => navigate('/profile')}
      onLogout={() => auth.signOut()}
    />
  )
}
```

The component uses a design system with `ods-*` CSS classes and includes responsive behavior, accessibility features, and clean separation of concerns through optional props.
