Responsive application header component for the OpenFrame platform, rendering navigation controls, search, organization filtering, notifications, user menu, and the Mingo AI launcher across desktop and mobile breakpoints.

## Key Components

### `AppHeader` (default export)
Memoized header component that adapts its layout based on viewport size via `useMdUp`. Renders a sticky top bar with configurable slots.

### `AppHeaderProps`
Full interface controlling all header features:

| Prop | Description |
|---|---|
| `showSearch` | Toggles global search (full bar on desktop, icon on mobile) |
| `showOrganizations` | Shows org filter dropdown (desktop only) |
| `showNotifications` | Shows bell icon with unread badge |
| `showMingoAI` | Enables the Mingo AI drawer launcher |
| `isMingoAIActive` | Visual pressed state for Mingo button |
| `isMobileMenuOpen` | Controls burger/X icon toggle |
| `disabled` | Dims and disables all controls except the mobile burger |

### `NotificationsHeaderButton` (internal)
Connects to `useOptionalNotifications` context for live unread count and panel toggle; falls back to `fallbackUnreadCount` prop when the context is unavailable.

## Usage Example

```typescript
import { AppHeader } from './app-header'

export function Layout() {
  const [mobileOpen, setMobileOpen] = React.useState(false)
  const [mingoOpen, setMingoOpen] = React.useState(false)

  return (
    <AppHeader
      isMobileMenuOpen={mobileOpen}
      onToggleMobileMenu={() => setMobileOpen(v => !v)}
      showSearch
      onSearch={(q) => console.log('search:', q)}
      showOrganizations
      organizations={[{ id: '1', name: 'Acme Corp' }]}
      selectedOrgId="1"
      onOrgChange={(id) => console.log('org:', id)}
      showNotifications
      unreadCount={3}
      showMingoAI
      onMingoAI={() => setMingoOpen(v => !v)}
      isMingoAIActive={mingoOpen}
      showUser
      userName="Jane Smith"
      userEmail="jane@acme.com"
      onLogout={() => signOut()}
    />
  )
}
```
