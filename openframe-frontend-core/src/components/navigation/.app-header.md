Responsive application header component for the OpenFrame platform, rendering navigation controls, search, organization filtering, notifications, user menu, and the Mingo AI launcher button.

## Key Components

### `AppHeader` (default export)
A memoized React component that composes all header sections with responsive behavior — mobile shows a burger menu and logo, desktop shows the full control bar.

### `AppHeaderProps`
Full interface for configuring header features. Key props:

| Prop | Type | Description |
|------|------|-------------|
| `showSearch` | `boolean` | Enables global search (desktop full-bar, mobile icon) |
| `showOrganizations` | `boolean` | Renders org filter (desktop `lg+` only) |
| `showNotifications` | `boolean` | Bell icon with unread dot indicator |
| `showMingoAI` | `boolean` | Mingo AI drawer launcher button |
| `isMingoAIActive` | `boolean` | Visual pressed state for Mingo button |
| `isMobileMenuOpen` | `boolean` | Controls burger/X icon toggle |
| `disabled` | `boolean` | Dims all controls except the mobile burger |

### `NotificationsHeaderButton` (internal)
Connects to `NotificationsContext` via `useOptionalNotifications()`, falling back to the `fallbackUnreadCount` prop when no context provider is present.

## Usage Example

```typescript
<AppHeader
  showSearch
  onSearch={(query) => console.log(query)}
  showOrganizations
  organizations={[{ id: 'org-1', name: 'Acme Corp' }]}
  selectedOrgId="org-1"
  onOrgChange={(id) => setSelectedOrg(id)}
  showNotifications
  unreadCount={3}
  showMingoAI
  onMingoAI={() => setDrawerOpen(true)}
  isMingoAIActive={isDrawerOpen}
  showUser
  userName="Jane Doe"
  userEmail="jane@acme.com"
  onProfile={() => router.push('/profile')}
  onLogout={handleLogout}
  isMobileMenuOpen={mobileMenuOpen}
  onToggleMobileMenu={() => setMobileMenuOpen(v => !v)}
/>
```

## Behavior Notes

- Uses `useMdUp()` to switch between mobile (`h-12`) and desktop (`h-14`) layouts
- The `disabled` prop sets `pointer-events-none opacity-50` on all controls **except** the mobile burger toggle
- Mingo AI button renders icon-only on mobile, full label on desktop (`md+`)
- User dropdown is desktop-only (`md+`); displays avatar, name, email, profile link, and logout
