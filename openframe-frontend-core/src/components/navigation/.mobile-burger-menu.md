A mobile burger menu component that provides a full-screen overlay navigation interface for mobile devices, featuring user profile display, grid-based navigation items, and smooth slide animations.

## Key Components

- **MobileBurgerMenu** - Main component that renders a slide-down mobile menu with backdrop blur
- **MobileBurgerMenuProps** - Interface defining menu configuration, user data, and callback functions
- **Navigation Grid System** - Renders primary navigation items in a 2-column grid layout
- **User Profile Card** - Displays user avatar, name, email, role badge, and action buttons
- **Keyboard & Scroll Management** - Handles ESC key closing and prevents body scroll when open

## Usage Example

```typescript
import { MobileBurgerMenu } from './mobile-burger-menu'
import { HomeIcon, SettingsIcon } from '../icons'

function MobileNav() {
  const [isMenuOpen, setIsMenuOpen] = useState(false)
  
  const navigationConfig = {
    items: [
      {
        id: 'home',
        label: 'Dashboard',
        icon: <HomeIcon />,
        path: '/dashboard',
        isActive: true
      },
      {
        id: 'settings',
        label: 'Settings',
        icon: <SettingsIcon />,
        path: '/settings',
        section: 'secondary'
      }
    ],
    onNavigate: (path: string) => router.push(path)
  }

  return (
    <MobileBurgerMenu
      isOpen={isMenuOpen}
      onClose={() => setIsMenuOpen(false)}
      config={navigationConfig}
      user={{
        userName: "John Doe",
        userEmail: "john@example.com",
        userRole: "admin"
      }}
      onLogout={() => signOut()}
      onEditProfile={() => router.push('/profile')}
    />
  )
}
```

The component automatically positions itself below the header (48px), provides smooth slide-down animations, and includes accessibility features like keyboard navigation and ARIA labels.
