A React component that provides a dropdown filter for switching between organizations in a header layout, displaying the selected organization name and device count.

## Key Components

- **`HeaderOrganizationFilterOrganization`** - Interface defining organization structure with id, name, and optional device count
- **`HeaderOrganizationFilterProps`** - Props interface supporting organization list, selection state, change callbacks, and styling
- **`HeaderOrganizationFilter`** - Main component rendering a dropdown trigger button with organization selection menu

## Usage Example

```typescript
import { HeaderOrganizationFilter } from './header-organization-filter'

const organizations = [
  { id: 'org1', name: 'Acme Corp', deviceCount: 150 },
  { id: 'org2', name: 'TechStart Inc', deviceCount: 75 }
]

function Header() {
  const [selectedOrgId, setSelectedOrgId] = useState('')
  
  return (
    <HeaderOrganizationFilter
      organizations={organizations}
      selectedOrgId={selectedOrgId}
      onOrgChange={setSelectedOrgId}
      totalDeviceCount={225}
      className="border-r"
    />
  )
}
```

The component displays "All Organizations" when no specific organization is selected, shows individual organization names when selected, and includes device counts below the organization name. It uses a fixed width design (240px) with hover effects and proper text truncation for long organization names.
