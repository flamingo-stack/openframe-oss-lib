This module serves as the main export hub for all navigation-related components and types in the application, providing a centralized interface for importing header, sidebar, and mobile navigation functionality.

## Key Components

**Components:**
- `Header` - Main application header component
- `ClientOnlyHeader` - Header that renders only on client-side
- `HeaderSkeleton` - Loading placeholder for header
- `MobileNavPanel` - Mobile-specific navigation panel
- `SlidingSidebar` - Animated sidebar that slides in/out
- `StickySectionNav` - Section navigation that sticks to viewport
- `NavigationSidebar` - Main navigation sidebar component  
- `AppHeader` - Application-level header wrapper

**Hooks:**
- `useSectionNavigation` - Hook for managing section navigation state

**Types:**
- Component props types for all exported components
- Core navigation types (`NavigationItem`, `MobileNavConfig`, etc.)

## Usage Example

```typescript
import { 
  Header, 
  MobileNavPanel, 
  NavigationSidebar,
  type HeaderProps,
  type NavigationItem 
} from './components/navigation'

const navItems: NavigationItem[] = [
  { label: 'Home', href: '/' },
  { label: 'About', href: '/about' }
]

function App() {
  return (
    <>
      <Header title="My App" navigation={navItems} />
      <NavigationSidebar items={navItems} />
      <MobileNavPanel items={navItems} />
    </>
  )
}
```
