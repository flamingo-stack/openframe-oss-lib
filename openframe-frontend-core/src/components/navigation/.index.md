Main entry point for the navigation component library, providing a clean public API with comprehensive exports for headers, sidebars, mobile navigation, and layout components along with their TypeScript type definitions.

## Key Components

**Core Navigation Components:**
- `Header` - Main header component with configuration support
- `AppHeader` - Application-level header implementation  
- `AppLayout` - Full page layout wrapper with navigation
- `NavigationSidebar` - Primary sidebar navigation component

**Mobile Navigation:**
- `MobileNavPanel` - Mobile-optimized navigation panel
- `MobileBurgerMenu` - Hamburger menu for mobile devices
- `ClientOnlyHeader` - Header that only renders on client-side

**Interactive Elements:**
- `SlidingSidebar` - Animated sidebar with slide transitions
- `StickySectionNav` - Section navigation with scroll tracking
- `HeaderGlobalSearch` - Global search functionality in header
- `HeaderOrganizationFilter` - Organization filtering component

**Utility Components:**
- `HeaderSkeleton` - Loading placeholder for header
- `HeaderButton` - Styled button component for header actions
