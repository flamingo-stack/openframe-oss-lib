Barrel export file for the navigation component library, re-exporting all navigation-related components, hooks, and TypeScript types from their respective modules.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Header` / `AppHeader` | Component | Primary and app-level header components |
| `ClientOnlyHeader` | Component | Header rendered exclusively on the client side |
| `HeaderSkeleton` | Component | Loading placeholder for the header |
| `AppLayout` | Component | Root layout wrapper with drawer support |
| `AppLayoutDrawer` + primitives | Components | Composable drawer system (trigger, close, content, header, body, footer) |
| `SlidingSidebar` | Component | Animated sliding sidebar panel |
| `NavigationSidebar` | Component | Full navigation sidebar |
| `MobileNavPanel` | Component | Mobile navigation overlay panel |
| `MobileBurgerMenu` | Component | Hamburger menu toggle for mobile |
| `StickySectionNav` | Component | Sticky in-page section navigation |
| `useSectionNavigation` | Hook | Controls active section tracking |
| `useAppLayoutDrawerContainer` | Hook | Accesses the drawer container context |
| `HeaderButton` / `HeaderMingoButton` | Components | Header action buttons (generic and Mingo AI-specific) |
| `HeaderGlobalSearch` | Component | Global search trigger in the header |
| `HeaderOrganizationFilter` | Component | Organization switcher/filter in the header |

## Usage Example

```typescript
import {
  AppLayout,
  AppHeader,
  NavigationSidebar,
  AppLayoutDrawer,
  AppLayoutDrawerTrigger,
  AppLayoutDrawerContent,
  useAppLayoutDrawerContainer,
} from '@/components/navigation'

export function Shell() {
  return (
    <AppLayout>
      <AppHeader>
        <HeaderGlobalSearch />
        <HeaderOrganizationFilter />
        <HeaderMingoButton />
      </AppHeader>
      <NavigationSidebar />
      <AppLayoutDrawer>
        <AppLayoutDrawerTrigger>Open</AppLayoutDrawerTrigger>
        <AppLayoutDrawerContent>
          <AppLayoutDrawerHeader>
            <AppLayoutDrawerTitle>Details</AppLayoutDrawerTitle>
          </AppLayoutDrawerHeader>
          <AppLayoutDrawerBody>...</AppLayoutDrawerBody>
        </AppLayoutDrawerContent>
      </AppLayoutDrawer>
    </AppLayout>
  )
}
```
