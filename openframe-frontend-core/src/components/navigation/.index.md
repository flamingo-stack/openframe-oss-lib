Main barrel export file that provides a centralized interface for all navigation components and their TypeScript types used throughout the application.

## Key Components

**Navigation Components:**
- `Header` - Main header component with configuration support
- `ClientOnlyHeader` - Client-side rendered header variant
- `HeaderSkeleton` - Loading placeholder for header
- `MobileNavPanel` - Mobile navigation interface
- `SlidingSidebar` - Collapsible sidebar navigation
- `StickySectionNav` - Sticky section navigation with scroll tracking
- `NavigationSidebar` - Main sidebar navigation component
- `AppHeader` - Application-level header wrapper

**Navigation Utilities:**
- `useSectionNavigation` - Hook for managing section navigation state

**Type Exports:**
- Component prop types for all navigation components
- Shared navigation types (`NavigationItem`, `MobileNavConfig`, etc.)

## Usage Example

```typescript
import { 
  Header, 
  MobileNavPanel, 
  NavigationSidebar,
  useSectionNavigation
} from '@/components/navigation'

import type { 
  HeaderProps, 
  NavigationItem,
  MobileNavConfig 
} from '@/components/navigation'

// Use navigation components with proper typing
const navItems: NavigationItem[] = [
  { label: 'Home', href: '/' },
  { label: 'About', href: '/about' }
]

const config: MobileNavConfig = {
  items: navItems,
  showSearch: true
}

function App() {
  const { activeSection } = useSectionNavigation()
  
  return (
    <>
      <Header items={navItems} />
      <NavigationSidebar config={config} />
      <MobileNavPanel isOpen={false} items={navItems} />
    </>
  )
}
```
