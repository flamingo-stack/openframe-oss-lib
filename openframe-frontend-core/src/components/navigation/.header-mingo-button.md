Accessible launcher button for the **Mingo AI** drawer in `AppHeader`, rendering the Mingo logo + wordmark at rest and swapping to a close icon when the drawer is active on narrow viewports.

## Key Components

- **`HeaderMingoButton`** — Primary export. A styled `<button>` that integrates with `AppHeader`'s `divide-x` divider system and mirrors the `HeaderButton` visual contract.
- **`HeaderMingoButtonProps`** — Extends `React.ButtonHTMLAttributes<HTMLButtonElement>` with three custom props:
  - `isActive` — Reflects open/closed drawer state via `aria-pressed`
  - `iconOnly` — Collapses the button to icon-only (hides "Mingo AI" wordmark); on active state, swaps the Mingo logo for an `XmarkIcon`
  - `className` — Allows style overrides

## Usage Example

```typescript
// Default — full label visible
<HeaderMingoButton
  isActive={isMingoOpen}
  onClick={() => setIsMingoOpen(prev => !prev)}
/>

// Narrow viewport / mobile — icon only
<HeaderMingoButton
  isActive={isMingoOpen}
  iconOnly
  onClick={() => setIsMingoOpen(prev => !prev)}
/>
```

## Behavior Notes

- **Accessibility:** Sets `aria-pressed` for drawer state and updates `aria-label` to `"Close Mingo AI"` when `iconOnly && isActive`
- **Icon swap:** `iconOnly + isActive` renders `XmarkIcon`; all other states render `MingoIcon` with ODS cyan accent colors
- **Styling:** Uses ODS tokens (`ods-card`, `ods-bg-hover`, `ods-text-primary`) and `focus-visible:ring-2` for keyboard navigation — no active background change
- **Figma reference:** `7532:222103 — button-full`
