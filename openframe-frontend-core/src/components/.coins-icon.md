A React SVG icon component that renders a coins symbol with two overlapping circles and decorative elements.

## Key Components

- **CoinsIcon**: Main React functional component that renders an SVG icon
- **CoinsIconProps**: TypeScript interface defining optional props for customization
  - `className`: CSS class for styling
  - `width`: Icon width (default: 20px)
  - `height`: Icon height (default: 20px)

## Usage Example

```typescript
import { CoinsIcon } from './coins-icon';

// Basic usage with defaults
<CoinsIcon />

// Custom size and styling
<CoinsIcon 
  width={24} 
  height={24} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a button or container
<button className="flex items-center gap-2">
  <CoinsIcon width={16} height={16} />
  Balance
</button>
```

The icon uses `currentColor` for the stroke, making it inherit the text color of its parent element for easy theming.
