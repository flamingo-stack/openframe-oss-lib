A React hook and utility module that provides unified filtering logic using AND operations for Next.js applications, managing URL-based filter state with smooth transitions.

## Key Components

- **`useUnifiedFiltering`**: Main hook for managing filter state and URL synchronization
- **`FilterState`**: Interface defining all possible filter parameters (search, categories, tags, etc.)
- **`FilterConfig`**: Configuration interface specifying supported filters and base path
- **Filter operations**: `addFilter`, `removeFilter`, `toggleFilter`, `clearAllFilters`
- **`getFilterChips`**: Generates displayable filter chips with removal handlers
- **Predefined configs**: `vendorFilterConfig` and `blogFilterConfig`
