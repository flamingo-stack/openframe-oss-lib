A React hook that detects whether the current viewport is in mobile mode based on a 768px breakpoint, providing responsive behavior for client-side components.

## Key Components

- **`useIsMobile()`** - Main hook that returns a boolean indicating mobile viewport status
- **`MOBILE_BREAKPOINT`** - Constant set to 768px defining the mobile/desktop threshold
- **Media query listener** - Automatically updates state when viewport size changes

## Usage Example

```typescript
import { useIsMobile } from "./use-mobile"

function ResponsiveComponent() {
  const isMobile = useIsMobile()

  return (
    <div>
      {isMobile ? (
        <MobileNavigation />
      ) : (
        <DesktopNavigation />
      )}
      <main className={isMobile ? "mobile-layout" : "desktop-layout"}>
        Content here
      </main>
    </div>
  )
}

// Conditional rendering based on screen size
function App() {
  const isMobile = useIsMobile()
  
  return (
    <div className={`app ${isMobile ? 'mobile' : 'desktop'}`}>
      {isMobile && <MobileHeader />}
      <Content />
    </div>
  )
}
```

The hook handles SSR by initially returning `false` (via `!!undefined`), then updating once the component hydrates on the client side.
