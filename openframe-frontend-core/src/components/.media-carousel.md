A responsive, accessible media carousel component supporting images, videos, and YouTube embeds with thumbnail navigation, touch/swipe gestures, and keyboard controls.

## Key Components

### `MediaCarousel` (default export)
Memoized carousel component accepting a `MediaItem[]` array and rendering a main display area with optional thumbnail strip.

**Props (`MediaCarouselProps`):**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `media` | `MediaItem[]` | — | Array of media items to display |
| `className` | `string` | — | Additional CSS classes |
| `aspectRatio` | `'16/9' \| '4/3' \| '3/2' \| '1/1'` | `'16/9'` | Container aspect ratio |
| `showThumbnails` | `boolean` | `true` | Toggle thumbnail strip |
| `autoPlay` | `boolean` | `false` | Auto-advance slides |
| `objectFit` | `'contain' \| 'cover'` | `'contain'` | Image fit behavior |

### Internal Renderers
- **`renderImage`** — Renders `<img>` with lazy loading and error suppression
- **`renderVideo`** — Delegates to `<Video>` (MuxPlayer/HLS/MP4 via SSoT)
- **`renderYouTubeEmbed`** — Delegates to `<Video kind="youtube">` (lite-youtube facade)
- **`renderThumbnail`** — Thumbnail button with active state, video play overlay, and YouTube `mqdefault` poster

## Usage Example

```typescript
import { MediaCarousel } from './media-carousel';
import { MediaItem } from '../utils/media-carousel-utils-stub';

const media: MediaItem[] = [
  { type: 'image', src: '/hero.jpg', alt: 'Hero image' },
  { type: 'youtube', src: 'https://youtube.com/watch?v=dQw4w9WgXcQ', alt: 'Demo video' },
  { type: 'video', src: '/demo.mp4', poster: '/demo-poster.jpg' },
];

<MediaCarousel
  media={media}
  aspectRatio="16/9"
  showThumbnails={true}
  objectFit="contain"
/>
```

## Behavior Notes

- **Index clamping** — Automatically adjusts `currentIndex` when items are removed at runtime, keeping thumbnail active state in sync
- **Keyboard nav** — Arrow keys work when the carousel container is focused (multi-item only)
- **Touch/swipe** — 50px minimum swipe threshold triggers prev/next navigation
- **Accessibility** — `role="region"` + `aria-label` applied only for multi-item carousels
