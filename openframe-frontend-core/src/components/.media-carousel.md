A memoized React carousel component for displaying mixed media (images, videos, YouTube embeds) with thumbnail navigation, swipe/keyboard support, and configurable aspect ratios.

## Key Components

**`MediaCarousel`** — Primary exported component (memoized)
- Accepts a `MediaItem[]` array supporting `'image'`, `'video'`, and `'youtube'` types
- Delegates video/YouTube rendering to the shared `<Video>` component (MuxPlayer/lite-youtube facade)
- Auto-clamps `currentIndex` when the media array shrinks to prevent stale selection state

**Internal Renderers**
- `renderImage` — Lazy-loaded `<img>` with configurable `objectFit`
- `renderVideo` — MuxPlayer via `<Video>` (handles HLS + MP4)
- `renderYouTubeEmbed` — lite-youtube facade via `<Video>`
- `renderThumbnail` — Clickable thumbnail buttons; auto-generates YouTube poster URLs via `extractYouTubeId`

**Navigation**
- Arrow buttons (hover-revealed), keyboard (`ArrowLeft`/`ArrowRight`), and touch swipe (50px threshold)
- `ChevronLeftIcon` / `ChevronRightIcon` — inline SVG nav icons

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `media` | `MediaItem[]` | — | Array of media items to display |
| `aspectRatio` | `'16/9' \| '4/3' \| '3/2' \| '1/1'` | `'16/9'` | Container aspect ratio |
| `showThumbnails` | `boolean` | `true` | Show thumbnail strip |
| `autoPlay` | `boolean` | `false` | Reserved for future use |
| `objectFit` | `'contain' \| 'cover'` | `'contain'` | Image fit mode |

## Usage Example

```typescript
import { MediaCarousel } from './media-carousel';

const items = [
  { type: 'image', src: '/hero.png', alt: 'Hero shot' },
  { type: 'youtube', src: 'https://www.youtube.com/watch?v=dQw4w9WgXcQ', alt: 'Demo' },
  { type: 'video', src: '/demo.mp4', poster: '/demo-thumb.jpg', alt: 'Walkthrough' },
];

export function ProductGallery() {
  return (
    <MediaCarousel
      media={items}
      aspectRatio="16/9"
      objectFit="cover"
      showThumbnails={true}
    />
  );
}
```
