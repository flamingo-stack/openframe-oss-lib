A React component that provides a responsive, touch-enabled carousel for displaying mixed media content including images, videos, and YouTube embeds with thumbnail navigation and keyboard controls.

## Key Components

- **MediaCarousel**: Main carousel component with navigation, touch gestures, and keyboard support
- **CarouselYouTubeEmbed**: YouTube embed component with click-to-play functionality
- **MediaCarouselProps**: TypeScript interface defining component configuration options
- **Navigation Icons**: Built-in SVG icons for carousel controls (ChevronLeft, ChevronRight, Play)
- **getYouTubeVideoId**: Utility function to extract YouTube video IDs from URLs

## Usage Example

```typescript
import { MediaCarousel } from './media-carousel';

const mediaItems = [
  { type: 'image', src: '/photo.jpg', alt: 'Photo' },
  { type: 'video', src: '/video.mp4', poster: '/thumb.jpg' },
  { type: 'youtube', src: 'https://youtube.com/watch?v=VIDEO_ID' }
];

function Gallery() {
  return (
    <MediaCarousel
      media={mediaItems}
      aspectRatio="16/9"
      showThumbnails={true}
      objectFit="cover"
      autoPlay={false}
      className="max-w-4xl mx-auto"
    />
  );
}
```

Features include swipe gestures on mobile, arrow key navigation, automatic video pausing when switching slides, and fallback error handling for unsupported media formats.
