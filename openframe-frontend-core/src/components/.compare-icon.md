A React functional component that renders an SVG icon representing a comparison or balance scale symbol, commonly used for compare features or weighing options.

## Key Components

- **CompareIcon**: Main React functional component that renders the SVG icon
- **CompareIconProps**: TypeScript interface defining optional props for customization
  - `className`: CSS class names for styling
  - `width`: Icon width in pixels (default: 24)
  - `height`: Icon height in pixels (default: 24)

## Usage Example

```typescript
import { CompareIcon } from './compare-icon';

// Basic usage with defaults
<CompareIcon />

// Customized size and styling
<CompareIcon 
  width={32} 
  height={32} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a button component
<button className="flex items-center gap-2">
  <CompareIcon width={16} height={16} />
  Compare Products
</button>
```

The icon uses `currentColor` for the fill property, making it inherit the text color of its parent element for easy theming. The SVG depicts a balance scale with a central post and two weighing pans, making it ideal for comparison features in e-commerce sites, product listings, or decision-making interfaces.
