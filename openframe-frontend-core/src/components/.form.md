A comprehensive form component library built on top of react-hook-form and Radix UI primitives, providing a complete set of accessible form building blocks with automatic error handling and validation states.

## Key Components

- **Form** - Main form provider (alias for `FormProvider`)
- **FormField** - Wrapper for controlled form fields with context
- **FormItem** - Container component that provides unique IDs for form elements
- **FormLabel** - Accessible label that automatically handles error states
- **FormControl** - Wrapper for form inputs with proper ARIA attributes
- **FormDescription** - Helper text component for form fields
- **FormMessage** - Error/validation message display component
- **useFormField** - Hook to access field state and IDs within form components

## Usage Example

```typescript
import { useForm } from "react-hook-form"
import { Form, FormField, FormItem, FormLabel, FormControl, FormDescription, FormMessage } from "./form"
import { Input } from "./ui/input"

function MyForm() {
  const form = useForm()
  
  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit(onSubmit)}>
        <FormField
          control={form.control}
          name="email"
          render={({ field }) => (
            <FormItem>
              <FormLabel>Email</FormLabel>
              <FormControl>
                <Input placeholder="Enter your email" {...field} />
              </FormControl>
              <FormDescription>We'll never share your email.</FormDescription>
              <FormMessage />
            </FormItem>
          )}
        />
      </form>
    </Form>
  )
}
```
