A React radio group component built on Radix UI primitives that provides accessible radio button functionality with custom styling and proper keyboard navigation.

## Key Components

- **`RadioGroup`** - Main container component that wraps radio group items with grid layout
- **`RadioGroupItem`** - Individual radio button with circular design and indicator
- Both components use `forwardRef` for proper ref handling and extend Radix UI primitives

## Usage Example

```typescript
import { RadioGroup, RadioGroupItem } from "./components/ui/radio-group"
import { Label } from "./components/ui/label"

function SettingsForm() {
  const [theme, setTheme] = React.useState("light")
  
  return (
    <div className="space-y-4">
      <Label>Theme Preference</Label>
      <RadioGroup value={theme} onValueChange={setTheme}>
        <div className="flex items-center space-x-2">
          <RadioGroupItem value="light" id="light" />
          <Label htmlFor="light">Light</Label>
        </div>
        <div className="flex items-center space-x-2">
          <RadioGroupItem value="dark" id="dark" />
          <Label htmlFor="dark">Dark</Label>
        </div>
        <div className="flex items-center space-x-2">
          <RadioGroupItem value="system" id="system" />
          <Label htmlFor="system">System</Label>
        </div>
      </RadioGroup>
    </div>
  )
}
```

The component includes built-in accessibility features, keyboard navigation, focus management, and disabled states through Radix UI primitives.
