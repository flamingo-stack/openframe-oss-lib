A reusable radio group component built on top of Radix UI primitives with custom styling and accessibility features. This component provides a styled wrapper around Radix UI's radio group functionality with proper focus states and keyboard navigation.

## Key Components

- **`RadioGroup`** - The root container component that manages radio button state and layout
- **`RadioGroupItem`** - Individual radio button item with custom styling and visual indicator

## Usage Example

```typescript
import { RadioGroup, RadioGroupItem } from "./radio-group"

function SettingsForm() {
  const [theme, setTheme] = React.useState("light")

  return (
    <RadioGroup value={theme} onValueChange={setTheme}>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="light" id="light" />
        <label htmlFor="light">Light</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="dark" id="dark" />
        <label htmlFor="dark">Dark</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="system" id="system" />
        <label htmlFor="system">System</label>
      </div>
    </RadioGroup>
  )
}
```

The component uses Radix UI for accessibility and state management, while providing custom Tailwind CSS styling including focus rings, disabled states, and a circular indicator icon.
