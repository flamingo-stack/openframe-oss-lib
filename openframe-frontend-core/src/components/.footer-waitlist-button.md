A footer-specific wrapper component that provides smart navigation to the waitlist form, either scrolling to it if already on the page or navigating there from other pages.

## Key Components

**FooterWaitlistButton**: React component that wraps `JoinWaitlistButton` with intelligent click handling for footer placement.

**FooterWaitlistButtonProps**: Interface defining optional `className` prop for styling customization.

**handleClick**: Callback function that determines whether to scroll to the waitlist form (if already on `/waitlist`) or navigate to the waitlist page.

## Usage Example

```typescript
import { FooterWaitlistButton } from './footer-waitlist-button';

export function Footer() {
  return (
    <footer className="site-footer">
      <div className="footer-content">
        <FooterWaitlistButton className="footer-cta-button" />
      </div>
    </footer>
  );
}

// Or with custom styling
export function CustomFooter() {
  return (
    <FooterWaitlistButton className="px-6 py-3 bg-blue-500 text-white rounded-lg" />
  );
}
```

The component automatically detects if the user is already on the waitlist page and smoothly scrolls to the form while focusing the email input. For users on other pages, it navigates directly to `/waitlist#waitlist-form`.
