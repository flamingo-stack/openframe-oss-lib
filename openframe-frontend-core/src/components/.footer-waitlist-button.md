A specialized footer component that wraps `JoinWaitlistButton` with intelligent navigation logic, either scrolling to the waitlist form on the current page or navigating to the waitlist page.

## Key Components

- **FooterWaitlistButton**: Main component that handles smart navigation to waitlist functionality
- **FooterWaitlistButtonProps**: Interface defining optional className prop
- **handleClick**: Callback function that determines whether to scroll to form or navigate to waitlist page

## Usage Example

```typescript
import { FooterWaitlistButton } from './footer-waitlist-button';

// Basic usage in footer
function Footer() {
  return (
    <footer className="bg-gray-800 text-white p-4">
      <div className="flex justify-center">
        <FooterWaitlistButton className="bg-blue-600 hover:bg-blue-700" />
      </div>
    </footer>
  );
}

// With custom styling
function CustomFooter() {
  return (
    <FooterWaitlistButton className="px-6 py-3 rounded-lg font-semibold" />
  );
}
```

The component automatically detects if the user is on `/waitlist` and smoothly scrolls to the `#waitlist-form` element while focusing the email input. Otherwise, it navigates to `/waitlist#waitlist-form`.
