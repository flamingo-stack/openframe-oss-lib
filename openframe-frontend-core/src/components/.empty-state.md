A React component that displays contextual empty states with smart call-to-action buttons based on the current page and content type.

## Key Components

- **EmptyStateProps**: Interface defining component configuration options including type, content customization, and CTA behavior
- **EmptyState**: Main component that renders different empty states for vendors, posts, search results, or generic content
- **getDefaultContent()**: Method that provides type-specific icons, titles, and descriptions using Lucide React icons
- **getSmartCTA()**: Intelligent function that generates contextual call-to-action buttons based on current URL path and empty state type

## Usage Example

```typescript
// Basic usage with default smart CTA
<EmptyState type="vendors" />

// Custom content with back button
<EmptyState
  type="search"
  title="No matching results"
  description="Try different search terms"
  showBackButton={true}
  onGoBack={() => router.back()}
/>

// Custom CTA override
<EmptyState
  type="posts"
  showCTA={true}
  ctaText="Create New Post"
  onCtaClick={() => router.push('/create')}
  ctaVariant="secondary"
/>
```

The component automatically adapts its CTA behavior - for example, showing "Reset Filters" on filtered pages or "Browse Vendors" when appropriate, making it highly reusable across different application contexts.
