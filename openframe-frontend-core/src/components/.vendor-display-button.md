A React component that renders clickable buttons displaying vendor information with logo and title, supporting both default and compact visual variants.

## Key Components

- **VendorDisplayButton**: Main component that renders vendor information as an interactive button
- **VendorDisplayButtonProps**: Interface defining component props including vendor data, click handler, visual variant, and optional external URL
- **Variant Support**: Two display modes - `default` (full-sized) and `compact` (inline teaser style)
- **Logo Handling**: Automatic fallback to vendor initials when logo is unavailable
- **External Navigation**: Built-in support for opening vendor pages in new tabs

## Usage Example

```typescript
import { VendorDisplayButton } from './vendor-display-button'
import { VendorWithMedia } from '../utils/vendor-media-stub'

const vendor: VendorWithMedia = {
  slug: 'example-vendor',
  title: 'Example Vendor',
  // ... other vendor properties
}

// Default variant with click handler
<VendorDisplayButton 
  vendor={vendor}
  onClick={(slug) => console.log(`Clicked vendor: ${slug}`)}
/>

// Compact variant with external URL
<VendorDisplayButton 
  vendor={vendor}
  variant="compact"
  externalUrl="https://example.com"
/>

// External navigation using environment variable
<VendorDisplayButton 
  vendor={vendor}
  externalUrl={process.env.NEXT_PUBLIC_OPENMSP_URL}
/>
```

The component automatically handles image proxying, provides hover effects with brand color accents, and ensures responsive text truncation for long vendor names.
