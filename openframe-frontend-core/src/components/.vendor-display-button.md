A reusable React component that displays vendor information in a clickable button format, supporting both default and compact variants with logo display.

## Key Components

- **VendorDisplayButton**: Main component that renders vendor information with logo and title
- **VendorDisplayButtonProps**: Interface defining component props including vendor data, click handler, variant, and external URL
- **Variant Support**: Two display modes - 'default' (larger with full styling) and 'compact' (smaller for embedded use)
- **Logo Handling**: Automatic fallback to vendor title initials when logo is unavailable
- **Click Behavior**: Supports both external URL navigation and custom click handlers

## Usage Example

```typescript
import { VendorDisplayButton } from './vendor-display-button'

// Basic usage with click handler
<VendorDisplayButton 
  vendor={vendorData}
  onClick={(slug) => console.log('Vendor clicked:', slug)}
/>

// Compact variant with external URL
<VendorDisplayButton 
  vendor={vendorData}
  variant="compact"
  externalUrl="https://example.com"
/>

// Default variant for card layouts
<VendorDisplayButton 
  vendor={vendorData}
  onClick={handleVendorClick}
  variant="default"
/>
```

The component automatically handles logo proxying, responsive design, and provides hover effects with consistent theming across both variants.
