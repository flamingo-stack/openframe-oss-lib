A comprehensive chart component library built on top of Recharts that provides theme-aware styling, customizable tooltips, and legends with a unified configuration system.

## Key Components

- **ChartConfig**: Type definition for chart configuration with labels, icons, and theme-aware colors
- **ChartContainer**: Main wrapper component that provides context and responsive container
- **ChartTooltipContent**: Customizable tooltip with indicators, labels, and formatters
- **ChartLegendContent**: Configurable legend component with icon and color support
- **useChart**: Hook to access chart configuration within components
- **ChartStyle**: Dynamic CSS generation for theme-aware chart styling

## Usage Example

```typescript
import { ChartContainer, ChartTooltipContent, ChartConfig } from './chart'
import { BarChart, Bar, XAxis, YAxis } from 'recharts'

const chartConfig: ChartConfig = {
  sales: {
    label: "Sales",
    color: "#8884d8",
  },
  revenue: {
    label: "Revenue", 
    theme: {
      light: "#82ca9d",
      dark: "#4ade80"
    }
  }
}

function MyChart({ data }) {
  return (
    <ChartContainer config={chartConfig}>
      <BarChart data={data}>
        <XAxis dataKey="month" />
        <YAxis />
        <Bar dataKey="sales" />
        <ChartTooltip content={<ChartTooltipContent />} />
      </BarChart>
    </ChartContainer>
  )
}
```
