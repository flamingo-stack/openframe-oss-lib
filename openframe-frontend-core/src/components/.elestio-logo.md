A React SVG component that renders the official Elestio logo with unique IDs to prevent gradient and mask collisions when multiple instances are used on the same page.

## Key Components

- **ElestioLogo**: Main React component that renders the Elestio logo as an SVG
- **Unique ID generation**: Uses `React.useId()` to create collision-free element IDs
- **Responsive design**: Accepts standard SVG props and custom className
- **Complex gradients**: Features multiple linear gradients for the colorful logo design

## Usage Example

```typescript
import { ElestioLogo } from './elestio-logo';

// Basic usage
<ElestioLogo />

// With custom styling
<ElestioLogo 
  className="w-12 h-12" 
  style={{ color: 'red' }}
/>

// Multiple instances (IDs won't conflict)
<div>
  <ElestioLogo className="header-logo" />
  <ElestioLogo className="footer-logo" />
</div>

// With additional SVG props
<ElestioLogo 
  width={50}
  height={48}
  onClick={() => console.log('Logo clicked')}
/>
```

The component automatically handles ID conflicts by generating unique identifiers for each instance, making it safe to use multiple logos on the same page without SVG rendering issues.
