A smooth-animating accordion component built on top of Radix UI primitives, featuring dynamic height calculation with ResizeObserver for seamless expand/collapse animations.

## Key Components

- **`SmoothAccordion`** - Root wrapper component (re-exports Radix AccordionPrimitive.Root)
- **`SmoothAccordionItem`** - Individual accordion item container
- **`SmoothAccordionTrigger`** - Clickable header with label and animated chevron icon
- **`SmoothAccordionContent`** - Content panel with smooth height transitions using ResizeObserver

## Usage Example

```typescript
import { 
  SmoothAccordion, 
  SmoothAccordionItem, 
  SmoothAccordionTrigger, 
  SmoothAccordionContent 
} from './smooth-accordion';

function FAQ() {
  return (
    <SmoothAccordion type="single" collapsible>
      <SmoothAccordionItem value="item-1">
        <SmoothAccordionTrigger label="What is your refund policy?">
        </SmoothAccordionTrigger>
        <SmoothAccordionContent>
          <div className="px-6 md:px-8">
            We offer full refunds within 30 days of purchase.
          </div>
        </SmoothAccordionContent>
      </SmoothAccordionItem>
      
      <SmoothAccordionItem value="item-2">
        <SmoothAccordionTrigger label="How do I contact support?">
        </SmoothAccordionTrigger>
        <SmoothAccordionContent>
          <div className="px-6 md:px-8">
            Email us at support@example.com
          </div>
        </SmoothAccordionContent>
      </SmoothAccordionItem>
    </SmoothAccordion>
  );
}
```

The component automatically handles content height measurement and provides 350ms ease-in-out transitions with proper opacity animations.
