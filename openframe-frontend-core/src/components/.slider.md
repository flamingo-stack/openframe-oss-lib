A React slider component built on top of Radix UI primitives with custom styling and proper accessibility features. This component provides a customizable range input with thumb controls and visual feedback.

## Key Components

- **Slider** - The main slider component that wraps Radix UI's slider primitive with styled track, range, and thumb elements
- **SliderPrimitive.Root** - Base container for the slider
- **SliderPrimitive.Track** - The slider track background
- **SliderPrimitive.Range** - The filled portion of the track
- **SliderPrimitive.Thumb** - The draggable control handle

## Usage Example

```typescript
import { Slider } from "./slider"

// Basic slider
<Slider defaultValue={[50]} max={100} step={1} />

// Range slider with multiple values
<Slider 
  defaultValue={[20, 80]} 
  max={100} 
  step={5}
  className="w-60"
/>

// Controlled slider
const [value, setValue] = React.useState([25])

<Slider 
  value={value} 
  onValueChange={setValue}
  min={0}
  max={100}
  disabled={isDisabled}
/>
```

The component supports all standard Radix UI slider props including `value`, `onValueChange`, `min`, `max`, `step`, and `disabled`. It uses CSS classes for theming with primary colors for active states and proper focus indicators for accessibility.
