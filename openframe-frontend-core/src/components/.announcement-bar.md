A React component that displays dismissible announcement banners with platform-specific persistence and automatic refresh capabilities.

## Key Components

**AnnouncementBar Component**
- Main export that renders announcement banners
- Handles API fetching, caching, and dismissal state
- Supports both PNG and SVG icons with customizable styling
- Platform-aware localStorage keys for multi-app environments

**Key Features**
- Automatic refresh every 60 seconds
- Mobile-responsive design with touch-friendly interactions
- CTA button support with customizable colors and icons
- Platform-specific caching using `getAppType()`

**Helper Functions**
- `getSvgIcon()` - Renders SVG icons with responsive sizing
- `getDismissKey()` - Generates platform-specific dismissal keys
- `getCacheKey()` - Creates platform-specific cache keys

## Usage Example

```typescript
import { AnnouncementBar } from './components/announcement-bar';

// Basic usage - component handles all state internally
export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <div>
      <AnnouncementBar />
      <main>{children}</main>
    </div>
  );
}

// The component automatically:
// - Fetches announcements from /api/announcements/active
// - Caches results in localStorage with platform-specific keys
// - Handles dismissal state per announcement ID
// - Refreshes every 60 seconds
```

The component expects an `Announcement` type with properties like `title`, `description`, `cta_url`, `background_color`, and icon configuration.
