A React tabs component library built on top of Radix UI primitives with customizable styling and accessibility features. This file provides styled wrapper components for creating tabbed interfaces.

## Key Components

- `Tabs` - Root container component (alias for `TabsPrimitive.Root`)
- `TabsList` - Container for tab triggers with default horizontal layout styling
- `TabsTrigger` - Individual tab button with active/inactive states and focus management
- `TabsContent` - Content panel that displays when corresponding tab is active

## Usage Example

```typescript
import { Tabs, TabsList, TabsTrigger, TabsContent } from "./tabs"

function ExampleTabs() {
  return (
    <Tabs defaultValue="account" className="w-[400px]">
      <TabsList className="grid w-full grid-cols-2">
        <TabsTrigger value="account">Account</TabsTrigger>
        <TabsTrigger value="password">Password</TabsTrigger>
      </TabsList>
      <TabsContent value="account">
        <p>Make changes to your account here.</p>
      </TabsContent>
      <TabsContent value="password">
        <p>Change your password here.</p>
      </TabsContent>
    </Tabs>
  )
}
```

All components are forwardRef-enabled and accept standard HTML props plus custom `className` overrides. The styling uses CSS utility classes and includes focus states, disabled states, and active tab indicators.
