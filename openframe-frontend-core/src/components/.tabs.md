A React tabs component built on Radix UI primitives with styled forwarded refs for accessible tabbed interfaces. Provides a complete tab system with customizable styling through className props.

## Key Components

- **`Tabs`** - Root container component (alias for `TabsPrimitive.Root`)
- **`TabsList`** - Container for tab triggers with horizontal flex layout and muted background
- **`TabsTrigger`** - Individual tab button with active/inactive states and focus management
- **`TabsContent`** - Content panels that show/hide based on active tab selection

## Usage Example

```typescript
import { Tabs, TabsList, TabsTrigger, TabsContent } from "./tabs"

function MyTabs() {
  return (
    <Tabs defaultValue="tab1" className="w-[400px]">
      <TabsList className="grid w-full grid-cols-2">
        <TabsTrigger value="tab1">Account</TabsTrigger>
        <TabsTrigger value="tab2">Settings</TabsTrigger>
      </TabsList>
      
      <TabsContent value="tab1" className="space-y-2">
        <h3 className="text-lg font-medium">Account Information</h3>
        <p>Manage your account details here.</p>
      </TabsContent>
      
      <TabsContent value="tab2" className="space-y-2">
        <h3 className="text-lg font-medium">Settings</h3>
        <p>Configure your preferences here.</p>
      </TabsContent>
    </Tabs>
  )
}
```

All components support standard React props and include built-in accessibility features, keyboard navigation, and focus management through Radix UI primitives.
