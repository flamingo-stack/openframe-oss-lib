A customizable badge component built with class-variance-authority for displaying labels, statuses, and tags with multiple visual variants.

## Key Components

- **`Badge`** - Main React component that renders a styled div element
- **`badgeVariants`** - CVA configuration defining style variants (default, secondary, destructive, outline, success)
- **`BadgeProps`** - TypeScript interface extending HTML div attributes with variant props

## Usage Example

```typescript
import { Badge } from './badge'

// Basic usage
<Badge>Default Badge</Badge>

// With variants
<Badge variant="success">Success</Badge>
<Badge variant="destructive">Error</Badge>
<Badge variant="outline">Outlined</Badge>

// With custom className
<Badge className="text-lg" variant="secondary">
  Custom Badge
</Badge>

// As status indicator
<Badge variant={user.isActive ? "success" : "destructive"}>
  {user.isActive ? "Online" : "Offline"}
</Badge>
```

The component uses Tailwind CSS classes and supports all standard div attributes while providing consistent styling through predefined variants.
