A React component that renders comment cards with contextual user/vendor information, supporting both profile and vendor page contexts with appropriate permissions and styling.

## Key Components

- **CommentCard** - Main component for displaying comment content with metadata
- **CommentCardProps** - TypeScript interface defining comment structure and display options
- **formatActivityTime()** - Utility function for human-readable timestamps
- **canUserDeleteComment()** - Permission logic for comment deletion
- **Context switching** - Displays either vendor info (profile context) or user info (vendor context)

## Usage Example

```typescript
import { CommentCard } from './comment-card'

// Profile page - shows vendor info for each comment
<CommentCard
  comment={{
    id: "123",
    content: "Great product with excellent support",
    title: "Highly Recommended",
    type: "pro",
    createdAt: "2024-01-15T10:30:00Z",
    vendor: {
      id: 456,
      title: "Awesome Software",
      slug: "awesome-software",
      logo: "/logo.png"
    },
    canDelete: true
  }}
  context="profile"
  onViewProduct={(slug) => router.push(`/vendors/${slug}`)}
  onDeleteComment={(id) => handleDelete(id)}
  showVendorInfo={true}
/>

// Vendor page - shows user info for each comment
<CommentCard
  comment={{
    id: "124",
    content: "Works well but lacks some features",
    createdAt: "2024-01-15T09:15:00Z",
    user: {
      id: "user789",
      name: "John Smith",
      profilePicture: "/avatar.jpg"
    }
  }}
  context="vendor"
  compact={true}
/>
```
