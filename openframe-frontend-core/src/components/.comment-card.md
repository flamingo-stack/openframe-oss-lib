A React component that displays comment cards with user information, vendor details, and interaction capabilities for profile and vendor contexts.

## Key Components

- **CommentCard** - Main component that renders comment content with user/vendor info
- **CommentCardProps** - TypeScript interface defining comment structure and display options
- **Context-aware rendering** - Displays different layouts for 'profile' vs 'vendor' contexts
- **Delete functionality** - Shows delete button based on user permissions
- **VendorDisplayButton** - Displays vendor information when in profile context
- **UserSummary** - Shows user details when in vendor context

## Usage Example

```typescript
import { CommentCard } from './comment-card'

// Profile context - showing vendor info
<CommentCard
  comment={{
    id: "123",
    content: "Great product for our MSP",
    title: "Excellent service",
    type: "pro",
    createdAt: "2024-01-15T10:00:00Z",
    vendor: {
      id: 1,
      title: "VendorCorp",
      slug: "vendorcorp",
      logo: "/logo.png"
    },
    canDelete: true
  }}
  context="profile"
  onViewProduct={(slug) => router.push(`/vendors/${slug}`)}
  onDeleteComment={(id) => deleteComment(id)}
  showVendorInfo={true}
  compact={false}
/>

// Vendor context - showing user info
<CommentCard
  comment={comment}
  context="vendor"
  showVendorInfo={false}
/>
```
