A React switch component built on top of Radix UI primitives, providing a styled toggle input with accessibility features and smooth animations.

## Key Components

- **Switch**: A forwardRef component that wraps `SwitchPrimitives.Root` with custom styling
- Uses Radix UI's switch primitives for accessibility and behavior
- Includes a thumb element that slides between checked/unchecked states
- Supports all standard switch props and ref forwarding

## Usage Example

```typescript
import { Switch } from "./components/ui/switch"
import { useState } from "react"

function SettingsPanel() {
  const [notifications, setNotifications] = useState(false)
  
  return (
    <div className="flex items-center space-x-2">
      <Switch 
        checked={notifications}
        onCheckedChange={setNotifications}
        id="notifications"
      />
      <label htmlFor="notifications">
        Enable notifications
      </label>
    </div>
  )
}

// With custom styling
<Switch 
  className="data-[state=checked]:bg-green-500" 
  defaultChecked={true}
  disabled={false}
/>
```

The component automatically handles focus states, keyboard navigation, and provides smooth transitions between states. The styling uses CSS custom properties and data attributes for different states (checked/unchecked, disabled, focused).
