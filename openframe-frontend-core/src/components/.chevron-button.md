A customizable chevron button component that rotates based on expansion state, commonly used for dropdowns, accordions, and collapsible content sections.

## Key Components

- **ChevronButton**: Main component with configurable size, colors, and expansion state
- **ChevronButtonProps**: Interface defining props including `isExpanded`, `size`, `backgroundColor`, `borderColor`, and `onClick`
- **Dynamic styling**: Hover effects and conditional styling based on props
- **Size variants**: `sm`, `default`, `lg`, and `icon` sizes with corresponding chevron dimensions

## Usage Example

```typescript
import { ChevronButton } from './chevron-button';

function DropdownMenu() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <ChevronButton
        isExpanded={isOpen}
        size="default"
        backgroundColor="#f3f4f6"
        borderColor="#d1d5db"
        onClick={() => setIsOpen(!isOpen)}
        className="ml-2"
      />
      {isOpen && (
        <div>Dropdown content</div>
      )}
    </div>
  );
}
```

The chevron automatically rotates 180 degrees when `isExpanded` is true, providing smooth visual feedback for interactive elements.
