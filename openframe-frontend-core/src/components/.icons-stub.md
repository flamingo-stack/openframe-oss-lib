A stub file providing icon mappings from custom icon names to corresponding Lucide React icons, serving as temporary implementations or fallbacks for missing custom icons.

## Key Components

- **OpenSourceIcon** - Maps to Code icon for open source related UI
- **CoinsIcon** - Maps to DollarSign icon for financial/pricing features  
- **VendorDirectoryIcon** - Maps to Building icon for vendor listings
- **CommunityHubIcon** - Maps to Users icon for community features
- **VendorsIcon** - Alternative Building icon export for vendor-related UI
- **CommunityIcon** - Maps to MessageSquare icon for community discussions
- **CompareIcon** - Maps to GitCompare icon for comparison features

## Usage Example

```typescript
import { 
  OpenSourceIcon, 
  CoinsIcon, 
  CommunityHubIcon 
} from './icons-stub';

function FeatureCard({ type }: { type: string }) {
  const getIcon = () => {
    switch(type) {
      case 'opensource': return <OpenSourceIcon className="w-6 h-6" />;
      case 'pricing': return <CoinsIcon className="w-6 h-6" />;
      case 'community': return <CommunityHubIcon className="w-6 h-6" />;
      default: return null;
    }
  };

  return <div className="flex items-center">{getIcon()}</div>;
}
```

This approach allows consistent icon usage across the application while providing flexibility to replace with custom implementations later.
