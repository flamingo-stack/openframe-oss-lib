A React component that displays boolean values with appropriate icons and text, along with utility logic for evaluating feature values across different data types.

## Key Components

- **YesNoDisplay** - Main component that renders boolean values with check/x icons
- **evaluateFeatureValue** - Utility function that converts various data types and formats into boolean values

## Usage Example

```typescript
import { YesNoDisplay, evaluateFeatureValue } from './yes-no-display'

// Basic boolean display
<YesNoDisplay value={true} />

// Custom text labels
<YesNoDisplay 
  value={false} 
  yesText="Available" 
  noText="Not Available" 
/>

// Custom text override
<YesNoDisplay 
  value={true} 
  customText="Premium Feature" 
/>

// Evaluate feature values from data
const hasFeature = evaluateFeatureValue("âœ…", "text") // returns true
const isEnabled = evaluateFeatureValue("0", "number") // returns false
const isActive = evaluateFeatureValue("yes", "boolean") // returns true

<YesNoDisplay value={hasFeature} />
```

The component uses Lucide icons and custom styling classes, displaying green check icons for positive values and red X icons for negative values. The `evaluateFeatureValue` function handles complex data conversion scenarios including emojis, various text formats, and numeric values.
