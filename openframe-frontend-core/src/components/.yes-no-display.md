A React component for displaying boolean values as visual yes/no indicators with icons, plus a utility function for evaluating feature values across different data types.

## Key Components

- **`YesNoDisplay`** - React component that renders boolean values with appropriate icons (check/X) and text
- **`evaluateFeatureValue()`** - Utility function that converts various string/null values to boolean based on data type
- **`YesNoDisplayProps`** - TypeScript interface defining component props including optional custom text and styling

## Usage Example

```typescript
import { YesNoDisplay, evaluateFeatureValue } from './yes-no-display'

// Basic boolean display
<YesNoDisplay value={true} />
<YesNoDisplay value={false} yesText="Available" noText="Not Available" />

// Custom text with icon
<YesNoDisplay value={true} customText="Feature enabled" />

// Evaluate feature values
const isEnabled = evaluateFeatureValue("âœ…", "text") // true
const hasValue = evaluateFeatureValue("null", "boolean") // false
const isPositive = evaluateFeatureValue("5", "number") // true

// Combined usage
const featureValue = "yes"
const evaluated = evaluateFeatureValue(featureValue, "boolean")
<YesNoDisplay value={evaluated} />
```

The component displays green check icons for positive values and red X icons for negative values, with customizable text labels and CSS classes.
