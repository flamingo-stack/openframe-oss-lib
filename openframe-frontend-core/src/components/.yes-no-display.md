A React component for displaying boolean values with appropriate icons and text, plus utility logic for evaluating feature values across different data types.

## Key Components

- **YesNoDisplay**: Main React component that renders boolean values with check/X icons
- **evaluateFeatureValue**: Utility function that determines boolean state from various data formats (boolean, text, number)

The component supports customizable yes/no text, custom text override, and flexible styling through className props.

## Usage Example

```typescript
import { YesNoDisplay, evaluateFeatureValue } from './yes-no-display'

// Basic boolean display
<YesNoDisplay value={true} />

// Custom text labels
<YesNoDisplay 
  value={false} 
  yesText="Active" 
  noText="Inactive" 
/>

// Custom text override
<YesNoDisplay 
  value={true} 
  customText="Premium Feature Available" 
/>

// Evaluate feature values
const isEnabled = evaluateFeatureValue("✅", "text")  // true
const isActive = evaluateFeatureValue("0", "number")  // false
const hasFeature = evaluateFeatureValue("yes", "boolean")  // true

// Use together
<YesNoDisplay 
  value={evaluateFeatureValue(featureData.value, featureData.type)}
  className="my-2"
/>
```

The `evaluateFeatureValue` function handles various input formats including emojis (✅/❌), text values ("yes"/"no"), numbers (>0 is true), and explicit null/empty states.
