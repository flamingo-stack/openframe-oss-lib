A reusable React component that renders authentication provider buttons with consistent styling and default text for popular OAuth providers.

## Key Components

- **ProviderButton** - Main component that extends standard button functionality
- **ProviderButtonProps** - TypeScript interface defining component props including:
  - `provider`: Supports 'google', 'microsoft', 'slack', 'github'
  - `variant`: Button style variants ('primary' or 'outline')
  - `size`: Button size options ('default', 'sm', 'lg')
  - `loading`: Optional loading state

## Usage Example

```typescript
import { ProviderButton } from './provider-button'

// Basic usage with default text
<ProviderButton provider="google" />

// Custom styling and content
<ProviderButton 
  provider="github" 
  variant="primary"
  size="lg"
  className="mb-4"
>
  Connect GitHub Account
</ProviderButton>

// With loading state
<ProviderButton 
  provider="microsoft"
  loading={isAuthenticating}
  onClick={handleMicrosoftAuth}
/>
```

The component automatically generates default text like "Sign in with Google" but can be overridden with custom children. It forwards refs and extends all standard HTML button attributes for maximum flexibility.
