A styled label component built on top of Radix UI's Label primitive with consistent typography and styling for form labels.

## Key Components

- **`Label`** - A forwardRef component that wraps Radix UI's Label primitive
- **`labelVariants`** - CVA (Class Variance Authority) configuration for label styling
- **`cn`** - Utility function for merging CSS classes

## Usage Example

```typescript
import { Label } from "./components/ui/label"

// Basic usage
<Label htmlFor="email">Email Address</Label>

// With custom className
<Label htmlFor="password" className="text-red-500">
  Password
</Label>

// In a form context
<div>
  <Label htmlFor="username">Username</Label>
  <input id="username" type="text" />
</div>
```

The component provides semantic accessibility through proper label-input associations and includes built-in styling with DM Sans font, primary text color, and responsive text sizing. It maintains all native label functionality while ensuring consistent visual presentation across the application.
