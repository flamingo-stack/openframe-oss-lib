A React component that provides a styled form label built on top of Radix UI's Label primitive with consistent typography and styling.

## Key Components

- **Label** - A forwardRef component that wraps `@radix-ui/react-label`
- **labelVariants** - CVA variant configuration defining the label's visual styling
- Accessibility features inherited from Radix UI Label primitive

## Usage Example

```typescript
import { Label } from "./components/ui/label"

function FormField() {
  return (
    <div>
      <Label htmlFor="email">Email Address</Label>
      <input 
        id="email" 
        type="email" 
        placeholder="Enter your email"
      />
    </div>
  )
}

// With custom styling
function CustomLabel() {
  return (
    <Label 
      htmlFor="username" 
      className="text-blue-600"
    >
      Username
    </Label>
  )
}
```

The component uses DM Sans font family with semibold weight, uppercase text transformation, and responsive text sizing. It automatically applies proper semantic labeling for form accessibility when used with the `htmlFor` attribute.
