A React component that renders an SVG icon representing vendors or partnerships, designed with customizable dimensions and styling.

## Key Components

- **VendorsIcon**: React functional component that renders an SVG icon
- **VendorsIconProps**: TypeScript interface defining optional props for className, width, and height
- **SVG paths**: Two path elements creating the vendor/partnership visual representation

## Usage Example

```typescript
import { VendorsIcon } from './vendors-icon';

// Basic usage with default size (24x24)
<VendorsIcon />

// Custom size and styling
<VendorsIcon 
  width={32} 
  height={32} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <VendorsIcon width={16} height={16} />
  Vendors
</button>
```

The icon uses `currentColor` for the fill property, making it easy to style with CSS text color utilities. Default dimensions are 24x24 pixels, but can be customized through props.
