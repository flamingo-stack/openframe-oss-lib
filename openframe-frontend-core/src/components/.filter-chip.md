A unified React component for displaying filterable chips with consistent styling across the application. Supports multiple visual variants, sizing options, and optional removal functionality.

## Key Components

- **FilterChip**: Main component with customizable appearance and behavior
- **FilterChipProps**: TypeScript interface defining component props
- **Variant system**: Six visual styles (`selected`, `unselected`, `category`, `subcategory`, `tag`, `info`)
- **Size options**: Two sizes (`sm`, `md`) with responsive design
- **Accessibility**: Full ARIA support with keyboard navigation and screen reader compatibility

## Usage Example

```typescript
import { FilterChip } from './filter-chip'

// Basic filter chip
<FilterChip
  id="tech"
  label="Technology"
  onClick={() => console.log('Clicked')}
/>

// Removable category chip
<FilterChip
  id="react"
  label="React Development"
  variant="category"
  removable={true}
  onRemove={() => console.log('Removed')}
/>

// Small info chip (non-interactive)
<FilterChip
  id="count"
  label="5 items"
  variant="info"
  size="sm"
/>

// Disabled selected chip
<FilterChip
  id="disabled"
  label="Unavailable"
  variant="selected"
  disabled={true}
/>
```

The component uses Tailwind CSS classes with custom design tokens for theming, and includes hover animations, focus states, and mobile-optimized touch targets.
