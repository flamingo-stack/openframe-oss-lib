A React component library that provides a customizable sheet (slide-out panel) UI component built on top of Radix UI Dialog primitives. The sheet can slide in from any of the four sides with smooth animations and includes optional header, footer, title, and description sections.

## Key Components

- **Sheet** - Root container component
- **SheetTrigger** - Button/element that opens the sheet
- **SheetContent** - Main sheet panel with configurable slide direction (`top`, `bottom`, `left`, `right`)
- **SheetOverlay** - Semi-transparent backdrop
- **SheetHeader/SheetFooter** - Layout containers for sheet content
- **SheetTitle/SheetDescription** - Typography components for sheet headings
- **SheetClose** - Close button component

## Usage Example

```typescript
import { Sheet, SheetContent, SheetDescription, SheetHeader, SheetTitle, SheetTrigger } from "./sheet"

function MyComponent() {
  return (
    <Sheet>
      <SheetTrigger asChild>
        <button>Open Sheet</button>
      </SheetTrigger>
      <SheetContent side="right">
        <SheetHeader>
          <SheetTitle>Settings</SheetTitle>
          <SheetDescription>
            Configure your preferences here.
          </SheetDescription>
        </SheetHeader>
        <div>Sheet content goes here</div>
      </SheetContent>
    </Sheet>
  )
}
```

The sheet supports four slide directions and includes built-in animations, accessibility features, and a close button with screen reader support.
