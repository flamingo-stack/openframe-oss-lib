A comprehensive React component for interactive image cropping with drag/resize functionality, custom aspect ratios, and automatic scaling.

## Key Components

- **`ImageCropper`** - Main React component providing cropping interface
- **`ImageCropperResult`** - Interface for cropped image output (dataUrl + blob)
- **`ImageCropperProps`** - Configuration interface with aspect ratio, circular mode, and size limits
- **`loadImage()`** - Utility for loading images with cross-origin support
- **`exportCrop()`** - Canvas-based image processing for final output

## Usage Example

```typescript
import { ImageCropper } from './components/image-cropper'

function AvatarUploader() {
  const [imageFile, setImageFile] = useState<string>()

  const handleCropComplete = (result: ImageCropperResult) => {
    // Use result.dataUrl for preview or result.blob for upload
    uploadAvatar(result.blob)
  }

  return (
    <>
      {imageFile && (
        <ImageCropper
          src={imageFile}
          aspectRatio={1} // Square crop
          circular={true} // Avatar mode
          maxSizePx={256}
          onComplete={handleCropComplete}
          onCancel={() => setImageFile(undefined)}
        />
      )}
    </>
  )
}
```

The component includes zoom/rotation controls, keyboard accessibility (Enter/Esc), and exports PNG format with automatic scaling to prevent oversized outputs.
