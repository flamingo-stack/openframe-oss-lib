A utility function for dynamically rendering SVG icons from a predefined set of Lucide React icons and custom logos. Provides a centralized way to render icons by name with fallback handling.

## Key Components

- **`renderSvgIcon()`** - Main function that takes an icon name string and optional SVG props, returns the corresponding React icon component
- **Icon mapping** - Supports Lucide icons (`megaphone`, `bell`, `info`, `star`, `rocket`, `package`) and custom logos (`openframe-logo`, `openmsp-logo`, `flamingo`)
- **Fallback handling** - Defaults to `Megaphone` icon for unrecognized names

## Usage Example

```typescript
import { renderSvgIcon } from './icon-utils';

// Basic usage
const NotificationIcon = () => renderSvgIcon('bell');

// With custom props
const StyledIcon = () => renderSvgIcon('star', { 
  size: 24, 
  color: 'blue',
  className: 'my-icon'
});

// Custom logo
const Logo = () => renderSvgIcon('openframe-logo', { width: 100 });

// Fallback behavior - renders Megaphone for unknown names
const UnknownIcon = () => renderSvgIcon('unknown-icon');
```
