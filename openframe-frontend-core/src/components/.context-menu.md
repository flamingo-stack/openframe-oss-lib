A React context menu component library built on top of Radix UI primitives, providing styled and accessible right-click context menus with support for items, checkboxes, radio buttons, submenus, and keyboard shortcuts.

## Key Components

- **ContextMenu** - Root wrapper component for context menu functionality
- **ContextMenuTrigger** - Element that triggers the context menu on right-click
- **ContextMenuContent** - Container for menu items with portal rendering and animations
- **ContextMenuItem** - Individual clickable menu item with hover states
- **ContextMenuCheckboxItem** - Checkbox menu item with check indicator
- **ContextMenuRadioItem** - Radio button menu item for single selections
- **ContextMenuLabel** - Non-interactive label for grouping menu items
- **ContextMenuSeparator** - Visual separator between menu sections
- **ContextMenuShortcut** - Text component for displaying keyboard shortcuts
- **ContextMenuSub** - Submenu container with chevron indicator
- **ContextMenuGroup/RadioGroup** - Grouping components for related items
