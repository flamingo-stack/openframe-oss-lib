This file provides a complete set of styled React components for building context menus, wrapping Radix UI's context menu primitives with consistent design tokens and animations.

## Key Components

- **ContextMenu** - Root container component
- **ContextMenuTrigger** - Element that triggers the context menu on right-click
- **ContextMenuContent** - Main menu container with animations and portal rendering
- **ContextMenuItem** - Individual menu items with hover states
- **ContextMenuCheckboxItem** - Checkable menu items with check indicators
- **ContextMenuRadioItem** - Radio button menu items for single selection
- **ContextMenuLabel** - Non-interactive labels for grouping items
- **ContextMenuSeparator** - Visual dividers between menu sections
- **ContextMenuShortcut** - Text component for displaying keyboard shortcuts
- **Submenu components** - ContextMenuSub, ContextMenuSubTrigger, ContextMenuSubContent for nested menus
