A reusable avatar component built on Radix UI primitives that displays user profile images with automatic fallback support. The component handles image loading states and provides a consistent circular avatar design.

## Key Components

- **Avatar** - Root container component with circular styling and fixed dimensions (40x40px by default)
- **AvatarImage** - Image component that displays the actual avatar picture with aspect ratio preservation
- **AvatarFallback** - Fallback content shown when the image fails to load or is unavailable

## Usage Example

```typescript
import { Avatar, AvatarImage, AvatarFallback } from "./avatar"

// Basic avatar with image and fallback
<Avatar>
  <AvatarImage src="/user-profile.jpg" alt="User Name" />
  <AvatarFallback>UN</AvatarFallback>
</Avatar>

// Custom sized avatar with initials fallback
<Avatar className="h-16 w-16">
  <AvatarImage src="https://github.com/username.png" />
  <AvatarFallback className="text-lg font-semibold">
    JD
  </AvatarFallback>
</Avatar>

// Avatar with only fallback content
<Avatar>
  <AvatarFallback className="bg-blue-500 text-white">
    ðŸ‘¤
  </AvatarFallback>
</Avatar>
```

The component automatically shows the fallback content when the image fails to load, making it perfect for user profiles, comment sections, and team member displays.
