A custom toast notification component that wraps the Sonner library with predefined dark theme styling and custom CSS classes for consistent design system integration.

## Key Components

- **Toaster**: Main component that renders toast notifications with dark theme and bottom-right positioning
- **ToasterProps**: Type alias extending Sonner's component props for type safety

## Usage Example

```typescript
import { Toaster } from "./sonner"

// Add to your app layout or root component
function App() {
  return (
    <div>
      {/* Your app content */}
      <Toaster />
    </div>
  )
}

// Trigger toasts from anywhere in your app
import { toast } from "sonner"

function MyComponent() {
  const handleClick = () => {
    toast.success("Operation completed!")
    toast.error("Something went wrong")
    toast("Info message")
  }
  
  return <button onClick={handleClick}>Show Toast</button>
}
```

The component uses a custom design system with `ods-*` CSS classes for consistent theming across different toast types (success, error, info) and includes styled action/cancel buttons.
