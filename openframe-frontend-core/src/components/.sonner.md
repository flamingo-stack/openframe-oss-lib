A React component that wraps the Sonner toast notification library with custom styling and dark theme configuration for consistent toast messages across the application.

## Key Components

- **`Toaster`** - Main toast notification component with pre-configured styling and positioning
- **`ToasterProps`** - Type definition extending Sonner's component props for type safety

## Usage Example

```typescript
import { Toaster } from './sonner'
import { toast } from 'sonner'

// Add the Toaster component to your app root
function App() {
  return (
    <div>
      {/* Your app content */}
      <Toaster />
    </div>
  )
}

// Trigger toast notifications anywhere in your app
function handleSuccess() {
  toast.success("Operation completed successfully!")
}

function handleError() {
  toast.error("Something went wrong!")
}

function handleInfo() {
  toast("Info message", {
    description: "Additional details here"
  })
}
```

The component is pre-styled with ODS (design system) color tokens and positioned at the bottom-right of the screen. It includes custom styling for different toast types (success, error, info) and interactive elements like action and cancel buttons. The dark theme and custom CSS classes ensure consistent appearance with the application's design system.
