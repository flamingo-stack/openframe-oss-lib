This file provides custom React SVG icon components for status indicators, specifically check (success) and X (error) circle icons.

## Key Components

- **`IconProps`** - TypeScript interface defining optional className prop for styling
- **`CheckCircleIcon`** - Green circle icon with checkmark for success states
- **`XCircleIcon`** - Red circle icon with X mark for error/failure states

## Usage Example

```typescript
import { CheckCircleIcon, XCircleIcon } from './custom-icons';

function StatusIndicator({ isSuccess }: { isSuccess: boolean }) {
  return (
    <div className="flex items-center gap-2">
      {isSuccess ? (
        <CheckCircleIcon className="w-6 h-6" />
      ) : (
        <XCircleIcon className="w-6 h-6" />
      )}
      <span>{isSuccess ? 'Success' : 'Error'}</span>
    </div>
  );
}

// Or use individually
<CheckCircleIcon className="text-green-500 hover:scale-110" />
<XCircleIcon className="ml-2 opacity-75" />
```

Both icons are sized at 25x24 pixels by default and accept custom CSS classes for styling flexibility. The SVGs use semantic colors (green for success, red for error) and are optimized for web use.
