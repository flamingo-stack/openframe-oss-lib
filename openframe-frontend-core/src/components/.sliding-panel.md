A sliding panel component that animates in from the right side of the screen, built on top of Headless UI Dialog for accessibility and proper focus management.

## Key Components

- **SlidingPanel**: Main component that renders a slide-out panel with header, content area, and optional footer
- **SlidingPanelProps**: Interface defining component props including size variants, content, and event handlers
- Built-in accessibility features via Headless UI Dialog
- Automatic body scroll prevention when panel is open

## Usage Example

```typescript
import { SlidingPanel } from './sliding-panel';
import { useState } from 'react';

function MyComponent() {
  const [isPanelOpen, setIsPanelOpen] = useState(false);

  return (
    <>
      <button onClick={() => setIsPanelOpen(true)}>
        Open Panel
      </button>

      <SlidingPanel
        isOpen={isPanelOpen}
        onClose={() => setIsPanelOpen(false)}
        title="Settings"
        subtitle="Configure your preferences"
        size="lg"
        footer={
          <div className="flex gap-2">
            <button>Cancel</button>
            <button>Save</button>
          </div>
        }
      >
        <div className="p-6">
          Panel content goes here
        </div>
      </SlidingPanel>
    </>
  );
}
```

The component supports six size variants (`sm`, `md`, `lg`, `xl`, `half`, `fullscreen`) and includes smooth transitions with backdrop blur effects.
