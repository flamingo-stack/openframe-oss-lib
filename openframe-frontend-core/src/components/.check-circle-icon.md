A React component that renders an SVG check circle icon with customizable styling and properties.

## Key Components

- **CheckCircleIcon**: Main functional component that renders a check mark inside a circle
- **CheckCircleIconProps**: Interface extending React's SVGProps with optional className
- **Default props**: Sets default color to '#1A1A1A' (dark gray) and 24x24 dimensions

## Usage Example

```typescript
import { CheckCircleIcon } from './check-circle-icon'

// Basic usage
<CheckCircleIcon />

// With custom styling
<CheckCircleIcon 
  className="success-icon" 
  color="#22C55E" 
  width={32} 
  height={32}
/>

// With click handler
<CheckCircleIcon 
  onClick={() => console.log('Clicked!')}
  style={{ cursor: 'pointer' }}
/>
```

The component accepts all standard SVG props through spread operator, making it highly flexible for different use cases like form validation indicators, status badges, or interactive elements.
