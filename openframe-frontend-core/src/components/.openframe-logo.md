A React SVG component that renders the OpenFrame logo with customizable colors and standard SVG properties.

## Key Components

- **OpenFrameLogo**: Main component that renders a 32x32 SVG logo
- **Props**: Extends `React.SVGProps<SVGSVGElement>` with additional color customization options
  - `className`: Optional CSS class name
  - `lowerPathColor`: Color for the white square elements (default: '#ffffff')
  - `upperPathColor`: Color for the main frame/border (default: '#1A1A1A')

## Usage Example

```typescript
import { OpenFrameLogo } from './openframe-logo';

// Basic usage
<OpenFrameLogo />

// With custom styling
<OpenFrameLogo 
  className="my-logo" 
  width={64} 
  height={64}
/>

// With custom colors
<OpenFrameLogo 
  lowerPathColor="#ff0000"
  upperPathColor="#0000ff"
/>

// With additional SVG props
<OpenFrameLogo 
  onClick={() => console.log('Logo clicked')}
  style={{ cursor: 'pointer' }}
/>
```

The logo consists of a main frame/border path and four white square elements positioned in the bottom-right area, creating a distinctive geometric design that can be easily customized for different themes and contexts.
