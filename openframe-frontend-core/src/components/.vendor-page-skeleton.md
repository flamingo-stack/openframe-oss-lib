A simple React component that provides a skeleton loading state for vendor pages by wrapping the vendor detail layout skeleton component.

## Key Components

- **`VendorPageSkeleton`** - Main export function that renders a skeleton loading UI for vendor pages

## Usage Example

```typescript
import { VendorPageSkeleton } from './vendor-page-skeleton';

// Use as a loading fallback in React Suspense
function VendorPage() {
  return (
    <Suspense fallback={<VendorPageSkeleton />}>
      <VendorContent />
    </Suspense>
  );
}

// Or conditionally render while data is loading
function VendorContainer({ isLoading }: { isLoading: boolean }) {
  if (isLoading) {
    return <VendorPageSkeleton />;
  }
  
  return <ActualVendorPage />;
}
```

This component serves as a thin wrapper around `VendorDetailLayoutSkeleton`, providing a consistent loading experience for vendor-related pages throughout the application.
