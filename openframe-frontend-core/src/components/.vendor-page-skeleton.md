A React component that provides skeleton loading state for vendor pages by wrapping the vendor detail layout skeleton component.

## Key Components

- **`VendorPageSkeleton`** - Main export component that renders skeleton loading UI for vendor pages

## Usage Example

```typescript
import { VendorPageSkeleton } from './vendor-page-skeleton';

// Use as loading state while vendor data is being fetched
function VendorPage({ vendorId }: { vendorId: string }) {
  const { data: vendor, isLoading } = useVendor(vendorId);
  
  if (isLoading) {
    return <VendorPageSkeleton />;
  }
  
  return <VendorDetailView vendor={vendor} />;
}

// Or use with Suspense boundary
<Suspense fallback={<VendorPageSkeleton />}>
  <VendorPageContent />
</Suspense>
```

This component serves as a thin wrapper around `VendorDetailLayoutSkeleton` to provide consistent loading states across vendor-related pages in the application.
