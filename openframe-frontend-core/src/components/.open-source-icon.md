A React SVG icon component that renders an open source/code symbol with angle brackets and a forward slash, commonly used to represent open source projects or code-related content.

## Key Components

- **OpenSourceIcon**: Main React functional component that renders the SVG icon
- **OpenSourceIconProps**: TypeScript interface defining optional props for styling and sizing
- **SVG paths**: Two path elements creating the visual design - one for the forward slash and one for the angle brackets

## Usage Example

```typescript
import { OpenSourceIcon } from './open-source-icon';

// Basic usage with default size (19x18)
<OpenSourceIcon />

// Custom size and styling
<OpenSourceIcon 
  width={24} 
  height={22} 
  className="text-blue-500 hover:text-blue-700" 
/>

// In a button or link
<button className="flex items-center gap-2">
  <OpenSourceIcon width={16} height={15} />
  View Source
</button>
```
