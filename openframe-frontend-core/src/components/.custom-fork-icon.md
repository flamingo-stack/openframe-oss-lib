A customizable React SVG component that renders a fork icon, commonly used to represent repository forking or branching actions in version control interfaces.

## Key Components

- **`CustomForkIconProps`** - Interface defining optional props for styling and sizing
- **`CustomForkIcon`** - Main functional component that renders the fork SVG with customizable properties

## Usage Example

```typescript
import { CustomForkIcon } from './custom-fork-icon';

// Basic usage with defaults
<CustomForkIcon />

// Customized styling
<CustomForkIcon 
  className="hover:opacity-75" 
  fill="#3b82f6" 
  width={32} 
  height={32} 
/>

// With additional SVG props
<CustomForkIcon 
  onClick={handleFork}
  role="button"
  aria-label="Fork repository"
/>
```

The component accepts standard SVG props through rest parameters, making it flexible for various use cases. Default values provide a 24x24 gray icon that works well in most UI contexts.
