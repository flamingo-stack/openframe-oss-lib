A customizable close/dismiss button component that renders an X icon with configurable styling, hover effects, and size variants.

## Key Components

- **`XButton`** - Main React component that wraps a Button with an X icon
- **`XButtonProps`** - TypeScript interface defining component props:
  - `className` - Additional CSS classes
  - `size` - Button size variant ('sm' | 'default' | 'lg' | 'icon')
  - `backgroundColor` - Custom background color
  - `borderColor` - Custom border color
  - `onClick` - Click event handler

## Usage Example

```typescript
import { XButton } from './components/x-button';

// Basic close button
<XButton onClick={() => setModalOpen(false)} />

// Customized styling
<XButton 
  size="lg"
  backgroundColor="rgba(255, 0, 0, 0.1)"
  borderColor="red"
  className="absolute top-2 right-2"
  onClick={handleClose}
/>

// Small variant for compact spaces
<XButton 
  size="sm"
  onClick={() => removeItem(id)}
/>
```

The component features hover state management that makes the background transparent on hover, responsive sizing based on the `size` prop, and dynamic styling through inline styles for background and border colors. The X icon scales proportionally with the button size and uses the Lucide React icon library.
