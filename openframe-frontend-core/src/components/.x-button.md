A customizable close/dismiss button component that renders an X icon with configurable styling and hover effects.

## Key Components

- **XButton**: Main component that renders a button with an X icon from Lucide React
- **XButtonProps**: Interface defining component props including size, colors, and click handler
- **Size variants**: `sm`, `default`, `lg`, and `icon` with corresponding dimensions
- **Dynamic styling**: Configurable background and border colors with hover state management

## Usage Example

```typescript
import { XButton } from './x-button';

// Basic usage
<XButton onClick={(e) => console.log('Close clicked')} />

// Custom styling with border
<XButton 
  size="lg"
  backgroundColor="#ef4444"
  borderColor="#dc2626"
  className="shadow-md"
  onClick={handleClose}
/>

// Small variant for compact layouts
<XButton 
  size="sm"
  onClick={() => setVisible(false)}
/>
```

The component uses the `Button` base component with "ghost" variant and includes smooth transitions. On hover, the background becomes transparent regardless of the initial `backgroundColor` prop, creating a consistent interactive experience.
