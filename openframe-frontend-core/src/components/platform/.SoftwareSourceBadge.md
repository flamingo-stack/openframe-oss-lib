A React component that displays styled badges for different software source types from Fleet MDM, supporting all osquery software sources with appropriate platform-specific icons and labels.

## Key Components

- **`SoftwareSource` type**: Union type defining all supported software source types (apps, chrome_extensions, homebrew_packages, etc.)
- **`SoftwareSourceBadgeProps` interface**: Component props including required `source` and optional `className`
- **`sourceConfig` object**: Maps each source type to human-readable labels and platform-specific icons
- **`SoftwareSourceBadge` component**: Main component that renders a button-styled badge with icon and label

## Usage Example

```typescript
import { SoftwareSourceBadge } from './SoftwareSourceBadge'

// Display different software source badges
function SoftwareList() {
  return (
    <div className="flex gap-2">
      <SoftwareSourceBadge source="apps" />
      <SoftwareSourceBadge source="chrome_extensions" />
      <SoftwareSourceBadge source="homebrew_packages" />
      <SoftwareSourceBadge source="apt_sources" className="custom-styling" />
    </div>
  )
}
```

The component automatically selects appropriate icons (MacOS, Windows, Linux, or generic Package icons) and displays concise labels for each software source type, making it easy to identify software installation methods in Fleet MDM interfaces.
