A React component that displays styled badges for different software source types from Fleet MDM, with appropriate icons and labels for package managers, browsers, applications, and system-specific sources across macOS, Windows, and Linux platforms.

## Key Components

- **`SoftwareSource`** - TypeScript union type defining 18+ supported software source types
- **`SoftwareSourceBadgeProps`** - Component props interface with source type and optional className
- **`sourceConfig`** - Configuration object mapping source types to human-readable labels and platform-specific icons
- **`SoftwareSourceBadge`** - Main component that renders a button-styled badge with icon and label

## Usage Example

```typescript
import { SoftwareSourceBadge } from './components/SoftwareSourceBadge'

// Basic usage with different source types
<SoftwareSourceBadge source="apps" />
<SoftwareSourceBadge source="chrome_extensions" />
<SoftwareSourceBadge source="python_packages" />

// With custom styling
<SoftwareSourceBadge 
  source="homebrew_packages" 
  className="ml-2" 
/>

// Platform-specific examples
<SoftwareSourceBadge source="chocolatey_packages" /> // Windows
<SoftwareSourceBadge source="apt_sources" />        // Linux/Debian
<SoftwareSourceBadge source="homebrew_packages" />  // macOS
```

The component automatically selects appropriate icons (MacOSIcon, WindowsIcon, LinuxIcon, PackageIcon) based on the source type and provides fallback handling for unknown sources.
