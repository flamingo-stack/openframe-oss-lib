A React component that renders a visual badge displaying shell/script types with their corresponding icons and labels, specifically designed for Tactical RMM shell type identification.

## Key Components

- **ShellTypeBadge**: Main component that accepts a `shellType` prop and renders an appropriate icon-text badge
- **ShellTypeBadgeProps**: TypeScript interface defining component props (`shellType`, optional `className`)
- **renderIcon()**: Internal method that maps shell types to their respective icon components
- **Icon Components**: Imports for PowerShell, CMD, Bash, Python, Nushell, Deno, and generic Shell icons

## Usage Example

```typescript
import { ShellTypeBadge } from './components/ShellTypeBadge'

// Display different shell type badges
function ScriptList() {
  return (
    <div className="flex gap-2">
      <ShellTypeBadge shellType="powershell" />
      <ShellTypeBadge shellType="bash" />
      <ShellTypeBadge shellType="python" className="custom-styling" />
    </div>
  )
}

// In a table or list context
function ScriptRow({ script }) {
  return (
    <tr>
      <td>{script.name}</td>
      <td>
        <ShellTypeBadge shellType={script.shellType} />
      </td>
    </tr>
  )
}
```

The component automatically normalizes shell type strings to uppercase and provides fallback to a generic shell icon for unrecognized types. It leverages the existing Button component with "table-display" variant for consistent styling across the application.
