React component that displays a badge for shell/script types with appropriate icons and labels, supporting all Tactical RMM shell types including PowerShell, CMD, Bash, Python, Nushell, Deno, and generic Shell.

## Key Components

- **`ShellTypeBadge`** - Main React component that renders shell type badge with icon and label
- **`shellIconMap`** - Typed configuration mapping shell types to their corresponding icons and styling
- **`ShellIconConfig`** - Interface defining icon component structure with size and styling props
- **`ShellTypeBadgeProps`** - Component props interface accepting shell type and optional className

## Usage Example

```typescript
import { ShellTypeBadge } from './components/ShellTypeBadge'
import { ShellTypeValues } from './types/shell.types'

// Basic usage with PowerShell
<ShellTypeBadge shellType={ShellTypeValues.POWERSHELL} />

// With custom styling
<ShellTypeBadge 
  shellType={ShellTypeValues.PYTHON}
  className="bg-gray-100 p-2 rounded"
/>

// Dynamic shell type from data
<ShellTypeBadge shellType={script.shellType} />
```

The component automatically normalizes shell types to uppercase, displays appropriate brand icons (PowerShell, Python) or generic icons for other types, and falls back to a default shell icon for unrecognized types. Each icon is configured with specific sizing and color properties for consistent visual presentation across different shell types.
