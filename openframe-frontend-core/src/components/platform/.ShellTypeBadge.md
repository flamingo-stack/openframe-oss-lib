A React component that renders a badge displaying shell/script types with corresponding icons and labels, supporting all Tactical RMM shell types including PowerShell, CMD, Bash, Python, Nushell, Deno, and Shell.

## Key Components

- **ShellTypeBadge**: Main component that displays a shell type badge with icon and label
- **ShellTypeBadgeProps**: TypeScript interface defining component props
- **renderIcon()**: Internal function that maps shell types to their appropriate icons

## Usage Example

```typescript
import { ShellTypeBadge } from './ShellTypeBadge'

// Basic usage with different shell types
<ShellTypeBadge shellType="POWERSHELL" />
<ShellTypeBadge shellType="bash" />
<ShellTypeBadge shellType="python" />

// With custom styling
<ShellTypeBadge 
  shellType="CMD" 
  className="custom-badge-styles" 
/>

// In a table or list context
const scripts = [
  { id: 1, name: "Deploy Script", type: "POWERSHELL" },
  { id: 2, name: "Backup Script", type: "BASH" }
]

{scripts.map(script => (
  <ShellTypeBadge 
    key={script.id} 
    shellType={script.type} 
  />
))}
```

The component automatically normalizes shell type strings to uppercase and falls back to a generic shell icon for unrecognized types.
