Displays a badge for a given shell/script type, pairing the appropriate brand icon with a human-readable label. Supports all Tactical RMM shell types.

## Key Components

**`ShellTypeBadge`** (`React.FC<ShellTypeBadgeProps>`)
Main exported component. Renders an icon + label pair for a given shell type.

**`ShellIconConfig`** (interface)
Describes the shape of each entry in `shellIconMap`, typing the icon component and its props.

**`shellIconMap`** (`Record<ShellType, ShellIconConfig>`)
Exhaustive typed mapping of every `ShellType` value to its icon component and display props. Falls back to `defaultIconConfig` (generic `ShellIcon`) for unrecognized types.

**Props — `ShellTypeBadgeProps`**

| Prop | Type | Description |
|---|---|---|
| `shellType` | `ShellType` | The shell/script type to display |
| `className` | `string?` | Additional classes for the wrapper `div` |
| `iconClassName` | `string?` | Additional classes merged onto the icon |

**Supported shell types:** `POWERSHELL` · `CMD` · `BASH` · `PYTHON` · `NUSHELL` · `DENO` · `SHELL`

## Usage Example

```typescript
import { ShellTypeBadge } from './ShellTypeBadge'
import { ShellTypeValues } from '../../types/shell.types'

// Basic usage
<ShellTypeBadge shellType={ShellTypeValues.POWERSHELL} />

// With custom wrapper and icon classes
<ShellTypeBadge
  shellType={ShellTypeValues.PYTHON}
  className="px-2 py-1 rounded bg-ods-surface-secondary"
  iconClassName="opacity-80"
/>
```

> Input `shellType` is normalized to uppercase before lookup, so lowercase string values from API responses are handled safely.
