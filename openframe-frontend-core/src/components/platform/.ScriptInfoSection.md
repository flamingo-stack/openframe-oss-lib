This React component renders script information in a structured card layout with responsive design that adapts from mobile to desktop viewing.

## Key Components

- **ScriptInfoSection** - Main component that displays script metadata in a card format
- **InfoCell** - Internal component for rendering label-value pairs with optional avatar
- **ScriptAuthor** - Interface defining author information (name, initials, photo)
- **ScriptInfoSectionProps** - Props interface for the main component
- **formatSupportedPlatforms()** - Utility function to format platform arrays into readable strings
- **getInitials()** - Helper function to extract initials from full names

## Usage Example

```typescript
import { ScriptInfoSection } from './ScriptInfoSection'

function ScriptDetails() {
  return (
    <ScriptInfoSection
      headline="System Backup Script"
      subheadline="Comprehensive Linux system backup with compression and encryption"
      shellType="BASH"
      supportedPlatforms={['linux', 'darwin']}
      category="System Maintenance"
      author={{
        name: 'John Doe',
        photoUrl: '/avatars/john.jpg',
        initials: 'JD'
      }}
      className="max-w-4xl"
    />
  )
}
```

The component features a responsive grid layout: 2x2 grid on mobile/tablet with row dividers, and a single 4-column row on desktop. It includes author avatars with fallback initials, platform formatting using utility functions, and consistent typography styling.
