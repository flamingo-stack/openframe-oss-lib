This file serves as the main export barrel for OpenFrame platform components, providing a centralized interface for device management and system information display components.

## Key Components

**Components:**
- `SoftwareInfo` - Displays software package information and metadata
- `SoftwareSourceBadge` - Shows visual badges for software package sources
- `CveLink` - Renders clickable links for CVE (Common Vulnerabilities and Exposures) references
- `ToolBadge` - Displays categorized badges for development tools
- `ShellTypeBadge` - Shows shell type indicators (bash, zsh, etc.)

**Types:**
- Component prop interfaces for all exported components
- `SoftwareSource` - Enum/union type for software package sources
- `ToolType` - Type definitions for categorizing development tools
- `ShellType` - Type definitions for shell environments

## Usage Example

```typescript
import { 
  SoftwareInfo, 
  SoftwareSourceBadge, 
  CveLink,
  ToolBadge,
  ShellTypeBadge 
} from './platform-components'
import type { 
  SoftwareInfoProps, 
  ToolType, 
  ShellType 
} from './platform-components'

// Display software package information
<SoftwareInfo 
  name="nodejs" 
  version="18.17.0" 
  source="npm" 
/>

// Show tool categories
<ToolBadge type="development" />

// Display shell environment
<ShellTypeBadge type="bash" />
```
