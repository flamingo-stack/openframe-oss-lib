A React component for managing script arguments as dynamic key-value pairs with add/delete functionality. Each argument consists of a key (argument name) and value field, with empty values representing flag-type arguments.

## Key Components

**Main Export:**
- `ScriptArguments` - Main component for argument management

**Types:**
- `ScriptArgument` - Interface defining argument structure (id, key, value)
- `ScriptArgumentsProps` - Component props with callbacks and customization options

**Key Features:**
- Dynamic add/remove arguments with unique IDs
- Key-value input pairs with placeholders
- Delete buttons with trash icons
- Responsive layout with proper labeling
- Disabled state support

## Usage Example

```typescript
import { ScriptArguments, ScriptArgument } from './ScriptArguments'

function MyComponent() {
  const [arguments, setArguments] = useState<ScriptArgument[]>([
    { id: '1', key: 'port', value: '3000' },
    { id: '2', key: 'verbose', value: '' }, // Flag argument
  ])

  return (
    <ScriptArguments
      arguments={arguments}
      onArgumentsChange={setArguments}
      titleLabel="Script Arguments"
      keyPlaceholder="Argument name"
      valuePlaceholder="Value (empty for flags)"
      addButtonLabel="Add New Argument"
    />
  )
}
```

The component automatically generates unique IDs for new arguments and provides intuitive controls for managing command-line style arguments in forms.
