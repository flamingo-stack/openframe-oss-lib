React component for managing script arguments as editable key-value pairs with add/delete functionality. Provides a dynamic interface for users to configure command-line arguments or parameters for script execution.

## Key Components

- **`ScriptArgument`** - Interface defining a script argument with id, key, and value properties
- **`ScriptArgumentsProps`** - Component props including arguments array, change handlers, and customization options
- **`ScriptArguments`** - Main component rendering argument inputs with add/delete controls
- **Event handlers** - `handleKeyChange`, `handleValueChange`, `handleDelete`, `handleAdd` for managing argument state

## Usage Example

```typescript
import { ScriptArguments, ScriptArgument } from './ScriptArguments'

function MyForm() {
  const [arguments, setArguments] = useState<ScriptArgument[]>([
    { id: '1', key: 'port', value: '3000' },
    { id: '2', key: 'verbose', value: '' }, // empty value = flag
  ])

  return (
    <ScriptArguments
      arguments={arguments}
      onArgumentsChange={setArguments}
      titleLabel="Script Parameters"
      keyPlaceholder="Parameter name"
      valuePlaceholder="Parameter value (empty for flags)"
      addButtonLabel="Add Parameter"
      disabled={false}
    />
  )
}
```

The component automatically generates unique IDs for new arguments and supports both valued parameters and flag-style arguments (with empty values).
