A React component for managing dynamic key-value script arguments with add/remove functionality and customizable placeholders.

## Key Components

- **`ScriptArgument`** - Interface defining a script argument with id, key, and value properties
- **`ScriptArgumentsProps`** - Props interface with callbacks, placeholders, and styling options
- **`ScriptArguments`** - Main component that renders argument pairs with input fields and management buttons

## Usage Example

```typescript
import { ScriptArguments, ScriptArgument } from './ScriptArguments'

function MyComponent() {
  const [arguments, setArguments] = useState<ScriptArgument[]>([
    { id: '1', key: 'port', value: '3000' },
    { id: '2', key: 'verbose', value: '' }, // Empty value acts as flag
  ])

  return (
    <ScriptArguments
      arguments={arguments}
      onArgumentsChange={setArguments}
      titleLabel="Script Parameters"
      keyPlaceholder="Parameter name"
      valuePlaceholder="Value (leave empty for flags)"
      addButtonLabel="Add Parameter"
    />
  )
}
```

The component automatically generates unique IDs for new arguments and provides intuitive controls for managing script parameters. Empty values are treated as boolean flags, making it suitable for command-line argument builders.
