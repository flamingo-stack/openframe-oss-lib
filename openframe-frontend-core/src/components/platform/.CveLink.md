A React component that renders CVE (Common Vulnerabilities and Exposures) identifiers as clickable links that redirect to the NIST National Vulnerability Database. Part of the OpenFrame Design System (ODS) platform components.

## Key Components

- **CveLink**: Main component that creates an external link to NIST NVD
- **CveLinkProps**: Interface defining the component's props
- **CustomExternalLinkIcon**: Icon component indicating external link behavior

## Usage Example

```typescript
import { CveLink } from './CveLink'

function SecurityReport() {
  return (
    <div>
      <p>Critical vulnerability found: <CveLink cveId="CVE-2024-1234" /></p>
      
      <p>
        Multiple CVEs identified:
        <CveLink 
          cveId="CVE-2023-5678" 
          className="ml-2 text-red-600" 
        />
      </p>
    </div>
  )
}
```

The component automatically generates NIST NVD URLs, opens links in new tabs with security attributes, and includes hover effects with ODS design tokens. The external link icon provides visual indication that clicking will navigate away from the current site.
