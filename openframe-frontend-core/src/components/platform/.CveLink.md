A React component that renders CVE identifiers as clickable links to the NIST National Vulnerability Database, featuring hover states and accessibility compliance.

## Key Components

- **CveLink**: Main component that displays a CVE ID with an external link icon
- **CveLinkProps**: TypeScript interface defining `cveId` (required) and `className` (optional) props
- **CustomExternalLinkIcon**: External link icon component that indicates the link opens in a new tab

## Usage Example

```typescript
import { CveLink } from './components/CveLink'

// Basic usage
<CveLink cveId="CVE-2024-1234" />

// With custom styling
<CveLink 
  cveId="CVE-2023-5678" 
  className="text-red-500 font-bold" 
/>

// In a vulnerability report
<div>
  <h3>Critical Vulnerability</h3>
  <p>Affected by <CveLink cveId="CVE-2024-0001" /></p>
</div>
```

The component automatically constructs NIST NVD URLs, opens links in new tabs with proper security attributes (`noopener noreferrer`), and includes hover animations with design system colors. It uses monospace font styling to properly display CVE identifiers and maintains accessibility standards for external links.
