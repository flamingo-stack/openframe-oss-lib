A React component that renders CVE (Common Vulnerabilities and Exposures) identifiers as clickable links to the NIST National Vulnerability Database. The component displays the CVE ID with an external link icon and applies hover effects for enhanced user experience.

## Key Components

- **CveLink**: Main component that renders a CVE ID as an external link
- **CveLinkProps**: TypeScript interface defining component props
  - `cveId`: Required CVE identifier string
  - `className`: Optional CSS classes for styling customization

## Usage Example

```typescript
import { CveLink } from './components/CveLink'

// Basic usage
<CveLink cveId="CVE-2024-1234" />

// With custom styling
<CveLink 
  cveId="CVE-2023-5678" 
  className="text-red-500 hover:text-red-700" 
/>

// In a vulnerability report
<div className="vulnerability-item">
  <h3>Critical Security Issue</h3>
  <p>Reference: <CveLink cveId="CVE-2024-9999" /></p>
</div>
```

The component automatically generates NIST NVD URLs, opens links in new tabs with proper security attributes, and includes accessibility features. It uses the OpenFrame Design System (ODS) styling tokens and includes smooth hover transitions for both the text and external link icon.
