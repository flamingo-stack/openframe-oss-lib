A React component that renders CVE identifiers as clickable links to the NIST National Vulnerability Database, complete with an external link icon.

## Key Components

- **CveLink**: Main component that creates an external link to NIST NVD
- **CveLinkProps**: TypeScript interface defining component props
- **CustomExternalLinkIcon**: Icon component indicating external navigation

## Usage Example

```typescript
import { CveLink } from './CveLink'

// Basic usage
<CveLink cveId="CVE-2024-1234" />

// With custom styling
<CveLink 
  cveId="CVE-2023-5678" 
  className="text-red-500 text-lg" 
/>

// In a vulnerability report
<div>
  <h3>Security Advisory</h3>
  <p>Critical vulnerability: <CveLink cveId="CVE-2024-9999" /></p>
</div>
```

The component automatically constructs the NIST URL, opens links in a new tab with proper security attributes, and includes hover effects for enhanced user experience. Styled with OpenFrame Design System tokens for consistent theming.
