A React component that displays software information including name, vendor, and version in a structured layout using the OpenFrame Design System styling.

## Key Components

- **SoftwareInfo**: Main functional component that renders software details
- **SoftwareInfoProps**: TypeScript interface defining component props
  - `name` (required): Software name string
  - `vendor` (optional): Software vendor string  
  - `version` (optional): Software version string
  - `className` (optional): Additional CSS classes

## Usage Example

```typescript
import { SoftwareInfo } from './SoftwareInfo'

// Basic usage with name only
<SoftwareInfo name="Chrome" />

// With vendor information
<SoftwareInfo name="Visual Studio Code" vendor="Microsoft" />

// Complete information with version
<SoftwareInfo 
  name="Node.js" 
  vendor="OpenJS Foundation" 
  version="v18.17.0"
  className="my-custom-class"
/>
```

The component renders the software name prominently, displays the version in a styled badge if provided, and shows vendor information below in secondary text styling. All text uses the DM Sans font family with ODS design tokens for consistent theming.
