A React component for displaying software information including name, vendor, and version in a clean, hierarchical layout as part of the OpenFrame Design System.

## Key Components

**SoftwareInfo** - Main component that renders software details with typography and spacing from the ODS design system

**SoftwareInfoProps** - TypeScript interface defining:
- `name` (required) - Software name displayed prominently
- `vendor` (optional) - Vendor name shown below the software name
- `version` (optional) - Version displayed as a badge next to the name
- `className` (optional) - Additional CSS classes for customization

## Usage Example

```typescript
import { SoftwareInfo } from './components/SoftwareInfo'

// Basic usage with name and vendor
<SoftwareInfo 
  name="Visual Studio Code" 
  vendor="Microsoft" 
/>

// With version badge
<SoftwareInfo 
  name="Node.js" 
  vendor="OpenJS Foundation"
  version="v18.17.0"
/>

// Name only
<SoftwareInfo name="Chrome" />

// With custom styling
<SoftwareInfo 
  name="Docker Desktop"
  vendor="Docker Inc."
  version="4.21.1"
  className="bg-white p-4 rounded-lg"
/>
```

The component uses ODS design tokens for consistent typography (DM Sans font family) and color schemes, with the version displayed as a subtle badge and vendor information appearing below the main software name.
