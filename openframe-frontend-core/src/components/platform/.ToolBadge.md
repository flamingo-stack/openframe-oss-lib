A React component that renders a badge displaying tool type information with an icon, designed for OpenFrame integrated tools in table displays.

## Key Components

- **ToolBadge**: Main component that renders a button-style badge with tool icon and label
- **ToolBadgeProps**: Interface defining the component's props including `toolType` and optional `className`
- **ToolType**: Re-exported type from tool types module for external use

## Usage Example

```typescript
import { ToolBadge } from './components/ToolBadge'

// Display different tool types
<ToolBadge toolType="TACTICAL" />
<ToolBadge toolType="FLEET" />
<ToolBadge toolType="MESHCENTRAL" />

// With custom styling
<ToolBadge 
  toolType="TACTICAL" 
  className="my-custom-class" 
/>

// In a table cell
<td>
  <ToolBadge toolType={device.toolType} />
</td>
```

The component uses a table-display button variant with left-aligned icon and text, automatically fetching the appropriate tool label and icon based on the provided tool type. It's commonly used in data tables to visually identify which management tool is associated with each device or resource.
