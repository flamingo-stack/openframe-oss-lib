A React component that renders a badge displaying the tool type with an associated icon for OpenFrame integrated tools like Tactical RMM, Fleet MDM, and MeshCentral.

## Key Components

- **ToolBadge**: Main component that accepts `toolType` and optional `className` props
- **ToolBadgeProps**: TypeScript interface defining component props
- **ToolType**: Re-exported type from tool types module
- Uses `ToolIcon` component for displaying tool-specific icons
- Uses `getToolLabel()` utility to convert tool type to display label

## Usage Example

```typescript
import { ToolBadge } from './components/ToolBadge'

// Basic usage in a table cell
<ToolBadge toolType="TACTICAL" />

// With custom styling
<ToolBadge 
  toolType="FLEET" 
  className="custom-badge-style" 
/>

// Multiple tool badges
const toolTypes = ["TACTICAL", "FLEET", "MESHCENTRAL"]
{toolTypes.map(tool => 
  <ToolBadge key={tool} toolType={tool} />
)}
```

The component renders as a button with left-aligned icon and label, styled with the `table-display` variant for consistent appearance in data tables.
