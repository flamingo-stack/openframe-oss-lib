A React component that displays a badge with icon and label for OpenFrame integrated tools, providing visual identification of tool sources in UI elements like tables and lists.

## Key Components

- **ToolBadge**: Main component that renders a tool badge with icon and label
- **ToolBadgeProps**: Interface defining the component props including tool type and optional styling
- **ToolType**: Re-exported type from tool.types for tool identification

## Usage Example

```typescript
import { ToolBadge } from './components/ToolBadge'

// Display different tool badges
function ToolList() {
  return (
    <div className="space-y-2">
      <ToolBadge toolType="TACTICAL" />
      <ToolBadge toolType="FLEET" />
      <ToolBadge toolType="MESHCENTRAL" />
      <ToolBadge toolType="CUSTOM" className="opacity-75" />
    </div>
  )
}

// In a table cell
function DeviceTable() {
  return (
    <td>
      <ToolBadge toolType="TACTICAL" />
    </td>
  )
}
```

The component automatically handles icon selection and label formatting through utility functions, with responsive text sizing and flexible styling through className props. It's commonly used to identify data sources across the OpenFrame platform.
