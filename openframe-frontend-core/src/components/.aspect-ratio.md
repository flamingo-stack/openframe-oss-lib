A wrapper component that maintains a specific width-to-height ratio for its content, built on top of Radix UI's AspectRatio primitive.

## Key Components

- **AspectRatio**: The main component that constrains child content to a specified aspect ratio
- Re-exports `@radix-ui/react-aspect-ratio` Root component with client-side rendering

## Usage Example

```typescript
import { AspectRatio } from "./aspect-ratio"

// 16:9 video aspect ratio
<AspectRatio ratio={16 / 9} className="bg-muted">
  <img
    src="/placeholder.jpg"
    alt="Photo"
    className="rounded-md object-cover w-full h-full"
  />
</AspectRatio>

// Square aspect ratio for avatars
<AspectRatio ratio={1} className="w-32">
  <div className="bg-gradient-to-br from-blue-400 to-purple-600 rounded-full" />
</AspectRatio>

// Wide banner format
<AspectRatio ratio={21 / 9}>
  <video controls className="w-full h-full">
    <source src="/video.mp4" type="video/mp4" />
  </video>
</AspectRatio>
```

The component is particularly useful for responsive images, videos, and other media that need to maintain consistent proportions across different screen sizes.
