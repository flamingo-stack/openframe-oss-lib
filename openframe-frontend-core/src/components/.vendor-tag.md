A React component that displays categorized vendor tags with icons and styling for different vendor types like open-source, commercial, free, AI-selected, and more.

## Key Components

- **VendorTag**: Main component that renders styled tags with icons and text
- **VendorTagProps**: TypeScript interface defining component properties including type, text, className, size, hidden state, and accent color
- **getTagContent()**: Internal function that returns appropriate icon, text, and styling based on tag type

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

// Basic vendor tags
<VendorTag type="open-source" />
<VendorTag type="commercial" text="Enterprise Vendor" />
<VendorTag type="ai" size="sm" />

// Custom styling and colors
<VendorTag 
  type="recommended" 
  text="Staff Pick"
  className="ml-2"
  accentColor="#ff6b35"
/>

// Classification with dynamic content
<VendorTag 
  type="classification" 
  text="openframe_selected"
/>

// Hidden placeholder tag
<VendorTag 
  type="placeholder" 
  hidden={!isLoading}
/>
```

The component supports 12 different tag types including open-source, commercial, free/freemium/paid tiers, AI vs manual selection, recommendations, and special classifications. Each type has predefined icons and styling that can be customized with props.
