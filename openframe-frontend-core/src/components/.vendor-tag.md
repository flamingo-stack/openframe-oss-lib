A reusable React component that renders styled classification tags for vendors and tools in the OpenFrame platform, supporting various license types, selection methods, and vendor classifications.

## Key Components

### `VendorTagProps`
| Prop | Type | Description |
|------|------|-------------|
| `type` | `union` | Tag variant — see supported types below |
| `text` | `string?` | Override default display text |
| `className` | `string?` | Additional CSS classes |
| `size` | `'sm' \| 'md'` | Controls padding and font size (default: `'md'`) |
| `hidden` | `boolean?` | Renders tag invisibly (preserves layout space) |
| `accentColor` | `string?` | Custom accent color override for the icon badge |

### Supported `type` values

| Type | Default Text | Icon |
|------|-------------|------|
| `open-source` | "Open Source" | `OpenSourceIcon` on accent bg |
| `commercial` | "Commercial Vendor" | `CoinsIcon` on border bg |
| `free` / `freemium` | "Free" / "Freemium" | `$` on accent bg |
| `paid` | "Paid" | `CoinsIcon` on border bg |
| `enterprise` | "Enterprise" | `E` on accent bg |
| `recommended` | "Recommended" | `★` on accent bg |
| `ai` | "AI Selected" | `Sparkles` icon |
| `manual` | "Manually Selected" | `Hand` icon |
| `openframe_selected` | "OpenFrame Selected" | `OpenFrameLogo` |
| `classification` | Delegates to sub-type via `text` prop | Varies |
| `placeholder` | "Placeholder" | `Sparkles` icon |

### `VendorTag`
Main exported component. The `classification` type uses the `text` prop to further resolve into `open_source`, `commercial`, or `openframe_selected` sub-variants.

## Usage Example

```typescript
import { VendorTag } from "./vendor-tag"

// Basic open-source tag
<VendorTag type="open-source" />

// Small AI-selected tag with custom class
<VendorTag type="ai" size="sm" className="ml-2" />

// Classification tag delegating to open_source sub-type
<VendorTag type="classification" text="open_source" />

// OpenFrame selected with custom accent color
<VendorTag type="open-source" accentColor="#FFC008" text="Recommended OSS" />

// Hidden tag (preserves layout space)
<VendorTag type="recommended" hidden />
```
