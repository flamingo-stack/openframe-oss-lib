A React component that renders standardized vendor/classification tags with icons for different software types and pricing models in the OpenFrame platform.

## Key Components

**VendorTag Component**
- Main component that displays categorized tags with appropriate icons and styling
- Supports multiple tag types: `open-source`, `commercial`, `free`, `freemium`, `paid`, `enterprise`, `recommended`, `classification`, `ai`, `manual`, `openframe_selected`, and `placeholder`
- Configurable size (`sm` or `md`) and custom styling options
- Dynamic icon rendering based on tag type (Sparkles for AI, Hand for manual, OpenFrame logo for selected items)

**VendorTagProps Interface**
- Defines the component's configuration options including type, custom text, styling, and visibility controls
- Supports accent color customization for open-source tags

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

// Basic usage with different tag types
<VendorTag type="open-source" />
<VendorTag type="commercial" text="Enterprise Solution" />
<VendorTag type="ai" size="sm" />

// Custom styling and accent color
<VendorTag 
  type="open-source" 
  text="Community Driven" 
  accentColor="#00ff00"
  className="ml-2" 
/>

// Classification with dynamic content
<VendorTag type="classification" text="openframe_selected" />
<VendorTag type="openframe_selected" size="md" />
```

The component automatically handles icon selection, color schemes, and text display based on the tag type, providing consistent visual indicators for software categorization across the platform.
