A client-side React component that renders styled classification badges for vendors and tools, supporting multiple tag types (licensing, AI selection, OpenFrame status, etc.) with configurable size and appearance.

## Key Components

### `VendorTagProps` Interface
| Prop | Type | Description |
|------|------|-------------|
| `type` | union string | Tag category — see supported types below |
| `text` | `string` | Override default label text |
| `className` | `string` | Additional CSS classes |
| `size` | `'sm' \| 'md'` | Controls padding and font size |
| `hidden` | `boolean` | Renders invisibly (preserves layout space) |
| `accentColor` | `string` | Custom background color for the icon badge (open-source only) |

### Supported `type` Values
| Type | Default Label |
|------|--------------|
| `open-source` | Open Source |
| `commercial` | Commercial Vendor |
| `free` / `freemium` | Free / Freemium |
| `paid` | Paid |
| `enterprise` | Enterprise |
| `recommended` | Recommended |
| `ai` | AI Selected |
| `manual` | Manually Selected |
| `openframe_selected` | OpenFrame Selected |
| `classification` | Dynamic — resolves based on `text` value |
| `placeholder` | Placeholder |

### `VendorTag` Function
Main export. Uses `getTagContent()` internally to map `type` → `{ text, icon, textColor }` then renders a flex container with an icon badge and a monospace uppercase label.

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

// Basic open-source badge
<VendorTag type="open-source" />

// AI-selected badge, small size
<VendorTag type="ai" size="sm" />

// Custom accent color for open-source
<VendorTag type="open-source" accentColor="#4ECDC4" />

// Classification resolved from text
<VendorTag type="classification" text="openframe_selected" />

// Hidden (invisible but still occupies space)
<VendorTag type="recommended" hidden />
```
