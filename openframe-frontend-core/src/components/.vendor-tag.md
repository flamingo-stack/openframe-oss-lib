A React component that renders categorization tags for vendor types, licensing models, and selection methods with consistent styling and iconography.

## Key Components

- **VendorTag** - Main component that renders styled tags based on type
- **VendorTagProps** - TypeScript interface defining component properties
- **getTagContent()** - Internal function that returns appropriate text, icon, and styling for each tag type

## Supported Tag Types

The component supports 12 different tag types:

- **Licensing**: `open-source`, `commercial`, `free`, `freemium`, `paid`, `enterprise`
- **Status**: `recommended`, `openframe_selected` 
- **Selection**: `ai`, `manual`, `classification`
- **Development**: `placeholder`

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

// Basic usage
<VendorTag type="open-source" />

// With custom text and styling
<VendorTag 
  type="commercial" 
  text="Premium Vendor"
  size="sm"
  className="ml-2"
/>

// With accent color override
<VendorTag 
  type="open-source"
  accentColor="#00ff00"
  hidden={false}
/>

// Classification with dynamic content
<VendorTag 
  type="classification"
  text="openframe_selected"
/>
```

The component uses a dark theme with configurable accent colors, responsive sizing (sm/md), and integrates with the OpenFrame design system through ODS CSS classes and custom iconography.
