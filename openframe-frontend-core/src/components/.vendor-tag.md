A React component that renders categorized vendor tags with icons and styling for different software types and classifications.

## Key Components

- **VendorTag**: Main component that displays a styled tag with icon and text based on vendor type
- **VendorTagProps**: Interface defining props including type, text, styling options, and size variants
- **getTagContent()**: Internal function that maps vendor types to their corresponding icons, colors, and display text

## Supported Tag Types

The component supports various vendor classifications:
- **Licensing**: `open-source`, `commercial`, `free`, `freemium`, `paid`, `enterprise`
- **Selection**: `ai`, `manual`, `openframe_selected`, `recommended`
- **Generic**: `classification`, `placeholder`

Each type has predefined styling, icons, and color schemes with support for custom accent colors and text overrides.

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

// Basic open source tag
<VendorTag type="open-source" />

// Custom commercial tag with accent color
<VendorTag 
  type="commercial" 
  text="Enterprise Solution" 
  accentColor="#ff6b35"
  size="sm"
/>

// AI-selected tag with custom styling
<VendorTag 
  type="ai"
  className="ml-2"
  hidden={!showTags}
/>

// Classification tag with dynamic content
<VendorTag 
  type="classification"
  text="openframe_selected"
/>
```

The component uses Tailwind CSS classes with a dark theme design system (`ods-*` custom colors) and includes responsive sizing options.
