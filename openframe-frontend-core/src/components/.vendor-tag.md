A React component that renders contextual tags to categorize and label different types of vendors, services, or classifications with appropriate icons and styling.

## Key Components

- **VendorTag** - Main component that renders a labeled tag with icon based on type
- **VendorTagProps** - TypeScript interface defining component props including type, text, size, and styling options
- **getTagContent()** - Internal function that maps tag types to their display text, icons, and colors

## Usage Example

```typescript
import { VendorTag } from './vendor-tag'

function ServiceList() {
  return (
    <div className="flex gap-2">
      {/* Basic usage with predefined types */}
      <VendorTag type="open-source" />
      <VendorTag type="commercial" />
      <VendorTag type="freemium" />
      
      {/* Custom text and styling */}
      <VendorTag 
        type="recommended" 
        text="Staff Pick" 
        size="sm"
        accentColor="#ff6b35"
      />
      
      {/* Classification with dynamic content */}
      <VendorTag 
        type="classification" 
        text="openframe_selected" 
      />
      
      {/* AI vs manual selection indicators */}
      <VendorTag type="ai" />
      <VendorTag type="manual" />
    </div>
  )
}
```

The component supports 12 different tag types including pricing models (free, paid, enterprise), selection methods (AI, manual), and special classifications (OpenFrame selected). Each type has predefined icons, colors, and text that can be customized through props.
