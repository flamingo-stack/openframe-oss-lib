A React progress bar component built on top of Radix UI primitives, providing a customizable visual indicator for completion status or loading states.

## Key Components

- **Progress**: A forwardRef component that wraps `@radix-ui/react-progress`
- **Props**: Extends Radix Progress props with optional `indicatorClassName` for styling the progress indicator
- **Styling**: Uses Tailwind classes with `cn` utility for conditional styling

## Usage Example

```typescript
import { Progress } from "./components/ui/progress"

// Basic usage
<Progress value={75} />

// With custom styling
<Progress 
  value={50} 
  className="h-4 bg-gray-200" 
  indicatorClassName="bg-green-500"
/>

// Controlled progress state
const [progress, setProgress] = useState(0)

useEffect(() => {
  const timer = setTimeout(() => setProgress(66), 500)
  return () => clearTimeout(timer)
}, [])

<Progress value={progress} />
```

The component automatically handles the visual transformation of the indicator based on the `value` prop (0-100), using CSS transforms for smooth animations.
