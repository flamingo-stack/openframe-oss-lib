A React component that renders a customizable X/close icon as an SVG element with a circular background.

## Key Components

- **XIcon**: Main component export that renders an X icon
- **XIconProps**: TypeScript interface defining component props
  - `width?: number` - Icon width (default: 16px)
  - `height?: number` - Icon height (default: 16px) 
  - `className?: string` - Additional CSS classes

## Usage Example

```typescript
import { XIcon } from './x-icon';

// Basic usage with defaults
<XIcon />

// Custom size
<XIcon width={24} height={24} />

// With custom styling
<XIcon 
  width={20} 
  height={20} 
  className="hover:opacity-75 cursor-pointer" 
/>

// As a close button
<button onClick={handleClose} className="p-2">
  <XIcon width={18} height={18} />
</button>
```

The icon features a white X symbol on a circular background, making it suitable for close buttons, modal overlays, and other dismissal interactions in UI components.
