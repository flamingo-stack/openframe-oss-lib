A customizable SVG icon component representing a legal scale/balance symbol, commonly used for license or legal-related UI elements.

## Key Components

- **`CustomLicenseIconProps`** - Interface defining the component's optional props
- **`CustomLicenseIcon`** - React functional component that renders an SVG of scales/balance symbol
- **SVG Paths** - Two path elements creating the scale icon: the central pole/base and the two balance scales

## Usage Example

```typescript
import { CustomLicenseIcon } from './custom-license-icon';

// Basic usage with defaults
<CustomLicenseIcon />

// Customized styling
<CustomLicenseIcon 
  className="license-icon"
  fill="#2563eb"
  width={32}
  height={32}
/>

// With additional props
<CustomLicenseIcon 
  fill="currentColor"
  className="text-gray-600 hover:text-blue-500"
  role="img"
  aria-label="License information"
/>
```

The component accepts standard SVG props through spread operators, making it highly flexible for styling and accessibility attributes. Default values provide a gray 24x24 pixel icon suitable for most UI contexts.
