A customizable React SVG icon component that renders a license or certificate symbol with a scales of justice design.

## Key Components

- **CustomLicenseIconProps**: Interface defining component props including styling and size options
- **CustomLicenseIcon**: Main React component that renders an SVG icon with customizable appearance

## Usage Example

```typescript
import { CustomLicenseIcon } from './custom-license-icon';

// Basic usage with defaults
<CustomLicenseIcon />

// Customized with different size and color
<CustomLicenseIcon 
  width={32} 
  height={32} 
  fill="#007bff" 
  className="license-icon"
/>

// With additional props passed through
<CustomLicenseIcon 
  onClick={handleClick}
  aria-label="License information"
  fill="#28a745"
/>
```

The component accepts standard SVG props through spread operator, making it flexible for various use cases. Default dimensions are 24x24 pixels with a gray fill color (#888888).
