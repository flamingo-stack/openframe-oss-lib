A reusable React component that displays vendor information in a compact card format with expandable content, pricing details, and interactive elements.

## Key Components

**Main Export:**
- `VendorCompactCard` - Primary component for rendering vendor cards

**Key Props:**
- `vendor` - Vendor data including title, logo, classifications, and pricing
- `score` - Optional scoring information (score/total format)
- `isExpanded/showChevron` - Controls for expandable UI state
- `showPricing` - Toggles pricing information display
- `isEditableText` - Enables inline text editing
- `rightAddon/stackedInfoTop/stackedInfoBottom` - Custom content slots
- `linkToVendor` - Enables click-to-navigate functionality

**Conditional Rendering:**
- Empty state with OpenMSP logo for placeholder cards
- Different pricing displays for open-source vs commercial vendors
- Editable text input when `isEditableText` is true
- Chevron and X buttons based on expansion state

## Usage Example

```typescript
import { VendorCompactCard } from './vendor-compact-card';

// Basic usage
<VendorCompactCard
  vendor={{
    title: "ExampleCorp",
    slug: "examplecorp",
    logo: "/logos/example.png",
    vendor_classification: [{ classification: 'commercial' }]
  }}
  score={{ score: 85, total: 100 }}
  showPricing={true}
  onChevronClick={(e) => toggleExpansion()}
  linkToVendor={true}
/>

// With custom content
<VendorCompactCard
  vendor={vendorData}
  rightAddon={<CustomBadge />}
  stackedInfoTop={<PricingInfo />}
  subtitle="Security Platform"
  isEditableText={editMode}
  onEditableChange={handleNameChange}
/>
```
