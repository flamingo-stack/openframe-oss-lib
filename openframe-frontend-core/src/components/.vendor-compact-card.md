A React component that renders a compact vendor card display with vendor information, pricing, and interactive controls for comparison tables and vendor selection interfaces.

## Key Components

- **VendorCompactCard**: Main component that displays vendor information in a horizontal layout
- **VendorIcon/EmptyVendorIcon**: Vendor logo display with fallback states
- **ChevronButton/XButton**: Interactive controls for expanding/collapsing and removing items
- **PricingDisplay**: Formatted pricing information for commercial vendors
- **OpenmspLogo**: Logo component for empty state display

## Usage Example

```typescript
import { VendorCompactCard } from './vendor-compact-card';

// Basic vendor card
<VendorCompactCard
  vendor={{
    id: 1,
    title: "Example Vendor",
    slug: "example-vendor",
    logo: "/logos/example.png",
    vendor_classification: [{ classification: 'commercial' }],
    vendor_pricing: [{
      model: "per_user",
      price: 29,
      unit: "user",
      billing_cycle: "monthly"
    }]
  }}
  showPricing={true}
  linkToVendor={true}
/>

// Expandable card with controls
<VendorCompactCard
  vendor={vendor}
  isExpanded={true}
  onChevronClick={(e) => setExpanded(!expanded)}
  onXClick={(e) => removeVendor(vendor.id)}
  score={{ score: 8, total: 10 }}
  rightAddon={<CustomBadge />}
/>

// Editable text mode
<VendorCompactCard
  vendor={vendor}
  isEditableText={true}
  editableValue={inputValue}
  onEditableChange={setInputValue}
  onEditableKeyDown={handleKeyDown}
/>
```

The component supports various display modes including comparison context, empty states, editable text input, and flexible content layout with optional addons and stacked information sections.
