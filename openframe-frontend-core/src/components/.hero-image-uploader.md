A reusable React component for uploading and managing hero-style cover images with drag-and-drop support, validation, and customizable upload handling.

## Key Components

- **`HeroImageUploader`** - Main component with image upload, preview, and removal functionality
- **Props Interface** - `HeroImageUploaderProps` defining all configuration options including:
  - `onChange` - Callback for image URL changes
  - `uploadEndpoint` - API endpoint for uploads
  - `deferUpload` - Option to return base64 preview instead of uploading
  - `onUpload`/`onDelete` - Custom handlers for authenticated operations

## Usage Example

```typescript
import { HeroImageUploader } from './hero-image-uploader';

function BlogEditor() {
  const [heroImage, setHeroImage] = useState<string>();

  return (
    <HeroImageUploader
      imageUrl={heroImage}
      onChange={setHeroImage}
      uploadEndpoint="/api/upload"
      height={400}
      objectFit="cover"
      showReplaceButton={true}
    />
  );
}

// For deferred uploads (unauthenticated flows)
<HeroImageUploader
  imageUrl={heroImage}
  onChange={setHeroImage}
  uploadEndpoint="/api/upload"
  deferUpload={true} // Returns base64 data URL
/>

// With custom upload handler
<HeroImageUploader
  imageUrl={heroImage}
  onChange={setHeroImage}
  uploadEndpoint="/api/upload"
  onUpload={async (file) => {
    const formData = new FormData();
    formData.append('file', file);
    const response = await authenticatedFetch('/api/upload', formData);
    return response.url;
  }}
/>
```

The component validates file types (JPEG, PNG, WebP, GIF) and enforces a 5MB size limit with toast notifications for errors.
