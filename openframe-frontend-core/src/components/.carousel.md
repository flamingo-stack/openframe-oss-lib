A comprehensive carousel component built with React and Embla Carousel that provides a fully accessible, keyboard-navigable slider with customizable orientation and navigation controls.

## Key Components

- **Carousel** - Main container component that manages carousel state and context
- **CarouselContent** - Wrapper for carousel slides with overflow handling
- **CarouselItem** - Individual slide component with proper ARIA attributes
- **CarouselPrevious/CarouselNext** - Navigation buttons with automatic enable/disable states
- **useCarousel** - Hook to access carousel context and controls
- **CarouselApi** - Type for accessing the underlying Embla carousel API

## Usage Example

```typescript
import { 
  Carousel, 
  CarouselContent, 
  CarouselItem, 
  CarouselPrevious, 
  CarouselNext 
} from './carousel'

function MyCarousel() {
  return (
    <Carousel 
      opts={{ align: "start", loop: true }}
      orientation="horizontal"
      className="max-w-sm mx-auto"
    >
      <CarouselContent>
        <CarouselItem>
          <div className="p-6 bg-gray-100">Slide 1</div>
        </CarouselItem>
        <CarouselItem>
          <div className="p-6 bg-gray-200">Slide 2</div>
        </CarouselItem>
        <CarouselItem>
          <div className="p-6 bg-gray-300">Slide 3</div>
        </CarouselItem>
      </CarouselContent>
      <CarouselPrevious />
      <CarouselNext />
    </Carousel>
  )
}
```

The component supports both horizontal and vertical orientations, keyboard navigation (arrow keys), and includes accessibility features like ARIA labels and screen reader announcements.
