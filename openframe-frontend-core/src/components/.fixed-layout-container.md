A collection of React container components designed to provide stable layout structures that prevent content shifting during dynamic operations like filtering, searching, and pagination.

## Key Components

- **`FixedLayoutContainer`** - Main wrapper component with basic dark background styling
- **`FixedContentArea`** - Container for dynamic content areas with loading state support
- **`StableControlsContainer`** - Simple wrapper for search and filter controls
- **`StablePaginationContainer`** - Wrapper for pagination with disabled state handling

All components accept a `contentType` prop ('blog' | 'vendor') for content-specific configurations and include className customization support.

## Usage Example

```typescript
import { 
  FixedLayoutContainer, 
  FixedContentArea, 
  StableControlsContainer,
  StablePaginationContainer 
} from './fixed-layout-container'

function ContentPage() {
  const [isLoading, setIsLoading] = useState(false)

  return (
    <FixedLayoutContainer contentType="blog" className="min-h-screen">
      <StableControlsContainer className="p-4">
        <SearchBar />
        <FilterControls />
      </StableControlsContainer>

      <FixedContentArea 
        contentType="blog" 
        isLoading={isLoading}
        className="px-4"
      >
        <ContentGrid />
      </FixedContentArea>

      <StablePaginationContainer className="p-4" disabled={isLoading}>
        <Pagination />
      </StablePaginationContainer>
    </FixedLayoutContainer>
  )
}
```
