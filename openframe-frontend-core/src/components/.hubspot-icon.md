A React component that renders the HubSpot brand logo as an SVG icon with customizable styling and props.

## Key Components

- **HubspotIcon**: Main React functional component that renders the HubSpot logo SVG
- **Props**: Extends `React.SVGProps<SVGSVGElement>` with optional `className` and `fill` properties
- **Default styling**: Uses HubSpot's brand orange color (`#FF7A59`) and inline-block display

## Usage Example

```typescript
import { HubspotIcon } from './hubspot-icon';

// Basic usage
<HubspotIcon />

// With custom size and styling
<HubspotIcon className="h-8 w-8" />

// With custom color
<HubspotIcon fill="#000000" className="h-6 w-6" />

// With additional SVG props
<HubspotIcon 
  className="h-5 w-5" 
  onClick={handleClick}
  aria-label="HubSpot logo"
/>
```

The component automatically applies the `cn` utility for className merging and includes the `inline-block` class by default. All standard SVG props are supported through the spread operator.
