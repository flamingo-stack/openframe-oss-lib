A comprehensive select dropdown component built on top of Radix UI's select primitives with custom styling for dark themes. Provides a complete set of composable select components with consistent visual design and accessibility features.

## Key Components

- **Select** - Root select component wrapper
- **SelectTrigger** - Clickable trigger button with chevron icon
- **SelectContent** - Dropdown content container with animations and scroll buttons
- **SelectItem** - Individual selectable option with check indicator
- **SelectValue** - Displays the selected value in the trigger
- **SelectLabel** - Section labels within the dropdown
- **SelectGroup** - Groups related select items
- **SelectSeparator** - Visual separator between items

## Usage Example

```typescript
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from './select'

function Example() {
  return (
    <Select>
      <SelectTrigger>
        <SelectValue placeholder="Select an option..." />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="option1">Option 1</SelectItem>
        <SelectItem value="option2">Option 2</SelectItem>
        <SelectItem value="option3">Option 3</SelectItem>
      </SelectContent>
    </Select>
  )
}
```

The components feature responsive typography, hover states, focus management, and smooth animations for opening/closing the dropdown.
