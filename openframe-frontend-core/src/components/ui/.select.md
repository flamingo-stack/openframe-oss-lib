A React Select component built with Radix UI primitives, providing a fully accessible dropdown selection interface with custom styling that matches the ODS design system.

## Key Components

- **Select** - Root component that manages selection state
- **SelectTrigger** - Button that opens the dropdown with chevron icon
- **SelectContent** - Dropdown panel with portal rendering and animations
- **SelectItem** - Individual selectable options with check indicators
- **SelectValue** - Displays the currently selected value
- **SelectGroup** - Groups related options together
- **SelectLabel** - Section headers within the dropdown
- **SelectSeparator** - Visual dividers between option groups
- **SelectScrollUpButton/SelectScrollDownButton** - Navigation for long lists

## Usage Example

```typescript
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "./select"

function LanguageSelect() {
  return (
    <Select defaultValue="en">
      <SelectTrigger className="w-48">
        <SelectValue placeholder="Select language" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="en">English</SelectItem>
        <SelectItem value="es">Spanish</SelectItem>
        <SelectItem value="fr">French</SelectItem>
      </SelectContent>
    </Select>
  )
}
```

The component features responsive typography (14px on mobile, 18px on desktop), hover/focus states with accent colors, and automatic width matching between trigger and content. All components are forwardRef-enabled for proper ref handling.
