A comprehensive card component library providing flexible, reusable card elements with consistent styling and accessibility features for React applications.

## Key Components

- **Card**: Base container component with rounded corners, border, and shadow
- **CardHeader**: Top section with padding and vertical spacing for titles/metadata
- **CardTitle**: Large, semibold heading text for card titles
- **CardDescription**: Smaller, muted text for card descriptions
- **CardContent**: Main content area with appropriate padding
- **CardFooter**: Bottom section aligned for actions/buttons
- **CardHorizontal**: Specialized horizontal layout card with icon, title, and description

## Usage Example

```typescript
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, CardHorizontal } from './card'

// Standard vertical card
<Card className="w-[350px]">
  <CardHeader>
    <CardTitle>Project Overview</CardTitle>
    <CardDescription>View project details and metrics</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Your project content goes here...</p>
  </CardContent>
  <CardFooter>
    <button>View Details</button>
  </CardFooter>
</Card>

// Horizontal card for categories
<CardHorizontal 
  icon={<IconComponent />}
  title="Analytics"
  description="Track and analyze performance"
  borderLeft={true}
/>
```

All components use `forwardRef` for proper ref handling and accept standard HTML div attributes with optional className overrides.
