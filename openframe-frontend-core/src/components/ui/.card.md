A collection of reusable card components built with React and TypeScript, providing a flexible foundation for displaying content in bordered containers with consistent styling.

## Key Components

- **Card**: Base container component with rounded borders, shadow, and card background styling
- **CardHeader**: Top section with vertical spacing, typically for titles and descriptions
- **CardTitle**: Large, semibold heading component for card titles
- **CardDescription**: Muted text component for subtitles or descriptions
- **CardContent**: Main content area with padding (excludes top padding for header spacing)
- **CardFooter**: Bottom section with flexbox layout for actions or additional info
- **CardHorizontal**: Specialized horizontal layout card with icon, title, and description

All components are built with `React.forwardRef` for proper ref forwarding and accept standard HTML div attributes. The `cn` utility function allows for easy className merging and customization.

## Usage Example

```typescript
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, CardHorizontal } from './card'

// Standard vertical card
<Card className="max-w-sm">
  <CardHeader>
    <CardTitle>Project Settings</CardTitle>
    <CardDescription>Manage your project configuration</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Your project content goes here...</p>
  </CardContent>
  <CardFooter>
    <button>Save Changes</button>
  </CardFooter>
</Card>

// Horizontal card with icon
<CardHorizontal
  icon={<StarIcon />}
  title="Featured Project"
  description="This project has been highlighted by the community"
  borderLeft={true}
/>
```
