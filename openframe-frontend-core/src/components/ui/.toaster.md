Custom toast notification system built on top of [Sonner](https://sonner.emilkowal.ski/), providing styled toast components with variant support, progress indicators, and an interactive command approval flow.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Toaster` | Component | Root provider; renders Sonner with unstyled base and injected `toast-progress` keyframe animation |
| `ToastCard` | Component | Standard toast with title, description, colored dot indicator, dismiss button, and animated progress bar |
| `CommandApprovalToast` | Component | Expandable toast that surfaces a command string with Approve/Reject actions; defaults to `duration: Infinity` |
| `showToast` | Function | Imperative helper to fire a `ToastCard`; accepts a plain string or `ShowToastOptions` |
| `showCommandApprovalToast` | Function | Imperative helper to fire a `CommandApprovalToast` |
| `ToastVariant` | Type | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` |
| `dotColorByVariant` | Const | Maps each variant to an ODS color class for the status dot |
| `progressColorByVariant` | Const | Maps each variant to an ODS color class for the progress bar |

## Usage Example

```typescript
// 1. Mount the provider once at the app root
import { Toaster } from './toaster'

export default function RootLayout({ children }) {
  return (
    <>
      {children}
      <Toaster position="bottom-right" />
    </>
  )
}

// 2. Fire a simple toast
import { showToast } from './toaster'

showToast({ title: 'Saved', description: 'Changes committed.', variant: 'success' })

// Short form — title only
showToast('Operation complete')

// 3. Fire a command approval toast
import { showCommandApprovalToast } from './toaster'

showCommandApprovalToast({
  command: 'rm -rf /tmp/cache',
  toolType: 'shell',
  onApprove: () => executeCommand(),
  onReject: () => console.log('Rejected'),
})
```

> **Note:** `CommandApprovalToast` defaults to `duration: Infinity` and must be dismissed via Approve, Reject, or the close button. The command panel is collapsed by default and expands via "Show Command"; set `defaultExpanded: true` to open it immediately.
