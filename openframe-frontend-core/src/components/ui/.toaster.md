A React client-side component that provides a customized toast notification system using Sonner with theme-aware styling.

## Key Components

- **Toaster**: Main component that renders the Sonner toast container with custom positioning and styling
- Positioned at bottom-right with custom CSS classes for consistent theming
- Supports different toast types (error, success, warning, info) with distinct visual styles

## Usage Example

```typescript
import { Toaster } from './toaster'
import { toast } from 'sonner'

// Include in your app layout
function Layout({ children }: { children: React.ReactNode }) {
  return (
    <div>
      {children}
      <Toaster />
    </div>
  )
}

// Trigger toasts from anywhere in your app
function MyComponent() {
  const handleSuccess = () => {
    toast.success('Operation completed successfully!')
  }

  const handleError = () => {
    toast.error('Something went wrong')
  }

  return (
    <div>
      <button onClick={handleSuccess}>Success Toast</button>
      <button onClick={handleError}>Error Toast</button>
    </div>
  )
}
```

The component uses ODS (design system) CSS custom properties for consistent theming across toast states, with semi-transparent backgrounds and matching borders for visual hierarchy.
