Custom toast notification system built on top of [Sonner](https://sonner.emilkowal.ski/), providing styled `ToastCard` and `CommandApprovalToast` components with variant theming, progress indicators, and an approval workflow UI.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Toaster` | Component | Drop-in `<Toaster>` wrapper with ODS defaults (`bottom-right`, offset, gap, unstyled base) |
| `ToastCard` | Component | Standard toast with title, description, colored dot, dismiss button, and animated progress bar |
| `CommandApprovalToast` | Component | Expandable toast with approve/reject actions, command preview, and optional `ToolIcon` |
| `showToast` | Function | Imperative helper to fire a `ToastCard` via Sonner's `custom` API |
| `showCommandApprovalToast` | Function | Imperative helper to fire a `CommandApprovalToast` |
| `ToastVariant` | Type | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` |

## Usage Example

**Mount the toaster once at the app root:**

```typescript
import { Toaster } from '@/components/toaster'

export default function RootLayout({ children }) {
  return (
    <>
      {children}
      <Toaster position="bottom-right" />
    </>
  )
}
```

**Fire a simple toast:**

```typescript
import { showToast } from '@/components/toaster'

showToast({ title: 'Saved', description: 'Changes saved successfully.', variant: 'success' })

// Shorthand string form
showToast('Operation complete')
```

**Fire a command approval toast (e.g., Mingo AI tool execution):**

```typescript
import { showCommandApprovalToast } from '@/components/toaster'

showCommandApprovalToast({
  title: 'Tech Required',
  command: 'Restart-Service -Name "Spooler"',
  toolType: 'powershell',
  approvalDescription: 'This will restart the Print Spooler on WORKSTATION-01.',
  onApprove: () => executeCommand(),
  onReject: () => cancelCommand(),
})
```

> **Note:** `CommandApprovalToast` defaults to `duration: Infinity` — it persists until the technician explicitly approves, rejects, or dismisses it.
