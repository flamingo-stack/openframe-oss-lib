A reusable interactive card component that provides consistent hover effects and clickable behavior with customizable accent colors.

## Key Components

- **InteractiveCard**: Main component with forwardRef support
- **InteractiveCardProps**: TypeScript interface defining component props
- **Hover Effects**: Automatic border and text color changes on hover
- **Custom Styling**: Support for custom accent colors via `hoverAccentColor` prop

## Usage Example

```typescript
import { InteractiveCard } from './interactive-card'

// Basic clickable card with default styling
<InteractiveCard onClick={() => navigate('/details')}>
  <h3>Product Title</h3>
  <p>Product description</p>
</InteractiveCard>

// Card with custom hover accent color
<InteractiveCard 
  onClick={handleClick}
  hoverAccentColor="#00D4AA"
  className="p-4 border rounded-lg"
>
  <h3>Custom Accent</h3>
  <p className="text-ods-text-primary">This text changes color on hover</p>
</InteractiveCard>

// Non-interactive card (display only)
<InteractiveCard clickable={false}>
  <h3>Static Content</h3>
  <p>No hover effects</p>
</InteractiveCard>
```

The component automatically applies hover states to h3 elements and primary text, excludes button text from color changes, and provides smooth transitions. It uses the `group` class for coordinated child element styling.
