A React badge component built with class-variance-authority that provides styled inline elements for displaying status indicators, labels, or tags with multiple visual variants.

## Key Components

- **`Badge`** - Main React component that renders a styled div element
- **`badgeVariants`** - CVA configuration defining visual styles for different badge types
- **`BadgeProps`** - TypeScript interface extending HTML div attributes with variant props
- **Variants**: `default`, `secondary`, `destructive`, `outline`, `success`

## Usage Example

```typescript
import { Badge } from "./components/ui/badge"

function StatusDisplay() {
  return (
    <div className="flex gap-2">
      <Badge variant="default">Active</Badge>
      <Badge variant="success">Completed</Badge>
      <Badge variant="destructive">Error</Badge>
      <Badge variant="outline">Pending</Badge>
      <Badge className="custom-class">Custom Badge</Badge>
    </div>
  )
}
```

The component uses Tailwind CSS classes for styling and supports hover states, focus rings, and smooth transitions. Additional CSS classes can be passed via the `className` prop for customization.
