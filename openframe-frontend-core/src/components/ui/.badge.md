A reusable React badge component built with class-variance-authority for consistent styling and variant management. Provides multiple visual styles for status indicators, labels, and categorization elements.

## Key Components

- **`Badge`** - Main React component that renders a styled div element
- **`badgeVariants`** - CVA configuration defining visual variants and base styles
- **`BadgeProps`** - TypeScript interface extending HTML div attributes with variant props

## Usage Example

```typescript
import { Badge } from "./components/ui/badge"

// Basic usage with different variants
<Badge variant="default">Default</Badge>
<Badge variant="secondary">Secondary</Badge>
<Badge variant="destructive">Error</Badge>
<Badge variant="outline">Outline</Badge>
<Badge variant="success">Success</Badge>

// With custom styling
<Badge variant="success" className="text-lg px-4 py-2">
  Large Success Badge
</Badge>

// As status indicator
<Badge variant={user.isActive ? "success" : "destructive"}>
  {user.isActive ? "Active" : "Inactive"}
</Badge>
```

The component uses Tailwind CSS classes and supports all standard HTML div attributes while providing five distinct visual variants for different use cases.
