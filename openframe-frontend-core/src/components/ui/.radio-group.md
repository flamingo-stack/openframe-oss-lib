A styled radio group component built on top of Radix UI primitives, providing accessible radio button functionality with custom styling and focus management.

## Key Components

- **`RadioGroup`** - Container component for radio button groups with grid layout
- **`RadioGroupItem`** - Individual radio button with circular indicator and accessibility features

## Usage Example

```typescript
import { RadioGroup, RadioGroupItem } from "./radio-group"

export function PaymentMethodSelector() {
  return (
    <RadioGroup defaultValue="card" className="flex flex-col space-y-2">
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="card" id="card" />
        <label htmlFor="card">Credit Card</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="paypal" id="paypal" />
        <label htmlFor="paypal">PayPal</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="bank" id="bank" />
        <label htmlFor="bank">Bank Transfer</label>
      </div>
    </RadioGroup>
  )
}
```

The component supports all standard Radix UI RadioGroup props including `value`, `onValueChange`, and `disabled`. The items feature a filled circle indicator when selected and include proper focus ring styling for accessibility.
