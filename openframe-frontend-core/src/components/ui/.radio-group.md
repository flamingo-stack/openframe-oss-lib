A React radio group component built on Radix UI primitives that provides accessible single-selection functionality with custom styling. The component uses Lucide React icons and supports custom CSS classes through a utility function.

## Key Components

- **`RadioGroup`** - The root container component that manages radio button group state and layout
- **`RadioGroupItem`** - Individual radio button component with custom indicator styling
- **Radix UI Integration** - Built on `@radix-ui/react-radio-group` for accessibility and behavior
- **Circle Icon** - Uses Lucide React's Circle component as the selection indicator

## Usage Example

```typescript
import { RadioGroup, RadioGroupItem } from "./radio-group"

function SettingsForm() {
  const [value, setValue] = React.useState("option1")
  
  return (
    <RadioGroup value={value} onValueChange={setValue}>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="option1" id="option1" />
        <label htmlFor="option1">Option 1</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="option2" id="option2" />
        <label htmlFor="option2">Option 2</label>
      </div>
      <div className="flex items-center space-x-2">
        <RadioGroupItem value="option3" id="option3" />
        <label htmlFor="option3">Option 3</label>
      </div>
    </RadioGroup>
  )
}
```

The component provides built-in focus management, keyboard navigation, and disabled state styling while maintaining full accessibility compliance through Radix UI's primitives.
