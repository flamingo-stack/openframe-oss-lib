A React component library for displaying alert messages with customizable variants and structured content areas.

## Key Components

- **Alert** - Main container component with `default` and `destructive` variants, includes icon positioning styles
- **AlertTitle** - Styled heading component for alert titles with medium font weight
- **AlertDescription** - Content wrapper for alert descriptions with relaxed line spacing
- **alertVariants** - CVA configuration defining visual variants and base styles

## Usage Example

```typescript
import { Alert, AlertTitle, AlertDescription } from "./alert"
import { AlertTriangle } from "lucide-react"

// Basic alert
<Alert>
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>
    You can add components to your app using the CLI.
  </AlertDescription>
</Alert>

// Destructive alert with icon
<Alert variant="destructive">
  <AlertTriangle className="h-4 w-4" />
  <AlertTitle>Error</AlertTitle>
  <AlertDescription>
    Your session has expired. Please log in again.
  </AlertDescription>
</Alert>

// Custom styling
<Alert className="border-blue-200">
  <AlertTitle>Custom Alert</AlertTitle>
  <AlertDescription>
    Additional custom styles applied.
  </AlertDescription>
</Alert>
```
