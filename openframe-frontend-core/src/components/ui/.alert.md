A reusable alert component system built with React and class-variance-authority that provides styled notification containers with support for icons, titles, and descriptions.

## Key Components

- **`Alert`** - Main alert container component with variant styling (default, destructive)
- **`AlertTitle`** - Styled heading component for alert titles
- **`AlertDescription`** - Content wrapper for alert body text
- **`alertVariants`** - CVA configuration defining visual variants and icon positioning

## Usage Example

```typescript
import { Alert, AlertTitle, AlertDescription } from "./components/ui/alert"
import { AlertCircle, CheckCircle } from "lucide-react"

// Basic alert
<Alert>
  <AlertCircle className="h-4 w-4" />
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>
    You can add components to your app using the cli.
  </AlertDescription>
</Alert>

// Destructive variant
<Alert variant="destructive">
  <AlertCircle className="h-4 w-4" />
  <AlertTitle>Error</AlertTitle>
  <AlertDescription>
    Your session has expired. Please log in again.
  </AlertDescription>
</Alert>

// Title only
<Alert>
  <CheckCircle className="h-4 w-4" />
  <AlertTitle>Success! Your changes have been saved.</AlertTitle>
</Alert>
```

The component uses CSS selectors to automatically position SVG icons and adjust spacing for child elements, providing a consistent layout regardless of content structure.
