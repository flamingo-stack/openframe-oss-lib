A React component that renders markdown content with syntax highlighting, GitHub-flavored markdown support, and custom styling for code blocks, headings, and other elements.

## Key Components

- **SimpleMarkdownRenderer**: Main component that converts markdown strings to styled JSX
- **SimpleMarkdownRendererProps**: Interface defining required `content` string and optional `className`
- **Custom components**: Styled renderers for code blocks, blockquotes, headings, paragraphs, links, and lists
- **Hydration handling**: Loading state to prevent SSR mismatches

## Usage Example

```typescript
import { SimpleMarkdownRenderer } from './simple-markdown-renderer';

function DocumentViewer({ markdownContent }: { markdownContent: string }) {
  return (
    <div className="prose-container">
      <SimpleMarkdownRenderer 
        content={markdownContent}
        className="custom-markdown-styles"
      />
    </div>
  );
}

// Example with code blocks and formatting
const sampleMarkdown = `
# API Documentation

\`\`\`typescript
interface User {
  id: string;
  name: string;
}
\`\`\`

> This is a **blockquote** with formatting support.

- Feature 1
- Feature 2
`;

<SimpleMarkdownRenderer content={sampleMarkdown} />
```

The component includes built-in support for syntax highlighting, responsive design, and follows GitHub-flavored markdown standards with custom CSS variable theming.
