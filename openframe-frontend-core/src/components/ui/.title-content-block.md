A reusable React component that displays a title-content pair with customizable styling, semantic HTML elements, and consistent spacing using the Azeret Mono font family.

## Key Components

- **`TitleContentBlock`** - Main component that renders a title and content block
- **`TitleContentBlockProps`** - TypeScript interface defining component props
- **`spacingMap`** - Configuration object mapping spacing sizes to Tailwind CSS classes

## Usage Example

```typescript
import { TitleContentBlock } from './title-content-block';

// Basic usage
<TitleContentBlock 
  title="Revenue" 
  content="$2.4M" 
/>

// Advanced customization
<TitleContentBlock
  title="User Growth"
  content={<span>+24% <small>this month</small></span>}
  titleAs="h3"
  contentAs="p"
  spacing="lg"
  titleColor="text-blue-600"
  containerClassName="border p-4 rounded"
/>

// Custom styling
<TitleContentBlock
  title="Status"
  content="Active"
  titleClassName="text-sm font-bold"
  contentClassName="text-2xl text-green-600"
/>
```

The component provides flexible semantic HTML rendering (h1-h6, div, p), three spacing options (sm/md/lg), and extensive styling customization while maintaining consistent typography defaults.
