A flexible React component for displaying title-content pairs with consistent typography and spacing, commonly used for labeled data displays and section headings.

## Key Components

- **`TitleContentBlock`** - Main component that renders a title above content with customizable styling
- **`TitleContentBlockProps`** - TypeScript interface defining all component props including styling and semantic options
- **`spacingMap`** - Internal mapping object for consistent vertical spacing between title and content

## Usage Example

```typescript
import { TitleContentBlock } from './title-content-block';

// Basic usage with default styling
<TitleContentBlock
  title="Project Status"
  content="In Progress"
/>

// Advanced usage with custom styling and semantic elements
<TitleContentBlock
  title="Revenue"
  content="$1,234,567"
  titleAs="h3"
  contentAs="p"
  spacing="lg"
  titleColor="text-blue-600"
  contentClassName="text-green-600 font-bold text-xl"
  containerClassName="p-4 bg-gray-50 rounded"
/>

// Dynamic content example
<TitleContentBlock
  title="User Count"
  content={<span>{userCount.toLocaleString()}</span>}
  spacing="sm"
/>
```

The component uses Azeret Mono font family by default and provides three spacing options (sm/md/lg) with responsive text sizing. Both title and content elements can be rendered as different HTML tags for proper semantic structure.
