A React component that displays service information in a card format with interactive rows featuring copy, reveal, and external link actions.

## Key Components

- **`ServiceCard`** - Main card component that displays service title, subtitle, icon, tag, and data rows
- **`ServiceCardRowItem`** - Individual row component with interactive actions (copy, reveal secrets, open links)
- **`MaskedValue`** - Utility component for showing/hiding secret values with bullet masking
- **Types**: `ServiceCardProps`, `ServiceCardRow`, `ServiceCardRowAction`, `ServiceCardTag`

## Usage Example

```typescript
import { ServiceCard } from './service-card'

const serviceData = {
  title: "Production API",
  subtitle: "Main application endpoint",
  tag: { label: "Active" },
  rows: [
    {
      label: "URL",
      value: "https://api.example.com",
      href: "https://api.example.com"
    },
    {
      label: "API Key",
      value: "sk_live_abc123...",
      isSecret: true,
      monospace: true,
      copyValue: "sk_live_abc123def456"
    },
    {
      label: "Status",
      value: "Healthy",
      actions: { copy: false, reveal: false }
    }
  ]
}

<ServiceCard {...serviceData} />
```

The component automatically provides copy, reveal (for secrets), and external link actions based on row properties, with customizable action controls per row.
