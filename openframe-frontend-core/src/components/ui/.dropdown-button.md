A trigger button component that opens a dropdown menu of actions, combining a labeled area with a rotating chevron indicator into a single clickable surface.

## Key Components

### `DropdownButton`
The main exported component. Renders a card-styled button with two visual zones — a label/icon area and a chevron divider — wrapped in a single `<button>` element that triggers an `ActionsMenuDropdown`.

### `DropdownButtonProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Button label text |
| `icon` | `React.ReactNode` | — | Optional leading icon |
| `items` | `ActionsMenuItem[]` | — | Dropdown menu items |
| `disabled` | `boolean` | — | Disables interaction and dims the button |
| `align` | `'start' \| 'center' \| 'end'` | `'end'` | Dropdown horizontal alignment |
| `side` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'bottom'` | Dropdown preferred side |
| `ariaLabel` | `string` | `label` | Accessible label override |

**Visual behavior:**
- Background lightens on hover and while open (`bg-ods-bg-hover`)
- Chevron rotates 180° when the dropdown is open
- `disabled` blocks open state changes and applies `opacity-50 cursor-not-allowed`

## Usage Example

```typescript
import { DropdownButton } from './dropdown-button'
import { SettingsIcon } from '../icons-v2-generated'

<DropdownButton
  label="Actions"
  icon={<SettingsIcon />}
  align="end"
  side="bottom"
  items={[
    { label: 'Edit', onClick: () => handleEdit() },
    { label: 'Delete', onClick: () => handleDelete(), variant: 'destructive' },
  ]}
/>
```

> Used internally by `PageActions` when a `PageActionButton` has `dropdownItems` configured.
