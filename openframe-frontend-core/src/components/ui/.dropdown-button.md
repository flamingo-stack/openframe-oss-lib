A compound trigger component that combines a labeled button with a trailing chevron to open an `ActionsMenuDropdown`. The button visually separates the label area from the chevron via a border seam, while acting as a single unified click target.

## Key Components

- **`DropdownButton`** — Primary export; a stateful button that manages dropdown open/close state and delegates rendering to `ActionsMenuDropdown`
- **`DropdownButtonProps`** — Interface defining label, optional icon, menu items, positioning (`align`/`side`), and accessibility props
- **`ActionsMenuDropdown`** — Underlying dropdown primitive consumed internally

## Behavior Notes

- Chevron rotates 180° when the menu is open
- Background lightens (`ods-bg-hover`) both on hover and while open
- `disabled` prop blocks pointer events and applies 50% opacity
- Defaults: `align="end"`, `side="bottom"`

## Usage Example

```typescript
import { DropdownButton } from './dropdown-button'
import { SettingsIcon } from '../icons-v2-generated'

<DropdownButton
  label="Actions"
  icon={<SettingsIcon />}
  align="end"
  side="bottom"
  items={[
    { label: 'Edit', onClick: () => handleEdit() },
    { label: 'Delete', onClick: () => handleDelete(), variant: 'danger' },
  ]}
/>
```
