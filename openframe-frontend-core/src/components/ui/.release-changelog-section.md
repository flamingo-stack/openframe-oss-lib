A React component that renders a structured changelog section with entries, displaying titles, descriptions, and optional breaking change indicators.

## Key Components

**ReleaseChangelogSection** - Main component that renders changelog entries with:
- `title` - Section heading text
- `entries` - Array of ChangelogEntry objects containing title and description
- `isBreaking` - Optional flag to style as breaking changes (red text)
- `hideTitle` - Optional flag to hide the section title
- `SimpleMarkdownRenderer` - Component for rendering markdown content

## Usage Example

```typescript
import { ReleaseChangelogSection } from './release-changelog-section';
import { SimpleMarkdownRenderer } from './markdown-renderer';

const changelogEntries = [
  {
    title: "New Authentication System",
    description: "Added **OAuth 2.0** support with social login providers"
  },
  {
    title: "Performance Improvements",
    description: "Reduced load times by 50%"
  }
];

function ChangelogPage() {
  return (
    <ReleaseChangelogSection
      title="New Features"
      entries={changelogEntries}
      isBreaking={false}
      SimpleMarkdownRenderer={SimpleMarkdownRenderer}
    />
  );
}
```

The component automatically handles empty entries by returning null, displays entry counts in badges, and applies consistent styling with border indicators for each changelog item.
