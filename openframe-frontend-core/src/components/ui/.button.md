A flexible React button component with extensive variant and styling options for the Flamingo/OpenFrame design system, built with Radix UI Slot, class-variance-authority, and Next.js integration.

## Key Components

- **`buttonVariants`** - CVA configuration defining 20+ visual variants (primary, secondary, outline, ghost, destructive, success, etc.)
- **`Button`** - Main component with navigation, loading states, and icon support
- **`ButtonProps`** - TypeScript interface extending HTML button attributes with custom props

## Key Features

- **Navigation Support**: Built-in Next.js Link integration with `href` prop
- **Dual Navigation**: `navigateUrl` prop for same-tab navigation with optional new-tab external link
- **Icon Positioning**: `leftIcon`, `rightIcon`, and `centerIcon` support
- **Loading States**: Built-in loading prop with disabled state
- **Responsive Design**: Mobile-first with full-width on mobile by default
- **Accessibility**: Focus-visible states and proper disabled handling

## Usage Example

```typescript
import { Button } from './button'
import { ExternalLink, ChevronRight } from 'lucide-react'

// Primary CTA button
<Button variant="primary" size="lg">
  Get Started
</Button>

// Navigation button with external link option
<Button 
  href="/dashboard"
  showExternalLinkOnHover
  rightIcon={<ChevronRight />}
>
  View Dashboard
</Button>

// Loading state
<Button variant="secondary" loading disabled>
  Processing...
</Button>

// Icon-only button
<Button variant="ghost" size="icon" centerIcon={<ExternalLink />} />

// Flamingo brand variant
<Button variant="flamingo-primary">
  Try Flamingo
</Button>
```
