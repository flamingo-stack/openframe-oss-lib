A comprehensive button component built with Radix UI Slot, class-variance-authority, and Next.js Link integration. It provides extensive styling variants, loading states, icon support, and flexible navigation options for the OpenFrame design system.

## Key Components

- **buttonVariants**: CVA configuration with 20+ visual variants including primary, secondary, ghost, flamingo-themed, and specialized variants for navigation, filters, and device actions
- **ButtonProps**: Extended interface supporting icons, loading states, navigation modes, and accessibility features
- **Button**: Forwardref component with conditional rendering for links, external navigation, and icon positioning

## Usage Example

```typescript
import { Button } from './button'
import { ExternalLink, Search } from 'lucide-react'

// Basic primary button
<Button variant="primary" size="lg">
  Save Changes
</Button>

// Button with icons and loading state
<Button 
  variant="secondary" 
  leftIcon={<Search />}
  rightIcon={<ExternalLink />}
  loading={isSubmitting}
>
  Search Products
</Button>

// Navigation button with external link behavior
<Button 
  variant="ghost-nav"
  href="/docs"
  showExternalLinkOnHover
  openInNewTab
>
  View Documentation
</Button>

// Icon-only button
<Button variant="outline" size="icon" centerIcon={<Search />} />

// Flamingo-themed CTA
<Button variant="flamingo-primary" size="touch">
  Get Started
</Button>
```

The component supports dual navigation modes, responsive sizing, accessibility features, and integrates seamlessly with the ODS (OpenFrame Design System) color tokens and typography system.
