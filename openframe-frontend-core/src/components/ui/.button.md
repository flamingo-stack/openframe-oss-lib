A reusable button component built with Radix UI Slot and class-variance-authority for comprehensive styling variants. Supports both button and link functionality with extensive customization options.

## Key Components

- **Button Component**: Main forwardRef component with TypeScript support
- **buttonVariants**: CVA configuration with 20+ style variants and 10+ size options
- **ButtonProps Interface**: Extends HTMLButtonAttributes with custom props

**Main Variants:**
- `primary`, `secondary`, `white` - Core action buttons
- `outline`, `ghost`, `transparent` - Alternative styles
- `link`, `destructive`, `success` - Specialized variants
- `flamingo-primary`, `flamingo-secondary` - Brand-specific styles
- `filter`, `section`, `device-action` - Component-specific variants

**Key Props:**
- `href` - Renders as Next.js Link when provided
- `leftIcon`, `rightIcon`, `centerIcon` - Icon positioning
- `loading` - Shows spinner state
- `openInNewTab`, `showExternalLinkOnHover` - Link behavior
- `asChild` - Radix Slot composition

## Usage Example

```typescript
import { Button } from "./components/button"
import { Search, ExternalLink } from "lucide-react"

// Primary CTA button
<Button variant="primary" size="lg">
  Get Started
</Button>

// Button with icons
<Button variant="outline" leftIcon={<Search />} rightIcon={<ExternalLink />}>
  Search Products
</Button>

// Link button
<Button 
  href="/products" 
  variant="secondary"
  openInNewTab
  showExternalLinkOnHover
>
  View All Products
</Button>

// Loading state
<Button variant="primary" loading disabled>
  Processing...
</Button>

// Icon-only button
<Button variant="ghost" size="icon" centerIcon={<Search />} />
```
