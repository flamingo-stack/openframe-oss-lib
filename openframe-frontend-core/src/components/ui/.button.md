A flexible Button component built with Radix UI and class-variance-authority that supports multiple variants, sizes, icons, and can render as either a button or link.

## Key Components

- **buttonVariants**: CVA configuration with 20+ style variants including primary, secondary, outline, ghost, and specialized variants for specific use cases
- **ButtonProps**: TypeScript interface extending HTML button attributes with additional props for icons, loading states, and navigation
- **Button**: Polymorphic component that can render as button, link (Next.js), or custom element via Radix Slot

## Usage Example

```typescript
import { Button } from "./components/ui/button"
import { ExternalLink, Plus } from "lucide-react"

// Basic button
<Button variant="primary" size="default">
  Click me
</Button>

// Button with icons
<Button variant="secondary" leftIcon={<Plus />} rightIcon={<ExternalLink />}>
  Add Item
</Button>

// Link button with external navigation
<Button 
  href="/external-link" 
  openInNewTab 
  showExternalLinkOnHover
  variant="outline"
>
  Visit Site
</Button>

// Icon-only button
<Button variant="ghost" size="icon" centerIcon={<Plus />} />

// Loading state
<Button variant="primary" loading disabled>
  Processing...
</Button>

// Custom alignment and full width control
<Button 
  variant="filter" 
  alignment="left" 
  fullWidthOnMobile={false}
>
  Filter Options
</Button>
```

The component automatically handles external link detection, loading states, accessibility features, and responsive design patterns.
