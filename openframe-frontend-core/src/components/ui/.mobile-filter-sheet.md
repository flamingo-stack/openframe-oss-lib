A React component that provides mobile-optimized filtering and sorting interfaces using sheet and modal patterns. This component enables users to apply filters and sort data on mobile devices with an intuitive touch-friendly interface.

## Key Components

- **MobileFilterSheet**: Bottom sheet component with filtering and sorting options
- **MobileFilterModal**: Modal version of the filter interface
- **FilterCheckboxItem**: Individual checkbox item for filter options
- **SortColumnItem**: Sortable column item with directional controls
- **FilterGroup**: Interface for grouping related filter options
- **SortConfig**: Configuration for sortable columns and current sort state

## Usage Example

```typescript
import { MobileFilterSheet, FilterGroup, SortConfig } from './mobile-filter-sheet'

const filterGroups: FilterGroup[] = [
  {
    id: 'status',
    title: 'Status',
    options: [
      { id: 'active', label: 'Active', count: 42 },
      { id: 'inactive', label: 'Inactive', count: 8 }
    ]
  }
]

const sortConfig: SortConfig = {
  columns: [
    { key: 'name', label: 'Name' },
    { key: 'date', label: 'Date Created' }
  ],
  sortBy: 'name',
  sortDirection: 'asc'
}

export function DataTable() {
  const [filterOpen, setFilterOpen] = useState(false)
  const [currentFilters, setCurrentFilters] = useState({})

  return (
    <>
      <button onClick={() => setFilterOpen(true)}>
        Open Filters
      </button>
      
      <MobileFilterSheet
        open={filterOpen}
        onOpenChange={setFilterOpen}
        title="Filter Data"
        filterGroups={filterGroups}
        currentFilters={currentFilters}
        onFilterChange={setCurrentFilters}
        sortConfig={sortConfig}
        onSort={(column, direction) => console.log('Sort:', column, direction)}
        onSortClear={(column) => console.log('Clear sort:', column)}
      />
    </>
  )
}
```

The component supports both sheet (bottom slide-up) and modal presentations, with checkbox-based filtering, three-state sorting (none → asc → desc → none), and customizable styling through the OpenFrame design system tokens.
