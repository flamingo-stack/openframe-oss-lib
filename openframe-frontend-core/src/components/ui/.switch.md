A React switch component built on Radix UI primitives that provides both controlled and uncontrolled functionality with smooth animations and accessible keyboard navigation.

## Key Components

- **Switch**: Main component that renders a toggle switch with customizable styling
- **Props**: Extends Radix Switch props with `checked` and `onCheckedChange` for state management
- **Internal state management**: Handles both controlled and uncontrolled usage patterns

## Usage Example

```typescript
import { Switch } from "./switch"
import { useState } from "react"

// Controlled usage
function ControlledExample() {
  const [enabled, setEnabled] = useState(false)
  
  return (
    <Switch 
      checked={enabled}
      onCheckedChange={setEnabled}
      className="my-custom-class"
    />
  )
}

// Uncontrolled usage
function UncontrolledExample() {
  return (
    <Switch 
      onCheckedChange={(checked) => console.log('Switch changed:', checked)}
    />
  )
}
```

The component automatically handles visual state transitions, applies proper ARIA attributes for accessibility, and supports both controlled and uncontrolled patterns. The switch thumb animates smoothly between positions and changes the background color based on the checked state.
