A React switch/toggle component built on top of Radix UI primitives with support for both controlled and uncontrolled usage patterns. The component provides smooth animations and consistent styling with custom accent colors.

## Key Components

- **Switch**: Main toggle component that extends Radix UI Switch with enhanced styling and state management
- Supports both controlled (`checked` prop) and uncontrolled usage modes
- Built-in state synchronization between internal and external state
- Accessible focus management and keyboard navigation

## Usage Example

```typescript
import { Switch } from "./switch"

// Uncontrolled usage
function BasicToggle() {
  return <Switch />
}

// Controlled usage
function ControlledToggle() {
  const [enabled, setEnabled] = useState(false)
  
  return (
    <Switch 
      checked={enabled}
      onCheckedChange={setEnabled}
    />
  )
}

// With custom styling
function CustomSwitch() {
  return (
    <Switch 
      className="scale-150" 
      onCheckedChange={(checked) => console.log('Toggle:', checked)}
    />
  )
}
```

The component automatically handles the visual transition between states, applying `bg-ods-accent` when checked and `bg-gray-600` when unchecked, with a smooth sliding thumb animation.
