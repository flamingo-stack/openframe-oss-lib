A reusable React component that renders a horizontal collection of icon placeholders with customizable sizing and styling.

## Key Components

- **IconsBlock**: Main component that displays an array of icon placeholders
- **IconsBlockProps**: Interface extending HTML div attributes with:
  - `icons`: Array of strings representing icons to display
  - `size`: Size variant ('sm' | 'md' | 'lg') controlling icon dimensions

## Usage Example

```typescript
import { IconsBlock } from './components/icons-block'

// Basic usage with default medium size
<IconsBlock icons={['icon1', 'icon2', 'icon3']} />

// Small icons with custom styling
<IconsBlock 
  icons={['react', 'typescript', 'tailwind']} 
  size="sm"
  className="justify-center p-4"
/>

// Large icons
<IconsBlock 
  icons={['github', 'linkedin']} 
  size="lg" 
/>
```

The component renders placeholder boxes for each icon in a flex layout. Icons are displayed as rounded containers with a secondary background and internal placeholder content. The size prop controls icon dimensions: small (16px), medium (24px), or large (32px).
