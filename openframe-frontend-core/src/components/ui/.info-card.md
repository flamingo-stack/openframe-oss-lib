A React component that renders structured information in a card format with support for icons, copyable values, dotted leaders, and optional progress indicators.

## Key Components

- **`InfoCardData`** - Interface defining the card's data structure including title, items, and progress
- **`InfoCardProps`** - Component props interface with data and optional className
- **`InfoCard`** - Main component that renders a dark-themed information card with:
  - Optional title with icon
  - Optional subtitle
  - List of label-value pairs with dotted leaders
  - Copyable values with clipboard functionality
  - Optional progress bar at the bottom

## Usage Example

```typescript
import { InfoCard } from './components/info-card'
import { ServerIcon } from './icons'

const cardData = {
  title: "System Status",
  subtitle: "Current metrics",
  icon: <ServerIcon className="w-5 h-5" />,
  items: [
    { label: "Status", value: "Online", copyable: false },
    { label: "IP Address", value: "192.168.1.100", copyable: true },
    { label: "Ports", value: ["80", "443", "8080"], copyable: true }
  ],
  progress: {
    value: 75,
    warningThreshold: 80,
    criticalThreshold: 90,
    inverted: false
  }
}

<InfoCard data={cardData} className="w-full max-w-md" />
```

The component automatically handles array values by displaying multiple rows, shows dotted leaders between labels and values, and provides one-click copying for designated fields.
