A customizable information display card component that presents structured data with optional title, subtitle, progress indicator, and copyable values in a dark-themed UI.

## Key Components

- **InfoCard**: Main component that renders a structured card with dark styling
- **InfoCardData**: Interface defining the card's data structure with title, subtitle, icon, items array, and optional progress
- **InfoCardProps**: Component props interface including data and optional className
- **ProgressBar**: Integrated progress visualization with configurable thresholds

## Usage Example

```typescript
import { InfoCard } from './info-card'
import { ServerIcon } from '../icons'

const serverData = {
  title: "Server Status",
  subtitle: "Production Environment",
  icon: <ServerIcon />,
  items: [
    { label: "Host", value: "server.example.com", copyable: true },
    { label: "Status", value: "Online" },
    { label: "IPs", value: ["192.168.1.1", "10.0.0.1"] }
  ],
  progress: {
    value: 75,
    warningThreshold: 80,
    criticalThreshold: 90,
    inverted: true
  }
}

<InfoCard data={serverData} className="max-w-md" />
```

The component automatically handles array values by displaying them as separate rows, includes clipboard copying for copyable items, and integrates a progress bar with customizable color thresholds.
