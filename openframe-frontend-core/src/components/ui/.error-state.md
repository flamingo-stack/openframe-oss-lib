A comprehensive React component for displaying error states with customizable styling, icons, and action buttons. Includes pre-configured convenience components for common error scenarios.

## Key Components

**Main Component:**
- `ErrorState` - Flexible error display component with configurable variants (error, warning, info)

**Convenience Components:**
- `PageError` - Pre-configured for page-level errors with retry/home actions
- `LoadError` - Pre-configured for loading failures with retry action
- `NotFoundError` - Pre-configured for 404-style errors with home action

**Key Features:**
- Three visual variants with appropriate color schemes
- Optional icon display using Lucide React icons
- Configurable retry and home navigation buttons
- Customizable styling via className props

## Usage Example

```typescript
import { ErrorState, PageError, LoadError } from './error-state'

// Basic error state
<ErrorState 
  message="Something went wrong" 
  variant="error"
/>

// With retry functionality
<ErrorState
  title="Connection Failed"
  message="Unable to connect to server"
  showRetry={true}
  onRetry={() => refetch()}
/>

// Using convenience components
<PageError 
  message="Failed to load page content"
  onRetry={() => window.location.reload()}
  onHome={() => router.push('/')}
/>

<LoadError 
  message="Failed to fetch data"
  onRetry={handleRetry}
/>
```
