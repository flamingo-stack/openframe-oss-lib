A React component for displaying error states with customizable variants, icons, and action buttons for consistent error handling across the application.

## Key Components

- **ErrorState** - Main component with configurable title, message, variant (error/warning/info), and optional retry/home actions
- **PageError** - Convenience component for general page errors with retry and home navigation options
- **LoadError** - Specialized component for loading failures with retry functionality
- **NotFoundError** - Component for 404-style errors with home navigation option

## Usage Example

```typescript
import { ErrorState, PageError, LoadError, NotFoundError } from './error-state'

// Basic error with custom message
<ErrorState 
  message="Something went wrong while processing your request" 
  variant="error"
  showRetry 
  onRetry={() => window.location.reload()}
/>

// Page error with navigation options
<PageError 
  message="Failed to load the dashboard"
  onRetry={() => refetch()}
  onHome={() => router.push('/')}
/>

// Loading error with retry
<LoadError 
  message="Unable to fetch user data"
  onRetry={() => retry()}
/>

// Not found error
<NotFoundError 
  message="The requested project was not found"
  onHome={() => navigate('/projects')}
/>
```

The component uses the OpenFrame Design System (ODS) color tokens and provides consistent styling across error, warning, and info variants with proper accessibility through semantic colors and icons.
