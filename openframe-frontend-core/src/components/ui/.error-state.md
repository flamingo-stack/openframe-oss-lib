A flexible React component for displaying error states with customizable variants, actions, and styling options. The component supports error, warning, and info variants with optional retry and navigation actions.

## Key Components

- **ErrorState** - Main error display component with configurable variant styling
- **PageError** - Convenience component for general page errors  
- **LoadError** - Pre-configured component for loading failures
- **NotFoundError** - Specialized component for 404-style errors
- **ErrorStateProps** - TypeScript interface defining all configuration options

## Usage Example

```typescript
import { ErrorState, PageError, LoadError } from './error-state'

// Basic error display
<ErrorState
  title="Something went wrong"
  message="Unable to load the requested data"
  variant="error"
  showRetry={true}
  onRetry={() => window.location.reload()}
/>

// Using convenience components
<PageError 
  message="Failed to load user profile"
  onRetry={handleRetry}
  onHome={() => navigate('/')}
/>

<LoadError 
  message="Network connection failed"
  onRetry={refetchData}
/>

<NotFoundError onHome={() => navigate('/')} />

// Custom warning variant
<ErrorState
  variant="warning"
  message="Some features may be unavailable"
  showIcon={false}
/>
```

The component uses ODS design tokens for consistent styling and includes built-in accessibility features like semantic markup and proper icon usage.
