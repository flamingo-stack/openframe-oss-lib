A React checkbox component built on top of Radix UI primitives that provides an accessible, customizable checkbox with consistent styling and a check icon indicator.

## Key Components

- **`Checkbox`** - Main checkbox component that forwards refs and accepts all standard Radix checkbox props
- **Styling** - Uses custom CSS classes with `ods-` prefix for theming (border, background, accent colors)
- **Indicator** - Displays a check icon from Lucide React when the checkbox is checked

## Usage Example

```typescript
import { Checkbox } from "./components/ui/checkbox"

function MyForm() {
  const [isChecked, setIsChecked] = React.useState(false)

  return (
    <div className="flex items-center space-x-2">
      <Checkbox 
        id="terms"
        checked={isChecked}
        onCheckedChange={setIsChecked}
      />
      <label htmlFor="terms">Accept terms and conditions</label>
    </div>
  )
}

// With custom styling
<Checkbox 
  className="h-5 w-5" 
  disabled={isLoading}
  defaultChecked={true}
/>
```

The component automatically handles focus states, disabled states, and accessibility features through the underlying Radix primitive.
