A React component that renders a file management toolbar with standard file operations like copy, cut, paste, upload, and folder creation.

## Key Components

- **FileManagerActionBar**: Main component that renders action buttons for file management operations
- **Button Integration**: Uses custom Button component with outline variant and small sizing
- **Icon Integration**: Leverages Lucide React icons for visual button indicators
- **Conditional States**: Handles disabled states for copy/cut (requires selection) and paste (requires clipboard content)

## Usage Example

```typescript

  const [selectedFiles, setSelectedFiles] = useState<string[]>([])
  const [clipboardHasContent, setClipboardHasContent] = useState(false)

        onUpload={() => console.log('Upload files')}
        onSelectAll={() => console.log('Select all files')}
        className="mb-4"
      />
      {/* File list components */}
    </div>
  )
}
```

The component provides a responsive toolbar with wrapped buttons, appropriate disabled states, and consistent styling using Tailwind classes and the cn utility for conditional styling.
