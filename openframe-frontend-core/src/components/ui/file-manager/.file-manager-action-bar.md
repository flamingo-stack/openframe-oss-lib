A React component that renders an action bar with file management controls including folder creation, file operations (copy/cut/paste), upload, and selection functionality.

## Key Components

- **FileManagerActionBar**: Main component that renders a horizontal bar of action buttons
- **Props**: 
  - `canPaste`, `hasSelection`: Boolean flags for button states
  - `onNewFolder`, `onPaste`, `onCopy`, `onCut`, `onUpload`, `onSelectAll`: Action handlers
  - `className`: Optional styling override

## Usage Example

```typescript
import { FileManagerActionBar } from './file-manager-action-bar'

function FileExplorer() {
  const [selectedItems, setSelectedItems] = useState([])
  const [clipboard, setClipboard] = useState(null)

  return (
    <div>
      <FileManagerActionBar
        hasSelection={selectedItems.length > 0}
        canPaste={clipboard !== null}
        onNewFolder={() => createFolder()}
        onCopy={() => copyToClipboard(selectedItems)}
        onCut={() => cutToClipboard(selectedItems)}
        onPaste={() => pasteFromClipboard()}
        onUpload={() => openUploadDialog()}
        onSelectAll={() => selectAllItems()}
        className="mb-4"
      />
      {/* File list component */}
    </div>
  )
}
```
