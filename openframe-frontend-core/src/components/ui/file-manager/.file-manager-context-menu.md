A React component that provides a context menu for file management operations with customizable actions and conditional item visibility based on file type and selection state.

## Key Components

- **FileManagerContextMenu**: Main component that renders a dropdown menu with file operations
- **menuItems**: Configuration array defining available actions (download, copy, rename, cut, delete)
- **Props**: `open`, `onOpenChange`, `onAction`, `fileType`, `hasSelection`, `trigger`, `className`
- **Actions**: 'download' | 'copy' | 'rename' | 'cut' | 'delete'

## Usage Example

```typescript
import { FileManagerContextMenu } from './file-manager-context-menu'

function FileItem() {
  const [contextMenuOpen, setContextMenuOpen] = useState(false)
  
  const handleAction = (action: string) => {
    switch (action) {
      case 'download':
        // Handle download
        break
      case 'delete':
        // Handle delete
        break
      // ... other actions
    }
  }

  return (
    <div>
      <FileManagerContextMenu
        open={contextMenuOpen}
        onOpenChange={setContextMenuOpen}
        onAction={handleAction}
        fileType="file"
        hasSelection={false}
      />
    </div>
  )
}
```

The menu automatically filters items based on context - download only shows for files, rename is hidden when multiple items are selected, and actions are separated logically with visual dividers.
