A React context menu component that provides standard file management operations like download, copy, rename, cut, and delete for file manager interfaces.

## Key Components

- **FileManagerContextMenu**: Main component that renders a dropdown menu with file operation actions
- **menuItems**: Configuration array defining available actions with icons, labels, and visibility conditions
- **Props Interface**: Accepts `FileManagerContextMenuProps` for customization and event handling

## Features

- **Dynamic Actions**: Shows/hides menu items based on file type and selection state
- **Event Prevention**: Prevents menu clicks from bubbling up to parent elements
- **Custom Trigger**: Supports custom trigger elements or defaults to a "more" icon button
- **Conditional Rendering**: Filters menu items based on context (single file vs. multiple selection)
- **Keyboard Accessible**: Built on shadcn/ui dropdown components with proper ARIA support

## Usage Example

```typescript
import { FileManagerContextMenu } from './file-manager-context-menu'

function FileItem({ file, isSelected }) {
  const [menuOpen, setMenuOpen] = useState(false)

  const handleAction = (action: string) => {
    switch (action) {
      case 'download':
        downloadFile(file)
        break
      case 'copy':
        copyToClipboard(file)
        break
      case 'rename':
        setRenaming(true)
        break
      case 'delete':
        deleteFile(file)
        break
    }
    setMenuOpen(false)
  }

  return (
    <div className="file-item">
      <span>{file.name}</span>
      <FileManagerContextMenu
        open={menuOpen}
        onOpenChange={setMenuOpen}
        onAction={handleAction}
        fileType={file.type}
        hasSelection={isSelected}
      />
    </div>
  )
}
```
