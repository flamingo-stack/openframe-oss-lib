This file serves as the main entry point for a file manager component library, providing a centralized export interface for all file management UI components and their associated TypeScript types.

## Key Components

### Components
- **FileManager** - Main file manager container component
- **FileManagerSkeleton** - Loading state placeholder component
- **FileManagerBreadcrumb** - Navigation breadcrumb component
- **FileManagerActionBar** - Toolbar with file actions (create, upload, etc.)
- **FileManagerTable** - File listing table component
- **FileManagerTableRow** - Individual file row component
- **FileManagerContextMenu** - Right-click context menu
- **FileManagerEmpty** - Empty state component
- **FileIcon** - File type icon renderer

### Types
- **FileType, FileAction, FileItem** - Core file system types
- **BreadcrumbItem** - Navigation path item type
- **Component Props** - TypeScript interfaces for all component props

## Usage Example

```typescript
import { 
  FileManager, 
  FileManagerProps, 
  FileItem 
} from './file-manager'

// Use the main file manager component
const files: FileItem[] = [
  { id: '1', name: 'document.pdf', type: 'file', size: 1024 }
]

const MyComponent = () => (
  <FileManager 
    files={files}
    onFileSelect={(file) => console.log(file)}
    onFileAction={(action, files) => console.log(action, files)}
  />
)
```
