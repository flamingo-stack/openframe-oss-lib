This is the main entry point for a comprehensive file manager component library, providing all necessary components and types for building file system interfaces.

## Key Components

**Components:**
- `FileManager` - Main file manager container component
- `FileManagerSkeleton` - Loading state placeholder
- `FileManagerBreadcrumb` - Navigation breadcrumb component
- `FileManagerActionBar` - Toolbar with file actions
- `FileManagerTable` - File listing table
- `FileManagerTableRow` - Individual file/folder row
- `FileManagerContextMenu` - Right-click context menu
- `FileManagerEmpty` - Empty state display
- `FileIcon` - File type icon component

**Types:**
- `FileType`, `FileAction`, `FileItem` - Core file system types
- `BreadcrumbItem` - Navigation item type
- Component prop types for all exported components

## Usage Example

```typescript
import { 
  FileManager, 
  FileManagerBreadcrumb,
  FileManagerActionBar,
  type FileItem 
} from './file-manager'

const MyApp = () => {
  const files: FileItem[] = [
    { id: '1', name: 'document.pdf', type: 'file', size: 1024 }
  ]

  return (
    <div>
      <FileManagerBreadcrumb items={breadcrumbs} />
      <FileManagerActionBar onUpload={handleUpload} />
      <FileManager files={files} onFileSelect={handleSelect} />
    </div>
  )
}
```
