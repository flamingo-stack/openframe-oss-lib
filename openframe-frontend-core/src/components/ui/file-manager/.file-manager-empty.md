A React component that displays an empty state for file managers with a customizable message, description, and optional action button.

## Key Components

- **FileManagerEmpty**: Main component that renders an empty state with folder icon, message, description, and optional action button
- **Props**:
  - `message`: Custom heading text (default: "No files or folders found")
  - `description`: Explanatory text below the heading
  - `action`: Optional object with `label` and `onClick` for action button
  - `className`: Additional CSS classes for styling

## Usage Example

```typescript
import { FileManagerEmpty } from './file-manager-empty'

// Basic usage
<FileManagerEmpty />

// With custom content and action
<FileManagerEmpty 
  message="No documents found"
  description="Upload your first document to get started with file management."
  action={{
    label: "Upload Files",
    onClick: () => handleUpload()
  }}
  className="my-8"
/>

// Minimal usage with just custom message
<FileManagerEmpty 
  message="Folder is empty"
/>
```

The component uses Lucide React icons and follows a design system with ODS (presumably "Our Design System") color tokens for consistent theming.
