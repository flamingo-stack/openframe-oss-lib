A React component that displays an empty state UI for file manager interfaces when no files or folders are present.

## Key Components

- **FileManagerEmpty** - Main component that renders an empty state with icon, message, description, and optional action button
- **Props Interface** - Uses `FileManagerEmptyProps` type for component configuration including custom messages and actions
- **Styling** - Uses CSS utility classes with ODS (design system) tokens for consistent theming
- **Icon Integration** - Leverages Lucide React's `FolderOpen` icon for visual context

## Usage Example

```typescript
import { FileManagerEmpty } from './file-manager-empty'

function FileExplorer() {
  const handleCreateFolder = () => {
    // Logic to create new folder
    console.log('Creating new folder...')
  }

  return (
    <div className="file-explorer">
      <FileManagerEmpty
        message="No documents found"
        description="Upload your first document or create a new folder to organize your files."
        action={{
          label: "Create Folder",
          onClick: handleCreateFolder
        }}
        className="min-h-96"
      />
    </div>
  )
}

// Basic usage without action button
function SimpleEmpty() {
  return (
    <FileManagerEmpty
      message="Folder is empty"
      description="No files have been uploaded to this location yet."
    />
  )
}
```

The component provides a user-friendly empty state that guides users toward taking action, with customizable messaging and optional call-to-action buttons for improved user experience.
