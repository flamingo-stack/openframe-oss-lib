A comprehensive table component for displaying and managing files with selection, loading states, and responsive height management. Handles both file and folder navigation with optional checkbox-based multi-selection.

## Key Components

- **FileManagerTable**: Main table component with header, file rows, and selection controls
- **Loading State**: Skeleton rows with proper column alignment during data fetching  
- **Empty State**: Falls back to `FileManagerEmpty` when no files are present
- **Dynamic Height**: Auto-adjusts table height based on viewport using `ResizeObserver`
- **Selection Logic**: Manages individual and bulk file selection with visual indicators

## Usage Example

```typescript
import { FileManagerTable } from './file-manager-table'

function FileExplorer() {
  const [selectedFiles, setSelectedFiles] = useState<string[]>([])
  
  return (
    <FileManagerTable
      files={fileList}
      selectedFiles={selectedFiles}
      showCheckboxes={true}
      loading={false}
      isSearchResult={false}
      onSelectFile={(id, selected) => {
        setSelectedFiles(prev => 
          selected ? [...prev, id] : prev.filter(f => f !== id)
        )
      }}
      onSelectAll={(selectAll) => {
        setSelectedFiles(selectAll ? fileList.map(f => f.id) : [])
      }}
      onFolderOpen={(folder) => navigateToFolder(folder)}
      onFileAction={(action, fileId) => handleFileAction(action, fileId)}
    />
  )
}
```
