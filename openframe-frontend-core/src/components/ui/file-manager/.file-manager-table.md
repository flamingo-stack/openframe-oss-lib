A React component that renders a file browser table with selection, loading states, and responsive height management for file management interfaces.

## Key Components

- **FileManagerTable** - Main table component with header, rows, and selection controls
- **Dynamic height calculation** - Uses `useLayoutEffect` to calculate optimal table height based on viewport
- **Selection management** - Handles individual file selection and "select all" functionality with checkbox states
- **Loading skeleton** - Renders skeleton placeholders during data fetching
- **Responsive layout** - Fixed column widths with flexible name column and overflow handling

## Usage Example

```typescript
import { FileManagerTable } from './file-manager-table'

function MyFileManager() {
  const [files, setFiles] = useState([])
  const [selectedFiles, setSelectedFiles] = useState<string[]>([])
  const [loading, setLoading] = useState(true)

  return (
    <FileManagerTable
      files={files}
      selectedFiles={selectedFiles}
      loading={loading}
      showCheckboxes={true}
      isSearchResult={false}
      onSelectFile={(fileId, selected) => {
        setSelectedFiles(prev => 
          selected ? [...prev, fileId] : prev.filter(id => id !== fileId)
        )
      }}
      onSelectAll={(checked) => {
        setSelectedFiles(checked ? files.map(f => f.id) : [])
      }}
      onFileClick={(file) => console.log('File clicked:', file)}
      onFolderOpen={(folder) => console.log('Folder opened:', folder)}
      onFileAction={(action, fileId) => console.log(action, fileId)}
      className="h-full"
    />
  )
}
```

The component automatically handles responsive height calculations, skeleton loading states, and provides a clean table interface with NAME, SIZE, and EDITED columns.
