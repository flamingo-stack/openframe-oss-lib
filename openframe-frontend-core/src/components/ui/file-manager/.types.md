This file defines comprehensive TypeScript types and interfaces for a file manager component system, providing type safety for file operations, UI components, and user interactions.

## Key Components

**Core Types:**
- `FileType` - Distinguishes between 'file' and 'folder' items
- `FileAction` - Union type for all possible file operations (download, copy, rename, etc.)
- `FileItem` - Complete file/folder data structure with metadata
- `BreadcrumbItem` - Navigation breadcrumb structure

**Component Interface Props:**
- `FileManagerProps` - Main file manager component configuration
- `FileManagerTableProps` - File listing table with sorting capabilities
- `FileManagerActionBarProps` - Action toolbar component
- `FileManagerSearchBarProps` - Search functionality component
- `FileManagerContextMenuProps` - Right-click context menu component

## Usage Example

```typescript
import { FileItem, FileAction, FileManagerProps } from './types'

// Define file data
const files: FileItem[] = [
  {
    id: '1',
    name: 'document.pdf',
    type: 'file',
    size: '2.5 MB',
    modified: '2024-01-15',
    path: '/documents/document.pdf'
  },
  {
    id: '2',
    name: 'images',
    type: 'folder',
    modified: '2024-01-14',
    path: '/images'
  }
]

// Handle file actions
const handleFileAction = (action: FileAction, fileId?: string) => {
  switch (action) {
    case 'download':
      console.log(`Downloading file: ${fileId}`)
      break
    case 'delete':
      console.log(`Deleting file: ${fileId}`)
      break
  }
}

// File manager props
const props: FileManagerProps = {
  files,
  currentPath: '/documents',
  selectedFiles: [],
  onFileAction: handleFileAction,
  onNavigate: (path) => console.log(`Navigating to: ${path}`)
}
```
