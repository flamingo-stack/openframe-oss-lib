Defines TypeScript interfaces and types for a comprehensive file manager component system. This file provides the type definitions needed to build a full-featured file browser with actions, navigation, and selection capabilities.

## Key Components

**Core Types:**
- `FileType` - Union type for file or folder items
- `FileAction` - Available file operations (download, copy, rename, etc.)
- `FileItem` - Complete file/folder data structure with metadata

**Component Interfaces:**
- `FileManagerProps` - Main file manager component configuration
- `FileManagerTableProps` - File list table with sorting and selection
- `FileManagerBreadcrumbProps` - Navigation breadcrumb component
- `FileManagerActionBarProps` - Toolbar with file actions
- `FileManagerSearchBarProps` - Search functionality
- `FileManagerContextMenuProps` - Right-click context menu
- `FileIconProps` - File type icon display
- `FileManagerEmptyProps` - Empty state display

## Usage Example

```typescript
import { FileItem, FileAction, FileManagerProps } from './types'

const files: FileItem[] = [
  {
    id: '1',
    name: 'document.pdf',
    type: 'file',
    size: '2.5 MB',
    modified: '2024-01-15',
    path: '/documents/document.pdf'
  }
]

const handleFileAction = (action: FileAction, fileId?: string) => {
  if (action === 'delete' && fileId) {
    // Handle file deletion
  }
}

const props: FileManagerProps = {
  files,
  currentPath: '/documents',
  selectedFiles: [],
  onFileAction: handleFileAction
}
```
