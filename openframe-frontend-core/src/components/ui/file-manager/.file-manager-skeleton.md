A React component that displays a skeleton loading state for a file manager interface, showing placeholder elements for breadcrumbs, search bar, and file table rows while data is loading.

## Key Components

**FileManagerSkeleton**: Main component that renders skeleton placeholders
- `rows` (number): Number of file rows to display (default: 8)
- `showSearch` (boolean): Whether to show search bar skeleton (default: true)  
- `showActions` (boolean): Whether to show action buttons skeleton (default: true)

## Usage Example

```typescript
import { FileManagerSkeleton } from './components/file-manager-skeleton'

// Basic usage with defaults
function LoadingFileManager() {
  return <FileManagerSkeleton />
}

// Customized skeleton
function CustomLoadingState() {
  return (
    <FileManagerSkeleton 
      rows={5}
      showSearch={false}
      showActions={true}
    />
  )
}

// Conditional rendering during data fetch
function FileManagerContainer() {
  const { data, isLoading } = useFileData()
  
  if (isLoading) {
    return <FileManagerSkeleton rows={10} />
  }
  
  return <FileManager data={data} />
}
```

The component mimics a complete file manager layout including breadcrumb navigation, search functionality, table headers (Name, Size, Edited), checkboxes, file icons, and action buttons.
