A client-side React component that renders a clickable breadcrumb navigation trail for file manager interfaces. It displays a hierarchical path with chevron separators and highlights the current location.

## Key Components

- **FileManagerBreadcrumb**: Main component that renders the breadcrumb navigation
- **Props**: `items` (breadcrumb items array), `onItemClick` (click handler), `className` (optional styling)
- **Dependencies**: Uses Lucide React for ChevronRight icons and custom `cn` utility for class merging

## Usage Example

```typescript
import { FileManagerBreadcrumb } from './file-manager-breadcrumb'

const breadcrumbItems = [
  { label: 'Home', path: '/' },
  { label: 'Documents', path: '/documents' },
  { label: 'Projects', path: '/documents/projects' }
]

function FileManager() {
  const handleBreadcrumbClick = (path: string) => {
    // Navigate to the selected path
    navigateToPath(path)
  }

  return (
    <FileManagerBreadcrumb
      items={breadcrumbItems}
      onItemClick={handleBreadcrumbClick}
      className="mb-4"
    />
  )
}
```

The component automatically adds chevron separators between items and applies special styling to the last item (current location) with bold font weight.
