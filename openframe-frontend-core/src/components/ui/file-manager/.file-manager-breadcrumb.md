A React component that renders a clickable breadcrumb navigation for file system hierarchies, displaying the path with chevron separators and hover interactions.

## Key Components

- **FileManagerBreadcrumb** - Main component that renders breadcrumb navigation
- **Props Interface** - Accepts `items` array, `onItemClick` callback, and optional `className`
- **Navigation Items** - Each breadcrumb segment is a clickable button with path and label
- **Responsive Design** - Handles text wrapping and flexible layout for long paths

## Usage Example

```typescript
import { FileManagerBreadcrumb } from './file-manager-breadcrumb'

const breadcrumbItems = [
  { path: '/', label: 'Root' },
  { path: '/documents', label: 'Documents' },
  { path: '/documents/projects', label: 'Projects' }
]

function FileExplorer() {
  const handleBreadcrumbClick = (path: string) => {
    // Navigate to the selected path
    console.log('Navigating to:', path)
  }

  return (
    <FileManagerBreadcrumb
      items={breadcrumbItems}
      onItemClick={handleBreadcrumbClick}
      className="mb-4"
    />
  )
}
```

The component features responsive text wrapping, hover states with color transitions, and highlights the current (last) item with bold text. Chevron separators between items provide clear visual hierarchy.
