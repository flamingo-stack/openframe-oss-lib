A comprehensive React file management component that provides a complete file browser interface with breadcrumb navigation, search functionality, and file operations.

## Key Components

- **FileManager** - Main component orchestrating the file browser interface
- **Breadcrumb Navigation** - Cross-platform path handling (Windows/Unix)
- **Search Integration** - File search with loading states
- **Action Bar** - File operations (copy, cut, paste, upload, new folder)
- **File Table** - Display and selection of files/folders
- **Path Management** - Intelligent path construction for navigation

## Usage Example

```typescript
import { FileManager } from './file-manager'

function MyFileBrowser() {
  const [files, setFiles] = useState([])
  const [currentPath, setCurrentPath] = useState('')
  const [selectedFiles, setSelectedFiles] = useState<string[]>([])

  return (
    <FileManager
      files={files}
      currentPath={currentPath}
      selectedFiles={selectedFiles}
      searchQuery=""
      loading={false}
      showCheckboxes={true}
      showSearch={true}
      showActions={true}
      onNavigate={setCurrentPath}
      onSelectFile={(id, selected) => {
        setSelectedFiles(prev => 
          selected ? [...prev, id] : prev.filter(f => f !== id)
        )
      }}
      onFileAction={(action) => console.log('Action:', action)}
      onFolderOpen={(folder) => console.log('Open:', folder)}
    />
  )
}
```

The component handles complex path logic for both Windows (`C:\folder\`) and Unix (`/folder/`) file systems, provides rich file operations through callbacks, and maintains selection state across navigation.
