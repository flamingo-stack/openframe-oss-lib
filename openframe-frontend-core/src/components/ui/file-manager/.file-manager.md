A comprehensive file manager component that provides a complete file browser interface with navigation, search, and file operations capabilities.

## Key Components

- **FileManager** - Main component that orchestrates the file management interface
- **Breadcrumb Navigation** - Cross-platform path handling (Windows/Unix) with clickable navigation
- **Search Integration** - Built-in search bar with loading states and result counts
- **File Operations** - Action bar with copy, cut, paste, upload, and new folder actions
- **File Table** - Displays files with selection checkboxes and interactive operations
- **Path Management** - Intelligent path handling for both Windows (`C:\path`) and Unix (`/path`) file systems

## Usage Example

```typescript
import { FileManager } from './file-manager'

function App() {
  const [files, setFiles] = useState([])
  const [selectedFiles, setSelectedFiles] = useState<string[]>([])
  const [currentPath, setCurrentPath] = useState('')

  return (
    <FileManager
      files={files}
      currentPath={currentPath}
      selectedFiles={selectedFiles}
      searchQuery=""
      onNavigate={(path) => setCurrentPath(path)}
      onSelectFile={(fileId, selected) => {
        if (selected) {
          setSelectedFiles(prev => [...prev, fileId])
        } else {
          setSelectedFiles(prev => prev.filter(id => id !== fileId))
        }
      }}
      onFileAction={(action) => {
        console.log('File action:', action)
      }}
      onSearch={(query) => {
        // Implement search logic
      }}
    />
  )
}
```

The component automatically generates breadcrumb navigation from the current path and handles both Windows drive letters and Unix-style paths. It provides a complete file management experience with customizable visibility for search, actions, and checkboxes.
