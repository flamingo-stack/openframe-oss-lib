A comprehensive file manager component that provides file navigation, breadcrumbs, search functionality, and file operations with support for both Windows and Unix path formats.

## Key Components

- **FileManager** - Main component that orchestrates file management UI
- **Breadcrumb navigation** - Dynamic path navigation with cross-platform support
- **Search integration** - Optional search bar with loading states
- **Action bar** - File operations like copy, cut, paste, upload, new folder
- **File table** - Display and selection of files with checkboxes
- **Path handling** - Intelligent Windows/Unix path parsing and navigation

## Usage Example

```typescript
import { FileManager } from './file-manager'

function App() {
  const [files, setFiles] = useState([])
  const [currentPath, setCurrentPath] = useState('/')
  const [selectedFiles, setSelectedFiles] = useState<string[]>([])

  return (
    <FileManager
      files={files}
      currentPath={currentPath}
      selectedFiles={selectedFiles}
      searchQuery=""
      showCheckboxes={true}
      showSearch={true}
      showActions={true}
      onNavigate={(path) => setCurrentPath(path)}
      onSelectFile={(id, selected) => {
        setSelectedFiles(prev => 
          selected ? [...prev, id] : prev.filter(f => f !== id)
        )
      }}
      onFileAction={(action) => console.log('Action:', action)}
      onFolderOpen={(folder) => console.log('Opening:', folder.name)}
    />
  )
}
```
