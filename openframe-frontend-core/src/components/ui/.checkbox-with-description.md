A reusable checkbox component with integrated title and description text, built on Radix UI primitives and designed for form sections requiring detailed explanations.

## Key Components

- **CheckboxWithDescription** - Main component combining checkbox, title label, and description text
- **CheckboxWithDescriptionProps** - TypeScript interface defining component props including required `id`, `checked`, `onCheckedChange`, `title`, and `description`

## Usage Example

```typescript
import { CheckboxWithDescription } from "./checkbox-with-description"
import { useState } from "react"

function SettingsForm() {
  const [notificationsEnabled, setNotificationsEnabled] = useState(false)

  return (
    <CheckboxWithDescription
      id="notifications"
      checked={notificationsEnabled}
      onCheckedChange={setNotificationsEnabled}
      title="Email Notifications"
      description="Receive updates about your account activity and new features via email"
      disabled={false}
      className="max-w-md"
    />
  )
}
```

The component renders as a bordered card with the checkbox aligned to the top-left, making it ideal for settings panels, consent forms, or any interface requiring detailed option explanations. The checkbox state is fully controlled through the `checked` and `onCheckedChange` props.
