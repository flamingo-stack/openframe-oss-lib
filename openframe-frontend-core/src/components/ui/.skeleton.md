A comprehensive skeleton loading component library providing animated placeholders for various UI patterns during content loading states.

## Key Components

- **Skeleton** - Base animated placeholder div with pulse animation
- **SkeletonText** - Multi-line text placeholder with configurable line count
- **SkeletonCard** - Card layout placeholder with optional image area
- **SkeletonGrid** - Responsive grid of skeleton cards with customizable columns
- **SkeletonButton** - Button placeholder in small, default, or large sizes
- **SkeletonHeading** - Heading placeholder with level-based sizing (h1-h6)
- **SkeletonList** - List item placeholders with avatar and text content
- **SkeletonNavigation** - Navigation bar placeholder with multiple items

## Usage Example

```typescript
import { 
  SkeletonText, 
  SkeletonCard, 
  SkeletonGrid 
} from './skeleton'

// Simple text loading
<SkeletonText lines={3} />

// Card with image placeholder
<SkeletonCard showImage className="w-80" />

// Grid of content cards
<SkeletonGrid 
  columns={3} 
  items={6} 
  showImages={true} 
/>

// Navigation placeholder
<SkeletonNavigation items={5} />
```

All components are forwardRef-enabled, accept standard HTML div props, and use the `cn` utility for className merging with consistent styling patterns.
