A React component module providing two color selection UI components: a preset-based color selector (`ColorPresetSelect`) and a full color picker with HEX/RGB/HSL input modes (`ColorPickerInput`).

## Key Components

### Exports

| Export | Type | Description |
|--------|------|-------------|
| `ColorPresetSelect` | Component | Dropdown selector for named color presets with a custom fallback option |
| `ColorPickerInput` | Component | Popover-based full color picker with `react-colorful` and multi-format channel inputs |
| `TICKET_STATUS_COLOR_PRESETS` | `readonly ColorPreset[]` | 12-color palette for ticket status labels |
| `ColorPreset` | Interface | `{ key, label, color }` shape for preset entries |
| `ColorPresetSelectProps` | Interface | Props for `ColorPresetSelect` |
| `ColorPickerInputProps` | Interface | Props for `ColorPickerInput` |
| `CUSTOM_PRESET_KEY` | `string` | Sentinel key `'custom'` for user-defined colors |
| `DEFAULT_CUSTOM_STATUS_COLOR` | `string` | Default fallback `#888888` for custom entries |

### Internal Components

- **`ColorSwatch`** — 16px rounded color block used as icons in triggers and menu items
- **`HexInputs`** — Controlled text input with draft state for `#rrggbb` hex entry
- **`ChannelInput`** — Numeric-only input with clamped commit logic for R/G/B and H/S/L channels
- **`RgbInputs`** / **`HslInputs`** — Triplet channel inputs converting to/from hex via `ods-color-utils`

## Usage Example

```typescript
// Preset-based color selector (e.g. ticket status)
<ColorPresetSelect
  value={statusColor}
  presetKey={statusPresetKey}
  onChange={({ color, preset }) => {
    setStatusColor(color)
    setStatusPresetKey(preset)
  }}
/>

// Full color picker with HEX/RGB/HSL inputs
<ColorPickerInput
  value={customColor}        // always a hex string: '#rrggbb'
  onChange={setCustomColor}
/>

// Custom preset list
<ColorPresetSelect
  value={color}
  presets={[
    { key: 'brand', label: 'Brand Blue', color: '#0066ff' },
    { key: 'accent', label: 'Accent', color: '#ff6600' },
  ]}
  onChange={({ color, preset }) => handleChange(color, preset)}
/>
```

> **Note:** Both components use hex strings (`#rrggbb`) as their `value` contract. All HEX ↔ RGB ↔ HSL conversions are handled internally via `ods-color-utils`.
