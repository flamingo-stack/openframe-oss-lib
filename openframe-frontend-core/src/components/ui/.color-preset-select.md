Provides two color selection components: a preset-based color selector (`ColorPresetSelect`) and a full custom color picker (`ColorPickerInput`) with HEX/RGB/HSL input support.

## Key Components

### Exports

- **`ColorPresetSelect`** — Dropdown that lets users choose from a predefined palette or switch to "Custom" mode. Renders a color swatch + label trigger backed by `ActionsMenuDropdown`.
- **`ColorPickerInput`** — Full color picker using `react-colorful`'s `HexColorPicker`, with a format switcher (HEX / RGB / HSL) and channel inputs for precise values.
- **`TICKET_STATUS_COLOR_PRESETS`** — Default 12-color palette (green, teal, sky, red, pink, etc.) for ticket status use cases.
- **`ColorPreset`** — Interface `{ key, label, color }` describing a named preset entry.

### Internal Helpers

- **`HexInputs`** — Controlled text input with draft state; commits on blur or Enter, validates against `HEX_PATTERN`.
- **`ChannelInput`** — Numeric input (strips non-digits) clamped to `[min, max]`; used by RGB and HSL rows.
- **`RgbInputs`** / **`HslInputs`** — Three-channel input rows that convert to/from hex via `ods-color-utils`.

## Usage Example

```typescript
// Preset selector (e.g., ticket status color)
<ColorPresetSelect
  value={color}
  presetKey={presetKey}
  onChange={({ color, preset }) => {
    setColor(color)
    setPresetKey(preset)
  }}
/>

// Full custom picker (always hex in/out)
<ColorPickerInput
  value={hexColor}
  onChange={(hex) => setHexColor(hex)}
/>
```

> **Note:** Both components accept `disabled` and `className` props. `ColorPresetSelect` accepts a custom `presets` array; it defaults to `TICKET_STATUS_COLOR_PRESETS`.
