A compact React component that renders device information (name and organization) in a space-efficient vertical layout, primarily designed for use in table cells or other constrained spaces.

## Key Components

- **`DeviceCardCompactProps`** - Interface extending HTML div attributes with optional `deviceName` and `organization` props
- **`DeviceCardCompact`** - React functional component that conditionally renders device details in a stacked layout
- **Value validation** - Built-in filtering for null, undefined, '-', and string 'null' values
- **Empty state handling** - Returns empty fragment when no valid data is present

## Usage Example

```typescript
import { DeviceCardCompact } from './device-card-compact'

// Basic usage in a table cell
<td>
  <DeviceCardCompact 
    deviceName="iPad Pro 12.9"
    organization="Engineering Team"
  />
</td>

// With custom styling
<DeviceCardCompact 
  deviceName="MacBook Air"
  organization="Design Department"
  className="border rounded p-2"
/>

// Handles missing/invalid data gracefully
<DeviceCardCompact 
  deviceName={null}
  organization="-"
  // Returns empty fragment - nothing rendered
/>

// Only organization available
<DeviceCardCompact 
  deviceName=""
  organization="Marketing Team"
  // Only shows organization
/>
```

The component uses Tailwind CSS with custom font styling and automatically truncates long text to prevent layout overflow.
