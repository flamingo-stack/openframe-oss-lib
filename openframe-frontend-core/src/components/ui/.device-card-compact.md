A compact React component for displaying device information in table cells with a stacked layout showing device name and organization.

## Key Components

- **DeviceCardCompact**: Main component that renders device name and organization in a vertical stack
- **DeviceCardCompactProps**: Interface extending HTML div attributes with optional `deviceName` and `organization` props

## Usage Example

```typescript
import { DeviceCardCompact } from './device-card-compact'

// Basic usage in a table cell
<DeviceCardCompact 
  deviceName="iPhone 14 Pro" 
  organization="ACME Corp" 
/>

// With custom styling
<DeviceCardCompact 
  deviceName="MacBook Pro" 
  organization="Tech Solutions Inc"
  className="bg-gray-50 rounded-md"
/>

// Handles missing data gracefully
<DeviceCardCompact 
  deviceName={null} 
  organization="Some Org"
  // Only shows organization
/>

// Returns empty fragment when both are null/invalid
<DeviceCardCompact 
  deviceName={null} 
  organization={null}
  // Renders nothing
/>
```

The component intelligently handles null, undefined, '-', and string 'null' values by filtering them out. It returns an empty fragment if both values are invalid, making it safe to use in data tables where some rows might have missing information.
