A reusable React component for displaying status badges with multiple visual variants and color schemes using class-variance-authority for consistent styling.

## Key Components

- **StatusBadge**: Main React component that renders a styled badge with text
- **statusBadgeVariants**: CVA configuration defining styling variants for different badge types
- **StatusBadgeProps**: TypeScript interface extending HTMLDivElement attributes with variant props

**Variants:**
- `variant`: "card" (larger) or "button" (compact with multi-line text support)
- `colorScheme`: 12 options including themed colors (cyan, pink, yellow, etc.), semantic colors (success, error, warning), and border-only variants

## Usage Example

```typescript
import { StatusBadge } from './components/status-badge';

// Basic usage
<StatusBadge text="Active" colorScheme="success" />

// Compact button variant with multi-line text
<StatusBadge 
  text="In Progress" 
  variant="button" 
  colorScheme="cyan" 
/>

// Border-only variant for task types
<StatusBadge 
  text="Priority Task" 
  colorScheme="accentBorder" 
  className="custom-spacing" 
/>

// With custom props
<StatusBadge 
  text="Error State" 
  colorScheme="error" 
  onClick={handleClick}
  role="button"
/>
```
