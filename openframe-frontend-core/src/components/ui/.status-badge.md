A customizable status badge component that displays text with various visual styles and color schemes using class-variance-authority for consistent theming across the Flamingo design system.

## Key Components

- **StatusBadge** - Main React component for rendering status indicators
- **StatusBadgeProps** - TypeScript interface extending HTMLDivElement attributes with variant props
- **statusBadgeVariants** - CVA configuration defining visual variants and color schemes
- **renderText()** - Internal method that handles text splitting for button variant multi-line display

## Usage Example

```typescript
import { StatusBadge } from './status-badge';

// Basic status badge
<StatusBadge text="ACTIVE" colorScheme="success" />

// Button variant with compact styling
<StatusBadge 
  text="IN PROGRESS" 
  variant="button" 
  colorScheme="warning" 
/>

// Border-only style for task types
<StatusBadge 
  text="URGENT" 
  colorScheme="errorBorder" 
  className="mb-2" 
/>

// Multi-word text automatically splits on button variant
<StatusBadge 
  text="DEPLOYMENT READY" 
  variant="button" 
  colorScheme="green" 
/>
```

The component supports 11 color schemes including Flamingo brand colors (cyan, pink, yellow, green, purple), semantic states (success, error, warning), and border-only variants. Text is automatically rendered in uppercase with monospace font and proper spacing.
