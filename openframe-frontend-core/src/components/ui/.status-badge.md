A React component for displaying styled status badges with multiple visual variants and color schemes, built using class-variance-authority for consistent styling patterns.

## Key Components

- **StatusBadge** - Main component that renders a styled badge with customizable appearance
- **statusBadgeVariants** - CVA configuration defining visual variants and color schemes
- **StatusBadgeProps** - TypeScript interface extending HTML div attributes with variant props

## Usage Example

```typescript
import { StatusBadge } from './status-badge';

// Basic usage with default styling
<StatusBadge text="Active" />

// Card variant with success color
<StatusBadge 
  text="Completed" 
  variant="card" 
  colorScheme="success" 
/>

// Button variant with custom styling
<StatusBadge 
  text="In Progress" 
  variant="button" 
  colorScheme="cyan"
  className="ml-2" 
/>

// Border-only variant for task types
<StatusBadge 
  text="Critical" 
  variant="button" 
  colorScheme="errorBorder" 
/>
```

The component automatically handles text wrapping for the button variant by splitting words across multiple lines when spaces are present. It uses the Azeret Mono font family and supports Flamingo's design system color variables, including both filled and border-only color schemes for different use cases like status indicators and task type badges.
