A reusable React component for displaying status badges with various visual styles, color schemes, and layout variants for different UI contexts.

## Key Components

- **StatusBadge** - Main component that renders status badges with customizable styling
- **statusBadgeVariants** - Class variance authority (CVA) configuration defining visual variants
- **StatusBadgeProps** - TypeScript interface extending HTML div attributes with component-specific props

The component supports two main variants:
- `card` - Larger badges (px-3 py-1.5, text-sm) for card displays
- `button` - Compact badges (px-2 py-0.5, text-[10px]) with multi-line text support

Color schemes include Flamingo brand colors (cyan, pink, yellow, green, purple), semantic states (success, error, warning), and border-only variants for task type indicators.

## Usage Example

```typescript
import { StatusBadge } from './status-badge';

function TaskCard() {
  return (
    <div className="flex gap-2">
      {/* Card variant with brand color */}
      <StatusBadge 
        text="ACTIVE" 
        variant="card" 
        colorScheme="green" 
      />
      
      {/* Button variant with multi-line text */}
      <StatusBadge 
        text="HIGH PRIORITY" 
        variant="button" 
        colorScheme="error" 
      />
      
      {/* Border-only variant for task types */}
      <StatusBadge 
        text="API TASK" 
        variant="card" 
        colorScheme="accentBorder" 
      />
    </div>
  );
}
```

The component automatically handles text wrapping for button variants, splitting space-separated text into multiple lines for better readability in narrow spaces.
