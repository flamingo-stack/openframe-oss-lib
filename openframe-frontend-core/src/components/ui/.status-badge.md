A flexible status badge component with multiple visual variants and color schemes for displaying status information in cards and buttons.

## Key Components

- **`StatusBadge`** - Main React component that renders status badges with customizable appearance
- **`statusBadgeVariants`** - CVA (Class Variance Authority) configuration defining visual variants and color schemes
- **`StatusBadgeProps`** - TypeScript interface extending HTML div attributes with variant props and required text

## Usage Example

```typescript
import { StatusBadge } from './components/status-badge';

// Basic usage
<StatusBadge text="ACTIVE" />

// Card variant with success color
<StatusBadge 
  text="COMPLETED" 
  variant="card" 
  colorScheme="success" 
/>

// Button variant with custom styling
<StatusBadge 
  text="IN PROGRESS" 
  variant="button" 
  colorScheme="cyan"
  className="ml-2"
/>

// Border-only variant for task types
<StatusBadge 
  text="URGENT TASK" 
  colorScheme="errorBorder" 
/>
```

The component automatically handles text wrapping for button variants by splitting multi-word text across multiple lines. It uses the Azeret Mono font and supports 12 different color schemes including brand colors (cyan, pink, yellow, green, purple), semantic states (success, error, warning), and border-only variants.
