A React component that renders a responsive table row for displaying log entries with different layouts for desktop and mobile views.

## Key Components

- **`LogTableRow`** - Main component that displays log data in a card-like row format
- **`LogTableRowProps`** - TypeScript interface defining the component's props structure
- **`FleetMDMIcon`** - Internal SVG icon component for fleet management display

## Usage Example

```typescript
import { LogTableRow } from './log-table-row'

function LogList() {
  return (
    <div className="space-y-2">
      <LogTableRow
        logId="LOG-001"
        timestamp="2024-01-15 10:30:00"
        status={{
          label: "Success",
          variant: "success"
        }}
        source={{
          name: "Fleet MDM",
          icon: <CustomIcon />
        }}
        device={{
          name: "MacBook Pro",
          organization: "Acme Corp"
        }}
        description={{
          title: "Device enrolled successfully",
          details: "User completed enrollment process"
        }}
        onMoreClick={() => console.log('More actions')}
        onDetailsClick={() => console.log('View details')}
      />
    </div>
  )
}
```

The component automatically switches between desktop (multi-column) and mobile (condensed) layouts using responsive CSS classes. It includes interactive buttons for additional actions and detail views.
