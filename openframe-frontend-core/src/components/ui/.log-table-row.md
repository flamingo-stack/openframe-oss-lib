A React component that renders a single row in a log table, displaying log information with status, source, device details, and actions. Features responsive design with separate desktop and mobile layouts.

## Key Components

- **LogTableRow** - Main component rendering a log entry with dark theme styling
- **LogTableRowProps** - TypeScript interface defining all props including log data, status, device info, and event handlers
- **FleetMDMIcon** - Internal SVG icon component for MDM source indicators

## Usage Example

```typescript
import { LogTableRow } from './log-table-row'

function LogsList() {
  return (
    <div className="space-y-2">
      <LogTableRow
        logId="LOG-001"
        timestamp="2024-01-15 14:30:00"
        status={{ 
          label: "Success", 
          variant: "success" 
        }}
        source={{ 
          name: "Fleet MDM",
          icon: <FleetIcon />
        }}
        device={{
          name: "MacBook-Pro-123",
          organization: "Engineering Dept"
        }}
        description={{
          title: "Configuration updated",
          details: "Security policy applied successfully"
        }}
        onMoreClick={() => console.log('More actions')}
        onDetailsClick={() => console.log('Show details')}
      />
    </div>
  )
}
```

The component automatically switches between desktop (tabular) and mobile (compact) layouts using responsive CSS classes. Desktop layout shows all information in aligned columns, while mobile layout prioritizes essential information with condensed formatting.
