A reusable autocomplete/multi-select component built with Radix UI primitives that supports tagging, keyboard navigation, search filtering, and customizable rendering options.

## Key Components

- **AutocompleteOption<T>**: Interface defining option structure with label and value
- **AutocompleteProps<T>**: Comprehensive props interface with 20+ configuration options
- **AutocompleteInner<T>**: Main component implementation with state management and event handling

Key features include:
- Multi-select with removable tags
- Search/filter functionality with custom filter support
- Keyboard navigation (arrows, enter, escape, backspace)
- Tag overflow handling with customizable "+N" display
- Free solo mode for creating new options
- Loading states and validation styling
- Accessible design with ARIA labels

## Usage Example

```typescript
import { Autocomplete, AutocompleteOption } from './autocomplete'

const options: AutocompleteOption[] = [
  { label: 'React', value: 'react' },
  { label: 'Vue', value: 'vue' },
  { label: 'Angular', value: 'angular' }
]

function TechSelector() {
  const [selected, setSelected] = useState<string[]>([])
  
  return (
    <Autocomplete
      options={options}
      value={selected}
      onChange={setSelected}
      placeholder="Select technologies..."
      maxItems={3}
      limitTags={2}
      showClearAll
      label="Tech Stack"
    />
  )
}
```

The component integrates seamlessly with form libraries and supports both controlled and uncontrolled usage patterns with comprehensive TypeScript support.
