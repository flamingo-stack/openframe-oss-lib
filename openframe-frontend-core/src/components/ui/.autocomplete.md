A multi-select autocomplete component built on Radix UI primitives that provides advanced filtering, keyboard navigation, and customizable tag display with overflow handling.

## Key Components

**Main Component:**
- `Autocomplete` - Forwarded ref component for multi-select autocomplete functionality

**Interfaces:**
- `AutocompleteOption<T>` - Generic option structure with label and value
- `AutocompleteProps<T>` - Comprehensive props interface supporting customization, validation, and advanced features

**Key Features:**
- Multi-select with removable tags
- Keyboard navigation (arrow keys, enter, escape, backspace)
- Search filtering with custom filter functions
- Tag limit display with overflow indicator
- Free-solo mode for creating new options
- Loading states and validation styling
- Custom renderers for options and tags

## Usage Example

```typescript
import { Autocomplete, AutocompleteOption } from './autocomplete'

const teamMembers: AutocompleteOption<string>[] = [
  { label: 'John Doe', value: 'john' },
  { label: 'Jane Smith', value: 'jane' },
  { label: 'Bob Johnson', value: 'bob' }
]

function TeamSelector() {
  const [selectedMembers, setSelectedMembers] = useState<string[]>([])

  return (
    <Autocomplete
      label="Select Team Members"
      options={teamMembers}
      value={selectedMembers}
      onChange={setSelectedMembers}
      placeholder="Search team members..."
      maxItems={5}
      limitTags={3}
      freeSolo={true}
      showClearAll={true}
      filterOptions={(options, inputValue) =>
        options.filter(opt => 
          opt.label.toLowerCase().includes(inputValue.toLowerCase())
        )
      }
    />
  )
}
```
