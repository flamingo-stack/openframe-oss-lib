A React sheet/drawer component built on top of Radix UI Dialog primitives with customizable slide directions and styled overlays. Provides a complete set of composable components for creating slide-out panels from any screen edge.

## Key Components

- **Sheet**: Root container component
- **SheetTrigger**: Button/element that opens the sheet
- **SheetContent**: Main sheet panel with configurable slide direction (`top`, `bottom`, `left`, `right`)
- **SheetOverlay**: Background overlay with fade animations
- **SheetHeader/SheetFooter**: Layout containers for sheet sections
- **SheetTitle/SheetDescription**: Accessible heading and description components
- **SheetClose**: Close button component

## Usage Example

```typescript
import {
  Sheet,
  SheetContent,
  SheetDescription,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from "./sheet"

function MyComponent() {
  return (
    <Sheet>
      <SheetTrigger asChild>
        <button>Open Sheet</button>
      </SheetTrigger>
      <SheetContent side="right">
        <SheetHeader>
          <SheetTitle>Settings</SheetTitle>
          <SheetDescription>
            Configure your preferences here.
          </SheetDescription>
        </SheetHeader>
        <div className="grid gap-4 py-4">
          {/* Sheet content */}
        </div>
      </SheetContent>
    </Sheet>
  )
}
```

The component supports four slide directions and includes automatic close button, overlay animations, and responsive sizing.
