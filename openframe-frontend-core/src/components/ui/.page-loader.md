A React component library providing full-page and compact loading states with customizable text and animated branding. Features a pulsing logo animation and flexible styling options for different container sizes.

## Key Components

- **`PageLoader`** - Full viewport loading screen with centered content
- **`CompactPageLoader`** - Smaller loading component for contained areas
- **`PageLoaderProps`** - Shared interface defining component configuration
- **`OpenFrameLogo`** - Animated logo component with pulse effect

## Usage Example

```typescript
import { PageLoader, CompactPageLoader } from './components/loaders/page-loader'

// Full page loading screen
function App() {
  const [isLoading, setIsLoading] = useState(true)
  
  if (isLoading) {
    return (
      <PageLoader 
        title="Setting up your workspace"
        description="This may take a few moments..."
        className="bg-gray-50"
      />
    )
  }
  
  return <MainContent />
}

// Compact loader for sections
function DataPanel() {
  return (
    <div className="border rounded-lg">
      <CompactPageLoader 
        title="Loading charts"
        showIcon={false}
      />
    </div>
  )
}
```
