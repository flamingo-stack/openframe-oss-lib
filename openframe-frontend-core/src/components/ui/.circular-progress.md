A customizable circular progress indicator component that displays percentage completion using SVG with smooth animations and configurable styling options.

## Key Components

- **CircularProgressProps**: Interface defining component props including percentage, size, stroke width, and color options
- **CircularProgress**: Main component function that renders an animated circular progress bar with centered percentage text
- **SVG Implementation**: Uses stroke-dasharray and stroke-dashoffset to create the progress arc effect
- **Percentage Normalization**: Clamps input values between 0-100 to prevent invalid states

## Usage Example

```typescript
import { CircularProgress } from './components/ui/circular-progress'

// Basic usage
<CircularProgress percentage={75} />

// Customized appearance
<CircularProgress 
  percentage={42}
  size={80}
  strokeWidth={8}
  color="#ff6b6b"
  trackColor="#e0e0e0"
  className="my-4"
/>

// Dynamic progress
const [progress, setProgress] = useState(0)

<CircularProgress 
  percentage={progress}
  size={120}
  color="#4ade80"
/>
```

The component automatically handles percentage clamping, smooth transitions, and responsive sizing. The progress arc animates smoothly when the percentage value changes, making it ideal for loading indicators, skill meters, or completion trackers.
