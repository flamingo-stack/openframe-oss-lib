A comprehensive React navigation menu component library built on Radix UI primitives with styled components and smooth animations. Provides accessible, keyboard-navigable dropdown menus with customizable styling.

## Key Components

- **NavigationMenu** - Root container component that manages menu state and positioning
- **NavigationMenuList** - Container for menu items with flex layout
- **NavigationMenuItem** - Individual menu item wrapper (alias for Radix primitive)
- **NavigationMenuTrigger** - Clickable trigger button with chevron icon and hover states
- **NavigationMenuContent** - Dropdown content panel with slide-in/fade animations
- **NavigationMenuLink** - Navigation link component (alias for Radix primitive)
- **NavigationMenuViewport** - Animated viewport for content panels with zoom transitions
- **NavigationMenuIndicator** - Visual indicator arrow pointing to active content
- **navigationMenuTriggerStyle** - CVA utility for consistent trigger button styling

## Usage Example

```typescript
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
} from "./navigation-menu"

function MainNav() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Products</NavigationMenuTrigger>
          <NavigationMenuContent>
            <div className="grid gap-3 p-6 w-[400px]">
              <NavigationMenuLink href="/products/web">
                Web Development
              </NavigationMenuLink>
              <NavigationMenuLink href="/products/mobile">
                Mobile Apps
              </NavigationMenuLink>
            </div>
          </NavigationMenuContent>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  )
}
```
