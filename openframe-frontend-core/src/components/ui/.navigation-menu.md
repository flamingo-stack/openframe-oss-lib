A complete navigation menu component system built on Radix UI primitives with custom styling and animations for creating accessible dropdown navigation menus.

## Key Components

- **NavigationMenu** - Root container component that manages the overall navigation structure
- **NavigationMenuList** - Wrapper for navigation menu items with flex layout
- **NavigationMenuItem** - Individual menu item container (re-exported from Radix)
- **NavigationMenuTrigger** - Clickable trigger button with ChevronDown icon and rotation animation
- **NavigationMenuContent** - Dropdown content panel with slide/fade animations
- **NavigationMenuLink** - Navigation link component (re-exported from Radix)
- **NavigationMenuViewport** - Portal container for dropdown content with zoom animations
- **NavigationMenuIndicator** - Visual indicator showing active menu state
- **navigationMenuTriggerStyle** - CVA utility for consistent trigger button styling

## Usage Example

```typescript
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuList,
  NavigationMenuTrigger,
} from "./navigation-menu"

function MainNav() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Products</NavigationMenuTrigger>
          <NavigationMenuContent>
            <div className="grid gap-3 p-6 md:w-[400px]">
              <a href="/product-1">Product 1</a>
              <a href="/product-2">Product 2</a>
            </div>
          </NavigationMenuContent>
        </NavigationMenuItem>
        
        <NavigationMenuItem>
          <NavigationMenuTrigger>About</NavigationMenuTrigger>
          <NavigationMenuContent>
            <div className="p-6">
              <p>About our company...</p>
            </div>
          </NavigationMenuContent>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  )
}
```

The component system provides full keyboard navigation, ARIA compliance, and smooth animations for professional navigation experiences.
