A reusable dropdown menu component that displays contextual actions behind a three-dot (ellipsis) button trigger. Built on top of Radix UI's DropdownMenu with customizable positioning and styling.

## Key Components

- **MoreActionsMenu**: Main component that renders the dropdown menu
- **MoreActionsItem**: Type definition for individual menu items
- **MoreActionsMenuProps**: Interface for component configuration

## Usage Example

```typescript
import { MoreActionsMenu, MoreActionsItem } from './more-actions-menu'
import { Edit, Trash, Share } from 'lucide-react'

const menuItems: MoreActionsItem[] = [
  {
    label: 'Edit',
    onClick: () => console.log('Edit clicked'),
    icon: <Edit size={16} />
  },
  {
    label: 'Share',
    onClick: () => handleShare(),
    icon: <Share size={16} />
  },
  {
    label: 'Delete',
    onClick: () => handleDelete(),
    icon: <Trash size={16} />,
    danger: true
  }
]

function MyComponent() {
  return (
    <MoreActionsMenu
      items={menuItems}
      align="end"
      side="bottom"
      ariaLabel="Post actions"
    />
  )
}
```

The component supports disabled states, danger styling for destructive actions, custom icons, and flexible positioning options.
