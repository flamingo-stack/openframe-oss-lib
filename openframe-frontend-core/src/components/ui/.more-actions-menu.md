Renders a compact dropdown menu triggered by an ellipsis icon button, built on Radix `DropdownMenu`. Supports both link-based and click-based menu items with optional icons, danger states, and controlled open behavior.

> **Deprecated:** Use `ActionsMenuDropdown` from `./actions-menu` instead. This component will be removed in a future release.

## Key Components

- **`MoreActionsMenu`** — Main component rendering an ellipsis-triggered dropdown with configurable alignment, side, and offset
- **`MoreActionsItem`** — Item shape supporting `label`, `onClick`, `href`, `openInNewTab`, `icon`, `disabled`, and `danger`
- **`MoreActionsMenuProps`** — Full props interface including controlled `open`/`onOpenChange`, custom `trigger`, and `onCloseAutoFocus`

## Usage Example

```typescript
import { MoreActionsMenu } from './more-actions-menu'
import { TrashIcon, EditIcon } from '../icons-v2-generated'

<MoreActionsMenu
  items={[
    {
      label: 'Edit',
      icon: <EditIcon />,
      onClick: () => handleEdit(),
    },
    {
      label: 'Open in docs',
      icon: <ExternalIcon />,
      href: 'https://flamingo.run/docs',
      openInNewTab: true,
    },
    {
      label: 'Delete',
      icon: <TrashIcon />,
      danger: true,
      onClick: () => handleDelete(),
    },
  ]}
  align="end"
  side="bottom"
/>
```

Replace with `ActionsMenuDropdown` from `./actions-menu`:
