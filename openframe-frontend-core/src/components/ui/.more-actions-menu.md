A reusable dropdown menu component that displays contextual actions behind a three-dot (ellipsis) trigger button. Built on top of Radix UI's DropdownMenu primitives with custom styling for dark themes.

## Key Components

- **`MoreActionsMenu`** - Main component that renders an ellipsis button triggering a dropdown menu
- **`MoreActionsItem`** - Type definition for individual menu items with optional icons and states
- **`MoreActionsMenuProps`** - Interface for component configuration including alignment and positioning

## Usage Example

```typescript
import { MoreActionsMenu } from './more-actions-menu'
import { Edit, Trash2, Share } from 'lucide-react'

function MyComponent() {
  const actions = [
    {
      label: 'Edit',
      onClick: () => console.log('Edit clicked'),
      icon: <Edit className="h-4 w-4" />
    },
    {
      label: 'Share',
      onClick: () => console.log('Share clicked'),
      icon: <Share className="h-4 w-4" />
    },
    {
      label: 'Delete',
      onClick: () => console.log('Delete clicked'),
      icon: <Trash2 className="h-4 w-4" />,
      danger: true
    }
  ]

  return (
    <MoreActionsMenu 
      items={actions}
      align="end"
      ariaLabel="Post actions"
    />
  )
}
```

The component supports disabled states, danger styling for destructive actions, and flexible positioning options. Each menu item can include an icon and will render with proper focus states and accessibility attributes.
