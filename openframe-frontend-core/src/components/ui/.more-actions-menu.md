A deprecated dropdown menu component that renders a list of action items triggered by an ellipsis icon button, built on top of Radix UI's `DropdownMenu`.

## Key Components

### `MoreActionsItem` *(deprecated)*
Type definition for individual menu items. Supports:
- `label` — display text
- `onClick` — click handler
- `href` — renders item as a Next.js `<Link>` (SEO-friendly)
- `openInNewTab` — applies `target="_blank"` when using `href`
- `icon` — optional React node rendered left of the label
- `disabled` — disables interaction
- `danger` — applies error color styling to the icon

### `MoreActionsMenuProps` *(deprecated)*
Interface for the menu component, supporting controlled open state (`open` / `onOpenChange`), custom trigger override, alignment/positioning props, and focus management via `onCloseAutoFocus`.

### `MoreActionsMenu` *(deprecated)*
The primary exported component. Renders an ellipsis icon button (or custom `trigger`) that opens a styled dropdown. Each item renders as either a `<Link>` (when `href` is set) or a button `DropdownMenuItem`.

## Usage Example

```typescript
import { MoreActionsMenu } from './more-actions-menu'
import { TrashIcon } from '../icons-v2-generated'

<MoreActionsMenu
  items={[
    {
      label: 'View Details',
      href: '/tickets/123',
      icon: <InfoIcon />
    },
    {
      label: 'Open in new tab',
      href: '/tickets/123',
      openInNewTab: true
    },
    {
      label: 'Delete',
      onClick: () => handleDelete(),
      icon: <TrashIcon />,
      danger: true
    }
  ]}
  align="end"
  side="bottom"
/>
```

> **Deprecated:** Migrate to `ActionsMenuDropdown` from `./actions-menu`, which supports all the same features plus grouped items, checkboxes, and submenus.
