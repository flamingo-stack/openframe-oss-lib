A flexible tab navigation component that supports both URL synchronization and controlled state modes for managing tab selection in Next.js applications.

## Key Components

- **TabNavigation**: Main component with dual operation modes - URL sync for browser history integration or controlled state for programmatic control
- **TabItem**: Interface defining tab structure with id, label, icon, optional component, and alert indicators
- **TabNavigationUrlSyncOptions**: Configuration for URL synchronization behavior (parameter name, history method)
- **getTabById**: Utility function to find a tab by its ID
- **getTabComponent**: Utility function to extract a tab's associated component

## Usage Example

```typescript
// URL sync mode (recommended)
<TabNavigation
  urlSync={{ paramName: 'section', replaceState: true }}
  defaultTab="overview"
  tabs={[
    {
      id: 'overview',
      label: 'Overview',
      icon: HomeIcon,
      component: OverviewPanel,
      hasAlert: true,
      alertType: 'warning'
    },
    {
      id: 'settings',
      label: 'Settings',
      icon: SettingsIcon,
      component: SettingsPanel
    }
  ]}
>
  {(activeTab) => {
    const TabComponent = getTabComponent(tabs, activeTab)
    return TabComponent ? <TabComponent /> : null
  }}
</TabNavigation>

// Controlled mode
<TabNavigation
  activeTab={currentTab}
  onTabChange={setCurrentTab}
  tabs={tabs}
/>
```
