A flexible tab navigation component with URL synchronization support that manages tab state through URL parameters or controlled props.

## Key Components

**Interfaces:**
- `TabItem` - Configuration for individual tabs with id, label, icon, component, and status indicator
- `TabNavigationUrlSyncOptions` - Settings for URL parameter synchronization
- `TabNavigationProps` - Main component props supporting both controlled and URL-sync modes

**Main Exports:**
- `TabNavigation` - Primary tab navigation component with dual operation modes
- `getTabById()` - Utility to find tab configuration by ID
- `getTabComponent()` - Utility to extract tab component by ID

## Usage Example

```typescript
import { TabNavigation, TabItem } from './tab-navigation'
import { HomeIcon, SettingsIcon } from './icons'

// Define tabs with icons and optional indicators
const tabs: TabItem[] = [
  { 
    id: 'dashboard', 
    label: 'Dashboard', 
    icon: HomeIcon,
    indicator: 'good'
  },
  { 
    id: 'settings', 
    label: 'Settings', 
    icon: SettingsIcon,
    component: SettingsPanel
  }
]

// URL-synced mode (syncs active tab with ?tab=dashboard)
<TabNavigation
  tabs={tabs}
  urlSync={{ paramName: 'tab', replaceState: true }}
  defaultTab="dashboard"
>
  {(activeTab) => {
    const TabComponent = getTabComponent(tabs, activeTab)
    return TabComponent ? <TabComponent /> : null
  }}
</TabNavigation>

// Controlled mode
const [activeTab, setActiveTab] = useState('dashboard')
<TabNavigation
  tabs={tabs}
  activeTab={activeTab}
  onTabChange={setActiveTab}
/>
```

The component features visual indicators, smooth transitions, horizontal scrolling for overflow, and automatic URL synchronization with browser history.
