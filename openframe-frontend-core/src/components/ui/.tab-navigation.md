A flexible tab navigation component that supports both controlled state management and URL synchronization for maintaining tab state across page reloads and browser navigation.

## Key Components

**Main Export:**
- `TabNavigation` - The primary tab navigation component with URL sync capabilities

**Interfaces:**
- `TabItem` - Defines tab structure with id, label, icon, optional component, and status indicator
- `TabNavigationUrlSyncOptions` - Configuration options for URL parameter synchronization

**Utility Functions:**
- `getTabById()` - Retrieves a tab item by its ID
- `getTabComponent()` - Extracts the React component from a tab item

## Usage Example

```typescript
import { TabNavigation, TabItem } from './tab-navigation'
import { HomeIcon, SettingsIcon } from 'lucide-react'

const tabs: TabItem[] = [
  {
    id: 'home',
    label: 'Home',
    icon: HomeIcon,
    component: HomeComponent,
    indicator: 'good'
  },
  {
    id: 'settings',
    label: 'Settings', 
    icon: SettingsIcon,
    component: SettingsComponent,
    indicator: 'warning'
  }
]

// URL sync mode (maintains state in URL)
function MyApp() {
  return (
    <TabNavigation
      tabs={tabs}
      urlSync={{ paramName: 'view', replaceState: true }}
      defaultTab="home"
    >
      {(activeTab) => {
        const TabComponent = getTabComponent(tabs, activeTab)
        return TabComponent ? <TabComponent /> : null
      }}
    </TabNavigation>
  )
}

// Controlled mode
function ControlledApp() {
  const [activeTab, setActiveTab] = useState('home')
  
  return (
    <TabNavigation
      tabs={tabs}
      activeTab={activeTab}
      onTabChange={setActiveTab}
    />
  )
}
```

The component features gradient backgrounds, status indicators, horizontal scrolling for overflow, and smooth transitions between tab states.
