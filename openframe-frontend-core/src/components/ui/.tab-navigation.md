A flexible tab navigation component that supports both controlled state and URL synchronization for maintaining tab state across page navigation.

## Key Components

- **TabNavigation** - Main component that renders navigational tabs with optional URL sync
- **TabItem** - Interface defining tab structure with id, label, icon, component, and alert options
- **TabNavigationUrlSyncOptions** - Configuration for URL synchronization behavior
- **getTabById** - Utility to find a tab by its ID
- **getTabComponent** - Utility to extract a tab's associated component

## Features

- **Dual Mode Operation**: Works as controlled component or with URL synchronization
- **URL Sync**: Automatically syncs active tab with URL query parameters
- **Navigation Support**: Handles browser back/forward navigation when URL sync is enabled
- **Alert Indicators**: Visual indicators for tabs requiring attention (warning/error)
- **Responsive Design**: Horizontal scrolling for tab overflow
- **Render Props**: Children function receives active tab for content rendering

## Usage Example

```typescript
import { TabNavigation, TabItem } from './tab-navigation'
import { HomeIcon, SettingsIcon } from './icons'

const tabs: TabItem[] = [
  {
    id: 'dashboard',
    label: 'Dashboard',
    icon: HomeIcon,
    hasAlert: true,
    alertType: 'warning'
  },
  {
    id: 'settings',
    label: 'Settings',
    icon: SettingsIcon
  }
]

// URL sync mode
<TabNavigation
  tabs={tabs}
  urlSync={{ paramName: 'section', replaceState: true }}
  defaultTab="dashboard"
>
  {(activeTab) => (
    <div>Content for {activeTab}</div>
  )}
</TabNavigation>

// Controlled mode
<TabNavigation
  tabs={tabs}
  activeTab={currentTab}
  onTabChange={setCurrentTab}
/>
```
