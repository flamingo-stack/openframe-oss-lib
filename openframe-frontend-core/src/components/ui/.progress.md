A reusable progress bar component built on top of Radix UI's Progress primitive, providing a customizable visual indicator for completion status or loading states.

## Key Components

- **Progress** - Main progress bar component with customizable styling
- Accepts standard Radix Progress props plus optional `indicatorClassName` for styling the progress indicator
- Uses CSS transforms for smooth progress transitions

## Usage Example

```typescript
import { Progress } from "./components/ui/progress"

function LoadingExample() {
  const [progress, setProgress] = React.useState(0)
  
  React.useEffect(() => {
    const timer = setTimeout(() => setProgress(66), 500)
    return () => clearTimeout(timer)
  }, [])

  return (
    <div className="space-y-4">
      {/* Basic usage */}
      <Progress value={progress} />
      
      {/* Custom styling */}
      <Progress 
        value={75} 
        className="h-3" 
        indicatorClassName="bg-green-500" 
      />
    </div>
  )
}
```

The component automatically handles the visual progress animation through CSS transforms, translating the indicator based on the provided `value` prop (0-100).
