A React label component built on Radix UI primitives with customizable size and spacing variants using class-variance-authority for consistent styling.

## Key Components

- **Label** - Main component that extends Radix UI's Label primitive with variant-based styling
- **labelVariants** - CVA configuration defining size and spacing variants
- **VariantProps** - TypeScript interface for component props

## Usage Example

```typescript
import { Label } from "./label"

export function FormExample() {
  return (
    <div>
      {/* Default label */}
      <Label htmlFor="email">Email Address</Label>
      
      {/* Small label with tight spacing */}
      <Label 
        htmlFor="password" 
        variant="small" 
        spacing="tight"
      >
        Password
      </Label>
      
      {/* Large label with loose spacing */}
      <Label 
        htmlFor="description" 
        variant="large" 
        spacing="loose"
      >
        Description
      </Label>
    </div>
  )
}
```

The component supports four size variants (`default`, `small`, `medium`, `large`) and four spacing options (`default`, `tight`, `normal`, `loose`) for flexible form layouts.
