A client-side React component that provides a flexible label element built on Radix UI primitives with customizable typography and spacing variants.

## Key Components

- **`labelVariants`** - CVA configuration defining size and spacing variants for consistent label styling
- **`Label`** - Main component extending Radix UI Label with variant props and forward ref support

## Usage Example

```typescript
import { Label } from "./label"

function FormExample() {
  return (
    <div>
      {/* Default label */}
      <Label htmlFor="email">Email Address</Label>
      <input id="email" type="email" />
      
      {/* Small label with tight spacing */}
      <Label htmlFor="username" variant="small" spacing="tight">
        Username
      </Label>
      <input id="username" type="text" />
      
      {/* Large label with loose spacing */}
      <Label htmlFor="description" variant="large" spacing="loose">
        Description
      </Label>
      <textarea id="description" />
    </div>
  )
}
```

The component supports four size variants (`default`, `small`, `medium`, `large`) and four spacing options (`default`, `tight`, `normal`, `loose`), providing flexible typography scaling and margin control for form layouts.
