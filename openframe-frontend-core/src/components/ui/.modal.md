A composable modal component system built with React that provides a fully accessible overlay dialog with backdrop, keyboard navigation, and scroll management.

## Key Components

- **Modal** - Main modal container with backdrop overlay, ESC key handling, and body scroll prevention
- **ModalContent** - Wrapper for modal body content
- **ModalHeader** - Header section with bottom border styling
- **ModalTitle** - Semantic h2 heading for modal titles
- **ModalFooter** - Footer section with right-aligned button layout

## Usage Example

```typescript
import { Modal, ModalHeader, ModalTitle, ModalContent, ModalFooter } from './modal'

function MyComponent() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
      <ModalHeader>
        <ModalTitle>Confirm Action</ModalTitle>
      </ModalHeader>
      <ModalContent>
        <p>Are you sure you want to continue?</p>
      </ModalContent>
      <ModalFooter>
        <button onClick={() => setIsOpen(false)}>Cancel</button>

}
```

The modal automatically handles accessibility attributes, prevents background scrolling when open, and closes on ESC key press or backdrop click.
