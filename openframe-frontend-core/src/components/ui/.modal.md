A React modal component system providing a complete set of composable modal dialog elements with accessibility features, keyboard navigation, and backdrop click handling.

## Key Components

- **Modal**: Main modal container with backdrop and escape key handling
- **ModalContent**: Wrapper component for modal body content
- **ModalHeader**: Header section with bottom border styling
- **ModalTitle**: Semantic heading element for modal titles
- **ModalFooter**: Footer section with right-aligned action buttons

## Usage Example

```typescript
import { Modal, ModalHeader, ModalTitle, ModalContent, ModalFooter } from './modal'
import { useState } from 'react'

function MyComponent() {
  const [isModalOpen, setIsModalOpen] = useState(false)

  return (
    <>
      <button onClick={() => setIsModalOpen(true)}>
        Open Modal
      </button>
      
      <Modal isOpen={isModalOpen} onClose={() => setIsModalOpen(false)}>
        <ModalHeader>
          <ModalTitle>Confirm Action</ModalTitle>
        </ModalHeader>
        <ModalContent className="p-6">
          <p>Are you sure you want to continue?</p>
        </ModalContent>
        <ModalFooter>
          <button onClick={() => setIsModalOpen(false)}>Cancel</button>
          <button onClick={() => setIsModalOpen(false)}>Confirm</button>
        </ModalFooter>
      </Modal>
    </>
  )
}
```

The modal automatically handles ESC key presses, prevents background scrolling when open, and includes proper ARIA attributes for accessibility. All components are forwardRef-compatible and accept custom className props for styling customization.
