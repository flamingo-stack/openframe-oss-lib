A React component that renders a visual separator line, built on top of Radix UI's Separator primitive with customizable orientation and styling.

## Key Components

- **Separator** - A forwardRef component that creates horizontal or vertical divider lines
- Uses Radix UI's `SeparatorPrimitive.Root` for accessibility and behavior
- Supports both horizontal (default) and vertical orientations
- Includes automatic styling based on orientation

## Usage Example

```typescript
import { Separator } from "./separator"

function MyComponent() {
  return (
    <div>
      <p>Section 1</p>
      
      {/* Horizontal separator (default) */}
      <Separator />
      
      <p>Section 2</p>
      
      {/* Vertical separator */}
      <div className="flex h-20 items-center">
        <span>Left content</span>
        <Separator orientation="vertical" className="mx-4" />
        <span>Right content</span>
      </div>
      
      {/* Custom styling */}
      <Separator className="my-6 bg-red-200" />
    </div>
  )
}
```

The component automatically applies appropriate dimensions (1px height for horizontal, 1px width for vertical) and can be customized with additional CSS classes.
