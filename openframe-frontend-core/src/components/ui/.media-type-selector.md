A React dropdown selector component for choosing media file types with icons, built on top of a custom Select component.

## Key Components

- **`MediaType`** - Union type defining supported media types (video, images, documents, etc.)
- **`MediaTypeSelector`** - Main component that renders a dropdown with media type options
- **`mediaTypeOptions`** - Array of predefined options with labels and Lucide React icons
- **`MediaTypeSelectorProps`** - Interface defining component props including value, onChange, and styling options

## Usage Example

```typescript
import { MediaTypeSelector, MediaType } from './media-type-selector'

function MediaUploadForm() {
  const [selectedType, setSelectedType] = useState<MediaType>('video')

  return (
    <div>
      <MediaTypeSelector
        value={selectedType}
        onValueChange={setSelectedType}
        placeholder="Choose file type"
        className="w-48"
      />
    </div>
  )
}
```

The component includes a force re-render mechanism using a key state to ensure proper Select component updates when the value changes externally. Each option displays an appropriate icon alongside its label for better user experience.
