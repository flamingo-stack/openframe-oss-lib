A React dropdown component that allows users to select from different media/file types with visual icons and labels.

## Key Components

- **`MediaType`** - Union type defining supported media formats (video, images, documents, etc.)
- **`MediaTypeSelector`** - Main component providing a styled select dropdown
- **`MediaTypeOption`** - Interface defining the structure for each selectable option
- **`mediaTypeOptions`** - Array of predefined media types with corresponding Lucide icons

## Usage Example

```typescript
import { MediaTypeSelector, MediaType } from "./media-type-selector"

function FileUploadForm() {
  const [selectedType, setSelectedType] = React.useState<MediaType>('video')

  return (
    <div>
      <MediaTypeSelector
        value={selectedType}
        onValueChange={setSelectedType}
        placeholder="Choose file type"
        className="w-full"
        disabled={false}
      />
    </div>
  )
}
```

The component uses a force re-render mechanism with a key state to ensure the Select component properly updates when the value changes externally. It supports 9 different media types including videos, various image formats, documents, archives, and content types like guides and checklists.
