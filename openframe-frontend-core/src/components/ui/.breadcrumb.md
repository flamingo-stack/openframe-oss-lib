A React component library for building accessible breadcrumb navigation with customizable separators and ellipsis support. Built with Radix UI primitives and includes proper ARIA attributes for screen readers.

## Key Components

- **Breadcrumb** - Root navigation container with `aria-label="breadcrumb"`
- **BreadcrumbList** - Ordered list wrapper with responsive flex styling
- **BreadcrumbItem** - Individual breadcrumb item container
- **BreadcrumbLink** - Clickable breadcrumb link with hover states and Slot support
- **BreadcrumbPage** - Current page indicator with `aria-current="page"`
- **BreadcrumbSeparator** - Customizable separator (defaults to ChevronRight icon)
- **BreadcrumbEllipsis** - Truncation indicator for long breadcrumb trails

## Usage Example

```typescript
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
  BreadcrumbEllipsis,
} from "./breadcrumb"

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/docs">Documentation</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbEllipsis />
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```
