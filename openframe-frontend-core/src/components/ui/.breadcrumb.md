A complete breadcrumb navigation component system built with React and Radix UI primitives. This file provides accessible, composable components for creating navigation breadcrumbs with proper semantic HTML and ARIA attributes.

## Key Components

- **Breadcrumb**: Main navigation wrapper with `nav` element and breadcrumb aria-label
- **BreadcrumbList**: Ordered list container with responsive styling
- **BreadcrumbItem**: Individual breadcrumb item wrapper (`li` element)
- **BreadcrumbLink**: Clickable breadcrumb links with hover states and `asChild` prop support
- **BreadcrumbPage**: Current page indicator (non-clickable) with proper ARIA attributes
- **BreadcrumbSeparator**: Visual separator between items (defaults to ChevronRight icon)
- **BreadcrumbEllipsis**: Truncation indicator using MoreHorizontal icon

## Usage Example

```typescript
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator
} from "./breadcrumb"

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/products">Products</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```
