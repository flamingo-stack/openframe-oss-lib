A reusable selection card component that provides an interactive interface for choosing options with visual feedback, styled with a dark theme and golden selection state.

## Key Components

- **SelectCard**: Main component that renders a clickable card with title, optional subtitle, icon, and selection state
- **SelectCardProps**: Interface extending HTML div attributes with card-specific properties
- **Conditional rendering**: Device type icons, custom icons, subtitles, and selection indicators
- **Styling states**: Default, hover, and selected appearances with consistent color theming

## Usage Example

```typescript
import { SelectCard } from './select-card'
import { Monitor } from 'lucide-react'

function DeviceSelector() {
  const [selected, setSelected] = useState<string | null>(null)

  return (
    <div className="space-y-3">
      <SelectCard
        title="Desktop Computer"
        subtitle="High performance workstation"
        type="desktop"
        selected={selected === 'desktop'}
        onSelect={() => setSelected('desktop')}
      />
      
      <SelectCard
        title="Custom Device"
        subtitle="Configure manually"
        icon={<Monitor />}
        selected={selected === 'custom'}
        onSelect={() => setSelected('custom')}
      />
    </div>
  )
}
```

The component automatically handles visual states, displays check marks when selected, and supports both device type icons and custom React nodes.
