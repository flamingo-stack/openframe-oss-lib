A reusable React component for rendering selectable card items with device type icons, customizable content, and visual feedback for selection states.

## Key Components

- **SelectCard**: Main component that renders a clickable card with icon, title, subtitle, and selection indicator
- **SelectCardProps**: TypeScript interface defining component props including title, device type, custom icon, selection state, and event handlers
- **Device Type Integration**: Uses `getDeviceTypeIcon` utility for automatic device-specific iconography
- **Selection States**: Visual feedback with hover effects, selected styling, and check mark indicator

## Usage Example

```typescript
import { SelectCard } from './select-card'
import { Smartphone } from 'lucide-react'

function DeviceSelector() {
  const [selectedDevice, setSelectedDevice] = useState<string>()

  return (
    <div className="space-y-2">
      <SelectCard
        title="iPhone 15 Pro"
        type="mobile"
        subtitle="iOS 17.0"
        selected={selectedDevice === 'iphone'}
        onSelect={() => setSelectedDevice('iphone')}
      />
      
      <SelectCard
        title="Custom Device"
        icon={<Smartphone />}
        subtitle="Custom configuration"
        selected={selectedDevice === 'custom'}
        onSelect={() => setSelectedDevice('custom')}
      />
    </div>
  )
}
```

The component features a dark theme with yellow accent colors for selected states, smooth transitions, and responsive design. It accepts either a `type` prop for automatic device icons or a custom `icon` prop for flexibility.
