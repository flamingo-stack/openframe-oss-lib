A date picker component built with Radix UI Popover and react-day-picker, supporting both single date and date range selection modes with customizable styling and locale support.

## Key Components

### Types
- **`DatePickerMode`** - Union type for picker modes: "single" | "range"
- **`DatePickerBaseProps`** - Shared props interface with placeholder, formatting, and constraints
- **`SingleDatePickerProps`** - Props for single date selection mode
- **`RangeDatePickerProps`** - Props for date range selection mode
- **`DatePickerProps`** - Union of single and range props

### Components
- **`CalendarNavButton`** - Navigation button for previous/next month controls
- **`DatePickerCalendar`** - Core calendar component with month navigation and date selection
- **Helper functions** - `defaultFormatDate` and `formatDateRange` for date formatting

## Usage Example

```typescript
import { DatePicker } from "./date-picker";

// Single date picker
<DatePicker
  mode="single"
  value={selectedDate}
  onChange={setSelectedDate}
  placeholder="Select a date"
  formatDate={(date) => date.toLocaleDateString()}
/>

// Date range picker
<DatePicker
  mode="range"
  value={selectedRange}
  onChange={setSelectedRange}
  numberOfMonths={2}
  placeholder="Select date range"
  fromDate={new Date()}
  toDate={new Date(Date.now() + 365 * 24 * 60 * 60 * 1000)}
/>
```

The component features custom dark theme styling with Flamingo brand colors (#ffc008), responsive design, and accessibility support through ARIA labels and keyboard navigation.
