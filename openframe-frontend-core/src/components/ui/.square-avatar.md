A flexible avatar component that displays user images with automatic fallback to initials when no image is provided.

## Key Components

- **`SquareAvatar`** - Main avatar component supporting both image and text fallbacks
- **`getInitials`** - Utility function that extracts initials from names (first + last)
- **`SquareAvatarProps`** - Interface extending HTML div attributes with avatar-specific props

## Usage Example

```typescript
import { SquareAvatar } from './components/square-avatar';

// With image
<SquareAvatar 
  src="/user-photo.jpg" 
  alt="John Doe" 
  size="lg" 
  variant="round" 
/>

// Fallback to initials
<SquareAvatar 
  fallback="Jane Smith" 
  size="md" 
  variant="square" 
/>

// Custom styling
<SquareAvatar 
  src="/avatar.png"
  alt="User Avatar"
  size="xl"
  className="border-2 border-primary"
/>
```

The component automatically falls back to displaying initials when no image source is provided, with support for four sizes (sm, md, lg, xl) and two variants (square, round). It uses Tailwind CSS classes and forwards refs for full integration with React patterns.
