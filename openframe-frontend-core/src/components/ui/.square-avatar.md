A flexible avatar component that displays user profile images with automatic fallback to initials when no image is provided, supporting both square and round variants with multiple size options.

## Key Components

- **`getInitials`** - Utility function that extracts initials from a name (first letter of first and last word)
- **`SquareAvatar`** - Main avatar component with image display and fallback functionality
- **`SquareAvatarProps`** - TypeScript interface extending HTML div attributes with avatar-specific props

### Props

- `src` - Image source URL (optional)
- `alt` - Alt text for the image
- `size` - Predefined sizes: `'sm' | 'md' | 'lg' | 'xl'`
- `fallback` - Text to generate initials from when no image is provided
- `variant` - Shape variant: `'square'` (rounded corners) or `'round'` (circular)

## Usage Example

```typescript
import { SquareAvatar } from './components/square-avatar';

// With image
<SquareAvatar 
  src="/user-photo.jpg" 
  alt="John Doe" 
  size="lg" 
  variant="round" 
/>

// Fallback to initials
<SquareAvatar 
  fallback="John Doe" 
  alt="John Doe" 
  size="md" 
  variant="square" 
/>

// Minimal usage with question mark fallback
<SquareAvatar alt="Unknown User" />
```

The component automatically falls back to displaying initials in a muted background when no image source is provided, and shows a question mark when no fallback text is available.
