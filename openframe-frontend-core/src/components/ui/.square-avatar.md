A flexible avatar component that renders a user image with an automatic initials-based fallback, supporting multiple sizes and shape variants.

## Key Components

### `SquareAvatarProps`
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image URL |
| `alt` | `string` | — | Image alt text (also used for initials fallback) |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Controls dimensions (32/40/48/64px) |
| `fallback` | `string` | — | Preferred source string for generating initials |
| `variant` | `'square' \| 'round'` | `'square'` | Border radius style |
| `initialsClassName` | `string` | — | Tailwind overrides for initials font size/color |

### `SquareAvatar`
A memoized, forwarded-ref `div` component. Renders an optimized `next/image` when `src` is provided; falls back to extracted initials via `getFirstLastInitials()`. On image load error, the image is hidden and initials are revealed automatically.

## Usage Example

```typescript
// Basic image avatar
<SquareAvatar
  src="https://example.com/avatar.png"
  alt="Jane Doe"
  size="md"
/>

// Initials-only fallback (no src)
<SquareAvatar
  fallback="Jane Doe"
  size="lg"
  variant="round"
/>

// Compact avatar with custom initials styling
<SquareAvatar
  src={user.avatarUrl}
  fallback={user.displayName}
  size="sm"
  initialsClassName="text-[10px] text-ods-flamingo-pink"
/>

// Accent-fill background (e.g. current user indicator)
<SquareAvatar
  fallback={currentUser.name}
  size="md"
  className="bg-ods-flamingo-pink"
/>
```

> **Accessibility note:** Initials use `text-ods-text-primary` to maintain WCAG AA contrast on both default (`bg-ods-bg`) and accent-fill backgrounds (`bg-ods-flamingo-pink`, `bg-ods-flamingo-cyan`).
