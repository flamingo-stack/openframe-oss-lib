A customizable textarea component built with React and forwardRef that provides consistent styling and behavior matching the design system's input components.

## Key Components

- **TextareaProps** - Interface extending React's native textarea HTML attributes
- **Textarea** - Forwardable React component with built-in styling and theming

## Usage Example

```typescript
import { Textarea } from "./components/ui/textarea"

function ContactForm() {
  return (
    <div>
      <label htmlFor="message">Message</label>
      <Textarea
        id="message"
        placeholder="Enter your message..."
        rows={4}
        className="mb-4"
      />
      
      {/* With ref */}
      <Textarea
        ref={textareaRef}
        disabled={isSubmitting}
        onChange={(e) => setMessage(e.target.value)}
      />
    </div>
  )
}
```

The component features responsive typography, dark theme styling with ODS design tokens, smooth transitions, focus states with accent colors, and vertical resize capability. It maintains visual consistency with other form inputs while providing the multi-line text input functionality of a textarea.
