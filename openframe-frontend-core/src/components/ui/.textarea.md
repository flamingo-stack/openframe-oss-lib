A customizable textarea component built with React forwardRef that provides consistent styling and error state handling for form inputs in the Flamingo design system.

## Key Components

- **TextareaProps**: Interface extending React's textarea attributes with an `invalid` prop for error states
- **Textarea**: Forwardref component that renders a styled textarea with comprehensive visual states
- **cn**: Utility function for conditional class name merging

The component features dark theme styling, focus states with accent color rings, hover effects, and vertical resizing capability. Error states are handled through the `invalid` prop which applies red border styling.

## Usage Example

```typescript
import { Textarea } from "./textarea"
import { useState } from "react"

function CommentForm() {
  const [comment, setComment] = useState("")
  const [hasError, setHasError] = useState(false)

  return (
    <div className="space-y-4">
      <Textarea 
        placeholder="Enter your comment..."
        value={comment}
        onChange={(e) => setComment(e.target.value)}
        invalid={hasError}
        rows={4}
      />
      
      <Textarea
        placeholder="Optional feedback"
        className="min-h-[120px]"
        disabled
      />
    </div>
  )
}
```

The component automatically handles focus management, supports all standard textarea attributes, and maintains visual consistency with the broader form input system through shared design tokens and responsive behaviors.
