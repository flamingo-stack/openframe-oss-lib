A React textarea component with consistent styling that matches the design system's Input component, featuring responsive typography and admin theme colors.

## Key Components

- **Textarea** - Forwardable React component extending HTML textarea attributes
- **TextareaProps** - TypeScript interface extending native textarea HTML attributes
- **cn** utility - For conditional CSS class merging

## Usage Example

```typescript
import { Textarea } from "./components/ui/textarea"

function ContactForm() {
  return (
    <div className="space-y-4">
      <Textarea 
        placeholder="Enter your message..."
        className="min-h-[120px]"
      />
      
      <Textarea 
        value={message}
        onChange={(e) => setMessage(e.target.value)}
        disabled={isLoading}
        rows={6}
      />
    </div>
  )
}
```

The component provides:
- **Design System Consistency** - Typography, colors, and focus states match Input components
- **Admin Theme** - Dark background (`#161616`) with accent colors and proper contrast
- **Responsive Design** - Text scales from 14px on mobile to 18px on desktop
- **Accessibility** - Proper focus indicators, disabled states, and touch optimization
- **Flexibility** - Vertical resize capability and customizable minimum height
- **Forward Ref Support** - Direct access to underlying textarea element for form libraries
