A comprehensive dropdown menu component library built on top of Radix UI primitives with custom styling and enhanced functionality. Provides a complete set of dropdown menu components including triggers, content containers, items, separators, and interactive elements.

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Button or element that opens the dropdown
- **DropdownMenuContent** - Main container for dropdown items with portal rendering
- **DropdownMenuItem** - Individual selectable menu item
- **DropdownMenuCheckboxItem** - Menu item with checkbox functionality
- **DropdownMenuRadioItem** - Radio button menu item for single selection
- **DropdownMenuLabel** - Section label for grouping items
- **DropdownMenuSeparator** - Visual divider between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub*** - Components for nested submenus
- **DropdownMenuGroup** - Container for grouping related items

## Usage Example

```typescript
import { 
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from "./dropdown-menu"

function MyDropdown() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <button>Open Menu</button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuLabel>My Account</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem>Profile</DropdownMenuItem>
        <DropdownMenuItem>Settings</DropdownMenuItem>
        <DropdownMenuItem>Logout</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```
