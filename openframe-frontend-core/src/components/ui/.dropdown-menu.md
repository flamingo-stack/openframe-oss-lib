A comprehensive React dropdown menu component library built on top of Radix UI primitives with custom styling and design system integration. Provides accessible dropdown menus with support for nested submenus, checkbox items, radio groups, and keyboard navigation.

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Element that opens the dropdown when clicked
- **DropdownMenuContent** - Main dropdown panel with Portal rendering
- **DropdownMenuItem** - Basic selectable menu item with hover/focus states
- **DropdownMenuCheckboxItem** - Checkbox item with check indicator
- **DropdownMenuRadioItem** - Radio button item with circle indicator  
- **DropdownMenuLabel** - Non-interactive label for grouping items
- **DropdownMenuSeparator** - Visual divider between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub/SubTrigger/SubContent** - Nested submenu components
- **DropdownMenuGroup/RadioGroup** - Logical grouping containers

## Usage Example

```typescript
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
  DropdownMenuCheckboxItem,
  DropdownMenuShortcut,
} from "./dropdown-menu"

function UserMenu() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline">Open Menu</Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent className="w-56">
        <DropdownMenuLabel>My Account</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem>
          Profile
          <DropdownMenuShortcut>⇧⌘P</DropdownMenuShortcut>
        </DropdownMenuItem>
        <DropdownMenuCheckboxItem checked>
          Show Notifications
        </DropdownMenuCheckboxItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```
