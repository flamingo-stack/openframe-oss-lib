A comprehensive dropdown menu component built on top of Radix UI's dropdown menu primitives, providing a full suite of styled menu components with consistent theming using the OpenFrame Design System (ODS).

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Button that opens the dropdown
- **DropdownMenuContent** - Main content container with portal rendering
- **DropdownMenuItem** - Interactive menu items with hover states
- **DropdownMenuCheckboxItem** - Checkable menu items with check indicators
- **DropdownMenuRadioItem** - Radio button menu items for single selection
- **DropdownMenuLabel** - Non-interactive section labels
- **DropdownMenuSeparator** - Visual dividers between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub/SubTrigger/SubContent** - Nested submenu components
- **DropdownMenuGroup/RadioGroup** - Grouping containers

## Usage Example

```typescript
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from "./dropdown-menu"

function UserMenu() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger>Open Menu</DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuLabel>My Account</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem>Profile</DropdownMenuItem>
        <DropdownMenuItem>Settings</DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem>Logout</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```

All components use ODS tokens for consistent theming and include smooth animations, focus management, and accessibility features from Radix UI.
