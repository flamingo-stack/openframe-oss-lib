A reusable React component for managing a collection of tags with add/remove functionality, input validation, and customizable styling options.

## Key Components

- **TagsInput**: Main component that renders an input field, add button, and displays existing tags as removable badges
- **TagsInputProps**: Interface defining component props including value array, onChange handler, styling options, and validation settings
- **handleAddTag**: Function that validates and adds new tags while preventing duplicates and enforcing max tag limits
- **handleRemoveTag**: Function that removes specific tags from the collection

## Usage Example

```typescript
import { TagsInput } from "./tags-input"
import { useState } from "react"

function MyForm() {
  const [tags, setTags] = useState<string[]>([])

  return (
    <TagsInput
      value={tags}
      onChange={setTags}
      label="Skills"
      placeholder="Add a skill..."
      maxTags={5}
      className="w-full"
      badgeClassName="bg-blue-100"
    />
  )
}
```

The component supports keyboard interaction (Enter to add), prevents duplicate tags, enforces maximum tag limits, and provides visual feedback with tag count display. Tags are displayed as badges with individual remove buttons when not disabled.
