A responsive page actions component that renders different button layouts optimized for desktop and mobile viewports, with support for primary buttons, icon buttons, and dropdown menus.

## Key Components

- **PageActions**: Main component with three variants (`icon-buttons`, `primary-buttons`, `menu-primary`)
- **PageActionButton**: Type definition for action buttons with label, onClick, icon, variant, and responsive visibility options
- **IconButtonsVariant**: Shows labeled buttons on desktop, single icon button or dropdown menu on mobile
- **PrimaryButtonsVariant**: Displays normal button layout on desktop, fixed bottom bar on mobile
- **MenuPrimaryVariant**: Combines dropdown menu with primary buttons on desktop, unified bottom bar on mobile
- **usePageActionsBottomPadding**: Hook to add bottom padding when using mobile bottom bars

## Usage Example

```typescript
import { PageActions, usePageActionsBottomPadding } from './page-actions'
import { EditIcon, DeleteIcon, ShareIcon } from '../icons'

function MyPage() {
  const paddingClass = usePageActionsBottomPadding('primary-buttons')

  const actions = [
    {
      label: 'Edit',
      onClick: () => console.log('edit'),
      icon: <EditIcon />,
      variant: 'outline' as const
    },
    {
      label: 'Save',
      onClick: () => console.log('save'),
      variant: 'primary' as const,
      loading: false
    }
  ]

  const menuActions = [
    {
      label: 'Share',
      onClick: () => console.log('share'),
      icon: <ShareIcon />
    },
    {
      label: 'Delete',
      onClick: () => console.log('delete'),
      icon: <DeleteIcon />,
      danger: true
    }
  ]

  return (
    <div className={paddingClass}>
      <PageActions
        variant="menu-primary"
        actions={actions}
        menuActions={menuActions}
        gap="md"
      />
    </div>
  )
}
```
