A responsive React component that renders page-level action buttons with adaptive layouts for desktop and mobile views, supporting multiple display variants including icon buttons, primary buttons, and menu combinations.

## Key Components

- **`PageActions`** - Main component that renders action buttons based on variant
- **`IconButtonsVariant`** - Shows buttons with icons on desktop, collapses to menu on mobile
- **`PrimaryButtonsVariant`** - Displays primary/outline buttons, uses fixed bottom bar on mobile
- **`MenuPrimaryVariant`** - Combines dropdown menu with primary buttons
- **`usePageActionsBottomPadding`** - Hook that returns padding classes for mobile bottom bar spacing

## Usage Example

```typescript
import { PageActions, usePageActionsBottomPadding } from './page-actions'

function MyPage() {
  const actions = [
    {
      label: 'Save',
      onClick: () => console.log('Save clicked'),
      icon: <SaveIcon />,
      variant: 'primary' as const
    },
    {
      label: 'Cancel',
      onClick: () => console.log('Cancel clicked'),
      variant: 'outline' as const
    }
  ]

  const menuActions = [
    { label: 'Delete', onClick: () => {}, icon: <DeleteIcon />, danger: true }
  ]

  const paddingClass = usePageActionsBottomPadding('menu-primary')

  return (
    <div className={paddingClass}>
      <PageActions 
        variant="menu-primary"
        actions={actions}
        menuActions={menuActions}
        gap="md"
      />
    </div>
  )
}
```

The component automatically adapts between desktop (horizontal layout) and mobile (fixed bottom bar or collapsed menu) based on screen size, with three variants optimized for different use cases.
