A flexible React component for displaying features or content in a responsive grid layout with customizable borders, backgrounds, and styling options.

## Key Components

- **`FeatureCardItem`**: Interface defining individual card properties including icon, title, badge, content, and styling overrides
- **`FeatureCardGridProps`**: Configuration interface for the grid layout, columns, borders, and spacing
- **`FeatureCardGrid`**: Main component that renders items in a responsive grid with intelligent border handling

## Usage Example

```typescript
import { FeatureCardGrid } from './feature-card';
import { SomeIcon } from './icons';

const features = [
  {
    icon: SomeIcon,
    iconColor: '#3b82f6',
    title: 'Feature Title',
    badge: { text: 'NEW', variant: 'card', colorScheme: 'green' },
    content: <p>Feature description content</p>
  },
  {
    title: 'Another Feature',
    content: <div>Rich content here</div>,
    noBackground: true
  }
];

<FeatureCardGrid
  items={features}
  columns={3}
  showBorders={true}
  roundedCorners={true}
  cardGap="gap-6"
/>
```

The component automatically handles responsive layouts (1 column on mobile, 2-3 on desktop) and provides granular control over borders, backgrounds, and spacing for each individual card.
