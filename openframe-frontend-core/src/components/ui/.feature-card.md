A client-side React component that renders a responsive grid of feature cards with customizable styling, borders, and content layout.

## Key Components

### Types
- **`FeatureCardItem`** - Interface defining individual card properties including icon, title, badge, content, and styling overrides
- **`FeatureCardGridProps`** - Configuration interface for the grid layout, columns, spacing, and global styling options

### Components
- **`FeatureCardGrid`** - Main component that renders a responsive grid of feature cards with intelligent border management and customizable layouts

## Usage Example

```typescript
import { FeatureCardGrid } from './feature-card';
import { SomeIcon } from './icons';

const features = [
  {
    icon: SomeIcon,
    iconColor: '#0EA5E9',
    title: 'Advanced Analytics',
    badge: {
      text: 'New',
      variant: 'card' as const,
      colorScheme: 'cyan' as const
    },
    content: <p>Real-time data insights and reporting</p>
  },
  {
    title: 'Custom Integration',
    content: <div>Flexible API connections</div>,
    noBackground: true
  }
];

<FeatureCardGrid 
  items={features}
  columns={2}
  showBorders={true}
  roundedCorners={true}
  cardGap="gap-6"
/>
```

The component features intelligent responsive border handling, supports 2-3 column layouts, and provides extensive customization options for backgrounds, borders, and spacing. Each card can override global styles with item-specific properties like `noBackground` or `removeAllBorders`.
