A React slider component that provides a customizable range input with cross-browser styling and array-based value handling for consistency with multi-range slider APIs.

## Key Components

- **`Slider`** - Main slider component built on HTML range input
- **`SliderProps`** - Interface extending HTML input props with custom value handling
- **`handleChange`** - Internal handler that converts single values to array format

## Usage Example

```typescript
import { Slider } from './slider'

function VolumeControl() {
  const [volume, setVolume] = useState([50])

  return (
    <div className="w-64">
      <label>Volume: {volume[0]}%</label>
      <Slider
        value={volume}
        onValueChange={setVolume}
        min={0}
        max={100}
        step={5}
        className="mt-2"
      />
    </div>
  )
}

// Basic usage with default range (0-100)
function SimpleSlider() {
  return (
    <Slider
      onValueChange={(value) => console.log('Value:', value[0])}
    />
  )
}
```

The component uses array-based values (`number[]`) to maintain API consistency with potential multi-range implementations, while internally handling single range inputs. It includes comprehensive cross-browser styling for webkit and Mozilla slider thumbs.
