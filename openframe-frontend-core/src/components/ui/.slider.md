A customizable range slider component built on top of HTML's native input[type="range"] with enhanced styling and TypeScript support.

## Key Components

- **Slider**: Forward-ref React component that extends HTML input attributes
- **SliderProps**: Interface defining component props including value array, change handler, and range constraints
- **handleChange**: Internal handler that converts single input values to array format for consistency

## Usage Example

```typescript
import { Slider } from "./components/ui/slider"
import { useState } from "react"

function VolumeControl() {
  const [volume, setVolume] = useState([50])
  
  return (
    <div className="w-full max-w-md">
      <label>Volume: {volume[0]}%</label>
      <Slider
        value={volume}
        onValueChange={setVolume}
        min={0}
        max={100}
        step={5}
        className="mt-2"
      />
    </div>
  )
}

// With custom range
<Slider
  value={[25]}
  onValueChange={(val) => console.log(val)}
  min={-10}
  max={50}
  step={0.5}
/>
```

The component uses array values for API consistency (likely with multi-range sliders) and includes cross-browser thumb styling for webkit and Firefox.
