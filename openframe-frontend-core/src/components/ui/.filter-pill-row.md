Responsive filter pill row component that renders a labeled row of selectable filter buttons inside a card container, used across catalog and library views.

## Key Components

### `FilterPillRow`
Main component rendering a card-style filter bar with an icon, label, pill buttons, and optional count display.

### Interfaces

| Interface | Description |
|---|---|
| `FilterPillRowOption` | `value` + `label` pair for each filter option |
| `FilterPillRowProps` | Full props definition for the component |

### Props

| Prop | Type | Required | Description |
|---|---|---|---|
| `label` | `string` | ✅ | Filter section label (e.g. `"Platform"`) |
| `selectedValue` | `string` | ✅ | Currently active filter value |
| `onValueChange` | `(value: string) => void` | ✅ | Selection change callback |
| `options` | `FilterPillRowOption[]` | ✅ | Available filter pills |
| `countLabel` | `string` | ❌ | Right-aligned count text (e.g. `"Showing 1-10 of 42"`) |
| `children` | `ReactNode` | ❌ | Replaces default pills with custom filter content |

## Usage Example

```typescript
import { FilterPillRow } from '@/components/ui/filter-pill-row'

const PLATFORM_OPTIONS = [
  { value: 'all', label: 'All' },
  { value: 'windows', label: 'Windows' },
  { value: 'linux', label: 'Linux' },
  { value: 'macos', label: 'macOS' },
]

export function PlatformFilter() {
  const [platform, setPlatform] = useState('all')

  return (
    <FilterPillRow
      label="Platform"
      selectedValue={platform}
      onValueChange={setPlatform}
      options={PLATFORM_OPTIONS}
      countLabel="Showing 12 of 48 scripts"
    />
  )
}
```

> Pass `children` instead of `options` when you need custom filter controls (e.g. date pickers, search inputs) while keeping the consistent card layout.
