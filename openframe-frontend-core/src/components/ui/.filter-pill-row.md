Responsive filter pill row component that renders a labeled set of toggle buttons inside a styled card container, used across catalog and admin views to filter lists by a single selected value.

## Key Components

**`FilterPillRow`** — Main export. Renders a horizontal (flex-wrap) row with a `Filter` icon, a label, pill-style toggle buttons, and an optional count label.

**`FilterPillRowOption`** — Shape of each filter option: `{ value: string, label: string }`.

**`FilterPillRowProps`** — Full prop interface:

| Prop | Type | Required | Description |
|---|---|---|---|
| `label` | `string` | ✅ | Text next to the filter icon (e.g. `"Section"`) |
| `selectedValue` | `string` | ✅ | Currently active filter value |
| `onValueChange` | `(value: string) => void` | ✅ | Selection change callback |
| `options` | `FilterPillRowOption[]` | ✅ | Available filter pills |
| `countLabel` | `string` | ❌ | Right-aligned summary (e.g. `"Showing 1–10 of 42"`) |
| `children` | `ReactNode` | ❌ | Replaces `options` rendering for custom content |

## Usage Example

```typescript
import { FilterPillRow } from '@/components/ui/filter-pill-row'

const PLATFORM_OPTIONS = [
  { value: 'all', label: 'All' },
  { value: 'windows', label: 'Windows' },
  { value: 'linux', label: 'Linux' },
  { value: 'macos', label: 'macOS' },
]

export function PlatformFilter() {
  const [platform, setPlatform] = useState('all')

  return (
    <FilterPillRow
      label="Platform"
      selectedValue={platform}
      onValueChange={setPlatform}
      options={PLATFORM_OPTIONS}
      countLabel="Showing 12 of 48 scripts"
    />
  )
}
```

> **Note:** Pass `children` instead of `options` when you need custom filter controls (e.g. a date range picker or search input) while keeping the shared card layout.
