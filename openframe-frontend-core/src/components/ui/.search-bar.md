A React search bar component with customizable icons, controlled/uncontrolled state support, and loading states. The component provides a dark-themed search input with flexible styling options.

## Key Components

- **SearchBar**: Main component that renders a search input with icon support
- **SearchBarProps**: Interface defining component props including `onSubmit`, `loading`, `icon`, and standard input attributes
- **Controlled/Uncontrolled modes**: Supports both external state management and internal state

## Usage Example

```typescript
import { SearchBar } from './components/search-bar'
import { Filter } from 'lucide-react'

// Basic usage
<SearchBar 
  placeholder="Search products..." 
  onSubmit={(value) => console.log('Search:', value)} 
/>

// With custom icon and loading state
<SearchBar
  icon={<Filter />}
  loading={true}
  placeholder="Filter results..."
  onSubmit={handleSearch}
/>

// Controlled component
const [searchValue, setSearchValue] = useState('')

<SearchBar
  value={searchValue}
  onSubmit={(value) => {
    setSearchValue(value)
    performSearch(value)
  }}
  disabled={isSearching}
/>
```
