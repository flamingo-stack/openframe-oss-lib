A reusable search input component with loading state, customizable icon, and both controlled and uncontrolled value management.

## Key Components

- **`SearchBar`** - Main search input component with icon support
- **`SearchBarProps`** - TypeScript interface defining component props including icon, placeholder, onSubmit callback, loading state, and standard input attributes

## Usage Example

```typescript
import { SearchBar } from './search-bar'
import { UserIcon } from 'lucide-react'

// Uncontrolled usage
<SearchBar
  placeholder="Search users..."
  onSubmit={(value) => handleSearch(value)}
  loading={isSearching}
/>

// Controlled usage with custom icon
const [searchQuery, setSearchQuery] = useState('')

<SearchBar
  icon={<UserIcon />}
  value={searchQuery}
  onSubmit={setSearchQuery}
  placeholder="Find team members..."
  disabled={!hasPermission}
/>

// With custom styling
<SearchBar
  className="max-w-md"
  placeholder="Search documentation..."
  onSubmit={(query) => performSearch(query)}
/>
```

The component automatically handles both controlled and uncontrolled states, displays a loading spinner when `loading` is true, and includes responsive sizing with dark theme styling optimized for the Flamingo design system.
