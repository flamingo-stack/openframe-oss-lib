A full-screen image gallery modal component with keyboard navigation, prev/next buttons, and image counter functionality. Includes error handling for unsupported formats like HEIC files.

## Key Components

**`ImageGalleryModal`** - Main export component that renders a modal with image gallery functionality

**Props:**
- `images`: Array of image URLs to display
- `isOpen`: Boolean to control modal visibility
- `onClose`: Callback function to close the modal
- `initialIndex`: Optional starting image index (defaults to 0)

**Features:**
- Keyboard navigation (arrow keys, escape)
- Navigation buttons for multi-image galleries
- Image counter display
- Error handling with download fallback
- Responsive design with viewport-relative sizing

## Usage Example

```typescript
import { ImageGalleryModal } from './image-gallery-modal';

function MyComponent() {
  const [isGalleryOpen, setIsGalleryOpen] = useState(false);
  const images = ['/image1.jpg', '/image2.png', '/image3.heic'];

  return (
    <>
      <button onClick={() => setIsGalleryOpen(true)}>
        View Gallery
      </button>
      
      <ImageGalleryModal
        images={images}
        isOpen={isGalleryOpen}
        onClose={() => setIsGalleryOpen(false)}
        initialIndex={0}
      />
    </>
  );
}
```
