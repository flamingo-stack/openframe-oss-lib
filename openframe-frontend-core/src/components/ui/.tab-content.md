A React component that renders dynamic tab content with automatic fallback to an empty state when no component is provided.

## Key Components

- **TabContent**: Main component that conditionally renders either a dynamic tab component or an empty state
- **TabContentProps**: TypeScript interface defining component props including active tab name, component to render, and customization options
- **Empty State**: Built-in fallback UI with customizable title and description for missing or invalid tabs

## Usage Example

```typescript
import { TabContent } from './tab-content'
import { UserProfile } from './user-profile'

function MyTabs() {
  const [activeTab, setActiveTab] = useState('profile')
  
  // Component mapping
  const tabComponents = {
    profile: UserProfile,
    settings: null // Will show empty state
  }
  
  return (
    <TabContent
      activeTab={activeTab}
      TabComponent={tabComponents[activeTab]}
      componentProps={{ userId: '123' }}
      className="bg-white rounded-lg"
      emptyStateTitle="Coming Soon"
      emptyStateDescription="This feature is under development"
      minHeight="min-h-[500px]"
    />
  )
}
```

The component automatically handles missing components by displaying a centered empty state with customizable messaging, making it ideal for tab interfaces where some tabs may not be implemented yet.
