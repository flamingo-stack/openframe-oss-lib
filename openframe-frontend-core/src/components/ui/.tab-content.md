A React component that renders dynamic tab content with fallback empty state handling when no component is provided.

## Key Components

- **TabContent**: Main component that conditionally renders either a dynamic tab component or an empty state message
- **TabContentProps**: Interface defining component properties including active tab name, component to render, styling options, and empty state customization
- **cn utility**: Used for conditional CSS class merging

## Usage Example

```typescript
import { TabContent } from './tab-content'
import { UserProfile } from './user-profile'

function TabContainer() {
  const [activeTab, setActiveTab] = useState('profile')
  
  return (
    <TabContent
      activeTab="profile"
      TabComponent={UserProfile}
      componentProps={{ userId: '123' }}
      className="p-4 border rounded-lg"
      emptyStateTitle="Content Unavailable"
      emptyStateDescription="This tab is currently under maintenance."
      minHeight="min-h-[500px]"
    />
  )
}

// Renders empty state when no component provided
<TabContent
  activeTab="missing-tab"
  TabComponent={null}
/>
```

The component automatically handles empty states with customizable titles and descriptions, making it ideal for dynamic tab systems where content may not always be available.
