A reusable React component that displays key-value pairs in a horizontal layout with a decorative divider line between the label and value.

## Key Components

- **InfoRow**: Main component that renders a label-value pair with optional icon
- **InfoRowProps**: TypeScript interface defining the component's props:
  - `label`: Display text for the left side
  - `value`: Display text for the right side  
  - `icon`: Optional React node (typically an icon) displayed after the value

## Usage Example

```typescript
import { InfoRow } from './info-row'
import { CheckIcon } from 'lucide-react'

// Basic usage
<InfoRow label="Status" value="Active" />

// With icon
<InfoRow 
  label="Email Verified" 
  value="Yes" 
  icon={<CheckIcon className="w-4 h-4 text-green-500" />} 
/>

// Multiple rows for a details section
<div className="space-y-2">
  <InfoRow label="Name" value="John Doe" />
  <InfoRow label="Role" value="Administrator" />
  <InfoRow label="Last Login" value="2 hours ago" />
</div>
```

The component features a clean design with DM Sans typography, ellipsis overflow handling, and a subtle gray divider line that automatically fills the available space between the label and value.
