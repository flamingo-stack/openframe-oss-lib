A reusable UI component that renders a labeled key-value row with a dotted separator line, commonly used in detail panels and summary cards.

## Key Components

### `InfoRow`
A client-side React component that displays a label and value on the same horizontal line, separated by a flexible spacer line. Optionally renders an icon next to the value.

**Props (`InfoRowProps`):**

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `label` | `string` | ✅ | Left-side descriptor text |
| `value` | `string` | ✅ | Right-side value text |
| `icon` | `React.ReactNode` | ❌ | Optional icon rendered alongside the value |

## Usage Example

```typescript
import { InfoRow } from "@/components/info-row"
import { CheckCircleIcon } from "lucide-react"

export function DeviceSummary() {
  return (
    <div className="flex flex-col gap-2">
      <InfoRow label="Hostname" value="workstation-01" />
      <InfoRow label="OS" value="Windows 11 Pro" />
      <InfoRow
        label="Status"
        value="Online"
        icon={<CheckCircleIcon className="w-4 h-4 text-green-500" />}
      />
    </div>
  )
}
```

> **Note:** Both `label` and `value` use `text-ellipsis` and `whitespace-nowrap`, so long strings are truncated rather than wrapping — ensure parent containers have a constrained width when displaying dynamic data.
