A reusable UI component that renders a labeled key-value row with a dotted/line separator, commonly used in detail panels and info cards.

## Key Components

### `InfoRow`
A client-side React component that displays a label and value pair separated by a horizontal line, with an optional icon rendered beside the value.

**Props (`InfoRowProps`):**

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `label` | `string` | ✅ | Left-side descriptor text |
| `value` | `string` | ✅ | Right-side display value |
| `icon` | `React.ReactNode` | ❌ | Optional icon rendered next to the value |

## Usage Example

```typescript
import { InfoRow } from "@/components/info-row"
import { CheckCircle } from "lucide-react"

// Basic usage
<InfoRow label="Status" value="Active" />

// With an icon
<InfoRow
  label="Uptime"
  value="99.9%"
  icon={<CheckCircle className="w-4 h-4 text-green-500" />}
/>

// In a detail card
<div className="flex flex-col gap-2">
  <InfoRow label="Hostname" value="server-01.local" />
  <InfoRow label="IP Address" value="192.168.1.10" />
  <InfoRow label="OS" value="Ubuntu 22.04" />
</div>
```

> The separator line (`flex-1 h-px`) stretches to fill available space between the label and value, ensuring consistent alignment regardless of text length. Both label and value use `text-ellipsis` to gracefully handle overflow.
