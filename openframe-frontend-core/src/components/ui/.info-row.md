A React component that displays a labeled information row with an optional icon, connected by a dotted line separator.

## Key Components

- **InfoRowProps** - TypeScript interface defining component props
  - `label: string` - Left-side label text
  - `value: string` - Right-side value text  
  - `icon?: React.ReactNode` - Optional icon displayed after the value
- **InfoRow** - Main component that renders a horizontal layout with label, separator line, and value

## Usage Example

```typescript
import { InfoRow } from './info-row'
import { CheckIcon } from 'lucide-react'

function UserProfile() {
  return (
    <div className="space-y-2">
      <InfoRow label="Name" value="John Doe" />
      <InfoRow label="Email" value="john@example.com" />
      <InfoRow 
        label="Status" 
        value="Active" 
        icon={<CheckIcon className="w-4 h-4 text-green-400" />} 
      />
    </div>
  )
}
```

The component uses a flex layout with DM Sans font, featuring text overflow handling and a flexible separator line that automatically adjusts to fill available space between the label and value.
