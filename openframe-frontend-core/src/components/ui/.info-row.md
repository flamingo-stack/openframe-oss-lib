A React component that renders a styled information row with a label, dotted separator line, and value with optional icon.

## Key Components

- **InfoRowProps** - Interface defining component props with label, value, and optional icon
- **InfoRow** - Main component that displays label-value pairs with visual separation

## Usage Example

```typescript
import { InfoRow } from './info-row'
import { Clock } from 'lucide-react'

function UserProfile() {
  return (
    <div className="space-y-4">
      <InfoRow 
        label="Email" 
        value="user@example.com" 
      />
      <InfoRow 
        label="Status" 
        value="Active" 
        icon={<Clock size={16} />}
      />
      <InfoRow 
        label="Role" 
        value="Administrator" 
      />
    </div>
  )
}
```

The component uses a flex layout with a dotted separator line that expands to fill available space between the label and value. Both text elements support overflow handling with ellipsis truncation. The optional icon appears after the value text.
