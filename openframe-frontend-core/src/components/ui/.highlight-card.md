A React component library for displaying highlighted category cards with responsive design and customizable grid layouts.

## Key Components

- **`HighlightCardItem`** - Interface defining card data structure with category, colors, and description
- **`HighlightCard`** - Individual card component displaying a highlighted category with dynamic background positioning
- **`HighlightCardProps`** - Props interface for the card component
- **`HighlightCardGrid`** - Grid container component for arranging multiple cards with responsive borders
- **`HighlightCardGridProps`** - Props interface for grid configuration (2 or 3 columns)

## Usage Example

```typescript
import { HighlightCard, HighlightCardGrid } from './highlight-card';

const cardData = {
  category: "Web",
  categoryColor: "#3B82F6",
  highlightBg: "#EFF6FF",
  description: "development projects"
};

const gridData = [
  { category: "Web", categoryColor: "#3B82F6", highlightBg: "#EFF6FF", description: "projects" },
  { category: "Mobile", categoryColor: "#10B981", highlightBg: "#ECFDF5", description: "apps" }
];

// Single card
<HighlightCard item={cardData} />

// Grid layout
<HighlightCardGrid 
  items={gridData}
  columns={2}
  className="my-4"
/>
```

The component features responsive text sizing, dynamic highlight background positioning based on category length, and sophisticated border management for seamless grid layouts.
