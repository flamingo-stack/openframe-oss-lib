A customizable segmented progress bar component with threshold-based color coding that adapts to container width and supports both normal and inverted color schemes.

## Key Components

- **ProgressBar**: Main React component that renders a horizontal progress bar using individual segments
- **ProgressBarProps**: TypeScript interface defining customizable properties including progress value, thresholds, dimensions, and color inversion
- **getColor()**: Internal function that determines segment color based on progress value and threshold settings
- **ResizeObserver**: Automatically calculates segment count based on container width

## Usage Example

```typescript
import { ProgressBar } from './progress-bar';

// Standard disk usage (high = bad)
<ProgressBar 
  progress={75} 
  warningThreshold={70}
  criticalThreshold={90}
/>

// Inverted for battery health (high = good)
<ProgressBar 
  progress={85}
  warningThreshold={30}
  criticalThreshold={70}
  inverted={true}
  height={20}
  segmentWidth={8}
/>

// Custom styling
<ProgressBar 
  progress={45}
  segmentWidth={4}
  segmentGap={1}
  height={12}
/>
```

The component automatically adjusts the number of segments based on available width and uses CSS custom properties for theming. Progress values range from 0-100, with customizable warning and critical thresholds that determine color transitions.
