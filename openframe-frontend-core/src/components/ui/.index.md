Central barrel export file for the Flamingo UI Kit library, re-exporting all client-side UI components organized by category for clean, single-import consumption across the platform.

## Key Components

Organized into the following export groups:

| Category | Components |
|----------|-----------|
| **UI Inputs** | `input`, `button`, `checkbox`, `select`, `switch`, `tags-input`, `date-picker`, `autocomplete`, `radio-group`, `textarea` |
| **Layout** | `dialog`, `modal`, `modal-v2`, `sheet`, `drawer`, `tabs`, `separator`, `alert-dialog` |
| **Navigation** | `accordion`, `breadcrumb`, `dropdown-menu`, `navigation-menu`, `tab-navigation` |
| **Feedback** | `alert`, `badge`, `progress`, `status-badge`, `toaster`, `skeleton` |
| **Table** | `Table`, `TableRow`, `TableHeader`, `TableCell`, `TableEmptyState`, `CursorPagination`, `DataTable` |
| **Query Report** | `QueryReportTable`, `deriveColumns`, `exportToCSV` |
| **Search & Filter** | `search-input`, `filter-list`, `tag-search-input`, `tag-key-value-filter` |
| **Ticket** | `ticket-status-tag`, `assignee-dropdown`, `ticket-notes-section`, `ticket-attachments-list` |
| **Chat & Layout** | `chat`, `page-container`, `page-heading`, `page-layout` |
| **Misc** | `FloatingTooltip`, `TruncateText`, `markdown-editor`, `file-upload`, `phone-input` |

```typescript
  TruncateText,

  CheckIcon,

```
