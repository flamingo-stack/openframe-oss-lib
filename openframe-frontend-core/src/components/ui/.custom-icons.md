A collection of reusable SVG icon components built with React for common UI interactions like checkmarks, close buttons, and status indicators.

## Key Components

- **CheckIcon** - Simple checkmark icon for success states and confirmations
- **XIcon** - Close/cancel icon for dismissing dialogs or clearing inputs
- **MinusIcon** - Horizontal line icon for collapsing content or decrementing values
- **CheckCircleIcon** - Checkmark within a circle for completed states and success messages
- **XCircleIcon** - X mark within a circle for error states and failed operations

All components accept an optional `className` prop for styling customization and use consistent SVG properties (24x24 viewBox, currentColor stroke, 2px stroke width).

## Usage Example

```typescript
import { CheckIcon, XIcon, CheckCircleIcon } from './custom-icons'

// Basic usage with Tailwind CSS classes
<CheckIcon className="w-5 h-5 text-green-500" />
<XIcon className="w-4 h-4 text-red-500 hover:text-red-700" />

// In a success notification
<div className="flex items-center gap-2">
  <CheckCircleIcon className="w-6 h-6 text-green-600" />
  <span>Operation completed successfully</span>
</div>

// As interactive button
<button className="p-2">
  <XIcon className="w-5 h-5" />
</button>
```
