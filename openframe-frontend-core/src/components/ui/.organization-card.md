A reusable React card component for displaying organization information with visual parity to OpenMSP's VendorCard styling. Features hover effects, organization icons, device counts, action buttons, and flexible footer stats.

## Key Components

- **OrganizationCard** - Main card component with hover effects and click navigation
- **Organization** - TypeScript interface defining organization data structure
- **OrganizationCardProps** - Comprehensive props interface with customization options
- **InteractiveCard** - Wrapper providing hover accent color functionality
- **OrganizationIcon** - 60x60px logo display with fallback initials

## Usage Example

```typescript
// Basic organization card
<OrganizationCard
  organization={{
    id: "org-123",
    name: "Acme Corp",
    industry: "Technology",
    description: "Leading software solutions provider"
  }}
/>

// Card with pre-fetched image and stats
const fetchedImages = useBatchImages(imageUrls)
<OrganizationCard
  organization={organization}
  fetchedImageUrl={fetchedImages[organization.imageUrl]}
  deviceCount={1250}
  footerStats={[
    { value: 1250, label: 'devices' },
    { value: '$5,000', label: 'MRR' }
  ]}
/>

// Card with action button
<OrganizationCard
  organization={organization}
  showActionButton
  actionButton={{
    icon: <Trash2 className="h-4 w-4" />,
    label: 'Remove',
    onClick: (org) => handleRemove(org.id)
  }}
/>
```
