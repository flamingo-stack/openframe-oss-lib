A reusable card component for displaying organization information with VendorCard-style hover effects and interactive functionality. Integrates with Next.js navigation and provides flexible customization options for various organizational data display scenarios.

## Key Components

**Main Exports:**
- `OrganizationCard` - Primary card component with hover effects and navigation
- `Organization` - Interface defining organization data structure
- `OrganizationCardProps` - Component props interface

**Key Features:**
- Organization icon/logo display with 60x60px sizing
- Fixed height description section (48px) with text truncation
- Optional footer stats display with custom formatting
- Action buttons with ghost/primary variants
- Device count display in top-right corner
- Customizable hover accent colors

## Usage Example

```typescript
// Basic organization card
<OrganizationCard organization={org} />

// With pre-fetched image and footer stats
<OrganizationCard
  organization={org}
  fetchedImageUrl={fetchedImages[org.imageUrl]}
  footerStats={[
    { value: org.totalDevices, label: 'devices' },
    { value: `$${org.mrrUsd}`, label: 'MRR' }
  ]}
/>

// With action button
<OrganizationCard
  organization={org}
  showActionButton
  actionButton={{
    icon: <Trash2 className="h-4 w-4" />,
    label: 'Remove',
    onClick: (org) => handleRemove(org.id),
    variant: 'ghost'
  }}
/>

// With custom click handler
<OrganizationCard
  organization={org}
  onClick={(org) => setSelectedOrg(org)}
  deviceCount={125}
/>
```
