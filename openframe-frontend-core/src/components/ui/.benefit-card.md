A React component system for displaying benefits or features in card format with flexible grid layouts and multiple visual variants.

## Key Components

- **`BenefitCard`** - Individual benefit card component with icon, title, description, and three visual variants (default, dark, auth-figma)
- **`BenefitCardGrid`** - Container component that arranges BenefitCards in responsive grids with automatic border management
- **`BenefitCardProps`** - Interface defining card properties including optional icon, title, description, variant, and styling
- **`BenefitCardGridProps`** - Interface for grid container with support for 2, 3, or 4 column layouts

## Usage Example

```typescript
import { BenefitCard, BenefitCardGrid } from './benefit-card'
import { IconCheck } from 'lucide-react'

// Single card
<BenefitCard
  icon={<IconCheck size={24} />}
  title="Fast Performance"
  description="Lightning-fast load times and smooth interactions"
  variant="dark"
/>

// Grid of cards
<BenefitCardGrid columns={3}>
  <BenefitCard
    icon={<IconCheck />}
    title="Secure"
    description="Enterprise-grade security"
  />
  <BenefitCard
    icon={<IconCheck />}
    title="Scalable"
    description="Grows with your business"
  />
  <BenefitCard
    icon={<IconCheck />}
    title="Reliable"
    description="99.9% uptime guarantee"
  />
</BenefitCardGrid>
```

The grid automatically handles responsive layouts and applies proper borders between cards based on the column configuration.
