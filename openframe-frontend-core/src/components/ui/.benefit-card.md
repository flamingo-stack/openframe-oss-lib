A flexible React component system for displaying benefit cards with icons, titles, and descriptions, supporting multiple visual variants and responsive grid layouts.

## Key Components

- **`BenefitCard`** - Main card component with three style variants (`default`, `dark`, `auth-figma`)
- **`BenefitCardGrid`** - Responsive grid container supporting 2, 3, or 4-column layouts with automatic border management
- **`BenefitCardProps`** - Interface defining card properties including optional icon, title, description, variant, and styling
- **`BenefitCardGridProps`** - Interface for grid configuration with column count and styling options

## Usage Example

```typescript
import { BenefitCard, BenefitCardGrid } from './components/benefit-card'
import { CheckIcon } from './icons'

// Single card
<BenefitCard
  icon={<CheckIcon />}
  title="Fast Performance"
  description="Lightning-fast load times and smooth interactions"
  variant="dark"
/>

// Grid layout
<BenefitCardGrid columns={3}>
  <BenefitCard
    title="Secure"
    description="Enterprise-grade security"
    variant="default"
  />
  <BenefitCard
    title="Scalable"
    description="Grows with your business"
    variant="default"
  />
  <BenefitCard
    title="Reliable"
    description="99.9% uptime guarantee"
    variant="default"
  />
</BenefitCardGrid>
```

The component automatically handles responsive breakpoints and applies appropriate borders between grid items based on the selected column count.
