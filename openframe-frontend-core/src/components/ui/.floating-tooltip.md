A floating tooltip component built with Floating UI that provides precise positioning, color markup support, and ODS design system styling.

## Key Components

- **FloatingTooltip**: Main component that renders a floating tooltip with customizable positioning and styling
- **parseColoredText**: Utility function that converts color markup syntax `[COLOR]text[/COLOR]` into styled JSX elements
- **FloatingTooltipProps**: TypeScript interface defining component props including content, positioning, and styling options

## Usage Example

```typescript
import { FloatingTooltip } from './components/ui/floating-tooltip'

// Basic tooltip
<FloatingTooltip content="This is a helpful tooltip" side="top">
  <button>Hover me</button>
</FloatingTooltip>

// Tooltip with colored text markup
<FloatingTooltip 
  content="Status: [GREEN]Active[/GREEN] | [YELLOW]Warning[/YELLOW]"
  side="right"
  delayDuration={300}
>
  <div className="status-indicator">
    Status Icon
  </div>
</FloatingTooltip>

// Custom styled tooltip
<FloatingTooltip
  content="Custom tooltip content"
  className="bg-red-100 border-red-300"
  side="bottom"
>
  <span>Custom Element</span>
</FloatingTooltip>
```

The component supports color markup (`[YELLOW]`, `[GREEN]`, `[RED]`, `[BLUE]`, `[PINK]`, `[CYAN]`) that maps to ODS design system color classes, includes an arrow pointer, and handles positioning automatically with collision detection and fallback positioning.
