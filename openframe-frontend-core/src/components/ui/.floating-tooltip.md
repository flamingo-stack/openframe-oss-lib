A React component that provides floating tooltips with intelligent positioning, color markup parsing, and ODS design system styling using Floating UI for precise positioning and interactions.

## Key Components

- **`FloatingTooltip`** - Main tooltip component with hover interactions and smart positioning
- **`parseColoredText`** - Utility function that converts color markup syntax (`[YELLOW]text[/YELLOW]`) into styled JSX elements
- **`FloatingTooltipProps`** - Interface defining component props including content, positioning, styling, and delay options

## Usage Example

```typescript
import { FloatingTooltip } from './floating-tooltip'

// Basic tooltip
<FloatingTooltip content="Simple tooltip text">
  <button>Hover me</button>
</FloatingTooltip>

// Tooltip with color markup
<FloatingTooltip 
  content="This is [YELLOW]highlighted[/YELLOW] and [RED]error[/RED] text"
  side="top"
  delayDuration={500}
>
  <span>Hover for colored tooltip</span>
</FloatingTooltip>

// Custom styling
<FloatingTooltip 
  content="Custom styled tooltip"
  className="max-w-sm"
  side="bottom"
>
  <div>Trigger element</div>
</FloatingTooltip>
```

The component automatically handles positioning, collision detection, and provides smooth hover interactions with customizable delays. Color markup supports YELLOW, GREEN, RED, BLUE, PINK, and CYAN colors mapped to ODS design tokens.
