A reusable dashboard metrics card component that displays key performance indicators with optional progress visualization and navigation capabilities.

## Key Components

- **DashboardInfoCard**: Main component that renders a card with title, value, optional percentage, and circular progress indicator
- **Props Interface**: Comprehensive configuration including styling, navigation (`href`), click handlers, and progress display options
- **Navigation Logic**: Supports both programmatic navigation via `href` prop and custom click handlers via `onClick`
- **Value Formatting**: Automatically formats numeric values with locale-appropriate thousand separators

## Usage Example

```typescript
import { DashboardInfoCard } from './dashboard-info-card'

// Basic usage with navigation
<DashboardInfoCard
  title="Total Users"
  value={1250}
  href="/dashboard/users"
/>

// With progress indicator and custom styling
<DashboardInfoCard
  title="Server Uptime"
  value="99.9"
  percentage={99.9}
  showProgress={true}
  progressColor="#00ff88"
  className="col-span-2"
/>

// With custom click handler
<DashboardInfoCard
  title="Active Sessions"
  value={847}
  onClick={() => console.log('Sessions clicked')}
/>
```

The component uses the `InteractiveCard` wrapper for consistent hover states and accessibility, with special styling for navigation-enabled cards that highlight on hover. Progress visualization is provided through the `CircularProgress` component when enabled.
