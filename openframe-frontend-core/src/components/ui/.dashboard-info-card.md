A Next.js dashboard component that displays key metrics with optional progress indicators and navigation capabilities. Renders as either a clickable link or interactive card based on provided props.

## Key Components

- **DashboardInfoCard**: Main component that displays a title, formatted value, and optional percentage with circular progress
- **DashboardInfoCardProps**: Interface defining title, value, percentage, navigation options (href/onClick), and styling props
- **InteractiveCard**: Wrapper component used for click handlers when no href is provided
- **CircularProgress**: Visual progress indicator component for percentage display

## Usage Example

```typescript
import { DashboardInfoCard } from './dashboard-info-card'

// Basic info card
<DashboardInfoCard 
  title="Total Sales" 
  value={125000} 
/>

// With progress indicator
<DashboardInfoCard 
  title="Completion Rate" 
  value="85" 
  percentage={85}
  showProgress={true}
  progressColor="#22c55e"
/>

// With navigation
<DashboardInfoCard 
  title="Active Users" 
  value={1250}
  href="/users"
  openInNewTab={false}
/>

// With click handler
<DashboardInfoCard 
  title="Revenue" 
  value={75000}
  onClick={() => console.log('Card clicked')}
/>
```

The component automatically formats numeric values with locale-specific formatting and provides hover effects for interactive states.
