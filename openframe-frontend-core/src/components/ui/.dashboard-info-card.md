A reusable React component that displays dashboard statistics with optional interactive features, progress indicators, and navigation capabilities.

## Key Components

- **`DashboardInfoCard`**: Main component that renders a card with title, value, optional percentage, and circular progress indicator
- **`DashboardInfoCardProps`**: Interface defining component props including navigation options and styling customization
- **Navigation modes**: Supports both Next.js Link-based navigation and onClick callback handlers
- **Progress display**: Optional circular progress indicator with customizable colors

## Usage Example

```typescript
import { DashboardInfoCard } from './dashboard-info-card'

function Dashboard() {
  return (
    <div className="grid grid-cols-3 gap-4">
      {/* Basic info card */}
      <DashboardInfoCard
        title="Total Users"
        value={1234}
      />

      {/* Card with progress indicator */}
      <DashboardInfoCard
        title="Completion Rate"
        value="85"
        percentage={85}
        showProgress={true}
        progressColor="text-green-500"
      />

      {/* Interactive card with navigation */}
      <DashboardInfoCard
        title="Active Projects"
        value={12}
        href="/projects"
        openInNewTab={false}
      />

      {/* Card with click handler */}
      <DashboardInfoCard
        title="Revenue"
        value="$24,500"
        onClick={() => console.log('Revenue clicked')}
      />
    </div>
  )
}
```

The component automatically formats numeric values with locale-appropriate separators and provides hover states with accent color transitions. When both `href` and `onClick` are provided, the `href` navigation takes precedence.
