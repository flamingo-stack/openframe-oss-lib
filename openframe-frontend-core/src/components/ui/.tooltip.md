A React tooltip component built on top of Radix UI primitives with custom styling and animations. Provides accessible hover/focus tooltips with dark theme styling and smooth transitions.

## Key Components

- **TooltipProvider** - Context provider that manages tooltip state across the app
- **Tooltip** - Root component that wraps the tooltip system
- **TooltipTrigger** - Element that triggers the tooltip on hover/focus
- **TooltipContent** - The actual tooltip content with custom styling and animations

## Usage Example

```typescript
import { 
  Tooltip, 
  TooltipContent, 
  TooltipProvider, 
  TooltipTrigger 
} from "./tooltip"

function App() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <button>Hover me</button>
        </TooltipTrigger>
        <TooltipContent>
          <p>This is a helpful tooltip</p>
        </TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}
```

The component features a dark theme with smooth fade/zoom animations, high z-index for layering, and customizable positioning. The `TooltipContent` accepts standard props plus custom `className` for styling overrides.
