A React tooltip component built on Radix UI primitives with dark theme styling and maximum z-index for overlay visibility.

## Key Components

- **TooltipProvider** - Context provider for tooltip functionality
- **Tooltip** - Root tooltip container component
- **TooltipTrigger** - Element that triggers the tooltip on hover/focus
- **TooltipContent** - Styled tooltip content with portal rendering and animations

## Usage Example

```typescript
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from "./tooltip"

function App() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <button>Hover me</button>
        </TooltipTrigger>
        <TooltipContent>
          <p>This is a tooltip message</p>
        </TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}
```

The component features a dark theme with `#1A1A1A` background, smooth fade/zoom animations, and maximum z-index (`2147483647`) to ensure tooltips appear above all other content. The content automatically positions itself relative to the trigger with a 4px offset and includes directional slide animations.
