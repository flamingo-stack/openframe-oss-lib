A React component library providing cursor-based pagination controls for navigating through paginated data sets. It offers both full-featured and simplified pagination interfaces with customizable styling and behavior.

## Key Components

- **`CursorPagination`** - Main pagination component with item count display, navigation buttons, and reset functionality
- **`CursorPaginationSimple`** - Minimal pagination component with icon-only navigation buttons
- **`CursorPaginationProps`** - TypeScript interface defining all available props and configuration options

## Usage Example

```typescript
import { CursorPagination, CursorPaginationSimple } from './cursor-pagination'

// Full pagination with item info
<CursorPagination
  hasNextPage={true}
  hasPreviousPage={false}
  isFirstPage={true}
  startCursor="cursor123"
  endCursor="cursor456"
  currentCount={10}
  totalCount={100}
  itemName="products"
  onNext={(cursor) => fetchNextPage(cursor)}
  onPrevious={(cursor) => fetchPreviousPage(cursor)}
  onReset={() => goToFirstPage()}
  showInfo={true}
  compact={false}
/>

// Simple icon-only pagination
<CursorPaginationSimple
  hasNextPage={true}
  hasPreviousPage={false}
  onNext={(cursor) => fetchNextPage(cursor)}
  onReset={() => goToFirstPage()}
/>
```

The components handle cursor-based pagination state, provide accessible navigation controls, and support both compact and full layouts with customizable styling.
