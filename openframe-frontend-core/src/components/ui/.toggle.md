A customizable toggle button component built on Radix UI's Toggle primitive with configurable variants for styling and sizing.

## Key Components

- **`Toggle`** - Main toggle component with forwardRef support
- **`toggleVariants`** - CVA-based styling variants with configurable options:
  - `variant`: `"default"` | `"outline"`
  - `size`: `"default"` | `"sm"` | `"lg"`

## Usage Example

```typescript
import { Toggle } from "./toggle"

// Basic toggle
<Toggle>Toggle me</Toggle>

// Outlined variant
<Toggle variant="outline">Outlined toggle</Toggle>

// Small size with custom styling
<Toggle 
  size="sm" 
  className="text-blue-500"
  pressed={isPressed}
  onPressedChange={setIsPressed}
>
  Small toggle
</Toggle>

// Controlled toggle with state
const [bold, setBold] = useState(false)

<Toggle 
  variant="outline" 
  size="lg"
  pressed={bold}
  onPressedChange={setBold}
  aria-label="Toggle bold"
>
  Bold
</Toggle>
```

The component inherits all Radix Toggle props including `pressed`, `onPressedChange`, and accessibility features. Styling uses a custom design system with `ods-` prefixed CSS variables for consistent theming.
