A reusable button component that displays chevron icons (up/down arrows) with customizable styling and state-aware icon switching. Perfect for collapsible sections, dropdowns, and expandable UI elements.

## Key Components

- **ChevronButton**: Main component with forwardRef support
- **ChevronButtonProps**: Interface extending button HTML attributes with custom props:
  - `direction`: Controls default chevron direction ('up' | 'down')
  - `size`: Icon size variants ('sm' | 'md' | 'lg')
  - `isExpanded`: Overrides direction based on expansion state
  - `backgroundColor`/`borderColor`: Custom styling options

## Usage Example

```typescript
import { ChevronButton } from './chevron-button'

// Basic usage
<ChevronButton onClick={() => toggleExpanded()} />

// With expansion state (automatically switches between up/down)
<ChevronButton 
  isExpanded={isOpen}
  onClick={() => setIsOpen(!isOpen)}
/>

// Customized styling and size
<ChevronButton
  direction="up"
  size="lg"
  backgroundColor="#f3f4f6"
  borderColor="#e5e7eb"
  className="custom-styles"
/>

// With ref
const buttonRef = useRef<HTMLButtonElement>(null)
<ChevronButton ref={buttonRef} />
```

The component intelligently switches between ChevronUp and ChevronDown icons based on the `isExpanded` prop or falls back to the `direction` prop.
