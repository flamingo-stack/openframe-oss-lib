A flexible button component that displays chevron icons (up/down arrows) with customizable direction, size, and styling options.

## Key Components

- **ChevronButton** - Main component that renders a button with chevron icons from lucide-react
- **ChevronButtonProps** - Interface extending `React.ButtonHTMLAttributes` with additional props:
  - `direction` - Controls icon direction ('up' | 'down')
  - `size` - Icon size variant ('sm' | 'md' | 'lg')
  - `isExpanded` - Overrides direction based on expansion state
  - `backgroundColor` - Custom background color
  - `borderColor` - Custom border color

## Usage Example

```typescript
import { ChevronButton } from './chevron-button'

// Basic usage
<ChevronButton onClick={handleClick} />

// With custom direction and size
<ChevronButton 
  direction="up" 
  size="lg"
  onClick={toggleSection}
/>

// Responsive to expansion state
<ChevronButton 
  isExpanded={isOpen}
  backgroundColor="#f0f0f0"
  borderColor="#ccc"
  onClick={() => setIsOpen(!isOpen)}
/>

// With additional props
<ChevronButton 
  size="sm"
  disabled={loading}
  className="custom-chevron"
  aria-label="Expand section"
/>
```

The component automatically switches between ChevronUp and ChevronDown icons based on the `isExpanded` prop or falls back to the `direction` prop, making it ideal for collapsible sections and dropdown toggles.
