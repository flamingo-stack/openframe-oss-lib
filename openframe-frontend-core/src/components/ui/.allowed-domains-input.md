A controlled React component for managing a list of allowed domains, supporting add/remove operations with optional validation, error display, and keyboard interaction.

## Key Components

**`AllowedDomainsInput`** — A `forwardRef` component that renders a domain list manager with:
- Existing domain entries displayed as disabled inputs with remove (`X`) buttons
- A text input for entering new domains (supports `Enter` key to add)
- An "Add Domain" button with a `PlusCircle` icon
- Inline error and helper text display

**`AllowedDomainsInputProps`** — Interface defining the component's API:

| Prop | Type | Description |
|------|------|-------------|
| `value` | `string[]` | Controlled list of current domains |
| `onChange` | `(domains: string[]) => void` | Called when the domain list changes |
| `onValidate` | `(domain: string) => { valid, error?, cleanedDomain? }` | Optional custom validator; supports domain normalization via `cleanedDomain` |
| `disabled` | `boolean` | Disables all interactions |
| `error` | `string \| null` | External error message (takes priority over local errors) |
| `helperText` | `string` | Descriptive text shown when no error is present |

## Usage Example

```typescript
import { AllowedDomainsInput } from "@/components/ui/allowed-domains-input"

function SecuritySettings() {
  const [domains, setDomains] = React.useState<string[]>(["flamingo.run"])

  const validateDomain = (domain: string) => {
    const cleaned = domain.toLowerCase().replace(/^https?:\/\//, "")
    const isValid = /^[a-z0-9.-]+\.[a-z]{2,}$/.test(cleaned)
    return {
      valid: isValid,
      error: isValid ? undefined : "Enter a valid domain (e.g. example.com)",
      cleanedDomain: cleaned,
    }
  }

  return (
    <AllowedDomainsInput
      value={domains}
      onChange={setDomains}
      onValidate={validateDomain}
      label="Allowed Email Domains"
      helperText="Only users from these domains can sign up."
    />
  )
}
```

> **Note:** Duplicate domains are silently ignored. The external `error` prop overrides any locally generated validation error.
