A controlled React component for managing a list of allowed domains, supporting add/remove operations with optional validation, error display, and keyboard interaction.

## Key Components

**`AllowedDomainsInputProps`** — Interface defining the component's props:

| Prop | Type | Description |
|------|------|-------------|
| `value` | `string[]` | Current list of domains |
| `onChange` | `(domains: string[]) => void` | Callback fired on domain list change |
| `onValidate` | `(domain: string) => { valid, error?, cleanedDomain? }` | Optional custom validator |
| `label` | `string` | Field label (default: `"Allowed Domains"`) |
| `placeholder` | `string` | Input placeholder (default: `"example.com"`) |
| `disabled` | `boolean` | Disables all interactions |
| `error` | `string \| null` | External error message |
| `helperText` | `string` | Secondary helper text (hidden when error is shown) |

**`AllowedDomainsInput`** — `forwardRef` component forwarding to the root `<div>`. Manages local input state and validation errors internally.

## Key Behaviors

- **Add domain** via `Enter` key or "Add Domain" button — deduplicates entries automatically
- **`onValidate`** receives raw input; returned `cleanedDomain` (e.g., stripped protocol) is stored instead of raw value
- **Error priority**: external `error` prop overrides internal `localError`
- Existing domains render as disabled `Input` fields with an `X` remove button each

## Usage Example

```typescript
import { AllowedDomainsInput } from "@/components/ui/allowed-domains-input"

function SettingsForm() {
  const [domains, setDomains] = React.useState<string[]>(["flamingo.run"])

  const validateDomain = (domain: string) => {
    const cleaned = domain.replace(/^https?:\/\//, "").toLowerCase()
    const valid = /^[a-z0-9.-]+\.[a-z]{2,}$/.test(cleaned)
    return { valid, cleanedDomain: cleaned, error: valid ? undefined : "Invalid domain format" }
  }

  return (
    <AllowedDomainsInput
      value={domains}
      onChange={setDomains}
      onValidate={validateDomain}
      helperText="Only users from these domains can sign in."
    />
  )
}
```
