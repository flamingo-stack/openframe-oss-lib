A React component that displays device information in a structured card layout with customizable actions and status indicators.

## Key Components

### Interfaces
- **`Device`** - Core device data structure with properties like name, type, OS, status, and metadata
- **`ActionButton`** - Configuration for custom action buttons with labels and click handlers
- **`DeviceCardProps`** - Component props including device data, actions config, and event handlers

### Main Export
- **`DeviceCard`** - Primary component that renders device information in a multi-row card layout

## Usage Example

```typescript
import { DeviceCard, Device } from './device-card'

const device: Device = {
  id: '123',
  name: 'John's MacBook Pro',
  type: 'laptop',
  operatingSystem: 'darwin',
  organization: 'Acme Corp',
  status: 'active',
  lastSeen: '2024-01-15T10:30:00Z',
  tags: ['development', 'remote']
}

// Basic usage
<DeviceCard device={device} />

// With custom actions and click handler
<DeviceCard 
  device={device}
  onDeviceClick={(device) => navigate(`/devices/${device.id}`)}
  actions={{
    moreButton: { onClick: () => showMenu() },
    detailsButton: { component: <ViewButton /> },
    customActions: [
      { label: 'Connect', onClick: () => connect() },
      { label: 'Restart', onClick: () => restart() }
    ]
  }}
  statusBadgeComponent={<StatusBadge status={device.status} />}
/>
```
