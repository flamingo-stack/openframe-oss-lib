A React component that renders a device information card with interactive elements, status indicators, and customizable actions for displaying system/device management data.

## Key Components

- **Device Interface**: Comprehensive device data structure supporting various device types, operating systems, and metadata
- **DeviceCard Component**: Main card component with flexible action configuration and status display
- **ActionButton Interface**: Configuration for custom action buttons with variants and visibility controls
- **Interactive Features**: Clickable device names, more button, custom actions, and hover states
- **Status Display**: OS badges, last seen timestamps, status indicators, and tag system

## Usage Example

```typescript
import { DeviceCard, Device, ActionButton } from './device-card'

const sampleDevice: Device = {
  id: 'device-123',
  name: 'MacBook Pro',
  type: 'laptop',
  operatingSystem: 'darwin',
  organization: 'Acme Corp',
  status: 'active',
  lastSeen: new Date(),
  tags: ['development', 'remote'],
  ipAddress: '192.168.1.100'
}

const customActions: ActionButton[] = [
  {
    label: 'Connect',
    onClick: () => console.log('Connecting...'),
    variant: 'default',
    visible: true
  }
]

<DeviceCard
  device={sampleDevice}
  actions={{
    moreButton: { visible: true, onClick: () => console.log('More options') },
    detailsButton: { visible: true, component: <Button>Details</Button> },
    customActions: customActions
  }}
  statusBadgeComponent={<StatusBadge status={sampleDevice.status} />}
  onDeviceClick={(device) => console.log('Device clicked:', device)}
  className="max-w-md"
/>
```
