A versatile status tag component for displaying operational states with customizable styling, icons, and responsive typography in the Flamingo design system.

## Key Components

- **StatusTag**: Main component for displaying status indicators with predefined color schemes
- **StatusTagProps**: TypeScript interface defining component properties including label, variant, mobile responsiveness, and icon support
- **variantStyles**: Internal mapping of status variants to Tailwind CSS classes for consistent theming

## Variants

The component supports 8 status variants with specific color schemes:
- `success`/`active`: Green theme for positive states
- `warning`: Yellow theme for caution states  
- `error`/`offline`: Red theme for error states
- `critical`: High-contrast red for urgent issues
- `info`/`inactive`: Gray theme for neutral states

## Usage Example

```typescript
import { StatusTag } from "./status-tag"
import { CheckIcon, AlertIcon } from "lucide-react"

// Basic status tag
<StatusTag label="Online" variant="success" />

// With icons and mobile optimization
<StatusTag 
  label="Warning" 
  variant="warning"
  isMobile={true}
  leftIcon={<AlertIcon size={16} />}
  className="ml-2"
/>

// Critical status with right icon
<StatusTag 
  label="System Down" 
  variant="critical"
  rightIcon={<AlertIcon size={16} />}
/>
```

The component uses the Azeret Mono font family with responsive sizing and supports custom CSS classes for additional styling flexibility.
