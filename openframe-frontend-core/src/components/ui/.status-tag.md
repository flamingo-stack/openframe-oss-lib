A React component that renders styled status tags with various predefined variants for displaying system states, service statuses, or operational conditions.

## Key Components

- **StatusTag**: Main component that renders a colored status indicator
- **StatusTagProps**: Interface defining the component's properties
- **variantStyles**: Object mapping status variants to their respective color schemes

## Usage Example

```typescript
import { StatusTag } from './status-tag'

// Basic usage with different variants
<StatusTag label="Online" variant="success" />
<StatusTag label="Warning" variant="warning" />
<StatusTag label="Error" variant="error" />
<StatusTag label="Critical" variant="critical" />

// Mobile-optimized version
<StatusTag 
  label="Active" 
  variant="active" 
  isMobile={true} 
/>

// With custom styling
<StatusTag 
  label="Offline" 
  variant="offline" 
  className="ml-4" 
/>
```

The component supports 8 predefined variants (success, warning, error, critical, info, active, inactive, offline) with distinct color schemes. It uses Azeret Mono font for consistent monospace styling and includes responsive text sizing for mobile displays. The component automatically applies appropriate background and text colors based on the selected variant, defaulting to 'success' if no variant is specified.
