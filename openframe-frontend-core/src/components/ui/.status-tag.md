A React component that displays status information as a styled tag with customizable variants and responsive text sizing.

## Key Components

- **`StatusTag`** - Main component that renders a styled status indicator
- **`StatusTagProps`** - TypeScript interface defining component props
- **Variants** - 8 predefined status types: `success`, `warning`, `error`, `critical`, `info`, `active`, `inactive`, `offline`

## Usage Example

```typescript
import { StatusTag } from './status-tag'

// Basic usage
<StatusTag label="Online" variant="success" />

// Mobile responsive version
<StatusTag 
  label="Error" 
  variant="error" 
  isMobile={true} 
/>

// With custom styling
<StatusTag 
  label="Processing" 
  variant="warning"
  className="ml-4"
/>

// Default variant (success)
<StatusTag label="Ready" />
```

The component uses the Azeret Mono font family and applies different background/text color combinations based on the selected variant. The `isMobile` prop adjusts font size and tracking for better mobile display. Custom styling can be added via the `className` prop.
