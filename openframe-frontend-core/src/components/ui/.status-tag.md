A reusable React component that displays status indicators with customizable variants, colors, and optional icons for UI state communication.

## Key Components

- **StatusTag** - Main component that renders a styled status badge
- **StatusTagProps** - Interface defining component props including label, variant, mobile support, and icon options
- **variantStyles** - Color scheme mapping for different status types (success, warning, error, critical, info, active, inactive, offline)

## Usage Example

```typescript
import { StatusTag } from './status-tag'
import { CheckIcon, AlertIcon } from 'lucide-react'

// Basic status tag
<StatusTag label="Online" variant="success" />

// With icons and mobile optimization
<StatusTag 
  label="Warning" 
  variant="warning" 
  isMobile={true}
  leftIcon={<AlertIcon size={16} />}
  className="ml-2"
/>

// Critical status with right icon
<StatusTag 
  label="Critical" 
  variant="critical" 
  rightIcon={<CheckIcon size={16} />}
/>
```

The component provides 8 predefined variants with appropriate color schemes, supports responsive design through the `isMobile` prop, and allows icon placement on either side of the label. Text is rendered in uppercase monospace font with precise spacing and tracking adjustments.
