A React component that renders a styled status indicator tag with customizable variants, icons, and responsive design for displaying system states and statuses.

## Key Components

- **StatusTag** - Main component that renders a status indicator with text and optional icons
- **StatusTagProps** - TypeScript interface defining component properties including label, variant, mobile support, and icon slots
- **variantStyles** - Color mapping object defining 8 status variants: success, warning, error, critical, info, active, inactive, and offline

## Usage Example

```typescript
import { StatusTag } from './status-tag'
import { CheckIcon, AlertIcon } from 'lucide-react'

function ServerStatus() {
  return (
    <div className="flex gap-2">
      <StatusTag 
        label="Online" 
        variant="success" 
        leftIcon={<CheckIcon size={16} />} 
      />
      
      <StatusTag 
        label="Warning" 
        variant="warning" 
        rightIcon={<AlertIcon size={16} />}
        isMobile={true}
      />
      
      <StatusTag 
        label="Critical" 
        variant="critical" 
        className="ml-2"
      />
    </div>
  )
}
```

The component features a monospace font with uppercase text styling, responsive sizing based on the `isMobile` prop, and supports both left and right icon placement with automatic gap spacing.
