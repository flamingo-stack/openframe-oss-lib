A reusable React component that renders status indicators with customizable styling variants and optional icons for displaying system states, alerts, and operational statuses.

## Key Components

- **StatusTag** - Main component that renders a styled status indicator
- **StatusTagProps** - TypeScript interface defining component properties
- **variantStyles** - Object mapping status variants to their color schemes

## Variants

The component supports 8 predefined status variants:
- `success` / `active` - Green theme for positive states
- `warning` - Yellow theme for caution states  
- `error` / `offline` - Red theme for error/offline states
- `critical` - High-contrast red for urgent alerts
- `info` / `inactive` - Gray theme for neutral/disabled states

## Usage Example

```typescript
import { StatusTag } from "./status-tag"
import { CheckIcon, AlertIcon } from "lucide-react"

// Basic status tag
<StatusTag label="Online" variant="success" />

// With icons
<StatusTag 
  label="Warning" 
  variant="warning"
  leftIcon={<AlertIcon size={16} />}
/>

// Mobile optimized
<StatusTag 
  label="Error" 
  variant="error"
  isMobile={true}
  rightIcon={<CheckIcon size={16} />}
/>

// Custom styling
<StatusTag 
  label="Custom" 
  variant="info"
  className="shadow-lg"
/>
```

The component uses Azeret Mono font family and automatically adjusts text size and spacing for mobile devices when the `isMobile` prop is enabled.
