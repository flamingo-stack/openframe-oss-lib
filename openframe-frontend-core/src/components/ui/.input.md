A reusable input component built with React forwardRef that provides consistent styling and error state handling across the application.

## Key Components

- **Input**: Forwarded ref component extending standard HTML input attributes
- **InputProps**: Interface adding optional `invalid` prop for error styling
- **cn utility**: Class name concatenation helper for conditional styling

## Usage Example

```typescript
import { Input } from "./components/input"
import { useState } from "react"

function LoginForm() {
  const [email, setEmail] = useState("")
  const [hasError, setHasError] = useState(false)

  return (
    <form>
      {/* Basic input */}
      <Input 
        type="email"
        placeholder="Enter your email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
      />
      
      {/* Error state */}
      <Input 
        type="password"
        placeholder="Password"
        invalid={hasError}
        className="mt-4"
      />
    </form>
  )
}
```

The component features admin theme styling with dark backgrounds, accent colors on focus, and automatic error styling when `invalid` is true. It maintains full compatibility with native input attributes while providing consistent visual design.
