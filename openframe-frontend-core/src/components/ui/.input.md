A comprehensive input component built on React's forwardRef pattern that supports adornments, validation states, and preview modes with consistent Flamingo design system styling.

## Key Components

- **Input** - Main forwardRef component extending `HTMLInputElement` attributes
- **InputProps** - Interface defining component props including validation and adornment options
- **cn** - Utility function for conditional class name merging

Key props:
- `invalid` - Applies red error styling for validation states
- `startAdornment`/`endAdornment` - React nodes displayed at input edges
- `preview` - Non-interactive display mode showing current value

## Usage Example

```typescript
import { Input } from "./input";
import { Search, Lock } from "lucide-react";

// Basic input
<Input placeholder="Enter your email" />

// With validation
<Input 
  type="email" 
  invalid={!isValidEmail} 
  placeholder="email@example.com" 
/>

// With adornments

  type="search"
  placeholder="Search..."

/>

// Preview mode (read-only display)

  startAdornment={<Lock size={16} />}
/>
```

The component automatically handles three rendering modes: basic input, input with adornments (wrapped in label), and preview mode with non-interactive styling. All modes support the Flamingo dark theme with proper focus states and accessibility features.
