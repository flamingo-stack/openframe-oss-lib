A reusable input component built with React and forwardRef that provides consistent styling, theming, and error state handling across the application.

## Key Components

- **`Input`** - The main input component with forwardRef support for direct DOM access
- **`InputProps`** - TypeScript interface extending native input attributes with custom `invalid` prop
- **`invalid`** - Boolean prop that applies error styling (red border and focus ring)

## Usage Example

```typescript
import { Input } from "./components/ui/input"
import { useState } from "react"

function LoginForm() {
  const [email, setEmail] = useState("")
  const [isEmailValid, setIsEmailValid] = useState(true)

  return (
    <form>
      <Input
        type="email"
        placeholder="Enter your email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        invalid={!isEmailValid}
        className="mb-4"
      />
      
      <Input
        type="file"
        accept="image/*"
        className="mb-4"
      />
    </form>
  )
}
```

The component features responsive typography (14px on mobile, 18px on desktop), dark theme styling with ODS design tokens, smooth transitions, and special handling for file inputs. It maintains a consistent 60px minimum height and integrates seamlessly with form libraries through forwardRef.
