A styled, accessible input component that extends the native HTML `<input>` with support for labels, validation states, adornments, and a loading spinner. Handles both standard text inputs and range sliders with distinct rendering logic.

## Key Components

- **`Input`** — The primary `forwardRef` component wrapping a native `<input>` with ODS design tokens
- **`InputProps`** — Extended interface adding `invalid`, `startAdornment`, `endAdornment`, `label`, `error`, `errorVariant`, and `loading` to standard HTML input attributes
- **`FieldWrapper`** — Wraps content with optional label and error message display
- **`invalidBorderClasses`** — Lookup map for `"error"` (red) and `"warning"` (yellow) border styling variants
- **Range input branch** — Special-cased rendering for `type="range"` that skips the label wrapper and adornments, applying custom slider track/thumb styles

## Usage Example

```typescript
import { Input } from "@/components/ui/input";
import { Search, X } from "lucide-react";

// Basic usage with label
<Input label="Email" type="email" placeholder="you@example.com" />

// With adornments and error state
<Input
  label="Username"
  startAdornment={<Search />}
  endAdornment={<X />}
  error="Username already taken"
  errorVariant="warning"
/>

// Loading state (replaces endAdornment)
<Input label="Search" loading={true} />

// Range slider
<Input
  type="range"
  label="Volume"
  min={0}
  max={100}
  defaultValue={50}
/>
```
