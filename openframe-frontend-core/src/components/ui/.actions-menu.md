A flexible, accessible actions menu component built on Radix UI's DropdownMenu primitive, supporting grouped items, checkboxes, submenus, navigation links, and optional secondary icon actions per row.

## Key Components

- **`ActionsMenu`** — Standalone menu panel (no trigger) rendering grouped `ActionsMenuItem` entries inside a scrollable container.
- **`ActionsMenuDropdown`** — Full dropdown wrapper combining a configurable trigger button with `ActionsMenu`; supports controlled/uncontrolled open state.
- **`MenuItem`** — Internal row renderer handling `item`, `checkbox`, `submenu`, and `separator` types, with optional link navigation and keyboard accessibility.
- **`SecondaryAction`** — Internal 40px icon button/link rendered on the right of a row with a vertical divider, independently clickable from the main row action.

**Key interfaces:**
- `ActionsMenuItem` — Single menu entry with optional `icon`, `href`, `danger`, `checked`, `submenu`, and `iconAction`.
- `ActionsMenuGroup` — Array of items with optional group separator.
- `ActionsMenuDropdownProps` — Extends `ActionsMenuProps` with trigger customization, alignment, positioning, and open-state control.

## Usage Example

```typescript
import { ActionsMenuDropdown } from "./actions-menu";

const groups = [
  {
    items: [
      {
        id: "edit",
        label: "Edit",
        icon: <EditIcon />,
        onClick: () => console.log("edit"),
      },
      {
        id: "open-link",
        label: "Open in browser",
        href: "https://example.com",
        openInNewTab: true,
        iconAction: {
          icon: <ExternalLinkIcon />,
          "aria-label": "Open in new tab",
          href: "https://example.com",
          openInNewTab: true,
        },
      },
      { id: "sep", label: "", type: "separator" },
      {
        id: "delete",
        label: "Delete",
        icon: <TrashIcon />,
        danger: true,
        onClick: () => console.log("delete"),
      },
    ],
  },
];

<ActionsMenuDropdown
  groups={groups}
  align="end"
  side="bottom"
  triggerAriaLabel="Record actions"
  onItemClick={(item) => console.log("clicked", item.id)}
/>
```
