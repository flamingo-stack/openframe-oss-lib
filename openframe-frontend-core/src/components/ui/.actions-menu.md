A comprehensive React component for creating context menus with support for nested submenus, checkboxes, separators, and external links with intelligent positioning and portal-based rendering.

## Key Components

- **`ActionsMenu`** - Main menu component that renders groups of menu items
- **`MenuItem`** - Individual menu item component with support for icons, checkboxes, and submenus
- **`ActionsMenuItem`** - Interface defining menu item properties (label, icon, onClick, type, etc.)
- **`ActionsMenuGroup`** - Interface for grouping menu items with optional separators
- **`ActionsMenuProps`** - Main component props interface

## Usage Example

```typescript
import { ActionsMenu, ActionsMenuItem } from './actions-menu'
import { Settings, Download, Check } from 'lucide-react'

const menuGroups = [
  {
    items: [
      {
        id: 'download',
        label: 'Download',
        icon: <Download />,
        onClick: () => console.log('Download clicked')
      },
      {
        id: 'settings',
        label: 'Settings',
        icon: <Settings />,
        type: 'submenu' as const,
        submenu: [
          {
            id: 'auto-save',
            label: 'Auto-save',
            type: 'checkbox' as const,
            checked: true,
            onClick: () => console.log('Toggle auto-save')
          }
        ]
      }
    ],
    separator: true
  }
]

<ActionsMenu 
  groups={menuGroups}
  onItemClick={(item) => console.log('Item clicked:', item)}
/>
```
