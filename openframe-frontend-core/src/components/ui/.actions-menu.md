A flexible, accessible actions/context menu component built on Radix UI's DropdownMenu primitive, supporting items, checkboxes, submenus, separators, navigation links, and optional secondary icon actions per row.

## Key Components

- **`ActionsMenu`** — Standalone menu panel (no trigger) that renders grouped items directly into the DOM
- **`ActionsMenuDropdown`** — Full dropdown with a configurable trigger button (ellipsis by default), wrapping `ActionsMenu` with controlled/uncontrolled open state
- **`MenuItem`** — Internal component handling all item variants: standard, checkbox, submenu, separator, and link
- **`SecondaryAction`** — Internal 40px-wide icon button/link rendered on the right side of a row, independently clickable from the primary action

### Key Interfaces

| Interface | Purpose |
|---|---|
| `ActionsMenuItem` | Defines a single menu row (label, icon, type, href, danger, iconAction, etc.) |
| `ActionsMenuGroup` | Groups items together with optional separators between groups |
| `ActionsMenuProps` | Props for the standalone `ActionsMenu` panel |
| `ActionsMenuDropdownProps` | Extends `ActionsMenuProps` with trigger, alignment, open state, and focus controls |

## Usage Example

```typescript
import { ActionsMenuDropdown } from "./actions-menu";

const groups = [
  {
    items: [
      {
        id: "edit",
        label: "Edit",
        icon: <EditIcon />,
        onClick: () => handleEdit(),
      },
      {
        id: "open-link",
        label: "Open in Browser",
        href: "https://example.com",
        openInNewTab: true,
        iconAction: {
          icon: <ExternalLinkIcon />,
          "aria-label": "Open in new tab",
          href: "https://example.com",
          openInNewTab: true,
        },
      },
      { id: "sep", type: "separator" },
      {
        id: "delete",
        label: "Delete",
        danger: true,
        icon: <TrashIcon />,
        onClick: () => handleDelete(),
      },
    ],
  },
];

<ActionsMenuDropdown
  groups={groups}
  align="end"
  onItemClick={(item) => console.log("Clicked:", item.id)}
/>
```
