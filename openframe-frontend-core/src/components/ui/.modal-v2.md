Animated modal dialog component with open/close state management, keyboard handling, and responsive layout (bottom sheet on mobile, centered on desktop).

## Key Components

| Export | Description |
|--------|-------------|
| `ModalV2` | Root container — manages mount/unmount lifecycle, backdrop, scroll lock, and `Escape` key handler |
| `ModalV2Header` | Header row with optional close button (desktop-only `×` icon via `ModalContext`) |
| `ModalV2Title` | Styled `<h2>` heading inside the header |
| `ModalV2Content` | Scrollable body area (`overflow-y-auto`, `flex-1`) |
| `ModalV2Footer` | Flex row for action buttons |

**Internal:** `ModalContext` passes `onClose` to `ModalHeader` without prop drilling.

## Usage Example

```typescript
import {
  ModalV2,
  ModalV2Header,
  ModalV2Title,
  ModalV2Content,
  ModalV2Footer,
} from "@/components/modal-v2"

export function ExampleModal() {
  const [open, setOpen] = useState(false)

  return (
    <>
      <button onClick={() => setOpen(true)}>Open</button>

      <ModalV2 isOpen={open} onClose={() => setOpen(false)}>
        <ModalV2Header>
          <ModalV2Title>Confirm Action</ModalV2Title>
        </ModalV2Header>

        <ModalV2Content>
          <p>Are you sure you want to continue?</p>
        </ModalV2Content>

        <ModalV2Footer>
          <button onClick={() => setOpen(false)}>Cancel</button>
          <button onClick={handleConfirm}>Confirm</button>
        </ModalV2Footer>
      </ModalV2>
    </>
  )
}
```

## Behaviour Notes

- **Exit animation** — the modal stays mounted for `200 ms` after `isOpen` becomes `false` so the fade/slide-out animation completes before unmount.
- **Scroll lock** — `document.body.style.overflow = 'hidden'` is applied while open and restored on close.
- **Responsive layout** — slides up from the bottom on mobile; zooms in centered on `md+` screens.
- **Backdrop click** — clicking outside the panel calls `onClose`.
