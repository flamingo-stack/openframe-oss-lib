Animated modal dialog component with open/close transitions, keyboard support, and a composable sub-component API.

## Key Components

| Export | Description |
|--------|-------------|
| `ModalV2` | Root modal container — manages mount/unmount lifecycle, backdrop, Escape key handling, and scroll lock |
| `ModalV2Header` | Header row with optional close button (desktop only) via `ModalContext` |
| `ModalV2Title` | Styled `<h2>` heading for the modal title |
| `ModalV2Content` | Scrollable content area (`overflow-y-auto`, `flex-1`) |
| `ModalV2Footer` | Flex row for action buttons |

**Internal:** `ModalContext` passes `onClose` down to `ModalHeader` without prop drilling.

## Behaviour Notes

- **Animation** — backdrop and panel use `data-[state=open/closed]` + Tailwind animate-in/out utilities (200 ms). `fill-mode-forwards` prevents an opacity flash before unmount.
- **Responsive motion** — mobile slides up from the bottom; desktop fades + zooms from center.
- **Body scroll lock** — `document.body.style.overflow = 'hidden'` while open, restored on close.
- **Keyboard** — `Escape` triggers `onClose`.
- **Delayed unmount** — component stays mounted for 200 ms after `isOpen` becomes `false` so the exit animation completes.

## Usage Example

```typescript
import {
  ModalV2,
  ModalV2Header,
  ModalV2Title,
  ModalV2Content,
  ModalV2Footer,
} from "@/components/ui/modal-v2"

function DeleteDialog({ open, onClose }: { open: boolean; onClose: () => void }) {
  return (
    <ModalV2 isOpen={open} onClose={onClose}>
      <ModalV2Header>
        <ModalV2Title>Confirm Delete</ModalV2Title>
      </ModalV2Header>

      <ModalV2Content>
        <p>This action cannot be undone.</p>
      </ModalV2Content>

      <ModalV2Footer>
        <button onClick={onClose}>Cancel</button>
        <button onClick={() => { /* delete */ onClose() }}>Delete</button>
      </ModalV2Footer>
    </ModalV2>
  )
}
```
