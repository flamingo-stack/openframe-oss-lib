A React component that renders a single row in a data table with support for selection, click handling, and dynamic column rendering.

## Key Components

- **TableRow**: Generic React component that renders table row data
- **handleRowClick**: Click handler that respects `data-no-row-click` elements
- **handleSelect**: Selection handler for checkbox interactions
- **getCellValue**: Utility function that extracts and formats cell values with dot notation support

## Usage Example

```typescript
import { TableRow } from './table-row'

// Define your data structure
interface User {
  id: string
  name: string
  email: string
  profile: {
    role: string
  }
}

// Configure columns
const columns = [
  { key: 'name', label: 'Name' },
  { key: 'email', label: 'Email' },
  { key: 'profile.role', label: 'Role' }, // Nested property access
  {
    key: 'actions',
    label: 'Actions',
    renderCell: (user: User) => (
      <button data-no-row-click>Edit</button>
    )
  }
]

// Render table row
<TableRow
  item={userData}
  columns={columns}
  selectable={true}

  onClick={handleRowClick}
  index={0}
/>
```

The component supports nested property access via dot notation, custom cell rendering, row selection with checkboxes, and prevents row click events on elements marked with `data-no-row-click`.
