A responsive table row component that renders data across desktop and mobile layouts with support for selection, actions, and custom rendering.

## Key Components

- **TableRow<T>** - Main component with generic type support for flexible data structures
- **handleRowClick** - Click handler with selective event propagation blocking
- **getCellValue** - Data accessor supporting nested properties and type-aware formatting
- **Responsive layouts** - Separate desktop (grid-based) and mobile (stacked) rendering

## Usage Example

```typescript
import { TableRow } from './table-row'

interface User {
  id: string
  name: string
  email: string
  status: boolean
}

const columns = [
  { key: 'name', label: 'Name' },
  { key: 'email', label: 'Email', hideOnMobile: true },
  { key: 'status', label: 'Active' }
]

<TableRow<User>
  item={{ id: '1', name: 'John Doe', email: 'john@example.com', status: true }}
  columns={columns}
  rowKey="id"
  selectable
  selected={false}
  onSelect={(user) => console.log('Selected:', user)}
  onClick={(user) => console.log('Clicked:', user)}
  rowActions={[
    { 
      icon: <EditIcon />, 
      onClick: (user) => editUser(user) 
    }
  ]}
  mobileColumns={['name', 'status']}
/>
```

The component automatically handles responsive breakpoints, nested property access via dot notation, and provides customizable mobile rendering patterns.
