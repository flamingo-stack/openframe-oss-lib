A responsive table row component that renders data items in both desktop and mobile layouts with support for selection, actions, and custom renderers.

## Key Components

- **TableRow**: Generic component accepting typed data items with configurable columns
- **handleRowClick**: Click handler that respects no-click zones via `data-no-row-click` attribute
- **getCellValue**: Value extraction supporting nested object properties via dot notation
- **Desktop Layout**: Full column display with consistent spacing and alignment
- **Mobile Layout**: Condensed view showing primary columns with optional custom renderer
- **Selection System**: Checkbox integration for multi-select functionality
- **Row Actions**: Configurable action buttons with overflow menu support

## Usage Example

```typescript
interface User {
  id: string
  name: string
  email: string
  profile: { department: string }
}

const columns = [
  { key: 'name', label: 'Name' },
  { key: 'email', label: 'Email' },
  { key: 'profile.department', label: 'Department' } // Nested property
]

<TableRow
  item={user}
  columns={columns}
  rowKey="id"
  onClick={(item) => console.log('Row clicked:', item)}
  selectable={true}
  selected={selectedIds.includes(user.id)}
  onSelect={(item) => toggleSelection(item.id)}
  mobileColumns={['name', 'email']}
  rowActions={[
    {
      icon: <Edit className="h-4 w-4" />,
      onClick: (item) => editUser(item)
    }
  ]}
/>
```
