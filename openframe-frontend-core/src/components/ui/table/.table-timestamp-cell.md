A reusable React component that displays timestamp and ID information in table cells with consistent ODS design token styling. The component automatically formats timestamps and provides optional ID labeling.

## Key Components

- **`TableTimestampCell`** - Main component for rendering timestamp/ID pairs in table cells
- **`TableTimestampCellProps`** - TypeScript interface defining component props
- **`formatTimestampValue`** - Internal utility function that formats dates and ISO strings to locale strings

## Usage Example

```typescript
import { TableTimestampCell } from './table-timestamp-cell'

// Basic timestamp and ID display
<TableTimestampCell
  timestamp="2024-01-15T10:30:00Z"
  id="abc123"
/>

// With custom ID label
<TableTimestampCell
  timestamp={new Date()}
  id="log-123"
  idLabel="Log ID"
/>

// Disable automatic formatting for pre-formatted timestamps
<TableTimestampCell
  timestamp="1/15/2024, 10:30:00 AM"
  id="chat-456"
  formatTimestamp={false}
  className="custom-styling"
/>
```

The component renders timestamps as primary text with IDs as secondary text below. It intelligently handles Date objects, ISO strings, and pre-formatted timestamps, with automatic locale-based formatting when appropriate.
