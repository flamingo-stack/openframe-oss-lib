A React component for displaying timestamp and ID data in table cells with automatic timestamp formatting and consistent ODS design token styling.

## Key Components

- **`TableTimestampCell`** - Main component that renders timestamp as primary text and ID as secondary text
- **`TableTimestampCellProps`** - TypeScript interface defining component props including timestamp, id, optional label, and formatting options
- **`formatTimestampValue`** - Utility function that intelligently formats timestamps from Date objects, ISO strings, or pre-formatted strings

## Usage Example

```typescript
import { TableTimestampCell } from './table-timestamp-cell'

// Basic usage with ISO timestamp
<TableTimestampCell
  timestamp="2024-01-15T10:30:00Z"
  id="abc123"
/>

// With custom ID label
<TableTimestampCell
  timestamp={new Date()}
  id="log-123"
  idLabel="Log ID"
/>

// Pre-formatted timestamp (no auto-formatting)
<TableTimestampCell
  timestamp="1/15/2024, 10:30:00 AM"
  id="chat-456"
  formatTimestamp={false}
  className="min-w-[200px]"
/>
```

The component automatically detects and formats ISO strings and Date objects using `toLocaleString()`, but preserves already-formatted timestamps. It uses ODS design tokens for consistent typography and spacing, with the timestamp displayed prominently above the ID in a flex column layout.
