A loading skeleton component that renders placeholder cards for table data with responsive design patterns for both desktop and mobile layouts.

## Key Components

- **`TableCardSkeleton`** - Main component that renders animated skeleton cards mimicking table rows
- **`ROW_HEIGHT_DESKTOP`** - Constant for consistent desktop row height using CSS clamp
- **`ROW_HEIGHT_MOBILE`** - Constant for consistent mobile row height using CSS clamp
- **`TableCardSkeletonProps`** - Type interface defining component props (imported from types)

## Usage Example

```typescript
import { TableCardSkeleton } from './table-skeleton'

// Basic usage
<TableCardSkeleton
  columns={[
    { key: 'name', width: 'w-1/3' },
    { key: 'email', width: 'w-1/3' },
    { key: 'status', width: 'w-1/4' }
  ]}
  rows={5}
/>

// With actions column
<TableCardSkeleton
  columns={columns}
  rows={8}
  hasActions={true}
  className="mb-4"
  rowClassName="hover:bg-gray-50"
/>
```

The component automatically adapts between desktop table-like layout and mobile stacked card layout. Each skeleton row includes animated placeholder elements that simulate loading content, with optional action buttons and multi-line text placeholders.
