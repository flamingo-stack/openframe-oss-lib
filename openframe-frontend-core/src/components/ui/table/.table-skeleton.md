A skeleton loading component for table cards that provides responsive placeholder animations while data is being fetched, maintaining consistent row heights across desktop and mobile layouts.

## Key Components

- **TableCardSkeleton** - Main skeleton component that renders animated placeholder rows
- **ROW_HEIGHT_DESKTOP/MOBILE** - Exported height constants using CSS clamp for responsive sizing
- **TableCardSkeletonProps** - TypeScript interface defining component props (imported from types)

The component uses CSS `clamp()` functions to ensure consistent responsive row heights:
- Desktop: 72px minimum, 5vw preferred, 88px maximum
- Mobile: 72px minimum, 18vw preferred, 96px maximum

## Usage Example

```typescript
import { TableCardSkeleton } from './table-skeleton'

// Basic usage with column definitions
<TableCardSkeleton
  columns={[
    { key: 'name', width: 'w-48' },
    { key: 'status', width: 'w-32' },
    { key: 'date' } // flex-1 width
  ]}
  rows={5}
  hasActions={true}
  className="mb-4"
/>

// Minimal usage
<TableCardSkeleton
  columns={[{ key: 'content' }]}
  rows={8}
/>
```

The skeleton automatically handles responsive layouts, showing a multi-column desktop view and a simplified mobile card layout with actions positioned appropriately for each breakpoint.
