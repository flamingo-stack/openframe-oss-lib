This is the main entry point for a comprehensive table component system that exports all table-related components and types in a single module.

## Key Components

**Table Components:**
- `Table` - Main table container component
- `TableHeader` - Table header with sorting and filtering capabilities
- `TableRow` - Individual table row component
- `TableCell` - Basic table cell component
- `TableDescriptionCell` - Specialized cell for description content
- `TableTimestampCell` - Specialized cell for timestamp formatting
- `TableCardSkeleton` - Loading skeleton with responsive row heights
- `TableEmptyState` - Empty state component for tables with no data

**Constants:**
- `ROW_HEIGHT_DESKTOP` - Desktop row height constant
- `ROW_HEIGHT_MOBILE` - Mobile row height constant

**Type Exports:**
Core table types including `TableProps`, `TableColumn`, `RowAction`, `BulkAction`, `FilterOption`, pagination types (`CursorPagination`, `PagePagination`), and component-specific prop types.

## Usage Example

```typescript
import { 
  Table, 
  TableHeader, 
  TableRow, 
  TableCell,
  TableEmptyState,
  type TableProps,
  type TableColumn 
} from './table'

const columns: TableColumn[] = [
  { key: 'name', label: 'Name', sortable: true },
  { key: 'email', label: 'Email' }
]

function DataTable({ data }: { data: any[] }) {
  return (
    <Table>
      <TableHeader columns={columns} />
      {data.length === 0 ? (
        <TableEmptyState message="No data available" />
      ) : (
        data.map(row => (
          <TableRow key={row.id}>
            <TableCell>{row.name}</TableCell>
            <TableCell>{row.email}</TableCell>
          </TableRow>
        ))
      )}
    </Table>
  )
}
```
