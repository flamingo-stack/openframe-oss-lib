This file serves as the main entry point for a table component library, consolidating exports for all table-related components and their TypeScript type definitions.

## Key Components

**Components:**
- `Table` - Main table component
- `TableHeader` - Table header with columns and actions
- `TableRow` - Individual table row component
- `TableCell` - Basic table cell component
- `TableDescriptionCell` - Specialized cell for descriptions
- `TableTimestampCell` - Specialized cell for timestamp display
- `TableCardSkeleton` - Loading skeleton component
- `TableEmptyState` - Empty state component

**Constants:**
- `ROW_HEIGHT_DESKTOP` - Standard desktop row height
- `ROW_HEIGHT_MOBILE` - Standard mobile row height

**Types:**
- Core table types (`TableProps`, `TableColumn`, `RowAction`, `BulkAction`)
- Filter types (`FilterOption`, `FilterSection`, `TableFilters`)
- Pagination types (`CursorPagination`, `PagePagination`)
- Component-specific prop types

## Usage Example

```typescript
import { 
  Table, 
  TableHeader, 
  TableRow, 
  TableCell,
  type TableProps,
  type TableColumn 
} from './table'

const columns: TableColumn[] = [
  { key: 'name', title: 'Name' },
  { key: 'email', title: 'Email' }
]

<Table columns={columns} data={users}>
  <TableHeader />
  {users.map(user => (
    <TableRow key={user.id}>
      <TableCell>{user.name}</TableCell>
      <TableCell>{user.email}</TableCell>
    </TableRow>
  ))}
</Table>
```
