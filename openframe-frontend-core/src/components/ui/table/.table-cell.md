A flexible table cell component that provides text alignment options and responsive width handling for table layouts.

## Key Components

- **`TableCell`** - Main component that renders a table cell with configurable alignment and styling
- **`getAlignment()`** - Internal helper function that maps alignment props to Tailwind CSS classes
- **`TableCellProps`** - Type interface imported from `./types` defining component props

## Usage Example

```typescript
import { TableCell } from './table-cell'

// Basic usage with text content
<TableCell>
  User Name
</TableCell>

// Right-aligned cell with custom width
<TableCell align="right" width="w-32">
  $1,234.56
</TableCell>

// Center-aligned cell with custom styling
<TableCell 
  align="center" 
  className="bg-blue-100"
>
  Active
</TableCell>

// Cell with custom React content
<TableCell>
  <button className="text-blue-500">
    Edit User
  </button>
</TableCell>
```

The component automatically applies appropriate text styling for string/number content and supports custom React elements. It uses flexbox layout with responsive width handling and text truncation for overflow scenarios.
