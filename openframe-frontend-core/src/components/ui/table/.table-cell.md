A flexible table cell component that renders content with configurable alignment and styling, automatically handling text truncation for string and number values.

## Key Components

- **TableCell**: Main component that accepts children, alignment, custom className, and width props
- **getAlignment()**: Internal function that maps alignment props to CSS classes
- **Conditional rendering**: Applies specific typography styles for text content while allowing custom JSX elements

## Usage Example

```typescript
import { TableCell } from './table-cell'

// Basic text cell
<TableCell>User Name</TableCell>

// Right-aligned numeric cell
<TableCell align="right">$1,234.56</TableCell>

// Custom width and styling
<TableCell 
  align="center" 
  width="w-32" 
  className="bg-blue-100"
>
  Status
</TableCell>

// Custom JSX content
<TableCell>
  <button className="text-blue-500">
    Edit User
  </button>
</TableCell>
```

The component uses Tailwind CSS classes and includes built-in text truncation for overflow handling. String and number children receive default typography styling, while custom JSX elements render as-is.
