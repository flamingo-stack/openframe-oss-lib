A responsive table header component that provides sorting, filtering, and selection functionality for data tables with both desktop and mobile-optimized layouts.

## Key Components

- **TableHeader** - Main component that renders sortable column headers with optional filtering and row selection
- **getAlignment()** - Utility function that maps alignment props to Tailwind CSS classes
- **handleSort()** - Manages column sorting logic and direction toggling
- **getSortIcon()** - Returns appropriate sort indicator icons based on current sort state

## Usage Example

```typescript
import { TableHeader } from './table-header'

const columns = [
  { key: 'name', label: 'Name', sortable: true, align: 'left' },
  { 
    key: 'status', 
    label: 'Status', 
    filterable: true, 
    filterOptions: [
      { id: 'active', label: 'Active' },
      { id: 'inactive', label: 'Inactive' }
    ]
  },
  { key: 'date', label: 'Date', sortable: true, align: 'right' }
]

<TableHeader
  columns={columns}
  sortBy="name"
  sortDirection="asc"
  onSort={(key, direction) => console.log('Sort:', key, direction)}
  filters={{ status: ['active'] }}
  onFilterChange={(filters) => console.log('Filters:', filters)}
  showFilters={true}
  selectable={true}
  allSelected={false}
  onSelectAll={(checked) => console.log('Select all:', checked)}
/>
```

The component automatically handles responsive behavior, showing filter dropdowns inline on mobile and within column headers on desktop. It integrates with Lucide React icons for sort indicators and uses a custom FiltersDropdown component for advanced filtering capabilities.
