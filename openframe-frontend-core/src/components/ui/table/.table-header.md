A responsive table header component that provides sorting, filtering, and selection functionality for data tables in the Flamingo platform.

## Key Components

- **TableHeader**: Generic table header component with support for:
  - Column sorting with visual indicators (arrow icons)
  - Column filtering with dropdown interface
  - Row selection with checkbox controls
  - Responsive visibility controls for mobile/desktop
  - Customizable column alignment and widths

## Usage Example

```typescript
import { TableHeader } from './table-header'
import type { TableColumn } from './types'

interface User {
  id: string
  name: string
  email: string
  status: 'active' | 'inactive'
}

const columns: TableColumn<User>[] = [
  {
    key: 'name',
    label: 'Name',
    sortable: true,
    align: 'left'
  },
  {
    key: 'status',
    label: 'Status', 
    filterable: true,
    filterOptions: [
      { label: 'Active', value: 'active' },
      { label: 'Inactive', value: 'inactive' }
    ]
  }
]

<TableHeader
  columns={columns}
  sortBy="name"
  sortDirection="asc"
  onSort={(key, direction) => handleSort(key, direction)}
  filters={{ status: ['active'] }}
  onFilterChange={handleFilterChange}
  selectable={true}
  allSelected={false}
  onSelectAll={handleSelectAll}
  totalItemsCount={50}
/>
```

The component automatically handles responsive breakpoints, hiding non-essential columns on mobile while preserving actions and filterable columns. Sort icons change color and direction based on active state, and filter icons highlight when filters are applied.
