A React table header component that provides sorting, filtering, and selection functionality with responsive design for mobile and desktop views.

## Key Components

- **TableHeader**: Main generic component that renders sortable column headers with filtering capabilities
- **Mobile Filter Header**: Horizontal scrollable filter bar for mobile devices
- **Desktop Header**: Traditional column-based header with inline filters
- **Sorting Logic**: Handles column sorting with visual indicators (chevron icons)
- **Filter Integration**: Uses `FiltersDropdown` component for column-based filtering
- **Selection Support**: Optional row selection with "select all" checkbox

## Usage Example

```typescript
import { TableHeader } from './table-header'
import type { TableColumn } from './types'

interface User {
  id: string
  name: string
  status: string
}

const columns: TableColumn<User>[] = [
  { 
    key: 'name', 
    label: 'Name', 
    sortable: true,
    width: 'flex-1'
  },
  { 
    key: 'status', 
    label: 'Status', 
    filterable: true,
    filterOptions: [
      { label: 'Active', value: 'active' },
      { label: 'Inactive', value: 'inactive' }
    ]
  }
]

<TableHeader
  columns={columns}
  sortBy="name"
  sortDirection="asc"
  onSort={(key, direction) => console.log('Sort:', key, direction)}
  filters={{ status: ['active'] }}
  onFilterChange={(filters) => console.log('Filters:', filters)}
  selectable
  showFilters
/>
```
