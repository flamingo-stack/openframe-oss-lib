A React component for displaying truncated description text in table cells with configurable line clamping and consistent typography using ODS design tokens.

## Key Components

- **`TableDescriptionCell`** - Main component that renders description text with line clamping
- **`TableDescriptionCellProps`** - TypeScript interface defining component props including text, styling, and line limits

## Usage Example

```typescript
import { TableDescriptionCell } from './table-description-cell'

// Basic usage with default 3-line limit
<TableDescriptionCell text="This is a long description that will be truncated after three lines of text to maintain consistent table row heights." />

// Custom line limit
<TableDescriptionCell 
  text="Short description" 
  maxLines={1} 
/>

// With additional styling
<TableDescriptionCell 
  text="Custom styled description text"
  maxLines={2}
  className="text-blue-600"
/>
```

The component automatically applies appropriate Tailwind CSS line-clamp utilities based on the `maxLines` prop (1-6 lines supported) and uses the DM Sans font with ODS secondary text color for consistent styling across table interfaces.
