A React component for displaying truncated description text in table cells with consistent styling and configurable line clamping using ODS design tokens.

## Key Components

- **`TableDescriptionCell`** - Main component that renders description text with line clamping
- **`TableDescriptionCellProps`** - TypeScript interface defining component props
  - `text` - Required description text to display
  - `className` - Optional additional CSS classes
  - `maxLines` - Maximum lines before truncation (default: 3)

## Usage Example

```typescript
import { TableDescriptionCell } from './table-description-cell'

// Basic usage with default 3-line truncation
<TableDescriptionCell text="This is a long description that will be truncated after three lines..." />

// Custom line limit and styling
<TableDescriptionCell 
  text="A shorter description"
  maxLines={2}
  className="text-blue-600"
/>

// Single line truncation
<TableDescriptionCell 
  text="Very long single line text that will be truncated with ellipsis"
  maxLines={1}
/>
```

The component uses Tailwind CSS line-clamp utilities and DM Sans font with ODS design tokens for consistent text presentation across table interfaces. It automatically handles text overflow with appropriate truncation based on the specified line limit.
