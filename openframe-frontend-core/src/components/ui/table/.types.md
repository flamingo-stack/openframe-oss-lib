This file defines TypeScript interfaces and types for a comprehensive, reusable table component system with advanced features like sorting, filtering, pagination, and mobile responsiveness.

## Key Components

- **`TableColumn<T>`** - Defines column configuration with sorting, filtering, custom rendering, and responsive options
- **`TableProps<T>`** - Main table component props supporting data, styling, interactions, selection, and pagination
- **`CursorPagination`** / **`PagePagination`** - Two pagination strategies for different data loading patterns
- **`BulkAction<T>`** / **`RowAction<T>`** - Action interfaces for bulk operations and individual row actions
- **Filter Types** - `FilterOption`, `FilterSection`, `TableFilters` for advanced filtering capabilities
- **Component Props** - Supporting interfaces for table subcomponents (header, row, cell, skeleton, empty state)

## Usage Example

```typescript
// Define a user table with custom columns
const userColumns: TableColumn<User>[] = [
  {
    key: 'name',
    label: 'Name',
    sortable: true,
    renderCell: (user) => <span className="font-medium">{user.name}</span>
  },
  {
    key: 'email',
    label: 'Email',
    hideOnMobile: true,
    filterable: true
  }
]

// Configure table with pagination and actions
const tableProps: TableProps<User> = {
  data: users,
  columns: userColumns,
  rowKey: 'id',
  selectable: true,
  cursorPagination: {
    hasNextPage: true,
    onNext: (cursor) => loadMoreUsers(cursor)
  },
  bulkActions: [
    {
      label: 'Delete Selected',
      onClick: (selected) => deleteUsers(selected),
      variant: 'destructive'
    }
  ]
}
```
