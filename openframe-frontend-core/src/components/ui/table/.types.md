TypeScript interface definitions for a comprehensive table component system with sorting, filtering, pagination, and selection capabilities.

## Key Components

**Core Interfaces:**
- `TableColumn<T>` - Column configuration with rendering, sorting, and filtering options
- `TableProps<T>` - Main table component props supporting cursor/page pagination
- `CursorPagination` / `PagePagination` - Two pagination strategies for different data sources
- `TableFilters` - Filter state management with multiple filter types per column

**Action System:**
- `RowAction<T>` - Individual row actions (edit, delete, view)
- `BulkAction<T>` - Actions for selected rows (bulk delete, export)
- `FilterOption` / `FilterSection` - Configurable filter UI components

**Component Props:**
- `TableHeaderProps<T>` - Header with sorting and filter controls
- `TableRowProps<T>` - Individual row rendering with mobile responsiveness
- `TableCellProps` - Cell-level styling and alignment
- `TableCardSkeletonProps` - Loading state configuration

## Usage Example

```typescript
// Define your data type
interface User {
  id: string
  name: string
  email: string
  status: 'active' | 'inactive'
}

// Configure table columns
const columns: TableColumn<User>[] = [
  {
    key: 'name',
    label: 'Name',
    sortable: true,
    filterable: true,
    renderCell: (user) => <strong>{user.name}</strong>
  },
  {
    key: 'email',
    label: 'Email',
    align: 'left',
    hideOnMobile: true
  },
  {
    key: 'status',
    label: 'Status',
    filterOptions: [
      { id: 'active', label: 'Active', value: 'active' },
      { id: 'inactive', label: 'Inactive', value: 'inactive' }
    ]
  }
]

// Table props with pagination and actions
const tableProps: TableProps<User> = {
  data: users,
  columns,
  rowKey: 'id',
  loading: false,
  selectable: true,
  cursorPagination: {
    hasNextPage: true,
    onNext: (cursor) => loadMore(cursor),
    totalCount: 150
  },
  bulkActions: [
    {
      label: 'Delete Selected',
      variant: 'destructive',
      onClick: (users) => deleteUsers(users)
    }
  ],
  onRowClick: (user) => navigateToUser(user.id)
}
```
