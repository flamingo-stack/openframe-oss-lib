A React component that displays an empty state message for tables when no data is available, featuring customizable icons, messages, and action buttons.

## Key Components

- **TableEmptyState**: Main component that renders a centered empty state with icon, message, and optional action button
- **Props**: Accepts `message`, `icon`, `action` (with label and onClick), and `className` for customization
- **Default styling**: Dark theme with rounded borders and consistent spacing

## Usage Example

```typescript
import { TableEmptyState } from './table-empty-state'
import { Plus, Search } from 'lucide-react'

// Basic usage
<TableEmptyState message="No users found" />

// With custom icon and action
<TableEmptyState
  message="No products available"
  icon={<Search className="w-12 h-12" />}
  action={{
    label: "Add Product",
    onClick: () => setShowCreateModal(true)
  }}
/>

// With custom styling
<TableEmptyState
  message="Your cart is empty"
  icon={<Plus className="w-10 h-10" />}
  className="min-h-[300px] bg-gray-50"
/>
```

The component provides a consistent empty state experience across tables with flexible customization options for different use cases.
