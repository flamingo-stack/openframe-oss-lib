A React component that renders an empty state display for data tables, providing users with feedback when no data is available and optionally offering an action to remedy the situation.

## Key Components

- **TableEmptyState**: Main functional component that displays an empty state with icon, message, and optional action button
- **Props**: Accepts `message` (custom text), `icon` (custom icon element), `action` (button configuration), and `className` for styling
- **Default Icon**: Uses Lucide React's `FileX2` icon when no custom icon is provided
- **Styling**: Dark theme optimized with rounded borders and consistent spacing

## Usage Example

```typescript
import { TableEmptyState } from './table-empty-state'
import { Plus, Database } from 'lucide-react'

// Basic usage with default message and icon
<TableEmptyState />

// Custom message and icon
<TableEmptyState 
  message="No users found"
  icon={<Database className="w-12 h-12" />}
/>

// With action button
<TableEmptyState
  message="No projects available"
  icon={<Plus className="w-12 h-12" />}
  action={{
    label: "Create Project",
    onClick: () => handleCreateProject()
  }}
  className="min-h-[300px]"
/>
```

The component follows a centered layout pattern with consistent typography using DM Sans font family and maintains the dark theme aesthetic with appropriate hover states for interactive elements.
