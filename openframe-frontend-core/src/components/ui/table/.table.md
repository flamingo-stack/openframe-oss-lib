This file provides a comprehensive, feature-rich data table component built with React and TypeScript. It supports responsive design, sorting, filtering, row selection, bulk actions, pagination, and various customization options.

## Key Components

- **Table** - Main table component with generic type support
- **injectActionsColumn** - Utility function that automatically adds an actions column when row actions are provided
- **TableHeader** - Header component with sorting/filtering capabilities
- **TableRow** - Individual row component with mobile responsive layout
- **Selection system** - Built-in row selection with bulk actions support
- **Pagination** - Both cursor-based and page-based pagination options

## Usage Example

```typescript
import { Table } from './table'

interface User {
  id: string
  name: string
  email: string
}

const users: User[] = [
  { id: '1', name: 'John Doe', email: 'john@example.com' }
]

const columns = [
  { key: 'name', label: 'Name', sortable: true },
  { key: 'email', label: 'Email' }
]

<Table
  data={users}
  columns={columns}
  rowKey="id"
  loading={false}
  selectable
  rowActions={[
    {
      label: 'Edit',
      onClick: (user) => console.log('Edit', user),
      variant: 'outline'
    }
  ]}
  onRowClick={(user) => console.log('Row clicked', user)}
  cursorPagination={{
    hasNextPage: true,
    hasPreviousPage: false,
    onNext: () => {},
    onPrevious: () => {}
  }}
/>
```
