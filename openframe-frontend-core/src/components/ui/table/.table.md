A comprehensive table component for displaying tabular data with support for sorting, filtering, pagination, row selection, and bulk actions.

## Key Components

- **`Table<T>`** - Main generic table component that renders data in both desktop and mobile responsive layouts
- **`injectActionsColumn()`** - Utility function that automatically adds an actions column when row actions are provided
- **Selection Logic** - Built-in row selection with bulk operations support
- **Responsive Design** - Automatically adapts between desktop table view and mobile card layout
- **Pagination Support** - Supports both cursor-based and page-based pagination

## Usage Example

```typescript
import { Table } from './components/ui/table'

interface User {
  id: string
  name: string
  email: string
  role: string
}

const users: User[] = [
  { id: '1', name: 'John Doe', email: 'john@example.com', role: 'Admin' },
  { id: '2', name: 'Jane Smith', email: 'jane@example.com', role: 'User' }
]

const columns = [
  { key: 'name', label: 'Name', sortable: true },
  { key: 'email', label: 'Email' },
  { key: 'role', label: 'Role', filterable: true }
]

const rowActions = [
  {
    label: 'Edit',
    onClick: (user: User) => console.log('Edit', user),
    variant: 'outline' as const
  },
  {
    label: 'Delete',
    onClick: (user: User) => console.log('Delete', user),
    variant: 'destructive' as const
  }
]

export default function UsersTable() {
  return (
    <Table
      data={users}
      columns={columns}
      rowKey="id"
      rowActions={rowActions}
      selectable={true}
      onRowClick={(user) => console.log('Clicked', user)}
      bulkActions={[
        {
          label: 'Delete Selected',
          onClick: (selected) => console.log('Bulk delete', selected),
          requiresSelection: true
        }
      ]}
    />
  )
}
```
