A configurable row component for managing ticket status entries in a settings UI, supporting both system-defined (read-only) and custom (editable, draggable) status configurations.

## Key Components

**`TicketStatusConfigRow`** — Main export. Renders a single row with:
- Drag handle (custom rows) or disabled dragger icon (system rows)
- Name input (editable for custom, read-only for system)
- Color preset selector and/or custom color picker input (custom rows only)
- Live `TicketStatusTag` preview
- Delete button (custom) or info tooltip (system)

**`WithLeftTooltip`** — Internal helper that conditionally wraps any element in a left-aligned `TooltipProvider`. Returns children unwrapped if no `content` is provided.

**`TicketStatusConfigRowProps`** — Exported interface defining all props.

## Key Props

| Prop | Type | Description |
|------|------|-------------|
| `variant` | `'system' \| 'custom'` | Controls read-only vs. editable behavior |
| `statusKey` | `string` | Canonical key forwarded to `TicketStatusTag` |
| `dragHandleProps` | `DraggableSyntheticListeners` | From `@dnd-kit/core` for drag-and-drop |
| `deleteDisabledReason` | `string` | Tooltip shown when delete is disabled |

## Usage Example

```typescript
import { TicketStatusConfigRow } from './ticket-status-config-row'

// Custom draggable row
<TicketStatusConfigRow
  variant="custom"
  statusKey="custom-123"
  name="Waiting on Client"
  color="#FFC008"
  presetKey="amber"
  onNameChange={(val) => updateName(val)}
  onColorChange={({ color, preset }) => updateColor(color, preset)}
  onDelete={() => removeStatus('custom-123')}
  dragHandleProps={listeners}
  dragHandleAttributes={attributes}
/>

// System (read-only) row
<TicketStatusConfigRow
  variant="system"
  statusKey="RESOLVED"
  name="Resolved"
  systemTagVariant="outline"
  systemTooltip="Built-in status — cannot be deleted"
/>
```
