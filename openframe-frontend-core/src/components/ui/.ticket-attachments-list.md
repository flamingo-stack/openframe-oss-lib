A read-only list component that renders ticket file attachments with thumbnail previews, file metadata, and optional download actions.

## Key Components

**`TicketAttachment`** — Shape of a single attachment entry:
- `id` — unique key
- `fileName` / `fileSize` — display metadata
- `thumbnailSrc` — optional image preview (falls back to a generic `FileIcon`)
- `onDownload` — optional callback; omitting it hides the download button

**`TicketAttachmentsListProps`** — Component props:
- `attachments` — array of `TicketAttachment`; renders `null` when empty
- `size` — `'default'` (full-row layout) | `'compact'` (condensed for in-message / conversation-feed use)
- `className` — optional wrapper override

**`TicketAttachmentsList`** — Main export. Renders a bordered card list; each row shows a thumbnail or file icon, file name + size, and an accessible download button when `onDownload` is provided.

## Usage Example

```typescript
import { TicketAttachmentsList, TicketAttachment } from "./ticket-attachments-list"

const attachments: TicketAttachment[] = [
  {
    id: "1",
    fileName: "screenshot.png",
    fileSize: "142 KB",
    thumbnailSrc: "/uploads/screenshot.png",
    onDownload: () => window.open("/uploads/screenshot.png"),
  },
  {
    id: "2",
    fileName: "error-log.txt",
    fileSize: "8 KB",
    onDownload: () => console.log("Downloading error-log.txt"),
  },
]

// Default layout (ticket detail panel)
<TicketAttachmentsList attachments={attachments} />

// Compact layout (inside a conversation message bubble)
<TicketAttachmentsList attachments={attachments} size="compact" />
```
