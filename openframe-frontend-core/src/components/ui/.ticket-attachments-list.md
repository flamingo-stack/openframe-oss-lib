A client-side React component that renders a styled list of ticket file attachments, supporting thumbnail previews, file metadata, and optional download actions.

## Key Components

**Interfaces**
- `TicketAttachment` — Describes a single attachment: `id`, `fileName`, `fileSize`, `thumbnailSrc?`, `onDownload?`
- `TicketAttachmentsListProps` — Component props: `attachments[]`, `className?`, `size?` (`'default' | 'compact'`)

**Component**
- `TicketAttachmentsList` — Renders a bordered list of attachments; returns `null` when the array is empty

## Behavior Notes

- **Thumbnail vs. icon:** Shows a `SquareAvatar` if `thumbnailSrc` is provided, otherwise falls back to a `FileIcon` placeholder
- **Size variants:** `compact` reduces padding, icon size, and typography for inline use inside conversation feeds; `default` uses a roomier layout for standalone surfaces
- **Download button:** Only rendered when `onDownload` is defined on the attachment; includes an accessible `aria-label`
- **Dividers:** A bottom border is applied between rows (not after the last item)

## Usage Example

```typescript
import { TicketAttachmentsList } from "./ticket-attachments-list"

const attachments = [
  {
    id: "1",
    fileName: "screenshot.png",
    fileSize: "1.2 MB",
    thumbnailSrc: "/thumbs/screenshot.png",
    onDownload: () => console.log("Downloading..."),
  },
  {
    id: "2",
    fileName: "report.pdf",
    fileSize: "340 KB",
  },
]

// Full-size layout (e.g. ticket detail page)
<TicketAttachmentsList attachments={attachments} />

// Compact layout (e.g. inside a message bubble)
<TicketAttachmentsList attachments={attachments} size="compact" />
```
