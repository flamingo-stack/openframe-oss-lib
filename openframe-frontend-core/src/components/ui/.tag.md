A reusable Tag component built with class-variance-authority that provides consistent styling for labels, status indicators, and dismissible tags across the OpenFrame design system.

## Key Components

- **`Tag`** - Main component that renders customizable tag elements with optional icons and close functionality
- **`tagVariants`** - CVA variant configuration defining 7 color variants (primary, outline, good, warning, error, critical, grey)
- **`TagProps`** - TypeScript interface extending HTML div attributes with variant props, icon support, and close handler

## Usage Example

```typescript
import { Tag } from './tag'
import { CheckCircle, AlertTriangle } from 'lucide-react'

// Basic tag
<Tag>Default</Tag>

// Status tags with variants
<Tag variant="good">Success</Tag>
<Tag variant="error">Error</Tag>
<Tag variant="warning">Warning</Tag>

// Tag with icon
<Tag variant="good" icon={<CheckCircle />}>
  Verified
</Tag>

// Dismissible tag
<Tag 
  variant="outline" 
  onClose={() => console.log('Tag removed')}
>
  Removable
</Tag>

// Custom styling
<Tag variant="primary" className="text-lg">
  Custom Tag
</Tag>
```

The component uses the OpenFrame design system CSS variables and includes hover/active states, smooth transitions, and accessibility features like proper ARIA labels for the close button.
