A React component for displaying labeled tags with multiple visual variants and optional close functionality. The component uses class-variance-authority for consistent styling across different states and contexts.

## Key Components

- **`Tag`** - Main component that renders a styled tag with optional icon and close button
- **`tagVariants`** - CVA configuration defining styling variants (primary, outline, good, warning, error, critical, grey)
- **`TagProps`** - Interface extending HTML div attributes with variant props, icon, and close handler

## Usage Example

```typescript
import { Tag } from './tag'
import { AlertCircle } from 'lucide-react'

// Basic tag
<Tag>Development</Tag>

// Tag with variant
<Tag variant="error">Critical Issue</Tag>

// Tag with icon and close functionality
<Tag 
  variant="success"
  icon={<AlertCircle />}
  onClose={() => console.log('Tag removed')}
>
  Success
</Tag>

// Custom styling
<Tag 
  variant="outline"
  className="custom-spacing"
>
  Custom Tag
</Tag>
```

The component supports seven visual variants with hover and active states, uses CSS custom properties for theming, and includes accessibility features like proper ARIA labels for the close button. The close button prevents event propagation to avoid triggering parent click handlers.
