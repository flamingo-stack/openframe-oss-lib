A comprehensive menubar component built on Radix UI primitives with styled components for creating accessible desktop-style menu interfaces. This component provides a complete suite of menu elements including triggers, content panels, items, and interactive elements with consistent styling and keyboard navigation.

## Key Components

- **Menubar** - Root container component with horizontal layout
- **MenubarMenu** - Individual menu wrapper
- **MenubarTrigger** - Clickable menu button that opens dropdown content
- **MenubarContent** - Dropdown panel containing menu items
- **MenubarItem** - Basic selectable menu item
- **MenubarCheckboxItem** - Toggleable menu item with check indicator
- **MenubarRadioItem** - Single-select menu item with radio indicator
- **MenubarSubTrigger/MenubarSubContent** - Nested submenu components
- **MenubarSeparator** - Visual divider between menu sections
- **MenubarLabel** - Non-interactive section headers
- **MenubarShortcut** - Keyboard shortcut display component
