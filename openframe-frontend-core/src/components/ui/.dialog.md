A styled dialog component system built on top of Radix UI's Dialog primitive, providing a complete modal implementation with overlay, animations, and customizable styling.

## Key Components

- **Dialog** - Root dialog component (alias for DialogPrimitive.Root)
- **DialogTrigger** - Button to open the dialog (alias for DialogPrimitive.Trigger)
- **DialogContent** - Main dialog container with overlay, close button, and animations
- **DialogOverlay** - Backdrop overlay with blur effect and fade animations
- **DialogHeader/DialogFooter** - Layout containers for dialog sections
- **DialogTitle/DialogDescription** - Semantic text components with proper styling
- **DialogClose** - Close button component (alias for DialogPrimitive.Close)

## Usage Example

```typescript
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "./dialog"

function MyDialog() {
  return (
    <Dialog>
      <DialogTrigger asChild>
        <button>Open Dialog</button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Confirm Action</DialogTitle>
          <DialogDescription>
            This action cannot be undone. Are you sure?
          </DialogDescription>
        </DialogHeader>
        <DialogFooter>
          <button type="submit">Confirm</button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}
```

The dialog features automatic backdrop blur, smooth zoom/slide animations, responsive design, and accessibility features including focus management and keyboard navigation.
