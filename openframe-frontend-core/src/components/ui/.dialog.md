A styled dialog component library built on top of Radix UI primitives, providing pre-configured modal dialogs with consistent styling and animations.

## Key Components

- **Dialog** - Root dialog container component
- **DialogTrigger** - Button/element that opens the dialog
- **DialogContent** - Main dialog content container with backdrop and close button
- **DialogOverlay** - Semi-transparent backdrop behind the dialog
- **DialogHeader/DialogFooter** - Layout components for dialog sections
- **DialogTitle/DialogDescription** - Semantic text components for dialog content
- **DialogClose** - Component for closing the dialog

## Usage Example

```typescript
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "./dialog"

export function MyDialog() {
  return (
    <Dialog>
      <DialogTrigger asChild>
        <button>Open Dialog</button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Confirm Action</DialogTitle>
          <DialogDescription>
            Are you sure you want to proceed?
          </DialogDescription>
        </DialogHeader>
        <DialogFooter>
          <button type="submit">Confirm</button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}
```

The components automatically handle accessibility, focus management, and smooth animations for opening/closing states.
