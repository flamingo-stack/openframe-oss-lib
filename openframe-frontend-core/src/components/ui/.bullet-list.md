A client-side React component that renders a customizable bullet list with flexible styling options and support for custom bullet icons.

## Key Components

- **`BulletListItem`** - Interface defining list item structure with text and optional ID
- **`BulletListProps`** - Props interface with extensive customization options including bullet styling, spacing, and CSS classes
- **`BulletList`** - Main component that renders items with bullets, supporting both string arrays and object arrays

## Usage Example

```typescript
import { BulletList } from './bullet-list';

// Simple string array
<BulletList 
  items={['First item', 'Second item', 'Third item']}
  spacing="lg"
/>

// With custom styling and bullet
<BulletList
  items={[
    { text: 'Custom item 1', id: 'item1' },
    { text: 'Custom item 2', id: 'item2' }
  ]}
  bulletColor="#ff6b6b"
  bulletSize={20}
  spacing="sm"
  textClassName="text-lg font-bold"
  containerClassName="max-w-md"
/>
```

The component automatically normalizes string arrays to object format and provides consistent spacing through predefined spacing maps (sm, md, lg).
