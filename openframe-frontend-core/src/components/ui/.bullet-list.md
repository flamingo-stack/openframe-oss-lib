A React component for rendering customizable bulleted lists with flexible styling and icon options.

## Key Components

- **`BulletList`** - Main component that renders a list of items with customizable bullet icons
- **`BulletListItem`** - Interface for individual list items with optional ID
- **`BulletListProps`** - Configuration interface for bullet appearance, spacing, and styling
- **`spacingMap`** - Predefined spacing options (sm/md/lg) using Tailwind classes

## Usage Example

```typescript
import { BulletList } from './bullet-list';
import { CustomIcon } from './icons';

// Simple string array
<BulletList 
  items={["First item", "Second item", "Third item"]}
  spacing="lg"
/>

// Advanced configuration with custom styling
<BulletList 
  items={[
    { text: "Custom bullet color", id: "item-1" },
    { text: "Different icon", id: "item-2" }
  ]}
  bulletColor="#ff6b6b"
  bulletIcon={CustomIcon}
  bulletSize={20}
  spacing="sm"
  textClassName="text-lg font-bold"
  containerClassName="bg-gray-100 p-4 rounded-lg"
/>
```

The component automatically normalizes string arrays to `BulletListItem` objects and provides responsive typography with DM Sans font by default. Each item is rendered with a flex layout for proper bullet alignment.
