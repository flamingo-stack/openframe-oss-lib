A React component that displays status information with a colored dot indicator and optional link functionality. The component supports four status types (success, pending, error, missing) with corresponding color schemes.

## Key Components

- **StatusIndicatorProps**: Interface defining component props with status type, label text, and optional href
- **StatusIndicator**: Main functional component that renders a status dot and label
- **getStatusColor**: Internal helper function that maps status values to Tailwind CSS color classes

## Usage Example

```typescript
import { StatusIndicator } from './status-indicator';

// Basic status indicator
<StatusIndicator 
  status="success" 
  label="Deployment Complete" 
/>

// Status indicator with link
<StatusIndicator 
  status="error" 
  label="Build Failed" 
  href="https://logs.example.com/build-123" 
/>

// Different status types
<StatusIndicator status="pending" label="In Progress" />
<StatusIndicator status="missing" label="Not Found" />
```

The component renders a circular dot with status-appropriate colors (green for success, orange for pending/missing, red for error) alongside a labeled text. When an `href` prop is provided, the entire indicator becomes a clickable link that opens in a new tab.
