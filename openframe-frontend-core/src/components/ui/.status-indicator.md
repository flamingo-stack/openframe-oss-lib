A React component that displays a status indicator with a colored dot and label, optionally wrapped in a clickable link.

## Key Components

- **`StatusIndicatorProps`** - Interface defining the component props:
  - `status`: One of 'success', 'pending', 'error', or 'missing'
  - `label`: Text to display next to the status dot
  - `href`: Optional URL to make the indicator clickable
- **`StatusIndicator`** - Main component that renders a colored dot with text label
- **`getStatusColor`** - Internal helper function that maps status types to Tailwind CSS color classes

## Usage Example

```typescript
import { StatusIndicator } from './status-indicator';

// Basic status indicator
<StatusIndicator status="success" label="Build Complete" />

// With clickable link
<StatusIndicator 
  status="error" 
  label="Test Failed" 
  href="https://example.com/logs" 
/>

// Different status types
<StatusIndicator status="pending" label="Deploying..." />
<StatusIndicator status="missing" label="Config Not Found" />
```

The component uses a color-coded system: green for success, orange for pending/missing states, and red for errors. When an `href` is provided, the entire indicator becomes a clickable link that opens in a new tab. The styling uses Tailwind CSS classes with DM Sans font for consistent typography.
