A React component that renders an SVG icon representing a community or group of people, commonly used for user groups, teams, or social features.

## Key Components

- **CommunityIcon**: Main React functional component that renders a scalable SVG icon
- **CommunityIconProps**: TypeScript interface defining optional props for customization
  - `className`: CSS class name for styling
  - `width`: Icon width (default: 24px)
  - `height`: Icon height (default: 24px)

## Usage Example

```typescript
import { CommunityIcon } from './community-icon';

// Basic usage with defaults
<CommunityIcon />

// Customized size and styling
<CommunityIcon 
  width={32} 
  height={32} 
  className="text-blue-500 hover:text-blue-600" 
/>

// In a button or nav item
<button className="flex items-center gap-2">
  <CommunityIcon width={20} height={20} />
  Community
</button>
```

The icon uses `currentColor` for the fill, making it easily themeable through CSS text color properties. The SVG maintains its aspect ratio and scales cleanly at different sizes.
