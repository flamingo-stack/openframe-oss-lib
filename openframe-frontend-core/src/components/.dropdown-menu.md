A complete dropdown menu component library built on Radix UI primitives with custom styling and enhanced functionality. This module provides all the necessary components to create accessible dropdown menus with features like checkboxes, radio buttons, submenus, and keyboard shortcuts.

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Button element that opens the dropdown
- **DropdownMenuContent** - Main dropdown container with portal rendering
- **DropdownMenuItem** - Individual clickable menu items
- **DropdownMenuCheckboxItem** - Menu items with checkbox functionality
- **DropdownMenuRadioItem** - Menu items for radio button groups
- **DropdownMenuLabel** - Section headers within the menu
- **DropdownMenuSeparator** - Visual dividers between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub/SubTrigger/SubContent** - Nested submenu components
- **DropdownMenuGroup/RadioGroup** - Logical grouping containers
