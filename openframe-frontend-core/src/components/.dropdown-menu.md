A comprehensive dropdown menu component library built on top of Radix UI primitives with custom styling and enhanced functionality. Provides a complete set of dropdown menu elements including items, checkboxes, radio buttons, separators, and nested submenus.

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Button that opens the dropdown
- **DropdownMenuContent** - Main dropdown container with portal rendering
- **DropdownMenuItem** - Individual menu item with optional inset styling
- **DropdownMenuCheckboxItem** - Checkbox item with check indicator
- **DropdownMenuRadioItem** - Radio button item with circle indicator
- **DropdownMenuLabel** - Section label with semibold styling
- **DropdownMenuSeparator** - Visual divider between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub** - Nested submenu components (SubTrigger, SubContent)
- **DropdownMenuGroup/RadioGroup** - Grouping containers

## Usage Example

```typescript
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuCheckboxItem,
  DropdownMenuSeparator,
  DropdownMenuShortcut
} from "./dropdown-menu"

<DropdownMenu>
  <DropdownMenuTrigger>Open Menu</DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuItem>
      Profile
      <DropdownMenuShortcut>⇧⌘P</DropdownMenuShortcut>
    </DropdownMenuItem>
    <DropdownMenuCheckboxItem checked={true}>
      Show toolbar
    </DropdownMenuCheckboxItem>
    <DropdownMenuSeparator />
    <DropdownMenuItem>Settings</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```
