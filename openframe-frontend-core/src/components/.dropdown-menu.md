A comprehensive dropdown menu component library built on top of Radix UI primitives with custom styling using the OpenFrame design system (ODS). Provides a complete set of accessible dropdown menu components including items, checkboxes, radio groups, submenus, and separators.

## Key Components

- **DropdownMenu** - Root container component
- **DropdownMenuTrigger** - Button that opens the menu
- **DropdownMenuContent** - The dropdown panel with animation and positioning
- **DropdownMenuItem** - Individual menu items with hover states
- **DropdownMenuCheckboxItem** - Checkbox menu items with check indicators
- **DropdownMenuRadioItem** - Radio button menu items for single selection
- **DropdownMenuLabel** - Section headers within the menu
- **DropdownMenuSeparator** - Visual dividers between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuSub/SubContent/SubTrigger** - Nested submenu components
- **DropdownMenuGroup/RadioGroup** - Logical grouping containers
