A React SVG component that renders the X (formerly Twitter) logo icon with customizable styling and SVG properties.

## Key Components

- **XLogo**: Main React component that renders the X logo as an inline SVG
- **XLogoProps**: TypeScript interface extending standard SVG props with optional className

## Usage Example

```typescript
import { XLogo } from './x-logo';

// Basic usage
<XLogo />

// With custom styling
<XLogo className="w-8 h-8 text-blue-500" />

// With custom size and click handler
<XLogo 
  width={32} 
  height={32} 
  onClick={() => window.open('https://x.com')}
  className="cursor-pointer hover:opacity-80"
/>

// With aria label for accessibility
<XLogo 
  aria-label="Follow us on X" 
  role="img"
  className="social-icon"
/>
```

The component accepts all standard SVG element properties through prop spreading, making it highly flexible for different use cases. The default size is 24x24 pixels with a white fill color (#FAFAFA).
