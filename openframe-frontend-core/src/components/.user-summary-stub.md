A client-side React component that displays user profile information with avatar, name, email, and optional MSP (Managed Service Provider) details in both compact and full layouts.

## Key Components

**Main Export:**
- `UserSummary` - Primary component with two display modes (compact/full)

**Helper Functions:**
- `getAuthProviderIcon()` - Returns provider-specific icons (Google, Microsoft, Slack)
- `formatNumber()` - Abbreviates large numbers (1.2K, 15M, 2B format)

**Key Features:**
- Responsive design with mobile/desktop variants
- Avatar with optional MSP logo badge overlay
- Authentication provider indicators
- Editable photo support
- MSP preview with company metrics

## Usage Example

```typescript
import { UserSummary } from './user-summary-stub';

// Basic usage
<UserSummary 
  name="John Doe" 
  email="john@company.com" 
  avatarUrl="/avatars/john.jpg"
/>

// Compact mode for comments/lists
<UserSummary 
  name="Jane Smith" 
  email="jane@msp.com"
  compact={true}
  avatarSize={32}
/>

// Full profile with MSP details
<UserSummary 
  name="Mike Johnson"
  email="mike@techcorp.com"
  authProviders={["google", "microsoft"]}
  showEditButton={true}
  mspPreview={{
    name: "TechCorp MSP",
    seatCount: 150,
    technicianCount: 12,
    annualRevenue: 2500000,
    logoUrl: "/logos/techcorp.png"
  }}
/>
```
