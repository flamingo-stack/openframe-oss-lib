A reusable alert component system built with React and class-variance-authority that provides accessible notification and message display with customizable styling variants.

## Key Components

- **Alert**: Main container component with `default` and `destructive` variants, includes proper ARIA role and icon positioning
- **AlertTitle**: Heading component for alert titles with medium font weight and tight tracking
- **AlertDescription**: Content wrapper for alert descriptions with relaxed line spacing
- **alertVariants**: CVA configuration defining base styles and variant-specific styling

## Usage Example

```typescript
import { Alert, AlertTitle, AlertDescription } from "./components/alert"
import { AlertCircle } from "lucide-react"

// Basic alert
<Alert>
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>
    You can add components to your app using the cli.
  </AlertDescription>
</Alert>

// Destructive alert with icon
<Alert variant="destructive">
  <AlertCircle className="h-4 w-4" />
  <AlertTitle>Error</AlertTitle>
  <AlertDescription>
    Your session has expired. Please log in again.
  </AlertDescription>
</Alert>

// Custom styling
<Alert className="border-blue-200 bg-blue-50">
  <AlertTitle>Custom Alert</AlertTitle>
  <AlertDescription>This alert uses custom classes.</AlertDescription>
</Alert>
```
