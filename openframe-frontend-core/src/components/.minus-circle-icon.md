A React SVG icon component that renders a minus sign inside a circle, commonly used for decrement actions or removing items.

## Key Components

- **MinusCircleIcon**: Main functional component that renders the SVG icon
- **MinusCircleIconProps**: Interface extending `React.SVGProps<SVGSVGElement>` with optional `className` prop
- **SVG Elements**: Two paths creating a circular outline and horizontal minus line

## Usage Example

```typescript
import { MinusCircleIcon } from './minus-circle-icon'

// Basic usage
<MinusCircleIcon />

// With custom styling
<MinusCircleIcon 
  className="text-red-500 hover:text-red-700" 
  width="32" 
  height="32" 
/>

// As a button for decrement functionality
<button onClick={handleDecrement}>
  <MinusCircleIcon className="w-5 h-5" />
</button>

// With additional SVG props
<MinusCircleIcon 
  fill="#FF0000"
  onClick={handleClick}
  aria-label="Remove item"
/>
```

The component accepts all standard SVG element props and forwards them to the underlying `<svg>` element, making it highly flexible for different use cases and styling approaches.
