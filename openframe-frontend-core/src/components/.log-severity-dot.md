A React component that renders a colored dot indicator representing different log severity levels with configurable sizing options.

## Key Components

- **LogSeverityDot**: Forwardable React component that displays severity-colored dots
- **Props**: Accepts `severity`, `size` ('sm'|'md'|'lg'), and `className`
- **Size variants**: Three predefined sizes with corresponding dot dimensions
- **Color mapping**: Uses `LOG_SEVERITY_COLORS` constant for dynamic color assignment
- **Accessibility**: Includes proper ARIA labels using `LOG_SEVERITY_LABELS`

## Usage Example

```typescript
import { LogSeverityDot } from './components/log-severity-dot'

// Basic usage with default medium size
<LogSeverityDot severity="error" />

// Custom size and styling
<LogSeverityDot 
  severity="warning" 
  size="lg" 
  className="mr-2" 
/>

// With ref for DOM access
const dotRef = useRef<HTMLDivElement>(null)
<LogSeverityDot 
  ref={dotRef}
  severity="info" 
  size="sm" 
/>
```

The component automatically applies appropriate colors based on the severity level and provides consistent sizing across different contexts. It's commonly used in log viewers, status indicators, and dashboard components where visual severity representation is needed.
