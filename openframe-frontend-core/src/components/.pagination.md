A React pagination component that provides navigation through multi-page data with smart page number generation, ellipsis handling, and full accessibility support.

## Key Components

- **`Pagination`** - Main pagination component with intelligent page number display logic
- **`PaginationContent`** - Container component for pagination items  
- **`PaginationItem`** - Individual pagination item wrapper
- **`PaginationLink`** - Clickable page number button
- **`PaginationEllipsis`** - Ellipsis indicator for truncated page ranges
- **`PaginationPrevious`** - Previous page navigation button
- **`PaginationNext`** - Next page navigation button

The main `Pagination` component automatically handles:
- Smart page range calculation (shows max 5 pages)
- Ellipsis insertion for large page counts
- First and last page always visible
- Proper ARIA labels and navigation semantics
- Disabled states for boundary pages

## Usage Example

```typescript
import { Pagination } from './pagination'

function DataTable() {
  const [currentPage, setCurrentPage] = useState(1)
  const totalPages = 25

  return (
    <div>
      {/* Your data content */}
      
      <Pagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
        className="mt-8"
      />
    </div>
  )
}

// Using individual components for custom layouts
function CustomPagination() {
  return (
    <nav>
      <PaginationContent>
        <PaginationItem>
          <PaginationPrevious onClick={() => goToPrevious()} />
        </PaginationItem>
        <PaginationItem>
          <PaginationLink onClick={() => goToPage(1)}>1</PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationEllipsis />
        </PaginationItem>
        <PaginationItem>
          <PaginationNext onClick={() => goToNext()} />
        </PaginationItem>
      </PaginationContent>
    </nav>
  )
}
```
