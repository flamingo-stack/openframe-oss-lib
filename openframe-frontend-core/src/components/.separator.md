A React component that renders a visual separator line, built on top of Radix UI's Separator primitive with customizable orientation and styling.

## Key Components

- **Separator**: A forwardRef component that creates horizontal or vertical divider lines
- Uses Radix UI's `SeparatorPrimitive.Root` for accessibility features
- Supports both horizontal (default) and vertical orientations
- Includes proper semantic attributes for screen readers

## Usage Example

```typescript
import { Separator } from "./separator"

// Horizontal separator (default)
<Separator />

// Vertical separator
<Separator orientation="vertical" />

// Custom styling
<Separator className="bg-red-500" />

// Non-decorative separator for accessibility
<Separator decorative={false} />

// In a layout context
<div className="space-y-4">
  <div>Content above</div>
  <Separator />
  <div>Content below</div>
</div>

// Vertical separator in a flex layout
<div className="flex items-center space-x-4 h-10">
  <span>Item 1</span>
  <Separator orientation="vertical" />
  <span>Item 2</span>
</div>
```

The component automatically applies appropriate dimensions (1px thickness) and uses the `cn` utility for conditional class merging.
