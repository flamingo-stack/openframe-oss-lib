A React component for displaying pricing information with configurable styling and support for both structured pricing data and legacy string formats.

## Key Components

- **`PricingDisplay`** - Main component that renders pricing with configurable styles
- **`PRICING_STYLES`** - Predefined style configurations for different contexts (comparison, compact, card)
- **`formatPriceValue`** - Utility function to format price values with optional tilde prefix
- **`LegacyPricingDisplay`** - Backward compatibility component for string pricing inputs
- **`formatPricingForDisplay`** - Utility function for preprocessing pricing strings

## Usage Example

```typescript
import { PricingDisplay, PRICING_STYLES } from './pricing-display'

// Using structured pricing data
const pricingData = [{
  ranges: [{ min: 10, unit: 'month' }]
}]

// Basic usage with default styling
<PricingDisplay pricing={pricingData} />

// With custom style configuration
<PricingDisplay 
  pricing={pricingData}
  styleConfig={PRICING_STYLES.compact}
  className="custom-class"
/>

// Legacy string support
<PricingDisplay 
  pricing="$10/month, $120/year"
  styleConfig={PRICING_STYLES.card}
/>

// Multiple pricing items
const multiplePricing = [
  { ranges: [{ min: 10, unit: 'month' }] },
  { ranges: [{ min: 100, unit: 'year' }] }
]
<PricingDisplay pricing={multiplePricing} />
```
