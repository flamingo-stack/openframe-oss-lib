A React component that provides consistent, configurable pricing display with support for both structured pricing data and legacy string formats.

## Key Components

- **PricingDisplay** - Main component accepting `StructuredPricingItem[]`, single `StructuredPricingItem`, or legacy string pricing data
- **PRICING_STYLES** - Predefined style configurations for different UI contexts (comparison tables, compact dropdowns, vendor cards)
- **PricingStyleConfig** - Interface defining typography, colors, and formatting options
- **LegacyPricingDisplay** - Backward compatibility component for string-based pricing
- **formatPriceValue** - Utility for consistent price formatting with optional tilde prefix
- **formatPricingForDisplay** - Export utility for preprocessing pricing strings

## Usage Example

```typescript
import { PricingDisplay, PRICING_STYLES } from './pricing-display'

// Using structured pricing data
const structuredPricing = [{
  ranges: [{ min: 29, unit: 'month' }]
}]

// Different style contexts
<PricingDisplay 
  pricing={structuredPricing} 
  styleConfig={PRICING_STYLES.comparison}
/>

<PricingDisplay 
  pricing="$10/device/month, $120/device/year" 
  styleConfig={PRICING_STYLES.compact}
  className="custom-spacing"
/>

// Multiple pricing tiers
<PricingDisplay 
  pricing={[
    { ranges: [{ min: 10, unit: 'month' }] },
    { ranges: [{ min: 100, unit: 'year' }] }
  ]}
  styleConfig={PRICING_STYLES.card}
/>
```

The component handles various pricing formats including free tiers, price ranges, multiple billing cycles, and provides consistent typography and color styling across different UI contexts.
