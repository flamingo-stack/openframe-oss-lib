A customizable date and time picker component built on top of react-datepicker with dark theme styling and consistent design system integration.

## Key Components

- **DateTimePicker** - Main component that wraps react-datepicker with custom styling
- **DateTimePickerProps** - Interface defining all configuration options including date formats, time settings, and styling props
- **inputClassName** - Pre-configured CSS classes matching the design system's input styling

## Usage Example

```typescript
import { DateTimePicker } from './date-time-picker';
import { useState } from 'react';

function MyComponent() {
  const [selectedDate, setSelectedDate] = useState<Date | null>(null);

  return (
    <DateTimePicker
      value={selectedDate}
      onChange={setSelectedDate}
      label="Event Date"
      placeholder="Choose when the event starts"
      showTimeSelect={true}
      timeIntervals={30}
      dateFormat="MMM dd, yyyy h:mm aa"
      disabled={false}
    />
  );
}

// Date-only picker
<DateTimePicker
  value={date}
  onChange={setDate}
  showTimeSelect={false}
  dateFormat="MMMM d, yyyy"
/>
```

The component includes comprehensive dark theme styling, mobile-responsive design, and integrates with the ODS design system colors and typography.
