A reusable date and time picker component built with react-datepicker, featuring dark theme styling and consistent form integration with the design system.

## Key Components

- **DateTimePicker**: Main component that wraps react-datepicker with custom styling
- **DateTimePickerProps**: TypeScript interface defining component configuration options
- **inputClassName**: Shared styling constants that match the design system's Input component pattern

## Usage Example

```typescript
import { DateTimePicker } from './components/date-time-picker';
import { useState } from 'react';

function EventForm() {
  const [eventDate, setEventDate] = useState<Date | null>(null);

  return (
    <div className="space-y-4">
      <DateTimePicker
        label="Event Date & Time"
        value={eventDate}
        onChange={setEventDate}
        placeholder="When is your event?"
        showTimeSelect={true}
        timeIntervals={30}
        dateFormat="MMM dd, yyyy h:mm aa"
      />
      
      {/* Date-only picker */}
      <DateTimePicker
        label="Deadline"
        value={eventDate}
        onChange={setEventDate}
        showTimeSelect={false}
        dateFormat="MMMM d, yyyy"
      />
    </div>
  );
}
```

The component features full-width responsive design, dark theme integration, customizable date/time formats, and matches the ODS design system colors including the signature yellow (#FFC008) accent color for selections.
