A React input component with dark theme styling, error states, and consistent design matching the admin theme palette.

## Key Components

- **Input**: Forwardable React component extending standard HTML input attributes
- **InputProps**: Interface adding an optional `invalid` boolean prop for error styling

## Usage Example

```typescript
import { Input } from './components/input'

function LoginForm() {
  const [email, setEmail] = useState('')
  const [hasError, setHasError] = useState(false)

  return (
    <form>
      <Input
        type="email"
        placeholder="Enter your email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        invalid={hasError}
      />
      
      <Input
        type="file"
        accept=".pdf,.doc"
        className="mt-4"
      />
    </form>
  )
}
```

The component features a dark theme (`bg-[#161616]`), yellow accent colors (`#FFC008`), responsive typography, and automatic error styling when `invalid={true}` is passed. It maintains accessibility with proper focus states and supports file inputs with custom styling.
