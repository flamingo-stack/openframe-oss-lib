A React component that renders an SVG icon representing a community hub, featuring interconnected user figures in a scalable vector format.

## Key Components

- **CommunityHubIcon**: Main functional component that renders the community hub SVG icon
- **CommunityHubIconProps**: TypeScript interface defining optional props for customization
  - `className`: Optional CSS class name for styling
  - `width`: Icon width (defaults to 19px)
  - `height`: Icon height (defaults to 16px)

## Usage Example

```typescript
import { CommunityHubIcon } from './community-hub-icon';

// Basic usage with defaults
<CommunityHubIcon />

// Custom size and styling
<CommunityHubIcon 
  width={24} 
  height={20} 
  className="text-blue-500 hover:text-blue-600" 
/>

// In a button or navigation
<button className="flex items-center gap-2">
  <CommunityHubIcon width={16} height={14} />
  Community
</button>
```

The icon uses `currentColor` for fill values, making it easily themeable through CSS color properties. The SVG maintains its aspect ratio and scales cleanly at different sizes.
