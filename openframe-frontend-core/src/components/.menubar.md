A comprehensive menubar component library built on Radix UI primitives with custom styling and enhanced functionality. Provides a complete set of accessible menubar components including triggers, items, separators, and submenus with built-in keyboard navigation and ARIA support.

## Key Components

- **Menubar** - Root container component with horizontal layout and border styling
- **MenubarTrigger** - Clickable trigger buttons that open menu content
- **MenubarContent/MenubarSubContent** - Dropdown containers with portal rendering and animations
- **MenubarItem** - Basic menu items with hover states and optional inset styling
- **MenubarCheckboxItem** - Checkable items with check mark indicators
- **MenubarRadioItem** - Radio button items with circle indicators for single selection
- **MenubarLabel** - Non-interactive labels for grouping menu sections
- **MenubarSeparator** - Visual dividers between menu sections
- **MenubarShortcut** - Keyboard shortcut display component
- **MenubarSubTrigger** - Items that trigger nested submenus with chevron icons

## Usage Example

```typescript
import {
  Menubar,
  MenubarContent,
  MenubarItem,
  MenubarMenu,
  MenubarSeparator,
  MenubarShortcut,
  MenubarTrigger,
} from "./menubar"

function AppMenubar() {
  return (
    <Menubar>
      <MenubarMenu>
        <MenubarTrigger>File</MenubarTrigger>
        <MenubarContent>
          <MenubarItem>
            New File <MenubarShortcut>⌘N</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            Open <MenubarShortcut>⌘O</MenubarShortcut>
          </MenubarItem>
          <MenubarSeparator />
          <MenubarItem>Exit</MenubarItem>
        </MenubarContent>
      </MenubarMenu>
    </Menubar>
  )
}
```
