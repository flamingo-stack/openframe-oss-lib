A styled React component library for creating accessible menubar interfaces, built as wrapper components around Radix UI's menubar primitives with consistent design system styling.

## Key Components

- **Menubar** - Root container with horizontal layout and border styling
- **MenubarTrigger** - Interactive button that opens menu content
- **MenubarContent/MenubarSubContent** - Dropdown containers with animations and portal rendering
- **MenubarItem** - Basic clickable menu items with hover states
- **MenubarCheckboxItem** - Toggleable items with check indicators
- **MenubarRadioItem/MenubarRadioGroup** - Single-selection radio button items
- **MenubarSeparator** - Visual dividers between menu sections
- **MenubarLabel** - Non-interactive section headers
- **MenubarShortcut** - Keyboard shortcut display component
