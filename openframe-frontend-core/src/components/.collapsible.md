A React component wrapper that re-exports Radix UI's Collapsible primitive components for creating expandable/collapsible content sections.

## Key Components

- **`Collapsible`** - Root container component that manages the collapsible state
- **`CollapsibleTrigger`** - Interactive element (button) that toggles the collapsible content
- **`CollapsibleContent`** - Container for the content that can be expanded or collapsed

## Usage Example

```typescript
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from "./collapsible"

function MyCollapsibleSection() {
  return (
    <Collapsible>
      <CollapsibleTrigger className="flex items-center gap-2 p-2">
        <span>Toggle Section</span>
        <ChevronDownIcon />
      </CollapsibleTrigger>
      <CollapsibleContent className="overflow-hidden">
        <div className="p-4 border-t">
          <p>This content can be expanded or collapsed.</p>
          <p>Add any content here that should be toggleable.</p>
        </div>
      </CollapsibleContent>
    </Collapsible>
  )
}
```

The components inherit all props and functionality from Radix UI's Collapsible primitives, providing accessible keyboard navigation, ARIA attributes, and smooth animations out of the box.
