This file provides a comprehensive sidebar component system with collapsible functionality, mobile responsiveness, and keyboard shortcuts. It includes a context provider, main sidebar component, and various sub-components for building flexible sidebar layouts.

## Key Components

- **`SidebarProvider`** - Context provider that manages sidebar state and mobile detection
- **`Sidebar`** - Main sidebar container with variants (sidebar, floating, inset) and collapsible modes
- **`SidebarTrigger`** - Button component to toggle sidebar visibility
- **`SidebarRail`** - Invisible rail for hover-based sidebar interaction
- **`SidebarInset`** - Main content area that adjusts based on sidebar state
- **`SidebarHeader/Footer`** - Layout components for sidebar sections
- **`SidebarInput`** - Styled input component for sidebar use
- **`SidebarSeparator`** - Divider component for sidebar sections
- **`useSidebar`** - Hook to access sidebar context and controls

## Usage Example

```typescript
import { 
  SidebarProvider, 
  Sidebar, 
  SidebarContent, 
  SidebarHeader, 
  SidebarTrigger,
  SidebarInset 
} from "./sidebar"

function App() {
  return (
    <SidebarProvider>
      <Sidebar>
        <SidebarHeader>
          <h2>Navigation</h2>
        </SidebarHeader>
        <SidebarContent>
          {/* Sidebar navigation items */}
        </SidebarContent>
      </Sidebar>
      <SidebarInset>
        <header>
          <SidebarTrigger />
        </header>
        <main>{/* Main content */}</main>
      </SidebarInset>
    </SidebarProvider>
  )
}
```

The sidebar automatically handles mobile responsiveness, keyboard shortcuts (Cmd/Ctrl+B), and persists state via cookies.
