A comprehensive, customizable sidebar component system built with React and Radix UI that supports responsive behavior, multiple variants, and keyboard shortcuts. The component provides context-based state management with persistent cookie storage and mobile-first design patterns.

## Key Components

- **SidebarProvider** - Context provider managing sidebar state with cookie persistence
- **Sidebar** - Main sidebar container with responsive behavior and multiple variants
- **SidebarTrigger** - Button component to toggle sidebar visibility
- **SidebarRail** - Interactive edge element for resizing/toggling
- **SidebarInset** - Main content area that adapts to sidebar state
- **SidebarHeader/Footer** - Standardized header and footer sections
- **SidebarInput** - Styled input component for sidebar search/forms
- **SidebarSeparator** - Visual divider component
- **useSidebar** - Hook for accessing sidebar context and controls

## Usage Example

```typescript
import { 
  SidebarProvider, 
  Sidebar, 
  SidebarContent, 
  SidebarHeader,
  SidebarTrigger,
  SidebarInset 
} from './sidebar'

function App() {
  return (
    <SidebarProvider defaultOpen={true}>
      <div className="flex">
        <Sidebar variant="sidebar" collapsible="icon">
          <SidebarHeader>
            <SidebarTrigger />
            <h2>Navigation</h2>
          </SidebarHeader>
          <SidebarContent>
            {/* Navigation items */}
          </SidebarContent>
        </Sidebar>
        
        <SidebarInset>
          <main>
            {/* Main content */}
          </main>
        </SidebarInset>
      </div>
    </SidebarProvider>
  )
}
```

The system supports three variants (sidebar, floating, inset), three collapsible modes (offcanvas, icon, none), and includes automatic mobile detection with sheet-based mobile navigation. Keyboard shortcut (Cmd/Ctrl + B) toggles the sidebar globally.
