A reusable React component that creates animated skeleton loading placeholders to display while content is being fetched or processed.

## Key Components

- **Skeleton**: A functional component that renders a pulsing placeholder div with customizable styling and standard HTML div attributes support.

## Usage Example

```typescript
import { Skeleton } from "./skeleton"

// Basic skeleton
<Skeleton className="h-4 w-[250px]" />

// Card skeleton layout
<div className="flex items-center space-x-4">
  <Skeleton className="h-12 w-12 rounded-full" />
  <div className="space-y-2">
    <Skeleton className="h-4 w-[250px]" />
    <Skeleton className="h-4 w-[200px]" />
  </div>
</div>

// List of skeletons
<div className="space-y-3">
  {Array.from({ length: 3 }).map((_, i) => (
    <Skeleton key={i} className="h-[125px] w-full rounded-xl" />
  ))}
</div>
```

The component uses Tailwind CSS classes for styling and includes a pulse animation, muted background, and rounded corners by default. Additional classes can be passed via the `className` prop to customize dimensions and appearance.
