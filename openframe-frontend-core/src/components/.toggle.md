A reusable toggle component built on Radix UI primitives with customizable styling variants for size and appearance.

## Key Components

- **`Toggle`** - Main toggle component with forwardRef support
- **`toggleVariants`** - CVA variant configuration for styling
- **Variants:**
  - `variant`: `default` (transparent) | `outline` (bordered)
  - `size`: `default` | `sm` | `lg`

## Usage Example

```typescript
import { Toggle } from "@/components/ui/toggle"
import { Bold } from "lucide-react"

// Basic toggle
<Toggle aria-label="Toggle bold">
  <Bold />
</Toggle>

// With variants
<Toggle variant="outline" size="lg" aria-label="Toggle feature">
  Enable notifications
</Toggle>

// Controlled state
const [pressed, setPressed] = useState(false)
<Toggle 
  pressed={pressed} 
  onPressedChange={setPressed}
  variant="outline"
>
  {pressed ? "On" : "Off"}
</Toggle>
```

The component automatically handles pressed states with `data-[state=on]` styling and includes proper accessibility attributes. Icons are automatically sized and styled within the toggle.
