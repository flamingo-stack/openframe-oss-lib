Central barrel export file for the Flamingo/OpenFrame shared UI component library, re-exporting all components, utilities, and types from their respective subdirectories and modules for unified package consumption.

## Key Components

Organized into the following export groups:

| Group | Path | Description |
|-------|------|-------------|
| **Core UI** | `./ui` | Base UI primitives and page containers |
| **Chat** | `./chat` | AI chat interface components (Mingo/Fae) |
| **Navigation** | `./navigation` | App navigation components |
| **Platform** | `./platform` | MSP platform-specific components |
| **Features** | `./features` | Feature-flag-driven components |
| **Shared/Onboarding** | `./shared/onboarding` | Client/technician onboarding flows |
| **Shared/Doc Search** | `./shared/doc-search` | RAG-powered search dropdown (data-room + guide catalog) |
| **Shared/Dev Section** | `./shared/dev-section` | Roadmap, delivery, and DevSectionView chrome |
| **Shared/Legal** | `./shared/legal-document` | Privacy policy, terms of service renderer |
| **Loading** | `./loading` | Skeleton and loading state components |
| **Auth** | `./auth-stub` | Authentication stub components |

Notable named export:

```typescript
export { DetailPageSkeleton, type DetailPageSkeletonProps } from './shared/detail-page-skeleton'
```

## Usage Example

```typescript
// Import any component directly from the shared package
import {
  AnnouncementBar,
  EmptyState,
  MetricValue,
  PricingDisplay,
  VendorIcon,
  DetailPageSkeleton,
  type DetailPageSkeletonProps,
} from '@flamingo/components'

// MSP-specific display components
import { MspDisplay, UserDisplay, VendorDisplayButton } from '@flamingo/components'

// Shared search + onboarding
import { DocSearchBar, OnboardingGuide } from '@flamingo/components'
```

> **Note:** Marked `"use client"` — all exports are React Client Components and must be used within Next.js client boundaries. Server Components should import from individual submodules directly to avoid bundling the full client graph.
