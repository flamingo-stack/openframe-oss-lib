A reusable page header component that renders a title, optional subtitle, entity image, back button, and action controls in a responsive layout.

## Key Components

- **`TitleBlockProps`** — Interface defining all configurable props
- **`TitleBlock`** — Main exported component composing `BackButton`, `EntityImage`, and `PageActions`
- **`variant`** prop — Controls visual style: `plain` (default, transparent) or `card` (bordered card on mobile, collapses to plain on `md+`)
- **`selector`** prop — Desktop-only slot for rendering supplemental controls (e.g. `TabSelector`) alongside actions

## Usage Example

```typescript
import { TitleBlock } from './title-block'

// Basic usage
<TitleBlock
  title="Managed Devices"
  subtitle="42 endpoints"
  backButton={{ label: 'Back', onClick: () => router.back() }}
  actions={[
    { label: 'Add Device', icon: PlusIcon, onClick: handleAdd },
  ]}
  actionsVariant="primary-buttons"
/>

// With entity image and card variant (mobile-friendly)
<TitleBlock
  title="Acme Corp"
  subtitle="client@acme.com"
  image={{ src: '/logos/acme.png', alt: 'Acme Corp' }}
  variant="card"
  menuActions={[
    { items: [{ label: 'Edit', onClick: handleEdit }] },
  ]}
  selector={<TabSelector tabs={tabs} />}
/>
```

## Responsive Behavior

| Breakpoint | Layout |
|---|---|
| Mobile | Row: title left, actions right |
| `md` | Column: stacked, `card` variant shows border/background |
| `lg` | Row: title left, actions right (same as mobile) |
