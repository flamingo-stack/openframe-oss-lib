A reusable page header component that renders a title, optional subtitle, entity image, back button, and action controls in a responsive layout.

## Key Components

**`TitleBlockProps`** — Interface defining all configurable props:
- `title` / `subtitle` — Primary and secondary heading text
- `image` — Entity image with `src` and `alt`
- `backButton` — Back navigation with optional label and click handler
- `actions` / `menuActions` — Action buttons or grouped menu actions passed to `PageActions`
- `actionsVariant` — Controls button rendering style (`icon-buttons`, `primary-buttons`, `menu-primary`)
- `selector` — Desktop-only slot (e.g. `TabSelector`) rendered alongside actions
- `variant` — `plain` (default, transparent) or `card` (bordered/padded on mobile, collapses on `md+`)

**`TitleBlock`** — Main exported component with responsive flex layout that stacks vertically on `md` and returns to horizontal on `lg`.

## Usage Example

```typescript
import { TitleBlock } from '@/components/layout/title-block'

// Basic usage
<TitleBlock
  title="Managed Devices"
  subtitle="All enrolled endpoints"
  backButton={{ label: 'Back', onClick: () => router.back() }}
  actions={[
    { label: 'Add Device', icon: PlusIcon, onClick: handleAdd },
  ]}
  actionsVariant="primary-buttons"
/>

// With entity image and card variant (mobile)
<TitleBlock
  variant="card"
  title="Acme Corp"
  subtitle="Enterprise client"
  image={{ src: '/logos/acme.png', alt: 'Acme Corp' }}
  menuActions={[
    { label: 'Settings', items: [{ label: 'Edit', onClick: handleEdit }] },
  ]}
/>
```
