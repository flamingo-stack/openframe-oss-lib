A standardized React layout component that provides consistent structure and styling for list-based pages throughout the OpenFrame application. It combines a page container with search functionality and error handling to create a unified user experience across data tables and grids.

## Key Components

- **ListPageLayout**: Main component that renders the complete layout structure
- **ListPageLayoutProps**: TypeScript interface defining all available props including title, search, actions, and styling options
- **ListPageContainer**: Underlying container component for page structure
- **SearchBar**: Integrated search functionality with full-width positioning
- **PageError**: Error state component for graceful error handling

## Usage Example

```typescript
import { ListPageLayout } from './list-page-layout'

function DevicesPage() {
  const [searchTerm, setSearchTerm] = useState('')
  const [error, setError] = useState(null)

  return (
    <ListPageLayout
      title="Devices"
      headerActions={
        <div className="flex gap-2">
          <Button onClick={handleRefresh}>Refresh</Button>
          <Button onClick={handleAdd}>Add Device</Button>
        </div>
      }
      searchPlaceholder="Search devices..."
      searchValue={searchTerm}
      onSearch={setSearchTerm}
      error={error}
      padding="md"
      background="card"
    >
      <DeviceTable data={filteredDevices} />
    </ListPageLayout>
  )
}
```

The component automatically handles error states, provides consistent spacing and responsive design, and supports customizable styling through padding and background props.
