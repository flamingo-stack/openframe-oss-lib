A unified React container component that provides both legacy layout functionality and modern page structure patterns with built-in headers, navigation, and responsive design.

## Key Components

- **`PageContainer`** - Main component that auto-detects usage pattern (legacy vs advanced)
- **`LegacyPageContainerProps`** - Interface for backward-compatible layout usage
- **`AdvancedPageContainerProps`** - Interface for modern UI patterns with headers and navigation
- **Convenience exports**: `ListPageContainer`, `DetailPageContainer`, `FormPageContainer`, `ContentPageContainer`

## Usage Example

```typescript
// Legacy usage (layout patterns)
<PageContainer 
  backgroundClassName="bg-gray-100" 
  contentPadding="px-4 py-6"
>
  <h1>Your Content</h1>
</PageContainer>

// Advanced usage (UI patterns)
<PageContainer 
  variant="detail"
  title="User Profile"
  subtitle="Manage user information and settings"
  backButton={{
    label: "Back to Users",
    onClick: () => router.back()
  }}
  headerActions={
    <Button>Edit Profile</Button>
  }
>
  <UserForm />
</PageContainer>

// Convenience components
<DetailPageContainer
  title="Product Details"
  backButton={{ onClick: handleBack }}
>
  <ProductInfo />
</DetailPageContainer>
```

The component automatically detects which interface you're using based on the props provided and renders the appropriate layout structure.
