A React component that provides both legacy and modern page layout patterns with automatic detection and rendering of the appropriate interface.

## Key Components

- **`PageContainer`** - Main unified component supporting both legacy and advanced props
- **`LegacyPageContainerProps`** - Interface for backward-compatible layout version
- **`AdvancedPageContainerProps`** - Interface for modern UI patterns with built-in headers
- **Convenience exports**: `ListPageContainer`, `DetailPageContainer`, `FormPageContainer`, `ContentPageContainer`
- **`isAdvancedProps()`** - Type guard function to detect which interface is being used

## Usage Example

**Legacy Pattern (backward compatible):**
```typescript
<PageContainer 
  backgroundClassName="bg-gray-100"
  contentPadding="px-8 py-6"
  maxWidth="max-w-6xl"
>
  <h1>Your Content</h1>
</PageContainer>
```

**Advanced Pattern (modern UI):**
```typescript
<PageContainer 
  variant="detail"
  title="User Profile"
  subtitle="Manage user information"
  backButton={{ onClick: () => router.back(), label: "Back to Users" }}
  headerActions={<Button>Edit</Button>}
  padding="lg"
  background="card"
>
  <UserForm />
</PageContainer>

// Or use convenience exports
<DetailPageContainer
  title="Dashboard"
  headerActions={<CreateButton />}
>
  <DashboardContent />
</DetailPageContainer>
```

The component automatically detects which interface to use based on the presence of advanced props like `variant`, `title`, or `backButton`.
