A React component that displays a metric value with an accompanying label in a visually distinct format, commonly used for showing statistics or measurements with units.

## Key Components

- **MetricValue** - Main component that renders a value-label pair
- **MetricValueProps** - TypeScript interface defining component props:
  - `value`: The primary metric (string or number)
  - `label`: Descriptive text shown in smaller, secondary text
  - `className`: Optional styling override

## Usage Example

```typescript
import { MetricValue } from './metric-value';

function Dashboard() {
  return (
    <div className="flex gap-4">
      <MetricValue value="30s" label="Generation Time" />
      <MetricValue value={1247} label="Total Users" />
      <MetricValue 
        value="99.9%" 
        label="Uptime" 
        className="text-green-600" 
      />
    </div>
  );
}
```

The component uses a flex layout with the main value displayed prominently and the label shown in smaller, secondary-colored text. The DM Sans font family is applied with responsive spacing and proper text alignment.
