A React component for displaying metrics with a prominent value and accompanying descriptive label in a consistent format.

## Key Components

- **MetricValue**: Main component that renders a metric value with its label
- **MetricValueProps**: Interface defining the component's props including value, label, and optional styling

## Usage Example

```typescript
import { MetricValue } from './metric-value';

function Dashboard() {
  return (
    <div className="flex gap-6">
      <MetricValue 
        value={30} 
        label="Generation Time" 
        className="text-green-600" 
      />
      <MetricValue 
        value="99.9%" 
        label="Uptime" 
      />
      <MetricValue 
        value={1247} 
        label="Active Users" 
      />
    </div>
  );
}
```

The component uses DM Sans font family and displays the value in primary text color while the label appears in smaller, secondary-colored text. Both elements are aligned to the bottom baseline and separated by a small gap. The `cn` utility function allows for conditional class merging with the provided `className` prop.
