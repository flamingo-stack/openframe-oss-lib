This file serves as the main export point for a comprehensive collection of education and academic-themed icons used throughout the Flamingo platform's UI components.

## Key Components

The index exports 47 icon components organized into several educational categories:

- **Books & Reading**: BookIcon, BookOpenIcon, BooksIcon, BookTextIcon, BookAltIcon, BookHeartIcon, NotebookIcon, NotebookAltIcon
- **Bookmarks**: BookmarkIcon, BookmarkCheckIcon, BookmarkPlusIcon, BookmarkMinusIcon, BookmarkOffIcon, BookmarkXmarkIcon, BookmarksIcon, BookBookmarkIcon  
- **Academic Tools**: CalculatorIcon variants, RulerIcon, RulerTriangleIcon, DrawingCompassIcon, MicroscopeIcon, TelescopeIcon, FlaskIcon, FlaskVialIcon, VialIcon
- **Geography & Science**: GlobeIcon variants, GlobeStandIcon, PlanetIcon, AtomIcon
- **Academic Achievement**: GraduationCapIcon, HonorIcon, HonorStarIcon, Point100Icon
- **Geometric Shapes**: ShapeCircleIcon, ShapeSquareIcon, ShapeTriangleIcon, ShapeHexagonIcon, ShapeOctagonIcon, ShapeDiamondSquareIcon, ShapesIcon
- **Miscellaneous**: BackpackIcon, PianoKeyIcon, QuoteLeftIcon, QuoteRightIcon, QuoteSquareIcon, StationeryIcon

## Usage Example

```typescript
import { BookIcon, GraduationCapIcon, CalculatorIcon } from '@flamingo/icons/education';

function CourseCard() {
  return (
    <div className="course-card">
      <BookIcon className="course-icon" />
      <h3>Mathematics 101</h3>
      <div className="course-tools">
        <CalculatorIcon size={16} />
        <GraduationCapIcon size={16} />
      </div>
    </div>
  );
}
```

This centralized export pattern enables clean imports and consistent icon usage across educational interfaces in the OpenFrame platform.
