This file serves as the main entry point for a transportation and logistics icon library, providing centralized exports for all transportation-related icon components.

## Key Components

The module exports 56 icon components organized into several categories:

**Transportation Vehicles:**
- Cars: `CarIcon`, `CarSideIcon`, `CarSideElectricIcon`, `CarTaxiIcon`
- Planes: `PlaneIcon`, `PlaneAltIcon`, `PlaneVerticalIcon`, `PlaneOffIcon`
- Trains: `TrainIcon`, `SpeedTrainIcon`, `TramIcon`, `TrainTunnelIcon`
- Water transport: `FerryIcon`, `SailboatIcon`
- Other vehicles: `BikeIcon`, `MotorbikeIcon`, `BusIcon`, `TruckIcon`, `HelicopterIcon`, `RocketIcon`

**Logistics & Delivery:**
- Parcel management: `Parcel01Icon`, `Parcel02Icon`, `Parcel03Icon`, `ParcelsIcon`
- Parcel states: `Parcel01CheckIcon`, `Parcel01DelayIcon`, `Parcel01FastIcon`, `Parcel01LockIcon`
- Truck variants: `TruckFastIcon`, `TruckDelayIcon`, `TruckMedicalIcon`

**Infrastructure & Accessories:**
- Gauges, pumps, traffic signals, car batteries, helmets, and road barriers

## Usage Example

```typescript
import { 
  CarIcon, 
  TruckFastIcon, 
  Parcel01CheckIcon,
  TrafficSignalIcon 
} from './transportation-icons';

// Use in a React component
function DeliveryStatus() {
  return (
    <div>
      <CarIcon />
      <TruckFastIcon />
      <Parcel01CheckIcon />
      <TrafficSignalIcon />
    </div>
  );
}
```

This barrel export pattern allows for clean imports and better tree-shaking in bundlers while maintaining organized icon categories for transportation and logistics applications.
