Central export module for design and editing tool icons used throughout the Flamingo platform's UI components. This barrel export provides a single import point for all design-related SVG icons.

## Key Components

This module exports 60+ design tool icons organized into several categories:

**Alignment & Layout**
- `AlignBottomIcon`, `AlignLeftIcon`, `AlignRightIcon`, `AlignTopIcon` - Object alignment controls
- `DistributeHorizontalIcon`, `DistributeVerticalIcon` - Object distribution tools
- `GridIcon`, `GridLayoutIcon`, `LayoutIcon` - Layout and grid system icons

**Drawing & Editing Tools**
- `PencilIcon`, `BrushIcon`, `PenEditIcon`, `MarkerIcon` - Drawing implements
- `EraserIcon`, `ScissorsIcon`, `CropIcon` - Editing tools
- `MagicWandIcon`, `DropperIcon`, `FillPourIcon` - Selection and color tools

**Shapes & Vectors**
- `VectorCircleIcon`, `VectorSquareIcon`, `VectorTriangleIcon` - Basic shapes
- `VectorBezierCurveIcon` - Advanced curve tools
- `ShapeCircleDashIcon`, `ShapeSquareDashIcon` - Dashed shape variants

**Layers & Organization**
- `LayersIcon`, `LayersPlusIcon`, `LayersMinusIcon` - Layer management
- `ArrangeIcon`, `MoveIcon`, `ReflectHrIcon`, `ReflectVrIcon` - Object manipulation

## Usage Example

```typescript
import { 
  BrushIcon, 
  LayersIcon, 
  VectorCircleIcon,
  AlignLeftIcon 
} from '@flamingo/icons/design';

// Use in a design toolbar component
function DesignToolbar() {
  return (
    <div className="toolbar">
      <button><BrushIcon /></button>
      <button><VectorCircleIcon /></button>
      <button><LayersIcon /></button>
      <button><AlignLeftIcon /></button>
    </div>
  );
}
```
