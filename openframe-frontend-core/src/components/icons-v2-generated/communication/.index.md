A centralized export module that provides access to all communication-related icon components used throughout the Flamingo platform's UI library.

## Key Components

This module serves as the main entry point for:

- **Call Icons** - Complete set of telephony UI elements (call, hangup, missed, inbound/outbound, etc.)
- **Chat Icons** - Real-time messaging interface components with various states and indicators
- **Email Icons** - Email management and status indicators (read, locked, banned, etc.)
- **Message Icons** - General messaging components including bots, quotes, and reactions
- **Inbox Icons** - Email inbox and mailbox interface elements
- **Send Icons** - Multiple variants of send/submit action buttons
- **Miscellaneous** - Additional communication tools like voicemail, story features, and retro telephone

## Usage Example

```typescript
import { 
  CallIcon, 
  CallHangupIcon, 
  ChatTextIcon, 
  EmailIcon,
  Send01Icon 
} from '@flamingo/icons/communication';

// Use in React component
function CommunicationPanel() {
  return (
    <div>
      <CallIcon className="w-6 h-6" />
      <ChatTextIcon className="w-5 h-5 text-blue-500" />
      <EmailIcon className="w-4 h-4" />
      <Send01Icon className="w-6 h-6 text-green-600" />
    </div>
  );
}
```

All icons follow consistent naming conventions and are designed for the Flamingo platform's communication features including VoIP, messaging, email management, and collaboration tools.
