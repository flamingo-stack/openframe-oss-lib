A barrel export file that provides convenient access to all home, appliance, and tool-related icon components from a single import location.

## Key Components

This index file exports 68 different icon components organized into several categories:

**Appliances & Electronics:**
- `AirConditionerIcon`, `BlenderIcon`, `DryerIcon`, `MicrowaveIcon`, `RefrigeratorIcon`, `TvRetroIcon`, `VacuumIcon`, `WasherIcon`

**Furniture & Fixtures:**
- `BedIcon`, `BedBunkIcon`, `BedFrontIcon`, `ChairIcon`, `CouchIcon`, `LampIcon`, `MirrorIcon`

**Kitchen & Dining:**
- `ForkKnifeIcon`, `KettleIcon`, `OvenIcon`, `PotIcon`, `SpatulaIcon`, `CleaverIcon`

**Tools & Equipment:**
- `AxeIcon`, `HammerIcon`, `ShovelIcon`, `WrenchIcon`, `FlashlightIcon`, `FireExtinguisherIcon`

**Home Structure:**
- `DoorIcon`, `WindowIcon`, `HouseIcon`, `StairsIcon`, `CurtainIcon`

**Utilities & Safety:**
- `ElectricalPlugIcon`, `LightbulbIcon`, `SirenIcon`, `SolarPanelIcon`

## Usage Example

```typescript
// Import specific icons as needed
import { 
  RefrigeratorIcon, 
  MicrowaveIcon, 
  LightbulbIcon,
  DoorIcon 
} from './icons';

// Use in React component
function KitchenDashboard() {
  return (
    <div>
      <RefrigeratorIcon />
      <MicrowaveIcon />
      <LightbulbIcon />
      <DoorIcon />
    </div>
  );
}

// Or import all for dynamic usage
import * as Icons from './icons';
const IconComponent = Icons[`${iconName}Icon`];
```
