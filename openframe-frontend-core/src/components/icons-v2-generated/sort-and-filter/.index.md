A barrel export file that consolidates all filter, sort, and toggle icon components into a single import entry point for the OpenFrame UI icon library.

## Key Components

This index file exports 23 icon components organized into three main categories:

**Filter Icons:**
- `Filter01CheckIcon`, `Filter01ClockIcon`, `Filter01Icon`, `Filter01ListIcon`
- `Filter02Icon`, `Filter03HrIcon`, `Filter03VrIcon`  
- `Filter04HrIcon`, `Filter04VrIcon`

**Sort Icons:**
- Numeric sorting: `Sort19DownIcon`, `Sort19UpIcon`, `Sort91DownIcon`, `Sort91UpIcon`
- Alphabetic sorting: `SortAzDownIcon`, `SortAzUpIcon`, `SortZaDownIcon`, `SortZaUpIcon`
- Generic sorting: `SortItemDownIcon`, `SortItemUpIcon`

**Control Icons:**
- Sliders: `SliderRightIcon`, `SlidersIcon`
- Toggles: `ToggleAltRightIcon`, `ToggleRightIcon`, `TogglesIcon`

## Usage Example

```typescript
import { 
  Filter01Icon, 
  SortAzUpIcon, 
  ToggleRightIcon 
} from '@/components/icons/filter-sort';

// Use in a data table component
function DataTableControls() {
  return (
    <div className="flex gap-2">
      <button><Filter01Icon className="w-4 h-4" /></button>
      <button><SortAzUpIcon className="w-4 h-4" /></button>
      <button><ToggleRightIcon className="w-4 h-4" /></button>
    </div>
  );
}

// Import all icons for dynamic usage
import * as FilterSortIcons from '@/components/icons/filter-sort';
const iconName = 'Filter01Icon';
const DynamicIcon = FilterSortIcons[iconName];
```
