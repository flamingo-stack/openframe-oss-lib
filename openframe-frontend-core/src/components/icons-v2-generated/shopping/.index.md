Barrel export file for the **commerce & shopping** icon set, re-exporting all commerce-related icon components from their individual modules as a single unified entry point.

## Key Components

| Export | Description |
|--------|-------------|
| `Bag*Icon` | Shopping bag variants (check, minus, plus, xmark, arrows, etc.) |
| `Basket*Icon` | Shopping basket variants (blank, check, exclamation, arrows, etc.) |
| `Cart*Icon` | Shopping cart variants (check, exclamation, minus, plus, arrows, etc.) |
| `Badge*Icon` | Dollar and percent badge icons |
| `Tag*Icon` | Price/label tag variants (dollar, percent, tags) |
| `Receipt*Icon` | Receipt document variants |
| `Voucher*Icon` | Voucher/coupon variants (dollar, percent) |
| `Megaphone*Icon` | Promotional megaphone variants |
| `Scale*Icon` | Balanced/unbalanced scale icons |
| `Certified*Icon` | Certification badge variants |
| Misc | `CrownIcon`, `StarIcon`, `GiftIcon`, `GiftcardIcon`, `TicketIcon`, `DiamondGemIcon`, `FeedbackIcon`, `ShopIcon`, `ShopLocationIcon`, `MonitorChartIcon`, `CodeBarIcon`, `CodeQrIcon`, etc. |

## Usage Example

```typescript
import {
  CartPlusIcon,
  TagPercentIcon,
  BadgeDollarIcon,
  Receipt01Icon,
  GiftIcon,
} from '@/components/icons/commerce';

export function ProductCard() {
  return (
    <div>
      <CartPlusIcon />
      <TagPercentIcon />
      <BadgeDollarIcon />
      <Receipt01Icon />
      <GiftIcon />
    </div>
  );
}
```
