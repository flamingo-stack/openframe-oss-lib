This file serves as the main entry point for exporting business and commerce-related icon components from an icon library.

## Key Components

The index file exports 58 icon components organized into several categories:

- **Badge Icons**: `BadgeDollarIcon`, `BadgePercentIcon`
- **Bag/Shopping Icons**: `BagIcon`, `BagCheckIcon`, `BagPlusIcon`, `BagMinusIcon`, etc.
- **Basket Icons**: `BasketIcon`, `BasketArrowDownIcon`, `BasketCheckIcon`, etc.
- **Cart Icons**: `CartIcon`, `CartPlusIcon`, `CartCheckIcon`, etc.
- **Code/Scanning Icons**: `CodeBarIcon`, `CodeQrIcon`, `CodeScanIcon`, etc.
- **Business Icons**: `CrownIcon`, `DiamondGemIcon`, `CertifiedIcon`, etc.
- **Commerce Icons**: `TagIcon`, `TicketIcon`, `VoucherIcon`, `GiftIcon`, etc.
- **Feedback Icons**: `ThumbsUpIcon`, `ThumbsDownIcon`, `StarIcon`, etc.

## Usage Example

```typescript
import { 
  CartPlusIcon, 
  TagDollarIcon, 
  StarIcon,
  BagCheckIcon 
} from './path/to/icons';

// Use in a React component
function ShoppingInterface() {
  return (
    <div>
      <CartPlusIcon className="w-6 h-6" />
      <TagDollarIcon size={24} />
      <StarIcon color="#ffd700" />
      <BagCheckIcon />
    </div>
  );
}
```

This barrel export pattern simplifies imports and provides a clean API for consuming business-focused icon components throughout the application.
