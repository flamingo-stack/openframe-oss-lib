A React SVG icon component that renders the Elestio cloud hosting platform logo with customizable size, color, and styling properties.

## Key Components

- **ElestioLogoIconProps** - TypeScript interface extending SVG props with custom options for className, size, and color
- **ElestioLogoIcon** - Main functional component that renders the Elestio logo as an inline SVG with gradient fills and masking effects

## Usage Example

```typescript
import { ElestioLogoIcon } from './elestio-logo-icon';

// Basic usage with default 24px size
<ElestioLogoIcon />

// Custom size and styling
<ElestioLogoIcon 
  size={32} 
  className="hover:opacity-80" 
  color="#A91A1F" 
/>

// With additional SVG props
<ElestioLogoIcon 
  size={48}
  onClick={handleClick}
  aria-label="Elestio Platform"
  role="img"
/>
```

The component features a complex multi-colored gradient design representing the Elestio "e" logo, with default dimensions of 24x24px and inherits the current text color unless specified otherwise.
