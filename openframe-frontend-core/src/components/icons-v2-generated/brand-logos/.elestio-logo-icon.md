A React component that renders the Elestio logo as an SVG icon with customizable size, color, and styling properties.

## Key Components

- **ElestioLogoIconProps**: TypeScript interface extending SVG props with custom properties for className, size, and color
- **ElestioLogoIcon**: Functional component that renders the Elestio logo SVG with gradient fills and masking effects
- **SVG Structure**: Complex path definitions with linear gradients creating the distinctive Elestio "e" logo design

## Usage Example

```typescript
import { ElestioLogoIcon } from './elestio-logo-icon';

// Basic usage with default props (24px, gray)
<ElestioLogoIcon />

// Custom size and color
<ElestioLogoIcon size={32} color="#FF0000" />

// With additional styling
<ElestioLogoIcon 
  className="my-custom-class" 
  size={48}
  onClick={handleClick}
  style={{ cursor: 'pointer' }}
/>

// All SVG props are supported
<ElestioLogoIcon 
  size={64}
  aria-label="Elestio Logo"
  role="img"
/>
```

The component uses a masked SVG approach with multiple gradient-filled paths to create the colorful Elestio logo. The size prop controls both width and height, maintaining the square aspect ratio, while the color prop affects the overall fill (though the internal gradients may override this for the actual logo design).
