React SVG icon component for the MeshCentral logo, rendering a 24×24 PNG-based logo embedded as a base64 pattern within an SVG container.

## Key Components

### `MeshcentralLogoIconProps`
Interface extending `SVGProps<SVGSVGElement>` (omitting `width`/`height`) with:
- `className` — optional CSS class string (default: `""`)
- `size` — icon dimensions in pixels (default: `24`)
- `color` — fill color token (default: `"currentColor"`)

### `MeshcentralLogoIcon`
Functional component that renders the MeshCentral logo as an SVG element using an embedded base64 PNG image via an SVG `<pattern>` definition.

## Usage Example

```typescript
import { MeshcentralLogoIcon } from "./meshcentral-logo-icon";

// Default usage
<MeshcentralLogoIcon />

// Custom size
<MeshcentralLogoIcon size={48} />

// With Tailwind class
<MeshcentralLogoIcon size={32} className="opacity-80 hover:opacity-100" />

// As part of a labeled integration card
<div className="flex items-center gap-2">
  <MeshcentralLogoIcon size={20} />
  <span>MeshCentral</span>
</div>
```

> **Note:** The logo image is fully self-contained as a base64-encoded PNG embedded in the SVG `<defs>` block — no external image assets required. The `color` prop is accepted for API consistency but does not affect the rendered PNG logo.
