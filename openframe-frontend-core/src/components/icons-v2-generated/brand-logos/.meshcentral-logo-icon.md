A reusable React icon component for the MeshCentral logo that provides customizable size, color, and styling options.

## Key Components

- **MeshcentralLogoIconProps**: Interface extending SVG props with custom properties for className, size, and color
- **MeshcentralLogoIcon**: Functional component that renders an SVG icon with embedded base64 image data

## Usage Example

```typescript
import { MeshcentralLogoIcon } from './meshcentral-logo-icon';

// Basic usage with default props (24px, gray)
<MeshcentralLogoIcon />

// Custom size and color
<MeshcentralLogoIcon 
  size={32} 
  color="#FF6B35" 
  className="my-logo-class" 
/>

// With additional SVG props
<MeshcentralLogoIcon 
  size={48}
  onClick={handleClick}
  title="MeshCentral Logo"
  role="img"
/>
```

The component uses a base64-encoded PNG image as the icon source and applies it through an SVG pattern fill. Default size is 24px with a gray (#888888) color, and it accepts all standard SVG element props while providing convenient shortcuts for common customizations.
