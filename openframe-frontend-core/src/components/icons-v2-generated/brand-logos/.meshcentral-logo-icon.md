React SVG icon component that renders the MeshCentral logo using an embedded base64 PNG image pattern.

## Key Components

- **MeshcentralLogoIconProps**: TypeScript interface extending SVG props with custom properties for size, color, and className
- **MeshcentralLogoIcon**: Functional React component that renders a 24x24 SVG containing the MeshCentral logo as a pattern fill
- **SVG Pattern Definition**: Uses a base64-encoded PNG image (216x216) as a pattern fill for the logo display

## Usage Example

```typescript
import { MeshcentralLogoIcon } from './meshcentral-logo-icon';

// Basic usage with default 24px size
<MeshcentralLogoIcon />

// Custom size and styling
<MeshcentralLogoIcon 
  size={32} 
  className="logo-icon" 
  color="currentColor" 
/>

// With additional SVG props
<MeshcentralLogoIcon 
  size={48}
  onClick={handleClick}
  style={{ cursor: 'pointer' }}
/>
```

The component creates a rounded rectangle (4px border radius) filled with the MeshCentral logo pattern. The embedded base64 PNG data contains the actual logo graphics, scaled appropriately within the SVG viewBox. All standard SVG properties are supported through prop spreading, while size, color, and className provide convenient customization options.
