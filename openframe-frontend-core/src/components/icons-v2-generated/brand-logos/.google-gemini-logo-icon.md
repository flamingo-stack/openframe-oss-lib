A React component that renders the Google Gemini logo as an SVG icon with customizable size, color, and styling properties.

## Key Components

- **GoogleGeminiLogoIconProps**: Interface extending SVGProps with custom properties for className, size, and color
- **GoogleGeminiLogoIcon**: Main component function that renders the Gemini logo SVG with configurable styling

## Usage Example

```typescript
import { GoogleGeminiLogoIcon } from './google-gemini-logo-icon';

// Basic usage with default 24px size
<GoogleGeminiLogoIcon />

// Customized with larger size and specific color
<GoogleGeminiLogoIcon 
  size={48} 
  color="#4285f4" 
  className="my-gemini-icon" 
/>

// Using currentColor for theme compatibility
<GoogleGeminiLogoIcon 
  size={32} 
  color="currentColor" 
  onClick={handleClick}
/>
```

The component features a complex multi-layered SVG with gradient fills and filter effects to accurately represent the Google Gemini logo. It accepts standard SVG properties while providing convenient props for the most common customizations (size, color, className).
