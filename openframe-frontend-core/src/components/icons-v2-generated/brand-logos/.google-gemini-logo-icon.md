A React SVG icon component rendering the Google Gemini logo with a multi-colored gradient star/sparkle design.

## Key Components

- **`GoogleGeminiLogoIconProps`** ‚Äî Interface extending `SVGProps<SVGSVGElement>` (excluding `width`/`height`) with:
  - `className` ‚Äî optional CSS class string (default: `""`)
  - `size` ‚Äî uniform width/height in pixels (default: `24`)
  - `color` ‚Äî fill color token (default: `"currentColor"`)

- **`GoogleGeminiLogoIcon`** ‚Äî Functional component rendering a 24√ó24 viewBox SVG with layered masked paths, Gaussian blur filters, and a linear gradient to replicate the official Gemini branding colors (blue, red, green, yellow).

## Usage Example

```typescript
import { GoogleGeminiLogoIcon } from "./google-gemini-logo-icon";

// Default usage
<GoogleGeminiLogoIcon />

// Custom size
<GoogleGeminiLogoIcon size={32} />

// With Tailwind class
<GoogleGeminiLogoIcon size={20} className="opacity-80" />

// Pass-through SVG props
<GoogleGeminiLogoIcon
  size={48}
  aria-label="Google Gemini"
  role="img"
/>
```
