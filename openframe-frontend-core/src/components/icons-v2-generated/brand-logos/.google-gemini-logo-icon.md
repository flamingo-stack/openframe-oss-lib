A React SVG icon component that renders the Google Gemini logo with customizable size, color, and styling properties.

## Key Components

- **GoogleGeminiLogoIconProps**: TypeScript interface extending SVGProps with custom properties for className, size, and color
- **GoogleGeminiLogoIcon**: Main component function that renders the Gemini logo as an SVG with gradient effects and filters

## Usage Example

```typescript
import { GoogleGeminiLogoIcon } from './google-gemini-logo-icon';

// Basic usage with default props (24px, gray)
<GoogleGeminiLogoIcon />

// Custom size and color
<GoogleGeminiLogoIcon 
  size={48} 
  color="#4893FC" 
  className="gemini-icon" 
/>

// With additional SVG properties
<GoogleGeminiLogoIcon 
  size={32}
  onClick={handleClick}
  style={{ cursor: 'pointer' }}
  aria-label="Google Gemini"
/>
```

The component features a complex multi-colored star/sparkle design with gradient fills, blur effects, and masks to create the distinctive Gemini branding. The SVG is optimized for scalability and includes proper accessibility attributes.
