Barrel export file that re-exports all brand and third-party logo icon components from their individual modules, providing a single import entry point for the icons library.

## Key Components

A flat collection of named icon exports organized by brand/platform category:

| Category | Icons |
|----------|-------|
| **Adobe** | `AdobeAeIcon`, `AdobeAiIcon`, `AdobePdfIcon`, `AdobeIdIcon`, `AdobePrIcon`, `AdobePsIcon`, `AdobeXdIcon` |
| **Apple** | `AppleIcon`, `AppleIncIcon`, `AppleLogoIcon`, `AppStoreIcon` |
| **Google** | `GoogleIcon`, `GoogleLogoIcon`, `GoogleDriveIcon`, `GooglePlayIcon`, `GoogleSheetsIcon`, `GoogleGeminiLogoIcon` |
| **Microsoft** | `MicrosoftLogoIcon`, `MicrosoftLogoGreyIcon`, `Office365LogoIcon`, `WindowsIcon`, `WindowsLogoIcon` |
| **AI Platforms** | `AnthropicLogoIcon`, `GeminiLogoIcon`, `OpenaiLogoIcon` (+ grey variants) |
| **Social** | `FacebookIcon`, `InstagramIcon`, `LinkedinIcon`, `TwitterIcon`, `TwitterXIcon`, `TiktokIcon`, `YoutubeIcon` |
| **MSP/IT Tools** | `TacticalRmmLogoIcon`, `MeshcentralLogoIcon`, `FleetMdmLogoIcon`, `OsqueryLogoIcon`, `PowershellLogoIcon` |
| **Dev Tools** | `GithubIcon`, `FigmaIcon`, `FramerIcon`, `TrelloIcon`, `ClickupLogoIcon` |

> Many brands include both a standard and a `Grey` variant (e.g., `SlackLogoIcon` / `SlackLogoGreyIcon`), useful for active/inactive states.

## Usage Example

```typescript
// Import one or many icons from the single barrel entry point
import {
  GithubIcon,
  SlackLogoIcon,
  SlackLogoGreyIcon,
  TacticalRmmLogoIcon,
  OpenaiLogoIcon,
} from '@/components/icons/brands';

// Use in a React/Vue component
export function IntegrationBadge({ active }: { active: boolean }) {
  const SlackIcon = active ? SlackLogoIcon : SlackLogoGreyIcon;
  return <SlackIcon className="w-6 h-6" />;
}
```
