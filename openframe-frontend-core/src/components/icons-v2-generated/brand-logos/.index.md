This TypeScript module serves as a centralized export hub for a comprehensive collection of brand and platform icon components used throughout the Flamingo platform.

## Key Components

The module exports 65+ icon components organized into several categories:

**Creative Software Icons:**
- Adobe Creative Suite (After Effects, Illustrator, InDesign, Premiere Pro, Photoshop, XD)
- Design platforms (Figma, Framer, Behance, Dribbble)

**Social Media & Communication:**
- Major platforms (Facebook, Instagram, LinkedIn, Twitter/X, TikTok, YouTube)
- Professional networks (Discord, Slack, Messenger)

**Tech Platforms & Operating Systems:**
- Cloud services (Google Drive, Office 365, Azure AD)
- Operating systems (Android, Apple, Linux, Windows)
- Development tools (GitHub variations, Chrome)

**MSP & Security Tools:**
- Fleet MDM, MeshCentral, Tactical RMM, Osquery
- Authentication (Authentik, Azure AD)

**AI & Modern Services:**
- OpenAI, Google Gemini, Anthropic logos
- Various business platforms

## Usage Example

```typescript
import { 
  GoogleLogoIcon, 
  MicrosoftLogoIcon, 
  TacticalRmmLogoIcon 
} from '@/icons';

function IntegrationsList() {
  return (
    <div className="integrations-grid">
      <div className="integration-card">
        <GoogleLogoIcon className="w-8 h-8" />
        <span>Google Workspace</span>
      </div>
      <div className="integration-card">
        <TacticalRmmLogoIcon className="w-8 h-8" />
        <span>Tactical RMM</span>
      </div>
    </div>
  );
}
```

This barrel export pattern provides clean imports and maintains a consistent icon system across the entire application.
