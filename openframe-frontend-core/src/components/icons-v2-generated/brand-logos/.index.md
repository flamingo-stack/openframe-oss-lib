This is a centralized barrel export file that provides easy access to a comprehensive collection of brand and platform icons for the Flamingo/OpenFrame ecosystem.

## Key Components

The file exports **62 icon components** organized into categories:

**Design Tools**: Adobe Creative Suite icons (AE, AI, ID, PR, PS, XD), Figma, Framer, Behance, Dribbble
**Platforms**: Android, Apple, Windows, Linux operating system icons
**Social Media**: Facebook, Instagram, LinkedIn, Twitter/X, TikTok, Pinterest, YouTube variants
**Development**: GitHub variations, Chrome, Google services
**MSP & Security**: Authentik, Azure AD, Fleet MDM, MeshCentral, Osquery, PowerShell, Tactical RMM
**AI Services**: Anthropic, OpenAI, Google Gemini logos
**Business**: Microsoft Office 365, PayPal, Slack, Trello, Google Drive

Each export follows the pattern `{ IconName } from './icon-file'` for consistent naming and tree-shaking optimization.

## Usage Example

```typescript
import { 
  GithubIcon, 
  SlackLogoIcon, 
  MicrosoftLogoIcon,
  TacticalRmmLogoIcon 
} from './index';

// Use in React components
const IntegrationList = () => (
  <div>
    <GithubIcon size={24} />
    <SlackLogoIcon size={24} />
    <MicrosoftLogoIcon size={24} />
    <TacticalRmmLogoIcon size={24} />
  </div>
);

// Or import all icons
import * as BrandIcons from './index';
```

This barrel export enables clean imports and ensures all brand icons are easily discoverable for UI components across the Flamingo platform.
