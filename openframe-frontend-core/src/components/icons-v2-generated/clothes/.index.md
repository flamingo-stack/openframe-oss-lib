Centralized export module for clothing and accessory icon components. This file provides a single entry point for importing various fashion and wearable item icons used throughout the application.

## Key Components

This module exports 24 distinct icon components organized into categories:

**Clothing Items:**
- `TShirtIcon`, `TankTopIcon`, `TankTopNumberIcon` - Shirt variations
- `PantsIcon`, `UnderpantsIcon` - Bottom wear
- `SocksIcon`, `StockingsIcon` - Hosiery

**Footwear:**
- `ShoesIcon`, `ShoesRunIcon` - General footwear
- `HeelsChunkyIcon`, `HeelsHighIcon` - High heel variations
- `SkateBootsIcon` - Specialized boots
- `FootprintsIcon` - Foot traces

**Accessories:**
- `Glasses01Icon`, `Glasses02Icon` - Eyewear variations
- `HatIcon`, `ChefHatIcon` - Headwear
- `TieIcon`, `BowTieIcon` - Neckwear
- `HangerIcon`, `ApronIcon` - Clothing accessories
- `GloveSmittenIcon`, `LifeVestIcon` - Protective wear
- `MustacheIcon` - Facial hair accessory

## Usage Example

```typescript
import { 
  TShirtIcon, 
  ShoesIcon, 
  Glasses01Icon,
  HatIcon 
} from './index';

// Use in a wardrobe component
function WardrobeSelector() {
  return (
    <div className="clothing-options">
      <TShirtIcon className="w-6 h-6" />
      <ShoesIcon className="w-6 h-6" />
      <Glasses01Icon className="w-6 h-6" />
      <HatIcon className="w-6 h-6" />
    </div>
  );
}

// Bulk import for icon libraries
import * as ClothingIcons from './index';
```
