Central export hub for analytics and data visualization icon components, providing a comprehensive collection of chart, graph, and activity icons for the Flamingo platform.

## Key Components

The module exports 35+ icon components organized into several categories:

**Activity Icons:**
- `ActivityIcon`, `ActivityCircleIcon`, `ActivitySquareIcon` - General activity indicators
- `ProgressIcon` - Progress tracking visualization

**Chart Icons:**
- `ChartBarIcon` variants - Horizontal and vertical bar charts (01, 02, 03, Square)
- `ChartDonutIcon`, `ChartPieIcon`, `ChartPyramidIcon` - Circular and hierarchical charts
- `ChartLineIcon` variants - Line charts with trend indicators (Up, Down, Heart, Star)
- `ChartRadarIcon`, `ChartScatterIcon` - Specialized chart types

**Graph Icons:**
- `GraphBarIcon`, `GraphMixedIcon`, `GraphMixSquareIcon` - Graph representations
- `GraphLineIcon` variants - Trending line graphs
- `GraphCurveArrowIcon` - Curved flow indicators

**Presentation & Analysis:**
- `PresentationIcon`, `PresentationBarIcon`, `PresentationLineIcon` - Presentation tools
- `AnalysisIcon` - Data analysis representation
- `NetworkIcon`, `VennDiagramIcon` - Network and relationship diagrams
- `Hierarchy01Icon`, `Hierarchy02Icon` - Organizational structures

**Trend Indicators:**
- `TrendUpIcon`, `TrendDownIcon` - Directional trend arrows

## Usage Example

```typescript
import { 
  ChartBarVerIcon, 
  ActivityCircleIcon, 
  TrendUpIcon,
  PresentationIcon 
} from './analytics-icons';

// In a React component
function DashboardCard() {
  return (
    <div className="dashboard-card">
      <ChartBarVerIcon className="chart-icon" />
      <ActivityCircleIcon className="status-icon" />
      <TrendUpIcon className="trend-positive" />
      <PresentationIcon className="action-icon" />
    </div>
  );
}

// Import all icons at once
import * as AnalyticsIcons from './analytics-icons';
const iconName = 'ChartPieIcon';
const IconComponent = AnalyticsIcons[iconName];
```
