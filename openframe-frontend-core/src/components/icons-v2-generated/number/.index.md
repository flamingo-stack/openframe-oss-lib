Central export module for number, age restriction, and dice icon components. This barrel file provides a single import point for all numeric and gaming-related icons in the icon library.

## Key Components

**Age Restriction Icons:**
- Age limit icons from 0-21 years (`AgeLimit0Icon` through `AgeLimit21Icon`)
- Commonly used for content rating and access control

**Numeric Icons:**
- Basic number icons 0-9 (`Number0Icon` through `Number9Icon`)
- Circle-styled numbers (`Numer0CircleIcon` through `Numer9CircleIcon`)
- Square-styled numbers (`Numer0SquareIcon` through `Numer9SquareIcon`)

**Gaming Icons:**
- Standard six-sided dice faces (`Dice1Icon` through `Dice6Icon`)
- Perfect for gaming applications and random number displays

## Usage Example

```typescript
import { 
  AgeLimit18Icon, 
  Number5Icon, 
  Dice3Icon,
  Numer7CircleIcon 
} from './index';

// Age restriction display
<AgeLimit18Icon className="text-red-500" />

// Basic numbering
<Number5Icon className="w-6 h-6" />

// Gaming interface
<Dice3Icon className="animate-bounce" />

// Styled numeric indicator
<Numer7CircleIcon className="bg-blue-500 text-white rounded-full" />
```

This module consolidates all numeric and gaming icon exports, making it easy to import specific icons or entire categories for consistent UI elements across the Flamingo platform.
