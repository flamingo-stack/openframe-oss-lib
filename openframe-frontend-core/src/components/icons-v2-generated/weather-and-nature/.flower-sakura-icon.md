A React component that renders a decorative sakura (cherry blossom) flower icon as an SVG, commonly used in UI designs for spring themes or Japanese-inspired interfaces.

## Key Components

- **FlowerSakuraIconProps** - TypeScript interface extending SVG props with custom styling options
- **FlowerSakuraIcon** - Main functional component that renders the sakura flower SVG

The component accepts standard SVG properties while providing convenient props for size, color, and CSS class customization.

## Usage Example

```typescript
import { FlowerSakuraIcon } from './flower-sakura-icon';

// Basic usage with defaults (24px, gray)
<FlowerSakuraIcon />

// Custom size and color
<FlowerSakuraIcon 
  size={32} 
  color="#ff69b4" 
  className="animate-pulse" 
/>

// With additional SVG properties
<FlowerSakuraIcon 
  size={48}
  color="#e91e63"
  onClick={handleClick}
  style={{ cursor: 'pointer' }}
  aria-label="Cherry blossom decoration"
/>
```

The icon features a detailed sakura flower design with petals and center detail, perfect for decorative elements, navigation icons, or seasonal UI themes. All props except `width` and `height` are passed through to the underlying SVG element.
