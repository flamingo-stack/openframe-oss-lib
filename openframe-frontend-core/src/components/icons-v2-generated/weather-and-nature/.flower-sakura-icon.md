A customizable React SVG component that renders a decorative sakura (cherry blossom) flower icon. The component provides flexible styling options including custom size, color, and CSS classes while maintaining proper TypeScript types.

## Key Components

- **FlowerSakuraIconProps**: TypeScript interface extending SVG props with custom options
  - `className?: string` - Optional CSS class for styling
  - `size?: number` - Icon dimensions (default: 24px)
  - `color?: string` - Fill color (default: "currentColor")
- **FlowerSakuraIcon**: React functional component that renders the SVG icon
- **SVG Path Data**: Two path elements creating the sakura flower design

## Usage Example

```typescript
import { FlowerSakuraIcon } from './flower-sakura-icon';

// Basic usage with defaults
<FlowerSakuraIcon />

// Custom size and color
<FlowerSakuraIcon 
  size={32} 
  color="#ff6b9d" 
  className="sakura-icon" 
/>

// Using currentColor for theme integration
<FlowerSakuraIcon 
  size={48} 
  className="text-pink-500 hover:text-pink-600" 
/>

// With additional SVG props
<FlowerSakuraIcon 
  size={20}
  onClick={handleClick}
  role="button"
  aria-label="Sakura decoration"
/>
```
