A TypeScript barrel file that exports a comprehensive collection of nature, weather, and environmental icons for the Flamingo UI component library.

## Key Components

This index file provides centralized access to 56 icon components organized into several thematic categories:

**Weather Icons**: Cloud variations, precipitation (rain, snow, drizzle), temperature indicators, wind, and atmospheric conditions
**Celestial Icons**: Sun, moon, stars, and sunrise/sunset variations  
**Nature Icons**: Trees, leaves, flowers, seedlings, and plant life
**Water Icons**: Droplets, waves, and water-related elements
**Natural Disasters**: Fire, flood, tornado, hurricane, avalanche, and explosion icons
**Animals**: Dog and paw-related icons
**Decorative**: Sparkles, radiation symbols, and UFO icons

## Usage Example

```typescript
import { 
  CloudWeatherRainIcon, 
  SunIcon, 
  TreeIcon,
  FlowerSakuraIcon 
} from '@flamingo/icons/nature';

// Use in React component
function WeatherDisplay() {
  return (
    <div>
      <CloudWeatherRainIcon className="weather-icon" />
      <SunIcon size={24} color="yellow" />
      <TreeIcon aria-label="Forest area" />
      <FlowerSakuraIcon className="seasonal-decoration" />
    </div>
  );
}

// Import all icons for dynamic usage
import * as NatureIcons from '@flamingo/icons/nature';
const iconName = 'FireIcon';
const DynamicIcon = NatureIcons[iconName];
```

This barrel export pattern enables efficient tree-shaking while providing convenient access to the entire nature-themed icon collection.
