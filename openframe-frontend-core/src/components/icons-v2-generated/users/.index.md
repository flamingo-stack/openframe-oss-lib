This file serves as a centralized export hub for user and face-related icons, providing a single import point for UI components across the Flamingo platform.

## Key Components

The file exports 34 icon components organized into three main categories:

**User Management Icons:**
- `UserIcon`, `UsersIcon`, `UsersGroupIcon` - Basic user representations
- `UserPlusIcon`, `UserMinusIcon`, `UsersPlusIcon`, `UserXmarkIcon` - User actions
- `UserCheckIcon`, `UserBannedIcon`, `UserLockIcon` - User status indicators
- `UserEditIcon`, `UserSearchIcon`, `UserSettingsIcon` - User operations

**Profile & Identity Icons:**
- `AvatarCircleIcon`, `AvatarSquareIcon` - Avatar containers
- `AddressBookIcon`, `BookUserIcon`, `IdCardIcon` - Identity management
- `HouseUserIcon`, `HeadHeartIcon`, `HeadSettingsIcon` - User relationships

**Facial Expression Icons:**
- `FaceSmile01Icon`, `FaceSmile02Icon`, `FaceLaughIcon` - Positive emotions
- `FaceSad01Icon`, `FaceSad02Icon`, `FaceAngryIcon` - Negative emotions
- `FaceMeh01Icon`, `FaceMeh02Icon`, `FaceWorryIcon` - Neutral emotions
- `FaceSurpriseIcon`, `FaceWinkIcon`, `FaceSleepingIcon`, `FaceBabyIcon` - Special expressions
