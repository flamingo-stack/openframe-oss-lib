Barrel export file for the **users & people** icon category, re-exporting all user, avatar, face expression, and contact-related icon components from their individual modules.

## Key Components

| Export | Description |
|---|---|
| `UserIcon`, `UsersIcon`, `UsersGroupIcon` | Core user and group icons |
| `UserCheckIcon`, `UserBannedIcon`, `UserXmarkIcon` | User status/action icons |
| `UserPlusIcon`, `UserMinusIcon`, `UserEditIcon` | User management action icons |
| `UserLockIcon`, `UserSearchIcon`, `UserSettingsIcon` | User permission/utility icons |
| `AvatarCircleIcon`, `AvatarSquareIcon` | Avatar shape variants |
| `FaceSmile01Icon`, `FaceLaughIcon`, `FaceAngryIcon`, etc. | Face expression icons |
| `AddressBookIcon`, `BookUserIcon`, `IdCardIcon` | Contact/identity icons |
| `HeadHeartIcon`, `HeadSettingsIcon` | Head/cognitive icons |
| `HouseUserIcon`, `UsersPlusIcon` | User context icons |

## Usage Example

```typescript
import {
  UserIcon,
  UsersGroupIcon,
  UserCheckIcon,
  FaceSmile01Icon,
  AvatarCircleIcon,
} from './index';

// Use in a React component
const UserProfile = () => (
  <div>
    <AvatarCircleIcon />
    <UserIcon />
    <UserCheckIcon />
    <FaceSmile01Icon />
  </div>
);

// Or import the full set for dynamic rendering
import * as UserIcons from './index';

const icon = UserIcons['UserSettingsIcon'];
```

> All icons in this category share a consistent API. Import only what you need to support tree-shaking and minimize bundle size.
