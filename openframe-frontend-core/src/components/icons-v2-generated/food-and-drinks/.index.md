This file serves as the main export barrel for a comprehensive collection of food and drink-themed icon components, providing a centralized access point for all food-related UI icons.

## Key Components

The file exports 70+ food and beverage icon components organized across multiple categories:

- **Fruits**: `AppleIcon`, `BananaIcon`, `CherriesIcon`, `GrapesIcon`, `KiwiIcon`, `LemonIcon`, `PeachIcon`, `StrawberryIcon`, `WatermelonIcon`
- **Vegetables**: `AvocadoIcon`, `CarrotIcon`, `EggplantIcon`, `GarlicIcon`, `LeafyGreenIcon`, `MushroomIcon`
- **Beverages**: `BeerIcon`, `CoffeeBeanIcon`, `CupBubbleTeaIcon`, `MugHotIcon`, `SodaCanIcon`, `WaterBottleIcon`
- **Baked Goods**: `BagelIcon`, `BreadLongIcon`, `CakeBirthdayIcon`, `CookieIcon`, `CupcakeIcon`, `PieIcon`
- **Meals**: `HamburgerCheeseIcon`, `HotdogIcon`, `PizzaSliceIcon`, `SandwichIcon`, `SteakIcon`
- **Dairy & Proteins**: `CheeseIcon`, `EggIcon`, `FishIcon`, `MeatIcon`, `MilkBottleIcon`, `ShrimpIcon`
- **Sweets**: `CandyIcon`, `CandyCaneIcon`, `HoneyPotIcon`, `IceCreamIcon`
