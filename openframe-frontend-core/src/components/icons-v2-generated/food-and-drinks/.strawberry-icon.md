A customizable React SVG icon component that renders a strawberry icon with configurable size, color, and styling properties.

## Key Components

- **StrawberryIconProps**: Interface extending SVG props with custom properties for className, size, and color
- **StrawberryIcon**: Main functional component that renders the strawberry SVG with default styling and prop spreading

## Usage Example

```typescript
import { StrawberryIcon } from './strawberry-icon';

// Basic usage with defaults (24px, gray)
<StrawberryIcon />

// Custom size and color
<StrawberryIcon size={32} color="#ff6b6b" />

// With custom styling and additional props
<StrawberryIcon 
  className="my-icon" 
  size={48} 
  color="#e74c3c"
  onClick={handleClick}
  aria-label="Strawberry icon"
/>
```

The component provides a clean strawberry icon with seeds and accepts standard SVG properties while offering convenient props for common customizations. The icon maintains its aspect ratio and scales uniformly based on the size prop.
