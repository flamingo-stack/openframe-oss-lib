A React SVG icon component that renders a decorative strawberry icon with customizable size, color, and styling properties.

## Key Components

- **StrawberryIconProps**: TypeScript interface extending SVG props with custom properties for `className`, `size`, and `color`
- **StrawberryIcon**: React functional component that renders an SVG strawberry icon with spots and decorative details

## Usage Example

```typescript
import { StrawberryIcon } from './strawberry-icon';

// Basic usage with default props (24px, currentColor)
<StrawberryIcon />

// Custom size and color
<StrawberryIcon 
  size={32} 
  color="#ff6b6b" 
  className="my-strawberry-icon" 
/>

// With additional SVG props
<StrawberryIcon 
  size={48}
  color="red"
  onClick={() => console.log('Strawberry clicked!')}
  style={{ cursor: 'pointer' }}
/>
```

The component accepts standard SVG element props (excluding `width` and `height` which are controlled by the `size` prop) and renders a 24x24 viewBox SVG with two path elements forming a stylized strawberry shape with decorative spots.
