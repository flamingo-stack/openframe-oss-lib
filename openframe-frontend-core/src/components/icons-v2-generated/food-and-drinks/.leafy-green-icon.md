A React SVG icon component that renders a leafy green plant or nature-themed icon with customizable size, color, and styling properties.

## Key Components

- **LeafyGreenIconProps** - Interface extending SVG props with custom `size`, `color`, and `className` options
- **LeafyGreenIcon** - Main functional component that renders the SVG icon with default styling

## Usage Example

```typescript
import { LeafyGreenIcon } from './leafy-green-icon';

// Basic usage with defaults (24px, currentColor)
<LeafyGreenIcon />

// Custom size and color
<LeafyGreenIcon 
  size={32} 
  color="#22c55e" 
  className="my-icon" 
/>

// With additional SVG props
<LeafyGreenIcon 
  size={48}
  color="var(--primary-color)"
  onClick={handleClick}
  aria-label="Nature icon"
/>
```

The component accepts standard SVG props while providing convenient defaults for common use cases. The `size` prop controls both width and height, `color` defaults to inheriting the current text color, and additional props are spread to the underlying SVG element for maximum flexibility.
