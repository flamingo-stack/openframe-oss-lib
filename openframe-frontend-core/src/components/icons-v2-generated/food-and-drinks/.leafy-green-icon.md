A React SVG icon component rendering a leafy green plant symbol, designed for flexible use across UI contexts.

## Key Components

### `LeafyGreenIconProps`
Extends `SVGProps<SVGSVGElement>` (omitting `width`/`height`) with:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `""` | CSS class for styling |
| `size` | `number` | `24` | Width and height in pixels |
| `color` | `string` | `"currentColor"` | Fill color for the SVG paths |

### `LeafyGreenIcon`
Functional component rendering a 24×24 viewBox SVG with two layered paths forming a leafy green icon. Inherits any additional `SVGSVGElement` props via spread.

## Usage Example

```typescript
import { LeafyGreenIcon } from "./leafy-green-icon";

// Default usage
<LeafyGreenIcon />

// Custom size and color
<LeafyGreenIcon size={32} color="#22c55e" />

// Inherits text color from parent
<span className="text-green-500">
  <LeafyGreenIcon size={16} />
</span>

// With Tailwind class
<LeafyGreenIcon className="opacity-80 hover:opacity-100" />
```
