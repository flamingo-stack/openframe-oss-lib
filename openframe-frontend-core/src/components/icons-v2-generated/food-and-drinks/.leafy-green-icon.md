A customizable SVG icon component that renders a decorative leafy green design, built with TypeScript and React.

## Key Components

- **LeafyGreenIconProps** - Interface extending SVG properties with custom size and color options
- **LeafyGreenIcon** - React functional component that renders the SVG icon with configurable styling

## Usage Example

```typescript
import { LeafyGreenIcon } from './leafy-green-icon';

function MyComponent() {
  return (
    <div>
      {/* Default icon */}
      <LeafyGreenIcon />
      
      {/* Custom size and color */}
      <LeafyGreenIcon 
        size={32} 
        color="#22c55e" 
        className="my-icon"
      />
      
      {/* With additional SVG props */}
      <LeafyGreenIcon 
        size={48}
        color="#059669"
        onClick={() => console.log('Icon clicked')}
        style={{ cursor: 'pointer' }}
      />
    </div>
  );
}
```

The component accepts standard SVG properties while providing convenient props for common customizations. The default size is 24px with a gray color (#888888), and it maintains a 24x24 viewBox for consistent scaling.
