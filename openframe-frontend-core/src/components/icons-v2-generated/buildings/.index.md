Icon module barrel export file that provides centralized access to all building and infrastructure-related icon components in the Flamingo design system.

## Key Components

This module exports 20 building and infrastructure icon components:

**Structural Icons:**
- `ArchwayIcon` - Decorative archway structure
- `Bridge01Icon`, `Bridge02Icon` - Different bridge styles
- `BuildingIcon`, `BuildingsIcon` - Single and multiple buildings
- `FenceIcon` - Barrier/fencing structure

**Specialized Buildings:**
- `CastleIcon`, `FortIcon` - Fortified structures
- `ChurchIcon` - Religious building
- `HospitalIcon`, `SchoolIcon` - Public service buildings
- `HotelIcon`, `HotelAltIcon` - Hospitality venues

**Functional Structures:**
- `GarageIcon`, `WarehouseIcon` - Storage buildings
- `CampgroundIcon`, `ParkIcon` - Recreation areas
- `LighthouseIcon` - Maritime navigation
- `BuildingTreeIcon`, `TerraceIcon` - Landscaped structures

## Usage Example

```typescript
import { 
  BuildingIcon, 
  HospitalIcon, 
  CastleIcon 
} from '@flamingo/icons/buildings';

// Use in React components
export const LocationCard = ({ type }: { type: string }) => {
  const getIcon = () => {
    switch (type) {
      case 'medical': return <HospitalIcon />;
      case 'historic': return <CastleIcon />;
      default: return <BuildingIcon />;
    }
  };

  return (
    <div className="location-card">
      {getIcon()}
      <span>{type}</span>
    </div>
  );
};
```
