A centralized export module for sports and fitness-related icon components, providing a single import point for all athletic and competitive activity icons.

## Key Components

The module exports 29 sports and fitness icon components organized into categories:

**Ball Sports:**
- `AmericanFootballIcon`, `BaseballIcon`, `BasketballIcon`, `FootballIcon`, `PingpongIcon`, `VolleyballIcon`

**Combat & Target Sports:**
- `ArcheryArrowIcon`, `ArcheryBowIcon`, `ArcheryTargetIcon`, `BoxingIcon`

**Individual Activities:**
- `BikingIcon`, `RunningIcon`, `SkippingRopeIcon`, `SwimmingIcon`, `SnorkelIcon`

**Equipment & Facilities:**
- `BasketballBasketIcon`, `BowlingIcon`, `CourtIcon`, `DumbellIcon`, `WhistleIcon`

**Awards & Recognition:**
- `Medal01Icon`, `Medal02Icon`, `TrophyIcon`, `RankingIcon`
- `RewardBadgeIcon`, `RewardBadgeCheckIcon`, `RewardBadgeStarIcon`

**Games:**
- `ChessIcon`

## Usage Example

```typescript
import { 
  BasketballIcon, 
  TrophyIcon, 
  RunningIcon,
  RewardBadgeStarIcon 
} from './sports-icons';

// Use in a sports dashboard
const SportsActivity = () => (
  <div className="activity-grid">
    <div className="sport">
      <BasketballIcon />
      <span>Basketball</span>
    </div>
    <div className="achievement">
      <TrophyIcon />
      <RewardBadgeStarIcon />
    </div>
  </div>
);

// Import all icons for dynamic rendering
import * as SportsIcons from './sports-icons';
```
