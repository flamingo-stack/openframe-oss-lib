Central index file that exports symbol and special character icons for the OpenFrame icon library. This module provides a single entry point for importing various symbolic icons including currency symbols, mathematical operators, status indicators, and special characters.

## Key Components

The module exports 52 symbol icon components organized into categories:

- **Status Icons**: `CheckIcon`, `CheckCircleIcon`, `XmarkIcon`, `ExclamationIcon`, `InfoIcon`, `QuestionIcon`
- **Currency Icons**: `DollarIcon`, `EuroIcon`, `BitcoinIcon`, `PoundIcon` (with circle variants)
- **Mathematical Operators**: `PlusIcon`, `MinusIcon`, `PercentIcon`, `AsteriskIcon`, `HashIcon`
- **Gender/Sign Icons**: `GenderMaleIcon`, `GenderFemaleIcon`, `SignMaleIcon`, `SignFemaleIcon`
- **Temperature Icons**: `DegreeCelsiusIcon`, `DegreeFahrenheitIcon`
- **Special Icons**: `KeyCommandIcon`, `KeyOptionIcon`, `RecycleIcon`, `YinYangIcon`, `SmokingIcon`
- **Navigation Icons**: `TurnLeftSignIcon`, `TurnRightSignIcon`

## Usage Example

```typescript
import { 
  CheckIcon, 
  DollarIcon, 
  PlusCircleIcon, 
  InfoCircleIcon 
} from '@/icons/symbols';

// Use in React components
function StatusIndicator({ success }: { success: boolean }) {
  return success ? <CheckIcon /> : <XmarkIcon />;
}

function PriceDisplay({ amount }: { amount: number }) {
  return (
    <div>
      <DollarIcon />
      {amount}
    </div>
  );
}

function AddButton({ onClick }: { onClick: () => void }) {
  return (
    <button onClick={onClick}>
      <PlusCircleIcon />
      Add Item
    </button>
  );
}
```
