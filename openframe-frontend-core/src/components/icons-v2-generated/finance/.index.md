A centralized export module that provides access to a comprehensive set of financial and payment-related icon components for the Flamingo platform's UI system.

## Key Components

This module exports 33 financial icon components organized by category:

**Banking & ATM Icons:**
- `AtmCardIcon`, `AtmMoneyIcon`, `BankIcon`, `BankInterestIcon`

**Currency & Coins:**
- `CoinIcon`, `CoinInsertIcon`, `CoinsIcon`, `CoinsExchangeIcon`, `CoinsExchangeCurrencyIcon`

**Credit Cards & Payments:**
- `CreditCardIcon`, `CreditCardAltIcon`, `CreditCardEditIcon`, `CreditCardLockIcon`
- `CreditCardPlusIcon`, `CreditCardMinusIcon`, `CreditCardSearchIcon`, `CreditCardShieldIcon`
- `CreditCardArrowUpIcon`, `CreditCardArrowDownIcon`, `CreditCardXmarkIcon`, `CreditCardsExchangeIcon`

**Money & Cash:**
- `MoneyBillIcon`, `MoneyBillDollarIcon`, `MoneyBillWavyIcon`, `MoneyBillWavyDollarIcon`
- `MoneyChequeEditIcon`, `MoneyChequeDollarIcon`, `HandCoinIcon`, `HandDollarIcon`

**Wallets & Storage:**
- `Wallet01Icon`, `Wallet02Icon`, `PiggyBankIcon`, `SackIcon`, `PosMachineIcon`

## Usage Example

```typescript
import { 
  CreditCardIcon, 
  BankIcon, 
  CoinsIcon,
  Wallet01Icon 
} from './financial-icons';

// Use in a payment interface
<div className="payment-options">
  <CreditCardIcon size={24} />
  <BankIcon size={24} />
  <CoinsIcon size={24} />
  <Wallet01Icon size={24} />
</div>
```
