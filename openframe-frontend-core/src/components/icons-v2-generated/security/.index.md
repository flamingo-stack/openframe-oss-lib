This module serves as the main entry point for exporting all security and privacy-related icon components, providing a centralized barrel export pattern for easier consumption.

## Key Components

The module exports 35 security-themed icon components organized into several categories:

**Authentication Icons:**
- `AuditIcon`, `FingerprintIcon`, `PasscodeIcon`, `ScanIcon`, `SignatureIcon`, `SignatureDocumentIcon`

**Lock/Unlock Icons:**
- `LockIcon`, `LockAltIcon`, `LockCheckIcon`, `LockCircleIcon`, `LockKeyholeIcon`, `LockKeyholeOffIcon`, `LockOpenIcon`, `LockOpenKeyholeIcon`, `LockSquareIcon`, `UnlockIcon`, `UnlockKeyholeIcon`

**Key Icons:**
- `KeyIcon`, `KeySkeletonIcon`, `KeyholeCircleIcon`, `KeyholeSquareIcon`

**Shield Protection Icons:**
- `ShieldIcon`, `ShieldAltIcon`, `ShieldCheckIcon`, `ShieldExclamationIcon`, `ShieldKeyholeIcon`, `ShieldLightningIcon`, `ShieldLockIcon`, `ShieldMinusIcon`, `ShieldOffIcon`, `ShieldPlusIcon`, `ShieldUserIcon`, `ShieldWifiIcon`, `ShieldXmarkIcon`

**Access Control Icons:**
- `BannedIcon`

## Usage Example

```typescript
import { 
  LockIcon, 
  ShieldCheckIcon, 
  FingerprintIcon,
  KeyIcon 
} from './security-icons';

// Use individual icons in your components
function SecurityDashboard() {
  return (
    <div>
      <LockIcon className="text-blue-500" />
      <ShieldCheckIcon className="text-green-500" />
      <FingerprintIcon className="text-purple-500" />
      <KeyIcon className="text-yellow-500" />
    </div>
  );
}
```
