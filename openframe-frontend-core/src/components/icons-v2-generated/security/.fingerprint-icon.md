A customizable fingerprint icon component that renders an SVG fingerprint symbol with configurable size, color, and styling properties.

## Key Components

- **FingerprintIconProps**: Interface extending SVG props with custom `size`, `color`, and `className` properties
- **FingerprintIcon**: React functional component that renders a fingerprint SVG icon

## Usage Example

```typescript
import { FingerprintIcon } from './fingerprint-icon';

// Basic usage with defaults (24px, gray)
<FingerprintIcon />

// Custom size and color
<FingerprintIcon 
  size={32} 
  color="#FFC008" 
  className="hover:opacity-80" 
/>

// With additional SVG props
<FingerprintIcon 
  size={48}
  color="#ff0000"
  onClick={() => console.log('Fingerprint clicked')}
  aria-label="Biometric authentication"
/>
```

The component accepts all standard SVG element properties except `width` and `height` (which are controlled by the `size` prop). Default values are 24px size with `#888888` gray color. The fingerprint design uses detailed path elements to create a realistic fingerprint pattern suitable for authentication or security-related UI elements.
