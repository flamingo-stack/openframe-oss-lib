A customizable SVG fingerprint icon component built for React applications, providing flexible styling options and accessibility features.

## Key Components

- **`FingerprintIconProps`** - TypeScript interface extending SVG props with custom options for className, size, and color
- **`FingerprintIcon`** - Main React component that renders a fingerprint SVG with customizable properties

The component accepts all standard SVG props while providing convenient defaults for common customizations like size (24px) and color (currentColor).

## Usage Example

```typescript
import { FingerprintIcon } from './fingerprint-icon';

// Basic usage with defaults
<FingerprintIcon />

// Customized size and color
<FingerprintIcon size={32} color="#3b82f6" />

// With custom styling and props
<FingerprintIcon 
  className="my-fingerprint-icon"
  size={48}
  color="currentColor"
  onClick={handleClick}
  aria-label="Biometric authentication"
/>

// Responsive sizing
<FingerprintIcon 
  size={window.innerWidth > 768 ? 32 : 24}
  className="text-blue-500 hover:text-blue-700"
/>
```

The icon uses `currentColor` by default, making it easy to inherit text color from parent elements and work seamlessly with CSS utility frameworks like Tailwind CSS.
