Barrel export file that re-exports all text-formatting and typography-related icon components from their individual modules, providing a single import entry point for the icon library.

## Key Components

| Export | Description |
|--------|-------------|
| `BlockQuoteIcon` | Blockquote formatting icon |
| `BoldIcon`, `BoldAltIcon` | Bold text variants |
| `ItalicIcon`, `UnderlineIcon`, `StrikeThroughIcon` | Inline text styling icons |
| `HeadingIcon`, `HeadingH1Icon`–`HeadingH6Icon`, `HeadingAltIcon` | Heading level icons |
| `GridDots*Icon` | Grid alignment/positioning icons (blank, bottom, center, left, right, top, outer, HR/VR variants) |
| `ListBulletIcon`, `ListCheckIcon`, `ListNumberIcon`, `ListTreeIcon` | List type icons |
| `TextAlign*Icon` | Text alignment icons (center, justify, left, right) |
| `FontIcon`, `FontAltIcon` | Font selection icons |
| `ParagraphIcon`, `IndentIcon`, `LineHeightIcon` | Paragraph formatting icons |
| `SpacingHrIcon`, `SpacingVrIcon` | Horizontal/vertical spacing icons |
| `ICursorIcon`, `TextAreaDraggerIcon` | Cursor and textarea interaction icons |

## Usage Example

```typescript
import {
  BoldIcon,
  ItalicIcon,
  UnderlineIcon,
  HeadingH1Icon,
  ListBulletIcon,
  TextAlignLeftIcon,
} from '@/components/icons';

export function EditorToolbar() {
  return (
    <div className="toolbar">
      <BoldIcon />
      <ItalicIcon />
      <UnderlineIcon />
      <HeadingH1Icon />
      <ListBulletIcon />
      <TextAlignLeftIcon />
    </div>
  );
}
```
