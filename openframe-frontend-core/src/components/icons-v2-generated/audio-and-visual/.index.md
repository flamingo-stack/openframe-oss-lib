A centralized export module that provides access to a comprehensive collection of media and entertainment-related React icon components for the Flamingo platform.

## Key Components

This index file exports 66 icon components organized into several categories:

- **Camera & Photography**: `CameraIcon`, `CameraOffIcon`, `CameraRetro01Icon`, `ApertureIcon`, `FlashIcon`, etc.
- **Audio & Music**: `HeadphoneIcon`, `MicrophoneStandIcon`, `MusicNote01Icon`, `VolumeUpIcon`, `RadioIcon`, etc.
- **Video & Film**: `MovieIcon`, `ClapperboardIcon`, `VideoRecorderIcon`, `FilmHrIcon`, etc.
- **Media Collections**: `CollectionAlbumIcon`, `CollectionAudioIcon`, `CollectionVideoIcon`, etc.
- **Visual Controls**: `Contrast01Icon`, `ColorPaletteIcon`, `SlidersSquareHrIcon`, etc.
- **Media Standards**: `Standard4kIcon`, `StandardHdIcon`, `StandardVrIcon`, etc.
- **Interactive Elements**: `Hand1FingerIcon`, `PlaylistIcon`, `PhotoFilterIcon`, etc.

## Usage Example

```typescript
import { 
  CameraIcon, 
  VolumeUpIcon, 
  MovieIcon, 
  Standard4kIcon 
} from '@flamingo/icons/media';

// Use in React components
function MediaControls() {
  return (
    <div className="media-toolbar">
      <CameraIcon className="icon-camera" />
      <VolumeUpIcon className="icon-volume" />
      <MovieIcon className="icon-movie" />
      <Standard4kIcon className="icon-quality" />
    </div>
  );
}

// Import all icons
import * as MediaIcons from '@flamingo/icons/media';
```

This barrel export pattern allows developers to easily import any media-related icon component while maintaining clean, organized code structure and tree-shaking capabilities.
