Barrel export file for the **media & audio/visual icon set**, re-exporting all media-related SVG icon components from their individual modules as a single unified entry point.

## Key Components

This file exports **71 media and multimedia icons** grouped into the following categories:

| Category | Icons |
|----------|-------|
| **Camera** | `CameraIcon`, `CameraOffIcon`, `CameraRetro01Icon`, `CameraRetro02Icon`, `ApertureIcon`, `FlashIcon`, `FlashAutoIcon`, `FlashOffIcon` |
| **Video** | `VideoRecorderIcon`, `VideoRecorderAltIcon`, `VideoRecorderOffIcon`, `VideoRecorderPlayIcon`, `ClapperboardIcon`, `ClapperboardPlayIcon`, `MovieIcon`, `FilmHrIcon`, `FilmVrIcon` |
| **Audio** | `HeadphoneIcon`, `HeadsetIcon`, `MicrophoneStandIcon`, `RadioIcon`, `PodcastIcon`, `GramophoneIcon`, `CassetteTapeIcon`, `DiscVinylIcon` |
| **Music** | `MusicNote01Icon`, `MusicNote02Icon`, `PlaylistIcon`, `PlaylistMusicIcon`, `CollectionMusicIcon`, `CollectionAudioIcon` |
| **Image** | `ImageIcon`, `ImageAIIcon`, `ImageOffIcon`, `ImagePlusIcon`, `ImageCheckIcon`, `ImageArrowDownIcon`, `ImageArrowUpIcon`, `ImageXmarkIcon`, `PhotoFilterIcon` |
| **Volume** | `VolumeUpIcon`, `VolumeDownIcon`, `VolumeOffIcon`, `VolumePlusIcon`, `VolumeMinusIcon`, `VolumeXmarkIcon` |
| **Standards** | `Standard3dIcon`, `Standard4kIcon`, `StandardHdIcon`, `StandardSdIcon`, `StandardVrIcon`, `StandardArIcon`, `StandardAdIcon`, `StandardCcIcon` |
| **Collections** | `CollectionAlbumIcon`, `CollectionImageIcon`, `CollectionVideoIcon` |
| **Misc** | `ColorPaletteIcon`, `ColorsIcon`, `Contrast01Icon`, `Contrast02Icon`, `LightningIcon`, `LightningFastIcon`, `CarouselHrIcon`, `CarouselVrIcon` |

## Usage Example

```typescript
// Import individual icons directly from the barrel
import {
  CameraIcon,
  VideoRecorderIcon,
  MusicNote01Icon,
  VolumeUpIcon,
  Standard4kIcon
} from './icons/media';

// Use in a React/Vue component
const MediaControls = () => (
  <div>
    <CameraIcon size={24} />
    <VideoRecorderIcon size={24} />
    <VolumeUpIcon size={24} />
  </div>
);
```
