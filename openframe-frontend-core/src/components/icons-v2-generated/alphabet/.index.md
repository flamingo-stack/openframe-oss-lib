## Key Components

The library exports three variants for each letter A-Z:

- **Basic Icons** (26 components): `AlphabetAIcon`, `AlphabetBIcon`, etc. - Simple letter representations
- **Circle Icons** (26 components): `AlphabetACircleIcon`, `AlphabetBCircleIcon`, etc. - Letters enclosed in circular backgrounds  
- **Square Icons** (26 components): `AlphabetASquareIcon`, `AlphabetBSquareIcon`, etc. - Letters enclosed in square backgrounds
