Barrel export file that re-exports all medical/health-related SVG icon components from their individual modules, providing a single entry point for the medical icon collection.

## Key Components

| Export | Description |
|--------|-------------|
| `BandageIcon`, `BandagesIcon` | Wound care icons |
| `BloodPressureIcon`, `HeartPulseIcon`, `PulseIcon` | Cardiovascular monitoring icons |
| `BrainIcon`, `BrainAIIcon` | Neurology and AI-assisted brain icons |
| `CapsuleIcon`, `CapsuleTabletIcon`, `SyringeIcon` | Medication and treatment icons |
| `HeartIcon`, `HeartBrokenIcon` | Heart condition icons |
| `StethoscopeIcon`, `ThermometerIcon` | Medical instrument icons |
| `Virus01Icon`, `Virus02Icon` | Pathogen/infection icons |
| `WheelchairIcon`, `FaceMaskIcon` | Accessibility and PPE icons |
| `DnaIcon`, `SkullIcon` | Biology and anatomy icons |

## Usage Example

```typescript
// Import specific icons
import { StethoscopeIcon, HeartPulseIcon, BrainAIIcon } from './medical-icons';

// Use in a React component
const PatientDashboard = () => (
  <div>
    <StethoscopeIcon size={24} />
    <HeartPulseIcon size={24} color="red" />
    <BrainAIIcon size={24} />
  </div>
);

// Or import multiple at once for an icon picker
import * as MedicalIcons from './medical-icons';

const iconList = Object.keys(MedicalIcons);
// ['BandageIcon', 'BandagesIcon', 'BloodPressureIcon', ...]
```

> **46 medical icons** are exported from this barrel file, covering anatomy, diagnostics, treatment, equipment, and patient care categories.
