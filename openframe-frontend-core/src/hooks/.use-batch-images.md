A React hook and utility library for efficiently batch fetching authenticated images with automatic caching and deduplication.

## Key Components

**Functions:**
- `configureBatchImageFetch()` - Global configuration setter for tenant URLs and dev mode
- `batchFetchAuthenticatedImages()` - Core function to fetch multiple images with authentication
- `useBatchImages()` - React hook for batch image fetching with caching

**Types:**
- `BatchImageFetchConfig` - Configuration interface for tenant URLs and auth settings

## Usage Example

```typescript
// App initialization
configureBatchImageFetch({
  tenantHostUrl: 'https://tenant.openframe.dev',
  enableDevMode: true
})

// In a React component
function OrganizationList({ organizations }) {
  const imageUrls = useMemo(() => 
    organizations.map(org => org.imageUrl).filter(Boolean),
    [organizations]
  )
  
  const fetchedImages = useBatchImages(imageUrls)
  
  return (
    <div>
      {organizations.map(org => (
        <img 
          key={org.id}
          src={fetchedImages[org.imageUrl] || '/placeholder.jpg'} 
          alt={org.name} 
        />
      ))}
    </div>
  )
}

// Direct API usage
const imageMap = await batchFetchAuthenticatedImages([
  '/api/organizations/123/image',
  '/api/organizations/456/image'
])
```

The hook automatically handles deduplication, caching, and cleanup of blob URLs, making it ideal for displaying multiple authenticated images efficiently.
