Converts nested GraphQL input types into flat URL parameter mappings for cleaner URL state management. Transforms complex object structures like `{ filter: { severity: [...] } }` into simple URL parameters like `?severity=...`.

## Key Components

- **`FlattenedParam`** - Interface defining URL parameter configuration with GraphQL path mapping
- **`flattenQueryVariables()`** - Main function that flattens GraphQL query variables to URL parameter schema
- **`mergeDefaults()`** - Merges default values into the flattened schema
- **`validateSchema()`** - Validates schema for URL parameter name conflicts
- **`getArrayParams()`** - Returns parameter names that should use repeated URL values
- **`getRequiredParams()`** - Returns required parameter names
- **`shouldIncludeInUrl()`** - Determines if a parameter value should be included in the URL
