Provides bidirectional conversion between URL search parameters and GraphQL variables for state management, handling type coercion, nested object paths, and array parameters.

## Key Components

- `urlParamsToVariables()` - Converts URLSearchParams to nested GraphQL variables object
- `variablesToUrlParams()` - Flattens GraphQL variables to URLSearchParams for clean URLs
- `coerceValue()` - Type conversion for URL string values to JavaScript types
- `setNestedValue()` / `getNestedValue()` - Utilities for working with dot-notation object paths
- `mergeVariables()` - Merge parameter updates into existing variables
- `clearParams()` - Remove specific parameters from variables
- `validateVariables()` - Schema validation with error reporting

## Usage Example

```typescript
import { urlParamsToVariables, variablesToUrlParams } from './url-converter'

// URL to Variables
const searchParams = new URLSearchParams('?search=error&severity=critical&severity=warning')
const schema = {
  search: { graphqlPath: 'search', type: 'string' },
  severity: { graphqlPath: 'filter.severity', type: 'array' }
}

const variables = urlParamsToVariables(searchParams, schema)
// Result: { search: 'error', filter: { severity: ['critical', 'warning'] } }

// Variables to URL
const newParams = variablesToUrlParams(variables, schema)
// Result: URLSearchParams with clean parameter encoding

// Merge updates
const updated = mergeVariables(variables, { search: 'new-search' }, schema)
```
