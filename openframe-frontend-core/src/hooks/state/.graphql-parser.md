A utility module that parses GraphQL queries to extract variable definitions and convert them to JavaScript types for URL state management.

## Key Components

- **`extractVariablesFromQuery(query: DocumentNode)`** - Main function that parses a GraphQL DocumentNode and returns variable definitions
- **`VariableDefinition`** - Interface describing extracted variables with name, JS type, required status, and array information
- **`JSType`** - Type union for supported URL parameter types (string, number, boolean, array, object)
- **`isScalarType()`** / **`isInputObjectType()`** - Type checking utilities for GraphQL types
- **`parseGraphQLType()`** - Internal parser for handling nested GraphQL type structures (NonNull, List, Named)
- **`mapGraphQLTypeToJS()`** - Maps GraphQL scalar types to JavaScript equivalents

## Usage Example

```typescript
import { gql } from '@apollo/client'
import { extractVariablesFromQuery } from './graphql-parser'

const SEARCH_QUERY = gql`
  query SearchLogs(
    $search: String!
    $limit: Int
    $filters: LogFilterInput
    $tags: [String!]
  ) {
    logs(search: $search, limit: $limit, filters: $filters, tags: $tags) {
      id
      message
    }
  }
`

const variables = extractVariablesFromQuery(SEARCH_QUERY)
// Returns:
// {
//   search: { name: 'search', type: 'string', required: true, isArray: false },
//   limit: { name: 'limit', type: 'number', required: false, isArray: false },
//   filters: { name: 'filters', type: 'object', required: false, isArray: false },
//   tags: { name: 'tags', type: 'string', required: false, isArray: true }
// }
```
