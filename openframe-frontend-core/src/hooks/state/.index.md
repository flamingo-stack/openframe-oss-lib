A comprehensive TypeScript module providing URL state management hooks with schema-driven parameter handling for both GraphQL and REST APIs. This module offers runtime type validation, automatic URL synchronization, and pagination utilities.

## Key Components

**Main Hooks:**
- `useQueryParams` - GraphQL query parameter management with automatic schema introspection
- `useApiParams` - REST API parameter handling with custom schema definitions
- `useCursorPaginationState` - Cursor-based pagination state management

**Utilities:**
- `createSearchParams` - Helper for creating URL search parameters
- `introspector` / `GraphQLIntrospector` - Runtime GraphQL schema introspection
- Parameter validation, coercion, and transformation utilities

**Types:**
- `ParamSchema`, `VariableDefinition` - Schema definition interfaces
- Hook return types and configuration options

## Usage Example

```typescript
// GraphQL usage
import { useQueryParams } from '@flamingo/ui-kit/hooks'

const { variables, setParam } = useQueryParams(LOGS_QUERY)
const { data } = useQuery(LOGS_QUERY, { variables })

// REST API usage  
import { useApiParams } from '@flamingo/ui-kit/hooks'

const { params, urlSearchParams } = useApiParams({
  search: { type: 'string', default: '' },
  page: { type: 'number', default: 1 }
})

// Cursor pagination
import { useCursorPaginationState } from '@flamingo/ui-kit/hooks'

const { hasNextPage, loadMore, reset } = useCursorPaginationState({
  pageSize: 20
})
```
