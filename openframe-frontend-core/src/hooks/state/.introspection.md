A singleton GraphQL schema introspection manager that fetches schemas at runtime and provides intelligent caching for URL state management. Handles authentication-protected endpoints and maps GraphQL input types to JavaScript types for URL serialization.

## Key Components

- **GraphQLIntrospector**: Main class that manages schema fetching, caching, and type introspection
- **introspector**: Singleton instance for shared use across the application
- **fetchSchema()**: Fetches schema via introspection query with caching support
- **getInputTypeFields()**: Extracts field definitions from GraphQL input object types
- **SchemaCache**: Interface for localStorage cache structure with versioning

## Usage Example

```typescript
import { introspector } from './introspection'

// Initialize with authentication
await introspector.fetchSchema(
  'https://api.example.com/graphql',
  { 'Authorization': 'Bearer token123' }
)

// Extract input type fields for URL state
const filterFields = introspector.getInputTypeFields('LogFilterInput')
// Returns: { severity: { name: 'severity', type: 'array', required: false }, ... }

// Check if schema is ready
if (introspector.isLoaded()) {
  const hasType = introspector.hasType('UserInput')
}

// Clear cache when needed
introspector.clearCache()
```

The introspector automatically caches schemas in localStorage for 24 hours and provides efficient type mapping for URL state serialization.
