A React hook that manages animated hint cycles for quick action interfaces, highlighting different actions in a random sequence to guide user discovery.

## Key Components

- **`useQuickActionHint`** - Main hook that cycles through action indices with configurable timing
- **`UseQuickActionHintOptions`** - Configuration interface for action count, cycle duration, and enabled state
- **`UseQuickActionHintReturn`** - Return type providing active index, stop function, and container ref
- **`shuffleIndices`** - Fisher-Yates shuffle algorithm for randomized, non-repeating hint sequences
- **`advanceHint`** - Core logic that progresses through shuffled action indices
- **`stopHint`** - Manual control to halt hint animations

## Usage Example

```typescript
import { useQuickActionHint } from './use-quick-action-hint'

function QuickActionPanel() {
  const { activeHintIndex, stopHint, containerRef } = useQuickActionHint({
    actionCount: 4,           // Number of quick actions
    cycleDuration: 3000,      // 3 seconds per hint
    enabled: true             // Enable hint system
  })

  const actions = ['Copy', 'Edit', 'Delete', 'Share']

  return (
    <div ref={containerRef} onMouseEnter={stopHint}>
      {actions.map((action, index) => (
        <button
          key={action}
          className={activeHintIndex === index ? 'highlighted' : ''}
          onClick={stopHint}
        >
          {action}
        </button>
      ))}
    </div>
  )
}
```

The hook automatically starts on mount, respects reduced motion preferences, and provides infinite cycling until user interaction stops the hints.
