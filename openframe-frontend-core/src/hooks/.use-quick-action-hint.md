A React hook that manages cycling hint animations for quick action buttons, highlighting each action in a random sequence to guide user discovery.

## Key Components

- **`useQuickActionHint`** - Main hook that cycles through action indices
- **`UseQuickActionHintOptions`** - Configuration interface with actionCount, cycleDuration, and enabled options
- **`UseQuickActionHintReturn`** - Return type with activeHintIndex, stopHint function, and containerRef
- **`shuffleIndices`** - Fisher-Yates shuffle for randomized, non-repeating sequences
- **`stopHint`** - Manual stop function for user interactions
- **`advanceHint`** - Internal function that progresses through the hint sequence

## Usage Example

```typescript
import { useQuickActionHint } from './use-quick-action-hint'

function QuickActionBar() {
  const { activeHintIndex, stopHint } = useQuickActionHint({
    actionCount: 4,
    cycleDuration: 3000,
    enabled: true
  })

  const actions = ['save', 'share', 'copy', 'delete']

  return (
    <div>
      {actions.map((action, index) => (
        <button
          key={action}
          className={activeHintIndex === index ? 'highlighted' : ''}
          onClick={stopHint}
        >
          {action}
        </button>
      ))}
    </div>
  )
}
```

The hook automatically starts on mount, respects reduced motion preferences, and stops when users interact with actions.
