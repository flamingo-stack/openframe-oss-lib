A React hook that manages dynamic theming with customizable colors and automatic dark mode detection based on system preferences.

## Key Components

- **`ThemeConfig`** - Interface defining theme color properties (primary, secondary, accent, background, text)
- **`defaultTheme`** - Default light theme configuration with predefined colors
- **`useDynamicTheming()`** - Main hook that provides theme state management and dark mode detection
- **`updateTheme()`** - Function to partially update theme colors
- **`toggleDark()`** - Function to manually toggle dark mode state

## Usage Example

```typescript
import { useDynamicTheming } from './use-dynamic-theming';

function App() {
  const { theme, isDark, updateTheme, toggleDark } = useDynamicTheming();

  const handleBrandingChange = () => {
    updateTheme({
      primaryColor: '#FF6B6B',
      secondaryColor: '#4ECDC4'
    });
  };

  return (
    <div style={{
      backgroundColor: theme.backgroundColor,
      color: theme.textColor
    }}>
      <button onClick={toggleDark}>
        Toggle {isDark ? 'Light' : 'Dark'} Mode
      </button>
      <button onClick={handleBrandingChange}>
        Update Brand Colors
      </button>
    </div>
  );
}
```

The hook automatically detects system dark mode preferences and provides reactive theme management for consistent UI theming across your application.
