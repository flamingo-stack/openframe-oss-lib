A React hook and utility wrapper for the Sonner toast library that provides a simplified interface for displaying notifications with consistent styling and variants.

## Key Components

- **`ToastOptions`** - Interface defining toast configuration options including title, description, variant types, and duration
- **`toast()`** - Main function that accepts either a string message or options object and displays the appropriate toast variant
- **`useToast()`** - React hook that returns the toast function along with direct access to all Sonner methods

## Usage Example

```typescript
import { useToast } from './use-toast'

function MyComponent() {
  const { toast } = useToast()

  const handleSuccess = () => {
    toast({
      title: 'Success!',
      description: 'Your action was completed.',
      variant: 'success',
      duration: 3000
    })
  }

  const handleError = () => {
    toast({
      title: 'Error',
      description: 'Something went wrong.',
      variant: 'destructive'
    })
  }

  const handleSimple = () => {
    toast('Quick message')
  }

  // Direct sonner access
  const handleCustom = () => {
    toast.promise(fetchData(), {
      loading: 'Loading...',
      success: 'Data loaded!',
      error: 'Failed to load data'
    })
  }
}
```
