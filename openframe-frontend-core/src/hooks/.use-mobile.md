A React hook that detects mobile screen sizes by monitoring viewport width and automatically updates on window resize events.

## Key Components

- **`useMobile()`** - Custom hook that returns a boolean indicating if the current viewport is mobile-sized (< 768px)

## Usage Example

```typescript
import { useMobile } from './use-mobile';

function ResponsiveComponent() {
  const isMobile = useMobile();

  return (
    <div>
      {isMobile ? (
        <MobileNavigation />
      ) : (
        <DesktopNavigation />
      )}
    </div>
  );
}

// Example with conditional styling
function AdaptiveLayout() {
  const isMobile = useMobile();
  
  return (
    <div className={isMobile ? 'mobile-layout' : 'desktop-layout'}>
      <Header compact={isMobile} />
      <Content stackVertical={isMobile} />
    </div>
  );
}
```

The hook uses a 768px breakpoint to determine mobile status and automatically cleans up event listeners to prevent memory leaks. It's ideal for conditional rendering, styling, or behavior based on screen size.
