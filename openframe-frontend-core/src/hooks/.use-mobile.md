A React hook that detects whether the current viewport is considered mobile-sized based on screen width. The hook automatically updates when the window is resized and uses a 768px breakpoint to determine mobile vs desktop views.

## Key Components

- **`useMobile()`** - Custom React hook that returns a boolean indicating if the viewport width is less than 768px
- **State management** - Uses `useState` to track the current mobile status
- **Resize listener** - Automatically updates on window resize events with proper cleanup

## Usage Example

```typescript
import { useMobile } from './use-mobile';

function ResponsiveComponent() {
  const isMobile = useMobile();

  return (
    <div>
      {isMobile ? (
        <MobileNavigation />
      ) : (
        <DesktopNavigation />
      )}
      <main className={isMobile ? 'mobile-layout' : 'desktop-layout'}>
        Content here
      </main>
    </div>
  );
}

// Conditional rendering based on screen size
function App() {
  const isMobile = useMobile();
  
  return (
    <div className="app">
      {isMobile && <MobileHeader />}
      <MainContent />
    </div>
  );
}
```

The hook is particularly useful for conditional rendering of components, applying different CSS classes, or adjusting layouts based on screen size without requiring CSS media queries.
