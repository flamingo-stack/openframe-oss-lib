A React hook for fetching and displaying authenticated images with automatic blob URL management and proper cleanup.

## Key Components

- **`useAuthenticatedImage`** - Main hook that fetches images with authentication
- **`AuthenticatedImageConfig`** - Configuration interface for tenant URLs and dev mode
- **`configureAuthenticatedImage`** - Global configuration function
- **`getImageConfig`** - Internal configuration getter with defaults

## Usage Example

```typescript
import { useAuthenticatedImage, configureAuthenticatedImage } from './use-authenticated-image'

// Configure globally in app initialization
configureAuthenticatedImage({
  tenantHostUrl: process.env.NEXT_PUBLIC_TENANT_HOST_URL || '',
  enableDevMode: process.env.NODE_ENV === 'development'
})

// Use in components
function UserAvatar({ user }: { user: User }) {
  const { imageUrl, isLoading, error } = useAuthenticatedImage(
    user.avatarUrl,
    user.avatarVersion // Force refresh when version changes
  )

  if (isLoading) return <div>Loading...</div>
  if (error) return <div>Error: {error}</div>
  
  return imageUrl ? (
    <img src={imageUrl} alt="User avatar" />
  ) : (
    <div>No image</div>
  )
}
```

The hook handles cookie-based authentication, optional Bearer tokens for development, automatic blob URL cleanup, and cache-busting with refresh keys.
