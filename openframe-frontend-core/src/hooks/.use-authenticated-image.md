A React hook for fetching images that require authentication, supporting both cookie-based and Bearer token authentication with automatic blob URL management.

## Key Components

- **`useAuthenticatedImage()`** - Main hook for fetching authenticated images with loading/error states
- **`configureAuthenticatedImage()`** - Global configuration function for tenant URLs and dev mode settings
- **`AuthenticatedImageConfig`** - TypeScript interface for configuration options
- **Automatic cleanup** - Manages blob URL lifecycle to prevent memory leaks

## Usage Example

```typescript
// Configure once in app initialization
configureAuthenticatedImage({
  tenantHostUrl: 'https://tenant.openframe.dev',
  enableDevMode: true
})

// Use in component
function OrganizationLogo({ organization }) {
  const { imageUrl, isLoading, error } = useAuthenticatedImage(
    organization?.imageUrl,
    organization?.imageVersion // Optional refresh key
  )

  if (isLoading) return <div>Loading...</div>
  if (error) return <div>Error: {error}</div>
  
  return imageUrl ? (
    <img src={imageUrl} alt="Organization logo" />
  ) : (
    <div>No image</div>
  )
}
```

The hook automatically handles URL construction, authentication headers, blob conversion, and cleanup. It supports cache-busting through refresh keys and includes comprehensive error handling.
