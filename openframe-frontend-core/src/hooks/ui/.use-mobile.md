A React hook that detects whether the current viewport is mobile-sized based on a 768px breakpoint. The hook uses `matchMedia` API and window resize events to provide real-time mobile detection with proper cleanup.

## Key Components

- **`useIsMobile()`** - Custom hook that returns a boolean indicating if the viewport is mobile-sized
- **`MOBILE_BREAKPOINT`** - Constant set to 768px defining the mobile/desktop threshold

## Usage Example

```typescript
import { useIsMobile } from './use-mobile'

function ResponsiveComponent() {
  const isMobile = useIsMobile()
  
  return (
    <div>
      {isMobile ? (
        <MobileNavigation />
      ) : (
        <DesktopNavigation />
      )}
    </div>
  )
}

// Conditional styling based on device type
function Layout() {
  const isMobile = useIsMobile()
  
  return (
    <div className={isMobile ? 'mobile-layout' : 'desktop-layout'}>
      <Content />
    </div>
  )
}
```

The hook handles SSR compatibility by initially returning `false` until the client-side measurement is available, and automatically updates when the window is resized across the breakpoint threshold.
