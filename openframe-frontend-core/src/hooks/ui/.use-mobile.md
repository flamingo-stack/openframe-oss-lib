A React hook that detects mobile devices based on screen width using a responsive breakpoint approach.

## Key Components

- **`useIsMobile`** - Custom hook that returns boolean indicating if current viewport is mobile-sized
- **`MOBILE_BREAKPOINT`** - Constant set to 768px defining the mobile/desktop threshold
- **Media Query Listener** - Uses `matchMedia` API for efficient viewport change detection

## Features

- **SSR-Safe**: Initial state is `undefined` to prevent hydration mismatches
- **Performance Optimized**: Uses `matchMedia` instead of resize event listeners
- **Responsive Design**: 768px breakpoint aligns with common CSS frameworks
- **Automatic Cleanup**: Properly removes event listeners on unmount

## Usage Example

```typescript
import { useIsMobile } from './use-mobile'

function ResponsiveComponent() {
  const isMobile = useIsMobile()

  // Handle SSR/initial render
  if (isMobile === undefined) {
    return <div>Loading...</div>
  }

  return (
    <div>
      {isMobile ? (
        <MobileNavigation />
      ) : (
        <DesktopNavigation />
      )}
    </div>
  )
}
```
