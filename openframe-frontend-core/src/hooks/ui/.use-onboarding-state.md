A React hook that manages onboarding flow state with localStorage persistence and cross-tab synchronization. It provides methods to track step completion, skipping, and dismissal while maintaining state consistency across browser tabs.

## Key Components

- **`useOnboardingState(storageKey?)`** - Main hook for managing onboarding state
- **`OnboardingStepConfig`** - Interface defining the structure of onboarding steps
- **`OnboardingState`** - Type representing the current onboarding state
- **State methods**: `markComplete`, `markSkipped`, `dismissOnboarding`, `markMultipleComplete`
- **Query methods**: `isStepComplete`, `isStepSkipped`, `allStepsComplete`

## Usage Example

```typescript
import { useOnboardingState, OnboardingStepConfig } from './use-onboarding-state'

const steps: OnboardingStepConfig[] = [
  {
    id: 'welcome',
    title: 'Welcome',
    description: 'Get started with our app',
    actionIcon: () => <Icon />,
    actionText: 'Continue',
    completedText: 'Completed',
    onAction: () => console.log('Welcome step action')
  }
]

function OnboardingFlow() {
  const { 
    state, 
    markComplete, 
    isStepComplete, 
    allStepsComplete 
  } = useOnboardingState()

  const handleStepComplete = (stepId: string) => {
    markComplete(stepId)
  }

  return (
    <div>
      {!allStepsComplete(steps) && (
        <OnboardingComponent steps={steps} onComplete={handleStepComplete} />
      )}
    </div>
  )
}
```
