A React hook that provides persistent state management using localStorage with cross-tab synchronization and SSR safety.

## Key Components

- **`useLocalStorage<T>`** - Main hook that returns a stateful value and setter function
- **Cross-tab sync** - Automatically syncs state changes across browser tabs using storage events
- **SSR compatibility** - Safe initialization that works with server-side rendering
- **Type safety** - Generic type support for any JSON-serializable data

## Usage Example

```typescript
import { useLocalStorage } from './use-local-storage'

function UserPreferences() {
  const [theme, setTheme] = useLocalStorage('theme', 'light')
  const [settings, setSettings] = useLocalStorage('settings', { 
    notifications: true, 
    language: 'en' 
  })

  return (
    <div>
      <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
        Current theme: {theme}
      </button>
      
      <button onClick={() => setSettings(prev => ({ 
        ...prev, 
        notifications: !prev.notifications 
      }))}>
        Notifications: {settings.notifications ? 'On' : 'Off'}
      </button>
    </div>
  )
}
```

The hook automatically persists changes to localStorage and synchronizes state across all components and browser tabs using the same key.
