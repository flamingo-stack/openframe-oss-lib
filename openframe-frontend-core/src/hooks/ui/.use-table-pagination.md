A React hook that provides unified pagination configuration for both client-side and server-side table pagination, converting input parameters into props compatible with a cursor pagination component.

## Key Components

- **`useTablePagination`** - Main hook that accepts pagination config and returns cursor pagination props
- **`ClientPaginationConfig`** - Interface for client-side pagination with page numbers and totals
- **`ServerPaginationConfig`** - Interface for server-side pagination with cursors and next/previous flags
- **`PaginationConfig`** - Union type combining both pagination approaches

## Usage Example

```typescript
// Client-side pagination
const clientPagination = useTablePagination({
  type: 'client',
  currentPage: 1,
  totalPages: 10,
  itemCount: 20,
  itemName: 'users',
  onNext: () => setPage(p => p + 1),
  onPrevious: () => setPage(p => p - 1),
  showInfo: true
})

// Server-side pagination
const serverPagination = useTablePagination({
  type: 'server',
  hasNextPage: pageInfo.hasNextPage,
  hasLoadedBeyondFirst: true,
  startCursor: pageInfo.startCursor,
  endCursor: pageInfo.endCursor,
  itemCount: items.length,
  itemName: 'items',
  onNext: fetchNextPage,
  onReset: fetchFirstPage
})
```

The hook returns `undefined` when pagination isn't needed (single page for client-side, no page info for server-side).
