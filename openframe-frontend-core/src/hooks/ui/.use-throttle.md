A React hook that throttles value updates to limit how frequently a value can change, useful for performance optimization in scenarios like search inputs or scroll handlers.

## Key Components

- **`useThrottle<T>`** - Generic hook that accepts a value and optional limit (default 200ms), returns the throttled value

## Usage Example

```typescript
import { useThrottle } from './use-throttle'
import { useState } from 'react'

function SearchComponent() {
  const [searchTerm, setSearchTerm] = useState('')
  const throttledSearchTerm = useThrottle(searchTerm, 300)

  // API call only triggers when throttled value changes
  useEffect(() => {
    if (throttledSearchTerm) {
      fetchSearchResults(throttledSearchTerm)
    }
  }, [throttledSearchTerm])

  return (
    <input
      value={searchTerm}
      onChange={(e) => setSearchTerm(e.target.value)}
      placeholder="Search..."
    />
  )
}

// Throttling scroll position for performance
function ScrollComponent() {
  const [scrollY, setScrollY] = useState(0)
  const throttledScrollY = useThrottle(scrollY, 100)

  useEffect(() => {
    const handleScroll = () => setScrollY(window.scrollY)
    window.addEventListener('scroll', handleScroll)
    return () => window.removeEventListener('scroll', handleScroll)
  }, [])

  return <div>Scroll position: {throttledScrollY}</div>
}
```
