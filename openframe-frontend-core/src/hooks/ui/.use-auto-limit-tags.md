Calculates how many tags fit in a single-line container using real DOM measurements, supporting both a fixed numeric limit and an automatic layout-based mode.

## Key Components

**`UseAutoLimitTagsOptions`** — Configuration interface:
- `count` — Total number of tags
- `limitTags` — Fixed number or `"auto"` (default) for DOM-based measurement
- `placeholder` — Text used to reserve input width in auto mode
- `reserveInputWidth` — Whether to reserve trailing space for an autocomplete input (default `true`)

**`UseAutoLimitTagsReturn`** — Returned refs and state:
- `visibleCount` — Number of tags to render visibly
- `middleRef` — Container div holding tags + input (must have `overflow-hidden`, `gap`, `padding`)
- `measureRef` — Off-screen div holding measurement copies of all tags
- `textMeasureRef` — Hidden `<span>` measuring placeholder text width
- `badgeRef` — Ref to the `+N` overflow badge button
- `inputRef` — Ref to the autocomplete input element

**`useAutoLimitTags()`** — Core hook that:
1. Reads real CSS metrics (`padding`, `gap`, `clientWidth`) from the container
2. Measures each tag's `offsetWidth` from the off-screen clone container
3. Greedily fits tags left-to-right, reserving space for the `+N` badge when overflow occurs
4. Re-runs on input changes and container resize via `ResizeObserver`

## Usage Example

```typescript
const { visibleCount, middleRef, measureRef, textMeasureRef, badgeRef, inputRef } =
  useAutoLimitTags({ count: tags.length, placeholder: "Search..." })

// Render container
<div ref={middleRef} className="flex overflow-hidden gap-2 px-3">
  {tags.slice(0, visibleCount).map(tag => <Chip key={tag.id} label={tag.name} />)}
  {visibleCount < tags.length && (
    <button ref={badgeRef}>+{tags.length - visibleCount}</button>
  )}
  <input ref={inputRef} placeholder="Search..." />
</div>

// Off-screen measurement container (hidden)
<div ref={measureRef} className="absolute invisible flex gap-2">
  {tags.map(tag => <Chip key={tag.id} label={tag.name} />)}
</div>
<span ref={textMeasureRef} className="absolute invisible whitespace-pre">Search...</span>
```
