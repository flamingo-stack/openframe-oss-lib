Calculates how many tag chips fit in a single-line container using DOM measurement, supporting both fixed limits and automatic responsive overflow detection.

## Key Components

**`UseAutoLimitTagsOptions`** — Configuration interface:
- `count` — Total number of tags
- `limitTags` — Fixed number or `"auto"` for DOM-based measurement (default: `"auto"`)
- `placeholder` — Text used to reserve input width in auto mode
- `reserveInputWidth` — Whether to reserve space for an autocomplete input (default: `true`)

**`UseAutoLimitTagsReturn`** — Returned refs and state:
- `visibleCount` — Number of tags that fit in the current container width
- `middleRef` — Attach to the tag container (requires `overflow-hidden`, `gap`, `padding`)
- `measureRef` — Attach to an off-screen div holding copies of all tags for width measurement
- `textMeasureRef` — Attach to a hidden `<span>` measuring placeholder text width
- `badgeRef` — Attach to the `+N` overflow badge button
- `inputRef` — Attach to the autocomplete input element

**`useAutoLimitTags()`** — Core hook. Reads real CSS padding and gap from `middleRef`, sums off-screen tag widths, reserves space for the input and `+N` badge, and uses a `ResizeObserver` to recalculate on container resize.

## Usage Example

```typescript
function TagInput({ tags }: { tags: string[] }) {
  const { visibleCount, middleRef, measureRef, textMeasureRef, badgeRef, inputRef } =
    useAutoLimitTags({
      count: tags.length,
      placeholder: "Search...",
      reserveInputWidth: true,
    })

  const hiddenCount = tags.length - visibleCount

  return (
    <>
      {/* Visible tag row */}
      <div ref={middleRef} className="flex overflow-hidden gap-1 px-2">
        {tags.slice(0, visibleCount).map((tag) => (
          <Chip key={tag} label={tag} />
        ))}
        {hiddenCount > 0 && (
          <button ref={badgeRef}>+{hiddenCount}</button>
        )}
        <input ref={inputRef} placeholder="Search..." />
      </div>

      {/* Off-screen measurement container */}
      <div ref={measureRef} className="absolute invisible flex gap-1">
        {tags.map((tag) => <Chip key={tag} label={tag} />)}
      </div>
      <span ref={textMeasureRef} className="absolute invisible whitespace-pre">
        Search...
      </span>
    </>
  )
}
```

> **Note:** The `measureRef` container must be styled identically to the visible tag chips (same font, padding, border) to ensure accurate width measurements.
