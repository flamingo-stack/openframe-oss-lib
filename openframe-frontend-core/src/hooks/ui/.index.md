A barrel file that consolidates and re-exports all UI-related React hooks from the current module, providing a single import entry point for consuming components.

## Key Components

This index file exports the following custom React hooks:

- **use-debounce** - Delays function execution until after a specified wait time
- **use-local-storage** - Manages state persistence in browser's localStorage
- **use-media-query** - Responds to CSS media query changes for responsive design
- **use-memoized-callback** - Optimizes callback function memoization
- **use-onboarding-state** - Manages user onboarding flow state
- **use-table-pagination** - Handles table pagination logic and state
- **use-throttle** - Limits function execution frequency to improve performance
- **use-window-size** - Tracks browser window dimensions and resize events

## Usage Example

```typescript
// Import individual hooks
import { useDebounce, useLocalStorage, useMediaQuery } from './hooks'

// Or import all hooks
import * as UIHooks from './hooks'

function MyComponent() {
  // Use debounced search input
  const debouncedSearch = useDebounce(searchTerm, 300)
  
  // Persist theme preference
  const [theme, setTheme] = useLocalStorage('theme', 'light')
  
  // Responsive design
  const isMobile = useMediaQuery('(max-width: 768px)')
  
  return <div>...</div>
}
```

This barrel export pattern simplifies imports and provides a clean API surface for the UI hooks library.
