A React hook that delays updating a value until a specified time period has passed without changes, commonly used for search inputs and API calls to prevent excessive requests.

## Key Components

- **`useDebounce<T>`** - Main hook function that accepts a generic value and optional delay parameter
- **Parameters:**
  - `value: T` - The value to debounce
  - `delay: number` - Delay in milliseconds (defaults to 500ms)
- **Returns:** The debounced value of type `T`

## Usage Example

```typescript
import { useDebounce } from './use-debounce'
import { useState, useEffect } from 'react'

function SearchComponent() {
  const [searchTerm, setSearchTerm] = useState('')
  const debouncedSearchTerm = useDebounce(searchTerm, 300)

  useEffect(() => {
    if (debouncedSearchTerm) {
      // API call only happens after user stops typing for 300ms
      searchAPI(debouncedSearchTerm)
    }
  }, [debouncedSearchTerm])

  return (
    <input
      value={searchTerm}
      onChange={(e) => setSearchTerm(e.target.value)}
      placeholder="Search..."
    />
  )
}
```

The hook automatically cleans up timers to prevent memory leaks and supports any data type through TypeScript generics.
