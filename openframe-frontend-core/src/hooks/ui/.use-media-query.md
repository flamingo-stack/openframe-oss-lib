A collection of React hooks for responsive design and media query handling with SSR-safe client-side detection.

## Key Components

- **`useMediaQuery(query: string)`** - Hook that returns boolean indicating if a CSS media query matches
- **`useWindowSize()`** - Hook that returns current window dimensions `{width, height}`
- **`useMobile(breakpoint?: string)`** - Hook that returns true if screen is below specified breakpoint
- **`breakpoints`** - Predefined media query strings for common screen sizes (sm, md, lg, xl, 2xl)

## Usage Example

```typescript
import { useMediaQuery, useWindowSize, useMobile, breakpoints } from './use-media-query'

function ResponsiveComponent() {
  const isMobile = useMobile()
  const isTablet = useMediaQuery(breakpoints.md)
  const { width, height } = useWindowSize()
  
  return (
    <div>
      {isMobile && <MobileLayout />}
      {isTablet && !isMobile && <TabletLayout />}
      <p>Window: {width} x {height}</p>
    </div>
  )
}

// Custom breakpoint usage
function CustomBreakpoint() {
  const isWideScreen = useMediaQuery('(min-width: 1600px)')
  return isWideScreen ? <WideLayout /> : <StandardLayout />
}
```

All hooks are SSR-safe and handle client-side hydration properly by returning `false` or `{width: 0, height: 0}` during server rendering.
