A React hooks library for responsive design that provides client-safe media query matching and window dimension tracking with built-in SSR/hydration protection.

## Key Components

- **`useMediaQuery(query)`** - Hook that returns boolean indicating if a CSS media query matches
- **`useWindowSize()`** - Hook that returns current window dimensions (`{width, height}`)
- **`useMobile(breakpoint?)`** - Convenience hook to detect mobile screens using customizable breakpoint
- **`breakpoints`** - Predefined Tailwind CSS-compatible breakpoint constants

## Usage Example

```typescript
import { useMediaQuery, useWindowSize, useMobile, breakpoints } from "./use-media-query"

function ResponsiveComponent() {
  // Check specific media query
  const isDesktop = useMediaQuery("(min-width: 1024px)")
  
  // Get current window dimensions
  const { width, height } = useWindowSize()
  
  // Check if mobile (using default md breakpoint)
  const isMobile = useMobile()
  
  
  return (
    <div>
      <p>Desktop: {isDesktop ? "Yes" : "No"}</p>
      <p>Window: {width} x {height}</p>
      <p>Mobile: {isMobile ? "Yes" : "No"}</p>
    </div>
  )
}
```

The hooks handle client-side hydration safely by returning `false`/`{0,0}` during SSR and updating after mounting, preventing hydration mismatches.
