Implements a hook injection system that allows ui-kit components to access API hooks from consuming projects without creating direct dependencies.

## Key Components

**Type Definitions:**
- `UseVendorsHook` - Type for vendor data fetching hook with filtering and pagination options
- `UseSubcategoryCountHook` - Type for subcategory count fetching hook

**Hook Setters:**
- `setUseVendorsHook()` - Injects vendor hook from main project
- `setUseSubcategoryCountHook()` - Injects subcategory count hook from main project

**Hook Getters:**
- `useVendors()` - Returns injected vendor hook or stub data with error logging
- `useSubcategoryCountByCategory()` - Returns injected subcategory hook or stub data

## Usage Example

```typescript
// In main project - inject real hooks
import { setUseVendorsHook, setUseSubcategoryCountHook } from './api-hooks-stub';
import { useVendorsAPI, useSubcategoryCount } from './api/hooks';

// Setup during app initialization
setUseVendorsHook(useVendorsAPI);
setUseSubcategoryCountHook(useSubcategoryCount);

// In ui-kit components - use injected hooks
import { useVendors, useSubcategoryCountByCategory } from './api-hooks-stub';

function VendorList() {
  const { data, isLoading, error } = useVendors({ 
    pageSize: 10, 
    filtering: ['active'] 
  });
  
  const subcategoryCount = useSubcategoryCountByCategory('electronics');
  
  return <div>{/* component JSX */}</div>;
}
```
