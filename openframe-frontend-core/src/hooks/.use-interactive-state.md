A React hook that manages interactive UI states (hover, focus, press, active) and provides event handlers for common user interactions.

## Key Components

- **`InteractiveState`**: Interface defining the four boolean state properties
- **`useInteractiveState`**: Main hook that returns state values, event handlers, and utility functions
- **State properties**: `isHovered`, `isFocused`, `isPressed`, `isActive`
- **Event handlers**: Pre-configured mouse and keyboard event callbacks
- **Utility functions**: `setActive`, `setLoading`, `setDisabled` for programmatic control

## Usage Example

```typescript
import { useInteractiveState } from './use-interactive-state';

function InteractiveButton({ children }: { children: React.ReactNode }) {
  const {
    isHovered,
    isFocused,
    isPressed,
    handlers,
    setActive,
    ref
  } = useInteractiveState();

  return (
    <button
      ref={ref}
      {...handlers}
      className={`
        ${isHovered ? 'bg-blue-100' : 'bg-white'}
        ${isFocused ? 'ring-2 ring-blue-500' : ''}
        ${isPressed ? 'scale-95' : 'scale-100'}
      `}
      onClick={() => setActive(true)}
    >
      {children}
    </button>
  );
}
```

The hook automatically manages state transitions based on user interactions and provides a clean API for building responsive UI components.
