Font configuration file that sets up and exports Google Fonts (Azeret Mono and DM Sans) for use throughout a Next.js application with optimized loading and CSS variable integration.

## Key Components

- **azeretMono** - Azeret Mono monospace font configured with weights 400-700
- **dmSans** - DM Sans sans-serif font configured with weights 400, 500, 700
- Both fonts include CSS variables for easy styling and swap display for performance

## Usage Example

```typescript
// In your layout or page component
import { azeretMono, dmSans } from './fonts'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en" className={`${azeretMono.variable} ${dmSans.variable}`}>
      <body className="font-sans">
        {children}
      </body>
    </html>
  )
}
```

```css
/* In your CSS/Tailwind config */
.font-mono {
  font-family: var(--font-azeret-mono);
}

.font-sans {
  font-family: var(--font-dm-sans);
}
```

The fonts are optimized with `display: "swap"` for better loading performance and include only Latin character subsets to reduce bundle size.
