Main entry point for the @flamingo/ui-kit package that re-exports all components, hooks, utilities, and types to provide a single import interface for consumers.

## Key Components

- **UI Components**: Basic UI elements from `./components/ui`
- **Feature Components**: Advanced feature components from `./components/features` 
- **Icons**: Icon components from `./components/icons`
- **Navigation**: Navigation-related components
- **Platform Components**: Platform-specific components
- **Chat Components**: Chat interface components
- **Onboarding**: User onboarding flow components
- **Logs Components**: `LogsList`, `LogSeverityDot`, `ToolIcon` with associated types
- **Hooks**: Custom React hooks
- **Utils**: Utility functions and helpers
- **Types**: TypeScript type definitions
- **Assets**: Static assets and resources

## Usage Example

```typescript
// Import specific components
import { LogsList, LogSeverityDot, ToolIcon } from '@flamingo/ui-kit'

// Import UI components (examples)
import { Button, Input, Modal } from '@flamingo/ui-kit'

// Import hooks and utilities
import { useCustomHook, formatData } from '@flamingo/ui-kit'
```
