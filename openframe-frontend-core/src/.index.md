Main export barrel file for the `@flamingo/ui-kit` package, aggregating and re-exporting all components, hooks, utilities, types, and assets from the library.

## Key Components

| Export | Source | Description |
|--------|--------|-------------|
| `ui` | `./components/ui` | Base UI primitives (buttons, inputs, modals, etc.) |
| `features` | `./components/features` | Feature-specific composite components |
| `icons` | `./components/icons` | Icon set components |
| `navigation` | `./components/navigation` | Navigation components (sidebars, menus, breadcrumbs) |
| `platform` | `./components/platform` | Platform-level layout components |
| `chat` | `./components/chat` | Chat/messaging UI components (Mingo AI, Fae) |
| `onboarding` | `./components/shared/onboarding` | Onboarding flow components |
| `LogsList` | `./components/logs-list` | Renders a list of log entries |
| `LogSeverityDot` | `./components/log-severity-dot` | Visual severity indicator dot for logs |
| `ToolIcon` | `./components/tool-icon` | Icon component for tool representations |
| `logs.types` | `./types/logs.types` | TypeScript types for log data structures |
| `hooks` | `./hooks` | Shared React hooks |
| `utils` | `./utils` | Utility/helper functions |
| `types` | `./types` | Global TypeScript type definitions |
| `assets` | `./assets` | Static assets (images, SVGs, fonts) |

## Usage Example

```typescript
// Import UI primitives
import { Button, Input, Modal } from '@flamingo/ui-kit'

// Import chat components for Mingo AI / Fae
import { ChatWindow, ChatMessage } from '@flamingo/ui-kit'

// Import log components
import { LogsList, LogSeverityDot } from '@flamingo/ui-kit'

// Import types
import type { LogEntry } from '@flamingo/ui-kit'

// Import hooks and utilities
import { useTheme, formatDate } from '@flamingo/ui-kit'
```

> This is a barrel file — import directly from `@flamingo/ui-kit` rather than from deep subpaths to ensure consistent access and tree-shaking compatibility.
