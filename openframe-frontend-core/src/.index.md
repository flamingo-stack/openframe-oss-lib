Main entry point for the @flamingo/ui-kit package, providing a centralized export interface for all UI components, utilities, and types.

## Key Components

The package exports are organized into the following categories:

- **UI Components**: Basic building blocks (`./components/ui`)
- **Feature Components**: Complex functionality (`./components/features`) 
- **Icons**: Icon library (`./components/icons`)
- **Navigation**: Navigation components (`./components/navigation`)
- **Platform**: Platform-specific components (`./components/platform`)
- **Chat**: Chat-related components (`./components/chat`)
- **Onboarding**: Shared onboarding flows (`./components/shared/onboarding`)
- **Logs**: Log display components (`LogsList`, `LogSeverityDot`, `ToolIcon`)
- **Utilities**: Helper functions and custom hooks (`./hooks`, `./utils`)
- **Types**: TypeScript definitions (`./types`, `./types/logs.types`)
- **Assets**: Static resources (`./assets`)
