Storybook stories for the `OnboardingGuideCard` component, covering all three display densities used across the Flamingo/OpenFrame UI.

## Key Components

| Export | Description |
|--------|-------------|
| `ChatInlineSm` | Compact `sm` size — used by the chat dispatch with `displayAction: true` for Ask/Display affordance |
| `Default` | Horizontal step-card layout at `default` density |
| `Catalog` | Full hero + author grid layout at `catalog` density |

## Usage Example

```typescript
import { OnboardingGuideCard } from '../components/chat/entity-cards'

// Chat inline (dispatch default)
<OnboardingGuideCard
  guide={onboardingGuide}
  href={`/guides/${onboardingGuide.slug}`}
  size="sm"
  target="_blank"
  rel="noopener noreferrer"
/>

// Catalog hero layout
<OnboardingGuideCard
  guide={onboardingGuide}
  href={`/guides/${onboardingGuide.slug}`}
  size="catalog"
  target="_blank"
  rel="noopener noreferrer"
/>
```

## Notes

- All stories are wrapped in `ChatColumnDecorator` to simulate the chat column width constraint
- The `sm` size mirrors exactly what the chat dispatch renders — always paired with `displayAction: true` in production
- Fixture data (`onboardingGuide`) lives in `./__fixtures__/chat-cards`
