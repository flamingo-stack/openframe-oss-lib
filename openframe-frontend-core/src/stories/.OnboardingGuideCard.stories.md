Storybook stories for the `OnboardingGuideCard` component, covering all three display densities used across the Flamingo/OpenFrame UI.

## Key Components

| Export | Description |
|--------|-------------|
| `ChatInlineSm` | `size="sm"` — compact inline variant used by chat dispatch with `displayAction` affordance |
| `Default` | `size="default"` — horizontal step-card layout |
| `Catalog` | `size="catalog"` — hero + author grid, full-featured catalog display |

## Usage Example

```typescript
import { OnboardingGuideCard } from '../components/chat/entity-cards'

// Chat dispatch usage (sm size, as rendered by Mingo AI)
<OnboardingGuideCard
  guide={onboardingGuide}
  href={`/guides/${onboardingGuide.slug}`}
  size="sm"
  displayAction={true}
  target="_blank"
  rel="noopener noreferrer"
/>

// Catalog page usage
<OnboardingGuideCard
  guide={onboardingGuide}
  href={`/guides/${onboardingGuide.slug}`}
  size="catalog"
  target="_blank"
  rel="noopener noreferrer"
/>
```

## Notes

- All stories are wrapped in `ChatColumnDecorator` to simulate the chat column width context
- Fixture data is sourced from `./__fixtures__/chat-cards` (`onboardingGuide`)
- The `sm` size is the canonical chat-dispatch variant — always paired with `displayAction: true` in production to show the Ask/Display affordance
