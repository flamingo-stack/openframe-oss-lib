Storybook stories for the `RenameChatModal` and `ArchiveChatModal` components, providing interactive previews of the chat header modals triggered from the `⋯` menu and dialog-history row menus.

## Key Components

| Export | Description |
|--------|-------------|
| `Rename` | Story for `RenameChatModal` — pre-seeded input with current chat name, Cancel + Save actions |
| `Archive` | Story for `ArchiveChatModal` — destructive confirmation dialog with a red action button |

**Meta config:**
- `title`: `Chat/MingoChatModals`
- `layout`: `fullscreen` with a `bg-ods-bg` decorator wrapper
- References Figma frames `7592:225962` (Rename) and `7592:226181` (Archive)
- Both modals are built on `ModalV2` (Azeret Mono title, `p-40` / `gap-24`, in-house overlay)

## Usage Example

```typescript
// Rename story — toggle open via button, logs saved name to console
<RenameChatModal
  isOpen={open}
  initialName="Troubleshooting Exchange Online connection issues"
  onClose={() => setOpen(false)}
  onSave={(name) => {
    console.log('save', name)
    setOpen(false)
  }}
/>

// Archive story — destructive confirmation, logs on confirm
<ArchiveChatModal
  isOpen={open}
  onClose={() => setOpen(false)}
  onConfirm={() => {
    console.log('archive confirmed')
    setOpen(false)
  }}
/>
```

Both stories render an **"Open … modal"** trigger button so the modal can be reopened after closing during interactive testing in Storybook.
