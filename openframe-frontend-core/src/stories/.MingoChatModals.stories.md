Storybook stories for the `RenameChatModal` and `ArchiveChatModal` components, providing interactive previews of the chat header modals used in the Mingo AI interface.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Rename` | `Story` | Interactive story for `RenameChatModal` — input pre-seeded with the current chat name, Cancel + Save actions |
| `Archive` | `Story` | Interactive story for `ArchiveChatModal` — destructive confirmation dialog with a red action button |

## Usage Example

```typescript
// Storybook story usage (see MingoChatModals.stories.tsx)
import { RenameChatModal, ArchiveChatModal } from '../components/chat/mingo-chat-modals'

// RenameChatModal
<RenameChatModal
  isOpen={open}
  initialName="Troubleshooting Exchange Online connection issues"
  onClose={() => setOpen(false)}
  onSave={(name) => {
    console.log('save', name)
    setOpen(false)
  }}
/>

// ArchiveChatModal
<ArchiveChatModal
  isOpen={open}
  onClose={() => setOpen(false)}
  onConfirm={() => {
    console.log('archive confirmed')
    setOpen(false)
  }}
/>
```

## Notes

- Both stories default to `isOpen: true` so the modal renders immediately on load in Storybook
- A toggle button is included in each story to reopen the modal after dismissal
- Both modals are built on `ModalV2` (Azeret Mono title, `p-40` / `gap-24`, in-house overlay)
- Figma references: Rename → `7592:225962`, Archive → `7592:226181`
