Storybook configuration file that defines stories for the AppLayout component, showcasing various layout configurations with sidebar navigation, header options, and main content areas.

## Key Components

- **Meta Configuration**: Storybook metadata with fullscreen layout and component documentation
- **Navigation Items**: Pre-configured sidebar navigation with primary and secondary sections
- **Story Variants**: Multiple story examples demonstrating different AppLayout configurations
- **Interactive Examples**: Stories with stateful navigation and user interaction handling

## Usage Example

```typescript
import { AppLayout } from '../components/navigation/app-layout'

// Basic layout with sidebar and header
<AppLayout
  sidebarConfig={{
    items: navigationItems,
    onNavigate: handleNavigate,
    onToggleMinimized: toggleSidebar,
  }}
  headerProps={{
    showNotifications: true,
    showUser: true,
    userName: 'John Doe',
    userEmail: 'john@example.com',
    onProfile: handleProfile,
    onLogout: handleLogout,
  }}
  mobileBurgerMenuProps={{
    user: {
      userName: 'Alex Developer',
      userEmail: 'alex@openframe.dev',
      userAvatarUrl: 'https://github.com/shadcn.png',
      userRole: 'Admin',
    },
  }}
>
  <div className="p-4">
    <h1>Main Content</h1>
  </div>
</AppLayout>
```

The stories demonstrate features like responsive sidebar behavior, search functionality, organization selection, custom loading states, and interactive navigation with proper state management.
