Storybook stories for the `PageContainer` layout component, covering all variants, header configurations, padding, and background options.

## Key Components

- **`meta`** — Storybook metadata registering `PageContainer` under `Layout/PageContainer` with controls for `variant`, `padding`, `background`, and `showHeader`
- **`PlaceholderContent`** — Internal helper component rendering a placeholder card for story layouts
- **Story exports** — Individual named stories demonstrating every supported configuration

## Story Groups

| Group | Stories |
|-------|---------|
| **Legacy** | `Legacy`, `LegacyCustomBackground`, `LegacyContentWidthBackground` |
| **List Variant** | `ListPage`, `ListPageMultipleActions` |
| **Detail Variant** | `DetailPage`, `DetailPageWithActions` |
| **Form Variant** | `FormPage` |
| **Content Variant** | `ContentPage` |
| **Header Variations** | `CustomHeaderContent`, `WithHeaderActions`, `HiddenHeader` |
| **Padding & Background** | `WithPaddingAndBackground` |

## Usage Example

```typescript
// Render a list page with an action button
export const ListPage: Story = {
  args: {
    variant: 'list',
    title: 'Devices',
    subtitle: '24 devices found',
    actions: [
      {
        label: 'Add Device',
        onClick: () => {},
        icon: <Plus className="w-5 h-5" />,
        variant: 'accent',
      },
    ],
    children: <PlaceholderContent />,
  },
};

// Render a detail page with back navigation and destructive action
export const DetailPageWithActions: Story = {
  args: {
    variant: 'detail',
    title: 'Script Details',
    backButton: { onClick: () => {} },
    actions: [
      { label: 'Delete', onClick: () => {}, icon: <Trash2 className="w-5 h-5" />, variant: 'outline' },
      { label: 'Run Script', onClick: () => {}, variant: 'accent' },
    ],
    children: <PlaceholderContent />,
  },
};
```
