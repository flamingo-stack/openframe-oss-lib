Storybook stories configuration for the PageContainer component, showcasing various layouts and usage patterns for different page types in the application.

## Key Components

- **Meta Configuration** - Storybook metadata with interactive controls for variant, padding, background, and header visibility
- **Story Variants** - Comprehensive examples including legacy, list, detail, form, and content page layouts
- **PlaceholderContent** - Reusable component for demonstrating page content areas
- **Interactive Controls** - Argtype definitions for testing different component configurations

The stories demonstrate legacy mode compatibility, advanced variants (list, detail, form, content), header customization options, and various padding/background combinations.
