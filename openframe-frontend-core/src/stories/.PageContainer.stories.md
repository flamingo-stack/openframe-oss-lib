Storybook configuration file that defines interactive stories for the PageContainer component, showcasing various layout variants, styling options, and usage patterns for the Flamingo platform's page layout system.

## Key Components

- **Meta Configuration**: Defines component controls for variant, padding, background, and header visibility
- **Legacy Stories**: Basic container usage with custom backgrounds and content-width options
- **Variant Stories**: Comprehensive examples for list, detail, form, and content page variants
- **Header Variations**: Stories demonstrating custom header content, actions, and visibility controls
- **Layout Options**: Examples showing different padding and background combinations

## Usage Example

```typescript
// Basic legacy usage
<PageContainer className="px-6">
  <div>Page content</div>
</PageContainer>

// Advanced list page variant
<PageContainer
  variant="list"
  title="Devices"
  subtitle="24 devices found"
  actions={[
    {
      label: 'Add Device',
      onClick: () => {},
      icon: <Plus className="w-5 h-5" />,
      variant: 'primary'
    }
  ]}
>
  <div>List content</div>
</PageContainer>

// Detail page with back navigation
<PageContainer
  variant="detail"
  title="Device Details"
  backButton={{ label: 'Back to Devices', onClick: () => {} }}
  actions={[
    { label: 'Delete', onClick: () => {}, variant: 'outline' }
  ]}
>
  <div>Detail content</div>
</PageContainer>
```

The stories demonstrate the PageContainer's flexibility for different page types, from simple legacy layouts to advanced variants with headers, actions, and navigation controls.
