Storybook stories for the `InvestorUpdateCard` entity card component, covering both compact chat-inline (`sm`) and full (`default`) size variants.

## Key Components

- **`ChatInlineSm`** — Compact `sm` size variant intended for inline display within chat messages
- **`Default`** — Full-size card variant for standard display contexts
- **`ChatColumnDecorator`** — Wraps each story in a simulated chat column layout
- **`investorUpdate`** — Shared fixture providing mock investor update data

## Usage Example

```typescript
import { InvestorUpdateCard } from '../components/chat/entity-cards'

// Compact chat-inline variant
<InvestorUpdateCard
  update={investorUpdate}
  href={`/investor-updates/${investorUpdate.slug}`}
  size="sm"
  target="_blank"
  rel="noopener noreferrer"
/>

// Default full-size variant
<InvestorUpdateCard
  update={investorUpdate}
  href={`/investor-updates/${investorUpdate.slug}`}
  size="default"
  target="_blank"
  rel="noopener noreferrer"
/>
```

## Notes

- Period badge (e.g. `"Mar 2025 - May 2025"`) is formatted via `formatInvestorUpdatePeriod` internally
- Both stories link to `/investor-updates/<slug>` and open in a new tab via `target="_blank"`
- Stories are registered under the `Chat/EntityCards/InvestorUpdateCard` Storybook path with `fullscreen` layout
