Storybook stories for the `InvestorUpdateCard` component, covering the `investor_update` doc type in both compact chat-inline (`sm`) and standard (`default`) sizes.

## Key Components

- **`ChatInlineSm`** — Compact card variant (`size: 'sm'`) intended for inline rendering within chat messages
- **`Default`** — Standard-size card variant (`size: 'default'`) for full-width display
- **`ChatColumnDecorator`** — Wraps each story in a chat column layout to simulate real rendering context
- **`investorUpdate`** — Shared fixture providing mock investor update data (slug, period, etc.)

## Usage Example

```typescript
import { InvestorUpdateCard } from '../components/chat/entity-cards'

<InvestorUpdateCard
  update={investorUpdate}
  href={`/investor-updates/${investorUpdate.slug}`}
  size="sm"
  target="_blank"
  rel="noopener noreferrer"
/>
```

## Notes

- The period badge (e.g. `"Mar 2025 - May 2025"`) is composed internally via `formatInvestorUpdatePeriod` — no manual formatting required
- Both stories use `target="_blank"` with `rel="noopener noreferrer"` to reflect typical chat link behavior
- The `fullscreen` layout parameter ensures the decorator column renders without Storybook padding interference
