A comprehensive Storybook file that showcases the SelectButton component with various configurations and states for testing and documentation purposes.

## Key Components

- **Meta Configuration**: Defines the component's Storybook metadata with title, parameters, and control types
- **Static Stories**: Individual story variants demonstrating different SelectButton states:
  - `Default`, `Selected` - Basic button states
  - `WithIcon`, `WithImage` - Visual elements with icons and images
  - `WithTag` - Additional tag/badge display
  - `NoDescription`, `Disabled` - Edge cases and accessibility states
- **Interactive Stories**: Live examples with state management:
  - `InteractiveGroup` - Multi-option selection with descriptions
  - `InteractiveIconTitleOnly` - Simplified icon-only selection group

## Usage Example

```typescript
// Basic story definition
export const WithIcon: Story = {
  args: {
    title: 'Option',
    description: 'Description text',
    icon: <Smile size={24} />,
    selected: true,
  },
}

// Interactive story with state management
export const InteractiveGroup = {
  render: () => {
    const [selectedId, setSelectedId] = useState<string | null>('monitor')
    
    return (
      <div className="flex flex-col gap-2">
        {options.map((option) => (
          <SelectButton
            key={option.id}
            title={option.title}
            selected={selectedId === option.id}
            onClick={() => setSelectedId(option.id)}
          />
        ))}
      </div>
    )
  },
}
```

The stories cover all major SelectButton features including icons from Lucide React, image support, tags, disabled states, and interactive selection patterns for comprehensive component testing and documentation.
