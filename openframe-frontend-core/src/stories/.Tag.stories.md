Storybook stories for the `Tag` UI component, covering all variants, icon/close button combinations, disabled states, and gallery renders.

## Key Components

| Export | Description |
|--------|-------------|
| `Primary` | Default primary variant |
| `Outline` / `Success` / `Warning` / `Error` / `Critical` / `Grey` | Individual variant stories |
| `WithIcon` | Tag with a leading icon |
| `WithCloseButton` | Tag with an `onClose` dismiss handler |
| `WithIconAndClose` | Combined icon + close button |
| `Disabled` / `DisabledWithIconAndClose` | Disabled state stories |
| `AllVariants` / `AllWithIcons` / `AllWithClose` / `AllDisabled` / `AllWithIconAndClose` | Gallery renders showing every variant side-by-side |

## Usage Example

```typescript
import { Tag } from '../components/ui/tag';
import { Smile } from 'lucide-react';

// Basic variant
<Tag variant="success" label="Resolved" />

// With icon
<Tag variant="primary" label="Open" icon={<Smile className="size-5" />} />

// With close button
<Tag variant="warning" label="Pending" onClose={() => removeTag(id)} />

// Fully featured
<Tag
  variant="error"
  label="Critical"
  icon={<Smile className="size-5" />}
  onClose={() => removeTag(id)}
  disabled={false}
/>
```
