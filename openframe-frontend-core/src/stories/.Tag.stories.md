Storybook stories for the `Tag` UI component, showcasing all variants, icon combinations, close button behavior, and disabled states.

## Key Components

| Export | Description |
|--------|-------------|
| `Primary` | Default amber/primary variant |
| `Outline` | Outlined style variant |
| `Success` / `Warning` / `Error` / `Critical` / `Grey` | Semantic color variants |
| `WithIcon` | Tag with a leading icon slot |
| `WithCloseButton` | Tag with an `onClose` dismiss handler |
| `WithIconAndClose` | Combined icon + close button |
| `Disabled` / `DisabledWithIconAndClose` | Disabled state stories |
| `AllVariants` / `AllWithIcons` / `AllWithClose` / `AllWithIconAndClose` / `AllDisabled` | Gallery stories rendering every variant side-by-side |

## Usage Example

```typescript
import { Tag } from '../components/ui/tag';
import { Smile } from 'lucide-react';

// Basic variant
<Tag variant="success" label="Active" />

// With icon
<Tag variant="primary" label="Label" icon={<Smile className="size-5" />} />

// With close handler
<Tag variant="warning" label="Pending" onClose={() => removeTag(id)} />

// Disabled
<Tag variant="error" label="Inactive" disabled />
```

## Variants

| Variant | Use Case |
|---------|----------|
| `primary` | General purpose / default |
| `outline` | Subtle / secondary emphasis |
| `success` | Positive / completed state |
| `warning` | Cautionary state |
| `error` | Non-critical failure |
| `critical` | High-priority alert |
| `grey` | Neutral / inactive |
