Storybook configuration file that defines interactive stories for the Tag UI component, showcasing all available variants and features including icons and close functionality.

## Key Components

- **Meta Configuration**: Defines the component title, controls, and argTypes for the Tag component
- **Individual Variant Stories**: Separate stories for each tag variant (primary, outline, good, warning, error, critical, grey)
- **Feature Stories**: Stories demonstrating icons, close buttons, and combined functionality
- **Collection Stories**: Comprehensive displays showing all variants together with different feature combinations

## Usage Example

```typescript
import { Tag } from '../components/ui/tag';
import { Smile } from 'lucide-react';

// Basic tag usage
<Tag variant="primary">Primary Tag</Tag>

// Tag with icon
<Tag variant="good" icon={<Smile className="size-5" />}>
  Success
</Tag>

// Tag with close functionality
<Tag variant="error" onClose={() => handleClose()}>
  Closeable Tag
</Tag>

// Tag with both icon and close
<Tag 
  variant="warning" 
  icon={<Smile className="size-5" />}
  onClose={() => handleClose()}
>
  Interactive Tag
</Tag>
```

The stories provide a complete visual documentation of the Tag component's capabilities, including seven semantic variants for different use cases, optional icon support using Lucide React icons, and closeable functionality for interactive scenarios.
