A comprehensive Storybook configuration file that showcases the TabNavigation component through various usage scenarios and interactive examples.

## Key Components

- **Default Tab Collections**: Pre-configured sets of tabs (`defaultTabs`, `tabsWithIndicators`) with Lucide React icons
- **Story Variants**: Eight different story configurations demonstrating controlled mode, URL synchronization, indicators, and customization options
- **Interactive Examples**: Stories with React state management showing real-time tab switching behavior
- **Configuration Options**: Storybook meta configuration with argTypes for component properties and Next.js integration settings

## Usage Example

```typescript
// Basic controlled tab navigation story
export const Controlled: Story = {
  render: () => {
    const [active, setActive] = useState('overview');
    return (
      <TabNavigation
        tabs={defaultTabs}
        activeTab={active}
        onTabChange={setActive}
      />
    );
  },
};

// URL synchronized tabs with custom parameter
export const UrlSyncCustomParam: Story = {
  args: {
    tabs: defaultTabs,
    urlSync: { paramName: 'section', replaceState: true },
    defaultTab: 'overview',
  },
  parameters: {
    nextjs: {
      navigation: { query: { section: 'settings' } },
    },
  },
};
```

The stories cover controlled state management, URL synchronization, status indicators (good/warning/error), content rendering with children props, and styling customization scenarios.
