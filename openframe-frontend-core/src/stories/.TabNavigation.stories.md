Storybook stories for the TabNavigation component demonstrating various usage patterns including controlled state management, URL synchronization, and visual indicators.

## Key Components

- **defaultTabs**: Basic tab configuration with Home, Users, Settings, and Notifications
- **tabsWithIndicators**: Extended tab set including status indicators (good, warning, error)
- **Controlled**: Standard controlled component story with local state management
- **WithContent**: Demonstrates children render prop pattern for tab content
- **WithIndicators**: Shows tabs with visual status indicators
- **UrlSync**: URL parameter synchronization using default "tab" param
- **UrlSyncCustomParam**: Custom URL parameter name and replace state behavior

## Usage Example

```typescript
// Basic controlled tab navigation
const [activeTab, setActiveTab] = useState('overview');

<TabNavigation
  tabs={defaultTabs}
  activeTab={activeTab}
  onTabChange={setActiveTab}
/>

// With status indicators and content
<TabNavigation
  tabs={tabsWithIndicators}
  activeTab={activeTab}
  onTabChange={setActiveTab}
>
  {(activeTab) => (
    <div>Content for {activeTab}</div>
  )}
</TabNavigation>

// URL synchronized tabs
<TabNavigation
  tabs={defaultTabs}
  urlSync={{ paramName: 'section', replaceState: true }}
  defaultTab="overview"
/>
```

The stories cover all major interaction patterns including two-tab layouts, default tab overrides, and custom styling through className props.
