Shared Storybook fixtures providing realistic mock data for all chat entity-card variants in the `CHAT_CARD_REGISTRY`, enabling visual testing without a live chat runtime.

## Key Components

### Constants & Helpers
- `NOW` / `DAY` — reference timestamp and millisecond offset for deterministic date generation
- `isoDaysAgo(days)` — returns an ISO 8601 string relative to the fixed `NOW` anchor
- `SAMPLE_AUTHOR` — reusable author object (name, avatar, job title)

## Usage Example

```typescript
import {
  githubCommitItem,
  hubspotTicketItem,
  financialKpiItem,
} from '@/stories/fixtures/chat-cards'

// Use in a Storybook story
export const HubSpotTicketStory: Story = {
  args: {
    item: hubspotTicketItem,
    href: hubspotTicketItem.url,
  },
}

// Or drive a unit test
expect(githubCommitItem.kind).toBe('commit')
expect(financialKpiItem.facts).toHaveLength(3)
```

> **Note:** Server-only fields (status timestamps, AI flags, view counts) are populated with type-correct placeholders. The presentation cards do not consume them at render time, but the wire types require them for TypeScript compatibility.
