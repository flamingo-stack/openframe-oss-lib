Shared fixtures providing typed mock data for all chat entity-card Storybook stories, covering every `docType` in `CHAT_CARD_REGISTRY` without requiring the chat runtime.

## Key Components

### Constants & Helpers
- `NOW` / `DAY` — baseline timestamp and day offset for relative dates
- `isoDaysAgo(days)` — returns an ISO string N days before the fixture baseline
- `SAMPLE_AUTHOR` — reusable author object (name, avatar, job title)

## Usage Example

```typescript
import {
  githubCommitItem,
  hubspotTicketItem,
  financialKpiItem,
  blogPostSummary,
} from '@/stories/fixtures/chat-cards'

// Storybook story
export const HubSpotTicketCard: Story = {
  args: {
    item: hubspotTicketItem,
    href: hubspotTicketItem.url,
  },
}

// Multiple cards preview
const allItems = [githubCommitItem, hubspotTicketItem, financialKpiItem]
```

> **Note:** Server-only fields (status timestamps, AI flags, view counts) are populated with type-correct placeholders. These values are not read at render time but are required by the wire types.
