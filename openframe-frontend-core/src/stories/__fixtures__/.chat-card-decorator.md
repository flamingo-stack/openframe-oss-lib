Storybook decorator and helper utilities for rendering chat entity-card stories in their native chat-column context.

## Key Components

- **`ChatColumnDecorator`** — A React wrapper component that replicates the assistant message column layout (constrained 420px max-width, ODS background, padding). Use it as a Storybook decorator so entity-cards render at their real in-chat dimensions without booting the full chat runtime.

- **`makeAnchorProps`** — Factory that returns `{ href, onClick }` props matching the `*AnchorProps` interface expected by entity-cards. The `onClick` calls `preventDefault` to suppress iframe navigation in Storybook while keeping anchor behavior (hover, copy-link) intact.

## Usage Example

```typescript
// .storybook story file
import { ChatColumnDecorator, makeAnchorProps } from './chat-card-decorator'

export default {
  title: 'Chat/TicketCard',
  component: TicketCard,
  decorators: [(Story) => (
    <ChatColumnDecorator>
      <Story />
    </ChatColumnDecorator>
  )],
}

export const Default = {
  args: {
    title: 'Printer not working',
    anchorProps: makeAnchorProps('https://example.com/tickets/42'),
  },
}
```

> **Note:** Both utilities are Storybook-only concerns. `ChatColumnDecorator` mirrors the column width and background applied by the dispatch pipeline at runtime, so visual regression tests reflect production layout.
