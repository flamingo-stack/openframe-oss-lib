Storybook stories for the `ProductReleaseCard` component, covering the two supported density variants (`sm` and `lg`) for the `product_release` doc type.

## Key Components

| Export | Description |
|---|---|
| `ChatInlineSm` | Compact `sm` card for inline chat display |
| `ChatInlineSmWithVideoCover` | Compact `sm` card with `hasVideoCover` flag enabled |
| `Large` | Rich `lg` card with hero cover, version pill, changelog stats, and metadata grid footer |

## Usage Example

```typescript
import { ProductReleaseCard } from '../components/chat/entity-cards'

// Compact inline variant
<ProductReleaseCard
  {...productReleaseSmProps}
  size="sm"
  anchorProps={{ href: '/releases/2-5-0' }}
/>

// Rich large variant (catalog row, DevCenter tab, investor updates)
<ProductReleaseCard
  {...productReleaseLgProps}
  size="lg"
  anchorProps={{ href: '/releases/2-5-0' }}
/>
```

Both variants share a flat prop signature (no `item` wrapper). The `lg` variant renders a hero cover, version pill, changelog stats strip, and a 4-cell metadata grid footer (Type / Status / Released / Author). Stories are wrapped in `ChatColumnDecorator` to simulate the chat column layout context.
