Storybook stories for the `ProductReleaseCard` component, covering the two supported density variants (`sm` and `lg`) of the `product_release` entity card.

## Key Components

| Export | Description |
|--------|-------------|
| `ChatInlineSm` | Compact inline card used within chat responses |
| `ChatInlineSmWithVideoCover` | Compact variant with video cover thumbnail enabled |
| `Large` | Rich catalog-row card used on `/releases`, DevCenter tab, and investor-update related-content sections |

## Usage Example

```typescript
// Render the large variant programmatically
import { ProductReleaseCard } from '../components/chat/entity-cards'
import { productReleaseLgProps } from './__fixtures__/chat-cards'
import { makeAnchorProps } from './__fixtures__/chat-card-decorator'

<ProductReleaseCard
  {...productReleaseLgProps}
  size="lg"
  anchorProps={makeAnchorProps('/releases/2-5-0')}
/>
```

## Notes

- Both `sm` and `lg` share a **flat prop signature** — no `item` wrapper object required.
- The `lg` variant renders a hero cover, version pill, changelog stats strip, and a 4-cell metadata grid footer (`Type / Status / Released / Author`).
- The `sm` variant optionally accepts `hasVideoCover: true` to display a video thumbnail.
- All stories are wrapped in `ChatColumnDecorator` to simulate the chat column layout context.
