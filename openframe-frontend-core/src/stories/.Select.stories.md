Storybook stories file demonstrating the Select component and its various states and configurations for the OpenFrame UI library.

## Key Components

- **Meta Configuration** - Storybook metadata with title, component reference, layout parameters, and custom decorator
- **Story Variants** - Seven different stories showcasing select functionality:
  - `Default` - Basic select with placeholder
  - `WithValue` - Pre-selected value example
  - `Disabled` - Non-interactive disabled state
  - `WithGroups` - Categorized options with labels and separators
  - `WithDisabledItem` - Individual disabled items within options
  - `ManyItems` - Scrollable list with timezone options
  - `AllVariants` - Side-by-side comparison view

## Usage Example

```typescript
// Basic select implementation
<Select>
  <SelectTrigger>
    <SelectValue placeholder="Select an option..." />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">Option 1</SelectItem>
    <SelectItem value="option2">Option 2</SelectItem>
  </SelectContent>
</Select>

// Grouped select with categories
<Select>
  <SelectTrigger>
    <SelectValue placeholder="Select a fruit..." />
  </SelectTrigger>
  <SelectContent>
    <SelectGroup>
      <SelectLabel>Fruits</SelectLabel>
      <SelectItem value="apple">Apple</SelectItem>
    </SelectGroup>
    <SelectSeparator />
    <SelectGroup>
      <SelectLabel>Vegetables</SelectLabel>
      <SelectItem value="carrot">Carrot</SelectItem>
    </SelectGroup>
  </SelectContent>
</Select>
```

The stories use a custom decorator that provides consistent styling with padding, background color using CSS custom properties (`--ods-bg`), and a maximum width constraint for optimal visual presentation in Storybook.
