Comprehensive Storybook stories for the DatePicker component, demonstrating all variants including single date, date range, time selection, and input-style pickers with interactive examples.

## Key Components

- **Meta Configuration**: Defines Storybook metadata with UI category, component documentation, and control configurations
- **Story Variants**: 15+ interactive stories covering different DatePicker modes and configurations
- **DatePicker Stories**: Basic picker variants (single, range, disabled, constraints)
- **DatePickerInput Stories**: Input-style variants with time selection and formatting options
- **Composite Stories**: All variants showcase and complex usage examples

## Usage Example

```typescript
// Basic single date picker story
export const Single: Story = {
  render: function Render() {
    const [date, setDate] = useState<Date | undefined>();
    return (
      <DatePicker
        mode="single"
        placeholder="Select date"
        value={date}
        onChange={setDate}
      />
    );
  },
};

// Date range picker with two months
export const RangeTwoMonths: Story = {
  render: function Render() {
    const [range, setRange] = useState<DateRange | undefined>();
    return (
      <DatePicker
        mode="range"
        placeholder="Select date range"
        value={range}
        onChange={setRange}
        numberOfMonths={2}
      />
    );
  },
};

// Input-style picker with time selection
export const InputWithTime: Story = {
  render: function Render() {
    const [date, setDate] = useState<Date | undefined>(new Date());
    return (
      <DatePickerInput
        placeholder="Select date"
        value={date}
        onChange={setDate}
        showTime
      />
    );
  },
};
```

The stories provide comprehensive coverage of DatePicker functionality including state management, formatting, constraints, and different UI presentations for Storybook documentation.
