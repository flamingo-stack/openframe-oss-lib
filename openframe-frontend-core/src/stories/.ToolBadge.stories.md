Storybook configuration file for the ToolBadge component, providing interactive examples and documentation for all supported OpenFrame integrated tool types.

## Key Components

- **meta**: Main Storybook configuration with component metadata, controls, and documentation
- **Story exports**: Individual stories showcasing different tool types (Tactical RMM, Fleet MDM, MeshCentral, etc.)
- **AllTools story**: Comparison view showing multiple tool badges together
- **argTypes**: Interactive controls for `toolType` selection and `className` customization

## Usage Example

```typescript
// Individual tool badge stories
export const Default: Story = {
  args: {
    toolType: 'TACTICAL_RMM',
  },
};

// Custom render for multiple badges
export const AllTools: Story = {
  render: () => (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
      <ToolBadge toolType="OPENFRAME" />
      <ToolBadge toolType="FLEET_MDM" />
      <ToolBadge toolType="MESHCENTRAL" />
    </div>
  ),
};
```

This Storybook configuration enables developers and designers to preview all available tool badge variants, test different configurations, and understand the component's visual hierarchy within the OpenFrame platform interface.
