Storybook configuration for the ShellTypeBadge component, defining visual documentation and interactive examples for all supported Tactical RMM shell types including PowerShell, CMD, Bash, Python, Nushell, Deno, and generic Shell.

## Key Components

- **Meta Configuration**: Storybook metadata with component description, layout settings, and argument controls
- **Story Variants**: Individual stories for each shell type (PowerShell, CMD, Bash, Python, Nushell, Deno, Shell)
- **AllShellTypes Story**: Composite story displaying all shell type badges for visual comparison
- **ArgTypes**: Interactive controls for `shellType` selection and `className` customization

## Usage Example

```typescript
// Import the stories for Storybook development
import { Default, AllShellTypes } from './ShellTypeBadge.stories';

// The component is documented with these story configurations:
// - Individual shell type examples
// - Interactive controls for testing different shell types
// - Visual comparison of all supported shell types

// Story structure provides:
const exampleStory = {
  args: {
    shellType: 'POWERSHELL', // One of the supported ShellType values
    className: 'custom-class' // Optional additional styling
  }
};
```

The stories provide comprehensive visual documentation for the ShellTypeBadge component, making it easy for developers to understand the available shell types and see how each badge renders with its appropriate icon and styling.
