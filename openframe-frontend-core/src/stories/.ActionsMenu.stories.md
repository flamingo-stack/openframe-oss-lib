Storybook stories for the `ActionsMenu` and `ActionsMenuDropdown` UI components, providing interactive visual documentation and usage examples for all supported menu variants.

## Key Components

**Exported Stories**

| Story | Description |
|---|---|
| `Default` | Single group with icons |
| `GroupedWithDanger` | Multiple groups with a destructive delete row |
| `WithCheckboxes` | Checkbox-type items with toggled `checked` state |
| `WithSubmenu` | Nested submenu via `ActionsMenuDropdown` (requires Radix context) |
| `WithDisabledItem` | Non-interactive dimmed row |
| `WithSecondaryAction` | Row with an independent secondary icon button (e.g. open in new tab) |
| `Dropdown` | `ActionsMenuDropdown` attached to an ellipsis trigger button |

## Usage Example

```typescript
// Inline flat panel
<ActionsMenu
  groups={[
    {
      id: 'primary',
      separator: true,
      items: [
        { id: 'edit', label: 'Edit', icon: <PencilIcon />, onClick: handleEdit },
        { id: 'download', label: 'Download', icon: <Download01Icon />, onClick: handleDownload },
      ],
    },
    {
      id: 'destructive',
      items: [
        { id: 'delete', label: 'Delete', icon: <TrashBlankIcon />, danger: true, onClick: handleDelete },
      ],
    },
  ]}
  onItemClick={(id) => console.log('Clicked:', id)}
/>

// Dropdown variant — attaches menu to a trigger button
<ActionsMenuDropdown
  groups={groups}
  onItemClick={(id) => console.log('Clicked:', id)}
/>
```

> **Note:** Submenu items (`type: 'submenu'`) require a Radix `DropdownMenu` context and must be rendered through `ActionsMenuDropdown`, not the inline `ActionsMenu` panel.
