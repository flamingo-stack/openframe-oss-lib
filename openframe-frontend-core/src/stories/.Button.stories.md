This file defines Storybook stories for the Button component, providing comprehensive examples and interactive controls for all button variants, sizes, states, and icon combinations used throughout the Flamingo platform.

## Key Components

**Story Configuration:**
- `Meta` configuration with extensive argTypes for variant, size, disabled, loading, alignment, and styling options
- 20+ button variants including primary, secondary, destructive, Flamingo-themed, and specialized variants (search, filter, device-action)
- Multiple size options from small to touch-friendly with icon-specific sizing

**Story Categories:**
- **Variant Stories**: Primary, Secondary, White, Outline, Ghost, Destructive, Success, Warning, Info, Flamingo variants
- **Size Stories**: Small, Default, Large, Touch, Icon-only, and specialized search sizes  
- **Icon Stories**: Left, right, center, and combined icon placements with Lucide React icons
- **State Stories**: Disabled, loading, link behaviors, and external link handling
- **Layout Stories**: Alignment options, padding controls, and mobile responsiveness
- **Showcase Stories**: Comprehensive displays of all variants, sizes, and icon combinations

## Usage Example

```typescript
// Basic button story with variant and icon
export const Primary: Story = {
  args: {
    children: 'Primary Button',
    variant: 'primary',
  },
};

// Icon button with Lucide React icon
export const WithLeftIcon: Story = {
  args: {
    children: 'Add Item',
    leftIcon: <Plus />,
  },
};

// Complex showcase story with custom render function
export const AllVariants: Story = {
  render: () => (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
      <Button variant="primary">Primary</Button>
      <Button variant="secondary">Secondary</Button>
      <Button variant="destructive">Destructive</Button>
    </div>
  ),
};
```

The stories provide interactive controls for testing all button properties and include specialized showcases for icon buttons with numbers, external links, and comprehensive variant comparisons.
