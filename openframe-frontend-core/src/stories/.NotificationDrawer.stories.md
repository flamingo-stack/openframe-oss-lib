Storybook stories for the notification drawer feature, covering static tile variants, live animation states, drawer states, and an interactive playground that simulates real-time notification delivery.

## Key Components

| Export | Description |
|--------|-------------|
| `AllTileVariants` | Static preview of all five `NotificationVariant` tiles (`default`, `success`, `error`, `warning`, `info`) in settled state |
| `LiveTile` | Demonstrates the animated progress bar and dismiss button shown for ~4 seconds on arrival, with a replay button |
| `DrawerEmpty` | Drawer opened automatically with no notifications (empty state) |
| `DrawerWithSeedData` | Drawer pre-seeded with 6 notifications matching the Figma reference, with history callback wired |
| `LivePlayground` | Full interactive playground simulating NATS-style realtime delivery via `addNotification`, with pop-ups and drawer controls |

**Internal helpers:**
- `AutoOpen` — mounts invisibly and calls `useNotifications().open()` on load so drawer stories render open by default
- `PlaygroundControls` — renders fire/dismiss/clear buttons and exposes all `useNotifications()` state

## Usage Example

```typescript
// Simulate what a NATS subscription would do at the app root:
function App() {
  return (
    <NotificationsProvider
      initialNotifications={[]}
      onHistoryClick={() => router.push('/notifications')}
      onShowPopupsChange={(enabled) => console.log('popups', enabled)}
    >
      <NotificationDrawer />
      <NotificationPopups />
      <YourApp />
    </NotificationsProvider>
  );
}

// From any child component:
function SomeFeature() {
  const { addNotification } = useNotifications();

  useEffect(() => {
    natsSubscription.on('event', (msg) => {
      addNotification({
        variant: 'success',
        title: msg.title,
        description: msg.body,
        onClick: () => router.push(`/runs/${msg.id}`),
      });
    });
  }, []);
}
```
