Storybook stories for the `BlogCard` entity card component, covering both compact chat-inline (`sm`) and full vertical catalog (`default`) display densities.

## Key Components

| Export | Description |
|---|---|
| `meta` | Storybook metadata — registers `BlogCard` under `Chat/EntityCards/BlogCard`, wraps stories in `ChatColumnDecorator` |
| `ChatInlineSm` | Compact horizontal card (~80px tall) used inline in chat responses |
| `ChatInlineWithVideoBadge` | `sm` variant with `hasEmbeddedVideo: true` to render the video badge overlay |
| `Default` | Full vertical catalog card used in related-content rails (not dispatched inline in chat) |

## Usage Example

```typescript
// Render the compact chat-inline card
<BlogCard
  post={blogPostSummary}
  href={`/blog/${blogPostSummary.slug}`}
  size="sm"
  target="_blank"
  rel="noopener noreferrer"
/>

// Render the full catalog card with video badge
<BlogCard
  post={blogPostSummary}
  href={`/blog/${blogPostSummary.slug}`}
  size="sm"
  hasEmbeddedVideo={true}
  target="_blank"
  rel="noopener noreferrer"
/>

// Full vertical card for related-content rail
<BlogCard
  post={blogPostSummary}
  href={`/blog/${blogPostSummary.slug}`}
  size="default"
  target="_blank"
  rel="noopener noreferrer"
/>
```

> **Note:** The card renders its own `<a>` tag internally. Callers are responsible for resolving the `href` value via `buildContentURL` or `useNavLink` before passing it as a prop.
