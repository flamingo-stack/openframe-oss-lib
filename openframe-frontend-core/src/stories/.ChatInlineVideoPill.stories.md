Storybook stories for `ChatInlineVideoPill`, an inline span component that renders a small video sentinel pill alongside markdown text in chat assistant turns.

## Key Components

- **`ChatInlineVideoPill`** — Local re-implementation of the `video` doc-type inline pill from `dispatch.tsx` (lines 318–326). Renders a `▶` icon + truncated title in a bordered pill; stays inline within paragraph text rather than breaking into a card.
- **`Default`** — Baseline story with a short video title rendered mid-sentence.
- **`LongTitleTruncates`** — Verifies that long titles truncate cleanly (`max-w-[220px]`) without wrapping adjacent paragraph text.

## Usage Example

```typescript
// Matches the source component in dispatch.tsx
<ChatInlineVideoPill title="On-call rotation walkthrough" />

// Renders as an inline span within a paragraph:
// "Here is the demo from last week's call: [▶ On-call rotation walkthrough] — watch the part..."
```

## Notes

- This pill is the **only entity card** designed to remain inline — all other entity cards use a card wrapper with Ask/Display affordances.
- The actual video player is **hoisted out** of the paragraph by the chat pre-scan pipeline and rendered above the assistant turn; this pill acts purely as a sentinel.
- Wrapped with `ChatColumnDecorator` to simulate realistic chat column width constraints during story rendering.
