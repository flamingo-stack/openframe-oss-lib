Storybook stories for `ChatInlineVideoPill`, an inline video sentinel pill rendered within chat markdown paragraphs for the `video` doc type.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `ChatInlineVideoPill` | `function` | Local re-implementation of the inline pill from `dispatch.tsx` — renders a small `▶ title` span inline with surrounding text |
| `Default` | `Story` | Standard pill with a short, readable title |
| `LongTitleTruncates` | `Story` | Validates `max-w-[220px]` truncation behavior on long titles |

## Design Notes

- **Inline-only** — unlike other entity cards, this pill has no card wrapper, no Ask/Display affordance, and no separate container. It sits inline alongside paragraph text using `inline-flex`.
- **Sentinel pattern** — the pill acts as a placeholder; the actual video player is hoisted *above* the assistant turn by the chat pre-scan pipeline (`dispatch.tsx`), so this pill is purely visual/contextual.
- The decorator wraps each story in `ChatColumnDecorator` with a realistic prose sentence to verify inline alignment.

## Usage Example

```typescript
// Rendered inside a paragraph alongside markdown text
<p className="text-sm text-ods-text-primary leading-relaxed">
  Here is the demo from last week's call:{' '}
  <ChatInlineVideoPill title="On-call rotation walkthrough" />
  — watch the part where Jordan walks through the on-call rotation.
</p>
```

> **Note:** `ChatInlineVideoPill` is defined locally in `dispatch.tsx` (lines 318–326). This story re-implements it byte-for-byte for isolated rendering — any changes to the source component must be manually mirrored here.
