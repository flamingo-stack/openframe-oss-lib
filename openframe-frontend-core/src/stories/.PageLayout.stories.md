Storybook stories for `PageLayout`, demonstrating all supported layout variants including title-only, subtitle/image combinations, action button styles, mobile responsive behavior, and long-text truncation.

## Key Components

- **`TitleOnly`** — Baseline header with title and content, no extras
- **`WithSubtitleAndImage`** — Full organization detail header: thumbnail, subtitle, back button, and `icon-buttons` actions
- **`MobileActionsMenu`** — Same as above at mobile viewport; actions collapse into a `…` dropdown
- **`WithSubtitleOnly`** — Title + context subtitle with `primary-buttons` actions
- **`WithImageOnly`** — Branded entity header with thumbnail but no subtitle line
- **`WithSelector`** — Desktop view toggle (`TabSelector`) rendered before action buttons; hidden on mobile
- **`WithSelectorMobile`** — `WithSelector` at mobile viewport; selector hidden, actions remain
- **`LongTitleTruncates`** — Verifies ellipsis truncation when title/subtitle overflow the header row

## Usage Example

```typescript
// Minimal title-only layout
<PageLayout title="Devices">
  <Table data={devices} columns={columns} rowKey="id" />
</PageLayout>

// Full org detail header
<PageLayout
  title="TechFlow Solutions"
  subtitle="techflow.com • Software Development"
  image={{ src: logoUri, alt: 'TechFlow Solutions logo' }}
  backButton={{ label: 'Back to Organizations', onClick: handleBack }}
  actionsVariant="icon-buttons"
  actions={[
    { label: 'Archive Organization', onClick: handleArchive, variant: 'outline', icon: <BoxArchiveIcon size={24} /> },
    { label: 'Edit Organization',    onClick: handleEdit,    variant: 'outline', icon: <PenEditIcon size={24} /> },
  ]}
>
  <Table data={devices} columns={columns} rowKey="id" />
</PageLayout>

// With desktop-only view selector
<PageLayout
  title="Devices"
  actionsVariant="icon-buttons"
  actions={[{ label: 'Add Device', onClick: fn(), variant: 'outline', icon: <PlusCircleIcon size={24} /> }]}
  selector={
    <TabSelector
      value={view}
      onValueChange={setView}
      items={[
        { id: 'table', label: '', icon: <TableCellIcon size={24} /> },
        { id: 'grid',  label: '', icon: <GridLayoutIcon size={24} /> },
      ]}
    />
  }
>
  <Table data={devices} columns={columns} rowKey="id" />
</PageLayout>
```

> **`actionsVariant`** controls how header actions render: `icon-buttons` collapses into a `…` menu on mobile, `primary-buttons` renders full-width accent buttons, and `menu-primary` combines a dropdown with a primary CTA.
