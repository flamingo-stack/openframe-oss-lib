Storybook stories for `PageLayout`, demonstrating all supported layout configurations including titles, subtitles, thumbnail images, back buttons, header actions, tab selectors, and responsive mobile behavior.

## Key Components

| Export | Description |
|--------|-------------|
| `TitleOnly` | Baseline header with title and table content |
| `WithSubtitleAndImage` | Full org detail header with logo, subtitle, back button, and icon-button actions |
| `MobileActionsMenu` | `WithSubtitleAndImage` at mobile viewport — actions collapse into a `…` dropdown |
| `WithSubtitleOnly` | Title + one-line context note, primary-button actions |
| `WithImageOnly` | Branded entity header without a subtitle line |
| `WithSelector` | Desktop `TabSelector` for table/grid toggle rendered before action buttons |
| `WithSelectorMobile` | `WithSelector` at mobile viewport — selector hidden, only `…` dropdown remains |
| `LongTitleTruncates` | Verifies ellipsis truncation when title/subtitle overflow |

## Usage Example

```typescript
// Minimal title-only layout
<PageLayout title="Devices">
  <Table data={devices} columns={columns} rowKey="id" />
</PageLayout>

// Full org detail layout with image, subtitle, and icon-button actions
<PageLayout
  title="TechFlow Solutions"
  subtitle="techflow.com • Software Development"
  image={{ src: logoUri, alt: 'TechFlow Solutions logo' }}
  backButton={{ label: 'Back to Organizations', onClick: handleBack }}
  actionsVariant="icon-buttons"
  actions={[
    { label: 'Archive Organization', onClick: handleArchive, variant: 'outline', icon: <BoxArchiveIcon size={24} /> },
    { label: 'Edit Organization',    onClick: handleEdit,    variant: 'outline', icon: <PenEditIcon size={24} />    },
  ]}
>
  <Table data={devices} columns={columns} rowKey="id" />
</PageLayout>

// With a desktop-only tab selector (hidden on mobile)
<PageLayout
  title="Devices"
  actionsVariant="icon-buttons"
  actions={[{ label: 'Add Device', onClick: fn(), variant: 'outline', icon: <PlusCircleIcon size={24} /> }]}
  selector={
    <TabSelector
      value={view}
      onValueChange={setView}
      items={[
        { id: 'table', label: '', icon: <TableCellIcon size={24} /> },
        { id: 'grid',  label: '', icon: <GridLayoutIcon size={24} /> },
      ]}
    />
  }
>
  {content}
</PageLayout>
```

> The `actionsVariant` prop controls rendering: `icon-buttons` collapses to a `…` dropdown on mobile, `primary-buttons` renders accent/outline buttons at all breakpoints, and `menu-primary` combines a menu trigger with a primary action button.
