Storybook configuration file for the PageActions UI component, demonstrating its three main variants: icon buttons for list pages, primary buttons for forms, and a menu+primary combination for complex actions.

## Key Components

- **Meta Configuration**: Storybook metadata defining component parameters, layout settings, and documentation
- **Story Variants**: Multiple story examples showcasing different PageActions configurations:
  - `IconButtons` - Icon-based actions that collapse to dropdown on mobile
  - `PrimaryButtons` - Primary and outline button combinations
  - `MenuPrimary` - MoreActionsMenu with primary button setup
- **Component Props**: Demonstrates `variant`, `actions`, `menuActions`, `gap`, and responsive behavior
- **State Examples**: Loading states, disabled states, and mobile-only actions

## Usage Example

```typescript
// Basic icon buttons story
export const IconButtons: Story = {
  args: {
    variant: 'icon-buttons',
    actions: [
      {
        label: 'Edit Categories',
        onClick: fn(),
        icon: <ColorsIcon size={24} />,
      },
      {
        label: 'Add Script', 
        onClick: fn(),
        icon: <PlusCircleIcon size={24} />,
      },
    ],
  },
}

// Menu + primary variant with dropdown actions
export const MenuPrimary: Story = {
  args: {
    variant: 'menu-primary',
    actions: [
      {
        label: 'Run Script',
        onClick: fn(),
        variant: 'primary',
        icon: <PlayCircleIcon size={24} />,
      },
    ],
    menuActions: [
      {
        label: 'Edit Script',
        onClick: fn(),
        icon: <PenEditIcon size={24} />,
      },
    ],
  },
}
```
