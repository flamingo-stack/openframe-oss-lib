A Storybook file for documenting and testing the PageActions component with comprehensive examples across three layout variants and various interaction states.

## Key Components

- **Meta Configuration** - Storybook metadata with component description and controls
- **Variant Stories** - Examples for `icon-buttons`, `primary-buttons`, and `menu-primary` layouts
- **State Examples** - Stories demonstrating loading, disabled, and mobile-only behaviors
- **Layout Options** - Different gap sizes and button configurations

## Usage Example

```typescript
// Basic icon buttons variant
export const IconButtons: Story = {
  args: {
    variant: 'icon-buttons',
    actions: [
      {
        label: 'Edit Categories',
        onClick: fn(),
        icon: <ColorsIcon size={24} />,
      },
      {
        label: 'Add Script',
        onClick: fn(),
        icon: <PlusCircleIcon size={24} />,
      },
    ],
  },
}

// Menu + primary variant with additional menu actions
export const MenuPrimary: Story = {
  args: {
    variant: 'menu-primary',
    actions: [
      {
        label: 'Run Script',
        onClick: fn(),
        variant: 'primary',
        icon: <PlayCircleIcon size={24} />,
      },
    ],
    menuActions: [
      {
        label: 'Edit Script',
        onClick: fn(),
        icon: <PenEditIcon size={24} />,
      },
    ],
  },
}
```

The stories cover responsive behavior where icon buttons collapse to dropdown menus on mobile, and primary buttons become fixed bottom bars. Special attention is given to mobile-only actions using the `showOnlyMobile` prop.
