Storybook configuration file that demonstrates all variations and states of the PageActions component, which provides flexible action button layouts for page headers.

## Key Components

- **Meta Configuration**: Defines the component story structure with layout parameters and comprehensive documentation
- **Story Variants**: 
  - `IconButtons` - Icon-based buttons that collapse to dropdown on mobile
  - `PrimaryButtons` - Primary and outline button combinations
  - `MenuPrimary` - More actions menu with primary button
- **State Examples**: Stories showing disabled, loading, and gap variations
- **ArgTypes**: Control definitions for variant, actions, gap, and className props

## Usage Example

```typescript
// Import the stories for Storybook development
import { IconButtons, PrimaryButtons, MenuPrimary } from './PageActions.stories'

// Example action configuration used in stories
const exampleActions = [
  {
    label: 'Save Script',
    onClick: () => console.log('Saving...'),
    variant: 'primary' as const,
    icon: <CheckCircleIcon size={24} />
  },
  {
    label: 'Test Script', 
    onClick: () => console.log('Testing...'),
    variant: 'outline' as const,
    icon: <VialIcon size={24} />
  }
]

// The stories demonstrate PageActions usage with PageContainer
<PageActions
  variant="primary-buttons"
  actions={exampleActions}
  gap="md"
/>
```

The stories showcase responsive behavior where buttons adapt from desktop layouts to mobile-optimized patterns like fixed bottom bars and dropdown menus.
