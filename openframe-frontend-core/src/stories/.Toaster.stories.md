Storybook stories for the `Toaster` / `ToastCard` component system, covering static variant previews, interactive playground, command-approval flows, and live Sonner-triggered toasts.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `AllVariants` | `Story` | Static grid of every `ToastVariant` plus edge cases (title-only, no-close, persistent) |
| `Playground` | `Story` | Arg-driven interactive preview wired to all `argTypes` controls |
| `CommandApprovalCollapsed` | `Story` | `CommandApprovalToast` in its default collapsed state |
| `CommandApprovalExpanded` | `Story` | `CommandApprovalToast` with `defaultExpanded` showing Approve / Reject row |
| `CommandApprovalPerTool` | `Story` | One `CommandApprovalToast` per supported tool type to verify icon mapping |
| `CommandApprovalLive` | `Story` | Buttons that fire real Sonner command-approval toasts with `onApprove`/`onReject` callbacks |
| `LiveTriggers` | `Story` | Button grid that fires every variant + edge case as live Sonner toasts |

## Usage Example

```typescript
// Render Toaster once near your app root
import { Toaster } from '@/components/ui/toaster';
<Toaster />

// Fire a toast from anywhere
import { showToast } from '@/components/ui/toaster';
import { showCommandApprovalToast } from '@/components/ui/toaster';
import { toast } from '@/hooks/use-toast';

// Simple variants
showToast({ variant: 'success', title: 'Saved', description: 'Changes applied.' });
showToast({ variant: 'error',   title: 'Failed', dismissible: false });
showToast({ variant: 'info',    title: 'Persistent', duration: Infinity });

// Command approval with Approve / Reject callbacks
showCommandApprovalToast({
  title: 'Tech Required',
  description: 'Approval is required to execute the command.',
  command: 'fleetctl query --hosts mac-* "SELECT * FROM system_info"',
  toolType: ToolTypeValues.FLEET_MDM,
  approvalDescription: 'Queries system info across all macOS endpoints.',
  onApprove: () => toast({ variant: 'success', title: 'Command approved' }),
  onReject:  () => toast({ variant: 'error',   title: 'Command rejected' }),
});
```

> **Supported variants:** `default` · `success` · `warning` · `error` · `info`
>
> **Supported tool types:** `FLEET_MDM` · `TACTICAL_RMM` · `MESHCENTRAL` · `AUTHENTIK` · `OSQUERY` · `OPENFRAME`
