Storybook stories for the `GuideWelcome` component, covering the guide-mode chat empty state with various configurations of quick actions, slash-command lists, and greeting copy.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `Default` | `Story` | Full empty state with slash-command list and quick-action chips |
| `NoCommands` | `Story` | Greeting and quick actions only, no slash-command list |
| `NoQuickActions` | `Story` | Slash-command list only, chip row omitted |
| `CustomCopy` | `Story` | Overridden title/subtitle with full content |

**Supporting fixtures:**

- `SAMPLE_QUICK_ACTIONS` — 7 chip items demonstrating inline display + `⋯` overflow menu
- `SAMPLE_LIST` — three `MingoOnboardingCard` entries (Product Releases, Blog Posts, Onboarding Guides) wrapped in a bordered container
- `ACTIONS` — shared Recent / Search / Find callbacks passed to each card

## Usage Example

```typescript
// Renders GuideWelcome inside a simulated 440×760 guide drawer panel
<div className="flex h-[760px] w-[440px] flex-col overflow-hidden rounded-md border border-ods-border bg-ods-bg">
  <GuideModeBanner />
  <div className="flex flex-1 min-h-0 flex-col p-4">
    <GuideWelcome
      title="Ask the Guide"
      subtitle="A temporary session for exploring docs and tickets."
      quickActions={SAMPLE_QUICK_ACTIONS}
      onQuickAction={(action) => console.log('quick action', action.id)}
    >
      {SAMPLE_LIST}
    </GuideWelcome>
  </div>
</div>
```
