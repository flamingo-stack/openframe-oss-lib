Storybook stories for the `ChatContainer` component, demonstrating the full interactive Fae client-facing chat shell with multiple layout and configuration variants.

## Key Components

- **`FaeChatShell`** — Interactive story wrapper that composes `ChatContainer`, `ChatHeader`, `ChatContent`, `ChatMessageList`, and `ChatInput` with controlled state; typing and sending appends new user messages to the list
- **`FAE_MESSAGES`** — Seed conversation between the Fae assistant and a user reporting a printer issue, used to pre-populate all stories
- **`Fae`** — Default story: standard centered layout with Fae persona header
- **`FaeWithTicket`** — Fae shell with `ticketInfo` populated on the header (ticket title, meta, status badge)
- **`FaeFullWidth`** — Drops the centered 600px content column; suited for side panels or mobile drawers
- **`FaeBareHeader`** — Removes card chrome (background, border, shadow) so the header blends flush with a host-supplied container

## Usage Example

```typescript
// Render the default Fae chat shell in a fullscreen layout
<FaeChatShell />

// With an active ticket displayed in the header
<FaeChatShell withTicketInfo />

// Full-width layout for narrow side panels
<FaeChatShell fullWidth />

// Bare header + full-width for drawers that supply their own card chrome
<FaeChatShell bare fullWidth />
```

## Props — `FaeChatShell`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `withTicketInfo` | `boolean` | `false` | Shows ticket title, meta, and status on the header |
| `bare` | `boolean` | `false` | Removes card chrome from the header |
| `fullWidth` | `boolean` | `false` | Disables centered column on header, list, and footer |
