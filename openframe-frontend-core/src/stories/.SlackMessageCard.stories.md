Storybook stories for `SlackMessageCard`, a chat entity card component that renders Slack message previews inline. Covers both display variants used across the Flamingo/OpenFrame chat UI.

## Key Components

- **`Compact` story** — renders the card as it appears inline within a chat message bubble
- **`RowVariant` story** — renders the card in list/row contexts (e.g., search results, grouped entity lists)
- **`ChatColumnDecorator`** — wraps stories in a realistic chat column layout for accurate sizing
- **`slackMessageItem`** — shared fixture representing a `slack_message` doc type with PII-sanitized preview data
- **`makeAnchorProps`** — helper that generates link props from the item's URL

## Usage Example

```typescript
import { SlackMessageCard } from '../components/chat/entity-cards'

// Inline chat usage
<SlackMessageCard
  item={slackMessageItem}
  variant="compact"
  anchorProps={makeAnchorProps(slackMessageItem.url ?? '#')}
/>

// List/row context
<SlackMessageCard
  item={slackMessageItem}
  variant="row"
  anchorProps={makeAnchorProps(slackMessageItem.url ?? '#')}
/>
```

## Notes

- Component is **pure presentation** — no side effects or data fetching
- Slack message previews are **PII-sanitized server-side** before reaching this component
- Both variants share the same `item` and `anchorProps` interface; only layout differs
