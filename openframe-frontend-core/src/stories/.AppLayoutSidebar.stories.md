Storybook stories for the `AppLayoutSidebar` component, demonstrating how `AppLayout` integrates with `AppLayoutDrawer` to render a toggleable Mingo AI chat sidebar within the main content area.

## Key Components

- **`Default`** — Story showing the Mingo AI header button toggling a right-side `AppLayoutDrawer`; navigation sidebar and header remain interactive while the drawer is open
- **`Resizable`** — Story demonstrating a draggable/resizable chat sidebar with configurable `minSize`, `defaultSize`, and `storageKey` persistence; starts open by default
- **`DashboardChildren`** — Internal helper component rendering a mock dashboard with device stats and a prompt to open the Mingo AI panel
- **`navigationItems`** — Shared `NavigationSidebarConfig` fixture with Dashboard, Customers, Devices, Scripts, and Settings entries

## Usage Example

```typescript
// Default: toggle Mingo AI sidebar via header button
const [open, setOpen] = useState(false)

<AppLayout
  sidebarConfig={{ items: navigationItems, onNavigate: fn() }}
  headerProps={{
    showMingoAI: true,
    onMingoAI: () => setOpen((prev) => !prev),
    isMingoAIActive: open,
  }}
  drawer={
    <AppLayoutDrawer open={open} onOpenChange={setOpen}>
      <AppLayoutDrawerContent side="right">
        <AppLayoutDrawerHeader>
          <AppLayoutDrawerTitle>Mingo AI</AppLayoutDrawerTitle>
        </AppLayoutDrawerHeader>
        <AppLayoutDrawerBody>Chat content here.</AppLayoutDrawerBody>
      </AppLayoutDrawerContent>
    </AppLayoutDrawer>
  }
>
  <DashboardContent />
</AppLayout>

// Resizable variant — add resizable props to AppLayoutDrawerContent
<AppLayoutDrawerContent
  side="right"
  resizable
  minSize={360}
  defaultSize={560}
  storageKey="app:mingo-sidebar"
/>
```
