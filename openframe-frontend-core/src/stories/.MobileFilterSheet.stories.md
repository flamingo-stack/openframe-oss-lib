A Storybook configuration file that defines interactive stories for the MobileFilterSheet component, demonstrating various filtering and sorting scenarios for mobile interfaces.

## Key Components

The file exports several story configurations:

- **Default** - Basic mobile filter sheet with status filters matching Figma design
- **Interactive** - Fully functional example with state management and filter counter
- **MultipleGroups** - Comprehensive example showing multiple filter categories (statuses, device types, OS, locations, departments, roles)
- **WithoutCounts** - Simple filter list without item counts
- **WithSorting** - Combined filtering and sorting functionality
- **SortingOnly** - Sorting interface without filters
- **InteractiveWithSorting** - Full interactive example with both filtering and sorting state management
