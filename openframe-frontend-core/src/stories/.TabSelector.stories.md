Storybook stories for the `TabSelector` UI component, showcasing all supported variants, states, and real-world usage patterns.

## Key Components

### Story Exports

| Story | Description |
|-------|-------------|
| `Primary` | Active tab with amber/accent background (`primary` variant) |
| `Secondary` | Active tab with soft grey background (`secondary` variant) |
| `FiveTabs` | Five-tab layout matching Figma design spec |
| `WithIcons` | Tabs with Lucide icons (Monitor, Tablet, Smartphone) |
| `WithLabel` | Optional label rendered above the selector |
| `WithDisabled` | Single tab item marked `disabled: true` |
| `Disabled` | Entire selector disabled via top-level `disabled` prop |
| `TwoTabs` | Minimal two-tab toggle (Enabled / Disabled) |
| `PlatformExample` | Real-world OS picker with icons and label |
| `VariantComparison` | Side-by-side primary vs. secondary rendering |
| `WithBadges` | Tabs with custom badge elements (e.g., "Soon" pill) |

### `TabSelectorItem` Shape

```typescript
interface TabSelectorItem {
  id: string;
  label: string;
  icon?: React.ReactNode;       // Optional Lucide or custom icon
  disabled?: boolean;           // Disables individual tab
  badge?: React.ReactNode;      // Custom badge element (e.g., "Soon" pill)
}
```

## Usage Example

```typescript
import { TabSelector, type TabSelectorItem } from '@/components/ui/tab-selector';
import { useState } from 'react';
import { Monitor, Apple, Globe } from 'lucide-react';

const items: TabSelectorItem[] = [
  { id: 'windows', label: 'Windows', icon: <Monitor className="w-5 h-5" /> },
  { id: 'macos',   label: 'macOS',   icon: <Apple className="w-5 h-5" /> },
  { id: 'linux',   label: 'Linux',   icon: <Globe className="w-5 h-5" />, disabled: true },
];

export function PlatformPicker() {
  const [platform, setPlatform] = useState('macos');

  return (
    <TabSelector
      label="Select Platform"
      variant="primary"      // 'primary' | 'secondary'
      value={platform}
      onValueChange={setPlatform}
      items={items}
    />
  );
}
```
