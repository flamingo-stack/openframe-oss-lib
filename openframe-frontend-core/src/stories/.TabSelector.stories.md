Storybook stories for the `TabSelector` UI component, demonstrating all supported variants, states, and real-world usage patterns.

## Key Components

### Story Exports

| Story | Description |
|-------|-------------|
| `Primary` | Accent background on active tab (default variant) |
| `Secondary` | Soft grey background on active tab |
| `FiveTabs` | Five-tab layout matching Figma spec |
| `WithIcons` | Tabs with Lucide icon prefixes |
| `WithLabel` | Optional label rendered above the selector |
| `WithDisabled` | Single tab item disabled via `disabled: true` |
| `Disabled` | Entire selector disabled via top-level `disabled` prop |
| `TwoTabs` | Minimal two-tab toggle pattern |
| `PlatformExample` | Real-world platform selector (Windows/macOS/Linux) |
| `VariantComparison` | Side-by-side `primary` vs `secondary` rendering |
| `WithBadges` | Tabs with inline badge elements (e.g. "Soon" pill) |

### Data Fixtures

- `basicItems` — 3 plain text tabs
- `fiveItems` — 5 plain text tabs
- `itemsWithIcons` — Desktop / Tablet / Mobile with Lucide icons

## Usage Example

```typescript
import { TabSelector, type TabSelectorItem } from '@/components/ui/tab-selector';
import { useState } from 'react';

const items: TabSelectorItem[] = [
  { id: 'windows', label: 'Windows' },
  { id: 'macos', label: 'macOS' },
  { id: 'linux', label: 'Linux', disabled: true, badge: <span>Soon</span> },
];

export function PlatformPicker() {
  const [value, setValue] = useState('windows');
  return (
    <TabSelector
      value={value}
      onValueChange={setValue}
      items={items}
      variant="primary"
      label="Platform"
    />
  );
}
```

> All stories use a controlled `useState` render wrapper so tab selection is interactive inside Storybook, not just a static snapshot.
