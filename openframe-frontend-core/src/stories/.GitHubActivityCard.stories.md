Storybook stories for `GitHubActivityCard`, covering GitHub commit, pull request, and PR review entity cards rendered in a chat column layout.

## Key Components

| Export | Description |
|--------|-------------|
| `Commit` | Compact card for `github_commit` doc type |
| `PullRequest` | Compact card for `github_pull_request` doc type |
| `PullRequestReview` | Compact card for `github_pr_review` doc type with `[review:STATE]` badge parsing |
| `RowVariant` | Row-layout variant using the pull request fixture |

## Usage Example

```typescript
import { GitHubActivityCard } from '../components/chat/entity-cards'
import { makeAnchorProps } from './__fixtures__/chat-card-decorator'
import { githubCommitItem } from './__fixtures__/chat-cards'

// Compact commit card
<GitHubActivityCard
  item={githubCommitItem}
  variant="compact"
  anchorProps={makeAnchorProps(githubCommitItem.url ?? '#')}
/>

// Row layout for pull requests
<GitHubActivityCard
  item={githubPullRequestItem}
  variant="row"
  anchorProps={makeAnchorProps(githubPullRequestItem.url ?? '#')}
/>
```

## Notes

- Supports `compact` and `row` variants
- `owner/repo` is derived from the item URL when `item.repo` is omitted
- Long commit SHAs are automatically shortened
- PR review titles with `[review:STATE]` markers drive the state badge color
- Wrapped in `ChatColumnDecorator` to simulate the chat column layout
- Covers `_public` doc type variants implicitly through the same component
