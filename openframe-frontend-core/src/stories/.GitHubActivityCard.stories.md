Storybook stories for `GitHubActivityCard`, covering GitHub commit, pull request, and PR review entity card variants used in chat.

## Key Components

| Export | Description |
|--------|-------------|
| `Commit` | Compact card story for `github_commit` doc type |
| `PullRequest` | Compact card story for `github_pull_request` doc type |
| `PullRequestReview` | Compact card story for `github_pr_review` doc type |
| `RowVariant` | Row-layout variant using `github_pull_request` fixture |

## Usage Example

```typescript
import { GitHubActivityCard } from '../components/chat/entity-cards'
import { makeAnchorProps } from './__fixtures__/chat-card-decorator'
import { githubPullRequestItem } from './__fixtures__/chat-cards'

// Render a compact PR card
<GitHubActivityCard
  item={githubPullRequestItem}
  variant="compact"
  anchorProps={makeAnchorProps(githubPullRequestItem.url ?? '#')}
/>

// Render a row-layout PR card
<GitHubActivityCard
  item={githubPullRequestItem}
  variant="row"
  anchorProps={makeAnchorProps(githubPullRequestItem.url ?? '#')}
/>
```

## Notes

- Supports `compact` and `row` display variants
- Pure presentational component — callers must pass resolved `anchorProps`
- Automatically derives `owner/repo` from URL when `item.repo` is absent
- Shortens long commit SHAs for display
- Parses `[review:STATE]` markers from review titles to determine badge color
- Covers both standard and `_public` doc type variants
