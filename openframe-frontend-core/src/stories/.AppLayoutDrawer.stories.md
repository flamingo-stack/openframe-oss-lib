Storybook stories for `AppLayoutDrawer`, demonstrating the in-layout drawer component that renders within `AppLayout`'s main content area without covering the global chrome (header/sidebar).

## Key Components

| Export | Description |
|--------|-------------|
| `Right` | Default right-side drawer; persistent — does not close on sidebar/header clicks |
| `ResizableRight` | Right-side drawer with drag-to-resize handle, clamped to the main content area |
| `Left` | Left-side drawer variant, suitable for filters or secondary navigation panels |
| `DashboardChildren` | Internal helper rendering a mock dashboard with a trigger button |

## Usage Example

```typescript
import {
  AppLayout,
  AppLayoutDrawer,
  AppLayoutDrawerBody,
  AppLayoutDrawerContent,
  AppLayoutDrawerHeader,
  AppLayoutDrawerTitle,
} from '../components/navigation'

function MyPage() {
  const [open, setOpen] = useState(false)

  return (
    <AppLayout
      sidebarConfig={{ items: navigationItems, onNavigate: handleNav }}
      drawer={
        <AppLayoutDrawer open={open} onOpenChange={setOpen}>
          <AppLayoutDrawerContent
            side="right"
            resizable
            minSize={360}
            defaultSize={560}
            storageKey="my-app:drawer"
          >
            <AppLayoutDrawerHeader>
              <AppLayoutDrawerTitle>Details</AppLayoutDrawerTitle>
            </AppLayoutDrawerHeader>
            <AppLayoutDrawerBody>
              <p>Panel content here.</p>
            </AppLayoutDrawerBody>
          </AppLayoutDrawerContent>
        </AppLayoutDrawer>
      }
    >
      <Button onClick={() => setOpen(true)}>Open Drawer</Button>
    </AppLayout>
  )
}
```

## Notes

- Use `AppLayoutDrawer` for page-scoped panels (chat, filters, detail views) where the sidebar and header must remain accessible.
- Use the standard `Drawer` from `components/ui` instead when a full-viewport overlay is needed (e.g. auth modals, notifications).
- The `resizable` prop on `AppLayoutDrawerContent` enables drag-to-resize; `storageKey` persists the panel size across sessions.
