Manages a singleton NATS WebSocket connection shared across multiple consumers, handling reference counting, deferred cleanup, and coordinated reconnection with exponential backoff.

## Key Components

**`NATS_DEFAULTS`** — Compile-time constants for timeouts, ping intervals, and retry backoff parameters.

**`SharedConnection`** — Interface representing the singleton connection state, including the `NatsClient`, `refCount`, deferred close/retry timers, and `retryOwner` (prevents multiple consumers racing to reconnect simultaneously).

**`acquireClient(url, opts?)`** — Increments `refCount` and returns the shared connection, creating a new one if the URL differs. Cancels any pending deferred close on re-acquire.

**`releaseClient(url, opts?)`** — Decrements `refCount` and schedules a deferred `close()` (default 3s) when the last consumer releases, allowing fast re-acquisition without a full reconnect cycle.

**`getSharedConnection()` / `getSharedConnectionFor(url)`** — Accessors for the current singleton, useful for status checks without affecting the ref count.

**`startConnectionLifecycle(options)`** — Attaches a status observer and retry loop to a shared connection. The first caller claims `retryOwner` and drives reconnect; subsequent callers observe only, preventing concurrent connect races. Returns a `stop()` handle to unsubscribe and release ownership.

**`NatsReconnectionBackoff`** — Configures the retry strategy: optional fast-retry phase (fixed delay) followed by a jittered exponential backoff capped at `maxDelayMs`.

## Usage Example

```typescript
import { acquireClient, releaseClient, startConnectionLifecycle } from './shared-connection'

const conn = acquireClient('wss://nats.example.com')

const lifecycle = startConnectionLifecycle({
  conn,
  wsUrl: 'wss://nats.example.com',
  getFreshUrl: () => 'wss://nats.example.com',
  backoff: { fastRetries: 2, fastRetryDelayMs: 500, maxDelayMs: 15_000 },
  onStatusChange: (status) => console.log('NATS status:', status),
})

// On teardown:
lifecycle.stop()
releaseClient('wss://nats.example.com')
```
