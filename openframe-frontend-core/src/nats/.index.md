Barrel export file that re-exports all public APIs from the NATS messaging module.

## Key Components

| Export Source | Description |
|---|---|
| `./nats` | Core NATS client utilities |
| `./shared-connection` | Shared connection management |
| `./nats-provider` | NATS context provider |
| `./use-nats-subscription` | React hook for NATS subscriptions |

## Usage Example

```typescript
// Import everything from a single entry point
import {
  NatsProvider,
  useNatsSubscription,
  createSharedConnection
} from '@/lib/nats'
```
