Barrel export file that re-exports all public APIs from the NATS messaging module.

## Key Components

| Export Source | Description |
|---|---|
| `./nats` | Core NATS client utilities and types |
| `./shared-connection` | Shared NATS connection management |
| `./nats-provider` | React/Vue provider for NATS context |
| `./use-nats-subscription` | Composable/hook for subscribing to NATS subjects |

## Usage Example

```typescript
// Import anything from the NATS module via this single entry point
import {
  NatsProvider,
  useNatsSubscription,
  SharedConnection,
} from '@flamingo/nats'

// Subscribe to a NATS subject
const { messages } = useNatsSubscription('tickets.>')
```

> All sub-modules are accessible through this single entry point, keeping imports clean across the codebase.
