A browser/WebSocket NATS client factory for the `nats.ws` library, providing typed publish, subscribe, request/reply, and JetStream ordered consumer operations with connection lifecycle management and exponential backoff reconnection.

## Key Components

### Types & Interfaces
- **`NatsClient`** — Main client interface with `connect`, `close`, publish, request, subscribe, and status listener methods
- **`NatsClientOptions`** — Connection config: servers, auth (token/user-pass), reconnect behavior, exponential backoff, ping, and timeout settings
- **`NatsSubscribeOptions`** — Per-subscription options: queue group, max messages, and `AbortSignal`
- **`JetStreamOrderedSubscribeOptions`** — Ephemeral JetStream consumer config with `deliverPolicy` (`'new'` | `'byStartSequence'`), optional start sequence, idle threshold, and `AbortSignal`
- **`NatsStatusEvent`** / **`NatsStatus`** — Connection lifecycle events: `'connecting' | 'connected' | 'disconnected' | 'reconnecting' | 'closed' | 'error'`

### Functions
- **`createNatsClient(options)`** — Factory that returns a `NatsClient` instance; browser/Tauri only (throws if `window` is undefined)
- **`createExponentialBackoffHandler(opts)`** — Internal backoff handler with configurable initial delay, max delay, multiplier, and jitter
- **`toNatsHeaders(nats, init)`** — Normalizes `Record<string, string>` or native `NatsHeaders` into a `nats.ws` headers object

## Usage Example

```typescript
import { createNatsClient } from './nats'

const client = createNatsClient({
  servers: 'wss://nats.example.com:443',
  name: 'my-app',
  token: 'secret',
  exponentialBackoff: {
    initialDelayMs: 500,
    maxDelayMs: 15_000,
    multiplier: 2,
    jitter: true,
  },
})

// Monitor connection status
const off = client.onStatus(({ status, data }) => {
  console.log('NATS status:', status, data)
})

await client.connect()

// Publish & request
client.publishJson('orders.created', { id: 42 })
const response = await client.requestJson<{ ok: boolean }>(
  'orders.validate',
  { id: 42 },
  { timeoutMs: 5000 },
)

// Subscribe
const handle = client.subscribeJson<{ id: number }>(
  'orders.*',
  (payload, msg) => console.log('Received:', payload),
  { queue: 'workers' },
)

// JetStream ordered consumer (live-tail)
const jsHandle = await client.subscribeJetStreamOrdered(
  (msg) => console.log('JS msg seq:', msg.seq),
  { streamName: 'ORDERS', filterSubject: 'orders.>', deliverPolicy: 'new' },
)

// Cleanup
handle.unsubscribe()
jsHandle.unsubscribe()
off()
await client.close()
```
