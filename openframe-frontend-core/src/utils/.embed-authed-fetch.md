Shared `fetch` wrapper for embedded surfaces (chat, ticket center, widgets) that injects bearer-act-as proxy auth headers and supports optional host-supplied auth adapters with 401 self-healing.

## Key Components

### `EmbedAuthAdapter` (interface)
Opt-in adapter contract for hosts with their own auth model. Provides:
- `getHeaders()` — per-request header factory (reactive token refresh aware)
- `credentials` — override `RequestCredentials` mode (e.g. `'include'` for cross-origin cookies)
- `refresh()` — optional 401 self-heal callback; resolves `true` to trigger a single retry

### `setEmbedAuthAdapter(adapter)`
Registers or clears a host-owned auth adapter at module level. Stored on `globalThis` to survive bundler chunk duplication. Warns in dev if overwriting an existing adapter.

### `embedAuthedFetch(url, init?)`
Drop-in `fetch` replacement that:
- Enforces same-origin URLs synchronously (throws before any promise)
- Merges proxy-auth headers via `applyProxyAuth` (proxy creds win over caller headers)
- Applies adapter headers on top (adapter wins over proxy)
- Defaults `Content-Type: application/json` when no headers are supplied
- Triggers deduped 401 refresh-then-retry when an adapter provides `refresh()`

### `dedupedRefresh()`
Internal deduplication guard — collapses a thundering herd of concurrent 401s into a single `adapter.refresh()` call using a `globalThis`-scoped in-flight promise.

## Usage Example

```typescript
// Register a host auth adapter (e.g. in a provider on mount)
import { setEmbedAuthAdapter, embedAuthedFetch } from './embed-authed-fetch'

setEmbedAuthAdapter({
  getHeaders: () => ({
    Authorization: token ? `Bearer ${token}` : undefined,
  }),
  credentials: 'include',
  refresh: async () => {
    const ok = await refreshAccessToken()
    return ok
  },
})

// Use as a drop-in fetch replacement for embedded API surfaces
const response = await embedAuthedFetch('/api/chat/message', {
  method: 'POST',
  body: JSON.stringify({ message: 'Hello' }),
})

// Clear adapter on unmount
setEmbedAuthAdapter(null)
```
