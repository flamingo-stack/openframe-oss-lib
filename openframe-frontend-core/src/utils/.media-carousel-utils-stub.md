A utility module that defines the core data structure and helper functions for handling different types of media items in carousel components.

## Key Components

- **`MediaItem` interface**: Comprehensive type definition for media objects supporting images, videos, and YouTube content with optional metadata like captions, dimensions, and duration
- **`getMediaType()`**: Returns the media type ('image', 'video', or 'youtube') for a given media item
- **`getMediaSrc()`**: Extracts the source URL from a media item
- **`getMediaPoster()`**: Retrieves the poster/thumbnail URL for video content

## Usage Example

```typescript
import { MediaItem, getMediaType, getMediaSrc } from './media-carousel-utils-stub';

const mediaItems: MediaItem[] = [
  {
    id: '1',
    type: 'image',
    src: '/images/photo.jpg',
    alt: 'Sample photo',
    width: 1920,
    height: 1080
  },
  {
    id: '2',
    type: 'video',
    src: '/videos/demo.mp4',
    poster: '/thumbnails/demo.jpg',
    duration: 120
  }
];

// Get media type and source
const firstItem = mediaItems[0];
const type = getMediaType(firstItem); // 'image'
const src = getMediaSrc(firstItem); // '/images/photo.jpg'
```
