Provides utility functions and type definitions for handling media items in a carousel component, supporting images, videos, and YouTube content.

## Key Components

- **`MediaItem` interface**: Core type definition for media objects with properties for different media types including metadata like dimensions, captions, and video-specific fields
- **`getMediaType()`**: Returns the media type of a given item
- **`getMediaSrc()`**: Extracts the source URL from a media item
- **`getMediaPoster()`**: Retrieves the poster/thumbnail image for video content

## Usage Example

```typescript
import { MediaItem, getMediaType, getMediaSrc, getMediaPoster } from './media-carousel-utils-stub';

const mediaItem: MediaItem = {
  id: 'video-1',
  type: 'video',
  src: '/videos/sample.mp4',
  poster: '/images/video-thumbnail.jpg',
  title: 'Sample Video',
  width: 1920,
  height: 1080,
  duration: 120
};

const type = getMediaType(mediaItem); // 'video'
const source = getMediaSrc(mediaItem); // '/videos/sample.mp4'
const poster = getMediaPoster(mediaItem); // '/images/video-thumbnail.jpg'
```
