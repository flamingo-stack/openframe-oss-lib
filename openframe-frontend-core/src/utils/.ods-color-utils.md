A comprehensive utility library for working with ODS (Open Design System) color tokens across multiple platforms. It provides functions for token retrieval, platform theme switching, and runtime color management with support for server-side and client-side environments.

## Key Components

**Types & Interfaces:**
- `Platform`: Platform identifiers (`'openmsp' | 'admin-hub' | 'openframe' | 'flamingo'`)
- `ColorCategory`: Token categories (`'open' | 'flamingo' | 'system' | 'attention'`)
- `ColorVariant`: Color states (`'base' | 'hover' | 'active' | 'focus' | 'disabled'`)

**Core Functions:**
- `getODSToken()`: Retrieves raw token values by category/color/variant
- `getPlatformAccentColor()`: Gets platform-specific accent colors as CSS variables
- `getCurrentPlatform()`: Detects current platform from environment or DOM
- `switchPlatformTheme()`: Changes platform theme and dispatches events
- `tokenToTailwindClass()`: Converts tokens to Tailwind CSS classes
- `usePlatformColors()`: React hook for platform-aware color management

**Utility Functions:**
- `getTokensByCategory()`: Filters tokens by category
- `interpolateColors()`: Creates color transitions between tokens
- `generatePlatformCSS()`: Generates platform-specific CSS custom properties

## Usage Example

```typescript
import { 
  getODSToken, 
  getPlatformAccentColor, 
  usePlatformColors,
  switchPlatformTheme 
} from './ods-color-utils';

// Get specific color token
const primaryColor = getODSToken('open', 'yellow', 'base');

// Get platform accent color
const accentColor = getPlatformAccentColor('flamingo');
// Returns: 'var(--ods-flamingo-pink-base)'

// React component usage
function MyComponent() {
  const { accentColor, getToken, switchTheme } = usePlatformColors();
  
  const handleThemeSwitch = () => {
    switchTheme('admin-hub');
  };
  
  return (
    <button 
      style={{ backgroundColor: accentColor }}
      onClick={handleThemeSwitch}
    >
      Switch Theme
    </button>
  );
}

// Switch platform theme
switchPlatformTheme('openframe');
```
