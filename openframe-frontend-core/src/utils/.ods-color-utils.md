Runtime utility library for working with ODS (OpenFrame Design System) color tokens across Flamingo's multi-platform stack (`openmsp`, `openframe`, `flamingo`).

## Key Components

### Types
- `Platform` — `'openmsp' | 'openframe' | 'flamingo'`
- `ColorCategory` — `'open' | 'flamingo' | 'system' | 'attention'`
- `ColorVariant` — `'base' | 'hover' | 'active' | 'focus' | 'disabled'`

### Token Utilities
- `getODSToken(category, color, variant?)` — Looks up a raw token value by composite key
- `getTokensByCategory(category)` — Returns all tokens matching a category prefix
- `isValidODSToken(tokenKey)` — Validates token existence in the ODS registry
- `tokenToTailwindClass(tokenKey, type?)` — Maps a semantic token to a Tailwind utility class (e.g. `bg-ods-accent`)

### Platform / Theme
- `getCurrentPlatform()` — Resolves active platform from DOM attribute or `NEXT_PUBLIC_APP_TYPE` env
- `getPlatformAccentColor(platform?)` — Returns the CSS variable for the platform's accent color
- `getPlatformConfig(platform?)` — Returns a config object including accent color, brand name, and dark/light flags
- `switchPlatformTheme(platform)` — Updates `data-app-type` on `<html>` and dispatches `platformThemeChanged`
- `generatePlatformCSS(platform)` — Emits a CSS block with platform-scoped custom property overrides
- `getSemanticColor(semanticName, platform?)` — Reads a computed `--color-*` property from a live DOM probe

### Color Math
- `hexToRgb(hex)` / `rgbToHex(r, g, b)` — Hex ↔ RGB conversion with clamping
- `rgbToHsl(r, g, b)` / `hslToRgb(h, s, l)` — RGB ↔ HSL conversion
- `getReadableTextColor(hex)` — Returns `#212121` or `#fafafa` based on Rec. 601 luminance
- `deriveHoverColor(hex)` / `deriveActiveColor(hex)` — Darkens by 10/20 steps to match design-system presets
- `interpolateColors(startToken, endToken, progress)` — Linearly blends two hex token values

### React Integration
- `usePlatformColors(platform?)` — Returns platform config + helper callbacks for use in React components

## Usage Example

```typescript
import {
  getCurrentPlatform,
  getPlatformConfig,
  getODSToken,
  deriveHoverColor,
  usePlatformColors
} from './ods-color-utils';

// Resolve active platform at runtime
const platform = getCurrentPlatform(); // 'openframe'

// Get full platform config
const config = getPlatformConfig();
// { platform: 'openframe', accentColor: 'var(--ods-open-yellow-base)', isDarkTheme: true, brandName: 'OpenFrame' }

// Look up a raw ODS token
const tokenValue = getODSToken('system', 'greys-background', 'base');

// Derive interaction states for a custom hex color
const hoverColor = deriveHoverColor('#3a86ff'); // '#2e7af5' (darkened by 10)

// In a React component
const { accentColor, isDarkTheme, getToken } = usePlatformColors();
```
