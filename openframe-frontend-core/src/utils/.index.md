A central export hub that provides a comprehensive collection of client-side utilities for the Flamingo platform, including formatting functions, platform utilities, validation helpers, and dynamic components.

## Key Components

- **Formatting utilities**: `cn`, `formatDate`, `formatNumber`, `formatPrice`, `formatBytes` for data presentation
- **Platform utilities**: `getPlatformAccentColor`, `getCurrentPlatform` for multi-platform support
- **Common helpers**: `delay`, `generateRandomString`, `truncateString`, `deepClone` for general operations
- **Validation tools**: Complete validation utility suite for data integrity
- **AI confidence helpers**: Utilities for handling AI-generated content confidence scores
- **Date formatters**: Specialized date and time formatting functions including relative time
- **Dynamic icons**: Icon registry system for runtime icon management
- **Access code client**: Client-side access code handling utilities

## Usage Example

```typescript
import { 
  cn, 
  formatPrice, 
  getCurrentPlatform, 
  delay,
  truncateString 
} from '@flamingo/utils'

// Combine CSS classes conditionally
const buttonClass = cn('btn', 'btn-primary', { 'disabled': isLoading })

// Format pricing for display
const price = formatPrice(1299, 'USD') // "$12.99"

// Get platform-specific styling
const platform = getCurrentPlatform()
const accentColor = getPlatformAccentColor(platform)

// Common operations
await delay(1000) // Wait 1 second
const shortText = truncateString(longDescription, 100)
```

This utility collection serves as the foundation for consistent data formatting, platform adaptation, and common operations across the Flamingo ecosystem.
