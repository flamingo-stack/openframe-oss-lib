This file serves as the main entry point for client-side utility functions, providing a centralized export interface for various helper modules used throughout the application.

## Key Components

- **Styling & Formatting**: `cn`, `formatDate`, `formatNumber`, `formatPrice`, `formatBytes` - Common formatting and className utilities
- **Platform Detection**: `getPlatformAccentColor`, `getCurrentPlatform`, `ColorCategory` - Platform-specific styling and detection
- **Common Utilities**: `delay`, `generateRandomString`, `truncateString`, `deepClone`, `getSlackCommunityJoinUrl`, `getBaseUrl` - General purpose helper functions
- **Platform Configuration**: Complete platform config exports
- **OS Platform Utils**: Operating system detection and utilities
- **Access Code Client**: Client-side access code management
- **Validation Utils**: Form and data validation helpers
- **Confidence Helpers**: AI confidence scoring utilities
- **Date Formatters**: Specialized date formatting and relative time utilities

## Usage Example

```typescript
import { 
  cn, 
  formatPrice, 
  getCurrentPlatform, 
  delay,
  truncateString 
} from './index'

// Combine className utilities
const buttonClass = cn('btn', 'btn-primary', { 'btn-disabled': disabled })

// Format pricing
const price = formatPrice(29.99, 'USD')

// Platform detection
const platform = getCurrentPlatform()

// Common utilities
await delay(1000)
const shortText = truncateString('Long text here', 50)
```
