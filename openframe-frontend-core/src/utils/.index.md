Barrel (index) file that re-exports the full public utility API for `@flamingo-stack/openframe-frontend-core/utils`, aggregating client-side helpers, formatters, platform config, chat utilities, and server-safe pure functions into a single import surface.

## Key Components

| Export Group | Source Module | Description |
|---|---|---|
| `cn`, `getAllPlatformBaseDomains` | `./cn` | Class name merging + domain helpers |
| `formatDate`, `formatNumber`, `formatPrice`, `formatBytes` | `./format` | Core number/currency/byte/date formatters |
| `getPlatformAccentColor`, `getCurrentPlatform` | `./ods-color-utils` | ODS design token color utilities |
| `delay`, `generateRandomString`, `truncateString`, `deepClone` | `./common` | General-purpose helpers |
| `getBaseUrl`, `getPlatformProductionUrl` | `../utils/cn` | Environment URL resolvers |
| `getDynamicIcon`, `ICON_REGISTRY`, `getIconComponent` | `../components/chat/utils/icon-registry` | Dynamic icon registry |
| `embedAuthedFetch`, `setEmbedAuthAdapter` | `./embed-authed-fetch` | Embed-surface auth fetch wrapper |
| `getEmbedProxyAuth`, `setEmbedProxyAuth` | `./embed-proxy-auth-storage` | Proxy auth storage (cross-surface) |
| `readLeadingDecisionFrame` | `./sse-decision-frame` | SSE leading-frame reader for confirm-tool route |
| `escapeMarkdownInline`, `buildChatAttachmentViewUrl` | `../components/chat/utils/chat-attachment-markdown` | Chat attachment markdown helpers |
| `scrollElementIntoView` | `./scroll-into-view` | Smooth scroll with sticky-chrome offset |
| `buildListUrl` | `./list-url` | Shared list-API URL builder for RAG mappers |

| `formatLargeNumber`, `formatCurrency`, `formatPercent`, `getTrendColors` | `./format` | Extended metric/trend/text formatters |
| `formatRelativeTime`, `formatAbsoluteDate`, `isToday` | `./date-utils` | Relative/absolute date helpers |
| `SOURCE_ICON_NAMES`, `getSourceLabel` | `./source-icons` | Chat source icons and labels (server-safe) |
| `CUSTOM_ITEM_ID`, `getTaskTypeLabel` | `../components/chat/utils/clickup-task-type-utils` | ClickUp taxonomy constants (server-safe) |

| `DEV_SECTION_PARAM_KEYS` | `./dev-sections/dev-section-param-keys` | Dev-center URL param key constants |

Also re-exports full namespaces: `platform-config`, `os-platforms`, `access-code-client`, `validation-utils`, `confidence-helpers`, `date-formatters`, `release-badge`, `release-cover`, `tool-utils`, `shell-utils`, `os-utils`, `country-phone-utils`, `generic-domain-utils`, `color-analysis`, `dev-sections`.

## Usage Example

```typescript
// Pull any util from the single barrel — no need to chase individual modules
import {
  cn,
  formatCurrency,
  formatRelativeTime,
  getDynamicIcon,
  embedAuthedFetch,
  scrollElementIntoView,
  buildListUrl,
  DEV_SECTION_PARAM_KEYS,
} from '@flamingo-stack/openframe-frontend-core/utils'

// Class merging
const cls = cn('base-class', isActive && 'active')

// Currency formatter
const price = formatCurrency(1999.99) // "$1,999.99"

// Relative date
const label = formatRelativeTime(new Date('2024-01-01')) // "6 months ago"

// Dynamic icon lookup
const Icon = getDynamicIcon('slack', 'md')

// Embed-surface authenticated fetch
const data = await embedAuthedFetch('/api/tickets')

// Smooth scroll with sticky-header offset
scrollElementIntoView(document.getElementById('section-3'), { offsetPx: 64 })

// Build a typed deep-link for the dev center
const url = `/dev?${DEV_SECTION_PARAM_KEYS.search}=flamingo`
```

> **Note:** Server-safe exports (source icons, ClickUp taxonomy, content-href, list-url) are explicitly isolated so Next.js server components can import from this barrel without crossing a `"use client"` boundary.
