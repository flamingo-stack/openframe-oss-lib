Barrel export index for the `utils` package, re-exporting all utility functions, types, and constants from their canonical source files to provide a single stable import path for consumers across the frontend library and hub.

## Key Components

| Export Group | Source | Description |
|---|---|---|
| `cn`, `getAllPlatformBaseDomains`, `getBaseUrl`, `getPlatformProductionUrl` | `./cn` | CSS class merging and platform URL helpers |
| `formatDate`, `formatNumber`, `formatPrice`, `formatBytes`, `formatCurrency`, `formatPercent`, etc. | `./format` | Number, currency, byte, date, and metric formatters |
| `getPlatformAccentColor`, `getCurrentPlatform` | `./ods-color-utils` | Platform theming and color utilities |
| `getDynamicIcon`, `ICON_REGISTRY`, `getIconComponent` | `chat/utils/icon-registry` | Dynamic icon registry |
| `embedAuthedFetch`, `getEmbedProxyAuth`, `setEmbedProxyAuth` | `./embed-*` | Embed-surface authentication helpers |
| `formatRelativeTime`, `formatAbsoluteDate`, `isToday`, `createUTCTimestamp` | `./date-utils` | Date/time formatting and comparison |
| `escapeMarkdownInline`, `buildChatAttachmentViewUrl` | `chat/utils/chat-attachment-markdown` | Chat attachment markdown helpers |
| `scrollElementIntoView` | `./scroll-into-view` | Smooth scroll with sticky-chrome offset |
| `readLeadingDecisionFrame` | `./sse-decision-frame` | SSE frame reader for confirm-tool route |
| `isCrossOriginUrl` | `chat/utils/is-cross-origin-url` | Cross-origin URL detection (server-safe) |

## Usage Example

```typescript
// Import any utility directly from the utils barrel
import {
  cn,
  formatCurrency,
  formatRelativeTime,
  getDynamicIcon,
  getEmbedProxyAuth,
  scrollElementIntoView,
  getCurrentPlatform,
} from '@flamingo-stack/openframe-frontend-core/utils'

// CSS class merging
const className = cn('base-class', isActive && 'active')

// Format a currency value
const price = formatCurrency(4999, 'USD') // "$49.99"

// Relative time display
const label = formatRelativeTime(new Date('2024-01-01')) // "6 months ago"

// Retrieve embed proxy auth for non-chat surfaces
const auth = getEmbedProxyAuth()

// Scroll an element into view respecting sticky chrome
scrollElementIntoView(targetElement, { offsetPx: 64 })
```

> **Note:** Several exports are intentionally re-routed through this barrel to keep server-safe utilities (e.g. `CUSTOM_ITEM_ID`, `getSourceLabel`, `isCrossOriginUrl`) accessible from SSR/server code without crossing `"use client"` boundaries.
