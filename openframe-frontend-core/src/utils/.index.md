A centralized export hub for client-side utility functions, providing a clean API surface for commonly used helpers across styling, platform detection, formatting, and validation.

## Key Components

- **Styling utilities**: `cn` for className merging, various formatting functions (`formatDate`, `formatNumber`, `formatPrice`, `formatBytes`)
- **Platform detection**: `getPlatformAccentColor`, `getCurrentPlatform` with `ColorCategory` type
- **Common helpers**: `delay`, `generateRandomString`, `truncateString`, `deepClone`, URL generators
- **Configuration modules**: Platform config, OS platforms, access code client
- **Validation utilities**: Form and data validation helpers
- **AI confidence helpers**: Utilities for handling AI response confidence levels
- **Date/time formatting**: Release date formatting and relative time utilities

## Usage Example

```typescript

  cn, 
  formatPrice, 
  getCurrentPlatform,
```
