Barrel export module for `@flamingo-stack/openframe-frontend-core/utils`, consolidating all utility functions, formatters, and helpers into a single import surface for both client and server consumers.

## Key Components

| Export Group | Source Module | Description |
|---|---|---|
| `cn`, `getAllPlatformBaseDomains` | `./cn` | CSS class merging + platform domain helpers |
| `formatDate`, `formatNumber`, `formatPrice`, `formatBytes` | `./format` | Core number/currency/date formatters |
| `getPlatformAccentColor`, `getCurrentPlatform` | `./ods-color-utils` | Platform accent color + ODS color utilities |
| `delay`, `generateRandomString`, `truncateString` | `./common` | General-purpose helpers |
| `getBaseUrl`, `getPlatformProductionUrl` | `./cn` | URL builders |
| `getDynamicIcon`, `ICON_REGISTRY`, `getIconComponent` | `../components/chat/utils/icon-registry` | Dynamic icon registry |
| `formatLargeNumber`, `formatCurrency`, `formatPercent` | `./format` | Extended numeric/text formatters |
| `formatRelativeTime`, `formatAbsoluteDate`, `isToday` | `./date-utils` | Date/time helpers |
| `embedAuthedFetch`, `setEmbedAuthAdapter` | `./embed-authed-fetch` | Embed surface authentication |
| `getEmbedProxyAuth`, `setEmbedProxyAuth` | `./embed-proxy-auth-storage` | Proxy auth storage |
| `readLeadingDecisionFrame` | `./sse-decision-frame` | SSE frame reader for confirm-tool route |

## Usage Example

```typescript
// Named imports from the utils barrel
import {
  cn,
  formatCurrency,
  formatRelativeTime,
  getCurrentPlatform,
  getDynamicIcon,
  embedAuthedFetch,
  scrollElementIntoView,
  buildListUrl,
} from '@flamingo-stack/openframe-frontend-core/utils'

// CSS class merging
const className = cn('base-class', isActive && 'active', variant === 'primary' && 'btn-primary')

// Currency formatting
const price = formatCurrency(1999.99) // "$1,999.99"

// Relative time
const ago = formatRelativeTime(new Date('2024-01-01')) // "3 months ago"

// Dynamic icon lookup
const IconComponent = getDynamicIcon('slack', 'md')

// Embed-surface authenticated fetch (replaces chat-prefixed equivalent)
const data = await embedAuthedFetch('/api/tickets')

// Smooth scroll with sticky-chrome offset
scrollElementIntoView(document.getElementById('section'), { offset: 64 })

// List API URL for RAG mapper
const url = buildListUrl('vendor', ['id-1', 'id-2'], '/content')
```

> **Note:** Modules sourced from `../components/chat/utils/` are re-exported here specifically to avoid the `'use client'` boundary — server-side hub callers (e.g. `doc-chat-utils`, `rag-mappers`) should import exclusively from this barrel rather than the chat component barrel.
