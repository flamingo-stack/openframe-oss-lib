Central utility exports module providing client-side helper functions and type definitions for the Flamingo platform. This barrel export file aggregates commonly used utilities across different categories.

## Key Components

### **Core Utilities**
- `cn` - Class name utility for conditional styling
- `formatDate`, `formatNumber`, `formatPrice`, `formatBytes` - Data formatting functions
- `getAllPlatformBaseDomains` - Platform domain configuration helper
- `getBaseUrl` - Base URL resolution utility

### **Platform & Color Management**
- `getPlatformAccentColor`, `getCurrentPlatform` - Platform detection and theming
- `ColorCategory` - TypeScript type for color categorization
- Platform configuration exports from `./platform-config`
- OS platform utilities from `./os-platforms`

### **Helper Functions**
- `delay` - Async delay utility
- `generateRandomString`, `truncateString` - String manipulation
- `deepClone` - Object cloning utility
- `getSlackCommunityJoinUrl` - Community integration helper

### **Specialized Modules**
- Access code client utilities
- Validation helpers
- AI confidence scoring
- Date/time formatting
- Dynamic icon registry
- Tool, shell, and OS type utilities

## Usage Example

```typescript
import { 
  cn, 
  formatPrice, 
  getPlatformAccentColor, 
  delay 
} from '@/utils'

// Conditional styling
const buttonClass = cn('btn', {
  'btn-primary': isPrimary,
  'btn-disabled': isDisabled
})

// Format currency
const price = formatPrice(29.99, 'USD')

// Get platform theme color
const accentColor = getPlatformAccentColor('flamingo')

// Async delay
await delay(1000)
```
