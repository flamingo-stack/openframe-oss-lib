A utility library for handling vendor media assets (logos, images, videos) with support for multiple data source formats and URL fixing for Supabase storage.

## Key Components

- **`VendorMedia`** - Interface for individual media items with type and URL
- **`VendorWithMedia`** - Interface for vendor objects with media arrays and legacy logo fields
- **`getVendorLogo()`** - Retrieves vendor logo with fallback priority (logo_url → logo → vendor_media)
- **`getVendorImage()`** / **`getVendorVideo()`** - Get specific media types from vendor_media array
- **`getVendorMediaByType()`** - Get all media URLs for a specific type
- **`getVendorMediaGrouped()`** - Returns all media organized by type
- **`addVendorMedia()`** / **`removeVendorMedia()`** - Manage media arrays for admin interfaces
- **`hasVendorMedia()`** / **`getVendorMediaCount()`** - Check media presence and count

## Usage Example

```typescript
import { getVendorLogo, getVendorMediaGrouped, VendorWithMedia } from './vendor-media-stub';

const vendor: VendorWithMedia = {
  id: 1,
  title: "Acme Corp",
  logo_url: "https://storage.supabase.co/logo.png",
  vendor_media: [
    { media_type: 'image', media_url: 'https://storage.supabase.co/banner.jpg' },
    { media_type: 'video', media_url: 'https://storage.supabase.co/demo.mp4' }
  ]
};

// Get logo with automatic fallback handling
const logoUrl = getVendorLogo(vendor);

// Get all media organized by type
const { logos, images, videos } = getVendorMediaGrouped(vendor);
```
