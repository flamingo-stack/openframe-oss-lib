Provides unified date formatting utilities for consistent timestamp display across the OpenMSP platform. Serves as the single source of truth for all date-related formatting operations.

## Key Components

- **`formatRelativeTime()`** - Converts timestamps to relative format (e.g., "2h ago", "3d ago")
- **`formatAbsoluteDate()`** - Formats dates in standard absolute format
- **`formatDateTime()`** - Formats timestamps with both date and time
- **`getDetailedTimeDifference()`** - Returns verbose time differences
- **`isToday()`** - Checks if timestamp is from today
- **`isWithinMinutes()`** - Validates if timestamp is within specified time window
- **`createUTCTimestamp()`** - Generates ISO UTC timestamp strings

## Usage Example

```typescript
import { 
  formatRelativeTime, 
  formatAbsoluteDate, 
  isWithinMinutes,
  createUTCTimestamp 
} from './date-utils';

// Display relative times
const relativeTime = formatRelativeTime('2024-01-01T12:00:00Z'); // "2h ago"

// Format absolute dates
const absoluteDate = formatAbsoluteDate(new Date(), { 
  weekday: 'short' 
}); // "Mon, Jan 15, 2024"
```
