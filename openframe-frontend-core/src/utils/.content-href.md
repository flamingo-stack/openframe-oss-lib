Unified content-href authority for all embeddable surfaces — resolves content links to in-app routes or external hub URLs from a single configuration, ensuring consistent navigation behavior across page views and chat cards.

## Key Components

**`DEFAULT_CONTENT_SUFFIXES`** — Map of content document types to URL path segments (e.g. `product_release → releases`). Must stay manually in sync with the hub's `PUBLIC_URL_PATHS`.

**`ComposeContentUrlInput`** — Input interface accepted by all callers. Page views pass `type` + `identifier` (slug); chat rows additionally pass `externalUrl` (canonical hub URL) and `targetPlatform`.

**`ContentHrefOptions`** — Configuration for `makeComposeContentUrl`: `hostedTypes` (in-app types), `contentOrigin` (hub origin fallback), optional `suffixes` override, and per-type `overrides`.

**`ComposeContentUrl`** — Seam type on `ChatRuntime`. Always returns `{ href, targetPlatform }`, never null.

**`makeComposeContentUrl(opts)`** — Builds the embedder's composer. Resolution order:
1. Per-type `overrides` entry (wins all)
2. `hostedTypes` member → relative `/<suffix>/<slug>` (soft-nav)
3. `externalUrl` present → verbatim (RAG-authoritative)
4. Fallback → `${contentOrigin}/<suffix>/<identifier>`

**`buildDefaultHref(basePath, slug)`** — Same-origin fallback when no composer is wired.

**`resolveContentHref(composeContentUrl, args)`** — Centralizes the ternary used by every catalog/detail view: delegates to the composer when available, otherwise calls `buildDefaultHref`.

## Usage Example

```typescript
// Wire the embedder's composer (e.g. in ChatRuntime config)
const composeContentUrl = makeComposeContentUrl({
  hostedTypes: new Set(['onboarding_guide', 'product_release']),
  contentOrigin: 'https://openframe.app',
  overrides: {
    blog_post: (id) => ({ href: `/news/${id}`, targetPlatform: null }),
  },
})

// Page view (slug as identifier)
composeContentUrl({ type: 'product_release', identifier: 'v2-launch' })
// → { href: '/releases/v2-launch', targetPlatform: null }

// Chat row (id as identifier, hub URL for slug recovery)
composeContentUrl({
  type: 'product_release',
  identifier: '42',
  externalUrl: 'https://openframe.app/releases/v2-launch',
})
// → { href: '/releases/v2-launch', targetPlatform: null }

// Non-hosted type with external URL
composeContentUrl({
  type: 'blog_post',
  identifier: '99',
  externalUrl: 'https://openframe.app/blog/my-post',
  targetPlatform: 'openframe',
})
// → { href: 'https://openframe.app/blog/my-post', targetPlatform: 'openframe' }

// Catalog/detail view helper
resolveContentHref(composeContentUrl, {
  type: 'onboarding_guide',
  slug: 'getting-started',
  basePath: '/onboarding-guides',
})
// → { href: '/onboarding-guides/getting-started', targetPlatform: null }
```
