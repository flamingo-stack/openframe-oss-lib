Pure utility module for building image proxy URLs and related image-loading helpers, with no side effects or environment dependencies.

## Key Components

| Export | Type | Description |
|--------|------|-------------|
| `GetProxiedImageUrlOptions` | Type | Configuration shape for proxy prefix, skip domains, and direct-HTTPS bypass |
| `getProxiedImageUrl` | Function | Core URL resolver â€” routes an image URL through the proxy or returns it unchanged based on resolution rules |
| `urlPathLooksLikeSvg` | Function | Heuristic check for `.svg` file extension in a URL path |
| `shouldProxyImage` | Function | Boolean guard indicating whether a URL requires proxying |
| `generateImageSizes` | Function | Returns a standard responsive `sizes` attribute string for `<img>` tags |

## Resolution Order (`getProxiedImageUrl`)

```mermaid
graph TD
    A["imageUrl input"] --> B{"Already contains proxyPrefix?"}
    B -- Yes --> C["Return unchanged (self-skip)"]
    B -- No --> D{"Relative / data: / blob:?"}
    D -- Yes --> E["Return unchanged"]
    D -- No --> F{"directHttps + starts with https://?"}
    F -- Yes --> G["Return unchanged"]
    F -- No --> H{"proxyPrefix unset?"}
    H -- Yes --> I["Return unchanged"]
    H -- No --> J{"Matches skipDomains?"}
    J -- Yes --> K["Return unchanged"]
    J -- No --> L["Return proxyPrefix?url=encoded"]
```

## Usage Example

```typescript
import {
  getProxiedImageUrl,
  urlPathLooksLikeSvg,
  shouldProxyImage,
} from './image-proxy';

const proxyPrefix = '/api/image-proxy';
const skipDomains = ['cdn.mycompany.com'];

// Standard proxy wrap
getProxiedImageUrl('https://external.com/photo.jpg', { proxyPrefix, skipDomains });
// â†’ '/api/image-proxy?url=https%3A%2F%2Fexternal.com%2Fphoto.jpg'

// Skip proxy for own CDN
getProxiedImageUrl('https://cdn.mycompany.com/logo.png', { proxyPrefix, skipDomains });
// â†’ 'https://cdn.mycompany.com/logo.png'

// Bypass proxy for SVGs only
const url = 'https://external.com/icon.svg';
getProxiedImageUrl(url, {
  proxyPrefix,
  directHttps: urlPathLooksLikeSvg(url), // true â†’ bypass
});
// â†’ 'https://external.com/icon.svg'

// Guard before proxying
shouldProxyImage('https://external.com/img.png', proxyPrefix); // â†’ true
shouldProxyImage('/relative/path.png', proxyPrefix);            // â†’ false
```

## Notes

- **Caller-owned config** â€” no env reads; proxy prefix and skip list are passed in at the call site, typically from `ChatRuntime.endpoints.imageProxyUrlPrefix`.
- **HTTP stays proxied** even when `directHttps: true` to avoid mixed-content issues.
- Relative URLs, `data:`, and `blob:` URIs always pass through unchanged.
