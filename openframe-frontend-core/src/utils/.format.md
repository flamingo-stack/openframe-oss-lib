A comprehensive utility module providing formatting functions for dates, numbers, prices, file sizes, and abbreviated number representations with internationalization support.

## Key Components

- **`formatDate()`** - Converts Date objects or ISO strings to localized date strings with customizable formatting options
- **`formatNumber()`** - Adds thousands separators to numbers using locale-specific formatting
- **`formatPrice()`** - Formats numbers as currency with symbol and proper decimal places
- **`formatBytes()`** - Converts byte values to human-readable file sizes (KB, MB, GB, etc.)
- **`formatLargeNumber()`** - Abbreviates large numbers to K/M/B format with no decimals
- **`formatAbbreviatedNumber()`** - Similar to formatLargeNumber but preserves one decimal place when needed

## Usage Example

```typescript
import { 
  formatDate, 
  formatPrice, 
  formatBytes, 
  formatAbbreviatedNumber 
} from './format';

// Date formatting
formatDate(new Date(), { month: 'short', day: 'numeric' }); // "Dec 25"

// Currency formatting
formatPrice(1299.99, 'EUR'); // "â‚¬1,299.99"

// File size formatting
formatBytes(1048576); // "1 MB"

// Number abbreviation
formatAbbreviatedNumber(1200); // "1.2K"
formatAbbreviatedNumber(15000); // "15K"
formatAbbreviatedNumber(2000000); // "2M"
```

All functions include input validation and handle edge cases like invalid dates and zero values gracefully.
