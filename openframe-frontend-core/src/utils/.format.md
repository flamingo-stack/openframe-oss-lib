This module provides a comprehensive set of utility functions for formatting various types of data into human-readable strings, including dates, numbers, currency, file sizes, and abbreviated large numbers.

## Key Components

- **`formatDate`** - Converts Date objects or ISO strings to localized date strings with customizable options
- **`formatNumber`** - Adds thousands separators to numbers using locale-specific formatting
- **`formatPrice`** - Formats numbers as currency with proper symbols and localization
- **`formatBytes`** - Converts byte values to human-readable file sizes (KB, MB, GB, etc.)
- **`formatLargeNumber`** - Abbreviates large numbers to compact form (K, M, B) without decimals
- **`formatAbbreviatedNumber`** - Similar to `formatLargeNumber` but includes decimal precision for better readability

## Usage Example

```typescript
import { 
  formatDate, 
  formatNumber, 
  formatPrice, 
  formatBytes, 
  formatLargeNumber,
  formatAbbreviatedNumber 
} from './format'

// Date formatting
const date = new Date('2024-01-15')
console.log(formatDate(date)) // "January 15, 2024"
console.log(formatDate(date, { month: 'short', day: 'numeric' })) // "Jan 15"

// Number formatting
console.log(formatNumber(1234567)) // "1,234,567"
console.log(formatPrice(29.99, 'USD')) // "$29.99"
console.log(formatBytes(1024000)) // "1000 KB"

// Large number abbreviation
console.log(formatLargeNumber(1500)) // "1K"
console.log(formatAbbreviatedNumber(1500)) // "1.5K"
```
