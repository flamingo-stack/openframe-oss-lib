A utility function that converts timestamps into human-readable relative time strings, supporting both past and future dates with proper pluralization and edge case handling.

## Key Components

- **`formatRelativeTime(timestamp: string | number): string`** - Main function that accepts ISO date strings or millisecond timestamps and returns formatted relative time strings

## Usage Example

```typescript
import { formatRelativeTime } from './format-relative-time';

// Past timestamps
formatRelativeTime('2024-01-01T00:00:00Z'); // "5 days ago"
formatRelativeTime(Date.now() - 3600000);   // "1 hour ago"

// Future timestamps  
formatRelativeTime(Date.now() + 86400000);  // "in 1 day"
formatRelativeTime(Date.now() + 1800000);   // "in 30 minutes"

// Edge cases
formatRelativeTime('');                      // "Not scheduled"
formatRelativeTime('invalid-date');         // "Invalid date"
formatRelativeTime(Date.now());             // "just now"

// Practical usage in components
const lastRun = formatRelativeTime(job.lastExecuted);
const nextRun = formatRelativeTime(job.nextScheduled);
```

The function automatically handles pluralization, validates input dates, and provides meaningful fallbacks for empty or invalid timestamps. Perfect for displaying job schedules, activity feeds, or any time-sensitive data in user interfaces.
