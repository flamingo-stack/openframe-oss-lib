Defines operating system platform configurations with associated metadata including names and icons for Windows, Linux, and macOS platforms.

## Key Components

- **`OSPlatformId`** - Union type for supported OS platforms (`'windows'`, `'linux'`, `'darwin'`)
- **`OSPlatformOption`** - Interface defining platform structure with id, display name, and icon component
- **`OS_PLATFORMS`** - Array of all supported platform configurations
- **`DEFAULT_OS_PLATFORM`** - Default platform constant set to Windows

## Usage Example

```typescript
import { OS_PLATFORMS, DEFAULT_OS_PLATFORM, type OSPlatformId } from './os-platforms';

// Render platform selector
const PlatformSelector = () => (
  <div>
    {OS_PLATFORMS.map(platform => (
      <button key={platform.id}>
        <platform.icon />
        {platform.name}
      </button>
    ))}
  </div>
);

// Get platform by ID
const findPlatform = (id: OSPlatformId) => 
  OS_PLATFORMS.find(p => p.id === id);

// Use default platform
const currentPlatform = DEFAULT_OS_PLATFORM; // 'windows'
```
