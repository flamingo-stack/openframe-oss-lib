A utility module providing common helper functions for class name merging, data formatting, and URL generation in a Next.js/React application.

## Key Components

- **`cn()`** - Combines and merges CSS class names using clsx and tailwind-merge
- **`formatDate()`** - Converts dates to human-readable strings with localization support
- **`formatNumber()`** - Formats numbers with thousands separators
- **`formatPrice()`** - Formats currency values with proper symbols and locale
- **`formatBytes()`** - Converts byte values to human-readable file sizes (KB, MB, GB, etc.)
- **`getBaseUrl()`** - Determines the correct base URL for different deployment environments

## Usage Example

```typescript
import { cn, formatDate, formatPrice, formatBytes, getBaseUrl } from './cn'

// Combine CSS classes with Tailwind conflict resolution
const buttonClass = cn(
  'px-4 py-2 bg-blue-500',
  isActive && 'bg-blue-700',
  'hover:bg-blue-600'
)

// Format various data types
const formatted = {
  date: formatDate(new Date(), { month: 'short', day: 'numeric' }),
  price: formatPrice(1299.99), // "$1,299.99"
  fileSize: formatBytes(1048576), // "1.00 MB"
  baseUrl: getBaseUrl() // Environment-aware URL
}
```

The module handles edge cases like invalid dates and provides environment-aware URL generation for Vercel deployments with production domain fallbacks.
