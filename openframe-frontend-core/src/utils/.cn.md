A utility module providing common helper functions for class name management, data formatting, and environment configuration.

## Key Components

- **`cn(...inputs)`** - Combines and merges CSS class names using clsx and tailwind-merge
- **`formatDate(date, options?)`** - Converts dates to localized strings with validation
- **`formatNumber(num)`** - Adds thousands separators to numbers
- **`formatPrice(price, currency?)`** - Formats currency values with proper symbols
- **`formatBytes(bytes, decimals?)`** - Converts byte values to human-readable units (KB, MB, etc.)
- **`getBaseUrl()`** - Returns environment-appropriate base URL for the application

## Usage Example

```typescript
import { cn, formatDate, formatPrice, formatBytes, getBaseUrl } from './cn'

// Combine CSS classes with Tailwind merge
const buttonStyles = cn(
  'px-4 py-2 rounded',
  isActive && 'bg-blue-500',
  'bg-blue-600' // This will override bg-blue-500 when merged
)

// Format various data types
const date = formatDate(new Date(), { month: 'short', day: 'numeric' })
const price = formatPrice(29.99) // "$29.99"
const fileSize = formatBytes(1536) // "1.5 KB"
const apiUrl = `${getBaseUrl()}/api/users`
```
