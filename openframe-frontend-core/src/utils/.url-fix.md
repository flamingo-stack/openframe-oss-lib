Provides utility functions for cleaning and properly constructing URLs, with special handling for Supabase storage URLs to prevent double slash issues that can cause SEO problems.

## Key Components

- **`fixUrlDoubleSlashes(url: string)`** - Removes consecutive slashes from URLs while preserving protocol slashes (https://)
- **`joinUrlPath(...segments: string[])`** - Safely joins URL path segments without creating double slashes
- **`fixSupabaseStorageUrl(url: string)`** - Specifically fixes Supabase storage URLs that contain `/storage/v1/object/public/` path segments

## Usage Example

```typescript
import { fixUrlDoubleSlashes, joinUrlPath, fixSupabaseStorageUrl } from './url-fix';

// Fix double slashes in URLs
const cleanUrl = fixUrlDoubleSlashes('https://example.com//api//users');
// Returns: 'https://example.com/api/users'

// Join path segments safely
const apiPath = joinUrlPath('/api/', '/users/', 'profile');
// Returns: 'api/users/profile'

// Fix Supabase storage URLs
const storageUrl = fixSupabaseStorageUrl(
  'https://app.example.com/storage/v1/object/public/logos///image.png'
);
// Returns: 'https://app.example.com/storage/v1/object/public/logos/image.png'
```

These utilities help prevent URL construction issues that can lead to 404 errors and SEO warnings from search engines.
