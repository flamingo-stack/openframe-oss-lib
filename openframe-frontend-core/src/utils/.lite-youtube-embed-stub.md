A lightweight React component that provides a simple YouTube embed wrapper using an iframe for displaying YouTube videos.

## Key Components

- **`LiteYouTubeEmbedProps`** - Interface defining component props including video ID, title, poster image, and CSS class
- **`LiteYouTubeEmbed`** - React functional component that renders a YouTube video embed with configurable properties

## Usage Example

```typescript
import { LiteYouTubeEmbed } from './lite-youtube-embed-stub';

function VideoPlayer() {
  return (
    <div>
      <LiteYouTubeEmbed 
        id="dQw4w9WgXcQ" 
        title="Rick Astley - Never Gonna Give You Up"
        className="video-container"
      />
      
      {/* With custom poster */}
      <LiteYouTubeEmbed 
        id="jNQXAC9IVRw"
        title="Me at the zoo"
        poster="custom-thumbnail.jpg"
        className="featured-video"
      />
    </div>
  );
}
```

The component automatically generates the YouTube embed URL using the provided video ID and includes standard iframe permissions for media playback features like autoplay, fullscreen, and gyroscope access.
