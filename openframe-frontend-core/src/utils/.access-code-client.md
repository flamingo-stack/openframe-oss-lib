Client-side utilities for integrating access code validation and consumption into registration forms. Provides functions to validate codes, consume them after successful registration, and a React hook for seamless integration.

## Key Components

- **`validateAccessCode(email, code)`** - Validates an access code for a given email address
- **`consumeAccessCode(email, code)`** - Marks an access code as used after successful registration
- **`validateAndConsumeAccessCode(email, code)`** - Convenience function that validates then consumes in one step
- **`useAccessCodeIntegration()`** - React hook providing validation/consumption with loading states

## Usage Example

```typescript
import { validateAccessCode, consumeAccessCode, useAccessCodeIntegration } from './access-code-client';

// Basic validation
const validation = await validateAccessCode('user@example.com', 'ABC123XY');
if (validation.valid) {
  console.log(`Welcome to ${validation.cohort_name}!`);
}

// Consume after registration
const consumption = await consumeAccessCode('user@example.com', 'ABC123XY');

// React hook usage
function RegistrationForm() {
  const { validate, consume, isValidating } = useAccessCodeIntegration();

    const result = await validate(formData.email, formData.code);

}
```
