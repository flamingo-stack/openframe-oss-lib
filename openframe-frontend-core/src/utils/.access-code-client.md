Client-side utilities for integrating access code validation and consumption into registration forms, providing both standalone functions and a React hook for seamless user registration workflows.

## Key Components

- **validateAccessCode** - Validates an access code against a user's email
- **consumeAccessCode** - Marks an access code as used after successful registration
- **validateAndConsumeAccessCode** - Convenience function combining validation and consumption
- **useAccessCodeIntegration** - React hook with loading states for form integration

## Usage Example

```typescript
import { validateAccessCode, consumeAccessCode, useAccessCodeIntegration } from './access-code-client';

// Standalone validation
const validation = await validateAccessCode('user@example.com', 'ABC123XY');
if (validation.valid) {
  console.log(`Welcome to ${validation.cohort_name}!`);
  
  // After successful registration, consume the code
  await consumeAccessCode('user@example.com', 'ABC123XY');
}

// React component integration
```
