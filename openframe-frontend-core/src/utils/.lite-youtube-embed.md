A lightweight React component that provides lazy loading for YouTube videos by showing a thumbnail placeholder before loading the actual iframe embed.

## Key Components

- **LiteYouTubeEmbed**: Main component that renders either a clickable thumbnail or YouTube iframe
- **LiteYouTubeEmbedProps**: Interface defining component props including video ID, title, custom thumbnail, and CSS classes
- **State Management**: Uses React.useState to track loading state and conditionally render content

## Usage Example

```typescript
import { LiteYouTubeEmbed } from './lite-youtube-embed';

// Basic usage with video ID
<LiteYouTubeEmbed 
  id="dQw4w9WgXcQ" 
  title="Never Gonna Give You Up"
/>

// With custom thumbnail and styling
<LiteYouTubeEmbed 
  id="dQw4w9WgXcQ"
  title="Custom Video Title"
  thumbnail="https://example.com/custom-thumb.jpg"
  className="w-full h-64 rounded-lg"
/>
```

The component initially displays a thumbnail image with a play button overlay. When clicked, it replaces the placeholder with a full YouTube iframe embed, improving page load performance by deferring the heavy iframe until user interaction.
