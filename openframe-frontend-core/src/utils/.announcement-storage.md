A browser-safe utility module for managing announcement data persistence using localStorage with SSR compatibility and graceful error handling.

## Key Components

- **`AnnouncementStorageOptions`** - Interface defining storage configuration with key and optional default value
- **`getStoredAnnouncement(key: string)`** - Retrieves and parses stored announcement data from localStorage
- **`setStoredAnnouncement(key: string, value: any)`** - Serializes and stores announcement data to localStorage
- **`clearStoredAnnouncement(key?: string)`** - Removes stored announcement data (defaults to 'announcement' key)

## Usage Example

```typescript
import { 
  getStoredAnnouncement, 
  setStoredAnnouncement, 
  clearStoredAnnouncement 
} from './announcement-storage';

// Store announcement data
const announcementData = { 
  message: "New feature available!", 
  dismissed: false 
};
setStoredAnnouncement('feature-announcement', announcementData);

// Retrieve announcement data
const stored = getStoredAnnouncement('feature-announcement');
if (stored && !stored.dismissed) {
  showAnnouncement(stored.message);
}

// Clear announcement
clearStoredAnnouncement('feature-announcement');
```

All functions include window checks for SSR compatibility and silent error handling for storage failures.
