Centralized configuration module for managing platform-specific branding, styling, and display properties across the MSP platform ecosystem. Provides consistent icons, colors, names, descriptions, and utility functions for platform identification and UI rendering.

## Key Components

- **Platform Data Objects**: Static mappings for icons, colors, display names, descriptions, slogans, and default configurations
- **Utility Functions**: Helper functions for retrieving platform-specific properties with fallbacks
- **Icon Components**: Specialized icon rendering functions for different UI contexts (small filters, admin panels)
- **Transform Functions**: Converters for adapting platform configs to UI component requirements

## Usage Example

```typescript
import { 
  getPlatformIcon, 
  getPlatformColor, 
  getPlatformDisplayName,
  transformPlatformConfigsToOptions,
  getSmallPlatformIcon 
} from './platform-config';

// Get platform branding elements
const icon = getPlatformIcon('flamingo');
const color = getPlatformColor('openframe');
const displayName = getPlatformDisplayName('openmsp');
```
