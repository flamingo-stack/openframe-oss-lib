This file provides centralized configuration and utilities for managing platform-specific branding elements including icons, colors, names, and descriptions across a multi-platform React application.

## Key Components

**Configuration Objects:**
- `platformIcons` - React components for each platform's icon
- `platformColors` - Tailwind CSS background color classes
- `platformDisplayNames` - Human-readable platform names
- `platformDescriptions` - Marketing descriptions for each platform
- `platformSlogans` - Platform taglines
- `platformHexColors` - Hex color codes for default configurations
- `platformIconNames` - Icon identifiers for configuration

**Utility Functions:**
- `getDefaultColorForPlatform()` - Retrieves default hex color
- `getDefaultIconForPlatform()` - Retrieves default icon name
- `transformPlatformConfigsToOptions()` - Converts configs to UI options
- `getPlatform*()` - Getter functions for various platform properties
- `getSmallPlatformIcon()` - Returns 4x4 icons with white colors
- `getPlatformIconComponent()` - Returns 6x6 icons for admin interfaces

## Usage Example

import { 
  getPlatformIcon, 
  getPlatformDisplayName,
  transformPlatformConfigsToOptions 
} from './platform-config';

// Get platform branding elements
