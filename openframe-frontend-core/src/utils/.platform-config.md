This file provides centralized platform configuration and utilities for managing multiple brand platforms within the Flamingo ecosystem, including OpenMSP, OpenFrame, Flamingo, and TMCG with consistent theming and branding.

## Key Components

### Configuration Objects
- **`platformIcons`** - React icon components for each platform with consistent styling
- **`platformColors`** - Tailwind CSS background color classes for platform theming
- **`platformDisplayNames`** - Human-readable names for UI display
- **`platformDescriptions`** - Detailed platform descriptions for marketing content
- **`platformSlogans`** - Brand slogans for each platform
- **`platformHexColors`** - Hex color codes for default configurations
- **`platformIconNames`** - Icon identifier strings for database storage

### Utility Functions
- **`transformPlatformConfigsToOptions()`** - Converts database platform configs to UI-compatible options
- **`getPlatform*()` functions** - Getter functions for platform properties with fallbacks
- **`getSmallPlatformIcon()`** - Returns 4x4 icons optimized for filter buttons
- **`getPlatformIconComponent()`** - Returns 6x6 icons for admin/selector components
- **`getDefaultColorForPlatform()`** - Returns default hex colors for platform initialization
