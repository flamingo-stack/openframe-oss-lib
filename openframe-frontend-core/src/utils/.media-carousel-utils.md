A utility module for handling media carousel functionality, providing type definitions and helper functions for processing media items in carousel components.

## Key Components

- **`MediaItem`** - Interface defining the structure of media items with support for images, videos, and embeds
- **`processMediaItems()`** - Function that normalizes raw media data into standardized MediaItem objects
- **`getMediaThumbnail()`** - Function that returns the thumbnail URL for a media item

## Usage Example

```typescript
import { MediaItem, processMediaItems, getMediaThumbnail } from './media-carousel-utils';

// Process raw media data
const rawItems = [
  { url: 'image1.jpg', alt: 'Photo 1' },
  { id: 'vid1', type: 'video', url: 'video.mp4', caption: 'Demo video' }
];

const mediaItems: MediaItem[] = processMediaItems(rawItems);

// Get thumbnail for carousel display
mediaItems.forEach(item => {
  const thumbnail = getMediaThumbnail(item);
  console.log(`${item.type}: ${thumbnail}`);
});

// Create a new media item
const newItem: MediaItem = {
  id: 'media-1',
  type: 'image',
  url: 'photo.jpg',
  alt: 'Description',
  caption: 'Photo caption'
};
```
