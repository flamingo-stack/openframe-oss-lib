A collection of utility functions for common operations including class name merging, string manipulation, async operations, and environment-specific URL generation for multi-platform applications.

## Key Components

- **`cn()`** - Merges class names using clsx and tailwind-merge for conflict-free Tailwind CSS classes
- **`delay()`** - Creates a Promise-based delay for async operations
- **`generateRandomString()`** - Generates cryptographically random strings with customizable length
- **`truncateString()`** - Safely truncates strings with ellipsis or custom suffix
- **`deepClone()`** - Creates deep copies of objects using JSON serialization
- **`getSlackCommunityJoinUrl()`** - Retrieves Slack community URL from environment with fallback
- **`getBaseUrl()`** - Platform-aware URL resolution supporting multiple app types and environments

## Usage Example

```typescript
import { cn, delay, generateRandomString, getBaseUrl } from './common'

// Merge Tailwind classes safely
const buttonClass = cn(
  'px-4 py-2 bg-blue-500',
  isActive && 'bg-blue-600',
  'hover:bg-blue-700'
)

// Async operations with delay
async function processWithDelay() {
  await delay(1000) // Wait 1 second
  console.log('Processing complete')
}

// Generate unique identifiers
const sessionId = generateRandomString(16)

// Get environment-appropriate URLs
const apiUrl = getBaseUrl('admin-hub') // https://admin-hub.flamingo.so
const devUrl = getBaseUrl() // Empty string in dev, production URL in prod
```
