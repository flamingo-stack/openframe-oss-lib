A collection of utility functions for common operations including CSS class merging, async operations, string manipulation, and environment-specific URL handling.

## Key Components

- **`cn()`** - Merges Tailwind CSS class names using clsx and tailwind-merge
- **`delay()`** - Creates a promise-based delay for async operations
- **`generateRandomString()`** - Generates random alphanumeric strings
- **`truncateString()`** - Truncates strings with customizable suffix
- **`deepClone()`** - Creates deep copies of objects via JSON serialization
- **`getSlackCommunityJoinUrl()`** - Returns Slack community URL from environment variables
- **`getBaseUrl()`** - Determines appropriate base URL for different app types and environments

## Usage Example

```typescript
import { cn, delay, generateRandomString, getBaseUrl } from './common'

// Merge CSS classes
const className = cn('text-red-500', 'font-bold', { 'underline': isActive })

// Async delay
await delay(1000) // Wait 1 second

// Generate random ID
const id = generateRandomString(12)

// Get environment-appropriate URL
const apiUrl = getBaseUrl('admin-hub') // Returns platform-specific URL
const devUrl = getBaseUrl(undefined, true) // Force absolute URL in dev
```
