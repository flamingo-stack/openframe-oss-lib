Utility functions for styling and categorizing AI confidence scores with consistent visual feedback using Tailwind CSS classes.

## Key Components

- **`getConfidenceColorClass`** - Returns combined border and text color classes
- **`getConfidenceLevel`** - Categorizes confidence into 'high', 'medium', 'low', or 'none'
- **`getConfidenceBorderClass`** - Returns border color class only
- **`getConfidenceTextClass`** - Returns text color class only
- **`getConfidenceBgClass`** - Returns subtle background color class
- **`getConfidenceLabel`** - Returns human-readable confidence description

All functions use a consistent threshold system: ≥80 = high/green, ≥50 = medium/yellow, <50 = low/red.

## Usage Example

```typescript
import { 
  getConfidenceColorClass, 
  getConfidenceLevel, 
  getConfidenceLabel 
} from './confidence-helpers';

// Style a confidence badge
const confidence = 75;
const badgeClasses = `px-2 py-1 rounded border ${getConfidenceColorClass(confidence)}`;

// Categorize confidence level
const level = getConfidenceLevel(confidence); // 'medium'
const label = getConfidenceLabel(confidence); // 'Medium'

// Use in JSX
<span className={badgeClasses}>
  {label} ({confidence}%)
</span>
```
