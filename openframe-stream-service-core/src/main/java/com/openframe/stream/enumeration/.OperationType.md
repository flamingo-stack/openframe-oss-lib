A simple enumeration defining the four basic CRUD (Create, Read, Update, Delete) operations for data manipulation in the OpenFrame streaming system.

## Key Components

- **READ** - Represents data retrieval operations
- **CREATE** - Represents data insertion operations  
- **UPDATE** - Represents data modification operations
- **DELETE** - Represents data removal operations

## Usage Example

```java
import com.openframe.stream.enumeration.OperationType;

public class DataProcessor {
    
    public void processOperation(OperationType operation, Object data) {
        switch (operation) {
            case READ:
                // Handle read operation
                retrieveData(data);
                break;
            case CREATE:
                // Handle create operation
                insertData(data);
                break;
            case UPDATE:
                // Handle update operation
                modifyData(data);
                break;
            case DELETE:
                // Handle delete operation
                removeData(data);
                break;
        }
    }
    
    // Method to check operation type
    public boolean isWriteOperation(OperationType operation) {
        return operation == OperationType.CREATE || 
               operation == OperationType.UPDATE || 
               operation == OperationType.DELETE;
    }
}
```
