An enumeration that defines the standard CRUD (Create, Read, Update, Delete) operation types used throughout the OpenFrame streaming system to categorize database and data manipulation operations.

## Key Components

- **READ** - Represents data retrieval operations
- **CREATE** - Represents data insertion operations  
- **UPDATE** - Represents data modification operations
- **DELETE** - Represents data removal operations

## Usage Example

```java
import com.openframe.stream.enumeration.OperationType;

// Example usage in a data access layer
public class DataProcessor {
    
    public void processOperation(OperationType operation, Object data) {
        switch (operation) {
            case CREATE:
                insertData(data);
                break;
            case READ:
                retrieveData(data);
                break;
            case UPDATE:
                modifyData(data);
                break;
            case DELETE:
                removeData(data);
                break;
        }
    }
    
    // Example in logging or auditing
    public void logOperation(OperationType type, String entityId) {
        System.out.println("Performing " + type + " operation on entity: " + entityId);
    }
}
```

This enum provides type safety and consistency when working with data operations across the streaming platform, ensuring standardized operation categorization for logging, auditing, and processing workflows.
