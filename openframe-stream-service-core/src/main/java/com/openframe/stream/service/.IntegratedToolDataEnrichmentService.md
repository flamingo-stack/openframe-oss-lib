A Spring service that enriches Debezium stream messages with machine and organization information by looking up cached data based on agent IDs.

## Key Components

- **IntegratedToolDataEnrichmentService**: Main service class implementing `DataEnrichmentService<DeserializedDebeziumMessage>`
- **getExtraParams()**: Enriches messages with machine ID, hostname, organization ID, and organization name
- **getType()**: Returns the service type identifier `INTEGRATED_TOOLS_EVENTS`
- **MachineIdCacheService**: Injected dependency for retrieving cached machine and organization data

## Usage Example

```java
@Autowired
private IntegratedToolDataEnrichmentService enrichmentService;

public void processMessage(DeserializedDebeziumMessage message) {
    // Enrich the message with machine and organization data
    IntegratedToolEnrichedData enrichedData = enrichmentService.getExtraParams(message);
    
    // Access enriched information
    String machineId = enrichedData.getMachineId();
    String hostname = enrichedData.getHostname();
    Long organizationId = enrichedData.getOrganizationId();
    String organizationName = enrichedData.getOrganizationName();
    
    // Check service type
    DataEnrichmentServiceType type = enrichmentService.getType();
    // Returns: INTEGRATED_TOOLS_EVENTS
}
```

The service handles null safety by returning empty enriched data when the input message or agent ID is null, and logs warnings when machine information cannot be found in the cache.
