Service that enriches Debezium messages with cached machine and organization data for integrated tool events.

## Key Components

- **IntegratedToolDataEnrichmentService**: Main service implementing `DataEnrichmentService<DeserializedDebeziumMessage>`
- **getExtraParams()**: Core enrichment method that retrieves machine and organization details from cache
- **getType()**: Returns `INTEGRATED_TOOLS_EVENTS` service type identifier
- **MachineIdCacheService**: Injected dependency for accessing cached machine and organization data

## Usage Example

```java
@Autowired
private IntegratedToolDataEnrichmentService enrichmentService;

// Enrich a Debezium message with machine/organization data
DeserializedDebeziumMessage message = new DeserializedDebeziumMessage();
message.setAgentId("agent-123");

IntegratedToolEnrichedData enrichedData = enrichmentService.getExtraParams(message);

// Access enriched data
String machineId = enrichedData.getMachineId();
String hostname = enrichedData.getHostname();
String orgId = enrichedData.getOrganizationId();
String orgName = enrichedData.getOrganizationName();

// Check service type
DataEnrichmentServiceType type = enrichmentService.getType();
// Returns: INTEGRATED_TOOLS_EVENTS
```

The service handles null safety by returning empty enriched data objects when the input message or agent ID is null, and logs warnings when machine information cannot be found in the cache.
