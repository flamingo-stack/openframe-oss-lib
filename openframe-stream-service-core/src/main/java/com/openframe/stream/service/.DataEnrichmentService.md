A service interface that defines the contract for enriching streaming data messages with additional parameters and metadata.

## Key Components

- **`getExtraParams(T message)`** - Generic method that takes a message of type T and returns enriched data containing additional parameters
- **`getType()`** - Returns the specific type of data enrichment service implementation
- **Generic Type Parameter `<T>`** - Allows implementations to work with different message types while maintaining type safety
- **`IntegratedToolEnrichedData`** - The standardized return type containing enriched data from integrated tools
- **`DataEnrichmentServiceType`** - Enum that categorizes different enrichment service implementations

## Usage Example

```java
// Implementation example
public class MetricsEnrichmentService implements DataEnrichmentService<MetricMessage> {
    
    @Override
    public IntegratedToolEnrichedData getExtraParams(MetricMessage message) {
        // Enrich message with additional metrics data
        return IntegratedToolEnrichedData.builder()
            .enrichedParams(extractMetadata(message))
            .timestamp(Instant.now())
            .build();
    }
    
    @Override
    public DataEnrichmentServiceType getType() {
        return DataEnrichmentServiceType.METRICS;
    }
}

// Usage in streaming pipeline
DataEnrichmentService<MetricMessage> enrichmentService = new MetricsEnrichmentService();
IntegratedToolEnrichedData enrichedData = enrichmentService.getExtraParams(incomingMessage);
```

This interface enables pluggable data enrichment strategies in streaming pipelines, allowing different implementations to enhance messages with context-specific additional data while maintaining a consistent API contract.
