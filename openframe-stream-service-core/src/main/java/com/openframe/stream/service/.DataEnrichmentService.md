Service interface for enriching streaming data with additional parameters and metadata from various data sources.

## Key Components

- **`getExtraParams(T message)`** - Core enrichment method that takes a generic message type and returns enriched data wrapped in `IntegratedToolEnrichedData`
- **`getType()`** - Returns the specific `DataEnrichmentServiceType` enum value identifying the enrichment service implementation
- **Generic type `<T>`** - Allows implementations to work with different message types while maintaining type safety

## Usage Example

```java
// Implementation example
public class DatabaseEnrichmentService implements DataEnrichmentService<UserEvent> {
    
    @Override
    public IntegratedToolEnrichedData getExtraParams(UserEvent message) {
        // Fetch additional user data from database
        UserProfile profile = userRepository.findById(message.getUserId());
        
        return IntegratedToolEnrichedData.builder()
            .userId(message.getUserId())
            .userProfile(profile)
            .enrichmentTimestamp(Instant.now())
            .build();
    }
    
    @Override
    public DataEnrichmentServiceType getType() {
        return DataEnrichmentServiceType.DATABASE_LOOKUP;
    }
}

// Usage in stream processing
DataEnrichmentService<UserEvent> enrichmentService = new DatabaseEnrichmentService();
IntegratedToolEnrichedData enrichedData = enrichmentService.getExtraParams(userEvent);
```
