A Spring Kafka listener service that handles JSON messages from multiple integrated tool topics including MeshCentral, Tactical RMM, and Fleet MDM events.

## Key Components

- **JsonKafkaListener**: Main service class annotated with `@Service` for Spring dependency injection
- **messageProcessor**: Injected `GenericJsonMessageProcessor` that handles the actual message processing logic
- **listenIntegratedToolsEvents()**: Kafka listener method that processes messages from multiple configured topics

## Usage Example

This listener automatically consumes messages when they arrive on the configured topics:

```java
// Configuration in application.yml
openframe:
  oss-tenant:
    kafka:
      topics:
        inbound:
          meshcentral-events: "meshcentral.events"
          tactical-rmm-events: "tactical-rmm.events"
          fleet-mdm-events: "fleet-mdm.events"
          fleet-mdm-query-result-events: "fleet-mdm.query-results"

spring:
  oss-tenant:
    kafka:
      consumer:
        group-id: "openframe-stream-consumer"
```

The listener receives Debezium-formatted messages with message type headers and delegates processing to the injected `GenericJsonMessageProcessor`. Messages from all four integrated tool topics are handled uniformly through the same processing pipeline.
