Abstract base class for processing Debezium CDC (Change Data Capture) events and publishing them as `IntegratedToolEvent` messages to Kafka topics within the OpenFrame stream pipeline.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `DebeziumKafkaMessageHandler` | Abstract Class | Extends `DebeziumMessageHandler<IntegratedToolEvent, DeserializedDebeziumMessage>` |
| `kafkaProducer` | `BaseRetryingKafkaProducer` | Publishes transformed events to Kafka with retry support |
| `validator` | `DebeziumEventValidator` | Validates incoming Debezium messages before processing |
| `getTopic()` | Abstract Method | Implemented by subclasses to define the target Kafka topic |
| `transform()` | Override | Maps `DeserializedDebeziumMessage` + `IntegratedToolEnrichedData` into an `IntegratedToolEvent` |
| `isValidMessage()` | Override | Passes only visible and validator-approved messages |
| `buildMessageBrokerKey()` | Private | Constructs a Kafka partition key using `deviceId`, `userId`, or `toolType` |
| `handleCreate/Read/Update/Delete()` | Protected | CRUD operation handlers; create/read/update publish to Kafka, delete is a no-op |

## Usage Example

```java
@Component
public class EndpointSecurityEventHandler extends DebeziumKafkaMessageHandler {

    public EndpointSecurityEventHandler(
            BaseRetryingKafkaProducer kafkaProducer,
            ObjectMapper objectMapper,
            DebeziumEventValidator validator) {
        super(kafkaProducer, objectMapper, validator);
    }

    @Override
    protected String getTopic() {
        return "openframe.endpoint-security.events";
    }
}
```

## Notes

- **Routing key priority**: `deviceId` → `userId` → `toolType`, ensuring Kafka messages are partitioned by the most specific available identifier.
- `getType()` returns `EventHandlerType.COMMON_TYPE` and `getDestination()` returns `Destination.KAFKA`, used for handler discovery and routing.
- `handleDelete()` is intentionally a no-op — CDC delete events are silently dropped at this layer.
