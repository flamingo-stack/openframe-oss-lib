An abstract base class for handling Debezium message processing with built-in JSON deserialization and CRUD operation routing. This class provides a template for implementing message handlers that transform Debezium messages into domain objects and route them to appropriate CRUD handlers.

## Key Components

- **GenericMessageHandler<T, U, V>**: Abstract generic class with three type parameters for transformed data, Debezium message, and enriched data
- **ObjectMapper**: Pre-configured Jackson mapper with JavaTimeModule and lenient deserialization settings
- **handle()**: Main entry point that orchestrates message validation, transformation, and routing
- **transform()**: Abstract method for converting Debezium messages to domain objects
- **pushData()**: Routes transformed data to appropriate CRUD handlers based on operation type
- **handleCreate/Read/Update/Delete()**: Abstract methods for implementing specific CRUD operations

## Usage Example

```java
public class UserMessageHandler extends GenericMessageHandler<User, UserDebeziumMessage, UserEnrichedData> {
    
    public UserMessageHandler(ObjectMapper mapper) {
        super(mapper);
    }
    
    @Override
    protected User transform(UserDebeziumMessage message, UserEnrichedData extraParams) {
        return mapper.convertValue(message.getAfter(), User.class);
    }
    
    @Override
    protected OperationType getOperationType(UserDebeziumMessage message) {
        return OperationType.valueOf(message.getOp().toUpperCase());
    }
    
    @Override
    protected void handleCreate(User user) {
        userRepository.save(user);
    }
    
    // Implement other CRUD methods...
}
```
