An abstract base class that provides a generic framework for handling Debezium messages with CRUD operation routing and Jackson JSON transformation capabilities.

## Key Components

- **GenericMessageHandler<T, U, V>** - Abstract generic class with type parameters for transformed data (T), Debezium message (U), and enriched data (V)
- **ObjectMapper** - Pre-configured Jackson mapper with JavaTimeModule and lenient deserialization settings
- **handle()** - Main processing method that validates, transforms, and routes messages based on operation type
- **pushData()** - Routes transformed data to appropriate CRUD handlers using switch expressions
- **Abstract methods** - `transform()`, `getOperationType()`, and CRUD handlers (`handleCreate()`, `handleRead()`, `handleUpdate()`, `handleDelete()`) for concrete implementation

## Usage Example

```java
public class UserMessageHandler extends GenericMessageHandler<User, UserDebeziumMessage, UserEnrichedData> {
    
    public UserMessageHandler(ObjectMapper mapper) {
        super(mapper);
    }
    
    @Override
    protected User transform(UserDebeziumMessage message, UserEnrichedData extraParams) {
        return mapper.convertValue(message.getAfter(), User.class);
    }
    
    @Override
    protected OperationType getOperationType(UserDebeziumMessage message) {
        return OperationType.valueOf(message.getOp().toUpperCase());
    }
    
    @Override
    protected void handleCreate(User data) {
        userRepository.save(data);
    }
    
    @Override
    protected void handleUpdate(User data) {
        userRepository.update(data);
    }
    
    @Override
    protected void handleDelete(User data) {
        userRepository.delete(data.getId());
    }
    
    @Override
    protected void handleRead(User data) {
        // Handle read operations
    }
}
```
