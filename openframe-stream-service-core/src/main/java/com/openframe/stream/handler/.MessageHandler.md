Defines a generic interface for handling deserialized Debezium messages with enriched data in the OpenFrame streaming framework. This interface establishes a contract for processing CDC (Change Data Capture) events with additional context.

## Key Components

- **MessageHandler<U, V>**: Generic interface with type parameters for message and enriched data types
- **getType()**: Returns the specific event handler type classification
- **getDestination()**: Specifies the target destination for processed messages
- **handle()**: Core processing method that accepts a message and enriched parameters

## Usage Example

```java
@Component
public class UserMessageHandler implements MessageHandler<UserDebeziumMessage, UserEnrichedData> {
    
    @Override
    public EventHandlerType getType() {
        return EventHandlerType.USER_UPDATE;
    }
    
    @Override
    public Destination getDestination() {
        return Destination.ELASTICSEARCH;
    }
    
    @Override
    public void handle(UserDebeziumMessage message, UserEnrichedData extraParams) {
        // Process user change events
        processUserUpdate(message.getPayload(), extraParams.getMetadata());
    }
}
```
