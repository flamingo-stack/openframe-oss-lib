Abstract handler for processing Debezium change data capture messages, extending the generic message handler to provide CDC-specific functionality and operation type mapping.

## Key Components

- **DebeziumMessageHandler** - Abstract base class extending `GenericMessageHandler` for Debezium CDC message processing
- **getOperationType()** - Maps Debezium operation codes (`c`, `r`, `u`, `d`) to `OperationType` enum values
- **transform()** - Abstract method for concrete implementations to transform Debezium messages with enriched data

## Usage Example

```java
@Component
public class UserDebeziumHandler extends DebeziumMessageHandler<UserEvent, UserDebeziumMessage> {
    
    public UserDebeziumHandler(ObjectMapper mapper) {
        super(mapper);
    }
    
    @Override
    protected UserEvent transform(UserDebeziumMessage debeziumMessage, IntegratedToolEnrichedData extraParams) {
        OperationType operation = getOperationType(debeziumMessage);
        
        return UserEvent.builder()
            .userId(debeziumMessage.getPayload().getAfter().getId())
            .operationType(operation)
            .timestamp(debeziumMessage.getPayload().getTimestamp())
            .enrichedData(extraParams)
            .build();
    }
}
```
