Abstract base class that handles Debezium CDC (Change Data Capture) messages by extending the generic message handler with specialized operation type mapping functionality.

## Key Components

- **DebeziumMessageHandler<T, U>** - Abstract generic class extending `GenericMessageHandler` for processing Debezium messages
- **getOperationType()** - Maps Debezium operation codes to `OperationType` enum values
- **transform()** - Abstract method for transforming deserialized Debezium messages with enriched data
- **Constructor** - Accepts `ObjectMapper` for JSON serialization/deserialization

## Usage Example

```java
public class UserDebeziumHandler extends DebeziumMessageHandler<UserEvent, UserDebeziumMessage> {
    
    public UserDebeziumHandler(ObjectMapper mapper) {
        super(mapper);
    }
    
    @Override
    protected UserEvent transform(UserDebeziumMessage debeziumMessage, 
                                  IntegratedToolEnrichedData extraParams) {
        OperationType operation = getOperationType(debeziumMessage);
        
        return UserEvent.builder()
            .operation(operation)
            .userData(debeziumMessage.getPayload().getAfter())
            .enrichedData(extraParams)
            .build();
    }
}
```

The handler automatically converts Debezium's single-character operation codes ("c", "r", "u", "d") to strongly-typed `OperationType` enum values, providing a clean abstraction for CDC event processing.
