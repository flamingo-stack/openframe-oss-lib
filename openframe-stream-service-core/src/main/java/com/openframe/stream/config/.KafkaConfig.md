Configuration class that sets up Kafka infrastructure components and message type conversion for the OpenFrame streaming application.

## Key Components

- **messageTypeConverter()** - Bean that converts byte arrays to `MessageType` enum values using UTF-8 encoding
- Uses Spring Boot's auto-configuration with `KafkaProperties` and `SslBundles`
- Imports `GenericKafkaProducer` and various Kafka factory classes for producer/consumer setup

## Usage Example

```java
@Autowired
private Converter<byte[], MessageType> messageTypeConverter;

// Convert Kafka message headers or payloads to MessageType enum
byte[] messageBytes = "USER_ACTION".getBytes(StandardCharsets.UTF_8);
MessageType type = messageTypeConverter.convert(messageBytes);

if (type != null) {
    // Process the message based on type
    handleMessage(type);
}
```

The converter handles invalid enum values gracefully by returning `null` instead of throwing exceptions, making it suitable for processing unreliable message streams where malformed data might be encountered.
