A utility class for parsing timestamps from database change events captured via Debezium CDC, which standardizes all database timestamps to ISO 8601 format.

## Key Components

- **parseIso8601()** - Static method that converts ISO 8601 timestamp strings to epoch milliseconds with error handling

## Usage Example

```java
import com.openframe.stream.util.TimestampParser;

// Parse a Debezium CDC timestamp
Optional<Long> epochMillis = TimestampParser.parseIso8601("2024-01-15T10:30:45.123Z");
if (epochMillis.isPresent()) {
    long timestamp = epochMillis.get();
    // Process the timestamp
} else {
    // Handle invalid timestamp
}

// Handle malformed timestamps gracefully
Optional<Long> invalid = TimestampParser.parseIso8601("invalid-date");
// Returns Optional.empty() and logs warning
```

This parser is specifically designed for Debezium's timestamp format normalization, providing safe conversion with automatic error handling and logging for invalid timestamps.
