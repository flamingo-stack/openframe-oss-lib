Utility class for parsing ISO 8601 timestamps from Debezium CDC events into epoch milliseconds. Provides safe timestamp parsing with error handling for database change data capture scenarios.

## Key Components

- **parseIso8601(String)** - Static method that converts ISO 8601 timestamp strings to epoch milliseconds, returning `Optional<Long>` for safe null handling

## Usage Example

```java
import com.openframe.stream.util.TimestampParser;
import java.util.Optional;

// Parse a Debezium CDC timestamp
String debeziumTimestamp = "2023-10-15T14:30:00.123Z";
Optional<Long> epochMillis = TimestampParser.parseIso8601(debeziumTimestamp);

if (epochMillis.isPresent()) {
    long timestamp = epochMillis.get();
    System.out.println("Parsed timestamp: " + timestamp);
} else {
    System.out.println("Failed to parse timestamp");
}

// Handle invalid timestamps gracefully
Optional<Long> invalid = TimestampParser.parseIso8601("invalid-date");
// Returns Optional.empty() and logs warning
```

The parser handles malformed timestamps gracefully by returning an empty Optional and logging warnings, making it ideal for robust stream processing pipelines.
