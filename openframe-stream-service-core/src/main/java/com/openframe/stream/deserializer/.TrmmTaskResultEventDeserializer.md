Deserializes Tactical RMM task result change events from the `autotasks_taskresult` database table into normalized `TACTICAL_RMM_TASK_RESULT_EVENT` messages for the OpenFrame event stream.

## Key Components

| Member | Description |
|--------|-------------|
| `getType()` | Returns `MessageType.TACTICAL_RMM_TASK_RESULT_EVENT` to identify this deserializer |
| `getAgentId()` | Resolves the numeric `agent_id` PK to a canonical agent identifier via `TacticalRmmCacheService` |
| `getSourceEventType()` | Maps `status` field values (`passing`, `failing`, `pending`) to typed event strings (e.g., `task_result.completed`) |
| `getMessage()` | Builds a human-readable message using the task name (resolved from cache) and current status |
| `getSourceEventTimestamp()` | Parses the `last_run` ISO 8601 timestamp via `TimestampParser` |
| `getError()` | Serializes `stderr` content or failure details (stdout + retcode) into a JSON error payload |
| `getResult()` | Serializes `stdout`, `execution_time`, `retcode`, and `task_id` into a structured JSON result payload |
| `getDetails()` | Emits supplemental metadata (`sync_status`, `run_status`, `task_id`) as a JSON details payload |

## Usage Example

```java
// Injected automatically via Spring component scan
@Autowired
TrmmTaskResultEventDeserializer deserializer;

// Identify the deserializer type
MessageType type = deserializer.getType();
// → MessageType.TACTICAL_RMM_TASK_RESULT_EVENT

// Deserialize a CDC "after" node from a Debezium/change-stream event
JsonNode afterNode = objectMapper.readTree("""
    {
      "id": "42",
      "agent_id": "7",
      "task_id": "15",
      "status": "passing",
      "stdout": "Backup completed successfully",
      "stderr": "",
      "retcode": "0",
      "execution_time": "3.14",
      "last_run": "2024-11-01T10:30:00Z"
    }
    """);

// Framework calls individual getters during normalization:
// getSourceEventType(afterNode) → "task_result.completed"
// getMessage(afterNode)         → "Task 'Daily Backup' passing"
// getResult(afterNode)          → {"output":"Backup completed successfully","execution_time":3.14,"retcode":0,"task_id":"15"}
```
