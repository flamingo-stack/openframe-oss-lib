A Spring component that deserializes Tactical RMM agent history events from database change streams, transforming them into standardized integrated tool events for command and script execution tracking.

## Key Components

- **TrmmAgentHistoryEventDeserializer**: Main deserializer class extending `IntegratedToolEventDeserializer`
- **getType()**: Returns `TACTICAL_RMM_AGENT_HISTORY_EVENT` message type
- **getAgentId()**: Resolves agent ID using `TacticalRmmCacheService`
- **getSourceEventType()**: Determines event type (cmd_run/script_run) with started/finished states
- **getMessage()**: Generates human-readable messages for command/script events
- **getResult()**: Extracts execution results from stdout and execution time
- **getError()**: Extracts error information from stderr
- **TacticalRmmCacheService**: Dependency for agent and script metadata lookup
