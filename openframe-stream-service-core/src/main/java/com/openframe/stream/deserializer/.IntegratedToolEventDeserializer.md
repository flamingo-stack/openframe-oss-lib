This abstract class serves as a base deserializer for converting Kafka Debezium messages from integrated security tools into a unified event format. It handles common deserialization logic, timestamp processing, ID generation, and event metadata extraction.

## Key Components

- **deserialize()** - Main method that transforms CommonDebeziumMessage to DeserializedDebeziumMessage
- **buildDetailsJson()** - Combines error, result, and additional fields into structured JSON
- **generateCompositeId()** - Creates deterministic UUIDs for event identification
- **Abstract methods** - getAgentId(), getSourceEventType(), getEventToolId(), getMessage(), getDetails()
- **Utility methods** - parseStringField(), getEffectiveTimestamp(), extractTableName()

## Usage Example

```java
public class MyToolDeserializer extends IntegratedToolEventDeserializer {
    
    public MyToolDeserializer(ObjectMapper mapper, List<String> eventsToSkip, List<String> eventsInvisible) {
        super(mapper, eventsToSkip, eventsInvisible);
    }

    @Override
    protected Optional<String> getAgentId(JsonNode afterField) {
        return parseStringField(afterField, "agent_id");
    }

    @Override
    protected Optional<String> getSourceEventType(JsonNode afterField) {
        return parseStringField(afterField, "event_type");
    }

    @Override
    protected String getDetails(JsonNode after) {
        return parseStringField(after, "metadata").orElse("{}");
    }
}
```
