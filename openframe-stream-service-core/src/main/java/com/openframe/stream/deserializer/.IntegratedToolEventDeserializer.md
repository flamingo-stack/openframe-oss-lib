A Kafka message deserializer that transforms Debezium CDC events from integrated tools into a unified format for stream processing. This abstract class provides common deserialization logic while allowing tool-specific implementations to handle field extraction.

## Key Components

- **`deserialize()`** - Main method that converts CommonDebeziumMessage to DeserializedDebeziumMessage
- **`buildDetailsJson()`** - Combines error, result, and additional dynamic fields into structured JSON
- **`generateCompositeId()`** - Creates deterministic UUIDs for event identification using tool/table/ID patterns
- **`getEffectiveTimestamp()`** - Prioritizes source event timestamps over Debezium processing timestamps
- **Abstract methods** - `getAgentId()`, `getSourceEventType()`, `getEventToolId()`, `getMessage()`, `getDetails()` for tool-specific field extraction
- **`parseStringField()`** - Utility method for safe field extraction from JsonNode

## Usage Example

```java
public class GitHubEventDeserializer extends IntegratedToolEventDeserializer {
    
    public GitHubEventDeserializer(ObjectMapper mapper, List<String> eventsToSkip, List<String> eventsInvisible) {
        super(mapper, eventsToSkip, eventsInvisible);
    }
    
    @Override
    protected Optional<String> getAgentId(JsonNode afterField) {
        return parseStringField(afterField, "actor_id");
    }
    
    @Override
    protected Optional<String> getSourceEventType(JsonNode afterField) {
        return parseStringField(afterField, "type");
    }
    
    @Override
    protected Optional<String> getEventToolId(JsonNode afterField) {
        return parseStringField(afterField, "id");
    }
    
    @Override
    protected String getDetails(JsonNode afterField) {
        ObjectNode details = mapper.createObjectNode();
        parseStringField(afterField, "repository").ifPresent(repo -> details.put("repository", repo));
        parseStringField(afterField, "action").ifPresent(action -> details.put("action", action));
        return details.toString();
    }
}
```
