A specialized deserializer for processing TRMM (Tactical RMM) audit events from JSON data streams, extending the base integrated tool event deserializer with TRMM-specific field mappings and event type handling.

## Key Components

- **TrmmAuditEventDeserializer**: Main class extending `IntegratedToolEventDeserializer`
- **Field Constants**: Predefined field names for consistent JSON parsing (`FIELD_AGENT_ID`, `FIELD_OBJECT_TYPE`, etc.)
- **Event Type Mapping**: Handles `AGENT_EXECUTE_SCRIPT` and `AGENT_EXECUTE_COMMAND` tactical events
- **Field Extraction Methods**: Overridden methods for extracting agent ID, event type, timestamps, and other audit data

## Usage Example

```java
@Autowired
private TrmmAuditEventDeserializer deserializer;

// Process incoming TRMM audit event
JsonNode auditEvent = objectMapper.readTree(jsonString);
MessageType type = deserializer.getType(); // Returns TACTICAL_RMM_AUDIT_EVENT

// Extract specific fields
Optional<String> agentId = deserializer.getAgentId(auditEvent);
Optional<String> eventType = deserializer.getSourceEventType(auditEvent);
Optional<Long> timestamp = deserializer.getSourceEventTimestamp(auditEvent);

// Event type combines object_type and action fields
// Example: "script.execute" or "command.run"
```

The deserializer automatically handles ISO 8601 timestamp parsing and combines object type and action fields to create meaningful event identifiers for tactical RMM operations.
