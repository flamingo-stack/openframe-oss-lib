A Spring component that deserializes Tactical RMM audit events from JSON streams into standardized message format. Extends the base `IntegratedToolEventDeserializer` to handle RMM-specific audit data transformation.

## Key Components

- **TrmmAuditEventDeserializer** - Main deserializer class extending `IntegratedToolEventDeserializer`
- **Field Constants** - Static field name constants for JSON parsing (`FIELD_AGENT_ID`, `FIELD_OBJECT_TYPE`, etc.)
- **getType()** - Returns `MessageType.TACTICAL_RMM_AUDIT_EVENT` message type
- **Field Extraction Methods** - Overridden methods to extract specific data from JSON nodes:
  - `getAgentId()` - Extracts agent identifier
  - `getSourceEventType()` - Combines object type and action for event classification
  - `getEventToolId()` - Extracts event ID
  - `getMessage()` - Extracts audit message
  - `getSourceEventTimestamp()` - Parses ISO8601 timestamp from entry_time
  - `getDetails()` - Extracts after_value as event details

## Usage Example

```java
@Autowired
private TrmmAuditEventDeserializer deserializer;

// JSON input from Tactical RMM audit stream
String jsonInput = """
    {
        "agentid": "agent-123",
        "object_type": "script",
        "action": "execute",
        "id": "audit-456",
        "message": "Script executed successfully",
        "entry_time": "2024-01-15T10:30:00Z",
        "after_value": "{'result': 'success'}"
    }
    """;

// Deserialize to standardized event
MessageType type = deserializer.getType(); // Returns TACTICAL_RMM_AUDIT_EVENT
// The deserializer handles supported event types: AGENT_EXECUTE_SCRIPT, AGENT_EXECUTE_COMMAND
```
