Deserializes Fleet MDM policy activity events from raw JSON stream data into structured `FLEET_MDM_POLICY_ACTIVITY_EVENT` messages, enriching them with policy metadata via cache lookups.

## Key Components

| Member | Type | Description |
|--------|------|-------------|
| `getType()` | Method | Returns `MessageType.FLEET_MDM_POLICY_ACTIVITY_EVENT` |
| `getMessage()` | Method | Builds a human-readable message combining activity type mapping and policy name |
| `getResult()` | Method | Serializes enriched policy details (name, query, resolution, platform, etc.) to JSON |
| `getPolicyInfo()` | Method | Resolves a `Policy` object via `FleetMdmCacheService`, evicting cache on mutation events |
| `getPolicyName()` | Method | Extracts policy name from activity details JSON first, then falls back to cache |
| `extractPolicyId()` | Method | Parses `policy_id` from the nested `details` JSON string |
| `POLICY_MUTATION_TYPES` | Constant | Set of activity types (`edited_policy`, `deleted_policy`) that trigger cache eviction |

## Usage Example

```java
// Spring auto-wires this deserializer as a @Component
// Incoming stream event JSON node (after field):
// {
//   "agentId": "agent-123",
//   "activity_type": "edited_policy",
//   "id": "evt-456",
//   "details": "{\"policy_id\": 99, \"policy_name\": \"Windows Patch Check\"}",
//   "created_at": "2024-06-01T12:00:00Z"
// }

FleetPolicyActivityDeserializer deserializer = context.getBean(FleetPolicyActivityDeserializer.class);

MessageType type = deserializer.getType();
// → MessageType.FLEET_MDM_POLICY_ACTIVITY_EVENT

// On deserialization, getMessage() returns:
// → "Edited policy 'Windows Patch Check'"

// getResult() returns enriched JSON:
// → {"policy_name":"Windows Patch Check","query":"SELECT ...","platform":"windows",...}
```

> **Cache behavior:** When an `edited_policy` or `deleted_policy` event is processed, the deserializer automatically evicts the cached `Policy` entry via `FleetMdmCacheService` before fetching fresh data, ensuring result accuracy after mutations.
