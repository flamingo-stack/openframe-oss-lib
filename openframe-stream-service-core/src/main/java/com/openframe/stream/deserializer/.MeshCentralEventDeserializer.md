A specialized event deserializer that processes MeshCentral remote management events, extracting agent information, event types, and metadata from JSON payloads within the OpenFrame streaming data pipeline.

## Key Components

**Main Class:**
- `MeshCentralEventDeserializer` - Extends `IntegratedToolEventDeserializer` to handle MeshCentral-specific event parsing

**Core Methods:**
- `getAgentId()` - Extracts node ID from MeshCentral events
- `getSourceEventType()` - Combines event type and action fields (e.g., "user.login")
- `getEventToolId()` - Retrieves MongoDB ObjectId from event metadata
- `getMessage()` - Extracts human-readable message content
- `getSourceEventTimestamp()` - Parses MongoDB date format timestamps

**Utility Methods:**
- `parseJson()` - Safely parses nested JSON strings with error handling
- `extractField()` - Generic field extraction with null safety
- `extractEventId()` - Handles MongoDB ObjectId extraction from `_id.$oid` structure

## Usage Example

```java
@Autowired
private MeshCentralEventDeserializer deserializer;

// Process MeshCentral event from stream
JsonNode eventNode = // ... from Kafka/stream
Optional<String> agentId = deserializer.getAgentId(eventNode);
Optional<String> eventType = deserializer.getSourceEventType(eventNode);

// Example event structure:
// {"nodeid": "agent123", "etype": "user", "action": "login", 
//  "_id": {"$oid": "507f1f77bcf86cd799439011"}, 
//  "time": {"$date": 1640995200000}}
```

The deserializer handles MeshCentral's nested JSON format and MongoDB-specific data structures, making them compatible with OpenFrame's unified event processing system.
