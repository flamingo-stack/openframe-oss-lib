Deserializes MeshCentral remote management tool events from JSON format into structured message objects. This component extends the base integrated tool deserializer to handle MeshCentral-specific event parsing and field extraction.

## Key Components

- **MeshCentralEventDeserializer** - Main deserializer class extending `IntegratedToolEventDeserializer`
- **getAgentId()** - Extracts node identifier from event data
- **getSourceEventType()** - Combines event type and action fields to create composite event types
- **getEventToolId()** - Retrieves MongoDB ObjectId from nested ID structure
- **parseJson()** - Safely parses JSON text content with error handling
- **extractEventId()** - Handles nested ObjectId extraction from MongoDB document structure

## Usage Example

```java
@Autowired
private MeshCentralEventDeserializer deserializer;

// The deserializer is automatically configured to handle:
// - Message type: MESHCENTRAL_EVENT
// - Source event types: MeshCentral.SERVER_TIMELINE_STATS

// Example JSON input structure:
{
  "nodeid": "mesh-node-123",
  "etype": "node",
  "action": "connect",
  "_id": {"$oid": "507f1f77bcf86cd799439011"},
  "msg": "Agent connected",
  "time": "2023-10-15T14:30:00Z"
}

// Automatically extracts:
// - Agent ID: "mesh-node-123"
// - Event type: "node.connect"
// - Tool ID: "507f1f77bcf86cd799439011"
// - Message: "Agent connected"
// - Timestamp: parsed ISO8601 format
```
