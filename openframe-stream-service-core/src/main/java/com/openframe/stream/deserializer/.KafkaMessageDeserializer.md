Interface for deserializing Kafka messages, specifically handling Debezium CDC (Change Data Capture) messages and converting them into application-specific deserialized formats.

## Key Components

- **`getType()`** - Returns the MessageType that this deserializer handles
- **`deserialize(CommonDebeziumMessage, MessageType)`** - Converts raw Debezium messages into DeserializedDebeziumMessage objects

## Usage Example

```java
public class UserMessageDeserializer implements KafkaMessageDeserializer {
    
    @Override
    public MessageType getType() {
        return MessageType.USER_UPDATE;
    }
    
    @Override
    public DeserializedDebeziumMessage deserialize(CommonDebeziumMessage message, MessageType type) {
        // Parse the Debezium message payload
        UserPayload payload = extractUserPayload(message);
        
        return DeserializedDebeziumMessage.builder()
            .messageType(type)
            .payload(payload)
            .timestamp(message.getTimestamp())
            .build();
    }
}

// Register and use the deserializer
KafkaMessageDeserializer deserializer = new UserMessageDeserializer();
DeserializedDebeziumMessage result = deserializer.deserialize(incomingMessage, MessageType.USER_UPDATE);
```

This interface follows the strategy pattern, allowing different implementations for various message types while maintaining a consistent deserialization contract.
