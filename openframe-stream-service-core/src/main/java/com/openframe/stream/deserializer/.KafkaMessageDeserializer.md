Interface for deserializing Kafka messages in the OpenFrame streaming system, specifically handling Debezium CDC messages with type-based deserialization strategies.

## Key Components

- **`getType()`** - Returns the MessageType that this deserializer handles
- **`deserialize()`** - Converts CommonDebeziumMessage to DeserializedDebeziumMessage based on message type

## Usage Example

```java
public class UserMessageDeserializer implements KafkaMessageDeserializer {
    
    @Override
    public MessageType getType() {
        return MessageType.USER_UPDATE;
    }
    
    @Override
    public DeserializedDebeziumMessage deserialize(CommonDebeziumMessage message, MessageType type) {
        // Parse the common Debezium message structure
        // Extract payload and metadata
        // Convert to strongly-typed DeserializedDebeziumMessage
        return new DeserializedDebeziumMessage(
            message.getPayload(),
            message.getSource(),
            type
        );
    }
}

// Usage in message processing
KafkaMessageDeserializer deserializer = getDeserializerForType(MessageType.USER_UPDATE);
DeserializedDebeziumMessage result = deserializer.deserialize(incomingMessage, MessageType.USER_UPDATE);
```

This interface enables a strategy pattern for handling different types of Debezium change data capture messages, allowing the streaming system to process various database events with type-specific deserialization logic.
