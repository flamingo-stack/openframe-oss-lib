A utility class that maps tool-specific event types to standardized unified event types, supporting multiple RMM platforms like MeshCentral and Tactical RMM. It enables consistent event processing across different integrated tools by translating native event formats into a common schema.

## Key Components

- **Static Mapping Registry**: Thread-safe `HashedMap` storing tool-specific event type mappings
- **Event Type Mapper**: `mapToUnifiedType()` method that converts source events to `UnifiedEventType`
- **Registration System**: `registerMapping()` method for adding new event type mappings
- **Default Mappings**: Comprehensive predefined mappings for MeshCentral and Tactical RMM events
- **Fallback Handling**: Returns `UnifiedEventType.UNKNOWN` for unmapped event types

## Usage Example

```java
// Map a MeshCentral login event
UnifiedEventType eventType = EventTypeMapper.mapToUnifiedType(
    IntegratedToolType.MESHCENTRAL, 
    "user.login"
);
// Returns: UnifiedEventType.LOGIN

// Map a Tactical RMM device registration
UnifiedEventType deviceEvent = EventTypeMapper.mapToUnifiedType(
    IntegratedToolType.TACTICAL,
    "agent_add"  
);
// Returns: UnifiedEventType.DEVICE_REGISTERED

// Handle unknown event type
UnifiedEventType unknownEvent = EventTypeMapper.mapToUnifiedType(
    IntegratedToolType.MESHCENTRAL,
    "custom.event"
);
// Returns: UnifiedEventType.UNKNOWN
```

The mapper supports comprehensive event categories including authentication, device management, user operations, remote sessions, monitoring, scripting, and system events across multiple RMM platforms.
