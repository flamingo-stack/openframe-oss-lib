Represents the relationship between a Fleet policy and a host, capturing compliance status and audit metadata for policy enforcement tracking.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `policyId` | `Integer` | ID of the associated policy |
| `hostId` | `Integer` | ID of the host evaluated against the policy |
| `passes` | `Boolean` | Whether the host passes the policy check |
| `createdAt` | `String` | Timestamp when the membership record was created |
| `updatedAt` | `String` | Timestamp of the last update |
| `automationIteration` | `Integer` | Tracks which automation cycle last evaluated this membership |

## Usage Example

```java
PolicyMembership membership = new PolicyMembership();
membership.setPolicyId(42);
membership.setHostId(101);
membership.setPasses(true);
membership.setCreatedAt("2024-01-15T10:30:00Z");
membership.setUpdatedAt("2024-01-15T12:00:00Z");
membership.setAutomationIteration(3);

// Check host compliance
if (Boolean.TRUE.equals(membership.getPasses())) {
    System.out.println("Host " + membership.getHostId() + " is compliant.");
}
```

## Notes

- Uses **Lombok `@Data`** to auto-generate getters, setters, `equals`, `hashCode`, and `toString`.
- **Jackson `@JsonProperty`** annotations map snake_case JSON fields (e.g., `policy_id`) to camelCase Java fields, ensuring compatibility with Fleet API responses.
- Part of the `com.openframe.stream.model.fleet` package within the **OpenFrame** platform's Fleet integration layer.
