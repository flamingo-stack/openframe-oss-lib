This model class represents a fleet activity record with user information and activity details, supporting JSON serialization/deserialization for API communication.

## Key Components

- **Activity** - Main data model class with Lombok `@Data` annotation for automatic getters/setters
- **Core Fields** - `id`, `createdAt`, `userId`, `userName`, `activityType`, `details`
- **JSON Mapping** - `@JsonProperty` annotations for snake_case to camelCase conversion
- **Enrichment Fields** - `agentId` and `hostId` for additional context from external sources

## Usage Example

```java
// Creating a new activity record
Activity activity = new Activity();
activity.setUserId(12345L);
activity.setUserName("john.doe");
activity.setActivityType("LOGIN");
activity.setDetails("{\"ip\":\"192.168.1.1\",\"location\":\"US\"}");
activity.setStreamed(1);

// JSON serialization will automatically map to snake_case
// {
//   "user_id": 12345,
//   "user_name": "john.doe",
//   "activity_type": "LOGIN",
//   ...
// }

// Enrichment data can be added separately
activity.setAgentId("agent-001");
activity.setHostId(567);
```
