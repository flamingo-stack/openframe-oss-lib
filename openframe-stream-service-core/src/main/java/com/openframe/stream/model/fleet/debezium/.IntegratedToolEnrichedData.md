A simple data model class representing enriched machine information with organizational and user context for fleet management operations.

## Key Components

- **machineId** - Unique identifier for the machine/device
- **hostname** - Network hostname of the machine
- **organizationId** - Unique identifier for the organization
- **organizationName** - Human-readable organization name
- **userId** - Identifier for the associated user

## Usage Example

```java
// Create enriched data object
IntegratedToolEnrichedData enrichedData = new IntegratedToolEnrichedData();
enrichedData.setMachineId("machine-001");
enrichedData.setHostname("workstation-alpha");
enrichedData.setOrganizationId("org-123");
enrichedData.setOrganizationName("Acme Corp");
enrichedData.setUserId("user-456");

// Access data (Lombok generates getters/setters)
String machineInfo = String.format("Machine %s (%s) belongs to %s", 
    enrichedData.getMachineId(), 
    enrichedData.getHostname(),
    enrichedData.getOrganizationName()
);
```

This class is typically used in Debezium change data capture pipelines to carry enriched machine metadata alongside streaming events, providing complete context for fleet management operations.
