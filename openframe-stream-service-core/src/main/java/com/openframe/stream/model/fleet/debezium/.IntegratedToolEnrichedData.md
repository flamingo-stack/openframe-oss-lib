A data transfer object (DTO) that holds enriched contextual metadata about a machine and its organizational associations, used within the Debezium-based fleet streaming pipeline.

## Key Components

| Field | Type | Description |
|---|---|---|
| `machineId` | `String` | Unique identifier for the machine |
| `hostname` | `String` | Hostname of the machine |
| `organizationId` | `String` | ID of the associated organization |
| `organizationName` | `String` | Human-readable organization name |
| `userId` | `String` | Associated user identifier |
| `tenantId` | `String` | Tenant scoping identifier |

The class is annotated with Lombok's `@Data`, which auto-generates getters, setters, `equals()`, `hashCode()`, and `toString()` at compile time.

## Usage Example

```java
IntegratedToolEnrichedData enrichedData = new IntegratedToolEnrichedData();
enrichedData.setMachineId("machine-001");
enrichedData.setHostname("workstation-nyc-01");
enrichedData.setOrganizationId("org-abc123");
enrichedData.setOrganizationName("Acme Corp");
enrichedData.setUserId("user-xyz789");
enrichedData.setTenantId("tenant-001");

// Access enriched fields downstream in the fleet event pipeline
String org = enrichedData.getOrganizationName(); // "Acme Corp"
```
