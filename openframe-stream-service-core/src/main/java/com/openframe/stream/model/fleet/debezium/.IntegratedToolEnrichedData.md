A simple data model class that represents enriched fleet data with organization and user context information, typically used in streaming data processing pipelines with Debezium integration.

## Key Components

- **machineId** - Unique identifier for the machine/device
- **hostname** - Network hostname of the machine
- **organizationId** - Unique identifier for the associated organization
- **organizationName** - Human-readable name of the organization
- **userId** - Identifier for the associated user

## Usage Example

```java
// Creating enriched data object
IntegratedToolEnrichedData enrichedData = new IntegratedToolEnrichedData();
enrichedData.setMachineId("machine-001");
enrichedData.setHostname("dev-server-01");
enrichedData.setOrganizationId("org-123");
enrichedData.setOrganizationName("Acme Corp");
enrichedData.setUserId("user-456");

// Accessing data (Lombok generates getters/setters)
String machineInfo = enrichedData.getMachineId() + " at " + enrichedData.getHostname();
```

This class serves as a data transfer object for fleet management systems, combining machine identification with organizational context for comprehensive data tracking and processing.
