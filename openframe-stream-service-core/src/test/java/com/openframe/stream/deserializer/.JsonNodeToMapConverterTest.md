JUnit test class that validates the conversion of Jackson JsonNode objects to flat Map structures with dot-notation and array indexing. Tests various JSON scenarios including nested objects, arrays, and primitive values.

## Key Components

- **JsonNodeToMapConverterTest** - Main test class that exercises JSON-to-Map conversion logic
- **convertJsonNodeToMap()** - Private helper method (copied from production code) that recursively flattens JsonNode into Map<String, String>
- **Test Methods**:
  - `testSimpleObjectConversion()` - Basic object field conversion
  - `testNestedObjectConversion()` - Multi-level object nesting with dot notation
  - `testArrayConversion()` - Array elements with bracket notation
  - `testNullAndEmptyValues()` - Null value handling and edge cases
  - `testComplexNestedStructure()` - Real-world JSON structure with mixed nesting

## Usage Example

```java
// The test validates this conversion pattern:
ObjectMapper mapper = new ObjectMapper();
JsonNode node = mapper.readTree("""
    {
        "user": {
            "id": 123,
            "tags": ["admin", "active"]
        }
    }
    """);

Map<String, String> result = new HashMap<>();
convertJsonNodeToMap(node, "", result);

// Results in flattened map:
// "user.id" -> "123"
// "user.tags[0]" -> "admin"
// "user.tags[1]" -> "active"

assertEquals("123", result.get("user.id"));
assertEquals("admin", result.get("user.tags[0]"));
```

The conversion creates dot-separated keys for nested objects and bracket notation for arrays, while preserving all primitive values as strings and excluding null values from the final map.
