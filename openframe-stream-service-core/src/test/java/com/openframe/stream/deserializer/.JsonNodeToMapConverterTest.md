Test suite for validating JSON-to-Map conversion functionality that flattens nested JSON structures into key-value pairs with dot notation for objects and bracket notation for arrays.

## Key Components

- **setUp()** - Initializes ObjectMapper for JSON parsing
- **testSimpleObjectConversion()** - Tests basic object property conversion
- **testNestedObjectConversion()** - Tests nested object flattening with dot notation
- **testArrayConversion()** - Tests array element conversion with bracket notation
- **testNullAndEmptyValues()** - Tests handling of null/empty values
- **testComplexNestedStructure()** - Tests real-world complex JSON structures
- **convertJsonNodeToMap()** - Helper method that recursively flattens JsonNode to Map

## Usage Example

```java
// Test demonstrates how the converter handles nested JSON
String json = """
{
    "user": {
        "id": 456,
        "profile": {"name": "John"}
    },
    "tags": ["tag1", "tag2"]
}
""";

JsonNode node = objectMapper.readTree(json);
Map<String, String> result = new HashMap<>();
convertJsonNodeToMap(node, "", result);

// Results in flattened map:
// "user.id" -> "456"
// "user.profile.name" -> "John"
// "tags[0]" -> "tag1"
// "tags[1]" -> "tag2"
```

The tests verify that complex JSON structures are correctly flattened into string key-value pairs, with null values excluded and proper handling of various data types.
