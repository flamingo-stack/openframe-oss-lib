A data model class representing contact person information within an organization, designed as an embedded document rather than a standalone collection.

## Key Components

**ContactPerson Class**
- `contactName` - Full name of the contact person
- `title` - Job title or position within the organization
- `phone` - Contact phone number
- `email` - Email address

**Lombok Annotations**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Provides builder pattern implementation
- `@NoArgsConstructor` / `@AllArgsConstructor` - Generate constructors

## Usage Example

```java
// Using builder pattern
ContactPerson contact = ContactPerson.builder()
    .contactName("John Smith")
    .title("Operations Manager")
    .phone("+1-555-0123")
    .email("john.smith@company.com")
    .build();

// Using constructor
ContactPerson contact2 = new ContactPerson(
    "Jane Doe", 
    "HR Director", 
    "+1-555-0456", 
    "jane.doe@company.com"
);

// Accessing properties
String name = contact.getContactName();
contact.setEmail("john.smith@newdomain.com");
```

This class is typically embedded within organization documents to store contact information without requiring separate database collections.
