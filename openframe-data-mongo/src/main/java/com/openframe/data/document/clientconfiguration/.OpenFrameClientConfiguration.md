This MongoDB document entity represents client configuration settings for the OpenFrame application, storing versioned download configurations with automatic timestamp tracking.

## Key Components

- **@Document**: Maps to MongoDB collection "openframe_client_configuration"
- **@Id**: MongoDB document identifier
- **version**: Configuration version string
- **downloadConfiguration**: List of `DownloadConfiguration` objects
- **@CreatedDate/@LastModifiedDate**: Automatic timestamp fields for audit tracking
- **Lombok annotations**: Provides getters, setters, builders, and constructors

## Usage Example

```java
// Create new client configuration
OpenFrameClientConfiguration config = OpenFrameClientConfiguration.builder()
    .version("1.2.0")
    .downloadConfiguration(List.of(
        new DownloadConfiguration("file1.zip", "https://example.com/download1"),
        new DownloadConfiguration("file2.jar", "https://example.com/download2")
    ))
    .build();

// Repository usage
@Autowired
private MongoRepository<OpenFrameClientConfiguration, String> repository;

// Save configuration
OpenFrameClientConfiguration saved = repository.save(config);

// Find by version
Optional<OpenFrameClientConfiguration> found = repository.findByVersion("1.2.0");
```
