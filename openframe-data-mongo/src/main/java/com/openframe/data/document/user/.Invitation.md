A MongoDB document entity representing user invitations with email-based user management and role assignment capabilities. This class uses Lombok annotations for boilerplate code generation and Spring Data MongoDB for persistence.

## Key Components

- **Document Mapping**: `@Document(collection = "invitations")` maps to MongoDB collection
- **Email Management**: Indexed email field with automatic trimming and lowercase conversion
- **Role System**: List of `UserRole` objects for permission management
- **Status Tracking**: `InvitationStatus` enum to track invitation lifecycle
- **Audit Fields**: Automatic timestamps for creation and modification tracking
- **Expiration**: `expiresAt` field for time-limited invitations

## Usage Example

```java
// Create a new invitation
Invitation invitation = Invitation.builder()
    .email("user@example.com")
    .roles(List.of(UserRole.ADMIN, UserRole.USER))
    .expiresAt(Instant.now().plusSeconds(86400)) // 24 hours
    .build();

// The email will be automatically normalized to lowercase
invitation.setEmail("User@Example.COM"); // stored as "user@example.com"

// Status defaults to PENDING
assert invitation.getStatus() == InvitationStatus.PENDING;
```
