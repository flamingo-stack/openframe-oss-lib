A MongoDB document class representing system events with type, payload, timestamp, and user tracking capabilities.

## Key Components

- **@Document(collection = "events")** - Maps to MongoDB "events" collection
- **@Data** - Lombok annotation providing getters, setters, toString, equals, and hashCode
- **@Builder** - Lombok annotation enabling builder pattern for object creation
- **Fields:**
  - `id` - MongoDB document identifier
  - `type` - Event classification/category
  - `payload` - Event data as string
  - `timestamp` - When the event occurred
  - `userId` - Associated user identifier

## Usage Example

```java
// Create an event using builder pattern
Event userLoginEvent = Event.builder()
    .type("USER_LOGIN")
    .payload("{\"deviceType\": \"mobile\", \"location\": \"US\"}")
    .timestamp(Instant.now())
    .userId("user123")
    .build();

// Access event properties
String eventType = userLoginEvent.getType();
Instant when = userLoginEvent.getTimestamp();

// Event for system operations
Event systemEvent = Event.builder()
    .type("SYSTEM_BACKUP")
    .payload("{\"status\": \"completed\", \"duration\": \"2.5s\"}")
    .timestamp(Instant.now())
    .build(); // userId can be null for system events
```
