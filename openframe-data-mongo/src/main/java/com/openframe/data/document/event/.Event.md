A MongoDB document entity representing system events with basic metadata and payload information.

## Key Components

- **Event**: Main entity class annotated with `@Document` for MongoDB collection mapping
- **Fields**:
  - `id`: MongoDB document identifier
  - `type`: Event classification/category
  - `payload`: Event data as string
  - `timestamp`: When the event occurred
  - `userId`: Associated user identifier
- **Annotations**: Uses Lombok `@Data` and `@Builder` for automatic getter/setter/builder generation

## Usage Example

```java
// Creating an event using the builder pattern
Event event = Event.builder()
    .type("USER_LOGIN")
    .payload("{\"ip\": \"192.168.1.1\", \"browser\": \"Chrome\"}")
    .timestamp(Instant.now())
    .userId("user123")
    .build();

// Accessing event data
String eventType = event.getType();
Instant when = event.getTimestamp();
String data = event.getPayload();
```

This class is typically used with Spring Data MongoDB repositories for persisting and querying event data in the "events" collection.
