A MongoDB document entity representing physical or virtual devices in the system, tracking their status, configuration, and health metrics.

## Key Components

- **Device**: Main entity class with Lombok `@Data` annotation for automatic getters/setters
- **Fields**:
  - `id`: MongoDB document identifier
  - `machineId`: Reference to associated Machine entity
  - `serialNumber`, `model`, `osVersion`: Hardware/software specifications
  - `status`: Operational state (ACTIVE, OFFLINE, MAINTENANCE)
  - `type`: Device category enum (DESKTOP, LAPTOP, SERVER, etc.)
  - `lastCheckin`: Timestamp of last system communication
  - `configuration`: Device settings object
  - `health`: Device health metrics object

## Usage Example

```java
// Creating a new device
Device device = new Device();
device.setMachineId("machine-123");
device.setSerialNumber("SN123456789");
device.setModel("Dell OptiPlex 7090");
device.setOsVersion("Windows 11");
device.setStatus("ACTIVE");
device.setType(DeviceType.DESKTOP);
device.setLastCheckin(Instant.now());

// Repository usage
@Autowired
private MongoRepository<Device, String> deviceRepository;

Device savedDevice = deviceRepository.save(device);
List<Device> activeDevices = deviceRepository.findByStatus("ACTIVE");
```
