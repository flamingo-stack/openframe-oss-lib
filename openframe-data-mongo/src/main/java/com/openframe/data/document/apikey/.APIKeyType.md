Enumeration that defines the different types of API key authentication methods supported by the OpenFrame document system.

## Key Components

- **HEADER** - API key passed as a custom header in HTTP requests
- **BEARER_TOKEN** - API key used as a Bearer token in the Authorization header
- **NONE** - No API key authentication required

## Usage Example

```java
import com.openframe.data.document.apikey.APIKeyType;

// Configure API key authentication type
APIKeyType authType = APIKeyType.BEARER_TOKEN;

// Use in conditional logic
switch (authType) {
    case HEADER:
        // Add custom header with API key
        request.addHeader("X-API-Key", apiKey);
        break;
    case BEARER_TOKEN:
        // Add Bearer token to Authorization header
        request.addHeader("Authorization", "Bearer " + apiKey);
        break;
    case NONE:
        // No authentication required
        break;
}

// Check if authentication is required
boolean requiresAuth = authType != APIKeyType.NONE;
```
