An enumeration that defines the different types of API key authentication methods supported by the OpenFrame system.

## Key Components

- **HEADER** - API key is passed as a custom header in HTTP requests
- **BEARER_TOKEN** - API key is used as a Bearer token in the Authorization header
- **NONE** - No API key authentication is required

## Usage Example

```java
// Setting API key type for header-based authentication
APIKeyType authType = APIKeyType.HEADER;

// Using in conditional logic
switch (apiKeyType) {
    case HEADER:
        request.addHeader("X-API-Key", apiKey);
        break;
    case BEARER_TOKEN:
        request.addHeader("Authorization", "Bearer " + apiKey);
        break;
    case NONE:
        // No authentication required
        break;
}

// Checking if authentication is required
boolean requiresAuth = apiKeyType != APIKeyType.NONE;
```

This enum is typically used in configuration objects or authentication handlers to specify how API keys should be transmitted when making requests to external services or APIs.
