A MongoDB document entity that tracks usage statistics and metrics for API keys in the OpenFrame system.

## Key Components

- **@Document**: Maps to the `api_key_stats` MongoDB collection
- **id**: Unique identifier for the statistics record
- **totalRequests**: Counter for all API requests made with the key
- **successfulRequests**: Counter for successful API calls
- **failedRequests**: Counter for failed API attempts
- **lastUsed**: Timestamp of the most recent API key usage
- **Lombok annotations**: Automatically generates getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create new API key statistics
ApiKeyStats stats = ApiKeyStats.builder()
    .totalRequests(150L)
    .successfulRequests(142L)
    .failedRequests(8L)
    .lastUsed(LocalDateTime.now())
    .build();

// Update statistics after API call
stats.setTotalRequests(stats.getTotalRequests() + 1);

double successRate = (double) stats.getSuccessfulRequests() / stats.getTotalRequests() * 100;
```
