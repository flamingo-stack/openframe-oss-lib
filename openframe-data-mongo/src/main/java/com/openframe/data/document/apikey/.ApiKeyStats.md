This document represents API key usage statistics stored in MongoDB, tracking request counts and last usage timestamp for API key monitoring and analytics.

## Key Components

- **ApiKeyStats**: Main entity class for storing API key usage metrics
- **totalRequests**: Counter for all API requests made with the key
- **successfulRequests**: Counter for successful API requests
- **failedRequests**: Counter for failed API requests  
- **lastUsed**: Timestamp of the most recent API key usage
- **MongoDB Integration**: Configured as a MongoDB document with collection name "api_key_stats"

## Usage Example

```java
// Create new API key statistics
ApiKeyStats stats = ApiKeyStats.builder()
    .totalRequests(150L)
    .successfulRequests(145L)
    .failedRequests(5L)
    .lastUsed(LocalDateTime.now())
    .build();

// Update statistics after API call
stats.setTotalRequests(stats.getTotalRequests() + 1);
stats.setSuccessfulRequests(stats.getSuccessfulRequests() + 1);
stats.setLastUsed(LocalDateTime.now());

// Calculate success rate
double successRate = (double) stats.getSuccessfulRequests() / stats.getTotalRequests() * 100;
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it ideal for data transfer and persistence operations.
