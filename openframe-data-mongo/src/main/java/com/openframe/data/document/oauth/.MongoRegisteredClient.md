A MongoDB document entity representing OAuth 2.0 registered client configurations for Spring Security OAuth2 authorization server implementations.

## Key Components

- **MongoRegisteredClient** - Main entity class with MongoDB document mapping
- **clientId** - Unique indexed identifier for OAuth client registration
- **authenticationMethods** - Supported client authentication methods (e.g., client_secret_basic)
- **grantTypes** - Allowed OAuth2 grant types (e.g., authorization_code, refresh_token)
- **redirectUris** - Valid redirect URIs for authorization callbacks
- **scopes** - Available OAuth2 scopes for the client
- **Token configuration** - TTL settings and refresh token reuse policy
- **Security flags** - PKCE and authorization consent requirements

## Usage Example

```java
// Create a new OAuth2 client registration
MongoRegisteredClient client = MongoRegisteredClient.builder()
    .clientId("my-app-client")
    .clientSecret("{noop}secret123")
    .authenticationMethods(Set.of("client_secret_basic"))
    .grantTypes(Set.of("authorization_code", "refresh_token"))
    .redirectUris(Set.of("http://localhost:8080/login/oauth2/code/my-app"))
    .scopes(Set.of("read", "write"))
    .requireProofKey(true)
    .requireAuthorizationConsent(false)
    .accessTokenTtlSeconds(3600)
    .refreshTokenTtlSeconds(86400)
    .reuseRefreshTokens(true)
    .build();

// Save to MongoDB repository
registeredClientRepository.save(client);
```
