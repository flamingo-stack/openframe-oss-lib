Enumeration that defines unique identifiers for integrated tools within the OpenFrame document management system.

## Key Components

- **FLEET_SERVER_ID** - Identifier for Fleet MDM server integration (`"fleetmdm-server"`)
- **TACTICAL_SERVER_ID** - Identifier for Tactical RMM server integration (`"tactical-rmm"`)
- **value** - String representation of the tool identifier (private field with getter)

## Usage Example

```java
// Get the identifier value for Fleet MDM
String fleetId = IntegratedToolId.FLEET_SERVER_ID.getValue();
// Returns: "fleetmdm-server"

// Get the identifier value for Tactical RMM
String tacticalId = IntegratedToolId.TACTICAL_SERVER_ID.getValue();
// Returns: "tactical-rmm"

// Use in configuration or API calls
public void configureIntegration(IntegratedToolId toolId) {
    String identifier = toolId.getValue();
    // Configure integration using the tool identifier
}
```

This enum provides type-safe access to tool identifiers used throughout the system for managing integrations with external fleet management and remote monitoring tools.
