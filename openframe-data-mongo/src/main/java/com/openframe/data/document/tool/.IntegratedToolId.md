This enum defines identifiers for integrated tools used within the system, providing a centralized way to reference different server types by their string values.

## Key Components

- **FLEET_SERVER_ID**: Enum constant for Fleet MDM server with value "fleetmdm-server"
- **TACTICAL_SERVER_ID**: Enum constant for Tactical RMM server with value "tactical-rmm"
- **value**: Private final field storing the string identifier for each tool
- **getValue()**: Lombok-generated getter method to access the string value

## Usage Example

```java
// Get the string value of a tool ID
String fleetId = IntegratedToolId.FLEET_SERVER_ID.getValue();
// Returns: "fleetmdm-server"

// Use in conditional logic
public void processToolData(IntegratedToolId toolId) {
    switch (toolId) {
        case FLEET_SERVER_ID:
            handleFleetServerData();
            break;
        case TACTICAL_SERVER_ID:
            handleTacticalServerData();
            break;
    }
}

// Compare tool types
if (currentTool == IntegratedToolId.TACTICAL_SERVER_ID) {
    // Handle tactical RMM specific logic
}
```
