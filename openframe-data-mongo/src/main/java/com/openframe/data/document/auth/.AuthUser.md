A MongoDB document model representing authenticated users in a multi-tenant authorization system, extending the base User class with authentication-specific fields and tenant isolation.

## Key Components

- **AuthUser**: Main entity class extending `User` with authentication capabilities
- **Tenant Isolation**: `tenantId` field with compound index ensuring unique email per tenant
- **Authentication Fields**: 
  - `passwordHash` - encrypted password storage
  - `emailVerified` - email verification status
  - `loginProvider` - authentication method (LOCAL, GOOGLE, etc.)
  - `externalUserId` - third-party provider user ID
  - `lastLogin` - user activity tracking
- **getFullName()**: Utility method for display name generation

## Usage Example

```java
// Create a new authenticated user
AuthUser authUser = AuthUser.builder()
    .tenantId("tenant-123")
    .email("user@example.com")
    .firstName("John")
    .lastName("Doe")
    .passwordHash("$2a$10$encrypted_password")
    .emailVerified(true)
    .loginProvider("LOCAL")
    .lastLogin(Instant.now())
    .build();

// Get display name
String displayName = authUser.getFullName(); // Returns "John Doe"

// Use with MongoDB repository
@Repository
public interface AuthUserRepository extends MongoRepository<AuthUser, String> {
    Optional<AuthUser> findByTenantIdAndEmail(String tenantId, String email);
}
```
