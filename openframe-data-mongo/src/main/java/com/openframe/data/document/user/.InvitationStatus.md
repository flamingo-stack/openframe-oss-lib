An enumeration that defines the possible states of user invitations within the OpenFrame system.

## Key Components

- **PENDING** - Invitation has been sent but not yet responded to by the recipient
- **ACCEPTED** - Invitation has been accepted and the user has joined
- **REVOKED** - Invitation has been cancelled or withdrawn before acceptance

## Usage Example

```java
// Check invitation status
if (invitation.getStatus() == InvitationStatus.PENDING) {
    // Send reminder email
    emailService.sendInvitationReminder(invitation);
}

// Update invitation status
invitation.setStatus(InvitationStatus.ACCEPTED);
userService.activateUser(invitation.getUserId());

// Revoke pending invitation
if (invitation.getStatus() == InvitationStatus.PENDING) {
    invitation.setStatus(InvitationStatus.REVOKED);
    invitationRepository.save(invitation);
}
```

This enum provides type safety for invitation state management and ensures consistent status tracking across the user invitation workflow.
