An enumeration that defines the possible states of user invitations within the OpenFrame system.

## Key Components

- **PENDING** - Invitation has been sent but not yet responded to
- **ACCEPTED** - Invitation has been accepted by the recipient
- **REVOKED** - Invitation has been cancelled or withdrawn

## Usage Example

```java
// Setting invitation status when creating a new invitation
InvitationStatus status = InvitationStatus.PENDING;

// Checking invitation state
if (invitation.getStatus() == InvitationStatus.ACCEPTED) {
    // Process accepted invitation
    activateUserAccount();
} else if (invitation.getStatus() == InvitationStatus.REVOKED) {
    // Handle revoked invitation
    cleanupInvitation();
}

// Updating invitation status
invitation.setStatus(InvitationStatus.ACCEPTED);

// Using in switch statements
switch (invitation.getStatus()) {
    case PENDING:
        sendReminderEmail();
        break;
    case ACCEPTED:
        completeOnboarding();
        break;
    case REVOKED:
        logRevocation();
        break;
}
```
