A filter class for building MongoDB queries to search and filter organization documents based on various criteria.

## Key Components

- **category** - Filter organizations by category/type
- **minEmployees** - Minimum employee count threshold
- **maxEmployees** - Maximum employee count threshold  
- **hasActiveContract** - Filter by active contract status
- **@Builder** - Enables fluent builder pattern for object creation
- **@Data** - Provides getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create filter with builder pattern
OrganizationQueryFilter filter = OrganizationQueryFilter.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Create filter for organizations with active contracts only
OrganizationQueryFilter activeFilter = OrganizationQueryFilter.builder()
    .hasActiveContract(true)
    .build();

// Create filter for large organizations in specific category
OrganizationQueryFilter largeCompanyFilter = OrganizationQueryFilter.builder()
    .category("Manufacturing")
    .minEmployees(1000)
    .build();
```
