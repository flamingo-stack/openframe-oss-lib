A filter class that defines query criteria for searching organizations in MongoDB, supporting filtering by category, employee count ranges, and contract status.

## Key Components

- **category** - String filter for organization category/type
- **minEmployees** - Integer minimum employee count threshold
- **maxEmployees** - Integer maximum employee count threshold  
- **hasActiveContract** - Boolean flag for active contract status
- **Lombok annotations** - Provides automatic getters, setters, builders, and constructors

## Usage Example

```java
// Build filter for medium-sized tech companies with active contracts
OrganizationQueryFilter filter = OrganizationQueryFilter.builder()
    .category("Technology")
    .minEmployees(50)
    .maxEmployees(500)
    .hasActiveContract(true)
    .build();

// Filter for any organization with 100+ employees
OrganizationQueryFilter largeOrgsFilter = OrganizationQueryFilter.builder()
    .minEmployees(100)
    .build();

// Filter by category only
OrganizationQueryFilter categoryFilter = new OrganizationQueryFilter();
categoryFilter.setCategory("Healthcare");
```

This filter class is typically used with repository methods or query builders to construct MongoDB queries for organization searches with flexible criteria combinations.
