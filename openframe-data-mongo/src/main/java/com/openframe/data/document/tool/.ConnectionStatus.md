Defines the possible connection states for database or network operations within the document tool framework.

## Key Components

- **CONNECTED** - Indicates an active, successful connection
- **DISCONNECTED** - Indicates no connection is established
- **ERROR** - Indicates a connection failure or error state

## Usage Example

```java
public class DatabaseManager {
    private ConnectionStatus status = ConnectionStatus.DISCONNECTED;
    
    public void connect() {
        try {
            // Connection logic here
            this.status = ConnectionStatus.CONNECTED;
        } catch (Exception e) {
            this.status = ConnectionStatus.ERROR;
        }
    }
    
    public boolean isConnected() {
        return status == ConnectionStatus.CONNECTED;
    }
    
    public void handleConnectionChange(ConnectionStatus newStatus) {
        switch (newStatus) {
            case CONNECTED:
                System.out.println("Connection established");
                break;
            case DISCONNECTED:
                System.out.println("Connection closed");
                break;
            case ERROR:
                System.out.println("Connection error occurred");
                break;
        }
    }
}
```
