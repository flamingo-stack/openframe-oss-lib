A simple enumeration that represents the possible states of a connection in the OpenFrame data document tool system.

## Key Components

- **CONNECTED** - Indicates an active, successful connection
- **DISCONNECTED** - Indicates no connection is currently established  
- **ERROR** - Indicates a connection failure or error state occurred

## Usage Example

```java
// Check connection status
ConnectionStatus status = getConnectionStatus();

switch (status) {
    case CONNECTED:
        // Proceed with data operations
        processData();
        break;
    case DISCONNECTED:
        // Attempt to reconnect
        establishConnection();
        break;
    case ERROR:
        // Handle error condition
        logError("Connection error occurred");
        retryConnection();
        break;
}

// Method signature example
public void handleConnectionChange(ConnectionStatus newStatus) {
    if (newStatus == ConnectionStatus.CONNECTED) {
        enableDataSync();
    } else {
        disableDataSync();
    }
}
```

This enum provides a type-safe way to represent connection states throughout the application, enabling clear status handling and improved code readability when working with connection-dependent operations.
