An enumeration that defines the different types of sessions available in the OpenFrame tool agent system, representing various contexts or modes of operation.

## Key Components

- **SERVICE** - Represents service-level sessions, typically for system or background operations
- **CONSOLE** - Represents console-based sessions for command-line interactions
- **USER** - Represents user-initiated sessions for interactive operations

## Usage Example

```java
// Setting session type for different contexts
SessionType sessionType = SessionType.USER;

// Using in conditional logic
switch (sessionType) {
    case SERVICE:
        initializeServiceSession();
        break;
    case CONSOLE:
        initializeConsoleSession();
        break;
    case USER:
        initializeUserSession();
        break;
}

// Comparing session types
if (currentSession.getType() == SessionType.CONSOLE) {
    enableConsoleFeatures();
}
```

This enum is commonly used to categorize and handle different session contexts within the OpenFrame document tool agent, allowing the system to apply appropriate behaviors, permissions, or configurations based on the session type.
