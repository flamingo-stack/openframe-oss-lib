A MongoDB document entity representing tags that can be used for categorizing and organizing content within different organizations.

## Key Components

- **Tag**: Main entity class with MongoDB document mapping
- **id**: Primary key field with `@Id` annotation
- **name**: Unique indexed field for tag identification
- **description**: Optional descriptive text for the tag
- **color**: Optional color code for UI representation
- **organizationId**: Scoping field to isolate tags by organization
- **createdAt/createdBy**: Audit fields for tracking tag creation

## Usage Example

```java
// Create a new tag
Tag tag = new Tag();
tag.setName("urgent");
tag.setDescription("High priority items");
tag.setColor("#FF0000");
tag.setOrganizationId("org-123");
tag.setCreatedAt(Instant.now());
tag.setCreatedBy("user-456");

// Save to MongoDB (assuming repository exists)
tagRepository.save(tag);

// Query by organization
List<Tag> orgTags = tagRepository.findByOrganizationId("org-123");
```

The class uses Lombok's `@Data` annotation for automatic getter/setter generation and Spring Data MongoDB annotations for persistence configuration. The unique index on the `name` field prevents duplicate tag names within the collection.
