A MongoDB entity representing tags that can be applied to documents and other resources within an organization, with unique name constraints and creation tracking.

## Key Components

- **@Document(collection = "tags")** - MongoDB collection mapping for tag storage
- **@Id String id** - Primary key for the tag entity
- **@Indexed(unique = true) String name** - Unique tag name within the system
- **String description** - Optional detailed description of the tag's purpose
- **String color** - Optional hex color code for UI display
- **String organizationId** - Scopes tags to specific organizations for multi-tenancy
- **Instant createdAt** - Timestamp tracking when the tag was created
- **String createdBy** - User identifier who created the tag

## Usage Example

```java
// Create a new tag for project categorization
Tag projectTag = new Tag();
projectTag.setName("high-priority");
projectTag.setDescription("Items requiring immediate attention");
projectTag.setColor("#FF6B6B");
projectTag.setOrganizationId("org-123");
projectTag.setCreatedAt(Instant.now());
projectTag.setCreatedBy("user-456");

// The unique index on 'name' ensures no duplicate tag names
// The organizationId allows different orgs to have their own tag namespaces
```
