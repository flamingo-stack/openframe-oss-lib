A data model representing contact information for an organization, including contact persons and addresses. This is designed as an embedded document rather than a standalone database entity.

## Key Components

- **ContactInformation** - Main data class with Lombok annotations for automatic getters/setters/builders
- **contacts** - List of ContactPerson objects representing organization contacts
- **physicalAddress** - Address object for the organization's physical location
- **mailingAddress** - Address object for mailing correspondence
- **mailingAddressSameAsPhysical** - Boolean flag to indicate address duplication

## Usage Example

```java
// Create contact information with builder pattern
ContactInformation contactInfo = ContactInformation.builder()
    .contacts(List.of(
        ContactPerson.builder()
            .name("John Doe")
            .email("john@example.com")
            .build()
    ))
    .physicalAddress(Address.builder()
        .street("123 Main St")
        .city("Springfield")
        .build())
    .mailingAddressSameAsPhysical(true)
    .build();

// Add additional contacts
contactInfo.getContacts().add(new ContactPerson("Jane Smith", "jane@example.com"));
```
