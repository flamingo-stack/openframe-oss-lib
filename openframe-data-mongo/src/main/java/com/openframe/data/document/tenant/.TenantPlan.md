Enumeration defining subscription tiers available for tenant organizations in the OpenFrame system. This enum provides a simple classification system for different service levels and billing plans.

## Key Components

- **FREE** - Basic tier with limited features and resources
- **BASIC** - Entry-level paid subscription with standard features
- **PREMIUM** - Advanced tier with enhanced capabilities and higher limits
- **ENTERPRISE** - Full-featured tier for large organizations with maximum resources

## Usage Example

```java
// Setting a tenant's subscription plan
TenantPlan currentPlan = TenantPlan.PREMIUM;

// Checking plan capabilities in service logic
if (tenant.getPlan() == TenantPlan.ENTERPRISE) {
    // Enable advanced features
    enableAdvancedAnalytics();
    increaseResourceLimits();
}

// Plan-based feature gating
public boolean canAccessFeature(TenantPlan plan, String feature) {
    return switch (plan) {
        case FREE -> basicFeatures.contains(feature);
        case BASIC -> basicFeatures.contains(feature) || standardFeatures.contains(feature);
        case PREMIUM -> !enterpriseOnlyFeatures.contains(feature);
        case ENTERPRISE -> true; // All features available
    };
}

// Upgrade/downgrade operations
TenantPlan newPlan = TenantPlan.valueOf("BASIC");
tenant.updatePlan(newPlan);
```
