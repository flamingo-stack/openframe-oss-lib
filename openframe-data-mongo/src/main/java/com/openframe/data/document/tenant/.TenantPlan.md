This enum defines the available subscription tiers for multi-tenant applications, representing different service levels that tenants can subscribe to.

## Key Components

- **FREE** - Entry-level tier with basic functionality
- **BASIC** - Standard paid tier with additional features
- **PREMIUM** - Advanced tier with enhanced capabilities
- **ENTERPRISE** - Top-tier plan with full feature access

## Usage Example

```java
// Setting a tenant's plan
Tenant tenant = new Tenant();
tenant.setPlan(TenantPlan.PREMIUM);

// Checking plan level for feature access
if (tenant.getPlan() == TenantPlan.ENTERPRISE) {
    enableAdvancedAnalytics();
}

// Switch statement for plan-based logic
switch (tenant.getPlan()) {
    case FREE:
        setStorageLimit(1_000_000); // 1GB
        break;
    case BASIC:
        setStorageLimit(10_000_000); // 10GB
        break;
    case PREMIUM:
        setStorageLimit(100_000_000); // 100GB
        break;
    case ENTERPRISE:
        setStorageLimit(-1); // Unlimited
        break;
}
```
