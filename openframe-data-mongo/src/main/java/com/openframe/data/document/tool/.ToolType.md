This enum defines the different types of remote management and monitoring tools supported by the document system.

## Key Components

- **MESHCENTRAL** - Represents MeshCentral remote management tool
- **TACTICAL_RMM** - Represents Tactical RMM (Remote Monitoring and Management) tool  
- **FLEET_MDM** - Represents Fleet MDM (Mobile Device Management) tool

## Usage Example

```java
import com.openframe.data.document.tool.ToolType;

// Using the enum in method parameters
public void configureRemoteTool(ToolType toolType) {
    switch (toolType) {
        case MESHCENTRAL:
            setupMeshCentralConnection();
            break;
        case TACTICAL_RMM:
            initializeTacticalRMM();
            break;
        case FLEET_MDM:
            configureMDMSettings();
            break;
    }
}

// Creating tool configurations
ToolType selectedTool = ToolType.MESHCENTRAL;
Document toolDocument = new Document(selectedTool);
```

This enum is typically used for type-safe selection and configuration of different remote management tools within the document processing system.
