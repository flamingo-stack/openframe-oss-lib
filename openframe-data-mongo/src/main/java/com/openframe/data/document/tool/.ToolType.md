This enum defines the supported remote management and monitoring tool types available in the OpenFrame document management system.

## Key Components

- **MESHCENTRAL** - Remote access and management platform for devices and systems
- **TACTICAL_RMM** - Tactical RMM (Remote Monitoring and Management) tool integration
- **FLEET_MDM** - Mobile Device Management solution for fleet management

## Usage Example

```java
import com.openframe.data.document.tool.ToolType;

// Check tool type in a switch statement
public void handleTool(ToolType toolType) {
    switch (toolType) {
        case MESHCENTRAL:
            // Configure MeshCentral integration
            break;
        case TACTICAL_RMM:
            // Setup Tactical RMM connection
            break;
        case FLEET_MDM:
            // Initialize Fleet MDM service
            break;
    }
}

// Use in configuration or service selection
ToolType selectedTool = ToolType.MESHCENTRAL;
String toolName = selectedTool.name(); // "MESHCENTRAL"
```

The enum provides type safety when working with different remote management tools throughout the OpenFrame platform, ensuring consistent tool identification and reducing errors in tool-specific implementations.
