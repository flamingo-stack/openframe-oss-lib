A MongoDB document entity representing external application events with metadata and payload information, designed for event tracking and auditing in distributed systems.

## Key Components

- **ExternalApplicationEvent** - Main entity class with MongoDB mapping
- **EventMetadata** - Nested static class containing event source information
- **Core Fields**:
  - `id` - Unique MongoDB identifier
  - `type` - Event type classification
  - `payload` - JSON string containing event data
  - `timestamp` - Event occurrence time
  - `userId` - Associated user identifier
  - `metadata` - Source, version, and tag information

## Usage Example

```java
// Creating a new external application event
ExternalApplicationEvent event = new ExternalApplicationEvent();
event.setType("user.login");
event.setPayload("{\"username\":\"john_doe\",\"ip\":\"192.168.1.1\"}");
event.setTimestamp(Instant.now());
event.setUserId("user123");

// Setting metadata
ExternalApplicationEvent.EventMetadata metadata = new ExternalApplicationEvent.EventMetadata();
metadata.setSource("mobile-app");
metadata.setVersion("1.0.0");
metadata.setTags(Map.of("environment", "production", "region", "us-east"));
event.setMetadata(metadata);

// Save to MongoDB repository
eventRepository.save(event);
```

The class uses Lombok annotations for automatic getter/setter generation and Spring Data MongoDB annotations for persistence mapping to the "external_application_events" collection.
