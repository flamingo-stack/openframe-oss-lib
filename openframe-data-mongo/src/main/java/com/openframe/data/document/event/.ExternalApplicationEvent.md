This class represents a MongoDB document entity for storing external application events, designed to capture and persist event data from external sources with associated metadata.

## Key Components

- **ExternalApplicationEvent**: Main entity class annotated with `@Document` for MongoDB persistence
- **EventMetadata**: Nested static class containing event source information, version, and custom tags
- **Fields**: 
  - `id`: MongoDB document identifier
  - `type`: Event classification
  - `payload`: Event data as string
  - `timestamp`: Event occurrence time
  - `userId`: Associated user identifier
  - `metadata`: Nested metadata object

## Usage Example

```java
// Creating a new external application event
ExternalApplicationEvent event = new ExternalApplicationEvent();
event.setType("USER_LOGIN");
event.setPayload("{\"username\":\"john.doe\",\"ip\":\"192.168.1.1\"}");
event.setTimestamp(Instant.now());
event.setUserId("user123");

// Setting metadata
ExternalApplicationEvent.EventMetadata metadata = new ExternalApplicationEvent.EventMetadata();
metadata.setSource("mobile-app");
metadata.setVersion("2.1.0");
metadata.setTags(Map.of("platform", "iOS", "region", "US"));
event.setMetadata(metadata);

// Save to MongoDB repository
externalEventRepository.save(event);
```

The class leverages Lombok's `@Data` annotation for automatic getter/setter generation and Spring Data MongoDB for persistence in the "external_application_events" collection.
