Enumeration defining the possible status values for tenants in the authorization server system. This enum provides standardized status indicators to track whether a tenant is currently operational or disabled.

## Key Components

- **ACTIVE** - Indicates the tenant is currently operational and can access system resources
- **INACTIVE** - Indicates the tenant is disabled and cannot access system resources

## Usage Example

```java
// Setting tenant status
TenantStatus status = TenantStatus.ACTIVE;

// Conditional logic based on status
if (tenant.getStatus() == TenantStatus.ACTIVE) {
    // Allow tenant operations
    processRequest(tenant);
} else {
    throw new TenantInactiveException("Tenant access denied");
}

// Switch statement for status handling
switch (tenantStatus) {
    case ACTIVE:
        enableTenantServices();
        break;
    case INACTIVE:
        disableTenantServices();
        break;
}
```
