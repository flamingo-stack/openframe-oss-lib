An enumeration that defines the possible status values for tenant entities in the OpenFrame authorization server system.

## Key Components

- **ACTIVE** - Indicates a tenant is currently active and operational
- **INACTIVE** - Indicates a tenant is currently inactive or disabled

## Usage Example

```java
// Setting tenant status during creation
Tenant tenant = new Tenant();
tenant.setStatus(TenantStatus.ACTIVE);

// Checking if a tenant is active
if (tenant.getStatus() == TenantStatus.ACTIVE) {
    // Process active tenant operations
    processActiveTenant(tenant);
}

// Deactivating a tenant
tenant.setStatus(TenantStatus.INACTIVE);

// Using in conditional logic
switch (tenant.getStatus()) {
    case ACTIVE:
        enableTenantServices(tenant);
        break;
    case INACTIVE:
        disableTenantServices(tenant);
        break;
}
```

This enum provides type safety and clarity when managing tenant lifecycle states throughout the authorization server, ensuring consistent status handling across the application.
