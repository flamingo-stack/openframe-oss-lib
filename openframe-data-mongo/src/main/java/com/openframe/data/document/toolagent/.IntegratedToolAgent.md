A MongoDB entity class representing an integrated tool agent configuration with installation, runtime, and management settings.

## Key Components

- **@Document**: Maps to "integrated_tool_agents" MongoDB collection
- **Core Properties**: `id`, `toolId`, `version`, `releaseVersion` for tool identification
- **Session Management**: `sessionType` enum for session handling
- **Configuration Lists**: `downloadConfigurations` and `assets` for tool resources  
- **Command Arguments**: Installation, run, agent tool ID, and uninstallation command lists
- **Update Flags**: `allowVersionUpdate` and `allowConfigurationUpdate` for maintenance control
- **Status Tracking**: `status` enum for current tool agent state

## Usage Example

```java
@Autowired
private MongoTemplate mongoTemplate;

// Create new tool agent
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setToolId("my-tool-v1");
agent.setVersion("1.0.0");
agent.setReleaseVersion(true);
agent.setSessionType(SessionType.PERSISTENT);
agent.setInstallationCommandArgs(List.of("install", "--config=prod"));
agent.setRunCommandArgs(List.of("run", "--port=8080"));
agent.setAllowVersionUpdate(true);
agent.setStatus(ToolAgentStatus.ACTIVE);

mongoTemplate.save(agent);

// Query by tool ID
IntegratedToolAgent found = mongoTemplate.findOne(
    Query.query(Criteria.where("toolId").is("my-tool-v1")), 
    IntegratedToolAgent.class
);
```
