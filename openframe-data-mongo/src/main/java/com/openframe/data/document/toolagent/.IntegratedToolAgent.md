A MongoDB document entity representing integrated tool agents in the OpenFrame platform, managing tool deployment configurations and execution parameters.

## Key Components

**Core Fields:**
- `toolId` - References the associated tool this agent manages
- `sessionType` - Defines the type of session (enum value)
- `releaseVersion` - Boolean flag indicating if this is a release version
- `version` - Version string for the tool agent

**Configuration Management:**
- `downloadConfigurations` - List of download settings for the tool
- `assets` - Associated tool agent assets
- `publishState` - Current publication status (from client configuration)

**Command Execution:**
- `installationCommandArgs` - Arguments for installation commands
- `runCommandArgs` - Arguments for running the tool
- `agentToolIdCommandArgs` - Tool ID-specific command arguments  
- `uninstallationCommandArgs` - Arguments for uninstallation

**Control Flags:**
- `allowVersionUpdate` - Whether version updates are permitted
- `allowConfigurationUpdate` - Whether configuration changes are allowed
- `status` - Current operational status of the tool agent

## Usage Example

```java
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setToolId("monitoring-tool-v2");
agent.setVersion("2.1.0");
agent.setReleaseVersion(true);
agent.setSessionType(SessionType.PERSISTENT);

// Configure installation
agent.setInstallationCommandArgs(List.of("install", "--config", "/opt/config"));
agent.setRunCommandArgs(List.of("run", "--daemon"));
agent.setUninstallationCommandArgs(List.of("uninstall", "--cleanup"));

// Set update permissions
agent.setAllowVersionUpdate(true);
agent.setAllowConfigurationUpdate(false);
agent.setStatus(ToolAgentStatus.ACTIVE);
```
