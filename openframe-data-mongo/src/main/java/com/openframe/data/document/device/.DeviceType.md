Enumeration defining the different types of devices that can be managed within the OpenFrame system. This enum provides standardized device categorization for inventory management, monitoring, and configuration purposes.

## Key Components

- **DESKTOP** - Desktop workstation computers
- **LAPTOP** - Portable laptop computers  
- **SERVER** - Server hardware systems
- **MOBILE_DEVICE** - Smartphones and mobile devices
- **TABLET** - Tablet computers and similar devices
- **NETWORK_DEVICE** - Routers, switches, and networking equipment
- **IOT_DEVICE** - Internet of Things connected devices
- **VIRTUAL_MACHINE** - Virtualized computer instances
- **CONTAINER_HOST** - Systems hosting containerized applications
- **OTHER** - Fallback category for unclassified devices

## Usage Example

```java
// Device classification during registration
DeviceType deviceType = DeviceType.LAPTOP;

// Switch statement for device-specific handling
switch (device.getType()) {
    case SERVER:
        // Apply server-specific monitoring
        break;
    case MOBILE_DEVICE:
        // Configure mobile device policies
        break;
    case IOT_DEVICE:
        // Set up IoT device communication protocols
        break;
    default:
        // Handle generic device operations
        break;
}

// Filtering devices by type
List<Device> servers = devices.stream()
    .filter(d -> d.getType() == DeviceType.SERVER)
    .collect(Collectors.toList());
```
