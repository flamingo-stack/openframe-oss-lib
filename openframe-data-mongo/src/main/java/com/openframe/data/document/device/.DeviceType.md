An enumeration that defines standardized device type categories for classification and filtering within the OpenFrame document system.

## Key Components

- **DESKTOP** - Traditional desktop computers
- **LAPTOP** - Portable laptop computers  
- **SERVER** - Server hardware systems
- **MOBILE_DEVICE** - Smartphones and mobile devices
- **TABLET** - Tablet devices
- **NETWORK_DEVICE** - Routers, switches, and network equipment
- **IOT_DEVICE** - Internet of Things devices and sensors
- **VIRTUAL_MACHINE** - Virtualized computing instances
- **CONTAINER_HOST** - Container orchestration platforms
- **OTHER** - Fallback category for unclassified devices

## Usage Example

```java
import com.openframe.data.document.device.DeviceType;

// Device classification
DeviceType userDevice = DeviceType.LAPTOP;
DeviceType serverType = DeviceType.SERVER;

// Switch-based logic
switch (deviceType) {
    case MOBILE_DEVICE:
    case TABLET:
        enableMobileFeatures();
        break;
    case DESKTOP:
    case LAPTOP:
        enableDesktopFeatures();
        break;
    case IOT_DEVICE:
        enableIoTProtocols();
        break;
    default:
        useDefaultConfiguration();
}

// Filtering devices by type
List<Device> mobileDevices = devices.stream()
    .filter(d -> d.getType() == DeviceType.MOBILE_DEVICE)
    .collect(Collectors.toList());
```
