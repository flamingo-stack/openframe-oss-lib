A MongoDB document entity representing API keys with authentication, authorization, and lifecycle management capabilities. This class provides secure storage and validation for API credentials used in the OpenFrame system.

## Key Components

- **@Document(collection = "api_keys")** - MongoDB collection mapping for API key storage
- **Security Fields** - `hashedKey` for secure credential storage, `userId` for ownership tracking
- **Authorization** - `scopes` and `roles` lists for permission-based access control  
- **Lifecycle Management** - `enabled`, `createdAt`, `updatedAt`, and `expiresAt` fields for key management
- **Validation Methods** - `isExpired()` and `isActive()` for runtime key status checking
- **Metadata** - `name` and `description` fields for human-readable identification

## Usage Example

```java
// Create a new API key
ApiKey apiKey = ApiKey.builder()
    .keyId("key_123")
    .hashedKey("$2a$10$hashed_value")
    .name("Production API Key")
    .description("Key for production service access")
    .userId("user_456")
    .scopes(List.of("read:data", "write:reports"))
    .roles(List.of("api_user"))
    .enabled(true)
    .createdAt(Instant.now())
    .expiresAt(Instant.now().plusSeconds(86400 * 30)) // 30 days
    .build();

// Validate key status
if (apiKey.isActive()) {
    // Process API request
    System.out.println("API key is valid and active");
} else {
    System.out.println("API key is disabled or expired");
}
```
