A MongoDB document entity that represents the relationship between machines and tags, storing when and by whom a machine was tagged.

## Key Components

- **MachineTag**: Main entity class with compound index on machineId and tagId
- **Fields**:
  - `id`: Primary key identifier
  - `machineId`: Reference to the machine being tagged
  - `tagId`: Reference to the applied tag
  - `taggedAt`: Timestamp when the tag was applied
  - `taggedBy`: User who applied the tag
- **Compound Index**: Ensures unique machine-tag combinations for data integrity

## Usage Example

```java
// Create a new machine tag relationship
MachineTag machineTag = new MachineTag();
machineTag.setMachineId("machine-123");
machineTag.setTagId("tag-456");
machineTag.setTaggedAt(Instant.now());
machineTag.setTaggedBy("user@example.com");

// Repository usage (assuming standard Spring Data MongoDB repository)
machineTagRepository.save(machineTag);

// Query by machine ID
List<MachineTag> machineTags = machineTagRepository.findByMachineId("machine-123");
```

The compound index prevents duplicate tag assignments to the same machine, making it ideal for implementing a many-to-many relationship between machines and tags with audit trail information.
