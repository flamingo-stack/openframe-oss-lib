A data model class representing address information designed as an embedded document for MongoDB collections rather than a standalone entity.

## Key Components

- **Address Class**: Main data model with Lombok annotations for automatic getter/setter generation
- **Fields**: 
  - `street1` - Primary street address line (required)
  - `street2` - Secondary address line (optional)
  - `city` - City name
  - `state` - State or province
  - `postalCode` - ZIP or postal code
  - `country` - Country name
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for boilerplate code generation

## Usage Example

```java
// Create address using builder pattern
Address address = Address.builder()
    .street1("123 Main Street")
    .street2("Apt 4B")
    .city("New York")
    .state("NY")
    .postalCode("10001")
    .country("USA")
    .build();

// Create with constructor
Address simpleAddress = new Address(
    "456 Oak Ave", null, "Boston", "MA", "02101", "USA"
);

// Use as embedded document in organization
Organization org = Organization.builder()
    .name("Tech Corp")
    .address(address)
    .build();
```
