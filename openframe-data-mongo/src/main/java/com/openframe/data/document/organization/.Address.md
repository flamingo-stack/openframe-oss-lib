A data model class representing address information that is designed to be embedded within other documents rather than stored as a separate collection.

## Key Components

- **Address** - Main data class with Lombok annotations for automatic getter/setter generation
- **street1** - Primary street address line (required)
- **street2** - Secondary address line (optional)
- **city** - City name
- **state** - State or province
- **postalCode** - ZIP or postal code
- **country** - Country name

## Usage Example

```java
// Create address using builder pattern
Address address = Address.builder()
    .street1("123 Main Street")
    .street2("Apt 4B")
    .city("New York")
    .state("NY")
    .postalCode("10001")
    .country("USA")
    .build();

// Create minimal address
Address simpleAddress = Address.builder()
    .street1("456 Oak Avenue")
    .city("Los Angeles")
    .state("CA")
    .postalCode("90210")
    .country("USA")
    .build();

// Use in organization documents
Organization org = Organization.builder()
    .name("Tech Company")
    .address(address)
    .build();
```

The class uses Lombok annotations to automatically generate constructors, getters, setters, and builder methods, making it easy to create and manipulate address objects throughout the application.
