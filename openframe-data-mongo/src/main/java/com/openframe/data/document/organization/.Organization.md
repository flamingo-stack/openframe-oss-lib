A MongoDB document entity representing organizations/companies in the system with business information, contract details, and soft delete functionality.

## Key Components

### Main Fields
- `organizationId` - Unique immutable UUID identifier (indexed)
- `name` - Organization/company name (indexed)
- `isDefault` - Flag for tenant's default organization (indexed)
- `category` - Business industry/category
- `numberOfEmployees` - Employee count
- `contactInformation` - Contact details and addresses
- `monthlyRevenue` - Financial information as BigDecimal

### Contract Management
- `contractStartDate` / `contractEndDate` - Contract validity period
- `isContractActive()` - Method to check if contract is currently valid

### Soft Delete Support
- `deleted` - Soft delete flag (indexed)
- `deletedAt` - Deletion timestamp
- `isDeleted()` - Convenience method to check deletion status

### Audit Fields
- `createdAt` / `updatedAt` - Automatic timestamp tracking via Spring Data

## Usage Example

```java
// Create a new organization
Organization org = Organization.builder()
    .name("Tech Corp")
    .organizationId("uuid-123-456")
    .isDefault(false)
    .category("Technology")
    .numberOfEmployees(150)
    .monthlyRevenue(new BigDecimal("50000.00"))
    .contractStartDate(LocalDate.now())
    .contractEndDate(LocalDate.now().plusYears(1))
    .build();

// Check contract status
if (org.isContractActive()) {
    System.out.println("Contract is valid");
}

// Soft delete
org.setDeleted(true);
org.setDeletedAt(Instant.now());
```
