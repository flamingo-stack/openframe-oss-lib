A data transfer object that defines filtering criteria for querying events, supporting user-based, type-based, and date range filtering.

## Key Components

- **userIds** - List of user identifiers to filter events by specific users
- **eventTypes** - List of event type strings to filter by event categories
- **startDate** - Beginning date boundary for date range filtering
- **endDate** - Ending date boundary for date range filtering

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern support.

## Usage Example

```java
// Create filter using builder pattern
EventQueryFilter filter = EventQueryFilter.builder()
    .userIds(List.of("user123", "user456"))
    .eventTypes(List.of("LOGIN", "LOGOUT", "PURCHASE"))
    .startDate(LocalDate.of(2024, 1, 1))
    .endDate(LocalDate.of(2024, 12, 31))
    .build();

// Create filter for specific date range only
EventQueryFilter dateFilter = EventQueryFilter.builder()
    .startDate(LocalDate.now().minusDays(30))
    .endDate(LocalDate.now())
    .build();

// Use with event query service
List<Event> events = eventService.findEvents(filter);
```
