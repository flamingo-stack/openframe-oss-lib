A simple data class representing security alerts with tracking information for detection, resolution status, and severity levels.

## Key Components

- **SecurityAlert** - Main data class with Lombok `@Data` annotation for automatic getters/setters
- **id** - Unique identifier for the security alert
- **severity** - Alert severity level (e.g., "HIGH", "MEDIUM", "LOW")
- **description** - Detailed description of the security issue
- **detectedAt** - Timestamp when the alert was first detected
- **resolved** - Boolean flag indicating if the alert has been addressed
- **resolvedAt** - Timestamp when the alert was resolved (null if unresolved)
- **resolution** - Description of how the alert was resolved

## Usage Example

```java
// Create a new security alert
SecurityAlert alert = new SecurityAlert();
alert.setId("SEC-001");
alert.setSeverity("HIGH");
alert.setDescription("Unauthorized access attempt detected");
alert.setDetectedAt(Instant.now());
alert.setResolved(false);

// Later, when resolving the alert
alert.setResolved(true);
alert.setResolvedAt(Instant.now());
alert.setResolution("IP address blocked and user account suspended");

// Check alert status
if (!alert.isResolved()) {
    System.out.println("Alert " + alert.getId() + " requires attention");
}
```
