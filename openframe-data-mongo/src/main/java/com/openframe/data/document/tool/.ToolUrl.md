A simple data class representing a URL configuration for tools in the OpenFrame system, combining URL, port, and type information into a structured object.

## Key Components

- **url** - String field storing the base URL
- **port** - String field for the port number
- **type** - ToolUrlType enum indicating the URL's purpose or protocol
- **Lombok annotations** - Auto-generates constructors, getters, setters, and builder pattern

## Usage Example

```java
// Using builder pattern
ToolUrl toolUrl = ToolUrl.builder()
    .url("https://api.example.com")
    .port("8080")
    .type(ToolUrlType.REST_API)
    .build();

// Using constructor
ToolUrl toolUrl = new ToolUrl("https://api.example.com", "8080", ToolUrlType.REST_API);

// Accessing properties
String endpoint = toolUrl.getUrl() + ":" + toolUrl.getPort();
ToolUrlType urlType = toolUrl.getType();
```

This class serves as a configuration container for tool endpoints, allowing the system to manage different types of tool URLs with their associated port configurations in a type-safe manner.
