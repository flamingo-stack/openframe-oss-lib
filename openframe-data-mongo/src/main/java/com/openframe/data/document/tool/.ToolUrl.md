A simple data class that represents a tool URL configuration with connection details and type classification.

## Key Components

- **ToolUrl** - Main data class with Lombok annotations for automatic getter/setter generation
- **url** - String field for the base URL or hostname
- **port** - String field for the connection port
- **type** - ToolUrlType enum field for categorizing the tool type

## Usage Example

```java
// Create using builder pattern
ToolUrl toolUrl = ToolUrl.builder()
    .url("localhost")
    .port("8080")
    .type(ToolUrlType.DATABASE)
    .build();

// Create using constructor
ToolUrl toolUrl = new ToolUrl("https://api.example.com", "443", ToolUrlType.API);

// Access properties
String fullEndpoint = toolUrl.getUrl() + ":" + toolUrl.getPort();
ToolUrlType toolType = toolUrl.getType();

// Modify properties
toolUrl.setPort("9090");
toolUrl.setUrl("staging.example.com");
```

The class leverages Lombok annotations to automatically generate boilerplate code including constructors, getters, setters, equals, hashCode, and toString methods, making it ideal for configuration objects and data transfer scenarios.
