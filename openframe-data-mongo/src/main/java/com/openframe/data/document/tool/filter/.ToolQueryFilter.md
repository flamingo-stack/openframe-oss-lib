A data transfer object for filtering tool queries, providing criteria-based search parameters for tool discovery and retrieval operations.

## Key Components

- **enabled** - Boolean flag to filter tools by their active/inactive status
- **type** - String field for filtering by tool type classification
- **category** - String field for filtering by tool category
- **platformCategory** - String field for filtering by platform-specific categories
- Uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create filter using builder pattern
ToolQueryFilter filter = ToolQueryFilter.builder()
    .enabled(true)
    .type("API")
    .category("INTEGRATION")
    .platformCategory("CLOUD")
    .build();

// Create filter for active tools only
ToolQueryFilter activeFilter = ToolQueryFilter.builder()
    .enabled(true)
    .build();

// Create empty filter (no criteria)
ToolQueryFilter noFilter = new ToolQueryFilter();

// Use in a service method
List<Tool> tools = toolService.findTools(filter);
```
