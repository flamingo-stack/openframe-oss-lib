A configuration data class that defines download settings for client applications, including operating system specifications and file naming conventions.

## Key Components

- **os** - Target operating system identifier for the download
- **linkTemplate** - URL template pattern for generating download links
- **fileName** - Name of the main file to be downloaded
- **agentFileName** - Name of the agent/auxiliary file associated with the download

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods.

## Usage Example

```java
// Creating a download configuration for Windows
DownloadConfiguration config = new DownloadConfiguration();
config.setOs("windows");
config.setLinkTemplate("https://downloads.example.com/{version}/{os}/");
config.setFileName("application-installer.exe");
config.setAgentFileName("setup-agent.exe");

// Using in a download service
String downloadUrl = config.getLinkTemplate()
    .replace("{os}", config.getOs())
    .replace("{version}", "1.2.3");

// Access configuration properties
System.out.println("Target OS: " + config.getOs());
System.out.println("Main file: " + config.getFileName());
```

This class is typically used in client configuration systems where different download options need to be managed based on operating system requirements.
