A simple data transfer object that defines the configuration parameters needed for downloading OpenFrame client files, including platform-specific details and file naming conventions.

## Key Components

- **os** - Target operating system identifier (e.g., "windows", "linux", "macos")
- **linkTemplate** - URL template or pattern for generating download links
- **fileName** - Name of the main downloadable file
- **agentFileName** - Name of the agent-specific file component

## Usage Example

```java
// Create download configuration for Windows client
DownloadConfiguration config = new DownloadConfiguration();
config.setOs("windows");
config.setLinkTemplate("https://releases.openframe.com/{version}/openframe-{os}.zip");
config.setFileName("openframe-installer.exe");
config.setAgentFileName("openframe-agent.jar");

// Use in download service
String downloadUrl = config.getLinkTemplate()
    .replace("{os}", config.getOs())
    .replace("{version}", "1.2.3");

// Access file names for UI display
String mainFile = config.getFileName();
String agentFile = config.getAgentFileName();
```

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods, making it ideal for configuration mapping and serialization scenarios.
