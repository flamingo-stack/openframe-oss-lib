A Java data class that defines the configuration settings for downloadable client software packages, specifying platform-specific download parameters and installation requirements.

## Key Components

### Fields
- **`os`** - Target operating system identifier (e.g., "windows", "linux", "macos")
- **`linkTemplate`** - URL template pattern for generating download links
- **`fileName`** - Original name of the downloadable file
- **`targetFileName`** - Desired filename after download/installation
- **`installationType`** - Installation method (defaults to `InstallationType.STANDARD`)
- **`bundleId`** - Unique identifier for the software bundle

### Annotations
- **`@Data`** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create download configuration for Windows client
DownloadConfiguration config = new DownloadConfiguration();
config.setOs("windows");
config.setLinkTemplate("https://releases.openframe.ai/client/{version}/openframe-{os}.exe");
config.setFileName("openframe-windows-v1.2.3.exe");
config.setTargetFileName("OpenFrame-Setup.exe");
config.setInstallationType(InstallationType.MSI);
config.setBundleId("com.openframe.client.windows");

// Configuration automatically provides getters/setters via Lombok
String downloadUrl = buildDownloadUrl(config.getLinkTemplate(), config.getOs());
```

This configuration class is typically used by OpenFrame's client distribution system to manage platform-specific software packages and their download metadata.
