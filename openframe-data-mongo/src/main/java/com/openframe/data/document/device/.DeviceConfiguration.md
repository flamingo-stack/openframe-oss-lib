A data model class that represents the configuration settings and metadata for a device, including system settings, installed software, network configuration, and security parameters.

## Key Components

- **settings** - Map storing key-value pairs for general device settings
- **installedSoftware** - List of installed software packages or applications
- **networkConfig** - Map containing network-related configuration parameters
- **security** - SecuritySettings object containing security-related configurations
- **@Data** - Lombok annotation providing automatic getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a new device configuration
DeviceConfiguration config = new DeviceConfiguration();

// Set general device settings
Map<String, String> settings = Map.of(
    "display_brightness", "75",
    "auto_sleep", "30",
    "timezone", "UTC"
);
config.setSettings(settings);

// Configure installed software
List<String> software = List.of(
    "OpenFrameOS v2.1",
    "WebKit Browser",
    "Media Player"
);
config.setInstalledSoftware(software);

// Set network configuration
Map<String, String> network = Map.of(
    "wifi_ssid", "OpenFrame-Network",
    "ip_address", "192.168.1.100",
    "dns_server", "8.8.8.8"
);
config.setNetworkConfig(network);

// Apply security settings
SecuritySettings securitySettings = new SecuritySettings();
config.setSecurity(securitySettings);
```
