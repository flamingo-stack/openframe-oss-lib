A data model class that represents the configuration settings and software state of a device, using Lombok annotations for automatic getter/setter generation.

## Key Components

- **settings** - Map containing general device configuration key-value pairs
- **installedSoftware** - List of installed software packages or applications
- **networkConfig** - Map storing network-related configuration parameters
- **security** - SecuritySettings object containing security-specific configurations
- **@Data** - Lombok annotation that generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create a new device configuration
DeviceConfiguration config = new DeviceConfiguration();

// Set general settings
Map<String, String> settings = new HashMap<>();
settings.put("timezone", "UTC");
settings.put("language", "en-US");
config.setSettings(settings);

// Configure installed software
List<String> software = Arrays.asList("antivirus-v2.1", "office-suite", "browser");
config.setInstalledSoftware(software);

// Set network configuration
Map<String, String> network = new HashMap<>();
network.put("dhcp", "enabled");
network.put("dns", "8.8.8.8");
config.setNetworkConfig(network);

// Access configuration data
String timezone = config.getSettings().get("timezone");
boolean hasAntivirus = config.getInstalledSoftware().contains("antivirus-v2.1");
```
