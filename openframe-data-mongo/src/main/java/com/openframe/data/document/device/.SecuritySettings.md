A data class that represents security configuration and status information for a device, including firewall settings, antivirus status, and security policies.

## Key Components

- **firewallEnabled** - Boolean flag indicating if firewall protection is active
- **antivirusEnabled** - Boolean flag indicating if antivirus protection is active  
- **encryptionStatus** - String describing current encryption state
- **lastSecurityScan** - String timestamp of the most recent security scan
- **securityPolicies** - Map containing key-value pairs of security policy configurations

## Usage Example

```java
SecuritySettings settings = new SecuritySettings();
settings.setFirewallEnabled(true);
settings.setAntivirusEnabled(true);
settings.setEncryptionStatus("AES-256 Enabled");
settings.setLastSecurityScan("2024-01-15T10:30:00Z");

Map<String, String> policies = new HashMap<>();
policies.put("passwordPolicy", "strong");
policies.put("accessControl", "role-based");
settings.setSecurityPolicies(policies);

// Access security information
if (settings.isFirewallEnabled()) {
    System.out.println("Firewall is active");
}
```
