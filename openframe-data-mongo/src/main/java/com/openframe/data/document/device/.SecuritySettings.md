A data class representing security configuration settings for OpenFrame devices, containing firewall status, antivirus protection, encryption details, and security policies.

## Key Components

- **firewallEnabled** - Boolean flag indicating firewall protection status
- **antivirusEnabled** - Boolean flag for antivirus software status  
- **encryptionStatus** - String describing current encryption state
- **lastSecurityScan** - Timestamp of the most recent security scan
- **securityPolicies** - Map of security policy names to their configuration values

## Usage Example

```java
SecuritySettings settings = new SecuritySettings();
settings.setFirewallEnabled(true);
settings.setAntivirusEnabled(true);
settings.setEncryptionStatus("AES-256 Enabled");
settings.setLastSecurityScan("2024-01-15T10:30:00Z");

Map<String, String> policies = new HashMap<>();
policies.put("passwordPolicy", "strong");
policies.put("sessionTimeout", "30min");
settings.setSecurityPolicies(policies);

// Check security status
if (settings.isFirewallEnabled() && settings.isAntivirusEnabled()) {
    System.out.println("Device security is properly configured");
}
```

This class uses Lombok's `@Data` annotation to automatically generate getters, setters, equals, hashCode, and toString methods, making it ideal for data transfer and configuration management in device security monitoring systems.
