A simple configuration class that stores local filename and operating system information for the document tool agent system.

## Key Components

- **filename** - String field storing the local filename path or name
- **os** - String field indicating the target operating system
- **@Data** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create and configure a local filename setting
LocalFilenameConfiguration config = new LocalFilenameConfiguration();
config.setFilename("/home/user/documents/report.pdf");
config.setOs("linux");

// Use in tool agent operations
String targetFile = config.getFilename();
String targetOS = config.getOs();

// Configuration can be passed to document processing tools
if ("windows".equals(config.getOs())) {
    // Handle Windows-specific path formatting
    String windowsPath = config.getFilename().replace("/", "\\");
}
```

This configuration class is typically used within the OpenFrame document tool agent system to specify local file operations that may need OS-specific handling, such as path separators or file system permissions.
