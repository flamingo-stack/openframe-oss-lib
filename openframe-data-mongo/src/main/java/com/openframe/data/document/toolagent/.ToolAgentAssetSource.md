A simple enumeration that defines the available source types for tool agent assets within the OpenFrame document management system.

## Key Components

- **ARTIFACTORY** - Represents assets sourced from an Artifactory repository
- **TOOL_API** - Represents assets sourced from a Tool API endpoint

## Usage Example

```java
import com.openframe.data.document.toolagent.ToolAgentAssetSource;

public class AssetManager {
    
    public void processAsset(ToolAgentAssetSource source) {
        switch (source) {
            case ARTIFACTORY:
                // Handle Artifactory-sourced assets
                loadFromArtifactory();
                break;
            case TOOL_API:
                // Handle Tool API-sourced assets
                loadFromToolApi();
                break;
        }
    }
    
    public boolean isRepositorySource(ToolAgentAssetSource source) {
        return source == ToolAgentAssetSource.ARTIFACTORY;
    }
}
```

This enum provides type safety when specifying asset sources and enables clean conditional logic for handling different asset retrieval mechanisms in tool agent workflows.
