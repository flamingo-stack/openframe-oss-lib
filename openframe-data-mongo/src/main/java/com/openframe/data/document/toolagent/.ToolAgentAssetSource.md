This enum defines the available sources from which tool agent assets can be retrieved in the OpenFrame document system.

## Key Components

- **ARTIFACTORY** - Indicates assets are sourced from an Artifactory repository
- **TOOL_API** - Indicates assets are sourced directly from a tool's API endpoint

## Usage Example

```java
import com.openframe.data.document.toolagent.ToolAgentAssetSource;

public class AssetManager {
    
    public void configureAssetSource(ToolAgentAssetSource source) {
        switch (source) {
            case ARTIFACTORY:
                // Configure Artifactory connection
                setupArtifactoryClient();
                break;
            case TOOL_API:
                // Configure direct API access
                setupToolApiClient();
                break;
        }
    }
    
    public boolean isArtifactorySource(ToolAgentAssetSource source) {
        return source == ToolAgentAssetSource.ARTIFACTORY;
    }
}
```

This enum is typically used in configuration classes or service methods to specify where tool agent assets should be fetched from, allowing the system to handle different asset retrieval strategies based on the source type.
