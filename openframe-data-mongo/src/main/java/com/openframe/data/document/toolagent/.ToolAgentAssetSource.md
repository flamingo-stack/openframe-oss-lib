A simple enumeration that defines the available sources from which tool agent assets can be retrieved in the OpenFrame platform.

## Key Components

- **ARTIFACTORY** - Assets sourced from Artifactory repository manager
- **TOOL_API** - Assets retrieved through dedicated tool APIs
- **GITHUB** - Assets pulled from GitHub repositories

## Usage Example

```java
// Specify asset source when configuring tool agent
ToolAgentAssetSource source = ToolAgentAssetSource.GITHUB;

// Use in conditional logic
switch (assetSource) {
    case ARTIFACTORY:
        // Handle Artifactory-specific asset retrieval
        break;
    case TOOL_API:
        // Handle API-based asset fetching
        break;
    case GITHUB:
        // Handle GitHub repository access
        break;
}

// Or in method parameters
public void downloadAsset(String assetId, ToolAgentAssetSource source) {
    // Implementation varies based on source type
}
```

This enum provides type safety and clarity when specifying where tool agent assets should be sourced from, supporting OpenFrame's flexible asset management across different repositories and APIs.
