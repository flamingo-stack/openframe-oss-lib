A data transfer object that encapsulates authentication credentials for external tools, supporting both traditional username/password authentication and API key-based authentication methods.

## Key Components

- **username** - String field for traditional authentication username
- **password** - String field for traditional authentication password  
- **apiKey** - ToolApiKey object for API-based authentication
- **@Data** - Lombok annotation providing automatic getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Traditional username/password authentication
ToolCredentials basicAuth = new ToolCredentials();
basicAuth.setUsername("admin");
basicAuth.setPassword("secretPassword");

// API key authentication
ToolApiKey apiKey = new ToolApiKey();
// Configure apiKey properties...

ToolCredentials apiAuth = new ToolCredentials();
apiAuth.setApiKey(apiKey);

// Mixed authentication (if tool supports both)
ToolCredentials mixedAuth = new ToolCredentials();
mixedAuth.setUsername("serviceAccount");
mixedAuth.setApiKey(apiKey);

// Accessing credentials
String user = mixedAuth.getUsername();
ToolApiKey key = mixedAuth.getApiKey();
```
