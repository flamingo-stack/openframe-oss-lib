A simple data model class that encapsulates authentication credentials required for tool integrations, supporting both traditional username/password authentication and API key-based authentication.

## Key Components

- **username/password fields** - String properties for basic authentication
- **apiKey** - ToolApiKey object for API key-based authentication
- **@Data annotation** - Lombok annotation that automatically generates getters, setters, toString, equals, and hashCode methods

## Usage Example

```java
// Create credentials with username/password
ToolCredentials basicAuth = new ToolCredentials();
basicAuth.setUsername("john.doe");
basicAuth.setPassword("secret123");

// Create credentials with API key
ToolCredentials apiAuth = new ToolCredentials();
ToolApiKey key = new ToolApiKey();
// Configure the API key object
apiAuth.setApiKey(key);

// Use in tool configuration
SomeTool tool = new SomeTool();
tool.authenticate(basicAuth);
```

This class serves as a flexible credential container that can be used across different tool integrations where various authentication mechanisms may be required.
