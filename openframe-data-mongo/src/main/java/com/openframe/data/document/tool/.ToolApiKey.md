A simple data model representing an API key configuration for tools, containing the key value, type, and display name.

## Key Components

- **key** - The actual API key string value
- **type** - An enum (`APIKeyType`) specifying the type/provider of the API key
- **keyName** - A human-readable name or identifier for the API key
- **@Data** - Lombok annotation providing auto-generated getters, setters, equals, hashCode, and toString methods

## Usage Example

```java
// Create a new tool API key
ToolApiKey apiKey = new ToolApiKey();
apiKey.setKey("sk-abc123def456");
apiKey.setType(APIKeyType.OPENAI);
apiKey.setKeyName("Production OpenAI Key");

// Access the properties
String keyValue = apiKey.getKey();
APIKeyType keyType = apiKey.getType();
String displayName = apiKey.getKeyName();

// Use in tool configuration
if (apiKey.getType() == APIKeyType.OPENAI) {
    // Configure OpenAI tool with this key
    configureTool(apiKey.getKey());
}
```

This class is typically used to store and manage API key credentials for various external tools and services within the application.
