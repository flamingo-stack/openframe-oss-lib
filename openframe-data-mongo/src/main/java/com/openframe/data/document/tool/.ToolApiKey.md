A simple data model class that represents an API key configuration for tools within the OpenFrame system.

## Key Components

- **key** - The actual API key value as a string
- **type** - An enum value from `APIKeyType` indicating the type/provider of the API key
- **keyName** - A human-readable identifier for the API key

## Usage Example

```java
import com.openframe.data.document.tool.ToolApiKey;
import com.openframe.data.document.apikey.APIKeyType;

// Create a new tool API key
ToolApiKey apiKey = new ToolApiKey();
apiKey.setKey("sk-1234567890abcdef");
apiKey.setType(APIKeyType.OPENAI);
apiKey.setKeyName("Main OpenAI Key");

// Access properties
String keyValue = apiKey.getKey();
APIKeyType keyType = apiKey.getType();
String displayName = apiKey.getKeyName();

// Or use constructor with Lombok's @Data
ToolApiKey anotherKey = new ToolApiKey();
anotherKey.setKey("api_key_12345");
anotherKey.setType(APIKeyType.ANTHROPIC);
anotherKey.setKeyName("Claude API Key");
```

This class uses Lombok's `@Data` annotation to automatically generate getters, setters, `toString()`, `equals()`, and `hashCode()` methods, making it a clean data transfer object for API key management in tool configurations.
