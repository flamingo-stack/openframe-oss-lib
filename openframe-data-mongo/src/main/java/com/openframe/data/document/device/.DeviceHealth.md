A data model representing device health metrics and status information, including CPU, memory, disk usage, and system alerts.

## Key Components

- **cpuUsage** - Current CPU utilization percentage
- **memoryUsage** - Current memory utilization percentage  
- **diskUsage** - Current disk space utilization percentage
- **activeProcesses** - List of currently running processes
- **alerts** - Collection of system alerts or warnings

## Usage Example

```java
import com.openframe.data.document.device.DeviceHealth;
import java.util.Arrays;
import java.util.ArrayList;

// Create device health instance
DeviceHealth deviceHealth = new DeviceHealth();
deviceHealth.setCpuUsage(75.5);
deviceHealth.setMemoryUsage(82.3);
deviceHealth.setDiskUsage(45.7);
deviceHealth.setActiveProcesses(Arrays.asList("java", "chrome", "system"));

// Access health metrics
if (deviceHealth.getCpuUsage() > 80.0) {
    System.out.println("High CPU usage detected: " + deviceHealth.getCpuUsage() + "%");
}

// Monitor system processes
List<String> processes = deviceHealth.getActiveProcesses();
System.out.println("Running processes: " + processes.size());
```

The class uses Lombok's `@Data` annotation to automatically generate getters, setters, and other common methods.
