A data model class representing the health status and performance metrics of a device in the OpenFrame system.

## Key Components

- **cpuUsage** - Current CPU utilization percentage (0-100)
- **memoryUsage** - Current memory utilization percentage (0-100)
- **diskUsage** - Current disk utilization percentage (0-100)
- **activeProcesses** - List of currently running process names
- **alerts** - List of Alert objects for system warnings or errors

## Usage Example

```java
DeviceHealth deviceHealth = new DeviceHealth();
deviceHealth.setCpuUsage(45.2);
deviceHealth.setMemoryUsage(67.8);
deviceHealth.setDiskUsage(23.1);
deviceHealth.setActiveProcesses(List.of("chrome", "java", "nodejs"));
deviceHealth.setAlerts(List.of(new Alert("High memory usage detected")));

// Access health metrics
if (deviceHealth.getCpuUsage() > 80) {
    System.out.println("CPU usage critical: " + deviceHealth.getCpuUsage() + "%");
}

// Check for alerts
if (!deviceHealth.getAlerts().isEmpty()) {
    deviceHealth.getAlerts().forEach(alert -> 
        System.out.println("Alert: " + alert.getMessage()));
}
```

This class uses Lombok's `@Data` annotation to automatically generate getters, setters, toString, equals, and hashCode methods, making it ideal for data transfer and monitoring applications.
