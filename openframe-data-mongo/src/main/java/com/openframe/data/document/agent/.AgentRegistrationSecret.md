A MongoDB document entity that represents secret keys used for agent registration authentication in the OpenFrame system.

## Key Components

- **@Document** - Maps to the `agent_registration_secrets` MongoDB collection
- **secretKey** - Unique indexed field containing the authentication secret
- **createdAt** - Timestamp tracking when the secret was generated
- **active** - Boolean flag to enable/disable the secret without deletion
- **@Data** - Lombok annotation providing getters, setters, equals, hashCode, and toString

## Usage Example

```java
// Create a new registration secret
AgentRegistrationSecret secret = new AgentRegistrationSecret();
secret.setSecretKey("ars_" + UUID.randomUUID().toString());
secret.setCreatedAt(Instant.now());
secret.setActive(true);

// Save to MongoDB
agentRegistrationSecretRepository.save(secret);

// Find active secrets for validation
List<AgentRegistrationSecret> activeSecrets = 
    agentRegistrationSecretRepository.findByActiveTrue();

// Validate an incoming secret
Optional<AgentRegistrationSecret> validSecret = 
    agentRegistrationSecretRepository.findBySecretKeyAndActiveTrue(incomingKey);
```

The unique index on `secretKey` ensures no duplicate secrets exist, while the `active` flag allows for secret rotation without data loss.
