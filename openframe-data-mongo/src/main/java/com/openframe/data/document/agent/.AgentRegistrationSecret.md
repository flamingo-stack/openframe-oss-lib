This MongoDB document class represents secure registration tokens used for agent authentication and onboarding processes.

## Key Components

- **AgentRegistrationSecret**: Main document entity with Lombok `@Data` annotation for automatic getters/setters
- **secretKey**: Unique indexed field containing the registration token
- **createdAt**: Timestamp tracking when the secret was generated
- **active**: Boolean flag to enable/disable the registration secret

## Usage Example

```java
// Create a new registration secret
AgentRegistrationSecret secret = new AgentRegistrationSecret();
secret.setSecretKey("abc123-def456-ghi789");
secret.setCreatedAt(Instant.now());
secret.setActive(true);

// Save to MongoDB
mongoTemplate.save(secret);

// Query by secret key
AgentRegistrationSecret found = mongoTemplate.findOne(
    Query.query(Criteria.where("secretKey").is("abc123-def456-ghi789")),
    AgentRegistrationSecret.class
);

// Deactivate expired secrets
found.setActive(false);
mongoTemplate.save(found);
```

The unique index on `secretKey` ensures no duplicate registration tokens exist in the system, while the `active` flag allows for secret lifecycle management without deletion.
