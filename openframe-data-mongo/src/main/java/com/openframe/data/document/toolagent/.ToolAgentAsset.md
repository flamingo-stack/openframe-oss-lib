A data model class representing an asset or file resource used by tool agents in the OpenFrame system. This class encapsulates metadata about files that tool agents can access and manipulate.

## Key Components

- **id** - Unique identifier for the asset
- **localFilename** - The filename as stored locally in the system
- **source** - Enum/object indicating where the asset originated from
- **path** - File system path to the asset location
- **executable** - Boolean flag indicating if the asset can be executed

## Usage Example

```java
// Create a new tool agent asset
ToolAgentAsset asset = new ToolAgentAsset();
asset.setId("script-001");
asset.setLocalFilename("data-processor.py");
asset.setPath("/tools/scripts/data-processor.py");
asset.setExecutable(true);
asset.setSource(ToolAgentAssetSource.USER_UPLOAD);

// Check if asset can be executed
if (asset.isExecutable()) {
    // Process executable asset
    System.out.println("Executing: " + asset.getLocalFilename());
}

// Get asset information
String assetPath = asset.getPath();
String filename = asset.getLocalFilename();
```

This model serves as a container for file metadata that tool agents need to manage and execute various resources within the OpenFrame platform.
