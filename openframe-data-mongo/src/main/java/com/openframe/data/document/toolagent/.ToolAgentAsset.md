A data model class representing an asset or file resource associated with a tool agent, containing metadata about the asset's location, source, and properties.

## Key Components

- **id**: Unique identifier for the asset
- **localFilename**: The filename when stored locally
- **source**: Source information wrapped in `ToolAgentAssetSource` object
- **path**: File system path to the asset
- **executable**: Boolean flag indicating if the asset is executable

## Usage Example

```java
// Create a new tool agent asset
ToolAgentAsset asset = new ToolAgentAsset();
asset.setId("asset-123");
asset.setLocalFilename("my-script.sh");
asset.setPath("/opt/tools/scripts/my-script.sh");
asset.setExecutable(true);

// Set source information
ToolAgentAssetSource source = new ToolAgentAssetSource();
asset.setSource(source);

// Access asset properties
if (asset.isExecutable()) {
    System.out.println("Asset " + asset.getLocalFilename() + 
                      " can be executed from " + asset.getPath());
}
```

This class uses Lombok annotations to automatically generate getters and setters for all fields, making it a clean data transfer object for tool agent asset management.
