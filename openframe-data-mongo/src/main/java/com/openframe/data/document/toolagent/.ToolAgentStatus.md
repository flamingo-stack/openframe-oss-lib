Simple enumeration that defines the operational status states for tool agents within the OpenFrame system.

## Key Components

- **ENABLED** - Indicates a tool agent is active and operational
- **DISABLED** - Indicates a tool agent is inactive or suspended

## Usage Example

```java
// Setting tool agent status
ToolAgentStatus status = ToolAgentStatus.ENABLED;

// Checking status in conditional logic
if (toolAgent.getStatus() == ToolAgentStatus.ENABLED) {
    toolAgent.executeTask();
} else {
    logger.warn("Tool agent is disabled");
}

// Using in switch statements
switch (toolAgent.getStatus()) {
    case ENABLED:
        processToolAgent(toolAgent);
        break;
    case DISABLED:
        skipToolAgent(toolAgent);
        break;
}
```

This enum provides type-safe status management for tool agents, ensuring only valid states can be assigned and preventing runtime errors from invalid string comparisons.
