An enumeration that defines the operational status states for tool agents within the OpenFrame system.

## Key Components

- **ENABLED** - Indicates the tool agent is active and operational
- **DISABLED** - Indicates the tool agent is inactive and not processing requests

## Usage Example

```java
// Set a tool agent to enabled status
ToolAgentStatus status = ToolAgentStatus.ENABLED;

// Check if a tool agent is operational
if (status == ToolAgentStatus.ENABLED) {
    // Process tool agent requests
    processToolAgentRequest();
}

// Toggle between states
ToolAgentStatus newStatus = (currentStatus == ToolAgentStatus.ENABLED) 
    ? ToolAgentStatus.DISABLED 
    : ToolAgentStatus.ENABLED;

// Use in conditional logic
switch (toolAgent.getStatus()) {
    case ENABLED:
        executeToolOperation();
        break;
    case DISABLED:
        logDisabledAttempt();
        break;
}
```

This enum provides a simple boolean-like state management system for controlling tool agent availability and processing within the OpenFrame document management system.
