A MongoDB document entity representing cryptographic keys associated with specific tenants, used for secure operations within the multi-tenant architecture.

## Key Components

- **@Document(collection = "tenant_keys")** - MongoDB collection mapping for tenant key storage
- **@Data** - Lombok annotation providing getters, setters, toString, equals, and hashCode methods
- **@Id** - Primary key identifier for MongoDB document
- **tenantId** - Reference to the owning tenant
- **keyId** - Unique identifier for the key pair
- **publicPem** - Public key in PEM format for encryption/verification
- **privateEncrypted** - Encrypted private key for secure storage
- **active** - Flag indicating if the key is currently active
- **createdAt/rotatedAt** - Timestamps for key lifecycle management

## Usage Example

```java
// Create a new tenant key
TenantKey tenantKey = new TenantKey();
tenantKey.setTenantId("tenant-123");
tenantKey.setKeyId("key-456");
tenantKey.setPublicPem("-----BEGIN PUBLIC KEY-----\n...");
tenantKey.setPrivateEncrypted("encrypted-private-key-data");
tenantKey.setActive(true);
tenantKey.setCreatedAt(Instant.now());

// Save to MongoDB
mongoTemplate.save(tenantKey);

// Query active keys for a tenant
Query query = new Query(Criteria.where("tenantId").is("tenant-123")
    .and("active").is(true));
List<TenantKey> activeKeys = mongoTemplate.find(query, TenantKey.class);
```
