A MongoDB document entity that represents cryptographic key pairs associated with specific tenants, supporting key rotation and encryption functionality.

## Key Components

- **TenantKey** - Main entity class with Lombok `@Data` annotation for automatic getters/setters
- **MongoDB Mapping** - Stored in "tenant_keys" collection with Spring Data MongoDB annotations
- **Key Fields**:
  - `id` - MongoDB document identifier
  - `tenantId` - Associates key with specific tenant
  - `keyId` - Unique identifier for the key pair
  - `publicPem` - Public key in PEM format
  - `privateEncrypted` - Encrypted private key
  - `active` - Key status flag
  - `createdAt`/`rotatedAt` - Timestamp tracking

## Usage Example

```java
// Create a new tenant key
TenantKey tenantKey = new TenantKey();
tenantKey.setTenantId("tenant-123");
tenantKey.setKeyId("key-456");
tenantKey.setPublicPem("-----BEGIN PUBLIC KEY-----...");
tenantKey.setPrivateEncrypted("encrypted_private_key_data");
tenantKey.setActive(true);
tenantKey.setCreatedAt(Instant.now());

// Repository usage (typical Spring Data MongoDB)
tenantKeyRepository.save(tenantKey);

// Find active keys for a tenant
List<TenantKey> activeKeys = tenantKeyRepository
    .findByTenantIdAndActiveTrue("tenant-123");
```
