A Spring Data MongoDB document class for storing OAuth 2.0 authentication tokens with expiration tracking and user association.

## Key Components

- **@Document**: Maps to MongoDB `oauth_tokens` collection
- **@Data**: Lombok annotation providing getters, setters, equals, hashCode, and toString
- **Token Fields**: Stores access and refresh tokens with their expiration timestamps
- **User Association**: Links tokens to specific users via `userId`
- **Client Information**: Tracks OAuth client and granted scopes

## Usage Example

```java
// Create a new OAuth token record
OAuthToken token = new OAuthToken();
token.setUserId("user123");
token.setAccessToken("eyJhbGciOiJIUzI1NiIs...");
token.setRefreshToken("refresh_token_value");
token.setAccessTokenExpiry(Instant.now().plusSeconds(3600)); // 1 hour
token.setRefreshTokenExpiry(Instant.now().plusDays(30)); // 30 days
token.setClientId("mobile_app");
token.setScopes(new String[]{"read", "write"});

// Save using Spring Data MongoDB repository
oauthTokenRepository.save(token);

// Check if access token is expired
if (token.getAccessTokenExpiry().isBefore(Instant.now())) {
    // Token needs refresh
}
```

This entity enables secure OAuth token management with automatic MongoDB persistence and proper expiration handling for authentication workflows.
