A MongoDB document model representing OAuth 2.0 tokens with their associated metadata and expiration information.

## Key Components

- **@Document annotation**: Maps to the "oauth_tokens" MongoDB collection
- **@Data annotation**: Lombok-generated getters, setters, toString, equals, and hashCode methods
- **Core fields**: 
  - `id` - Primary key identifier
  - `userId` - Associated user identifier
  - `accessToken` - OAuth access token string
  - `refreshToken` - OAuth refresh token string
  - `accessTokenExpiry` - Access token expiration timestamp
  - `refreshTokenExpiry` - Refresh token expiration timestamp
  - `clientId` - OAuth client identifier
  - `scopes` - Array of authorized permission scopes

## Usage Example

```java
// Creating a new OAuth token document
OAuthToken token = new OAuthToken();
token.setUserId("user123");
token.setAccessToken("abc123token");
token.setRefreshToken("refresh456token");
token.setAccessTokenExpiry(Instant.now().plusSeconds(3600));
token.setRefreshTokenExpiry(Instant.now().plusDays(30));
token.setClientId("myapp");
token.setScopes(new String[]{"read", "write"});

// Save to MongoDB repository
oauthTokenRepository.save(token);

// Check if access token is expired
if (token.getAccessTokenExpiry().isBefore(Instant.now())) {
    // Token needs refresh
}
```
