A MongoDB document entity representing integrated third-party tools and services within the OpenFrame platform. This class encapsulates tool metadata, configuration settings, and monitoring parameters for external system integrations.

## Key Components

- **Core Properties**: `id`, `name`, `description`, `icon` - Basic tool identification
- **URL Configuration**: `toolUrls` - List of endpoints for the integrated tool
- **Categorization**: `type`, `toolType`, `category`, `platformCategory` - Tool classification fields
- **Layer Management**: `layer`, `layerOrder`, `layerColor` - UI positioning and appearance
- **Monitoring Settings**: `metricsPath`, `healthCheckEndpoint`, timeout configurations
- **Security**: `credentials` - Authentication information for tool access
- **Data Integration**: `debeziumConnectors` - CDC connector configurations

## Usage Example

```java
IntegratedTool tool = IntegratedTool.builder()
    .name("Jenkins CI/CD")
    .description("Continuous integration and deployment platform")
    .type("CI_CD")
    .category("DevOps")
    .enabled(true)
    .layer("BUILD")
    .layerOrder(1)
    .healthCheckEndpoint("/api/health")
    .healthCheckInterval(300)
    .connectionTimeout(5000)
    .build();

// Save to MongoDB
mongoTemplate.save(tool);

// Query enabled tools by category
List<IntegratedTool> devOpsTools = mongoTemplate.find(
    Query.query(Criteria.where("category").is("DevOps")
        .and("enabled").is(true)), 
    IntegratedTool.class
);
```
