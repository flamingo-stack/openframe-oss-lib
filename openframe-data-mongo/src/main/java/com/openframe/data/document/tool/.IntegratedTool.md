A MongoDB document entity representing an integrated tool within the OpenFrame platform, containing comprehensive configuration including URLs, credentials, monitoring settings, and layer organization.

## Key Components

- **@Document(collection = "integrated_tools")** - MongoDB collection mapping for tool persistence
- **Basic Properties** - Core tool identification (`name`, `description`, `icon`, `type`, `category`)
- **URL Management** - `toolUrls` list for multiple endpoint configurations
- **Authentication** - `ToolCredentials` object for secure access management
- **Layer Organization** - `layer`, `layerOrder`, and `layerColor` for UI positioning and visual hierarchy
- **Health Monitoring** - Complete health check configuration with endpoints, intervals, and timeouts
- **Data Integration** - `debeziumConnectors` array for change data capture configuration

## Usage Example

```java
IntegratedTool tool = IntegratedTool.builder()
    .name("Analytics Dashboard")
    .description("Real-time analytics and reporting tool")
    .type("dashboard")
    .category("analytics")
    .enabled(true)
    .layer("monitoring")
    .layerOrder(1)
    .healthCheckEndpoint("/health")
    .healthCheckInterval(30)
    .connectionTimeout(5000)
    .allowedEndpoints(new String[]{"/api/metrics", "/api/reports"})
    .build();

// Save to MongoDB
repository.save(tool);
```
