This data class represents a filter object for querying machine/device records with multiple filter criteria.

## Key Components

- **statuses** - List of machine status values to filter by
- **deviceTypes** - List of device type identifiers for filtering  
- **osTypes** - List of operating system types to include
- **organizationIds** - List of organization identifiers for scoping results
- **tagNames** - List of tag names associated with machines

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and utility methods.

## Usage Example

```java
// Create filter for active Windows devices in specific organizations
MachineQueryFilter filter = new MachineQueryFilter();
filter.setStatuses(List.of("ACTIVE", "ONLINE"));
filter.setDeviceTypes(List.of("LAPTOP", "DESKTOP"));
filter.setOsTypes(List.of("WINDOWS_10", "WINDOWS_11"));
filter.setOrganizationIds(List.of("org-123", "org-456"));
filter.setTagNames(List.of("production", "critical"));

// Or use the all-args constructor
MachineQueryFilter filter2 = new MachineQueryFilter(
    List.of("ACTIVE"),
    List.of("SERVER"),
    List.of("LINUX"),
    List.of("org-789"),
    List.of("database")
);
```

This filter would typically be passed to a service or repository method to build dynamic queries for machine data retrieval.
