A data filter class for querying machine/device records with multiple criteria including status, device type, operating system, organization, and key-value tag filtering.

## Key Components

- **statuses** - List of machine status values to filter by
- **deviceTypes** - List of device type identifiers for filtering
- **osTypes** - List of operating system types to match
- **organizationIds** - List of organization identifiers for scoping queries
- **tagKeys** - List of tag keys for tag-based filtering
- **tagValues** - List of tag values for filtering by specific tag values

## Usage Example

```java
// Create filter for active Windows devices with specific tags
MachineQueryFilter filter = new MachineQueryFilter();
filter.setStatuses(List.of("ACTIVE", "ONLINE"));
filter.setDeviceTypes(List.of("DESKTOP", "LAPTOP"));
filter.setOsTypes(List.of("WINDOWS_10", "WINDOWS_11"));
filter.setOrganizationIds(List.of("org-123", "org-456"));
filter.setTagKeys(List.of("site", "environment"));
filter.setTagValues(List.of("production"));
```

This filter class uses Lombok annotations to automatically generate getters, setters, constructors, and common methods, making it ideal for use in repository query methods or search APIs.
