Represents an installed agent entity stored in MongoDB, tracking agent installations across different machines with version and timestamp information.

## Key Components

- **InstalledAgent** - MongoDB document entity with Lombok data annotations
- **Fields:**
  - `id` - Primary key identifier
  - `machineId` - Identifier for the host machine
  - `agentType` - Type/category of the installed agent
  - `version` - Agent version string
  - `createdAt/updatedAt` - Timestamp fields for audit tracking

## Usage Example

```java
// Create a new installed agent record
InstalledAgent agent = new InstalledAgent();
agent.setMachineId("server-001");
agent.setAgentType("monitoring");
agent.setVersion("2.1.0");
agent.setCreatedAt("2024-01-15T10:30:00Z");

// Access agent information
String machineId = agent.getMachineId();
String version = agent.getVersion();

// Update version
agent.setVersion("2.1.1");
agent.setUpdatedAt("2024-01-20T14:45:00Z");
```

This entity is typically used with Spring Data MongoDB repositories to perform CRUD operations on the `installed_agents` collection, enabling tracking and management of software agents deployed across an infrastructure.
