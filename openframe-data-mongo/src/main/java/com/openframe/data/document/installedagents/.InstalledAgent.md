A MongoDB document entity that represents installed software agents on machines, tracking agent metadata and versioning information.

## Key Components

- **@Document**: Maps to the "installed_agents" MongoDB collection
- **@Data**: Lombok annotation providing getters, setters, toString, equals, and hashCode
- **@Id**: MongoDB document identifier
- **machineId**: Reference to the machine where the agent is installed
- **agentType**: Classification of the agent software
- **version**: Current version of the installed agent
- **createdAt/updatedAt**: Timestamp fields for audit tracking

## Usage Example

```java
// Create a new installed agent record
InstalledAgent agent = new InstalledAgent();
agent.setMachineId("machine-123");
agent.setAgentType("monitoring");
agent.setVersion("1.2.3");
agent.setCreatedAt("2024-01-15T10:30:00Z");
agent.setUpdatedAt("2024-01-15T10:30:00Z");

// Access agent information
String machineId = agent.getMachineId();
String currentVersion = agent.getVersion();

// Update agent version
agent.setVersion("1.2.4");
agent.setUpdatedAt("2024-01-16T09:15:00Z");
```

This entity is typically used in conjunction with Spring Data MongoDB repositories to persist and query agent installation data across machine infrastructure.
