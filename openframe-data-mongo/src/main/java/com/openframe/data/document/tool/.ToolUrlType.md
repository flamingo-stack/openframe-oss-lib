An enumeration that defines standardized URL types for different tool and service categories within the OpenFrame document system.

## Key Components

- **DASHBOARD** - Web interface URLs for monitoring and management
- **API** - REST API endpoint URLs
- **WS** - WebSocket connection URLs
- **NATS** - NATS messaging system URLs
- **DATABASE** - Database connection URLs
- **CACHE** - Cache service URLs (Redis, etc.)
- **BROKER** - Message broker URLs
- **COORDINATOR** - Service coordination URLs
- **CONTROLLER** - Controller service URLs
- **SERVER** - General server URLs
- **METRICS** - Monitoring and metrics URLs
- **LOGS** - Logging service URLs
- **INTERNAL** - Internal service communication URLs
- **MINION** - Worker/minion service URLs

## Usage Example

```java
import com.openframe.data.document.tool.ToolUrlType;

// Categorize different service URLs
public class ServiceRegistry {
    public void registerService(String url, ToolUrlType type) {
        switch (type) {
            case API:
                configureApiEndpoint(url);
                break;
            case DATABASE:
                setupDatabaseConnection(url);
                break;
            case METRICS:
                initializeMetricsCollection(url);
                break;
        }
    }
}

// Usage in service configuration
registerService("http://api.example.com", ToolUrlType.API);
registerService("jdbc:postgresql://db:5432/app", ToolUrlType.DATABASE);
```
