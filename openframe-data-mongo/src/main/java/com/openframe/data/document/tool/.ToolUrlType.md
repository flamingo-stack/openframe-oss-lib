An enumeration that defines the different types of URLs that tools within the OpenFrame system can expose, categorizing endpoints by their functional purpose.

## Key Components

- **DASHBOARD** - Web interface endpoints for tool management and monitoring
- **API** - RESTful API endpoints for programmatic access
- **WS** - WebSocket endpoints for real-time communication
- **NATS** - NATS messaging system endpoints
- **DATABASE** - Database connection endpoints
- **CACHE** - Caching service endpoints (Redis, etc.)
- **BROKER** - Message broker endpoints
- **COORDINATOR** - Service coordination endpoints
- **CONTROLLER** - Control plane endpoints
- **SERVER** - General server endpoints
- **METRICS** - Monitoring and metrics collection endpoints
- **LOGS** - Log aggregation and access endpoints
- **INTERNAL** - Internal service communication endpoints
- **MINION** - Worker/minion service endpoints

## Usage Example

```java
// Define tool URLs by type
Map<ToolUrlType, String> toolUrls = new HashMap<>();
toolUrls.put(ToolUrlType.DASHBOARD, "http://localhost:8080/dashboard");
toolUrls.put(ToolUrlType.API, "http://localhost:8080/api/v1");
toolUrls.put(ToolUrlType.METRICS, "http://localhost:9090/metrics");

// Check URL type in service configuration
if (urlType == ToolUrlType.DATABASE) {
    configureDataSource(url);
} else if (urlType == ToolUrlType.WS) {
    establishWebSocketConnection(url);
}
```
