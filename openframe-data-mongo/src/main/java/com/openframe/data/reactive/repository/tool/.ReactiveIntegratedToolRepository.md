A reactive MongoDB repository interface for managing IntegratedTool documents in reactive web applications. This repository extends Spring's ReactiveMongoRepository and provides non-blocking database operations using Project Reactor types.

## Key Components

- **ReactiveIntegratedToolRepository**: Main repository interface extending `ReactiveMongoRepository<IntegratedTool, String>`
- **BaseIntegratedToolRepository**: Parent interface providing common tool repository operations
- **findByType()**: Custom query method to find tools by their type field
- **@ConditionalOnWebApplication**: Auto-configuration annotation that only activates in reactive web contexts

## Usage Example

```java
@Service
public class ToolService {
    
    @Autowired
    private ReactiveIntegratedToolRepository toolRepository;
    
    public Mono<IntegratedTool> getToolByType(String toolType) {
        return toolRepository.findByType(toolType);
    }
    
    public Mono<IntegratedTool> saveTool(IntegratedTool tool) {
        return toolRepository.save(tool);
    }
    
    public Flux<IntegratedTool> getAllTools() {
        return toolRepository.findAll();
    }
    
    public Mono<Void> deleteTool(String id) {
        return toolRepository.deleteById(id);
    }
}
```

The repository automatically provides reactive CRUD operations and integrates seamlessly with Spring WebFlux applications for non-blocking data access patterns.
