A custom runtime exception that indicates when an API key lookup operation fails to locate the specified key.

## Key Components

- **ApiKeyNotFoundException** - Runtime exception class for API key lookup failures
- **Constructor overloads:**
  - `ApiKeyNotFoundException(String keyId)` - Basic constructor with key ID
  - `ApiKeyNotFoundException(String keyId, String userId)` - Constructor with key ID and user context
  - `ApiKeyNotFoundException(String message, Throwable cause)` - Generic constructor with custom message and cause

## Usage Example

```java
// Throwing when API key not found
public ApiKey findApiKey(String keyId) {
    ApiKey key = repository.findByKeyId(keyId);
    if (key == null) {
        throw new ApiKeyNotFoundException(keyId);
    }
    return key;
}

// Throwing with user context
public ApiKey findUserApiKey(String keyId, String userId) {
    ApiKey key = repository.findByKeyIdAndUserId(keyId, userId);
    if (key == null) {
        throw new ApiKeyNotFoundException(keyId, userId);
    }
    return key;
}

// Handling the exception
try {
    ApiKey key = service.findApiKey("invalid-key-123");
} catch (ApiKeyNotFoundException e) {
    logger.error("API key lookup failed: " + e.getMessage());
    return ResponseEntity.notFound().build();
}
```
