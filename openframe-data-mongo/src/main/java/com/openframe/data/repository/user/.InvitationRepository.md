A Spring Data MongoDB repository interface for managing user invitation documents, providing database operations for the invitation system.

## Key Components

- **InvitationRepository** - MongoDB repository interface extending `MongoRepository<Invitation, String>`
- **findByEmailAndStatus()** - Custom query method to find invitations by email address and status
- **Standard CRUD operations** - Inherited from MongoRepository (save, findById, findAll, delete, etc.)

## Usage Example

```java
@Service
public class InvitationService {
    
    @Autowired
    private InvitationRepository invitationRepository;
    
    public Optional<Invitation> findPendingInvitation(String email) {
        return invitationRepository.findByEmailAndStatus(email, "PENDING");
    }
    
    public Invitation createInvitation(String email) {
        Invitation invitation = new Invitation();
        invitation.setEmail(email);
        invitation.setStatus("PENDING");
        return invitationRepository.save(invitation);
    }
    
    public List<Invitation> getAllInvitations() {
        return invitationRepository.findAll();
    }
}
```

The repository uses Spring Data MongoDB's automatic query derivation to generate the implementation for `findByEmailAndStatus()` based on the method name, eliminating the need for manual query writing.
