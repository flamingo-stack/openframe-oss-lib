A Spring Data MongoDB repository interface for managing OAuth token persistence and retrieval operations in the OpenFrame authentication system.

## Key Components

- **OAuthTokenRepository** - MongoDB repository interface extending `MongoRepository<OAuthToken, String>`
- **findByAccessToken()** - Retrieves OAuth token by access token value
- **findByRefreshToken()** - Retrieves OAuth token by refresh token value
- Inherits standard CRUD operations from `MongoRepository`

## Usage Example

```java
@Service
public class TokenService {
    
    @Autowired
    private OAuthTokenRepository tokenRepository;
    
    public boolean validateAccessToken(String accessToken) {
        Optional<OAuthToken> token = tokenRepository.findByAccessToken(accessToken);
        return token.isPresent() && !token.get().isExpired();
    }
    
    public OAuthToken refreshToken(String refreshToken) {
        Optional<OAuthToken> existingToken = tokenRepository.findByRefreshToken(refreshToken);
        if (existingToken.isPresent()) {
            OAuthToken newToken = generateNewToken();
            tokenRepository.delete(existingToken.get());
            return tokenRepository.save(newToken);
        }
        throw new InvalidTokenException("Refresh token not found");
    }
    
    public void revokeToken(String accessToken) {
        tokenRepository.findByAccessToken(accessToken)
            .ifPresent(tokenRepository::delete);
    }
}
```
