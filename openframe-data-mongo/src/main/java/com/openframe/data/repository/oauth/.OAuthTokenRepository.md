This repository interface provides MongoDB data access operations for OAuth token management, extending Spring Data MongoDB's base repository functionality.

## Key Components

- **OAuthTokenRepository** - MongoDB repository interface for OAuthToken documents
- **findByAccessToken()** - Retrieves token by access token string
- **findByRefreshToken()** - Retrieves token by refresh token string
- Inherits standard CRUD operations from `MongoRepository<OAuthToken, String>`

## Usage Example

```java
@Service
public class TokenService {
    
    @Autowired
    private OAuthTokenRepository tokenRepository;
    
    public boolean validateAccessToken(String accessToken) {
        Optional<OAuthToken> token = tokenRepository.findByAccessToken(accessToken);
        return token.isPresent() && !token.get().isExpired();
    }
    
    public OAuthToken refreshToken(String refreshToken) {
        Optional<OAuthToken> existingToken = tokenRepository.findByRefreshToken(refreshToken);
        if (existingToken.isPresent()) {
            OAuthToken newToken = generateNewToken();
            return tokenRepository.save(newToken);
        }
        throw new TokenNotFoundException("Invalid refresh token");
    }
    
    public void revokeToken(String tokenId) {
        tokenRepository.deleteById(tokenId);
    }
}
```
