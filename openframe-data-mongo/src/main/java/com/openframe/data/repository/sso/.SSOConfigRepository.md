A Spring Data MongoDB repository interface for managing Single Sign-On (SSO) configuration documents in the database. This repository provides data access operations for SSO provider settings and configurations.

## Key Components

- **SSOConfigRepository**: Main repository interface extending `MongoRepository<SSOConfig, String>`
- **findByProvider()**: Retrieves SSO configuration by provider name
- **findByEnabledTrue()**: Returns all active/enabled SSO configurations

## Usage Example

```java
@Service
public class SSOService {
    
    @Autowired
    private SSOConfigRepository ssoConfigRepository;
    
    public Optional<SSOConfig> getProviderConfig(String providerName) {
        return ssoConfigRepository.findByProvider(providerName);
    }
    
    public List<SSOConfig> getActiveProviders() {
        return ssoConfigRepository.findByEnabledTrue();
    }
    
    public SSOConfig saveConfig(SSOConfig config) {
        return ssoConfigRepository.save(config);
    }
    
    public void deleteConfig(String configId) {
        ssoConfigRepository.deleteById(configId);
    }
}
```

The repository automatically inherits standard CRUD operations from `MongoRepository` while providing custom query methods for SSO-specific use cases like finding configurations by provider or retrieving only enabled configurations.
