A Spring Data MongoDB repository interface for managing tenant encryption keys, providing methods to find and count active tenant keys.

## Key Components

- **TenantKeyRepository** - MongoDB repository interface extending `MongoRepository<TenantKey, String>`
- **findFirstByTenantIdAndActiveTrue()** - Retrieves the first active key for a specific tenant
- **countByTenantIdAndActiveTrue()** - Counts active keys for a tenant

## Usage Example

```java
@Service
public class TenantKeyService {
    
    @Autowired
    private TenantKeyRepository tenantKeyRepository;
    
    public Optional<TenantKey> getActiveTenantKey(String tenantId) {
        return tenantKeyRepository.findFirstByTenantIdAndActiveTrue(tenantId);
    }
    
    public boolean hasActiveKeys(String tenantId) {
        return tenantKeyRepository.countByTenantIdAndActiveTrue(tenantId) > 0;
    }
    
    public void saveNewKey(TenantKey key) {
        tenantKeyRepository.save(key);
    }
}
```

The repository follows Spring Data naming conventions to automatically generate query implementations based on method names, filtering by both `tenantId` and `active=true` status.
