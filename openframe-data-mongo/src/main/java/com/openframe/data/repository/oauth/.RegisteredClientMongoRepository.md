A Spring Data MongoDB repository interface for managing OAuth2 registered client entities in a MongoDB database.

## Key Components

- **RegisteredClientMongoRepository** - Repository interface extending `MongoRepository`
- **findByClientId()** - Custom query method to find clients by their OAuth2 client ID
- **MongoRegisteredClient** - The document entity being managed
- **@Repository** - Spring stereotype annotation for repository beans

## Usage Example

```java
@Service
public class OAuth2ClientService {
    
    @Autowired
    private RegisteredClientMongoRepository clientRepository;
    
    public Optional<MongoRegisteredClient> getClientByClientId(String clientId) {
        return clientRepository.findByClientId(clientId);
    }
    
    public MongoRegisteredClient saveClient(MongoRegisteredClient client) {
        return clientRepository.save(client);
    }
    
    public void deleteClient(String id) {
        clientRepository.deleteById(id);
    }
    
    public List<MongoRegisteredClient> getAllClients() {
        return clientRepository.findAll();
    }
}
```

This repository provides standard CRUD operations through `MongoRepository` inheritance, plus a custom finder method for OAuth2 client lookup by client ID, which is essential for OAuth2 authorization flows.
