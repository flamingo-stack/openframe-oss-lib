A MongoDB repository interface for managing OAuth2 registered client data, extending Spring Data MongoDB's repository functionality with custom query methods.

## Key Components

- **RegisteredClientMongoRepository** - Main repository interface extending `MongoRepository<MongoRegisteredClient, String>`
- **findByClientId()** - Custom query method to find registered clients by their client ID
- Uses `MongoRegisteredClient` as the document entity type
- Annotated with `@Repository` for Spring component scanning

## Usage Example

```java
@Service
public class OAuth2ClientService {
    
    @Autowired
    private RegisteredClientMongoRepository clientRepository;
    
    public Optional<MongoRegisteredClient> getClientByClientId(String clientId) {
        return clientRepository.findByClientId(clientId);
    }
    
    public MongoRegisteredClient saveClient(MongoRegisteredClient client) {
        return clientRepository.save(client);
    }
    
    public void deleteClient(String id) {
        clientRepository.deleteById(id);
    }
}
```

The repository provides standard CRUD operations through `MongoRepository` plus the custom `findByClientId` method for OAuth2 client lookup scenarios. This is typically used in OAuth2 authorization server implementations where clients need to be validated and retrieved by their unique client identifier.
