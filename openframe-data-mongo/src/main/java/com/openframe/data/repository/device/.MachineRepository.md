A Spring Data MongoDB repository interface that provides data access operations for Machine entities, including custom query methods for filtering by hostname, device type, and organization.

## Key Components

- **MachineRepository Interface**: Extends `MongoRepository<Machine, String>` and `CustomMachineRepository`
- **findByMachineId()**: Retrieves a machine by its unique machine identifier
- **findByHostnameContainingIgnoreCase()**: Searches machines by hostname with case-insensitive partial matching
- **findByTypeAndHostnameContainingIgnoreCase()**: Filters machines by device type and hostname
- **findByType()**: Retrieves all machines of a specific device type
- **existsByOrganizationId()**: Checks if any machines exist for a given organization

## Usage Example

```java
@Autowired
private MachineRepository machineRepository;

// Find specific machine
Optional<Machine> machine = machineRepository.findByMachineId("MACHINE_001");

// Search by hostname
List<Machine> servers = machineRepository.findByHostnameContainingIgnoreCase("web-server");

// Filter by type and hostname
List<Machine> webServers = machineRepository.findByTypeAndHostnameContainingIgnoreCase(
    DeviceType.SERVER, "web"
);

// Check organization existence
boolean hasDevices = machineRepository.existsByOrganizationId("org-123");

// Standard CRUD operations inherited from MongoRepository
Machine newMachine = new Machine();
machineRepository.save(newMachine);
```
