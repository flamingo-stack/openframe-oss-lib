MongoDB repository interface for managing application release version information as a single document in the database.

## Key Components

- **ReleaseVersionRepository**: Spring Data MongoDB repository interface extending `MongoRepository<ReleaseVersion, String>`
- **findFirstByOrderByCreatedAtAsc()**: Custom query method to retrieve the single release version document
- **@ConditionalOnWebApplication**: Conditionally loads only in servlet-based web applications
- **@Repository**: Marks as a Spring Data repository component

## Usage Example

```java
@Service
public class VersionService {
    
    @Autowired
    private ReleaseVersionRepository versionRepository;
    
    public Optional<ReleaseVersion> getCurrentVersion() {
        return versionRepository.findFirstByOrderByCreatedAtAsc();
    }
    
    public ReleaseVersion updateVersion(String newVersion) {
        ReleaseVersion version = versionRepository.findFirstByOrderByCreatedAtAsc()
            .orElse(new ReleaseVersion());
        version.setVersion(newVersion);
        return versionRepository.save(version);
    }
    
    public void initializeVersion(String initialVersion) {
        if (versionRepository.count() == 0) {
            ReleaseVersion version = new ReleaseVersion();
            version.setVersion(initialVersion);
            versionRepository.save(version);
        }
    }
}
```
