Spring Data MongoDB repository interface for managing tenant entities with domain-based lookup capabilities. Provides CRUD operations and specialized queries for multi-tenant applications.

## Key Components

- **TenantRepository Interface**: Main repository extending `MongoRepository` and `BaseTenantRepository`
- **Domain Operations**: `findByDomain()`, `existsByDomain()`, `findByDomainIn()`
- **DomainView Interface**: Projection for retrieving only domain fields
- **Conditional Loading**: Only active in servlet-based web applications

## Usage Example

```java
@Service
public class TenantService {
    
    @Autowired
    private TenantRepository tenantRepository;
    
    public Optional<Tenant> getTenantByDomain(String domain) {
        return tenantRepository.findByDomain(domain);
    }
    
    public boolean isDomainTaken(String domain) {
        return tenantRepository.existsByDomain(domain);
    }
    
    public List<TenantRepository.DomainView> getDomainsSubset(List<String> domains) {
        return tenantRepository.findByDomainIn(domains);
    }
    
    public long getTotalTenants() {
        return tenantRepository.count();
    }
}
```

The repository automatically provides standard CRUD operations through `MongoRepository` while adding tenant-specific domain queries for multi-tenant application scenarios.
