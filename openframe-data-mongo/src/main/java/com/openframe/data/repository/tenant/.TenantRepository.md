A MongoDB repository interface for managing tenant documents with domain-based operations. This repository extends Spring Data MongoDB's MongoRepository and implements a base tenant repository interface for multi-tenant application support.

## Key Components

- **TenantRepository** - Main repository interface extending `MongoRepository<Tenant, String>`
- **BaseTenantRepository** - Base interface implementation with generic type parameters for tenant operations
- **DomainView** - Projection interface for retrieving only domain information
- **Domain Operations** - Methods for finding and checking tenant existence by domain
- **Conditional Bean** - Only active in servlet-based web applications via `@ConditionalOnWebApplication`

## Usage Example

```java
@Service
public class TenantService {
    
    @Autowired
    private TenantRepository tenantRepository;
    
    public Optional<Tenant> getTenantByDomain(String domain) {
        return tenantRepository.findByDomain(domain);
    }
    
    public boolean isDomainRegistered(String domain) {
        return tenantRepository.existsByDomain(domain);
    }
    
    public List<String> getExistingDomains(List<String> domains) {
        return tenantRepository.findByDomainIn(domains)
                .stream()
                .map(TenantRepository.DomainView::getDomain)
                .collect(Collectors.toList());
    }
    
    public long getTotalTenantCount() {
        return tenantRepository.count();
    }
}
```

The repository provides efficient domain-based queries and includes a projection interface for optimized domain-only lookups in multi-tenant scenarios.
