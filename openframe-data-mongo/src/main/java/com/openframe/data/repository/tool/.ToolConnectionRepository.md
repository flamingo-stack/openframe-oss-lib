A Spring Data MongoDB repository interface for managing ToolConnection documents, providing methods to query tool connections by machine ID, agent tool ID, and tool type.

## Key Components

- **ToolConnectionRepository** - MongoDB repository interface extending `MongoRepository<ToolConnection, String>`
- **findByMachineId()** - Retrieves all tool connections for a specific machine
- **findByMachineIdIn()** - Batch retrieval for multiple machine IDs
- **findFirstByAgentToolIdOrderByConnectedAtDesc()** - Gets the most recent connection by agent tool ID
- **findByMachineIdAndToolType()** - Finds connection by machine and tool type combination

## Usage Example

```java
@Service
public class ToolConnectionService {
    
    @Autowired
    private ToolConnectionRepository toolConnectionRepository;
    
    public List<ToolConnection> getMachineConnections(String machineId) {
        return toolConnectionRepository.findByMachineId(machineId);
    }
    
    public Optional<ToolConnection> getLatestAgentConnection(String agentToolId) {
        return toolConnectionRepository
            .findFirstByAgentToolIdOrderByConnectedAtDesc(agentToolId);
    }
    
    public Optional<ToolConnection> findSpecificConnection(String machineId, ToolType toolType) {
        return toolConnectionRepository
            .findByMachineIdAndToolType(machineId, toolType);
    }
}
```

The repository leverages Spring Data MongoDB's query method naming conventions to automatically generate query implementations, with the `findFirstBy...OrderBy...Desc` method providing chronological sorting for retrieving the most recent tool connection.
