Repository interface for managing tool connection data in MongoDB, providing methods to query tool connections by machine ID, agent tool ID, and tool type.

## Key Components

- **ToolConnectionRepository**: Main repository interface extending `MongoRepository<ToolConnection, String>`
- **findByMachineId()**: Retrieves all tool connections for a specific machine
- **findByMachineIdIn()**: Batch query for multiple machine IDs
- **findFirstByAgentToolIdOrderByConnectedAtDesc()**: Gets the most recent connection by agent tool ID
- **findByMachineIdAndToolType()**: Finds connection by machine and tool type combination

## Usage Example

```java
@Autowired
private ToolConnectionRepository toolConnectionRepo;

// Get all connections for a machine
List<ToolConnection> machineConnections = toolConnectionRepo.findByMachineId("machine123");

// Get connections for multiple machines
List<String> machineIds = Arrays.asList("machine1", "machine2");
List<ToolConnection> multipleConnections = toolConnectionRepo.findByMachineIdIn(machineIds);

// Get latest connection for an agent tool
Optional<ToolConnection> latestConnection = toolConnectionRepo
    .findFirstByAgentToolIdOrderByConnectedAtDesc("agent456");

// Find specific tool type connection
Optional<ToolConnection> specificTool = toolConnectionRepo
    .findByMachineIdAndToolType("machine123", ToolType.SCANNER);
```
