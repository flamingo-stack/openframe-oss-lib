A Spring Data MongoDB repository interface that provides data access operations for IntegratedToolAgent documents, supporting queries by status and release version filtering.

## Key Components

- **IntegratedToolAgentRepository** - Main repository interface extending MongoRepository
- **findByStatus()** - Query method to find agents by their current status
- **findByReleaseVersionTrue()** - Query method to find agents marked as release versions
- **MongoRepository inheritance** - Provides standard CRUD operations (save, findById, delete, etc.)

## Usage Example

```java
@Service
public class ToolAgentService {
    
    @Autowired
    private IntegratedToolAgentRepository repository;
    
    // Find all active tool agents
    public List<IntegratedToolAgent> getActiveAgents() {
        return repository.findByStatus(ToolAgentStatus.ACTIVE);
    }
    
    // Get all release version agents
    public List<IntegratedToolAgent> getReleaseAgents() {
        return repository.findByReleaseVersionTrue();
    }
    
    // Standard CRUD operations
    public IntegratedToolAgent saveAgent(IntegratedToolAgent agent) {
        return repository.save(agent);
    }
    
    public Optional<IntegratedToolAgent> findById(String id) {
        return repository.findById(id);
    }
}
```

The repository uses Spring Data's query derivation to automatically implement the custom finder methods based on method names and the IntegratedToolAgent document structure.
