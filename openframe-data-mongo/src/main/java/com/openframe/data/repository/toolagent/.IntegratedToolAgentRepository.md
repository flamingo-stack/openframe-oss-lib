A MongoDB repository interface for managing IntegratedToolAgent documents, providing data access methods for tool agent entities with status and release version filtering capabilities.

## Key Components

- **IntegratedToolAgentRepository**: Main repository interface extending MongoRepository
- **findByStatus()**: Query method to retrieve tool agents by their status
- **findByReleaseVersionTrue()**: Query method to find tool agents marked as release versions
- **MongoRepository inheritance**: Provides standard CRUD operations (save, findById, findAll, delete, etc.)

## Usage Example

```java
@Service
public class ToolAgentService {
    
    @Autowired
    private IntegratedToolAgentRepository repository;
    
    public List<IntegratedToolAgent> getActiveAgents() {
        return repository.findByStatus(ToolAgentStatus.ACTIVE);
    }
    
    public List<IntegratedToolAgent> getReleaseVersionAgents() {
        return repository.findByReleaseVersionTrue();
    }
    
    public IntegratedToolAgent saveAgent(IntegratedToolAgent agent) {
        return repository.save(agent);
    }
    
    public Optional<IntegratedToolAgent> findById(String id) {
        return repository.findById(id);
    }
}
```
