This interface defines a MongoDB repository for managing OAuth client data, extending Spring Data MongoDB's base repository functionality with custom query methods.

## Key Components

- **OAuthClientRepository**: MongoDB repository interface for `OAuthClient` entities
- **findByClientId()**: Retrieves an OAuth client by its unique client identifier
- **existsByMachineId()**: Checks if an OAuth client exists for a specific machine ID
- Inherits standard CRUD operations from `MongoRepository<OAuthClient, String>`

## Usage Example

```java
@Service
public class OAuthService {
    
    @Autowired
    private OAuthClientRepository oauthClientRepository;
    
    public Optional<OAuthClient> getClientByClientId(String clientId) {
        return oauthClientRepository.findByClientId(clientId);
    }
    
    public boolean isMachineRegistered(String machineId) {
        return oauthClientRepository.existsByMachineId(machineId);
    }
    
    public OAuthClient saveClient(OAuthClient client) {
        return oauthClientRepository.save(client);
    }
}
```

The repository automatically implements the custom methods using Spring Data's query derivation from method names, eliminating the need for manual query implementation.
