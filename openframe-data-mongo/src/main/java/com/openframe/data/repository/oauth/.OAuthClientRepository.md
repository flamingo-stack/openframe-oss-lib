Spring Data MongoDB repository interface for managing OAuth client configurations in the database.

## Key Components

- **OAuthClientRepository** - MongoDB repository interface extending `MongoRepository<OAuthClient, String>`
- **findByClientId()** - Retrieves an OAuth client by its unique client identifier
- **existsByMachineId()** - Checks if an OAuth client exists for a specific machine ID

## Usage Example

```java
@Service
public class OAuthClientService {
    
    @Autowired
    private OAuthClientRepository oauthClientRepository;
    
    public Optional<OAuthClient> getClientById(String clientId) {
        return oauthClientRepository.findByClientId(clientId);
    }
    
    public boolean isMachineRegistered(String machineId) {
        return oauthClientRepository.existsByMachineId(machineId);
    }
    
    public OAuthClient saveClient(OAuthClient client) {
        return oauthClientRepository.save(client);
    }
}
```

This repository follows Spring Data's naming conventions for automatic query generation, providing CRUD operations for OAuth client management without requiring implementation code.
