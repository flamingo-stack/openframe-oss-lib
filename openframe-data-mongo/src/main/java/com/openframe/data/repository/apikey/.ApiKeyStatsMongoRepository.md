A MongoDB repository interface for managing API key statistics data using Spring Data MongoDB.

## Key Components

- **ApiKeyStatsMongoRepository** - Repository interface extending `MongoRepository<ApiKeyStats, String>`
- **@Repository** - Spring stereotype annotation for repository layer components
- **Generic Types** - Uses `ApiKeyStats` as the document type and `String` as the ID type

## Usage Example

```java
@Service
public class ApiKeyStatsService {
    
    @Autowired
    private ApiKeyStatsMongoRepository repository;
    
    public ApiKeyStats saveStats(ApiKeyStats stats) {
        return repository.save(stats);
    }
    
    public Optional<ApiKeyStats> findStatsById(String id) {
        return repository.findById(id);
    }
    
    public List<ApiKeyStats> findAllStats() {
        return repository.findAll();
    }
    
    public void deleteStats(String id) {
        repository.deleteById(id);
    }
}
```

This repository inherits all standard CRUD operations from `MongoRepository`, including save, find, delete, and count methods. The interface requires no additional method declarations as it uses the default MongoDB repository functionality for basic data operations on `ApiKeyStats` documents.
