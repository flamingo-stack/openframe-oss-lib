A Spring Data MongoDB repository interface for managing API key statistics data persistence operations.

## Key Components

- **ApiKeyStatsMongoRepository**: Repository interface extending `MongoRepository<ApiKeyStats, String>`
- Inherits all standard CRUD operations from MongoRepository
- Uses `String` as the ID type for ApiKeyStats documents
- Annotated with `@Repository` for Spring component scanning

## Usage Example

```java
@Service
public class ApiKeyStatsService {
    
    @Autowired
    private ApiKeyStatsMongoRepository repository;
    
    public ApiKeyStats saveStats(ApiKeyStats stats) {
        return repository.save(stats);
    }
    
    public Optional<ApiKeyStats> findById(String id) {
        return repository.findById(id);
    }
    
    public List<ApiKeyStats> findAllStats() {
        return repository.findAll();
    }
    
    public void deleteStats(String id) {
        repository.deleteById(id);
    }
}
```

The repository automatically provides standard database operations through Spring Data MongoDB's built-in methods, eliminating the need for custom implementation code.
