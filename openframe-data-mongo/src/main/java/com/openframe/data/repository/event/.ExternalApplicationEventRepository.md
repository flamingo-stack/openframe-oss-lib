A Spring Data MongoDB repository interface for managing external application events with custom query methods for retrieving events by user and time range or by type and metadata tags.

## Key Components

- **ExternalApplicationEventRepository**: Main repository interface extending `MongoRepository<ExternalApplicationEvent, String>`
- **findByUserIdAndTimestampBetween()**: Query method to find events for a specific user within a time range
- **findByTypeAndTags()**: Custom query method using `@Query` annotation to find events by type and metadata tags

## Usage Example

```java
@Autowired
private ExternalApplicationEventRepository eventRepository;

// Find events for a user within a time range
Instant startTime = Instant.now().minus(Duration.ofDays(7));
Instant endTime = Instant.now();
List<ExternalApplicationEvent> userEvents = eventRepository
    .findByUserIdAndTimestampBetween("user123", startTime, endTime);

// Find events by type and specific tags
Map<String, String> tags = Map.of("environment", "prod", "module", "auth");
List<ExternalApplicationEvent> taggedEvents = eventRepository
    .findByTypeAndTags("LOGIN_EVENT", tags);

// Standard MongoDB repository operations
ExternalApplicationEvent event = new ExternalApplicationEvent();
eventRepository.save(event);
```
