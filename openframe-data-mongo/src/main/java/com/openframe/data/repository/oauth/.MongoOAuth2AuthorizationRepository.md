A Spring Data MongoDB repository interface that provides data access operations for OAuth2 authorization records stored in MongoDB.

## Key Components

- **MongoOAuth2AuthorizationRepository** - Repository interface extending `MongoRepository<MongoOAuth2Authorization, String>`
- **findByState()** - Retrieves authorization by OAuth2 state parameter
- **findByAuthorizationCodeValue()** - Finds authorization by authorization code
- **findByAccessTokenValue()** - Locates authorization by access token
- **findByRefreshTokenValue()** - Searches authorization by refresh token

## Usage Example

```java
@Service
public class OAuth2AuthorizationService {
    
    @Autowired
    private MongoOAuth2AuthorizationRepository repository;
    
    public MongoOAuth2Authorization findByAccessToken(String token) {
        return repository.findByAccessTokenValue(token)
            .orElseThrow(() -> new AuthorizationNotFoundException("Invalid access token"));
    }
    
    public void revokeByRefreshToken(String refreshToken) {
        repository.findByRefreshTokenValue(refreshToken)
            .ifPresent(auth -> {
                // Invalidate and save
                auth.setActive(false);
                repository.save(auth);
            });
    }
    
    public MongoOAuth2Authorization validateAuthorizationCode(String code) {
        return repository.findByAuthorizationCodeValue(code)
            .filter(auth -> !auth.isExpired())
            .orElseThrow(() -> new InvalidAuthorizationCodeException("Code expired or invalid"));
    }
}
```

This repository supports the complete OAuth2 authorization flow by providing query methods for all token types and state management, enabling secure token validation and lifecycle management in MongoDB-backed OAuth2 implementations.
