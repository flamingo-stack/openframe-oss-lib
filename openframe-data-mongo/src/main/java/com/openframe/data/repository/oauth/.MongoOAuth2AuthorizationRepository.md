Spring Data MongoDB repository interface for managing OAuth2 authorization documents in MongoDB. Provides specialized query methods for retrieving OAuth2 authorization records by various token and state identifiers.

## Key Components

- **MongoOAuth2AuthorizationRepository** - Repository interface extending `MongoRepository<MongoOAuth2Authorization, String>`
- **findByState()** - Retrieves authorization by OAuth2 state parameter
- **findByAuthorizationCodeValue()** - Finds authorization by authorization code
- **findByAccessTokenValue()** - Queries authorization by access token
- **findByRefreshTokenValue()** - Locates authorization by refresh token

## Usage Example

```java
@Service
public class OAuth2Service {
    
    @Autowired
    private MongoOAuth2AuthorizationRepository authRepository;
    
    public void handleAuthorizationCallback(String state, String code) {
        Optional<MongoOAuth2Authorization> auth = authRepository.findByState(state);
        if (auth.isPresent()) {
            // Process authorization
            MongoOAuth2Authorization authorization = auth.get();
            // Update with authorization code
        }
    }
    
    public boolean validateAccessToken(String token) {
        return authRepository.findByAccessTokenValue(token).isPresent();
    }
    
    public void refreshToken(String refreshToken) {
        authRepository.findByRefreshTokenValue(refreshToken)
            .ifPresent(auth -> {
                // Generate new access token
                authRepository.save(auth);
            });
    }
}
```
