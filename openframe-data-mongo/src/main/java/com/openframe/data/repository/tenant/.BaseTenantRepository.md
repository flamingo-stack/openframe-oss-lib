A technology-agnostic base interface for tenant repository operations that supports both reactive and non-reactive implementations through generic type parameters.

## Key Components

- **Generic Type Parameters**: 
  - `T` - Return type wrapper (Optional for blocking, Mono for reactive)
  - `B` - Boolean return type (boolean for blocking, Mono<Boolean> for reactive)
  - `ID` - Identifier type (String for tenant domains)

- **Core Methods**:
  - `findByDomain(String domain)` - Retrieves tenant by domain name
  - `existsByDomain(String domain)` - Checks tenant existence by domain

## Usage Example

```java
// Blocking implementation
public interface BlockingTenantRepository 
    extends BaseTenantRepository<Optional<Tenant>, Boolean, String> {
    
    @Override
    Optional<Tenant> findByDomain(String domain);
    
    @Override
    Boolean existsByDomain(String domain);
}

// Reactive implementation  
public interface ReactiveTenantRepository 
    extends BaseTenantRepository<Mono<Tenant>, Mono<Boolean>, String> {
    
    @Override
    Mono<Tenant> findByDomain(String domain);
    
    @Override
    Mono<Boolean> existsByDomain(String domain);
}

// Usage in service layer
@Service
public class TenantService {
    
    public void validateTenant(String domain) {
        if (!tenantRepository.existsByDomain(domain)) {
            throw new TenantNotFoundException("Domain not found: " + domain);
        }
    }
}
```
