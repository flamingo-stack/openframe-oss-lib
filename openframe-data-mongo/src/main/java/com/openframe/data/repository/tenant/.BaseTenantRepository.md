A generic base interface that defines common tenant repository operations, designed to work with both blocking and reactive implementations through type parameterization.

## Key Components

- **`BaseTenantRepository<T, B, ID>`** - Generic interface with type parameters for return type wrapper (T), boolean return type (B), and ID type (ID)
- **`findByDomain(String domain)`** - Retrieves a tenant by domain name
- **`existsByDomain(String domain)`** - Checks if a tenant exists with the given domain

## Usage Example

```java
// Blocking implementation
public interface BlockingTenantRepository extends BaseTenantRepository<Optional<Tenant>, Boolean, String> {
    // Inherits methods with concrete types:
    // Optional<Tenant> findByDomain(String domain)
    // Boolean existsByDomain(String domain)
}

// Reactive implementation  
public interface ReactiveTenantRepository extends BaseTenantRepository<Mono<Tenant>, Mono<Boolean>, String> {
    // Inherits methods with reactive types:
    // Mono<Tenant> findByDomain(String domain)
    // Mono<Boolean> existsByDomain(String domain)
}

// Usage
Optional<Tenant> tenant = blockingRepo.findByDomain("example.com");
Mono<Tenant> reactiveTenant = reactiveRepo.findByDomain("example.com");
```

This design allows the same interface contract to be used across different persistence technologies while maintaining type safety.
