A Spring Data MongoDB repository interface for managing Organization documents, providing both standard CRUD operations and custom query methods for organization-specific lookups.

## Key Components

- **OrganizationRepository**: Main repository interface extending `MongoRepository<Organization, String>` and `CustomOrganizationRepository`
- **findByOrganizationId()**: Retrieves organization by unique identifier
- **findByName()**: Finds organization by name
- **findByIsDefaultTrue()**: Locates the default organization
- **findByOrganizationIdIn()**: Batch retrieval of organizations by multiple IDs

## Usage Example

```java
@Autowired
private OrganizationRepository organizationRepository;

// Find organization by ID
Optional<Organization> org = organizationRepository.findByOrganizationId("ORG123");

// Get default organization
Optional<Organization> defaultOrg = organizationRepository.findByIsDefaultTrue();

// Batch load organizations
Set<String> orgIds = Set.of("ORG123", "ORG456", "ORG789");
List<Organization> organizations = organizationRepository.findByOrganizationIdIn(orgIds);

// Standard repository operations
Organization newOrg = new Organization();
organizationRepository.save(newOrg);
```

The repository is conditionally loaded only in servlet-based web applications and supports both individual and batch organization queries for efficient data access patterns.
