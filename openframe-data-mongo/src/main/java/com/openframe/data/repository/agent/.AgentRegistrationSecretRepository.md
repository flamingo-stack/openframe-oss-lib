A Spring Data MongoDB repository interface for managing agent registration secrets, providing database access operations for the `AgentRegistrationSecret` document collection.

## Key Components

- **AgentRegistrationSecretRepository**: Main repository interface extending `MongoRepository<AgentRegistrationSecret, String>`
- **findByActiveTrue()**: Retrieves the currently active registration secret
- **existsAny()**: Custom query method to check if any registration secrets exist in the collection

## Usage Example

```java
@Service
public class AgentRegistrationService {
    
    @Autowired
    private AgentRegistrationSecretRepository secretRepository;
    
    public Optional<AgentRegistrationSecret> getActiveSecret() {
        return secretRepository.findByActiveTrue();
    }
    
    public boolean hasAnySecrets() {
        return secretRepository.existsAny();
    }
    
    public AgentRegistrationSecret createNewSecret(AgentRegistrationSecret secret) {
        return secretRepository.save(secret);
    }
}
```

The repository uses Spring Data's query derivation to automatically implement `findByActiveTrue()` based on the method name, while `existsAny()` uses a custom MongoDB query `"{}"` (empty filter) with the `exists = true` parameter to efficiently check for document existence without retrieving actual data.
