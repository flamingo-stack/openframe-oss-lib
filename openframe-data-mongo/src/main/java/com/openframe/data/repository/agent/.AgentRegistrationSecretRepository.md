A Spring Data MongoDB repository interface for managing agent registration secrets, providing methods to retrieve active secrets and check for document existence.

## Key Components

- **AgentRegistrationSecretRepository**: Repository interface extending `MongoRepository<AgentRegistrationSecret, String>`
- **findByActiveTrue()**: Finds the currently active registration secret
- **existsAny()**: Checks if any registration secret documents exist in the collection

## Usage Example

```java
@Service
public class AgentRegistrationService {
    
    @Autowired
    private AgentRegistrationSecretRepository repository;
    
    public Optional<AgentRegistrationSecret> getCurrentSecret() {
        return repository.findByActiveTrue();
    }
    
    public boolean hasRegistrationSecrets() {
        return repository.existsAny();
    }
    
    public void createSecret(AgentRegistrationSecret secret) {
        repository.save(secret);
    }
}
```

The repository follows Spring Data conventions for automatic query generation based on method names, with the `findByActiveTrue()` method automatically querying documents where the `active` field is `true`. The custom `existsAny()` query uses an empty filter to check if any documents exist in the collection.
