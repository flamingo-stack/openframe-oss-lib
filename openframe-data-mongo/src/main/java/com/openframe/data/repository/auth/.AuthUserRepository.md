A Spring Data MongoDB repository interface for managing authentication user data with support for multi-tenant operations and user status filtering.

## Key Components

- **AuthUserRepository** - Main repository interface extending `MongoRepository<AuthUser, String>`
- **findByEmailAndStatus()** - Retrieves active users by email across all tenants
- **findByEmailAndTenantIdAndStatus()** - Finds active users by email within specific tenant
- **existsByEmailAndTenantId()** - Checks if user exists in specific tenant
- **findByEmailAndTenantId()** - Retrieves users by email and tenant regardless of status

## Usage Example

```java
@Service
public class AuthService {
    
    @Autowired
    private AuthUserRepository authUserRepository;
    
    // Find active user for login
    public Optional<AuthUser> authenticateUser(String email) {
        return authUserRepository.findByEmailAndStatus(email, UserStatus.ACTIVE);
    }
    
    // Multi-tenant user lookup
    public Optional<AuthUser> findTenantUser(String email, String tenantId) {
        return authUserRepository.findByEmailAndTenantIdAndStatus(
            email, tenantId, UserStatus.ACTIVE);
    }
    
    // Check if user already exists before registration
    public boolean userExists(String email, String tenantId) {
        return authUserRepository.existsByEmailAndTenantId(email, tenantId);
    }
}
```
