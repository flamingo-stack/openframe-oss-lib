Custom MongoDB repository implementation that provides advanced querying capabilities for integrated tools with filtering, searching, and sorting functionality.

## Key Components

**Main Methods:**
- `buildToolQuery()` - Constructs MongoDB queries with dynamic filters and text search
- `findToolsWithFilters()` - Executes filtered queries without sorting
- `findToolsWithFiltersAndSort()` - Executes filtered queries with configurable sorting
- `findDistinctTypes/Categories/PlatformCategories()` - Retrieves unique field values
- `isSortableField()` - Validates sortable field names
- `getDefaultSortField()` - Returns default sorting field

**Key Features:**
- Dynamic query building based on `ToolQueryFilter` parameters
- Case-insensitive regex search across name and description fields
- Configurable sorting with fallback to ID field
- Field validation for security and consistency
- Null-safe operations with stream processing

## Usage Example

```java
@Autowired
private CustomIntegratedToolRepository toolRepository;

// Filter and search tools
ToolQueryFilter filter = new ToolQueryFilter();
filter.setEnabled(true);
filter.setType("MSP_TOOL");

List<IntegratedTool> tools = toolRepository.findToolsWithFiltersAndSort(
    filter, 
    "monitoring",  // search term
    "name",        // sort field
    "ASC"          // sort direction
);

// Get distinct categories for filters
List<String> categories = toolRepository.findDistinctCategories();

// Validate sort field
boolean isValid = toolRepository.isSortableField("name"); // true
```
