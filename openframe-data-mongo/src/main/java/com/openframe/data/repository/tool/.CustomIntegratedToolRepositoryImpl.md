Custom MongoDB repository implementation that provides advanced querying capabilities for integrated tools, including dynamic filtering, search functionality, and metadata retrieval operations.

## Key Components

- **buildToolQuery()** - Constructs MongoDB queries dynamically based on filter criteria and search terms
- **findToolsWithFilters()** - Executes filtered tool searches with optional text search across name and description fields
- **findDistinctTypes()** - Retrieves all unique tool types available in the database
- **findDistinctCategories()** - Fetches distinct categories for filtering options
- **findDistinctPlatformCategories()** - Returns unique platform categories for advanced filtering

## Usage Example

```java
@Autowired
private CustomIntegratedToolRepository toolRepository;

// Create filter for enabled development tools
ToolQueryFilter filter = new ToolQueryFilter();
filter.setEnabled(true);
filter.setType("DEVELOPMENT");
filter.setCategory("IDE");

// Search for tools matching "java" in name or description
List<IntegratedTool> tools = toolRepository.findToolsWithFilters(filter, "java");

// Get all available categories for UI dropdown
List<String> categories = toolRepository.findDistinctCategories();

// Build custom query for complex filtering
Query customQuery = toolRepository.buildToolQuery(filter, "spring framework");
```

The implementation uses MongoDB's flexible query building with regex-based case-insensitive search and supports null-safe filtering across multiple tool attributes. All distinct value queries automatically filter null values and return sorted results for consistent UI display.
