Implementation of custom MongoDB repository operations for IntegratedTool entities, providing advanced querying capabilities with dynamic filtering and search functionality.

## Key Components

- **buildToolQuery()** - Constructs MongoDB queries based on filter criteria and search terms
- **findToolsWithFilters()** - Retrieves IntegratedTool documents with applied filters and search
- **findDistinctTypes()** - Returns unique tool types from the collection
- **findDistinctCategories()** - Returns unique categories from the collection
- **findDistinctPlatformCategories()** - Returns unique platform categories from the collection

## Usage Example

```java
@Service
public class ToolService {
    
    @Autowired
    private CustomIntegratedToolRepository toolRepository;
    
    public List<IntegratedTool> searchTools() {
        ToolQueryFilter filter = new ToolQueryFilter();
        filter.setEnabled(true);
        filter.setType("API");
        
        // Find tools with filters and search
        List<IntegratedTool> tools = toolRepository.findToolsWithFilters(filter, "integration");
        
        // Get available filter options
        List<String> types = toolRepository.findDistinctTypes();
        List<String> categories = toolRepository.findDistinctCategories();
        
        return tools;
    }
}
```

The repository supports case-insensitive regex search across tool names and descriptions, with null-safe filtering for enabled status, type, category, and platform category fields.
