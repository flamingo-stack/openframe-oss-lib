A Spring Data MongoDB repository interface for managing installed agent records, providing queries to find agents by machine ID and agent type.

## Key Components

- **InstalledAgentRepository**: Main repository interface extending MongoRepository
- **findByMachineId()**: Retrieves all agents installed on a specific machine
- **findByMachineIdIn()**: Bulk query to find agents across multiple machines
- **findByMachineIdAndAgentType()**: Finds a specific agent type on a given machine

## Usage Example

```java
@Service
public class AgentService {
    
    @Autowired
    private InstalledAgentRepository agentRepository;
    
    public List<InstalledAgent> getAgentsForMachine(String machineId) {
        return agentRepository.findByMachineId(machineId);
    }
    
    public Optional<InstalledAgent> findSpecificAgent(String machineId, String agentType) {
        return agentRepository.findByMachineIdAndAgentType(machineId, agentType);
    }
    
    public List<InstalledAgent> getAgentsForMultipleMachines(List<String> machineIds) {
        return agentRepository.findByMachineIdIn(machineIds);
    }
}
```

The repository automatically inherits standard CRUD operations from MongoRepository for basic create, read, update, and delete functionality.
