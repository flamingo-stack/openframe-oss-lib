A Spring Data MongoDB repository interface that provides data access operations for managing installed agents across different machines in the system.

## Key Components

- **InstalledAgentRepository** - Repository interface extending `MongoRepository<InstalledAgent, String>`
- **findByMachineId()** - Retrieves all agents installed on a specific machine
- **findByMachineIdIn()** - Batch query for agents across multiple machines
- **findByMachineIdAndAgentType()** - Finds a specific agent type on a particular machine

## Usage Example

```java
@Service
public class AgentService {
    
    @Autowired
    private InstalledAgentRepository agentRepository;
    
    public List<InstalledAgent> getAgentsForMachine(String machineId) {
        return agentRepository.findByMachineId(machineId);
    }
    
    public Optional<InstalledAgent> findSpecificAgent(String machineId, String agentType) {
        return agentRepository.findByMachineIdAndAgentType(machineId, agentType);
    }
    
    public List<InstalledAgent> getAgentsForMultipleMachines(List<String> machineIds) {
        return agentRepository.findByMachineIdIn(machineIds);
    }
}
```

This repository provides efficient querying capabilities for tracking which agents are installed on which machines, supporting both individual machine lookups and batch operations for multiple machines.
