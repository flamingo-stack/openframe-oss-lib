A custom MongoDB repository interface for advanced event querying operations that extends beyond standard Spring Data capabilities.

## Key Components

- **`buildEventQuery()`** - Constructs MongoDB Query objects from EventQueryFilter and search terms
- **`findEventsWithCursor()`** - Implements cursor-based pagination for efficient large dataset traversal
- **`findDistinctUserIds()`** - Retrieves unique user identifiers from the event collection
- **`findDistinctEventTypes()`** - Fetches all distinct event type values for filtering purposes

## Usage Example

```java
@Service
public class EventService {
    
    @Autowired
    private CustomEventRepository eventRepository;
    
    public EventResponse getEvents(EventQueryFilter filter, String searchTerm, 
                                 String cursor, int pageSize) {
        // Build complex query with filters and search
        Query query = eventRepository.buildEventQuery(filter, searchTerm);
        
        // Get paginated results with cursor
        List<Event> events = eventRepository.findEventsWithCursor(query, cursor, pageSize);
        
        return new EventResponse(events);
    }
    
    public EventMetadata getEventMetadata() {
        List<String> userIds = eventRepository.findDistinctUserIds();
        List<String> eventTypes = eventRepository.findDistinctEventTypes();
        
        return new EventMetadata(userIds, eventTypes);
    }
}
```

This interface enables complex event queries with custom filtering, cursor-based pagination, and metadata extraction for building dynamic UI components like filter dropdowns.
