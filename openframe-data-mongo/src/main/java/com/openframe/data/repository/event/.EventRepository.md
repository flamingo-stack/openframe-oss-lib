A Spring Data MongoDB repository interface for managing Event documents, combining built-in CRUD operations with custom query capabilities.

## Key Components

- **EventRepository**: Main repository interface extending `MongoRepository<Event, String>`
- **Custom Repository Integration**: Extends `CustomEventRepository` for specialized query methods
- **Generic Type Parameters**: Uses `Event` as entity type and `String` as ID type

## Usage Example

```java
@Service
public class EventService {
    
    @Autowired
    private EventRepository eventRepository;
    
    // Built-in CRUD operations
    public Event saveEvent(Event event) {
        return eventRepository.save(event);
    }
    
    public Optional<Event> findById(String eventId) {
        return eventRepository.findById(eventId);
    }
    
    public List<Event> findAllEvents() {
        return eventRepository.findAll();
    }
    
    public void deleteEvent(String eventId) {
        eventRepository.deleteById(eventId);
    }
    
    // Custom methods from CustomEventRepository
    // (implementation would be in CustomEventRepositoryImpl)
    public List<Event> findEventsByCustomCriteria() {
        return eventRepository.customQueryMethod();
    }
}
```

This repository provides full MongoDB integration for Event entities with both standard Spring Data operations and extensibility through the custom repository pattern.
