A Spring Data MongoDB repository interface for managing Tag documents, providing CRUD operations and custom query methods for tag-related data access.

## Key Components

- **TagRepository Interface**: Extends `MongoRepository<Tag, String>` for basic MongoDB operations
- **findByOrganizationId()**: Retrieves all tags belonging to a specific organization
- **findByNameAndOrganizationId()**: Finds a specific tag by name within an organization
- **findByNameIn()**: Fetches multiple tags by a list of tag names

## Usage Example

```java
@Autowired
private TagRepository tagRepository;

// Get all tags for an organization
List<Tag> orgTags = tagRepository.findByOrganizationId("org123");

// Find specific tag by name and organization
Tag tag = tagRepository.findByNameAndOrganizationId("development", "org123");

// Get multiple tags by names
List<String> tagNames = Arrays.asList("java", "spring", "mongodb");
List<Tag> selectedTags = tagRepository.findByNameIn(tagNames);

// Standard CRUD operations
Tag newTag = new Tag("frontend", "org123");
tagRepository.save(newTag);
tagRepository.deleteById("tagId");
```
