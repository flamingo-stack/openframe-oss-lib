Repository interface for managing Tag documents in MongoDB with organization-scoped operations and custom query methods for tag retrieval.

## Key Components

- **TagRepository Interface**: Extends `MongoRepository<Tag, String>` for CRUD operations
- **findByOrganizationId()**: Retrieves all tags for a specific organization
- **findByNameAndOrganizationId()**: Finds a unique tag by name within an organization
- **findByNameIn()**: Bulk retrieval of tags by multiple names

## Usage Example

```java
@Service
public class TagService {
    
    @Autowired
    private TagRepository tagRepository;
    
    // Get all tags for an organization
    public List<Tag> getOrganizationTags(String orgId) {
        return tagRepository.findByOrganizationId(orgId);
    }
    
    // Find specific tag within organization scope
    public Tag findTag(String tagName, String orgId) {
        return tagRepository.findByNameAndOrganizationId(tagName, orgId);
    }
    
    // Bulk retrieve tags by names
    public List<Tag> getTagsByNames(List<String> tagNames) {
        return tagRepository.findByNameIn(tagNames);
    }
    
    // Standard CRUD operations inherited from MongoRepository
    public Tag saveTag(Tag tag) {
        return tagRepository.save(tag);
    }
}
```

The repository supports organization-level tag isolation, ensuring tags are properly scoped to their respective organizations while providing efficient querying capabilities.
