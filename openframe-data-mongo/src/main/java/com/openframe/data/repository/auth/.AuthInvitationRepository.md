A Spring Data MongoDB repository interface for managing authentication invitation documents with tenant-specific operations and expiration handling.

## Key Components

**Repository Methods:**
- `findByTenantIdAndId()` - Retrieves invitation by tenant and ID
- `findByTenantIdAndEmailAndStatus()` - Finds invitation by tenant, email, and status
- `findByTenantIdAndStatusOrderByCreatedAtDesc()` - Lists invitations by tenant and status, ordered by creation date
- `deleteByTenantIdAndExpiresAtBefore()` - Removes expired invitations for a tenant

**Entity:** Works with `AuthInvitation` documents stored in MongoDB

## Usage Example

```java
@Service
public class InvitationService {
    
    @Autowired
    private AuthInvitationRepository repository;
    
    public Optional<AuthInvitation> findPendingInvitation(String tenantId, String email) {
        return repository.findByTenantIdAndEmailAndStatus(tenantId, email, "PENDING");
    }
    
    public List<AuthInvitation> getActiveInvitations(String tenantId) {
        return repository.findByTenantIdAndStatusOrderByCreatedAtDesc(tenantId, "ACTIVE");
    }
    
    public void cleanupExpiredInvitations(String tenantId) {
        repository.deleteByTenantIdAndExpiresAtBefore(tenantId, Instant.now());
    }
}
```
