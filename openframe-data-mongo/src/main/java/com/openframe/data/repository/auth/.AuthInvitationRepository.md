A Spring Data MongoDB repository interface for managing authentication invitations with tenant-scoped operations and automatic cleanup capabilities.

## Key Components

- **AuthInvitationRepository** - MongoDB repository interface extending `MongoRepository<AuthInvitation, String>`
- **findByTenantIdAndId()** - Retrieves a specific invitation by tenant and ID
- **findByTenantIdAndEmailAndStatus()** - Finds invitation by tenant, email, and status
- **findByTenantIdAndStatusOrderByCreatedAtDesc()** - Gets all invitations for a tenant with specific status, ordered by creation date
- **deleteByTenantIdAndExpiresAtBefore()** - Cleanup method to remove expired invitations for a tenant

## Usage Example

```java
@Service
public class InvitationService {
    @Autowired
    private AuthInvitationRepository invitationRepository;
    
    public Optional<AuthInvitation> getInvitation(String tenantId, String invitationId) {
        return invitationRepository.findByTenantIdAndId(tenantId, invitationId);
    }
    
    public List<AuthInvitation> getPendingInvitations(String tenantId) {
        return invitationRepository.findByTenantIdAndStatusOrderByCreatedAtDesc(
            tenantId, "PENDING"
        );
    }
    
    public void cleanupExpiredInvitations(String tenantId) {
        invitationRepository.deleteByTenantIdAndExpiresAtBefore(
            tenantId, Instant.now()
        );
    }
}
```

This repository provides tenant-isolated invitation management with built-in query methods for common operations like finding pending invitations and cleaning up expired ones.
