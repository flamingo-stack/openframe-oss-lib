A Spring Data MongoDB repository interface for managing integrated tool data with conditional web application activation.

## Key Components

- **IntegratedToolRepository**: Main repository interface extending `MongoRepository` for CRUD operations on `IntegratedTool` entities
- **BaseIntegratedToolRepository**: Base interface providing common repository methods with generic type parameters
- **CustomIntegratedToolRepository**: Custom interface for specialized query operations
- **@ConditionalOnWebApplication**: Annotation ensuring this repository is only active in servlet-based web applications
- **findByType()**: Method to retrieve integrated tools by their type identifier

## Usage Example

```java
@Service
public class ToolService {
    
    @Autowired
    private IntegratedToolRepository toolRepository;
    
    public IntegratedTool findToolByType(String toolType) {
        Optional<IntegratedTool> tool = toolRepository.findByType(toolType);
        return tool.orElse(null);
    }
    
    public IntegratedTool saveTool(IntegratedTool tool) {
        return toolRepository.save(tool);
    }
    
    public List<IntegratedTool> findAllTools() {
        return toolRepository.findAll();
    }
}
```

The repository automatically provides standard MongoDB operations while adding custom query capabilities through its extended interfaces. It's conditionally loaded only in servlet-based web applications.
