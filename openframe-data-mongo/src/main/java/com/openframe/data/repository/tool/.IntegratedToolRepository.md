A MongoDB repository interface for managing `IntegratedTool` documents in web applications, providing CRUD operations and custom query methods.

## Key Components

- **IntegratedToolRepository**: Main repository interface extending MongoDB functionality
- **MongoRepository<IntegratedTool, String>**: Provides standard CRUD operations with String IDs
- **BaseIntegratedToolRepository**: Base interface for common tool repository methods
- **CustomIntegratedToolRepository**: Interface for custom query implementations
- **@ConditionalOnWebApplication**: Ensures repository is only active in servlet-based web applications

## Usage Example

```java
@Service
public class ToolService {
    
    @Autowired
    private IntegratedToolRepository toolRepository;
    
    // Find tool by type
    public Optional<IntegratedTool> getToolByType(String toolType) {
        return toolRepository.findByType(toolType);
    }
    
    // Standard CRUD operations
    public IntegratedTool saveTool(IntegratedTool tool) {
        return toolRepository.save(tool);
    }
    
    public List<IntegratedTool> getAllTools() {
        return toolRepository.findAll();
    }
    
    public void deleteTool(String id) {
        toolRepository.deleteById(id);
    }
}
```

The repository automatically inherits MongoDB operations like `save()`, `findAll()`, `deleteById()`, plus custom methods from the extended interfaces. The conditional annotation ensures it's only instantiated in web application contexts.
