A Spring Data MongoDB custom repository implementation that provides cursor-based pagination and dynamic query building for Machine documents with filtering and search capabilities.

## Key Components

- **findMachinesWithCursor()** - Implements cursor-based pagination using ObjectId cursors with configurable limits and descending sort order
- **buildDeviceQuery()** - Constructs MongoDB queries based on MachineQueryFilter criteria (status, device type, OS type, organization)
- **applySearchCriteria()** - Adds case-insensitive regex search across multiple machine fields (hostname, IP, serial number, etc.)

## Usage Example

```java
@Autowired
private CustomMachineRepository customMachineRepository;

// Build a filtered query
MachineQueryFilter filter = new MachineQueryFilter();
filter.setStatuses(Arrays.asList("ACTIVE", "MAINTENANCE"));
filter.setDeviceTypes(Arrays.asList("SERVER", "WORKSTATION"));

Query query = customMachineRepository.buildDeviceQuery(filter, "dell");

// Execute with cursor pagination
String cursor = "507f1f77bcf86cd799439011"; // Previous page's last item ID
int limit = 20;
List<Machine> machines = customMachineRepository.findMachinesWithCursor(query, cursor, limit);

// For first page, pass null cursor
List<Machine> firstPage = customMachineRepository.findMachinesWithCursor(query, null, limit);
```
