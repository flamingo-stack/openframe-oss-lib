Custom repository interface for organization queries that provides advanced filtering, search capabilities, and cursor-based pagination support for MongoDB operations.

## Key Components

- **Query Building Methods**: `buildOrganizationQuery()` - Creates MongoDB queries with filters and optional cursor pagination
- **Filtered Search**: `findOrganizationsWithFilters()` - Executes database-level filtering and search operations
- **Cursor Pagination**: `findOrganizationsWithCursor()` - Implements efficient pagination using organization IDs as cursors

## Usage Example

```java
@Service
public class OrganizationService {
    
    @Autowired
    private CustomOrganizationRepository organizationRepository;
    
    public List<Organization> searchOrganizations(String searchTerm, int limit) {
        OrganizationQueryFilter filter = new OrganizationQueryFilter();
        filter.setActive(true);
        
        // Build and execute query with filters
        List<Organization> results = organizationRepository
            .findOrganizationsWithFilters(filter, searchTerm);
        
        return results.stream().limit(limit).collect(Collectors.toList());
    }
    
    public List<Organization> getNextPage(String cursor, int pageSize) {
        OrganizationQueryFilter filter = new OrganizationQueryFilter();
        Query query = organizationRepository.buildOrganizationQuery(filter, null, cursor);
        
        return organizationRepository.findOrganizationsWithCursor(query, cursor, pageSize);
    }
}
```
