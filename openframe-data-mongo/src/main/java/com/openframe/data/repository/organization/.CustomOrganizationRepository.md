This interface defines custom repository methods for querying Organization documents in MongoDB with advanced filtering, searching, and cursor-based pagination capabilities.

## Key Components

- **buildOrganizationQuery()** - Creates MongoDB Query objects with filters and search criteria, with optional cursor support for pagination
- **findOrganizationsWithFilters()** - Retrieves organizations using database-level filtering and search
- **findOrganizationsWithCursor()** - Implements cursor-based pagination for efficient large dataset traversal
- **OrganizationQueryFilter** - Filter criteria object for querying organizations
- **Organization** - Document entity representing organization data

## Usage Example

```java
@Autowired
private CustomOrganizationRepository organizationRepository;

// Build query with filters and search
OrganizationQueryFilter filter = new OrganizationQueryFilter();
Query query = organizationRepository.buildOrganizationQuery(filter, "tech");

// Find organizations with filters
List<Organization> orgs = organizationRepository.findOrganizationsWithFilters(filter, "startup");

// Cursor-based pagination
String cursor = "org123";
int limit = 20;
List<Organization> pagedOrgs = organizationRepository.findOrganizationsWithCursor(query, cursor, limit);

// Query with cursor for next page
Query nextPageQuery = organizationRepository.buildOrganizationQuery(filter, "tech", cursor);
```
