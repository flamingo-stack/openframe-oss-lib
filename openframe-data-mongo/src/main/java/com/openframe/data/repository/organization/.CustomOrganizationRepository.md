A custom repository interface for organization data access with advanced filtering, search, and pagination capabilities using MongoDB queries.

## Key Components

- **buildOrganizationQuery()** - Constructs MongoDB Query objects from filters and search terms
- **findOrganizationsWithFilters()** - Executes filtered organization searches
- **findOrganizationsWithCursor()** - Implements cursor-based pagination for large datasets
- **isSortableField()** - Validates field names for sorting operations
- **getDefaultSortField()** - Returns the default field for sorting results

## Usage Example

```java
@Component
public class OrganizationService {
    
    @Autowired
    private CustomOrganizationRepository organizationRepository;
    
    public List<Organization> searchOrganizations(String searchTerm, int limit) {
        OrganizationQueryFilter filter = new OrganizationQueryFilter();
        filter.setActive(true);
        
        return organizationRepository.findOrganizationsWithFilters(filter, searchTerm);
    }
    
    public List<Organization> getNextPage(String cursor, int limit) {
        OrganizationQueryFilter filter = new OrganizationQueryFilter();
        Query query = organizationRepository.buildOrganizationQuery(filter, null);
        
        String sortField = organizationRepository.getDefaultSortField();
        return organizationRepository.findOrganizationsWithCursor(
            query, cursor, limit, sortField, "ASC"
        );
    }
}
```
