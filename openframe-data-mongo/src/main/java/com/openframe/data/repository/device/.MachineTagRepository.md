A Spring Data MongoDB repository interface that manages the many-to-many relationship between machines and tags in the OpenFrame system. It provides methods for querying, creating, and deleting machine-tag associations.

## Key Components

- **MachineTagRepository**: MongoDB repository interface extending `MongoRepository<MachineTag, String>`
- **findByMachineId()**: Retrieves all tags associated with a specific machine
- **findByTagId()**: Retrieves all machines associated with a specific tag
- **deleteByMachineId()**: Removes all tag associations for a machine
- **deleteByMachineIdAndTagId()**: Removes a specific machine-tag association
- **findByMachineIdIn()**: Bulk query for multiple machines' tag associations
- **findByTagIdIn()**: Bulk query for multiple tags' machine associations

## Usage Example

```java
@Service
public class MachineTagService {
    @Autowired
    private MachineTagRepository machineTagRepository;
    
    // Get all tags for a machine
    public List<MachineTag> getMachineTags(String machineId) {
        return machineTagRepository.findByMachineId(machineId);
    }
    
    // Remove specific machine-tag association
    public void removeTagFromMachine(String machineId, String tagId) {
        machineTagRepository.deleteByMachineIdAndTagId(machineId, tagId);
    }
    
    // Clean up all tags when machine is deleted
    public void cleanupMachineTags(String machineId) {
        machineTagRepository.deleteByMachineId(machineId);
    }
    
    // Bulk query for multiple machines
    public List<MachineTag> getTagsForMachines(List<String> machineIds) {
        return machineTagRepository.findByMachineIdIn(machineIds);
    }
}
```
