A Spring Data MongoDB repository interface that manages MachineTag documents, providing CRUD operations and custom queries for machine-tag relationships in a device management system.

## Key Components

- **MachineTagRepository**: MongoDB repository interface extending `MongoRepository<MachineTag, String>`
- **findByMachineId()**: Retrieves all tags associated with a specific machine
- **findByTagId()**: Retrieves all machines associated with a specific tag
- **deleteByMachineId()**: Removes all tag associations for a machine
- **deleteByMachineIdAndTagId()**: Removes a specific machine-tag association
- **findByMachineIdIn()**: Batch retrieval for multiple machine IDs
- **findByTagIdIn()**: Batch retrieval for multiple tag IDs

## Usage Example

```java
@Autowired
private MachineTagRepository machineTagRepository;

// Find all tags for a machine
List<MachineTag> machineTags = machineTagRepository.findByMachineId("machine123");

// Find all machines with a specific tag
List<MachineTag> taggedMachines = machineTagRepository.findByTagId("tag456");

// Remove all tags from a machine
machineTagRepository.deleteByMachineId("machine123");

// Remove specific tag from machine
machineTagRepository.deleteByMachineIdAndTagId("machine123", "tag456");

// Batch operations
List<MachineTag> multipleMachines = machineTagRepository.findByMachineIdIn(
    Arrays.asList("machine1", "machine2", "machine3")
);
```
