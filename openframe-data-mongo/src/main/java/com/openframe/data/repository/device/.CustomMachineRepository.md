A repository interface that extends MongoDB query capabilities for Machine entities, providing custom filtering and cursor-based pagination functionality.

## Key Components

- **`buildDeviceQuery(MachineQueryFilter, String)`** - Constructs MongoDB queries using filter criteria and search terms
- **`findMachinesWithCursor(Query, String, int)`** - Executes paginated queries with cursor-based navigation
- **`MachineQueryFilter`** - Filter object for building targeted machine queries
- **`Machine`** - Document entity representing machine data

## Usage Example

```java
@Service
public class MachineService {
    
    @Autowired
    private CustomMachineRepository machineRepository;
    
    public List<Machine> searchMachines(String searchTerm, String cursor) {
        MachineQueryFilter filter = new MachineQueryFilter();
        filter.setStatus("ACTIVE");
        
        Query query = machineRepository.buildDeviceQuery(filter, searchTerm);
        return machineRepository.findMachinesWithCursor(query, cursor, 20);
    }
}
```

This interface enables flexible machine searches with pagination, allowing implementation classes to define specific MongoDB query building logic and cursor-based result fetching strategies.
