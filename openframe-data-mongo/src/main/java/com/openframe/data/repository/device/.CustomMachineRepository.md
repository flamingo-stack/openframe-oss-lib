A custom repository interface that extends MongoDB query capabilities for Machine documents, providing advanced filtering and cursor-based pagination functionality.

## Key Components

- **`buildDeviceQuery(MachineQueryFilter filter, String search)`** - Constructs a MongoDB Query object using the provided filter criteria and search terms
- **`findMachinesWithCursor(Query query, String cursor, int limit)`** - Executes cursor-based pagination to retrieve Machine documents efficiently

## Usage Example

```java
@Service
public class MachineService {
    
    @Autowired
    private CustomMachineRepository customMachineRepository;
    
    public List<Machine> searchMachines(String searchTerm, String lastCursor) {
        MachineQueryFilter filter = new MachineQueryFilter();
        filter.setStatus("ACTIVE");
        
        Query query = customMachineRepository.buildDeviceQuery(filter, searchTerm);
        return customMachineRepository.findMachinesWithCursor(query, lastCursor, 20);
    }
}
```

This interface is typically implemented by a concrete repository class that handles the MongoDB query construction logic and cursor-based pagination implementation for optimal performance when dealing with large datasets of Machine documents.
