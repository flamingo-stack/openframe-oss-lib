A Spring service for retrieving tool URLs from integrated tools based on their type.

## Key Components

- **`getUrlByToolType(IntegratedTool, ToolUrlType)`** - Finds the first URL matching a specific tool type from an integrated tool's URL collection

## Usage Example

```java
@Autowired
private ToolUrlService toolUrlService;

public void processToolUrl(IntegratedTool tool) {
    // Get API URL for the tool
    Optional<ToolUrl> apiUrl = toolUrlService.getUrlByToolType(tool, ToolUrlType.API);
    
    if (apiUrl.isPresent()) {
        String url = apiUrl.get().getUrl();
        // Process the API URL
        callExternalAPI(url);
    }
    
    // Get documentation URL
    Optional<ToolUrl> docsUrl = toolUrlService.getUrlByToolType(tool, ToolUrlType.DOCUMENTATION);
    docsUrl.ifPresent(toolUrl -> openDocumentation(toolUrl.getUrl()));
}
```

The service uses Stream API to filter through an integrated tool's URL collection, returning the first match for the specified type. Returns an `Optional<ToolUrl>` to handle cases where no URL of the requested type exists.
