A Spring service class for managing integrated tools within the OpenFrame platform, providing CRUD operations and repository access for tool configurations.

## Key Components

- **IntegratedToolService** - Main service class with conditional activation based on configuration properties
- **IntegratedToolRepository** - Injected repository for database operations
- **getAllTools()** - Retrieves all available integrated tools
- **getTool(String toolType)** - Finds a tool by its type identifier
- **getToolById(String toolId)** - Retrieves a specific tool by ID
- **saveTool(IntegratedTool tool)** - Persists or updates a tool configuration

## Usage Example

```java
@Autowired
private IntegratedToolService toolService;

// Get all available tools
List<IntegratedTool> tools = toolService.getAllTools();

// Find a specific tool by type
Optional<IntegratedTool> aiTool = toolService.getTool("AI_ASSISTANT");
if (aiTool.isPresent()) {
    log.info("Found AI tool: {}", aiTool.get().getName());
}

// Save or update a tool configuration
IntegratedTool newTool = new IntegratedTool();
newTool.setType("CUSTOM_TOOL");
newTool.setName("My Custom Tool");
IntegratedTool saved = toolService.saveTool(newTool);
```

The service is conditionally enabled via the `openframe.integration.tool.enabled` property and only activates in servlet-based web applications, making it suitable for microservice architectures where tool integration may be optional.
