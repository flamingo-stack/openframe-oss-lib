A Spring service that manages Organization entities with CRUD operations, soft deletion, and default organization creation for tenant initialization.

## Key Components

- **CRUD Operations**: `getOrganizationById()`, `getOrganizationByOrganizationId()`, `getOrganizationByName()` for retrieval; `createOrganization()`, `updateOrganization()` for persistence
- **Default Organization Management**: `getDefaultOrganization()`, `createDefaultOrganization()` for tenant initialization with fallback organization
- **Safe Deletion**: `deleteOrganization()` with validation to prevent deletion of organizations with associated machines or default organizations
- **Soft Delete Support**: All retrieval methods automatically filter out soft-deleted records
- **Dependencies**: Integrates with `OrganizationRepository` and `MachineRepository`

## Usage Example

```java
@Autowired
private OrganizationService organizationService;

// Create a new organization
Organization org = Organization.builder()
    .name("Engineering Team")
    .organizationId("eng-001")
    .category("Development")
    .build();

Optional<Organization> found = organizationService.getOrganizationByName("Engineering Team");
```
