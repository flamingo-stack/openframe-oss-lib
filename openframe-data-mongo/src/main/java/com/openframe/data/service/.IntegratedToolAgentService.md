Service layer for managing integrated tool agents within the OpenFrame platform. Provides CRUD operations and specialized queries for tool agent configurations.

## Key Components

- **save()** - Persists tool agent configurations
- **getAll()** - Retrieves all tool agents
- **getAllEnabled()** - Fetches only enabled tool agents filtered by status
- **findById()/getById()** - Locates agents by ID with optional vs. required semantics
- **findByReleaseVersionTrue()** - Queries agents marked for release versioning
- **updateVersionForReleaseAgents()** - Bulk updates version numbers for release agents

## Usage Example

```java
@Autowired
private IntegratedToolAgentService agentService;

// Create and save a new tool agent
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setName("Email Integration");
agent.setStatus(ToolAgentStatus.ENABLED);
agent.setReleaseVersion(true);
agentService.save(agent);

// Get all active agents
List<IntegratedToolAgent> enabledAgents = agentService.getAllEnabled();

// Update versions for release agents
agentService.updateVersionForReleaseAgents("v2.1.0");

// Safely retrieve an agent
Optional<IntegratedToolAgent> maybeAgent = agentService.findById("agent-123");
if (maybeAgent.isPresent()) {
    // Process agent
}
```

The service handles tool agent lifecycle management and supports version control workflows for release deployments.
