A Spring service that manages integrated tool agents, providing CRUD operations and version management functionality for tools in the OpenFrame system.

## Key Components

- **save()** - Persists IntegratedToolAgent entities to the database
- **getAll()** - Retrieves all tool agents regardless of status
- **getAllEnabled()** - Fetches only active/enabled tool agents
- **findById()** - Locates a specific agent by ID with Optional wrapper
- **findByReleaseVersionTrue()** - Gets agents marked for release version tracking
- **updateVersionForReleaseAgents()** - Bulk updates version numbers for release-tracked agents

## Usage Example

```java
@Autowired
private IntegratedToolAgentService toolAgentService;

// Create and save a new tool agent
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setName("DocumentProcessor");
agent.setStatus(ToolAgentStatus.ENABLED);
agent.setReleaseVersion(true);
toolAgentService.save(agent);

// Get all active agents
List<IntegratedToolAgent> activeAgents = toolAgentService.getAllEnabled();

// Update version for all release agents
toolAgentService.updateVersionForReleaseAgents("2.1.0");

// Find specific agent
Optional<IntegratedToolAgent> agent = toolAgentService.findById("agent-123");
if (agent.isPresent()) {
    // Process the agent
}
```

The service uses constructor injection with Lombok's `@RequiredArgsConstructor` and includes logging for version update operations.
