A Spring service class that manages CRUD operations and business logic for IntegratedToolAgent entities, providing methods to retrieve, save, and update tool agents with various filtering capabilities.

## Key Components

- **save()** - Persists an IntegratedToolAgent entity to the database
- **getAll()** - Retrieves all tool agents from the repository
- **getAllEnabled()** - Fetches only agents with ENABLED status
- **findById()** - Locates a specific agent by its ID
- **findByReleaseVersionTrue()** - Gets agents marked for release
- **updateVersionForReleaseAgents()** - Batch updates version for all release agents

## Usage Example

```java
@Autowired
private IntegratedToolAgentService toolAgentService;

// Create and save a new tool agent
IntegratedToolAgent agent = new IntegratedToolAgent();
agent.setStatus(ToolAgentStatus.ENABLED);
agent.setReleaseVersion(true);
IntegratedToolAgent savedAgent = toolAgentService.save(agent);

// Retrieve enabled agents
List<IntegratedToolAgent> enabledAgents = toolAgentService.getAllEnabled();

// Update version for all release agents
toolAgentService.updateVersionForReleaseAgents("v2.1.0");

// Find specific agent
Optional<IntegratedToolAgent> agent = toolAgentService.findById("agent-123");
if (agent.isPresent()) {
    // Process found agent
}
```
