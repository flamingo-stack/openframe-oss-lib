Configuration class that disables Spring Boot's default Kafka auto-configuration to allow for custom Kafka setup within the OpenFrame framework.

## Key Components

- **@EnableKafka**: Enables Kafka listener annotated endpoints
- **@EnableAutoConfiguration**: Configures Spring Boot auto-configuration with exclusions
- **KafkaAutoConfiguration exclusion**: Prevents default Kafka configuration from being applied

## Usage Example

```java
// This configuration class is automatically picked up by Spring Boot
// when placed in the component scan path

// Custom Kafka configuration can now be defined separately
@Configuration
public class CustomKafkaProducerConfig {
    
    @Bean
    public ProducerFactory<String, Object> producerFactory() {
        Map<String, Object> configProps = new HashMap<>();
        configProps.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        // Custom producer configuration
        return new DefaultKafkaProducerFactory<>(configProps);
    }
    
    @Bean
    public KafkaTemplate<String, Object> kafkaTemplate() {
        return new KafkaTemplate<>(producerFactory());
    }
}
```

This configuration enables developers to define custom Kafka producers, consumers, and listeners without conflicts from Spring Boot's default Kafka settings.
