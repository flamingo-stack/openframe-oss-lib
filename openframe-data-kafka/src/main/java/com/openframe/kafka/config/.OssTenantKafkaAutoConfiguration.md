Auto-configuration class that sets up Kafka beans for the OSS tenant cluster using Spring Boot's auto-configuration mechanism. Creates all necessary Kafka infrastructure components with string keys and JSON serialization.

## Key Components

- **`ossTenantKafkaProducerFactory`** - Factory for creating Kafka producers with String/JSON serialization
- **`ossTenantKafkaTemplate`** - Spring Kafka template for sending messages
- **`ossTenantKafkaConsumerFactory`** - Factory for creating Kafka consumers with String/JSON deserialization  
- **`ossTenantKafkaListenerContainerFactory`** - Container factory for @KafkaListener annotations with configurable concurrency and acknowledgment modes
- **`ossTenantKafkaProducer`** - High-level message producer wrapper implementing MessageProducer interface

## Usage Example

This configuration is automatically activated when the property is enabled:

```java
// application.yml
spring:
  oss-tenant:
    kafka:
      enabled: true
      bootstrap-servers: localhost:9092
      consumer:
        group-id: oss-tenant-group
      listener:
        concurrency: 3
        ack-mode: RECORD

// Using the auto-configured beans
@Component
public class MyService {
    
    @Autowired
    @Qualifier("ossTenantKafkaProducer")
    private MessageProducer producer;
    
    public void sendMessage(String topic, Object payload) {
        producer.send(topic, payload);
    }
    
    @KafkaListener(topics = "my-topic", 
                   containerFactory = "ossTenantKafkaListenerContainerFactory")
    public void handleMessage(Object message) {
        // Process message
    }
}
```

The configuration automatically applies JSON serialization for values and provides sensible defaults like RECORD acknowledgment mode for reliable message processing.
