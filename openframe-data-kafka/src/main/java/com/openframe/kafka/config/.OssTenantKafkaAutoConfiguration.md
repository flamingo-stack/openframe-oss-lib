Spring Boot auto-configuration class that sets up Kafka components specifically for the OSS (Open Source Software) tenant cluster. Configures producer, consumer, and listener components with JSON serialization and custom property bindings.

## Key Components

- **`ossTenantKafkaProducerFactory`** - Creates ProducerFactory with String keys and JSON value serialization
- **`ossTenantKafkaTemplate`** - Configures KafkaTemplate for sending messages with optional default topic
- **`ossTenantKafkaConsumerFactory`** - Creates ConsumerFactory with String key and JSON value deserialization  
- **`ossTenantKafkaListenerContainerFactory`** - Sets up listener container with configurable concurrency, acknowledgment modes, and polling settings
- **`ossTenantKafkaProducer`** - Wraps KafkaTemplate in a MessageProducer interface for simplified usage

## Usage Example

```java
// Enable OSS tenant Kafka in application.yml
spring:
  oss-tenant:
    kafka:
      enabled: true
      bootstrap-servers: localhost:9092
      
// Inject and use the producer
@Autowired
@Qualifier("ossTenantKafkaProducer")
private MessageProducer kafkaProducer;

// Use the listener factory for consuming
@KafkaListener(topics = "my-topic", 
               containerFactory = "ossTenantKafkaListenerContainerFactory")
public void handleMessage(MyMessage message) {
    // Process message
}
```

The configuration is conditionally activated when `spring.oss-tenant.kafka.enabled=true` and uses the `OssTenantKafkaProperties` for customizing Kafka client settings.
