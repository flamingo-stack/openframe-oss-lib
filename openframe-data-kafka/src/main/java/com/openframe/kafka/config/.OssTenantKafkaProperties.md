Configuration properties class for managing OSS tenant-specific Kafka cluster settings. Extends Spring's standard Kafka properties with tenant-specific configuration options and enable/disable functionality.

## Key Components

- **OssTenantKafkaProperties** - Main configuration class with `@ConfigurationProperties` annotation
- **enabled** - Boolean flag to enable/disable the OSS Kafka cluster (default: true)
- **kafka** - Embedded `KafkaProperties` instance providing access to all standard Kafka configuration options
- **Getters/Setters** - Standard accessors for the configuration properties

## Usage Example

```java
// In application.yml or application.properties
spring:
  oss-tenant:
    enabled: true
    kafka:
      bootstrap-servers: localhost:9092
      consumer:
        group-id: oss-tenant-group
        key-deserializer: org.apache.kafka.common.serialization.StringDeserializer
        value-deserializer: org.apache.kafka.common.serialization.StringDeserializer
      producer:
        key-serializer: org.apache.kafka.common.serialization.StringSerializer
        value-serializer: org.apache.kafka.common.serialization.StringSerializer

// Injecting into a Spring component
@Autowired
private OssTenantKafkaProperties ossTenantKafkaProperties;

public void configureKafka() {
    if (ossTenantKafkaProperties.isEnabled()) {
        KafkaProperties kafkaProps = ossTenantKafkaProperties.getKafka();
        // Use kafkaProps for Kafka client configuration
    }
}
```
