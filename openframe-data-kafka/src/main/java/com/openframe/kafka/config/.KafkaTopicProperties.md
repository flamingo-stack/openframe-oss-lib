Configuration properties class that defines Kafka topic settings for OpenFrame's multi-tenant architecture, supporting automatic topic creation and per-topic partition/replication configuration.

## Key Components

- **`KafkaTopicProperties`** - Main configuration class with Spring Boot configuration properties binding
- **`autoCreate`** - Boolean flag to enable/disable automatic topic creation (defaults to true)
- **`inbound`** - Map of topic configurations for inbound message processing
- **`TopicConfig`** - Nested static class defining individual topic settings (name, partitions, replication factor)

## Usage Example

```java
// Application configuration (application.yml)
openframe:
  oss-tenant:
    kafka:
      topics:
        auto-create: true
        inbound:
          user-events:
            name: "user-events-topic"
            partitions: 3
            replication-factor: 2
          system-logs:
            name: "system-logs-topic"
            partitions: 1
            replication-factor: 1

// Injecting in a service
@Service
public class KafkaTopicService {
    
    @Autowired
    private KafkaTopicProperties topicProperties;
    
    public void configureTopics() {
        if (topicProperties.isAutoCreate()) {
            topicProperties.getInbound().forEach((key, config) -> {
                createTopic(config.getName(), config.getPartitions(), 
                           config.getReplicationFactor());
            });
        }
    }
}
```
