A simple interface that defines standard header constants for Kafka messages, providing a centralized way to reference common message header keys.

## Key Components

- **MESSAGE_TYPE_HEADER** - String constant defining the standard header key "message-type" used to identify the type of Kafka message being sent or received

## Usage Example

```java
import com.openframe.kafka.enumeration.KafkaHeader;
import org.apache.kafka.clients.producer.ProducerRecord;
import org.apache.kafka.common.header.Headers;
import org.apache.kafka.common.header.internals.RecordHeaders;

// Creating a Kafka message with message type header
Headers headers = new RecordHeaders();
headers.add(KafkaHeader.MESSAGE_TYPE_HEADER, "user.created".getBytes());

ProducerRecord<String, String> record = new ProducerRecord<>(
    "user-events", 
    null, 
    "user123", 
    "{\"id\":\"user123\",\"name\":\"John\"}", 
    headers
);

// Reading the message type from received message
String messageType = new String(
    record.headers().lastHeader(KafkaHeader.MESSAGE_TYPE_HEADER).value()
);
```

This interface promotes consistency across the application by ensuring all components use the same header key when working with message types in Kafka communications.
