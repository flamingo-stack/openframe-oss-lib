This interface defines standard header constants used across Kafka messaging operations within the OpenFrame system.

## Key Components

- **MESSAGE_TYPE_HEADER**: A string constant defining the header key "message-type" used to identify the type of Kafka messages being sent or received.

## Usage Example

```java
import com.openframe.kafka.enumeration.KafkaHeader;
import org.apache.kafka.clients.producer.ProducerRecord;
import org.apache.kafka.common.header.Header;
import org.apache.kafka.common.header.internals.RecordHeader;

// Creating a Kafka message with message type header
ProducerRecord<String, String> record = new ProducerRecord<>(
    "my-topic", 
    "message-key", 
    "message-payload"
);

// Adding the message type header
Header messageTypeHeader = new RecordHeader(
    KafkaHeader.MESSAGE_TYPE_HEADER, 
    "USER_CREATED".getBytes()
);
record.headers().add(messageTypeHeader);

// Consuming and reading the header
String messageType = new String(
    record.headers()
          .lastHeader(KafkaHeader.MESSAGE_TYPE_HEADER)
          .value()
);
```

This interface provides a centralized location for Kafka header key definitions, ensuring consistency across the application when working with message metadata.
