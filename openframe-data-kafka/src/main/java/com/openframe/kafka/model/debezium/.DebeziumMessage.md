A generic data model class for deserializing Debezium change data capture (CDC) messages from Kafka topics. This class provides type-safe access to database change events with before/after states and metadata.

## Key Components

- **DebeziumMessage<T>** - Main wrapper class containing the message payload
- **Payload<T>** - Inner class holding the actual change data with before/after states
- **Source** - Nested class containing metadata about the source database and connector
- **Generic type T** - Allows typed access to the specific entity being tracked

## Usage Example

```java
// Deserialize a user change event
DebeziumMessage<User> userChange = objectMapper.readValue(
    kafkaMessage, new TypeReference<DebeziumMessage<User>>() {});

// Access the changed data
User beforeState = userChange.getPayload().getBefore();
User afterState = userChange.getPayload().getAfter();
String operation = userChange.getPayload().getOperation(); // "c", "u", "d"

// Access source metadata
DebeziumMessage.Payload.Source source = userChange.getPayload().getSource();
String tableName = source.getTable();
String database = source.getDatabase();
Long timestamp = source.getTimestamp();

// Handle different operations
switch (operation) {
    case "c": // CREATE
        processNewRecord(afterState);
        break;
    case "u": // UPDATE
        processUpdate(beforeState, afterState);
        break;
    case "d": // DELETE
        processDelete(beforeState);
        break;
}
```
