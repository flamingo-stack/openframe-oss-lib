Represents a Kafka message payload for events originating from integrated MSP tools (e.g., RMM, PSA), carrying normalized event metadata across tenants and devices.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `tenantId` | `String` | Identifies the MSP tenant |
| `toolEventId` | `String` | Unique identifier for the tool-generated event |
| `userId` | `String` | Associated user, if applicable |
| `deviceId` | `String` | Target device identifier |
| `hostname` | `String` | Device hostname |
| `organizationId` | `String` | Client organization identifier |
| `organizationName` | `String` | Human-readable organization name |
| `ingestDay` | `String` | Date partition key for ingestion |
| `toolType` | `String` | Type of integrated tool (e.g., RMM, AV) |
| `eventType` | `String` | Category of the event |
| `severity` | `String` | Event severity level |
| `summary` | `String` | Human-readable event summary |
| `eventTimestamp` | `Long` | Epoch timestamp of the original event |

## Usage Example

```java
IntegratedToolEvent event = new IntegratedToolEvent();
event.setTenantId("tenant-001");
event.setToolEventId("evt-abc123");
event.setDeviceId("dev-xyz789");
event.setHostname("workstation-42");
event.setOrganizationId("org-456");
event.setOrganizationName("Acme Corp");
event.setToolType("RMM");
event.setEventType("ALERT");
event.setSeverity("HIGH");
event.setSummary("Disk usage exceeded 90% threshold");
event.setEventTimestamp(System.currentTimeMillis());
event.setIngestDay("2025-01-15");

kafkaTemplate.send("integrated-tool-events", event.getTenantId(), event);
```

> Implements `KafkaMessage`, making it compatible with OpenFrame's unified Kafka routing and processing pipeline.
