A generic Debezium message implementation that uses `JsonNode` for flexible payload handling, extending the base `DebeziumMessage` class while implementing the `KafkaMessage` interface.

## Key Components

- **CommonDebeziumMessage**: Generic Debezium message class with `JsonNode` payload type
- **Lombok Annotations**: `@SuperBuilder`, `@Data`, `@NoArgsConstructor` for boilerplate code generation
- **Type Parameter**: Uses `JsonNode` allowing flexible JSON structure handling
- **Interface Implementation**: Implements `KafkaMessage` for Kafka message processing compatibility

## Usage Example

```java
// Create a CommonDebeziumMessage using the builder pattern
CommonDebeziumMessage message = CommonDebeziumMessage.builder()
    .payload(jsonNode)  // JsonNode containing the actual data
    .build();

// Process as a generic Kafka message
KafkaMessage kafkaMessage = message;

// Access payload data flexibly
JsonNode data = message.getPayload();
String field = data.get("fieldName").asText();
```

This class serves as a flexible container for Debezium change data capture events where the exact payload structure may vary, making it suitable for generic message processing pipelines that need to handle different database table schemas.
