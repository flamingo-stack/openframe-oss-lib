A generic Debezium message implementation that uses JsonNode for flexible payload handling, extending the base DebeziumMessage class while implementing the KafkaMessage interface.

## Key Components

- **CommonDebeziumMessage**: Main class extending `DebeziumMessage<JsonNode>`
- **JsonNode payload**: Uses Jackson's JsonNode for dynamic JSON data handling
- **Lombok annotations**: Provides builder pattern, getters/setters, and no-args constructor
- **KafkaMessage interface**: Implements standard Kafka message contract

## Usage Example

```java
// Create a CommonDebeziumMessage using the builder pattern
CommonDebeziumMessage message = CommonDebeziumMessage.builder()
    .payload(someJsonNode)
    .build();

// Process the message in a Kafka consumer
@KafkaListener(topics = "debezium-topic")
public void handleDebeziumMessage(CommonDebeziumMessage message) {
    JsonNode payload = message.getPayload();
    // Process the dynamic JSON payload
    String operation = payload.get("op").asText();
    JsonNode beforeData = payload.get("before");
    JsonNode afterData = payload.get("after");
}

// Use as a generic KafkaMessage
KafkaMessage kafkaMessage = new CommonDebeziumMessage();
```

This class is particularly useful when working with Debezium change events where the exact schema may vary or when you need flexible JSON processing capabilities.
