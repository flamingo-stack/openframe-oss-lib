This interface defines the contract for asynchronous and synchronous Kafka message publishing functionality within the OpenFrame framework.

## Key Components

- **`sendAsyncMessage`** - Sends a Kafka message asynchronously, returning a CompletableFuture for non-blocking operations
- **`sendAndAwaitMessage`** - Sends a Kafka message synchronously, blocking until completion
- **Parameters**:
  - `messageDestinationName` - Target Kafka topic name
  - `message` - KafkaMessage payload object
  - `specificKey` - Partition key for message routing

## Usage Example

```java
@Autowired
private MessageProducer messageProducer;

// Async message sending
CompletableFuture<SendResult<String, Object>> future = messageProducer
    .sendAsyncMessage("user-events", kafkaMessage, "user123");

future.thenAccept(result -> {
    System.out.println("Message sent successfully: " + result.getRecordMetadata());
});

// Sync message sending (blocks until complete)
messageProducer.sendAndAwaitMessage("order-events", orderMessage, "order456");
```

This interface provides flexibility for both fire-and-forget scenarios (async) and cases requiring guaranteed delivery confirmation (sync) in Kafka-based messaging systems.
