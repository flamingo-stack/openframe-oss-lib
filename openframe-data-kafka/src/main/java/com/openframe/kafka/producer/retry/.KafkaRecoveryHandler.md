A functional interface for handling failed Kafka message publishing attempts by defining recovery strategies when messages cannot be sent to Kafka topics.

## Key Components

- **`enqueue()`** - Core recovery method that processes failed message publishing attempts with full context including the exception, target topic, message key, and payload data

## Usage Example

```java
// Implementation for dead letter queue recovery
public class DeadLetterRecoveryHandler implements KafkaRecoveryHandler {
    
    @Override
    public void enqueue(Throwable ex, String topic, String key, Object payload) {
        // Log the error
        log.error("Failed to publish to topic: {}, key: {}", topic, key, ex);
        
        // Send to dead letter topic
        deadLetterProducer.send(topic + "-dlq", key, payload);
        
        // Store for retry queue
        retryQueue.add(new FailedMessage(topic, key, payload, ex));
    }
}

// Usage with Kafka producer retry mechanism
KafkaRecoveryHandler recoveryHandler = new DeadLetterRecoveryHandler();
// Handler will be invoked when all retry attempts are exhausted
```
