Implements Kafka message recovery handling by logging failed message details for debugging and monitoring purposes.

## Key Components

- **KafkaRecoveryHandlerImpl** - Service class that implements `KafkaRecoveryHandler` interface
- **enqueue()** - Core method that processes failed Kafka messages by logging comprehensive error information
- **ObjectMapper** - Jackson mapper instance for JSON processing (though currently unused)

## Usage Example

```java
@Autowired
private KafkaRecoveryHandler recoveryHandler;

// When Kafka message fails after retries
try {
    kafkaTemplate.send(topic, key, message);
} catch (Exception ex) {
    // Recovery handler logs the failure details
    recoveryHandler.enqueue(ex, "user-events", "user123", userEvent);
}

// Example log output:
// ERROR - Kafka RECOVER invoked: topic=user-events key=user123 
//         errorClass=org.springframework.kafka.KafkaException 
//         errorMsg=Failed to send message payload~={"userId":"user123","action":"login"}
```

The handler creates structured error logs containing the topic, key, exception details, and payload for failed messages, enabling effective monitoring and debugging of Kafka delivery issues.
