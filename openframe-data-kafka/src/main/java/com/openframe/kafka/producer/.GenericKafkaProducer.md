A Kafka producer abstraction that provides both asynchronous and synchronous message sending capabilities with comprehensive error handling and logging.

## Key Components

- **sendAsync()** - Non-blocking message sending that returns a CompletableFuture with success/failure logging
- **sendAndAwait()** - Blocking message sending with exception classification for retry strategies
- **NonRetryableKafkaException** - Thrown for fatal errors (authorization, serialization, invalid topics)
- **TransientKafkaSendException** - Thrown for temporary errors that can be retried
- **Error Classification** - Automatically categorizes Kafka exceptions as retryable or fatal
- **Security Features** - Key redaction and payload truncation for safe logging

## Usage Example

```java
@Service
public class OrderProducer extends GenericKafkaProducer {
    
    public OrderProducer(KafkaTemplate<String, Object> kafkaTemplate) {
        super(kafkaTemplate);
    }
    
    // Fire-and-forget async sending
    public CompletableFuture<SendResult<String, Object>> publishOrderAsync(Order order) {
        return sendAsync("orders", order.getId(), order);
    }
    
    // Blocking send with retry support (use with @Retryable)
    @Retryable(value = TransientKafkaSendException.class, maxAttempts = 3)
    public void publishOrderSync(Order order) {
        sendAndAwait("orders", order.getId(), order);
    }
}
```
