A generic base class for Kafka message producers that provides both synchronous and asynchronous sending capabilities with comprehensive error handling, logging, and payload redaction for security.

## Key Components

- **sendAsync()** - Non-blocking message sending that returns a CompletableFuture with callback-based logging
- **sendAndAwait()** - Blocking message sending with classified exception handling for retry strategies
- **Error Classification** - Separates fatal errors (NonRetryableKafkaException) from transient ones (TransientKafkaSendException)
- **Security Features** - Automatic payload truncation and key redaction in logs
- **Exception Handling** - Pattern matching for different Kafka exception types with appropriate logging levels

## Usage Example

```java
@Component
public class OrderEventProducer extends GenericKafkaProducer {
    
    public OrderEventProducer(KafkaTemplate<String, Object> kafkaTemplate) {
        super(kafkaTemplate);
    }
    
    // Fire-and-forget approach
    public void publishOrderCreated(String orderId, OrderEvent event) {
        sendAsync("order-events", orderId, event)
            .thenAccept(result -> log.info("Order event sent successfully"))
            .exceptionally(ex -> {
                log.error("Failed to send order event", ex);
                return null;
            });
    }
    
    // Blocking approach with retry support
    @Retryable(value = TransientKafkaSendException.class, maxAttempts = 3)
    public void publishCriticalOrder(String orderId, OrderEvent event) {
        sendAndAwait("critical-orders", orderId, event);
    }
}
```
