A Spring service that provides retry functionality for OSS tenant Kafka message production, conditionally enabled through configuration properties.

## Key Components

- **OssTenantRetryingKafkaProducer** - Main service class extending `BaseRetryingKafkaProducer`
- **@ConditionalOnProperty** - Enables the service only when `spring.oss-tenant.kafka.enabled=true`
- **MessageProducer** - Injected OSS tenant-specific Kafka producer
- **KafkaRecoveryHandler** - Handles failed message recovery logic

## Usage Example

```java
// Configuration to enable the service
# application.yml
spring:
  oss-tenant:
    kafka:
      enabled: true

// The service is automatically registered when conditions are met
@Autowired
private OssTenantRetryingKafkaProducer producer;

// Usage inherits from BaseRetryingKafkaProducer
public void sendMessage() {
    // Retry logic is handled by the base class
    producer.send("topic", "key", messagePayload);
}
```

This implementation follows the decorator pattern, wrapping the base OSS tenant Kafka producer with retry capabilities while maintaining conditional instantiation based on feature flags.
