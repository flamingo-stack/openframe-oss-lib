A Spring service that implements Kafka message producing with retry functionality specifically for OSS tenant operations, conditionally enabled via configuration properties.

## Key Components

- **OssTenantRetryingKafkaProducer** - Main service class extending `BaseRetryingKafkaProducer`
- **@ConditionalOnProperty** - Spring annotation enabling the service only when `spring.oss-tenant.kafka.enabled=true`
- **Constructor dependencies**:
  - `MessageProducer ossTenantKafkaProducer` - The underlying Kafka producer
  - `KafkaRecoveryHandler recoveryHandler` - Handles retry failure scenarios

## Usage Example

To enable this producer in your application:

```java
# application.properties
spring.oss-tenant.kafka.enabled=true
```

The service will be automatically instantiated by Spring when the condition is met:

```java
@Autowired
private OssTenantRetryingKafkaProducer producer;

// The producer inherits retry functionality from BaseRetryingKafkaProducer
// and delegates actual message production to the injected ossTenantKafkaProducer
```

This implementation follows the decorator pattern, adding retry capabilities to the base OSS tenant Kafka producer while maintaining the same interface through inheritance.
