A specialized Kafka producer implementation for OSS tenant operations, extending GenericKafkaProducer to provide asynchronous and synchronous message publishing capabilities.

## Key Components

- **OssTenantKafkaProducer** - Main producer class extending `GenericKafkaProducer` and implementing `MessageProducer`
- **sendAsyncMessage()** - Sends messages asynchronously returning a `CompletableFuture`
- **sendAndAwaitMessage()** - Sends messages synchronously with blocking behavior
- **Constructor** - Accepts a configured `KafkaTemplate<String, Object>` for OSS tenant operations

## Usage Example

```java
// Initialize the producer with a Kafka template
KafkaTemplate<String, Object> template = // ... configured template
OssTenantKafkaProducer producer = new OssTenantKafkaProducer(template);

// Send message asynchronously
KafkaMessage message = new KafkaMessage(/* message data */);
CompletableFuture<SendResult<String, Object>> future = 
    producer.sendAsyncMessage("tenant-topic", message, "tenant-123");

future.thenAccept(result -> 
    System.out.println("Message sent successfully: " + result.getRecordMetadata()));

// Send message synchronously
producer.sendAndAwaitMessage("tenant-topic", message, "tenant-123");
```

This producer provides a tenant-specific abstraction over Kafka messaging operations, allowing applications to send messages with both asynchronous and synchronous patterns while maintaining type safety with `KafkaMessage` objects.
