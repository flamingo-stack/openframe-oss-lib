A specialized Kafka producer implementation for OSS (Open Source Software) tenant messaging that extends the generic producer with tenant-specific functionality.

## Key Components

- **OssTenantKafkaProducer** - Main producer class that extends `GenericKafkaProducer` and implements `MessageProducer`
- **sendAsyncMessage()** - Sends messages asynchronously and returns a `CompletableFuture`
- **sendAndAwaitMessage()** - Sends messages synchronously and waits for completion
- **Constructor** - Accepts a `KafkaTemplate<String, Object>` for OSS tenant operations

## Usage Example

```java
// Initialize the producer with a KafkaTemplate
KafkaTemplate<String, Object> ossTenantTemplate = new KafkaTemplate<>(producerFactory);
OssTenantKafkaProducer producer = new OssTenantKafkaProducer(ossTenantTemplate);

// Send message asynchronously
KafkaMessage message = new KafkaMessage("Hello World", metadata);
CompletableFuture<SendResult<String, Object>> future = 
    producer.sendAsyncMessage("oss-tenant-topic", message, "tenant-123");

future.thenAccept(result -> 
    System.out.println("Message sent successfully: " + result.getProducerRecord())
);

// Send message synchronously
producer.sendAndAwaitMessage("oss-tenant-notifications", message, "tenant-456");
```

This producer provides a clean abstraction for OSS tenant-specific Kafka messaging operations while leveraging the base functionality from `GenericKafkaProducer`.
