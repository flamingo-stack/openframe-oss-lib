Enum defining the available data enrichment service types used for categorizing enrichment operations within the OpenFrame data pipeline.

## Key Components

| Enum Constant | Description |
|---|---|
| `INTEGRATED_TOOLS_EVENTS` | Represents enrichment operations sourced from integrated tool events across the MSP platform |

## Usage Example

```java
DataEnrichmentServiceType serviceType = DataEnrichmentServiceType.INTEGRATED_TOOLS_EVENTS;

switch (serviceType) {
    case INTEGRATED_TOOLS_EVENTS:
        // Handle enrichment from integrated tool events
        break;
}
```

> **Note:** This enum is expected to grow as additional enrichment service types are introduced to support new integrations within the OpenFrame platform.
