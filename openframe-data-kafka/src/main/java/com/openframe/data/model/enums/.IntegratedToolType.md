Enum defining the supported integrated tool types within the OpenFrame platform, mapping each tool to its corresponding database identifier string.

## Key Components

| Member | DB Name | Description |
|--------|---------|-------------|
| `MESHCENTRAL` | `"meshcentral"` | MeshCentral remote monitoring/management tool |
| `TACTICAL` | `"tacticalrmm"` | TacticalRMM remote monitoring and management |
| `FLEET` | `"fleet-mdm"` | Fleet MDM mobile device management |

**Methods:**

- `getDbName()` — Returns the database-persisted string identifier for the enum constant.

## Usage Example

```java
// Retrieve the DB name for a given tool type
IntegratedToolType tool = IntegratedToolType.TACTICAL;
String dbIdentifier = tool.getDbName(); // Returns "tacticalrmm"

// Common use case: persisting or querying by tool type
String query = "SELECT * FROM integrations WHERE tool_type = ?";
preparedStatement.setString(1, IntegratedToolType.FLEET.getDbName()); // "fleet-mdm"

// Switching on tool type
switch (tool) {
    case MESHCENTRAL -> handleMeshCentral();
    case TACTICAL    -> handleTacticalRMM();
    case FLEET       -> handleFleetMDM();
}
```

> **Note:** Always use `getDbName()` when persisting to or querying the database, rather than `name()` or `toString()`, to ensure consistency with the stored identifier format.
