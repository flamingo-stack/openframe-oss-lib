Defines the `MessageType` enum, which classifies incoming integration events by their source tool and routing metadata within the OpenFrame data pipeline.

## Key Components

| Field | Type | Description |
|---|---|---|
| `integratedToolType` | `IntegratedToolType` | Source MSP tool (e.g., MeshCentral, Fleet MDM) |
| `dataEnrichmentServiceType` | `DataEnrichmentServiceType` | Enrichment pipeline stage handling the event |
| `destinationList` | `List<Destination>` | Storage/streaming targets (Cassandra, Kafka) |
| `eventHandlerType` | `EventHandlerType` | Processing strategy applied to the event |

### Enum Constants

| Constant | Source Tool |
|---|---|
| `MESHCENTRAL_EVENT` | MeshCentral |
| `FLEET_MDM_EVENT` | Fleet MDM |
| `FLEET_MDM_QUERY_RESULT_EVENT` | Fleet MDM |
| `FLEET_MDM_POLICY_ACTIVITY_EVENT` | Fleet MDM |
| `FLEET_MDM_POLICY_MEMBERSHIP_EVENT` | Fleet MDM |

All constants route to both `CASSANDRA` and `KAFKA` destinations using the `COMMON_TYPE` event handler.

## Usage Example

```java
MessageType type = MessageType.FLEET_MDM_EVENT;

// Access routing metadata
IntegratedToolType tool = type.getIntegratedToolType();       // FLEET
List<Destination> destinations = type.getDestinationList();   // [CASSANDRA, KAFKA]
EventHandlerType handler = type.getEventHandlerType();        // COMMON_TYPE

// Pattern-match on type for dispatch logic
switch (type) {
    case MESHCENTRAL_EVENT -> processMeshCentralEvent(payload);
    case FLEET_MDM_POLICY_ACTIVITY_EVENT -> processPolicyEvent(payload);
    default -> processCommonEvent(payload);
}
```
