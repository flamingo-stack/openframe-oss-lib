Spring Boot configuration properties class that binds feature flag settings from the `openframe` namespace in application configuration files to a typed map.

## Key Components

| Component | Description |
|-----------|-------------|
| `@ConfigurationProperties(prefix = "openframe")` | Binds properties under the `openframe.*` namespace |
| `feFeatureFlag` | `Map<String, Boolean>` holding feature flag names and their enabled/disabled state |
| `@Data` | Lombok annotation generating getters, setters, `equals`, `hashCode`, and `toString` |

## Usage Example

Define flags in `application.yml`:

```yaml
openframe:
  fe-feature-flag:
    new-dashboard: true
    beta-reporting: false
    dark-mode: true
```

Inject and read flags in a Spring component:

```java
@Service
@RequiredArgsConstructor
public class FeatureFlagService {

    private final FeFeatureFlagProperties featureFlagProperties;

    public boolean isEnabled(String flagName) {
        return featureFlagProperties.getFeFeatureFlag()
                .getOrDefault(flagName, false);
    }
}
```

Enable binding in your Spring Boot app:

```java
@SpringBootApplication
@EnableConfigurationProperties(FeFeatureFlagProperties.class)
public class OpenFrameApplication { ... }
```
