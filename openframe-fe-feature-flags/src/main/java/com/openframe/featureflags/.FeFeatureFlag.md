A simple data model representing a feature flag with a name identifier and enabled/disabled state, used within the OpenFrame feature flags system.

## Key Components

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | Unique identifier for the feature flag |
| `enabled` | `boolean` | Toggle state of the feature flag |

Lombok annotations provide:
- `@Data` — getters, setters, `equals`, `hashCode`, and `toString`
- `@Builder` — fluent builder pattern
- `@NoArgsConstructor` / `@AllArgsConstructor` — default and full constructors

## Usage Example

```java
// Using the builder pattern
FeFeatureFlag flag = FeFeatureFlag.builder()
    .name("ai-assistant-enabled")
    .enabled(true)
    .build();

// Using the all-args constructor
FeFeatureFlag flag = new FeFeatureFlag("dark-mode", false);

// Checking flag state
if (flag.isEnabled()) {
    // activate feature
}
```
